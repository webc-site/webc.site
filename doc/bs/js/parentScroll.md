# parentScroll.js

Nabavite prethodni element kontejnera za pomicanje

## default(el)

Krećite se prema gore od navedenog elementa, pronađite i vratite element spremnika za pomicanje koji ga sadrži (kao što je `C-VS` ugrađeni spremnik za pomicanje u ShadowRoot)

parametar:
- `el` : početni DOM element

Povratna vrijednost: DOM element spremnika za pomicanje
