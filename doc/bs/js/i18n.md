# i18n.js

Višejezično upravljanje statusom i praćenje promjena

## LANG_LI

Lista dostupnih jezičkih konfiguracija

## lang()

Preuzmite ID trenutnog jezika

Povratna vrijednost: ID trenutnog jezika

## langCode()

Preuzmite trenutni kôd jezika

Povratna vrijednost: trenutni kod jezika, kao što je `"zh"`

## onLang(func)

Povežite povratni poziv slušanja za promjene jezika

parametar:
- `func` : funkcija povratnog poziva kada se promijeni jezik
  - parametar
    - `code` : Kôd jezika
    - `id` : ID jezika

Povratna vrijednost: Nevezana funkcija čišćenja

## langSet(id)

Prebacite i emitujte trenutni jezik

parametar:
- `id` : ID ciljnog jezika
