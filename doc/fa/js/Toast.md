# Toast.js

اعلان پیام پاپ آپ نان تست

## toast(render, timeout)

نمایش پنجره بازشوی اعلان استاندارد

پارامتر:
- `render` : عملکرد رندر که برای سفارشی کردن محتوای پاپ آپ استفاده می شود
  - پارامتر
    - `el` : عنصر DOM پنجره بازشو
- `timeout` : ثانیه تاخیر برای خاموش شدن خودکار، پیش‌فرض 9، بدون خاموش شدن خودکار در زمان 0

مقدار بازگشتی: عنصر DOM پنجره بازشو
