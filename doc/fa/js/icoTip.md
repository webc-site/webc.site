# icoTip.js

یک جزء نماد را با نکات بین المللی تعریف و ثبت کنید

## default(i18n_url, tag, key)

یک جزء عنصر HTML سفارشی را با حباب‌های بین‌المللی‌سازی ثبت کنید

پارامتر:
- `i18n_url` : درخواست آدرس برای پیکربندی بین المللی
- `tag` : نام تگ عنصر HTML را سفارشی کنید
- `key` : مقدار کلیدی متن درخواستی در فرهنگ لغت بین المللی سازی
