# md/streamRender.js

رندر جریان Markdown

## default(container, source)

پخش جریانی متن Markdown در ظرف DOM مشخص شده، پشتیبانی از جلوه های مکان نما ماشین تحریر و پیمایش خودکار

پارامتر:
- `container` : عنصر ظرف DOM میزبان محتوای ارائه شده
- `source` : منبع داده متن ReadableStream

ارزش برگشتی:

- `cancel` : عملکردی برای توقف رندر و خواندن جریان داده
- `setStop` : عملکردی برای تنظیم توقف موقت پیمایش خودکار
  - پارامتر
    - `val` : مقدار بولی که باید متوقف شود یا خیر
