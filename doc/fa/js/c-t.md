# c-t.js

بین المللی سازی HTML

تگ `<c-t>key</c-t>` سفارشی شده است و `key` کلید بسته زبان است.

## cTranSet(el, genUrl)

متن را برای `c-t` تنظیم کنید.

پارامتر:
- `el` : عنصر ریشه، سراسری را می توان در `document` ارسال کرد
- `genUrl` : تابعی برای به دست آوردن فرهنگ لغت بسته زبان
  - پارامتر
    - `code` : کد زبان
  - مقدار بازگشتی: یک شیء فرهنگ لغت بسته زبان یا یک Promise که به این شیء حل می شود
