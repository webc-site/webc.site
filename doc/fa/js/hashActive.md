# hashActive.js

برجسته کردن پیوند ناوبری بر اساس لنگرهای URL و پیمایش درگاه دید

## default(link_li, cls)

بر اساس موقعیت پیمایش صفحه و تغییرات نقطه لنگر URL، پیوندهای پیمایش مربوط به ناحیه قابل مشاهده فعلی را به طور خودکار برجسته کنید.

پارامتر:
- `link_li` : لیست عناصر DOM پیوند پیمایش
- `cls` : نام کلاس CSS فعال، پیش‌فرض `"A"`

مقدار برگشتی: تمام عملکردهای پاکسازی گوش دادن را لغو کنید
