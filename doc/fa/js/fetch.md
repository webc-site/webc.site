# fetch.js

عملکرد ابزار درخواست شبکه بر اساس کپسولاسیون واکشی

## F(url, opt)

یک درخواست شبکه ارسال کنید و یک شی پاسخ را برگردانید

پارامتر:
- `url` : درخواست آدرس
- `opt` : `fetch` گزینه‌ها شیء هستند

مقدار بازگشتی: شی پاسخ

## f(attr)

یک تابع مرتبه بالاتر که یک درخواست را بسته بندی می کند و محتوای برگشتی را در قالب مشخصی تجزیه می کند

پارامتر:
- `attr` : نام ویژگی تجزیه بدن پاسخ

مقدار بازگشتی: تابع درخواست و تجزیه محتوا به صورت ناهمزمان
  - پارامتر
    - `url` : درخواست آدرس
    - `opt` : `fetch` گزینه‌ها شیء هستند
  - مقدار بازگشتی: محتوای تجزیه شده

## fTxt

یک درخواست شبکه ارسال کنید و بدنه پاسخ را در قالب متن برگردانید

پارامتر:
- `url` : درخواست آدرس
- `opt` : `fetch` گزینه‌ها شیء هستند

مقدار بازگشتی: متن پاسخ

## fJson(url, opt)

یک درخواست شبکه ارسال کنید و بدنه پاسخ را در قالب JSON تجزیه و برگردانید

پارامتر:
- `url` : درخواست آدرس
- `opt` : `fetch` گزینه‌ها شیء هستند

مقدار بازگشتی: شیء JSON تجزیه شده

## fBin(url, opt)

یک درخواست شبکه ارسال کنید و بدنه پاسخ را به صورت یک آرایه بایت باینری برگردانید

پارامتر:
- `url` : درخواست آدرس
- `opt` : `fetch` گزینه‌ها شیء هستند

مقدار بازگشتی: آرایه بایت
