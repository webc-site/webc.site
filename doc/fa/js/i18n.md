# i18n.js

مدیریت وضعیت چند زبانه و نظارت بر تغییرات

## LANG_LI

لیست تنظیمات زبان موجود

## lang()

شناسه زبان فعلی را دریافت کنید

مقدار بازگشتی: شناسه زبان فعلی

## langCode()

کد زبان فعلی را دریافت کنید

مقدار بازگشتی: کد زبان فعلی، مانند `"zh"`

## onLang(func)

برای تغییرات زبان، تماس پاسخگوی گوش دادن را متصل کنید

پارامتر:
- `func` : عملکرد برگشت به تماس زمانی که زبان تغییر می کند
  - پارامتر
    - `code` : کد زبان
    - `id` : شناسه زبان

مقدار بازگشتی: تابع پاکسازی بدون محدودیت

## langSet(id)

زبان فعلی را تغییر دهید و پخش کنید

پارامتر:
- `id` : شناسه زبان مقصد
