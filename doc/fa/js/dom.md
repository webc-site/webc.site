# dom.js

اختصارات متغیر و توابع میانبر را مستند کنید

## D

`document` اشاره میانبر به شی

## B

`document.body` اشاره میانبر به شی

## newEl

عناصر DOM را ایجاد کنید

پارامتر:
- `tagName` : نام تگ عنصر

مقدار بازگشتی: عنصر DOM جدید ایجاد شده

## newLi(...tag_li)

چندین عنصر DOM را به صورت دسته ای ایجاد کنید

پارامتر:
- `tag_li` : فهرست نام برچسب

ارزش برگشتی:
- آرایه حاوی تمام عناصر DOM جدید ایجاد شده است

## cE(name, cls)

عناصر سفارشی Web Components را ثبت کنید

پارامتر:
- `name` : نام عنصر سفارشی، پیشوند `c-` به طور خودکار اضافه می شود
- `cls` : کلاس عنصر سفارشی
