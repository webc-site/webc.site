# fetchLang.js

بسته‌های زبان را به‌طور ناهمزمان بارگیری کنید و تماس‌های به‌روزرسانی را پیوند دهید

## fetchLang(getDict)

یک شنونده بسته زبان ایجاد کنید تا به صورت ناهمزمان بسته زبان را بدست آورید و هنگامی که زبان تغییر می کند، یک بازخوانی مجدد اجرا کنید.

پارامتر:
- `getDict` : تابع ناهمزمان بسته زبان را دریافت کنید
  - پارامتر
    - `code` : کد زبان
  - ارزش بازگشتی: فرهنگ لغت بسته زبان

مقدار بازگشتی: تابعی برای ثبت پاسخ تماس مجدد
- پارامتر
  - `refresh` : وقتی زبان به روز می شود، عملکرد پاسخ به تماس را تازه کنید
    - پارامتر
      - `i18n` : شیء فرهنگ لغت بسته زبان
      - `code` : کد زبان فعلی
- مقدار بازگشتی: عملکردی برای لغو ثبت مجدد تماس مجدد
