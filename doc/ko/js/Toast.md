# Toast.js

토스트 팝업 메시지 프롬프트

## toast(render, timeout)

표준 프롬프트 팝업 창 표시

매개변수:
- `render` : 팝업 콘텐츠를 사용자 정의하는 데 사용되는 렌더링 기능
  - 매개변수
    - `el` : 팝업 창의 DOM 요소
- `timeout` : 자동 종료 지연 초, 기본값 9, 0일 때 자동 종료 없음

반환값: 팝업창의 DOM 요소

## toastErr(render, timeout)

오류 메시지 팝업 표시

매개변수:
- `render` : 팝업 콘텐츠를 사용자 정의하는 데 사용되는 렌더링 기능
  - 매개변수
    - `el` : 팝업 창의 DOM 요소
- `timeout` : 자동 종료 지연 초, 기본값 9, 0일 때 자동 종료 없음

반환 값 : 오류 팝업창의 DOM 요소
