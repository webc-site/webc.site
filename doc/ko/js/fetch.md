# fetch.js

Fetch Encapsulation 기반의 네트워크 요청 도구 기능

## F(url, opt)

네트워크 요청을 보내고 응답 객체를 반환합니다.

매개변수:
- `url` : 요청 주소
- `opt` : `fetch` 옵션 객체

반환값: 응답 객체

## f(attr)

요청을 래핑하고 반환된 콘텐츠를 지정된 형식으로 구문 분석하는 고차 함수

매개변수:
- `attr` : 응답 본문 구문 분석 속성 이름

반환 값: 콘텐츠를 비동기적으로 요청하고 구문 분석하는 함수
  - 매개변수
    - `url` : 요청 주소
    - `opt` : `fetch` 옵션 객체
  - 반환 값: 구문 분석된 콘텐츠

## fTxt

네트워크 요청을 보내고 응답 본문을 텍스트 형식으로 반환합니다.

매개변수:
- `url` : 요청 주소
- `opt` : `fetch` 옵션 객체

반환 값: 응답 텍스트

## fJson(url, opt)

네트워크 요청을 보내고 응답 본문을 JSON 형식으로 구문 분석하여 반환합니다.

매개변수:
- `url` : 요청 주소
- `opt` : `fetch` 옵션 개체

반환 값: 구문 분석된 JSON 개체

## fBin(url, opt)

네트워크 요청을 보내고 응답 본문을 이진 바이트 배열로 반환합니다.

매개변수:
- `url` : 요청 주소
- `opt` : `fetch` 옵션 개체

반환 값: 바이트 배열
