# dom.js

문서 변수 약어 및 단축키 기능

## D

`document` 개체에 대한 바로가기 참조

## B

`document.body` 개체에 대한 바로가기 참조

## newEl

DOM 요소 생성

매개변수:
- `tagName` : 요소 태그 이름

반환 값: 새로 생성된 DOM 요소

## newLi(...tag_li)

여러 DOM 요소를 일괄적으로 생성

매개변수:
- `tag_li` : 태그 이름 목록

반환 값:
- 새로 생성된 모든 DOM 요소를 포함하는 배열

## cE(name, cls)

웹 구성 요소 사용자 정의 요소 등록

매개변수:
- `name` : 맞춤 요소 이름, `c-` 접두사가 자동으로 추가됩니다.
- `cls` : 맞춤 요소의 클래스
