# route.js

단일 페이지 프런트엔드 라우팅 관리

## nowUrl()

선행 슬래시가 제거된 현재 URL 경로를 가져옵니다.

반환 값: 현재 경로 문자열

## route(hook)

경로 변경에 대한 수신 콜백을 등록합니다. 등록시 현재 경로로 1회 실행됩니다.

매개변수:
- `hook` : 경로 모니터링 콜백 기능
  - 매개변수
    - `url` : 현재 경로
    - `pre` : 마지막 경로

반환 값: 등록되지 않은 정리 함수

## setPre(url)

마지막으로 기록된 URL을 저장하세요

매개변수:
- `url` : 경로 문자열

## preUrl()

마지막 레코드의 URL을 가져옵니다.

반환 값: 마지막 경로 문자열

## refresh(url)

모든 경로 수신 콜백이 지정된 경로에서 강제로 트리거되고 사전 업데이트가 이루어지도록 합니다.

매개변수:
- `url` : 대상 경로, 기본 현재 경로

## removeSlash(url)

경로 헤더에서 슬래시 제거

매개변수:
- `url` : 경로 문자열

반환 값: 선행 슬래시를 제거한 후의 경로

## setUrl(url)

브라우저 주소 표시줄 URL 상태 업데이트

매개변수:
- `url` : 대상 경로

반환 값: 경로가 실제로 변경되는 경우 1

## goto(url)

브라우저 주소 업데이트 및 경로 새로 고침 트리거

매개변수:
- `url` : 대상 경로
