# i18n.js

다국어 상태 관리 및 변경 모니터링

## LANG_LI

사용 가능한 언어 구성 목록

## lang()

현재 언어 ID 가져오기

반환 값: 현재 언어 ID

## langCode()

현재 언어 코드 가져오기

반환 값: `"zh"` 등의 현재 언어 코드

## onLang(func)

언어 변경에 대한 청취 콜백 바인딩

매개변수:
- `func` : 언어 변경 시 콜백 함수
  - 매개변수
    - `code` : 언어 코드
    - `id` : 언어 ID

반환값: 언바운드 정리 함수

## langSet(id)

현재 언어를 전환하고 방송하세요

매개변수:
- `id` : 대상 언어 ID
