# fetchLang.js

언어 팩을 비동기식으로 로드하고 업데이트 콜백을 바인드합니다.

## fetchLang(getDict)

언어 팩을 비동기적으로 가져오고 언어가 변경되면 새로 고침 콜백을 실행하는 언어 팩 수신기를 만듭니다.

매개변수:
- `getDict` : 언어 패키지의 비동기 기능을 가져옵니다.
  - 매개변수
    - `code` : 언어 코드
  - 반환 값: 언어 팩 사전

반환값: 새로고침 콜백을 등록하는 함수
- 매개변수
  - `refresh` : 언어 업데이트 시 콜백 새로고침 기능
    - 매개변수
      - `i18n` : 언어 팩 사전 개체
      - `code` : 현재 언어 코드
- 반환값: 새로고침 콜백 등록을 취소하는 함수
