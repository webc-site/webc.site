# i18n.js

Būsenos valdymas ir pokyčių stebėjimas keliomis kalbomis

## LANG_LI

Galimų kalbos konfigūracijų sąrašas

## lang()

Gaukite dabartinės kalbos ID

Grąžinama vertė: dabartinės kalbos ID

## langCode()

Gaukite dabartinės kalbos kodą

Grąžinama vertė: dabartinis kalbos kodas, pvz., `"zh"`

## onLang(func)

Susiekite klausymosi atgalinį skambutį, kad pakeistumėte kalbą

parametras:
- `func` : atgalinio skambinimo funkcija, kai pasikeičia kalba
  - parametras
    - `code` : kalbos kodas
    - `id` : kalbos ID

Grąžinama vertė: neapribota valymo funkcija

## langSet(id)

Perjunkite ir transliuokite dabartinę kalbą

parametras:
- `id` : tikslinės kalbos ID
