# dom/header.js

Sukurkite antraštės juostos DOM struktūrą

## default()

Sukurkite ir sujunkite antraštės DOM struktūrą iš antraštės, h1 ir b žymų

Grąžinimo vertė:

- `header` : antraštės juostos sudėtinio rodinio elementas
- `h1` : pavadinimo elementas
- `b` : subtitrai arba papildomos informacijos elementas
