# hashActive.js

Naršymo nuorodų paryškinimas, pagrįstas URL inkarais ir peržiūros srities slinkimu

## default(link_li, cls)

Automatiškai paryškinkite naršymo nuorodas, atitinkančias šiuo metu matomą sritį, atsižvelgdami į puslapio slinkimo padėtį ir URL inkaro taško pasikeitimus.

parametras:
- `link_li` : naršymo nuorodos DOM elementų sąrašas
- `cls` : aktyvios CSS klasės pavadinimas, numatytasis `"A"`

Grąžinama vertė: atšaukti visas klausymosi valymo funkcijas
