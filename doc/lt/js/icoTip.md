# icoTip.js

Apibrėžkite ir užregistruokite piktogramos komponentą su internacionalizavimo patarimais

## default(i18n_url, tag, key)

Užregistruokite tinkintą HTML elemento komponentą su internacionalizavimo raginimo burbulais

parametras:
- `i18n_url` : Internacionalizavimo konfigūracijos adreso užklausa
- `tag` : tinkinkite HTML elemento žymos pavadinimą
- `key` : pagrindinė raginimo teksto reikšmė internacionalizacijos žodyne
