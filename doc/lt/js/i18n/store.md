# i18n/store.js

Daugiakalbis vietinės saugyklos valdymas

## get()

Gaukite numatytąjį kalbos ID, apskaičiuotą vietoje talpyklos arba apskaičiuotą pagal naršyklės kalbos atitiktį

Grąžinama vertė: kalbos ID

## set(id)

Išsaugokite dabartinės kalbos ID talpykloje vietoje

parametras:
- `id` : kalbos ID
