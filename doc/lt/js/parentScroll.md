# parentScroll.js

Gaukite protėvių slinkties konteinerio elementą

## default(el)

Keliaukite aukštyn nuo nurodyto elemento, suraskite ir grąžinkite slinkties konteinerio elementą, kuriame jis yra (pvz., `C-VS`įtaisytasis slinkties konteineris „ShadowRoot“)

parametras:
- `el` : pradedantis DOM elementas

Grąžinama vertė: slinkties konteinerio DOM elementas
