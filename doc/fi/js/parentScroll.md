# parentScroll.js

Hanki esivanhempien vierityssäiliöelementti

## default(el)

Siirry määritetystä elementistä ylöspäin, etsi ja palauta sen sisältävä vierityssäiliöelementti (kuten `C-VS`:n ShadowRootin sisäänrakennettu vierityssäiliö)

parametri:
- `el` : DOM-elementin aloitus

Palautusarvo: vierityssäiliön DOM-elementti
