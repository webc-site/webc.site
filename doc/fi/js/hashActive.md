# hashActive.js

Navigointilinkin korostus URL-ankkureiden ja näkymän vierityksen perusteella

## default(link_li, cls)

Korosta automaattisesti tällä hetkellä näkyvää aluetta vastaavat navigointilinkit sivun vierityspaikan ja URL-ankkuripisteiden muutosten perusteella.

parametri:
- `link_li` : Navigointilinkin DOM-elementtiluettelo
- `cls` : Aktiivinen CSS-luokan nimi, oletusarvo `"A"`

Palautusarvo: Peruuta kaikki kuuntelun puhdistustoiminnot
