# i18n.js

Monikielinen tilanhallinta ja muutosten seuranta

## LANG_LI

Luettelo käytettävissä olevista kielikokoonpanoista

## lang()

Hanki nykyinen kielitunnus

Palautusarvo: nykyinen kielen tunnus

## langCode()

Hanki nykyinen kielikoodi

Palautusarvo: nykyinen kielikoodi, kuten `"zh"`

## onLang(func)

Sido kuuntelusoitto kielimuutoksia varten

parametri:
- `func` : takaisinsoittotoiminto, kun kieli vaihtuu
  - parametri
    - `code` : Kielikoodi
    - `id` : Kielitunnus

Palautusarvo: Siivoamaton puhdistustoiminto

## langSet(id)

Vaihda ja lähetä nykyinen kieli

parametri:
- `id` : Kohdekielen tunnus
