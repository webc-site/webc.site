# dom.js

စာရွက်စာတမ်း ပြောင်းလဲနိုင်သော အတိုကောက်များနှင့် ဖြတ်လမ်းလုပ်ဆောင်ချက်များ

## D

`document`အရာဝတ္ထုကို ရည်ညွှန်းသည့် ဖြတ်လမ်းလင့်ခ်

## B

`document.body`အရာဝတ္ထုကို ရည်ညွှန်းသည့် ဖြတ်လမ်းလင့်ခ်

## css(txt, root)

## newEl

DOM အစိတ်အပိုင်းများကို ဖန်တီးပါ။

ကန့်သတ်ချက်-
- `tagName` : အစိတ်အပိုင်း တဂ်အမည်

ပြန်ပို့တန်ဖိုး- အသစ်ဖန်တီးထားသော DOM ဒြပ်စင်

## newLi(...tag_li)

အစုအဝေးတွင် DOM အစိတ်အပိုင်းများစွာကို ဖန်တီးပါ။

ကန့်သတ်ချက်-
- `tag_li` : တဂ်အမည်စာရင်း

ပြန်ပို့တန်ဖိုး-
- အသစ်ဖန်တီးထားသော DOM ဒြပ်စင်များ ပါဝင်သော အခင်းအကျင်း

## cE(name, cls)

Web Components စိတ်ကြိုက်ဒြပ်စင်များကို စာရင်းသွင်းပါ။

ကန့်သတ်ချက်-
- `name` : စိတ်ကြိုက်ဒြပ်စင်အမည်၊ `c-` ရှေ့ဆက်ကို အလိုအလျောက် ထည့်သွင်းပါမည်
- `cls` : စိတ်ကြိုက်ဒြပ်စင်၏ အတန်း
