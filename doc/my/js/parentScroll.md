# parentScroll.js

ဘိုးဘေးလိပ်ကွန်တိန်နာဒြပ်စင်ကိုရယူပါ။

## default(el)

သတ်မှတ်ထားသော ဒြပ်စင်မှ အပေါ်ဘက်သို့ ကွေ့ကာ ၎င်းပါရှိသော လှိမ့်ကွန်တိန်နာဒြပ်စင်ကို ရှာပြီး ပြန်ပို့ပါ (ဥပမာ `C-VS`၏ ShadowRoot ပါ၀င်သော လှိမ့်ပုံကွန်တိန်နာ ကဲ့သို့)

ကန့်သတ်ချက်-
- `el` : DOM အစိတ်အပိုင်းကို စတင်သည်။

ပြန်ပို့သည့်တန်ဖိုး- ကွန်တိန်နာ DOM ဒြပ်စင်ကို ရွှေ့ပါ။
