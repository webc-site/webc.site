# i18n.js

ဘာသာစကားပေါင်းစုံ အခြေအနေကို စီမံခန့်ခွဲခြင်းနှင့် ပြောင်းလဲမှု စောင့်ကြည့်ခြင်း။

## LANG_LI

ရရှိနိုင်သော ဘာသာစကားဖွဲ့စည်းပုံများစာရင်း

## lang()

လက်ရှိ ဘာသာစကား ID ကို ရယူပါ။

ပြန်ပို့တန်ဖိုး- လက်ရှိ ဘာသာစကား ID

## langCode()

လက်ရှိဘာသာစကားကုဒ်ကို ရယူပါ။

ပြန်ပေးသည့်တန်ဖိုး- `"zh"` ကဲ့သို့သော လက်ရှိဘာသာစကားကုဒ်

## onLang(func)

ဘာသာစကားပြောင်းလဲမှုများအတွက် နားထောင်ခြင်းခေါ်ဆိုမှုအား ချည်နှောင်ပါ။

ကန့်သတ်ချက်-
- `func` : ဘာသာစကား ပြောင်းလဲသောအခါ ပြန်ခေါ်သည့် လုပ်ဆောင်ချက်
  - ကန့်သတ်ချက်
    - `code` : ဘာသာစကားကုဒ်
    - `id` : ဘာသာစကား ID

ပြန်ပေးတန်ဖိုး- Unbound ရှင်းလင်းရေး လုပ်ဆောင်ချက်

## langSet(id)

လက်ရှိဘာသာစကားကို ပြောင်းပြီး ထုတ်လွှင့်ပါ။

ကန့်သတ်ချက်-
- `id` : ပစ်မှတ် ဘာသာစကား ID
