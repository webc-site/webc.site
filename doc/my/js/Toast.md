# Toast.js

ပေါ်လာသော မက်ဆေ့ချ်ကို ဆန္ဒပြုပါ။

## toast(render, timeout)

စံသတ်မှတ်ချက် ပေါ်လာသော ဝင်းဒိုးကို ပြပါ။

ကန့်သတ်ချက်-
- `render` : တင်ဆက်ခြင်းလုပ်ဆောင်ချက်၊ ပေါ့ပ်အပ်အကြောင်းအရာကို စိတ်ကြိုက်ပြင်ဆင်ရန် အသုံးပြုသည်။
  - ကန့်သတ်ချက်
    - `el` : ပေါ့ပ်အပ်ဝင်းဒိုး၏ DOM အစိတ်အပိုင်း
- `timeout` : အလိုအလျောက်ပိတ်ခြင်းအတွက် စက္ကန့်နှောင့်နှေးခြင်း၊ မူရင်း 9၊ 0 တွင် အလိုအလျောက်ပိတ်ခြင်း မရှိပါ

ပြန်ပို့သည့်တန်ဖိုး- ပေါ့ပ်အပ်ဝင်းဒိုး၏ DOM အစိတ်အပိုင်း

## toastErr(render, timeout)

ကန့်သတ်ချက်-
- `render` : တင်ဆက်ခြင်းလုပ်ဆောင်ချက်၊ ပေါ့ပ်အပ်အကြောင်းအရာကို စိတ်ကြိုက်ပြင်ဆင်ရန် အသုံးပြုသည်။
  - ကန့်သတ်ချက်
    - `el` : ပေါ့ပ်အပ်ဝင်းဒိုး၏ DOM အစိတ်အပိုင်း
- `timeout` : အလိုအလျောက်ပိတ်ခြင်းအတွက် စက္ကန့်နှောင့်နှေးခြင်း၊ မူရင်း 9၊ 0 တွင် အလိုအလျောက်ပိတ်ခြင်း မရှိပါ

ပြန်ပို့သည့်တန်ဖိုး- အမှားအယွင်း ပေါ်လာသောဝင်းဒိုး၏ DOM အစိတ်အပိုင်း
