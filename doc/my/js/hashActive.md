# hashActive.js

URL anchors နှင့် viewport scrolling ကို အခြေခံ၍ အသားပေးဖော်ပြသော လမ်းကြောင်းပြလင့်ခ်

## default(link_li, cls)

စာမျက်နှာ scroll position နှင့် URL anchor point အပြောင်းအလဲများအပေါ် အခြေခံ၍ လက်ရှိမြင်နေရသော ဧရိယာနှင့် သက်ဆိုင်သည့် လမ်းကြောင်းပြလင့်ခ်များကို အလိုအလျောက် မီးမောင်းထိုးပြပါ။

ကန့်သတ်ချက်-
- `link_li` : လမ်းညွှန်လင့်ခ် DOM ဒြပ်စင်စာရင်း
- `cls` : အသုံးပြုနေသော CSS အတန်းအမည်၊ မူရင်း `"A"`

ပြန်ပို့သည့်တန်ဖိုး- နားထောင်မှုရှင်းလင်းရေးလုပ်ဆောင်ချက်အားလုံးကို ပယ်ဖျက်ပါ။
