# On.js

Batch binding event ကို စောင့်ကြည့်ခြင်း။

## On(elem, dict)

DOM ဒြပ်စင် ဖြစ်ရပ် နားထောင်သူများကို အစုအဝေးတွင် စည်းပါ။

ကန့်သတ်ချက်-
- `elem` : DOM အစိတ်အပိုင်း
- `dict` : အစီအစဉ်များနှင့် ပြန်ခေါ်မှုများကို ပုံဖော်ခြင်း။

ပြန်ပို့သည့်တန်ဖိုး- ချည်နှောင်ထားသော အဖြစ်အပျက် နားထောင်သူ လုပ်ဆောင်ချက်အားလုံးကို ဖယ်ရှားပါ။
