# fetch.js

fetch encapsulation ကို အခြေခံ၍ ကွန်ရက် တောင်းဆိုမှု ကိရိယာ လုပ်ဆောင်ချက်

## F(url, opt)

ကွန်ရက်တောင်းဆိုချက်ကို ပေးပို့ပြီး တုံ့ပြန်မှုအရာဝတ္တုကို ပြန်ပို့ပါ။

ကန့်သတ်ချက်-
- `url` : လိပ်စာ တောင်းဆိုပါ။
- `opt` : `fetch` ရွေးချယ်စရာများ အရာဝတ္ထု

ပြန်ပို့တန်ဖိုး- တုံ့ပြန်မှု အရာဝတ္ထု

## f(attr)

တောင်းဆိုချက်တစ်ခုကို ခြုံငုံပြီး ပြန်ပေးထားသော အကြောင်းအရာကို သတ်မှတ်ထားသော ဖော်မတ်ဖြင့် ခွဲခြမ်းစိတ်ဖြာသည့် အဆင့်မြင့် အမှာစာတစ်ခု

ကန့်သတ်ချက်-
- `attr` : တုံ့ပြန်မှုကိုယ်ထည် ပိုင်းခြားစိတ်ဖြာမှု ရည်ညွှန်းချက် အမည်

ပြန်ပေးသည့်တန်ဖိုး- အကြောင်းအရာကို တပြိုင်တည်း တောင်းဆိုရန်နှင့် ခွဲခြမ်းစိတ်ဖြာရန် လုပ်ဆောင်ချက်
  - ကန့်သတ်ချက်
    - `url` : လိပ်စာ တောင်းဆိုပါ။
    - `opt` : `fetch` ရွေးချယ်စရာများ အရာဝတ္ထု
  - ပြန်ပို့တန်ဖိုး- ပိုင်းခြားထားသော အကြောင်းအရာ

## fTxt

ကွန်ရက်တောင်းဆိုချက်ကို ပေးပို့ပြီး တုံ့ပြန်မှုကိုယ်ထည်ကို စာသားဖော်မတ်ဖြင့် ပြန်ပေးသည်။

ကန့်သတ်ချက်-
- `url` : လိပ်စာ တောင်းဆိုပါ။
- `opt` : `fetch` ရွေးချယ်စရာများ အရာဝတ္ထု

ပြန်ပို့တန်ဖိုး- တုံ့ပြန်ချက်စာသား

## fJson(url, opt)

ကွန်ရက်တောင်းဆိုမှုကို ပေးပို့ပြီး ခွဲခြမ်းစိတ်ဖြာပြီး တုံ့ပြန်မှုကိုယ်ထည်ကို JSON ဖော်မတ်ဖြင့် ပြန်ပေးသည်။

ကန့်သတ်ချက်-
- `url` : လိပ်စာ တောင်းဆိုပါ။
- `opt` : `fetch` ရွေးချယ်စရာများ အရာဝတ္ထု

ပြန်ပို့တန်ဖိုး- JSON အရာဝတ္ထုကို ပိုင်းခြားထားသည်။

## fBin(url, opt)

ကွန်ရက်တောင်းဆိုချက်ကို ပေးပို့ပြီး တုံ့ပြန်မှုကိုယ်ထည်အား binary byte ခင်းကျင်းမှုအဖြစ် ပြန်ပေးသည်။

ကန့်သတ်ချက်-
- `url` : လိပ်စာ တောင်းဆိုပါ။
- `opt` : `fetch` ရွေးချယ်စရာများ အရာဝတ္ထု

ပြန်ပို့တန်ဖိုး- ဘိုက်အခင်း
