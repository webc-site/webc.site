# icoTip.js

နိုင်ငံတကာပြုခြင်းဆိုင်ရာ အရိပ်အမြွက်များဖြင့် အိုင်ကွန်အစိတ်အပိုင်းတစ်ခုကို သတ်မှတ်ပြီး မှတ်ပုံတင်ပါ။

## default(i18n_url, tag, key)

နိုင်ငံတကာအသွင်ကူးပြောင်းမှု အပေးအယူစနစ် ပူဖောင်းများဖြင့် စိတ်ကြိုက် HTML အစိတ်အပိုင်းတစ်ခုကို မှတ်ပုံတင်ပါ။

ကန့်သတ်ချက်-
- `i18n_url` : နိုင်ငံတကာအသွင်ကူးပြောင်းမှုအတွက် လိပ်စာတောင်းဆိုပါ။
- `tag` : HTML အစိတ်အပိုင်း၏ တဂ်အမည်ကို စိတ်ကြိုက်ပြင်ဆင်ပါ။
- `key` : နိုင်ငံတကာအသွင်ပြောင်းအဘိဓာန်ရှိ အချက်ပြစာသား၏ အဓိကတန်ဖိုး
