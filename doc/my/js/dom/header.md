# dom/header.js

ခေါင်းစီးဘား DOM ဖွဲ့စည်းပုံကို ဖန်တီးပါ။

## default()

ခေါင်းစီး၊ h1 နှင့် b တဂ်များ၏ ခေါင်းစီး DOM ဖွဲ့စည်းပုံကို ဖန်တီးပြီး ပေါင်းစပ်ပါ။

ပြန်ပို့တန်ဖိုး-

- `header` : ခေါင်းစီးဘား၏ ကွန်တိန်နာအစိတ်အပိုင်း
- `h1` : ခေါင်းစဉ် အစိတ်အပိုင်း
- `b` : စာတန်းထိုး သို့မဟုတ် နောက်ထပ် အချက်အလက် အစိတ်အပိုင်း
