# fetchLang.js

ဘာသာစကားပက်ကေ့ခ်ျများကို အပြိုင်အဆိုင်ဖွင့်ပြီး အပ်ဒိတ်ပြန်ခေါ်မှုများကို ချည်နှောင်ပါ။

## fetchLang(getDict)

ဘာသာစကားပက်ကေ့ဂျ်ကို တပြိုင်တည်းရယူရန်နှင့် ဘာသာစကားပြောင်းလဲသည့်အခါ ပြန်လည်ဆန်းသစ်သောပြန်ခေါ်မှုကို လုပ်ဆောင်ရန် ဘာသာစကား pack နားထောင်သူအား ဖန်တီးပါ

ကန့်သတ်ချက်-
- `getDict` : ဘာသာစကားပက်ကေ့ဂျ်၏ မညီညွှတ်သောလုပ်ဆောင်ချက်ကို ရယူပါ။
  - ကန့်သတ်ချက်
    - `code` : ဘာသာစကားကုဒ်
  - ပြန်ပေးတန်ဖိုး- ဘာသာစကား အတွဲ အဘိဓာန်

Return value- ပြန်လည်စတင်ရန် ဖုန်းခေါ်ဆိုမှုကို မှတ်ပုံတင်ရန် လုပ်ဆောင်ချက်
- ကန့်သတ်ချက်
  - `refresh` : ဘာသာစကား အပ်ဒိတ်လုပ်သောအခါ ပြန်ခေါ်သည့် လုပ်ဆောင်ချက်ကို ပြန်လည်စတင်ပါ။
    - ကန့်သတ်ချက်
      - `i18n` : ဘာသာစကားအစုံ အဘိဓာန် အရာဝတ္ထု
      - `code` : လက်ရှိ ဘာသာစကားကုဒ်
- ပြန်ပို့သည့်တန်ဖိုး- ပြန်လည်ဆန်းသစ်ခေါ်ဆိုမှုကို မှတ်ပုံတင်ခြင်းမှ ပယ်ဖျက်ရန် လုပ်ဆောင်ချက်
