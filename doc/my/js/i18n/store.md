# i18n/store.js

ဘာသာစကားမျိုးစုံဖြင့် ဒေသတွင်းသိုလှောင်မှုစီမံခန့်ခွဲမှု

## get()

စက်တွင်း ကက်ရှ်လုပ်ထားသော သို့မဟုတ် ဘရောင်ဇာဘာသာစကားဖြင့် တွက်ချက်ထားသော မူရင်းဘာသာစကား ID ကို ရယူပါ။

ပြန်ပို့တန်ဖိုး- ဘာသာစကား ID

## set(id)

လက်ရှိ ဘာသာစကား ID ကို စက်တွင်းတွင် သိမ်းဆည်းပါ။

ကန့်သတ်ချက်-
- `id` : ဘာသာစကား ID
