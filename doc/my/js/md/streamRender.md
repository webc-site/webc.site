# md/streamRender.js

Markdown streaming rendering

## default(container, source)

သတ်မှတ်ထားသော DOM ကွန်တိန်နာထဲသို့ Markdown စာသားကို လွှင့်ထုတ်ခြင်း၊ လက်နှိပ်စက် cursor သက်ရောက်မှုများနှင့် အလိုအလျောက်လှိမ့်ခြင်းကို ပံ့ပိုးပေးခြင်း

ကန့်သတ်ချက်-
- `container` : DOM ကွန်တိန်နာဒြပ်စင်သည် ပြန်ဆိုထားသော အကြောင်းအရာကို လက်ခံဆောင်ရွက်ပေးသည်။
- `source` : ReadableStream စာသားဒေတာအရင်းအမြစ်

ပြန်ပို့တန်ဖိုး-

- `cancel` : တင်ဆက်ခြင်းနှင့် ဒေတာစီးကြောင်းဖတ်ရှုခြင်းကို ရပ်ရန် လုပ်ဆောင်ချက်
- `setStop` : အလိုအလျောက်လှိမ့်ခြင်းကို ခေတ္တရပ်ရန် သတ်မှတ်ရန် လုပ်ဆောင်ချက်
  - ကန့်သတ်ချက်
    - `val` : ခေတ္တရပ်သည်ဖြစ်စေ မရပ်သည်ဖြစ်စေ Boolean တန်ဖိုး
