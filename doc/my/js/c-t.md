# c-t.js

HTML နိုင်ငံတကာသို့ ပြောင်းလဲခြင်း

တဂ် `<c-t>key</c-t>`ကို စိတ်ကြိုက်ပြင်ဆင်ထားပြီး `key` သည် ဘာသာစကားထုပ်၏ သော့ဖြစ်သည်။

## cTranSet(el, genUrl)

`c-t`အတွက် စာသားကို သတ်မှတ်ပါ။

ကန့်သတ်ချက်-
- `el` : root ဒြပ်စင်၊ ဂလိုဘယ်ကို `document` တွင် ဖြတ်သန်းနိုင်သည်။
- `genUrl` : ဘာသာစကား pack အဘိဓာန်ကို ရယူရန် လုပ်ဆောင်ချက်
  - ကန့်သတ်ချက်
    - `code` : ဘာသာစကားကုဒ်
  - ပြန်ပေးသည့်တန်ဖိုး- ဘာသာစကား pack အဘိဓာန်အရာဝတ္ထု သို့မဟုတ် ဤအရာဝတ္တုကို ဖြေရှင်းပေးသည့် ကတိတစ်ခု
