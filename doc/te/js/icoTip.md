# icoTip.js

అంతర్జాతీయీకరణ సూచనలతో ఐకాన్ కాంపోనెంట్‌ను నిర్వచించండి మరియు నమోదు చేయండి

## default(i18n_url, tag, key)

అంతర్జాతీయీకరణ ప్రాంప్ట్ బబుల్‌లతో అనుకూల HTML మూలకం భాగాన్ని నమోదు చేయండి

పరామితి:
- `i18n_url` : అంతర్జాతీయీకరణ కాన్ఫిగరేషన్ కోసం అభ్యర్థన చిరునామా
- `tag` : HTML మూలకం యొక్క ట్యాగ్ పేరును అనుకూలీకరించండి
- `key` : అంతర్జాతీయీకరణ నిఘంటువులో ప్రాంప్ట్ టెక్స్ట్ యొక్క కీలక విలువ
