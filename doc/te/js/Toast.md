# Toast.js

టోస్ట్ పాప్-అప్ సందేశ ప్రాంప్ట్

## toast(render, timeout)

ప్రామాణిక ప్రాంప్ట్ పాప్-అప్ విండోను చూపు

పరామితి:
- `render` : రెండరింగ్ ఫంక్షన్, పాప్-అప్ కంటెంట్‌ని అనుకూలీకరించడానికి ఉపయోగించబడుతుంది
  - పరామితి
    - `el` : పాప్-అప్ విండో యొక్క DOM మూలకం
- `timeout` : ఆటోమేటిక్ షట్‌డౌన్ కోసం సెకన్లు ఆలస్యం, డిఫాల్ట్ 9, 0 ఉన్నప్పుడు ఆటోమేటిక్ షట్‌డౌన్ ఉండదు

రిటర్న్ విలువ: పాప్-అప్ విండో యొక్క DOM మూలకం

- `timeout` : ఆటోమేటిక్ షట్‌డౌన్ కోసం సెకన్లు ఆలస్యం, డిఫాల్ట్ 9, 0 ఉన్నప్పుడు ఆటోమేటిక్ షట్‌డౌన్ ఉండదు
