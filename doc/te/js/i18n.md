# i18n.js

బహుళ భాషా స్థితి నిర్వహణ మరియు మార్పు పర్యవేక్షణ

## LANG_LI

అందుబాటులో ఉన్న భాషా కాన్ఫిగరేషన్‌ల జాబితా

## lang()

ప్రస్తుత భాషా IDని పొందండి

రిటర్న్ విలువ: ప్రస్తుత భాష ID

## langCode()

ప్రస్తుత భాషా కోడ్‌ను పొందండి

రిటర్న్ విలువ: `"zh"` వంటి ప్రస్తుత భాష కోడ్

## onLang(func)

భాష మార్పుల కోసం వినే కాల్‌బ్యాక్‌ను బైండ్ చేయండి

పరామితి:
- `func` : భాష మారినప్పుడు కాల్ బ్యాక్ ఫంక్షన్
  - పరామితి
    - `code` : భాషా కోడ్
    - `id` : భాష ID

రిటర్న్ విలువ: అన్‌బౌండ్ క్లీనప్ ఫంక్షన్

## langSet(id)

ప్రస్తుత భాషను మార్చండి మరియు ప్రసారం చేయండి

పరామితి:
- `id` : లక్ష్య భాష ID
