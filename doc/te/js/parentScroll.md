# parentScroll.js

పూర్వీకుల స్క్రోల్ కంటైనర్ మూలకాన్ని పొందండి

## default(el)

పేర్కొన్న మూలకం నుండి పైకి వెళ్లండి, దానిని కలిగి ఉన్న స్క్రోల్ కంటైనర్ మూలకాన్ని కనుగొని, తిరిగి ఇవ్వండి (`C-VS`ShadowRoot అంతర్నిర్మిత స్క్రోల్ కంటైనర్ వంటివి)

పరామితి:
- `el` : DOM మూలకం ప్రారంభిస్తోంది

రిటర్న్ విలువ: స్క్రోల్ కంటైనర్ DOM మూలకం
