# md/streamRender.js

మార్క్‌డౌన్ స్ట్రీమింగ్ రెండరింగ్

## default(container, source)

పేర్కొన్న DOM కంటైనర్‌లో మార్క్‌డౌన్ టెక్స్ట్ యొక్క స్ట్రీమింగ్ రెండరింగ్, టైప్‌రైటర్ కర్సర్ ప్రభావాలు మరియు ఆటోమేటిక్ స్క్రోలింగ్ మద్దతు

పరామితి:
- `container` : DOM కంటైనర్ ఎలిమెంట్ రెండర్ చేయబడిన కంటెంట్‌ని హోస్టింగ్ చేస్తోంది
- `source` : రీడబుల్ స్ట్రీమ్ టెక్స్ట్ డేటా సోర్స్

రిటర్న్ విలువ:

- `cancel` : రెండరింగ్ మరియు డేటా స్ట్రీమ్ రీడింగ్‌ను ఆపడానికి ఫంక్షన్
- `setStop` : ఆటోమేటిక్ స్క్రోలింగ్‌ను పాజ్ చేయాలా వద్దా అని సెట్ చేయడానికి ఫంక్షన్
  - పరామితి
    - `val` : పాజ్ చేయాలా వద్దా అనే బూలియన్ విలువ
