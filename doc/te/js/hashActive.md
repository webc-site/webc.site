# hashActive.js

URL యాంకర్లు మరియు వ్యూపోర్ట్ స్క్రోలింగ్ ఆధారంగా నావిగేషన్ లింక్ హైలైట్ చేయడం

## default(link_li, cls)

పేజీ స్క్రోల్ స్థానం మరియు URL యాంకర్ పాయింట్ మార్పుల ఆధారంగా ప్రస్తుతం కనిపించే ప్రాంతానికి సంబంధించిన నావిగేషన్ లింక్‌లను ఆటోమేటిక్‌గా హైలైట్ చేయండి.

పరామితి:
- `link_li` : నావిగేషన్ లింక్ DOM మూలకం జాబితా
- `cls` : సక్రియ CSS తరగతి పేరు, డిఫాల్ట్ `"A"`

రిటర్న్ విలువ: అన్ని లిజనింగ్ క్లీనప్ ఫంక్షన్‌లను రద్దు చేయండి
