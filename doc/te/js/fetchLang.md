# fetchLang.js

భాషా ప్యాక్‌లను అసమకాలికంగా లోడ్ చేయండి మరియు అప్‌డేట్ కాల్‌బ్యాక్‌లను బైండ్ చేయండి

## fetchLang(getDict)

భాషా ప్యాక్‌ను అసమకాలికంగా పొందేందుకు లాంగ్వేజ్ ప్యాక్ లిజనర్‌ను సృష్టించండి మరియు భాష మారినప్పుడు రిఫ్రెష్ కాల్‌బ్యాక్‌ని అమలు చేయండి

పరామితి:
- `getDict` : భాష ప్యాకేజీ యొక్క అసమకాలిక ఫంక్షన్‌ను పొందండి
  - పరామితి
    - `code` : భాషా కోడ్
  - రిటర్న్ విలువ: భాషా ప్యాక్ నిఘంటువు

రిటర్న్ విలువ: రిఫ్రెష్ కాల్‌బ్యాక్‌ను నమోదు చేయడానికి ఫంక్షన్
- పరామితి
  - `refresh` : భాష నవీకరించబడినప్పుడు కాల్‌బ్యాక్ ఫంక్షన్‌ని రిఫ్రెష్ చేయండి
    - పరామితి
      - `i18n` : భాషా ప్యాక్ నిఘంటువు వస్తువు
      - `code` : ప్రస్తుత భాష కోడ్
- రిటర్న్ విలువ: రిఫ్రెష్ కాల్‌బ్యాక్‌ను అన్‌రిజిస్టర్ చేయడానికి ఫంక్షన్
