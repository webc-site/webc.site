# c-t.js

HTML అంతర్జాతీయీకరణ

`<c-t>key</c-t>` ట్యాగ్ అనుకూలీకరించబడింది మరియు `key` అనేది భాషా ప్యాక్ యొక్క కీ.

## cTranSet(el, genUrl)

`c-t`కి వచనాన్ని సెట్ చేయండి.

పరామితి:
- `el` : రూట్ ఎలిమెంట్, గ్లోబల్‌ను `document`లో పాస్ చేయవచ్చు
- `genUrl` : భాషా ప్యాక్ నిఘంటువును పొందేందుకు ఫంక్షన్
  - పరామితి
    - `code` : భాషా కోడ్
  - రిటర్న్ విలువ: లాంగ్వేజ్ ప్యాక్ డిక్షనరీ ఆబ్జెక్ట్ లేదా ఈ ఆబ్జెక్ట్‌ను పరిష్కరించే ప్రామిస్
