# dom.js

డాక్యుమెంట్ వేరియబుల్ సంక్షిప్తాలు మరియు షార్ట్‌కట్ ఫంక్షన్‌లు

## D

`document` వస్తువుకు సత్వరమార్గ సూచన

## B

`document.body` వస్తువుకు సత్వరమార్గ సూచన

## newEl

DOM మూలకాలను సృష్టించండి

పరామితి:
- `tagName` : మూలకం ట్యాగ్ పేరు

రిటర్న్ విలువ: కొత్తగా సృష్టించబడిన DOM మూలకం

## newLi(...tag_li)

బ్యాచ్‌లలో బహుళ DOM మూలకాలను సృష్టించండి

పరామితి:
- `tag_li` : ట్యాగ్ పేరు జాబితా

రిటర్న్ విలువ:
- కొత్తగా సృష్టించబడిన అన్ని DOM మూలకాలను కలిగి ఉన్న అర్రే

## cE(name, cls)

వెబ్ కాంపోనెంట్స్ అనుకూల మూలకాలను నమోదు చేయండి

పరామితి:
- `name` : అనుకూల మూలకం పేరు, `c-` ఉపసర్గ స్వయంచాలకంగా జోడించబడుతుంది
- `cls` : అనుకూల మూలకం యొక్క తరగతి
