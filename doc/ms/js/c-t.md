# c-t.js

Pengantarabangsaan HTML

Teg `<c-t>key</c-t>` disesuaikan dan `key` ialah kunci pek bahasa.

## cTranSet(el, genUrl)

Tetapkan teks untuk `c-t`.

parameter:
- `el` : elemen akar, global boleh dihantar dalam `document`
- `genUrl` : berfungsi untuk mendapatkan kamus pek bahasa
  - parameter
    - `code` : kod bahasa
  - Nilai pulangan: objek kamus pek bahasa atau Janji yang menyelesaikan objek ini
