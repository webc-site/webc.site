# dom.js

Dokumen singkatan pembolehubah dan fungsi pintasan

## D

`document` Rujukan pintasan kepada objek

## B

`document.body` Rujukan pintasan kepada objek

## newEl

Buat elemen DOM

parameter:
- `tagName` : nama teg unsur

Nilai pulangan: elemen DOM yang baru dibuat

## newLi(...tag_li)

Buat berbilang elemen DOM dalam kelompok

parameter:
- `tag_li` : senarai nama teg

Nilai pulangan:
- Tatasusunan yang mengandungi semua elemen DOM yang baru dibuat

## cE(name, cls)

Daftar elemen tersuai Komponen Web

parameter:
- `name` : Nama unsur tersuai, awalan `c-` akan ditambahkan secara automatik
- `cls` : kelas elemen tersuai
