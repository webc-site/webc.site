# hashActive.js

Penyerlahan pautan navigasi berdasarkan penambat URL dan penatalan port pandangan

## default(link_li, cls)

Serlahkan pautan navigasi secara automatik yang sepadan dengan kawasan yang kelihatan pada masa ini berdasarkan kedudukan tatal halaman dan perubahan titik sauh URL.

parameter:
- `link_li` : Senarai elemen DOM pautan navigasi
- `cls` : Nama kelas CSS aktif, lalai `"A"`

Nilai pulangan: Batalkan semua fungsi pembersihan mendengar
