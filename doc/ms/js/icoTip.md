# icoTip.js

Tentukan dan daftarkan komponen ikon dengan petunjuk pengantarabangsaan

## default(i18n_url, tag, key)

Daftarkan komponen elemen HTML tersuai dengan gelembung gesaan pengantarabangsaan

parameter:
- `i18n_url` : Minta alamat untuk konfigurasi pengantarabangsaan
- `tag` : Sesuaikan nama teg bagi elemen HTML
- `key` : Nilai utama teks gesaan dalam kamus pengantarabangsaan
