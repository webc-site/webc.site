# i18n.js

Pengurusan status berbilang bahasa dan pemantauan perubahan

## LANG_LI

Senarai konfigurasi bahasa yang tersedia

## lang()

Dapatkan ID bahasa semasa

Nilai pulangan: ID bahasa semasa

## langCode()

Dapatkan kod bahasa semasa

Nilai pulangan: kod bahasa semasa, seperti `"zh"`

## onLang(func)

Ikat panggilan balik pendengaran untuk perubahan bahasa

parameter:
- `func` : fungsi panggil balik apabila bahasa berubah
  - parameter
    - `code` : Kod bahasa
    - `id` : ID Bahasa

Nilai pulangan: Fungsi pembersihan tidak terikat

## langSet(id)

Tukar dan siarkan bahasa semasa

parameter:
- `id` : ID bahasa sasaran
