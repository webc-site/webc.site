# route.js

單頁前端路由管理

## nowUrl()

取得目前去除首部斜線的 URL 路徑

傳回值: 目前路徑字串

## route(hook)

註冊路由變更的監聽回調，註冊時會以目前路徑執行一次

參數:
- `hook` : 路由監聽回呼函數
  - 參數
    - `url` : 目前路徑
    - `pre` : 上一次的路徑

傳回值: 取消註冊的清理函數

## setPre(url)

儲存上一次記錄的 URL

參數:
- `url` : 路徑字串

## preUrl()

取得上一次記錄的 URL

傳回值: 上一次的路徑字串

## refresh(url)

強制以指定路徑觸發所有路由監聽回調，並更新 pre

參數:
- `url` : 目標路徑，預設目前路徑

## removeSlash(url)

移除路徑頭部的斜杠

參數:
- `url` : 路徑字串

回傳值: 移除頭部斜線後的路徑

## setUrl(url)

更新瀏覽器網址列 URL 狀態

參數:
- `url` : 目標路徑

傳回值: 若路徑實際變更則回傳 1

## goto(url)

更新瀏覽器位址並觸發路由刷新

參數:
- `url` : 目標路徑
