# fetch.js

基於 fetch 封裝的網路請求工具函數

## F(url, opt)

發送網路請求並傳回回應對象

參數:
- `url` : 請求位址
- `opt` : `fetch` 選項對象

傳回值: Response 對象

## f(attr)

包裝請求並以指定格式解析傳回內容的高階函數

參數:
- `attr` : 反應體解析屬性名

傳回值: 非同步請求並解析內容的函數
  - 參數
    - `url` : 請求位址
    - `opt` : `fetch` 選項對象
  - 傳回值: 解析後的內容

## fTxt

發送網路請求並以文字格式傳回回應體

參數:
- `url` : 請求位址
- `opt` : `fetch` 選項對象

傳回值: 回應文字

## fJson(url, opt)

發送網路請求並以 JSON 格式解析回傳回應體

參數:
- `url` : 請求位址
- `opt` : `fetch` 選項對象

傳回值: 解析後的 JSON 對象

## fBin(url, opt)

發送網路請求並以二進位位元組數組傳回響應體

參數:
- `url` : 請求位址
- `opt` : `fetch` 選項對象

傳回值: 位元組數組
