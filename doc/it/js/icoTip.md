# icoTip.js

Definire e registrare un componente icona con suggerimenti per l'internazionalizzazione

## default(i18n_url, tag, key)

Registra un componente di elemento HTML personalizzato con fumetti di richiesta di internazionalizzazione

parametro:
- `i18n_url` : Richiedi indirizzo per la configurazione dell'internazionalizzazione
- `tag`: personalizza il nome del tag dell'elemento HTML
- `key` : il valore chiave del testo del prompt nel dizionario di internazionalizzazione
