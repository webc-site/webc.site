# i18n.js

Gestione dello stato multilingue e monitoraggio delle modifiche

## LANG_LI

Elenco delle configurazioni linguistiche disponibili

## lang()

Ottieni l'ID della lingua corrente

Valore restituito: ID della lingua corrente

## langCode()

Ottieni il codice della lingua corrente

Valore restituito: codice della lingua corrente, ad esempio `"zh"`

## onLang(func)

Associa la richiamata di ascolto per i cambiamenti di lingua

parametro:
- `func`: funzione di callback quando cambia la lingua
  - parametro
    - `code`: codice della lingua
    - `id`: ID della lingua

Valore restituito: funzione di pulizia non associata

## langSet(id)

Cambia e trasmetti la lingua corrente

parametro:
- `id`: ID della lingua di destinazione
