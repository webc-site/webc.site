## B

## css(txt, root)

## newEl

Crea elementi DOM

parametro:
- `tagName`: nome del tag dell'elemento

Valore restituito: elemento DOM appena creato

## newLi(...tag_li)

Crea più elementi DOM in batch

parametro:
- `tag_li`: elenco dei nomi dei tag

Valore restituito:
- Array contenente tutti gli elementi DOM appena creati

## cE(name, cls)

Registrare elementi personalizzati di Web Components

parametro:
- `name`: il nome dell'elemento personalizzato, il prefisso `c-` verranno aggiunti automaticamente
- `cls` : la classe dell'elemento personalizzato
