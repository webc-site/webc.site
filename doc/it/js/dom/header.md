# dom/header.js

Crea la struttura DOM della barra dell'intestazione

## default()

Crea e combina la struttura DOM dell'intestazione dei tag header, h1 e b

Valore restituito:

- `header` : elemento contenitore della barra dell'intestazione
- `h1`: elemento del titolo
- `b` : sottotitolo o elemento informativo aggiuntivo
