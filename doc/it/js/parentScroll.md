# parentScroll.js

Ottieni l'elemento contenitore di scorrimento dell'antenato

## default(el)

Attraversa verso l'alto dall'elemento specificato, trova e restituisce l'elemento contenitore di scorrimento che lo contiene (come il contenitore di scorrimento integrato ShadowRoot di `C-VS`)

parametro:
- `el` : elemento DOM iniziale

Valore restituito: elemento DOM del contenitore di scorrimento
