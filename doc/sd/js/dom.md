## newEl

DOM عناصر ٺاهيو

پيرا ميٽر:
- `tagName` : عنصر ٽيگ جو نالو

واپسي جي قيمت: نئون ٺهيل DOM عنصر

## newLi(...tag_li)

بيچ ۾ گھڻن DOM عناصر ٺاھيو

پيرا ميٽر:
- `tag_li` : ٽيگ جي نالي جي فهرست

واپسي جو قدر:
- سڀني نئين ٺاهيل DOM عناصر تي مشتمل صف

## cE(name, cls)

رجسٽر ويب اجزاء ڪسٽم عناصر

پيرا ميٽر:
- `name` : حسب ضرورت عنصر جو نالو، `c-` اڳي ئي شامل ڪيو ويندو
- `cls` : ڪسٽم عنصر جو طبقو
