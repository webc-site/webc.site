# fetchLang.js

غير هم وقتي طور تي ٻولي پيڪ لوڊ ڪريو ۽ اپڊيٽ ڪال بيڪس کي پابند ڪريو

## fetchLang(getDict)

هڪ ٻولي پيڪ ٻڌندڙ ٺاھيو غير مطابقت پذير طور تي ٻولي پيڪ حاصل ڪرڻ ۽ ھڪڙي ريفريش ڪال بيڪ تي عمل ڪريو جڏھن ٻولي تبديل ٿي

پيرا ميٽر:
- `getDict` : ٻولي پيڪيج جي غير مطابقت واري فنڪشن حاصل ڪريو
  - پيرا ميٽر
    - `code` : ٻولي ڪوڊ
  - واپسي جي قيمت: ٻولي پيڪ ڊڪشنري

واپسي جي قيمت: ريفريش ڪال بڪ کي رجسٽر ڪرڻ لاءِ فنڪشن
- پيرا ميٽر
  - `refresh` : ڪال بڪ فنڪشن ريفريش ڪريو جڏهن ٻولي اپڊيٽ ڪئي وڃي
    - پيرا ميٽر
      - `i18n` : ٻولي پيڪ ڊڪشنري اعتراض
      - `code` : موجوده ٻولي ڪوڊ
- واپسي جي قيمت: ريفريش ڪالبڪ کي غير رجسٽر ڪرڻ لاءِ فنڪشن
