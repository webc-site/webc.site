# hashActive.js

URL اينڪرز ۽ وييو پورٽ اسڪرولنگ جي بنياد تي نيويگيشن لنڪ کي نمايان ڪرڻ

## default(link_li, cls)

پاڻمرادو نمايان نيويگيشن لنڪس جيڪي هن وقت ڏسڻ ۾ ايندڙ علائقي سان ملن ٿيون پيج اسڪرول پوزيشن ۽ URL اينڪر پوائنٽ تبديلين جي بنياد تي.

پيرا ميٽر:
- `link_li` : نيويگيشن لنڪ DOM عنصرن جي فهرست
- `cls` : فعال CSS ڪلاس جو نالو، ڊفالٽ `"A"`

واپسي جي قيمت: سڀني ٻڌڻ جي صفائي جي ڪم کي منسوخ ڪريو
