# route.js

سنگل پيج فرنٽ-اينڊ روٽنگ جو انتظام

## nowUrl()

موجوده URL جو رستو حاصل ڪريو اڳواٽ سليش سان هٽايو ويو

واپسي جي قيمت: موجوده رستو اسٽرنگ

## route(hook)

رستي جي تبديلين لاءِ ٻڌڻ واري ڪال بڪ کي رجسٽر ڪريو. جڏهن رجسٽريشن، اهو هڪ ڀيرو جاري ڪيو ويندو موجوده رستي سان.

پيرا ميٽر:
- `hook` : رستي جي نگراني ڪالبڪ فنڪشن
  - پيرا ميٽر
    - `url` : موجوده رستو
    - `pre` : آخري رستو

واپسي جي قيمت: غير رجسٽر ٿيل صفائي فنڪشن

## setPre(url)

آخري رڪارڊ ٿيل URL محفوظ ڪريو

پيرا ميٽر:
- `url` : رستو اسٽرنگ

## preUrl()

آخري رڪارڊ جو URL حاصل ڪريو

واپسي جي قيمت: آخري رستو اسٽرنگ

## refresh(url)

سڀني رستن جي ٻڌڻ واري ڪال بڪ کي مخصوص رستي تي شروع ٿيڻ لاءِ مجبور ڪريو ۽ اڳي کي اپڊيٽ ڪريو

پيرا ميٽر:
- `url` : ٽارگيٽ رستو، ڊفالٽ موجوده رستو

## removeSlash(url)

رستي جي هيڊرن مان سليش هٽايو

پيرا ميٽر:
- `url` : رستو اسٽرنگ

واپسي جي قيمت: اڳواٽ سليش کي هٽائڻ کان پوء رستو

## setUrl(url)

تازه ڪاري برائوزر ايڊريس بار URL اسٽيٽس

پيرا ميٽر:
- `url` : ٽارگيٽ رستو

واپسي جي قيمت: 1 جيڪڏهن رستو اصل ۾ تبديل ٿي وڃي

## goto(url)

برائوزر ايڊريس کي اپڊيٽ ڪريو ۽ رستو ريفريش ڪريو

پيرا ميٽر:
- `url` : ٽارگيٽ رستو
