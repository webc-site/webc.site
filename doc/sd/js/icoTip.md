# icoTip.js

بين الاقواميت جي اشارن سان هڪ آئڪن جزو جي وضاحت ۽ رجسٽر ڪريو

## default(i18n_url, tag, key)

رجسٽر ڪريو هڪ ڪسٽم HTML عنصر جزو بين الاقواميائيزيشن پرامپٹ بلبلز سان

پيرا ميٽر:
- `i18n_url` : درخواست ائڊريس لاءِ بين الاقوامي ترتيب جي ترتيب
- `tag` : HTML عنصر جي ٽيگ جو نالو ترتيب ڏيو
- `key` : انٽرنيشنلائيزيشن ڊڪشنري ۾ پرامپٽ ٽيڪسٽ جو اهم قدر
