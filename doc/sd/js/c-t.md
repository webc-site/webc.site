# c-t.js

HTML انٽرنيشنلائيزيشن

ٽيگ `<c-t>key</c-t>` ترتيب ڏنل آهي، ۽ `key` ٻولي پيڪ جي ڪنجي آهي.

## cTranSet(el, genUrl)

`c-t` لاءِ متن سيٽ ڪريو.

پيرا ميٽر:
- `el` : روٽ عنصر، گلوبل پاس ڪري سگھجي ٿو `document` ۾
- `genUrl` : ٻولي پيڪ ڊڪشنري حاصل ڪرڻ لاءِ فنڪشن
  - پيرا ميٽر
    - `code` : ٻولي ڪوڊ
  - واپسي جي قيمت: هڪ ٻولي پيڪ ڊڪشنري اعتراض، يا هڪ وعدو جيڪو هن اعتراض کي حل ڪري ٿو
