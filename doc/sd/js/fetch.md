# fetch.js

نيٽ ورڪ جي درخواست جو اوزار فعل آڻڻ جي بنياد تي encapsulation

## F(url, opt)

هڪ نيٽ ورڪ جي درخواست موڪليو ۽ هڪ جواب اعتراض واپس

پيرا ميٽر:
- `url` : درخواست ائڊريس
- `opt` : `fetch` آپشن اعتراض

واپسي جي قيمت: جوابي اعتراض

## f(attr)

هڪ اعلي-آرڊر فنڪشن جيڪو هڪ درخواست کي لپي ٿو ۽ واپس ڪيل مواد کي مخصوص فارميٽ ۾ پارس ڪري ٿو

پيرا ميٽر:
- `attr` : جوابي جسم کي پارس ڪرڻ واري خاصيت جو نالو

واپسي جي قيمت: فعل غير مطابقت واري طور تي درخواست ڪرڻ ۽ مواد کي پارس ڪرڻ لاءِ
  - پيرا ميٽر
    - `url` : درخواست ائڊريس
    - `opt` : `fetch` آپشن اعتراض
  - واپسي جي قيمت: تجزياتي مواد

## fTxt

نيٽ ورڪ جي درخواست موڪليو ۽ جوابي جسم کي ٽيڪسٽ فارميٽ ۾ واپس ڪريو

پيرا ميٽر:
- `url` : درخواست ائڊريس
- `opt` : `fetch` آپشن اعتراض

واپسي جو قدر: جوابي متن

## fJson(url, opt)

نيٽ ورڪ جي درخواست موڪليو ۽ پارس ڪريو ۽ جوابي جسم کي JSON فارميٽ ۾ واپس ڪريو

پيرا ميٽر:
- `url` : درخواست ائڊريس
- `opt` : `fetch` آپشن اعتراض

واپسي جي قيمت: parsed JSON اعتراض

## fBin(url, opt)

نيٽ ورڪ جي درخواست موڪليو ۽ جوابي جسم کي بائنري بائيٽ صف جي طور تي واپس ڪريو

پيرا ميٽر:
- `url` : درخواست ائڊريس
- `opt` : `fetch` آپشن اعتراض

واپسي جي قيمت: بائيٽ صف
