# Toast.js

ٽوسٽ پاپ اپ پيغام جو اشارو

## toast(render, timeout)

ڏيکاريو معياري تڪڙي پاپ اپ ونڊو

پيرا ميٽر:
- `render` : رينڊرنگ فنڪشن، استعمال ٿيل پاپ اپ مواد کي ترتيب ڏيڻ لاءِ
  - پيرا ميٽر
    - `el` : پاپ اپ ونڊو جو DOM عنصر
- `timeout` : خودڪار بند ڪرڻ لاءِ دير سيڪنڊ، ڊفالٽ 9، ڪو به خودڪار بند نه جڏهن 0

واپسي جي قيمت: پاپ اپ ونڊو جو DOM عنصر
