# md/streamRender.js

مارڪ ڊائون اسٽريمنگ رينڊرنگ

## default(container, source)

مخصوص DOM ڪنٽينر ۾ مارڪ ڊائون متن جي اسٽريمنگ رينڊنگ، ٽائپ رائيٽر ڪرسر اثرات ۽ خودڪار اسڪرولنگ کي سپورٽ ڪرڻ

پيرا ميٽر:
- `container` : DOM ڪنٽينر عنصر ميزباني ڪيل مواد
- `source` : پڙهڻ لائق اسٽريم ٽيڪسٽ ڊيٽا جو ذريعو

واپسي جو قدر:

- `cancel` : رينڊرنگ ۽ ڊيٽا اسٽريم ريڊنگ کي روڪڻ لاءِ فنڪشن
- `setStop` : فنڪشن مقرر ڪرڻ لاءِ ته ڇا خودڪار اسڪرولنگ کي روڪيو وڃي
  - پيرا ميٽر
    - `val` : بوليان قدر ته ڇا روڪيو وڃي يا نه
