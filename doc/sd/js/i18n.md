# i18n.js

ملٽي ٻولي جي صورتحال جو انتظام ۽ تبديلي جي نگراني

## LANG_LI

دستياب ٻولين جي ترتيبن جي فهرست

## lang()

موجوده ٻولي جي سڃاڻپ حاصل ڪريو

واپسي جو قدر: موجوده ٻولي ID

## langCode()

موجوده ٻولي ڪوڊ حاصل ڪريو

واپسي جي قيمت: موجوده ٻولي ڪوڊ، جيئن ته `"zh"`

## onLang(func)

ٻولي جي تبديلين لاءِ ٻڌڻ واري ڪال بڪ کي پابند ڪريو

پيرا ميٽر:
- `func` : ڪال بڪ فنڪشن جڏهن ٻولي تبديل ٿئي ٿي
  - پيرا ميٽر
    - `code` : ٻولي ڪوڊ
    - `id` : ٻولي جي سڃاڻپ

واپسي جي قيمت: اڻڄاتل صفائي فنڪشن

## langSet(id)

تبديل ڪريو ۽ موجوده ٻولي کي نشر ڪريو

پيرا ميٽر:
- `id` : ٽارگيٽ ٻولي ID
