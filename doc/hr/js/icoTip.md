# icoTip.js

Definirajte i registrirajte komponentu ikone sa savjetima za internacionalizaciju

## default(i18n_url, tag, key)

Registrirajte prilagođenu komponentu HTML elementa s oblačićima za internacionalizaciju

parametar:
- `i18n_url` : Adresa zahtjeva za konfiguraciju internacionalizacije
- `tag` : Prilagodite naziv oznake HTML elementa
- `key` : Vrijednost ključa teksta upita u rječniku internacionalizacije
