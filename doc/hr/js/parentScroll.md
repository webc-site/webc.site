# parentScroll.js

Nabavite element spremnika pomicanja pretka

## default(el)

Krenite prema gore od navedenog elementa, pronađite i vratite element spremnika za pomicanje koji ga sadrži (kao što je ugrađeni spremnik za pomicanje `C-VS`'s ShadowRoot)

parametar:
- `el` : početni DOM element

Povratna vrijednost: DOM element spremnika za pomicanje
