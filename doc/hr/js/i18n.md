# i18n.js

Višejezično upravljanje statusom i praćenje promjena

## LANG_LI

Popis dostupnih jezičnih konfiguracija

## lang()

Dohvatite trenutni ID jezika

Povratna vrijednost: trenutni ID jezika

## langCode()

Dobijte trenutni jezični kod

Povratna vrijednost: kod trenutnog jezika, kao što je `"zh"`

## onLang(func)

Povežite povratni poziv slušanja za promjene jezika

parametar:
- `func` : funkcija povratnog poziva kada se promijeni jezik
  - parametar
    - `code` : kôd jezika
    - `id` : ID jezika

Povratna vrijednost: Nevezana funkcija čišćenja

## langSet(id)

Promijenite i emitirajte trenutni jezik

parametar:
- `id` : ID ciljnog jezika
