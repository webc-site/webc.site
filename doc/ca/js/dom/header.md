# dom/header.js

Creeu l'estructura DOM de la barra de capçalera

## default()

Creeu i combineu l'estructura DOM de la capçalera de les etiquetes de capçalera, h1 i b

Valor de retorn:

- `header` : element contenidor de la barra de capçalera
- `h1` : element del títol
- `b` : subtítol o element d'informació addicional
