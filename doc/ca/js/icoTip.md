# icoTip.js

Definiu i registreu un component d'icona amb consells d'internacionalització

## default(i18n_url, tag, key)

Registreu un component d'element HTML personalitzat amb bombolles d'indicació d'internacionalització

paràmetre:
- `i18n_url` : sol·licitud d'adreça per a la configuració d'internacionalització
- `tag` : personalitza el nom de l'etiqueta de l'element HTML
- `key` : el valor clau del text de sol·licitud al diccionari d'internacionalització
