# parentScroll.js

Obteniu l'element contenidor de desplaçament de l'ancestre

## default(el)

Desplaceu-vos cap amunt des de l'element especificat, cerqueu i retorneu l'element del contenidor de desplaçament que el conté (com ara el contenidor de desplaçament integrat de ShadowRoot de `C-VS`).

paràmetre:
- `el` : element DOM inicial

Valor de retorn: element DOM del contenidor de desplaçament
