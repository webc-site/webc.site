# On.js

Supervisió d'esdeveniments d'enllaç per lots

## On(elem, dict)

Enllaceu els oients d'esdeveniments d'elements DOM per lots

paràmetre:
- `elem` : element DOM
- `dict` : mapeig d'esdeveniments i devolució de trucades

Valor de retorn: elimina totes les funcions d'escolta d'esdeveniments vinculades
