# i18n.js

Gestió de l'estat en diversos idiomes i seguiment de canvis

## LANG_LI

Llista de configuracions d'idiomes disponibles

## lang()

Obteniu l'identificador de l'idioma actual

Valor de retorn: identificador de l'idioma actual

## langCode()

Obteniu el codi d'idioma actual

Valor de retorn: codi d'idioma actual, com ara `"zh"`

## onLang(func)

Enllaça la devolució de trucada d'escolta per als canvis d'idioma

paràmetre:
- `func` : funció de devolució de trucada quan canvia l'idioma
  - paràmetre
    - `code` : codi d'idioma
    - `id` : ID d'idioma

Valor de retorn: funció de neteja sense vincular

## langSet(id)

Canvia i transmet l'idioma actual

paràmetre:
- `id` : identificador de l'idioma objectiu
