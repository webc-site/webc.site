# i18n/store.js

Gestió d'emmagatzematge local multilingüe

## get()

Obteniu l'identificador d'idioma predeterminat que es calcula localment a la memòria cau o que es calcula mitjançant la concordança d'idiomes del navegador

Valor de retorn: ID d'idioma

## set(id)

Emmagatzemeu a la memòria cau localment l'identificador de l'idioma actual

paràmetre:
- `id` : ID d'idioma
