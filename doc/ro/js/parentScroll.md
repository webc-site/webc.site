# parentScroll.js

Obțineți elementul container de defilare a strămoșilor

## default(el)

Traversați în sus de la elementul specificat, găsiți și returnați elementul container de defilare care îl conține (cum ar fi containerul de defilare încorporat ShadowRoot al `C-VS`)

parametru:
- `el` : elementul DOM de pornire

Valoare returnată: element DOM container de defilare
