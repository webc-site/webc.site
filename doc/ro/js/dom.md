## newEl

Creați elemente DOM

parametru:
- `tagName` : numele etichetei elementului

Valoare returnată: element DOM nou creat

## newLi(...tag_li)

Creați mai multe elemente DOM în loturi

parametru:
- `tag_li` : lista cu numele etichetelor

Valoarea returnată:
- Matrice care conține toate elementele DOM nou create

## cE(name, cls)

Înregistrați elementele personalizate ale componentelor web

parametru:
- `name` : numele elementului personalizat, prefixul `c-` va fi adăugat automat
- `cls` : clasa elementului personalizat
