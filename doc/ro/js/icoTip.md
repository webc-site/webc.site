# icoTip.js

Definiți și înregistrați o componentă de pictogramă cu indicii de internaționalizare

## default(i18n_url, tag, key)

Înregistrați o componentă de element HTML personalizată cu bule de prompt de internaționalizare

parametru:
- `i18n_url` : Solicitați adresa pentru configurarea internaționalizării
- `tag` : personalizați numele etichetei elementului HTML
- `key` : valoarea cheie a textului prompt din dicționarul de internaționalizare
