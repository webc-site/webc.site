# route.js

การจัดการเส้นทางส่วนหน้าหน้าเดียว

## nowUrl()

รับเส้นทาง URL ปัจจุบันโดยลบเครื่องหมายทับนำหน้าออก

ค่าที่ส่งคืน: สตริงเส้นทางปัจจุบัน

## route(hook)

ลงทะเบียนการโทรกลับเพื่อฟังการเปลี่ยนแปลงเส้นทาง เมื่อลงทะเบียน มันจะถูกดำเนินการครั้งเดียวกับเส้นทางปัจจุบัน

พารามิเตอร์:
- `hook` : ฟังก์ชั่นการโทรกลับการตรวจสอบเส้นทาง
  - พารามิเตอร์
    - `url` : เส้นทางปัจจุบัน
    - `pre` : เส้นทางสุดท้าย

ค่าที่ส่งกลับ: ฟังก์ชันการล้างข้อมูลที่ไม่ได้ลงทะเบียน

## setPre(url)

บันทึก URL ที่บันทึกไว้ล่าสุด

พารามิเตอร์:
- `url` : สตริงเส้นทาง

## preUrl()

รับ URL ของบันทึกล่าสุด

ค่าที่ส่งคืน: สตริงเส้นทางสุดท้าย

## refresh(url)

บังคับให้เรียกใช้การเรียกกลับการฟังเส้นทางทั้งหมดตามเส้นทางที่ระบุและอัปเดตล่วงหน้า

พารามิเตอร์:
- `url` : เส้นทางเป้าหมาย, เส้นทางปัจจุบันเริ่มต้น

## removeSlash(url)

ลบเครื่องหมายทับออกจากส่วนหัวของเส้นทาง

พารามิเตอร์:
- `url` : สตริงเส้นทาง

ค่าที่ส่งคืน: เส้นทางหลังจากลบเครื่องหมายทับนำหน้าแล้ว

## setUrl(url)

อัปเดตสถานะ URL ของแถบที่อยู่ของเบราว์เซอร์

พารามิเตอร์:
- `url` : เส้นทางเป้าหมาย

ส่งคืนค่า: 1 หากเส้นทางเปลี่ยนแปลงจริง

## goto(url)

อัปเดตที่อยู่เบราว์เซอร์และทริกเกอร์การรีเฟรชเส้นทาง

พารามิเตอร์:
- `url` : เส้นทางเป้าหมาย
