# c-t.js

การทำให้ HTML เป็นสากล

แท็ก `<c-t>key</c-t>` ได้รับการปรับแต่ง และ `key` เป็นคีย์ของแพ็กภาษา

## cTranSet(el, genUrl)

ตั้งค่าข้อความสำหรับ `c-t`

พารามิเตอร์:
- `el` : องค์ประกอบ root, global สามารถส่งผ่านใน `document`
- `genUrl` : ฟังก์ชั่นเพื่อรับพจนานุกรมชุดภาษา
  - พารามิเตอร์
    - `code` : รหัสภาษา
  - ส่งคืนค่า: ออบเจ็กต์พจนานุกรมชุดภาษา หรือ Promise ที่แก้ไขออบเจ็กต์นี้
