# Toast.js

ข้อความแจ้งข้อความป๊อปอัปขนมปังปิ้ง

## toast(render, timeout)

แสดงหน้าต่างป๊อปอัปพร้อมท์มาตรฐาน

พารามิเตอร์:
- `render` : ฟังก์ชั่นการเรนเดอร์ ใช้เพื่อปรับแต่งเนื้อหาป๊อปอัป
  - พารามิเตอร์
    - `el` : องค์ประกอบ DOM ของหน้าต่างป๊อปอัป
- `timeout` : หน่วงเวลาวินาทีสำหรับการปิดเครื่องอัตโนมัติ ค่าเริ่มต้น 9 ไม่มีการปิดเครื่องอัตโนมัติเมื่อ 0

ส่งคืนค่า: องค์ประกอบ DOM ของหน้าต่างป๊อปอัป

## toastErr(render, timeout)

แสดงป๊อปอัปข้อความแสดงข้อผิดพลาด

พารามิเตอร์:
- `render` : ฟังก์ชั่นการเรนเดอร์ ใช้เพื่อปรับแต่งเนื้อหาป๊อปอัป
  - พารามิเตอร์
    - `el` : องค์ประกอบ DOM ของหน้าต่างป๊อปอัป
- `timeout` : หน่วงเวลาวินาทีสำหรับการปิดเครื่องอัตโนมัติ ค่าเริ่มต้น 9 ไม่มีการปิดเครื่องอัตโนมัติเมื่อ 0

ค่าที่ส่งคืน: องค์ประกอบ DOM ของหน้าต่างป๊อปอัปข้อผิดพลาด
