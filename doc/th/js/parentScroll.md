# parentScroll.js

รับองค์ประกอบคอนเทนเนอร์การเลื่อนบรรพบุรุษ

## default(el)

สำรวจขึ้นไปจากองค์ประกอบที่ระบุ ค้นหาและส่งกลับองค์ประกอบคอนเทนเนอร์การเลื่อนที่มีองค์ประกอบนั้น (เช่น `C-VS` คอนเทนเนอร์การเลื่อนในตัว ShadowRoot)

พารามิเตอร์:
- `el` : เริ่มต้นองค์ประกอบ DOM

ส่งคืนค่า: องค์ประกอบ DOM ของคอนเทนเนอร์แบบเลื่อน
