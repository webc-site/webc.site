# fetchLang.js

โหลดชุดภาษาแบบอะซิงโครนัสและผูกการเรียกกลับการอัปเดต

## fetchLang(getDict)

สร้างตัวฟังชุดภาษาเพื่อรับชุดภาษาแบบอะซิงโครนัส และดำเนินการเรียกกลับรีเฟรชเมื่อมีการเปลี่ยนแปลงภาษา

พารามิเตอร์:
- `getDict` : รับฟังก์ชันอะซิงโครนัสของแพ็คเกจภาษา
  - พารามิเตอร์
    - `code` : รหัสภาษา
  - ค่าที่ส่งกลับ: พจนานุกรมชุดภาษา

ค่าที่ส่งคืน: ฟังก์ชั่นเพื่อลงทะเบียนการรีเฟรชการโทรกลับ
- พารามิเตอร์
  - `refresh` : รีเฟรชฟังก์ชันการโทรกลับเมื่อมีการอัพเดตภาษา
    - พารามิเตอร์
      - `i18n` : ออบเจ็กต์พจนานุกรมชุดภาษา
      - `code` : รหัสภาษาปัจจุบัน
- ส่งกลับค่า: ฟังก์ชั่นยกเลิกการลงทะเบียนการเรียกกลับรีเฟรช
