# fetch.js

ฟังก์ชันเครื่องมือคำขอเครือข่ายอิงตามการดึงข้อมูลแบบห่อหุ้ม

## F(url, opt)

ส่งคำขอเครือข่ายและส่งคืนออบเจ็กต์ตอบกลับ

พารามิเตอร์:
- `url` : ขอที่อยู่
- `opt` : `fetch` วัตถุตัวเลือก

ส่งกลับค่า: วัตถุตอบสนอง

## f(attr)

ฟังก์ชันลำดับที่สูงกว่าที่ล้อมคำขอและแยกวิเคราะห์เนื้อหาที่ส่งคืนในรูปแบบที่ระบุ

พารามิเตอร์:
- `attr` : ชื่อแอตทริบิวต์การแยกวิเคราะห์เนื้อหาการตอบสนอง

ส่งกลับค่า: ฟังก์ชั่นในการร้องขอและแยกวิเคราะห์เนื้อหาแบบอะซิงโครนัส
  - พารามิเตอร์
    - `url` : ขอที่อยู่
    - `opt` : `fetch` วัตถุตัวเลือก
  - ค่าส่งคืน: เนื้อหาที่แยกวิเคราะห์

## fTxt

ส่งคำขอเครือข่ายและส่งคืนเนื้อหาการตอบกลับในรูปแบบข้อความ

พารามิเตอร์:
- `url` : ขอที่อยู่
- `opt` : `fetch` วัตถุตัวเลือก

ค่าส่งคืน: ข้อความตอบกลับ

## fJson(url, opt)

ส่งคำขอเครือข่ายและแยกวิเคราะห์และส่งคืนเนื้อหาการตอบกลับในรูปแบบ JSON

พารามิเตอร์:
- `url` : ขอที่อยู่
- `opt` : `fetch` วัตถุตัวเลือก

ค่าที่ส่งคืน: ออบเจ็กต์ JSON ที่แยกวิเคราะห์

## fBin(url, opt)

ส่งคำขอเครือข่ายและส่งคืนเนื้อหาการตอบสนองเป็นอาร์เรย์ไบต์ไบนารี

พารามิเตอร์:
- `url` : ขอที่อยู่
- `opt` : `fetch` วัตถุตัวเลือก

ส่งคืนค่า: อาร์เรย์ไบต์
