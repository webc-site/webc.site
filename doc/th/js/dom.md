# dom.js

คำย่อตัวแปรเอกสารและฟังก์ชันทางลัด

## D

`document` การอ้างอิงทางลัดไปยังวัตถุ

## B

`document.body` การอ้างอิงทางลัดไปยังวัตถุ

## newEl

สร้างองค์ประกอบ DOM

พารามิเตอร์:
- `tagName` : ชื่อแท็กองค์ประกอบ

ค่าที่ส่งคืน: องค์ประกอบ DOM ที่สร้างขึ้นใหม่

## newLi(...tag_li)

สร้างองค์ประกอบ DOM หลายรายการเป็นชุด

พารามิเตอร์:
- `tag_li` : รายการชื่อแท็ก

ค่าส่งคืน:
- อาร์เรย์ที่มีองค์ประกอบ DOM ที่สร้างขึ้นใหม่ทั้งหมด

## cE(name, cls)

ลงทะเบียนองค์ประกอบที่กำหนดเองของ Web Components

พารามิเตอร์:
- `name` : ชื่อองค์ประกอบที่กำหนดเอง คำนำหน้า `c-` จะถูกเพิ่มโดยอัตโนมัติ
- `cls` : คลาสขององค์ประกอบที่กำหนดเอง
