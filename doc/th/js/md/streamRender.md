# md/streamRender.js

การเรนเดอร์สตรีมมิ่ง Markdown

## default(container, source)

การสตรีมการเรนเดอร์ข้อความ Markdown ลงในคอนเทนเนอร์ DOM ที่ระบุ รองรับเอฟเฟกต์เคอร์เซอร์ของเครื่องพิมพ์ดีดและการเลื่อนอัตโนมัติ

พารามิเตอร์:
- `container` : องค์ประกอบคอนเทนเนอร์ DOM ที่โฮสต์เนื้อหาที่แสดงผล
- `source` : แหล่งข้อมูลข้อความ ReadableStream

ค่าส่งคืน:

- `cancel` : ฟังก์ชั่นหยุดการเรนเดอร์และการอ่านสตรีมข้อมูล
- `setStop` : ฟังก์ชั่นตั้งค่าว่าจะหยุดการเลื่อนอัตโนมัติชั่วคราวหรือไม่
  - พารามิเตอร์
    - `val` : ค่าบูลีนว่าจะหยุดชั่วคราวหรือไม่
