# i18n.js

การจัดการสถานะหลายภาษาและการตรวจสอบการเปลี่ยนแปลง

## LANG_LI

รายการการกำหนดค่าภาษาที่ใช้ได้

## lang()

รับรหัสภาษาปัจจุบัน

ค่าส่งคืน: รหัสภาษาปัจจุบัน

## langCode()

รับรหัสภาษาปัจจุบัน

ค่าที่ส่งคืน: รหัสภาษาปัจจุบัน เช่น `"zh"`

## onLang(func)

ผูกการโทรกลับเพื่อฟังการเปลี่ยนแปลงภาษา

พารามิเตอร์:
- `func` : ฟังก์ชั่นโทรกลับเมื่อภาษาเปลี่ยน
  - พารามิเตอร์
    - `code` : รหัสภาษา
    - `id` : รหัสภาษา

ค่าที่ส่งกลับ: ฟังก์ชันการล้างข้อมูลที่ไม่ถูกผูกไว้

## langSet(id)

สลับและออกอากาศภาษาปัจจุบัน

พารามิเตอร์:
- `id` : รหัสภาษาเป้าหมาย
