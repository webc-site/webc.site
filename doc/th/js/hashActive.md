# hashActive.js

การเน้นลิงก์การนำทางตามจุดยึด URL และการเลื่อนวิวพอร์ต

## default(link_li, cls)

ไฮไลต์ลิงก์การนำทางที่สอดคล้องกับพื้นที่ที่มองเห็นได้โดยอัตโนมัติตามตำแหน่งการเลื่อนหน้าและการเปลี่ยนแปลงจุดยึด URL

พารามิเตอร์:
- `link_li` : รายการองค์ประกอบ DOM ของลิงก์การนำทาง
- `cls` : ชื่อคลาส CSS ที่ใช้งานอยู่ ค่าเริ่มต้น `"A"`

ค่าส่งคืน: ยกเลิกฟังก์ชันล้างข้อมูลการฟังทั้งหมด
