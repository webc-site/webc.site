# icoTip.js

กำหนดและลงทะเบียนองค์ประกอบไอคอนพร้อมคำแนะนำการทำให้เป็นสากล

## default(i18n_url, tag, key)

ลงทะเบียนคอมโพเนนต์องค์ประกอบ HTML ที่กำหนดเองด้วยฟองอากาศแจ้งการทำให้เป็นสากล

พารามิเตอร์:
- `i18n_url` : ขอที่อยู่สำหรับการกำหนดค่าสากล
- `tag` : ปรับแต่งชื่อแท็กขององค์ประกอบ HTML
- `key` : ค่าคีย์ของข้อความแจ้งในพจนานุกรมสากล
