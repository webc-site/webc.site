## newEl

Креирајте DOM елементи

параметар:
- `tagName` : име на ознака на елементот

Повратна вредност: новосоздаден DOM елемент

## newLi(...tag_li)

Креирајте повеќе DOM елементи во серии

параметар:
- `tag_li` : список со имиња на ознаки

Повратна вредност:
- Низа што ги содржи сите новосоздадени DOM елементи

## cE(name, cls)

Регистрирајте сопствени елементи на Web Components

параметар:
- `name` : приспособено име на елементот, префиксот `c-` ќе се додаде автоматски
- `cls` : класата на приспособениот елемент
