# i18n.js

Управување со повеќејазичен статус и следење на промени

## LANG_LI

Список на достапни јазични конфигурации

## lang()

Добијте идентификација на тековниот јазик

Повратна вредност: ID на тековниот јазик

## langCode()

Добијте го тековниот јазичен код

Повратна вредност: тековен јазичен код, како што е `"zh"`

## onLang(func)

Поврзете го повратниот повик за слушање за јазични промени

параметар:
- `func` : функција за повратен повик кога се менува јазикот
  - параметар
    - `code` : Јазичен код
    - `id` : ИД на јазик

Повратна вредност: Неврзана функција за чистење

## langSet(id)

Префрлете го и емитувајте го тековниот јазик

параметар:
- `id` : ID на целниот јазик
