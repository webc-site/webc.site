# Toast.js

Прашање за скокачка порака за тост

## toast(render, timeout)

Прикажи стандарден скокачки прозорец

параметар:
- `render` : Функција за рендерирање, што се користи за приспособување на содржината што се појавува
  - параметар
    - `el` : DOM елемент на скокачкиот прозорец
- `timeout` : одложување секунди за автоматско исклучување, стандардно 9, без автоматско исклучување кога 0

Повратна вредност: DOM елемент на скокачкиот прозорец

## toastErr(render, timeout)

параметар:
- `render` : Функција за рендерирање, што се користи за приспособување на содржината што се појавува
  - параметар
    - `el` : DOM елемент на скокачкиот прозорец
- `timeout` : одложување секунди за автоматско исклучување, стандардно 9, без автоматско исклучување кога 0

Повратна вредност: DOM елемент на скокачкиот прозорец за грешка
