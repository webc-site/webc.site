# icoTip.js

Дефинирајте и регистрирајте компонента за икона со совети за интернационализација

## default(i18n_url, tag, key)

Регистрирајте прилагодена компонента на HTML елемент со меурчиња за интернационализација

параметар:
- `i18n_url` : Барајте адреса за конфигурација на интернационализација
- `tag` : Приспособете го името на ознаката на елементот HTML
- `key` : клучната вредност на текстот на прашалникот во речникот за интернационализација
