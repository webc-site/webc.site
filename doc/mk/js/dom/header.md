# dom/header.js

Креирајте ја структурата DOM на лентата за заглавие

## default()

Креирајте и комбинирајте ја структурата DOM на заглавието на ознаките за заглавие, h1 и b

Повратна вредност:

- `header` : Елемент на контејнер на лентата за заглавие
- `h1` : елемент на насловот
- `b` : титл или дополнителен информативен елемент
