# md/streamRender.js

Рендерирање на стриминг Markdown

## default(container, source)

Пренесување на рендерирање на текстот Markdown во наведениот контејнер DOM, поддршка на ефектите на курсорот на машината за пишување и автоматско лизгање

параметар:
- `container` : Елемент на контејнер DOM што е хостирана рендерирана содржина
- `source` : Извор на текстуални податоци ReadableStream

Повратна вредност:

- `cancel` : Функција за запирање на прикажувањето и читањето на преносот на податоци
- `setStop` : Функција за поставување дали да се паузира автоматското лизгање
  - параметар
    - `val` : Булова вредност дали да се паузира или не
