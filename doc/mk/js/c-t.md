# c-t.js

HTML интернационализација

Ознаката `<c-t>key</c-t>` е приспособена, а `key` е клучот на јазичниот пакет.

## cTranSet(el, genUrl)

Поставете го текстот за `c-t`.

параметар:
- `el` : root елемент, глобален може да се пренесе во `document`
- `genUrl` : функција за добивање речник на јазичен пакет
  - параметар
    - `code` : јазичен код
  - Повратна вредност: објект за речник на јазичен пакет или ветување што се решава на овој објект
