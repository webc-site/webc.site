# hashActive.js

Истакнување на навигациската врска врз основа на прицврстувачите на URL-адресата и лизгањето на приказот

## default(link_li, cls)

Автоматски означете ги навигациските врски што одговараат на моментално видливата област врз основа на позицијата на лизгање на страницата и промените на точката на прицврстување на URL-то.

параметар:
- `link_li` : Список со елементи на DOM навигациска врска
- `cls` : Активно име на класа CSS, стандардно `"A"`

Повратна вредност: Откажете ги сите функции за чистење на слушање
