# i18n.js

Večjezično upravljanje stanja in spremljanje sprememb

## LANG_LI

Seznam razpoložljivih jezikovnih konfiguracij

## lang()

Pridobite ID trenutnega jezika

Povratna vrednost: ID trenutnega jezika

## langCode()

Pridobite trenutno kodo jezika

Povratna vrednost: koda trenutnega jezika, kot je `"zh"`

## onLang(func)

Povežite povratni klic poslušanja za jezikovne spremembe

parameter:
- `func` : funkcija povratnega klica ob spremembi jezika
  - parameter
    - `code` : Koda jezika
    - `id` : ID jezika

Povratna vrednost: Nevezana funkcija čiščenja

## langSet(id)

Preklopite in oddajajte trenutni jezik

parameter:
- `id` : ID ciljnega jezika
