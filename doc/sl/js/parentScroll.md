# parentScroll.js

Pridobite element vsebnika drsnega prednika

## default(el)

Pojdite navzgor od navedenega elementa, poiščite in vrnite element vsebnika drsenja, ki ga vsebuje (kot je vgrajen vsebnik drsenja ShadowRoot `C-VS`)

parameter:
- `el` : začetni element DOM

Povratna vrednost: element DOM vsebnika drsenja
