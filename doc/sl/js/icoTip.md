# icoTip.js

Definirajte in registrirajte komponento ikone z namigi za internacionalizacijo

## default(i18n_url, tag, key)

Registrirajte komponento elementa HTML po meri z mehurčki za internacionalizacijo

parameter:
- `i18n_url` : naslov zahteve za konfiguracijo internacionalizacije
- `tag` : Prilagodite ime oznake elementa HTML
- `key` : ključna vrednost besedila poziva v internacionalizacijskem slovarju
