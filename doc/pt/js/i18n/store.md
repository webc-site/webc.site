# i18n/store.js

Gerenciamento de armazenamento local multilíngue

## get()

Obtenha o ID do idioma padrão calculado em cache localmente ou calculado pela correspondência de idioma do navegador

Valor de retorno: ID do idioma

## set(id)

Armazene em cache o ID do idioma atual localmente

parâmetro:
- `id` : ID do idioma
