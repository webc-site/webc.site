# parentScroll.js

Obtenha o elemento do contêiner de rolagem ancestral

## default(el)

Percorra para cima a partir do elemento especificado, encontre e retorne o elemento do contêiner de rolagem que o contém (como o contêiner de rolagem integrado do `C-VS` ShadowRoot)

parâmetro:
- `el` : iniciando o elemento DOM

Valor de retorno: elemento DOM do contêiner de rolagem
