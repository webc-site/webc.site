# dom/header.js

Crie a estrutura DOM da barra de cabeçalho

## default()

Crie e combine a estrutura DOM do cabeçalho das tags de cabeçalho, h1 e b

Valor de retorno:

- `header` : Elemento contêiner da barra de cabeçalho
- `h1` : elemento de título
- `b` : legenda ou elemento de informação adicional
