# i18n.js

Gerenciamento de status multilíngue e monitoramento de alterações

## LANG_LI

Lista de configurações de idioma disponíveis

## lang()

Obtenha o ID do idioma atual

Valor de retorno: ID do idioma atual

## langCode()

Obtenha o código do idioma atual

Valor de retorno: código do idioma atual, como `"zh"`

## onLang(func)

Vincule o retorno de chamada de escuta para alterações de idioma

parâmetro:
- `func` : função de retorno de chamada quando o idioma muda
  - parâmetro
    - `code` : Código do idioma
    - `id` : ID do idioma

Valor de retorno: função de limpeza não consolidada

## langSet(id)

Alternar e transmitir o idioma atual

parâmetro:
- `id` : ID do idioma de destino
