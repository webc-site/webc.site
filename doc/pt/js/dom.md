# dom.js

Documente abreviações de variáveis ​​e funções de atalho

## D

`document` Referência de atalho para o objeto

## B

`document.body` Referência de atalho para o objeto

## newEl

Crie elementos DOM

parâmetro:
- `tagName` : nome da tag do elemento

Valor de retorno: elemento DOM recém-criado

## newLi(...tag_li)

Crie vários elementos DOM em lotes

parâmetro:
- `tag_li` : lista de nomes de tags

Valor de retorno:
- Array contendo todos os elementos DOM recém-criados

## cE(name, cls)

Registrar elementos personalizados de Web Components

parâmetro:
- `name`: nome do elemento personalizado, o prefixo `c-` será adicionado automaticamente
- `cls` : a classe do elemento personalizado
