# icoTip.js

Definir e registrar um componente de ícone com dicas de internacionalização

## default(i18n_url, tag, key)

Registre um componente de elemento HTML personalizado com balões de prompt de internacionalização

parâmetro:
- `i18n_url` : Endereço de solicitação para configuração de internacionalização
- `tag` : Personalize o nome da tag do elemento HTML
- `key` : o valor-chave do texto do prompt no dicionário de internacionalização
