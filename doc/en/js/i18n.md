# i18n.js

Multi-language status management and change monitoring

## LANG_LI

List of available language configurations

## lang()

Get current language ID

Return value: current language ID

## langCode()

Get the current language code

Return value: current language code, such as `"zh"`

## onLang(func)

Bind the listening callback for language changes

parameter:
- `func` : callback function when language changes
  - Parameter
    - `code` : Language code
    - `id` : Language ID

Return value: Unbound cleanup function

## langSet(id)

Switch and broadcast the current language

parameter:
- `id` : Target language ID
