# hashActive.js

Виділення навігаційних посилань на основі URL-адрес і прокручування вікна перегляду

## default(link_li, cls)

Автоматично виділяйте навігаційні посилання, що відповідають поточній видимій області на основі позиції прокручування сторінки та змін точки прив’язки URL-адреси.

параметр:
- `link_li` : список елементів DOM навігаційного посилання
- `cls` : назва активного класу CSS, за умовчанням `"A"`

Значення, що повертається: скасувати всі функції очищення прослуховування
