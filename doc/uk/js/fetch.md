# fetch.js

Функція інструменту мережевого запиту на основі інкапсуляції отримання

## F(url, opt)

Надіслати мережевий запит і повернути об’єкт відповіді

параметр:
- `url` : адреса запиту
- `opt` : об'єкт параметрів `fetch`

Значення, що повертається: об’єкт відповіді

## f(attr)

Функція вищого порядку, яка обгортає запит і аналізує повернутий вміст у заданому форматі

параметр:
- `attr` : назва атрибута розбору тіла відповіді

Повернене значення: функція для асинхронного запиту та аналізу вмісту
  - параметр
    - `url` : адреса запиту
    - `opt` : об'єкт параметрів `fetch`
  - Повернене значення: аналізований вміст

## fTxt

Надішліть мережевий запит і поверніть тіло відповіді в текстовому форматі

параметр:
- `url` : адреса запиту
- `opt` : об'єкт параметрів `fetch`

Повернене значення: текст відповіді

## fJson(url, opt)

Надішліть мережевий запит, проаналізуйте та поверніть тіло відповіді у форматі JSON

параметр:
- `url` : адреса запиту
- `opt` : об'єкт параметрів `fetch`

Повернене значення: проаналізований об’єкт JSON

## fBin(url, opt)

Надішліть мережевий запит і поверніть тіло відповіді у вигляді двійкового байтового масиву

параметр:
- `url` : адреса запиту
- `opt` : об'єкт параметрів `fetch`

Значення, що повертається: масив байтів
