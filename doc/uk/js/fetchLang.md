# fetchLang.js

Асинхронно завантажуйте мовні пакети та прив’язуйте зворотні виклики оновлення

## fetchLang(getDict)

Створіть слухач мовного пакета для асинхронного отримання мовного пакета та виконання зворотного виклику оновлення, коли мова змінюється

параметр:
- `getDict` : отримати асинхронну функцію мовного пакета
  - параметр
    - `code` : код мови
  - Повернене значення: словник мовного пакету

Повернене значення: функція для реєстрації зворотного виклику оновлення
- параметр
  - `refresh`: оновити функцію зворотного виклику після оновлення мови
    - параметр
      - `i18n` : об'єкт словника мовного пакета
      - `code` : поточний код мови
- Повернене значення: функція для скасування реєстрації зворотного виклику оновлення
