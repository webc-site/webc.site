# Toast.js

Спливаюче повідомлення Toast

## toast(render, timeout)

Показати стандартне спливаюче вікно підказки

параметр:
- `render` : функція візуалізації, яка використовується для налаштування вмісту спливаючих вікон
  - параметр
    - `el` : елемент DOM спливаючого вікна
- `timeout`: секунди затримки для автоматичного вимкнення, за умовчанням 9, без автоматичного вимкнення, коли 0

Повернене значення: елемент DOM спливаючого вікна

## toastErr(render, timeout)

Показати спливаюче повідомлення про помилку

параметр:
- `render` : функція візуалізації, яка використовується для налаштування вмісту спливаючих вікон
  - параметр
    - `el` : елемент DOM спливаючого вікна
- `timeout`: секунди затримки для автоматичного вимкнення, за умовчанням 9, без автоматичного вимкнення, коли 0

Значення, що повертається: елемент DOM у спливаючому вікні помилки
