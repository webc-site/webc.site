# i18n.js

Багатомовне керування статусом і моніторинг змін

## LANG_LI

Список доступних мовних конфігурацій

## lang()

Отримати ідентифікатор поточної мови

Повернене значення: ідентифікатор поточної мови

## langCode()

Отримати поточний код мови

Повернене значення: код поточної мови, наприклад `"zh"`

## onLang(func)

Прив’яжіть зворотний виклик прослуховування до змін мови

параметр:
- `func` : функція зворотного виклику при зміні мови
  - параметр
    - `code` : код мови
    - `id` : ідентифікатор мови

Значення, що повертається: Незв’язана функція очищення

## langSet(id)

Перемикайте та транслюйте поточну мову

параметр:
- `id` : ідентифікатор цільової мови
