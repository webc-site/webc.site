# parentScroll.js

Отримайте елемент-контейнер прокрутки предка

## default(el)

Перейдіть угору від зазначеного елемента, знайдіть і поверніть елемент контейнера прокручування, який його містить (наприклад, вбудований контейнер прокрутки ShadowRoot `C-VS`)

параметр:
- `el` : початковий елемент DOM

Повернене значення: елемент DOM контейнера прокручування
