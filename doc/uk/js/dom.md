## newEl

Створення елементів DOM

параметр:
- `tagName` : назва тегу елемента

Повернене значення: щойно створений елемент DOM

## newLi(...tag_li)

Пакетне створення кількох елементів DOM

параметр:
- `tag_li` : список імен тегів

Повернене значення:
- Масив, що містить усі щойно створені елементи DOM

## cE(name, cls)

Зареєструйте спеціальні елементи веб-компонентів

параметр:
- `name`: ім'я спеціального елемента, префікс `c-` буде додано автоматично
- `cls` : клас спеціального елемента
