# c-t.js

Інтернаціоналізація HTML

Тег `<c-t>key</c-t>` настроюється, а `key` є ключем мовного пакета.

## cTranSet(el, genUrl)

Установіть текст для `c-t`.

параметр:
- `el`: кореневий елемент, глобальний можна передати в `document`
- `genUrl` : функція для отримання словника мовного пакету
  - параметр
    - `code` : код мови
  - Значення, що повертається: об’єкт словника мовного пакету або Promise, який розв’язує цей об’єкт
