# icoTip.js

Визначте та зареєструйте компонент піктограми з підказками щодо інтернаціоналізації

## default(i18n_url, tag, key)

Зареєструйте спеціальний компонент HTML-елемента за допомогою бульбашок інтернаціоналізації

параметр:
- `i18n_url` : адреса запиту для налаштування інтернаціоналізації
- `tag` : налаштуйте назву тегу елемента HTML
- `key` : Ключове значення тексту підказки в словнику інтернаціоналізації
