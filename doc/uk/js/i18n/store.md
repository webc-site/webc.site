# i18n/store.js

Багатомовне керування локальним сховищем

## get()

Отримайте ідентифікатор мови за замовчуванням, розрахований у локальному кеш-пам’яті або розрахований за допомогою відповідності мови браузера

Повернене значення: ID мови

## set(id)

Кешувати ідентифікатор поточної мови локально

параметр:
- `id` : ідентифікатор мови
