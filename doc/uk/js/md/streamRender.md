# md/streamRender.js

Потокове рендеринг Markdown

## default(container, source)

Потокове відтворення тексту Markdown у вказаний контейнер DOM, підтримка ефектів курсору друкарської машинки та автоматичного прокручування

параметр:
- `container` : Елемент контейнера DOM, що містить відтворений вміст
- `source` : джерело текстових даних ReadableStream

Повернене значення:

- `cancel` : функція зупинки відтворення та читання потоку даних
- `setStop` : функція для встановлення, чи призупиняти автоматичне прокручування
  - параметр
    - `val` : логічне значення, чи потрібно призупиняти чи ні
