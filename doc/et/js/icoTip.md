# icoTip.js

Määratlege ja registreerige rahvusvahelistumise vihjetega ikoonikomponent

## default(i18n_url, tag, key)

Registreerige kohandatud HTML-i elemendi komponent rahvusvahelistumise viipu mullidega

parameeter:
- `i18n_url` : Küsi aadressi rahvusvahelistumise konfigureerimiseks
- `tag` : kohandage HTML-elemendi sildi nime
- `key` : rahvusvahelistumise sõnaraamatu viipa teksti võtmeväärtus
