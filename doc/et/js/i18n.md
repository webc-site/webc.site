# i18n.js

Mitmekeelne olekuhaldus ja muudatuste jälgimine

## LANG_LI

Saadaolevate keelekonfiguratsioonide loend

## lang()

Hankige praegune keele ID

Tagastatav väärtus: praeguse keele ID

## langCode()

Hankige praegune keelekood

Tagastusväärtus: praegune keelekood, näiteks `"zh"`

## onLang(func)

Keelemuutuste jaoks siduge kuulamise tagasihelistamine

parameeter:
- `func` : tagasihelistamise funktsioon, kui keel muutub
  - parameeter
    - `code` : keelekood
    - `id` : keele ID

Tagastusväärtus: sidumata puhastusfunktsioon

## langSet(id)

Vahetage ja edastage praegust keelt

parameeter:
- `id` : sihtkeele ID
