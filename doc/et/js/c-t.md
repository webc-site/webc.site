# c-t.js

HTML-i rahvusvahelistumine

Silt `<c-t>key</c-t>` on kohandatud ja `key` on keelepaketi võti.

## cTranSet(el, genUrl)

Määrake `c-t` tekst.

parameeter:
- `el` : juurelement, globaalse saab edasi anda elemendis `document`
- `genUrl` : funktsioon keelepaketi sõnastiku hankimiseks
  - parameeter
    - `code` : keelekood
  - Tagastusväärtus: keelepaketi sõnastiku objekt või lubadus, mis lahendab selle objekti
