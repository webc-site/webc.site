# fetch.js

وظيفة أداة طلب الشبكة تعتمد على تغليف الجلب

## F(url, opt)

إرسال طلب شبكة وإرجاع كائن استجابة

المعلمة:
- `url` : عنوان الطلب
- `opt` : كائن الخيارات `fetch`

قيمة الإرجاع: كائن الاستجابة

## f(attr)

دالة ذات ترتيب أعلى تقوم بتغليف الطلب وتحليل المحتوى الذي تم إرجاعه بتنسيق محدد

المعلمة:
- `attr` : اسم سمة تحليل نص الاستجابة

قيمة الإرجاع: وظيفة لطلب المحتوى وتحليله بشكل غير متزامن
  - المعلمة
    - `url` : عنوان الطلب
    - `opt` : كائن الخيارات `fetch`
  - قيمة الإرجاع: المحتوى الذي تم تحليله

## fTxt

أرسل طلب شبكة وأعد نص الاستجابة بتنسيق نصي

المعلمة:
- `url` : عنوان الطلب
- `opt` : كائن الخيارات `fetch`

قيمة الإرجاع: نص الاستجابة

## fJson(url, opt)

أرسل طلب شبكة وقم بتحليل وإرجاع نص الاستجابة بتنسيق JSON

المعلمة:
- `url` : عنوان الطلب
- `opt` : كائن الخيارات `fetch`

قيمة الإرجاع: كائن JSON الذي تم تحليله

## fBin(url, opt)

إرسال طلب شبكة وإرجاع نص الاستجابة كمصفوفة بايت ثنائية

المعلمة:
- `url` : عنوان الطلب
- `opt` : كائن الخيارات `fetch`

قيمة الإرجاع: صفيف البايت
