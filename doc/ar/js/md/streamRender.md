# md/streamRender.js

عرض تدفق Markdown

## default(container, source)

عرض متدفق لنص Markdown في حاوية DOM المحددة، ودعم تأثيرات مؤشر الآلة الكاتبة والتمرير التلقائي

المعلمة:
- `container` : عنصر حاوية DOM الذي يستضيف المحتوى المعروض
- `source` : مصدر البيانات النصية ReadableStream

قيمة الإرجاع:

- `cancel` : وظيفة إيقاف العرض وقراءة دفق البيانات
- `setStop` : وظيفة لتعيين ما إذا كان سيتم إيقاف التمرير التلقائي مؤقتًا أم لا
  - المعلمة
    - `val` : قيمة منطقية للإيقاف المؤقت أم لا
