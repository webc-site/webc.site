# fetchLang.js

تحميل حزم اللغات بشكل غير متزامن وربط عمليات الاسترجاعات للتحديث

## fetchLang(getDict)

قم بإنشاء مستمع لحزمة اللغة للحصول على حزمة اللغة بشكل غير متزامن وتنفيذ رد اتصال التحديث عندما تتغير اللغة

المعلمة:
- `getDict` : احصل على الوظيفة غير المتزامنة لحزمة اللغة
  - المعلمة
    - `code` : رمز اللغة
  - قيمة الإرجاع: قاموس حزمة اللغة

قيمة الإرجاع: وظيفة تسجيل رد اتصال التحديث
- المعلمة
  - `refresh` : تحديث وظيفة رد الاتصال عند تحديث اللغة
    - المعلمة
      - `i18n` : كائن قاموس حزمة اللغة
      - `code` : رمز اللغة الحالي
- قيمة الإرجاع: وظيفة إلغاء تسجيل رد الاتصال التحديث
