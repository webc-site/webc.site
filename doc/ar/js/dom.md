# dom.js

توثيق الاختصارات المتغيرة ووظائف الاختصار

## D

`document` مرجع مختصر للكائن

## B

`document.body` مرجع مختصر للكائن

## newEl

إنشاء عناصر DOM

المعلمة:
- `tagName` : اسم علامة العنصر

قيمة الإرجاع: عنصر DOM تم إنشاؤه حديثًا

## newLi(...tag_li)

إنشاء عناصر DOM متعددة على دفعات

المعلمة:
- `tag_li` : قائمة أسماء العلامات

قيمة الإرجاع:
- مصفوفة تحتوي على جميع عناصر DOM التي تم إنشاؤها حديثًا

## cE(name, cls)

تسجيل العناصر المخصصة لمكونات الويب

المعلمة:
- `name` : سيتم إضافة اسم العنصر المخصص، `c-` البادئة تلقائيًا
- `cls` : فئة العنصر المخصص
