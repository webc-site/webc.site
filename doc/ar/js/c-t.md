# c-t.js

تدويل HTML

تم تخصيص العلامة `<c-t>key</c-t>`، و`key` هو مفتاح حزمة اللغة.

## cTranSet(el, genUrl)

قم بتعيين النص لـ `c-t`.

المعلمة:
- `el` : العنصر الجذري، يمكن تمريره إلى `document`
- `genUrl` : وظيفة للحصول على قاموس حزمة اللغة
  - المعلمة
    - `code` : رمز اللغة
  - قيمة الإرجاع: كائن قاموس حزمة اللغة، أو وعد يحل هذا الكائن
