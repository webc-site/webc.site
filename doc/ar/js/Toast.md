# Toast.js

نخب رسالة منبثقة موجه

## toast(render, timeout)

إظهار النافذة المنبثقة للمطالبة القياسية

المعلمة:
- `render` : وظيفة العرض المستخدمة لتخصيص المحتوى المنبثق
  - المعلمة
    - `el` : عنصر DOM للنافذة المنبثقة
- `timeout` : تأخير ثواني للإغلاق التلقائي، الافتراضي 9، لا يوجد إيقاف تلقائي عند 0

قيمة الإرجاع: عنصر DOM للنافذة المنبثقة

## toastErr(render, timeout)

إظهار رسالة الخطأ المنبثقة

المعلمة:
- `render` : وظيفة العرض المستخدمة لتخصيص المحتوى المنبثق
  - المعلمة
    - `el` : عنصر DOM للنافذة المنبثقة
- `timeout` : تأخير ثواني للإغلاق التلقائي، الافتراضي 9، لا يوجد إيقاف تلقائي عند 0

قيمة الإرجاع: عنصر DOM في نافذة الخطأ المنبثقة
