# hashActive.js

تسليط الضوء على رابط التنقل بناءً على نقاط ربط URL وتمرير إطار العرض

## default(link_li, cls)

قم تلقائيًا بتمييز روابط التنقل المقابلة للمنطقة المرئية حاليًا بناءً على موضع تمرير الصفحة وتغييرات نقطة ربط عنوان URL.

المعلمة:
- `link_li` : قائمة عناصر DOM لرابط التنقل
- `cls` : اسم فئة CSS النشط، الافتراضي `"A"`

قيمة الإرجاع: إلغاء كافة وظائف تنظيف الاستماع
