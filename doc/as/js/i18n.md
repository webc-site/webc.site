# i18n.js

বহুভাষাৰ অৱস্থা ব্যৱস্থাপনা আৰু পৰিৱৰ্তন নিৰীক্ষণ

## LANG_LI

উপলব্ধ ভাষা বিন্যাসসমূহৰ তালিকা

## lang()

বৰ্তমানৰ ভাষা আইডি লাভ কৰক

ৰিটাৰ্ণ মান: বৰ্তমান ভাষা আইডি

## langCode()

বৰ্তমানৰ ভাষা ক'ড লাভ কৰক

ঘূৰাই দিয়া মান: বৰ্তমান ভাষা ক'ড, যেনে `"zh"`

## onLang(func)

ভাষা পৰিবৰ্তনৰ বাবে শুনা কলবেক বান্ধক

প্ৰাচল:
- `func` : ভাষা সলনি হ'লে কলবেক ফাংচন
  - প্ৰাচল
    - `code` : ভাষাৰ ক'ড
    - `id` : ভাষাৰ আইডি

ৰিটাৰ্ণ মান: আনবাউণ্ড ক্লিনআপ ফাংচন

## langSet(id)

বৰ্তমানৰ ভাষা সলনি কৰক আৰু সম্প্ৰচাৰ কৰক

প্ৰাচল:
- `id` : লক্ষ্য ভাষাৰ আইডি
