# c-t.js

HTML আন্তৰ্জাতিককৰণ

`<c-t>key</c-t>` টেগটো স্বনিৰ্বাচিত কৰা হৈছে, আৰু `key` ভাষা পেকৰ চাবি।

## cTranSet(el, genUrl)

`${b id="1"}`ৰ বাবে টেক্সট নিৰ্ধাৰণ কৰক।

প্ৰাচল:
- `el` : ৰূট উপাদান, গ্লোবেল `document` ত পাছ কৰিব পাৰি
- `${b id="1"}` : ভাষা পেকেজৰ অভিধান লাভ কৰাৰ ফাংচন
  - প্ৰাচল
    - `code` : ভাষাৰ ক'ড
  - ৰিটাৰ্ণ মান: এটা ভাষা পেক অভিধান বস্তু, বা এটা প্ৰতিশ্ৰুতি যি এই বস্তুৰ সৈতে সমাধান কৰে
