# Toast.js

টোষ্ট পপ-আপ বাৰ্তা প্ৰমপ্ট

## toast(render, timeout)

প্ৰামাণিক প্ৰমপ্ট পপ-আপ উইন্ডো দেখুৱাওক

প্ৰাচল:
- `render` : ৰেণ্ডাৰিং ফাংচন, পপ-আপ বিষয়বস্তু স্বনিৰ্বাচিত কৰিবলে ব্যৱহাৰ কৰা হয়
  - প্ৰাচল
    - `el` : পপ-আপ উইণ্ড'ৰ DOM উপাদান
- `timeout` : স্বয়ংক্ৰিয় বন্ধ কৰাৰ বাবে চেকেণ্ড বিলম্ব কৰক, অবিকল্পিত 9, কোনো স্বয়ংক্ৰিয় বন্ধ যেতিয়া 0

ৰিটাৰ্ণ মান: পপ-আপ উইণ্ড'ৰ DOM উপাদান

  - প্ৰাচল
    - `el` : পপ-আপ উইণ্ড'ৰ DOM উপাদান
- `timeout` : স্বয়ংক্ৰিয় বন্ধ কৰাৰ বাবে চেকেণ্ড বিলম্ব কৰক, অবিকল্পিত 9, কোনো স্বয়ংক্ৰিয় বন্ধ যেতিয়া 0

ৰিটাৰ্ণ মান: ভুল পপ-আপ উইণ্ড'ৰ DOM উপাদান
