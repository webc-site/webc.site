# fetch.js

নে'টৱৰ্ক অনুৰোধ সঁজুলি কাৰ্য্য ফেচ এনকেপচুলেচনৰ ওপৰত ভিত্তি কৰি

## F(url, opt)

এটা নে'টৱৰ্ক অনুৰোধ পঠাওক আৰু এটা সঁহাৰি বস্তু ঘূৰাই দিয়ক

প্ৰাচল:
- `url` : অনুৰোধ ঠিকনা
- `opt` : `fetch` বিকল্প বস্তু

ৰিটাৰ্ণ মান: প্ৰতিক্ৰিয়া বস্তু

## f(attr)

এটা উচ্চ-ক্ৰমৰ ফলন যি এটা অনুৰোধ ৰেপ কৰে আৰু ঘূৰাই দিয়া বিষয়বস্তু এটা ধাৰ্য্য কৰা বিন্যাসত বিশ্লেষণ কৰে

প্ৰাচল:
- `attr` : সঁহাৰিৰ মূখ্য অংশ বিশ্লেষণ বৈশিষ্ট্যৰ নাম

ৰিটাৰ্ণ মান: এচিনক্ৰ'নাছলি অনুৰোধ আৰু বিষয়বস্তু বিশ্লেষণ কৰিবলৈ ফাংচন
  - প্ৰাচল
    - `url` : অনুৰোধ ঠিকনা
    - `opt` : `fetch` বিকল্প বস্তু
  - ৰিটাৰ্ণ মান: বিশ্লেষণ কৰা বিষয়বস্তু

এটা নে'টৱৰ্ক অনুৰোধ পঠাওক আৰু সঁহাৰিৰ মূখ্য অংশ লিখনী বিন্যাসত ঘূৰাই দিয়ক

প্ৰাচল:

ৰিটাৰ্ণ মান: সঁহাৰি লিখনী

প্ৰাচল:
- `url` : অনুৰোধ ঠিকনা

ৰিটাৰ্ণ মান: বিশ্লেষণ কৰা JSON বস্তু

## fBin(url, opt)

এটা নে'টৱৰ্ক অনুৰোধ পঠাওক আৰু সঁহাৰিৰ মূখ্য অংশ এটা বাইনাৰী বাইট এৰে হিচাপে ঘূৰাই দিয়ক

প্ৰাচল:
- `url` : অনুৰোধ ঠিকনা
- `opt` : `fetch` বিকল্প বস্তু

ৰিটাৰ্ণ মান: বাইট এৰে
