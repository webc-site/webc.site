# fetchLang.js

ভাষা পেকসমূহ এচিনক্ৰ'নাছভাৱে লোড কৰক আৰু আপডেইট কলবেকসমূহ বান্ধক

## fetchLang(getDict)

ভাষা পেক এচিনক্ৰ'নাছলি লাভ কৰিবলে আৰু ভাষা সলনি হ'লে এটা সতেজ কলবেক এক্সিকিউট কৰিবলে এটা ভাষা পেক লিষ্টাৰ সৃষ্টি কৰক

প্ৰাচল:
- `getDict` : ভাষা সৰঞ্জামৰ এচিনক্ৰ’নাছ ফাংচন পাওক
  - প্ৰাচল
    - `code` : ভাষাৰ ক'ড
  - ৰিটাৰ্ণ মান: ভাষা পেক অভিধান

ৰিটাৰ্ণ মান: সতেজ কলবেক পঞ্জীয়ন কৰিবলৈ ফাংচন
- প্ৰাচল
  - `refresh` : ভাষা আপডেইট কৰাৰ সময়ত কলবেক ফলন সতেজ কৰক
    - প্ৰাচল
      - `i18n` : ভাষা পেক অভিধান বস্তু
      - `code` : বৰ্তমান ভাষাৰ ক'ড
- ৰিটাৰ্ণ মান: সতেজ কলবেক আনপঞ্জীয়ন কৰিবলৈ ফাংচন
