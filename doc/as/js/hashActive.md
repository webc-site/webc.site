# hashActive.js

URL এংকৰ আৰু দৰ্শনপৰ্ট স্ক্ৰলিঙৰ ওপৰত ভিত্তি কৰি নেভিগেচন লিংক হাইলাইটিং

## default(link_li, cls)

পৃষ্ঠা স্ক্ৰল অৱস্থান আৰু URL এংকৰ পইণ্ট পৰিবৰ্তনৰ ওপৰত ভিত্তি কৰি বৰ্তমানে দৃশ্যমান অঞ্চলৰ সৈতে সংগতি ৰাখি নেভিগেচন সংযোগসমূহ স্বয়ংক্ৰিয়ভাৱে হাইলাইট কৰক।

প্ৰাচল:
- `link_li` : নেভিগেচন লিংক DOM উপাদান তালিকা
- `cls` : সক্ৰিয় CSS শ্ৰেণীৰ নাম, অবিকল্পিত `"A"`

ঘূৰাই দিয়া মান: সকলো শুনা পৰিষ্কাৰ কাৰ্য্য বাতিল কৰক
