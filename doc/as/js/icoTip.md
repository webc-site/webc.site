# icoTip.js

আন্তৰ্জাতিককৰণৰ ইংগিতৰ সৈতে এটা আইকন উপাদান সংজ্ঞায়িত আৰু পঞ্জীয়ন কৰক

## default(i18n_url, tag, key)

আন্তৰ্জাতিককৰণ প্ৰমপ্ট বাবলসমূহৰ সৈতে এটা স্বনিৰ্বাচিত HTML উপাদান উপাদান পঞ্জীয়ন কৰক

প্ৰাচল:
- `i18n_url` : আন্তৰ্জাতিককৰণ বিন্যাসৰ বাবে অনুৰোধ ঠিকনা
- `tag` : HTML উপাদানৰ টেগ নাম স্বনিৰ্বাচিত কৰক
- `key` : আন্তৰ্জাতিককৰণ অভিধানত প্ৰমপ্ট লিখনীৰ মূল মান
