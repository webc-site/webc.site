# dom.js

চলক সংক্ষিপ্তকৰণ আৰু চৰ্টকাট ফাংচনসমূহ নথিভুক্ত কৰক

## D

`document` বস্তুটোৰ চৰ্টকাট প্ৰসংগ

## B

`document.body` বস্তুটোৰ চৰ্টকাট প্ৰসংগ

## newEl

DOM উপাদান সৃষ্টি কৰক

প্ৰাচল:
- `tagName` : উপাদান টেগ নাম

ৰিটাৰ্ণ মান: নতুনকৈ সৃষ্টি কৰা DOM উপাদান

## newLi(...tag_li)

বেচসমূহত একাধিক DOM উপাদান সৃষ্টি কৰক

প্ৰাচল:
- `tag_li` : টেগ নামৰ তালিকা

ৰিটাৰ্ণ মান:
- নতুনকৈ সৃষ্টি কৰা সকলো DOM উপাদান থকা এৰে

## cE(name, cls)

ৱেব উপাদানসমূহ স্বনিৰ্বাচিত উপাদানসমূহ পঞ্জীয়ন কৰক

প্ৰাচল:
- `name` : স্বনিৰ্বাচিত উপাদানৰ নাম, `c-` উপসৰ্গ স্বয়ংক্ৰিয়ভাৱে যোগ কৰা হ'ব
- `cls` : স্বনিৰ্বাচিত উপাদানৰ শ্ৰেণী
