# hashActive.js

Navigasieskakelverligting gebaseer op URL-ankers en deurkykpoort-blaai

## default(link_li, cls)

Merk outomaties navigasieskakels wat ooreenstem met die tans sigbare area gebaseer op bladsy-rolposisie en URL-ankerpuntveranderinge.

parameter:
- `link_li` : Navigasieskakel DOM element lys
- `cls` : Aktiewe CSS-klasnaam, verstek `"A"`

Retourwaarde: Kanselleer alle luisteropruimfunksies
