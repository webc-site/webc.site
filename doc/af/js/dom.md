## newEl

Skep DOM-elemente

parameter:
- `tagName` : elementmerkernaam

Terugkeerwaarde: nuutgeskepte DOM-element

## newLi(...tag_li)

Skep veelvuldige DOM-elemente in bondels

parameter:
- `tag_li` : merkernaamlys

Terugkeerwaarde:
- Skikking wat alle nuutgeskepte DOM-elemente bevat

## cE(name, cls)

Registreer webkomponente pasgemaakte elemente

parameter:
- `name` : Pasgemaakte elementnaam, `c-` voorvoegsel sal outomaties bygevoeg word
- `cls` : die klas van die pasgemaakte element
