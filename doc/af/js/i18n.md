# i18n.js

Multi-taal status bestuur en verandering monitering

## LANG_LI

Lys van beskikbare taalkonfigurasies

## lang()

Kry huidige taal-ID

Retourwaarde: huidige taal-ID

## langCode()

Kry die huidige taalkode

Retourwaarde: huidige taalkode, soos `"zh"`

## onLang(func)

Bind die luisterterugroep vir taalveranderinge

parameter:
- `func` : terugbelfunksie wanneer taal verander
  - parameter
    - `code` : Taalkode
    - `id` : Taal-ID

Terugkeerwaarde: Ongebonde opruimfunksie

## langSet(id)

Wissel en saai die huidige taal uit

parameter:
- `id` : Doeltaal-ID
