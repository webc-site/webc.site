# i18n.js

Pamamahala ng katayuan sa maraming wika at pagsubaybay sa pagbabago

## LANG_LI

Listahan ng mga magagamit na configuration ng wika

## lang()

Kunin ang kasalukuyang language ID

Return value: kasalukuyang language ID

## langCode()

Kunin ang kasalukuyang code ng wika

Return value: kasalukuyang code ng wika, gaya ng `"zh"`

## onLang(func)

Itali ang callback sa pakikinig para sa mga pagbabago sa wika

parameter:
- `func` : callback function kapag nagbago ang wika
  - parameter
    - `code` : Code ng wika
    - `id` : Language ID

Return value: Unbound cleanup function

## langSet(id)

Lumipat at i-broadcast ang kasalukuyang wika

parameter:
- `id` : Target na ID ng wika
