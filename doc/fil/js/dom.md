# dom.js

Idokumento ang mga variable na pagdadaglat at mga function ng shortcut

## D

`document` Shortcut reference sa object

## B

`document.body` Shortcut reference sa object

## css(txt, root)

## newEl

Lumikha ng mga elemento ng DOM

parameter:
- `tagName` : pangalan ng tag ng elemento

Return value: bagong likhang elemento ng DOM

## newLi(...tag_li)

Gumawa ng maraming elemento ng DOM sa mga batch

parameter:
- `tag_li` : listahan ng pangalan ng tag

Ibinalik na halaga:
- Array na naglalaman ng lahat ng bagong likhang elemento ng DOM

## cE(name, cls)

Magrehistro ng mga custom na elemento ng Web Components

parameter:
- `name` : Ang pangalan ng custom na elemento, `c-` prefix ay awtomatikong idaragdag
- `cls` : ang klase ng custom na elemento
