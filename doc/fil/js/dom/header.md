# dom/header.js

Lumikha ng istraktura ng DOM ng header bar

## default()

Gumawa at pagsamahin ang header na istraktura ng DOM ng mga tag ng header, h1 at b

Ibinalik na halaga:

- `header` : Container element ng header bar
- `h1` : elemento ng pamagat
- `b` : subtitle o karagdagang elemento ng impormasyon
