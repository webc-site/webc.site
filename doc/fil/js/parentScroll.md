# parentScroll.js

Kunin ang ancestor scroll container element

## default(el)

Tumawid pataas mula sa tinukoy na elemento, hanapin at ibalik ang scroll container na elemento na naglalaman nito (gaya ng ShadowRoot built-in na scroll container ng `C-VS`)

parameter:
- `el` : pagsisimula ng elemento ng DOM

Ibinalik na halaga: scroll container DOM na elemento
