# icoTip.js

Tukuyin at irehistro ang isang bahagi ng icon na may mga pahiwatig sa internasyonalisasyon

## default(i18n_url, tag, key)

Magrehistro ng custom na HTML element component na may mga internationalization prompt bubble

parameter:
- `i18n_url` : Humiling ng address para sa pagsasaayos ng internationalization
- `tag` : I-customize ang pangalan ng tag ng elemento ng HTML
- `key` : Ang pangunahing halaga ng prompt na teksto sa diksyunaryo ng internasyonalisasyon
