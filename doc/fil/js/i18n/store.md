# i18n/store.js

Pamamahala ng lokal na imbakan ng maraming wika

## get()

Kunin ang default na ID ng wika na nakalkula nang lokal na naka-cache o nakalkula sa pamamagitan ng pagtutugma ng wika ng browser

Return value: language ID

## set(id)

I-cache nang lokal ang kasalukuyang ID ng wika

parameter:
- `id` : Language ID
