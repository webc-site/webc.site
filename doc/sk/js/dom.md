## newEl

Vytvorte prvky DOM

parameter:
- `tagName` : názov značky prvku

Návratová hodnota: novovytvorený prvok DOM

## newLi(...tag_li)

Vytvorte viacero prvkov DOM v dávkach

parameter:
- `tag_li` : zoznam názvov značiek

Návratová hodnota:
- Pole obsahujúce všetky novovytvorené prvky DOM

## cE(name, cls)

Zaregistrujte vlastné prvky Web Components

parameter:
- `name` : Názov vlastného prvku, predpona `c-` sa pridá automaticky
- `cls` : trieda vlastného prvku
