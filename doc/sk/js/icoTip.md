# icoTip.js

Definujte a zaregistrujte komponent ikony s internacionalizačnými radami

## default(i18n_url, tag, key)

Zaregistrujte si vlastný komponent prvku HTML pomocou bublín s výzvou na internacionalizáciu

parameter:
- `i18n_url` : Žiadosť o adresu pre konfiguráciu internacionalizácie
- `tag` : Prispôsobte názov značky prvku HTML
- `key` : Kľúčová hodnota textu výzvy v slovníku internacionalizácie
