# i18n.js

Viacjazyčná správa stavu a monitorovanie zmien

## LANG_LI

Zoznam dostupných jazykových konfigurácií

## lang()

Získajte aktuálne ID jazyka

Návratová hodnota: ID aktuálneho jazyka

## langCode()

Získajte aktuálny kód jazyka

Návratová hodnota: aktuálny kód jazyka, napríklad `"zh"`

## onLang(func)

Prepojte spätné počúvanie pre zmeny jazyka

parameter:
- `func` : funkcia spätného volania pri zmene jazyka
  - parameter
    - `code` : Kód jazyka
    - `id` : ID jazyka

Návratová hodnota: Neviazaná funkcia čistenia

## langSet(id)

Prepnite a vysielajte aktuálny jazyk

parameter:
- `id` : ID cieľového jazyka
