# i18n/store.js

Viacjazyčná správa lokálneho úložiska

## get()

Získajte predvolené ID jazyka vypočítané lokálne vo vyrovnávacej pamäti alebo vypočítané podľa zhody jazyka prehliadača

Návratová hodnota: ID jazyka

## set(id)

Lokálne uložte ID aktuálneho jazyka do vyrovnávacej pamäte

parameter:
- `id` : ID jazyka
