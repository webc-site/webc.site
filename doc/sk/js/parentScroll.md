# parentScroll.js

Získajte prvok kontajnera rolovania predkov

## default(el)

Prejdite smerom nahor od určeného prvku, nájdite a vráťte prvok kontajnera posúvania, ktorý ho obsahuje (napríklad vstavaný kontajner posúvania `C-VS` ShadowRoot)

parameter:
- `el` : počiatočný prvok DOM

Návratová hodnota: prvok DOM rolovacieho kontajnera
