# i18n/store.js

பன்மொழி உள்ளூர் சேமிப்பு மேலாண்மை

## get()

உள்ளூரில் தேக்ககமாகக் கணக்கிடப்பட்ட அல்லது உலாவி மொழிப் பொருத்தத்தின் மூலம் கணக்கிடப்பட்ட இயல்புநிலை மொழி ஐடியைப் பெறுங்கள்

வருவாய் மதிப்பு: மொழி ஐடி

## set(id)

தற்போதைய மொழி ஐடியை உள்நாட்டில் தேக்ககப்படுத்தவும்

அளவுரு:
- `id` : மொழி ஐடி
