# route.js

ஒற்றை பக்க முன்-இறுதி ரூட்டிங் மேலாண்மை

## nowUrl()

முன்னணி ஸ்லாஷ் அகற்றப்பட்ட தற்போதைய URL பாதையைப் பெறவும்

திரும்ப மதிப்பு: தற்போதைய பாதை சரம்

## route(hook)

பாதை மாற்றங்களுக்கு கேட்கும் அழைப்பைப் பதிவு செய்யவும். பதிவு செய்யும் போது, ​​தற்போதைய பாதையில் ஒரு முறை செயல்படுத்தப்படும்.

அளவுரு:
- `hook` : ரூட் கண்காணிப்பு கால்பேக் செயல்பாடு
  - அளவுரு
    - `url` : தற்போதைய பாதை
    - `pre` : கடைசி பாதை

திரும்ப மதிப்பு: பதிவு செய்யப்படாத தூய்மைப்படுத்தும் செயல்பாடு

## setPre(url)

கடைசியாக பதிவு செய்யப்பட்ட URL ஐ சேமிக்கவும்

அளவுரு:
- `url` : பாதை சரம்

## preUrl()

கடைசி பதிவின் URL ஐப் பெறவும்

திரும்ப மதிப்பு: கடைசி பாதை சரம்

## refresh(url)

அனைத்து வழி கேட்கும் கால்பேக்குகளையும் குறிப்பிட்ட பாதையில் தூண்டி, முன்பே புதுப்பிக்கவும்

அளவுரு:
- `url` : இலக்கு பாதை, இயல்புநிலை தற்போதைய பாதை

## removeSlash(url)

பாதை தலைப்புகளில் இருந்து சாய்வுகளை அகற்றவும்

அளவுரு:
- `url` : பாதை சரம்

திரும்ப மதிப்பு: முன்னணி சாய்வை அகற்றிய பின் பாதை

## setUrl(url)

உலாவி முகவரிப் பட்டியின் URL நிலையைப் புதுப்பிக்கவும்

அளவுரு:
- `url` : இலக்கு பாதை

திரும்ப மதிப்பு: 1 பாதை உண்மையில் மாறினால்

## goto(url)

உலாவி முகவரியைப் புதுப்பித்து, பாதை புதுப்பிப்பைத் தூண்டவும்

அளவுரு:
- `url` : இலக்கு பாதை
