# hashActive.js

URL அறிவிப்பாளர்கள் மற்றும் வியூபோர்ட் ஸ்க்ரோலிங் அடிப்படையில் வழிசெலுத்தல் இணைப்பு தனிப்படுத்தல்

## default(link_li, cls)

பக்க ஸ்க்ரோல் நிலை மற்றும் URL ஆங்கர் புள்ளி மாற்றங்களின் அடிப்படையில் தற்போது காணக்கூடிய பகுதியுடன் தொடர்புடைய வழிசெலுத்தல் இணைப்புகளை தானாகவே முன்னிலைப்படுத்தவும்.

அளவுரு:
- `link_li` : வழிசெலுத்தல் இணைப்பு DOM உறுப்பு பட்டியல்
- `cls` : செயலில் உள்ள CSS வகுப்பின் பெயர், இயல்புநிலை `"A"`

திரும்ப மதிப்பு: அனைத்து கேட்கும் தூய்மைப்படுத்தும் செயல்பாடுகளையும் ரத்துசெய்யவும்
