# c-t.js

HTML சர்வதேசமயமாக்கல்

`<c-t>key</c-t>` குறிச்சொல் தனிப்பயனாக்கப்பட்டது, மேலும் `key` என்பது மொழிப் பொதியின் திறவுகோலாகும்.

## cTranSet(el, genUrl)

`c-t`க்கு உரையை அமைக்கவும்.

அளவுரு:
- `el` : ரூட் உறுப்பு, குளோபல் என்பதை `document`ல் அனுப்பலாம்
- `genUrl` : மொழி தொகுப்பு அகராதியைப் பெறுவதற்கான செயல்பாடு
  - அளவுரு
    - `code` : மொழி குறியீடு
  - திரும்பப்பெறும் மதிப்பு: மொழிப் பொதி அகராதிப் பொருள் அல்லது இந்தப் பொருளைத் தீர்க்கும் வாக்குறுதி
