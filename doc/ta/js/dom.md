# dom.js

ஆவண மாறி சுருக்கங்கள் மற்றும் குறுக்குவழி செயல்பாடுகள்

## D

`document` பொருளின் குறுக்குவழி குறிப்பு

## B

`document.body` பொருளின் குறுக்குவழி குறிப்பு

## newEl

DOM கூறுகளை உருவாக்கவும்

அளவுரு:
- `tagName` : உறுப்பு குறிச்சொல் பெயர்

திரும்ப மதிப்பு: புதிதாக உருவாக்கப்பட்ட DOM உறுப்பு

## newLi(...tag_li)

தொகுதிகளில் பல DOM கூறுகளை உருவாக்கவும்

அளவுரு:
- `tag_li` : குறிச்சொல் பெயர் பட்டியல்

வருவாய் மதிப்பு:
- புதிதாக உருவாக்கப்பட்ட அனைத்து DOM கூறுகளையும் கொண்ட அணிவரிசை

## cE(name, cls)

வலை கூறுகளின் தனிப்பயன் கூறுகளை பதிவு செய்யவும்

அளவுரு:
- `name` : தனிப்பயன் உறுப்பு பெயர், `c-` முன்னொட்டு தானாகவே சேர்க்கப்படும்
- `cls` : தனிப்பயன் உறுப்புகளின் வகுப்பு
