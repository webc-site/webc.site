# Toast.js

டோஸ்ட் பாப்-அப் செய்தி வரியில்

## toast(render, timeout)

நிலையான பாப்-அப் சாளரத்தைக் காட்டு

அளவுரு:
- `render` : ரெண்டரிங் செயல்பாடு, பாப்-அப் உள்ளடக்கத்தைத் தனிப்பயனாக்கப் பயன்படுகிறது
  - அளவுரு
    - `el` : பாப்-அப் சாளரத்தின் DOM உறுப்பு
- `timeout` : தானியங்கி பணிநிறுத்தத்திற்கு வினாடிகள் தாமதம், இயல்புநிலை 9, 0 ஆக இருக்கும்போது தானியங்கி பணிநிறுத்தம் இல்லை

திரும்ப மதிப்பு: பாப்-அப் சாளரத்தின் DOM உறுப்பு
