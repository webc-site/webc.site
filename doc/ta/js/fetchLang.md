# fetchLang.js

மொழிப் பொதிகளை ஒத்திசைவின்றி ஏற்றவும் மற்றும் புதுப்பிப்பு அழைப்புகளை பிணைக்கவும்

## fetchLang(getDict)

மொழிப் பொதியை ஒத்திசைவின்றிப் பெறுவதற்கு ஒரு மொழிப் பொதி கேட்பவரை உருவாக்கவும் மற்றும் மொழி மாறும்போது புதுப்பித்த அழைப்பைச் செயல்படுத்தவும்

அளவுரு:
- `getDict` : மொழி தொகுப்பின் ஒத்திசைவற்ற செயல்பாட்டைப் பெறவும்
  - அளவுரு
    - `code` : மொழி குறியீடு
  - வருவாய் மதிப்பு: மொழி தொகுப்பு அகராதி

திரும்ப மதிப்பு: புதுப்பிப்பு அழைப்பைப் பதிவு செய்வதற்கான செயல்பாடு
- அளவுரு
  - `refresh` : மொழி புதுப்பிக்கப்படும் போது, ​​திரும்ப அழைப்பின் செயல்பாட்டைப் புதுப்பிக்கவும்
    - அளவுரு
      - `i18n` : மொழி தொகுப்பு அகராதி பொருள்
      - `code` : தற்போதைய மொழி குறியீடு
- திரும்ப மதிப்பு: புதுப்பித்த அழைப்பை பதிவு நீக்குவதற்கான செயல்பாடு
