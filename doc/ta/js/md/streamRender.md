# md/streamRender.js

மார்க் டவுன் ஸ்ட்ரீமிங் ரெண்டரிங்

## default(container, source)

குறிப்பிட்ட DOM கண்டெய்னரில் மார்க் டவுன் உரையை ஸ்ட்ரீமிங் செய்தல், தட்டச்சுப்பொறி கர்சர் விளைவுகள் மற்றும் தானியங்கி ஸ்க்ரோலிங் ஆதரவு

அளவுரு:
- `container` : DOM கன்டெய்னர் உறுப்பு ரெண்டர் செய்யப்பட்ட உள்ளடக்கத்தை ஹோஸ்டிங் செய்கிறது
- `source` : ReadableStream உரை தரவு ஆதாரம்

வருவாய் மதிப்பு:

- `cancel` : ரெண்டரிங் மற்றும் டேட்டா ஸ்ட்ரீம் வாசிப்பை நிறுத்துவதற்கான செயல்பாடு
- `setStop` : தானியங்கி ஸ்க்ரோலிங்கை இடைநிறுத்த வேண்டுமா என்பதை அமைப்பதற்கான செயல்பாடு
  - அளவுரு
    - `val` : பூலியன் மதிப்பு இடைநிறுத்தப்பட வேண்டுமா இல்லையா
