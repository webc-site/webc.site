# parentScroll.js

மூதாதையர் உருள் கொள்கலன் உறுப்பைப் பெறுங்கள்

## default(el)

குறிப்பிட்ட உறுப்பிலிருந்து மேல்நோக்கிச் செல்லவும், அதைக் கொண்ட உருள் கொள்கலன் உறுப்பைக் கண்டுபிடித்து திருப்பி அனுப்பவும் (`C-VS`ன் ShadowRoot உள்ளமைக்கப்பட்ட உருள் கொள்கலன் போன்றவை)

அளவுரு:
- `el` : DOM உறுப்பைத் தொடங்குதல்

திரும்ப மதிப்பு: உருள் கொள்கலன் DOM உறுப்பு
