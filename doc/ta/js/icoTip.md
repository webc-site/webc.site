# icoTip.js

சர்வதேசமயமாக்கல் குறிப்புகளுடன் ஐகான் கூறுகளை வரையறுத்து பதிவு செய்யவும்

## default(i18n_url, tag, key)

சர்வதேசமயமாக்கல் உடனடி குமிழ்களுடன் தனிப்பயன் HTML உறுப்பு கூறுகளை பதிவு செய்யவும்

அளவுரு:
- `i18n_url` : சர்வதேசமயமாக்கல் உள்ளமைவுக்கான முகவரியைக் கோரவும்
- `tag` : HTML உறுப்பின் குறிச்சொல் பெயரைத் தனிப்பயனாக்கு
- `key` : சர்வதேசமயமாக்கல் அகராதியில் உள்ள உடனடி உரையின் முக்கிய மதிப்பு
