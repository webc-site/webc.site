# i18n.js

பல மொழி நிலை மேலாண்மை மற்றும் மாற்றம் கண்காணிப்பு

## LANG_LI

கிடைக்கக்கூடிய மொழி அமைப்புகளின் பட்டியல்

## lang()

தற்போதைய மொழி ஐடியைப் பெறுங்கள்

திரும்ப மதிப்பு: தற்போதைய மொழி ஐடி

## langCode()

தற்போதைய மொழிக் குறியீட்டைப் பெறவும்

திரும்ப மதிப்பு: தற்போதைய மொழி குறியீடு, அதாவது `"zh"`

## onLang(func)

மொழி மாற்றங்களுக்கு கேட்கும் அழைப்பை இணைக்கவும்

அளவுரு:
- `func` : மொழி மாறும்போது திரும்ப அழைக்கும் செயல்பாடு
  - அளவுரு
    - `code` : மொழி குறியீடு
    - `id` : மொழி ஐடி

திரும்ப மதிப்பு: வரம்பற்ற தூய்மைப்படுத்தும் செயல்பாடு

## langSet(id)

தற்போதைய மொழியை மாற்றி ஒளிபரப்பவும்

அளவுரு:
- `id` : இலக்கு மொழி ஐடி
