# fetch.js

ஃபெட்ச் என்காப்சுலேஷன் அடிப்படையில் நெட்வொர்க் கோரிக்கை கருவி செயல்பாடு

## F(url, opt)

பிணைய கோரிக்கையை அனுப்பவும் மற்றும் மறுமொழி பொருளை திருப்பி அனுப்பவும்

அளவுரு:
- `url` : முகவரியைக் கோரவும்
- `opt` : `fetch` விருப்பங்கள் பொருள்

திரும்ப மதிப்பு: பதில் பொருள்

## f(attr)

ஒரு உயர்-வரிசை செயல்பாடு கோரிக்கையை மூடி, குறிப்பிட்ட வடிவத்தில் திரும்பிய உள்ளடக்கத்தை பாகுபடுத்துகிறது

அளவுரு:
- `attr` : பதில் உடல் பாகுபடுத்தும் பண்புக்கூறு பெயர்

மதிப்பு திரும்ப: ஒத்திசைவின்றி கோரிக்கை மற்றும் உள்ளடக்கத்தை அலசுவதற்கான செயல்பாடு
  - அளவுரு
    - `url` : முகவரியைக் கோரவும்
    - `opt` : `fetch` விருப்பங்கள் பொருள்
  - திரும்ப மதிப்பு: பாகுபடுத்தப்பட்ட உள்ளடக்கம்

## fTxt

பிணைய கோரிக்கையை அனுப்பவும் மற்றும் உரை வடிவத்தில் பதிலளிப்பு உள்ளடக்கத்தை திருப்பி அனுப்பவும்

அளவுரு:
- `url` : முகவரியைக் கோரவும்
- `opt` : `fetch` விருப்பங்கள் பொருள்

திரும்ப மதிப்பு: பதில் உரை

## fJson(url, opt)

பிணைய கோரிக்கையை அனுப்பவும் மற்றும் பாகுபடுத்தவும் மற்றும் பதில் அமைப்பை JSON வடிவத்தில் திருப்பி அனுப்பவும்

அளவுரு:
- `url` : முகவரியைக் கோரவும்
- `opt` : `fetch` விருப்பங்கள் பொருள்

திரும்ப மதிப்பு: பாகுபடுத்தப்பட்ட JSON பொருள்

## fBin(url, opt)

பிணைய கோரிக்கையை அனுப்பவும் மற்றும் மறுமொழியை பைனரி பைட் வரிசையாக திருப்பி அனுப்பவும்

- `opt` : `fetch` விருப்பங்கள் பொருள்
