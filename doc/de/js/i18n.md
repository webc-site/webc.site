# i18n.js

Mehrsprachige Statusverwaltung und Änderungsüberwachung

## LANG_LI

Liste der verfügbaren Sprachkonfigurationen

## lang()

Holen Sie sich die aktuelle Sprach-ID

Rückgabewert: aktuelle Sprach-ID

## langCode()

Holen Sie sich den aktuellen Sprachcode

Rückgabewert: aktueller Sprachcode, z. B. `"zh"`

## onLang(func)

Binden Sie den Listening-Rückruf für Sprachänderungen

Parameter:
- `func`: Rückruffunktion bei Sprachwechsel
  - Parameter
    - `code`: Sprachcode
    - `id`: Sprach-ID

Rückgabewert: Ungebundene Bereinigungsfunktion

## langSet(id)

Wechseln und übertragen Sie die aktuelle Sprache

Parameter:
- `id`: Zielsprachen-ID
