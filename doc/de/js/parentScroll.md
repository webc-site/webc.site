# parentScroll.js

Holen Sie sich das Vorfahren-Scroll-Containerelement

## default(el)

Gehen Sie vom angegebenen Element nach oben, suchen Sie das Scroll-Container-Element, das es enthält, und geben Sie es zurück (z. B. den in ShadowRoot integrierten Scroll-Container von `C-VS`).

Parameter:
- `el`: Start-DOM-Element

Rückgabewert: Scroll-Container-DOM-Element
