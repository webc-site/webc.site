# icoTip.js

Definieren und registrieren Sie eine Symbolkomponente mit Internationalisierungshinweisen

## default(i18n_url, tag, key)

Registrieren Sie eine benutzerdefinierte HTML-Elementkomponente mit Internationalisierungs-Eingabeaufforderungsblasen

Parameter:
- `i18n_url`: Adresse für Internationalisierungskonfiguration anfordern
- `tag`: Passen Sie den Tag-Namen des HTML-Elements an
- `key`: Der Schlüsselwert des Eingabeaufforderungstextes im Internationalisierungswörterbuch
