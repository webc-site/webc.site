# Toast.js

ٹوسٹ پاپ اپ میسج پرامپٹ

## toast(render, timeout)

معیاری پرامپٹ پاپ اپ ونڈو دکھائیں۔

پیرامیٹر:
- `render` : رینڈرنگ فنکشن، جو پاپ اپ مواد کو حسب ضرورت بنانے کے لیے استعمال کیا جاتا ہے۔
  - پیرامیٹر
    - `el` : پاپ اپ ونڈو کا DOM عنصر
- `timeout` : خودکار شٹ ڈاؤن کے لیے سیکنڈ میں تاخیر، ڈیفالٹ 9، 0 ہونے پر کوئی خودکار شٹ ڈاؤن نہیں

واپسی کی قدر: پاپ اپ ونڈو کا DOM عنصر

## toastErr(render, timeout)

پیرامیٹر:
- `render` : رینڈرنگ فنکشن، جو پاپ اپ مواد کو حسب ضرورت بنانے کے لیے استعمال کیا جاتا ہے۔
  - پیرامیٹر
    - `el` : پاپ اپ ونڈو کا DOM عنصر
- `timeout` : خودکار شٹ ڈاؤن کے لیے سیکنڈ میں تاخیر، ڈیفالٹ 9، 0 ہونے پر کوئی خودکار شٹ ڈاؤن نہیں

ریٹرن ویلیو: ایرر پاپ اپ ونڈو کا DOM عنصر
