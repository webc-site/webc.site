# i18n.js

کثیر زبان کی حیثیت کا انتظام اور تبدیلی کی نگرانی

## LANG_LI

دستیاب زبان کی ترتیب کی فہرست

## lang()

موجودہ زبان کی شناخت حاصل کریں۔

واپسی کی قدر: موجودہ زبان کی شناخت

## langCode()

موجودہ زبان کا کوڈ حاصل کریں۔

واپسی کی قدر: موجودہ زبان کا کوڈ، جیسے `"zh"`

## onLang(func)

زبان کی تبدیلیوں کے لیے سننے والے کال بیک کو پابند کریں۔

پیرامیٹر:
- `func` : زبان تبدیل ہونے پر کال بیک فنکشن
  - پیرامیٹر
    - `code` : زبان کا کوڈ
    - `id` : زبان کی شناخت

ریٹرن ویلیو: ان باؤنڈ کلین اپ فنکشن

## langSet(id)

موجودہ زبان کو تبدیل اور نشر کریں۔

پیرامیٹر:
- `id` : ٹارگٹ لینگوئج ID
