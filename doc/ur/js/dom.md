# dom.js

دستاویز متغیر مخففات اور شارٹ کٹ فنکشنز

## D

`document` آبجیکٹ کا شارٹ کٹ حوالہ

## B

`document.body` آبجیکٹ کا شارٹ کٹ حوالہ

## newEl

DOM عناصر بنائیں

پیرامیٹر:
- `tagName` : عنصر ٹیگ کا نام

واپسی کی قدر: نیا تخلیق کردہ DOM عنصر

## newLi(...tag_li)

بیچوں میں متعدد DOM عناصر بنائیں

پیرامیٹر:
- `tag_li` : ٹیگ کے نام کی فہرست

واپسی کی قیمت:
- تمام نئے بنائے گئے DOM عناصر پر مشتمل صف

## cE(name, cls)

ویب اجزاء اپنی مرضی کے عناصر کو رجسٹر کریں۔

پیرامیٹر:
- `name` : حسب ضرورت عنصر کا نام، `c-` سابقہ ​​خود بخود شامل ہو جائے گا۔
- `cls` : حسب ضرورت عنصر کی کلاس
