# hashActive.js

یو آر ایل اینکرز اور ویو پورٹ اسکرولنگ پر مبنی نیویگیشن لنک کو نمایاں کرنا

## default(link_li, cls)

صفحہ اسکرول پوزیشن اور یو آر ایل اینکر پوائنٹ کی تبدیلیوں کی بنیاد پر خودکار طور پر نیویگیشن لنکس کو نمایاں کریں جو فی الحال نظر آنے والے علاقے کے مطابق ہیں۔

پیرامیٹر:
- `link_li` : نیویگیشن لنک DOM عنصر کی فہرست
- `cls` : فعال CSS کلاس کا نام، ڈیفالٹ `"A"`

واپسی کی قیمت: سننے کے تمام کلین اپ فنکشنز کو منسوخ کریں۔
