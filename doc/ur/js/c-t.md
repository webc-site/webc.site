# c-t.js

HTML بین الاقوامی کاری

ٹیگ `<c-t>key</c-t>` کو حسب ضرورت بنایا گیا ہے، اور `key` زبان کے پیک کی کلید ہے۔

## cTranSet(el, genUrl)

`c-t` کے لیے متن سیٹ کریں۔

پیرامیٹر:
- `el` : روٹ عنصر، گلوبل کو `document` میں پاس کیا جا سکتا ہے۔
- `genUrl` : زبان پیک لغت حاصل کرنے کے لیے فنکشن
  - پیرامیٹر
    - `code` : زبان کا کوڈ
  - واپسی کی قیمت: ایک زبان پیک لغت آبجیکٹ، یا ایک وعدہ جو اس اعتراض کو حل کرتا ہے۔
