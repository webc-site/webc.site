# icoTip.js

بین الاقوامی کاری کے اشارے کے ساتھ ایک آئیکن جزو کی وضاحت اور رجسٹر کریں۔

## default(i18n_url, tag, key)

انٹرنیشنلائزیشن پرامپٹ ببلز کے ساتھ ایک حسب ضرورت HTML عنصر کے جزو کو رجسٹر کریں۔

پیرامیٹر:
- `i18n_url` : انٹرنیشنلائزیشن کنفیگریشن کے لیے ایڈریس کی درخواست کریں۔
- `tag` : HTML عنصر کے ٹیگ نام کو حسب ضرورت بنائیں
- `key` : بین الاقوامی کاری لغت میں پرامپٹ ٹیکسٹ کی کلیدی قدر
