# md/streamRender.js

مارک ڈاؤن اسٹریمنگ رینڈرنگ

## default(container, source)

مخصوص DOM کنٹینر میں مارک ڈاؤن ٹیکسٹ کی سٹریمنگ رینڈرنگ، ٹائپ رائٹر کرسر اثرات اور خودکار اسکرولنگ کو سپورٹ کرنا

پیرامیٹر:
- `container` : DOM کنٹینر عنصر پیش کردہ مواد کی میزبانی کرتا ہے۔
- `source` : پڑھنے کے قابل اسٹریم ٹیکسٹ ڈیٹا سورس

واپسی کی قیمت:

- `cancel` : رینڈرنگ اور ڈیٹا اسٹریم ریڈنگ کو روکنے کا فنکشن
- `setStop` : خودکار اسکرولنگ کو روکنا ہے یا نہیں یہ سیٹ کرنے کے لیے فنکشن
  - پیرامیٹر
    - `val` : بولین قدر چاہے توقف کیا جائے یا نہ کیا جائے۔
