# fetch.js

نیٹ ورک کی درخواست کے آلے کا فنکشن بازیافت انکیپسولیشن پر مبنی ہے۔

## F(url, opt)

نیٹ ورک کی درخواست بھیجیں اور جوابی آبجیکٹ واپس کریں۔

پیرامیٹر:
- `url` : ایڈریس کی درخواست کریں۔
- `opt` : `fetch` اختیارات آبجیکٹ

ریٹرن ویلیو: ریسپانس آبجیکٹ

## f(attr)

ایک اعلیٰ ترتیب کا فنکشن جو ایک درخواست کو سمیٹتا ہے اور واپس کیے گئے مواد کو ایک مخصوص فارمیٹ میں پارس کرتا ہے۔

پیرامیٹر:
- `attr` : ریسپانس باڈی پارسنگ انتساب کا نام

واپسی کی قدر: غیر مطابقت پذیری کی درخواست کرنے اور مواد کو پارس کرنے کے لیے فنکشن
  - پیرامیٹر
    - `url` : ایڈریس کی درخواست کریں۔
    - `opt` : `fetch` اختیارات آبجیکٹ
  - واپسی کی قدر: تجزیہ شدہ مواد

## fTxt

نیٹ ورک کی درخواست بھیجیں اور جوابی باڈی کو ٹیکسٹ فارمیٹ میں واپس کریں۔

پیرامیٹر:
- `url` : ایڈریس کی درخواست کریں۔
- `opt` : `fetch` اختیارات آبجیکٹ

واپسی کی قدر: جوابی متن

## fJson(url, opt)

نیٹ ورک کی درخواست بھیجیں اور تجزیہ کریں اور JSON فارمیٹ میں رسپانس باڈی واپس کریں۔

پیرامیٹر:
- `url` : ایڈریس کی درخواست کریں۔
- `opt` : `fetch` اختیارات آبجیکٹ

واپسی کی قدر: تجزیہ کردہ JSON آبجیکٹ

## fBin(url, opt)

نیٹ ورک کی درخواست بھیجیں اور بائنری بائٹ سرنی کے طور پر رسپانس باڈی واپس کریں۔

پیرامیٹر:
- `url` : ایڈریس کی درخواست کریں۔
- `opt` : `fetch` اختیارات آبجیکٹ

واپسی کی قدر: بائٹ سرنی
