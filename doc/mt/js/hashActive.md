# hashActive.js

L-enfasi tal-link tan-navigazzjoni bbażata fuq l-ankri tal-URL u l-iskrolljar tal-viewport

## default(link_li, cls)

Enfasizza awtomatikament links tan-navigazzjoni li jikkorrispondu għaż-żona viżibbli bħalissa abbażi tal-pożizzjoni tal-iscroll tal-paġna u l-bidliet fil-punt tal-ankra tal-URL.

parametru:
- `link_li` : Lista ta' elementi DOM tal-link tan-navigazzjoni
- `cls` : Isem tal-klassi CSS attiv, default `"A"`

Valur tar-ritorn: Ikkanċella l-funzjonijiet kollha tat-tindif tas-smigħ
