# On.js

Monitoraġġ tal-avveniment vinkolanti tal-lott

## On(elem, dict)

Orbot lis-semmiegħa tal-avvenimenti tal-element DOM f'lottijiet

parametru:
- `elem` : Element DOM
- `dict` : Immappjar ta' avvenimenti u callbacks

Valur tar-ritorn: Neħħi l-funzjonijiet kollha tas-semmiegħa tal-avvenimenti marbuta
