# icoTip.js

Iddefinixxi u rreġistra komponent tal-ikona b'ideat ta' internazzjonalizzazzjoni

## default(i18n_url, tag, key)

Irreġistra komponent ta' element HTML personalizzat bi bżieżaq fil-pront għall-internazzjonalizzazzjoni

parametru:
- `i18n_url` : Itlob l-indirizz għall-konfigurazzjoni tal-internazzjonalizzazzjoni
- `tag` : Ippersonalizza l-isem tat-tikketta tal-element HTML
- `key` : Il-valur ewlieni tat-test fil-pront fid-dizzjunarju tal-internazzjonalizzazzjoni
