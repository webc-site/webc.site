# i18n.js

Ġestjoni tal-istatus b'ħafna lingwi u monitoraġġ tal-bidla

## LANG_LI

Lista ta' konfigurazzjonijiet lingwistiċi disponibbli

## lang()

Ikseb l-ID tal-lingwa attwali

Valur tar-ritorn: ID tal-lingwa attwali

## langCode()

Ikseb il-kodiċi tal-lingwa attwali

Valur tar-ritorn: kodiċi tal-lingwa attwali, bħal `"zh"`

## onLang(func)

Orbot is-sejħa lura tas-smigħ għal bidliet fil-lingwa

parametru:
- `func` : funzjoni ta' callback meta tinbidel il-lingwa
  - parametru
    - `code` : Kodiċi tal-lingwa
    - `id` : ID tal-lingwa

Valur tar-ritorn: Funzjoni ta' tindif mhux marbuta

## langSet(id)

Aqleb u xxandar il-lingwa attwali

parametru:
- `id` : ID tal-lingwa fil-mira
