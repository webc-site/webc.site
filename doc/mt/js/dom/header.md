# dom/header.js

Oħloq l-istruttura DOM tal-bar header

## default()

Oħloq u għaqqad l-istruttura DOM tal-header tat-tikketti header, h1 u b

Valur tar-ritorn:

- `header` : Element tal-kontenitur tal-bar header
- `h1` : element tat-titlu
- `b` : sottotitolu jew element ta' informazzjoni addizzjonali
