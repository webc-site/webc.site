## newEl

Oħloq elementi DOM

parametru:
- `tagName` : isem it-tikketta tal-element

Valur tar-ritorn: element DOM maħluq ġdid

## newLi(...tag_li)

Oħloq elementi DOM multipli f'lottijiet

parametru:
- `tag_li` : lista tal-ismijiet tat-tikketta

Valur tar-ritorn:
- Array li fih l-elementi DOM kollha maħluqa ġodda

## cE(name, cls)

Irreġistra elementi tad-dwana tal-Komponenti tal-Web

parametru:
- `name` : Isem tal-element personalizzat, `c-` prefiss se jiżdied awtomatikament
- `cls` : il-klassi tal-element tad-dwana
