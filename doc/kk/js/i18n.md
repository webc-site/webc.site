# i18n.js

Көптілді күйді басқару және өзгерістерді бақылау

## LANG_LI

Қол жетімді тіл конфигурацияларының тізімі

## lang()

Ағымдағы тіл идентификаторын алыңыз

Қайтарылатын мән: ағымдағы тіл идентификаторы

## langCode()

Ағымдағы тіл кодын алыңыз

Қайтарылатын мән: ағымдағы тіл коды, мысалы, `"zh"`

## onLang(func)

Тіл өзгерістері үшін тыңдау кері қоңырауды байланыстырыңыз

параметр:
- `func` : тіл өзгерген кезде кері шақыру функциясы
  - параметр
    - `code` : тіл коды
    - `id` : тіл идентификаторы

Қайтарылатын мән: байланыссыз тазалау функциясы

## langSet(id)

Ағымдағы тілді ауыстырыңыз және таратыңыз

параметр:
- `id` : мақсатты тіл идентификаторы
