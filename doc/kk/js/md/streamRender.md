# md/streamRender.js

Markdown ағынды көрсету

## default(container, source)

Белгіленген DOM контейнеріне Markdown мәтінін ағынмен көрсету, жазу машинкасының курсор әсерлерін және автоматты айналдыруды қолдау

параметр:
- `container` : көрсетілетін мазмұнды орналастыратын DOM контейнер элементі
- `source` : ReadableStream мәтіндік деректер көзі

Қайтарылатын мән:

- `cancel` : көрсетуді және деректер ағынын оқуды тоқтату функциясы
- `setStop` : Автоматты айналдыруды кідірту немесе кідіртуді орнату функциясы
  - параметр
    - `val` : кідірту керек пе, жоқ па логикалық мән
