# parentScroll.js

Негізгі айналдыру контейнер элементін алыңыз

## default(el)

Көрсетілген элементтен жоғары қарай жүріңіз, оны қамтитын айналдыру контейнері элементін тауып, қайтарыңыз (мысалы, ShadowRoot кірістірілген айналдыру контейнері)

параметр:
- `el` : DOM элементін бастау

Қайтару мәні: айналдыру контейнерінің DOM элементі
