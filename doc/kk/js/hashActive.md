# hashActive.js

URL анкерлеріне және көру портын айналдыруға негізделген шарлау сілтемесін бөлектеу

## default(link_li, cls)

Беттің айналдыру орны мен URL бекіту нүктесінің өзгерістеріне негізделген ағымдағы көрінетін аймаққа сәйкес шарлау сілтемелерін автоматты түрде бөлектеңіз.

параметр:
- `link_li` : шарлау сілтемесі DOM элементтерінің тізімі
- `cls` : Белсенді CSS класының атауы, әдепкі `"A"`

Қайтарылатын мән: тыңдауды тазалаудың барлық функцияларынан бас тарту
