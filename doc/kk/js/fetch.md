# fetch.js

Қабылдау инкапсуляциясына негізделген желілік сұрау құралының функциясы

## F(url, opt)

Желі сұрауын жіберіңіз және жауап нысанын қайтарыңыз

параметр:
- `url` : Мекенжайды сұрау
- `opt` : `fetch` опциялар нысаны

Қайтарылатын мән: Жауап нысаны

## f(attr)

Сұрауды орап, қайтарылған мазмұнды көрсетілген пішімде талдайтын жоғары дәрежелі функция

параметр:
- `attr` : жауап негізгі талдаушы атрибут атауы

Қайтарылатын мән: мазмұнды асинхронды сұрау және талдау функциясы
  - параметр
    - `url` : Мекенжайды сұрау
    - `opt` : `fetch` опциялар нысаны
  - Қайтарылатын мән: талданған мазмұн

## fTxt

Желі сұрауын жіберіңіз және жауап мәтінін мәтін пішімінде қайтарыңыз

параметр:
- `url` : Мекенжайды сұрау
- `opt` : `fetch` опциялар нысаны

Қайтарылатын мән: жауап мәтіні

## fJson(url, opt)

Желі сұрауын жіберіңіз және жауап мәтінін JSON пішімінде талдаңыз және қайтарыңыз

параметр:
- `url` : Мекенжайды сұрау
- `opt` : `fetch` опциялар нысаны

Қайтарылатын мән: талданған JSON нысаны

## fBin(url, opt)

Желі сұрауын жіберіңіз және жауап мәтінін екілік байт массиві ретінде қайтарыңыз

параметр:
- `url` : Мекенжайды сұрау
- `opt` : `fetch` опциялар нысаны

Қайтарылатын мән: байт массиві
