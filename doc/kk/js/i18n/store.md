# i18n/store.js

Көптілді жергілікті сақтауды басқару

## get()

Жергілікті кэштелген немесе браузер тілін сәйкестендіру арқылы есептелген әдепкі тіл идентификаторын алыңыз

Қайтарылатын мән: тіл идентификаторы

## set(id)

Ағымдағы тіл идентификаторын жергілікті түрде кэштеңіз

параметр:
- `id` : тіл идентификаторы
