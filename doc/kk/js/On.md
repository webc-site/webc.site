# On.js

Пакеттік байланыстыру оқиғасын бақылау

## On(elem, dict)

DOM элементінің оқиға тыңдаушыларын пакеттермен байланыстырыңыз

параметр:
- `elem` : DOM элементі
- `dict` : Оқиғалар мен кері қоңырауларды салыстыру

Қайтарылатын мән: барлық байланыстырылған оқиғаларды тыңдаушы функцияларын жойыңыз
