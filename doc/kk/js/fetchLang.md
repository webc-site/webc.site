# fetchLang.js

Тіл бумаларын синхронды түрде жүктеңіз және жаңарту кері шақыруларын байланыстырыңыз

## fetchLang(getDict)

Тіл бумасын асинхронды түрде алу және тіл өзгерген кезде жаңартуды кері шақыруды орындау үшін тіл бумасының тыңдаушысын жасаңыз.

параметр:
- `getDict` : Тіл бумасының асинхронды функциясын алыңыз
  - параметр
    - `code` : тіл коды
  - Қайтарылатын мән: тіл бумасының сөздігі

Қайтарылатын мән: жаңартуды кері шақыруды тіркеу функциясы
- параметр
  - `refresh` : Тіл жаңартылған кезде кері шақыру функциясын жаңарту
    - параметр
      - `i18n` : Тіл бумасы сөздік нысаны
      - `code` : Ағымдағы тіл коды
- Қайтару мәні: жаңарту кері шақыруды тіркеуден шығару функциясы
