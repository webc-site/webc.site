# dom.js

Құжат айнымалы аббревиатуралары және таңбаша функциялары

## D

`document` Нысанға сілтеме сілтемесі

## B

`document.body` Нысанға сілтеме сілтемесі

## newEl

DOM элементтерін жасаңыз

параметр:
- `tagName` : элемент тегінің аты

Қайтарылатын мән: жаңадан жасалған DOM элементі

## newLi(...tag_li)

Пакеттерде бірнеше DOM элементтерін жасаңыз

параметр:
- `tag_li` : тегтер тізімі

Қайтарылатын мән:
- Жаңадан жасалған барлық DOM элементтерін қамтитын массив

## cE(name, cls)

Web Components пайдаланушы элементтерін тіркеңіз

параметр:
- `name` : реттелетін элемент атауы, `c-` префиксі автоматты түрде қосылады
- `cls` : реттелетін элемент класы
