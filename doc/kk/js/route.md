# route.js

Бір беттің алдыңғы жағындағы маршруттауды басқару

## nowUrl()

Бастапқы қиғаш сызық жойылған ағымдағы URL жолын алыңыз

Қайтарылатын мән: ағымдағы жол жолы

## route(hook)

Бағдардағы өзгерістерді тыңдау үшін кері қоңырауды тіркеңіз. Тіркеу кезінде ол ағымдағы жолмен бір рет орындалады.

параметр:
- `hook` : маршрутты бақылау кері шақыру функциясы
  - параметр
    - `url` : ағымдағы жол
    - `pre` : соңғы жол

Қайтару мәні: тіркелмеген тазалау функциясы

## setPre(url)

Соңғы жазылған URL мекенжайын сақтаңыз

параметр:
- `url` : жол жолы

## preUrl()

Соңғы жазбаның URL мекенжайын алыңыз

Қайтарылатын мән: соңғы жол жолы

## refresh(url)

Барлық бағытты тыңдау кері шақыруларын көрсетілген жолда іске қосуға және алдын ала жаңартуға мәжбүрлеңіз

параметр:
- `url` : мақсатты жол, әдепкі ағымдағы жол

## removeSlash(url)

Жол тақырыптарындағы қиғаш сызықтарды алып тастаңыз

параметр:
- `url` : жол жолы

Қайтарылатын мән: алдыңғы қиғаш сызықты алып тастағаннан кейінгі жол

## setUrl(url)

Браузердің мекенжай жолағы URL күйін жаңартыңыз

параметр:
- `url` : мақсатты жол

Қайтарылатын мән: 1, егер жол шынымен өзгерсе

## goto(url)

Браузер мекенжайын жаңартыңыз және маршрутты жаңартуды іске қосыңыз

параметр:
- `url` : мақсатты жол
