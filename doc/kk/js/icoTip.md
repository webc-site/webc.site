# icoTip.js

Интернационалдандыру туралы кеңестері бар белгіше құрамдас бөлігін анықтаңыз және тіркеңіз

## default(i18n_url, tag, key)

Теңшелетін HTML элементінің құрамдас бөлігін интернационализацияға шақыру көпіршіктерімен тіркеңіз

параметр:
- `i18n_url` : Интернационалдандыру конфигурациясының мекенжайын сұрау
- `tag` : HTML элементінің тег атауын теңшеу
- `key` : Интернационалдандыру сөздігіндегі шақыру мәтінінің негізгі мәні
