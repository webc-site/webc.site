# c-t.js

HTML интернационализациясы

`<c-t>key</c-t>` тегі теңшелген, ал `key` тіл бумасының кілті болып табылады.

## cTranSet(el, genUrl)

`c-t` үшін мәтінді орнатыңыз.

параметр:
- `el` : түбір элементі, жаһандық `document` ішінде берілуі мүмкін
- `genUrl` : тіл бумасының сөздігін алу функциясы
  - параметр
    - `code` : тіл коды
  - Қайтарылатын мән: тіл бумасы сөздік нысаны немесе осы нысанды шешетін уәде
