# Toast.js

Тост қалқымалы хабарды шақыру

## toast(render, timeout)

Стандартты шақыру қалқымалы терезесін көрсету

параметр:
- `render` : Қалқымалы мазмұнды теңшеу үшін пайдаланылатын көрсету функциясы
  - параметр
    - `el` : қалқымалы терезенің DOM элементі
- `timeout` : автоматты өшіру үшін кідіріс секундтары, әдепкі 9, 0 болғанда автоматты өшіру жоқ

Қайтарылатын мән: қалқымалы терезенің DOM элементі

## toastErr(render, timeout)

Қате туралы хабардың қалқымалы терезесін көрсету

параметр:
- `render` : Қалқымалы мазмұнды теңшеу үшін пайдаланылатын көрсету функциясы
  - параметр
    - `el` : қалқымалы терезенің DOM элементі
- `timeout` : автоматты өшіру үшін кідіріс секундтары, әдепкі 9, 0 болғанда автоматты өшіру жоқ

Қайтарылатын мән: қате қалқымалы терезесінің DOM элементі
