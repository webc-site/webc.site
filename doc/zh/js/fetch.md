# fetch.js

基于 fetch 封装的网络请求工具函数

## F(url, opt)

发送网络请求并返回响应对象

参数:
- `url` : 请求地址
- `opt` : `fetch` 选项对象

返回值: Response 对象

## f(attr)

包装请求并以指定格式解析返回内容的高阶函数

参数:
- `attr` : 响应体解析属性名

返回值: 异步请求并解析内容的函数
  - 参数
    - `url` : 请求地址
    - `opt` : `fetch` 选项对象
  - 返回值: 解析后的内容

## fTxt

发送网络请求并以文本格式返回响应体

参数:
- `url` : 请求地址
- `opt` : `fetch` 选项对象

返回值: 响应文本

## fJson(url, opt)

发送网络请求并以 JSON 格式解析返回响应体

参数:
- `url` : 请求地址
- `opt` : `fetch` 选项对象

返回值: 解析后的 JSON 对象

## fBin(url, opt)

发送网络请求并以二进制字节数组返回响应体

参数:
- `url` : 请求地址
- `opt` : `fetch` 选项对象

返回值: 字节数组
