# route.js

单页面前端路由管理

## nowUrl()

获取当前去除首部斜杠的 URL 路径

返回值: 当前路径字符串

## route(hook)

注册路由变更的监听回调，注册时会以当前路径执行一次

参数:
- `hook` : 路由监听回调函数
  - 参数
    - `url` : 当前路径
    - `pre` : 上一次的路径

返回值: 取消注册的清理函数

## setPre(url)

保存上一次记录的 URL

参数:
- `url` : 路径字符串

## preUrl()

获取上一次记录的 URL

返回值: 上一次的路径字符串

## refresh(url)

强制以指定路径触发所有路由监听回调，并更新 pre

参数:
- `url` : 目标路径，默认当前路径

## removeSlash(url)

移除路径头部的斜杠

参数:
- `url` : 路径字符串

返回值: 移除头部斜杠后的路径

## setUrl(url)

更新浏览器地址栏 URL 状态

参数:
- `url` : 目标路径

返回值: 若路径实际发生变化则返回 1

## goto(url)

更新浏览器地址并触发路由刷新

参数:
- `url` : 目标路径
