# dom/header.js

Paghimo sa header bar DOM nga istruktura

## default()

Paghimo ug paghiusa sa header DOM nga istruktura sa mga tag sa header, h1 ug b

Ibalik nga bili:

- `header` : Container nga elemento sa header bar
- `h1` : elemento sa titulo
- `b` : subtitle o dugang nga elemento sa impormasyon
