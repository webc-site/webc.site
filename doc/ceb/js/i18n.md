# i18n.js

Pagdumala sa status sa daghang pinulongan ug pag-monitor sa pagbag-o

## LANG_LI

Listahan sa magamit nga mga configuration sa pinulongan

## lang()

Pagkuha sa kasamtangan nga ID sa pinulongan

Ibalik nga bili: kasamtangan nga ID sa pinulongan

## langCode()

Kuhaa ang kasamtangan nga code sa pinulongan

Ibalik nga bili: kasamtangan nga code sa pinulongan, sama sa `"zh"`

## onLang(func)

Ibugkos ang callback sa pagpaminaw alang sa mga kausaban sa pinulongan

parameter:
- `func` : callback function kung mausab ang lengguwahe
  - parametro
    - `code` : Kodigo sa pinulongan
    - `id` : Language ID

Ibalik nga bili: Wala'y kapuslanan nga pagpanglimpyo

## langSet(id)

Pagbalhin ug pagsibya sa kasamtangan nga pinulongan

parameter:
- `id` : Target nga pinulongan ID
