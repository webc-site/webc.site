# hashActive.js

Pag-highlight sa link sa nabigasyon base sa mga anchor sa URL ug pag-scroll sa viewport

## default(link_li, cls)

Awtomatikong i-highlight ang mga link sa nabigasyon nga katumbas sa makita karon nga lugar base sa posisyon sa pag-scroll sa panid ug mga pagbag-o sa URL anchor point.

parameter:
- `link_li` : Navigation link DOM element list
- `cls` : Aktibo nga CSS class name, default `"A"`

Ibalik nga bili: Ikansela ang tanan nga mga function sa paglimpyo sa pagpaminaw
