# parentScroll.js

Kuhaa ang ancestor scroll container element

## default(el)

Pagtabok pataas gikan sa gipiho nga elemento, pangitaa ug ibalik ang scroll container nga elemento nga adunay sulod niini (sama sa `C-VS`'s ShadowRoot built-in scroll container)

parameter:
- `el` : pagsugod sa elemento sa DOM

Ibalik nga bili: scroll container DOM nga elemento
