# icoTip.js

Ipasabut ug irehistro ang usa ka sangkap sa icon nga adunay mga pahiwatig sa internasyonalisasyon

## default(i18n_url, tag, key)

Irehistro ang usa ka naandan nga sangkap sa HTML nga adunay mga bula nga pang-internasyonalisasyon

parameter:
- `i18n_url` : Pangayo og adres para sa pag-configure sa internasyonalisasyon
- `tag` : I-customize ang tag name sa HTML nga elemento
- `key` : Ang yawe nga bili sa prompt text sa internationalization dictionary
