# On.js

Pag-monitor sa panghitabo sa batch binding

## On(elem, dict)

Ibugkos ang mga tigpaminaw sa panghitabo sa elemento sa DOM sa mga batch

parameter:
- `elem` : DOM nga elemento
- `dict` : Pagmapa sa mga panghitabo ug mga callback

Pagbalik nga bili: Kuhaa ang tanan nga gihigot nga mga function sa tigpaminaw sa panghitabo
