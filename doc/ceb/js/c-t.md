# c-t.js

internasyonalisasyon sa HTML

Ang tag `<c-t>key</c-t>` gipahiangay, ug ang `key` mao ang yawe sa pack sa pinulongan.

## cTranSet(el, genUrl)

Ibutang ang teksto para sa `c-t`.

parameter:
- `el` : gamut nga elemento, global mahimong ipasa sa `document`
- `genUrl` : function para makakuha og language pack dictionary
  - parametro
    - `code` : code sa pinulongan
  - Ibalik nga bili: usa ka butang nga diksyonaryo nga pakete sa pinulongan, o usa ka Saad nga nagsulbad niini nga butang
