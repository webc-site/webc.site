# i18n.js

Flerspråkig statushantering och förändringsövervakning

## LANG_LI

Lista över tillgängliga språkkonfigurationer

## lang()

Få aktuellt språk-ID

Returvärde: aktuellt språk-ID

## langCode()

Hämta den aktuella språkkoden

Returvärde: aktuell språkkod, som `"zh"`

## onLang(func)

Bind den lyssnande återuppringningen för språkändringar

parameter:
- `func` : återuppringningsfunktion när språket ändras
  - parameter
    - `code` : Språkkod
    - `id` : Språk-ID

Returvärde: Obunden rengöringsfunktion

## langSet(id)

Byt och sänd det aktuella språket

parameter:
- `id` : Målspråk-ID
