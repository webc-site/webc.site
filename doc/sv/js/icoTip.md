# icoTip.js

Definiera och registrera en ikonkomponent med internationaliseringstips

## default(i18n_url, tag, key)

Registrera en anpassad HTML-elementkomponent med bubblor med internationaliseringsprompt

parameter:
- `i18n_url` : Begär adress för internationaliseringskonfiguration
- `tag` : Anpassa taggnamnet för HTML-elementet
- `key` : Nyckelvärdet för prompttexten i internationaliseringsordboken
