# i18n.js

බහු භාෂා තත්ව කළමනාකරණය සහ වෙනස්වීම් අධීක්ෂණය

## LANG_LI

පවතින භාෂා වින්‍යාස ලැයිස්තුව

## lang()

වත්මන් භාෂා හැඳුනුම්පත ලබා ගන්න

ප්‍රතිලාභ අගය: වත්මන් භාෂා ID

## langCode()

වත්මන් භාෂා කේතය ලබා ගන්න

ප්‍රතිලාභ අගය: `"zh"` වැනි වත්මන් භාෂා කේතය

## onLang(func)

භාෂා වෙනස්කම් සඳහා සවන්දීමේ ඇමතුම බැඳ තබන්න

පරාමිතිය:
- `func` : භාෂාව වෙනස් වන විට ආපසු කැඳවීමේ කාර්යය
  - පරාමිතිය
    - `code` : භාෂා කේතය
    - `id` : භාෂා ID

ප්‍රතිලාභ අගය: නොබැඳි පිරිසිදු කිරීමේ කාර්යය

## langSet(id)

වත්මන් භාෂාව මාරු කර විකාශනය කරන්න

පරාමිතිය:
- `id` : ඉලක්ක භාෂා ID
