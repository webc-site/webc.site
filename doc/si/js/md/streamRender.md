# md/streamRender.js

මාර්ක්ඩවුන් ප්‍රවාහ විදැහුම්කරණය

## default(container, source)

නිශ්චිත DOM කන්ටේනරය තුළට මාර්ක්ඩවුන් පෙළ විදැහුම්කරණය, යතුරු ලියන කර්සරයේ ප්‍රයෝග සහ ස්වයංක්‍රීය අනුචලනය සඳහා සහාය වීම

පරාමිතිය:
- `container` : DOM බහාලුම් මූලද්‍රව්‍ය විදැහුම් කළ අන්තර්ගත සත්කාරක
- `source` : ReadableStream පෙළ දත්ත මූලාශ්‍රය

ප්‍රතිලාභ අගය:

- `cancel` : විදැහුම්කරණය සහ දත්ත ප්‍රවාහ කියවීම නැවැත්වීමේ කාර්යය
- `setStop` : ස්වයංක්‍රීය අනුචලනය විරාම කළ යුතුද යන්න සැකසීමේ කාර්යය
  - පරාමිතිය
    - `val` : බූලියන් අගය විරාම කළ යුතුද නැද්ද යන්න
