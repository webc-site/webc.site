# dom.js

ලේඛන විචල්‍ය කෙටි යෙදුම් සහ කෙටිමං ශ්‍රිත

## D

`document` වස්තුවට කෙටිමං යොමුව

## B

`document.body` වස්තුවට කෙටිමං යොමුව

## newEl

DOM මූලද්‍රව්‍ය සාදන්න

පරාමිතිය:
- `tagName` : මූලද්‍රව්‍ය ටැග් නම

ප්‍රතිලාභ අගය: අලුතින් සාදන ලද DOM මූලද්‍රව්‍යය

## newLi(...tag_li)

කණ්ඩායම් වශයෙන් DOM මූලද්‍රව්‍ය කිහිපයක් සාදන්න

පරාමිතිය:
- `tag_li` : ටැග් නාම ලැයිස්තුව

ප්‍රතිලාභ අගය:
- අලුතින් සාදන ලද සියලුම DOM මූලද්‍රව්‍ය අඩංගු අරාව

## cE(name, cls)

වෙබ් සංරචක අභිරුචි අංග ලියාපදිංචි කරන්න

පරාමිතිය:
- `name` : අභිරුචි මූලද්‍රව්‍ය නම, `c-` උපසර්ගය ස්වයංක්‍රීයව එකතු වේ
- `cls` : අභිරුචි මූලද්‍රව්‍යයේ පන්තිය
