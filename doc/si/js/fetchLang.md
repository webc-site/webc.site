# fetchLang.js

භාෂා ඇසුරුම් අසමමුහුර්තව පූරණය කර යාවත්කාලීන ඇමතුම් බැඳ තබන්න

## fetchLang(getDict)

භාෂා ඇසුරුම අසමමුහුර්තව ලබා ගැනීමට භාෂා ඇසුරුම් සවන්දෙන්නෙකු සාදන්න සහ භාෂාව වෙනස් වූ විට නැවත නැවත ඇමතුමක් ක්‍රියාත්මක කරන්න

පරාමිතිය:
- `getDict` : භාෂා පැකේජයේ අසමමුහුර්ත ශ්‍රිතය ලබා ගන්න
  - පරාමිතිය
    - `code` : භාෂා කේතය
  - ප්‍රතිලාභ අගය: භාෂා ඇසුරුම් ශබ්දකෝෂය

ප්‍රතිලාභ අගය: නැවත නැවත කැඳවීම ලියාපදිංචි කිරීමේ කාර්යය
- පරාමිතිය
  - `refresh` : භාෂාව යාවත්කාලීන වූ විට නැවත ඇමතුම් ශ්‍රිතය නැවුම් කරන්න
    - පරාමිතිය
      - `i18n` : භාෂා ඇසුරුම ශබ්ද කෝෂ වස්තුව
      - `code` : වත්මන් භාෂා කේතය
- ප්‍රතිලාභ අගය: නැවත නැවත කැඳවීම ලියාපදිංචියෙන් ඉවත් කිරීමේ කාර්යය
