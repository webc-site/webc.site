# c-t.js

HTML ජාත්‍යන්තරකරණය

ටැගය `<c-t>key</c-t>` අභිරුචිකරණය කර ඇති අතර `key` භාෂා ඇසුරුමේ යතුර වේ.

## cTranSet(el, genUrl)

`c-t` සඳහා පෙළ සකසන්න.

පරාමිතිය:
- `el` : මූල මූලද්‍රව්‍යය, ගෝලීය `document` තුළ සම්මත කළ හැක
- `genUrl` : භාෂා ඇසුරුම් ශබ්දකෝෂය ලබා ගැනීමේ කාර්යය
  - පරාමිතිය
    - `code` : භාෂා කේතය
  - ප්‍රතිලාභ අගය: භාෂා ඇසුරුම් ශබ්ද කෝෂ වස්තුවක්, හෝ මෙම වස්තුවට විසඳන පොරොන්දුවක්
