# hashActive.js

URL නැංගුරම් සහ viewport අනුචලනය මත පදනම්ව සංචාලන සබැඳි උද්දීපනය

## default(link_li, cls)

පිටු අනුචලන ස්ථානය සහ URL නැංගුරම් ලක්ෂ්‍ය වෙනස්වීම් මත පදනම්ව දැනට පෙනෙන ප්‍රදේශයට අනුරූප වන සංචාලන සබැඳි ස්වයංක්‍රීයව උද්දීපනය කරන්න.

පරාමිතිය:
- `link_li` : Navigation link DOM මූලද්‍රව්‍ය ලැයිස්තුව
- `cls` : ක්‍රියාකාරී CSS පන්තියේ නම, පෙරනිමිය `"A"`

ප්‍රතිලාභ අගය: සියලුම සවන්දීමේ පිරිසිදු කිරීමේ කාර්යයන් අවලංගු කරන්න
