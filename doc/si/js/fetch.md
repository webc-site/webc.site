# fetch.js

ෆෙච් එන්කැප්සුලේෂන් මත පදනම් වූ ජාල ඉල්ලීම් මෙවලම් ක්‍රියාකාරිත්වය

## F(url, opt)

ජාල ඉල්ලීමක් යවා ප්‍රතිචාර වස්තුවක් ආපසු එවන්න

පරාමිතිය:
- `url` : ලිපිනය ඉල්ලන්න
- `opt` : `fetch` විකල්ප වස්තුව

ප්‍රතිලාභ අගය: ප්‍රතිචාර වස්තුව

## f(attr)

ඉල්ලීමක් ඔතා සහ ආපසු ලබා දුන් අන්තර්ගතය නිශ්චිත ආකෘතියකින් විග්‍රහ කරන ඉහළ පෙළේ ශ්‍රිතයක්

පරාමිතිය:
- `attr` : ප්‍රතිචාර ශරීරය විග්‍රහ කිරීමේ ගුණාංග නාමය

ප්‍රතිලාභ අගය: අසමමුහුර්තව ඉල්ලීමට සහ අන්තර්ගතය විග්‍රහ කිරීමට ක්‍රියා කරයි
  - පරාමිතිය
    - `url` : ලිපිනය ඉල්ලන්න
    - `opt` : `fetch` විකල්ප වස්තුව
  - ප්‍රතිලාභ අගය: විග්‍රහ කළ අන්තර්ගතය

## fTxt

ජාල ඉල්ලීමක් යවා ප්‍රතිචාර දැක්වීමේ කොටස පෙළ ආකෘතියෙන් ආපසු එවන්න

පරාමිතිය:

- `opt` : `fetch` විකල්ප වස්තුව

ප්‍රතිලාභ අගය: ප්‍රතිචාර පෙළ

## fJson(url, opt)

ජාල ඉල්ලීමක් යවා විග්‍රහ කර JSON ආකෘතියෙන් ප්‍රතිචාර අන්තර්ගතය ආපසු එවන්න

පරාමිතිය:
- `url` : ලිපිනය ඉල්ලන්න
- `opt` : `fetch` විකල්ප වස්තුව

ප්‍රතිලාභ අගය: විග්‍රහ කළ JSON වස්තුව

ජාල ඉල්ලීමක් යවා ප්‍රතිචාරයේ අන්තර්ගතය ද්විමය බයිට් අරාවක් ලෙස ආපසු එවන්න
