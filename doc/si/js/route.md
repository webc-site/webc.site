# route.js

තනි පිටු ඉදිරිපස-අන්ත මාර්ගගත කළමනාකරණය

## nowUrl()

ප්‍රමුඛ ස්ලෑෂ් ඉවත් කර වත්මන් URL මාර්ගය ලබා ගන්න

ප්‍රතිලාභ අගය: වත්මන් මාර්ග පෙළ

## route(hook)

මාර්ග වෙනස් කිරීම් සඳහා සවන්දීමේ ඇමතුම ලියාපදිංචි කරන්න. ලියාපදිංචි වන විට, එය වත්මන් මාර්ගය සමඟ එක් වරක් ක්රියාත්මක වේ.

පරාමිතිය:
- `hook` : මාර්ග නිරීක්ෂණ ආපසු කැඳවීමේ කාර්යය
  - පරාමිතිය
    - `url` : වත්මන් මාර්ගය
    - `pre` : අවසාන මාර්ගය

ප්‍රතිලාභ අගය: ලියාපදිංචි නොකළ පිරිසිදු කිරීමේ කාර්යය

## setPre(url)

අවසන් වරට පටිගත කළ URL එක සුරකින්න

පරාමිතිය:
- `url` : මාර්ග පෙළ

## preUrl()

අවසන් වාර්තාවේ URL එක ලබා ගන්න

ප්‍රතිලාභ අගය: අවසාන මාර්ග පෙළ

## refresh(url)

සියලුම මාර්ග සවන්දීමේ ඇමතුම් නියමිත මාර්ගයේදී ක්‍රියාත්මක කිරීමට බල කර පෙර යාවත්කාලීන කරන්න

පරාමිතිය:
- `url` : ඉලක්ක මාර්ගය, පෙරනිමි වත්මන් මාර්ගය

## removeSlash(url)

මාර්ග ශීර්ෂවලින් ස්ලැෂ් ඉවත් කරන්න

පරාමිතිය:
- `url` : මාර්ග පෙළ

ප්‍රතිලාභ අගය: ප්‍රමුඛ ස්ලෑෂ් ඉවත් කිරීමෙන් පසු මාර්ගය

## setUrl(url)

බ්‍රවුසර ලිපින තීරුවේ URL තත්ත්වය යාවත්කාලීන කරන්න

පරාමිතිය:
- `url` : ඉලක්ක මාර්ගය

ප්‍රතිලාභ අගය: 1 මාර්ගය සැබවින්ම වෙනස් වන්නේ නම්

## goto(url)

බ්‍රවුසර ලිපිනය යාවත්කාලීන කර මාර්ග නැවුම් කිරීමක් ක්‍රියාරම්භ කරන්න

පරාමිතිය:
- `url` : ඉලක්ක මාර්ගය
