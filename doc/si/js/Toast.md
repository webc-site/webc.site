# Toast.js

ටෝස්ට් උත්පතන පණිවිඩ විමසුම

## toast(render, timeout)

සම්මත ක්ෂණික උත්පතන කවුළුව පෙන්වන්න

පරාමිතිය:
- `render` : විදැහුම්කරණ කාර්යය, උත්පතන අන්තර්ගතය අභිරුචිකරණය කිරීමට භාවිතා කරයි
  - පරාමිතිය
    - `el` : උත්පතන කවුළුවේ DOM මූලද්‍රව්‍යය
- `timeout` : ස්වයංක්‍රීය වසා දැමීම සඳහා තත්පර ප්‍රමාද, පෙරනිමි 9, 0 විට ස්වයංක්‍රීය වසා දැමීමක් නොමැත

ප්‍රතිලාභ අගය: උත්පතන කවුළුවේ DOM මූලද්‍රව්‍යය

## toastErr(render, timeout)

දෝෂ පණිවිඩ උත්පතන පෙන්වන්න

පරාමිතිය:
- `render` : විදැහුම්කරණ කාර්යය, උත්පතන අන්තර්ගතය අභිරුචිකරණය කිරීමට භාවිතා කරයි
  - පරාමිතිය
    - `el` : උත්පතන කවුළුවේ DOM මූලද්‍රව්‍යය
- `timeout` : ස්වයංක්‍රීය වසා දැමීම සඳහා තත්පර ප්‍රමාද, පෙරනිමි 9, 0 විට ස්වයංක්‍රීය වසා දැමීමක් නොමැත

ප්‍රතිලාභ අගය: දෝෂ උත්පතන කවුළුවේ DOM මූලද්‍රව්‍යය
