# icoTip.js

ජාත්‍යන්තරකරණ ඉඟි සහිත අයිකන සංරචකයක් නිර්වචනය කර ලියාපදිංචි කරන්න

## default(i18n_url, tag, key)

ජාත්‍යන්තරකරණ ඉක්මන් බුබුලු සහිත අභිරුචි HTML මූලද්‍රව්‍ය සංරචකයක් ලියාපදිංචි කරන්න

පරාමිතිය:
- `i18n_url` : ජාත්‍යන්තරකරණ වින්‍යාසය සඳහා ලිපිනය ඉල්ලන්න
- `tag` : HTML මූලද්‍රව්‍යයේ ටැග් නම අභිරුචිකරණය කරන්න
- `key` : ජාත්‍යන්තරකරණ ශබ්ද කෝෂයේ ප්‍රමාද පාඨයේ ප්‍රධාන අගය
