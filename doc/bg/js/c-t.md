# c-t.js

HTML интернационализация

Тагът `<c-t>key</c-t>` е персонализиран, а `key` е ключът на езиковия пакет.

## cTranSet(el, genUrl)

Задайте текста за `c-t`.

параметър:
- `el` : основен елемент, глобален може да бъде предаден в `document`
- `genUrl` : функция за получаване на речник на езиковия пакет
  - параметър
    - `code` : езиков код
  - Връщана стойност: речников обект на езиков пакет или обещание, което разрешава този обект
