# i18n/store.js

Многоезично управление на локално съхранение

## get()

Вземете идентификатора на езика по подразбиране, изчислен локално кеширан или изчислен чрез съвпадение на езика на браузъра

Върната стойност: ID на езика

## set(id)

Кеширайте текущия езиков идентификатор локално

параметър:
- `id` : ИД на език
