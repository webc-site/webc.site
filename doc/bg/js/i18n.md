# i18n.js

Многоезично управление на състоянието и наблюдение на промените

## LANG_LI

Списък с налични езикови конфигурации

## lang()

Вземете текущия езиков идентификатор

Върната стойност: ID на текущия език

## langCode()

Вземете текущия езиков код

Върната стойност: код на текущия език, като `"zh"`

## onLang(func)

Свържете слушащото обратно извикване за промени в езика

параметър:
- `func` : функция за обратно извикване при промяна на езика
  - параметър
    - `code` : Езиков код
    - `id` : ИД на език

Върната стойност: Необвързана функция за почистване

## langSet(id)

Превключете и излъчете текущия език

параметър:
- `id` : ID на целевия език
