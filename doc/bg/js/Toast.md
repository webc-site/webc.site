# Toast.js

Подкана за изскачащо съобщение за тост

## toast(render, timeout)

Показване на изскачащ прозорец със стандартна подкана

параметър:
- `render` : Функция за изобразяване, използвана за персонализиране на изскачащо съдържание
  - параметър
    - `el` : DOM елемент на изскачащия прозорец
- `timeout`: Секунди забавяне за автоматично изключване, по подразбиране 9, без автоматично изключване, когато 0

Върната стойност: DOM елемент на изскачащия прозорец

## toastErr(render, timeout)

Показване на изскачащ прозорец със съобщение за грешка

параметър:
- `render` : Функция за изобразяване, използвана за персонализиране на изскачащо съдържание
  - параметър
    - `el` : DOM елемент на изскачащия прозорец
- `timeout`: Секунди забавяне за автоматично изключване, по подразбиране 9, без автоматично изключване, когато 0

Върната стойност: DOM елемент от изскачащия прозорец за грешка
