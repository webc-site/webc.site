# hashActive.js

Осветяване на връзката за навигация въз основа на котви на URL адреси и превъртане на прозореца за изглед

## default(link_li, cls)

Автоматично маркирайте връзките за навигация, съответстващи на текущо видимата област въз основа на позицията на превъртане на страницата и промените в опорната точка на URL адреса.

параметър:
- `link_li` : Списък с DOM елементи на връзката за навигация
- `cls` : Име на активен CSS клас, по подразбиране `"A"`

Върната стойност: Отменете всички функции за почистване на слушане
