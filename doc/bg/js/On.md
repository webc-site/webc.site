# On.js

Мониторинг на събития за пакетно обвързване

## On(elem, dict)

Свързване на слушатели на събития на DOM елемент на партиди

параметър:
- `elem` : DOM елемент
- `dict` : Картографиране на събития и обратни извиквания

Върната стойност: Премахнете всички обвързани функции за приемане на събития
