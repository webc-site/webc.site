# icoTip.js

Дефинирайте и регистрирайте компонент на икона със съвети за интернационализация

## default(i18n_url, tag, key)

Регистрирайте персонализиран компонент на HTML елемент с мехурчета с подкана за интернационализация

параметър:
- `i18n_url` : Адрес на заявка за конфигурация за интернационализация
- `tag` : Персонализирайте името на етикета на HTML елемента
- `key` : Ключовата стойност на подканящия текст в речника за интернационализация
