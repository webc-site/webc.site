# dom/header.js

Създайте DOM структура на заглавната лента

## default()

Създайте и комбинирайте заглавната DOM структура на заглавката, h1 и b таговете

Върната стойност:

- `header` : Елемент контейнер на заглавната лента
- `h1` : заглавен елемент
- `b` : подзаглавие или допълнителен информационен елемент
