# dom/header.js

Krijo strukturën DOM të shiritit të kokës

## default()

Krijoni dhe kombinoni strukturën e titullit DOM të etiketave header, h1 dhe b

Vlera e kthimit:

- `header` : Elementi i kontejnerit të shiritit të kokës
- `h1` : elementi i titullit
- `b` : nëntitull ose element informacioni shtesë
