# icoTip.js

Définir et enregistrer un composant icône avec des conseils d'internationalisation

## default(i18n_url, tag, key)

Enregistrez un composant d'élément HTML personnalisé avec des bulles d'invite d'internationalisation

paramètre:
- `i18n_url` : Adresse de demande pour la configuration de l'internationalisation
- `tag` : Personnalisez le nom de la balise de l'élément HTML
- `key` : valeur clé du texte d'invite dans le dictionnaire d'internationalisation
