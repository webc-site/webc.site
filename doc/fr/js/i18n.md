# i18n.js

Gestion des statuts multilingues et suivi des modifications

## LANG_LI

Liste des configurations linguistiques disponibles

## lang()

Obtenir l'ID de langue actuel

Valeur de retour : ID de langue actuel

## langCode()

Obtenez le code de langue actuel

Valeur de retour : code de langue actuel, tel que `"zh"`

## onLang(func)

Lier le rappel d'écoute pour les changements de langue

paramètre:
- `func` : fonction de rappel lorsque la langue change
  - paramètre
    - `code` : Code langue
    - `id` : ID de langue

Valeur de retour : fonction de nettoyage indépendante

## langSet(id)

Changer et diffuser la langue actuelle

paramètre:
- `id` : ID de la langue cible
