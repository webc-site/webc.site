# parentScroll.js

Obtenez l'élément conteneur de défilement ancêtre

## default(el)

Parcourez vers le haut à partir de l'élément spécifié, recherchez et renvoyez l'élément de conteneur de défilement le contenant (tel que le conteneur de défilement intégré ShadowRoot de `C-VS`)

paramètre:
- `el` : élément DOM de départ

Valeur de retour : élément DOM du conteneur de défilement
