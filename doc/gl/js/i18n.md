# i18n.js

Xestión de estado multilingüe e seguimento de cambios

## LANG_LI

Lista de configuracións de idiomas dispoñibles

## lang()

Obter o ID de idioma actual

Valor de retorno: ID da lingua actual

## langCode()

Obter o código de idioma actual

Valor de retorno: código de idioma actual, como `"zh"`

## onLang(func)

Vincular a devolución de chamada de escoita para os cambios de idioma

parámetro:
- `func` : función de devolución de chamada cando cambia o idioma
  - parámetro
    - `code` : código de idioma
    - `id` : ID de idioma

Valor de retorno: función de limpeza sen vincular

## langSet(id)

Cambia e transmite o idioma actual

parámetro:
- `id` : ID da lingua de destino
