# dom/header.js

Crea a estrutura DOM da barra de cabeceira

## default()

Cree e combine a estrutura DOM de cabeceira das etiquetas de cabeceira, h1 e b

Valor de retorno:

- `header` : elemento contenedor da barra de cabeceira
- `h1` : elemento de título
- `b` : subtítulo ou elemento de información adicional
