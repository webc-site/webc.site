# icoTip.js

Defina e rexistre un compoñente de icona con consellos de internacionalización

## default(i18n_url, tag, key)

Rexistra un compoñente de elemento HTML personalizado con burbullas de indicadores de internacionalización

parámetro:
- `i18n_url` : Solicita o enderezo para a configuración de internacionalización
- `tag` : personaliza o nome da etiqueta do elemento HTML
- `key` : o valor clave do texto da solicitude no dicionario de internacionalización
