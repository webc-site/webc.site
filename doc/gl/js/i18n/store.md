# i18n/store.js

Xestión de almacenamento local multilingüe

## get()

Obtén o ID de idioma predeterminado calculado na memoria caché local ou calculado pola coincidencia do idioma do navegador

Valor de retorno: ID de idioma

## set(id)

Almacene na caché o ID de idioma actual localmente

parámetro:
- `id` : ID de idioma
