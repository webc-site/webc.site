## newEl

Crear elementos DOM

parámetro:
- `tagName` : nome da etiqueta do elemento

Valor de retorno: elemento DOM recén creado

## newLi(...tag_li)

Crea varios elementos DOM en lotes

parámetro:
- `tag_li` : lista de nomes de etiquetas

Valor de retorno:
- Matriz que contén todos os elementos DOM recén creados

## cE(name, cls)

Rexistrar elementos personalizados de Web Components

parámetro:
- `name` : o nome do elemento personalizado, o prefixo `c-` engadirase automaticamente
- `cls` : a clase do elemento personalizado
