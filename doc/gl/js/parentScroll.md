# parentScroll.js

Obtén o elemento contenedor de desprazamento do antepasado

## default(el)

Desprázase cara arriba desde o elemento especificado, busca e devolve o elemento do contedor de desprazamento que o contén (como o contenedor de desprazamento integrado de `C-VS` ShadowRoot)

parámetro:
- `el` : elemento DOM inicial

Valor de retorno: elemento DOM do contenedor de desprazamento
