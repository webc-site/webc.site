# fetchLang.js

Асинхроно учитајте језичке пакете и повежите повратне позиве за ажурирање

## fetchLang(getDict)

Креирајте слушалац језичког пакета да асинхроно добијете језички пакет и извршите повратни позив за освежавање када се језик промени

параметар:
- <б ид="1"> : Добијте асинхрону функцију језичког пакета
  - параметар
    - <б ид="1"> : код језика
  - Повратна вредност: речник језичког пакета

Повратна вредност: функција за регистровање повратног позива за освежавање
- параметар
  - <б ид="1"> : Освежите функцију повратног позива када се језик ажурира
    - параметар
      - <б ид="1"> : Речник језичког пакета
      - <б ид="1"> : Тренутни код језика
- Повратна вредност: функција за поништавање регистрације повратног позива за освежавање
