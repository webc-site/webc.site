# i18n.js

Вишејезично управљање статусом и праћење промена

## LANG_LI

Листа доступних језичких конфигурација

## lang()

Преузмите ИД тренутног језика

Повратна вредност: ИД тренутног језика

## langCode()

Преузмите тренутни код језика

Повратна вредност: тренутни код језика, као што је <б ид="1">

## onLang(func)

Повежите повратни позив слушања за промене језика

параметар:
- <б ид="1"> : функција повратног позива када се језик промени
  - параметар
    - <б ид="1"> : Код језика
    - <б ид="1"> : ИД језика

Повратна вредност: Невезана функција чишћења

## langSet(id)

Пребаците и емитујте тренутни језик

параметар:
- <б ид="1"> : ИД циљног језика
