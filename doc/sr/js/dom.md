# dom.js

Скраћенице променљивих докумената и функције пречица

## D

<б ид="1"> Пречица за објекат

## B

<б ид="1"> Пречица за објекат

## newEl

Креирајте ДОМ елементе

параметар:
- <б ид="1"> : назив ознаке елемента

Повратна вредност: ново креирани ДОМ елемент

## newLi(...tag_li)

Креирајте више ДОМ елемената у групама

параметар:
- <б ид="1"> : листа имена ознака

Повратна вредност:
- Низ који садржи све новокреиране ДОМ елементе

## cE(name, cls)

Региструјте прилагођене елементе веб компоненти

параметар:
- <б ид="1"> : назив прилагођеног елемента, префикс <б ид="2"> ће бити додат аутоматски
- <б ид="1"> : класа прилагођеног елемента
