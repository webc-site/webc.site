# Toast.js

Тост искачуће поруке

## toast(render, timeout)

Прикажи стандардни искачући прозор са упитом

параметар:
- <б ид="1"> : Функција приказивања, која се користи за прилагођавање искачућег садржаја
  - параметар
    - <б ид="1"> : ДОМ елемент искачућег прозора
- <б ид="1"> : Кашњење секунди за аутоматско искључивање, подразумевано 9, нема аутоматског искључивања када је 0

Повратна вредност: ДОМ елемент искачућег прозора

  - параметар
    - <б ид="1"> : ДОМ елемент искачућег прозора
- <б ид="1"> : Кашњење секунди за аутоматско искључивање, подразумевано 9, нема аутоматског искључивања када је 0

Повратна вредност: ДОМ елемент искачућег прозора са грешком
