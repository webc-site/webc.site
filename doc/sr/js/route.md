# route.js

Једностранично управљање рутирањем на фронт-енд

## nowUrl()

Преузмите тренутну путању УРЛ-а са уклоњеном водећим косом цртом

Повратна вредност: тренутни низ путање

## route(hook)

Региструјте повратни позив за слушање за промене руте. Приликом регистрације, биће извршено једном са тренутном путањом.

параметар:
- <б ид="1"> : Функција повратног позива праћења руте
  - параметар
    - <б ид="1"> : тренутна путања
    - <б ид="1"> : последња путања

Повратна вредност: Нерегистрована функција чишћења

## setPre(url)

Сачувајте последњи снимљени УРЛ

параметар:
- <б ид="1"> : низ путање

## preUrl()

Добијте УРЛ последњег записа

Повратна вредност: последњи низ путање

## refresh(url)

Присилите да се сви повратни позиви за слушање руте активирају на наведеној путањи и ажурирајте пре

параметар:
- <б ид="1"> : циљна путања, подразумевана тренутна путања

## removeSlash(url)

Уклоните косе црте из заглавља путање

параметар:
- <б ид="1"> : низ путање

Повратна вредност: путања након уклањања водеће косе црте

## setUrl(url)

Ажурирајте статус УРЛ адресе траке за адресу прегледача

параметар:
- <б ид="1"> : циљна путања

Повратна вредност: 1 ако се путања заиста промени

## goto(url)

Ажурирајте адресу прегледача и покрените освежавање руте

параметар:
- <б ид="1"> : циљна путања
