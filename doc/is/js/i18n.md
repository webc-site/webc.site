# i18n.js

Stöðustjórnun á mörgum tungumálum og eftirlit með breytingum

## LANG_LI

Listi yfir tiltækar tungumálastillingar

## lang()

Fáðu núverandi tungumálakenni

Skilagildi: núverandi tungumálauðkenni

## langCode()

Fáðu núverandi tungumálakóða

Skilagildi: núverandi tungumálakóði, eins og `"zh"`

## onLang(func)

Binddu hlustunarhringinguna fyrir tungumálabreytingar

færibreyta:
- `func` : svarhringingaraðgerð þegar tungumál breytist
  - færibreytu
    - `code` : Tungumálakóði
    - `id` : Tungumálakenni

Skilagildi: Óbundið hreinsunaraðgerð

## langSet(id)

Skiptu um og sendu út núverandi tungumál

færibreyta:
- `id` : Auðkenni markmáls
