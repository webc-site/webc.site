# dom.js

Skjalabreytu skammstafanir og flýtileiðir

## D

`document` Flýtivísun í hlutinn

## B

`document.body` Flýtivísun í hlutinn

## newEl

Búðu til DOM þætti

færibreyta:
- `tagName` : heiti frumefnismerkis

Skilagildi: nýstofnað DOM frumefni

## newLi(...tag_li)

Búðu til marga DOM þætti í lotum

færibreyta:
- `tag_li` : merkisnafnalisti

Skilagildi:
- Fylki sem inniheldur alla nýstofnaða DOM þætti

## cE(name, cls)

Skráðu sérsniðna þætti vefhluta

færibreyta:
- `name` : Heiti sérsniðins þáttar, `c-` forskeyti verður sjálfkrafa bætt við
- `cls` : flokkur sérsniðna þáttarins
