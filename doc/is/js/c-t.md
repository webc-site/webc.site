# c-t.js

HTML alþjóðavæðing

Merkið `<c-t>key</c-t>` er sérsniðið og `key` er lykillinn að tungumálapakkanum.

## cTranSet(el, genUrl)

Stilltu textann fyrir `c-t`.

færibreyta:
- `el` : rótarþáttur, alþjóðlegt er hægt að senda í `document`
- `genUrl` : aðgerð til að fá tungumálapakkaorðabók
  - færibreytu
    - `code` : tungumálakóði
  - Skilagildi: orðabókarhlutur í tungumálapakka eða loforð sem leysir þennan hlut
