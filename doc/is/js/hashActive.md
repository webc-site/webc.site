# hashActive.js

Leiðsögn tengil auðkenning byggt á vefslóð akkeri og útsýnisglugga

## default(link_li, cls)

Auðkenndu sjálfkrafa siglingatengla sem samsvara því svæði sem er sýnilegt eins og er, byggt á síðufletstöðu og breytingum á vefslóð akkerispunkti.

færibreyta:
- `link_li` : Listi yfir DOM-einingar fyrir siglingartengil
- `cls` : Virkt CSS flokksheiti, sjálfgefið `"A"`

Skilagildi: Hætta við allar hlustunarhreinsunaraðgerðir
