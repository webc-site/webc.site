# icoTip.js

Skilgreindu og skráðu táknhluta með vísbendingum um alþjóðavæðingu

## default(i18n_url, tag, key)

Skráðu sérsniðna HTML frumhlutahluta með alþjóðavæðingarbeiðni

færibreyta:
- `i18n_url` : Beðið um heimilisfang fyrir uppsetningu alþjóðavæðingar
- `tag` : Sérsníddu merkisheiti HTML-einingarinnar
- `key` : Lykilgildi boðtexta í alþjóðavæðingarorðabókinni
