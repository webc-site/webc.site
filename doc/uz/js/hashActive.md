# hashActive.js

URL langarlari va koʻrish maydonini aylantirish asosida navigatsiya havolasini ajratib koʻrsatish

## default(link_li, cls)

Sahifani aylantirish joyi va URL bogʻlanish nuqtasi oʻzgarishlari asosida hozirda koʻrinadigan hududga mos keladigan navigatsiya havolalarini avtomatik ravishda ajratib koʻrsatish.

parametr:
- `link_li` : Navigatsiya havolasi DOM elementlari roʻyxati
- `cls` : Faol CSS sinf nomi, birlamchi `"A"`

Qaytish qiymati: Tinglashni tozalashning barcha funksiyalarini bekor qiling
