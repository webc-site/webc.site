## default(i18n_url, tag, key)

parametr:
- `i18n_url` : Xalqarolashtirish konfiguratsiyasi uchun manzilni so'rash
- `tag` : HTML elementining teg nomini moslashtiring
- `key` : Xalqarolashtirish lug'atidagi taklif matnining asosiy qiymati
