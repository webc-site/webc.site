# Toast.js

Toast Pop-up Message Ufro

## toast(render, timeout)

Show Standard Prompt Pop-up Fënster

parameter:
- `render` : Rendering Funktioun, benotzt fir Pop-up Inhalt ze personaliséieren
  - parameter
    - `el` : DOM Element vun der Pop-up Fënster
- `timeout` : Sekonnen Verzögerung fir automatesch Ausschaltung, Standard 9, keng automatesch Ausschaltung wann 0

Retour Wäert: DOM Element vun der Pop-up Fënster

## toastErr(render, timeout)

Fehlermeldung Popup weisen

parameter:
- `render` : Rendering Funktioun, benotzt fir Pop-up Inhalt ze personaliséieren
  - parameter
    - `el` : DOM Element vun der Pop-up Fënster
- `timeout` : Sekonnen Verzögerung fir automatesch Ausschaltung, Standard 9, keng automatesch Ausschaltung wann 0

Retour Wäert: DOM Element vun der Feeler Pop-up Fënster
