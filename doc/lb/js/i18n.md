# i18n.js

Multi-Sprooch Status Gestioun an änneren Iwwerwachung

## LANG_LI

Lëscht vun verfügbare Sproochkonfiguratiounen

## lang()

Kréien aktuell Sprooch ID

Retour Wäert: aktuell Sprooch ID

## langCode()

Kréien déi aktuell Sprooch Code

Retourwäert: aktuelle Sproochcode, wéi `"zh"`

## onLang(func)

Bindt den Oflauschterskandal fir Sproochenännerungen

parameter:
- `func` : Réckrufffunktioun wann d'Sprooch ännert
  - parameter
    - `code` : Sproochcode
    - `id` : Sprooch ID

Retour Wäert: Unbound Botzen Funktioun

## langSet(id)

Wiesselt a verschéckt déi aktuell Sprooch

parameter:
- `id` : Zilsprooch ID
