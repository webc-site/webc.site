# c-t.js

HTML Internationaliséierung

Den Tag `<c-t>key</c-t>` ass personaliséiert, an `key` ass de Schlëssel vum Sproochepack.

## cTranSet(el, genUrl)

Setzt den Text fir `c-t`.

parameter:
- `el` : Root Element, global kann an `document` passéiert ginn
- `genUrl` : Funktioun fir Sproochepack Wierderbuch ze kréien
  - parameter
    - `code` : Sproochcode
  - Retour Wäert: e Sproochepak Wierderbuch Objet, oder e Verspriechen, datt zu dësem Objet entscheet
