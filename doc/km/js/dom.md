# dom.js

ឯកសារអក្សរកាត់អថេរ និងមុខងារផ្លូវកាត់

## D

`document` ផ្លូវកាត់​យោង​ទៅ​វត្ថុ

## B

`document.body` ផ្លូវកាត់​យោង​ទៅ​វត្ថុ

## newEl

បង្កើតធាតុ DOM

ប៉ារ៉ាម៉ែត្រ៖
- `tagName` : ឈ្មោះស្លាកធាតុ

តម្លៃត្រឡប់៖ ធាតុ DOM ដែលទើបបង្កើតថ្មី។

## newLi(...tag_li)

បង្កើតធាតុ DOM ច្រើនជាបាច់

ប៉ារ៉ាម៉ែត្រ៖
- `tag_li`៖ បញ្ជីឈ្មោះស្លាក

តម្លៃត្រឡប់៖
- អារេដែលមានធាតុ DOM ដែលទើបបង្កើតថ្មីទាំងអស់។

## cE(name, cls)

ចុះឈ្មោះធាតុផ្ទាល់ខ្លួនរបស់ Web Components

ប៉ារ៉ាម៉ែត្រ៖
- `name` : ឈ្មោះធាតុផ្ទាល់ខ្លួន `c-` បុព្វបទនឹងត្រូវបានបន្ថែមដោយស្វ័យប្រវត្តិ
- `cls` : ថ្នាក់នៃធាតុផ្ទាល់ខ្លួន
