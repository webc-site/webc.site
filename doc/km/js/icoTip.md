# icoTip.js

កំណត់ និងចុះឈ្មោះសមាសភាគរូបតំណាងជាមួយនឹងការណែនាំអំពីអន្តរភាវូបនីយកម្ម

## default(i18n_url, tag, key)

ចុះឈ្មោះសមាសភាគធាតុ HTML ផ្ទាល់ខ្លួនជាមួយនឹងពពុះដែលជំរុញឱ្យធ្វើអន្តរកម្ម

ប៉ារ៉ាម៉ែត្រ៖
- `i18n_url`៖ ស្នើសុំអាសយដ្ឋានសម្រាប់ការកំណត់រចនាសម្ព័ន្ធអន្តរកម្ម
- `tag`៖ ប្ដូរឈ្មោះស្លាករបស់ធាតុ HTML តាមបំណង
- `key`៖ តម្លៃគន្លឹះនៃអត្ថបទប្រអប់បញ្ចូលក្នុងវចនានុក្រមអន្តរជាតិ
