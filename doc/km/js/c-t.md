# c-t.js

HTML អន្តរជាតិ

ស្លាក `<c-t>key</c-t>`ត្រូវបានប្ដូរតាមបំណង ហើយ `key` គឺជាគន្លឹះនៃកញ្ចប់ភាសា។

## cTranSet(el, genUrl)

កំណត់អត្ថបទសម្រាប់ `c-t`។

ប៉ារ៉ាម៉ែត្រ៖
- `el` : ធាតុឫស សកលអាចត្រូវបានឆ្លងកាត់ក្នុង `document`
- `genUrl` : មុខងារដើម្បីទទួលបានវចនានុក្រមកញ្ចប់ភាសា
  - ប៉ារ៉ាម៉ែត្រ
    - `code` : កូដភាសា
  - តម្លៃត្រឡប់៖ វត្ថុវចនានុក្រមកញ្ចប់ភាសា ឬការសន្យាដែលដោះស្រាយចំពោះវត្ថុនេះ។
