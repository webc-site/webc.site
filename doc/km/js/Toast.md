# Toast.js

Toast бһҹбһ¶бһҡбһӣбҹҒбһ…бһЎбһҫбһ„

## toast(render, timeout)

бһ”бһ„бҹ’бһ бһ¶бһүвҖӢбһ”бһ„бҹ’бһўбһҪбһ…вҖӢбһӣбҹҒбһ…вҖӢбһЎбһҫбһ„вҖӢбһҹбҹ’бһҸбһ„бҹӢбһҠбһ¶бһҡвҖӢ

бһ”бҹүбһ¶бһҡбҹүбһ¶бһҳбҹүбҹӮбһҸбҹ’бһҡбҹ–
- `render`бҹ– бһҳбһ»бһҒбһ„бһ¶бһҡвҖӢбһ”бһ„бҹ’бһ бһ¶бһү бһ”бҹ’бһҡбһҫвҖӢбһҠбһҫбһҳбҹ’бһ”бһёвҖӢбһ”бҹ’бһҠбһјбһҡвҖӢбһҳбһ¶бһҸбһ·бһҖбһ¶вҖӢбһӣбҹҒбһ…вҖӢбһЎбһҫбһ„вҖӢбһҸбһ¶бһҳвҖӢбһ”бҹҶбһҺбһ„
  - бһ”бҹүбһ¶бһҡбҹүбһ¶бһҳбҹүбҹӮбһҸбҹ’бһҡ
    - `el` : бһ’бһ¶бһҸбһ» DOM бһ“бҹғбһ”бһ„бҹ’бһўбһҪбһ…бһӣбҹҒбһ…бһЎбһҫбһ„
- `timeout`бҹ– бһ–бһ“бҹ’бһҷбһ¶бһҡбһ–бҹҒбһӣбһңбһ·бһ“бһ¶бһ‘бһёбһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһҖбһ¶бһҡбһ”бһ·бһ‘бһҠбҹ„бһҷбһҹбҹ’бһңбҹҗбһҷбһ”бҹ’бһҡбһңбһҸбҹ’бһҸбһ· бһӣбҹҶбһ“бһ¶бҹҶбһҠбһҫбһҳ 9 бһӮбҹ’бһҳбһ¶бһ“бһҖбһ¶бһҡбһ”бһ·бһ‘бһҠбҹ„бһҷбһҹбҹ’бһңбҹҗбһҷбһ”бҹ’бһҡбһңбһҸбҹ’бһҸбһ·бһ“бҹ…бһ–бҹҒбһӣ 0

бһҸбһҳбҹ’бһӣбҹғбһҸбҹ’бһҡбһЎбһ”бҹӢбҹ– бһ’бһ¶бһҸбһ» DOM бһ“бҹғбһ”бһ„бҹ’бһўбһҪбһ…бһӣбҹҒбһ…бһЎбһҫбһ„

## toastErr(render, timeout)

бһ”бһ„бҹ’бһ бһ¶бһүбһҖбһ¶бһҡбһӣбҹҒбһ…бһЎбһҫбһ„бһҹбһ¶бһҡбһҖбҹҶбһ бһ»бһҹ

бһ”бҹүбһ¶бһҡбҹүбһ¶бһҳбҹүбҹӮбһҸбҹ’бһҡбҹ–
- `render`бҹ– бһҳбһ»бһҒбһ„бһ¶бһҡвҖӢбһ”бһ„бҹ’бһ бһ¶бһү бһ”бҹ’бһҡбһҫвҖӢбһҠбһҫбһҳбҹ’бһ”бһёвҖӢбһ”бҹ’бһҠбһјбһҡвҖӢбһҳбһ¶бһҸбһ·бһҖбһ¶вҖӢбһӣбҹҒбһ…вҖӢбһЎбһҫбһ„вҖӢбһҸбһ¶бһҳвҖӢбһ”бҹҶбһҺбһ„
  - бһ”бҹүбһ¶бһҡбҹүбһ¶бһҳбҹүбҹӮбһҸбҹ’бһҡ
    - `el` : бһ’бһ¶бһҸбһ» DOM бһ“бҹғбһ”бһ„бҹ’бһўбһҪбһ…бһӣбҹҒбһ…бһЎбһҫбһ„
- `timeout`бҹ– бһ–бһ“бҹ’бһҷбһ¶бһҡбһ–бҹҒбһӣбһңбһ·бһ“бһ¶бһ‘бһёбһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһҖбһ¶бһҡбһ”бһ·бһ‘бһҠбҹ„бһҷбһҹбҹ’бһңбҹҗбһҷбһ”бҹ’бһҡбһңбһҸбҹ’бһҸбһ· бһӣбҹҶбһ“бһ¶бҹҶбһҠбһҫбһҳ 9 бһӮбҹ’бһҳбһ¶бһ“бһҖбһ¶бһҡбһ”бһ·бһ‘бһҠбҹ„бһҷбһҹбҹ’бһңбҹҗбһҷбһ”бҹ’бһҡбһңбһҸбҹ’бһҸбһ·бһ“бҹ…бһ–бҹҒбһӣ 0

бһҸбһҳбҹ’бһӣбҹғбһҸбҹ’бһҡбһЎбһ”бҹӢбҹ– бһ’бһ¶бһҸбһ» DOM бһ“бҹғбһ”бһ„бҹ’бһўбһҪбһ…бһӣбҹҒбһ…бһЎбһҫбһ„бһҖбҹҶбһ бһ»бһҹ
