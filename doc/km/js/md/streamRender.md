бһҖбһ¶бһҡбһҹбҹ’бһ‘бҹ’бһҡбһёбһҳбһ”бһ„бҹ’бһ бһ¶бһүбһўбһҸбҹ’бһҗбһ”бһ‘ Markdown бһ‘бҹ…бһҖбҹ’бһ“бһ»бһ„бһ’бһ»бһ„ DOM бһҠбҹӮбһӣбһ”бһ¶бһ“бһ”бһүбҹ’бһҮбһ¶бһҖбҹӢ бһӮбһ¶бҹҶбһ‘бҹ’бһҡбһҘбһ‘бҹ’бһ’бһ·бһ–бһӣбһ‘бһҹбҹ’бһҹбһ“бҹҚбһ‘бҹ’бһҡбһ“бһ·бһ…бһҳбҹүбһ¶бһҹбҹҠбһёбһ“бһўбһ„бҹ’бһӮбһ»бһӣбһёбһӣбҹҒбһҒ бһ“бһ·бһ„бһҖбһ¶бһҡбһҡбһҳбһјбһҡбһҠбҹ„бһҷбһҹбҹ’бһңбҹҗбһҷбһ”бҹ’бһҡбһңбһҸбҹ’бһҸбһ·

бһ”бҹүбһ¶бһҡбҹүбһ¶бһҳбҹүбҹӮбһҸбҹ’бһҡбҹ–
- `container` : бһ’бһ¶бһҸбһ»бһҖбһ»бһ„бһҸбһәбһ“бҹҗбһҡ DOM бһҠбҹӮбһӣбһ”бһ„бҹ’бһ бҹ„бҹҮбһҳбһ¶бһҸбһ·бһҖбһ¶бһҠбҹӮбһӣбһ”бһ„бҹ’бһ бһ¶бһү
- `source`бҹ– бһ”бҹ’бһҡбһ—бһ–бһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷбһўбһҸбҹ’бһҗбһ”бһ‘ ReadableStream

бһҸбһҳбҹ’бһӣбҹғбһҸбҹ’бһҡбһЎбһ”бҹӢбҹ–

- `cancel`бҹ– бһҳбһ»бһҒбһ„бһ¶бһҡвҖӢбһҠбһҫбһҳбҹ’бһ”бһёвҖӢбһ”бһүбҹ’бһҲбһ”бҹӢвҖӢбһҖбһ¶бһҡвҖӢбһ”бһ„бҹ’бһ бһ¶бһү бһ“бһ·бһ„вҖӢбһҖбһ¶бһҡвҖӢбһўбһ¶бһ“вҖӢбһҹбҹ’бһ‘бҹ’бһҡбһёбһҳвҖӢбһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷ
- `setStop`бҹ– бһҳбһ»бһҒбһ„бһ¶бһҡбһҠбһҫбһҳбҹ’бһ”бһёбһҖбҹҶбһҺбһҸбҹӢбһҗбһ¶бһҸбһҫбһҸбҹ’бһҡбһјбһңбһ•бҹ’бһўбһ¶бһҖбһҖбһ¶бһҡбһҡбһҳбһјбһҡбһҠбҹ„бһҷбһҹбҹ’бһңбҹҗбһҷбһ”бҹ’бһҡбһңбһҸбҹ’бһҸбһ·
  - бһ”бҹүбһ¶бһҡбҹүбһ¶бһҳбҹүбҹӮбһҸбҹ’бһҡ
    - `val`бҹ– бһҸбһҳбҹ’бһӣбҹғбһ”бҹҠбһјбһӣбһёбһ“ бһҗбһ¶бһҸбһҫбһҸбҹ’бһҡбһјбһңбһ•бҹ’бһўбһ¶бһҖбһ¬бһўбһҸбҹӢ
