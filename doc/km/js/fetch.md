# fetch.js

មុខងារ​ឧបករណ៍​សំណើ​បណ្តាញ​ដែល​ផ្អែក​លើ​ការ​ប្រមូល​យក

## F(url, opt)

ផ្ញើសំណើបណ្តាញមួយ ហើយត្រឡប់វត្ថុឆ្លើយតប

ប៉ារ៉ាម៉ែត្រ៖
- `url`៖ ស្នើសុំអាសយដ្ឋាន
- `opt` : `fetch` វត្ថុជម្រើស

តម្លៃត្រឡប់៖ វត្ថុឆ្លើយតប

## f(attr)

អនុគមន៍លំដាប់ខ្ពស់ដែលបញ្ចប់សំណើ និងញែកមាតិកាដែលបានត្រឡប់មកវិញក្នុងទម្រង់ជាក់លាក់មួយ។

ប៉ារ៉ាម៉ែត្រ៖
- `attr`៖ ឈ្មោះ​គុណលក្ខណៈ​ញែក​តួការ​ឆ្លើយតប

តម្លៃត្រឡប់៖ មុខងារដើម្បីស្នើសុំដោយអសមកាល និងញែកមាតិកា
  - ប៉ារ៉ាម៉ែត្រ
    - `url`៖ ស្នើសុំអាសយដ្ឋាន
    - `opt` : `fetch` វត្ថុជម្រើស
  - តម្លៃត្រឡប់៖ មាតិកាដែលបានញែក
