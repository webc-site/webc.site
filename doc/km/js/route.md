# route.js

ការគ្រប់គ្រងការនាំផ្លូវផ្នែកខាងមុខតែមួយទំព័រ

## nowUrl()

ទទួលបានផ្លូវ URL បច្ចុប្បន្នជាមួយនឹងសញ្ញាដកនាំមុខត្រូវបានដកចេញ

តម្លៃត្រឡប់៖ ខ្សែផ្លូវបច្ចុប្បន្ន

## route(hook)

ចុះឈ្មោះការហៅត្រឡប់មកវិញសម្រាប់ការស្តាប់សម្រាប់ការផ្លាស់ប្តូរផ្លូវ។ នៅពេលចុះឈ្មោះវានឹងត្រូវបានប្រតិបត្តិម្តងជាមួយផ្លូវបច្ចុប្បន្ន។

ប៉ារ៉ាម៉ែត្រ៖
- `hook`៖ មុខងារ​ហៅ​ត្រឡប់​ការ​ត្រួតពិនិត្យ​ផ្លូវ
  - ប៉ារ៉ាម៉ែត្រ
    - `url`៖ ផ្លូវបច្ចុប្បន្ន
    - `pre`៖ ផ្លូវចុងក្រោយ

តម្លៃត្រឡប់៖ មុខងារសម្អាតដែលមិនបានចុះឈ្មោះ

## setPre(url)

រក្សាទុក URL ដែលបានកត់ត្រាចុងក្រោយ

ប៉ារ៉ាម៉ែត្រ៖
- `url`៖ ខ្សែអក្សរផ្លូវ

## preUrl()

ទទួលបាន URL នៃកំណត់ត្រាចុងក្រោយ

តម្លៃត្រឡប់៖ ខ្សែផ្លូវចុងក្រោយ

## refresh(url)

បង្ខំឱ្យការហៅត្រឡប់ក្នុងការស្តាប់ផ្លូវទាំងអស់ត្រូវបានកេះនៅផ្លូវដែលបានបញ្ជាក់ និងធ្វើបច្ចុប្បន្នភាពជាមុន

ប៉ារ៉ាម៉ែត្រ៖
- `url`៖ ផ្លូវគោលដៅ ផ្លូវបច្ចុប្បន្នលំនាំដើម

## removeSlash(url)

ដក​សញ្ញា​ពី​ក្បាល​ផ្លូវ

ប៉ារ៉ាម៉ែត្រ៖
- `url`៖ ខ្សែអក្សរផ្លូវ

តម្លៃត្រឡប់៖ ផ្លូវបន្ទាប់ពីដកសញ្ញានាំមុខចេញ

## setUrl(url)

ធ្វើបច្ចុប្បន្នភាពស្ថានភាព URL របារអាសយដ្ឋានកម្មវិធីរុករក

ប៉ារ៉ាម៉ែត្រ៖
- `url` : ផ្លូវគោលដៅ

តម្លៃត្រឡប់៖ 1 ប្រសិនបើផ្លូវពិតជាផ្លាស់ប្តូរ

## goto(url)

អាប់ដេតអាសយដ្ឋានកម្មវិធីរុករកតាមអ៊ីនធឺណិត ហើយចាប់ផ្តើមដំណើរការឡើងវិញនូវផ្លូវ

ប៉ារ៉ាម៉ែត្រ៖
- `url` : ផ្លូវគោលដៅ
