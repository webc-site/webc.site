# Toast.js

トーストポップアップメッセージプロンプト

## toast(render, timeout)

標準のプロンプト ポップアップ ウィンドウを表示する

パラメータ:
- `render` : レンダリング機能。ポップアップ コンテンツのカスタマイズに使用されます。
  - パラメータ
    - `el` : ポップアップ ウィンドウの DOM 要素
- `timeout` : 自動シャットダウンの遅延秒数、デフォルトは 9、0 の場合は自動シャットダウンなし

戻り値: ポップアップウィンドウのDOM要素
