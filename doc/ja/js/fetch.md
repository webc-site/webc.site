# fetch.js

フェッチカプセル化によるネットワークリクエストツール機能

## F(url, opt)

ネットワークリクエストを送信し、応答オブジェクトを返します。

パラメータ:
- `url` : リクエストアドレス
- `opt` : `fetch` オプション オブジェクト

戻り値: レスポンスオブジェクト

## f(attr)

リクエストをラップし、指定された形式で返されたコンテンツを解析する高階関数

パラメータ:
- `attr` : 応答本文の解析属性名

戻り値: コンテンツを非同期にリクエストして解析する関数
  - パラメータ
    - `url` : リクエストアドレス
    - `opt` : `fetch` オプション オブジェクト
  - 戻り値: 解析されたコンテンツ

## fTxt

ネットワーク要求を送信し、応答本文をテキスト形式で返します

パラメータ:
- `url` : リクエストアドレス
- `opt` : `fetch` オプション オブジェクト

戻り値: 応答テキスト

## fJson(url, opt)

ネットワークリクエストを送信し、解析してレスポンスボディをJSON形式で返します。

パラメータ:
- `url` : リクエストアドレス
- `opt` : `fetch` オプション オブジェクト

戻り値: 解析された JSON オブジェクト

## fBin(url, opt)

ネットワークリクエストを送信し、レスポンスボディをバイナリバイト配列として返します。

パラメータ:
- `url` : リクエストアドレス
- `opt` : `fetch` オプション オブジェクト

戻り値: バイト配列
