# i18n.js

多言語ステータス管理と変更監視

## LANG_LI

利用可能な言語設定のリスト

## lang()

現在の言語 ID を取得する

戻り値: 現在の言語 ID

## langCode()

現在の言語コードを取得する

戻り値: 現在の言語コード (`"zh"` など)

## onLang(func)

言語変更のリスニング コールバックをバインドする

パラメータ:
- `func` : 言語変更時のコールバック関数
  - パラメータ
    - `code` : 言語コード
    - `id` : 言語 ID

戻り値: アンバウンドクリーンアップ関数

## langSet(id)

現在の言語を切り替えてブロードキャストします

パラメータ:
- `id` : ターゲット言語 ID
