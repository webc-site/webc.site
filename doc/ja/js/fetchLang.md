# fetchLang.js

言語パックを非同期的にロードし、更新コールバックをバインドする

## fetchLang(getDict)

言語パックを非同期的に取得し、言語が変更されたときに更新コールバックを実行する言語パック リスナーを作成します。

パラメータ:
- `getDict` : 言語パッケージの非同期関数を取得します。
  - パラメータ
    - `code` : 言語コード
  - 戻り値: 言語パック辞書

戻り値: リフレッシュコールバックを登録する関数
- パラメータ
  - `refresh` : 言語が更新されたときにコールバック関数を更新します
    - パラメータ
      - `i18n` : 言語パック辞書オブジェクト
      - `code` : 現在の言語コード
- 戻り値: リフレッシュコールバックの登録を解除する関数
