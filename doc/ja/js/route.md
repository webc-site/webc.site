# route.js

単一ページのフロントエンドルーティング管理

## nowUrl()

先頭のスラッシュを削除した現在の URL パスを取得します

戻り値: 現在のパス文字列

## route(hook)

ルート変更のリスニング コールバックを登録します。登録時は現在のパスで一度実行されます。

パラメータ:
- `hook` : ルート監視コールバック関数
  - パラメータ
    - `url` : 現在のパス
    - `pre` : 最後のパス

戻り値: 未登録のクリーンアップ関数

## setPre(url)

最後に記録した URL を保存する

パラメータ:
- `url` : パス文字列

## preUrl()

最後のレコードのURLを取得する

戻り値: 最後のパス文字列

## refresh(url)

すべてのルート リスニング コールバックが指定されたパスで強制的にトリガーされ、事前に更新されます。

パラメータ:
- `url` : ターゲット パス、デフォルトの現在のパス

## removeSlash(url)

パスヘッダーからスラッシュを削除する

パラメータ:
- `url` : パス文字列

戻り値: 先頭のスラッシュを削除した後のパス

## setUrl(url)

ブラウザのアドレスバーの URL ステータスを更新する

パラメータ:
- `url` : ターゲットパス

戻り値: パスが実際に変更された場合は 1

## goto(url)

ブラウザのアドレスを更新し、ルートの更新をトリガーします

パラメータ:
- `url` : ターゲットパス
