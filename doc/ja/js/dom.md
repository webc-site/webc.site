# dom.js

ドキュメント変数の略語とショートカット関数

## D

`document` オブジェクトへのショートカット参照

## B

`document.body` オブジェクトへのショートカット参照

## newEl

DOM要素を作成する

パラメータ:
- `tagName` : 要素タグ名

戻り値: 新しく作成された DOM 要素

## newLi(...tag_li)

複数の DOM 要素をバッチで作成する

パラメータ:
- `tag_li` : タグ名リスト

戻り値:
- 新しく作成されたすべての DOM 要素を含む配列

## cE(name, cls)

Web コンポーネントのカスタム要素を登録する

パラメータ:
- `name` : カスタム要素名、`c-` プレフィックスが自動的に追加されます
- `cls` : カスタム要素のクラス
