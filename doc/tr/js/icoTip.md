# icoTip.js

Uluslararasılaştırma ipuçlarıyla bir simge bileşenini tanımlayın ve kaydedin

## default(i18n_url, tag, key)

Uluslararasılaştırma istemi baloncuklarıyla özel bir HTML öğesi bileşenini kaydedin

parametre:
- `i18n_url` : Uluslararasılaştırma yapılandırması için istek adresi
- `tag` : HTML öğesinin etiket adını özelleştirin
- `key` : Uluslararasılaştırma sözlüğündeki bilgi istemi metninin anahtar değeri
