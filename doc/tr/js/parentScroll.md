# parentScroll.js

Ata kaydırma kapsayıcı öğesini alın

## default(el)

Belirtilen öğeden yukarıya doğru ilerleyin, onu içeren kaydırma kapsayıcı öğesini bulun ve döndürün (`C-VS`'in ShadowRoot yerleşik kaydırma kapsayıcısı gibi)

parametre:
- `el` : DOM öğesini başlatıyoruz

Dönüş değeri: konteyner DOM öğesini kaydır
