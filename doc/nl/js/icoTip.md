# icoTip.js

Definieer en registreer een pictogramcomponent met internationaliseringstips

## default(i18n_url, tag, key)

Registreer een aangepaste HTML-elementcomponent met internationaliseringspromptbellen

parameter:
- `i18n_url` : Verzoekadres voor internationaliseringsconfiguratie
- `tag` : Pas de tagnaam van het HTML-element aan
- `key` : De sleutelwaarde van de prompttekst in het internationaliseringswoordenboek
