## css(txt, root)

## newEl

DOM-elementen maken

parameter:
- `tagName` : naam van elementtag

Retourwaarde: nieuw aangemaakt DOM-element

## newLi(...tag_li)

Maak meerdere DOM-elementen in batches

parameter:
- `tag_li` : lijst met tagnamen

Retourwaarde:
- Array met alle nieuw gemaakte DOM-elementen

## cE(name, cls)

Registreer aangepaste elementen van Web Components

parameter:
- `name` : Aangepaste elementnaam, `c-` voorvoegsel wordt automatisch toegevoegd
- `cls` : de klasse van het aangepaste element
