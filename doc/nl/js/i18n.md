# i18n.js

Meertalig statusbeheer en wijzigingsmonitoring

## LANG_LI

Lijst met beschikbare taalconfiguraties

## lang()

Haal de huidige taal-ID op

Retourwaarde: huidige taal-ID

## langCode()

Haal de huidige taalcode op

Retourwaarde: huidige taalcode, zoals `"zh"`

## onLang(func)

Bind de luister-callback voor taalwijzigingen

parameter:
- `func`: callback-functie wanneer de taal verandert
  - parameter
    - `code` : Taalcode
    - `id` : Taal-ID

Retourwaarde: ongebonden opschoonfunctie

## langSet(id)

Schakel en zend de huidige taal uit

parameter:
- `id` : Doeltaal-ID
