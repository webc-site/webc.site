# parentScroll.js

Haal het voorouder-scrollcontainerelement op

## default(el)

Beweeg omhoog vanaf het opgegeven element, zoek het scrollcontainerelement dat het bevat en retourneer het (zoals de ingebouwde ShadowRoot-scrollcontainer van `C-VS`)

parameter:
- `el` : startend DOM-element

Retourwaarde: scroll container DOM-element
