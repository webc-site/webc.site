# i18n/store.js

Πολυγλωσσική διαχείριση τοπικής αποθήκευσης

## get()

Λάβετε το προεπιλεγμένο αναγνωριστικό γλώσσας που υπολογίζεται τοπικά στην προσωρινή μνήμη ή υπολογίζεται με αντιστοίχιση γλώσσας προγράμματος περιήγησης

Επιστρεφόμενη τιμή: αναγνωριστικό γλώσσας

## set(id)

Αποθηκεύστε προσωρινά το αναγνωριστικό της τρέχουσας γλώσσας τοπικά

παράμετρος:
- `id` : Αναγνωριστικό γλώσσας
