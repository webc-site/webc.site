# icoTip.js

Ορίστε και καταχωρίστε ένα στοιχείο εικονιδίου με υποδείξεις διεθνοποίησης

## default(i18n_url, tag, key)

Καταχωρίστε ένα προσαρμοσμένο στοιχείο στοιχείου HTML με συννεφάκια προτροπής διεθνοποίησης

παράμετρος:
- `i18n_url` : Αίτημα διεύθυνσης για διαμόρφωση διεθνοποίησης
- `tag` : Προσαρμόστε το όνομα της ετικέτας του στοιχείου HTML
- `key` : Η βασική τιμή του κειμένου προτροπής στο λεξικό διεθνοποίησης
