# dom/header.js

Δημιουργήστε τη δομή DOM της γραμμής κεφαλίδας

## default()

Δημιουργήστε και συνδυάστε τη δομή DOM κεφαλίδας των ετικετών κεφαλίδας, h1 και b

Επιστρεφόμενη τιμή:

- `header` : Στοιχείο κοντέινερ της γραμμής κεφαλίδας
- `h1` : στοιχείο τίτλου
- `b` : υπότιτλος ή στοιχείο πρόσθετων πληροφοριών
