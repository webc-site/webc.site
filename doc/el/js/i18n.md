# i18n.js

Διαχείριση κατάστασης πολλών γλωσσών και παρακολούθηση αλλαγών

## LANG_LI

Λίστα με τις διαθέσιμες διαμορφώσεις γλώσσας

## lang()

Λάβετε το τρέχον αναγνωριστικό γλώσσας

Επιστρεφόμενη τιμή: αναγνωριστικό τρέχουσας γλώσσας

## langCode()

Λάβετε τον τρέχοντα κωδικό γλώσσας

Επιστρεφόμενη τιμή: τρέχων κωδικός γλώσσας, όπως `"zh"`

## onLang(func)

Συνδέστε την επανάκληση ακρόασης για αλλαγές γλώσσας

παράμετρος:
- `func` : λειτουργία επανάκλησης όταν αλλάζει η γλώσσα
  - παράμετρος
    - `code` : Κωδικός γλώσσας
    - `id` : Αναγνωριστικό γλώσσας

Επιστρεφόμενη τιμή: Μη δεσμευμένη λειτουργία καθαρισμού

## langSet(id)

Εναλλαγή και μετάδοση της τρέχουσας γλώσσας

παράμετρος:
- `id` : Αναγνωριστικό γλώσσας στόχου
