# md/streamRender.js

Απόδοση ροής Markdown

## default(container, source)

Μετάδοση ροής κειμένου Markdown στο καθορισμένο κοντέινερ DOM, υποστήριξη εφέ δρομέα γραφομηχανής και αυτόματη κύλιση

παράμετρος:
- `container` : Στοιχείο κοντέινερ DOM που φιλοξενεί αποδομένο περιεχόμενο
- `source` : Πηγή δεδομένων κειμένου ReadableStream

Επιστρεφόμενη τιμή:

- `cancel` : Λειτουργία για διακοπή απόδοσης και ανάγνωσης ροής δεδομένων
- `setStop` : Λειτουργία για να ρυθμίσετε εάν θα παύσει η αυτόματη κύλιση
  - παράμετρος
    - `val` : Boolean τιμή είτε θα γίνει παύση είτε όχι
