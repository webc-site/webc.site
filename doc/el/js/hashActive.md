# hashActive.js

Επισήμανση συνδέσμου πλοήγησης με βάση τις άγκυρες διευθύνσεων URL και την κύλιση θυρών προβολής

## default(link_li, cls)

Αυτόματη επισήμανση συνδέσμων πλοήγησης που αντιστοιχούν στην τρέχουσα ορατή περιοχή με βάση τη θέση κύλισης σελίδας και τις αλλαγές στο σημείο αγκύρωσης της διεύθυνσης URL.

παράμετρος:
- `link_li` : Λίστα στοιχείων DOM συνδέσμου πλοήγησης
- `cls` : Ενεργό όνομα κλάσης CSS, προεπιλογή `"A"`

Επιστρεφόμενη τιμή: Ακυρώστε όλες τις λειτουργίες καθαρισμού ακρόασης
