# fetchLang.js

Ασύγχρονη φόρτωση πακέτων γλωσσών και δέσμευση ανακλήσεων ενημέρωσης

## fetchLang(getDict)

Δημιουργήστε ένα πρόγραμμα ακρόασης πακέτου γλωσσών για να αποκτήσετε ασύγχρονα το πακέτο γλώσσας και να εκτελέσετε μια επανάκληση ανανέωσης όταν αλλάζει η γλώσσα

παράμετρος:
- `getDict` : Λάβετε την ασύγχρονη λειτουργία του πακέτου γλώσσας
  - παράμετρος
    - `code` : κωδικός γλώσσας
  - Επιστρεφόμενη τιμή: λεξικό πακέτου γλώσσας

Return value: λειτουργία για καταχώρηση ανανέωσης επανάκλησης
- παράμετρος
  - `refresh` : Ανανεώστε τη λειτουργία επανάκλησης όταν ενημερώνεται η γλώσσα
    - παράμετρος
      - `i18n` : Αντικείμενο λεξικού πακέτου γλωσσών
      - `code` : Κωδικός τρέχουσας γλώσσας
- Return value: λειτουργία για την κατάργηση της καταχώρισης της επανάκλησης ανανέωσης
