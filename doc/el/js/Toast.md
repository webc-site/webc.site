# Toast.js

Προτροπή αναδυόμενου μηνύματος τοστ

## toast(render, timeout)

Εμφάνιση τυπικού αναδυόμενου παραθύρου προτροπής

παράμετρος:
- `render` : Λειτουργία απόδοσης, που χρησιμοποιείται για την προσαρμογή του αναδυόμενου περιεχομένου
  - παράμετρος
    - `el` : στοιχείο DOM του αναδυόμενου παραθύρου
- `timeout` : Δευτερόλεπτα καθυστέρησης για αυτόματο τερματισμό λειτουργίας, προεπιλογή 9, χωρίς αυτόματο τερματισμό λειτουργίας όταν 0

Επιστρεφόμενη τιμή: στοιχείο DOM του αναδυόμενου παραθύρου
