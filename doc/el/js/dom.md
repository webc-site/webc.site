## newEl

Δημιουργία στοιχείων DOM

παράμετρος:
- `tagName` : όνομα ετικέτας στοιχείου

Επιστρεφόμενη τιμή: νέο στοιχείο DOM

## newLi(...tag_li)

Δημιουργήστε πολλαπλά στοιχεία DOM σε παρτίδες

παράμετρος:
- `tag_li` : λίστα ονομάτων ετικετών

Επιστρεφόμενη τιμή:
- Πίνακας που περιέχει όλα τα στοιχεία DOM που δημιουργήθηκαν πρόσφατα

## cE(name, cls)

Καταχωρίστε προσαρμοσμένα στοιχεία Web Components

παράμετρος:
- `name` : Προσαρμοσμένο όνομα στοιχείου, το πρόθεμα `c-` θα προστεθεί αυτόματα
- `cls` : η κλάση του προσαρμοσμένου στοιχείου
