# route.js

Διαχείριση δρομολόγησης front-end μιας σελίδας

## nowUrl()

Λάβετε την τρέχουσα διαδρομή URL με την πρώτη κάθετο αφαιρεθεί

Επιστρεφόμενη τιμή: συμβολοσειρά τρέχουσας διαδρομής

## route(hook)

Καταχωρίστε την επανάκληση ακρόασης για αλλαγές διαδρομής. Κατά την εγγραφή, θα εκτελεστεί μία φορά με την τρέχουσα διαδρομή.

παράμετρος:
- `hook` : Λειτουργία επανάκλησης παρακολούθησης διαδρομής
  - παράμετρος
    - `url` : τρέχουσα διαδρομή
    - `pre` : τελευταίο μονοπάτι

Επιστρεφόμενη τιμή: Μη καταχωρημένη λειτουργία καθαρισμού

## setPre(url)

Αποθηκεύστε την τελευταία καταγεγραμμένη διεύθυνση URL

παράμετρος:
- `url` : συμβολοσειρά διαδρομής

## preUrl()

Λάβετε τη διεύθυνση URL της τελευταίας εγγραφής

Επιστρεφόμενη τιμή: συμβολοσειρά τελευταίας διαδρομής

## refresh(url)

Αναγκάστε να ενεργοποιηθούν όλες οι επανακλήσεις ακρόασης διαδρομής στην καθορισμένη διαδρομή και ενημερώστε εκ των προτέρων

παράμετρος:
- `url` : διαδρομή προορισμού, προεπιλεγμένη τρέχουσα διαδρομή

## removeSlash(url)

Αφαιρέστε κάθετες από τις κεφαλίδες διαδρομής

παράμετρος:
- `url` : συμβολοσειρά διαδρομής

Επιστρεφόμενη τιμή: διαδρομή μετά την αφαίρεση της πρώτης κάθετης

## setUrl(url)

Ενημέρωση κατάστασης διεύθυνσης URL γραμμής διευθύνσεων προγράμματος περιήγησης

παράμετρος:
- `url` : διαδρομή στόχου

Επιστρεφόμενη τιμή: 1 εάν η διαδρομή αλλάξει πραγματικά

## goto(url)

Ενημερώστε τη διεύθυνση του προγράμματος περιήγησης και ενεργοποιήστε μια ανανέωση διαδρομής

παράμετρος:
- `url` : διαδρομή στόχου
