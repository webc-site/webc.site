# c-t.js

Διεθνοποίηση HTML

Η ετικέτα `<c-t>key</c-t>` είναι προσαρμοσμένη και η `key` είναι το κλειδί του πακέτου γλώσσας.

## cTranSet(el, genUrl)

Ορίστε το κείμενο για `c-t`.

παράμετρος:
- `el` : ριζικό στοιχείο, καθολικό μπορεί να μεταβιβαστεί στο `document`
- `genUrl` : λειτουργία λήψης λεξικού πακέτου γλωσσών
  - παράμετρος
    - `code` : κωδικός γλώσσας
  - Επιστρεφόμενη τιμή: ένα αντικείμενο λεξικού πακέτου γλώσσας ή μια υπόσχεση που επιλύεται σε αυτό το αντικείμενο
