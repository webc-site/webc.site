# i18n.js

Többnyelvű állapotkezelés és változásfigyelés

## LANG_LI

Az elérhető nyelvi konfigurációk listája

## lang()

Szerezze meg az aktuális nyelvazonosítót

Visszatérési érték: aktuális nyelvazonosító

## langCode()

Szerezze meg az aktuális nyelvkódot

Visszatérési érték: aktuális nyelvkód, például `"zh"`

## onLang(func)

Kapcsolja össze a visszahívást a nyelvi változtatásokhoz

paraméter:
- `func` : visszahívási funkció nyelvváltáskor
  - paraméter
    - `code` : Nyelvkód
    - `id` : Nyelvazonosító

Visszatérési érték: Kötetlen tisztítási funkció

## langSet(id)

Az aktuális nyelv váltása és sugárzása

paraméter:
- `id` : Célnyelv azonosítója
