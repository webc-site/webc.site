# icoTip.js

Határozzon meg és regisztráljon egy ikonkomponenst nemzetköziesítési tippekkel

## default(i18n_url, tag, key)

Regisztráljon egyéni HTML-elem-összetevőt nemzetközivé tételi buborékokkal

paraméter:
- `i18n_url` : Cím kérése nemzetköziesítési konfigurációhoz
- `tag` : A HTML-elem címkenevének testreszabása
- `key` : A prompt szöveg kulcsértéke a nemzetköziesítési szótárban
