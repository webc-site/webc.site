# i18n/store.js

Többnyelvű helyi tárkezelés

## get()

Lekérheti az alapértelmezett nyelvi azonosítót helyi gyorsítótárban, vagy a böngésző nyelvi egyeztetésével kiszámítva

Visszatérési érték: nyelvazonosító

## set(id)

Gyorsítótárazza az aktuális nyelvazonosítót helyileg

paraméter:
- `id` : Nyelvazonosító
