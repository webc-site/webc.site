# parentScroll.js

Szerezd meg az ős görgetőtároló elemet

## default(el)

Haladjon felfelé a megadott elemtől, keresse meg és adja vissza az azt tartalmazó görgetőtároló elemet (például `C-VS` ShadowRoot beépített görgetőtárolóját)

paraméter:
- `el` : kezdő DOM elem

Visszatérési érték: görgetőtároló DOM elem
