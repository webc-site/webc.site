# md/streamRender.js

માર્કડાઉન સ્ટ્રીમિંગ રેન્ડરિંગ

## default(container, source)

ઉલ્લેખિત DOM કન્ટેનરમાં માર્કડાઉન ટેક્સ્ટનું સ્ટ્રીમિંગ રેન્ડરિંગ, ટાઈપરાઈટર કર્સર ઈફેક્ટ્સ અને ઓટોમેટિક સ્ક્રોલિંગને સપોર્ટ કરે છે

પરિમાણ:
- `container` : DOM કન્ટેનર તત્વ હોસ્ટિંગ પ્રસ્તુત સામગ્રી
- `source` : વાંચવા યોગ્ય સ્ટ્રીમ ટેક્સ્ટ ડેટા સ્ત્રોત

વળતર મૂલ્ય:

- `cancel` : રેન્ડરિંગ અને ડેટા સ્ટ્રીમ રીડિંગને રોકવા માટેનું કાર્ય
- `setStop` : સ્વચાલિત સ્ક્રોલિંગને થોભાવવું કે નહીં તે સેટ કરવા માટેનું કાર્ય
  - પરિમાણ
    - `val` : બુલિયન મૂલ્ય થોભાવવું કે નહીં
