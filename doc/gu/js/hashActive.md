# hashActive.js

URL એન્કર અને વ્યુપોર્ટ સ્ક્રોલિંગ પર આધારિત નેવિગેશન લિંક હાઇલાઇટિંગ

## default(link_li, cls)

પૃષ્ઠ સ્ક્રોલ સ્થિતિ અને URL એન્કર પોઇન્ટ ફેરફારોના આધારે હાલમાં દૃશ્યમાન વિસ્તારને અનુરૂપ નેવિગેશન લિંક્સને આપમેળે હાઇલાઇટ કરો.

પરિમાણ:
- `link_li` : નેવિગેશન લિંક DOM ઘટક સૂચિ
- `cls` : સક્રિય CSS વર્ગનું નામ, ડિફોલ્ટ `"A"`

વળતર મૂલ્ય: સાંભળવાના તમામ ક્લીનઅપ કાર્યોને રદ કરો
