# i18n.js

મલ્ટી-લેંગ્વેજ સ્ટેટસ મેનેજમેન્ટ અને ચેન્જ મોનિટરિંગ

## LANG_LI

ઉપલબ્ધ ભાષા રૂપરેખાંકનોની યાદી

## lang()

વર્તમાન ભાષા ID મેળવો

વળતર મૂલ્ય: વર્તમાન ભાષા ID

## langCode()

વર્તમાન ભાષા કોડ મેળવો

વળતર મૂલ્ય: વર્તમાન ભાષા કોડ, જેમ કે `"zh"`

## onLang(func)

ભાષા ફેરફારો માટે સાંભળવાના કૉલબેકને બાંધો

પરિમાણ:
- `func` : જ્યારે ભાષા બદલાય ત્યારે કૉલબેક કાર્ય
  - પરિમાણ
    - `code` : ભાષા કોડ
    - `id` : ભાષા ID

વળતર મૂલ્ય: અનબાઉન્ડ સફાઈ કાર્ય

## langSet(id)

વર્તમાન ભાષાને સ્વિચ કરો અને બ્રોડકાસ્ટ કરો

પરિમાણ:
- `id` : લક્ષ્ય ભાષા ID
