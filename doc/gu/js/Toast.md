# Toast.js

ટોસ્ટ પોપ-અપ સંદેશ પ્રોમ્પ્ટ

## toast(render, timeout)

માનક પ્રોમ્પ્ટ પોપ-અપ વિન્ડો બતાવો

પરિમાણ:
- `render` : રેન્ડરીંગ ફંક્શન, પોપ-અપ સામગ્રીને કસ્ટમાઇઝ કરવા માટે વપરાય છે
  - પરિમાણ
    - `el` : પોપ-અપ વિન્ડોનું DOM ઘટક
- `timeout` : સ્વચાલિત શટડાઉન માટે વિલંબ સેકંડ, ડિફોલ્ટ 9, જ્યારે 0 હોય ત્યારે સ્વચાલિત શટડાઉન નહીં

વળતર મૂલ્ય: પોપ-અપ વિન્ડોનું DOM ઘટક
