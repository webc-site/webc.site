# route.js

સિંગલ પેજ ફ્રન્ટ-એન્ડ રૂટીંગ મેનેજમેન્ટ

## nowUrl()

અગ્રણી સ્લેશ દૂર કરીને વર્તમાન URL પાથ મેળવો

વળતર મૂલ્ય: વર્તમાન પાથ સ્ટ્રિંગ

## route(hook)

રૂટ ફેરફારો માટે સાંભળવાના કોલબેકની નોંધણી કરો. નોંધણી કરતી વખતે, તે વર્તમાન પાથ સાથે એકવાર ચલાવવામાં આવશે.

પરિમાણ:
- `hook` : રૂટ મોનિટરિંગ કૉલબેક કાર્ય
  - પરિમાણ
    - `url` : વર્તમાન પાથ
    - `pre` : છેલ્લો રસ્તો

વળતર મૂલ્ય: બિન નોંધાયેલ સફાઈ કાર્ય

## setPre(url)

છેલ્લું રેકોર્ડ કરેલ URL સાચવો

પરિમાણ:
- `url` : પાથ સ્ટ્રિંગ

## preUrl()

છેલ્લા રેકોર્ડનું URL મેળવો

વળતર મૂલ્ય: છેલ્લી પાથ સ્ટ્રિંગ

## refresh(url)

ઉલ્લેખિત પાથ પર ટ્રિગર થવા માટે તમામ રૂટ સાંભળવાના કૉલબૅક્સને દબાણ કરો અને પૂર્વ અપડેટ કરો

પરિમાણ:
- `url` : લક્ષ્ય પાથ, ડિફોલ્ટ વર્તમાન પાથ

## removeSlash(url)

પાથ હેડરોમાંથી સ્લેશ દૂર કરો

પરિમાણ:
- `url` : પાથ સ્ટ્રિંગ

વળતર મૂલ્ય: અગ્રણી સ્લેશ દૂર કર્યા પછીનો માર્ગ

## setUrl(url)

બ્રાઉઝર એડ્રેસ બાર URL સ્ટેટસ અપડેટ કરો

પરિમાણ:
- `url` : લક્ષ્ય પાથ

વળતર મૂલ્ય: 1 જો પાથ ખરેખર બદલાય છે

## goto(url)

બ્રાઉઝર સરનામું અપડેટ કરો અને રૂટ રિફ્રેશને ટ્રિગર કરો

પરિમાણ:
- `url` : લક્ષ્ય પાથ
