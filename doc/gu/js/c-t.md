# c-t.js

HTML આંતરરાષ્ટ્રીયકરણ

ટેગ `<c-t>key</c-t>` કસ્ટમાઇઝ કરેલ છે, અને `key` ભાષા પેકની ચાવી છે.

## cTranSet(el, genUrl)

`c-t` માટે ટેક્સ્ટ સેટ કરો.

પરિમાણ:
- `el` : રુટ એલિમેન્ટ, વૈશ્વિક `document` માં પસાર કરી શકાય છે
- `genUrl` : ભાષા પેક શબ્દકોશ મેળવવા માટેનું કાર્ય
  - પરિમાણ
    - `code` : ભાષા કોડ
  - રીટર્ન વેલ્યુ: લેંગ્વેજ પેક ડિક્શનરી ઑબ્જેક્ટ, અથવા પ્રોમિસ કે જે આ ઑબ્જેક્ટને ઉકેલે છે
