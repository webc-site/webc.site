# dom.js

દસ્તાવેજ ચલ સંક્ષેપ અને શોર્ટકટ કાર્યો

## D

`document` ઑબ્જેક્ટનો શૉર્ટકટ સંદર્ભ

## B

`document.body` ઑબ્જેક્ટનો શૉર્ટકટ સંદર્ભ

## css(txt, root)

## newEl

DOM તત્વો બનાવો

પરિમાણ:
- `tagName` : તત્વ ટેગ નામ

વળતર મૂલ્ય: નવું બનાવેલ DOM ઘટક

## newLi(...tag_li)

બેચમાં બહુવિધ DOM તત્વો બનાવો

પરિમાણ:
- `tag_li` : ટેગ નામની સૂચિ

વળતર મૂલ્ય:
- બધા નવા બનાવેલા DOM ઘટકો ધરાવતો અરે

## cE(name, cls)

વેબ ઘટકો કસ્ટમ ઘટકોની નોંધણી કરો

પરિમાણ:
- `name` : કસ્ટમ તત્વનું નામ, `c-` ઉપસર્ગ આપમેળે ઉમેરવામાં આવશે
- `cls` : કસ્ટમ ઘટકનો વર્ગ
