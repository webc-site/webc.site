# fetchLang.js

અસુમેળ રીતે ભાષા પૅક્સ લોડ કરો અને અપડેટ કૉલબૅક્સને જોડો

## fetchLang(getDict)

ભાષા પૅકને અસિંક્રોનસ રીતે મેળવવા માટે ભાષા પૅક શ્રોતા બનાવો અને જ્યારે ભાષા બદલાય ત્યારે રિફ્રેશ કૉલબૅક ચલાવો

પરિમાણ:
- `getDict` : ભાષા પેકેજનું અસુમેળ કાર્ય મેળવો
  - પરિમાણ
    - `code` : ભાષા કોડ
  - વળતર મૂલ્ય: ભાષા પેક શબ્દકોશ

રીટર્ન વેલ્યુ: રિફ્રેશ કોલબેક રજીસ્ટર કરવા માટે ફંક્શન
- પરિમાણ
  - `refresh` : જ્યારે ભાષા અપડેટ થાય ત્યારે કૉલબેક ફંક્શન રિફ્રેશ કરો
    - પરિમાણ
      - `i18n` : ભાષા પેક શબ્દકોશ ઑબ્જેક્ટ
      - `code` : વર્તમાન ભાષા કોડ
- રીટર્ન વેલ્યુ: રિફ્રેશ કોલબેકની નોંધણી રદ કરવા માટેનું કાર્ય
