# icoTip.js

આંતરરાષ્ટ્રીયકરણ સંકેતો સાથે આઇકોન ઘટકને વ્યાખ્યાયિત કરો અને નોંધણી કરો

## default(i18n_url, tag, key)

આંતરરાષ્ટ્રીયકરણ પ્રોમ્પ્ટ બબલ્સ સાથે કસ્ટમ HTML તત્વ ઘટકની નોંધણી કરો

પરિમાણ:
- `i18n_url` : આંતરરાષ્ટ્રીયકરણ ગોઠવણી માટે સરનામાની વિનંતી કરો
- `tag` : HTML ઘટકના ટેગ નામને કસ્ટમાઇઝ કરો
- `key` : આંતરરાષ્ટ્રીયકરણ શબ્દકોશમાં પ્રોમ્પ્ટ ટેક્સ્ટનું મુખ્ય મૂલ્ય
