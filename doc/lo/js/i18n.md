# i18n.js

ການຄຸ້ມຄອງສະຖານະການຫຼາຍພາສາແລະການຕິດຕາມການປ່ຽນແປງ

## LANG_LI

ບັນຊີລາຍຊື່ຂອງການຕັ້ງຄ່າພາສາທີ່ມີຢູ່

## lang()

ເອົາ ID ພາສາປັດຈຸບັນ

ຄ່າກັບຄືນ: ID ພາສາປັດຈຸບັນ

## langCode()

ເອົາລະຫັດພາສາປະຈຸບັນ

ມູນຄ່າກັບຄືນ: ລະຫັດພາສາປະຈຸບັນ, ເຊັ່ນ `"zh"`

## onLang(func)

ຜູກ​ມັດ​ການ​ຮັບ​ຟັງ​ສໍາ​ລັບ​ການ​ປ່ຽນ​ແປງ​ພາ​ສາ​

ພາລາມິເຕີ:
- `func` : ຟັງຊັນໂທກັບເມື່ອປ່ຽນພາສາ
  - ພາລາມິເຕີ
    - `code` : ລະຫັດພາສາ
    - `id` : ID ພາສາ

ມູນຄ່າກັບຄືນ: ຟັງຊັນການລ້າງບໍ່ຖືກຜູກມັດ

## langSet(id)

ສະຫຼັບ ແລະອອກອາກາດພາສາປັດຈຸບັນ

ພາລາມິເຕີ:
- `id` : ID ພາສາເປົ້າໝາຍ
