# fetch.js

ຟັງຊັນເຄື່ອງມືການຮ້ອງຂໍເຄືອຂ່າຍໂດຍອີງໃສ່ fetch encapsulation

## F(url, opt)

ສົ່ງຄໍາຮ້ອງຂໍເຄືອຂ່າຍແລະສົ່ງຄືນວັດຖຸຕອບສະຫນອງ

ພາລາມິເຕີ:
- `url` : ຮ້ອງຂໍທີ່ຢູ່
- `opt` : `fetch` ທາງເລືອກວັດຖຸ

ຄ່າຕອບແທນ: ຕອບສະໜອງວັດຖຸ

## f(attr)

ຟັງຊັນຄໍາສັ່ງທີ່ສູງກວ່າທີ່ຫໍ່ຄໍາຮ້ອງຂໍແລະວິເຄາະເນື້ອຫາທີ່ສົ່ງຄືນໃນຮູບແບບທີ່ກໍານົດ

ພາລາມິເຕີ:
- `attr` : ຊື່ຄຸນສົມບັດການວິເຄາະສ່ວນຕອບສະໜອງ

ກັບຄືນມູນຄ່າ: ຟັງຊັນເພື່ອຮ້ອງຂໍ asynchronously ແລະ parse ເນື້ອໃນ
  - ພາລາມິເຕີ
    - `url` : ຮ້ອງຂໍທີ່ຢູ່
    - `opt` : `fetch` ທາງເລືອກວັດຖຸ
  - ມູນຄ່າກັບຄືນ: ເນື້ອໃນທີ່ວິເຄາະແລ້ວ

## fTxt

ສົ່ງຄໍາຮ້ອງຂໍເຄືອຂ່າຍແລະສົ່ງຄືນເນື້ອໃນການຕອບສະຫນອງໃນຮູບແບບຂໍ້ຄວາມ

ພາລາມິເຕີ:
- `url` : ຮ້ອງຂໍທີ່ຢູ່
- `opt` : `fetch` ທາງເລືອກວັດຖຸ

ຄ່າກັບຄືນ: ຂໍ້ຄວາມຕອບ

## fJson(url, opt)

ສົ່ງຄໍາຮ້ອງຂໍເຄືອຂ່າຍແລະວິເຄາະແລະສົ່ງຄືນເນື້ອໃນການຕອບສະຫນອງໃນຮູບແບບ JSON

ພາລາມິເຕີ:
- `url` : ຮ້ອງຂໍທີ່ຢູ່
- `opt` : `fetch` ທາງເລືອກວັດຖຸ

ຄ່າກັບຄືນ: ແຍກວິເຄາະວັດຖຸ JSON

## fBin(url, opt)

ສົ່ງຄໍາຮ້ອງຂໍເຄືອຂ່າຍແລະສົ່ງຄືນຮ່າງກາຍຕອບສະຫນອງເປັນ array ໄບນາຣີ

ພາລາມິເຕີ:
- `url` : ຮ້ອງຂໍທີ່ຢູ່
- `opt` : `fetch` ທາງເລືອກວັດຖຸ

ຄ່າກັບຄືນ: byte array
