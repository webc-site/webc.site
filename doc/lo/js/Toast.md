# Toast.js

Toast ຂໍ້ຄວາມ pop-up prompt

## toast(render, timeout)

ສະແດງໜ້າຈໍປັອບອັບມາດຕະຖານ

ພາລາມິເຕີ:
- `render` : ຟັງຊັນການສະແດງຜົນ, ໃຊ້ເພື່ອປັບແຕ່ງເນື້ອຫາປັອບອັບ
  - ພາລາມິເຕີ
    - `el` : ອົງປະກອບ DOM ຂອງໜ້າຕ່າງປັອບອັບ
- `timeout` : ເລື່ອນວິນາທີສຳລັບການປິດອັດຕະໂນມັດ, ຄ່າເລີ່ມຕົ້ນ 9, ບໍ່ມີການປິດອັດຕະໂນມັດເມື່ອ 0

ຄ່າກັບຄືນ: ອົງປະກອບ DOM ຂອງປ່ອງຢ້ຽມປັອບອັບ

## toastErr(render, timeout)

ສະແດງຂໍ້ຄວາມສະແດງຂໍ້ຜິດພາດ

ພາລາມິເຕີ:
- `render` : ຟັງຊັນການສະແດງຜົນ, ໃຊ້ເພື່ອປັບແຕ່ງເນື້ອຫາປັອບອັບ
  - ພາລາມິເຕີ
    - `el` : ອົງປະກອບ DOM ຂອງໜ້າຕ່າງປັອບອັບ
- `timeout` : ເລື່ອນວິນາທີສຳລັບການປິດອັດຕະໂນມັດ, ຄ່າເລີ່ມຕົ້ນ 9, ບໍ່ມີການປິດອັດຕະໂນມັດເມື່ອ 0

ກັບຄືນມູນຄ່າ: ອົງປະກອບ DOM ຂອງປ່ອງຢ້ຽມປ໊ອບອັບຄວາມຜິດພາດ
