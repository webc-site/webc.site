# md/streamRender.js

Markdown ການສະແດງຜົນການຖ່າຍທອດ

## default(container, source)

ການຖ່າຍທອດການສະແດງຂໍ້ຄວາມ Markdown ເຂົ້າໄປໃນຖັງ DOM ທີ່ລະບຸ, ສະຫນັບສະຫນູນຜົນກະທົບຕົວກະພິບຂອງເຄື່ອງພິມດີດແລະການເລື່ອນອັດຕະໂນມັດ

ພາລາມິເຕີ:
- `container` : ອົງປະກອບບັນຈຸ DOM ທີ່ໂຮດເນື້ອຫາທີ່ສະແດງ
- `source` : ແຫຼ່ງຂໍ້ມູນຂໍ້ຄວາມ ReadableStream

ຄ່າຕອບແທນ:

- `cancel` : ຟັງຊັນເພື່ອຢຸດການສະແດງຜົນ ແລະການອ່ານການຖ່າຍທອດຂໍ້ມູນ
- `setStop` : ຟັງຊັນເພື່ອຕັ້ງວ່າຈະຢຸດການເລື່ອນອັດຕະໂນມັດຊົ່ວຄາວຫຼືບໍ່
  - ພາລາມິເຕີ
    - `val` : ຄ່າ Boolean ວ່າຈະຢຸດຊົ່ວຄາວຫຼືບໍ່
