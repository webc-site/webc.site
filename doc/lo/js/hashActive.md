# hashActive.js

ການເນັ້ນການເຊື່ອມຕໍ່ການນໍາທາງໂດຍອີງໃສ່ສະມໍ URL ແລະການເລື່ອນ viewport

## default(link_li, cls)

ເນັ້ນໃສ່ການເຊື່ອມຕໍ່ການນໍາທາງໂດຍອັດຕະໂນມັດທີ່ສອດຄ້ອງກັບພື້ນທີ່ທີ່ເຫັນໄດ້ໃນປັດຈຸບັນໂດຍອີງໃສ່ຕໍາແຫນ່ງເລື່ອນຫນ້າແລະການປ່ຽນແປງຈຸດສະມໍ URL.

ພາລາມິເຕີ:
- `link_li` : ການນຳທາງເຊື່ອມຕໍ່ລາຍການອົງປະກອບ DOM
- `cls` : ຊື່ຄລາສ CSS ທີ່ໃຊ້ໄດ້, ຄ່າເລີ່ມຕົ້ນ `"A"`

ກັບຄືນມູນຄ່າ: ຍົກເລີກຟັງຊັນທໍາຄວາມສະອາດການຟັງທັງໝົດ
