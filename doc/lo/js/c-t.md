# c-t.js

HTML ເປັນສາກົນ

ແທັກ `<c-t>key</c-t>`ຖືກປັບແຕ່ງ, ແລະ `key`ແມ່ນກະແຈຂອງຊຸດພາສາ.

## cTranSet(el, genUrl)

ກໍານົດຂໍ້ຄວາມສໍາລັບ `c-t`.

ພາລາມິເຕີ:
- `el` : ອົງປະກອບຮາກ, ທົ່ວໂລກສາມາດຜ່ານໄດ້ໃນ `document`
- `genUrl` : ຟັງຊັນເພື່ອຮັບເອົາວັດຈະນານຸກົມຊຸດພາສາ
  - ພາລາມິເຕີ
    - `code` : ລະຫັດພາສາ
  - ມູນຄ່າກັບຄືນ: ວັດຖຸວັດຈະນານຸກົມຊຸດພາສາ, ຫຼືຄໍາສັນຍາທີ່ແກ້ໄຂວັດຖຸນີ້
