# icoTip.js

ກໍາ​ນົດ​ແລະ​ລົງ​ທະ​ບຽນ​ອົງ​ປະ​ກອບ icon ທີ່​ມີ​ຄໍາ​ແນະ​ນໍາ​ການ​ສາ​ກົນ​

## default(i18n_url, tag, key)

ລົງທະບຽນອົງປະກອບອົງປະກອບ HTML ແບບກຳນົດເອງທີ່ມີ bubble ເຕືອນໃຫ້ສາກົນ

ພາລາມິເຕີ:
- `i18n_url` : ຮ້ອງຂໍທີ່ຢູ່ສໍາລັບການຕັ້ງຄ່າສາກົນ
- `tag` : ປັບແຕ່ງຊື່ແທັກຂອງອົງປະກອບ HTML
- `key` : ຄ່າຫຼັກຂອງຂໍ້ຄວາມເຕືອນໃນວັດຈະນານຸກົມສາກົນ
