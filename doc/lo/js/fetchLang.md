# fetchLang.js

ໂຫຼດຊຸດພາສາແບບ Asynchronously ແລະຜູກມັດການອັບເດດການໂທຄືນ

## fetchLang(getDict)

ສ້າງຊຸດຟັງພາສາເພື່ອຮັບຊຸດພາສາແບບບໍ່ກົງກັນ ແລະດໍາເນີນການໂທກັບຄືນໃຫມ່ເມື່ອພາສາມີການປ່ຽນແປງ

ພາລາມິເຕີ:
- `getDict` : ຮັບເອົາຟັງຊັນ asynchronous ຂອງຊຸດພາສາ
  - ພາລາມິເຕີ
    - `code` : ລະຫັດພາສາ
  - ມູນຄ່າກັບຄືນ: ວັດຈະນານຸກົມຊຸດພາສາ

ມູນຄ່າກັບຄືນ: ຟັງຊັນເພື່ອລົງທະບຽນການເອີ້ນຄືນຄືນໃຫມ່
- ພາລາມິເຕີ
  - `refresh` : ໂຫຼດຟັງຊັນໂທຄືນໃໝ່ເມື່ອມີການອັບເດດພາສາ
    - ພາລາມິເຕີ
      - `i18n` : ວັດຈະນານຸກົມຊຸດພາສາ
      - `code` : ລະຫັດພາສາປັດຈຸບັນ
- ກັບຄືນມູນຄ່າ: ຟັງຊັນເພື່ອຍົກເລີກການລົງທະບຽນການໂທຄືນໃຫມ່
