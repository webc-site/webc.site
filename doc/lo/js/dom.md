# dom.js

ເອກະສານຫຍໍ້ຕົວແປແລະຟັງຊັນທາງລັດ

## D

`document`ທາງລັດອ້າງອີງເຖິງວັດຖຸ

## B

`document.body`ທາງລັດອ້າງອີງເຖິງວັດຖຸ

## newEl

ສ້າງອົງປະກອບ DOM

ພາລາມິເຕີ:
- `tagName` : ຊື່ແທັກອົງປະກອບ

ມູນຄ່າກັບຄືນ: ອົງປະກອບ DOM ທີ່ສ້າງຂຶ້ນໃຫມ່

## newLi(...tag_li)

ສ້າງອົງປະກອບ DOM ຫຼາຍອັນໃນຊຸດ

ພາລາມິເຕີ:
- `tag_li` : ລາຍຊື່ແທັກ

ຄ່າຕອບແທນ:
- Array ມີອົງປະກອບ DOM ທີ່ສ້າງຂຶ້ນໃໝ່ທັງໝົດ

## cE(name, cls)

ລົງທະບຽນອົງປະກອບເວັບທີ່ກໍາຫນົດເອງ

ພາລາມິເຕີ:
- `name` : ຊື່ອົງປະກອບແບບກຳນົດເອງ, `c-` ຄຳນຳໜ້າຈະຖືກເພີ່ມໂດຍອັດຕະໂນມັດ
- `cls` : ຊັ້ນຂອງອົງປະກອບກຳນົດເອງ
