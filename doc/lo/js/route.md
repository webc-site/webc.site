# route.js

ການຈັດການເສັ້ນທາງໜ້າດຽວ

## nowUrl()

ເອົາເສັ້ນທາງ URL ປະຈຸບັນທີ່ມີ slash ຊັ້ນນໍາອອກ

ຄ່າກັບຄືນ: ສະຕຣິງເສັ້ນທາງປະຈຸບັນ

## route(hook)

ລົງທະບຽນການໂທກັບຟັງສໍາລັບການປ່ຽນແປງເສັ້ນທາງ. ເມື່ອລົງທະບຽນ, ມັນຈະຖືກປະຕິບັດຄັ້ງດຽວກັບເສັ້ນທາງປະຈຸບັນ.

ພາລາມິເຕີ:
- `hook` : ຟັງຊັນການຕິດຕາມເສັ້ນທາງການໂທກັບ
  - ພາລາມິເຕີ
    - `url` : ເສັ້ນທາງປະຈຸບັນ
    - `pre` : ເສັ້ນທາງສຸດທ້າຍ

ກັບຄືນມູນຄ່າ: ຟັງຊັນການທໍາຄວາມສະອາດທີ່ບໍ່ໄດ້ລົງທະບຽນ

## setPre(url)

ບັນທຶກ URL ທີ່ບັນທຶກໄວ້ຫຼ້າສຸດ

ພາລາມິເຕີ:
- `url` : ສະຕຣິງເສັ້ນທາງ

## preUrl()

ເອົາ URL ຂອງບັນທຶກສຸດທ້າຍ

ກັບຄືນມູນຄ່າ: ເສັ້ນເສັ້ນທາງສຸດທ້າຍ

## refresh(url)

ບັງຄັບໃຫ້ມີການໂທຫາການຟັງເສັ້ນທາງທັງໝົດໃນເສັ້ນທາງທີ່ກຳນົດໄວ້ ແລະອັບເດດກ່ອນ

ພາລາມິເຕີ:
- `url` : ເສັ້ນທາງເປົ້າໝາຍ, ເສັ້ນທາງປັດຈຸບັນເລີ່ມຕົ້ນ

## removeSlash(url)

ເອົາ slashes ອອກຈາກສ່ວນຫົວເສັ້ນທາງ

ພາລາມິເຕີ:
- `url` : ສະຕຣິງເສັ້ນທາງ

ກັບຄືນມູນຄ່າ: ເສັ້ນທາງຫຼັງຈາກຖອນ slash ຊັ້ນນໍາ

## setUrl(url)

ອັບເດດສະຖານະ URL ແຖບທີ່ຢູ່ຂອງຕົວທ່ອງເວັບ

ພາລາມິເຕີ:
- `url` : ເສັ້ນທາງເປົ້າໝາຍ

ມູນຄ່າກັບຄືນ: 1 ຖ້າເສັ້ນທາງປ່ຽນແປງຕົວຈິງ

## goto(url)

ອັບເດດທີ່ຢູ່ບຣາວເຊີ ແລະກະຕຸ້ນການໂຫຼດເສັ້ນທາງຄືນໃໝ່

ພາລາມິເຕີ:
- `url` : ເສັ້ນທາງເປົ້າໝາຍ
