# dom.js

Dəyişən qısaltmaları və qısayol funksiyalarını sənədləşdirin

## D

`document` Obyektə qısa yol istinadı

## B

`document.body` Obyektə qısa yol istinadı

## newEl

DOM elementləri yaradın

parametr:
- `tagName` : element teq adı

Qaytarma dəyəri: yeni yaradılmış DOM elementi

## newLi(...tag_li)

Partiyalarda çoxlu DOM elementləri yaradın

parametr:
- `tag_li` : teq adı siyahısı

Qaytarma dəyəri:
- Bütün yeni yaradılmış DOM elementlərini ehtiva edən massiv

## cE(name, cls)

Veb Komponentlərinin fərdi elementlərini qeydiyyatdan keçirin

parametr:
- `name` : Fərdi element adı, `c-` prefiksi avtomatik əlavə olunacaq
- `cls` : fərdi elementin sinfi
