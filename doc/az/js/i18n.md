# i18n.js

Çoxdilli statusun idarə edilməsi və dəyişiklik monitorinqi

## LANG_LI

Mövcud dil konfiqurasiyalarının siyahısı

## lang()

Cari dil identifikatorunu əldə edin

Qaytarılan dəyər: cari dil identifikatoru

## langCode()

Cari dil kodunu əldə edin

Qaytarılan dəyər: cari dil kodu, məsələn, `"zh"`

## onLang(func)

Dil dəyişiklikləri üçün dinləmə geri çağırışını bağlayın

parametr:
- `func` : dil dəyişdikdə geri çağırış funksiyası
  - parametr
    - `code` : Dil kodu
    - `id` : Dil ID-si

Qaytarma dəyəri: Bağlanmamış təmizləmə funksiyası

## langSet(id)

Cari dili dəyişdirin və yayımlayın

parametr:
- `id` : Hədəf dil ID-si
