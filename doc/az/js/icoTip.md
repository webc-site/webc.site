# icoTip.js

Beynəlxalqləşdirmə göstərişləri ilə simvol komponentini müəyyənləşdirin və qeyd edin

## default(i18n_url, tag, key)

Xüsusi HTML element komponentini beynəlmiləlləşdirmə xəbərdarlığı ilə qeyd edin

parametr:
- `i18n_url` : Beynəlxalqlaşdırma konfiqurasiyası üçün ünvan sorğusu
- `tag` : HTML elementinin teq adını fərdiləşdirin
- `key` : Beynəlmiləlləşdirmə lüğətindəki sorğu mətninin əsas dəyəri
