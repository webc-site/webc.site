# parentScroll.js

Ata sürüşdürmə konteyner elementini əldə edin

## default(el)

Göstərilən elementdən yuxarıya doğru hərəkət edin, onu ehtiva edən sürüşdürmə konteyner elementini tapın və qaytarın (məsələn, `C-VS` ShadowRoot daxili sürüşmə konteyneri)

parametr:
- `el` : DOM elementi başlayır

Qaytarma dəyəri: fırladın konteyner DOM elementi
