# hashActive.js

URL lövbərləri və görünüş portunun sürüşdürülməsinə əsaslanan naviqasiya linkinin vurğulanması

## default(link_li, cls)

Səhifə sürüşdürmə mövqeyinə və URL anker nöqtəsi dəyişikliklərinə əsasən hazırda görünən sahəyə uyğun naviqasiya bağlantılarını avtomatik olaraq vurğulayın.

parametr:
- `link_li` : Naviqasiya linki DOM element siyahısı
- `cls` : Aktiv CSS sinif adı, defolt `"A"`

Dəyəri qaytarın: Bütün dinləmə təmizləmə funksiyalarını ləğv edin
