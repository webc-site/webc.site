# i18n.js

Rheoli statws aml-iaith a monitro newid

## LANG_LI

Rhestr o ffurfweddau iaith sydd ar gael

## lang()

Cael ID iaith gyfredol

Gwerth dychwelyd: ID iaith gyfredol

## langCode()

Cael y cod iaith cyfredol

Gwerth dychwelyd: cod iaith cyfredol, megis `"zh"`

## onLang(func)

Rhwymwch yr alwad gwrando am newidiadau iaith

paramedr:
- `func` : swyddogaeth galw'n ôl pan fydd iaith yn newid
  - paramedr
    - `code` : Cod iaith
    - `id` : ID iaith

Gwerth dychwelyd: Swyddogaeth glanhau heb ei rwymo

## langSet(id)

Newid a darlledu'r iaith gyfredol

paramedr:
- `id` : ID iaith darged
