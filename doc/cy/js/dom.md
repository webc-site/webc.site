# dom.js

Dogfennu byrfoddau newidiol a swyddogaethau llwybr byr

## D

`document` Cyfeirnod llwybr byr at y gwrthrych

## B

`document.body` Cyfeirnod llwybr byr at y gwrthrych

## newEl

Creu elfennau DOM

paramedr:
- `tagName` : enw tag elfen

Gwerth dychwelyd: elfen DOM newydd ei chreu

## newLi(...tag_li)

Creu elfennau DOM lluosog mewn sypiau

paramedr:
- `tag_li` : rhestr enw tag

Gwerth dychwelyd:
- Arae yn cynnwys yr holl elfennau DOM sydd newydd eu creu

## cE(name, cls)

Cofrestru Elfennau Gwe elfennau arferiad

paramedr:
- `name` : Enw elfen personol, `c-` ychwanegir rhagddodiad yn awtomatig
- `cls` : dosbarth yr elfen arferiad
