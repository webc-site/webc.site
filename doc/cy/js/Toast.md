# Toast.js

Tost anogwr neges pop-up

## toast(render, timeout)

Dangos ffenestr naid brydlon safonol

paramedr:
- `render` : Swyddogaeth rendro, a ddefnyddir i addasu cynnwys naid
  - paramedr
    - `el` : elfen DOM y ffenestr naid
- `timeout` : Oedi eiliadau ar gyfer cau i lawr yn awtomatig, rhagosodiad 9, dim cau i lawr yn awtomatig pan 0

Gwerth dychwelyd: Elfen DOM y ffenestr naid
