# hashActive.js

Amlygiad cyswllt llywio yn seiliedig ar angorau URL a sgrolio porth gwylio

## default(link_li, cls)

Amlygwch yn awtomatig ddolenni llywio sy'n cyfateb i'r ardal sydd i'w gweld ar hyn o bryd yn seiliedig ar safle sgrolio'r dudalen a newidiadau pwynt angori URL.

paramedr:
- `link_li` : Dolen llywio rhestr elfennau DOM
- `cls` : Enw dosbarth CSS gweithredol, `"A"` rhagosodedig

Gwerth dychwelyd: Canslo'r holl swyddogaethau glanhau gwrando
