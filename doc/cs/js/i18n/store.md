# i18n/store.js

Vícejazyčná správa místního úložiště

## get()

Získejte výchozí ID jazyka vypočítané lokálně uložené v mezipaměti nebo vypočítané podle shody jazyka prohlížeče

Návratová hodnota: ID jazyka

## set(id)

Lokálně uložte ID aktuálního jazyka do mezipaměti

parametr:
- `id` : ID jazyka
