# icoTip.js

Definujte a zaregistrujte komponentu ikony s radami pro internacionalizaci

## default(i18n_url, tag, key)

Zaregistrujte si vlastní komponentu HTML elementu pomocí bublin s výzvou k internacionalizaci

parametr:
- `i18n_url` : Žádost o adresu pro konfiguraci internacionalizace
- `tag` : Přizpůsobte název značky prvku HTML
- `key` : Klíčová hodnota textu výzvy ve slovníku internacionalizace
