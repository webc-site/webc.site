# i18n.js

Vícejazyčná správa stavu a sledování změn

## LANG_LI

Seznam dostupných jazykových konfigurací

## lang()

Získejte aktuální ID jazyka

Návratová hodnota: aktuální ID jazyka

## langCode()

Získejte aktuální kód jazyka

Návratová hodnota: aktuální kód jazyka, například `"zh"`

## onLang(func)

Svažte zpětný poslech pro změny jazyka

parametr:
- `func` : funkce zpětného volání při změně jazyka
  - parametr
    - `code` : Kód jazyka
    - `id` : ID jazyka

Návratová hodnota: Nevázaná funkce čištění

## langSet(id)

Přepnout a vysílat aktuální jazyk

parametr:
- `id` : ID cílového jazyka
