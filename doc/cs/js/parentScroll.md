# parentScroll.js

Získejte prvek kontejneru posouvání předka

## default(el)

Přejděte nahoru od zadaného prvku, najděte a vraťte prvek kontejneru posouvání, který jej obsahuje (jako je vestavěný kontejner posouvání `C-VS` ShadowRoot)

parametr:
- `el` : počáteční prvek DOM

Návratová hodnota: prvek DOM kontejneru posuvníku
