# icoTip.js

Definējiet un reģistrējiet ikonas komponentu ar internacionalizācijas padomiem

## default(i18n_url, tag, key)

Reģistrējiet pielāgotu HTML elementa komponentu ar internacionalizācijas uzvednes burbuļiem

parametrs:
- `i18n_url` : Pieprasīt adresi internacionalizācijas konfigurācijai
- `tag` : pielāgojiet HTML elementa taga nosaukumu
- `key` : internacionalizācijas vārdnīcas uzvednes teksta galvenā vērtība
