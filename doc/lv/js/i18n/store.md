# i18n/store.js

Daudzvalodu vietējās krātuves pārvaldība

## get()

Iegūstiet noklusējuma valodas ID, kas tiek aprēķināts lokāli kešatmiņā vai aprēķināts pēc pārlūkprogrammas valodas atbilstības

Atgriešanas vērtība: valodas ID

## set(id)

Saglabājiet pašreizējās valodas ID lokāli

parametrs:
- `id` : valodas ID
