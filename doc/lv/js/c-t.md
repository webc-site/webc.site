# c-t.js

HTML internacionalizācija

Tags `<c-t>key</c-t>` ir pielāgots, un `key` ir valodas pakotnes atslēga.

## cTranSet(el, genUrl)

Iestatiet `c-t` tekstu.

parametrs:
- `el` : saknes elements, globālo var nodot `document`
- `genUrl` : funkcija valodu pakotnes vārdnīcas iegūšanai
  - parametrs
    - `code` : valodas kods
  - Atgriešanas vērtība: valodas pakotnes vārdnīcas objekts vai solījums, kas atrisina šo objektu
