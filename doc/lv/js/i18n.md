# i18n.js

Daudzvalodu statusa pārvaldība un izmaiņu uzraudzība

## LANG_LI

Pieejamo valodu konfigurāciju saraksts

## lang()

Iegūstiet pašreizējo valodas ID

Atgriešanas vērtība: pašreizējā valodas ID

## langCode()

Iegūstiet pašreizējo valodas kodu

Atgrieztā vērtība: pašreizējais valodas kods, piemēram, `"zh"`

## onLang(func)

Saistīt klausīšanās atzvanu, lai mainītu valodu

parametrs:
- `func` : atzvanīšanas funkcija, kad mainās valoda
  - parametrs
    - `code` : valodas kods
    - `id` : valodas ID

Atgriešanas vērtība: nesaistīta tīrīšanas funkcija

## langSet(id)

Pārslēdziet un pārraidiet pašreizējo valodu

parametrs:
- `id` : mērķa valodas ID
