# parentScroll.js

Iegūstiet priekšteča ritināšanas konteinera elementu

## default(el)

Brauciet uz augšu no norādītā elementa, atrodiet un atgrieziet ritināšanas konteinera elementu, kurā tas ir (piemēram, `C-VS`s ShadowRoot iebūvētais ritināšanas konteiners)

parametrs:
- `el` : sākas DOM elements

Atgrieztā vērtība: ritināšanas konteinera DOM elements
