# fetch.js

ल्याइएको encapsulation मा आधारित नेटवर्क अनुरोध उपकरण प्रकार्य

## F(url, opt)

नेटवर्क अनुरोध पठाउनुहोस् र प्रतिक्रिया वस्तु फिर्ता गर्नुहोस्

प्यारामिटर:
- `url` : ठेगाना अनुरोध गर्नुहोस्
- `opt` : `fetch` विकल्प वस्तु

रिटर्न मान: प्रतिक्रिया वस्तु

## f(attr)

उच्च-अर्डर प्रकार्य जसले अनुरोधलाई बेर्छ र फिर्ता गरिएको सामग्रीलाई निर्दिष्ट ढाँचामा पार्स गर्दछ

प्यारामिटर:
- `attr` : प्रतिक्रिया शरीर पार्सिङ विशेषता नाम

रिटर्न मान: एसिन्क्रोनस अनुरोध र सामग्री पार्स गर्न कार्य
  - प्यारामिटर
    - `url` : ठेगाना अनुरोध गर्नुहोस्
    - `opt` : `fetch` विकल्प वस्तु
  - रिटर्न मान: पार्स गरिएको सामग्री

## fTxt

नेटवर्क अनुरोध पठाउनुहोस् र पाठ ढाँचामा प्रतिक्रिया शरीर फर्काउनुहोस्

प्यारामिटर:
- `url` : ठेगाना अनुरोध गर्नुहोस्
- `opt` : `fetch` विकल्प वस्तु

फिर्ता मूल्य: प्रतिक्रिया पाठ

## fJson(url, opt)

नेटवर्क अनुरोध पठाउनुहोस् र JSON ढाँचामा पार्स गर्नुहोस् र प्रतिक्रिया मुख्य भाग फर्काउनुहोस्

प्यारामिटर:
- `url` : ठेगाना अनुरोध गर्नुहोस्
- `opt` : `fetch` विकल्प वस्तु

फिर्ता मान: पार्स गरिएको JSON वस्तु

## fBin(url, opt)

नेटवर्क अनुरोध पठाउनुहोस् र बाइनरी बाइट एरेको रूपमा प्रतिक्रिया मुख्य भाग फर्काउनुहोस्

प्यारामिटर:
- `url` : ठेगाना अनुरोध गर्नुहोस्
- `opt` : `fetch` विकल्प वस्तु

रिटर्न मान: बाइट एरे
