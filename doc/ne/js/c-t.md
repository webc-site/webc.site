# c-t.js

HTML अन्तर्राष्ट्रियकरण

ट्याग `<c-t>key</c-t>` अनुकूलित छ, र `key` भाषा प्याकको कुञ्जी हो।

## cTranSet(el, genUrl)

`c-t` को लागि पाठ सेट गर्नुहोस्।

प्यारामिटर:
- `el` : मूल तत्व, ग्लोबललाई `document` मा पास गर्न सकिन्छ
- `genUrl` : भाषा प्याक शब्दकोश प्राप्त गर्न कार्य
  - प्यारामिटर
    - `code` : भाषा कोड
  - रिटर्न मान: भाषा प्याक डिक्शनरी वस्तु, वा प्रतिज्ञा जसले यो वस्तुलाई समाधान गर्छ
