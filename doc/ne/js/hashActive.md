# hashActive.js

URL एङ्कर र भ्युपोर्ट स्क्रोलिङमा आधारित नेभिगेसन लिङ्क हाइलाइटिङ

## default(link_li, cls)

पृष्ठ स्क्रोल स्थिति र URL एंकर बिन्दु परिवर्तनहरूमा आधारित हाल देखिने क्षेत्रसँग सम्बन्धित नेभिगेसन लिङ्कहरू स्वचालित रूपमा हाइलाइट गर्नुहोस्।

प्यारामिटर:
- `link_li` : नेभिगेसन लिङ्क DOM तत्व सूची
- `cls` : सक्रिय CSS वर्ग नाम, पूर्वनिर्धारित `"A"`

फिर्ता मूल्य: सबै सुन्ने सफाई कार्यहरू रद्द गर्नुहोस्
