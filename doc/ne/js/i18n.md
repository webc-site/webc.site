# i18n.js

बहु-भाषा स्थिति व्यवस्थापन र परिवर्तन अनुगमन

## LANG_LI

उपलब्ध भाषा कन्फिगरेसनहरूको सूची

## lang()

हालको भाषा ID प्राप्त गर्नुहोस्

फिर्ता मान: हालको भाषा ID

## langCode()

हालको भाषा कोड प्राप्त गर्नुहोस्

फिर्ता मान: हालको भाषा कोड, जस्तै `"zh"`

## onLang(func)

भाषा परिवर्तनहरूको लागि सुन्ने कलब्याक बाँध्नुहोस्

प्यारामिटर:
- `func` : भाषा परिवर्तन हुँदा कलब्याक प्रकार्य
  - प्यारामिटर
    - `code` : भाषा कोड
    - `id` : भाषा ID

रिटर्न मान: अनबाउन्ड क्लीनअप प्रकार्य

## langSet(id)

हालको भाषा स्विच र प्रसारण गर्नुहोस्

प्यारामिटर:
- `id` : लक्षित भाषा ID
