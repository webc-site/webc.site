# icoTip.js

अन्तर्राष्ट्रियकरण संकेतहरूको साथ आइकन घटक परिभाषित र दर्ता गर्नुहोस्

## default(i18n_url, tag, key)

अन्तर्राष्ट्रियकरण प्रम्प्ट बबलहरूसँग आफू अनुकूल HTML तत्व घटक दर्ता गर्नुहोस्

प्यारामिटर:
- `i18n_url` : अन्तर्राष्ट्रियकरण कन्फिगरेसनको लागि अनुरोध ठेगाना
- `tag` : HTML तत्वको ट्याग नाम अनुकूलन गर्नुहोस्
- `key` : अन्तर्राष्ट्रियकरण शब्दकोशमा प्रम्प्ट पाठको प्रमुख मान
