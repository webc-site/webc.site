# Toast.js

टोस्ट पप-अप सन्देश प्रम्प्ट

## toast(render, timeout)

मानक प्रम्प्ट पप-अप विन्डो देखाउनुहोस्

प्यारामिटर:
- `render` : रेन्डरिङ प्रकार्य, पप-अप सामग्री अनुकूलन गर्न प्रयोग गरिन्छ
  - प्यारामिटर
    - `el` : पप-अप विन्डोको DOM तत्व
- `timeout` : स्वचालित बन्दको लागि सेकेन्ड ढिलाइ, पूर्वनिर्धारित 9, ० जब स्वचालित बन्द हुँदैन

फिर्ता मान: पप-अप विन्डोको DOM तत्व
