# route.js

Аднастаронкавае інтэрфейснае кіраванне маршрутызацыяй

## nowUrl()

Атрымаць бягучы URL-шлях з выдаленай касой рысай

Вяртанае значэнне: бягучы радок шляху

## route(hook)

Зарэгіструйце зваротны выклік праслухоўвання для змены маршруту. Пры рэгістрацыі ён будзе выкананы адзін раз з бягучым шляхам.

параметр:
- `hook` : Функцыя зваротнага выкліку маніторынгу маршруту
  - параметр
    - `url` : бягучы шлях
    - `pre` : апошні шлях

Вяртанае значэнне: незарэгістраваная функцыя ачысткі

## setPre(url)

Захавайце апошні запісаны URL

параметр:
- `url` : радок шляху

## preUrl()

Атрымаць URL апошняга запісу

Вяртанае значэнне: радок апошняга шляху

## refresh(url)

Прымусова запускаць усе зваротныя выклікі праслухоўвання маршруту па ўказаным шляху і папярэдне абнавіць

параметр:
- `url` : мэтавы шлях, бягучы шлях па змаўчанні

## removeSlash(url)

Выдаліць касыя рысы з загалоўкаў шляху

параметр:
- `url` : радок шляху

Вяртанае значэнне: шлях пасля выдалення пачатковай касой рысы

## setUrl(url)

Абнавіць статус URL адраснага радка браўзера

параметр:
- `url` : мэтавы шлях

Вяртанае значэнне: 1, калі шлях сапраўды змяняецца

## goto(url)

Абнавіце адрас браўзера і запусціце абнаўленне маршруту

параметр:
- `url` : мэтавы шлях
