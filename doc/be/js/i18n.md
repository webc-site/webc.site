# i18n.js

Шматмоўнае кіраванне статусам і маніторынг змен

## LANG_LI

Спіс даступных моўных канфігурацый

## lang()

Атрымаць ідэнтыфікатар актуальнай мовы

Вяртанае значэнне: ідэнтыфікатар бягучай мовы

## langCode()

Атрымаць бягучы код мовы

Значэнне, якое вяртаецца: бягучы код мовы, напрыклад `"zh"`

## onLang(func)

Прывязаць зваротны выклік праслухоўвання да змены мовы

параметр:
- `func` : функцыя зваротнага выкліку пры змене мовы
  - параметр
    - `code` : код мовы
    - `id` : Ідэнтыфікатар мовы

Вяртанае значэнне: функцыя ачысткі без прывязкі

## langSet(id)

Пераключэнне і трансляцыя бягучай мовы

параметр:
- `id` : Ідэнтыфікатар мэтавай мовы
