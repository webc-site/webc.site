# dom.js

Скарачэнні зменных дакументаў і функцыі хуткага доступу

## D

`document` Спасылка на аб'ект

## B

`document.body` Спасылка на аб'ект

## newEl

Стварэнне элементаў DOM

параметр:
- `tagName` : назва тэга элемента

Вяртанае значэнне: нядаўна створаны элемент DOM

## newLi(...tag_li)

Пакетна стварайце некалькі элементаў DOM

параметр:
- `tag_li` : спіс назваў тэгаў

Вяртанае значэнне:
- Масіў, які змяшчае ўсе нядаўна створаныя элементы DOM

## cE(name, cls)

Зарэгістраваць карыстальніцкія элементы вэб-кампанентаў

параметр:
- `name` : Імя карыстацкага элемента, прэфікс `c-` будзе дададзены аўтаматычна
- `cls` : клас карыстацкага элемента
