# c-t.js

Інтэрнацыяналізацыя HTML

Тэг `<c-t>key</c-t>` наладжваецца, а `key` з'яўляецца ключом моўнага пакета.

## cTranSet(el, genUrl)

Усталюйце тэкст для `c-t`.

параметр:
- `el` : каранёвы элемент, глабальны можа быць перададзены ў `document`
- `genUrl` : функцыя атрымання слоўніка моўнага пакета
  - параметр
    - `code` : код мовы
  - Вяртанае значэнне: аб'ект слоўніка моўнага пакета або Promise, які вырашае гэты аб'ект
