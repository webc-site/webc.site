# i18n/store.js

Шматмоўнае кіраванне лакальным сховішчам

## get()

Атрымайце ідэнтыфікатар мовы па змаўчанні, які разлічваецца ў лакальным кэшы або вылічваецца шляхам супадзення мовы браўзера

Вяртанае значэнне: ідэнтыфікатар мовы

## set(id)

Кэшаваць ідэнтыфікатар бягучай мовы лакальна

параметр:
- `id` : Ідэнтыфікатар мовы
