# parentScroll.js

Атрымайце элемент кантэйнера пракруткі продка

## default(el)

Перайдзіце ўверх ад вызначанага элемента, знайдзіце і вярніце элемент кантэйнера пракруткі, які змяшчае яго (напрыклад, убудаваны кантэйнер пракруткі ShadowRoot `C-VS`)

параметр:
- `el` : пачатковы элемент DOM

Вяртанае значэнне: элемент DOM кантэйнера пракруткі
