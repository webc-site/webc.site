# Toast.js

Падказка ўсплывальнага паведамлення Toast

## toast(render, timeout)

Паказаць стандартнае ўсплывальнае акно падказкі

параметр:
- `render` : Функцыя візуалізацыі, якая выкарыстоўваецца для наладжвання ўсплываючага кантэнту
  - параметр
    - `el` : элемент DOM усплывальнага акна
- `timeout` : секунды затрымкі для аўтаматычнага выключэння, па змаўчанні 9, без аўтаматычнага выключэння, калі 0

Вяртанае значэнне: элемент DOM усплывальнага акна

## toastErr(render, timeout)

Паказаць усплывальнае акно паведамлення пра памылку

параметр:
- `render` : Функцыя візуалізацыі, якая выкарыстоўваецца для наладжвання ўсплываючага кантэнту
  - параметр
    - `el` : элемент DOM усплывальнага акна
- `timeout` : секунды затрымкі для аўтаматычнага выключэння, па змаўчанні 9, без аўтаматычнага выключэння, калі 0

Вяртанае значэнне: элемент DOM усплывальнага акна з памылкай
