# icoTip.js

Вызначце і зарэгіструйце кампанент значка з падказкамі па інтэрнацыяналізацыі

## default(i18n_url, tag, key)

Зарэгіструйце карыстальніцкі кампанент элемента HTML з бурбалкамі з інтэрнацыяналізацыяй

параметр:
- `i18n_url` : Адрас запыту для канфігурацыі інтэрнацыяналізацыі
- `tag` : Наладзьце імя тэга элемента HTML
- `key` : Ключавое значэнне тэксту падказкі ў інтэрнацыяналізацыйным слоўніку
