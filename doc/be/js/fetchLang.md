# fetchLang.js

Асінхронна загружаць моўныя пакеты і прывязваць зваротныя выклікі абнаўлення

## fetchLang(getDict)

Стварыце слухач моўнага пакета для асінхроннага атрымання моўнага пакета і выканання зваротнага выкліку абнаўлення пры змене мовы

параметр:
- `getDict` : атрымаць асінхронную функцыю моўнага пакета
  - параметр
    - `code` : код мовы
  - Вяртанае значэнне: слоўнік моўнага пакета

Вяртанае значэнне: функцыя для рэгістрацыі зваротнага выкліку абнаўлення
- параметр
  - `refresh` : Абнавіць функцыю зваротнага выкліку пры абнаўленні мовы
    - параметр
      - `i18n` : аб'ект слоўніка моўнага пакета
      - `code` : бягучы код мовы
- Вяртанае значэнне: функцыя для адмены рэгістрацыі зваротнага выкліку абнаўлення
