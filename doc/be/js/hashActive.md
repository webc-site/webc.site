# hashActive.js

Вылучэнне навігацыйнай спасылкі на аснове прывязкі URL і пракруткі акна прагляду

## default(link_li, cls)

Аўтаматычна выдзяляць навігацыйныя спасылкі, якія адпавядаюць бачнай у дадзены момант вобласці, у залежнасці ад пазіцыі пракруткі старонкі і змяненняў кропкі прывязкі URL.

параметр:
- `link_li` : Спіс элементаў DOM навігацыйнай спасылкі
- `cls` : Імя актыўнага класа CSS, па змаўчанні `"A"`

Вяртанае значэнне: адмяніць усе функцыі ачысткі праслухоўвання
