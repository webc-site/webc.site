# md/streamRender.js

Струменевы рэндэрынг Markdown

## default(container, source)

Патокавы рэндэрынг тэксту Markdown у паказаны кантэйнер DOM, падтрымка эфектаў курсора машынкі і аўтаматычная пракрутка

параметр:
- `container` : Элемент кантэйнера DOM, які размяшчае адлюстраваны кантэнт
- `source` : Крыніца тэкставых даных ReadableStream

Вяртанае значэнне:

- `cancel` : Функцыя спынення візуалізацыі і чытання патоку даных
- `setStop` : Функцыя для ўстанаўлення прыпынення аўтаматычнай пракруткі
  - параметр
    - `val` : лагічнае значэнне незалежна ад таго, рабіць паўзу ці не
