# c-t.js

HTML ინტერნაციონალიზაცია

ტეგი `<c-t>key</c-t>` მორგებულია და `key` არის ენის პაკეტის გასაღები.

## cTranSet(el, genUrl)

დააყენეთ ტექსტი `c-t`-ისთვის.

პარამეტრი:
- `el` : root ელემენტი, გლობალური შეიძლება გადავიდეს `document`-ში
- `genUrl` : ფუნქცია ენის პაკეტის ლექსიკონის მისაღებად
  - პარამეტრი
    - `code` : ენის კოდი
  - დაბრუნების მნიშვნელობა: ენის პაკეტის ლექსიკონის ობიექტი, ან დაპირება, რომელიც გადაწყვეტს ამ ობიექტს
