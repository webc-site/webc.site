# Toast.js

სადღეგრძელო ამომხტარი შეტყობინების მოთხოვნა

## toast(render, timeout)

სტანდარტული მოთხოვნის ამომხტარი ფანჯრის ჩვენება

პარამეტრი:
- `render` : რენდერის ფუნქცია, რომელიც გამოიყენება ამომხტარი კონტენტის მოსარგებად
  - პარამეტრი
    - `el` : ამომხტარი ფანჯრის DOM ელემენტი
- `timeout` : ავტომატური გამორთვის წამების დაყოვნება, ნაგულისხმევი 9, ავტომატური გათიშვის გარეშე, როდესაც 0

დაბრუნების მნიშვნელობა: ამომხტარი ფანჯრის DOM ელემენტი

პარამეტრი:
- `render` : რენდერის ფუნქცია, რომელიც გამოიყენება ამომხტარი კონტენტის მოსარგებად
  - პარამეტრი
    - `el` : ამომხტარი ფანჯრის DOM ელემენტი
- `timeout` : ავტომატური გამორთვის წამების დაყოვნება, ნაგულისხმევი 9, ავტომატური გათიშვის გარეშე, როდესაც 0

დაბრუნების მნიშვნელობა: შეცდომის ამომხტარი ფანჯრის DOM ელემენტი
