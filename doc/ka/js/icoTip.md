## default(i18n_url, tag, key)

დაარეგისტრირეთ მორგებული HTML ელემენტის კომპონენტი ინტერნაციონალიზაციის მოთხოვნის ბუშტებით

პარამეტრი:
- `i18n_url` : მოითხოვეთ მისამართი ინტერნაციონალიზაციის კონფიგურაციისთვის
- `tag` : HTML ელემენტის ტეგის სახელის მორგება
- `key` : მოთხოვნის ტექსტის ძირითადი მნიშვნელობა ინტერნაციონალიზაციის ლექსიკონში
