# fetch.js

ქსელის მოთხოვნის ხელსაწყოს ფუნქცია Fetch encapsulation-ზე დაფუძნებული

## F(url, opt)

გაგზავნეთ ქსელის მოთხოვნა და დააბრუნეთ პასუხის ობიექტი

პარამეტრი:
- `url` : მოითხოვეთ მისამართი
- `opt` : `fetch` პარამეტრების ობიექტი

დაბრუნების მნიშვნელობა: საპასუხო ობიექტი

## f(attr)

უფრო მაღალი რიგის ფუნქცია, რომელიც ახვევს მოთხოვნას და აანალიზებს დაბრუნებულ შინაარსს მითითებულ ფორმატში

პარამეტრი:
- `attr` : საპასუხო სხეულის ანალიზების ატრიბუტის სახელი

დაბრუნების მნიშვნელობა: ფუნქცია ასინქრონულად მოითხოვოს და გააანალიზოს შინაარსი
  - პარამეტრი
    - `url` : მოითხოვეთ მისამართი
    - `opt` : `fetch` პარამეტრების ობიექტი
  - დაბრუნების მნიშვნელობა: გაანალიზებული შინაარსი

პარამეტრი:

დაბრუნების მნიშვნელობა: პასუხის ტექსტი

გაგზავნეთ ქსელის მოთხოვნა და გააანალიზეთ და დააბრუნეთ პასუხის სხეული JSON ფორმატში

პარამეტრი:
- `url` : მოითხოვეთ მისამართი
- `opt` : `fetch` პარამეტრების ობიექტი

დაბრუნების მნიშვნელობა: გაანალიზებული JSON ობიექტი

პარამეტრი:

- `opt` : `fetch` პარამეტრების ობიექტი

დაბრუნების მნიშვნელობა: ბაიტის მასივი
