# i18n.js

მრავალენოვანი სტატუსის მართვა და ცვლილებების მონიტორინგი

## LANG_LI

ხელმისაწვდომი ენის კონფიგურაციების სია

## lang()

მიიღეთ მიმდინარე ენის ID

დაბრუნების მნიშვნელობა: მიმდინარე ენის ID

## langCode()

მიიღეთ მიმდინარე ენის კოდი

დაბრუნების მნიშვნელობა: მიმდინარე ენის კოდი, როგორიცაა `"zh"`

## onLang(func)

დააკავშირეთ მოსმენის გამოძახება ენის ცვლილებებისთვის

პარამეტრი:
- `func` : გამოძახების ფუნქცია, როდესაც ენა იცვლება
  - პარამეტრი
    - `code` : ენის კოდი
    - `id` : ენის ID

დაბრუნების მნიშვნელობა: შეუზღუდავი გასუფთავების ფუნქცია

## langSet(id)

მიმდინარე ენის გადართვა და გადაცემა

პარამეტრი:
- `id` : სამიზნე ენის ID
