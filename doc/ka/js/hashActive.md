# hashActive.js

ნავიგაციის ბმულის ხაზგასმა URL-ის წამყვანებზე და ხედის გადახვევაზე დაყრდნობით

## default(link_li, cls)

ავტომატურად მონიშნეთ ნავიგაციის ბმულები, რომლებიც შეესაბამება ამჟამად ხილულ არეალს, გვერდის გადახვევის პოზიციისა და URL-ის წამყვანი წერტილის ცვლილებების საფუძველზე.

პარამეტრი:
- `link_li` : ნავიგაციის ბმული DOM ელემენტების სია
- `cls` : აქტიური CSS კლასის სახელი, ნაგულისხმევი `"A"`

დაბრუნების მნიშვნელობა: გააუქმეთ მოსმენის გასუფთავების ყველა ფუნქცია
