# md/streamRender.js

Markdown სტრიმინგის რენდერი

## default(container, source)

Markdown ტექსტის სტრიმინგი მითითებულ DOM კონტეინერში, საბეჭდი მანქანის კურსორის ეფექტების მხარდაჭერა და ავტომატური გადახვევა

პარამეტრი:
- `container` : DOM კონტეინერის ელემენტი, რომელიც მასპინძლობს რენდირებულ კონტენტს
- `source` : ReadableStream ტექსტის მონაცემთა წყარო

დაბრუნების ღირებულება:

- `cancel` : ფუნქცია შეაჩეროს რენდერი და მონაცემთა ნაკადის კითხვა
- `setStop` : ფუნქცია ავტომატური გადახვევის დაპაუზების დასაყენებლად
  - პარამეტრი
    - `val` : ლოგიკური მნიშვნელობა შეჩერდეს თუ არა
