# fetchLang.js

ასინქრონულად ჩატვირთეთ ენების პაკეტები და დააკავშირეთ განახლების გამოძახებები

## fetchLang(getDict)

შექმენით ენის პაკეტის მსმენელი, რათა ასინქრონულად მიიღოთ ენის პაკეტი და განახორციელოთ განახლების გამოძახება, როდესაც ენა იცვლება

პარამეტრი:
- `getDict` : მიიღეთ ენის პაკეტის ასინქრონული ფუნქცია
  - პარამეტრი
    - `code` : ენის კოდი
  - დასაბრუნებელი ღირებულება: ენის პაკეტის ლექსიკონი

დაბრუნების მნიშვნელობა: ფუნქცია განახლების დარეკვის რეგისტრაციისთვის
- პარამეტრი
  - `refresh` : განაახლეთ გამოძახების ფუნქცია, როდესაც ენა განახლდება
    - პარამეტრი
      - `i18n` : ენის პაკეტის ლექსიკონის ობიექტი
      - `code` : მიმდინარე ენის კოდი
- დაბრუნების მნიშვნელობა: ფუნქცია განახლების გამოძახების რეგისტრაციის გაუქმების მიზნით
