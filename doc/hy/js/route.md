# route.js

Մեկ էջի ճակատային երթուղիների կառավարում

## nowUrl()

Ստացեք ընթացիկ URL-ի ուղին՝ առաջատար շեղը հեռացնելով

Վերադարձի արժեքը՝ ընթացիկ ուղու տողը

## route(hook)

Գրանցեք երթուղու փոփոխությունների համար լսող հետադարձ զանգը: Գրանցվելիս այն կկատարվի մեկ անգամ ընթացիկ ճանապարհով։

պարամետր:
- `hook` : Երթուղու մոնիտորինգի հետադարձ զանգի գործառույթ
  - պարամետր
    - `url` : ընթացիկ ուղին
    - `pre` : վերջին ուղին

Վերադարձի արժեքը. չգրանցված մաքրման գործառույթ

## setPre(url)

Պահպանեք վերջին գրանցված URL-ը

պարամետր:
- `url` : ուղու տող

## preUrl()

Ստացեք վերջին գրանցման URL-ը

Վերադարձի արժեքը՝ վերջին ուղու տողը

## refresh(url)

Ստիպեք բոլոր երթուղիների հետադարձ զանգերը գործարկվեն նշված ուղու վրա և թարմացրեք նախապես

պարամետր:
- `url` : նպատակային ուղի, լռելյայն ընթացիկ ուղի

## removeSlash(url)

Հեռացրեք շեղերը ուղու վերնագրերից

պարամետր:
- `url` : ուղու տող

Վերադարձի արժեքը. ուղին առաջատար շեղը հեռացնելուց հետո

## setUrl(url)

Թարմացրեք դիտարկիչի հասցեագոտու URL կարգավիճակը

պարամետր:
- `url` : նպատակային ուղի

Վերադարձի արժեքը՝ 1, եթե ուղին իրականում փոխվում է

## goto(url)

Թարմացրեք դիտարկիչի հասցեն և գործարկեք երթուղու թարմացում

պարամետր:
- `url` : նպատակային ուղի
