# dom.js

Փաստաթղթերի փոփոխական հապավումներ և դյուրանցումների գործառույթներ

## D

`document` օբյեկտի դյուրանցման հղում

## B

`document.body` օբյեկտի դյուրանցման հղում

## newEl

Ստեղծեք DOM տարրեր

պարամետր:
- `tagName` : տարրի պիտակի անունը

Վերադարձի արժեքը՝ նոր ստեղծված DOM տարր

## newLi(...tag_li)

Ստեղծեք բազմաթիվ DOM տարրեր խմբաքանակներով

պարամետր:
- `tag_li` : պիտակների անունների ցանկ

Վերադարձի արժեքը.
- Զանգված, որը պարունակում է բոլոր նորաստեղծ DOM տարրերը

## cE(name, cls)

Գրանցեք Web Components մաքսային տարրերը

պարամետր:
- `name` . հատուկ տարրի անունը, `c-` նախածանցը կավելացվի ավտոմատ կերպով
- `cls`՝ հատուկ տարրի դասը
