# fetchLang.js

Ասինխրոն կերպով բեռնեք լեզվական փաթեթները և կապեք թարմացման հետադարձ զանգերը

## fetchLang(getDict)

Ստեղծեք լեզվական փաթեթի ունկնդիր, որպեսզի ասինխրոն կերպով ձեռք բերեք լեզվի փաթեթը և կատարեք թարմացում, երբ լեզուն փոխվի

պարամետր:
- `getDict`. Ստացեք լեզվական փաթեթի ասինխրոն գործառույթը
  - պարամետր
    - `code` : լեզվի կոդը
  - Վերադարձի արժեքը՝ լեզվի փաթեթի բառարան

Վերադարձի արժեք. ֆունկցիա՝ հետադարձ զանգի թարմացման գրանցման համար
- պարամետր
  - `refresh` . թարմացրեք հետադարձ կապի գործառույթը, երբ լեզուն թարմացվում է
    - պարամետր
      - `i18n` : Լեզուների փաթեթի բառարանի օբյեկտ
      - `code` : Ընթացիկ լեզվի կոդը
- Վերադարձի արժեք. ֆունկցիա՝ հետադարձ զանգը թարմացնելու համար
