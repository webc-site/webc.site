# On.js

Խմբաքանակի պարտադիր իրադարձությունների մոնիտորինգ

## On(elem, dict)

Կցեք DOM տարրերի իրադարձությունների ունկնդիրներին խմբաքանակով

պարամետր:
- `elem` : DOM տարր
- `dict`. իրադարձությունների և հետադարձ զանգերի քարտեզագրում

Վերադարձի արժեքը. Հեռացրեք բոլոր կապված իրադարձությունների լսողի գործառույթները
