# c-t.js

HTML միջազգայնացում

`<c-t>key</c-t>` պիտակը հարմարեցված է, և `key` լեզվական փաթեթի բանալին է:

## cTranSet(el, genUrl)

Սահմանեք տեքստը `c-t`-ի համար:

պարամետր:
- `el` : արմատային տարրը, գլոբալը կարող է փոխանցվել `document`-ում
- `genUrl` : ֆունկցիա լեզվական փաթեթի բառարան ստանալու համար
  - պարամետր
    - `code` : լեզվի կոդը
  - Վերադարձի արժեք. լեզվական փաթեթ բառարանի օբյեկտ կամ խոստում, որը լուծում է այս օբյեկտին
