# hashActive.js

Նավիգացիոն հղման ընդգծումը՝ հիմնված URL խարիսխների և տեսադաշտի ոլորման վրա

## default(link_li, cls)

Էջի ոլորման դիրքի և URL-ի խարիսխի կետի փոփոխությունների հիման վրա ավտոմատ կերպով ընդգծեք ներկայումս տեսանելի տարածքին համապատասխանող նավիգացիոն հղումները:

պարամետր:
- `link_li` . Նավիգացիոն հղման DOM տարրերի ցանկ
- `cls`: Ակտիվ CSS դասի անունը, լռելյայն `"A"`

Վերադարձի արժեքը. չեղարկել լսողության մաքրման բոլոր գործառույթները
