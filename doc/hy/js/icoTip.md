# icoTip.js

Սահմանեք և գրանցեք պատկերակի բաղադրիչ միջազգայնացման հուշումներով

## default(i18n_url, tag, key)

Գրանցեք հատուկ HTML տարրի բաղադրիչ միջազգայնացման հուշման փուչիկներով

պարամետր:
- `i18n_url` : Միջազգայնացման կոնֆիգուրացիայի համար հասցե պահանջեք
- `tag` . Անհատականացրեք HTML տարրի պիտակի անունը
- `key` . Միջազգայինացման բառարանում հուշման տեքստի հիմնական արժեքը
