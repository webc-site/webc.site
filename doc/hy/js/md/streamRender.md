# md/streamRender.js

Markdown հոսքային մատուցում

## default(container, source)

Markdown տեքստի ցուցադրում նշված DOM կոնտեյներով, գրամեքենայի կուրսորի էֆեկտների աջակցում և ավտոմատ ոլորում

պարամետր:
- `container` . DOM կոնտեյների տարր, որը հոսթինգ է մատուցված բովանդակություն
- `source` : ReadableStream տեքստային տվյալների աղբյուր

Վերադարձի արժեքը.

- `cancel`. ֆունկցիան դադարեցնելու ցուցադրումը և տվյալների հոսքի ընթերցումը
- `setStop` . ֆունկցիա՝ ավտոմատ ոլորումը դադարեցնելու համար
  - պարամետր
    - `val` . Բուլյան արժեք՝ դադարեցնել, թե ոչ
