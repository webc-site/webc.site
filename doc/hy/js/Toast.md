# Toast.js

Տոստ թռուցիկ հաղորդագրության հուշում

## toast(render, timeout)

Ցույց տալ ստանդարտ հուշման բացվող պատուհանը

պարամետր:
- `render` : Արտադրման ֆունկցիա, որն օգտագործվում է ելնող բովանդակությունը հարմարեցնելու համար
  - պարամետր
    - `el` : ելնող պատուհանի DOM տարր
- `timeout` . Ավտոմատ անջատման վայրկյաններ ուշացում, լռելյայն 9, ավտոմատ անջատում չկա, երբ 0

Վերադարձի արժեքը՝ բացվող պատուհանի DOM տարր
