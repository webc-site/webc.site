# fetch.js

Ցանցի հարցման գործիքի գործառույթը, որը հիմնված է բեռնման ինկապսուլյացիայի վրա

## F(url, opt)

Ուղարկեք ցանցի հարցում և վերադարձրեք պատասխանի օբյեկտ

պարամետր:
- `url` : Հարցրեք հասցե
- `opt` : `fetch` ընտրանքների օբյեկտ

Վերադարձի արժեքը՝ պատասխանի օբյեկտ

## f(attr)

Ավելի բարձր կարգի ֆունկցիա, որը փաթեթավորում է հարցումը և վերլուծում վերադարձված բովանդակությունը նշված ձևաչափով

պարամետր:
- `attr` : պատասխանի մարմնի վերլուծող հատկանիշի անունը

Վերադարձի արժեք. գործառույթ՝ բովանդակությունը ասինխրոն կերպով պահանջելու և վերլուծելու համար
  - պարամետր
    - `url` : Հարցրեք հասցե
    - `opt` : `fetch` ընտրանքների օբյեկտ
  - Վերադարձի արժեքը՝ վերլուծված բովանդակություն

պարամետր:

Վերադարձի արժեքը՝ պատասխանի տեքստ

## fJson(url, opt)

Ուղարկեք ցանցի հարցում և վերլուծեք և վերադարձրեք պատասխանի մարմինը JSON ձևաչափով

պարամետր:
- `url` : Հարցրեք հասցե
- `opt` : `fetch` ընտրանքների օբյեկտ

Վերադարձի արժեքը՝ վերլուծված JSON օբյեկտ
