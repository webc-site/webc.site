# i18n.js

Բազմալեզու կարգավիճակի կառավարում և փոփոխությունների մոնիտորինգ

## LANG_LI

Հասանելի լեզուների կոնֆիգուրացիաների ցանկ

## lang()

Ստացեք ընթացիկ լեզվի ID

Վերադարձի արժեքը՝ ընթացիկ լեզվի ID

## langCode()

Ստացեք ընթացիկ լեզվի կոդը

Վերադարձի արժեքը. ընթացիկ լեզվի կոդը, օրինակ՝ `"zh"`

## onLang(func)

Լեզվի փոփոխությունների համար կապակցեք լսող հետադարձ զանգը

պարամետր:
- `func` : հետ կանչելու ֆունկցիա, երբ լեզուն փոխվում է
  - պարամետր
    - `code` : Լեզվի կոդը
    - `id` : Լեզվի ID

Վերադարձի արժեք՝ չսահմանափակված մաքրման ֆունկցիա

## langSet(id)

Փոխեք և հեռարձակեք ընթացիկ լեզուն

պարամետր:
- `id` : Թիրախային լեզվի ID
