# parentScroll.js

Ստացեք նախնիների ոլորման կոնտեյների տարրը

## default(el)

Անցեք նշված տարրից դեպի վեր, գտեք և վերադարձրեք այն պարունակող ոլորման տարրը (օրինակ՝ `C-VS`-ի ShadowRoot ներկառուցված ոլորման կոնտեյները)

պարամետր:
- `el` : մեկնարկային DOM տարր

Վերադարձի արժեքը՝ ոլորման կոնտեյների DOM տարր
