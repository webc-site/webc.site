# i18n.js

ಬಹು-ಭಾಷಾ ಸ್ಥಿತಿ ನಿರ್ವಹಣೆ ಮತ್ತು ಬದಲಾವಣೆ ಮೇಲ್ವಿಚಾರಣೆ

## LANG_LI

ಲಭ್ಯವಿರುವ ಭಾಷಾ ಸಂರಚನೆಗಳ ಪಟ್ಟಿ

## lang()

ಪ್ರಸ್ತುತ ಭಾಷೆಯ ID ಪಡೆಯಿರಿ

ಹಿಂತಿರುಗಿಸುವ ಮೌಲ್ಯ: ಪ್ರಸ್ತುತ ಭಾಷೆಯ ID

## langCode()

ಪ್ರಸ್ತುತ ಭಾಷೆಯ ಕೋಡ್ ಪಡೆಯಿರಿ

ಹಿಂತಿರುಗಿಸುವ ಮೌಲ್ಯ: ಪ್ರಸ್ತುತ ಭಾಷೆಯ ಕೋಡ್, ಉದಾಹರಣೆಗೆ `"zh"`

## onLang(func)

ಭಾಷೆಯ ಬದಲಾವಣೆಗಳಿಗಾಗಿ ಕೇಳುವ ಕಾಲ್‌ಬ್ಯಾಕ್ ಅನ್ನು ಬಂಧಿಸಿ

ನಿಯತಾಂಕ:
- `func` : ಭಾಷೆ ಬದಲಾದಾಗ ಕಾಲ್‌ಬ್ಯಾಕ್ ಕಾರ್ಯ
  - ನಿಯತಾಂಕ
    - `code` : ಭಾಷಾ ಕೋಡ್
    - `id` : ಭಾಷಾ ID

ರಿಟರ್ನ್ ಮೌಲ್ಯ: ಅನ್‌ಬೌಂಡ್ ಕ್ಲೀನಪ್ ಕಾರ್ಯ

## langSet(id)

ಪ್ರಸ್ತುತ ಭಾಷೆಯನ್ನು ಬದಲಾಯಿಸಿ ಮತ್ತು ಪ್ರಸಾರ ಮಾಡಿ

ನಿಯತಾಂಕ:
- `id` : ಗುರಿ ಭಾಷೆ ID
