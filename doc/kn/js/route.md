# route.js

ಏಕ ಪುಟದ ಮುಂಭಾಗದ ರೂಟಿಂಗ್ ನಿರ್ವಹಣೆ

## nowUrl()

ಮುಂಚೂಣಿಯಲ್ಲಿರುವ ಸ್ಲ್ಯಾಷ್ ಅನ್ನು ತೆಗೆದುಹಾಕುವುದರೊಂದಿಗೆ ಪ್ರಸ್ತುತ URL ಮಾರ್ಗವನ್ನು ಪಡೆಯಿರಿ

ಹಿಂತಿರುಗಿಸುವ ಮೌಲ್ಯ: ಪ್ರಸ್ತುತ ಮಾರ್ಗ ಸ್ಟ್ರಿಂಗ್

## route(hook)

ಮಾರ್ಗ ಬದಲಾವಣೆಗಳಿಗಾಗಿ ಕೇಳುವ ಕಾಲ್ಬ್ಯಾಕ್ ಅನ್ನು ನೋಂದಾಯಿಸಿ. ನೋಂದಾಯಿಸುವಾಗ, ಅದನ್ನು ಪ್ರಸ್ತುತ ಮಾರ್ಗದೊಂದಿಗೆ ಒಮ್ಮೆ ಕಾರ್ಯಗತಗೊಳಿಸಲಾಗುತ್ತದೆ.

ನಿಯತಾಂಕ:
- `hook` : ರೂಟ್ ಮಾನಿಟರಿಂಗ್ ಕಾಲ್‌ಬ್ಯಾಕ್ ಕಾರ್ಯ
  - ನಿಯತಾಂಕ
    - `url` : ಪ್ರಸ್ತುತ ಮಾರ್ಗ
    - `pre` : ಕೊನೆಯ ಮಾರ್ಗ

ರಿಟರ್ನ್ ಮೌಲ್ಯ: ನೋಂದಾಯಿಸದ ಕ್ಲೀನಪ್ ಕಾರ್ಯ

## setPre(url)

ಕೊನೆಯದಾಗಿ ರೆಕಾರ್ಡ್ ಮಾಡಿದ URL ಅನ್ನು ಉಳಿಸಿ

ನಿಯತಾಂಕ:
- `url` : ಪಥ ಸ್ಟ್ರಿಂಗ್

## preUrl()

ಕೊನೆಯ ದಾಖಲೆಯ URL ಅನ್ನು ಪಡೆಯಿರಿ

ಹಿಂತಿರುಗಿಸುವ ಮೌಲ್ಯ: ಕೊನೆಯ ಮಾರ್ಗದ ಸ್ಟ್ರಿಂಗ್

## refresh(url)

ಎಲ್ಲಾ ಮಾರ್ಗ ಕೇಳುವ ಕಾಲ್‌ಬ್ಯಾಕ್‌ಗಳನ್ನು ನಿರ್ದಿಷ್ಟಪಡಿಸಿದ ಮಾರ್ಗದಲ್ಲಿ ಪ್ರಚೋದಿಸಲು ಒತ್ತಾಯಿಸಿ ಮತ್ತು ಪೂರ್ವವನ್ನು ನವೀಕರಿಸಿ

ನಿಯತಾಂಕ:
- `url` : ಗುರಿ ಮಾರ್ಗ, ಡೀಫಾಲ್ಟ್ ಪ್ರಸ್ತುತ ಮಾರ್ಗ

## removeSlash(url)

ಮಾರ್ಗದ ಹೆಡರ್‌ಗಳಿಂದ ಸ್ಲ್ಯಾಷ್‌ಗಳನ್ನು ತೆಗೆದುಹಾಕಿ

ನಿಯತಾಂಕ:
- `url` : ಪಥ ಸ್ಟ್ರಿಂಗ್

ಮೌಲ್ಯವನ್ನು ಹಿಂತಿರುಗಿಸಿ: ಪ್ರಮುಖ ಸ್ಲ್ಯಾಷ್ ಅನ್ನು ತೆಗೆದುಹಾಕಿದ ನಂತರ ಮಾರ್ಗ

## setUrl(url)

ಬ್ರೌಸರ್ ವಿಳಾಸ ಪಟ್ಟಿಯ URL ಸ್ಥಿತಿಯನ್ನು ನವೀಕರಿಸಿ

ನಿಯತಾಂಕ:
- `url` : ಗುರಿ ಮಾರ್ಗ

ಹಿಂತಿರುಗಿಸುವ ಮೌಲ್ಯ: 1 ಮಾರ್ಗವು ನಿಜವಾಗಿ ಬದಲಾದರೆ

## goto(url)

ಬ್ರೌಸರ್ ವಿಳಾಸವನ್ನು ನವೀಕರಿಸಿ ಮತ್ತು ಮಾರ್ಗವನ್ನು ರಿಫ್ರೆಶ್ ಮಾಡಿ

ನಿಯತಾಂಕ:
- `url` : ಗುರಿ ಮಾರ್ಗ
