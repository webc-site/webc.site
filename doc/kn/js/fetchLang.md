# fetchLang.js

ಭಾಷಾ ಪ್ಯಾಕ್‌ಗಳನ್ನು ಅಸಮಕಾಲಿಕವಾಗಿ ಲೋಡ್ ಮಾಡಿ ಮತ್ತು ಅಪ್‌ಡೇಟ್ ಕಾಲ್‌ಬ್ಯಾಕ್‌ಗಳನ್ನು ಬಂಧಿಸಿ

## fetchLang(getDict)

ಭಾಷಾ ಪ್ಯಾಕ್ ಅನ್ನು ಅಸಮಕಾಲಿಕವಾಗಿ ಪಡೆಯಲು ಭಾಷಾ ಪ್ಯಾಕ್ ಕೇಳುಗರನ್ನು ರಚಿಸಿ ಮತ್ತು ಭಾಷೆ ಬದಲಾದಾಗ ರಿಫ್ರೆಶ್ ಕಾಲ್‌ಬ್ಯಾಕ್ ಅನ್ನು ಕಾರ್ಯಗತಗೊಳಿಸಿ

ನಿಯತಾಂಕ:
- `getDict` : ಭಾಷಾ ಪ್ಯಾಕೇಜ್‌ನ ಅಸಮಕಾಲಿಕ ಕಾರ್ಯವನ್ನು ಪಡೆಯಿರಿ
  - ನಿಯತಾಂಕ
    - `code` : ಭಾಷಾ ಕೋಡ್
  - ರಿಟರ್ನ್ ಮೌಲ್ಯ: ಭಾಷಾ ಪ್ಯಾಕ್ ನಿಘಂಟು

ರಿಟರ್ನ್ ಮೌಲ್ಯ: ರಿಫ್ರೆಶ್ ಕಾಲ್‌ಬ್ಯಾಕ್ ಅನ್ನು ನೋಂದಾಯಿಸುವ ಕಾರ್ಯ
- ನಿಯತಾಂಕ
  - `refresh` : ಭಾಷೆಯನ್ನು ನವೀಕರಿಸಿದಾಗ ಕಾಲ್‌ಬ್ಯಾಕ್ ಕಾರ್ಯವನ್ನು ರಿಫ್ರೆಶ್ ಮಾಡಿ
    - ನಿಯತಾಂಕ
      - `i18n` : ಭಾಷಾ ಪ್ಯಾಕ್ ನಿಘಂಟು ವಸ್ತು
      - `code` : ಪ್ರಸ್ತುತ ಭಾಷೆಯ ಕೋಡ್
- ರಿಟರ್ನ್ ಮೌಲ್ಯ: ರಿಫ್ರೆಶ್ ಕಾಲ್‌ಬ್ಯಾಕ್ ಅನ್ನು ನೋಂದಾಯಿಸದ ಕಾರ್ಯ
