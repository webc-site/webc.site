# fetch.js

ಎನ್‌ಕ್ಯಾಪ್ಸುಲೇಶನ್ ಅನ್ನು ಪಡೆದುಕೊಳ್ಳುವುದರ ಆಧಾರದ ಮೇಲೆ ನೆಟ್‌ವರ್ಕ್ ವಿನಂತಿ ಟೂಲ್ ಕಾರ್ಯ

## F(url, opt)

ನೆಟ್ವರ್ಕ್ ವಿನಂತಿಯನ್ನು ಕಳುಹಿಸಿ ಮತ್ತು ಪ್ರತಿಕ್ರಿಯೆ ವಸ್ತುವನ್ನು ಹಿಂತಿರುಗಿ

ನಿಯತಾಂಕ:
- `url` : ವಿಳಾಸವನ್ನು ವಿನಂತಿಸಿ
- `opt` : `fetch` ಆಯ್ಕೆಗಳ ವಸ್ತು

ರಿಟರ್ನ್ ಮೌಲ್ಯ: ಪ್ರತಿಕ್ರಿಯೆ ವಸ್ತು

## f(attr)

ವಿನಂತಿಯನ್ನು ಸುತ್ತುವ ಮತ್ತು ಹಿಂತಿರುಗಿದ ವಿಷಯವನ್ನು ನಿರ್ದಿಷ್ಟ ಸ್ವರೂಪದಲ್ಲಿ ಪಾರ್ಸ್ ಮಾಡುವ ಉನ್ನತ-ಕ್ರಮದ ಕಾರ್ಯ

ನಿಯತಾಂಕ:
- `attr` : ಪ್ರತಿಕ್ರಿಯೆ ದೇಹದ ಪಾರ್ಸಿಂಗ್ ಗುಣಲಕ್ಷಣದ ಹೆಸರು

ಮೌಲ್ಯವನ್ನು ಹಿಂತಿರುಗಿಸಿ: ಅಸಮಕಾಲಿಕವಾಗಿ ವಿನಂತಿಸಲು ಮತ್ತು ವಿಷಯವನ್ನು ಪಾರ್ಸ್ ಮಾಡಲು ಕಾರ್ಯ
  - ನಿಯತಾಂಕ
    - `url` : ವಿಳಾಸವನ್ನು ವಿನಂತಿಸಿ
    - `opt` : `fetch` ಆಯ್ಕೆಗಳ ವಸ್ತು
  - ರಿಟರ್ನ್ ಮೌಲ್ಯ: ಪಾರ್ಸ್ ಮಾಡಿದ ವಿಷಯ

## fTxt

ನೆಟ್‌ವರ್ಕ್ ವಿನಂತಿಯನ್ನು ಕಳುಹಿಸಿ ಮತ್ತು ಪ್ರತಿಕ್ರಿಯೆಯನ್ನು ಪಠ್ಯ ಸ್ವರೂಪದಲ್ಲಿ ಹಿಂತಿರುಗಿ

ನಿಯತಾಂಕ:
- `url` : ವಿಳಾಸವನ್ನು ವಿನಂತಿಸಿ
- `opt` : `fetch` ಆಯ್ಕೆಗಳ ವಸ್ತು

ರಿಟರ್ನ್ ಮೌಲ್ಯ: ಪ್ರತಿಕ್ರಿಯೆ ಪಠ್ಯ

## fJson(url, opt)

ನೆಟ್‌ವರ್ಕ್ ವಿನಂತಿಯನ್ನು ಕಳುಹಿಸಿ ಮತ್ತು ಪಾರ್ಸ್ ಮಾಡಿ ಮತ್ತು ಪ್ರತಿಕ್ರಿಯೆಯನ್ನು JSON ಫಾರ್ಮ್ಯಾಟ್‌ನಲ್ಲಿ ಹಿಂತಿರುಗಿಸಿ

ನಿಯತಾಂಕ:
- `url` : ವಿಳಾಸವನ್ನು ವಿನಂತಿಸಿ
- `opt` : `fetch` ಆಯ್ಕೆಗಳ ವಸ್ತು

ಮೌಲ್ಯವನ್ನು ಹಿಂತಿರುಗಿಸಿ: ಪಾರ್ಸ್ ಮಾಡಿದ JSON ವಸ್ತು

## fBin(url, opt)

ನೆಟ್‌ವರ್ಕ್ ವಿನಂತಿಯನ್ನು ಕಳುಹಿಸಿ ಮತ್ತು ಪ್ರತಿಕ್ರಿಯೆ ದೇಹವನ್ನು ಬೈನರಿ ಬೈಟ್ ಅರೇ ಆಗಿ ಹಿಂತಿರುಗಿಸಿ

ನಿಯತಾಂಕ:
- `url` : ವಿಳಾಸವನ್ನು ವಿನಂತಿಸಿ
- `opt` : `fetch` ಆಯ್ಕೆಗಳ ವಸ್ತು

ಹಿಂತಿರುಗಿಸುವ ಮೌಲ್ಯ: ಬೈಟ್ ಅರೇ
