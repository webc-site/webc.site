# icoTip.js

ಅಂತರಾಷ್ಟ್ರೀಕರಣದ ಸುಳಿವುಗಳೊಂದಿಗೆ ಐಕಾನ್ ಘಟಕವನ್ನು ವಿವರಿಸಿ ಮತ್ತು ನೋಂದಾಯಿಸಿ

## default(i18n_url, tag, key)

ಅಂತರರಾಷ್ಟ್ರೀಕರಣ ಪ್ರಾಂಪ್ಟ್ ಬಬಲ್‌ಗಳೊಂದಿಗೆ ಕಸ್ಟಮ್ HTML ಅಂಶ ಘಟಕವನ್ನು ನೋಂದಾಯಿಸಿ

ನಿಯತಾಂಕ:
- `i18n_url` : ಅಂತರರಾಷ್ಟ್ರೀಕರಣದ ಸಂರಚನೆಗಾಗಿ ವಿನಂತಿಯ ವಿಳಾಸ
- `tag` : HTML ಅಂಶದ ಟ್ಯಾಗ್ ಹೆಸರನ್ನು ಕಸ್ಟಮೈಸ್ ಮಾಡಿ
- `key` : ಅಂತರರಾಷ್ಟ್ರೀಕರಣ ನಿಘಂಟಿನಲ್ಲಿ ಪ್ರಾಂಪ್ಟ್ ಪಠ್ಯದ ಪ್ರಮುಖ ಮೌಲ್ಯ
