# c-t.js

HTML ಅಂತರಾಷ್ಟ್ರೀಯೀಕರಣ

`<c-t>key</c-t>` ಟ್ಯಾಗ್ ಅನ್ನು ಕಸ್ಟಮೈಸ್ ಮಾಡಲಾಗಿದೆ ಮತ್ತು `key` ಭಾಷೆ ಪ್ಯಾಕ್‌ನ ಕೀಲಿಯಾಗಿದೆ.

## cTranSet(el, genUrl)

`c-t` ಗಾಗಿ ಪಠ್ಯವನ್ನು ಹೊಂದಿಸಿ.

ನಿಯತಾಂಕ:
- `el` : ಮೂಲ ಅಂಶ, ಜಾಗತಿಕವನ್ನು `document` ರಲ್ಲಿ ರವಾನಿಸಬಹುದು
- `genUrl` : ಭಾಷಾ ಪ್ಯಾಕ್ ನಿಘಂಟನ್ನು ಪಡೆಯುವ ಕಾರ್ಯ
  - ನಿಯತಾಂಕ
    - `code` : ಭಾಷಾ ಕೋಡ್
  - ಮೌಲ್ಯವನ್ನು ಹಿಂತಿರುಗಿಸಿ: ಭಾಷಾ ಪ್ಯಾಕ್ ನಿಘಂಟು ವಸ್ತು, ಅಥವಾ ಈ ವಸ್ತುವನ್ನು ಪರಿಹರಿಸುವ ಭರವಸೆ
