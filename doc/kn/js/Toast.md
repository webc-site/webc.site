# Toast.js

ಟೋಸ್ಟ್ ಪಾಪ್-ಅಪ್ ಸಂದೇಶ ಪ್ರಾಂಪ್ಟ್

## toast(render, timeout)

ಪ್ರಮಾಣಿತ ಪ್ರಾಂಪ್ಟ್ ಪಾಪ್-ಅಪ್ ವಿಂಡೋವನ್ನು ತೋರಿಸಿ

ನಿಯತಾಂಕ:
- `render` : ರೆಂಡರಿಂಗ್ ಕಾರ್ಯ, ಪಾಪ್-ಅಪ್ ವಿಷಯವನ್ನು ಕಸ್ಟಮೈಸ್ ಮಾಡಲು ಬಳಸಲಾಗುತ್ತದೆ
  - ನಿಯತಾಂಕ
    - `el` : ಪಾಪ್-ಅಪ್ ವಿಂಡೋದ DOM ಅಂಶ
- `timeout` : ಸ್ವಯಂಚಾಲಿತ ಶಟ್‌ಡೌನ್‌ಗಾಗಿ ಸೆಕೆಂಡುಗಳ ವಿಳಂಬ, ಡಿಫಾಲ್ಟ್ 9, 0 ಆಗಿರುವಾಗ ಸ್ವಯಂಚಾಲಿತ ಸ್ಥಗಿತಗೊಳ್ಳುವುದಿಲ್ಲ

ರಿಟರ್ನ್ ಮೌಲ್ಯ: ಪಾಪ್-ಅಪ್ ವಿಂಡೋದ DOM ಅಂಶ
