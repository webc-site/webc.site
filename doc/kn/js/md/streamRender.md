# md/streamRender.js

ಮಾರ್ಕ್‌ಡೌನ್ ಸ್ಟ್ರೀಮಿಂಗ್ ರೆಂಡರಿಂಗ್

## default(container, source)

ನಿರ್ದಿಷ್ಟಪಡಿಸಿದ DOM ಕಂಟೇನರ್‌ಗೆ ಮಾರ್ಕ್‌ಡೌನ್ ಪಠ್ಯದ ಸ್ಟ್ರೀಮಿಂಗ್ ರೆಂಡರಿಂಗ್, ಟೈಪ್ ರೈಟರ್ ಕರ್ಸರ್ ಪರಿಣಾಮಗಳನ್ನು ಬೆಂಬಲಿಸುವುದು ಮತ್ತು ಸ್ವಯಂಚಾಲಿತ ಸ್ಕ್ರೋಲಿಂಗ್

ನಿಯತಾಂಕ:
- `container` : DOM ಕಂಟೇನರ್ ಅಂಶವು ಸಲ್ಲಿಸಿದ ವಿಷಯವನ್ನು ಹೋಸ್ಟಿಂಗ್ ಮಾಡುತ್ತದೆ
- `source` : ಓದಬಹುದಾದ ಸ್ಟ್ರೀಮ್ ಪಠ್ಯ ಡೇಟಾ ಮೂಲ

ಹಿಂತಿರುಗಿಸುವ ಮೌಲ್ಯ:

- `cancel` : ರೆಂಡರಿಂಗ್ ಮತ್ತು ಡೇಟಾ ಸ್ಟ್ರೀಮ್ ಓದುವಿಕೆಯನ್ನು ನಿಲ್ಲಿಸುವ ಕಾರ್ಯ
- `setStop` : ಸ್ವಯಂಚಾಲಿತ ಸ್ಕ್ರೋಲಿಂಗ್ ಅನ್ನು ವಿರಾಮಗೊಳಿಸಬೇಕೆ ಎಂದು ಹೊಂದಿಸುವ ಕಾರ್ಯ
  - ನಿಯತಾಂಕ
    - `val` : ಬೂಲಿಯನ್ ಮೌಲ್ಯವನ್ನು ವಿರಾಮಗೊಳಿಸಬೇಕೆ ಅಥವಾ ಬೇಡವೇ
