# hashActive.js

URL ಆಂಕರ್‌ಗಳು ಮತ್ತು ವ್ಯೂಪೋರ್ಟ್ ಸ್ಕ್ರೋಲಿಂಗ್ ಅನ್ನು ಆಧರಿಸಿ ನ್ಯಾವಿಗೇಷನ್ ಲಿಂಕ್ ಹೈಲೈಟ್ ಮಾಡುವುದು

## default(link_li, cls)

ಪುಟ ಸ್ಕ್ರಾಲ್ ಸ್ಥಾನ ಮತ್ತು URL ಆಂಕರ್ ಪಾಯಿಂಟ್ ಬದಲಾವಣೆಗಳ ಆಧಾರದ ಮೇಲೆ ಪ್ರಸ್ತುತ ಗೋಚರಿಸುವ ಪ್ರದೇಶಕ್ಕೆ ಅನುಗುಣವಾಗಿ ನ್ಯಾವಿಗೇಷನ್ ಲಿಂಕ್‌ಗಳನ್ನು ಸ್ವಯಂಚಾಲಿತವಾಗಿ ಹೈಲೈಟ್ ಮಾಡಿ.

ನಿಯತಾಂಕ:
- `link_li` : ನ್ಯಾವಿಗೇಶನ್ ಲಿಂಕ್ DOM ಅಂಶ ಪಟ್ಟಿ
- `cls` : ಸಕ್ರಿಯ CSS ವರ್ಗ ಹೆಸರು, ಡೀಫಾಲ್ಟ್ `"A"`

ರಿಟರ್ನ್ ಮೌಲ್ಯ: ಎಲ್ಲಾ ಆಲಿಸುವ ಕ್ಲೀನಪ್ ಕಾರ್ಯಗಳನ್ನು ರದ್ದುಗೊಳಿಸಿ
