# dom.js

ಡಾಕ್ಯುಮೆಂಟ್ ವೇರಿಯಬಲ್ ಸಂಕ್ಷೇಪಣಗಳು ಮತ್ತು ಶಾರ್ಟ್‌ಕಟ್ ಕಾರ್ಯಗಳು

## D

`document` ವಸ್ತುವಿನ ಶಾರ್ಟ್‌ಕಟ್ ಉಲ್ಲೇಖ

## B

`document.body` ವಸ್ತುವಿನ ಶಾರ್ಟ್‌ಕಟ್ ಉಲ್ಲೇಖ

## newEl

DOM ಅಂಶಗಳನ್ನು ರಚಿಸಿ

ನಿಯತಾಂಕ:
- `tagName` : ಅಂಶ ಟ್ಯಾಗ್ ಹೆಸರು

ರಿಟರ್ನ್ ಮೌಲ್ಯ: ಹೊಸದಾಗಿ ರಚಿಸಲಾದ DOM ಅಂಶ

## newLi(...tag_li)

ಬ್ಯಾಚ್‌ಗಳಲ್ಲಿ ಬಹು DOM ಅಂಶಗಳನ್ನು ರಚಿಸಿ

ನಿಯತಾಂಕ:
- `tag_li` : ಟ್ಯಾಗ್ ಹೆಸರಿನ ಪಟ್ಟಿ

ಹಿಂತಿರುಗಿಸುವ ಮೌಲ್ಯ:
- ಹೊಸದಾಗಿ ರಚಿಸಲಾದ ಎಲ್ಲಾ DOM ಅಂಶಗಳನ್ನು ಒಳಗೊಂಡಿರುವ ಅರೇ

## cE(name, cls)

ವೆಬ್ ಘಟಕಗಳ ಕಸ್ಟಮ್ ಅಂಶಗಳನ್ನು ನೋಂದಾಯಿಸಿ

ನಿಯತಾಂಕ:
- `name` : ಕಸ್ಟಮ್ ಅಂಶದ ಹೆಸರು, `c-` ಪೂರ್ವಪ್ರತ್ಯಯವನ್ನು ಸ್ವಯಂಚಾಲಿತವಾಗಿ ಸೇರಿಸಲಾಗುತ್ತದೆ
- `cls` : ಕಸ್ಟಮ್ ಅಂಶದ ವರ್ಗ
