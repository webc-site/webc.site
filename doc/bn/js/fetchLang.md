# fetchLang.js

অ্যাসিঙ্ক্রোনাসভাবে ভাষা প্যাকগুলি লোড করুন এবং আপডেট কলব্যাকগুলি আবদ্ধ করুন৷

## fetchLang(getDict)

অ্যাসিঙ্ক্রোনাসভাবে ভাষা প্যাক পেতে একটি ভাষা প্যাক শ্রোতা তৈরি করুন এবং ভাষা পরিবর্তন হলে একটি রিফ্রেশ কলব্যাক চালান

প্যারামিটার:
- `getDict` : ভাষা প্যাকেজের অ্যাসিঙ্ক্রোনাস ফাংশন পান
  - প্যারামিটার
    - `code` : ভাষার কোড
  - রিটার্ন মান: ভাষা প্যাক অভিধান

রিটার্ন মান: রিফ্রেশ কলব্যাক নিবন্ধনের ফাংশন
- প্যারামিটার
  - `refresh` : ভাষা আপডেট হলে কলব্যাক ফাংশন রিফ্রেশ করুন
    - প্যারামিটার
      - `i18n` : ভাষা প্যাক অভিধান অবজেক্ট
      - `code` : বর্তমান ভাষার কোড
- রিটার্ন মান: রিফ্রেশ কলব্যাক নিবন্ধনমুক্ত করতে ফাংশন
