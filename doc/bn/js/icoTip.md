# icoTip.js

আন্তর্জাতিকীকরণ ইঙ্গিত সহ একটি আইকন উপাদান সংজ্ঞায়িত করুন এবং নিবন্ধন করুন

## default(i18n_url, tag, key)

আন্তর্জাতিকীকরণ প্রম্পট বুদবুদ সহ একটি কাস্টম HTML উপাদান উপাদান নিবন্ধন করুন৷

প্যারামিটার:
- `i18n_url` : আন্তর্জাতিকীকরণ কনফিগারেশনের জন্য ঠিকানার অনুরোধ করুন৷
- `tag` : HTML উপাদানের ট্যাগ নাম কাস্টমাইজ করুন
- `key` : আন্তর্জাতিকীকরণ অভিধানে প্রম্পট পাঠ্যের মূল মান
