# i18n.js

বহু-ভাষা অবস্থা ব্যবস্থাপনা এবং পরিবর্তন পর্যবেক্ষণ

## LANG_LI

উপলব্ধ ভাষা কনফিগারেশনের তালিকা

## lang()

বর্তমান ভাষা আইডি পান

রিটার্ন মান: বর্তমান ভাষা আইডি

## langCode()

বর্তমান ভাষার কোড পান

রিটার্ন মান: বর্তমান ভাষার কোড, যেমন `"zh"`

## onLang(func)

ভাষা পরিবর্তনের জন্য শোনার কলব্যাক আবদ্ধ করুন

প্যারামিটার:
- `func` : ভাষা পরিবর্তন হলে কলব্যাক ফাংশন
  - প্যারামিটার
    - `code` : ভাষার কোড
    - `id` : ভাষা আইডি

রিটার্ন মান: আনবাউন্ড ক্লিনআপ ফাংশন

## langSet(id)

বর্তমান ভাষা পরিবর্তন করুন এবং সম্প্রচার করুন

প্যারামিটার:
- `id` : টার্গেট ভাষা আইডি
