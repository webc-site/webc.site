# route.js

একক পৃষ্ঠা ফ্রন্ট-এন্ড রাউটিং ব্যবস্থাপনা

## nowUrl()

লিডিং স্ল্যাশ সরিয়ে বর্তমান URL পাথ পান

রিটার্ন মান: বর্তমান পাথ স্ট্রিং

## route(hook)

রুট পরিবর্তনের জন্য শোনার কলব্যাক নিবন্ধন করুন। নিবন্ধন করার সময়, এটি বর্তমান পথের সাথে একবার কার্যকর করা হবে।

প্যারামিটার:
- `hook` : রুট পর্যবেক্ষণ কলব্যাক ফাংশন
  - প্যারামিটার
    - `url` : বর্তমান পথ
    - `pre` : শেষ পথ

রিটার্ন মান: অনিবন্ধিত ক্লিনআপ ফাংশন

## setPre(url)

সর্বশেষ রেকর্ড করা URL সংরক্ষণ করুন

প্যারামিটার:
- `url` : পাথ স্ট্রিং

## preUrl()

শেষ রেকর্ডের URL পান

রিটার্ন মান: শেষ পাথ স্ট্রিং

## refresh(url)

সমস্ত রুট শোনার কলব্যাকগুলিকে নির্দিষ্ট পথে ট্রিগার করতে বাধ্য করুন এবং পূর্বে আপডেট করুন৷

প্যারামিটার:
- `url` : লক্ষ্য পথ, ডিফল্ট বর্তমান পথ

## removeSlash(url)

পাথ হেডার থেকে স্ল্যাশগুলি সরান

প্যারামিটার:
- `url` : পাথ স্ট্রিং

রিটার্ন মান: অগ্রণী স্ল্যাশ সরানোর পরে পথ

## setUrl(url)

ব্রাউজারের ঠিকানা বার URL স্থিতি আপডেট করুন

প্যারামিটার:
- `url` : লক্ষ্য পথ

রিটার্ন মান: 1 যদি পাথ আসলে পরিবর্তন হয়

## goto(url)

ব্রাউজার ঠিকানা আপডেট করুন এবং একটি রুট রিফ্রেশ ট্রিগার করুন

প্যারামিটার:
- `url` : লক্ষ্য পথ
