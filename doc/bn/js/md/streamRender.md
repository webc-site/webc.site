# md/streamRender.js

মার্কডাউন স্ট্রিমিং রেন্ডারিং

## default(container, source)

নির্দিষ্ট DOM কন্টেইনারে মার্কডাউন পাঠ্যের স্ট্রিমিং রেন্ডারিং, টাইপরাইটার কার্সার প্রভাব এবং স্বয়ংক্রিয় স্ক্রলিং সমর্থন করে

প্যারামিটার:
- `container` : DOM কন্টেইনার উপাদান হোস্টিং রেন্ডার করা সামগ্রী৷
- `source` : পঠনযোগ্য স্ট্রিম পাঠ্য ডেটা উৎস৷

রিটার্ন মান:

- `cancel` : রেন্ডারিং এবং ডেটা স্ট্রিম রিডিং বন্ধ করার ফাংশন
- `setStop` : স্বয়ংক্রিয় স্ক্রোলিং থামাতে হবে কিনা তা সেট করার ফাংশন৷
  - প্যারামিটার
    - `val` : বুলিয়ান মান বিরতি দেওয়া হোক বা না হোক
