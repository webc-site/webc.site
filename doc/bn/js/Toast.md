# Toast.js

টোস্ট পপ আপ বার্তা প্রম্পট

## toast(render, timeout)

স্ট্যান্ডার্ড প্রম্পট পপ-আপ উইন্ডো দেখান

প্যারামিটার:
- `render` : রেন্ডারিং ফাংশন, পপ-আপ সামগ্রী কাস্টমাইজ করতে ব্যবহৃত হয়
  - প্যারামিটার
    - `el` : পপ-আপ উইন্ডোর DOM উপাদান
- `timeout` : স্বয়ংক্রিয় শাটডাউনের জন্য বিলম্ব সেকেন্ড, ডিফল্ট 9, 0 হলে স্বয়ংক্রিয় শাটডাউন নেই

রিটার্ন মান: পপ-আপ উইন্ডোর DOM উপাদান

- `timeout` : স্বয়ংক্রিয় শাটডাউনের জন্য বিলম্ব সেকেন্ড, ডিফল্ট 9, 0 হলে স্বয়ংক্রিয় শাটডাউন নেই
