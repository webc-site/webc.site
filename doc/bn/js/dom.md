# dom.js

নথি পরিবর্তনশীল সংক্ষেপণ এবং শর্টকাট ফাংশন

## D

`document` বস্তুর শর্টকাট রেফারেন্স

## B

`document.body` বস্তুর শর্টকাট রেফারেন্স

## css(txt, root)

## newEl

DOM উপাদান তৈরি করুন

প্যারামিটার:
- `tagName` : উপাদান ট্যাগের নাম

রিটার্ন মান: নতুন তৈরি DOM উপাদান

## newLi(...tag_li)

ব্যাচে একাধিক DOM উপাদান তৈরি করুন

প্যারামিটার:
- `tag_li` : ট্যাগ নামের তালিকা

রিটার্ন মান:
- সমস্ত নতুন তৈরি DOM উপাদান ধারণকারী অ্যারে

## cE(name, cls)

রেজিস্টার ওয়েব কম্পোনেন্ট কাস্টম উপাদান

প্যারামিটার:
- `name` : কাস্টম উপাদানের নাম, `c-` উপসর্গ স্বয়ংক্রিয়ভাবে যোগ করা হবে
- `cls` : কাস্টম উপাদানের ক্লাস
