# fetch.js

এনক্যাপসুলেশন আনার উপর ভিত্তি করে নেটওয়ার্ক অনুরোধ টুল ফাংশন

## F(url, opt)

একটি নেটওয়ার্ক অনুরোধ পাঠান এবং একটি প্রতিক্রিয়া বস্তু ফেরত

প্যারামিটার:
- `url` : ঠিকানার অনুরোধ করুন
- `opt` : `fetch` অপশন অবজেক্ট

রিটার্ন মান: প্রতিক্রিয়া বস্তু

## f(attr)

একটি উচ্চ-অর্ডার ফাংশন যা একটি অনুরোধ মোড়ানো এবং একটি নির্দিষ্ট বিন্যাসে প্রত্যাবর্তিত সামগ্রীকে পার্স করে

প্যারামিটার:
- `attr` : রেসপন্স বডি পার্সিং অ্যাট্রিবিউটের নাম

রিটার্ন মান: অ্যাসিঙ্ক্রোনাসলি অনুরোধ এবং বিষয়বস্তু পার্স করার জন্য ফাংশন
  - প্যারামিটার
    - `url` : ঠিকানার অনুরোধ করুন
    - `opt` : `fetch` অপশন অবজেক্ট
  - রিটার্ন মান: পার্স করা বিষয়বস্তু

## fTxt

একটি নেটওয়ার্ক অনুরোধ পাঠান এবং পাঠ্য বিন্যাসে প্রতিক্রিয়া বডি ফেরত দিন

প্যারামিটার:
- `url` : ঠিকানার অনুরোধ করুন
- `opt` : `fetch` অপশন অবজেক্ট

রিটার্ন মান: প্রতিক্রিয়া পাঠ্য

## fJson(url, opt)

একটি নেটওয়ার্ক অনুরোধ পাঠান এবং পার্স করুন এবং JSON ফর্ম্যাটে প্রতিক্রিয়া বডি ফেরত দিন

প্যারামিটার:
- `url` : ঠিকানার অনুরোধ করুন
- `opt` : `fetch` অপশন অবজেক্ট

রিটার্ন মান: পার্স করা JSON অবজেক্ট

## fBin(url, opt)

একটি নেটওয়ার্ক অনুরোধ পাঠান এবং একটি বাইনারি বাইট অ্যারে হিসাবে প্রতিক্রিয়া বডি ফেরত দিন

প্যারামিটার:
- `url` : ঠিকানার অনুরোধ করুন
- `opt` : `fetch` অপশন অবজেক্ট

রিটার্ন মান: বাইট অ্যারে
