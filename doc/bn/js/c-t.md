# c-t.js

এইচটিএমএল আন্তর্জাতিকীকরণ

ট্যাগ `<c-t>key</c-t>` কাস্টমাইজ করা হয়েছে, এবং `key` হল ভাষা প্যাকের কী।

## cTranSet(el, genUrl)

`c-t` এর জন্য পাঠ্য সেট করুন৷

প্যারামিটার:
- `el` : মূল উপাদান, গ্লোবাল `document` এ পাস করা যেতে পারে
- `genUrl` : ভাষা প্যাক অভিধান পেতে ফাংশন
  - প্যারামিটার
    - `code` : ভাষার কোড
  - রিটার্ন মান: একটি ভাষা প্যাক অভিধান অবজেক্ট, বা একটি প্রতিশ্রুতি যা এই বস্তুর সমাধান করে
