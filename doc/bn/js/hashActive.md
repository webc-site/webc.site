# hashActive.js

ইউআরএল অ্যাঙ্কর এবং ভিউপোর্ট স্ক্রোলিং এর উপর ভিত্তি করে নেভিগেশন লিঙ্ক হাইলাইট করা

## default(link_li, cls)

পৃষ্ঠা স্ক্রোল অবস্থান এবং ইউআরএল অ্যাঙ্কর পয়েন্ট পরিবর্তনের উপর ভিত্তি করে বর্তমানে দৃশ্যমান এলাকার সাথে সম্পর্কিত নেভিগেশন লিঙ্কগুলি স্বয়ংক্রিয়ভাবে হাইলাইট করুন।

প্যারামিটার:
- `link_li` : নেভিগেশন লিঙ্ক DOM উপাদান তালিকা
- `cls` : সক্রিয় CSS ক্লাসের নাম, ডিফল্ট `"A"`৷

রিটার্ন মান: সমস্ত শোনার ক্লিনআপ ফাংশন বাতিল করুন
