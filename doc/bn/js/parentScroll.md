# parentScroll.js

পূর্বপুরুষ স্ক্রোল ধারক উপাদান পান

## default(el)

নির্দিষ্ট উপাদান থেকে উপরের দিকে যান, এটি ধারণকারী স্ক্রোল কন্টেইনার উপাদানটি খুঁজুন এবং ফেরত দিন (যেমন `C-VS` এর ShadowRoot বিল্ট-ইন স্ক্রোল কন্টেইনার)

প্যারামিটার:
- `el` : শুরু হওয়া DOM উপাদান

রিটার্ন মান: স্ক্রোল কন্টেইনার DOM উপাদান
