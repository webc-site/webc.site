# hashActive.js

URL अँकर आणि व्ह्यूपोर्ट स्क्रोलिंगवर आधारित नेव्हिगेशन लिंक हायलाइटिंग

## default(link_li, cls)

पृष्ठ स्क्रोल स्थिती आणि URL अँकर पॉइंट बदलांवर आधारित सध्या दृश्यमान क्षेत्राशी संबंधित नेव्हिगेशन लिंक स्वयंचलितपणे हायलाइट करा.

पॅरामीटर:
- `link_li` : नेव्हिगेशन लिंक DOM घटक सूची
- `cls` : सक्रिय CSS वर्ग नाव, डीफॉल्ट `"A"`

रिटर्न व्हॅल्यू: सर्व ऐकण्याची क्लीनअप फंक्शन्स रद्द करा
