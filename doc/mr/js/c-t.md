# c-t.js

HTML आंतरराष्ट्रीयीकरण

टॅग `<c-t>key</c-t>` सानुकूलित आहे आणि `key` भाषा पॅकची की आहे.

## cTranSet(el, genUrl)

`c-t` साठी मजकूर सेट करा.

पॅरामीटर:
- `el` : रूट घटक, ग्लोबल `document` मध्ये पास केला जाऊ शकतो
- `genUrl` : भाषा पॅक शब्दकोश मिळविण्यासाठी कार्य
  - पॅरामीटर
    - `code` : भाषा कोड
  - रिटर्न व्हॅल्यू: एक भाषा पॅक डिक्शनरी ऑब्जेक्ट, किंवा एक वचन जे या ऑब्जेक्टचे निराकरण करते
