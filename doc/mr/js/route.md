# route.js

सिंगल पेज फ्रंट-एंड राउटिंग व्यवस्थापन

## nowUrl()

अग्रगण्य स्लॅश काढून वर्तमान URL पथ मिळवा

परतावा मूल्य: वर्तमान पथ स्ट्रिंग

## route(hook)

मार्ग बदलांसाठी ऐकण्याच्या कॉलबॅकची नोंदणी करा. नोंदणी करताना, ते वर्तमान मार्गासह एकदा कार्यान्वित केले जाईल.

पॅरामीटर:
- `hook` : रूट मॉनिटरिंग कॉलबॅक फंक्शन
  - पॅरामीटर
    - `url` : वर्तमान मार्ग
    - `pre` : शेवटचा मार्ग

रिटर्न व्हॅल्यू: नोंदणी न केलेले क्लीनअप फंक्शन

## setPre(url)

शेवटची रेकॉर्ड केलेली URL जतन करा

पॅरामीटर:
- `url` : पथ स्ट्रिंग

## preUrl()

शेवटच्या रेकॉर्डची URL मिळवा

रिटर्न व्हॅल्यू: शेवटची पाथ स्ट्रिंग

## refresh(url)

सर्व मार्ग ऐकण्याचे कॉलबॅक निर्दिष्ट मार्गावर ट्रिगर करण्यासाठी सक्ती करा आणि पूर्व अद्यतनित करा

पॅरामीटर:
- `url` : लक्ष्य पथ, डीफॉल्ट वर्तमान मार्ग

## removeSlash(url)

पाथ हेडरमधून स्लॅश काढा

पॅरामीटर:
- `url` : पथ स्ट्रिंग

रिटर्न व्हॅल्यू: लीडिंग स्लॅश काढून टाकल्यानंतरचा मार्ग

## setUrl(url)

ब्राउझर ॲड्रेस बार URL स्थिती अद्यतनित करा

पॅरामीटर:
- `url` : लक्ष्य मार्ग

रिटर्न व्हॅल्यू: 1 मार्ग प्रत्यक्षात बदलल्यास

## goto(url)

ब्राउझर पत्ता अपडेट करा आणि रूट रिफ्रेश ट्रिगर करा

पॅरामीटर:
- `url` : लक्ष्य मार्ग
