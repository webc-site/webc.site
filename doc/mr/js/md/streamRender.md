# md/streamRender.js

मार्कडाउन स्ट्रीमिंग प्रस्तुतीकरण

## default(container, source)

मार्कडाउन मजकूराचे निर्दिष्ट DOM कंटेनरमध्ये प्रवाहित करणे, टाइपरायटर कर्सर प्रभावांना समर्थन देणे आणि स्वयंचलित स्क्रोलिंग

पॅरामीटर:
- `container` : प्रस्तुत सामग्री होस्टिंग DOM कंटेनर घटक
- `source` : वाचनीय प्रवाह मजकूर डेटा स्रोत

परतावा मूल्य:

- `cancel` : प्रस्तुतीकरण आणि डेटा प्रवाह वाचन थांबवण्याचे कार्य
- `setStop` : स्वयंचलित स्क्रोलिंग थांबवायचे की नाही हे सेट करण्यासाठी कार्य
  - पॅरामीटर
    - `val` : बूलियन मूल्य विराम द्यावा की नाही
