# fetch.js

नेटवर्क रिक्वेस्ट टूल फंक्शन फेच एन्कॅप्स्युलेशनवर आधारित आहे

## F(url, opt)

नेटवर्क विनंती पाठवा आणि प्रतिसाद ऑब्जेक्ट परत करा

पॅरामीटर:
- `url` : पत्ता विनंती करा
- `opt` : `fetch` पर्याय ऑब्जेक्ट

रिटर्न व्हॅल्यू: रिस्पॉन्स ऑब्जेक्ट

## f(attr)

उच्च-ऑर्डर फंक्शन जे विनंती गुंडाळते आणि परत केलेल्या सामग्रीचे एका निर्दिष्ट फॉरमॅटमध्ये विश्लेषण करते

पॅरामीटर:
- `attr` : प्रतिसाद मुख्य भाग विश्लेषण विशेषता नाव

रिटर्न व्हॅल्यू: ॲसिंक्रोनस रिक्वेस्ट आणि कंटेंट पार्स करण्यासाठी फंक्शन
  - पॅरामीटर
    - `url` : पत्ता विनंती करा
    - `opt` : `fetch` पर्याय ऑब्जेक्ट
  - परतावा मूल्य: विश्लेषित सामग्री

## fTxt

नेटवर्क विनंती पाठवा आणि मजकूर स्वरूपात प्रतिसाद मुख्य भाग परत करा

पॅरामीटर:
- `url` : पत्ता विनंती करा
- `opt` : `fetch` पर्याय ऑब्जेक्ट

परतावा मूल्य: प्रतिसाद मजकूर

## fJson(url, opt)

नेटवर्क विनंती पाठवा आणि विश्लेषण करा आणि प्रतिसाद मुख्य भाग JSON फॉरमॅटमध्ये परत करा

पॅरामीटर:
- `url` : पत्ता विनंती करा
- `opt` : `fetch` पर्याय ऑब्जेक्ट

परतावा मूल्य: विश्लेषित JSON ऑब्जेक्ट

## fBin(url, opt)

नेटवर्क विनंती पाठवा आणि बायनरी बाइट ॲरे म्हणून प्रतिसाद मुख्य भाग परत करा

पॅरामीटर:
- `url` : पत्ता विनंती करा
- `opt` : `fetch` पर्याय ऑब्जेक्ट

रिटर्न व्हॅल्यू: बाइट ॲरे
