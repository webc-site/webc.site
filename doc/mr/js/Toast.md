# Toast.js

टोस्ट पॉप-अप संदेश प्रॉम्प्ट

## toast(render, timeout)

मानक प्रॉम्प्ट पॉप-अप विंडो दर्शवा

पॅरामीटर:
- `render` : रेंडरिंग फंक्शन, पॉप-अप सामग्री कस्टमाइझ करण्यासाठी वापरले जाते
  - पॅरामीटर
    - `el` : पॉप-अप विंडोचा DOM घटक
- `timeout` : स्वयंचलित शटडाउनसाठी विलंब सेकंद, डीफॉल्ट 9, 0 असताना स्वयंचलित शटडाउन नाही

रिटर्न व्हॅल्यू: पॉप-अप विंडोचा DOM घटक

रिटर्न व्हॅल्यू: एरर पॉप-अप विंडोचा DOM घटक
