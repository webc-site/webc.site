# i18n.js

बहु-भाषा स्थिती व्यवस्थापन आणि बदल निरीक्षण

## LANG_LI

उपलब्ध भाषा कॉन्फिगरेशनची सूची

## lang()

वर्तमान भाषा आयडी मिळवा

परतावा मूल्य: वर्तमान भाषा आयडी

## langCode()

वर्तमान भाषा कोड मिळवा

परतावा मूल्य: वर्तमान भाषा कोड, जसे की `"zh"`

## onLang(func)

भाषा बदलांसाठी ऐकण्याच्या कॉलबॅकला बांधा

पॅरामीटर:
- `func` : भाषा बदलल्यावर कॉलबॅक कार्य
  - पॅरामीटर
    - `code` : भाषा कोड
    - `id` : भाषा आयडी

रिटर्न व्हॅल्यू: अनबाउंड क्लीनअप फंक्शन

## langSet(id)

वर्तमान भाषा स्विच आणि प्रसारित करा

पॅरामीटर:
- `id` : लक्ष्य भाषा आयडी
