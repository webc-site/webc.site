# icoTip.js

आंतरराष्ट्रीयकरण संकेतांसह आयकॉन घटक परिभाषित करा आणि नोंदणी करा

## default(i18n_url, tag, key)

आंतरराष्ट्रीयकरण प्रॉम्प्ट बबलसह सानुकूल HTML घटक घटकाची नोंदणी करा

पॅरामीटर:
- `i18n_url` : आंतरराष्ट्रीयीकरण कॉन्फिगरेशनसाठी विनंती पत्ता
- `tag` : HTML घटकाचे टॅग नाव सानुकूल करा
- `key` : आंतरराष्ट्रीयीकरण शब्दकोशातील प्रॉम्प्ट मजकूराचे मुख्य मूल्य
