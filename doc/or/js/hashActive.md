# hashActive.js

URL ଆଙ୍କର୍ ଏବଂ ଭ୍ୟୁପୋର୍ଟ ସ୍କ୍ରୋଲିଂ ଉପରେ ଆଧାର କରି ନାଭିଗେସନ୍ ଲିଙ୍କ୍ ହାଇଲାଇଟିଂ |

## default(link_li, cls)

ପୃଷ୍ଠା ସ୍କ୍ରୋଲ୍ ସ୍ଥିତି ଏବଂ URL ଆଙ୍କର୍ ପଏଣ୍ଟ ପରିବର୍ତ୍ତନ ଉପରେ ଆଧାର କରି ସମ୍ପ୍ରତି ଦୃଶ୍ୟମାନ ହେଉଥିବା କ୍ଷେତ୍ର ସହିତ ଥିବା ନାଭିଗେସନ୍ ଲିଙ୍କଗୁଡ଼ିକୁ ସ୍ୱୟଂଚାଳିତ ଭାବରେ ହାଇଲାଇଟ୍ କରନ୍ତୁ |

ପାରାମିଟର:
- `link_li`: ନାଭିଗେସନ୍ ଲିଙ୍କ୍ DOM ଉପାଦାନ ତାଲିକା |
- `cls`: ସକ୍ରିୟ CSS ଶ୍ରେଣୀ ନାମ, ଡିଫଲ୍ଟ `"A"` |

ଫେରସ୍ତ ମୂଲ୍ୟ: ସମସ୍ତ ଶୁଣିବା ପରିଷ୍କାର କାର୍ଯ୍ୟଗୁଡ଼ିକୁ ବାତିଲ କରନ୍ତୁ |
