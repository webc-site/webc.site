# fetchLang.js

ଅସନ୍ତୁଳିତ ଭାବରେ ଭାଷା ପ୍ୟାକ୍ ଲୋଡ୍ କରନ୍ତୁ ଏବଂ ଅପଡେଟ୍ କଲବ୍ୟାକ୍ ବାନ୍ଧନ୍ତୁ |

## fetchLang(getDict)

ଭାଷା ପ୍ୟାକ୍ ଶ୍ରୋତା ସୃଷ୍ଟି କରନ୍ତୁ ଅସନ୍ତୁଳିତ ଭାବରେ ଭାଷା ପ୍ୟାକ୍ ପାଇବା ଏବଂ ଭାଷା ପରିବର୍ତ୍ତନ ହେଲେ ଏକ ସତେଜ କଲବ୍ୟାକ୍ ଏକଜେକ୍ୟୁଟ୍ କରିବା |

ପାରାମିଟର:
- `getDict`: ଭାଷା ପ୍ୟାକେଜ୍ ର ଅସନ୍ତୁଳିତ କାର୍ଯ୍ୟ ପ୍ରାପ୍ତ କରନ୍ତୁ |
  - ପାରାମିଟର
    - `code`: ଭାଷା କୋଡ୍ |
  - ଫେରସ୍ତ ମୂଲ୍ୟ: ଭାଷା ପ୍ୟାକ୍ ଅଭିଧାନ |

ଫେରସ୍ତ ମୂଲ୍ୟ: ସତେଜ କଲବ୍ୟାକ୍ ପଞ୍ଜିକରଣ କରିବା କାର୍ଯ୍ୟ |
- ପାରାମିଟର
  - `refresh`: ଭାଷା ଅଦ୍ୟତନ ହେଲେ କଲବ୍ୟାକ୍ କାର୍ଯ୍ୟକୁ ସତେଜ କର |
    - ପାରାମିଟର
      - `i18n`: ଭାଷା ପ୍ୟାକ୍ ଅଭିଧାନ ବସ୍ତୁ |
      - `code`: ସାମ୍ପ୍ରତିକ ଭାଷା କୋଡ୍ |
- ଫେରସ୍ତ ମୂଲ୍ୟ: ସତେଜ କଲବ୍ୟାକ୍ ପଞ୍ଜିକରଣ କରିବା ପାଇଁ କାର୍ଯ୍ୟ |
