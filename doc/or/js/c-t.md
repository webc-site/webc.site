# c-t.js

HTML ଆନ୍ତର୍ଜାତୀୟକରଣ |

ଟ୍ୟାଗ୍ `<c-t>key</c-t>` କଷ୍ଟୋମାଇଜ୍ ହୋଇଛି, ଏବଂ `key` ହେଉଛି ଭାଷା ପ୍ୟାକ୍ ର ଚାବି |

## cTranSet(el, genUrl)

`c-t` ପାଇଁ ପାଠ ସେଟ୍ କରନ୍ତୁ |

ପାରାମିଟର:
- `el`: ମୂଳ ଉପାଦାନ, ଗ୍ଲୋବାଲ୍ `document` ରେ ପାସ୍ ହୋଇପାରିବ |
- `genUrl`: ଭାଷା ପ୍ୟାକ୍ ଅଭିଧାନ ପାଇବା ପାଇଁ କାର୍ଯ୍ୟ |
  - ପାରାମିଟର
    - `code`: ଭାଷା କୋଡ୍ |
  - ଫେରସ୍ତ ମୂଲ୍ୟ: ଏକ ଭାଷା ପ୍ୟାକ୍ ଅଭିଧାନ ବସ୍ତୁ, କିମ୍ବା ଏକ ପ୍ରତିଜ୍ଞା ଯାହା ଏହି ବସ୍ତୁକୁ ସମାଧାନ କରେ |
