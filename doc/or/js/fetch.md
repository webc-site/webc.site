# fetch.js

ଫେକ୍ ଏନକାପସୁଲେସନ୍ ଉପରେ ଆଧାର କରି ନେଟୱାର୍କ ଅନୁରୋଧ ଉପକରଣ କାର୍ଯ୍ୟ |

## F(url, opt)

ଏକ ନେଟୱର୍କ ଅନୁରୋଧ ପଠାନ୍ତୁ ଏବଂ ଏକ ପ୍ରତିକ୍ରିୟା ବସ୍ତୁ ଫେରସ୍ତ କରନ୍ତୁ |

ପାରାମିଟର:
- `url`: ଠିକଣା ଅନୁରୋଧ |
- `opt`: `fetch` ବିକଳ୍ପ ବସ୍ତୁ |

ଫେରସ୍ତ ମୂଲ୍ୟ: ପ୍ରତିକ୍ରିୟା ବସ୍ତୁ |

## f(attr)

ଏକ ଉଚ୍ଚ-ଅର୍ଡର ଫଙ୍କସନ୍ ଯାହା ଏକ ଅନୁରୋଧକୁ ଗୁଡ଼ାଏ ଏବଂ ଫେରସ୍ତ ହୋଇଥିବା ବିଷୟବସ୍ତୁକୁ ଏକ ନିର୍ଦ୍ଦିଷ୍ଟ ଫର୍ମାଟରେ ବିଶ୍ଳେଷଣ କରେ |

ପାରାମିଟର:
- `attr`: ପ୍ରତିକ୍ରିୟା ଶରୀର ଗୁଣଧର୍ମ ନାମ ବିଶ୍ଳେଷଣ କରେ |

ଫେରସ୍ତ ମୂଲ୍ୟ: ଅସନ୍ତୁଳିତ ଭାବରେ ଅନୁରୋଧ ଏବଂ ବିଷୟବସ୍ତୁ ବିଶ୍ଳେଷଣ କରିବାକୁ କାର୍ଯ୍ୟ |
  - ପାରାମିଟର
    - `url`: ଠିକଣା ଅନୁରୋଧ |
    - `opt`: `fetch` ବିକଳ୍ପ ବସ୍ତୁ |
  - ଫେରସ୍ତ ମୂଲ୍ୟ: ପାର୍ସଡ୍ ବିଷୟବସ୍ତୁ |

## fTxt

ପାରାମିଟର:

ଫେରସ୍ତ ମୂଲ୍ୟ: ପ୍ରତିକ୍ରିୟା ପାଠ

## fJson(url, opt)

ଏକ ନେଟୱାର୍କ ଅନୁରୋଧ ପଠାନ୍ତୁ ଏବଂ JSON ଫର୍ମାଟରେ ପ୍ରତିକ୍ରିୟା ବଡି ଫେରାନ୍ତୁ |

ପାରାମିଟର:
- `url`: ଠିକଣା ଅନୁରୋଧ |
- `opt`: `fetch` ବିକଳ୍ପ ବସ୍ତୁ |

ଫେରସ୍ତ ମୂଲ୍ୟ: ପାର୍ସଡ୍ JSON ବସ୍ତୁ |

## fBin(url, opt)
