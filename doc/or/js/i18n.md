# i18n.js

ଏକାଧିକ ଭାଷା ସ୍ଥିତି ପରିଚାଳନା ଏବଂ ପରିବର୍ତ୍ତନ ମନିଟରିଂ |

## LANG_LI

ଉପଲବ୍ଧ ଭାଷା ସଂରଚନାଗୁଡ଼ିକର ତାଲିକା |

## lang()

ସାମ୍ପ୍ରତିକ ଭାଷା ID ପ୍ରାପ୍ତ କରନ୍ତୁ |

ଫେରସ୍ତ ମୂଲ୍ୟ: ସାମ୍ପ୍ରତିକ ଭାଷା ID |

## langCode()

ସାମ୍ପ୍ରତିକ ଭାଷା କୋଡ୍ ପାଆନ୍ତୁ |

ଫେରସ୍ତ ମୂଲ୍ୟ: ସାମ୍ପ୍ରତିକ ଭାଷା କୋଡ୍, ଯେପରିକି `"zh"` |

## onLang(func)

ଭାଷା ପରିବର୍ତ୍ତନ ପାଇଁ ଶ୍ରବଣ କଲବ୍ୟାକ୍ ବାନ୍ଧ |

ପାରାମିଟର:
- `func`: ଭାଷା ପରିବର୍ତ୍ତନ ହେଲେ କଲବ୍ୟାକ୍ କାର୍ଯ୍ୟ |
  - ପାରାମିଟର
    - `code`: ଭାଷା କୋଡ୍ |
    - `id`: ଭାଷା ID |

ଫେରସ୍ତ ମୂଲ୍ୟ: ଅନ୍ବାଉଣ୍ଡ କ୍ଲିନଅପ୍ କାର୍ଯ୍ୟ |

## langSet(id)

ସାମ୍ପ୍ରତିକ ଭାଷା ସୁଇଚ୍ ଏବଂ ପ୍ରସାରଣ କର |

ପାରାମିଟର:
- `id`: ଲକ୍ଷ୍ୟ ଭାଷା ID |
