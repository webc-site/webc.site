# icoTip.js

ଆନ୍ତର୍ଜାତୀୟକରଣ ସୂଚନା ସହିତ ଏକ ଆଇକନ୍ ଉପାଦାନକୁ ବ୍ୟାଖ୍ୟା ଏବଂ ପଞ୍ଜିକରଣ କର |

## default(i18n_url, tag, key)

ଆନ୍ତର୍ଜାତୀୟକରଣ ପ୍ରମ୍ପ୍ଟ ବୁବୁଲ୍ସ ସହିତ ଏକ କଷ୍ଟମ୍ HTML ଉପାଦାନ ଉପାଦାନ ପଞ୍ଜିକରଣ କରନ୍ତୁ |

ପାରାମିଟର:
- `i18n_url`: ଆନ୍ତର୍ଜାତୀୟକରଣ ବିନ୍ୟାସ ପାଇଁ ଠିକଣା ଅନୁରୋଧ |
- `tag`: HTML ଉପାଦାନର ଟ୍ୟାଗ୍ ନାମ କଷ୍ଟୋମାଇଜ୍ କରନ୍ତୁ |
- `key`: ଆନ୍ତର୍ଜାତୀୟକରଣ ଅଭିଧାନରେ ପ୍ରମ୍ପ୍ଟ ପାଠ୍ୟର ମୁଖ୍ୟ ମୂଲ୍ୟ |
