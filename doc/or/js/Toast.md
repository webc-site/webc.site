# Toast.js

ଟୋଷ୍ଟ ପପ୍-ଅପ୍ ମେସେଜ୍ ପ୍ରମ୍ପ୍ଟ |

## toast(render, timeout)

ଷ୍ଟାଣ୍ଡାର୍ଡ ପ୍ରମ୍ପ୍ଟ ପପ୍-ଅପ୍ ୱିଣ୍ଡୋ ଦେଖାନ୍ତୁ |

ପାରାମିଟର:
- `render`: ରେଣ୍ଡରିଂ ଫଙ୍କସନ୍, ପପ୍-ଅପ୍ ବିଷୟବସ୍ତୁକୁ କଷ୍ଟମାଇଜ୍ କରିବା ପାଇଁ ବ୍ୟବହୃତ |
  - ପାରାମିଟର
    - `el`: ପପ୍-ଅପ୍ ୱିଣ୍ଡୋର DOM ଉପାଦାନ |
- `timeout`: ସ୍ୱୟଂଚାଳିତ ବନ୍ଦ ପାଇଁ ସେକେଣ୍ଡ ବିଳମ୍ବ, ଡିଫଲ୍ଟ 9, 0 ହେଲେ ସ୍ୱୟଂଚାଳିତ ବନ୍ଦ ନାହିଁ |

ଫେରସ୍ତ ମୂଲ୍ୟ: ପପ୍-ଅପ୍ ୱିଣ୍ଡୋର DOM ଉପାଦାନ |
