# i18n.js

Flerspråklig statusadministrasjon og endringsovervåking

## LANG_LI

Liste over tilgjengelige språkkonfigurasjoner

## lang()

Få gjeldende språk-ID

Returverdi: gjeldende språk-ID

## langCode()

Få gjeldende språkkode

Returverdi: gjeldende språkkode, for eksempel `"zh"`

## onLang(func)

Bind tilbakeringingen for språkendringer

parameter:
- `func` : tilbakeringingsfunksjon når språk endres
  - parameter
    - `code` : Språkkode
    - `id` : Språk-ID

Returverdi: Ubundet oppryddingsfunksjon

## langSet(id)

Bytt og kringkast gjeldende språk

parameter:
- `id` : Målspråk-ID
