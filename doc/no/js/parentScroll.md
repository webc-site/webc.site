# parentScroll.js

Skaff det forfedre rullebeholderelementet

## default(el)

Gå oppover fra det angitte elementet, finn og returner rullebeholderelementet som inneholder det (som `C-VS`s ShadowRoot innebygde rullebeholder)

parameter:
- `el` : startende DOM-element

Returverdi: rullebeholder DOM-element
