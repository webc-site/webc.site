# i18n/store.js

Многоязычное управление локальным хранилищем

## get()

Получите идентификатор языка по умолчанию, рассчитанный локально в кэше или рассчитанный путем сопоставления языка браузера.

Возвращаемое значение: идентификатор языка

## set(id)

Кэшировать текущий идентификатор языка локально

параметр:
- `id`: идентификатор языка.
