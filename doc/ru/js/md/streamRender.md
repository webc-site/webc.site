# md/streamRender.js

Потоковое рендеринг Markdown

## default(container, source)

Потоковая отрисовка текста Markdown в указанный контейнер DOM с поддержкой эффектов курсора пишущей машинки и автоматической прокрутки.

параметр:
- `container`: элемент контейнера DOM, в котором размещается отображаемый контент.
- `source` : источник текстовых данных ReadableStream.

Возвращаемое значение:

- `cancel`: функция остановки рендеринга и чтения потока данных.
- `setStop`: функция, позволяющая установить, приостанавливать ли автоматическую прокрутку.
  - параметр
    - `val`: логическое значение, следует ли приостанавливать или нет.
