# fetch.js

Функция инструмента сетевых запросов на основе инкапсуляции выборки

## F(url, opt)

Отправьте сетевой запрос и верните объект ответа

параметр:
- `url` : адрес запроса.
- `opt` : объект параметров `fetch`

Возвращаемое значение: объект ответа

## f(attr)

Функция высшего порядка, которая оборачивает запрос и анализирует возвращаемое содержимое в указанном формате.

параметр:
- `attr` : имя атрибута синтаксического анализа тела ответа.

Возвращаемое значение: функция для асинхронного запроса и анализа содержимого.
  - параметр
    - `url` : адрес запроса.
    - `opt` : объект параметров `fetch`
  - Возвращаемое значение: проанализированный контент

## fTxt

Отправьте сетевой запрос и верните тело ответа в текстовом формате.

параметр:
- `url` : адрес запроса.
- `opt` : объект параметров `fetch`

Возвращаемое значение: текст ответа

## fJson(url, opt)

Отправьте сетевой запрос, проанализируйте и верните тело ответа в формате JSON.

параметр:
- `url` : адрес запроса.
- `opt` : объект параметров `fetch`

Возвращаемое значение: проанализированный объект JSON.

## fBin(url, opt)

Отправьте сетевой запрос и верните тело ответа в виде двоичного массива байтов.

параметр:
- `url` : адрес запроса.
- `opt` : объект параметров `fetch`

Возвращаемое значение: массив байтов
