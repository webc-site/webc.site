# hashActive.js

Подсветка навигационных ссылок на основе привязок URL-адресов и прокрутки области просмотра.

## default(link_li, cls)

Автоматически выделяйте навигационные ссылки, соответствующие видимой в данный момент области, на основе положения прокрутки страницы и изменений точки привязки URL-адреса.

параметр:
- `link_li` : Список элементов DOM ссылки навигации.
- `cls`: имя активного класса CSS, по умолчанию `"A"`

Возвращаемое значение: Отменить все функции очистки прослушивания.
