# route.js

Управление маршрутизацией на одной странице

## nowUrl()

Получить текущий URL-путь с удаленной косой чертой.

Возвращаемое значение: текущая строка пути.

## route(hook)

Зарегистрируйте обратный вызов для прослушивания изменений маршрута. При регистрации он будет выполнен один раз с текущим путем.

параметр:
- `hook`: функция обратного вызова для мониторинга маршрута.
  - параметр
    - `url`: текущий путь
    - `pre` : последний путь

Возвращаемое значение: незарегистрированная функция очистки.

## setPre(url)

Сохраните последний записанный URL-адрес

параметр:
- `url` : строка пути.

## preUrl()

Получить URL последней записи

Возвращаемое значение: последняя строка пути.

## refresh(url)

Принудительно запускать все обратные вызовы прослушивания маршрута по указанному пути и предварительно обновить

параметр:
- `url`: целевой путь, текущий путь по умолчанию.

## removeSlash(url)

Удалить косую черту из заголовков путей

параметр:
- `url` : строка пути.

Возвращаемое значение: путь после удаления косой черты.

## setUrl(url)

Обновить статус URL адресной строки браузера

параметр:
- `url`: целевой путь.

Возвращаемое значение: 1, если путь действительно изменяется.

## goto(url)

Обновите адрес браузера и запустите обновление маршрута.

параметр:
- `url`: целевой путь.
