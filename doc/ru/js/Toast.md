# Toast.js

Всплывающее всплывающее сообщение

## toast(render, timeout)

Показать стандартное всплывающее окно с подсказками

параметр:
- `render`: функция рендеринга, используемая для настройки содержимого всплывающего окна.
  - параметр
    - `el` : элемент DOM всплывающего окна.
- `timeout`: задержка в секундах для автоматического выключения, по умолчанию 9, автоматическое выключение при 0 отсутствует.

Возвращаемое значение: элемент DOM всплывающего окна.

- `timeout`: задержка в секундах для автоматического выключения, по умолчанию 9, автоматическое выключение при 0 отсутствует.

Возвращаемое значение: элемент DOM всплывающего окна с ошибкой.
