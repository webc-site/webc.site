# icoTip.js

Определите и зарегистрируйте компонент значка с подсказками по интернационализации.

## default(i18n_url, tag, key)

Зарегистрируйте пользовательский компонент HTML-элемента с всплывающими подсказками по интернационализации.

параметр:
- `i18n_url` : адрес запроса для настройки интернационализации.
- `tag`: настройка имени тега элемента HTML.
- `key`: ключевое значение текста подсказки в словаре интернационализации.
