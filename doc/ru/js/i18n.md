# i18n.js

Многоязычное управление статусами и мониторинг изменений

## LANG_LI

Список доступных языковых конфигураций

## lang()

Получить текущий идентификатор языка

Возвращаемое значение: текущий идентификатор языка.

## langCode()

Получить текущий код языка

Возвращаемое значение: текущий код языка, например `"zh"`

## onLang(func)

Привяжите обратный вызов для прослушивания изменений языка

параметр:
- `func`: функция обратного вызова при смене языка.
  - параметр
    - `code`: код языка.
    - `id`: идентификатор языка.

Возвращаемое значение: несвязанная функция очистки.

## langSet(id)

Переключение и трансляция текущего языка

параметр:
- `id`: идентификатор целевого языка.
