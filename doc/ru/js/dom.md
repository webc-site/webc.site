# dom.js

Сокращения переменных документа и функции быстрого доступа

## D

`document` Ярлык ссылки на объект

## B

`document.body` Ярлык ссылки на объект

## newEl

Создание элементов DOM

параметр:
- `tagName` : имя тега элемента.

Возвращаемое значение: вновь созданный элемент DOM.

## newLi(...tag_li)

Пакетное создание нескольких элементов DOM

параметр:
- `tag_li`: список имен тегов.

Возвращаемое значение:
- Массив, содержащий все вновь созданные элементы DOM.

## cE(name, cls)

Регистрация пользовательских элементов веб-компонентов

параметр:
- `name`: имя пользовательского элемента, префикс `c-` будет добавлен автоматически.
- `cls`: класс пользовательского элемента.
