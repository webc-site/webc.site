# fetchLang.js

Асинхронно загружайте языковые пакеты и привязывайте обратные вызовы обновлений.

## fetchLang(getDict)

Создайте прослушиватель языкового пакета для асинхронного получения языкового пакета и выполнения обратного вызова обновления при изменении языка.

параметр:
- `getDict`: получить асинхронную функцию языкового пакета.
  - параметр
    - `code` : код языка.
  - Возвращаемое значение: словарь языкового пакета.

Возвращаемое значение: функция для регистрации обратного вызова обновления.
- параметр
  - `refresh`: обновлять функцию обратного вызова при обновлении языка.
    - параметр
      - `i18n` : объект словаря языкового пакета.
      - `code`: текущий код языка.
- Возвращаемое значение: функция для отмены регистрации обратного вызова обновления.
