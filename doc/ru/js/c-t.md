# c-t.js

Интернационализация HTML

Тег `<c-t>key</c-t>` настраивается, а `key` — это ключ языкового пакета.

## cTranSet(el, genUrl)

Задайте текст для `c-t`.

параметр:
- `el`: корневой элемент, глобальный может быть передан в `document`
- `genUrl` : функция для получения словаря языкового пакета.
  - параметр
    - `code` : код языка.
  - Возвращаемое значение: объект словаря языкового пакета или обещание, которое разрешается в этот объект.
