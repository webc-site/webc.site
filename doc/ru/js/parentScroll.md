# parentScroll.js

Получить элемент контейнера прокрутки предка

## default(el)

Перейдите вверх от указанного элемента, найдите и верните содержащий его элемент контейнера прокрутки (например, встроенный контейнер прокрутки `C-VS` в ShadowRoot).

параметр:
- `el` : запуск элемента DOM.

Возвращаемое значение: элемент DOM контейнера прокрутки
