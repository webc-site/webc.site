# Toast.js

Ujumbe wa pop-up wa toast

## toast(render, timeout)

Onyesha dirisha ibukizi la kidokezo cha kawaida

kigezo:
- `render` : Kitendaji cha uwasilishaji, kinachotumika kubinafsisha maudhui ibukizi
  - kigezo
    - `el` : Kipengele cha DOM cha dirisha ibukizi
- `timeout` : Sekunde za kuchelewa kwa kuzima kiotomatiki, chaguo-msingi 9, hakuna kuzima kiotomatiki wakati 0

Thamani ya kurejesha: kipengele cha DOM cha dirisha ibukizi
