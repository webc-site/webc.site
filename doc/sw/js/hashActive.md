# hashActive.js

Uangaziaji wa kiungo cha kusogeza kulingana na nanga za URL na usogezaji wa kituo cha kutazama

## default(link_li, cls)

Angazia kiotomatiki viungo vya kusogeza vinavyolingana na eneo linaloonekana kwa sasa kulingana na nafasi ya kusogeza ukurasa na mabadiliko ya alama za URL.

kigezo:
- `link_li` : Kiungo cha kusogeza orodha ya vipengele vya DOM
- `cls` : Jina la darasa la CSS linalotumika, chaguomsingi `"A"`

Thamani ya kurejesha: Ghairi vipengele vyote vya kusafisha usikilizaji
