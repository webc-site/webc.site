# c-t.js

HTML kimataifa

Lebo `<c-t>key</c-t>` imebinafsishwa, na `key` ndio ufunguo wa pakiti ya lugha.

## cTranSet(el, genUrl)

Weka maandishi ya `c-t`.

kigezo:
- `el` : kipengele cha mzizi, kimataifa kinaweza kupitishwa katika `document`
- `genUrl` : kipengele cha kupata kamusi ya pakiti ya lugha
  - kigezo
    - `code` : msimbo wa lugha
  - Thamani ya kurudisha: kipengee cha kamusi ya pakiti ya lugha, au Ahadi ambayo inasuluhisha kipengee hiki
