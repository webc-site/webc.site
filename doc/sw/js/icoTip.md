# icoTip.js

Bainisha na usajili kijenzi cha ikoni na vidokezo vya utangazaji wa kimataifa

## default(i18n_url, tag, key)

Sajili kijenzi maalum cha HTML na viputo vya haraka vya uanzishaji wa kimataifa

kigezo:
- `i18n_url` : Omba anwani kwa ajili ya usanidi wa kimataifa
- `tag` : Geuza kukufaa jina la lebo ya kipengele cha HTML
- `key` : Thamani muhimu ya matini ya papo hapo katika kamusi ya kimataifa
