# i18n.js

Manajemen status multi-basa lan ngawasi owah-owahan

## LANG_LI

Dhaptar konfigurasi basa sing kasedhiya

## lang()

Entuk ID basa saiki

Nilai bali: ID basa saiki

## langCode()

Entuk kode basa saiki

Nilai bali: kode basa saiki, kayata `"zh"`

## onLang(func)

Ikatan panggilan balik ngrungokake kanggo owah-owahan basa

parameter:
- `func` : fungsi callback nalika basa diganti
  - paramèter
    - `code` : Kode basa
    - `id` : ID Basa

Nilai bali: fungsi ngresiki unbound

## langSet(id)

Ngalih lan siaran basa saiki

parameter:
- `id` : ID basa target
