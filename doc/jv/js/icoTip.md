# icoTip.js

Netepake lan ndhaptar komponen lambang kanthi pitunjuk internasionalisasi

## default(i18n_url, tag, key)

Ndhaptar komponen unsur HTML khusus kanthi gelembung cepet internasionalisasi

parameter:
- `i18n_url` : Njaluk alamat kanggo konfigurasi internasionalisasi
- `tag` : Kustomisasi jeneng tag saka unsur HTML
- `key` : Nilai kunci teks pituduh ing kamus internasionalisasi
