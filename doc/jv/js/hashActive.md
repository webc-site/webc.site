# hashActive.js

Sorotan pranala navigasi adhedhasar jangkar URL lan scrolling viewport

## default(link_li, cls)

Nyorot pranala navigasi kanthi otomatis sing cocog karo area sing saiki katon adhedhasar posisi gulung kaca lan owah-owahan titik jangkar URL.

parameter:
- `link_li` : Daftar unsur DOM pranala navigasi
- `cls` : Jeneng kelas CSS aktif, standar `"A"`

Nilai bali: Batal kabeh fungsi ngresiki ngrungokake
