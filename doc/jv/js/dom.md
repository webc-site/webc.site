# dom.js

Dokumen singkatan variabel lan fungsi trabasan

## D

`document` Referensi trabasan menyang obyek

## B

`document.body` Referensi trabasan menyang obyek

## newEl

Nggawe unsur DOM

parameter:
- `tagName`: jeneng tag unsur

Nilai bali: unsur DOM sing mentas digawe

## newLi(...tag_li)

Nggawe pirang-pirang unsur DOM ing batch

parameter:
- `tag_li` : daftar jeneng tag

Nilai bali:
- Array ngemot kabeh unsur DOM sing mentas digawe

## cE(name, cls)

Ndhaptar unsur adat Web Komponen

parameter:
- `name` : Jeneng unsur khusus, awalan `c-` bakal ditambahake kanthi otomatis
- `cls` : kelas saka unsur khusus
