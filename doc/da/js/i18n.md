# i18n.js

Flersproget statusstyring og ændringsovervågning

## LANG_LI

Liste over tilgængelige sprogkonfigurationer

## lang()

Få det aktuelle sprog-id

Returværdi: nuværende sprog-id

## langCode()

Hent den aktuelle sprogkode

Returværdi: aktuel sprogkode, såsom `"zh"`

## onLang(func)

Bind det lyttende tilbagekald for sprogændringer

parameter:
- `func` : tilbagekaldsfunktion, når sproget ændres
  - parameter
    - `code` : Sprogkode
    - `id` : Sprog-id

Returværdi: Ubundet oprydningsfunktion

## langSet(id)

Skift og udsend det aktuelle sprog

parameter:
- `id` : Målsprog-id
