# parentScroll.js

Hent forfader-rullebeholderelementet

## default(el)

Gå opad fra det angivne element, find og returner det rullebeholderelement, der indeholder det (såsom `C-VS`s ShadowRoot indbyggede rullebeholder)

parameter:
- `el` : startende DOM-element

Returværdi: rullebeholder DOM-element
