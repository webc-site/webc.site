# icoTip.js

Tentukan dan daftarkan komponen ikon dengan petunjuk internasionalisasi

## default(i18n_url, tag, key)

Daftarkan komponen elemen HTML khusus dengan gelembung perintah internasionalisasi

parameter:
- `i18n_url` : Meminta alamat untuk konfigurasi internasionalisasi
- `tag` : Sesuaikan nama tag elemen HTML
- `key` : Nilai kunci dari teks perintah dalam kamus internasionalisasi
