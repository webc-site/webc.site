# i18n.js

Manajemen status multi-bahasa dan pemantauan perubahan

## LANG_LI

Daftar konfigurasi bahasa yang tersedia

## lang()

Dapatkan ID bahasa saat ini

Nilai yang dikembalikan: ID bahasa saat ini

## langCode()

Dapatkan kode bahasa saat ini

Nilai yang dikembalikan: kode bahasa saat ini, seperti `"zh"`

## onLang(func)

Ikat panggilan balik mendengarkan untuk perubahan bahasa

parameter:
- `func` : fungsi panggilan balik ketika bahasa berubah
  - parameter
    - `code` : Kode bahasa
    - `id` : ID Bahasa

Nilai kembalian: Fungsi pembersihan tidak terikat

## langSet(id)

Ganti dan siarkan bahasa saat ini

parameter:
- `id` : ID bahasa target
