# parentScroll.js

Dapatkan elemen wadah gulir leluhur

## default(el)

Telusuri ke atas dari elemen yang ditentukan, temukan dan kembalikan elemen wadah gulir yang berisi elemen tersebut (seperti wadah gulir bawaan ShadowRoot `C-VS`)

parameter:
- `el` : memulai elemen DOM

Nilai kembalian: gulir elemen DOM kontainer
