## css(txt, root)

## newEl

Buat elemen DOM

parameter:
- `tagName` : nama tag elemen

Nilai kembalian: elemen DOM yang baru dibuat

## newLi(...tag_li)

Buat beberapa elemen DOM secara berkelompok

parameter:
- `tag_li` : daftar nama tag

Nilai pengembalian:
- Array berisi semua elemen DOM yang baru dibuat

## cE(name, cls)

Daftarkan elemen khusus Komponen Web

parameter:
- `name` : Nama elemen khusus, awalan `c-` akan ditambahkan secara otomatis
- `cls` : kelas elemen khusus
