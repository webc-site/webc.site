# parentScroll.js

Obtener el elemento contenedor de desplazamiento de ancestros

## default(el)

Recorra hacia arriba desde el elemento especificado, busque y devuelva el elemento contenedor de desplazamiento que lo contiene (como el contenedor de desplazamiento integrado ShadowRoot de `C-VS`)

parámetro:
- `el`: elemento DOM inicial

Valor de retorno: elemento DOM del contenedor de desplazamiento
