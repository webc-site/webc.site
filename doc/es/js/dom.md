## newEl

Crear elementos DOM

parámetro:
- `tagName`: nombre de etiqueta del elemento

Valor de retorno: elemento DOM recién creado

## newLi(...tag_li)

Crea múltiples elementos DOM en lotes

parámetro:
- `tag_li`: lista de nombres de etiquetas

Valor de retorno:
- Matriz que contiene todos los elementos DOM recién creados

## cE(name, cls)

Registrar elementos personalizados de componentes web

parámetro:
- `name`: Nombre del elemento personalizado, el prefijo `c-` se agregará automáticamente
- `cls`: la clase del elemento personalizado
