# i18n/store.js

Gestión de almacenamiento local multilingüe

## get()

Obtenga el ID de idioma predeterminado calculado en caché local o calculado mediante la coincidencia de idioma del navegador

Valor de retorno: ID de idioma

## set(id)

Almacenar en caché el ID del idioma actual localmente

parámetro:
- `id` : ID de idioma
