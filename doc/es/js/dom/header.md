# dom/header.js

Crear la estructura DOM de la barra de encabezado

## default()

Cree y combine la estructura DOM del encabezado de las etiquetas de encabezado, h1 y b

Valor de retorno:

- `header` : Elemento contenedor de la barra de encabezado
- `h1`: elemento de título
- `b` : subtítulo o elemento de información adicional
