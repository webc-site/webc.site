# icoTip.js

Definir y registrar un componente de icono con sugerencias de internacionalización.

## default(i18n_url, tag, key)

Registre un componente de elemento HTML personalizado con burbujas de aviso de internacionalización

parámetro:
- `i18n_url` : Solicitar dirección para configuración de internacionalización
- `tag` : Personaliza el nombre de la etiqueta del elemento HTML
- `key`: El valor clave del texto del mensaje en el diccionario de internacionalización.
