# i18n.js

Gestión de estado en varios idiomas y seguimiento de cambios

## LANG_LI

Lista de configuraciones de idiomas disponibles

## lang()

Obtener ID de idioma actual

Valor de retorno: ID de idioma actual

## langCode()

Obtener el código de idioma actual

Valor de retorno: código de idioma actual, como `"zh"`

## onLang(func)

Vincular la devolución de llamada de escucha para cambios de idioma

parámetro:
- `func`: función de devolución de llamada cuando cambia el idioma
  - parámetro
    - `code` : Código de idioma
    - `id` : ID de idioma

Valor de retorno: función de limpieza independiente

## langSet(id)

Cambiar y transmitir el idioma actual

parámetro:
- `id` : ID del idioma de destino
