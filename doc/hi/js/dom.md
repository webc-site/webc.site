# dom.js

दस्तावेज़ चर संक्षिप्तीकरण और शॉर्टकट फ़ंक्शन

## D

`document`ऑब्जेक्ट का शॉर्टकट संदर्भ

## B

`document.body`ऑब्जेक्ट का शॉर्टकट संदर्भ

## newEl

DOM तत्व बनाएं

पैरामीटर:
- `tagName` : तत्व टैग नाम

वापसी मान: नव निर्मित DOM तत्व

## newLi(...tag_li)

बैचों में एकाधिक DOM तत्व बनाएं

पैरामीटर:
- `tag_li` : टैग नाम सूची

वापसी मूल्य:
- सरणी जिसमें सभी नव निर्मित DOM तत्व शामिल हैं

## cE(name, cls)

वेब घटक कस्टम तत्वों को पंजीकृत करें

पैरामीटर:
- `name` : कस्टम तत्व नाम, `c-` उपसर्ग स्वचालित रूप से जोड़ा जाएगा
- `cls` : कस्टम तत्व का वर्ग
