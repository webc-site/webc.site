# fetchLang.js

भाषा पैक को एसिंक्रोनस रूप से लोड करें और अपडेट कॉलबैक को बाइंड करें

## fetchLang(getDict)

भाषा पैक को अतुल्यकालिक रूप से प्राप्त करने के लिए एक भाषा पैक श्रोता बनाएं और भाषा बदलने पर ताज़ा कॉलबैक निष्पादित करें

पैरामीटर:
- `getDict` : भाषा पैकेज का एसिंक्रोनस फ़ंक्शन प्राप्त करें
  - पैरामीटर
    - `code` : भाषा कोड
  - वापसी मूल्य: भाषा पैक शब्दकोश

वापसी मूल्य: ताज़ा कॉलबैक पंजीकृत करने का कार्य
- पैरामीटर
  - `refresh` : भाषा अपडेट होने पर कॉलबैक फ़ंक्शन को रिफ्रेश करें
    - पैरामीटर
      - `i18n` : भाषा पैक शब्दकोश ऑब्जेक्ट
      - `code` : वर्तमान भाषा कोड
- वापसी मूल्य: ताज़ा कॉलबैक को अपंजीकृत करने का कार्य
