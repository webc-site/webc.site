# icoTip.js

अंतर्राष्ट्रीयकरण संकेतों के साथ एक आइकन घटक को परिभाषित और पंजीकृत करें

## default(i18n_url, tag, key)

अंतर्राष्ट्रीयकरण शीघ्र बुलबुले के साथ एक कस्टम HTML तत्व घटक पंजीकृत करें

पैरामीटर:
- `i18n_url` : अंतर्राष्ट्रीयकरण कॉन्फ़िगरेशन के लिए पते का अनुरोध करें
- `tag` : HTML तत्व का टैग नाम अनुकूलित करें
- `key` : अंतर्राष्ट्रीयकरण शब्दकोश में शीघ्र पाठ का मुख्य मान
