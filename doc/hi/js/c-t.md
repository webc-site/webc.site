# c-t.js

HTML अंतर्राष्ट्रीयकरण

टैग `<c-t>key</c-t>` अनुकूलित है, और `key` भाषा पैक की कुंजी है।

## cTranSet(el, genUrl)

`c-t` के लिए टेक्स्ट सेट करें।

पैरामीटर:
- `el` : मूल तत्व, वैश्विक को `document` में पारित किया जा सकता है
- `genUrl` : भाषा पैक शब्दकोश प्राप्त करने का कार्य
  - पैरामीटर
    - `code` : भाषा कोड
  - वापसी मूल्य: एक भाषा पैक शब्दकोश ऑब्जेक्ट, या एक वादा जो इस ऑब्जेक्ट का समाधान करता है
