# Toast.js

टोस्ट पॉप-अप संदेश प्रॉम्प्ट

## toast(render, timeout)

मानक प्रॉम्प्ट पॉप-अप विंडो दिखाएँ

पैरामीटर:
- `render` : रेंडरिंग फ़ंक्शन, पॉप-अप सामग्री को अनुकूलित करने के लिए उपयोग किया जाता है
  - पैरामीटर
    - `el` : पॉप-अप विंडो का DOM तत्व
- `timeout` : स्वचालित शटडाउन के लिए देरी सेकंड, डिफ़ॉल्ट 9, 0 होने पर कोई स्वचालित शटडाउन नहीं

वापसी मान: पॉप-अप विंडो का DOM तत्व
