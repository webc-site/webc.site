# md/streamRender.js

मार्कडाउन स्ट्रीमिंग रेंडरिंग

## default(container, source)

निर्दिष्ट DOM कंटेनर में मार्कडाउन टेक्स्ट का स्ट्रीमिंग रेंडरिंग, टाइपराइटर कर्सर प्रभाव और स्वचालित स्क्रॉलिंग का समर्थन करता है

पैरामीटर:
- `container` : DOM कंटेनर तत्व रेंडर की गई सामग्री को होस्ट करता है
- `source` : ReadableStream टेक्स्ट डेटा स्रोत

वापसी मूल्य:

- `cancel` : रेंडरिंग और डेटा स्ट्रीम रीडिंग को रोकने का कार्य
- `setStop` : स्वचालित स्क्रॉलिंग को रोकना है या नहीं, यह सेट करने का फ़ंक्शन
  - पैरामीटर
    - `val` : बूलियन मान, रोकना है या नहीं
