# i18n.js

बहुभाषी स्थिति प्रबंधन और परिवर्तन निगरानी

## LANG_LI

उपलब्ध भाषा विन्यास की सूची

## lang()

वर्तमान भाषा आईडी प्राप्त करें

वापसी मूल्य: वर्तमान भाषा आईडी

## langCode()

वर्तमान भाषा कोड प्राप्त करें

वापसी मान: वर्तमान भाषा कोड, जैसे `"zh"`

## onLang(func)

भाषा परिवर्तन के लिए श्रवण कॉलबैक को बाइंड करें

पैरामीटर:
- `func` : भाषा बदलने पर कॉलबैक फ़ंक्शन
  - पैरामीटर
    - `code` : भाषा कोड
    - `id` : भाषा आईडी

वापसी मूल्य: अनबाउंड क्लीनअप फ़ंक्शन

## langSet(id)

वर्तमान भाषा बदलें और प्रसारित करें

पैरामीटर:
- `id` : लक्ष्य भाषा आईडी
