# hashActive.js

הדגשת קישורי ניווט בהתבסס על עוגני כתובת URL וגלילה של נקודת מבט

## default(link_li, cls)

הדגש אוטומטית קישורי ניווט התואמים את האזור הנראה כעת בהתבסס על מיקום גלילה בדף ושינויים בנקודת העיגון של כתובת האתר.

פָּרָמֶטֶר:
- `link_li` : רשימת רכיבי DOM של קישורי ניווט
- `cls` : שם מחלקה פעילה של CSS, ברירת מחדל `"A"`

ערך החזרה: בטל את כל פונקציות ניקוי ההאזנה
