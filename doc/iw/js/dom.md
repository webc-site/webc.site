## newEl

צור רכיבי DOM

פָּרָמֶטֶר:
- `tagName` : שם תג אלמנט

ערך החזרה: רכיב DOM חדש שנוצר

## newLi(...tag_li)

צור רכיבי DOM מרובים באצוות

פָּרָמֶטֶר:
- `tag_li` : רשימת שמות תגים

ערך החזרה:
- מערך המכיל את כל רכיבי ה-DOM החדשים שנוצרו

## cE(name, cls)

רשום רכיבים מותאמים אישית של רכיבי אינטרנט

פָּרָמֶטֶר:
- `name` : שם רכיב מותאם אישית, קידומת `c-` תתווסף אוטומטית
- `cls` : המחלקה של האלמנט המותאם אישית
