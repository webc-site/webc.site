# c-t.js

בינאום HTML

התג `<c-t>key</c-t>` מותאם אישית, ו`key` הוא המפתח של ערכת השפה.

## cTranSet(el, genUrl)

הגדר את הטקסט עבור `c-t`.

פָּרָמֶטֶר:
- `el` : אלמנט שורש, גלובלי ניתן להעביר ב`document`
- `genUrl` : פונקציה להשגת מילון ערכת שפה
  - פָּרָמֶטֶר
    - `code` : קוד שפה
  - ערך החזרה: אובייקט מילון של ערכת שפה, או הבטחה שפותרת לאובייקט זה
