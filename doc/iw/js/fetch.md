# fetch.js

פונקציית כלי בקשת רשת המבוססת על אחזור אנקפסולציה

## F(url, opt)

שלח בקשת רשת והחזר אובייקט תגובה

פָּרָמֶטֶר:
- `url` : בקש כתובת
- `opt` : `fetch` אובייקט אפשרויות

ערך החזרה: אובייקט תגובה

## f(attr)

פונקציה מסדר גבוה העוטפת בקשה ומנתחת את התוכן המוחזר בפורמט מוגדר

פָּרָמֶטֶר:
- `attr` : שם תכונת ניתוח גוף תגובה

החזרת ערך: פונקציה לבקש ולנתח תוכן באופן אסינכרוני
  - פָּרָמֶטֶר
    - `url` : בקש כתובת
    - `opt` : `fetch` אובייקט אפשרויות
  - ערך החזרה: תוכן מנותח
