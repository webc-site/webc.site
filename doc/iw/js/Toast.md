# Toast.js

הודעה מוקפצת טוסט

## toast(render, timeout)

הצג חלון פקודה מוקפץ סטנדרטי

פָּרָמֶטֶר:
- `render` : פונקציית עיבוד, המשמשת להתאמה אישית של תוכן מוקפץ
  - פָּרָמֶטֶר
    - `el` : רכיב DOM של החלון המוקפץ
- `timeout` : עיכוב שניות עבור כיבוי אוטומטי, ברירת מחדל 9, ללא כיבוי אוטומטי כאשר 0

ערך החזרה: רכיב DOM של החלון המוקפץ
