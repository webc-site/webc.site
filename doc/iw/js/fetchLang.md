# fetchLang.js

טען חבילות שפה באופן אסינכרוני ואגד התקשרויות חוזרות של עדכונים

## fetchLang(getDict)

צור מאזין חבילת שפה כדי להשיג את חבילת השפה באופן אסינכרוני ולבצע התקשרות חוזרת רענון כאשר השפה משתנה

פָּרָמֶטֶר:
- `getDict` : קבל את הפונקציה האסינכרונית של חבילת השפה
  - פָּרָמֶטֶר
    - `code` : קוד שפה
  - ערך החזרה: מילון ערכת שפה

ערך החזרה: פונקציה לרישום התקשרות חוזרת לרענון
- פָּרָמֶטֶר
  - `refresh` : רענן את פונקציית ההתקשרות חזרה כאשר השפה מתעדכנת
    - פָּרָמֶטֶר
      - `i18n` : אובייקט מילון ערכת שפה
      - `code` : קוד שפה נוכחי
- ערך החזרה: פונקציה לביטול רישום החזרה לרענון
