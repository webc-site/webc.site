# i18n.js

ניהול סטטוס רב שפות וניטור שינויים

## LANG_LI

רשימה של תצורות שפה זמינות

## lang()

קבל מזהה שפה נוכחי

ערך החזרה: מזהה השפה הנוכחית

## langCode()

קבל את קוד השפה הנוכחי

ערך החזרה: קוד שפה נוכחי, כגון `"zh"`

## onLang(func)

קשר את ההתקשרות האחורית לשינויי שפה

פָּרָמֶטֶר:
- `func` : פונקציית התקשרות חוזרת כאשר השפה משתנה
  - פָּרָמֶטֶר
    - `code` : קוד שפה
    - `id` : מזהה שפה

ערך החזרה: פונקציית ניקוי לא קשורה

## langSet(id)

החלף ושדר את השפה הנוכחית

פָּרָמֶטֶר:
- `id` : מזהה שפת יעד
