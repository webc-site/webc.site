# md/streamRender.js

עיבוד סטרימינג של סימון

## default(container, source)

הזרמת רינדור של טקסט Markdown לתוך מיכל ה-DOM שצוין, תמיכה באפקטים של סמן מכונת כתיבה וגלילה אוטומטית

פָּרָמֶטֶר:
- `container` : רכיב מכיל DOM המארח תוכן מעובד
- `source` : מקור נתוני טקסט ReadableStream

ערך החזרה:

- `cancel` : פונקציה להפסקת רינדור וקריאת זרם נתונים
- `setStop` : פונקציה להגדרה אם להשהות את הגלילה האוטומטית
  - פָּרָמֶטֶר
    - `val` : ערך בוליאני אם להשהות או לא
