# hashActive.js

Đánh dấu liên kết điều hướng dựa trên neo URL và cuộn khung nhìn

## default(link_li, cls)

Tự động đánh dấu các liên kết điều hướng tương ứng với khu vực hiện có dựa trên vị trí cuộn trang và thay đổi điểm neo URL.

tham số:
- `link_li` : Danh sách thành phần DOM liên kết điều hướng
- `cls` : Tên lớp CSS hoạt động, mặc định `"A"`

Giá trị trả về: Hủy tất cả các chức năng dọn dẹp nghe
