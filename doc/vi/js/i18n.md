# i18n.js

Quản lý trạng thái đa ngôn ngữ và giám sát thay đổi

## LANG_LI

Danh sách cấu hình ngôn ngữ có sẵn

## lang()

Nhận ID ngôn ngữ hiện tại

Giá trị trả về: ID ngôn ngữ hiện tại

## langCode()

Lấy mã ngôn ngữ hiện tại

Giá trị trả về: mã ngôn ngữ hiện tại, chẳng hạn như `"zh"`

## onLang(func)

Liên kết cuộc gọi lại nghe để thay đổi ngôn ngữ

tham số:
- `func` : chức năng gọi lại khi ngôn ngữ thay đổi
  - tham số
    - `code` : Mã ngôn ngữ
    - `id` : ID ngôn ngữ

Giá trị trả về: Hàm dọn dẹp không liên kết

## langSet(id)

Chuyển đổi và phát ngôn ngữ hiện tại

tham số:
- `id` : ID ngôn ngữ mục tiêu
