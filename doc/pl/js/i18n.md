# i18n.js

Wielojęzyczne zarządzanie statusem i monitorowanie zmian

## LANG_LI

Lista dostępnych konfiguracji językowych

## lang()

Uzyskaj aktualny identyfikator języka

Wartość zwracana: bieżący identyfikator języka

## langCode()

Uzyskaj aktualny kod języka

Wartość zwracana: bieżący kod języka, np. `"zh"`

## onLang(func)

Powiąż wywołanie zwrotne nasłuchiwania w celu zmiany języka

parametr:
- `func` : funkcja wywołania zwrotnego w przypadku zmiany języka
  - parametr
    - `code` : Kod języka
    - `id` : Identyfikator języka

Wartość zwracana: Niepowiązana funkcja czyszcząca

## langSet(id)

Przełącz i nadaj bieżący język

parametr:
- `id` : Identyfikator języka docelowego
