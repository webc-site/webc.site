# icoTip.js

Zdefiniuj i zarejestruj komponent ikony ze wskazówkami dotyczącymi internacjonalizacji

## default(i18n_url, tag, key)

Zarejestruj niestandardowy komponent elementu HTML z dymkami monitującymi o internacjonalizację

parametr:
- `i18n_url` : Adres żądania dla konfiguracji internacjonalizacji
- `tag` : dostosuj nazwę znacznika elementu HTML
- `key` : Wartość klucza tekstu zachęty w słowniku internacjonalizacji
