# i18n.js

Hizkuntza anitzeko egoeraren kudeaketa eta aldaketen jarraipena

## LANG_LI

Eskuragarri dauden hizkuntza-konfigurazioen zerrenda

## lang()

Lortu uneko hizkuntzaren IDa

Itzultzeko balioa: uneko hizkuntzaren IDa

## langCode()

Lortu uneko hizkuntza-kodea

Itzultzeko balioa: uneko hizkuntza-kodea, adibidez, `"zh"`

## onLang(func)

Lotu entzuteko deia hizkuntza aldaketetarako

parametroa:
- `func`: hizkuntza aldatzen denean deitzeko funtzioa
  - parametroa
    - `code` : hizkuntza-kodea
    - `id` : Hizkuntza IDa

Itzultzeko balioa: lotu gabeko garbiketa-funtzioa

## langSet(id)

Aldatu eta igorri uneko hizkuntza

parametroa:
- `id` : xede-hizkuntzaren IDa
