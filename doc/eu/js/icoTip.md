# icoTip.js

Definitu eta erregistratu ikono-osagai bat nazioartekotzeko aholkuekin

## default(i18n_url, tag, key)

Erregistratu HTML elementu pertsonalizatuaren osagai bat nazioartekotzeko gonbita-burbuilekin

parametroa:
- `i18n_url` : Nazioartekotzeko konfiguraziorako helbidea eskatzea
- `tag` : pertsonalizatu HTML elementuaren etiketa-izena
- `key` : Nazioartekotze hiztegiko gonbidapen-testuaren funtsezko balioa
