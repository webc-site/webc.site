# i18n/store.js

Tokiko biltegiratze eleaniztunaren kudeaketa

## get()

Lortu hizkuntzaren ID lehenetsia lokalean cachean gordeta edo arakatzailearen hizkuntza parekatzearen arabera kalkulatua

Itzultzeko balioa: hizkuntzaren IDa

## set(id)

Cachean uneko hizkuntza IDa lokalean

parametroa:
- `id` : Hizkuntza IDa
