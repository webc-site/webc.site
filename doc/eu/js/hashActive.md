# hashActive.js

Nabigazio estekaren azpimarratzea URL ainguretan eta bistaratze-atalean korritzearen arabera

## default(link_li, cls)

Nabigatu automatikoki une honetan ikusgai dagoen eremuari dagozkion nabigazio-estekak orriaren korritze-posizioan eta URL aingura puntuen aldaketetan oinarrituta.

parametroa:
- `link_li` : nabigazio-esteka DOM elementuen zerrenda
- `cls` : CSS klasearen izena aktiboa, lehenetsia `"A"`

Itzultzeko balioa: bertan behera utzi entzuteko garbiketa-funtzio guztiak
