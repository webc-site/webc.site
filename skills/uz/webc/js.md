`src/js` Odatda ishlatiladigan JS skriptlari quyidagilar

- `import { D, newEl, newLi } from "../js/dom.js"`
  - `D` - `document` qisqartmasi
  - `newEl` `dom` elementini yaratadi. Agar siz bir nechta element yaratmoqchi bo'lsangiz, ularni to'plamlarda yaratish uchun `newLi` dan foydalaning
- `import { cE } from "../js/cE.js"` Maxsus element yarating
- `import { On } from "../js/On.js"` Hodisalarni bog'lash va o'chirishni amalga oshirish uchun ishlatiladi
