`src/js` ხშირად გამოყენებული js სკრიპტები შემდეგია

- `import { D, newEl, newLi } from "../js/dom.js"`
  - `D` არის `document`-ის აბრევიატურა
  - `newEl` ქმნის `dom` ელემენტს. თუ გსურთ შექმნათ მრავალი ელემენტი, გამოიყენეთ `newLi`, რომ შექმნათ ისინი ჯგუფურად
- `import { cE } from "../js/cE.js"` შექმენით მორგებული ელემენტი
- `import { On } from "../js/On.js"` გამოიყენება ღონისძიების დაკავშირებისა და გაუქმების განსახორციელებლად
