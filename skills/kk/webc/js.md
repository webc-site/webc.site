`src/js` Жиі қолданылатын JS сценарийлері төмендегідей

- `import { D, newEl, newLi } from "../js/dom.js"`
  - `D` — `document` аббревиатурасы
  - `newEl` `dom` элементін жасайды. Бірнеше элементтерді жасағыңыз келсе, оларды пакеттерде жасау үшін `newLi` пайдаланыңыз
- `import { cE } from "../js/dom.js"` `c-` арқылы басталатын теңшелетін элемент жасаңыз
- `import { On } from "../js/On.js"` Оқиғаны байланыстыру мен байланыстыруды жүзеге асыру үшін пайдаланылады
- `import { fTxt } from "../js/fetch.js"` Мәтінді алу
