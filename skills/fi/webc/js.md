`src/js` Yleisesti käytetyt js-skriptit ovat seuraavat

- `import { D, newEl, newLi } from "../js/dom.js"`
  - `D` on lyhenne sanoista `document`
  - `newEl` luo `dom` -elementin. Jos haluat luoda useita elementtejä, käytä `newLi` luodaksesi ne erissä
- `import { cE } from "../js/dom.js"` Luo mukautettu elementti
- `import { On } from "../js/On.js"` Käytetään tapahtumien sitomisen ja sitomisen purkamiseen
