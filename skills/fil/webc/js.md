`src/js` Ang mga karaniwang ginagamit na js script ay ang mga sumusunod

- `import { D, newEl, newLi } from "../js/dom.js"`
  - Ang `D` ay ang pagdadaglat ng `document`
  - `newEl` ay lumilikha ng `dom` elemento. Kung gusto mong lumikha ng maraming elemento, gamitin ang `newLi` upang gawin ang mga ito sa mga batch
- `import { cE } from "../js/cE.js"` Gumawa ng custom na elemento
- `import { On } from "../js/On.js"` Ginagamit para ipatupad ang pag-binding at pag-unbinding ng kaganapan
