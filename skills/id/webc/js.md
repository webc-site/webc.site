`src/js` Script js yang umum digunakan adalah sebagai berikut

- `import { D, newEl, newLi } from "../js/dom.js"`
  - `D` adalah singkatan dari `document`
  - `newEl` membuat elemen `dom`. Jika Anda ingin membuat beberapa elemen, gunakan `newLi` untuk membuatnya secara berkelompok
- `import { cE } from "../js/dom.js"` Buat elemen khusus
- `import { On } from "../js/On.js"` Digunakan untuk mengimplementasikan pengikatan dan pelepasan peristiwa
