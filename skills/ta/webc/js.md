`src/js` பொதுவாகப் பயன்படுத்தப்படும் js ஸ்கிரிப்டுகள் பின்வருமாறு

- `import { D, newEl, newLi } from "../js/dom.js"`
  - `D` என்பது `document` என்பதன் சுருக்கமாகும்
  - `newEl` ஒரு `dom` உறுப்பை உருவாக்குகிறது. நீங்கள் பல கூறுகளை உருவாக்க விரும்பினால், `newLi`ஐப் பயன்படுத்தி அவற்றைத் தொகுப்பாக உருவாக்கவும்
- `import { cE } from "../js/dom.js"` `c-` இல் தொடங்கும் தனிப்பயன் உறுப்பை உருவாக்கவும்
- `import { On } from "../js/On.js"` நிகழ்வு பிணைப்பு மற்றும் அன்பைண்டிங்கைச் செயல்படுத்தப் பயன்படுகிறது
- `import { fTxt } from "../js/fetch.js"` உரையைப் பெறவும்
