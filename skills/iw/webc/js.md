`src/js` סקריפטים של js הנפוצים הם כדלקמן

- `import { D, newEl, newLi } from "../js/dom.js"`
  - `D` הוא הקיצור של `document`
  - `newEl` יוצר אלמנט `dom`. אם ברצונך ליצור מספר אלמנטים, השתמש ב`newLi` כדי ליצור אותם באצווה
- `import { cE } from "../js/dom.js"` צור רכיב מותאם אישית שמתחיל ב-`c-`
- `import { On } from "../js/On.js"` משמש ליישום הכריכה וביטול של אירועים
- `import { fTxt } from "../js/fetch.js"` קבל טקסט
