`src/js` সাধাৰণতে ব্যৱহৃত js লিপিসমূহ তলত দিয়া ধৰণৰ

- `import { D, newEl, newLi } from "../js/dom.js"`
  - `D` হৈছে `document`ৰ সংক্ষিপ্ত ৰূপ
  - `newEl` এ এটা `dom` উপাদান সৃষ্টি কৰে। যদি আপুনি একাধিক উপাদান সৃষ্টি কৰিব বিচাৰে, সিহতক গোটত সৃষ্টি কৰিবলে `newLi` ব্যৱহাৰ কৰক
- `import { cE } from "../js/dom.js"` এটা স্বনিৰ্বাচিত উপাদান সৃষ্টি কৰক
- `import { On } from "../js/On.js"` ইভেন্ট বাইণ্ডিং আৰু আনবাইণ্ডিং প্ৰণয়ন কৰিবলে ব্যৱহাৰ কৰা হয়
