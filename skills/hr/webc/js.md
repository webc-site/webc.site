`src/js` Najčešće korištene js skripte su sljedeće

- `import { D, newEl, newLi } from "../js/dom.js"`
  - `D` je skraćenica od `document`
  - `newEl` stvara element `dom`. Ako želite stvoriti više elemenata, upotrijebite `newLi` da ih izradite u serijama
- `import { cE } from "../js/cE.js"` Izradite prilagođeni element
- `import { On } from "../js/On.js"` Koristi se za implementaciju povezivanja i odvezivanja događaja
