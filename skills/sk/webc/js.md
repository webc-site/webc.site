`src/js` Bežne používané skripty js sú nasledovné

- `import { D, newEl, newLi } from "../js/dom.js"`
  - `D` je skratka z `document`
  - `newEl` vytvorí prvok `dom`. Ak chcete vytvoriť viacero prvkov, pomocou `newLi` ich vytvorte v dávkach
- `import { cE } from "../js/dom.js"` Vytvorte vlastný prvok
- `import { On } from "../js/On.js"` Používa sa na implementáciu viazania a zrušenia viazania udalosti
