`src/js` බහුලව භාවිතා වන js ස්ක්‍රිප්ට් පහත පරිදි වේ

- `import { D, newEl, newLi } from "../js/dom.js"`
  - `D` යනු `document` හි කෙටි යෙදුමයි
  - `newEl` `dom` මූලද්‍රව්‍යයක් සාදයි. ඔබට බහු මූලද්‍රව්‍ය සෑදීමට අවශ්‍ය නම්, ඒවා කාණ්ඩ වශයෙන් සෑදීමට `newLi` භාවිතා කරන්න
- `import { cE } from "../js/dom.js"` `c-` සමඟින් ආරම්භ වන අභිරුචි අංගයක් සාදන්න
- `import { On } from "../js/On.js"` සිදුවීම් බන්ධනය සහ බැඳීම ක්‍රියාත්මක කිරීමට භාවිතා කරයි
- `import { fTxt } from "../js/fetch.js"` පෙළ ලබා ගන්න
