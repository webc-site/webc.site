`src/js` Normalean erabiltzen diren js scriptak hauek dira

- `import { D, newEl, newLi } from "../js/dom.js"`
  - `D` `document`-ren laburdura da
  - `newEl`k `dom` elementu bat sortzen du. Hainbat elementu sortu nahi badituzu, erabili `newLi` multzoka sortzeko
- `import { cE } from "../js/cE.js"` Sortu elementu pertsonalizatu bat
- `import { On } from "../js/On.js"` Gertaerak lotzea eta deslokatzea ezartzeko erabiltzen da
