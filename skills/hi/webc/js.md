`src/js` आमतौर पर उपयोग की जाने वाली js स्क्रिप्ट इस प्रकार हैं

- `import { D, newEl, newLi } from "../js/dom.js"`
  - `D`, `document` का संक्षिप्त रूप है
  - `newEl` एक `dom` तत्व बनाता है। यदि आप एकाधिक तत्व बनाना चाहते हैं, तो उन्हें बैचों में बनाने के लिए `newLi` का उपयोग करें
- `import { cE } from "../js/dom.js"` एक कस्टम तत्व बनाएं
- `import { On } from "../js/On.js"` इवेंट बाइंडिंग और अनबाइंडिंग को लागू करने के लिए उपयोग किया जाता है
