## 运行步骤

1. 运行 `./sh/jsDoc.js`，查看缺失文档的函数（若无输出，则无需补充）。
2. 根据输出的文件列表，逐个读取现有文档与代码并补全（依次处理，逐个击破）。

## 文档要求

如果导出是函数，写
- 参数与返回值：
  - 仅在有参数或返回值时编写对应说明，无则不写。
  - 参数使用无序列表介绍。
  - 若返回值为数组，使用无序列表逐个说明（否则，直接写，`返回值: 描述`，一行即可）。
  - 若参数或返回值为函数，其参数与返回值使用子无序列表说明（无则不写）。

如果导出是常量，就介绍这是什么，一行即可

- 用语简洁、专业，避免使用形容词。
- 要在现有的文档上修订，不要完全重写

## 格式模板

每个名字只写一次，避免重复，尽量一行表述清楚，参考格式如下

```
# 文件名

## 函数名(参数)

用途描述

参数:
- `xx` : 用途
  - 参数
    - `a` : 用途
  - 返回值 : 说明

返回值: 说明
```
