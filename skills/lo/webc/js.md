`src/js`ສະຄຣິບ js ທີ່ໃຊ້ທົ່ວໄປມີດັ່ງນີ້

- `import { D, newEl, newLi } from "../js/dom.js"`
  - `D`ແມ່ນຕົວຫຍໍ້ຂອງ `document`
  - `newEl` ສ້າງອົງປະກອບ `dom`. ຖ້າທ່ານຕ້ອງການສ້າງອົງປະກອບຫຼາຍອັນ, ໃຫ້ໃຊ້ `newLi`ເພື່ອສ້າງພວກມັນເປັນຊຸດ
- `import { cE } from "../js/cE.js"` ສ້າງອົງປະກອບແບບກຳນົດເອງ
- `import { On } from "../js/On.js"`ໃຊ້ເພື່ອປະຕິບັດການຜູກມັດເຫດການ ແລະການຍົກເລີກການຜູກມັດ
