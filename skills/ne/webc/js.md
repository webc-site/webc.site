`src/js` सामान्यतया प्रयोग हुने js स्क्रिप्टहरू निम्नानुसार छन्

- `import { D, newEl, newLi } from "../js/dom.js"`
  - `D` `document` को संक्षिप्त नाम हो
  - `newEl` ले `dom` तत्व सिर्जना गर्छ। यदि तपाइँ धेरै तत्वहरू सिर्जना गर्न चाहनुहुन्छ भने, तिनीहरूलाई ब्याचहरूमा सिर्जना गर्न `newLi` प्रयोग गर्नुहोस्
- `import { cE } from "../js/dom.js"` `c-` बाट सुरु हुने अनुकूलन तत्व सिर्जना गर्नुहोस्
- `import { On } from "../js/On.js"` घटना बाइंडिङ र अनबाइन्डिङ लागू गर्न प्रयोग गरिन्छ
- `import { fTxt } from "../js/fetch.js"` पाठ प्राप्त गर्नुहोस्
