`src/js` Ümumi istifadə edilən js skriptləri aşağıdakılardır

- `import { D, newEl, newLi } from "../js/dom.js"`
  - `D` `document`-nin abbreviaturasıdır
  - `newEl` `dom` elementi yaradır. Birdən çox element yaratmaq istəyirsinizsə, onları qruplar şəklində yaratmaq üçün `newLi` istifadə edin
- `import { cE } from "../js/dom.js"` Fərdi element yaradın
- `import { On } from "../js/On.js"` Hadisənin bağlanması və bağlanmasının həyata keçirilməsi üçün istifadə olunur
