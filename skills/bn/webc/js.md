`src/js` সাধারণত ব্যবহৃত js স্ক্রিপ্টগুলি নিম্নরূপ

- `import { D, newEl, newLi } from "../js/dom.js"`
  - `D` হল `document` এর সংক্ষিপ্ত রূপ
  - `newEl` একটি `dom` উপাদান তৈরি করে৷ আপনি যদি একাধিক উপাদান তৈরি করতে চান, সেগুলিকে ব্যাচে তৈরি করতে `newLi` ব্যবহার করুন৷
- `import { cE } from "../js/cE.js"` একটি কাস্টম উপাদান তৈরি করুন৷
- `import { On } from "../js/On.js"` ইভেন্ট বাইন্ডিং এবং আনবাইন্ডিং বাস্তবায়ন করতে ব্যবহৃত হয়
