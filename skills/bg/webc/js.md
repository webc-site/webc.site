`src/js` Често използваните js скриптове са както следва

- `import { D, newEl, newLi } from "../js/dom.js"`
  - `D` е съкращението на `document`
  - `newEl` създава елемент `dom`. Ако искате да създадете няколко елемента, използвайте `newLi`, за да ги създадете на групи
- `import { cE } from "../js/cE.js"` Създайте персонализиран елемент
- `import { On } from "../js/On.js"` Използва се за прилагане на обвързване и развързване на събития
