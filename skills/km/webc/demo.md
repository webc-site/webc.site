# ការបង្ហាញ

យោង​ទៅ `src/BoxX/demo` ហើយ​បង្កើត​ការបង្ហាញ​នៃ​សមាសភាគ​ក្នុង `src/component name/demo/`

`demo/_.pug` គំរូរងសាកល្បង

មានតែស្លាកសមាសភាគតិចតួចប៉ុណ្ណោះដែលត្រូវបានរួមបញ្ចូល មិនចាំបាច់មានទំព័រពេញលេញទេ ហើយការនាំចូលរចនាប័ទ្ម និងស្គ្រីបត្រូវបានហាមឃាត់។ ក្របខ័ណ្ឌនឹងចាក់ដោយស្វ័យប្រវត្តិ (ដូចជា `demo/_.styl`, `demo/_.js`)។

កុំសរសេរចំណងជើង (ឧទាហរណ៍ កុំសរសេរ xxx demo) កុំសរសេរឈ្មោះសមាសភាគ កុំណែនាំសមាសភាគ សរសេរតែស្ថានភាពនៃករណីប្រើប្រាស់សាកល្បងប៉ុណ្ណោះ (ប្រសិនបើគ្មានស្ថានភាព សរសេរគ្មានចំណងជើង)

ធាតុ​ដើម​របស់ `_.pug` ប្រើ `main.demo.Lg` ដែល​បាន​កំណត់​រចនាប័ទ្ម​ក្នុង `demo/_.styl` ហើយ​មិន​កំណត់​រចនាប័ទ្ម​របស់ `main.demo.Lg` ខ្លួន​វា​ទេ។

កុងតឺន័រនៃសមាសភាគបង្ហាញគួរតែត្រូវគ្នាជាមួយ `main.demo.Lg` ហើយមិនគួរមាន `margin` និង `padding` រវាងពួកវាទេ ព្រោះ `main.demo`ខ្លួនវា​បានកំណត់​រួចហើយ `padding` ហើយ​ការ​កំណត់​វាទទេ​នឹង​ត្រូវ​ធ្វើ​ម្តងទៀត។

`demo/_.styl`ប្រសិនបើ​មាន​សមាសធាតុ​បង្ហាញ​តែ​មួយ នោះ​ធុង​របស់​វា​នឹង​មិន​មាន​ស៊ុម​ខាងក្រៅ​ទេ ហើយ​កំណត់​តែ `max-width` (ព្រោះ​ធុង​បង្ហាញ​នឹង​បន្ថែម​ស៊ុម​ខាងក្រៅ​ដោយ​ស្វ័យ​ប្រវត្តិ)។ ប្រសិនបើមានច្រើន ធុងអាចត្រូវបានបំបែកដោយស៊ុមខាងក្រៅ។

កុំសរសេរ `img`ក្នុង `demo/_.pug` ប្រើ `demo/_.styl` ដើម្បីកំណត់រូបភាពផ្ទៃខាងក្រោយ

ប្រសិនបើសមាសធាតុមានទម្រង់ច្រើន សូមបង្ហាញពួកវាតាមលំដាប់លំដោយ (ដូចជាការផ្ទុកទិន្នន័យទទេ ទិន្នន័យបរាជ័យ។ល។) ប្លង់បញ្ឈរបត់បែន

`demo/_.styl`៖ `stylus`រចនាប័ទ្មនៃទំព័រសាកល្បង សូមអនុវត្តតាមការបញ្ជាក់កូដ និងសោភ័ណភាពនៃការរចនា [./styl.md](./styl.md)

កុំសរសេរ `@import '../_.styl'` ដើម្បីនាំចូលសមាសធាតុដែលត្រូវបង្ហាញ វានឹងត្រូវបានចាក់ដោយស្វ័យប្រវត្តិ

សូមនាំចូល `@import '../../Btn/_.styl'`ក្នុង `demo/_.styl` ហើយប្រើ `.Btn`

`demo/_.js`៖ ហៅ​ស្គ្រីប ទម្រង់ `export default (root) => { ... }` ប៉ារ៉ាម៉ែត្រ `root`គឺ​ជា​ម៉ាស៊ីន (`document` ឬ​ធាតុ​ឫស​នៃ​ប្រអប់​ខ្សាច់)

`D.createElement`ត្រូវបានហាមឃាត់ ប្រើ `newEl`ដែលបានរៀបរាប់ដោយ [./js.md](./js.md)

`js`ត្រូវបានហាមឃាត់សម្រាប់ការប្រើប្រាស់នៅក្នុង `c-t` និងសម្រាប់អន្តរជាតិ `fLang`

ប្រសិនបើ `../_.js` (`../_.js`គឺ​ដើម្បី​បង្ហាញ​សមាសភាគ​ដោយ​ខ្លួន​វា សមាសភាគ​អាស្រ័យ​តម្រូវ​ដោយ​ការ​បង្ហាញ​ផ្សេង​ទៀត​នៅ​តែ​ត្រូវ​ការ​នាំ​ចូល) តម្រូវ​ឱ្យ​មាន​តែ `import` ហើយ​មិន​មាន​មាតិកា​ជាក់លាក់​ណា​មួយ​ត្រូវ​បាន​នាំ​ចូល ដែល​អាច​ត្រូវ​បាន​លុប​ចោល (ក្របខ័ណ្ឌ​នឹង​បញ្ចូល​វា​ដោយ​ស្វ័យ​ប្រវត្តិ)

`pug`មិន​ចាំបាច់​នាំចូល `js` និង `styl` ក្របខ័ណ្ឌ​នឹង​បញ្ចូល​ពួកវា​ដោយ​ស្វ័យ​ប្រវត្តិ
