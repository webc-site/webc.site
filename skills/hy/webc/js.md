`src/js` Հաճախ օգտագործվող js սկրիպտները հետևյալն են

- `import { D, newEl, newLi } from "../js/dom.js"`
  - `D`-ը `document`-ի հապավումն է
  - `newEl` ստեղծում է `dom` տարր: Եթե ​​ցանկանում եք ստեղծել մի քանի տարրեր, օգտագործեք `newLi` դրանք խմբաքանակով ստեղծելու համար
- `import { cE } from "../js/cE.js"` Ստեղծեք հատուկ տարր
- `import { On } from "../js/On.js"` Օգտագործվում է իրադարձությունների կապակցման և անջատման համար
