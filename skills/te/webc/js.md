`src/js` సాధారణంగా ఉపయోగించే js స్క్రిప్ట్‌లు క్రింది విధంగా ఉన్నాయి

- `import { D, newEl, newLi } from "../js/dom.js"`
  - `D` అనేది `document` యొక్క సంక్షిప్తీకరణ
  - `newEl` `dom` మూలకాన్ని సృష్టిస్తుంది. మీరు బహుళ మూలకాలను సృష్టించాలనుకుంటే, వాటిని బ్యాచ్‌లలో సృష్టించడానికి `newLi`ని ఉపయోగించండి
- `import { cE } from "../js/dom.js"` అనుకూల మూలకాన్ని సృష్టించండి
- `import { On } from "../js/On.js"` ఈవెంట్ బైండింగ్ మరియు అన్‌బైండింగ్‌ని అమలు చేయడానికి ఉపయోగించబడుతుంది
