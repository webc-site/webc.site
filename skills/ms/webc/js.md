`src/js` Skrip js yang biasa digunakan adalah seperti berikut

- `import { D, newEl, newLi } from "../js/dom.js"`
  - `D` ialah singkatan daripada `document`
  - `newEl` mencipta elemen `dom`. Jika anda ingin mencipta berbilang elemen, gunakan `newLi` untuk menciptanya dalam kelompok
- `import { cE } from "../js/dom.js"` Buat elemen tersuai bermula dengan `c-`
- `import { On } from "../js/On.js"` Digunakan untuk melaksanakan peristiwa mengikat dan menyahikat
- `import { fTxt } from "../js/fetch.js"` Dapatkan teks
