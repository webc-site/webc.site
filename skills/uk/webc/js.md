`src/js` Нижче наведено типові сценарії js

- `import { D, newEl, newLi } from "../js/dom.js"`
  - `D` - це абревіатура `document`
  - `newEl` створює елемент `dom`. Якщо ви хочете створити кілька елементів, використовуйте `newLi`, щоб створювати їх у групах
- `import { cE } from "../js/cE.js"` Створіть спеціальний елемент
- `import { On } from "../js/On.js"` Використовується для реалізації зв’язування та скасування зв’язування події
