`src/js` Scripturile js utilizate în mod obișnuit sunt următoarele

- `import { D, newEl, newLi } from "../js/dom.js"`
  - `D` este abrevierea lui `document`
  - `newEl` creează un element `dom`. Dacă doriți să creați mai multe elemente, utilizați `newLi` pentru a le crea în loturi
- `import { cE } from "../js/dom.js"` Creați un element personalizat începând cu `c-`
- `import { On } from "../js/On.js"` Folosit pentru a implementa legarea și dezlegarea evenimentelor
- `import { fTxt } from "../js/fetch.js"` Obțineți text
