`src/js` Skriptet js të përdorura zakonisht janë si më poshtë

- `import { D, newEl, newLi } from "../js/dom.js"`
  - `D` është shkurtesa e `document`
  - `newEl` krijon një element `dom`. Nëse dëshironi të krijoni shumë elementë, përdorni `newLi` për t'i krijuar ato në grupe
- `import { cE } from "../js/dom.js"` Krijo një element të personalizuar
- `import { On } from "../js/On.js"` Përdoret për të zbatuar lidhjen dhe çlidhjen e ngjarjeve
