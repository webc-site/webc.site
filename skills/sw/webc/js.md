`src/js` Hati za js zinazotumiwa sana ni kama ifuatavyo

- `import { D, newEl, newLi } from "../js/dom.js"`
  - `D` ni ufupisho wa `document`
  - `newEl` huunda `dom` kipengele. Ikiwa ungependa kuunda vipengele vingi, tumia `newLi` kuviunda katika makundi
- `import { cE } from "../js/dom.js"` Unda kipengele maalum
- `import { On } from "../js/On.js"` Inatumika kutekeleza kufunga na kutofunga tukio
