`src/js` Најчесто користените js скрипти се како што следува

- `import { D, newEl, newLi } from "../js/dom.js"`
  - `D` е кратенка од `document`
  - `newEl` создава елемент `dom`. Ако сакате да креирате повеќе елементи, користете `newLi` за да ги креирате во серии
- `import { cE } from "../js/dom.js"` Направете приспособен елемент почнувајќи со `c-`
- `import { On } from "../js/On.js"` Се користи за имплементирање на врзување и разврзување на настани
- `import { fTxt } from "../js/fetch.js"` Добијте текст
