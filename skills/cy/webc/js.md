`src/js` Mae sgriptiau js a ddefnyddir yn gyffredin fel a ganlyn

- `import { D, newEl, newLi } from "../js/dom.js"`
  - `D` yw'r talfyriad o `document`
  - `newEl` yn creu elfen `dom`. Os ydych chi eisiau creu elfennau lluosog, defnyddiwch `newLi` i'w creu mewn sypiau
- `import { cE } from "../js/dom.js"` Creu elfen wedi'i haddasu
- `import { On } from "../js/On.js"` Defnyddir i weithredu rhwymiad a dadrwymiad digwyddiad
