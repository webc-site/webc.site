`src/js` Τα σενάρια js που χρησιμοποιούνται συνήθως είναι τα εξής

- `import { D, newEl, newLi } from "../js/dom.js"`
  - Το `D` είναι η συντομογραφία του `document`
  - Το `newEl` δημιουργεί ένα στοιχείο `dom`. Εάν θέλετε να δημιουργήσετε πολλά στοιχεία, χρησιμοποιήστε το `newLi` για να τα δημιουργήσετε σε παρτίδες
- `import { cE } from "../js/dom.js"` Δημιουργήστε ένα προσαρμοσμένο στοιχείο που ξεκινά με `c-`
- `import { On } from "../js/On.js"` Χρησιμοποιείται για την εφαρμογή δέσμευσης και αποσύνδεσης συμβάντων
- `import { fTxt } from "../js/fetch.js"` Λήψη κειμένου
