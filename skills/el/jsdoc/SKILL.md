## Εκτελέστε βήματα

1. Εκτελέστε το `./sh/jsDoc.js` για να προβάλετε συναρτήσεις με τεκμηρίωση που λείπει (αν δεν υπάρχει έξοδος, δεν χρειάζεται να την προσθέσετε).
2. Σύμφωνα με τη λίστα αρχείων εξόδου, διαβάστε ένα προς ένα τα υπάρχοντα έγγραφα και κωδικούς και συμπληρώστε τα (επεξεργαστείτε τα με τη σειρά και σπάστε τα ένα προς ένα).

## Απαιτήσεις τεκμηρίωσης

Εάν η εξαγωγή είναι συνάρτηση, γράψτε
- Παράμετροι και τιμές επιστροφής:
  - Γράψτε τις αντίστοιχες περιγραφές μόνο εάν υπάρχουν παράμετροι ή τιμές επιστροφής, διαφορετικά μην τις γράψετε.
  - Οι παράμετροι εισάγονται χρησιμοποιώντας μη ταξινομημένες λίστες.
  - Εάν η επιστρεφόμενη τιμή είναι πίνακας, χρησιμοποιήστε μια μη ταξινομημένη λίστα για να την περιγράψετε μία προς μία (διαφορετικά, γράψτε απευθείας, `Επιστρεφόμενη τιμή: περιγραφή`, μόνο μία γραμμή).
  - Εάν οι παράμετροι ή η επιστρεφόμενη τιμή είναι συναρτήσεις, χρησιμοποιήστε λίστες δευτερεύουσας σειράς για να περιγράψετε τις παραμέτρους και τις επιστρεφόμενες τιμές (αν δεν υπάρχουν, μην τις γράψετε).

Εάν η εξαγωγή είναι σταθερά, εισαγάγετε τι είναι, μόνο μία γραμμή

- Χρησιμοποιήστε συνοπτική, επαγγελματική γλώσσα και αποφύγετε τη χρήση επιθέτων.
- Αναθεωρήστε το υπάρχον έγγραφο, μην το ξαναγράψετε εντελώς

## πρότυπο μορφής

Γράψτε κάθε όνομα μόνο μία φορά για να αποφύγετε την επανάληψη και προσπαθήστε να το εκφράσετε καθαρά σε μία γραμμή. Η μορφή αναφοράς έχει ως εξής

```
# όνομα αρχείου

## Όνομα συνάρτησης (παράμετρος)

Περιγραφή χρήσης

παράμετρος:
- `xx` : Σκοπός
  - παράμετρος
    - `a` : Σκοπός
  - Επιστρεφόμενη τιμή: περιγραφή

Επιστρεφόμενη τιμή: Περιγραφή
```
