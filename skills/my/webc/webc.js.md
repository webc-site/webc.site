
  `js` အစိတ်အပိုင်းအတွက်၊ `src/webc/componentname.js` တွင် `Light DOM`ကို အသုံးပြု၍ အစိတ်အပိုင်းကို အကောင်အထည်ဖော်ပါ။

  `တင်သွင်း "./component name.js" ကဲ့သို့သော အခြားအစိတ်အပိုင်းများကို တင်သွင်းရန် ဆက်စပ်လမ်းကြောင်းများကို အသုံးပြုပါ (အစိတ်အပိုင်းတစ်ခုစီ၏ JS ဖိုင်တွင်၊ `တင်သွင်းခြင်း "./component name/_.styl";` ကဲ့သို့သော အစတွင် သက်ဆိုင်ရာ `တင်သွင်းရမည်)။

  `this`ဝဘ်အစိတ်အပိုင်း၏ မူရင်းပြန်ခေါ်ခြင်းလုပ်ဆောင်ချက်ကိုသာ သတ်မှတ်ပါ၊ အခြားအရာများကို သီးခြားလုပ်ဆောင်ချက်များအဖြစ် ရေးပါ၊ `this` နှင့် လိုအပ်သော ဘောင်များကို ဖြတ်သန်းပါ။

  ပြင်ပတွင် ထိတွေ့နေသော အင်တာဖေ့စ်များအပြင်၊ သစ်ပင်လှုပ်ခြင်း ပိုမိုကောင်းမွန်အောင်ပြုလုပ်ခြင်း နှင့် ကုဒ်ချုံ့ခြင်းတို့ကို လွယ်ကူချောမွေ့စေရန်အတွက်၊ ၎င်းအား `this`တွင် စိတ်ကြိုက်သီးသန့်ရည်ညွှန်းချက်များကို စည်းစည်းရန် တားမြစ်ထားပါသည် (ကြားခံကိန်းရှင်များကို ဖြုတ်ချရန်အတွက် `this.$`ကို အသုံးပြုပါ) နှင့် အလယ်အလတ်ကိန်းရှင်များကို ထိန်းထားရန် ပိတ်သိမ်းမှုများကို ပြန်လည်ပြုလုပ်ရန်အတွက် အဆင့်မြင့်လုပ်ဆောင်မှုများကို အသုံးပြုနိုင်ပါသည်။

  `constructor`ကို မရေးပါနှင့်၊ ရည်ညွှန်းချက်ကို `connectedCallback` တွင် ရရှိသည်။


  `src/js/`အောက်ရှိ အများသူငှာ လုပ်ဆောင်ချက်များကို ပြန်လည်အသုံးပြုရန် [./js.md](./js.md) ကို ကိုးကားပါ။

  ကွန်တိန်နာများအတွက်၊ ရည်ညွှန်းချက်များအစား `slot`ကို အသုံးပြုပါ။
  အကယ်၍ သင်သည် ဒေတာကို တင်ရန် လိုအပ်ပါက၊ တင်ခြင်းအင်တာဖေ့စ်ကို အပေါ်အလွှာသို့ ကွဲလွဲစွာ ပြန်ခေါ်သည့် လုပ်ဆောင်ချက်ပုံစံဖြင့် ပေါ်လာပြီး ပြန်တန်ဖိုး၏ ဖော်မတ်ကိုညွှန်ပြရန် မှတ်ချက်များကို အသုံးပြုပါ။
  အစိတ်အပိုင်းများသည် မှတ်ဉာဏ်ပေါက်ကြားမှုကို ရှောင်ရှားရပါမည်။ `disconnectedCallback` ဖြုတ်ပစ်သည့်ဖြစ်ရပ်များကို လုပ်ဆောင်ချက်တစ်ခုထဲသို့ `this.$` ပေါင်းစည်းပြီး ဖြုတ်ပစ်သည့်လုပ်ဆောင်ချက်များစွာကို မရေးပါနှင့်။
  အင်တာဖေ့စ်ပြန်ပေးသည့်တန်ဖိုးသည် အရာဝတ္ထုတစ်ခုမဟုတ်ဘဲ တန်ဖိုးတစ်ခု၊ array တစ်ခု သို့မဟုတ် array တစ်ခု၏ array တစ်ခု ဖြစ်နိုင်သည်။
  အစိတ်အပိုင်းတွင် ပြည်နယ်များစွာရှိပါက၊ ပြည်နယ်တစ်ခုစီ၏ ဒေတာဖော်မတ်ကို ရှင်းလင်းစွာဖော်ပြရန် လိုင်းပေါင်းစုံမှတ်ချက်များကို အသုံးပြုပါ။ ( `*` ဖြင့် မှတ်ချက်တစ်ခုစီကို ရှေ့မတင်ပါနှင့်)
  ဂဏန်းကိန်းသေများကို အသုံးပြု၍ ပြည်နယ်ပြောင်းခြင်း ( `const.js` ဟုသတ်မှတ်သည်)။
  ဖုန်းပြန်ခေါ်မှုကို ဖွင့်နေချိန်တွင်၊ ကာတွန်းကိုပြသရန် `const xx = newEl('b');xx.className="Ing"`ကိုအသုံးပြုပြီး အစိတ်အပိုင်း JS ဖိုင်တွင် မှီခိုမှုကို `import "./Ing.js"`မှတစ်ဆင့် မိတ်ဆက်ပါ။
