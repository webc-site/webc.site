`src/js` အသုံးများသော js script များသည် အောက်ပါအတိုင်းဖြစ်သည်။

- `import { D, newEl, newLi } from "../js/dom.js"`
  - `D` သည် `document` ၏ အတိုကောက်ဖြစ်သည်။
  - `newEl` သည် `dom` အစိတ်အပိုင်းတစ်ခုကို ဖန်တီးသည်။ ဒြပ်စင်များစွာကို ဖန်တီးလိုပါက၊ ၎င်းတို့ကို အတွဲလိုက်ဖန်တီးရန် `newLi`ကို အသုံးပြုပါ။
- `import { cE } from "../js/dom.js"`စိတ်ကြိုက်ဒြပ်စင်တစ်ခု ဖန်တီးပါ။
- `import { On } from "../js/On.js"`ဖြစ်ရပ်ကို ပေါင်းစပ်ခြင်းနှင့် စည်းနှောင်ခြင်းကို အကောင်အထည်ဖော်ရန် အသုံးပြုသည်။
