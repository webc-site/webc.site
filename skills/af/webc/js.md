`src/js` Algemene js-skrifte is soos volg

- `import { D, newEl, newLi } from "../js/dom.js"`
  - `D` is die afkorting van `document`
  - `newEl` skep 'n `dom` element. As jy veelvuldige elemente wil skep, gebruik `newLi` om hulle in bondels te skep
- `import { cE } from "../js/dom.js"` Skep 'n pasgemaakte element wat met `c-` begin
- `import { On } from "../js/On.js"` Word gebruik om gebeurtenisbinding en ontbinding te implementeer
- `import { fTxt } from "../js/fetch.js"` Kry teks
