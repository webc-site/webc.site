`src/js` اسکریپت های رایج js به شرح زیر است

- `import { D, newEl, newLi } from "../js/dom.js"`
  - `D` مخفف `document` است
  - `newEl` یک عنصر `dom` ایجاد می کند. اگر می خواهید چندین عنصر ایجاد کنید، از `newLi` برای ایجاد آنها به صورت دسته ای استفاده کنید
- `import { cE } from "../js/dom.js"` یک عنصر سفارشی ایجاد کنید که با `c-` شروع می شود
- `import { On } from "../js/On.js"` برای اجرای اتصال و عدم اتصال رویداد استفاده می شود
- `import { fTxt } from "../js/fetch.js"` دریافت متن
