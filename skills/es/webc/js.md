`src/js` Los scripts js más utilizados son los siguientes

- `import { D, newEl, newLi } from "../js/dom.js"`
  - `D` es la abreviatura de `document`
  - `newEl` crea un elemento `dom`. Si desea crear varios elementos, utilice `newLi` para crearlos en lotes
- `import { cE } from "../js/dom.js"` Crear un elemento personalizado
- `import { On } from "../js/On.js"` Se utiliza para implementar la vinculación y desvinculación de eventos.
