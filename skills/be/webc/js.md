`src/js` Звычайна выкарыстоўваюцца наступныя сцэнарыі js

- `import { D, newEl, newLi } from "../js/dom.js"`
  - `D` - гэта абрэвіятура `document`
  - `newEl` стварае элемент `dom`. Калі вы жадаеце стварыць некалькі элементаў, выкарыстоўвайце `newLi` для стварэння іх групамі
- `import { cE } from "../js/dom.js"` Стварыце ўласны элемент
- `import { On } from "../js/On.js"` Выкарыстоўваецца для рэалізацыі прывязкі і адмены прывязкі падзей
