# интернационализација

Случај: Погледајте <б ид="1"> да бисте применили интернационализацију компоненти и демонстрације.

правило:
- <б ид="1"> Немојте додавати <б ид="2"> на крају уноса, ставите <б ид="3"> у шаблон за пуг и јс код и спојите га
- Користите <б ид="2"> за вишередни текст од <б ид="1"> да бисте лако уклонили размаке на крају.
- Ако нема интернационализованог уноса, обришите датотеку <б ид="1"> (не пишите ништа, немојте писати <б ид="2">)
- Уноси морају бити на једном језику, а мешање више језика је строго забрањено.

процес:

1. Погледајте код изворног језика конфигурисан у основном директоријуму <б ид="1"> → <б ид="2"> → <б ид="3">

2. Унесите <б цласс="ЦОДЕ">срц/вебц/име компоненте</б> директоријум

3. Интернационализујте компоненте
   - Напишите текст који се користи у <б цласс="ЦОДЕ">име компоненте.јс</б> као унос у <б цласс="ЦОДЕ">срц/вебц/цомпонент наме/и18н/соурце лангуаге/јс.имл</б>.
   - Измените <б цласс="ЦОДЕ">срц/вебц/цомпонент наме/И18Н.јс</б> и конфигуришите константу <б ид="1"> имена кључа за унос, као што је <б ид="2">
   - Увод и повезивање у <б цласс="ЦОДЕ">срц/вебц/цомпонент наме.јс</б> (ако постоји <б ид="1">, ако не, није неопходно):
     ```js
     import { fetchLang } from "../js/fetchLang.js";
     импорт и18нУрл, { КЕИ } из "./име компоненте/И18Н.јс"; // Оквир ће аутоматски убацити и18нУрл, подразумевани подразумевани извоз, немојте га сами креирати

     const fLang = fetchLang(i18nUrl);
     fLang((i18n) => {
       btn.title = i18n[KEY];
     });
     ```
    Покушајте да се повежете у групама да бисте смањили фЛанг позиве
    <б цласс="ЦОДЕ">име компоненте.јс</б> компоненте не би требало да има стринг који садржи текст

4. Интернационализација презентација
   - У <б ид="1">, текст ознаке који треба да се преведе је заузет у облику <б цласс="ЦОДЕ">ц-т кључа</б> (ово је прилагођена ознака, посебан ред), а унос је уписан у <б цласс="ЦОДЕ">демо/и18н/соурце лангуаге/пуг.имл</б>
   - <б ид="1"> Напишите оригинални текстуални унос који се користи у <б цласс="ЦОДЕ">демо/и18н/соурце лангуаге/јс.имл</б> и ставите га у <б ид="2">
   - Уведите <б ид="2"> у <б ид="1"> и учитајте <б ид="3"> (ово је <б ид="4">), погледајте горенаведени процес интернационализације компоненти (на пример, <б ид="5"> нема <б ид="6"> испод, прескочите овај корак)
   - <б ид="1"> Конфигуришите константе имена кључа у <б цласс="ЦОДЕ">демо/и18н/соурце лангуаге/јс.имл</б> и извезите

5. Напишите опис компоненте у <б цласс="ЦОДЕ">срц/вебц/цомпонент наме/и18н/соурце лангуаге/РЕАДМЕ.мд</б>. Процес је следећи
  5.1 Напишите главни наслов. Наслов треба да почиње са <б ид="1">, да опише сврху компоненте у једној реченици (не пише назив компоненте) и не додаје знакове интерпункције на крају.
  5.2 Наведите главне карактеристике компоненте, почевши од главног наслова, а не пишите секундарне наслове. Ако постоји више карактеристика, користите неуређену листу (ако је опис неуређене листе релативно кратак, на крају неће бити тачке).
  5.3 Напишите <б цласс="ЦОДЕ">## Користећи демонстрацију</б>, демо код почиње са <б ид="1"><б ид="2">хтмл <б цласс="ЦОДЕ"> (у демо коду, ознака једног елемента није затворена), ако постоји </б>јс<б цласс="ЦОДЕ">, напишите интерну класу </б>><б>хтмл"<б цласс=",><б>хтмл"> ид="3"><б цласс="ЦОДЕ"> Ознака, </б>јс` Нема увлачење у првом реду садржаја.
  Ако је то чист јс, почните са <б ид="1"><б ид="2">јс <б цласс="ЦОДЕ">. </б>јс<б цласс="ЦОДЕ"> У демонстрацији, ако треба да увезете </б>цомпонент/_.јс<б цласс="ЦОДЕ">, користите </б>импорт ккк из "вебц.сите/цомпонент наме.јс";<б цласс="ЦОДЕ">. Ово је преведени </б>јс`, који ће аутоматски убацити јс и цсс.
  <б ид="1"> Користите сажето писање, као што је <б ид="2"> да бисте прогласили више константи

Главни текст уводи како се користи компонента, параметри интерфејса, пребацивање између <б ид="1"> и <б ид="2"> у различитим стањима итд.
Затим опишите параметре, повратну вредност, статус стила итд. (ако их има) и додајте секундарни наслов.

Користите речи сажето и избегавајте придеве.

Копирајтерство треба да избегава бесмислице и не пише бесмислене речи (као што су: универзално, контејнер итд.).

6. Покрените <б ид="1"> основног директоријума, превод
