Используйте стандартный `css`, не пишите `css`, начиная с `-webkit-` (фреймворк автоматически внедрит его)

Разрабатывайте стили компонентов с помощью современных функций `css`, таких как `css nesting` , `flex` и т. д.

Для одного и того же значения можно написать, определить и повторно использовать несколько селекторов CSS в одном месте, а функция `stylus` будет отключена.

Единицей пикселей является `px`, а значение должно быть целым числом.

Следуйте эстетике дизайна новейшего жидкого стекла Apple.

`svg` Поместите изображение в папку `svg` и используйте его в `styl` с относительным путем.

Не используйте внешние тени, чаще используйте внутренние тени, при этом внутренние тени не должны смещаться.

Не устанавливайте `font-family`

`stylus` Использует только самые основные переменные и формат синтаксиса отступов, без дополнительных функций.
