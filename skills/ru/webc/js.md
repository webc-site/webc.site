`src/js` Наиболее часто используемые сценарии js:

- `import { D, newEl, newLi } from "../js/dom.js"`
  - `D` — это сокращение от `document`
  - `newEl` создает элемент `dom`. Если вы хотите создать несколько элементов, используйте `newLi` для их пакетного создания.
- `import { cE } from "../js/dom.js"` Создать пользовательский элемент
- `import { On } from "../js/On.js"` Используется для реализации привязки и отмены привязки событий.
