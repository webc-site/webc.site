`src/js` عام طور تي استعمال ٿيندڙ js اسڪرپٽ هن ريت آهن

- `import { D, newEl, newLi } from "../js/dom.js"`
  - `D` جو مخفف آهي `document`
  - `newEl` هڪ `dom` عنصر ٺاهي ٿو. جيڪڏھن توھان گھڻا عنصر ٺاھيو ٿا، استعمال ڪريو `newLi` انھن کي بيچ ۾ ٺاھيو
- `import { cE } from "../js/dom.js"` سان شروع ٿيندڙ ڪسٽم عنصر ٺاهيو `c-`
- `import { On } from "../js/On.js"` ايونٽ بائنڊنگ ۽ ان بائنڊنگ کي لاڳو ڪرڻ لاءِ استعمال ڪيو ويو
- `import { fTxt } from "../js/fetch.js"` متن حاصل ڪريو
