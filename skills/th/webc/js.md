`src/js` สคริปต์ js ที่ใช้กันทั่วไปมีดังนี้

- `import { D, newEl, newLi } from "../js/dom.js"`
  - `D` เป็นตัวย่อของ `document`
  - `newEl` สร้าง `dom` องค์ประกอบ หากคุณต้องการสร้างองค์ประกอบหลายรายการ ให้ใช้ `newLi` เพื่อสร้างเป็นชุด
- `import { cE } from "../js/dom.js"` สร้างองค์ประกอบที่กำหนดเอง
- `import { On } from "../js/On.js"` ใช้เพื่อดำเนินการผูกและเลิกผูกเหตุการณ์
