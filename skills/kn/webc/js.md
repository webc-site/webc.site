`src/js` ಸಾಮಾನ್ಯವಾಗಿ ಬಳಸುವ js ಸ್ಕ್ರಿಪ್ಟ್‌ಗಳು ಈ ಕೆಳಗಿನಂತಿವೆ

- `import { D, newEl, newLi } from "../js/dom.js"`
  - `D` `document` ನ ಸಂಕ್ಷೇಪಣವಾಗಿದೆ
  - `newEl` ಒಂದು `dom` ಅಂಶವನ್ನು ರಚಿಸುತ್ತದೆ. ನೀವು ಬಹು ಅಂಶಗಳನ್ನು ರಚಿಸಲು ಬಯಸಿದರೆ, ಅವುಗಳನ್ನು ಬ್ಯಾಚ್‌ಗಳಲ್ಲಿ ರಚಿಸಲು `newLi` ಬಳಸಿ
- `import { cE } from "../js/cE.js"` ಕಸ್ಟಮ್ ಅಂಶವನ್ನು ರಚಿಸಿ
- `import { On } from "../js/On.js"` ಈವೆಂಟ್ ಬೈಂಡಿಂಗ್ ಮತ್ತು ಅನ್‌ಬೈಂಡಿಂಗ್ ಅನ್ನು ಕಾರ್ಯಗತಗೊಳಿಸಲು ಬಳಸಲಾಗುತ್ತದೆ
