## 단계 실행

1. 문서가 누락된 함수를 보려면 `./sh/jsDoc.js`을 실행하세요(출력이 없으면 추가할 필요가 없습니다).
2. 출력된 파일 목록에 따라 기존 문서와 코드를 하나씩 읽어서 완성합니다(순서대로 처리하고 하나씩 분리).

## 문서 요구 사항

내보내기가 함수인 경우 다음을 작성하십시오.
- 매개변수 및 반환 값:
  - 매개변수나 반환 값이 있는 경우에만 해당 설명을 작성하고, 그렇지 않은 경우에는 작성하지 않습니다.
  - 매개변수는 순서가 지정되지 않은 목록을 사용하여 도입됩니다.
  - 반환 값이 배열인 경우 순서가 지정되지 않은 목록을 사용하여 하나씩 설명합니다(그렇지 않으면 `반환 값: 설명`, 한 줄만 직접 작성).
  - 매개변수 또는 반환값이 함수인 경우 순서가 지정되지 않은 하위 목록을 사용하여 매개변수 및 반환값을 설명합니다(없으면 작성하지 마세요).

내보내기가 상수인 경우 그것이 무엇인지 소개하십시오. 한 줄만 입력하십시오.

- 간결하고 전문적인 언어를 사용하고 형용사 사용을 피하세요.
- 기존 문서를 수정하고 완전히 다시 작성하지 마세요.

## 형식 템플릿

각 이름은 중복되지 않도록 한 번씩만 쓰고, 한 줄로 명확하게 표현하도록 노력하세요. 참고 형식은 다음과 같습니다

```
# 파일 이름

## 함수명(파라미터)

사용법 설명

매개변수:
- `xx` : 목적
  - 매개변수
    - `a` : 목적
  - 반환 값: 설명

반환 값: 설명
```
