`src/js` Algengar js forskriftir eru eftirfarandi

- `import { D, newEl, newLi } from "../js/dom.js"`
  - `D` er skammstöfun á `document`
  - `newEl` býr til `dom` frumefni. Ef þú vilt búa til marga þætti, notaðu `newLi` til að búa þá til í lotum
- `import { cE } from "../js/dom.js"` Búðu til sérsniðna þátt sem byrjar á `c-`
- `import { On } from "../js/On.js"` Notað til að innleiða atburðabindingu og afbindingu
- `import { fTxt } from "../js/fetch.js"` Fáðu texta
