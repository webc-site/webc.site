用標準 `css`，不寫 `-webkit-` 開頭的 `css`（框架會自動注入）

以現代 `css` 特性（如 `css nesting` 、`flex` 等） 開發元件樣式

相同的值，可以寫多個 css 選擇器，一個定義重複使用，停用 `stylus` 的函數

像素單位用 `px`，數值必須是整數

遵循蘋果最新液態玻璃的設計美學

`svg` 圖片放到 `svg` 資料夾下，用相對路徑在 `styl` 中使用

不用外陰影，多用內陰影，內陰影不要有位移

不設定 `font-family`

`stylus` 只用最基本的變數、縮排語法格式，不用高階特性

禁止使用 `@import` 匯入其他元件樣式（包括元件 `_.styl` 和 `demo/_.styl`）；元件依賴樣式透過元件的 JS 檔案匯入
