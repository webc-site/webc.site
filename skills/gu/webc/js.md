`src/js` સામાન્ય રીતે ઉપયોગમાં લેવાતી js સ્ક્રિપ્ટો નીચે મુજબ છે

- `import { D, newEl, newLi } from "../js/dom.js"`
  - `D` એ `document`નું સંક્ષેપ છે
  - `newEl` એક `dom` ઘટક બનાવે છે. જો તમે બહુવિધ ઘટકો બનાવવા માંગતા હો, તો તેમને બેચમાં બનાવવા માટે `newLi` નો ઉપયોગ કરો
- `import { cE } from "../js/dom.js"` કસ્ટમ એલિમેન્ટ બનાવો
- `import { On } from "../js/On.js"` ઇવેન્ટ બાઈન્ડિંગ અને અનબાઈન્ડિંગને અમલમાં મૂકવા માટે વપરાય છે
