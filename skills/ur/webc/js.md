`src/js` عام طور پر استعمال ہونے والے js اسکرپٹ درج ذیل ہیں۔

- `import { D, newEl, newLi } from "../js/dom.js"`
  - `D` `document` کا مخفف ہے۔
  - `newEl` ایک `dom` عنصر بناتا ہے۔ اگر آپ متعدد عناصر بنانا چاہتے ہیں تو انہیں بیچوں میں بنانے کے لیے `newLi` استعمال کریں۔
- `import { cE } from "../js/cE.js"` ایک حسب ضرورت عنصر بنائیں
- `import { On } from "../js/On.js"` ایونٹ بائنڈنگ اور ان بائنڈنگ کو نافذ کرنے کے لیے استعمال کیا جاتا ہے۔
