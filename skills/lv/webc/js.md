`src/js` Parasti izmantotie js skripti ir šādi

- `import { D, newEl, newLi } from "../js/dom.js"`
  - `D` ir `document` saīsinājums
  - `newEl` izveido `dom` elementu. Ja vēlaties izveidot vairākus elementus, izmantojiet `newLi`, lai tos izveidotu grupās
- `import { cE } from "../js/dom.js"` Izveidojiet pielāgotu elementu, kas sākas ar `c-`
- `import { On } from "../js/On.js"` Izmanto, lai ieviestu notikumu saistīšanu un atsaistīšanu
- `import { fTxt } from "../js/fetch.js"` Saņemt tekstu
