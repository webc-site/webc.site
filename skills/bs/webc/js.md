`src/js` Najčešće korištene js skripte su sljedeće

- `import { D, newEl, newLi } from "../js/dom.js"`
  - `D` je skraćenica od `document`
  - `newEl` kreira `dom` element. Ako želite kreirati više elemenata, koristite `newLi` da ih kreirate u serijama
- `import { cE } from "../js/cE.js"` Kreirajte prilagođeni element
- `import { On } from "../js/On.js"` Koristi se za implementaciju povezivanja i poništavanja događaja
