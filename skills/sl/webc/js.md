`src/js` Pogosto uporabljeni skripti js so naslednji

- `import { D, newEl, newLi } from "../js/dom.js"`
  - `D` je okrajšava za `document`
  - `newEl` ustvari element `dom`. Če želite ustvariti več elementov, uporabite `newLi`, da jih ustvarite v serijah
- `import { cE } from "../js/dom.js"` Ustvarite element po meri
- `import { On } from "../js/On.js"` Uporablja se za implementacijo vezave in razveze dogodkov
