`src/js` Skrip js sing umum digunakake kaya ing ngisor iki

- `import { D, newEl, newLi } from "../js/dom.js"`
  - `D` iku singkatan saka `document`
  - `newEl` nggawe elemen `dom`. Yen sampeyan pengin nggawe macem-macem unsur, gunakake `newLi` kanggo nggawe batch
- `import { cE } from "../js/dom.js"` Gawe unsur khusus sing diwiwiti karo `c-`
- `import { On } from "../js/On.js"` Digunakake kanggo ngleksanakake acara ngiket lan ora ngiket
- `import { fTxt } from "../js/fetch.js"` Entuk teks
