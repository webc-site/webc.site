Gunakake standar `css`, aja nulis `css` diwiwiti karo `-webkit-` (framework bakal nyuntikake kanthi otomatis)

Gawe gaya komponen kanthi fitur `css` modern kayata `css nesting` , `flex` lsp.

Kanggo nilai sing padha, sawetara pamilih css bisa ditulis, ditetepake lan digunakake maneh ing sak panggonan, lan fungsi `stylus` dipateni.

Unit piksel yaiku `px`, lan nilai kasebut kudu integer.

`svg` Selehake gambar ing folder `svg` banjur gunakake ing `styl` kanthi path relatif

Aja nggunakake bayangan njaba, gunakake bayangan batin luwih kerep, lan bayangan batin ora kena dipindhah.

Aja nyetel `font-family`

`stylus` Mung nggunakake variabel paling dhasar lan format sintaks indentasi, tanpa fitur canggih

Dilarang nggunakake `@import` kanggo ngimpor gaya komponen liyane (kalebu komponen `_.styl` lan `demo/_.styl`); gaya gumantung komponen diimpor liwat file JS komponen
