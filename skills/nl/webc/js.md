`src/js` Veelgebruikte js-scripts zijn als volgt

- `import { D, newEl, newLi } from "../js/dom.js"`
  - `D` is de afkorting van `document`
  - `newEl` maakt een `dom` element. Als je meerdere elementen wilt maken, gebruik dan `newLi` om ze in batches te maken
- `import { cE } from "../js/dom.js"` Maak een aangepast element dat begint met `c-`
- `import { On } from "../js/On.js"` Wordt gebruikt om het binden en ontbinden van gebeurtenissen te implementeren
- `import { fTxt } from "../js/fetch.js"` Ontvang tekst
