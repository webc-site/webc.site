`src/js` सामान्यतः वापरल्या जाणाऱ्या js स्क्रिप्ट खालीलप्रमाणे आहेत

- `import { D, newEl, newLi } from "../js/dom.js"`
  - `D` हे `document` चे संक्षेप आहे
  - `newEl` एक `dom` घटक तयार करतो. तुम्हाला एकाधिक घटक तयार करायचे असल्यास, ते बॅचेसमध्ये तयार करण्यासाठी `newLi` वापरा
- `import { cE } from "../js/dom.js"` एक सानुकूल घटक तयार करा
- `import { On } from "../js/On.js"` इव्हेंट बाइंडिंग आणि अनबाइंडिंग लागू करण्यासाठी वापरले जाते
