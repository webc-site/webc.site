`src/js` Ang kasagarang gigamit nga js scripts mao ang mosunod

- `import { D, newEl, newLi } from "../js/dom.js"`
  - `D` mao ang minubo sa `document`
  - `newEl` nagmugna og `dom` elemento. Kung gusto nimo maghimo daghang mga elemento, gamita ang `newLi` aron mahimo kini nga mga batch
- `import { cE } from "../js/cE.js"` Paghimo ug custom nga elemento
- `import { On } from "../js/On.js"` Gigamit aron ipatuman ang pagbugkos ug dili pagbugkos sa panghitabo
