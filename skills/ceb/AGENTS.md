Nahuman

Dayon padagana ang ./test.sh nga pagsulay

`./dev.js component name` magsugod sa pagpalambo ug pag-debug sa component, ug ang browser mo-access sa `http://127.0.0.1:8888/component name` para sa debugging (walay `localhost`)

1. Isalikway ang walay sulod nga mga pulong ug matinahuron nga mga pulong
2. Hupti ang teknikal nga kinauyokan nga mga pulong
3. Ang code ug mga agianan nagpabilin nga pareho
4. Ang panguna nga mga lakang ug mga pasidaan sa problema nga nanginahanglan pagrepaso ug kumpirmasyon sa tiggamit kay gi-output sa pino nga bernakular.
