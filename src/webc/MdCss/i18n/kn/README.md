# ಓದುವ ಅನುಭವಕ್ಕಾಗಿ ಮುದ್ರಣಕಲೆ ಶೈಲಿಗಳನ್ನು ಹೊಂದುವಂತೆ ಮಾಡಲಾಗಿದೆ

- **ಮೂಲ ಟೈಪ್‌ಸೆಟ್ಟಿಂಗ್**: ಶೀರ್ಷಿಕೆಗಳು, ಪ್ಯಾರಾಗಳು, ದಪ್ಪ, ಇಟಾಲಿಕ್ಸ್, ಸ್ಟ್ರೈಕ್‌ಥ್ರೂ, ಇನ್‌ಲೈನ್ ಕೋಡ್, ಚಿತ್ರಗಳು ಮತ್ತು ವಿಭಜಿಸುವ ರೇಖೆಗಳಿಗೆ ಹೊಂದಿಕೊಳ್ಳಿ.
- **ಲಿಂಕ್ ಅನಿಮೇಷನ್**: ತೂಗಾಡುತ್ತಿರುವಾಗ ಅಂಡರ್‌ಲೈನ್ ಅನಿಮೇಷನ್ ಅನ್ನು ಪ್ರದರ್ಶಿಸಿ.
- **ಪ್ರಾಜೆಕ್ಟ್ ಪಟ್ಟಿ**: ಕ್ರಮಿಸದ ಪಟ್ಟಿಗಳು ಬುಲೆಟ್‌ಗಳನ್ನು ಬಳಸುತ್ತವೆ, ಆರ್ಡರ್ ಮಾಡಿದ ಪಟ್ಟಿಗಳು ದುಂಡಾದ ಹಿನ್ನೆಲೆ ಸರಣಿ ಸಂಖ್ಯೆಗಳನ್ನು ಬಳಸುತ್ತವೆ ಮತ್ತು ಬಹು-ಹಂತದ ಗೂಡುಕಟ್ಟುವಿಕೆಯನ್ನು ಬೆಂಬಲಿಸುತ್ತವೆ.
- **ಕಾರ್ಯ ಪಟ್ಟಿ**: ಚೆಕ್ ಬಾಕ್ಸ್ ಶೈಲಿಯನ್ನು ಒದಗಿಸುತ್ತದೆ, ಮಾಡಬೇಕಾದ ಮತ್ತು ಪೂರ್ಣಗೊಂಡ ಸ್ಥಿತಿಯನ್ನು ಬೆಂಬಲಿಸುತ್ತದೆ.
- **ಎಚ್ಚರಿಕೆ ಸಲಹೆಗಳು**: ಐದು ರೀತಿಯ ಎಚ್ಚರಿಕೆ ಬಾಕ್ಸ್‌ಗಳನ್ನು ಬೆಂಬಲಿಸುತ್ತದೆ: ಸೂಚನೆ, ಸಲಹೆ, ಪ್ರಮುಖ, ಎಚ್ಚರಿಕೆ ಮತ್ತು ಗಮನ.
- **ಟೇಬಲ್‌ಗಳು ಮತ್ತು ಕೋಡ್**: ಟೇಬಲ್ ರೋ ಹೋವರ್ ಹೈಲೈಟ್ ಮಾಡುವುದು ಮತ್ತು ನೆರಳುಗಳು ಮತ್ತು ಗ್ರೇಡಿಯಂಟ್ ಹಿನ್ನೆಲೆಗಳೊಂದಿಗೆ ಕೋಡ್ ಬ್ಲಾಕ್‌ಗಳನ್ನು ಒದಗಿಸುತ್ತದೆ.

## ಡೆಮೊ ಬಳಸಿ

```html
<div class="Md">
  <h1>ಶೀರ್ಷಿಕೆ</h1>
  <p>ಪ್ಯಾರಾಗ್ರಾಫ್ ಪಠ್ಯ ಮತ್ತು <a href="#">ಲಿಂಕ್</a></p>
  <hr>
  <h2>ಎರಡನೇ ಹಂತದ ಶೀರ್ಷಿಕೆ</h2>
  <ul><li><input type="checkbox" checked="" disabled=""> ಕಾರ್ಯ ಐಟಂ</li></ul>
  <blockquote class="q note">
    <b class="qt"><i class="qi"></i>`<p>ಟಿಪ್ಪಣಿ ವಿಷಯ</p>
  </blockquote>
</div>
```

## ಶೈಲಿ ವರ್ಗ

`.Md` ಕಂಟೈನರ್ ಅಡಿಯಲ್ಲಿ HTML ಅಂಶಗಳಿಗೆ ಶೈಲಿಗಳನ್ನು ಅನ್ವಯಿಸಲಾಗುತ್ತದೆ.

### ಎಚ್ಚರಿಕೆ ಬಾಕ್ಸ್ ಶೈಲಿ

ಎಚ್ಚರಿಕೆ ಬಾಕ್ಸ್ ಅನ್ನು ಟಾಗಲ್ ಮಾಡಲು `.q` ವರ್ಗದ ಹೆಸರಿನೊಂದಿಗೆ `blockquote` ಅಂಶದಲ್ಲಿ ಕೆಳಗಿನ ವರ್ಗ ಹೆಸರನ್ನು ಸೇರಿಸಿ:

- `note`: ಟಿಪ್ಪಣಿಗಳು
- `tip`: ಸಲಹೆ
- `important`: ಪ್ರಮುಖ
- `warning`: ಎಚ್ಚರಿಕೆ
- `caution`: ಗಮನಿಸಿ
