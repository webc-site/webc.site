# ტიპოგრაფიის სტილები ოპტიმიზირებულია კითხვის გამოცდილებისთვის

- **ძირითადი აკრეფა**: შეეგუეთ სათაურებს, აბზაცებს, თამამს, დახრილს, ხაზგასმულ კოდს, სურათებს და გამყოფ ხაზებს.
- **ბმულის ანიმაცია**: ხაზგასმული ანიმაციის ჩვენება ჰოვერირებისას.
- **პროექტების სია**: შეურიგებელი სიები იყენებენ ბურთებს, მოწესრიგებული სიები იყენებენ დამრგვალებულ ფონურ სერიულ ნომრებს და მხარს უჭერენ მრავალ დონის ბუდეს.
- **სამუშაოების სია**: გთავაზობთ გამშვები ველის სტილს, მხარს უჭერს სამუშაოს დასრულებულ სტატუსს.
- **გაფრთხილების რჩევები**: მხარს უჭერს ხუთი ტიპის გამაფრთხილებელ ველს: შენიშვნა, რჩევა, მნიშვნელოვანი, გაფრთხილება და ყურადღება.
- **ცხრილები და კოდი**: უზრუნველყოფს ცხრილის მწკრივის ხაზგასმას და კოდის ბლოკებს ჩრდილებით და გრადიენტური ფონით.

## გამოიყენეთ დემო ვერსია

```html
<div class="Md">
  <h1>სათაური</h1>
  <p>აბზაცის ტექსტი და <a href="#">ბმული</a></p>
  <hr>
  <h2>მეორე დონის სათაური</h2>
  <ul><li><input type="checkbox" checked="" disabled=""> დავალების ერთეული</li></ul>
  <blockquote class="q note">
    <b class="qt"><i class="qi"></i>`<p>შენიშვნის შინაარსი</p>
  </blockquote>
</div>
```

## სტილის კლასი

სტილები გამოიყენება HTML ელემენტებზე `.Md` კონტეინერის ქვეშ.

### გაფრთხილების ყუთის სტილი

დაამატეთ კლასის შემდეგი სახელი `blockquote` ელემენტზე `.q` კლასის სახელით, რათა გადართოთ გაფრთხილების ველი:

- `note`: შენიშვნები
- `tip`: რჩევა
- `important`: მნიშვნელოვანია
- `warning`: გაფრთხილება
- `caution`: შენიშვნა
