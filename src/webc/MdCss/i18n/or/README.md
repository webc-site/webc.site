## ଶ style ଳୀ ଶ୍ରେଣୀ

ଶ b ଳୀଗୁଡିକ HTML ଉପାଦାନଗୁଡ଼ିକରେ `.Md` ପାତ୍ରରେ ପ୍ରୟୋଗ କରାଯାଏ |

### ଆଲର୍ଟ ବାକ୍ସ ଶ style ଳୀ |

ଆଲର୍ଟ ବାକ୍ସକୁ ଟୋଗଲ୍ କରିବାକୁ `.q` ଶ୍ରେଣୀ ନାମ ସହିତ `blockquote` ଉପାଦାନରେ ନିମ୍ନ ଶ୍ରେଣୀ ନାମ ଯୋଡନ୍ତୁ:

- `note`: ଟିପ୍ପଣୀ |
- `tip`: ଟିପ୍ପଣୀ |
- `important`: ଗୁରୁତ୍ୱପୂର୍ଣ୍ଣ |
- `warning`: ଚେତାବନୀ |
- `caution`: ଟିପ୍ପଣୀ |
