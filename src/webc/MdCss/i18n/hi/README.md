# पढ़ने के अनुभव के लिए अनुकूलित टाइपोग्राफी शैलियाँ

- **बुनियादी टाइपसेटिंग**: शीर्षक, पैराग्राफ, बोल्ड, इटैलिक, स्ट्राइकथ्रू, इनलाइन कोड, चित्र और विभाजन रेखाओं के अनुकूल।
- **लिंक एनीमेशन**: होवर करते समय अंडरलाइन एनीमेशन प्रदर्शित करें।
- **प्रोजेक्ट सूची**: अव्यवस्थित सूची गोलियों का उपयोग करती है, क्रमित सूची गोलाकार पृष्ठभूमि क्रमांक का उपयोग करती है, और बहु-स्तरीय नेस्टिंग का समर्थन करती है।
- **कार्य सूची**: चेक बॉक्स शैली प्रदान करता है, कार्य करने और पूर्ण स्थिति का समर्थन करता है।
- **चेतावनी युक्तियाँ**: पाँच प्रकार के चेतावनी बक्सों का समर्थन करता है: नोट, टिप, महत्वपूर्ण, चेतावनी और ध्यान।
- **टेबल्स और कोड**: टेबल पंक्ति होवर हाइलाइटिंग, और छाया और ग्रेडिएंट पृष्ठभूमि के साथ कोड ब्लॉक प्रदान करता है।

## डेमो का प्रयोग करें

```html
<div class="Md">
  <h1>शीर्षक</h1>
  <p>अनुच्छेद पाठ और <a href="#">लिंक</a></p>
  <hr>
  <h2>दूसरे स्तर का शीर्षक</h2>
  <ul><li><input type="checkbox" checked="" disabled=""> कार्य आइटम</li></ul>
  <blockquote class="q note">
    <b class="qt"><i class="qi"></i>`<p>सामग्री नोट करें</p>
  </blockquote>
</div>
```

## शैली वर्ग

शैलियाँ `.Md` कंटेनर के अंतर्गत HTML तत्वों पर लागू की जाती हैं।

### अलर्ट बॉक्स शैली

अलर्ट बॉक्स को टॉगल करने के लिए `blockquote` तत्व पर `.q` क्लास नाम के साथ निम्न क्लास नाम जोड़ें:

- `note`: नोट्स
- `tip`: टिप
- `important`: महत्वपूर्ण
- `warning`: चेतावनी
- `caution`: नोट
