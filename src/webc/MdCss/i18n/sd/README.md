# ٽائپوگرافي اسلوب کي پڙهڻ جي تجربي لاءِ بهتر بڻايو ويو

- **بنيادي ٽائيپ سيٽنگ**: عنوانن، پيراگرافن، بولڊ، اطالوي، اسٽرائڪ تھرو، ان لائن ڪوڊ، تصويرن ۽ ورهائڻ واري لائنن کي ترتيب ڏيو.
- **لنڪ اينيميشن**: انڊر لائن اينيميشن ڏيکاريو جڏهن هور هجي.
- **پروجيڪٽ لسٽ**: غير ترتيب ڏنل فهرستون گوليون استعمال ڪن ٿيون، ترتيب ڏنل فهرستون گول پس منظر سيريل نمبر استعمال ڪن ٿيون، ۽ گھڻن سطحي نيسٽنگ کي سپورٽ ڪن ٿيون.
- **ٽاسڪ لسٽ**: چيڪ باڪس جو انداز مهيا ڪري ٿو، ڪم ڪرڻ ۽ مڪمل حالت کي سپورٽ ڪري ٿو.
- ** ڊيڄاريندڙ ٽوٽڪا**: پنجن قسمن جي ڊيڄاريندڙ باڪس کي سپورٽ ڪري ٿو: نوٽ، ٽپ، اھم، ڊيڄاريندڙ، ۽ ڌيان.
- **ٽيبل ۽ ڪوڊ**: مهيا ڪري ٿو ٽيبل جي قطار کي نمايان ڪرڻ، ۽ ڪوڊ بلاڪ کي پاڇين ۽ گريجوئيٽ پس منظر سان.

## ڊيمو استعمال ڪريو

```html
<div class="Md">
  <h1>عنوان</h1>
  <p>پيراگراف متن ۽ <a href="#">لنڪ</a></p>
  <hr>
  <h2>ٻئي سطح جو عنوان</h2>
  <ul><li><input type="checkbox" checked="" disabled=""> ٽاسڪ آئٽم</li></ul>
  <blockquote class="q note">
    <b class="qt"><i class="qi"></i>`<p>نوٽ مواد</p>
  </blockquote>
</div>
```

## انداز ڪلاس

طرزون HTML عناصر تي `.Md` ڪنٽينر جي تحت لاڳو ٿين ٿيون.

### الرٽ باڪس انداز

ھيٺ ڏنل ڪلاس جو نالو شامل ڪريو `blockquote` عنصر تي `.q` ڪلاس جي نالي سان الرٽ باڪس کي ٽوگل ڪرڻ لاءِ:

- `note`: نوٽس
- `tip`: ٽپ
- `important`: اهم
- `warning`: ڊيڄاريندڙ
- `caution`: نوٽ
