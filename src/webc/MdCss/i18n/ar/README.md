# أنماط الطباعة الأمثل لتجربة القراءة

- **التنضيد الأساسي**: التكيف مع العناوين والفقرات والخط العريض والمائل ويتوسطه خط والرمز المضمن والصور والخطوط الفاصلة.
- **رسوم الارتباط المتحركة**: عرض الرسوم المتحركة التي تحتها خط عند التمرير.
- **قائمة المشاريع**: تستخدم القوائم غير المرتبة الرموز النقطية، وتستخدم القوائم المرتبة أرقامًا تسلسلية مستديرة في الخلفية، وتدعم التداخل متعدد المستويات.
- **قائمة المهام**: توفر نمط خانة الاختيار، وتدعم المهام المطلوبة والحالة المكتملة.
- **نصائح تحذيرية**: يدعم خمسة أنواع من مربعات التحذير: ملاحظة، وتلميح، ومهم، وتحذير، وانتباه.
- **الجداول والتعليمات البرمجية**: توفر تمييزًا لصفوف الجدول وكتل التعليمات البرمجية ذات الظلال والخلفيات المتدرجة.

## استخدم العرض التوضيحي

```html
<div class="Md">
  <h1>عنوان</h1>
  <p>نص الفقرة و<a href="#">الرابط</a></p>
  <hr>
  <h2>عنوان المستوى الثاني</h2>
  <ul><li><input type="checkbox" checked="" disabled=""> عنصر المهمة</li></ul>
  <blockquote class="q note">
    <b class="qt"><i class="qi"></i>`<p>محتوى الملاحظة</p>
  </blockquote>
</div>
```

## فئة النمط

يتم تطبيق الأنماط على عناصر HTML ضمن الحاوية `.Md`.

### نمط صندوق التنبيه

أضف اسم الفئة التالي إلى العنصر `blockquote` مع اسم الفئة `.q` لتبديل مربع التنبيه:

- `note`: ملاحظات
- `tip`: نصيحة
- `important`: مهم
- `warning`: تحذير
- <b معرف = "1">: ملحوظة
