# סגנונות טיפוגרפיה מותאמים לחוויית קריאה

- **קביעת כתיבה בסיסית**: התאימו לכותרות, פסקאות, מודגשות, נטוי, קו חוצה, קוד מוטבע, תמונות וקווי הפרדה.
- **אנימציית קישור**: הצג אנימציית קו תחתון בעת ​​ריחוף.
- **רשימת פרויקטים**: רשימות לא מסודרות משתמשות בתבליטים, רשימות מסודרות משתמשות במספרי רקע סידוריים מעוגלים ותומכות בקינון רב-רמות.
- **רשימת משימות**: מספקת סגנון תיבת סימון, תומך במטלות ובסטטוס הושלם.
- **טיפים אזהרה**: תומך בחמישה סוגים של תיבות אזהרה: הערה, טיפ, חשוב, אזהרה ותשומת לב.
- **טבלאות וקוד**: מספק הדגשת ריחוף של שורת טבלה, ובלוקי קוד עם צללים ורקע הדרגתי.

## השתמש בהדגמה

```html
<div class="Md">
  <h1>כּוֹתֶרֶת</h1>
  <p>טקסט פסקה ו<a href="#">קישור</a></p>
  <hr>
  <h2>כותרת ברמה השנייה</h2>
  <ul><li><input type="checkbox" checked="" disabled=""> פריט משימה</li></ul>
  <blockquote class="q note">
    <b class="qt"><i class="qi"></i>`<p>תוכן הערה</p>
  </blockquote>
</div>
```

## שיעור סגנון

סגנונות מוחלים על רכיבי HTML מתחת למיכל `.Md`.

### סגנון תיבת התראה

הוסף את שם המחלקה הבא ברכיב `blockquote` עם שם המחלקה `.q` כדי להחליף את תיבת ההתראה:

- `note`: הערות
- `tip`: טיפ
- `important`: חשוב
- `warning`: אזהרה
- `caution`: הערה
