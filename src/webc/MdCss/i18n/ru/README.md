# Стили оформления, оптимизированные для удобства чтения

- **Базовый набор**. Адаптируйте заголовки, абзацы, жирный шрифт, курсив, зачеркивание, встроенный код, изображения и разделительные линии.
- **Анимация ссылки**: отображение анимации подчеркивания при наведении курсора.
- **Список проектов**. В неупорядоченных списках используются маркеры, в упорядоченных — округленные фоновые серийные номера, а также поддерживается многоуровневая вложенность.
- **Список задач**: обеспечивает стиль флажка, поддерживает список дел и статус завершения.
- **Советы по предупреждениям**. Поддерживается пять типов окон с предупреждениями: «Примечание», «Совет», «Важно», «Предупреждение» и «Внимание».
- **Таблицы и код**: обеспечивает подсветку строк таблицы при наведении, а также блоки кода с тенями и градиентным фоном.

## Используйте демо-версию

```html
<div class="Md">
  <h1>заголовок</h1>
  <p>Текст абзаца и <a href="#">ссылка</a></p>
  <hr>
  <h2>Название второго уровня</h2>
  <ul><li><input type="checkbox" checked="" disabled=""> Элемент задачи</li></ul>
  <blockquote class="q note">
    <b class="qt"><i class="qi"></i>`<p>Содержание примечания</p>
  </blockquote>
</div>
```

## класс стиля

Стили применяются к элементам HTML в контейнере `.Md`.

### Стиль окна оповещений

Добавьте следующее имя класса в элемент `blockquote` с именем класса `.q`, чтобы переключить окно предупреждения:

- `note`: Примечания
- `tip`: Совет
- `important`: Важно
- `warning`: предупреждение
- `caution`: Примечание.
