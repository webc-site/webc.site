# Տպագրության ոճերը օպտիմիզացված են ընթերցանության փորձի համար

- **Հիմնական շարադրություն**. Հարմարվել վերնագրերին, պարբերություններին, թավերին, շեղատառերին, գծանշմանը, ներդիր կոդի, նկարների և բաժանարար գծերի:
- **Հղման անիմացիա**. ցուցադրել ընդգծված անիմացիան սավառնելիս:
- **Նախագծերի ցուցակ**. չդասավորվող ցուցակներն օգտագործում են պարբերակներ, պատվիրված ցուցակները օգտագործում են կլորացված ֆոնային սերիական համարներ և աջակցում են բազմաստիճան բույն:
- **Առաջադրանքների ցանկ**. Ապահովում է վանդակի ոճը, աջակցում է անելիքների և ավարտված կարգավիճակին:
- **Նախազգուշացման խորհուրդներ**. Աջակցում է հինգ տեսակի նախազգուշացման տուփեր՝ Ծանոթագրություն, Հուշում, Կարևոր, Զգուշացում և Ուշադրություն:
- **Սեղաններ և ծածկագիր**. Ապահովում է աղյուսակի տողերի սավառնման ընդգծում և կոդի բլոկներ ստվերներով և գրադիենտ ֆոնով:

## Օգտագործեք ցուցադրությունը

```html
<div class="Md">
  <h1>վերնագիր</h1>
  <p>Պարբերության տեքստ և <a href="#">հղում</a></p>
  <hr>
  <h2>Երկրորդ մակարդակի կոչում</h2>
  <ul><li><input type="checkbox" checked="" disabled=""> Առաջադրանքի տարր</li></ul>
  <blockquote class="q note">
    <b class="qt"><i class="qi"></i>`<p>Նշման բովանդակություն</p>
  </blockquote>
</div>
```

## ոճի դաս

Ոճերը կիրառվում են HTML տարրերի վրա `.Md` կոնտեյների տակ:

### Զգուշացման տուփի ոճը

Ավելացրե՛ք հետևյալ դասի անունը `blockquote` տարրի վրա `.q` դասի անվան հետ` ահազանգերի տուփը փոխարկելու համար.

- `note`: Նշումներ
- `tip`: Հուշում
- `important`: Կարևոր է
- `warning`: զգուշացում
- `caution`: Նշում
