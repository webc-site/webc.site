# ຮູບແບບການພິມຖືກປັບໃຫ້ເໝາະສົມສຳລັບປະສົບການການອ່ານ

- **ການ​ຈັດ​ປະ​ເພດ​ພື້ນ​ຖານ**​: ປັບ​ຕົວ​ເຂົ້າ​ກັບ​ຫົວ​ຂໍ້​, ຫຍໍ້​ຫນ້າ​, ຕົວ​ຫນາ​, ຕົວ​ອຽງ​, ຂີດ​ຜ່ານ​, ລະ​ຫັດ​ໃນ​ແຖວ​, ຮູບ​ພາບ​ແລະ​ເສັ້ນ​ແບ່ງ​ປັນ​.
- **Link animation**: ສະແດງພາບເຄື່ອນໄຫວທີ່ຂີດກ້ອງເມື່ອເລື່ອນ.
- **ລາຍຊື່ໂຄງການ**: ລາຍຊື່ທີ່ບໍ່ໄດ້ຈັດລຳດັບໃຊ້ຫົວຂໍ້ຍ່ອຍ, ລາຍການຕາມລຳດັບຈະໃຊ້ເລກລຳດັບພື້ນຫຼັງແບບກົມ, ແລະຮອງຮັບການຕັ້ງຮັງຫຼາຍລະດັບ.
- ** ບັນ​ຊີ​ລາຍ​ການ​ວຽກ​ງານ **​: ສະ​ຫນອງ​ຮູບ​ແບບ​ປ່ອງ​ຢ້ຽມ​, ສະ​ຫນັບ​ສະ​ຫນູນ​ການ​ເຮັດ​ແລະ​ສະ​ຖາ​ນະ​ພາບ​ສໍາ​ເລັດ​.
- ** ຄໍາ​ແນະ​ນໍາ​ການ​ເຕືອນ​ໄພ **​: ສະ​ຫນັບ​ສະ​ຫນູນ​ຫ້າ​ປະ​ເພດ​ຂອງ​ປ່ອງ​ເຕືອນ​: ຫມາຍ​ເຫດ​, ຄໍາ​ແນະ​ນໍາ​, ສໍາ​ຄັນ​, ຄໍາ​ເຕືອນ​, ແລະ​ການ​ເອົາ​ໃຈ​ໃສ່​.
- **ຕາຕະລາງ ແລະລະຫັດ**: ສະໜອງການເນັ້ນໃສ່ແຖວແຖວຕາຕະລາງ, ແລະຕົວບລັອກລະຫັດທີ່ມີເງົາ ແລະພື້ນຫຼັງສີ.

## ໃຊ້ຕົວຢ່າງ

```html
<div class="Md">
  <h1>ຫົວຂໍ້</h1>
  <p>ຂໍ້ຄວາມຫຍໍ້ໜ້າ ແລະ <a href="#">ລິ້ງ</a></p>
  <hr>
  <h2>ຫົວຂໍ້ລະດັບທີສອງ</h2>
  <ul><li><input type="checkbox" checked="" disabled="">ລາຍການໜ້າວຽກ</li></ul>
  <blockquote class="q note">
    <b class="qt"><i class="qi"></i>`<p>ເນື້ອໃນບັນທຶກ</p>
  </blockquote>
</div>
```

## ຫ້ອງຮຽນແບບ

ຮູບແບບຖືກນຳໃຊ້ກັບອົງປະກອບ HTML ພາຍໃຕ້ກ່ອງບັນຈຸ `.Md`.

### ຮູບແບບກ່ອງແຈ້ງເຕືອນ

ເພີ່ມ​ຊື່​ຊັ້ນ​ຮຽນ​ຕໍ່​ໄປ​ນີ້​ໃນ​ອົງ​ປະ​ກອບ `blockquote` ດ້ວຍ `.q`ຊື່​ຫ້ອງ​ເພື່ອ​ສະ​ຫຼັບ​ປ່ອງ​ການ​ເຕືອນ​:

- `note`: ໝາຍເຫດ
- `tip`: ເຄັດລັບ
- `important`: ສຳຄັນ
- `warning`: ຄໍາເຕືອນ
- `caution`: ໝາຍເຫດ
