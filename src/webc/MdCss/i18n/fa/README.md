# سبک های تایپوگرافی برای تجربه خواندن بهینه شده است

- ** حروفچینی اساسی **: با عنوان، پاراگراف، پررنگ، مورب، خط خطی، کد درون خطی، تصاویر و خطوط تقسیم تطبیق دهید.
- ** انیمیشن پیوند **: انیمیشن زیر خطی را هنگام شناور نمایش می دهد.
- **لیست پروژه**: لیست های نامرتب از گلوله ها استفاده می کنند، لیست های مرتب شده از شماره سریال پس زمینه گرد استفاده می کنند و از تودرتوی چند سطحی پشتیبانی می کنند.
- **فهرست وظایف**: سبک چک باکس را ارائه می دهد، از وضعیت کارهای انجام شده و تکمیل شده پشتیبانی می کند.
- ** نکات هشدار **: از پنج نوع جعبه هشدار پشتیبانی می کند: توجه، نکته، مهم، هشدار و توجه.
- **جدول و کد**: برجسته کردن شناور ردیف جدول و بلوک های کد با سایه ها و پس زمینه گرادیان را ارائه می دهد.

## از دمو استفاده کنید

```html
<div class="Md">
  <h1>عنوان</h1>
  <p>متن پاراگراف و <a href="#">پیوند</a></p>
  <hr>
  <h2>عنوان سطح دوم</h2>
  <ul><li><input type="checkbox" checked="" disabled=""> مورد وظیفه</li></ul>
  <blockquote class="q note">
    <b class="qt"><i class="qi"></i>`<p>محتوای یادداشت</p>
  </blockquote>
</div>
```

## کلاس سبک

سبک‌ها به عناصر HTML در زیر ظرف `.Md` اعمال می‌شوند.

### سبک جعبه هشدار

نام کلاس زیر را به عنصر `blockquote` با نام کلاس `.q` اضافه کنید تا کادر هشدار را تغییر دهید:

- `note`: یادداشت ها
- `tip`: نکته
- `important`: مهم است
- `warning`: هشدار
- `caution`: توجه
