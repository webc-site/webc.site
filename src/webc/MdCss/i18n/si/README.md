# කියවීමේ අත්දැකීම සඳහා මුද්‍රණ ශෛලීන් ප්‍රශස්ත කර ඇත

- **මූලික අකුරු සැකසීම**: මාතෘකා, ඡේද, තදකුරු, ඇල අකුරු, වර්ජන, පේළිගත කේතය, පින්තූර සහ බෙදුම් රේඛා වලට අනුගත වන්න.
- **සබැඳි සජීවිකරණය**: සැරිසරන විට යටින් ඉරි සජීවිකරණය පෙන්වන්න.
- **ව්‍යාපෘති ලැයිස්තුව**: ඇණවුම් නොකළ ලැයිස්තු බුලට් භාවිත කරයි, ඇණවුම් කළ ලැයිස්තු වටකුරු පසුබිම් අනුක්‍රමික අංක භාවිත කරයි, සහ බහු මට්ටමේ කැදැල්ලට සහාය වේ.
- **කාර්ය ලැයිස්තුව**: පිරික්සුම් කොටු විලාසය සපයයි, කළ යුතු සහ සම්පූර්ණ කළ තත්ත්වයට සහාය වේ.
- **අනතුරු ඇඟවීමේ ඉඟි**: අනතුරු ඇඟවීමේ පෙට්ටි වර්ග පහකට සහය දක්වයි: සටහන, ඉඟිය, වැදගත්, අනතුරු ඇඟවීම සහ අවධානය.
- **වගු සහ කේතය**: වගු පේළි හෝවර් උද්දීපනය සහ සෙවනැලි සහ අනුක්‍රමණ පසුබිම් සහිත කේත කුට්ටි සපයයි.

## Demo භාවිතා කරන්න

```html
<div class="Md">
  <h1>මාතෘකාව</h1>
  <p>ඡේද පෙළ සහ <a href="#">සබැඳිය</a></p>
  <hr>
  <h2>දෙවන මට්ටමේ මාතෘකාව</h2>
  <ul><li><input type="checkbox" checked="" disabled=""> කාර්ය අයිතමය</li></ul>
  <blockquote class="q note">
    <b class="qt"><i class="qi"></i>`<p>අන්තර්ගතය සටහන් කරන්න</p>
  </blockquote>
</div>
```

## ශෛලිය පන්තිය

`.Md` බහාලුම යටතේ HTML මූලද්‍රව්‍ය සඳහා මෝස්තර යොදනු ලැබේ.

### ඇඟවීම් පෙට්ටි විලාසය

ඇඟවීම් පෙට්ටිය ටොගල් කිරීමට `blockquote` පන්තියේ නම සමඟ `blockquote` පන්තියේ නම එක් කරන්න:

- `note`: සටහන්
- `tip`: ඉඟිය
- `important`: වැදගත්
- `warning`: අනතුරු ඇඟවීම
- `caution`: සටහන
