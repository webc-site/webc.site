# စာဖတ်ခြင်းအတွေ့အကြုံအတွက် အကောင်းဆုံး စာစီစာရိုက်ပုံစံများ

- **အခြေခံအမျိုးအစားများသတ်မှတ်ခြင်း**- ခေါင်းစဉ်များ၊ စာပိုဒ်များ၊ ရဲရင့်မှု၊ စာလုံးစောင်းများ၊ အကွက်များ၊ အတွင်းကုဒ်များ၊ ရုပ်ပုံများနှင့် မျဉ်းကြောင်းများကို လိုက်လျောညီထွေဖြစ်အောင်ပါ။
- **လင့်ခ် ကာတွန်း**- ပျံဝဲနေချိန်တွင် မျဉ်းသားထားသော ကာတွန်းပုံကို ပြသပါ။
- **ပရောဂျက်စာရင်း**- အစီအစဥ်မထားသောစာရင်းများသည် ကျည်ဆန်များကိုအသုံးပြုသည်၊ မှာယူထားသောစာရင်းများသည် လုံးဝန်းသောနောက်ခံအမှတ်စဉ်နံပါတ်များကို အသုံးပြုကာ အဆင့်ပေါင်းများစွာ nesting ကို ပံ့ပိုးပေးသည်။
- ** Task List**- check box စတိုင်ကို ပံ့ပိုးပေးသည်၊ လုပ်စရာနှင့် ပြီးမြောက်မှု အခြေအနေတို့ကို ပံ့ပိုးပေးသည်။
- **သတိပေးချက်**- သတိပေးချက်သေတ္တာ အမျိုးအစားငါးခုကို ပံ့ပိုးသည်- မှတ်ချက်၊ အကြံပြုချက်၊ အရေးကြီး၊ သတိပေးချက်နှင့် အာရုံစူးစိုက်မှု။
- **Tables နှင့် Code**- စားပွဲအတန်းကို မီးမောင်းထိုးပြပြီး အရိပ်များနှင့် gradient နောက်ခံများဖြင့် ကုဒ်တုံးများကို ပံ့ပိုးပေးသည်။

## ဒီမိုကိုသုံးပါ။

```html
<div class="Md">
  <h1>ခေါင်းစဉ်</h1>
  <p>စာပိုဒ်စာသားနှင့် <a href="#">လင့်ခ်</a></p>
  <hr>
  <h2>ဒုတိယအဆင့် ခေါင်းစဉ်</h2>
  <ul><li><input type="checkbox" checked="" disabled=""> လုပ်ဆောင်စရာ အကြောင်းအရာ</li></ul>
  <blockquote class="q note">
    <b class="qt"><i class="qi"></i>`<p>မှတ်ချက် အကြောင်းအရာ</p>
  </blockquote>
</div>
```

## စတိုင်အတန်း

စတိုင်များကို `.Md` ကွန်တိန်နာအောက်ရှိ HTML ဒြပ်စင်များတွင် အသုံးပြုသည်။

### သတိပေးချက်သေတ္တာပုံစံ

သတိပေးချက်အကွက်ကို ပြောင်းရန် `.q`အတန်းအမည်ဖြင့် အောက်ဖော်ပြပါ အတန်းအမည်ကို `blockquote`ဒြပ်စင်ပေါ်တွင် ထည့်ပါ-

- `note`- မှတ်စုများ
- `tip`- အကြံပြုချက်
- `important`- အရေးကြီးသည်။
- `warning`- သတိပေးချက်
- `caution`- မှတ်ချက်
