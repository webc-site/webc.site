# Стылі друкарні, аптымізаваныя для чытання

- **Асноўная вёрстка**: адаптацыя да загалоўкаў, абзацаў, тлустага шрыфта, курсіву, закрэслівання, убудаванага кода, малюнкаў і раздзяляльных ліній.
- **Анімацыя спасылкі**: адлюстроўваць анімацыю падкрэслівання пры навядзенні курсора.
- **Спіс праектаў**: у неўпарадкаваных спісах выкарыстоўваюцца маркеры, у упарадкаваных спісах выкарыстоўваюцца акругленыя фонавыя серыйныя нумары і падтрымліваецца шматузроўневае ўкладанне.
- **Спіс задач**: забяспечвае стыль сцяжка, падтрымлівае спіс спраў і стан выканання.
- **Папярэджанні**: падтрымлівае пяць тыпаў папярэджанняў: Заўвага, Савет, Важна, Папярэджанне і Увага.
- **Табліцы і код**: забяспечвае вылучэнне радкоў табліцы пры навядзенні курсора і блокі кода з ценямі і градыентным фонам.

## Выкарыстоўвайце дэма

```html
<div class="Md">
  <h1>назва</h1>
  <p>Тэкст абзаца і <a href="#">спасылка</a></p>
  <hr>
  <h2>Назва другога ўзроўню</h2>
  <ul><li><input type="checkbox" checked="" disabled=""> Пункт задачы</li></ul>
  <blockquote class="q note">
    <b class="qt"><i class="qi"></i>`<p>Змест нататкі</p>
  </blockquote>
</div>
```

## клас стылю

Стылі прымяняюцца да элементаў HTML у кантэйнеры `.Md`.

### Стыль скрынкі абвестак

Дадайце наступнае імя класа ў элемент `blockquote` з імем класа `.q`, каб пераключыць поле папярэджання:

- `note`: Заўвагі
- `tip`: Парада
- `important`: Важна
- `warning`: папярэджанне
- `caution`: Заўвага
