# Στυλ τυπογραφίας βελτιστοποιημένα για εμπειρία ανάγνωσης

- **Βασική στοιχειοθεσία**: Προσαρμογή σε τίτλους, παραγράφους, έντονους χαρακτήρες, πλάγιους χαρακτήρες, διαγράμμιση, ενσωματωμένο κώδικα, εικόνες και διαχωριστικές γραμμές.
- **Κινούμενη εικόνα σύνδεσης**: Εμφάνιση κινούμενων εικόνων με υπογράμμιση όταν τοποθετείτε το δείκτη του ποντικιού.
- **Λίστα έργων**: Οι μη ταξινομημένες λίστες χρησιμοποιούν κουκκίδες, οι ταξινομημένες λίστες χρησιμοποιούν στρογγυλεμένους σειριακούς αριθμούς φόντου και υποστηρίζουν ένθεση πολλαπλών επιπέδων.
- **Λίστα εργασιών**: Παρέχει στυλ πλαισίου ελέγχου, υποστηρίζει την κατάσταση υποχρεώσεων και ολοκλήρωσης.
- **Συμβουλές προειδοποίησης**: Υποστηρίζει πέντε τύπους πλαισίων προειδοποίησης: Σημείωση, Συμβουλή, Σημαντικό, Προειδοποίηση και Προσοχή.
- **Πίνακες και κώδικας**: Παρέχει επισήμανση του δείκτη του ποντικιού σε σειρές πίνακα και μπλοκ κώδικα με σκιές και ντεγκραντέ φόντο.

## Χρησιμοποιήστε το demo

```html
<div class="Md">
  <h1>τίτλος</h1>
  <p>Κείμενο παραγράφου και <a href="#">σύνδεσμος</a></p>
  <hr>
  <h2>Τίτλος δεύτερου επιπέδου</h2>
  <ul><li><input type="checkbox" checked="" disabled=""> Στοιχείο εργασίας</li></ul>
  <blockquote class="q note">
    <b class="qt"><i class="qi"></i>`<p>Περιεχόμενο σημείωσης</p>
  </blockquote>
</div>
```

## τάξη στυλ

Τα στυλ εφαρμόζονται σε στοιχεία HTML κάτω από το κοντέινερ `.Md`.

### Στυλ κουτιού ειδοποίησης

Προσθέστε το ακόλουθο όνομα τάξης στο στοιχείο `blockquote` με το όνομα της τάξης `.q` για να αλλάξετε το πλαίσιο ειδοποίησης:

- `note`: Σημειώσεις
- `tip`: Συμβουλή
- `important`: Σημαντικό
- `warning`: προειδοποίηση
- `caution`: Σημείωση
