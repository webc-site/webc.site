# Оқу тәжірибесі үшін оңтайландырылған типография стильдері

- **Негізгі теру**: Тақырыптарға, абзацтарға, қалың, курсивке, сызылған, кірістірілген кодқа, суреттерге және бөлетін жолдарға бейімдеу.
- **Сілтеме анимациясы**: меңзерді апарған кезде астын сызу анимациясын көрсету.
- **Жобалар тізімі**: ретсіз тізімдер таңбалауыштарды пайдаланады, реттелген тізімдер дөңгелектенген фондық сериялық нөмірлерді пайдаланады және көп деңгейлі кірістіруді қолдайды.
- **Тапсырмалар тізімі**: құсбелгі мәнерін береді, істер мен аяқталған күйді қолдайды.
- **Ескерту кеңестері**: ескерту жолағының бес түрін қолдайды: Ескертпе, Кеңес, Маңызды, Ескерту және Назар аударыңыз.
- **Кестелер мен код**: кесте жолын жылжытуды бөлектеуді және көлеңкелер мен градиент фоны бар код блоктарын қамтамасыз етеді.

## Демонстрацияны пайдаланыңыз

```html
<div class="Md">
  <h1>тақырып</h1>
  <p>Абзац мәтіні және <a href="#">сілтеме</a></p>
  <hr>
  <h2>Екінші деңгейдегі тақырып</h2>
  <ul><li><input type="checkbox" checked="" disabled=""> Тапсырма элементі</li></ul>
  <blockquote class="q note">
    <b class="qt"><i class="qi"></i>`<p>Ескертпе мазмұны</p>
  </blockquote>
</div>
```

## стиль сыныбы

Мәнерлер `.Md` контейнерінің астындағы HTML элементтеріне қолданылады.

### Ескерту қорабы стилі

Ескерту жолағын ауыстыру үшін `blockquote` элементіне `.q` сынып атауымен келесі сынып атауын қосыңыз:

- `note`: ескертпелер
- `tip`: Кеңес
- `important`: Маңызды
- `warning`: ескерту
- `caution`: Ескерту
