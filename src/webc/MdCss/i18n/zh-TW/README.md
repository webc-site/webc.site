# 為閱讀體驗優化的排版樣式

- **基礎排版**：適配標題、段落、粗體、斜體、刪除線、行內程式碼、圖片與分割線。
- **連結動效**：懸停時顯示底線動畫。
- **項目清單**：無序號表使用項目符號，有序列表使用圓角背景序號，支援多層巢狀。
- **任務清單**：提供複選框樣式，支援待辦和已辦狀態。
- **警告提示**：支援便籤、提示、重要、警告、注意五種樣式的警告框。
- **表格與程式碼**：提供表格行懸停高亮，以及帶有陰影與漸層背景的程式碼區塊。

## 使用演示

```html
<div class="Md">
  <h1>標題</h1>
  <p>段落文字與 <a href="#">連結</a></p>
  <hr>
  <h2>二級標題</h2>
  <ul><li><input type="checkbox" checked="" disabled=""> 任務項目</li></ul>
  <blockquote class="q note">
    <b class="qt"><i class="qi"></i>`<p>便條提示內容</p>
  </blockquote>
</div>
```

## 樣式類

樣式應用於 `.Md` 容器下的 HTML 元素。

### 警告框樣式

在有 `.q` 類別名稱的 `blockquote` 元素上新增下列類別名稱以切換警告方塊：

- `note`：便籤
- `tip`：提示
- `important`：重要
- `warning`：警告
- `caution`：注意
