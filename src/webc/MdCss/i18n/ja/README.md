# 読書体験に最適化されたタイポグラフィ スタイル

- **基本的な組版**: タイトル、段落、太字、斜体、取り消し線、インライン コード、画像、分割線に適応します。
- **リンクアニメーション**: ホバー時に下線アニメーションを表示します。
- **プロジェクト リスト**: 順序なしリストでは箇条書きが使用され、順序付きリストでは丸められた背景のシリアル番号が使用され、マルチレベルのネストがサポートされます。
- **タスク リスト**: チェック ボックス スタイルを提供し、To Do および完了ステータスをサポートします。
- **警告ヒント**: 5 種類の警告ボックス (メモ、ヒント、重要、警告、注意) をサポートします。
- **テーブルとコード**: テーブル行のホバー強調表示、および影とグラデーションの背景を備えたコード ブロックを提供します。

## デモを使用する

```html
<div class="Md">
  <h1>タイトル</h1>
  <p>段落テキストと<a href="#">リンク</a></p>
  <hr>
  <h2>第二レベルの称号</h2>
  <ul><li><input type="checkbox" checked="" disabled=""> タスクアイテム</li></ul>
  <blockquote class="q note">
    <b class="qt"><i class="qi"></i>`<p>メモの内容</p>
  </blockquote>
</div>
```

## スタイルクラス

スタイルは、`.Md` コンテナの下の HTML 要素に適用されます。

### アラートボックスのスタイル

`.q` クラス名を持つ `blockquote` 要素に次のクラス名を追加して、アラート ボックスを切り替えます。

- `note`: 注記
- `tip`: ヒント
- `important`: 重要
- `warning`: 警告
- `caution`: 注
