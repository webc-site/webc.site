# পঢ়াৰ অভিজ্ঞতাৰ বাবে অনুকূলিত টাইপোগ্ৰাফী শৈলী

- **মূল টাইপছেটিং**: শিৰোনাম, অনুচ্ছেদ, বল্ড, ইটালিক, ষ্ট্ৰাইকথ্ৰু, ইনলাইন ক'ড, ছবি আৰু বিভাজন লাইনৰ সৈতে খাপ খুৱাই লওক।
- **লিংক এনিমেচন**: হ'ভাৰিং কৰাৰ সময়ত আণ্ডাৰলাইন এনিমেচন প্ৰদৰ্শন কৰক।
- **প্ৰকল্প তালিকা**: অক্ৰমিত তালিকাসমূহে বুলেটসমূহ ব্যৱহাৰ কৰে, ক্ৰমবদ্ধ তালিকাসমূহে ঘূৰণীয়া পটভূমি ক্ৰমিক সংখ্যাসমূহ ব্যৱহাৰ কৰে, আৰু বহু-স্তৰীয় নেষ্টিং সমৰ্থন কৰে।
- **কাৰ্য্য তালিকা**: চেক বাকচ শৈলী প্ৰদান কৰে, কৰিবলগীয়া আৰু সম্পূৰ্ণ অৱস্থা সমৰ্থন কৰে।
- **সতৰ্কবাণী টিপছ**: পাঁচ ধৰণৰ সতৰ্কবাণী বাকচ সমৰ্থন কৰে: টোকা, টিপচ্, গুৰুত্বপূৰ্ণ, সতৰ্কবাণী, আৰু মনোযোগ।
- **টেবুল আৰু ক'ড**: টেবুল শাৰী হোভাৰ হাইলাইটিং প্ৰদান কৰে, আৰু ছাঁ আৰু গ্ৰেডিয়েণ্ট পটভূমিৰ সৈতে ক'ড ব্লকসমূহ।

## ডেমো ব্যৱহাৰ কৰক

```html
<div class="Md">
  <h1>অলপ</h1>
  <p>অনুচ্ছেদৰ লিখনী আৰু <a href="#">লিংক</a></p>
  <hr>
  <h2>দ্বিতীয় পৰ্যায়ৰ খিতাপ</h2>
  <ul><li><input type="checkbox" checked="" disabled=""> কাৰ্য্য বস্তু</li></ul>
  <blockquote class="q note">
    <b class="qt"><i class="qi"></i>`<p>সামগ্ৰী মন কৰক</p>
  </blockquote>
</div>
```

## শৈলী শ্ৰেণী

শৈলীসমূহ `.Md` পাত্ৰৰ অন্তৰ্গত HTML উপাদানসমূহত প্ৰয়োগ কৰা হয়।

### সতৰ্ক বাকচ শৈলী

সতৰ্কবাণী বাকচ টগল কৰিবলে `blockquote` উপাদানত `.q` শ্ৰেণী নামৰ সৈতে নিম্নলিখিত শ্ৰেণী নাম যোগ কৰক:

- `note`: টোকাসমূহ
- `tip`: টিপচ্
- `important`: গুৰুত্বপূৰ্ণ
- `warning`: সতৰ্কবাণী
- `caution`: টোকা
