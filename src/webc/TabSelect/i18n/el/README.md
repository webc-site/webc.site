# Καρτέλα με αναπτυσσόμενη επιλογή

- Υποστηρίζει την ενσωμάτωση του αναπτυσσόμενου πλαισίου επιλογής προσαρμοστικού πλάτους (`<c-select>`) στην ετικέτα, το οποίο θα εμφανίζεται μόνο όταν ενεργοποιηθεί η τρέχουσα σελίδα καρτέλας.
- Δομική προδιαγραφή: Το θυγατρικό στοιχείο χρησιμοποιεί `nav > a` ως πλοήγηση ετικέτας και το άμεσο θυγατρικό στοιχείο `b` χρησιμεύει ως περιοχή περιεχομένου.
- Συσχετίζει αυτόματα την ετικέτα `a[value]`, την `a` με την `c-select` (με βάση το χαρακτηριστικό τιμής της ομάδας και την επιλεγμένη τιμή επιλογής) και τον κόμβο περιεχομένου `b[slot]`.
- Υποστηρίζει την αυτόματη παραμονή του ενεργοποιημένου στοιχείου σε `localStorage` μέσω του χαρακτηριστικού `key`.
- Υποστηρίζει την αυτόματη αποθήκευση και επαναφορά ιστορικών υποεπιλογών για κάθε ομάδα σε `localStorage` χρησιμοποιώντας `key + '/' + tab`.
- Υποστηρίζει τον καθορισμό της ενεργής καρτέλας μέσω του χαρακτηριστικού `value`.

## Χρησιμοποιήστε το demo

```html
<c-tab-select key="my-tab-select-key" value="language/js">
  <nav>
    <a value="language">
      <span>γλώσσα</span>
      <c-select>
        <option value="js">JavaScript</option>
        <option value="rs">Rust</option>
      </c-select>
    </a>
    <a value="os">
      <span>σύστημα</span>
      <c-select>
        <option value="mac">macOS</option>
        <option value="win">Windows</option>
      </c-select>
    </a>
  </nav>
  <b slot="language">
    <b slot="js">Η JavaScript είναι μια γλώσσα σεναρίου</b>
    <b slot="rs">Η Rust είναι μια γλώσσα προγραμματισμού σε επίπεδο συστήματος</b>
  </b>
  <b slot="os">
    <b slot="mac">Το macOS είναι ένα γραφικό λειτουργικό σύστημα που βασίζεται σε Unix</b>
    <b slot="win">Τα Windows είναι ένα λειτουργικό σύστημα που αναπτύχθηκε από τη Microsoft</b>
  </b>
</c-tab-select>
```

## ιδιοκτησία

- `key`: όνομα κλειδιού που χρησιμοποιείται για τη διατήρηση της τοπικής αποθήκευσης
- `value`: επιλεγμένη τιμή

## Προσαρμοσμένες εκδηλώσεις

- `change`: Ενεργοποιείται κατά την εναλλαγή καρτελών, το `e.value` είναι η τιμή `value` της τρέχουσας ενεργοποιημένης επιλογής.
