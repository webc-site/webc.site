# ড্রপ-ডাউন নির্বাচন সহ ট্যাব পৃষ্ঠা

- লেবেলে অভিযোজিত প্রস্থ ড্রপ-ডাউন নির্বাচন বাক্স (`<c-select>`) এম্বেড করা সমর্থন করে, যেটি শুধুমাত্র বর্তমান ট্যাব পৃষ্ঠা সক্রিয় হলেই প্রদর্শিত হবে।
- স্ট্রাকচারাল স্পেসিফিকেশন: চাইল্ড এলিমেন্ট `nav > a` লেবেল নেভিগেশন হিসেবে ব্যবহার করে, এবং সরাসরি চাইল্ড এলিমেন্ট `b` বিষয়বস্তু এলাকা হিসেবে কাজ করে।
- স্বয়ংক্রিয়ভাবে `a[value]` ট্যাগ, `a` সাথে `c-select` (গোষ্ঠীর মান বৈশিষ্ট্য এবং নির্বাচিত বিকল্প মানের উপর ভিত্তি করে) এবং `b[slot]` সামগ্রী নোড যুক্ত করে৷
- `key` অ্যাট্রিবিউটের মাধ্যমে স্বয়ংক্রিয়ভাবে বর্তমানে সক্রিয় করা আইটেমটিকে `localStorage` টিকে সমর্থন করে৷
- `key + '/' + tab` ব্যবহার করে প্রতিটি গোষ্ঠীর জন্য ঐতিহাসিক উপ-নির্বাচনগুলি স্বয়ংক্রিয়ভাবে সংরক্ষণ এবং পুনরুদ্ধার সমর্থন করে৷
- `value` অ্যাট্রিবিউটের মাধ্যমে সক্রিয় ট্যাব নির্দিষ্ট করা সমর্থন করে।

## ডেমো ব্যবহার করুন

```html
<c-tab-select key="my-tab-select-key" value="language/js">
  <nav>
    <a value="language">
      <span>ভাষা</span>
      <c-select>
        <option value="js">JavaScript</option>
        <option value="rs">Rust</option>
      </c-select>
    </a>
    <a value="os">
      <span>সিস্টেম</span>
      <c-select>
        <option value="mac">macOS</option>
        <option value="win">Windows</option>
      </c-select>
    </a>
  </nav>
  <b slot="language">
    <b slot="js">জাভাস্ক্রিপ্ট একটি স্ক্রিপ্টিং ভাষা</b>
    <b slot="rs">মরিচা একটি সিস্টেম-স্তরের প্রোগ্রামিং ভাষা</b>
  </b>
  <b slot="os">
    <b slot="mac">macOS একটি ইউনিক্স-ভিত্তিক গ্রাফিকাল অপারেটিং সিস্টেম</b>
    <b slot="win">উইন্ডোজ মাইক্রোসফ্ট দ্বারা তৈরি একটি অপারেটিং সিস্টেম</b>
  </b>
</c-tab-select>
```

## সম্পত্তি

- `key`: স্থানীয় স্টোরেজ অধ্যবসায়ের জন্য ব্যবহৃত কী নাম
- `value`: বর্তমানে নির্বাচিত মান

## কাস্টম ঘটনা

- `change`: ট্যাব স্যুইচ করার সময় ট্রিগার হয়, `e.value` হল `value` বর্তমানে সক্রিয় বিকল্পের মান।
