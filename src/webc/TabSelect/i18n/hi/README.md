# ड्रॉप-डाउन चयन के साथ टैब पृष्ठ

- लेबल में अनुकूली चौड़ाई ड्रॉप-डाउन चयन बॉक्स (<बी आईडी = "1">) को एम्बेड करने का समर्थन करता है, जो केवल वर्तमान टैब पृष्ठ सक्रिय होने पर प्रदर्शित किया जाएगा।
- संरचनात्मक विशिष्टता: चाइल्ड तत्व `nav > a` को लेबल नेविगेशन के रूप में उपयोग करता है, और प्रत्यक्ष चाइल्ड तत्व `b` सामग्री क्षेत्र के रूप में कार्य करता है।
- स्वचालित रूप से `a[value]` टैग, `a` को `c-select` (समूह के मूल्य विशेषता और चयनित विकल्प मान के आधार पर) और `b[slot]` सामग्री नोड के साथ संबद्ध करता है।
- `key` विशेषता के माध्यम से वर्तमान में सक्रिय आइटम को स्वचालित रूप से `localStorage` पर बनाए रखने का समर्थन करता है।
- `key + '/' + tab` का उपयोग करके प्रत्येक समूह के लिए ऐतिहासिक उप-चयनों को स्वचालित रूप से सहेजने और पुनर्स्थापित करने का समर्थन करता है।
- `value` विशेषता के माध्यम से सक्रिय टैब निर्दिष्ट करने का समर्थन करता है।

## डेमो का प्रयोग करें

```html
<c-tab-select key="my-tab-select-key" value="language/js">
  <nav>
    <a value="language">
      <span>भाषा</span>
      <c-select>
        <option value="js">JavaScript</option>
        <option value="rs">Rust</option>
      </c-select>
    </a>
    <a value="os">
      <span>प्रणाली</span>
      <c-select>
        <option value="mac">macOS</option>
        <option value="win">Windows</option>
      </c-select>
    </a>
  </nav>
  <b slot="language">
    <b slot="js">जावास्क्रिप्ट एक स्क्रिप्टिंग भाषा है</b>
    <b slot="rs">रस्ट एक सिस्टम-स्तरीय प्रोग्रामिंग भाषा है</b>
  </b>
  <b slot="os">
    <b slot="mac">macOS एक यूनिक्स-आधारित ग्राफिकल ऑपरेटिंग सिस्टम है</b>
    <b slot="win">विंडोज़ माइक्रोसॉफ्ट द्वारा विकसित एक ऑपरेटिंग सिस्टम है</b>
  </b>
</c-tab-select>
```

## संपत्ति

- `key`: लोकलस्टोरेज दृढ़ता के लिए उपयोग किया जाने वाला कुंजी नाम
- `value`: वर्तमान में चयनित मान

## कस्टम इवेंट

- `change`: टैब स्विच करते समय ट्रिगर, `e.value` वर्तमान में सक्रिय विकल्प का `value` मान है।
