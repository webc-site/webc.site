# Укладка з выпадальным выбарам

- Падтрымліваецца ўбудаванне выпаднога поля выбару адаптыўнай шырыні (`<c-select>`) у цэтлік, які будзе адлюстроўвацца толькі пры актывацыі бягучай старонкі ўкладкі.
- Структурная спецыфікацыя: даччыны элемент выкарыстоўвае `nav > a` у якасці навігацыі па цэтліках, а прамы даччыны элемент `b` служыць вобласцю кантэнту.
- Аўтаматычна звязвае тэг `a[value]`, `a` з `c-select` (на падставе атрыбута значэння групы і значэння абранага параметра) і вузла змесціва `b[slot]`.
- Падтрымлівае аўтаматычнае захаванне актываванага ў цяперашні час элемента `localStorage` праз атрыбут `key`.
- Падтрымлівае аўтаматычнае захаванне і аднаўленне гістарычных падвыбараў для кожнай групы ў `localStorage` з дапамогай `key + '/' + tab`.
- Падтрымліваецца ўказанне актыўнай укладкі праз атрыбут `value`.

## Выкарыстоўвайце дэма

```html
<c-tab-select key="my-tab-select-key" value="language/js">
  <nav>
    <a value="language">
      <span>мове</span>
      <c-select>
        <option value="js">JavaScript</option>
        <option value="rs">Rust</option>
      </c-select>
    </a>
    <a value="os">
      <span>сістэма</span>
      <c-select>
        <option value="mac">macOS</option>
        <option value="win">Windows</option>
      </c-select>
    </a>
  </nav>
  <b slot="language">
    <b slot="js">JavaScript - гэта мова сцэнарыяў</b>
    <b slot="rs">Rust - гэта мова праграмавання сістэмнага ўзроўню</b>
  </b>
  <b slot="os">
    <b slot="mac">macOS - гэта графічная аперацыйная сістэма на базе Unix</b>
    <b slot="win">Windows - гэта аперацыйная сістэма, распрацаваная Microsoft</b>
  </b>
</c-tab-select>
```

## уласнасць

- `key`: імя ключа, якое выкарыстоўваецца для захоўвання localStorage
- `value`: выбранае значэнне

## Карыстальніцкія падзеі

- `change`: Спрацоўвае пры пераключэнні ўкладак, `e.value` - гэта значэнне `value` бягучай актываванай опцыі.
