# డ్రాప్-డౌన్ ఎంపికతో ట్యాబ్ పేజీ

- లేబుల్‌లో అనుకూల వెడల్పు డ్రాప్-డౌన్ ఎంపిక పెట్టె (`<c-select>`) పొందుపరచడానికి మద్దతు ఇస్తుంది, ఇది ప్రస్తుత ట్యాబ్ పేజీ సక్రియం చేయబడినప్పుడు మాత్రమే ప్రదర్శించబడుతుంది.
- నిర్మాణ వివరణ: చైల్డ్ ఎలిమెంట్ `nav > a`ని లేబుల్ నావిగేషన్‌గా ఉపయోగిస్తుంది మరియు డైరెక్ట్ చైల్డ్ ఎలిమెంట్ `b` కంటెంట్ ప్రాంతంగా పనిచేస్తుంది.
- `a[value]` ట్యాగ్, `a`ని `c-select` (సమూహం యొక్క విలువ లక్షణం మరియు ఎంచుకున్న ఎంపిక విలువ ఆధారంగా) మరియు `b[slot]` కంటెంట్ నోడ్‌తో స్వయంచాలకంగా అనుబంధిస్తుంది.
- ప్రస్తుతం యాక్టివేట్ చేయబడిన అంశాన్ని `localStorage`కి `key` లక్షణం ద్వారా స్వయంచాలకంగా కొనసాగించడానికి మద్దతు ఇస్తుంది.
- `key + '/' + tab`ని ఉపయోగించి `localStorage`కి ప్రతి సమూహానికి హిస్టారికల్ ఉప-ఎంపికలను స్వయంచాలకంగా సేవ్ చేయడానికి మరియు పునరుద్ధరించడానికి మద్దతు ఇస్తుంది.
- `value` లక్షణం ద్వారా సక్రియ ట్యాబ్‌ను పేర్కొనడానికి మద్దతు ఇస్తుంది.

## డెమో ఉపయోగించండి

```html
<c-tab-select key="my-tab-select-key" value="language/js">
  <nav>
    <a value="language">
      <span>భాష</span>
      <c-select>
        <option value="js">JavaScript</option>
        <option value="rs">Rust</option>
      </c-select>
    </a>
    <a value="os">
      <span>వ్యవస్థ</span>
      <c-select>
        <option value="mac">macOS</option>
        <option value="win">Windows</option>
      </c-select>
    </a>
  </nav>
  <b slot="language">
    <b slot="js">జావాస్క్రిప్ట్ ఒక స్క్రిప్టింగ్ భాష</b>
    <b slot="rs">రస్ట్ అనేది సిస్టమ్స్-స్థాయి ప్రోగ్రామింగ్ భాష</b>
  </b>
  <b slot="os">
    <b slot="mac">macOS అనేది Unix-ఆధారిత గ్రాఫికల్ ఆపరేటింగ్ సిస్టమ్</b>
    <b slot="win">విండోస్ అనేది మైక్రోసాఫ్ట్ అభివృద్ధి చేసిన ఆపరేటింగ్ సిస్టమ్</b>
  </b>
</c-tab-select>
```

## ఆస్తి

- `key`: స్థానిక నిల్వ నిలకడ కోసం కీ పేరు ఉపయోగించబడుతుంది
- `value`: ప్రస్తుతం ఎంచుకున్న విలువ

## కస్టమ్ ఈవెంట్స్

- `change`: ట్యాబ్‌లను మార్చేటప్పుడు ట్రిగ్గర్ చేయబడింది, `e.value` అనేది ప్రస్తుతం యాక్టివేట్ చేయబడిన ఎంపిక యొక్క `value` విలువ.
