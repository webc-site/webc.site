# ଇନପୁଟ୍ ବାକ୍ସ ଯାହା ସ୍ୱୟଂଚାଳିତ ଭାବରେ ପ୍ଲେସହୋଲ୍ଡର୍ ସ୍ଥିତିକୁ ସୁଇଚ୍ କରେ |

- ପ୍ରବେଶ ନକରିବା ସମୟରେ ସ୍ଥାନଧାରୀ ଦେଖାନ୍ତୁ |
- ବିଷୟବସ୍ତୁ ଟାଇପ୍ କରିବା ସମୟରେ ସ୍ଥାନଧାରୀମାନେ ସହଜରେ ସ୍କେଲ ଏବଂ ଫ୍ଲୋଟ୍ କରନ୍ତି |
- ଶୁଦ୍ଧ CSS କାର୍ଯ୍ୟକାରିତା, କ Java ଣସି ଜାଭାସ୍କ୍ରିପ୍ଟ ନିର୍ଭରଶୀଳତା ନାହିଁ |

## ଡେମୋ ବ୍ୟବହାର କରନ୍ତୁ |

```html
<b class="Input Lg">
  <input id="email" type="text" placeholder=" ">
  <label for="email">ଇମେଲ୍ କରନ୍ତୁ |</label>
</b>
```

## ଗଠନମୂଳକ ଆବଶ୍ୟକତା |

- ଧାରଣକାରୀ: ଇନପୁଟ୍ ବାକ୍ସ ଏବଂ ଲେବଲ୍ ହୋଷ୍ଟ କରିବାକୁ `.Input` ଶ୍ରେଣୀର ଉପାଦାନଗୁଡିକ ବ୍ୟବହାର କରନ୍ତୁ |
- ଇନପୁଟ୍ ବାକ୍ସ: `input` ଉପାଦାନ, `placeholder=" "` (ଏକ ସ୍ଥାନ ଧାରଣ କରିଥିବା ସ୍ଥାନଧାରୀ) ରାଜ୍ୟ ସୁଇଚ୍ ଟ୍ରିଗର୍ କରିବାକୁ ସେଟ୍ ହେବା ଜରୁରୀ |
- ଟିପ୍ପଣୀ ଟ୍ୟାଗ୍: `label` ଉପାଦାନ, ତୁରନ୍ତ `input` ପରେ |

## ଶ style ଳୀ ଶ୍ରେଣୀ

- `.Input`: ମ Basic ଳିକ ଶ style ଳୀ, ଇନପୁଟ୍ ବକ୍ସ ଗଠନ ଏବଂ ସ୍ଥାନାନ୍ତରଣ ଆନିମେସନ୍ ପ୍ରଦାନ କରେ |
- `.Lg`: ହାଇଲାଇଟ୍ / ଫ୍ରଷ୍ଟଡ୍ ଗ୍ଲାସ୍ ରୂପାନ୍ତର ଶ style ଳୀ |
