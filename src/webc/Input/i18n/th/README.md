# กล่องป้อนข้อมูลที่จะสลับสถานะตัวยึดตำแหน่งโดยอัตโนมัติ

- แสดงตัวยึดตำแหน่งเมื่อไม่ได้ป้อน
- พื้นที่ที่สำรองไว้จะปรับขนาดและลอยอยู่เหนือได้อย่างราบรื่นเมื่อพิมพ์เนื้อหา
- การใช้งาน CSS อย่างแท้จริง ไม่มีการพึ่งพา JavaScript
- ค่าเริ่มต้นด้วยพื้นหลังกระจกฝ้าและการเปลี่ยนแบบโฮเวอร์

## ใช้การสาธิต

```html
<b class="Input Lg">
  <input id="email" type="text" placeholder=" ">
  <label for="email">อีเมล</label>
</b>
```

## ข้อกำหนดด้านโครงสร้าง

- คอนเทนเนอร์: ใช้องค์ประกอบของคลาส `.Input` เพื่อโฮสต์กล่องอินพุตและป้ายกำกับ
- ช่องป้อนข้อมูล: `input` องค์ประกอบ `placeholder=" "` (ตัวยึดตำแหน่งที่มีช่องว่าง) จะต้องตั้งค่าเพื่อทริกเกอร์สวิตช์สถานะ
- แท็กเคล็ดลับ: `label` องค์ประกอบ ทันทีหลังจาก `input`

## คลาสสไตล์

- `.Input`: รูปแบบพื้นฐาน ให้โครงสร้างกล่องอินพุตและภาพเคลื่อนไหวการเปลี่ยนแปลง
- `.Lg`: สไตล์การปรับเปลี่ยนไฮไลต์/กระจกฝ้า
