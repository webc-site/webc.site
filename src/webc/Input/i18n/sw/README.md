# Kisanduku cha kuingiza ambacho hubadilisha kiotomati hali ya kishika nafasi

- Onyesha kishika nafasi wakati haujaingizwa
- Vishika nafasi vyema na kuelea juu wakati wa kuandika maudhui
- Utekelezaji safi wa CSS, hakuna tegemezi za JavaScript

## Tumia onyesho

```html
<b class="Input Lg">
  <input id="email" type="text" placeholder=" ">
  <label for="email">Barua pepe</label>
</b>
```

## mahitaji ya muundo

- Chombo: Tumia vipengele vya darasa la `.Input` kupangisha visanduku vya ingizo na lebo.
- Kisanduku cha kuingiza: `input` kipengele, `placeholder=" "` (kishika nafasi kilicho na nafasi) lazima kiwekwe ili kuanzisha swichi ya hali.
- Lebo ya kidokezo: `label` kipengele, mara baada ya `input`.

## darasa la mtindo

- `.Input`: Mtindo msingi, unatoa muundo wa kisanduku cha ingizo na uhuishaji wa mpito.
- `.Lg`: Angazia/mtindo wa urekebishaji wa glasi iliyoganda.
