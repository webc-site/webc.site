# ইনপুট বাকচ যি স্বয়ংক্ৰিয়ভাৱে প্লেচহোল্ডাৰৰ অৱস্থা সলনি কৰে

- প্ৰৱেশ নকৰাৰ সময়ত প্লেচহোল্ডাৰ দেখুৱাওক
- প্লেচহোল্ডাৰসমূহে মসৃণভাৱে স্কেল কৰে আৰু বিষয়বস্তু টাইপ কৰাৰ সময়ত ওপৰত ভাসমান হয়
- বিশুদ্ধ CSS প্ৰণয়ন, কোনো জাভাস্ক্রিপ্ট নিৰ্ভৰশীলতা নাই

## ডেমো ব্যৱহাৰ কৰক

```html
<b class="Input Lg">
  <input id="email" type="text" placeholder=" ">
  <label for="email">ইমেইল</label>
</b>
```

## গাঁথনিগত প্ৰয়োজনীয়তা

- ধাৰক: ইনপুট বাকচ আৰু লেবেলসমূহ হস্ট কৰিবলে `.Input` শ্ৰেণীৰ উপাদানসমূহ ব্যৱহাৰ কৰক।
- ইনপুট বাকচ: `input` উপাদান, `placeholder=" "` (এটা স্থান ধাৰণ কৰা এটা স্থানধাৰী) অৱস্থা চুইচ ট্ৰিগাৰ কৰিবলে সংহতি কৰিব লাগিব।
- টিপ টেগ: `label` উপাদান, `input`ৰ ঠিক পিছত।

## শৈলী শ্ৰেণী

- `.Input`: মূল শৈলী, ইনপুট বাকচৰ গঠন আৰু পৰিৱৰ্তন এনিমেচন প্ৰদান কৰা।
- `.Lg`: হাইলাইট/ফ্ৰষ্টেড গ্লাছ পৰিৱৰ্তন শৈলী।
