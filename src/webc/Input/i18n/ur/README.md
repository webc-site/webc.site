# ان پٹ باکس جو خود بخود پلیس ہولڈر کی حالت کو تبدیل کرتا ہے۔

- داخل نہ ہونے پر پلیس ہولڈر دکھائیں۔
- مواد ٹائپ کرتے وقت پلیس ہولڈر آسانی سے پیمانے اور اوپر تیرتے ہیں۔
- خالص سی ایس ایس کا نفاذ، جاوا اسکرپٹ پر کوئی انحصار نہیں۔
- فراسٹڈ گلاس بیک گراؤنڈ اور ہوور ٹرانزیشن کے ساتھ ڈیفالٹ

## ڈیمو استعمال کریں۔

```html
<b class="Input Lg">
  <input id="email" type="text" placeholder=" ">
  <label for="email">ای میل</label>
</b>
```

## ساختی ضروریات

- کنٹینر: ان پٹ باکسز اور لیبلز کی میزبانی کے لیے `.Input` کلاس کے عناصر کا استعمال کریں۔
- ان پٹ باکس: `input` عنصر، `placeholder=" "` (اسپیس پر مشتمل پلیس ہولڈر) کو اسٹیٹ سوئچ کو متحرک کرنے کے لیے سیٹ کیا جانا چاہیے۔
- ٹپ ٹیگ: `label` عنصر، `input` کے فوراً بعد۔

## سٹائل کلاس

- `.Input`: بنیادی انداز، ان پٹ باکس کا ڈھانچہ اور ٹرانزیشن اینیمیشن فراہم کرتا ہے۔
- `.Lg`: ہائی لائٹ/فروسٹڈ شیشے میں ترمیم کا انداز۔
