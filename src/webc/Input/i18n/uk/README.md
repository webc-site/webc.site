# Поле введення, яке автоматично змінює стан заповнювача

- Показувати заповнювач, якщо не введено
- Заповнювачі плавно масштабуються та плавають під час введення вмісту
- Чиста реалізація CSS, відсутність залежностей від JavaScript

## Використовуйте демо

```html
<b class="Input Lg">
  <input id="email" type="text" placeholder=" ">
  <label for="email">Електронна пошта</label>
</b>
```

## структурні вимоги

- Контейнер: використовуйте елементи класу `.Input` для розміщення полів введення та міток.
- Поле введення: елемент `input`, `placeholder=" "` (заповнювач, що містить пробіл), має бути встановлено, щоб активувати перемикач стану.
- Тег підказки: елемент `label`, одразу після `input`.

## клас стилю

- `.Input`: базовий стиль із структурою поля введення та анімацією переходів.
- `.Lg`: стиль модифікації підсвічування/матового скла.
