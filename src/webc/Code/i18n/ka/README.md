# კოდის ღილაკი

- შეიცავდეს კოდის ხატულას ღილაკი
- მაუსის გადაადგილების სწრაფი კოდი, მხარს უჭერს მრავალ ენას

## გამოიყენეთ დემო ვერსია

```html
<c-code href="https://github.com"></c-code>
```

## ინტერფეისის პარამეტრები

კომპონენტის ეტიკეტი არის `<c-code>` და კოდის გადასვლის მისამართის კონფიგურაცია შესაძლებელია `href` ატრიბუტის მეშვეობით.
