# buton sa code

- Adunay buton nga icon sa code
- Mouse hover prompt code, nagsuporta sa daghang mga pinulongan

## Gamita ang demo

```html
<c-code href="https://github.com"></c-code>
```

## Mga parametro sa interface

Ang component label mao ang `<c-code>`, ug ang code jump address mahimong ma-configure pinaagi sa `href` attribute.
