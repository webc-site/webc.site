# tombol kode

- Berisi tombol ikon kode
- Kode prompt arahkan mouse, mendukung berbagai bahasa

## Gunakan demonya

```html
<c-code href="https://github.com"></c-code>
```

## Parameter antarmuka

Label komponennya adalah `<c-code>`, dan alamat lompatan kode dapat dikonfigurasi melalui atribut `href`.
