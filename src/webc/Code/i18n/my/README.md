# ကုဒ်ခလုတ်

- ကုဒ်သင်္ကေတပါရှိသောခလုတ်
- မောက်စ်ပေါ်တွင် ခလုတ်နှိပ်ကုဒ်၊ ဘာသာစကားများစွာကို ပံ့ပိုးပေးသည်။

## ဒီမိုကိုသုံးပါ။

```html
<c-code href="https://github.com"></c-code>
```

## မျက်နှာပြင်ဘောင်များ

အစိတ်အပိုင်းအညွှန်းသည် `<c-code>`ဖြစ်ပြီး၊ ကုဒ်ခုန်လိပ်စာကို `href` ရည်ညွှန်းချက်ဖြင့် ပြင်ဆင်သတ်မှတ်နိုင်သည်။
