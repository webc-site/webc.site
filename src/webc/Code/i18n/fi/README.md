# koodipainike

- Sisällytä koodi -kuvakepainike
- Hiiren hover-kehotekoodi, tukee useita kieliä

## Käytä demoa

```html
<c-code href="https://github.com"></c-code>
```

## Käyttöliittymän parametrit

Komponenttinimike on `<c-code>`, ja koodin hyppyosoite voidaan määrittää `href`-attribuutin avulla.
