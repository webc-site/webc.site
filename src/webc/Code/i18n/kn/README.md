# ಕೋಡ್ ಬಟನ್

- ಕೋಡ್ ಐಕಾನ್ ಬಟನ್ ಅನ್ನು ಒಳಗೊಂಡಿರುತ್ತದೆ
- ಮೌಸ್ ಹೋವರ್ ಪ್ರಾಂಪ್ಟ್ ಕೋಡ್, ಬಹು ಭಾಷೆಗಳನ್ನು ಬೆಂಬಲಿಸುತ್ತದೆ

## ಡೆಮೊ ಬಳಸಿ

```html
<c-code href="https://github.com"></c-code>
```

## ಇಂಟರ್ಫೇಸ್ ನಿಯತಾಂಕಗಳು

ಘಟಕ ಲೇಬಲ್ `<c-code>` ಆಗಿದೆ, ಮತ್ತು ಕೋಡ್ ಜಂಪ್ ವಿಳಾಸವನ್ನು `href` ಗುಣಲಕ್ಷಣದ ಮೂಲಕ ಕಾನ್ಫಿಗರ್ ಮಾಡಬಹುದು.
