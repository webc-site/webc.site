# कोड बटन

- कोड आइकन बटन शामिल है
- माउस होवर प्रॉम्प्ट कोड, कई भाषाओं का समर्थन करता है

## डेमो का प्रयोग करें

```html
<c-code href="https://github.com"></c-code>

<script type="module">
import "webc.site/Code.js";
</script>
```

## इंटरफ़ेस पैरामीटर

घटक लेबल `<c-code>` है, और कोड जंप पता `href` विशेषता के माध्यम से कॉन्फ़िगर किया जा सकता है।
