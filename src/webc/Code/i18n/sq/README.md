# butonin e kodit

- Butoni i ikonës përmban kodin
- Kodi i shpejtë i lëvizjes së miut, mbështet shumë gjuhë

## Përdorni demonstrimin

```html
<c-code href="https://github.com"></c-code>
```

## Parametrat e ndërfaqes

Etiketa e komponentit është `<c-code>` dhe adresa e kërcimit të kodit mund të konfigurohet nëpërmjet atributit `href`.
