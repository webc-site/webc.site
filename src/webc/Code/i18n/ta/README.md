# குறியீடு பொத்தான்

- குறியீடு ஐகான் பட்டனைக் கொண்டிருக்கும்
- மவுஸ் ஹோவர் ப்ராம்ட் குறியீடு, பல மொழிகளை ஆதரிக்கிறது

## டெமோவைப் பயன்படுத்தவும்

```html
<c-code href="https://github.com"></c-code>
```

## இடைமுக அளவுருக்கள்

கூறு லேபிள் `<c-code>` ஆகும், மேலும் குறியீடு ஜம்ப் முகவரியை `href` பண்புக்கூறு மூலம் கட்டமைக்க முடியும்.
