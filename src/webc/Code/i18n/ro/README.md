# butonul de cod

- Butonul pictogramă Conține cod
- Codul prompt al mouse-ului, acceptă mai multe limbi

## Utilizați demonstrația

```html
<c-code href="https://github.com"></c-code>
```

## Parametrii interfeței

Eticheta componentei este `<c-code>`, iar adresa de salt a codului poate fi configurată prin atributul `href`.
