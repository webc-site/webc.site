# kod düyməsi

- Kod simvolu düyməsini ehtiva edir
- Siçan üzərinə sürüşdürmə kodu, bir çox dili dəstəkləyir

## Demodan istifadə edin

```html
<c-code href="https://github.com"></c-code>
```

## İnterfeys parametrləri

Komponent etiketi `<c-code>`-dir və kodun keçid ünvanı `href` atributu vasitəsilə konfiqurasiya edilə bilər.
