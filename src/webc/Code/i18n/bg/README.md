# кодов бутон

- Бутон за икона на кода
- Код за подкана при задържане на мишката, поддържа множество езици

## Използвайте демонстрацията

```html
<c-code href="https://github.com"></c-code>
```

## Параметри на интерфейса

Етикетът на компонента е `<c-code>`, а адресът за прескачане на код може да бъде конфигуриран чрез атрибута `href`.
