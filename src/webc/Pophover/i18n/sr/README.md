# плутајући балон промпт

Прикажите информације о облачићу када се миш пређе мишем или када је елемент активиран.

- Заснован на изворној веб компоненти, лаган и високих перформанси
- Подржава прилагодљиво просторно позиционирање у четири правца (цласс="топ/боттом/лефт/ригхт")
- Користи позадину са градијентом текстуре црног мат стакла и беле знакове, са изузетним детаљима физичких ивица.
- Уграђени прорачун за прилагођавање положаја, аутоматски прилагођава смер искачућих прозора према преосталом простору у прозору за приказ
- Користите <б ид="1"> за приказ плутајућег садржаја упита

## Користите демо

```html
<!-- Аутоматски израчунај правац (препоручено, нема потребе за проглашавањем класе) -->
<c-pophover>
  <button>аутоматски</button>
  <div slot="pop">Ово је врх мехурића</div>
</c-pophover>

<!-- фиксни правац -->
<c-pophover class="left">
  <button>Лево</button>
  <div slot="pop">Ово је врх мехурића</div>
</c-pophover>

<c-pophover class="right">
  <button>право</button>
  <div slot="pop">Ово је врх мехурића</div>
</c-pophover>

<c-pophover class="bottom">
  <button>Доле</button>
  <div slot="pop">Ово је врх мехурића</div>
</c-pophover>

<c-pophover class="top">
  <button>супериоран</button>
  <div slot="pop">Ово је врх мехурића</div>
</c-pophover>
```

## ЦСС променљиве

Прилагођени стилови су подржани преко следећих ЦСС променљивих:

| име променљиве | подразумевана вредност | илуструју |
| :--- | :--- | :--- |
| `--pophover-gap` | `8px` | Простор између облачића и елемента за окидање |
| `--pophover-margin` | `24px` | Сигурносна маргина ивице приказа током прорачуна прилагодљивог позиционирања |
| `--pophover-bg-top` | `#18181ce0` | Горња боја позадине са градијентом мехурића |
| `--pophover-bg-mid` | `#121215e5` | Боја позадине стрелица на левој и десној страни облачића |
| `--pophover-bg-bottom` | `#0c0c0feb` | Боја доње и горње и доње бочне стрелице позадине градијента мехурића |

## Стил описа иконе (<б ид="1">)

За плутајућа дугмад са упитом чистог типа иконе (као што су „Уреди“, „Избриши“ итд.), можете применити јединствену икону и стил лебдења тако што ћете додати име класе <б ид="1"> у спољашњи контејнер или саму компоненту.

### Пример структуре

```html
<b class="Edit Ico">
  <c-pophover>
    <a></a>
    <div slot="pop">едит</div>
  </c-pophover>
</b>
```

Основни изглед, величина (18к18пк) и промене боје при лебдењу су обједињене у <б ид="1">. Програмери само треба да наведу одговарајућу датотеку икона одвојено, као што су:
```css
.Edit > c-pophover > a {
  mask-image: url("./svg/edit.svg")
}
```
