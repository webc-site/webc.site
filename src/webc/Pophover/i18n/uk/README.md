# плаваюча підказка

Відображати інформацію підказки під час наведення миші або активації елемента.

- На основі рідного веб-компонента, легкий і високопродуктивний
- Підтримує адаптивне просторове позиціонування в чотирьох напрямках (class="top/bottom/left/right")
- Він використовує градієнтний фон текстури чорного матового скла та білі символи з вишуканими підсвічуваннями фізичних контурів.
- Вбудований адаптивний розрахунок позиції автоматично регулює напрямок спливаючого вікна відповідно до вільного простору у вікні перегляду
- Використовуйте `slot="pop"` для відображення плаваючого вмісту запиту

## Використовуйте демо

```html
<!-- Автоматично розрахувати напрямок (рекомендовано, не потрібно оголошувати клас) -->
<c-pophover>
  <button>автоматичний</button>
  <div slot="pop">Це наконечник бульбашки</div>
</c-pophover>

<!-- фіксований напрямок -->
<c-pophover class="left">
  <button>Ліворуч</button>
  <div slot="pop">Це наконечник бульбашки</div>
</c-pophover>

<c-pophover class="right">
  <button>правильно</button>
  <div slot="pop">Це наконечник бульбашки</div>
</c-pophover>

<c-pophover class="bottom">
  <button>вниз</button>
  <div slot="pop">Це наконечник бульбашки</div>
</c-pophover>

<c-pophover class="top">
  <button>начальника</button>
  <div slot="pop">Це наконечник бульбашки</div>
</c-pophover>
```

## Змінні CSS

Підтримка налаштування стилів за допомогою наступних CSS змінних:

| ім'я змінної | значення за замовчуванням | ілюструвати |
| :--- | :--- | :--- |
| `--pophover-gap` | `8px` | Відстань між спливаючою підказкою та елементом-триггером |
| `--pophover-margin` | `24px` | Безпечні відступи від країв вікна під час автоматичного розташування |
| `--pophover-bg-top` | `#18181ce0` | Верхній колір тла бульбашки градієнта |
| `--pophover-bg-mid` | `#121215e5` | Фонова колір стрілок зліва та справа у спливаючій підказці |
| `--pophover-bg-bottom` | `#0c0c0feb` | Колір нижньої, верхньої та нижньої бічних стрілок бульбашкового градієнтного фону |

## Стиль підказки значка (`.Ico`)

Для плаваючих кнопок підказок чистого типу піктограм (таких як «Редагувати», «Видалити» тощо) ви можете застосувати єдиний стиль піктограми та наведення, додавши назву класу `.Ico` до зовнішнього контейнера або самого компонента.

### Приклад структури

```html
<b class="Edit Ico">
  <c-pophover>
    <a></a>
    <div slot="pop">редагувати</div>
  </c-pophover>
</b>
```

Основний макет, розмір (18x18 пікселів) і зміни кольору при наведенні курсора уніфіковано в `.Ico > c-pophover > a`. Розробникам потрібно лише окремо вказати відповідний файл значка, наприклад:
```css
.Edit > c-pophover > a {
  mask-image: url("./svg/edit.svg")
}
```
