# промпт за лебдечки меур

Прикажува информации за советот за алатка за лебдење кога глувчето е лебди или елементот е активиран.

- Заснована на домашна веб-компонента, лесна и со високи перформанси
- Поддржува адаптивно просторно позиционирање во четири насоки (класа = "горе/долу/лево/десно")
- Користи градиентна позадина со текстура на црно матирано стакло и бели знаци, со извонредни физички нагласувања на контурите на рабовите.
- Вградена адаптивна пресметка за позиција, автоматски ја прилагодува насоката на скокачкиот прозорец според преостанатиот простор во приказот
- Користете `slot="pop"` за да прикажете пловечка содржина за известување

## Користете го демо

```html
<!-- Автоматско пресметување на насоката (препорачано, нема потреба да се декларира класа) -->
<c-pophover>
  <button>автоматски</button>
  <div slot="pop">Ова е врв на меур</div>
</c-pophover>

<!-- фиксна насока -->
<c-pophover class="left">
  <button>Лево</button>
  <div slot="pop">Ова е врв на меур</div>
</c-pophover>

<c-pophover class="right">
  <button>право</button>
  <div slot="pop">Ова е врв на меур</div>
</c-pophover>

<c-pophover class="bottom">
  <button>Долу</button>
  <div slot="pop">Ова е врв на меур</div>
</c-pophover>

<c-pophover class="top">
  <button>надреден</button>
  <div slot="pop">Ова е врв на меур</div>
</c-pophover>
```

## CSS променливи

Прилагодените стилови се поддржани преку следните CSS променливи:

| име на променлива | стандардна вредност | илустрираат |
| :--- | :--- | :--- |
| `--pophover-gap` | `8px` | Просторот помеѓу советот за алатка со балон и елементот за активирање |
| `--pophover-margin` | `24px` | Безбедносна маржа на работ на приказот при адаптивното позиционирање |
| `--pophover-bg-top` | `#18181ce0` | Горна боја на позадината со градиент на меурчиња |
| `--pophover-bg-mid` | `#121215e5` | Бојата на позадината на стрелките на левата и десната страна на балонот |
| `--pophover-bg-bottom` | `#0c0c0feb` | Бојата на долните и горните и долните странични стрелки на позадината на градиентот на балонот |

## Стил на совет за икона (`.Ico`)

За копчињата за пловечки прашалници за чист тип на икона (како што се „Измени“, „Избриши“ итн.), може да примените унифицирана икона и стил на лебдење со додавање на името на класата `.Ico` на надворешниот контејнер или самата компонента.

### Пример за структура

```html
<b class="Edit Ico">
  <c-pophover>
    <a></a>
    <div slot="pop">уредување</div>
  </c-pophover>
</b>
```

Основниот распоред, големината (18x18px) и промените на бојата на лебдењето се обединети во `.Ico > c-pophover > a`. Програмерите треба само да ја наведат соодветната датотека со икони одделно, како на пример:
```css
.Edit > c-pophover > a {
  mask-image: url("./svg/edit.svg")
}
```
