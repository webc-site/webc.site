# फ्लोटिंग बबल प्रॉम्प्ट

जब माउस होवर किया जाता है या तत्व सक्रिय होता है तो होवर टूलटिप जानकारी प्रदर्शित करें।

- देशी वेब घटक पर आधारित, हल्का और उच्च प्रदर्शन वाला
- चार दिशाओं में अनुकूली स्थानिक स्थिति का समर्थन करता है (वर्ग = "ऊपर/नीचे/बाएँ/दाएँ")
- इसमें उत्कृष्ट भौतिक किनारे समोच्च हाइलाइट्स के साथ एक काले फ्रॉस्टेड ग्लास बनावट ढाल पृष्ठभूमि और सफेद अक्षरों का उपयोग किया जाता है।
- अंतर्निहित स्थिति अनुकूली गणना, व्यूपोर्ट में शेष स्थान के अनुसार पॉप-अप दिशा को स्वचालित रूप से समायोजित करती है
- फ़्लोटिंग प्रॉम्प्ट सामग्री प्रदर्शित करने के लिए `slot="pop"` का उपयोग करें

## डेमो का प्रयोग करें

```html
<!-- स्वचालित रूप से दिशा की गणना करें (अनुशंसित, वर्ग घोषित करने की कोई आवश्यकता नहीं) -->
<c-pophover>
  <button>स्वचालित</button>
  <div slot="pop">यह एक बबल टिप है</div>
</c-pophover>

<!-- निश्चित दिशा -->
<c-pophover class="left">
  <button>बाएं</button>
  <div slot="pop">यह एक बबल टिप है</div>
</c-pophover>

<c-pophover class="right">
  <button>सही</button>
  <div slot="pop">यह एक बबल टिप है</div>
</c-pophover>

<c-pophover class="bottom">
  <button>नीचे</button>
  <div slot="pop">यह एक बबल टिप है</div>
</c-pophover>

<c-pophover class="top">
  <button>बेहतर</button>
  <div slot="pop">यह एक बबल टिप है</div>
</c-pophover>
```

## सीएसएस चर

कस्टम शैलियाँ निम्नलिखित CSS वेरिएबल्स के माध्यम से समर्थित हैं:

| चर का नाम | डिफ़ॉल्ट मान | उदाहरण देकर स्पष्ट करना |
| :--- | :--- | :--- |
| `--pophover-gap` | `8px` | बबल टूलटिप और ट्रिगरिंग तत्व के बीच का स्थान |
| `--pophover-margin` | `24px` | अनुकूली स्थिति गणना के दौरान व्यूपोर्ट एज सुरक्षा मार्जिन |
| `--pophover-bg-top` | `#18181ce0` | बुलबुला ढाल पृष्ठभूमि शीर्ष रंग |
| `--pophover-bg-mid` | `#121215e5` | बुलबुले के बायीं और दायीं ओर तीरों का पृष्ठभूमि रंग |
| `--pophover-bg-bottom` | `#0c0c0feb` | बबल ग्रेडिएंट पृष्ठभूमि के निचले और ऊपरी तथा निचले पार्श्व तीरों का रंग |

## चिह्न टूलटिप शैली (`.Ico`)

शुद्ध आइकन प्रकार के फ़्लोटिंग प्रॉम्प्ट बटन (जैसे "संपादित करें", "हटाएँ", आदि) के लिए, आप बाहरी कंटेनर या घटक में `.Ico` क्लास नाम जोड़कर एक एकीकृत आइकन और होवर शैली लागू कर सकते हैं।

### संरचना उदाहरण

```html
<b class="Edit Ico">
  <c-pophover>
    <a></a>
    <div slot="pop">संपादन करना</div>
  </c-pophover>
</b>
```

मूल लेआउट, आकार (18x18px), और होवर रंग परिवर्तन को `.Ico > c-pophover > a` में एकीकृत किया गया है। डेवलपर्स को केवल संबंधित आइकन फ़ाइल को अलग से निर्दिष्ट करने की आवश्यकता है, जैसे:
```css
.Edit > c-pophover > a {
  mask-image: url("./svg/edit.svg")
}
```
