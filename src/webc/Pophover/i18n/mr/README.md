# फ्लोटिंग बबल प्रॉम्प्ट

जेव्हा माउस फिरवला जातो किंवा घटक सक्रिय केला जातो तेव्हा होव्हर टूलटिप माहिती प्रदर्शित करा.

- मूळ वेब घटक, हलके आणि उच्च-कार्यक्षमतेवर आधारित
- चार दिशांमध्ये अनुकूली अवकाशीय स्थितीचे समर्थन करते (वर्ग == वर/खाली/डावी/उजवीकडे")
- हे ब्लॅक फ्रॉस्टेड ग्लास टेक्सचर ग्रेडियंट पार्श्वभूमी आणि पांढरे वर्ण वापरते, उत्कृष्ट भौतिक किनार समोच्च हायलाइट्ससह.
- अंगभूत स्थिती अनुकूली गणना, व्ह्यूपोर्टमधील उर्वरित जागेनुसार पॉप-अप दिशा स्वयंचलितपणे समायोजित करते
- फ्लोटिंग प्रॉम्प्ट सामग्री प्रदर्शित करण्यासाठी `slot="pop"` वापरा

## डेमो वापरा

```html
<!-- स्वयंचलितपणे दिशा मोजा (शिफारस केलेले, वर्ग घोषित करण्याची आवश्यकता नाही) -->
<c-pophover>
  <button>स्वयंचलित</button>
  <div slot="pop">ही एक बबल टीप आहे</div>
</c-pophover>

<!-- निश्चित दिशा -->
<c-pophover class="left">
  <button>बाकी</button>
  <div slot="pop">ही एक बबल टीप आहे</div>
</c-pophover>

<c-pophover class="right">
  <button>बरोबर</button>
  <div slot="pop">ही एक बबल टीप आहे</div>
</c-pophover>

<c-pophover class="bottom">
  <button>खाली</button>
  <div slot="pop">ही एक बबल टीप आहे</div>
</c-pophover>

<c-pophover class="top">
  <button>श्रेष्ठ</button>
  <div slot="pop">ही एक बबल टीप आहे</div>
</c-pophover>
```

## CSS व्हेरिएबल्स

सानुकूल शैली खालील CSS व्हेरिएबल्सद्वारे समर्थित आहेत:

| व्हेरिएबल नाव | डीफॉल्ट मूल्य | स्पष्ट करणे |
| :--- | :--- | :--- |
| `--pophover-gap` | `8px` | बबल टूलटिप आणि ट्रिगरिंग घटकामधील जागा |
| `--pophover-margin` | `24px` | अडॅप्टिव्ह पोझिशनिंग गणना दरम्यान व्ह्यूपोर्ट एज सुरक्षा मार्जिन |
| `--pophover-bg-top` | `#18181ce0` | बबल ग्रेडियंट पार्श्वभूमी शीर्ष रंग |
| `--pophover-bg-mid` | `#121215e5` | बबलच्या डाव्या आणि उजव्या बाजूला असलेल्या बाणांचा पार्श्वभूमी रंग |
| `--pophover-bg-bottom` | `#0c0c0feb` | बबल ग्रेडियंट पार्श्वभूमीच्या खालच्या आणि वरच्या आणि खालच्या बाजूच्या बाणांचा रंग |

## चिन्ह टूलटिप शैली (`.Ico`)

शुद्ध आयकॉन प्रकारच्या फ्लोटिंग प्रॉम्प्ट बटणांसाठी (जसे की "संपादित करा", "हटवा", इ.), तुम्ही बाहेरील कंटेनर किंवा घटकामध्येच `.Ico` वर्ग नाव जोडून युनिफाइड आयकॉन आणि होव्हर शैली लागू करू शकता.

### रचना उदाहरण

```html
<b class="Edit Ico">
  <c-pophover>
    <a></a>
    <div slot="pop">संपादित करा</div>
  </c-pophover>
</b>
```

मूलभूत लेआउट, आकार (18x18px), आणि होव्हर रंग बदल `.Ico > c-pophover > a` मध्ये एकत्रित केले गेले आहेत. विकसकांना फक्त संबंधित आयकॉन फाइल स्वतंत्रपणे निर्दिष्ट करणे आवश्यक आहे, जसे की:
```css
.Edit > c-pophover > a {
  mask-image: url("./svg/edit.svg")
}
```
