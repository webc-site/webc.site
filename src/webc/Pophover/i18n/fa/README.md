# اعلان حباب شناور

هنگامی که ماوس در حالت شناور قرار می گیرد یا عنصر فعال می شود، اطلاعات راهنمای ابزار شناور را نمایش دهید.

- بر اساس کامپوننت وب بومی، سبک وزن و با کارایی بالا
- پشتیبانی از موقعیت مکانی تطبیقی ​​در چهار جهت (class="بالا/پایین/چپ/راست")
- از یک پس‌زمینه شیب بافت شیشه‌ای مات سیاه و کاراکترهای سفید، با برجسته‌سازی لبه‌های فیزیکی عالی استفاده می‌کند.
- محاسبه تطبیقی ​​موقعیت داخلی، به طور خودکار جهت پاپ آپ را با توجه به فضای باقیمانده در نمای دید تنظیم می کند.
- از `slot="pop"` برای نمایش محتوای درخواستی شناور استفاده کنید

## از دمو استفاده کنید

```html
<!-- محاسبه خودکار جهت (توصیه می شود، بدون نیاز به اعلام کلاس) -->
<c-pophover>
  <button>خودکار</button>
  <div slot="pop">این یک نوک حباب است</div>
</c-pophover>

<!-- جهت ثابت -->
<c-pophover class="left">
  <button>سمت چپ</button>
  <div slot="pop">این یک نوک حباب است</div>
</c-pophover>

<c-pophover class="right">
  <button>درست است</button>
  <div slot="pop">این یک نوک حباب است</div>
</c-pophover>

<c-pophover class="bottom">
  <button>پایین</button>
  <div slot="pop">این یک نوک حباب است</div>
</c-pophover>

<c-pophover class="top">
  <button>برتر</button>
  <div slot="pop">این یک نوک حباب است</div>
</c-pophover>
```

## متغیرهای CSS

سبک های سفارشی از طریق متغیرهای CSS زیر پشتیبانی می شوند:

| نام متغیر | مقدار پیش فرض | نشان دادن |
| :--- | :--- | :--- |
| `--pophover-gap` | `8px` | فضای بین راهنمای ابزار حباب و عنصر محرک |
| `--pophover-margin` | `24px` | حاشیه ایمنی لبه نما در حین محاسبه موقعیت یابی تطبیقی |
| `--pophover-bg-top` | `#18181ce0` | رنگ بالای پس زمینه گرادیان حباب |
| `--pophover-bg-mid` | `#121215e5` | رنگ پس زمینه فلش های سمت چپ و راست حباب |
| `--pophover-bg-bottom` | `#0c0c0feb` | رنگ فلش های کناری پایین و بالا و پایین پس زمینه گرادیان حباب |

## سبک راهنمای ابزار نماد (`.Ico`)

برای دکمه‌های اعلان شناور نوع نماد خالص (مانند "ویرایش"، "حذف"، و غیره)، می‌توانید یک نماد یکپارچه و سبک شناور را با افزودن نام کلاس `.Ico` به ظرف یا مؤلفه بیرونی اعمال کنید.

### نمونه ساختار

```html
<b class="Edit Ico">
  <c-pophover>
    <a></a>
    <div slot="pop">ویرایش کنید</div>
  </c-pophover>
</b>
```

طرح اولیه، اندازه (18x18px) و تغییرات رنگ شناور در `.Ico > c-pophover > a` یکسان شده است. توسعه دهندگان فقط باید فایل نماد مربوطه را به طور جداگانه مشخص کنند، مانند:
```css
.Edit > c-pophover > a {
  mask-image: url("./svg/edit.svg")
}
```
