# ভাসমান বাবল প্ৰমপ্ট

হ'ভাৰ সঁজুলিটিপ তথ্য প্ৰদৰ্শন কৰক যেতিয়া মাউছ হ'ভাৰ কৰা হয় বা উপাদান সক্ৰিয় কৰা হয়।

- স্থানীয় ৱেব উপাদানৰ ওপৰত ভিত্তি কৰি, লঘু আৰু উচ্চ-কাৰ্য্যক্ষমতা
- চাৰিটা দিশত অভিযোজিত স্থানীয় অৱস্থান সমৰ্থন কৰে (class="top/bottom/left/right")
- ইয়াত এটা ক'লা ফ্ৰষ্টেড গ্লাছ টেক্সচাৰ গ্ৰেডিয়েণ্ট পটভূমি আৰু বগা চৰিত্ৰ ব্যৱহাৰ কৰা হৈছে, সুন্দৰ ভৌতিক প্ৰান্ত কনট্যুৰ হাইলাইটৰ সৈতে।
- বিল্ট-ইন অৱস্থান অভিযোজিত গণনা, স্বয়ংক্ৰিয়ভাৱে দৃশ্যপৰ্টত বাকী থকা স্থান অনুসৰি পপ-আপ দিশ সামঞ্জস্য কৰে
- ভাসমান প্ৰমপ্ট বিষয়বস্তু প্ৰদৰ্শন কৰিবলে `slot="pop"` ব্যৱহাৰ কৰক

## ডেমো ব্যৱহাৰ কৰক

```html
<!-- স্বয়ংক্ৰিয়ভাৱে দিশ গণনা কৰক (পৰামৰ্শ দিয়া হৈছে, শ্ৰেণী ঘোষণা কৰাৰ প্ৰয়োজন নাই) -->
<c-pophover>
  <button>স্বয়ংক্ৰিয়</button>
  <div slot="pop">এইটো এটা বুদবুদৰ টিপ</div>
</c-pophover>

<!-- নিৰ্দিষ্ট দিশ -->
<c-pophover class="left">
  <button>বাওঁ</button>
  <div slot="pop">এইটো এটা বুদবুদৰ টিপ</div>
</c-pophover>

<c-pophover class="right">
  <button>শুদ্ধ</button>
  <div slot="pop">এইটো এটা বুদবুদৰ টিপ</div>
</c-pophover>

<c-pophover class="bottom">
  <button>তললৈ</button>
  <div slot="pop">এইটো এটা বুদবুদৰ টিপ</div>
</c-pophover>

<c-pophover class="top">
  <button>শ্ৰেষ্ঠ</button>
  <div slot="pop">এইটো এটা বুদবুদৰ টিপ</div>
</c-pophover>
```

## CSS চলকসমূহ

স্বনিৰ্বাচিত শৈলীসমূহ নিম্নলিখিত CSS চলকসমূহৰ যোগেদি সমৰ্থিত হয়:

| চলকৰ নাম | অবিকল্পিত মান | চিত্ৰিত কৰক |
| :--- | :--- | :--- |
| `--pophover-gap` | `8px` | বাবল টুলটিপ আৰু ট্ৰিগাৰিং উপাদানৰ মাজৰ স্থান |
| `--pophover-margin` | `24px` | অভিযোজিত অৱস্থান গণনাৰ সময়ত ভিউপ'ৰ্ট প্ৰান্ত সুৰক্ষা মাৰ্জিন |
| `--pophover-bg-top` | `#18181ce0` | বুদবুদ গ্ৰেডিয়েন্ট পটভূমি শীৰ্ষ ৰং |
| `--pophover-bg-mid` | `#121215e5` | বুদবুদৰ বাওঁ আৰু সোঁফালে থকা কাঁড় চিহ্নবোৰৰ পটভূমিৰ ৰং |
| `--pophover-bg-bottom` | `#0c0c0feb` | বুদবুদ গ্ৰেডিয়েণ্ট পটভূমিৰ তলৰ আৰু ওপৰৰ আৰু তলৰ কাষৰ কাঁড় চিহ্নৰ ৰং |

## আইকন টুলটিপ শৈলী (`.Ico`)

বিশুদ্ধ আইকন ধৰণৰ ভাসমান প্ৰমপ্ট বুটামসমূহৰ বাবে (যেনে "সম্পাদনা কৰক", "মচি পেলাওক", ইত্যাদি), আপুনি এটা সংহত আইকন আৰু হ'ভাৰ শৈলী প্ৰয়োগ কৰিব পাৰে `.Ico` শ্ৰেণী নামটো বাহিৰৰ ধাৰক বা উপাদানত নিজেই যোগ কৰি।

### গঠনৰ উদাহৰণ

```html
<b class="Edit Ico">
  <c-pophover>
    <a></a>
    <div slot="pop">সম্পাদনা কৰক</div>
  </c-pophover>
</b>
```

মূল পৰিকল্পনা, আকাৰ (18x18px), আৰু হোভাৰ ৰঙৰ পৰিবৰ্তনসমূহ `.Ico > c-pophover > a` ত একত্ৰিত কৰা হৈছে। ডেভেলপাৰসকলে কেৱল সংশ্লিষ্ট আইকন নথিপত্ৰ পৃথকে ধাৰ্য্য কৰিব লাগিব, যেনে:
```css
.Edit > c-pophover > a {
  mask-image: url("./svg/edit.svg")
}
```
