# ఫ్లోటింగ్ బబుల్ ప్రాంప్ట్

మౌస్ హోవర్ చేయబడినప్పుడు లేదా మూలకం సక్రియం చేయబడినప్పుడు హోవర్ టూల్‌టిప్ సమాచారాన్ని ప్రదర్శించండి.

- స్థానిక వెబ్ కాంపోనెంట్, తేలికైన మరియు అధిక-పనితీరు ఆధారంగా
- నాలుగు దిశలలో అనుకూల ప్రాదేశిక స్థానానికి మద్దతు ఇస్తుంది (తరగతి="పైన/దిగువ/ఎడమ/కుడి")
- ఇది నల్లని గడ్డకట్టిన గాజు ఆకృతి గ్రేడియంట్ బ్యాక్‌గ్రౌండ్ మరియు తెల్లని అక్షరాలు, సున్నితమైన భౌతిక అంచు ఆకృతి హైలైట్‌లను ఉపయోగిస్తుంది.
- అంతర్నిర్మిత స్థానం అనుకూల గణన, వీక్షణపోర్ట్‌లో మిగిలిన స్థలం ప్రకారం పాప్-అప్ దిశను స్వయంచాలకంగా సర్దుబాటు చేస్తుంది
- ఫ్లోటింగ్ ప్రాంప్ట్ కంటెంట్‌ని ప్రదర్శించడానికి `slot="pop"`ని ఉపయోగించండి

## డెమో ఉపయోగించండి

```html
<!-- దిశను స్వయంచాలకంగా లెక్కించండి (సిఫార్సు చేయబడింది, తరగతిని ప్రకటించాల్సిన అవసరం లేదు) -->
<c-pophover>
  <button>ఆటోమేటిక్</button>
  <div slot="pop">ఇది బబుల్ చిట్కా</div>
</c-pophover>

<!-- స్థిర దిశ -->
<c-pophover class="left">
  <button>ఎడమ</button>
  <div slot="pop">ఇది బబుల్ చిట్కా</div>
</c-pophover>

<c-pophover class="right">
  <button>కుడి</button>
  <div slot="pop">ఇది బబుల్ చిట్కా</div>
</c-pophover>

<c-pophover class="bottom">
  <button>డౌన్</button>
  <div slot="pop">ఇది బబుల్ చిట్కా</div>
</c-pophover>

<c-pophover class="top">
  <button>ఉన్నతమైన</button>
  <div slot="pop">ఇది బబుల్ చిట్కా</div>
</c-pophover>
```

## CSS వేరియబుల్స్

కింది CSS వేరియబుల్స్ ద్వారా అనుకూల శైలులకు మద్దతు ఉంది:

| వేరియబుల్ పేరు | డిఫాల్ట్ విలువ | వర్ణించండి |
| :--- | :--- | :--- |
| `--pophover-gap` | `8px` | బబుల్ టూల్‌టిప్ మరియు ట్రిగ్గరింగ్ ఎలిమెంట్ మధ్య ఖాళీ |
| `--pophover-margin` | `24px` | అడాప్టివ్ పొజిషనింగ్ గణన సమయంలో వీక్షణ పోర్ట్ అంచు భద్రత మార్జిన్ |
| `--pophover-bg-top` | `#18181ce0` | బబుల్ గ్రేడియంట్ బ్యాక్‌గ్రౌండ్ టాప్ కలర్ |
| `--pophover-bg-mid` | `#121215e5` | బబుల్ యొక్క ఎడమ మరియు కుడి వైపులా ఉన్న బాణాల నేపథ్య రంగు |
| `--pophover-bg-bottom` | `#0c0c0feb` | బబుల్ గ్రేడియంట్ బ్యాక్‌గ్రౌండ్ యొక్క దిగువ మరియు ఎగువ మరియు దిగువ వైపు బాణాల రంగు |

## ఐకాన్ టూల్‌టిప్ శైలి (`.Ico`)

స్వచ్ఛమైన ఐకాన్ రకం ఫ్లోటింగ్ ప్రాంప్ట్ బటన్‌ల కోసం ("సవరించు", "తొలగించు" మొదలైనవి), మీరు `.Ico` తరగతి పేరును బాహ్య కంటైనర్ లేదా కాంపోనెంట్‌కు జోడించడం ద్వారా ఏకీకృత చిహ్నాన్ని మరియు హోవర్ శైలిని వర్తింపజేయవచ్చు.

### నిర్మాణం ఉదాహరణ

```html
<b class="Edit Ico">
  <c-pophover>
    <a></a>
    <div slot="pop">సవరించు</div>
  </c-pophover>
</b>
```

ప్రాథమిక లేఅవుట్, పరిమాణం (18x18px) మరియు హోవర్ రంగు మార్పులు `.Ico > c-pophover > a`లో ఏకీకృతం చేయబడ్డాయి. డెవలపర్‌లు సంబంధిత ఐకాన్ ఫైల్‌ను ప్రత్యేకంగా పేర్కొనాలి, అవి:
```css
.Edit > c-pophover > a {
  mask-image: url("./svg/edit.svg")
}
```
