# மிதக்கும் குமிழி வரியில்

சுட்டியை நகர்த்தும்போது அல்லது உறுப்பு செயல்படுத்தப்படும்போது, ​​மிதவை உதவிக்குறிப்பு தகவலைக் காண்பி.

- சொந்த வலை கூறு, இலகுரக மற்றும் உயர் செயல்திறன் அடிப்படையில்
- நான்கு திசைகளில் அடாப்டிவ் ஸ்பேஷியல் பொசிஷனிங்கை ஆதரிக்கிறது (வகுப்பு="மேல்/கீழ்/இடது/வலது")
- இது ஒரு கருப்பு உறைந்த கண்ணாடி அமைப்பு சாய்வு பின்னணி மற்றும் வெள்ளை எழுத்துக்கள், நேர்த்தியான இயற்பியல் விளிம்பு விளிம்பு சிறப்பம்சங்கள் பயன்படுத்துகிறது.
- உள்ளமைக்கப்பட்ட நிலை அடாப்டிவ் கணக்கீடு, காட்சிப் பகுதியில் மீதமுள்ள இடத்தின் படி பாப்-அப் திசையை தானாகவே சரிசெய்கிறது
- மிதக்கும் உடனடி உள்ளடக்கத்தைக் காட்ட `slot="pop"` ஐப் பயன்படுத்தவும்

## டெமோவைப் பயன்படுத்தவும்

```html
<!-- திசையைத் தானாகக் கணக்கிடுங்கள் (பரிந்துரைக்கப்பட்டது, வகுப்பை அறிவிக்கத் தேவையில்லை) -->
<c-pophover>
  <button>தானியங்கி</button>
  <div slot="pop">இது ஒரு குமிழி முனை</div>
</c-pophover>

<!-- நிலையான திசை -->
<c-pophover class="left">
  <button>விட்டு</button>
  <div slot="pop">இது ஒரு குமிழி முனை</div>
</c-pophover>

<c-pophover class="right">
  <button>சரி</button>
  <div slot="pop">இது ஒரு குமிழி முனை</div>
</c-pophover>

<c-pophover class="bottom">
  <button>கீழே</button>
  <div slot="pop">இது ஒரு குமிழி முனை</div>
</c-pophover>

<c-pophover class="top">
  <button>மேலான</button>
  <div slot="pop">இது ஒரு குமிழி முனை</div>
</c-pophover>
```

## CSS மாறிகள்

தனிப்பயன் பாணிகள் பின்வரும் CSS மாறிகள் மூலம் ஆதரிக்கப்படுகின்றன:

| மாறி பெயர் | இயல்புநிலை மதிப்பு | விளக்குகின்றன |
| :--- | :--- | :--- |
| `--pophover-gap` | `8px` | குமிழி உதவிக்குறிப்புக்கும் தூண்டுதல் உறுப்புக்கும் இடையே உள்ள இடைவெளி |
| `--pophover-margin` | `24px` | அடாப்டிவ் பொசிஷனிங் கணக்கீட்டின் போது வியூபோர்ட் எட்ஜ் பாதுகாப்பு விளிம்பு |
| `--pophover-bg-top` | `#18181ce0` | குமிழி சாய்வு பின்னணி மேல் வண்ணம் |
| `--pophover-bg-mid` | `#121215e5` | குமிழியின் இடது மற்றும் வலது பக்கங்களில் உள்ள அம்புகளின் பின்னணி நிறம் |
| `--pophover-bg-bottom` | `#0c0c0feb` | குமிழி சாய்வு பின்னணியின் கீழ் மற்றும் மேல் மற்றும் கீழ் பக்க அம்புகளின் நிறம் |

## ஐகான் உதவிக்குறிப்பு நடை (`.Ico`)

தூய ஐகான் வகை ஃப்ளோட்டிங் ப்ராம்ட் பொத்தான்களுக்கு ("திருத்து", "நீக்கு", முதலியன), நீங்கள் ஒரு ஒருங்கிணைந்த ஐகானைப் பயன்படுத்தலாம் மற்றும் வெளிப்புறக் கொள்கலன் அல்லது கூறுகளில் `.Ico` வகுப்பின் பெயரைச் சேர்ப்பதன் மூலம் ஹோவர் பாணியைப் பயன்படுத்தலாம்.

### அமைப்பு உதாரணம்

```html
<b class="Edit Ico">
  <c-pophover>
    <a></a>
    <div slot="pop">திருத்த</div>
  </c-pophover>
</b>
```

அடிப்படை தளவமைப்பு, அளவு (18x18px) மற்றும் மிதவை வண்ண மாற்றங்கள் `.Ico > c-pophover > a` இல் ஒருங்கிணைக்கப்பட்டுள்ளன. டெவலப்பர்கள் தொடர்புடைய ஐகான் கோப்பை மட்டும் தனித்தனியாக குறிப்பிட வேண்டும், அதாவது:
```css
.Edit > c-pophover > a {
  mask-image: url("./svg/edit.svg")
}
```
