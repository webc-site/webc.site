# ଭାସମାନ ବବୁଲ୍ ପ୍ରମ୍ପ୍ଟ |

ଯେତେବେଳେ ମାଉସ୍ ହୋଭର୍ ହୁଏ କିମ୍ବା ଉପାଦାନ ସକ୍ରିୟ ହୁଏ, ହୋଭର ଟୁଲଟିପ୍ ସୂଚନା ପ୍ରଦର୍ଶନ କରନ୍ତୁ |

- ଦେଶୀ ୱେବ୍ ଉପାଦାନ ଉପରେ ଆଧାରିତ, ହାଲୁକା ଏବଂ ଉଚ୍ଚ-କାର୍ଯ୍ୟଦକ୍ଷତା |
- ଚାରୋଟି ଦିଗରେ ଆଡାପ୍ଟିଭ୍ ସ୍ପେସାଲ୍ ପୋଜିସନ୍ କୁ ସମର୍ଥନ କରେ (ଶ୍ରେଣୀ = "ଉପର / ତଳ / ବାମ / ଡାହାଣ")
- ଏହା ଏକ କଳା ଫ୍ରୋଷ୍ଟଡ୍ ଗ୍ଲାସ୍ ଟେକ୍ସଚର୍ ଗ୍ରେଡିଏଣ୍ଟ୍ ବ୍ୟାକଗ୍ରାଉଣ୍ଡ୍ ଏବଂ ଧଳା ବର୍ଣ୍ଣଗୁଡିକ ବ୍ୟବହାର କରେ, ସୂକ୍ଷ୍ମ ଶାରୀରିକ ଧାର କଣ୍ଟୁର ହାଇଲାଇଟ୍ ସହିତ |
- ବିଲ୍ଟ-ଇନ୍ ପୋଜିସନ୍ ଆଡାପ୍ଟିଭ୍ ଗଣନା, ଭ୍ୟୁପୋର୍ଟରେ ଅବଶିଷ୍ଟ ସ୍ଥାନ ଅନୁଯାୟୀ ସ୍ୱୟଂଚାଳିତ ଭାବରେ ପପ୍-ଅପ୍ ଦିଗକୁ ସଜାଡିଥାଏ |
- ଭାସମାନ ପ୍ରମ୍ପ୍ଟ ବିଷୟବସ୍ତୁ ପ୍ରଦର୍ଶନ କରିବାକୁ `slot="pop"` ବ୍ୟବହାର କରନ୍ତୁ |

## ଡେମୋ ବ୍ୟବହାର କରନ୍ତୁ |

```html
<!-- ସ୍ୱୟଂଚାଳିତ ଭାବରେ ଦିଗ ଗଣନା କରନ୍ତୁ (ପରାମର୍ଶ ଦିଆଯାଇଛି, ଶ୍ରେଣୀ ଘୋଷଣା କରିବାର ଆବଶ୍ୟକତା ନାହିଁ) -->
<c-pophover>
  <button>ସ୍ୱୟଂଚାଳିତ |</button>
  <div slot="pop">ଏହା ଏକ ବବୁଲ୍ ଟିପ୍ |</div>
</c-pophover>

<!-- ସ୍ଥିର ଦିଗ | -->
<c-pophover class="left">
  <button>ବାମ</button>
  <div slot="pop">ଏହା ଏକ ବବୁଲ୍ ଟିପ୍ |</div>
</c-pophover>

<c-pophover class="right">
  <button>ଠିକ୍</button>
  <div slot="pop">ଏହା ଏକ ବବୁଲ୍ ଟିପ୍ |</div>
</c-pophover>

<c-pophover class="bottom">
  <button>ତଳକୁ |</button>
  <div slot="pop">ଏହା ଏକ ବବୁଲ୍ ଟିପ୍ |</div>
</c-pophover>

<c-pophover class="top">
  <button>ଶ୍ରେଷ୍ଠ</button>
  <div slot="pop">ଏହା ଏକ ବବୁଲ୍ ଟିପ୍ |</div>
</c-pophover>
```

## CSS ଭେରିଏବଲ୍ |

ନିମ୍ନଲିଖିତ CSS ଭେରିଏବଲ୍ ମାଧ୍ୟମରେ କଷ୍ଟମ୍ ଶ yles ଳୀ ସମର୍ଥିତ:

| ଭେରିଏବଲ୍ ନାମ | ଡିଫଲ୍ଟ ମୂଲ୍ୟ | ବର୍ଣ୍ଣନା କର |
| :--- | :--- | :--- |
| `--pophover-gap` | `8px` | ବବୁଲ୍ ଟୁଲ୍ ଟିପ୍ ଏବଂ ଟ୍ରିଜିଙ୍ଗ୍ ଉପାଦାନ ମଧ୍ୟରେ ସ୍ଥାନ | |
| `--pophover-margin` | `24px` | ଆଡାପ୍ଟିଭ୍ ପୋଜିସନ୍ ଗଣନା ସମୟରେ ଭ୍ୟୁପୋର୍ଟ ଏଜ୍ ସୁରକ୍ଷା ମାର୍ଜିନ | |
| `--pophover-bg-top` | `#18181ce0` | ବବୁଲ୍ ଗ୍ରେଡିଏଣ୍ଟ୍ ପୃଷ୍ଠଭୂମି ଉପର ରଙ୍ଗ | |
| `--pophover-bg-mid` | `#121215e5` | ବବୁଲର ବାମ ଏବଂ ଡାହାଣ ପାର୍ଶ୍ୱରେ ତୀରଗୁଡ଼ିକର ପୃଷ୍ଠଭୂମି ରଙ୍ଗ | |
| `--pophover-bg-bottom` | `#0c0c0feb` | ବବୁଲ୍ ଗ୍ରେଡିଏଣ୍ଟ୍ ପୃଷ୍ଠଭୂମିର ତଳ ଏବଂ ଉପର ଏବଂ ତଳ ପାର୍ଶ୍ୱ ତୀରଗୁଡ଼ିକର ରଙ୍ଗ | |

## ଆଇକନ୍ ଟୁଲ୍ ଟିପ୍ ଶ style ଳୀ (`.Ico`)

ଶୁଦ୍ଧ ଆଇକନ୍ ପ୍ରକାର ଫ୍ଲୋଟିଂ ପ୍ରମ୍ପ୍ଟ ବଟନ୍ ପାଇଁ (ଯେପରିକି "ସଂପାଦନ", "ଡିଲିଟ୍" ଇତ୍ୟାଦି), ଆପଣ ବାହ୍ୟ ପାତ୍ର କିମ୍ବା ଉପାଦାନରେ `.Ico` ଶ୍ରେଣୀ ନାମ ଯୋଡି ଏକ ୟୁନିଫାଏଡ୍ ଆଇକନ୍ ଏବଂ ହୋଭର ଷ୍ଟାଇଲ୍ ପ୍ରୟୋଗ କରିପାରିବେ |

### ସଂରଚନା ଉଦାହରଣ |

```html
<b class="Edit Ico">
  <c-pophover>
    <a></a>
    <div slot="pop">ସମ୍ପାଦନ କରନ୍ତୁ |</div>
  </c-pophover>
</b>
```

ମ b ଳିକ ଲେଆଉଟ୍, ସାଇଜ୍ (18x18px), ଏବଂ ହୋଭର ରଙ୍ଗ ପରିବର୍ତ୍ତନ `.Ico > c-pophover > a` ରେ ଏକୀକୃତ ହୋଇଛି | ବିକାଶକାରୀମାନେ କେବଳ ସଂପୃକ୍ତ ଆଇକନ୍ ଫାଇଲକୁ ପୃଥକ ଭାବରେ ନିର୍ଦ୍ଦିଷ୍ଟ କରିବା ଆବଶ୍ୟକ କରନ୍ତି, ଯେପରି:
```css
.Edit > c-pophover > a {
  mask-image: url("./svg/edit.svg")
}
```
