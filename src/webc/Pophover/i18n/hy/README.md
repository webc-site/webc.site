# լողացող պղպջակների հուշում

Ցուցադրել սավառնող գործիքի հուշման տեղեկատվությունը, երբ մկնիկը սավառնում է կամ տարրն ակտիվացված է:

- Հիմնված է հայրենի վեբ բաղադրիչի վրա՝ թեթև և բարձր արդյունավետությամբ
- Աջակցում է հարմարվողական տարածական դիրքավորումը չորս ուղղություններով (class="վերև/ներքև/ձախ/աջ")
- Այն օգտագործում է սև ցրտաշունչ ապակու տեքստուրային գրադիենտ ֆոն և սպիտակ նիշեր՝ եզրերի ուրվագծերի նուրբ ֆիզիկական շեշտադրումներով:
- Ներկառուցված դիրքի հարմարվողական հաշվարկ, ավտոմատ կերպով կարգավորում է բացվող ուղղությունը՝ ըստ տեսադաշտի մնացած տարածության
- Օգտագործեք `slot="pop"`՝ լողացող հուշումների բովանդակությունը ցուցադրելու համար

## Օգտագործեք ցուցադրությունը

```html
<!-- Ավտոմատ հաշվարկել ուղղությունը (խորհուրդ է տրվում, դաս հայտարարելու կարիք չկա) -->
<c-pophover>
  <button>ավտոմատ</button>
  <div slot="pop">Սա պղպջակների հուշում է</div>
</c-pophover>

<!-- ֆիքսված ուղղություն -->
<c-pophover class="left">
  <button>Ձախ</button>
  <div slot="pop">Սա պղպջակների հուշում է</div>
</c-pophover>

<c-pophover class="right">
  <button>ճիշտ</button>
  <div slot="pop">Սա պղպջակների հուշում է</div>
</c-pophover>

<c-pophover class="bottom">
  <button>Ներքև</button>
  <div slot="pop">Սա պղպջակների հուշում է</div>
</c-pophover>

<c-pophover class="top">
  <button>վերադաս</button>
  <div slot="pop">Սա պղպջակների հուշում է</div>
</c-pophover>
```

## CSS փոփոխականներ

Հատուկ ոճերը աջակցվում են հետևյալ CSS փոփոխականների միջոցով.

| փոփոխական անուն | լռելյայն արժեք | նկարազարդել |
| :--- | :--- | :--- |
| `--pophover-gap` | `8px` | Պղպջակների գործիքի հուշման և գործարկող տարրի միջև ընկած տարածությունը |
| `--pophover-margin` | `24px` | Դիտակետի եզրային անվտանգության սահմանը հարմարվողական դիրքավորման հաշվարկի ժամանակ |
| `--pophover-bg-top` | `#18181ce0` | Պղպջակների գրադիենտ ֆոնի վերին գույնը |
| `--pophover-bg-mid` | `#121215e5` | Պղպջակի ձախ և աջ կողմերում գտնվող սլաքների ֆոնի գույնը |
| `--pophover-bg-bottom` | `#0c0c0feb` | Պղպջակների գրադիենտ ֆոնի ներքևի և վերին և ստորին կողային սլաքների գույնը |

## Պատկերակի գործիքի հուշման ոճ (`.Ico`)

Մաքուր պատկերակի տիպի լողացող հուշման կոճակների համար (օրինակ՝ «Խմբագրել», «Ջնջել» և այլն), կարող եք կիրառել միասնական պատկերակ և սավառնել ոճը՝ ավելացնելով `.Ico` դասի անունը արտաքին կոնտեյների կամ բաղադրիչի վրա:

### Կառուցվածքի օրինակ

```html
<b class="Edit Ico">
  <c-pophover>
    <a></a>
    <div slot="pop">խմբագրել</div>
  </c-pophover>
</b>
```

Հիմնական դասավորությունը, չափը (18x18px) և սավառնողի գույնի փոփոխությունները միավորվել են `.Ico > c-pophover > a`-ում: Մշակողները միայն պետք է առանձին նշեն համապատասխան պատկերակի ֆայլը, օրինակ՝
```css
.Edit > c-pophover > a {
  mask-image: url("./svg/edit.svg")
}
```
