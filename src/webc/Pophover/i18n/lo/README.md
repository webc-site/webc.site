# ກະຕຸ້ນຟອງລອຍ

ສະແດງຂໍ້ມູນຄໍາແນະນໍາເຄື່ອງມື hover ເມື່ອຫນູຖືກເລື່ອນຫຼືອົງປະກອບຖືກເປີດໃຊ້.

- ອີງໃສ່ອົງປະກອບເວັບພື້ນເມືອງ, ນ້ຳໜັກເບົາ ແລະ ປະສິດທິພາບສູງ
- ສະ​ຫນັບ​ສະ​ຫນູນ​ການ​ປັບ​ຕໍາ​ແຫນ່ງ​ທາງ​ກວ້າງ​ຂວາງ​ໃນ​ສີ່​ທິດ​ທາງ (class="top/bottom/left/right")
- ມັນໃຊ້ພື້ນຫຼັງແບບສີພື້ນແກ້ວເປັນສີດຳ ແລະຕົວລະຄອນສີຂາວ, ມີຈຸດເດັ່ນຂອງຂອບຮູບຊົງທີ່ສວຍງາມ.
- ການຄິດໄລ່ການປັບຕົວຕັ້ງຢູ່ໃນຕົວ, ປັບທິດທາງປັອບອັບໂດຍອັດຕະໂນມັດຕາມພື້ນທີ່ທີ່ຍັງເຫຼືອຢູ່ໃນຊ່ອງເບິ່ງ
- ໃຊ້ `slot="pop"` ເພື່ອສະແດງເນື້ອຫາການເຕືອນແບບເລື່ອນ

## ໃຊ້ຕົວຢ່າງ

```html
<!-- ຄິດ​ໄລ່​ອັດ​ຕະ​ໂນ​ມັດ​ທິດ​ທາງ (ແນະ​ນໍາ​, ບໍ່​ຈໍາ​ເປັນ​ຕ້ອງ​ປະ​ກາດ​ຫ້ອງ​) -->
<c-pophover>
  <button>ອັດຕະໂນມັດ</button>
  <div slot="pop">ນີ້ແມ່ນປາຍຟອງ</div>
</c-pophover>

<!-- ທິດທາງຄົງທີ່ -->
<c-pophover class="left">
  <button>ຊ້າຍ</button>
  <div slot="pop">ນີ້ແມ່ນປາຍຟອງ</div>
</c-pophover>

<c-pophover class="right">
  <button>ຖືກຕ້ອງ</button>
  <div slot="pop">ນີ້ແມ່ນປາຍຟອງ</div>
</c-pophover>

<c-pophover class="bottom">
  <button>ລົງ</button>
  <div slot="pop">ນີ້ແມ່ນປາຍຟອງ</div>
</c-pophover>

<c-pophover class="top">
  <button>ດີກວ່າ</button>
  <div slot="pop">ນີ້ແມ່ນປາຍຟອງ</div>
</c-pophover>
```

## ຕົວແປ CSS

ຮູບແບບແບບກຳນົດເອງແມ່ນຮອງຮັບຜ່ານຕົວແປ CSS ຕໍ່ໄປນີ້:

| ຊື່ຕົວແປ | ຄ່າເລີ່ມຕົ້ນ | ສະແດງໃຫ້ເຫັນ |
| :--- | :--- | :--- |
| `--pophover-gap` | `8px` | ຊ່ອງຫວ່າງລະຫວ່າງປາຍເຄື່ອງມືຟອງ ແລະອົງປະກອບກະຕຸ້ນ |
| `--pophover-margin` | `24px` | ຂອບດ້ານຄວາມປອດໄພຂອງ Viewport edge ໃນລະຫວ່າງການຄິດໄລ່ການຈັດຕຳແໜ່ງແບບປັບຕົວ |
| `--pophover-bg-top` | `#18181ce0` | Bubble gradient ສີພື້ນຫຼັງ |
| `--pophover-bg-mid` | `#121215e5` | ສີພື້ນຫຼັງຂອງລູກສອນຢູ່ດ້ານຊ້າຍ ແລະຂວາຂອງຟອງ |
| `--pophover-bg-bottom` | `#0c0c0feb` | ສີຂອງລູກສອນທາງລຸ່ມ ແລະດ້ານເທິງ ແລະລຸ່ມຂອງພື້ນຫຼັງ bubble gradient |

## ໄອຄອນຮູບແບບຄຳແນະນຳ (`.Ico`)

ສໍາລັບປະເພດໄອຄອນອັນບໍລິສຸດປຸ່ມເຕືອນແບບເລື່ອນ (ເຊັ່ນ: "ແກ້ໄຂ", "ລຶບ", ແລະອື່ນໆ), ທ່ານສາມາດນຳໃຊ້ໄອຄອນແບບຮວມກັນ ແລະຮູບແບບການເລື່ອນໄດ້ໂດຍການເພີ່ມ `.Ico`ຊື່ຊັ້ນໃສ່ກັບຖັງ ຫຼືອົງປະກອບທາງນອກຂອງມັນເອງ.

### ຕົວຢ່າງໂຄງສ້າງ

```html
<b class="Edit Ico">
  <c-pophover>
    <a></a>
    <div slot="pop">ແກ້ໄຂ</div>
  </c-pophover>
</b>
```

ການຈັດວາງພື້ນຖານ, ຂະໜາດ (18x18px), ແລະການປ່ຽນສີເລື່ອນໄດ້ຖືກຮວມຢູ່ໃນ `.Ico > c-pophover > a`. ນັກພັດທະນາພຽງແຕ່ຕ້ອງການລະບຸໄຟລ໌ໄອຄອນທີ່ສອດຄ້ອງກັນແຍກຕ່າງຫາກ, ເຊັ່ນ:
```css
.Edit > c-pophover > a {
  mask-image: url("./svg/edit.svg")
}
```
