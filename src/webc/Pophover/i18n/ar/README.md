# موجه الفقاعة العائمة

عرض معلومات تلميح أداة التمرير عند تحريك الماوس أو تنشيط العنصر.

- استنادًا إلى مكون الويب الأصلي، فهو خفيف الوزن وعالي الأداء
- يدعم تحديد المواقع المكانية التكيفية في أربعة اتجاهات (الفئة = "أعلى/أسفل/يسار/يمين")
- يستخدم خلفية متدرجة من الزجاج البلوري الأسود وأحرفًا بيضاء، مع إبرازات محيطية رائعة للحافة المادية.
- يقوم الحساب التكيفي للموضع المدمج بضبط الاتجاه المنبثق تلقائيًا وفقًا للمساحة المتبقية في إطار العرض
- استخدم `slot="pop"` لعرض محتوى المطالبة العائم

## استخدم العرض التوضيحي

```html
<!-- حساب الاتجاه تلقائيًا (موصى به، لا داعي للإعلان عن الفصل) -->
<c-pophover>
  <button>تلقائي</button>
  <div slot="pop">هذه نصيحة فقاعة</div>
</c-pophover>

<!-- اتجاه ثابت -->
<c-pophover class="left">
  <button>غادر</button>
  <div slot="pop">هذه نصيحة فقاعة</div>
</c-pophover>

<c-pophover class="right">
  <button>يمين</button>
  <div slot="pop">هذه نصيحة فقاعة</div>
</c-pophover>

<c-pophover class="bottom">
  <button>تحت</button>
  <div slot="pop">هذه نصيحة فقاعة</div>
</c-pophover>

<c-pophover class="top">
  <button>أرقى</button>
  <div slot="pop">هذه نصيحة فقاعة</div>
</c-pophover>
```

## متغيرات CSS

يتم دعم الأنماط المخصصة عبر متغيرات CSS التالية:

| اسم متغير | القيمة الافتراضية | يوضح |
| :--- | :--- | :--- |
| `--pophover-gap` | `8px` | المسافة بين تلميح أداة الفقاعة وعنصر التشغيل |
| `--pophover-margin` | `24px` | هامش أمان حافة منفذ العرض أثناء حساب تحديد المواقع التكيفي |
| `--pophover-bg-top` | `#18181ce0` | فقاعة خلفية متدرجة اللون العلوي |
| `--pophover-bg-mid` | `#121215e5` | لون خلفية الأسهم الموجودة على الجانبين الأيسر والأيمن للفقاعة |
| `--pophover-bg-bottom` | `#0c0c0feb` | لون الأسهم الجانبية السفلية والعلوية والسفلية لخلفية التدرج الفقاعي |

## نمط تلميح أداة الرمز (`.Ico`)

بالنسبة لأزرار المطالبة العائمة من نوع الأيقونة الخالصة (مثل "تحرير"، "حذف"، وما إلى ذلك)، يمكنك تطبيق رمز موحد ونمط التمرير عن طريق إضافة اسم الفئة `.Ico` إلى الحاوية الخارجية أو المكون نفسه.

### مثال الهيكل

```html
<b class="Edit Ico">
  <c-pophover>
    <a></a>
    <div slot="pop">يحرر</div>
  </c-pophover>
</b>
```

تم توحيد التخطيط الأساسي والحجم (18 × 18 بكسل) وتغييرات لون التمرير في `.Ico > c-pophover > a`. يحتاج المطورون فقط إلى تحديد ملف الرمز المقابل بشكل منفصل، مثل:
```css
.Edit > c-pophover > a {
  mask-image: url("./svg/edit.svg")
}
```
