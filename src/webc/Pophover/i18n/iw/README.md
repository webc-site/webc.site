# הודעת בועה צפה

הצג מידע על הסבר על ריחוף כאשר העכבר מרחף או האלמנט מופעל.

- מבוסס על רכיב אינטרנט מקורי, קל משקל וביצועים גבוהים
- תומך במיקום מרחבי אדפטיבי בארבעה כיוונים (class="top/bottom/left/right")
- הוא משתמש ברקע שיפוע מרקם זכוכית חלבית שחור ותווים לבנים, עם הדגשות מתאר קצה פיזיים מעודנים.
- חישוב מסתגל למיקום מובנה, מתאים אוטומטית את כיוון המוקפץ בהתאם לשטח הנותר בשדה התצוגה
- השתמש ב`slot="pop"` כדי להציג תוכן הנחיה צף

## השתמש בהדגמה

```html
<!-- חישוב אוטומטי של כיוון (מומלץ, אין צורך להכריז על מחלקה) -->
<c-pophover>
  <button>אוֹטוֹמָטִי</button>
  <div slot="pop">זה טיפ בועה</div>
</c-pophover>

<!-- כיוון קבוע -->
<c-pophover class="left">
  <button>שְׁמֹאל</button>
  <div slot="pop">זה טיפ בועה</div>
</c-pophover>

<c-pophover class="right">
  <button>יָמִינָה</button>
  <div slot="pop">זה טיפ בועה</div>
</c-pophover>

<c-pophover class="bottom">
  <button>לְמַטָה</button>
  <div slot="pop">זה טיפ בועה</div>
</c-pophover>

<c-pophover class="top">
  <button>מְעוּלֶה</button>
  <div slot="pop">זה טיפ בועה</div>
</c-pophover>
```

## משתני CSS

סגנונות מותאמים אישית נתמכים באמצעות משתני ה-CSS הבאים:

| שם המשתנה | ערך ברירת המחדל | לְהַדגִים |
| :--- | :--- | :--- |
| `--pophover-gap` | `8px` | הרווח בין קצה הכלי של הבועה לאלמנט המפעיל |
| `--pophover-margin` | `24px` | מרווח בטיחות קצה של Viewport במהלך חישוב מיקום אדפטיבי |
| `--pophover-bg-top` | `#18181ce0` | צבע עליון רקע שיפוע בועה |
| `--pophover-bg-mid` | `#121215e5` | צבע הרקע של החצים בצד שמאל וימין של הבועה |
| `--pophover-bg-bottom` | `#0c0c0feb` | הצבע של החצים בצד התחתון והעליון והתחתון של רקע שיפוע הבועה |

## סגנון תיאור כלי סמל (`.Ico`)

עבור כפתורי הנחיה צפים מסוג סמל טהור (כגון "ערוך", "מחק" וכו'), אתה יכול להחיל סמל מאוחד וסגנון ריחוף על ידי הוספת שם המחלקה `.Ico` למיכל החיצוני או לרכיב עצמו.

### דוגמה למבנה

```html
<b class="Edit Ico">
  <c-pophover>
    <a></a>
    <div slot="pop">לַעֲרוֹך</div>
  </c-pophover>
</b>
```

הפריסה הבסיסית, הגודל (18x18px) ושינויי צבע הריחוף אוחדו ב`.Ico > c-pophover > a`. מפתחים צריכים רק לציין בנפרד את קובץ הסמלים המתאים, כגון:
```css
.Edit > c-pophover > a {
  mask-image: url("./svg/edit.svg")
}
```
