# αιωρούμενη φυσαλίδα

Εμφάνιση πληροφοριών συμβουλής εργαλείου αιώρησης όταν το ποντίκι είναι τοποθετημένο ή το στοιχείο είναι ενεργοποιημένο.

- Βασισμένο στο εγγενές στοιχείο Web, ελαφρύ και υψηλής απόδοσης
- Υποστηρίζει προσαρμοστική χωρική τοποθέτηση σε τέσσερις κατευθύνσεις (class="top/bottom/left/right")
- Χρησιμοποιεί φόντο με ντεγκραντέ υφή μαύρου παγωμένου γυαλιού και λευκούς χαρακτήρες, με εξαιρετικές φυσικές ανταύγειες του περιγράμματος των άκρων.
- Ενσωματωμένος προσαρμοστικός υπολογισμός θέσης, προσαρμόζει αυτόματα την κατεύθυνση του αναδυόμενου παραθύρου σύμφωνα με τον υπόλοιπο χώρο στη θύρα προβολής
- Χρησιμοποιήστε το `slot="pop"` για να εμφανίσετε περιεχόμενο κινητής προτροπής

## Χρησιμοποιήστε το demo

```html
<!-- Αυτόματος υπολογισμός κατεύθυνσης (συνιστάται, δεν χρειάζεται να δηλώσετε τάξη) -->
<c-pophover>
  <button>αυτόματο</button>
  <div slot="pop">Αυτή είναι μια άκρη φούσκας</div>
</c-pophover>

<!-- σταθερή κατεύθυνση -->
<c-pophover class="left">
  <button>Αριστερά</button>
  <div slot="pop">Αυτή είναι μια άκρη φούσκας</div>
</c-pophover>

<c-pophover class="right">
  <button>δικαίωμα</button>
  <div slot="pop">Αυτή είναι μια άκρη φούσκας</div>
</c-pophover>

<c-pophover class="bottom">
  <button>Κάτω</button>
  <div slot="pop">Αυτή είναι μια άκρη φούσκας</div>
</c-pophover>

<c-pophover class="top">
  <button>ανώτερος</button>
  <div slot="pop">Αυτή είναι μια άκρη φούσκας</div>
</c-pophover>
```

## Μεταβλητές CSS

Τα προσαρμοσμένα στυλ υποστηρίζονται μέσω των ακόλουθων μεταβλητών CSS:

| όνομα μεταβλητής | προεπιλεγμένη τιμή | εικονογραφώ |
| :--- | :--- | :--- |
| `--pophover-gap` | `8px` | Ο χώρος μεταξύ της επεξήγησης εργαλείου φυσαλίδων και του στοιχείου ενεργοποίησης |
| `--pophover-margin` | `24px` | Περιθώριο ασφαλείας άκρου θύρας προβολής κατά τον υπολογισμό προσαρμοστικής τοποθέτησης |
| `--pophover-bg-top` | `#18181ce0` | Φούσκα ντεγκραντέ φόντο χρώμα επάνω |
| `--pophover-bg-mid` | `#121215e5` | Το χρώμα φόντου των βελών στην αριστερή και δεξιά πλευρά της φυσαλίδας |
| `--pophover-bg-bottom` | `#0c0c0feb` | Το χρώμα του κάτω και του άνω και κάτω πλευρικού βέλους του φόντου ντεγκραντέ φυσαλίδων |

## Στυλ συμβουλής εργαλείου εικονιδίου (`.Ico`)

Για πλήκτρα κινητής προτροπής καθαρού τύπου εικονιδίου (όπως "Επεξεργασία", "Διαγραφή" κ.λπ.), μπορείτε να εφαρμόσετε ένα ενοποιημένο εικονίδιο και να τοποθετήσετε το στυλ του ποντικιού προσθέτοντας το όνομα κλάσης `.Ico` στο ίδιο το εξωτερικό κοντέινερ ή στοιχείο.

### Παράδειγμα δομής

```html
<b class="Edit Ico">
  <c-pophover>
    <a></a>
    <div slot="pop">εκδίδω</div>
  </c-pophover>
</b>
```

Η βασική διάταξη, το μέγεθος (18x18px) και οι αλλαγές χρώματος του δείκτη έχουν ενοποιηθεί στο `.Ico > c-pophover > a`. Οι προγραμματιστές χρειάζεται μόνο να καθορίσουν το αντίστοιχο αρχείο εικονιδίων ξεχωριστά, όπως:
```css
.Edit > c-pophover > a {
  mask-image: url("./svg/edit.svg")
}
```
