# すりガラスの背景とエッジのハイライト効果

半透明のボーダーと内側のシャドウにより、マイクロレリーフエッジのハイライトが作成されます。
ボタンと入力ボックスのコンポーネントの基本的なスタイルの依存関係として。

## デモを使用する

```html
<main class="Lg">
  <article>
    <h3>タイトル</h3>
    <pre>コンテンツテキスト</pre>
  </article>
</main>
```

## スタイルクラス

### `.Lg`
カードの基本的なスタイル。 `backdrop-filter`ぼかし、`border`マイクロエンボス境界線、および`box-shadow`インナーシャドウを適用します。ホバー時に`svg/glass.svg#h`フィルターを適用します。

### `main.Lg`
ページのメイン ブロック コンテナー。角が丸い`24px`、レスポンシブパディング付き（幅が`800px`より小さい場合は`16px`、それ以外の場合は`24px`）。

### `.Btn`, `.BtnC`, `.Input`
基本コンポーネントのクラス名。このスタイルを導入すると、柔軟なレイアウト、階層関係、すりガラスの基本プロパティが自動的に得られます。
