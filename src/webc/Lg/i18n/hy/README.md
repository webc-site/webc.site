# Ցրտահարված ապակու ֆոն և եզրերի ընդգծման էֆեկտ

Կիսաթափանցիկ եզրագիծը և ներքին ստվերը ստեղծում են միկրոռելիեֆային եզրեր:
Որպես կոճակի և մուտքագրման տուփի բաղադրիչների հիմնական ոճային կախվածություն:

## Օգտագործեք ցուցադրությունը

```html
<main class="Lg">
  <article>
    <h3>վերնագիր</h3>
    <pre>Բովանդակության տեքստ</pre>
  </article>
</main>
```

## ոճի դաս

### `.Lg`
Քարտի հիմնական ոճը. Կիրառեք `backdrop-filter` լղոզումը, `border` միկրո դաջված եզրագիծը և `box-shadow` ներքին ստվերը: Կիրառել `svg/glass.svg#h` զտիչը սավառնելիս:

### `main.Lg`
Էջի հիմնական բլոկային կոնտեյները: Կլորացված անկյուններ `24px` արձագանքող լիցքով (`16px` եթե լայնությունը `800px`-ից փոքր է, `24px` հակառակ դեպքում):

### `.Btn`, `.BtnC`, `.Input`
Հիմնական բաղադրիչի դասի անվանումը. Այս ոճը ներմուծելուց հետո դուք ավտոմատ կերպով կստանաք ճկուն դասավորություն, հիերարխիկ հարաբերություններ և սառեցված ապակու հիմնական հատկությունները:
