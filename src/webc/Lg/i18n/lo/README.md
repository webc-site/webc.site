# ພື້ນຫຼັງແກ້ວມີອາກາດໜາວ ແລະ ເອັບເຟັກຂອບຂອບ

ຂອບເຄິ່ງໂປ່ງໃສ ແລະເງົາພາຍໃນສ້າງຈຸດເດັ່ນຂອງຂອບການບັນເທົາທຸກ.
ໃນ​ຖາ​ນະ​ເປັນ​ແບບ​ພື້ນ​ຖານ​ການ​ຂຶ້ນ​ກັບ​ປຸ່ມ​ແລະ​ອົງ​ປະ​ກອບ​ຂອງ​ປ່ອງ​ປ້ອນ​ຂໍ້​ມູນ​.

## ໃຊ້ຕົວຢ່າງ

```html
<main class="Lg">
  <article>
    <h3>ຫົວຂໍ້</h3>
    <pre>ເນື້ອໃນຂໍ້ຄວາມ</pre>
  </article>
</main>
```

## ຫ້ອງຮຽນແບບ

### `.Lg`
ຮູບແບບພື້ນຖານຂອງບັດ. ນຳໃຊ້ `backdrop-filter`ມົວ, `border`ຂອບຂະໜາດນ້ອຍ, ແລະ `box-shadow`ເງົາພາຍໃນ. ນຳໃຊ້ `svg/glass.svg#h`ຕົວກັ່ນຕອງໃສ່ເທິງເຄື່ອງ.

### `main.Lg`
ຖັງບລັອກຫຼັກຂອງໜ້າ. ມຸມມົນ `24px`ມີ padding ຕອບສະໜອງ (`16px`ຖ້າຄວາມກວ້າງໜ້ອຍກວ່າ `800px`, `24px`ຖ້າບໍ່ດັ່ງນັ້ນ).

### `.Btn`, `.BtnC`, `.Input`
ຊື່ຊັ້ນອົງປະກອບພື້ນຖານ. ຫຼັງຈາກແນະນໍາຮູບແບບນີ້, ທ່ານຈະໄດ້ຮັບຮູບແບບທີ່ມີຄວາມຍືດຫຍຸ່ນ, ການພົວພັນແບບລໍາດັບຊັ້ນແລະຄຸນສົມບັດພື້ນຖານຂອງແກ້ວອາກາດຫນາວໂດຍອັດຕະໂນມັດ.
