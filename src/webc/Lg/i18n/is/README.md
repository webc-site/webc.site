# Bakgrunnur úr matt gleri og hápunktur á brún

Hálfgagnsæir rammar og innri skuggi skapa örléttir brúnir.
Sem grunnstíll háð hnappa og inntaksbox íhlutum.

## Notaðu kynninguna

```html
<main class="Lg">
  <article>
    <h3>titill</h3>
    <pre>Efnistexti</pre>
  </article>
</main>
```

## stílaflokkur

### `.Lg`
Grunnstíll korta. Notaðu `backdrop-filter` óskýrleika, `border` örupphleyptan ramma og `box-shadow` innri skugga. Notaðu `svg/glass.svg#h` síu á sveimi.

### `main.Lg`
Aðalblokkagámur síðunnar. Ávöl horn `24px` með móttækilegum bólstrun (`16px` ef breiddin er minni en `800px`, `24px` annars).

### `.Btn`, `.BtnC`, `.Input`
Nafn grunnþáttaflokks. Eftir að þú hefur kynnt þennan stíl færðu sjálfkrafa sveigjanlegt skipulag, stigveldistengsl og grunneiginleika mattglers.
