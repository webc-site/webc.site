# फ्रोस्टेड ग्लास पृष्ठभूमि र किनारा हाइलाइट प्रभाव

अर्ध-पारदर्शी किनारा र भित्री छायाले माइक्रो-रिलीफ एज हाइलाइटहरू सिर्जना गर्दछ।
बटन र इनपुट बक्स घटकहरूको आधारभूत शैली निर्भरताको रूपमा।

## डेमो प्रयोग गर्नुहोस्

```html
<main class="Lg">
  <article>
    <h3>शीर्षक</h3>
    <pre>सामग्री पाठ</pre>
  </article>
</main>
```

## शैली वर्ग

### `.Lg`
कार्ड आधारभूत शैली। `backdrop-filter` ब्लर, `border` माइक्रो-इम्बोस्ड किनारा, र `box-shadow` भित्री छाया लागू गर्नुहोस्। होभरमा `svg/glass.svg#h` फिल्टर लागू गर्नुहोस्।

### `main.Lg`
पृष्ठको मुख्य ब्लक कन्टेनर। गोलाकार कुनाहरू `24px` प्रतिक्रियाशील प्याडिङको साथ (`16px` यदि चौडाइ `800px` भन्दा कम छ भने, `24px` अन्यथा)।

### `.Btn`, `.BtnC`, `.Input`
आधारभूत घटक वर्ग नाम। यो शैली परिचय गरेपछि, तपाईंले स्वचालित रूपमा लचिलो लेआउट, पदानुक्रमिक सम्बन्धहरू र फ्रोस्टेड ग्लासको आधारभूत गुणहरू प्राप्त गर्नुहुनेछ।
