# ଫ୍ରଷ୍ଟଡ୍ ଗ୍ଲାସ୍ ପୃଷ୍ଠଭୂମି ଏବଂ ଧାର ହାଇଲାଇଟ୍ ଇଫେକ୍ଟ |

ସେମି-ସ୍ୱଚ୍ଛ ସୀମା ଏବଂ ଭିତର ଛାୟା ମାଇକ୍ରୋ-ରିଲିଫ୍ ଧାରାର ହାଇଲାଇଟ୍ ସୃଷ୍ଟି କରେ |
ବଟନ୍ ଏବଂ ଇନପୁଟ୍ ବକ୍ସ ଉପାଦାନଗୁଡ଼ିକର ମ basic ଳିକ ଶ style ଳୀ ନିର୍ଭରଶୀଳତା |

## ଡେମୋ ବ୍ୟବହାର କରନ୍ତୁ |

```html
<main class="Lg">
  <article>
    <h3>ଆଖ୍ୟା</h3>
    <pre>ବିଷୟବସ୍ତୁ ପାଠ</pre>
  </article>
</main>
```

## ଶ style ଳୀ ଶ୍ରେଣୀ

### `.Lg`
କାର୍ଡ ମ basic ଳିକ ଶ style ଳୀ | `backdrop-filter` ଅସ୍ପଷ୍ଟ, `border` ମାଇକ୍ରୋ-ଏମ୍ବୋଡ୍ ସୀମା, ଏବଂ `box-shadow` ଭିତର ଛାୟା ପ୍ରୟୋଗ କରନ୍ତୁ | ହୋଭରରେ `svg/glass.svg#h` ଫିଲ୍ଟର୍ ପ୍ରୟୋଗ କରନ୍ତୁ |

### `main.Lg`
ପୃଷ୍ଠାର ମୁଖ୍ୟ ବ୍ଲକ ପାତ୍ର | ଗୋଲାକାର କୋଣଗୁଡିକ `24px` ପ୍ରତିକ୍ରିୟାଶୀଳ ପ୍ୟାଡିଂ ସହିତ (`16px` ଯଦି ମୋଟେଇ `800px` ଠାରୁ କମ୍, `24px` ଅନ୍ୟଥା) |

### `.Btn`, `.BtnC`, `.Input`
ମ Basic ଳିକ ଉପାଦାନ ଶ୍ରେଣୀ ନାମ | ଏହି ଶ style ଳୀକୁ ଉପସ୍ଥାପନ କରିବା ପରେ, ଆପଣ ସ୍ୱୟଂଚାଳିତ ଭାବରେ ନମନୀୟ ଲେଆଉଟ୍, କ୍ରମିକ ସମ୍ପର୍କ ଏବଂ ତରକାରୀ ଗ୍ଲାସର ମ basic ଳିକ ଗୁଣ ପାଇବେ |
