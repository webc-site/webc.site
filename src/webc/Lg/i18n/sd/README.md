# ڀريل گلاس پس منظر ۽ ڪنڊ نمايان اثر

نيم شفاف سرحد ۽ اندروني ڇانو ٺاهي مائڪرو-رليف ايج نمايان.
جيئن ته بٽڻ ۽ ان پٽ باڪس جي اجزاء جو بنيادي انداز انحصار.

## ڊيمو استعمال ڪريو

```html
<main class="Lg">
  <article>
    <h3>عنوان</h3>
    <pre>مواد جو متن</pre>
  </article>
</main>
```

## انداز ڪلاس

### `.Lg`
ڪارڊ بنيادي انداز. لاڳو ڪريو `backdrop-filter` blur، `border` micro-embossed بارڊر، ۽ `box-shadow` اندروني ڇانو. هور تي `svg/glass.svg#h` فلٽر لاڳو ڪريو.

### `main.Lg`
صفحي جو مکيه بلاڪ ڪنٽينر. گول ڪنڊ `24px` جوابي پيڊنگ سان (`16px` جيڪڏهن ويڪر `800px` کان گهٽ آهي، `24px` ٻي صورت ۾).

### `.Btn`, `.BtnC`, `.Input`
بنيادي جزو طبقي جو نالو. هن انداز کي متعارف ڪرائڻ کان پوء، توهان خودڪار طريقي سان لچڪدار ترتيب، ترتيب واري رشتي ۽ ڀريل شيشي جي بنيادي ملڪيت حاصل ڪندا.
