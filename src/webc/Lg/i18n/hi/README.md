# फ्रॉस्टेड ग्लास पृष्ठभूमि और किनारा हाइलाइट प्रभाव

अर्ध-पारदर्शी बॉर्डर और आंतरिक छाया सूक्ष्म-राहत किनारे हाइलाइट बनाते हैं।
बटन और इनपुट बॉक्स घटकों की मूल शैली निर्भरता के रूप में।

## डेमो का प्रयोग करें

```html
<main class="Lg">
  <article>
    <h3>शीर्षक</h3>
    <pre>सामग्री पाठ</pre>
  </article>
</main>
```

## शैली वर्ग

### `.Lg`
कार्ड मूल शैली. `backdrop-filter` धुंधलापन, `border` माइक्रो-उभरा बॉर्डर और `box-shadow` आंतरिक छाया लागू करें। होवर पर `svg/glass.svg#h` फ़िल्टर लागू करें।

### `main.Lg`
पेज का मुख्य ब्लॉक कंटेनर. गोलाकार कोने `24px` प्रतिक्रियाशील पैडिंग के साथ (`16px` यदि चौड़ाई `800px` से कम है, तो `24px` अन्यथा)।

### `.Btn`, `.BtnC`, `.Input`
मूल घटक वर्ग का नाम. इस शैली को पेश करने के बाद, आप स्वचालित रूप से लचीले लेआउट, पदानुक्रमित रिश्ते और फ्रॉस्टेड ग्लास के बुनियादी गुण प्राप्त करेंगे।
