# फ्रॉस्टेड ग्लास बॅकग्राउंड आणि एज हायलाइट इफेक्ट

अर्ध-पारदर्शक सीमा आणि आतील सावली सूक्ष्म-रिलीफ एज हायलाइट तयार करतात.
बटण आणि इनपुट बॉक्स घटकांची मूलभूत शैली अवलंबित्व म्हणून.

## डेमो वापरा

```html
<main class="Lg">
  <article>
    <h3>शीर्षक</h3>
    <pre>सामग्री मजकूर</pre>
  </article>
</main>
```

## शैली वर्ग

### `.Lg`
कार्ड मूलभूत शैली. `backdrop-filter` ब्लर, `border` मायक्रो एम्बॉस्ड बॉर्डर आणि `box-shadow` अंतर्गत सावली लागू करा. होवरवर `svg/glass.svg#h` फिल्टर लागू करा.

### `main.Lg`
पृष्ठाचा मुख्य ब्लॉक कंटेनर. गोलाकार कोपरे `24px` रिस्पॉन्सिव्ह पॅडिंगसह (`16px` रुंदी `800px` पेक्षा कमी असल्यास, `24px` अन्यथा).

### `.Btn`, `.BtnC`, `.Input`
मूलभूत घटक वर्गाचे नाव. या शैलीचा परिचय दिल्यानंतर, आपण आपोआप लवचिक मांडणी, श्रेणीबद्ध संबंध आणि फ्रॉस्टेड ग्लासचे मूलभूत गुणधर्म प्राप्त कराल.
