# Заднина од матирано стакло и ефект на истакнување на рабовите

Полутранспарентната граница и внатрешната сенка создаваат микрорелјефни рабови.
Како основна стилска зависност на компонентите на копчето и влезното поле.

## Користете го демо

```html
<main class="Lg">
  <article>
    <h3>наслов</h3>
    <pre>Текст на содржината</pre>
  </article>
</main>
```

## класа стил

### `.Lg`
Основен стил на картичката. Примени `backdrop-filter` заматување, `border` микро-релјефна граница и `box-shadow` внатрешна сенка. Примени `svg/glass.svg#h` филтер на лебди.

### `main.Lg`
Главниот блок контејнер на страницата. Заоблени агли `24px` со респонзивно полнење (`16px` ако ширината е помала од `800px`, инаку `24px`).

### `.Btn`, `.BtnC`, `.Input`
Име на класата на основна компонента. По воведувањето на овој стил, автоматски ќе добиете флексибилен распоред, хиерархиски односи и основни својства на матираното стакло.
