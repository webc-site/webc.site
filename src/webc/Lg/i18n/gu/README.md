# ફ્રોસ્ટેડ ગ્લાસ બેકગ્રાઉન્ડ અને એજ હાઇલાઇટ ઇફેક્ટ

અર્ધ-પારદર્શક સરહદ અને આંતરિક પડછાયો માઇક્રો-રિલીફ એજ હાઇલાઇટ્સ બનાવે છે.
બટન અને ઇનપુટ બોક્સ ઘટકોની મૂળભૂત શૈલી નિર્ભરતા તરીકે.

## ડેમોનો ઉપયોગ કરો

```html
<main class="Lg">
  <article>
    <h3>શીર્ષક</h3>
    <pre>સામગ્રી ટેક્સ્ટ</pre>
  </article>
</main>
```

## શૈલી વર્ગ

### `.Lg`
કાર્ડ મૂળભૂત શૈલી. `backdrop-filter` બ્લર, `border` માઇક્રો-એમ્બોસ્ડ બોર્ડર અને `box-shadow` આંતરિક પડછાયો લાગુ કરો. હોવર પર `svg/glass.svg#h` ફિલ્ટર લાગુ કરો.

### `main.Lg`
પૃષ્ઠનું મુખ્ય બ્લોક કન્ટેનર. ગોળાકાર ખૂણાઓ `24px` રિસ્પોન્સિવ પેડિંગ સાથે (`16px` જો પહોળાઈ `800px` કરતાં ઓછી હોય, `24px` અન્યથા).

### `.Btn`, `.BtnC`, `.Input`
મૂળભૂત ઘટક વર્ગનું નામ. આ શૈલી રજૂ કર્યા પછી, તમે આપમેળે લવચીક લેઆઉટ, વંશવેલો સંબંધો અને હિમાચ્છાદિત કાચના મૂળભૂત ગુણધર્મો મેળવશો.
