# పేజీ అంచున పాప్-అప్ విండోను ప్రాంప్ట్ చేయండి

- **ఫంక్షన్ కాల్**: సాధారణ మరియు ఎర్రర్ ప్రాంప్ట్ ఇంటర్‌ఫేస్‌లను అందిస్తుంది
- **ఆటో స్టాక్**: బహుళ చిట్కాలు స్వయంచాలకంగా అంతరాన్ని లెక్కించి నిలువుగా పేర్చుతాయి
- **షెడ్యూల్డ్ మరియు మాన్యువల్ షట్‌డౌన్**: షెడ్యూల్డ్ మరియు మాన్యువల్ షట్‌డౌన్‌కు మద్దతు ఇస్తుంది

## డెమో ఉపయోగించండి

```js
import { toast, toastErr } from "webc.site/js/Toast.js";

toast((el) => {
  el.textContent = "ప్రాంప్ట్ కంటెంట్";
});

toastErr((el) => {
  el.textContent = "ఎర్రర్ కంటెంట్";
});

toast(
  (el) => {
    el.textContent = "99 సెకన్ల తర్వాత స్వయంచాలకంగా మూసివేయండి";
  },
  99
);
```

## ఇంటర్ఫేస్ పారామితులు

### toast(render, timeout)

ప్రాంప్ట్‌లను సృష్టించండి మరియు ప్రదర్శించండి.

- `render`: `(el) => void`, కాల్‌బ్యాక్ రెండరింగ్, `el` అనేది ప్రాంప్ట్ బాక్స్ మూలకం.
- `timeout`: `Number`, గడువు ముగిసిన సెకన్లు. డిఫాల్ట్ 9. స్వయంచాలకంగా షట్ డౌన్ కాకుండా ఉండేలా 0కి సెట్ చేయండి.
- రిటర్న్ విలువ: ప్రాంప్ట్ బాక్స్ మూలకం. `el.close()` ఆఫ్‌కి మద్దతు ఇస్తుంది.

### toastErr(render, timeout)

దోష సందేశాలను సృష్టించండి మరియు ప్రదర్శించండి. పారామితులు `toast`, `.ERR` స్టైల్ క్లాస్‌తో సమానంగా ఉంటాయి.

## శైలి తరగతి

- `.Toast`: ప్రాంప్ట్ బాక్స్ బేస్ క్లాస్.
- `.ERR`: ఎర్రర్ స్థితి.
- `.animated` / `.fadeInR` / `.fadeOutR`: యానిమేటెడ్ ట్రాన్సిషన్.
- `.x`: క్లోజ్ బటన్.
