# หน้าต่างป๊อปอัปแจ้งที่ขอบของหน้า

- **การเรียกใช้ฟังก์ชัน**: จัดเตรียมอินเทอร์เฟซปกติและพร้อมท์ข้อผิดพลาด
- **เรียงซ้อนอัตโนมัติ**: เคล็ดลับหลายรายการจะคำนวณระยะห่างและเรียงซ้อนในแนวตั้งโดยอัตโนมัติ
- **การปิดระบบตามกำหนดเวลาและด้วยตนเอง**: รองรับการปิดระบบตามกำหนดเวลาและด้วยตนเอง

## ใช้การสาธิต

```js
import { toast, toastErr } from "webc.site/js/Toast.js";

toast((el) => {
  el.textContent = "เนื้อหาพร้อมท์";
});

toastErr((el) => {
  el.textContent = "เนื้อหามีข้อผิดพลาด";
});

toast(
  (el) => {
    el.textContent = "ปิดโดยอัตโนมัติหลังจาก 99 วินาที";
  },
  99
);
```

## พารามิเตอร์อินเทอร์เฟซ

### toast(render, timeout)

สร้างและแสดงพร้อมท์

- `render`: `(el) => void`, แสดงผลการโทรกลับ, `el` เป็นองค์ประกอบกล่องพร้อมท์
- `timeout`: `Number` วินาทีหมดเวลา ค่าเริ่มต้น 9. ตั้งค่าเป็น 0 เพื่อไม่ปิดเครื่องโดยอัตโนมัติ
- ส่งกลับค่า: องค์ประกอบกล่องพร้อมท์ รองรับ `el.close()` ปิด

### toastErr(render, timeout)

สร้างและแสดงข้อความแสดงข้อผิดพลาด พารามิเตอร์เหมือนกับ `toast` โดยมีคลาสสไตล์ `.ERR`

## คลาสสไตล์

- `.Toast`: คลาสฐานกล่องพร้อมท์
- `.ERR`: สถานะข้อผิดพลาด
- `.animated` / `.fadeInR` / `.fadeOutR`: การเปลี่ยนภาพเคลื่อนไหว
- `.x`: ปุ่มปิด
