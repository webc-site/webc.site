# Побарајте скокачки прозорец на работ на страницата

- **Повик за функција**: Обезбедува нормални интерфејси и интерфејси со барање за грешки
- **Автоматско редење**: Повеќе совети автоматски го пресметуваат растојанието и се редат вертикално
- **Закажано и рачно исклучување**: Поддржува закажано и рачно исклучување

## Користете го демо

```js
import { toast, toastErr } from "webc.site/js/Toast.js";

toast((el) => {
  el.textContent = "Праша содржина";
});

toastErr((el) => {
  el.textContent = "Содржина на грешка";
});

toast(
  (el) => {
    el.textContent = "Автоматско затворање по 99 секунди";
  },
  99
);
```

## Параметри на интерфејсот

### toast(render, timeout)

Креирајте и прикажувајте потсетници.

- `render`: `(el) => void`, рендерирање повратен повик, `el` е елементот на полето за праќање.
- `timeout`: `Number`, истек на секунди. Стандардно 9. Поставете на 0 за да не се исклучува автоматски.
- Повратна вредност: елемент за промптно поле. Поддржува исклучување на `el.close()`.

### toastErr(render, timeout)

Креирајте и прикажувајте пораки за грешка. Параметрите се исти како `toast`, со `.ERR` класа на стил.

## класа стил

- `.Toast`: основна класа на полето за праќање.
- `.ERR`: Статус на грешка.
- `.animated` / `.fadeInR` / `.fadeOutR`: анимирана транзиција.
- `.x`: Копче за затворање.
