# پنجره پاپ آپ را در لبه صفحه نمایش دهید

- ** فراخوانی عملکرد **: رابط های عادی و سریع خطا را ارائه می دهد
- ** پشته خودکار **: چند نکته به طور خودکار فاصله را محاسبه می کند و به صورت عمودی روی هم قرار می گیرد
- **خاموش شدن برنامه ریزی شده و دستی**: از خاموش شدن برنامه ریزی شده و دستی پشتیبانی می کند

## از دمو استفاده کنید

```js
import { toast, toastErr } from "webc.site/js/Toast.js";

toast((el) => {
  el.textContent = "محتوای درخواستی";
});

toastErr((el) => {
  el.textContent = "محتوای خطا";
});

toast(
  (el) => {
    el.textContent = "بعد از 99 ثانیه به صورت خودکار بسته می شود";
  },
  99
);
```

## پارامترهای رابط

### toast(render, timeout)

دستورات را ایجاد و نمایش دهید.

- `render`: `(el) => void`، پاسخ به تماس را ارائه می دهد، `el` عنصر جعبه درخواست است.
- `timeout`: `Number`، وقفه در ثانیه. پیش فرض 9. روی 0 تنظیم کنید تا به طور خودکار خاموش نشود.
- مقدار بازگشتی: عنصر جعبه اعلان. `el.close()` خاموش را پشتیبانی می کند.

### toastErr(render, timeout)

ایجاد و نمایش پیام های خطا پارامترها مانند `toast`، با کلاس سبک `.ERR` هستند.

## کلاس سبک

- `.Toast`: کلاس پایه جعبه درخواست.
- `.ERR`: وضعیت خطا.
- `.animated` / `.fadeInR` / `.fadeOutR`: انتقال متحرک.
- `.x`: دکمه بستن.
