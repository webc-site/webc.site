# صفحہ کے کنارے پر فوری پاپ اپ ونڈو

- **فنکشن کال**: نارمل اور ایرر پرامپٹ انٹرفیس فراہم کرتا ہے۔
- **آٹو اسٹیک**: متعدد ٹپس خود بخود وقفہ کاری کا حساب لگاتے ہیں اور عمودی طور پر اسٹیک کرتے ہیں
- **شیڈیولڈ اور مینوئل شٹ ڈاؤن**: شیڈول اور مینوئل شٹ ڈاؤن کو سپورٹ کرتا ہے۔

## ڈیمو استعمال کریں۔

```js
import { toast, toastErr } from "webc.site/js/Toast.js";

toast((el) => {
  el.textContent = "فوری مواد"؛
});

toastErr((el) => {
  el.textContent = "خرابی کا مواد"؛
});

toast(
  (el) => {
    el.textContent = "99 سیکنڈ کے بعد خود بخود بند"؛
  },
  99
);
```

## انٹرفیس کے پیرامیٹرز

### toast(render, timeout)

اشارے بنائیں اور ڈسپلے کریں۔

- `render`: `(el) => void`، کال بیک رینڈرنگ، `el` پرامپٹ باکس عنصر ہے۔
- `timeout`: `Number`، ٹائم آؤٹ سیکنڈ۔ ڈیفالٹ 9۔ خود بخود بند نہ ہونے کے لیے 0 پر سیٹ کریں۔
- واپسی کی قدر: پرامپٹ باکس عنصر۔ سپورٹ `el.close()` آف۔

### toastErr(render, timeout)

غلطی کے پیغامات بنائیں اور ڈسپلے کریں۔ پیرامیٹرز `toast` کی طرح ہیں، `.ERR` اسٹائل کلاس کے ساتھ۔

## سٹائل کلاس

- `.Toast`: پرامپٹ باکس بیس کلاس۔
- `.ERR`: خرابی کی صورتحال۔
- `.animated` / `.fadeInR` / `.fadeOutR`: متحرک منتقلی۔
- `.x`: بند کریں بٹن۔
