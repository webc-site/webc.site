# ກະຕຸ້ນປ່ອງຢ້ຽມປັອບອັບຢູ່ຂອບຂອງໜ້າ

- **Function call**: ໃຫ້​ການ​ໂຕ້​ຕອບ​ການ​ແຈ້ງ​ເຕືອນ​ທີ່​ເປັນ​ປົກ​ກະ​ຕິ​ແລະ​ຄວາມ​ຜິດ​ພາດ​
- ** stack ອັດ​ຕະ​ໂນ​ມັດ**​: ຫຼາຍ​ຄໍາ​ແນະ​ນໍາ​ອັດ​ຕະ​ໂນ​ມັດ​ການ​ຄິດ​ໄລ່​ຫ່າງ​ໄກ​ສອກ​ຫຼີກ​ແລະ stack ໃນ​ແນວ​ຕັ້ງ​
- ** ການ​ປິດ​ກໍາ​ນົດ​ເວ​ລາ​ແລະ​ຄູ່​ມື **​: ສະ​ຫນັບ​ສະ​ຫນູນ​ກໍາ​ນົດ​ເວ​ລາ​ແລະ​ການ​ປິດ​ດ້ວຍ​ມື​

## ໃຊ້ຕົວຢ່າງ

```js
import { toast, toastErr } from "webc.site/js/Toast.js";

toast((el) => {
  el.textContent = "ເນື້ອໃນທີ່ວ່ອງໄວ";
});

toastErr((el) => {
  el.textContent = "ເນື້ອໃນຂໍ້ຜິດພາດ";
});

toast(
  (el) => {
    el.textContent = "ປິດອັດຕະໂນມັດຫຼັງຈາກ 99 ວິນາທີ";
  },
  99
);
```

## ຕົວກໍານົດການໃນການໂຕ້ຕອບ

### toast(render, timeout)

ສ້າງແລະສະແດງການເຕືອນ.

- `render`: `(el) => void`, ການສະແດງການໂທກັບ, `el` ແມ່ນອົງປະກອບຂອງກ່ອງເຕືອນ.
- `timeout`: `Number`, ໝົດເວລາວິນາທີ. ຄ່າເລີ່ມຕົ້ນ 9. ຕັ້ງເປັນ 0 ເພື່ອບໍ່ໃຫ້ປິດອັດຕະໂນມັດ.
- ຄ່າກັບຄືນ: ອົງປະກອບກ່ອງເຕືອນ. ຮອງຮັບ `el.close()` ປິດ.

### toastErr(render, timeout)

ສ້າງ​ແລະ​ສະ​ແດງ​ຂໍ້​ຄວາມ​ຜິດ​ພາດ​. ພາລາມິເຕີແມ່ນຄືກັນກັບ `toast`, ທີ່ມີ `.ERR`ຊັ້ນສະໄຕລ໌.

## ຫ້ອງຮຽນແບບ

- `.Toast`: ຫ້ອງຮຽນຖານກ່ອງເຕືອນ.
- `.ERR`: ສະຖານະຄວາມຜິດພາດ.
- `.animated` / `.fadeInR` / `.fadeOutR`: ການປ່ຽນແບບເຄື່ອນໄຫວ.
- `.x`: ປຸ່ມປິດ.
