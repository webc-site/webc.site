# பக்கத்தின் விளிம்பில் உடனடியாக பாப்-அப் சாளரம்

- **செயல்பாட்டு அழைப்பு**: இயல்பான மற்றும் பிழை உடனடி இடைமுகங்களை வழங்குகிறது
- **ஆட்டோ ஸ்டேக்**: பல குறிப்புகள் தானாகவே இடைவெளியைக் கணக்கிட்டு செங்குத்தாக அடுக்கி வைக்கும்
- **திட்டமிடப்பட்ட மற்றும் கைமுறை பணிநிறுத்தம்**: திட்டமிடப்பட்ட மற்றும் கைமுறை பணிநிறுத்தத்தை ஆதரிக்கிறது

## டெமோவைப் பயன்படுத்தவும்

```js
import { toast, toastErr } from "webc.site/js/Toast.js";

toast((el) => {
  el.textContent = "உடனடியான உள்ளடக்கம்";
});

toastErr((el) => {
  el.textContent = "பிழை உள்ளடக்கம்";
});

toast(
  (el) => {
    el.textContent = "99 வினாடிகளுக்குப் பிறகு தானாக மூடு";
  },
  99
);
```

## இடைமுக அளவுருக்கள்

### toast(render, timeout)

கட்டளைகளை உருவாக்கி காண்பிக்கவும்.

- `render`: `(el) => void`, ரெண்டரிங் கால்பேக், `el` என்பது ப்ராம்ட் பாக்ஸ் உறுப்பு.
- `timeout`: `Number`, காலாவதி வினாடிகள். இயல்புநிலை 9. தானாக அணைக்கப்படாமல் இருக்க 0 என அமைக்கவும்.
- திரும்ப மதிப்பு: ப்ராம்ட் பாக்ஸ் உறுப்பு. `el.close()` முடக்கத்தை ஆதரிக்கிறது.

### toastErr(render, timeout)

பிழை செய்திகளை உருவாக்கி காட்டவும். அளவுருக்கள் `toast`, `.ERR` பாணி வகுப்பைப் போலவே இருக்கும்.

## பாணி வகுப்பு

- `.Toast`: ப்ராம்ப்ட் பாக்ஸ் அடிப்படை வகுப்பு.
- `.ERR`: பிழை நிலை.
- `.animated` / `.fadeInR` / `.fadeOutR`: அனிமேஷன் மாற்றம்.
- `.x`: மூடு பொத்தான்.
