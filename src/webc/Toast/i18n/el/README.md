# Αναδυόμενο παράθυρο προτροπής στην άκρη της σελίδας

- **Κλήση λειτουργίας**: Παρέχει κανονικές διεπαφές και διεπαφές προτροπής σφαλμάτων
- **Αυτόματη στοίβα**: Πολλές συμβουλές υπολογίζουν αυτόματα την απόσταση και στοιβάζονται κάθετα
- **Προγραμματισμένη και μη αυτόματη απενεργοποίηση**: Υποστηρίζει προγραμματισμένο και μη αυτόματο τερματισμό

## Χρησιμοποιήστε το demo

```js
import { toast, toastErr } from "webc.site/js/Toast.js";

toast((el) => {
  el.textContent = "Περιεχόμενο προτροπής";
});

toastErr((el) => {
  el.textContent = "Περιεχόμενο σφάλματος";
});

toast(
  (el) => {
    el.textContent = "Αυτόματο κλείσιμο μετά από 99 δευτερόλεπτα";
  },
  99
);
```

## Παράμετροι διεπαφής

### toast(render, timeout)

Δημιουργία και εμφάνιση προτροπών.

- `render`: `(el) => void`, απόδοση επανάκλησης, `el` είναι το στοιχείο του πλαισίου προτροπής.
- `timeout`: `Number`, χρονικό όριο λήξης δευτερόλεπτα. Προεπιλογή 9. Ρυθμίστε στο 0 για να μην τερματίζεται αυτόματα.
- Επιστρεφόμενη τιμή: στοιχείο του πλαισίου προτροπής. Υποστηρίζει `el.close()` απενεργοποιημένο.

### toastErr(render, timeout)

Δημιουργία και εμφάνιση μηνυμάτων σφάλματος. Οι παράμετροι είναι ίδιες με την `toast`, με την κατηγορία στυλ `.ERR`.

## τάξη στυλ

- `.Toast`: Βασική κλάση του πλαισίου προτροπής.
- `.ERR`: Κατάσταση σφάλματος.
- `.animated` / `.fadeInR` / `.fadeOutR`: κινούμενη μετάβαση.
- `.x`: Κουμπί κλεισίματος.
