# ಪುಟದ ಅಂಚಿನಲ್ಲಿ ಪ್ರಾಂಪ್ಟ್ ಪಾಪ್-ಅಪ್ ವಿಂಡೋ

- **ಫಂಕ್ಷನ್ ಕರೆ**: ಸಾಮಾನ್ಯ ಮತ್ತು ದೋಷ ಪ್ರಾಂಪ್ಟ್ ಇಂಟರ್‌ಫೇಸ್‌ಗಳನ್ನು ಒದಗಿಸುತ್ತದೆ
- **ಸ್ವಯಂ ಸ್ಟಾಕ್**: ಬಹು ಸಲಹೆಗಳು ಸ್ವಯಂಚಾಲಿತವಾಗಿ ಅಂತರವನ್ನು ಲೆಕ್ಕಾಚಾರ ಮಾಡಿ ಮತ್ತು ಲಂಬವಾಗಿ ಜೋಡಿಸಿ
- ** ನಿಗದಿತ ಮತ್ತು ಹಸ್ತಚಾಲಿತ ಸ್ಥಗಿತಗೊಳಿಸುವಿಕೆ **: ನಿಗದಿತ ಮತ್ತು ಹಸ್ತಚಾಲಿತ ಸ್ಥಗಿತವನ್ನು ಬೆಂಬಲಿಸುತ್ತದೆ

## ಡೆಮೊ ಬಳಸಿ

```js
import { toast, toastErr } from "webc.site/js/Toast.js";

toast((el) => {
  el.textContent = "ಪ್ರಾಂಪ್ಟ್ ವಿಷಯ";
});

toastErr((el) => {
  el.textContent = "ದೋಷ ವಿಷಯ";
});

toast(
  (el) => {
    el.textContent = "99 ಸೆಕೆಂಡುಗಳ ನಂತರ ಸ್ವಯಂಚಾಲಿತವಾಗಿ ಮುಚ್ಚು";
  },
  99
);
```

## ಇಂಟರ್ಫೇಸ್ ನಿಯತಾಂಕಗಳು

### toast(render, timeout)

ಪ್ರಾಂಪ್ಟ್‌ಗಳನ್ನು ರಚಿಸಿ ಮತ್ತು ಪ್ರದರ್ಶಿಸಿ.

- `render`: `(el) => void`, ರೆಂಡರಿಂಗ್ ಕಾಲ್‌ಬ್ಯಾಕ್, `el` ಪ್ರಾಂಪ್ಟ್ ಬಾಕ್ಸ್ ಅಂಶವಾಗಿದೆ.
- `timeout`: `Number`, ಸಮಯ ಮೀರುವ ಸೆಕೆಂಡುಗಳು. ಡೀಫಾಲ್ಟ್ 9. ಸ್ವಯಂಚಾಲಿತವಾಗಿ ಸ್ಥಗಿತಗೊಳ್ಳದಂತೆ 0 ಗೆ ಹೊಂದಿಸಿ.
- ರಿಟರ್ನ್ ಮೌಲ್ಯ: ಪ್ರಾಂಪ್ಟ್ ಬಾಕ್ಸ್ ಅಂಶ. `el.close()` ಅನ್ನು ಬೆಂಬಲಿಸುತ್ತದೆ.

### toastErr(render, timeout)

ದೋಷ ಸಂದೇಶಗಳನ್ನು ರಚಿಸಿ ಮತ್ತು ಪ್ರದರ್ಶಿಸಿ. `.ERR` ಶೈಲಿಯ ವರ್ಗದೊಂದಿಗೆ `toast` ಪ್ಯಾರಾಮೀಟರ್‌ಗಳು ಒಂದೇ ಆಗಿರುತ್ತವೆ.

## ಶೈಲಿ ವರ್ಗ

- `.Toast`: ಪ್ರಾಂಪ್ಟ್ ಬಾಕ್ಸ್ ಬೇಸ್ ಕ್ಲಾಸ್.
- `.ERR`: ದೋಷ ಸ್ಥಿತಿ.
- `.animated` / `.fadeInR` / `.fadeOutR`: ಅನಿಮೇಟೆಡ್ ಪರಿವರ್ತನೆ.
- `.x`: ಮುಚ್ಚು ಬಟನ್.
