# Карталар тізімін асинхронды түрде жүктеңіз және көрсетіңіз

- **Асинхронды жүктеу**: деректерді асинхронды функция арқылы алу
- **Күй дисплейі**: үш күйді қолдайды: жүктеу, бос деректер және көрсету картасы.
- **Икемді орналасу**: Карточкалар қатарға орналастырылған

## Демонстрацияны пайдаланыңыз

```html
<script type="module">
import CardLi from "webc.site/CardLi.js";

const sleep = (ms) => new Promise((resolve) => setTimeout(resolve, ms));

const card_li = CardLi(async () => {
  await sleep(2000);
  const data = [];
  for (let i = 1; i <= 13; ++i) {
    data.push(["Тақырып " + i, "Сипаттамасы" + i, "#" + i]);
  }
  return data;
});

document.body.append(card_li);
</script>
```

## Интерфейс параметрлері

### CardLi(loader)

- **жүктеуші**: `() => Promise<Array<[string, string, string]>>`, деректерді асинхронды жүктеу функциясы. Әрбір ішкі кілттің пішімі `[аты, кіріспе, сілтеме]` болып табылады.

## стиль сыныбы

- `.CardLi`: сыртқы элемент
- `.Card`: карта элементі
- `.Ing`: анимациялық элемент жүктелуде
