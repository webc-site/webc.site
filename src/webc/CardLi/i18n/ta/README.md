# கார்டு பட்டியலை ஒத்திசைவின்றி ஏற்றி காண்பிக்கவும்

- ** ஒத்திசைவற்ற ஏற்றுதல்**: ஒத்திசைவற்ற செயல்பாட்டின் மூலம் தரவைப் பெறவும்
- **நிலை காட்சி**: மூன்று நிலைகளை ஆதரிக்கிறது: ஏற்றுதல், வெற்று தரவு மற்றும் ரெண்டரிங் கார்டு.
- **நெகிழ்வான தளவமைப்பு**: அட்டைகள் வரிசையாக அமைக்கப்பட்டிருக்கும்

## டெமோவைப் பயன்படுத்தவும்

```html
<script type="module">
import CardLi from "webc.site/CardLi.js";

const sleep = (ms) => new Promise((resolve) => setTimeout(resolve, ms));

const card_li = CardLi(async () => {
  await sleep(2000);
  const data = [];
  for (let i = 1; i <= 13; ++i) {
    data.push(["தலைப்பு" + i, "விளக்கம்" + i, "#" + i]);
  }
  return data;
});

document.body.append(card_li);
</script>
```

## இடைமுக அளவுருக்கள்

### CardLi(loader)

- **லோடர்**: `() => Promise<Array<[string, string, string]>>`, ஒத்திசைவற்ற தரவு ஏற்றுதல் செயல்பாடு. ஒவ்வொரு துணை விசையின் வடிவம் `[பெயர், அறிமுகம், இணைப்பு]`.

## பாணி வகுப்பு

- `.CardLi`: வெளிப்புற உறுப்பு
- `.Card`: அட்டை உறுப்பு
- `.Ing`: அனிமேஷன் உறுப்பை ஏற்றுகிறது
