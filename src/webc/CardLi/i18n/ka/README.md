# ჩატვირთეთ და აჩვენეთ ბარათების სია ასინქრონულად

- **ასინქრონული ჩატვირთვა**: მიიღეთ მონაცემები ასინქრონული ფუნქციის მეშვეობით
- **სტატუსის ჩვენება**: მხარს უჭერს სამ მდგომარეობას: ჩატვირთვა, ცარიელი მონაცემები და ბარათის გაცემა.
- **მოქნილი განლაგება**: ბარათები განლაგებულია რიგებად

## გამოიყენეთ დემო ვერსია

```html
<script type="module">
import CardLi from "webc.site/CardLi.js";

const sleep = (ms) => new Promise((resolve) => setTimeout(resolve, ms));

const card_li = CardLi(async () => {
  await sleep(2000);
  const data = [];
  for (let i = 1; i <= 13; ++i) {
    data.push(["სათაური" + i, "აღწერა" + i, "#" + i]);
  }
  return data;
});

document.body.append(card_li);
</script>
```

## ინტერფეისის პარამეტრები

### CardLi(loader)

- **ჩამტვირთავი**: `() => Promise<Array<[string, string, string]>>`, მონაცემთა ასინქრონული ჩატვირთვის ფუნქცია. თითოეული ქვეკლავის ფორმატია `[სახელი, შესავალი, ბმული]`.

## სტილის კლასი

- `.CardLi`: გარე ელემენტი
- `.Card`: ბარათის ელემენტი
- `.Ing`: იტვირთება ანიმაციური ელემენტი
