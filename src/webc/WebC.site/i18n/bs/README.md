# Portal za usmjeravanje koji hostira dokumentaciju biblioteke komponenti i demo stranicu

- **Izgled preko celog ekrana**: Apsolutno pozicioniranje ispunjava okvir za prikaz, sa ugrađenim skrolovanjem i navigacijom.
- **Dinamičko usmjeravanje**: Parsirajte URL putanje i dinamički učitajte demonstracije komponenti ili početne stranice.
- **Višejezično**: odgovorite i ažurirajte navigacijsku traku i kopiju stranice u realnom vremenu.
- **Promjena statusa**: Automatski održavajte klase stilova za učitavanje i početnu stranicu.

## Koristite demo

```html
<c-webc-site></c-webc-site>
```

## stilska klasa

Komponenta će promijeniti sljedeće nazive klasa unutar sebe:

- `Ing`: Učitavanje rute ili raščlanjivanje jezičkog paketa.
- `A`: Trenutno na početnoj stranici.
