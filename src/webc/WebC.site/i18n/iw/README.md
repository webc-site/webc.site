# ניתוב תיעוד ספריית רכיבי אירוח בפורטל ואתר הדגמה

- **פריסת מסך מלא**: מיקום מוחלט ממלא את יציאת התצוגה, עם גלילה וניווט מובנים.
- **ניתוב דינמי**: נתח נתיבי כתובת אתר וטען באופן דינמי הדגמות או דפי בית של רכיבים.
- **ריבוי שפות**: השב ועדכן את סרגל הניווט והעתקת העמוד בזמן אמת.
- **החלפת סטטוס**: שמירה אוטומטית על מחלקות הסגנון של טעינה ודף הבית.

## השתמש בהדגמה

```html
<c-webc-site></c-webc-site>
```

## שיעור סגנון

הרכיב יחליף את שמות המחלקות הבאים בתוך עצמו:

- `Ing`: טעינת מסלול או ניתוח ערכת שפה.
- `A`: כרגע בדף הבית.
