# Portal za usmjeravanje koji sadrži dokumentaciju knjižnice komponenti i demo stranicu

- **Izgled preko cijelog zaslona**: Apsolutno pozicioniranje ispunjava okvir za prikaz, s ugrađenim pomicanjem i navigacijom.
- **Dinamičko usmjeravanje**: analizirajte URL staze i dinamički učitajte demonstracije komponenti ili početne stranice.
- **Više jezika**: odgovorite i ažurirajte navigacijsku traku i kopiju stranice u stvarnom vremenu.
- **Promjena statusa**: Automatski održavajte stilske klase učitavanja i početne stranice.

## Koristite demo

```html
<c-webc-site></c-webc-site>
```

## stilska klasa

Komponenta će unutar sebe promijeniti sljedeća imena klasa:

- `Ing`: Učitavanje rute ili raščlanjivanje jezičnog paketa.
- `A`: Trenutno na početnoj stranici.
