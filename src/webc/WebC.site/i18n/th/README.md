# เอกสารคู่มือไลบรารีคอมโพเนนต์การโฮสต์พอร์ทัลและไซต์สาธิต

- **เค้าโครงเต็มหน้าจอ**: การวางตำแหน่งที่แน่นอนจะเติมเต็มวิวพอร์ต พร้อมด้วยการเลื่อนและการนำทางในตัว
- **การกำหนดเส้นทางแบบไดนามิก**: แยกวิเคราะห์เส้นทาง URL และโหลดการสาธิตส่วนประกอบหรือหน้าแรกแบบไดนามิก
- **หลายภาษา**: ตอบสนองและอัปเดตแถบนำทางและสำเนาหน้าแบบเรียลไทม์
- **การสลับสถานะ**: รักษาคลาสสไตล์ของการโหลดและหน้าแรกโดยอัตโนมัติ

## ใช้การสาธิต

```html
<c-webc-site></c-webc-site>
```

## คลาสสไตล์

ส่วนประกอบจะเปลี่ยนชื่อคลาสต่อไปนี้ภายในตัวเอง:

- `Ing`: การโหลดเส้นทางหรือการแยกวิเคราะห์ชุดภาษา
- `A`: ขณะนี้อยู่ในหน้าแรก
