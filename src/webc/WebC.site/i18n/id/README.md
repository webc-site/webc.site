# Perutean portal hosting dokumentasi pustaka komponen dan situs demo

- **Tata letak layar penuh**: Pemosisian absolut memenuhi area pandang, dengan pengguliran dan navigasi bawaan.
- **Perutean Dinamis**: Mengurai jalur URL dan memuat demo atau beranda komponen secara dinamis.
- **Multi-bahasa**: Merespon dan memperbarui bilah navigasi dan salinan halaman secara real-time.
- **Peralihan status**: Secara otomatis mempertahankan kelas gaya pemuatan dan beranda.

## Gunakan demonya

```html
<c-webc-site></c-webc-site>
```

## kelas gaya

Komponen akan mengganti nama kelas berikut di dalamnya:

- `Ing`: Pemuatan rute atau penguraian paket bahasa.
- `A`: Saat ini berada di beranda.
