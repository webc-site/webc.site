# Routing Portal Hosting Komponent Bibliothéik Dokumentatioun an Demo Site

- ** Vollbildschierm Layout **: Absolut Positionéierung fëllt de Viewport, mat agebauter Scrollen an Navigatioun.
- **Dynamic Routing**: Parse URL Weeër an dynamesch lued Komponent Demos oder Homepages.
- ** Multi-Sprooch **: Äntwert an update d'Navigatiounsbar an d'Säitkopie an Echtzäit.
- ** Status Wiessel **: Erhalen automatesch d'Styl Klassen vun Luede an Homepage.

## Benotzt d'Demo

```html
<c-webc-site></c-webc-site>
```

## Stil Klass

De Komponent wiesselt déi folgend Klassennimm a sech selwer:

- `Ing`: Route Luede oder Sproochepack Parsing.
- `A`: Momentan op der Homepage.
