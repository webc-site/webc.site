# रूटिंग पोर्टल घटक लाइब्रेरी दस्तावेज़ीकरण और डेमो साइट की मेजबानी करता है

- **पूर्ण स्क्रीन लेआउट**: अंतर्निहित स्क्रॉलिंग और नेविगेशन के साथ, पूर्ण स्थिति व्यूपोर्ट को भर देती है।
- **डायनेमिक रूटिंग**: यूआरएल पथों को पार्स करें और घटक डेमो या होमपेज को गतिशील रूप से लोड करें।
- **बहुभाषी**: वास्तविक समय में नेविगेशन बार और पेज कॉपी का जवाब दें और अपडेट करें।
- **स्थिति स्विचिंग**: लोडिंग और होमपेज की शैली कक्षाएं स्वचालित रूप से बनाए रखें।

## शैली वर्ग

घटक निम्नलिखित वर्ग नामों को अपने भीतर बदल देगा:

- `Ing`: रूट लोडिंग या भाषा पैक पार्सिंग।
- `A`: वर्तमान में मुखपृष्ठ पर।
