# Портал маршрутизации, на котором размещена документация по библиотеке компонентов и демонстрационный сайт.

- **Полноэкранный макет**: абсолютное позиционирование заполняет область просмотра со встроенной прокруткой и навигацией.
- **Динамическая маршрутизация**: анализируйте URL-пути и динамически загружайте демо-версии компонентов или домашние страницы.
- **Многоязычность**: отвечайте и обновляйте панель навигации и копию страницы в режиме реального времени.
- **Переключение статуса**: автоматическое сохранение классов стилей загрузки и домашней страницы.

## Используйте демо-версию

```html
<c-webc-site></c-webc-site>
```

## класс стиля

Компонент будет переключать внутри себя следующие имена классов:

- `Ing`: загрузка маршрута или анализ языкового пакета.
- `A`: в настоящее время находится на главной странице.
