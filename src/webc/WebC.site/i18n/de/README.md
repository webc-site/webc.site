# Dokumentation und Demo-Site der Routing-Portal-Hosting-Komponentenbibliothek

- **Vollbild-Layout**: Die absolute Positionierung füllt das Ansichtsfenster aus, mit integriertem Scrollen und Navigation.
- **Dynamisches Routing**: URL-Pfade analysieren und Komponentendemos oder Homepages dynamisch laden.
- **Mehrsprachig**: Reagieren und aktualisieren Sie die Navigationsleiste und die Seitenkopie in Echtzeit.
- **Statusumschaltung**: Behalten Sie automatisch die Stilklassen des Ladens und der Homepage bei.

## Nutzen Sie die Demo

```html
<c-webc-site></c-webc-site>
```

## Stilklasse

Die Komponente wechselt in sich selbst die folgenden Klassennamen:

- `Ing`: Laden der Route oder Parsen des Sprachpakets.
- `A`: Derzeit auf der Startseite.
