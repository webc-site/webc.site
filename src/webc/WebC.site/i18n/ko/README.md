# 라우팅 포털 호스팅 구성 요소 라이브러리 문서 및 데모 사이트

- **전체 화면 레이아웃**: 절대 위치 지정이 내장된 스크롤 및 탐색 기능을 통해 뷰포트를 채웁니다.
- **동적 라우팅**: URL 경로를 구문 분석하고 구성요소 데모 또는 홈페이지를 동적으로 로드합니다.
- **다국어**: 탐색바 및 페이지 사본에 실시간으로 응답하고 업데이트합니다.
- **상태 전환**: 로딩 및 홈페이지의 스타일 클래스를 자동으로 유지합니다.

## 데모 사용

```html
<c-webc-site></c-webc-site>
```

## 스타일 수업

구성 요소는 자체 내에서 다음 클래스 이름을 전환합니다.

- `Ing`: 경로 로딩 또는 언어 팩 구문 분석.
- `A`: 현재 홈페이지에 있습니다.
