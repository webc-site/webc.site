# Portal de rutare care găzduiește documentația bibliotecii componente și site-ul demonstrativ

- **Aspect pe întregul ecran**: Poziționarea absolută umple fereastra de vizualizare, cu derulare și navigare încorporate.
- **Dynamic Routing**: Analizați căile URL și încărcați dinamic demonstrațiile componente sau paginile de pornire.
- **Multi-limbi**: Răspundeți și actualizați bara de navigare și copierea paginii în timp real.
- **Comutarea stării**: mențineți automat clasele de stil de încărcare și pagina de pornire.

## Utilizați demonstrația

```html
<c-webc-site></c-webc-site>
```

## clasa de stil

Componenta va schimba următoarele nume de clasă în interiorul ei:

- `Ing`: încărcarea rutei sau analizarea pachetului de limbi.
- `A`: în prezent pe pagina de pornire.
