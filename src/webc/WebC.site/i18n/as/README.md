# ৰুটিং পৰ্টেল হ'ষ্টিং উপাদান লাইব্ৰেৰী দস্তাবেজ আৰু ডেমো চাইট

- **সম্পূৰ্ণ পৰ্দা পৰিকল্পনা**: নিৰপেক্ষ অৱস্থানে দৃশ্যপৰ্ট পূৰণ কৰে, অন্তৰ্নিৰ্মিত স্ক্ৰলিং আৰু নেভিগেচনৰ সৈতে।
- **গতিশীল ৰাউটিং**: URL পথসমূহ বিশ্লেষণ কৰক আৰু পৰিবৰ্তনশীলভাৱে উপাদান ডেমো বা হোমপেজসমূহ লোড কৰক।
- **বহু-ভাষা**: সঁহাৰি আৰু নেভিগেচন বাৰ আৰু পৃষ্ঠা কপি বাস্তৱ সময়ত আপডেট.
- **অৱস্থা চুইচিং**: স্বয়ংক্ৰিয়ভাৱে লোডিং আৰু হোমপেজৰ শৈলী শ্ৰেণীসমূহ ৰক্ষণাবেক্ষণ কৰক।
