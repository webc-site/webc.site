# रूटिंग पोर्टल होस्टिंग घटक लायब्ररी दस्तऐवजीकरण आणि डेमो साइट

- **फुल स्क्रीन लेआउट**: अंगभूत स्क्रोलिंग आणि नॅव्हिगेशनसह परिपूर्ण पोझिशनिंग व्ह्यूपोर्ट भरते.
- **डायनॅमिक राउटिंग**: URL पाथ पार्स करा आणि घटक डेमो किंवा होमपेज डायनॅमिकली लोड करा.
- **बहु-भाषा**: रिअल टाइममध्ये नॅव्हिगेशन बार आणि पेज कॉपीला प्रतिसाद द्या आणि अपडेट करा.
- **स्थिती बदलणे**: लोडिंग आणि होमपेजचे स्टाइल क्लास स्वयंचलितपणे राखणे.

## डेमो वापरा

```html
<c-webc-site></c-webc-site>
```

## शैली वर्ग

घटक खालील वर्गांची नावे स्वतःमध्ये बदलेल:

- `Ing`: मार्ग लोडिंग किंवा भाषा पॅक पार्सिंग.
- `A`: सध्या मुख्यपृष्ठावर आहे.
