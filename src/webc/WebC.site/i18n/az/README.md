# Komponent kitabxana sənədləri və demo saytı yerləşdirən marşrutlaşdırma portalı

- **Tam ekran tərtibatı**: Mütləq yerləşdirmə daxili sürüşmə və naviqasiya ilə görünüş pəncərəsini doldurur.
- **Dinamik marşrutlaşdırma**: URL yollarını təhlil edin və komponent demolarını və ya əsas səhifələrini dinamik olaraq yükləyin.
- **Çoxdilli**: Naviqasiya panelini və səhifə surətini real vaxt rejimində cavablandırın və yeniləyin.
- **Statusun dəyişdirilməsi**: Yükləmə və ana səhifənin üslub siniflərini avtomatik olaraq qoruyun.


## stil sinfi

Komponent öz daxilində aşağıdakı sinif adlarını dəyişəcək:

- `Ing`: Marşrutun yüklənməsi və ya dil paketinin təhlili.
- `A`: Hazırda əsas səhifədədir.
