# Ruting portal hosting komponent bibliotek dokumentasjon og demo nettsted

- **Fullskjermoppsett**: Absolutt posisjonering fyller visningsporten, med innebygd rulling og navigering.
- **Dynamisk ruting**: Analyser URL-baner og last dynamisk inn komponentdemoer eller hjemmesider.
- **Flerspråklig**: Svar og oppdater navigasjonslinjen og sidekopien i sanntid.
- **Statusbytte**: Oppretthold automatisk stilklassene for lasting og hjemmeside.

## Bruk demoen

```html
<c-webc-site></c-webc-site>
```

## stilklasse

Komponenten vil bytte følgende klassenavn i seg selv:

- `Ing`: Ruteinnlasting eller språkpakkeanalyse.
- `A`: For øyeblikket på hjemmesiden.
