# Δρομολόγηση πύλης που φιλοξενεί τεκμηρίωση βιβλιοθήκης στοιχείων και ιστότοπο επίδειξης

- **Διάταξη πλήρους οθόνης**: Η απόλυτη τοποθέτηση γεμίζει το παράθυρο προβολής, με ενσωματωμένη κύλιση και πλοήγηση.
- **Δυναμική δρομολόγηση**: Ανάλυση διαδρομών URL και δυναμική φόρτωση επιδείξεων στοιχείων ή αρχικών σελίδων.
- **Πολυγλωσσική**: Απαντήστε και ενημερώστε τη γραμμή πλοήγησης και το αντίγραφο της σελίδας σε πραγματικό χρόνο.
- **Εναλλαγή κατάστασης**: Διατηρήστε αυτόματα τις κατηγορίες στυλ φόρτωσης και αρχικής σελίδας.

## Χρησιμοποιήστε το demo

```html
<c-webc-site></c-webc-site>
```

## τάξη στυλ

Το στοιχείο θα αλλάξει τα ακόλουθα ονόματα κλάσεων μέσα του:

- `Ing`: Φόρτωση διαδρομής ή ανάλυση πακέτων γλώσσας.
- `A`: Αυτήν τη στιγμή βρίσκεται στην αρχική σελίδα.
