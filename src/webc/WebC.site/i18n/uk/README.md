# Портал маршрутизації, що містить документацію бібліотеки компонентів і демонстраційний сайт

- **Повноекранний макет**: абсолютне позиціонування заповнює вікно перегляду з вбудованим прокручуванням і навігацією.
- **Динамічна маршрутизація**: аналізуйте URL-шляхи та динамічно завантажуйте демонстраційні версії компонентів або домашні сторінки.
- **Багатомовність**: відповідайте та оновлюйте панель навігації та копію сторінки в реальному часі.
- **Перемикання статусів**: автоматично підтримувати класи стилю завантаження та домашньої сторінки.

## Використовуйте демо

```html
<c-webc-site></c-webc-site>
```

## клас стилю

Компонент перемикає в собі такі імена класів:

- `Ing`: завантаження маршруту або аналіз мовного пакету.
- `A`: зараз на головній сторінці.
