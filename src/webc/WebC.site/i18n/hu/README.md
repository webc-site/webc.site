# Útválasztási portál hosting összetevő könyvtár dokumentációja és bemutató webhely

- **Teljes képernyős elrendezés**: Az abszolút pozicionálás kitölti a nézetablakot, beépített görgetéssel és navigációval.
- **Dinamikus útválasztás**: URL-útvonalak elemzése, valamint az összetevők bemutatóinak vagy kezdőlapjainak dinamikus betöltése.
- **Többnyelvű**: Válaszoljon és frissítse a navigációs sávot és az oldalmásolást valós időben.
- **Állapotváltás**: A betöltés és a kezdőlap stílusosztályainak automatikus karbantartása.

## Használd a demót

```html
<c-webc-site></c-webc-site>
```

## stílus osztály

A komponens a következő osztályneveket váltja át magában:

- `Ing`: Útvonal betöltése vagy nyelvi csomag elemzése.
- `A`: Jelenleg a kezdőlapon van.
