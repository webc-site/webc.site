# Maršrutēšanas portāla mitināšanas komponentu bibliotēkas dokumentācija un demonstrācijas vietne

- **Pilnekrāna izkārtojums**: absolūtā pozicionēšana aizpilda skata logu ar iebūvētu ritināšanu un navigāciju.
- **Dinamiskā maršrutēšana**: parsējiet URL ceļus un dinamiski ielādējiet komponentu demonstrācijas vai sākumlapas.
- **Vairākas valodas**: atbildiet un atjauniniet navigācijas joslu un lapas kopiju reāllaikā.
- **Statusa maiņa**: automātiski uzturiet ielādes un sākumlapas stila klases.

## Izmantojiet demonstrāciju

```html
<c-webc-site></c-webc-site>
```

## stila klase

Komponents sevī pārslēgs šādus klašu nosaukumus:

- `Ing`: maršruta ielāde vai valodas pakotnes parsēšana.
- `A`: pašlaik sākumlapā.
