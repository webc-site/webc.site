# Pagruruta ng portal na nagho-host ng component library documentation at demo site

- **Layout ng buong screen**: Pinuno ng ganap na pagpoposisyon ang viewport, na may built-in na pag-scroll at pag-navigate.
- **Dynamic na Pagruruta**: I-parse ang mga path ng URL at dynamic na naglo-load ng mga component demo o homepage.
- **Multi-language**: Tumugon at i-update ang navigation bar at page copy nang real time.
- **Paglipat ng katayuan**: Awtomatikong panatilihin ang mga klase ng istilo ng paglo-load at homepage.

## Gamitin ang demo

```html
<c-webc-site></c-webc-site>
```

## klase ng istilo

Papalitan ng component ang mga sumusunod na pangalan ng klase sa loob mismo nito:

- `Ing`: Pag-load ng ruta o pag-parse ng language pack.
- `A`: Kasalukuyang nasa homepage.
