# Směrovací portál hostující dokumentaci knihovny komponent a demo web

- **Rozvržení na celou obrazovku**: Absolutní umístění vyplní výřez s vestavěným posouváním a navigací.
- **Dynamické směrování**: Analyzujte cesty URL a dynamicky načtěte ukázky komponent nebo domovské stránky.
- **Vícejazyčné**: Odpovídejte a aktualizujte navigační panel a kopírování stránky v reálném čase.
- **Přepínání stavu**: Automaticky udržovat třídy stylu načítání a domovské stránky.

## Použijte demo

```html
<c-webc-site></c-webc-site>
```

## stylová třída

Komponenta v sobě přepne následující názvy tříd:

- `Ing`: Načítání trasy nebo analýza jazykového balíčku.
- `A`: Momentálně na domovské stránce.
