# بوابة التوجيه التي تستضيف وثائق مكتبة المكونات والموقع التجريبي

- **تخطيط ملء الشاشة**: يملأ الموضع المطلق إطار العرض، مع التمرير والتنقل المدمجين.
- ** التوجيه الديناميكي **: تحليل مسارات URL وتحميل العروض التوضيحية للمكونات أو الصفحات الرئيسية ديناميكيًا.
- **متعدد اللغات**: الرد على شريط التنقل ونسخة الصفحة وتحديثهما في الوقت الفعلي.
- **تبديل الحالة**: الحفاظ تلقائيًا على فئات أنماط التحميل والصفحة الرئيسية.

## استخدم العرض التوضيحي

```html
<c-webc-site></c-webc-site>
```

## فئة النمط

سيقوم المكون بتبديل أسماء الفئات التالية داخل نفسه:

- `Ing`: تحميل المسار أو تحليل حزمة اللغة.
- `A`: موجود حاليًا على الصفحة الرئيسية.
