# Portal tar-rotot li jospita dokumentazzjoni tal-librerija tal-komponenti u sit demo

- **Disinn tal-iskrin sħiħ**: Pożizzjonament assolut jimla l-vetrina, bi scrolling u navigazzjoni integrati.
- **Rotot Dinamic**: Analiżi l-mogħdijiet tal-URL u tagħbija dinamikament demos jew homepages tal-komponenti.
- **B'diversi lingwi**: Irrispondi u aġġorna l-bar tan-navigazzjoni u l-kopja tal-paġna f'ħin reali.
- **Bdil tal-istatus**: Żomm awtomatikament il-klassijiet tal-istil tat-tagħbija u l-homepage.

## Uża d-demo

```html
<c-webc-site></c-webc-site>
```

## klassi tal-istil

Il-komponent se jaqleb l-ismijiet tal-klassi li ġejjin fih innifsu:

- `Ing`: It-tagħbija tar-rotta jew l-analiżi tal-pakkett tal-lingwa.
- `A`: Bħalissa fuq il-paġna ewlenija.
