# Estilo ng listahan ng link ng navigation bar

- Pahalang na layout link mga bata
- Naglalaman ng default, hover at mga aktibong estado
- Built-in na transition at click-to-zoom na mga animation

## Gamitin ang demo

```html
<div class="NavALi">
  <a class="Card" href="#home">Link</a>
  <a class="Card A" href="#active">link sa pag-activate</a>
</div>
```

## katayuan ng istilo

### `.NavALi`

Nalalapat ang panlabas na listahan ng link ng pahalang na layout at espasyo.

### `.Card`

I-link ang mga bata.

### `.A`

Kapag na-activate, ilalapat ang highlight effect.
