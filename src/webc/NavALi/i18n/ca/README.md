# Estil de llista d'enllaços de la barra de navegació

- Enllaç de disseny horitzontal fills
- Conté els estats predeterminats, desplaçament i actiu
- Transició integrada i animacions de clic per fer zoom

## Utilitzeu la demostració

```html
<div class="NavALi">
  <a class="Card" href="#home">Enllaç</a>
  <a class="Card A" href="#active">enllaç d'activació</a>
</div>
```

## estat d'estil

### `.NavALi`

La llista d'enllaços exteriors aplica la disposició i l'espaiat horitzontals.

### `.Card`

Vincula els nens.

### `.A`

Quan està activat, s'aplica l'efecte de ressaltat.
