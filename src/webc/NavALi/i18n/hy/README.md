# Նավիգացիոն գծի հղումների ցանկի ոճը

- Հորիզոնական դասավորության կապող երեխաներ
- Պարունակում է լռելյայն, սավառնող և ակտիվ վիճակներ
- Ներկառուցված անցումային և սեղմումով խոշորացնելու անիմացիաներ

## Օգտագործեք ցուցադրությունը

```html
<div class="NavALi">
  <a class="Card" href="#home">Հղում</a>
  <a class="Card A" href="#active">ակտիվացման հղում</a>
</div>
```

## ոճի կարգավիճակը

### `.NavALi`

Արտաքին հղումների ցանկը կիրառում է հորիզոնական դասավորությունը և տարածությունը:

### `.Card`

Կապել երեխաներին:

### `.A`

Երբ ակտիվացված է, կիրառվում է ընդգծված էֆեկտը:
