# Stile dell'elenco dei collegamenti della barra di navigazione

- Il layout orizzontale collega i bambini
- Contiene gli stati predefinito, al passaggio del mouse e attivo
- Animazioni di transizione e clic per zoom integrate

## Utilizza la demo

```html
<div class="NavALi">
  <a class="Card" href="#home">Collegamento</a>
  <a class="Card A" href="#active">collegamento di attivazione</a>
</div>
```

## stato di stile

### `.NavALi`

L'elenco dei collegamenti esterni applica il layout e la spaziatura orizzontali.

### `.Card`

Collega i bambini.

### `.A`

Quando attivato, viene applicato l'effetto evidenziatore.
