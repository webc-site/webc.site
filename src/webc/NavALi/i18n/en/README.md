# Navigation bar link list style

- Horizontal layout link children
- Contains default, hover and active states
- Built-in transition and click-to-zoom animations

## Use the demo

```html
<div class="NavALi">
  <a class="Card" href="#home">Link</a>
  <a class="Card A" href="#active">Activation Link</a>
</div>
```

## Style Status

### `.NavALi`

The outer link list applies horizontal layout and spacing.

### `.Card`

Link children.

### `.A`

When activated, the highlight effect is applied.
