# Stil tal-lista tal-links tal-bar tan-navigazzjoni

- It-tfal tal-link tat-tqassim orizzontali
- Fih stati default, hover u attivi
- Animazzjonijiet ta' transizzjoni u click-to-zoom integrati

## Uża d-demo

```html
<div class="NavALi">
  <a class="Card" href="#home">Link</a>
  <a class="Card A" href="#active">rabta ta' attivazzjoni</a>
</div>
```

## status stil

### `.NavALi`

Il-lista tal-links ta' barra tapplika tqassim u spazjar orizzontali.

### `.Card`

Link tfal.

### `.A`

Meta jiġi attivat, jiġi applikat l-effett tal-qofol.
