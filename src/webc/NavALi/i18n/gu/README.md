# નેવિગેશન બાર લિંક સૂચિ શૈલી

- આડું લેઆઉટ લિંક બાળકો
- ડિફોલ્ટ, હોવર અને એક્ટિવ સ્ટેટ્સ ધરાવે છે
- બિલ્ટ-ઇન ટ્રાન્ઝિશન અને ક્લિક-ટુ-ઝૂમ એનિમેશન

## ડેમોનો ઉપયોગ કરો

```html
<div class="NavALi">
  <a class="Card" href="#home">લિંક</a>
  <a class="Card A" href="#active">સક્રિયકરણ લિંક</a>
</div>
```

## શૈલી સ્થિતિ

### `.NavALi`

બાહ્ય લિંક સૂચિ આડી લેઆઉટ અને અંતર લાગુ કરે છે.

### `.Card`

બાળકોને લિંક કરો.

### `.A`

જ્યારે સક્રિય થાય છે, ત્યારે હાઇલાઇટ અસર લાગુ થાય છે.
