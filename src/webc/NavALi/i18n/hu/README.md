# Navigációs sáv hivatkozáslista stílusa

- Vízszintes elrendezés link gyerekek
- Tartalmaz alapértelmezett, lebegtető és aktív állapotokat
- Beépített átmenet és kattintással nagyítható animációk

## Használd a demót

```html
<div class="NavALi">
  <a class="Card" href="#home">Link</a>
  <a class="Card A" href="#active">aktiváló link</a>
</div>
```

## stílus állapota

### `.NavALi`

A külső hivatkozáslista vízszintes elrendezést és térközt alkalmaz.

### `.Card`

Link gyerekek.

### `.A`

Aktiválása esetén a kiemelési effektus érvényesül.
