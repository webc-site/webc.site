# Nabigazio-barrako esteka-zerrenda estiloa

- Diseinu horizontala esteka seme-alabak
- Egoera lehenetsiak, pasatzea eta aktiboak ditu
- Trantsizio integratua eta klik egiteko zooma egiteko animazioak

## Erabili demoa

```html
<div class="NavALi">
  <a class="Card" href="#home">Esteka</a>
  <a class="Card A" href="#active">aktibazio esteka</a>
</div>
```

## estilo egoera

### `.NavALi`

Kanpoko esteken zerrendak diseinu horizontala eta tartea aplikatzen ditu.

### `.Card`

Lotu haurrak.

### `.A`

Aktibatzen denean, nabarmendu efektua aplikatzen da.
