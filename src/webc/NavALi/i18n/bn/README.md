# নেভিগেশন বার লিঙ্ক তালিকা শৈলী

- অনুভূমিক বিন্যাস লিঙ্ক শিশুদের
- ডিফল্ট, হোভার এবং সক্রিয় অবস্থা রয়েছে
- অন্তর্নির্মিত রূপান্তর এবং ক্লিক-টু-জুম অ্যানিমেশন

## ডেমো ব্যবহার করুন

```html
<div class="NavALi">
  <a class="Card" href="#home">লিঙ্ক</a>
  <a class="Card A" href="#active">সক্রিয়করণ লিঙ্ক</a>
</div>
```

## শৈলী অবস্থা

### `.NavALi`

বাইরের লিঙ্ক তালিকা অনুভূমিক বিন্যাস এবং ব্যবধান প্রয়োগ করে।

### `.Card`

শিশুদের লিঙ্ক করুন।

### `.A`

সক্রিয় হলে, হাইলাইট প্রভাব প্রয়োগ করা হয়।
