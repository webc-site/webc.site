# Kiểu danh sách liên kết thanh điều hướng

- Bố trí liên kết ngang con
- Chứa trạng thái mặc định, di chuột và hoạt động
- Tích hợp chuyển tiếp và hoạt ảnh nhấp để thu phóng

## Sử dụng bản trình diễn

```html
<div class="NavALi">
  <a class="Card" href="#home">liên kết</a>
  <a class="Card A" href="#active">liên kết kích hoạt</a>
</div>
```

## trạng thái phong cách

### `.NavALi`

Danh sách liên kết bên ngoài áp dụng bố cục và khoảng cách theo chiều ngang.

### `.Card`

Liên kết trẻ em.

### `.A`

Khi được kích hoạt, hiệu ứng nổi bật sẽ được áp dụng.
