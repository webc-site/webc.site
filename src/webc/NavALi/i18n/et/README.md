# Navigeerimisriba linkide loendi stiil

- Horisontaalne paigutus link lapsed
- Sisaldab vaike-, hõljutus- ja aktiivseid olekuid
- Sisseehitatud üleminek ja suumimiseks klõpsatavad animatsioonid

## Kasutage demo

```html
<div class="NavALi">
  <a class="Card" href="#home">Link</a>
  <a class="Card A" href="#active">aktiveerimise link</a>
</div>
```

## stiili staatus

### `.NavALi`

Välimine linkide loend rakendab horisontaalset paigutust ja vahesid.

### `.Card`

Link lapsed.

### `.A`

Kui see on aktiveeritud, rakendatakse esiletõstmise efekti.
