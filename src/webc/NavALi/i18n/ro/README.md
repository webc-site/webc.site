# Stilul listei de link-uri din bara de navigare

- Aspect orizontal link copii
- Conține stările implicite, hover și activ
- Tranziție încorporată și animații click-to-zoom

## Utilizați demonstrația

```html
<div class="NavALi">
  <a class="Card" href="#home">Legătură</a>
  <a class="Card A" href="#active">link de activare</a>
</div>
```

## statutul stilului

### `.NavALi`

Lista de linkuri exterioare aplică aspectul orizontal și spațierea.

### `.Card`

Leagă copiii.

### `.A`

Când este activat, se aplică efectul de evidențiere.
