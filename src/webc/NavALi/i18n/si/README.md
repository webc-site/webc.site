# සංචාලන තීරු සබැඳි ලැයිස්තු විලාසය

- තිරස් පිරිසැලසුම් සම්බන්ධක ළමුන්
- පෙරනිමි, සැරිසරන සහ සක්‍රිය තත්වයන් අඩංගු වේ
- බිල්ට්-ඉන් සංක්‍රාන්තිය සහ ක්ලික් කිරීමට විශාලනය කරන සජීවිකරණ

## Demo භාවිතා කරන්න

```html
<div class="NavALi">
  <a class="Card" href="#home">සබැඳිය</a>
  <a class="Card A" href="#active">සක්රිය කිරීමේ සබැඳිය</a>
</div>
```

## ශෛලිය තත්ත්වය

### `.NavALi`

පිටත සබැඳි ලැයිස්තුව තිරස් පිරිසැලසුම සහ පරතරය අදාළ වේ.

### `.Card`

ළමයින් සම්බන්ධ කරන්න.

### `.A`

සක්රිය කළ විට, උද්දීපන බලපෑම යොදනු ලැබේ.
