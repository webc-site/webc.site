# Stil popisa veza navigacijske trake

- Horizontalni izgled veze djece
- Sadrži zadano, lebdenje i aktivno stanje
- Ugrađeni prijelaz i animacije klika za zumiranje

## Koristite demo

```html
<div class="NavALi">
  <a class="Card" href="#home">Link</a>
  <a class="Card A" href="#active">poveznica za aktivaciju</a>
</div>
```

## status stila

### `.NavALi`

Popis vanjskih veza primjenjuje horizontalni raspored i razmak.

### `.Card`

Povežite djecu.

### `.A`

Kada se aktivira, primjenjuje se efekt isticanja.
