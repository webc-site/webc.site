# 탐색 모음 링크 목록 스타일

- 가로 레이아웃 링크 하위
- 기본, 마우스오버 및 활성 상태를 포함합니다.
- 내장된 전환 및 클릭하여 확대/축소 애니메이션

## 데모 사용

```html
<div class="NavALi">
  <a class="Card" href="#home">링크</a>
  <a class="Card A" href="#active">활성화 링크</a>
</div>
```

## 스타일 상태

### `.NavALi`

외부 링크 목록에는 가로 레이아웃과 간격이 적용됩니다.

### `.Card`

아이들을 연결하세요.

### `.A`

활성화 시 하이라이트 효과가 적용됩니다.
