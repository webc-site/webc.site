# Navigācijas joslas saišu saraksta stils

- Horizontālā izkārtojuma saite bērniem
- Ietver noklusējuma, kursora novietošanas un aktīvos stāvokļus
- Iebūvēta pāreja un “noklikšķināt, lai tuvinātu” animācijas

## Izmantojiet demonstrāciju

```html
<div class="NavALi">
  <a class="Card" href="#home">Saite</a>
  <a class="Card A" href="#active">aktivizācijas saite</a>
</div>
```

## stila statuss

### `.NavALi`

Ārējo saišu sarakstā tiek izmantots horizontāls izkārtojums un atstarpes.

### `.Card`

Saite bērnus.

### `.A`

Kad tas ir aktivizēts, tiek piemērots izcelšanas efekts.
