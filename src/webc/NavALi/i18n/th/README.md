# รูปแบบรายการลิงก์แถบนำทาง

- ลิงก์ลูกเค้าโครงแนวนอน
- ประกอบด้วยสถานะเริ่มต้น โฮเวอร์ และแอคทีฟ
- การเปลี่ยนแปลงในตัวและภาพเคลื่อนไหวคลิกเพื่อซูม

## ใช้การสาธิต

```html
<div class="NavALi">
  <a class="Card" href="#home">ลิงค์</a>
  <a class="Card A" href="#active">ลิงค์เปิดใช้งาน</a>
</div>
```

## สถานะสไตล์

### `.NavALi`

รายการลิงก์ภายนอกใช้เค้าโครงแนวนอนและระยะห่าง

### `.Card`

เชื่อมโยงเด็ก ๆ

### `.A`

เมื่อเปิดใช้งาน เอฟเฟกต์ไฮไลต์จะถูกใช้
