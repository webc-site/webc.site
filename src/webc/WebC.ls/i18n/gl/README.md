# Lista e mostra todos os compoñentes da biblioteca de compoñentes

- Amosado por estilo puro, compoñentes básicos e compoñentes compostos
- Apoiar a agrupación segundo as regras habituais
- Carga automaticamente a lista de compoñentes e a descrición do idioma correspondente ao cambiar de idioma
- Cambia automaticamente as clases de estado durante a carga asíncrona

## Use a demostración

```html
<c-webc-ls></c-webc-ls>
```

## clase de estilo

### `.Ing`

Engadido automaticamente ao cargar datos, usado para ocultar o contido da lista e aplicar o estilo de carga.
