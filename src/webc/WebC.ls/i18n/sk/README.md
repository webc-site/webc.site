# Zoznam a zobrazenie všetkých komponentov v knižnici komponentov

- Zobrazuje sa čistým štýlom, základnými komponentmi a kompozitnými komponentmi
- Podporte zoskupovanie podľa bežných pravidiel
- Automaticky načítať zoznam komponentov a popis príslušného jazyka pri prepínaní jazykov
- Automaticky prepínať triedy stavov počas asynchrónneho načítania

## Použite demo

```html
<c-webc-ls></c-webc-ls>
```

## štýlová trieda

### `.Ing`

Automaticky sa pridáva pri načítavaní údajov, používa sa na skrytie obsahu zoznamu a použitie štýlu načítania.
