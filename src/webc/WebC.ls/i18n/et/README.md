# Loetlege ja kuvage kõik komponenditeegi komponendid

- Kuvatakse puhta stiili, põhikomponentide ja komposiitkomponentide järgi
- Toetage rühmitamist tavareeglite järgi
- Laadige keelte vahetamisel automaatselt vastava keele komponentide loend ja kirjeldus
- Vahetage asünkroonse laadimise ajal automaatselt olekuklasse

## Kasutage demo

```html
<c-webc-ls></c-webc-ls>
```

## stiiliklass

### `.Ing`

Lisatakse automaatselt andmete laadimisel, kasutatakse loendi sisu peitmiseks ja laadimisstiili rakendamiseks.
