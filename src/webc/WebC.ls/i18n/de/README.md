# Listen Sie alle Komponenten in der Komponentenbibliothek auf und zeigen Sie sie an

- Dargestellt nach reinem Stil, Grundkomponenten und Verbundkomponenten
- Unterstützen Sie die Gruppierung nach regulären Regeln
- Laden Sie beim Wechseln der Sprache automatisch die Komponentenliste und die Beschreibung der entsprechenden Sprache
- Zustandsklassen während des asynchronen Ladens automatisch wechseln

## Nutzen Sie die Demo

```html
<c-webc-ls></c-webc-ls>
```

## Stilklasse

### `.Ing`

Wird beim Laden von Daten automatisch hinzugefügt, dient zum Ausblenden von Listeninhalten und zum Anwenden des Ladestils.
