# Lys en vertoon alle komponente in die komponentbiblioteek

- Getoon deur suiwer styl, basiese komponente en saamgestelde komponente
- Ondersteun groepering volgens gereelde reëls
- Laai outomaties die komponentlys en beskrywing van die ooreenstemmende taal wanneer tale verander word
- Skakel staatsklasse outomaties tydens asinchroniese laai

## Gebruik die demo

```html
<c-webc-ls></c-webc-ls>
```

## styl klas

### `.Ing`

Outomaties bygevoeg wanneer data gelaai word, gebruik om lysinhoud te versteek en laaistyl toe te pas.
