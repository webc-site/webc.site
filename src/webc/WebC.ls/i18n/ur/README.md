# اجزاء کی لائبریری میں تمام اجزاء کی فہرست بنائیں اور ڈسپلے کریں۔

- خالص سٹائل، بنیادی اجزاء اور جامع اجزاء کی طرف سے دکھایا گیا ہے
- باقاعدہ قواعد کے مطابق گروپ بندی کی حمایت کریں۔
- زبانوں کو تبدیل کرتے وقت اجزاء کی فہرست اور متعلقہ زبان کی تفصیل کو خودکار طور پر لوڈ کریں۔
- غیر مطابقت پذیر لوڈنگ کے دوران خودکار طور پر اسٹیٹ کلاسز کو تبدیل کریں۔
