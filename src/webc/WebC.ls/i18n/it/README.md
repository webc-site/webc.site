# Elenca e visualizza tutti i componenti nella libreria dei componenti

- Visualizzato da stile puro, componenti di base e componenti compositi
- Supportare il raggruppamento secondo regole regolari
- Carica automaticamente l'elenco dei componenti e la descrizione della lingua corrispondente quando si cambia lingua
- Cambia automaticamente le classi di stato durante il caricamento asincrono

## Utilizza la demo

```html
<c-webc-ls></c-webc-ls>
```

## lezione di stile

### `.Ing`

Aggiunto automaticamente durante il caricamento dei dati, utilizzato per nascondere il contenuto dell'elenco e applicare lo stile di caricamento.
