# Спіс і адлюстраванне ўсіх кампанентаў у бібліятэцы кампанентаў

- Адлюстроўваецца чысты стыль, асноўныя кампаненты і кампазітныя кампаненты
- Падтрымка групоўкі ў адпаведнасці са звычайнымі правіламі
- Аўтаматычна загружаць спіс кампанентаў і апісанне адпаведнай мовы пры пераключэнні моў
- Аўтаматычнае пераключэнне класаў стану падчас асінхроннай загрузкі
