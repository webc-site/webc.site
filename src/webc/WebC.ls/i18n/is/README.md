# Listaðu og sýndu alla hluti í íhlutasafninu

- Sýnd með hreinum stíl, grunnhlutum og samsettum íhlutum
- Stuðningshópur samkvæmt reglulegum reglum
- Hladdu sjálfkrafa íhlutalistanum og lýsingu á samsvarandi tungumáli þegar skipt er um tungumál
- Skiptu sjálfkrafa um ástandsflokka meðan á ósamstilltri hleðslu stendur
