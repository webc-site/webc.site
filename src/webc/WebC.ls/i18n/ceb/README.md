# Ilista ug ipakita ang tanang component sa component library

- Gipakita sa lunsay nga istilo, sukaranan nga mga sangkap ug mga komposisyon nga sangkap
- Suportahi ang paggrupo sumala sa regular nga mga lagda
- Awtomatikong i-load ang lista sa sangkap ug paghulagway sa katugbang nga sinultian kung magbalhin-balhin sa mga pinulongan
- Awtomatikong ibalhin ang mga klase sa estado sa panahon sa asynchronous nga pagkarga
