# Komponentlər kitabxanasındakı bütün komponentləri sadalayın və göstərin

- Təmiz üslub, əsas komponentlər və kompozit komponentlər tərəfindən göstərilir
- Adi qaydalara uyğun olaraq qruplaşmaya dəstək verin
- Dilləri dəyişdirərkən komponent siyahısını və müvafiq dilin təsvirini avtomatik yükləyin
- Asinxron yükləmə zamanı dövlət siniflərini avtomatik dəyişdirin
