# कम्पोनेन्ट लाइब्रेरीमा रहेका सबै कम्पोनेन्टहरू सूची र प्रदर्शन गर्नुहोस्

- शुद्ध शैली, आधारभूत कम्पोनेन्टहरू र कम्पोजिट कम्पोनेन्टहरूद्वारा प्रदर्शित
- नियमित नियम अनुसार समर्थन समूह
- भाषाहरू स्विच गर्दा कम्पोनेन्ट सूची र सम्बन्धित भाषाको विवरण स्वतः लोड गर्नुहोस्
- एसिन्क्रोनस लोडिङको बेला राज्य कक्षाहरू स्वचालित रूपमा स्विच गर्नुहोस्
