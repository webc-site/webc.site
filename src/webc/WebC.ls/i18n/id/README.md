# Daftar dan tampilkan semua komponen di perpustakaan komponen

- Ditampilkan dengan gaya murni, komponen dasar dan komponen komposit
- Dukung pengelompokan menurut aturan reguler
- Secara otomatis memuat daftar komponen dan deskripsi bahasa terkait saat berpindah bahasa
- Secara otomatis mengganti kelas status selama pemuatan asinkron

## Gunakan demonya

```html
<c-webc-ls></c-webc-ls>
```

## kelas gaya

### `.Ing`

Secara otomatis ditambahkan saat memuat data, digunakan untuk menyembunyikan konten daftar dan menerapkan gaya pemuatan.
