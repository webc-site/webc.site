# 키 값 쌍

상태, 버전 또는 키 값 정보를 표시하는 데 자주 사용되는 이중 텍스트 블록으로 배열된 캡슐 배지 스타일입니다.
- 이중 열 디스플레이: 왼쪽과 오른쪽으로 정렬된 두 개의 텍스트 블록을 지원하고 속성 이름과 프롬프트 내용을 구별할 수 있습니다.
- 액체 유리: 미묘한 내부 그림자와 호버에 대한 마이크로 줌 피드백을 갖춘 반투명 디자인을 따릅니다.
- 컨테이너 지원: 목록 컨테이너에서 단일 디스플레이 및 적응형 간격 배열 지원
