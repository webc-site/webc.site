# pares ng mahalagang halaga

Isang istilo ng capsule badge na nakaayos sa mga double text block, kadalasang ginagamit upang ipakita ang status, bersyon o impormasyon ng pangunahing halaga.
- Double-column display: sumusuporta sa dalawang text block na nakaayos sa kaliwa at kanan, at maaaring makilala ang mga pangalan ng attribute at prompt content
- Liquid Glass: Sinusundan ang isang frosted translucent na disenyo na may banayad na panloob na mga anino at micro-zoom na feedback sa hover
- Suporta sa container: sumusuporta sa iisang display at adaptive spacing arrangement sa mga list container

## Gamitin ang demo
```html
<main class="KvLi">
  <a href="#">
    <b>Version</b>
    <b>1.0.0</b>
  </a>
  <a href="#">
    <b>License</b>
    <b>MIT</b>
  </a>
</main>

<a class="Kv" href="#">
  <b>Status</b>
  <b>Active</b>
</a>
```
