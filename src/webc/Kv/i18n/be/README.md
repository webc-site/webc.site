# пара ключ-значэнне

Капсульны значок у выглядзе падвойных тэкставых блокаў, які часта выкарыстоўваецца для адлюстравання інфармацыі аб статусе, версіі або ключавым значэнні.
- Дысплей у два слупкі: падтрымлівае два тэкставыя блокі, размешчаныя злева і справа, і можа адрозніваць назвы атрыбутаў і змесціва падказкі
- Вадкае шкло: пасля матавага напаўпразрыстага дызайну з тонкімі ўнутранымі ценямі і мікрапавелічэннем пры навядзенні
- Падтрымка кантэйнераў: падтрымлівае адзін дысплей і адаптыўнае размяшчэнне інтэрвалаў у спісных кантэйнерах
