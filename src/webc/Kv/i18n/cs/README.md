# pár klíč-hodnota

Styl odznaku kapsle uspořádaný do dvojitých textových bloků, často používaný k zobrazení informací o stavu, verzi nebo klíčové hodnotě.
- Dvousloupcové zobrazení: podporuje dva textové bloky uspořádané vlevo a vpravo a dokáže rozlišit názvy atributů a obsah výzvy
- Tekuté sklo: Má matný průsvitný design s jemnými vnitřními stíny a zpětnou vazbou mikrozoomu při najetí myší
- Podpora kontejnerů: podporuje jedno zobrazení a adaptivní uspořádání mezer v kontejnerech seznamu

## Použijte demo
```html
<main class="KvLi">
  <a href="#">
    <b>Version</b>
    <b>1.0.0</b>
  </a>
  <a href="#">
    <b>License</b>
    <b>MIT</b>
  </a>
</main>

<a class="Kv" href="#">
  <b>Status</b>
  <b>Active</b>
</a>
```
