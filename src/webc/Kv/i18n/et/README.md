# võtmeväärtuste paar

Kapsli märgi stiil, mis on paigutatud topelttekstiplokkidesse, mida kasutatakse sageli oleku, versiooni või võtmeväärtuste teabe kuvamiseks.
- Kaheveeruline ekraan: toetab kahte vasakule ja paremale paigutatud tekstiplokki ning suudab eristada atribuutide nimesid ja viipade sisu
- Vedel klaas: järgib jäätunud poolläbipaistvat kujundust peente sisevarjude ja hõljumisel mikrosuumi tagasisidega
- Konteinerite tugi: toetab ühte kuvamist ja kohanduvat vahede paigutust loendikonteinerites

## Kasutage demo
```html
<main class="KvLi">
  <a href="#">
    <b>Version</b>
    <b>1.0.0</b>
  </a>
  <a href="#">
    <b>License</b>
    <b>MIT</b>
  </a>
</main>

<a class="Kv" href="#">
  <b>Status</b>
  <b>Active</b>
</a>
```
