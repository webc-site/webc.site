# nyckelvärdespar

En kapselmärkesstil arrangerad i dubbla textblock, som ofta används för att visa information om status, version eller nyckelvärde.
- Visning med dubbla kolumner: stöder två textblock arrangerade till vänster och höger, och kan särskilja attributnamn och promptinnehåll
- Flytande glas: Följer en frostad genomskinlig design med subtila inre skuggor och mikrozoom-feedback vid hovring
- Behållarstöd: stöder enkel display och adaptivt avståndsarrangemang i listbehållare

## Använd demon
```html
<main class="KvLi">
  <a href="#">
    <b>Version</b>
    <b>1.0.0</b>
  </a>
  <a href="#">
    <b>License</b>
    <b>MIT</b>
  </a>
</main>

<a class="Kv" href="#">
  <b>Status</b>
  <b>Active</b>
</a>
```
