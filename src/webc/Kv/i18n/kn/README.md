# ಪ್ರಮುಖ ಮೌಲ್ಯದ ಜೋಡಿ

ಡಬಲ್ ಟೆಕ್ಸ್ಟ್ ಬ್ಲಾಕ್‌ಗಳಲ್ಲಿ ಜೋಡಿಸಲಾದ ಕ್ಯಾಪ್ಸುಲ್ ಬ್ಯಾಡ್ಜ್ ಶೈಲಿಯನ್ನು ಸಾಮಾನ್ಯವಾಗಿ ಸ್ಥಿತಿ, ಆವೃತ್ತಿ ಅಥವಾ ಪ್ರಮುಖ ಮೌಲ್ಯದ ಮಾಹಿತಿಯನ್ನು ಪ್ರದರ್ಶಿಸಲು ಬಳಸಲಾಗುತ್ತದೆ.
- ಡಬಲ್-ಕಾಲಮ್ ಪ್ರದರ್ಶನ: ಎಡ ಮತ್ತು ಬಲಕ್ಕೆ ಜೋಡಿಸಲಾದ ಎರಡು ಪಠ್ಯ ಬ್ಲಾಕ್‌ಗಳನ್ನು ಬೆಂಬಲಿಸುತ್ತದೆ ಮತ್ತು ಗುಣಲಕ್ಷಣದ ಹೆಸರುಗಳು ಮತ್ತು ಪ್ರಾಂಪ್ಟ್ ವಿಷಯವನ್ನು ಪ್ರತ್ಯೇಕಿಸಬಹುದು
- ಲಿಕ್ವಿಡ್ ಗ್ಲಾಸ್: ಹೂವರ್‌ನಲ್ಲಿ ಸೂಕ್ಷ್ಮವಾದ ಒಳ ನೆರಳುಗಳು ಮತ್ತು ಮೈಕ್ರೋ-ಜೂಮ್ ಪ್ರತಿಕ್ರಿಯೆಯೊಂದಿಗೆ ಫ್ರಾಸ್ಟೆಡ್ ಅರೆಪಾರದರ್ಶಕ ವಿನ್ಯಾಸವನ್ನು ಅನುಸರಿಸುತ್ತದೆ
- ಕಂಟೈನರ್ ಬೆಂಬಲ: ಪಟ್ಟಿ ಕಂಟೈನರ್‌ಗಳಲ್ಲಿ ಏಕ ಪ್ರದರ್ಶನ ಮತ್ತು ಹೊಂದಾಣಿಕೆಯ ಅಂತರದ ವ್ಯವಸ್ಥೆಯನ್ನು ಬೆಂಬಲಿಸುತ್ತದೆ
