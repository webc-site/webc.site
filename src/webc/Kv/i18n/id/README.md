# pasangan nilai kunci

Gaya lencana kapsul yang disusun dalam blok teks ganda, sering digunakan untuk menampilkan informasi status, versi, atau nilai kunci.
- Tampilan kolom ganda: mendukung dua blok teks yang disusun kiri dan kanan, dan dapat membedakan nama atribut dan konten prompt
- Kaca Cair: Mengikuti desain tembus pandang buram dengan bayangan bagian dalam yang halus dan umpan balik zoom mikro saat mengarahkan kursor
- Dukungan kontainer: mendukung tampilan tunggal dan pengaturan spasi adaptif dalam kontainer daftar

## Gunakan demonya
```html
<main class="KvLi">
  <a href="#">
    <b>Version</b>
    <b>1.0.0</b>
  </a>
  <a href="#">
    <b>License</b>
    <b>MIT</b>
  </a>
</main>

<a class="Kv" href="#">
  <b>Status</b>
  <b>Active</b>
</a>
```
