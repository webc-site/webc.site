# ຄູ່ມູນຄ່າທີ່ສໍາຄັນ

ຮູບແບບປ້າຍແຄບຊູນຈັດລຽງເປັນຕົວໜັງສືຄູ່, ມັກໃຊ້ເພື່ອສະແດງສະຖານະ, ສະບັບ ຫຼືຂໍ້ມູນຄ່າຫຼັກ.
- ການ​ສະ​ແດງ​ສອງ​ຖັນ​: ສະ​ຫນັບ​ສະ​ຫນູນ​ສອງ​ຕົວ​ອັກ​ສອນ​ຈັດ​ລຽງ​ຊ້າຍ​ແລະ​ຂວາ​, ແລະ​ສາ​ມາດ​ຈໍາ​ແນກ​ຊື່​ຄຸນ​ສົມ​ບັດ​ແລະ​ເນື້ອ​ຫາ​ທີ່​ກະ​ຕຸ້ນ​ເຕືອນ​.
- Liquid Glass: ປະຕິບັດຕາມການອອກແບບທີ່ໂປ່ງໃສທີ່ມີອາກາດຫນາວ, ມີເງົາພາຍໃນທີ່ລະອຽດອ່ອນ ແລະສຽງສະທ້ອນຈາກ micro-zoom ໃນ hover
- ຮອງຮັບຕູ້ຄອນເທນເນີ: ຮອງຮັບການສະແດງຜົນດຽວ ແລະການຈັດລຽງຊ່ອງຫວ່າງທີ່ສາມາດປັບຕົວໄດ້ໃນກ່ອງບັນຈຸລາຍການ
