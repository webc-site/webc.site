# გასაღების მნიშვნელობის წყვილი

კაფსულის ბეჯის სტილი მოწყობილია ორმაგ ტექსტურ ბლოკებში, რომელიც ხშირად გამოიყენება სტატუსის, ვერსიის ან ძირითადი მნიშვნელობის ინფორმაციის საჩვენებლად.
- ორსვეტიანი ჩვენება: მხარს უჭერს მარცხნივ და მარჯვნივ განლაგებულ ორ ტექსტურ ბლოკს და შეუძლია განასხვავოს ატრიბუტების სახელები და მოთხოვნის შინაარსი
- თხევადი შუშა: მიჰყვება ყინვაგამძლე გამჭვირვალე დიზაინს დახვეწილი შიდა ჩრდილებით და მიკრო ზუმით გამოხმაურებით ცურვისას
- კონტეინერის მხარდაჭერა: მხარს უჭერს ერთ ეკრანს და ადაპტირებულ ინტერვალს სიის კონტეინერებში
