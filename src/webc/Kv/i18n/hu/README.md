# kulcsérték pár

Dupla szövegblokkba rendezett kapszulajelvénystílus, amelyet gyakran használnak állapot-, verzió- vagy kulcsérték-információk megjelenítésére.
- Kétoszlopos megjelenítés: két, balra és jobbra elrendezett szövegblokkot támogat, és képes megkülönböztetni az attribútumneveket és a prompt tartalmat
- Folyékony üveg: matt, áttetsző dizájnt követ, finom belső árnyékokkal és mikrozoom visszajelzéssel a lebegéskor
- Tárolótámogatás: támogatja az egyszeri megjelenítést és az adaptív térközelrendezést a listatárolókban

## Használd a demót
```html
<main class="KvLi">
  <a href="#">
    <b>Version</b>
    <b>1.0.0</b>
  </a>
  <a href="#">
    <b>License</b>
    <b>MIT</b>
  </a>
</main>

<a class="Kv" href="#">
  <b>Status</b>
  <b>Active</b>
</a>
```
