# Key Value Pair

A capsule badge style arranged in double text blocks, often used to display status, version or key value information.
- Double-column display: supports two text blocks arranged left and right, and can distinguish attribute names and prompt content
- Liquid Glass: Follows a frosted translucent design with subtle inner shadows and micro-zoom feedback on hover
- Container support: supports single display and adaptive spacing arrangement in list containers

## Use the demo
```html
<main class="KvLi">
  <a href="#">
    <b>Version</b>
    <b>1.0.0</b>
  </a>
  <a href="#">
    <b>License</b>
    <b>MIT</b>
  </a>
</main>

<a class="Kv" href="#">
  <b>Status</b>
  <b>Active</b>
</a>
```
