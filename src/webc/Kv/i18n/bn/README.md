# মূল মান জোড়া

ডবল টেক্সট ব্লকে সাজানো একটি ক্যাপসুল ব্যাজ শৈলী, যা প্রায়ই স্থিতি, সংস্করণ বা মূল মান তথ্য প্রদর্শন করতে ব্যবহৃত হয়।
- ডাবল-কলাম প্রদর্শন: বাম এবং ডানে সাজানো দুটি পাঠ্য ব্লক সমর্থন করে এবং বৈশিষ্ট্যের নাম এবং প্রম্পট বিষয়বস্তুকে আলাদা করতে পারে
- লিকুইড গ্লাস: সূক্ষ্ম অভ্যন্তরীণ ছায়া এবং হোভারে মাইক্রো-জুম প্রতিক্রিয়া সহ একটি হিমায়িত স্বচ্ছ নকশা অনুসরণ করে
- ধারক সমর্থন: তালিকা পাত্রে একক প্রদর্শন এবং অভিযোজিত ব্যবধান ব্যবস্থা সমর্থন করে
