# キーと値のペア

二重のテキスト ブロックに配置されたカプセル バッジ スタイル。ステータス、バージョン、またはキー値の情報を表示するためによく使用されます。
- 2 列表示: 左右に配置された 2 つのテキスト ブロックをサポートし、属性名とプロンプトの内容を区別できます。
- 液体ガラス: 微妙な内側の影とホバー時のマイクロズームフィードバックを備えたつや消し半透明のデザインに従います。
- コンテナーのサポート: リスト コンテナーでの単一表示と適応的な間隔配置をサポートします。

## デモを使用する
```html
<main class="KvLi">
  <a href="#">
    <b>Version</b>
    <b>1.0.0</b>
  </a>
  <a href="#">
    <b>License</b>
    <b>MIT</b>
  </a>
</main>

<a class="Kv" href="#">
  <b>Status</b>
  <b>Active</b>
</a>
```
