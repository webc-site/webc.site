# اهم قدر جوڙو

ڊبل ٽيڪسٽ بلاڪ ۾ ترتيب ڏنل ڪيپسول بيج جو انداز، اڪثر ڪري اسٽيٽس، ورزن يا اهم قدر جي معلومات ڏيکارڻ لاءِ استعمال ٿيندو آهي.
- ڊبل ڪالمن ڊسپلي: کاٻي ۽ ساڄي ترتيب ڏنل ٻه ٽيڪسٽ بلاڪ کي سپورٽ ڪري ٿو، ۽ خاصيتن جا نالا ۽ پرامپٹ مواد کي فرق ڪري سگھن ٿا
- Liquid Glass: ھور تي ٿلهي اندرين پاڇين ۽ مائڪرو زوم راءِ سان گڏ ھڪ ٿلهي پارباسي ڊيزائن جي پيروي ڪري ٿو
- ڪنٽينر سپورٽ: ھڪڙي ڊسپلي کي سپورٽ ڪري ٿو ۽ لسٽ ڪنٽينرز ۾ موافقت واري فاصلي جي ترتيب
