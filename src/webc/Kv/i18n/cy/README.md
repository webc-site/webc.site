# pâr gwerth allweddol

Arddull bathodyn capsiwl wedi'i drefnu mewn blociau testun dwbl, a ddefnyddir yn aml i arddangos gwybodaeth statws, fersiwn neu werth allweddol.
- Arddangosfa dwy golofn: yn cefnogi dau floc testun wedi'u trefnu i'r chwith a'r dde, a gall wahaniaethu rhwng enwau priodoleddau a chynnwys prydlon
- Gwydr Hylif: Yn dilyn dyluniad tryloyw barugog gyda chysgodion mewnol cynnil ac adborth micro-chwyddo ar hofran
- Cefnogaeth cynhwysydd: yn cefnogi arddangosfa sengl a threfniant bylchiad addasol mewn cynwysyddion rhestr
