# առանցքային արժեքների զույգ

Պարկուճի կրծքանշանի ոճ՝ դասավորված կրկնակի տեքստային բլոկներով, որը հաճախ օգտագործվում է կարգավիճակի, տարբերակի կամ հիմնական արժեքի մասին տեղեկությունները ցուցադրելու համար:
- Կրկնասյուն ցուցադրում. աջակցում է ձախ և աջ դասավորված երկու տեքստային բլոկների և կարող է տարբերակել ատրիբուտների անուններն ու հուշումների բովանդակությունը
- Հեղուկ ապակի. հետևում է ցրտահարված կիսաթափանցիկ ձևավորմանը՝ նուրբ ներքին ստվերներով և միկրո խոշորացման հետադարձ կապով սավառնելիս
- Կոնտեյների աջակցություն. աջակցում է մեկ ցուցադրման և հարմարվողական տարածությունների դասավորությանը ցուցակի բեռնարկղերում
