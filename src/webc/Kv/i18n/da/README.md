# nøgleværdipar

En kapsel-badge-stil arrangeret i dobbelte tekstblokke, der ofte bruges til at vise status, version eller nøgleværdiinformation.
- Visning med to kolonner: understøtter to tekstblokke arrangeret til venstre og højre og kan skelne attributnavne og promptindhold
- Flydende glas: Følger et frostet gennemsigtigt design med subtile indre skygger og mikrozoom-feedback ved svævning
- Containerunderstøttelse: understøtter enkelt skærm og adaptivt mellemrumsarrangement i listecontainere

## Brug demoen
```html
<main class="KvLi">
  <a href="#">

    <b>1.0.0</b>
  </a>

<a class="Kv" href="#">
  <b>Status</b>
  <b>Active</b>
</a>
```
