# Αντιγραφή κειμένου στο πρόχειρο

Κάντε κλικ στο κουμπί για να αντιγράψετε το κείμενο του καθορισμένου στοιχείου.

- Υποστηρίζει τη χρήση rel για τον καθορισμό του επιλογέα στοιχείων που θα αντιγραφεί
- Όταν δεν καθορίζεται, το προηγούμενο αδερφικό στοιχείο αντιγράφεται από προεπιλογή.
- Ένθετο πλαίσιο προτροπής Pophover
- Διεθνής υποστήριξη

## Χρησιμοποιήστε το demo

```html
<!-- Αντιγράψτε συγκεκριμένα στοιχεία -->
<span id="target">Εδώ είναι το κείμενο που πρέπει να αντιγραφεί</span>
<c-copy rel="#target"></c-copy>

<!-- Τα προηγούμενα στοιχεία αντιγράφονται από προεπιλογή -->
<span>το κείμενο του προηγούμενου στοιχείου</span>
<c-copy></c-copy>
```

## παράμετρος

| ιδιοκτησία | εικονογραφώ |
| --- | --- |
| rel | Αντιγράψτε τον επιλογέα του στοιχείου προορισμού. Προαιρετικός |

## στυλ κατάστασης

- `c-pophover.Done`: στυλ πλαισίου προτροπής μετά από επιτυχή αντιγραφή
