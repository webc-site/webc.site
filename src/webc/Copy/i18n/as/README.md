# লিখনী ক্লিপবোৰ্ডলৈ কপি কৰক

ধাৰ্য্য কৰা উপাদানৰ লিখনী কপি কৰিবলে বুটামটো ক্লিক কৰক।

- কপি কৰিবলগীয়া উপাদান নিৰ্বাচক ধাৰ্য্য কৰিবলে rel ব্যৱহাৰ কৰি সমৰ্থন কৰে
- যেতিয়া ধাৰ্য্য কৰা নহয়, পূৰ্বৱৰ্তী ভাই-ভনী উপাদান অবিকল্পিতভাৱে কপি কৰা হয়।
- নেষ্টেড পফ'ভাৰ প্ৰমপ্ট বক্স
- আন্তঃৰাষ্ট্ৰীয় সমৰ্থন

## ডেমো ব্যৱহাৰ কৰক

```html
<!-- নিৰ্দিষ্ট উপাদানসমূহ কপি কৰক -->
<span id="target">ইয়াত কপি কৰিবলগীয়া লিখনী দিয়া হৈছে</span>
<c-copy rel="#target"></c-copy>

<!-- পূৰ্বৰ উপাদানসমূহ অবিকল্পিতভাৱে কপি কৰা হয় -->
<span>পূৰ্বৰ উপাদানটোৰ লিখনী</span>
<c-copy></c-copy>
```
