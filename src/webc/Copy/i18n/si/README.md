# පෙළ පසුරු පුවරුවට පිටපත් කරන්න

නිශ්චිත මූලද්රව්යයේ පෙළ පිටපත් කිරීමට බොත්තම ක්ලික් කරන්න.

- පිටපත් කළ යුතු මූලද්‍රව්‍ය තේරීම නියම කිරීමට rel භාවිතා කිරීමට සහය දක්වයි
- නිශ්චිතව දක්වා නොමැති විට, පෙර සහෝදර අංගය පෙරනිමියෙන් පිටපත් කරනු ලැබේ.
