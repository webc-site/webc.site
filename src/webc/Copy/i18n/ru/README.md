# Скопировать текст в буфер обмена

Нажмите кнопку, чтобы скопировать текст указанного элемента.

- Поддерживает использование rel для указания селектора копируемого элемента.
- Если не указано, предыдущий родственный элемент копируется по умолчанию.
- Вложенное окно подсказки Pophover
- Международная поддержка

## Используйте демо-версию

```html
<!-- Копирование определенных элементов -->
<span id="target">Вот текст, который нужно скопировать</span>
<c-copy rel="#target"></c-copy>

<!-- Предыдущие элементы копируются по умолчанию. -->
<span>текст предыдущего элемента</span>
<c-copy></c-copy>
```

## параметр

| свойство | иллюстрировать |
| --- | --- |
| rel | Скопируйте селектор целевого элемента. Необязательный |

## статусный стиль

- `c-pophover.Done`: стиль окна подсказки после успешного копирования.
