# क्लिपबोर्डवर मजकूर कॉपी करा

निर्दिष्ट घटकाचा मजकूर कॉपी करण्यासाठी बटणावर क्लिक करा.

- कॉपी करण्यासाठी घटक निवडक निर्दिष्ट करण्यासाठी rel वापरण्याचे समर्थन करते
- निर्दिष्ट नसताना, मागील भाव घटक डीफॉल्टनुसार कॉपी केला जातो.
- नेस्टेड पोफोवर प्रॉम्प्ट बॉक्स
- आंतरराष्ट्रीय समर्थन

## डेमो वापरा

```html
<!-- विशिष्ट घटक कॉपी करा -->
<span id="target">येथे कॉपी करायचा मजकूर आहे</span>
<c-copy rel="#target"></c-copy>

<!-- आधीचे घटक डीफॉल्टनुसार कॉपी केले जातात -->
<span>मागील घटकाचा मजकूर</span>
<c-copy></c-copy>
```

## पॅरामीटर

| मालमत्ता | स्पष्ट करणे |
| --- | --- |
| rel | लक्ष्य घटकाचा निवडकर्ता कॉपी करा. ऐच्छिक |
