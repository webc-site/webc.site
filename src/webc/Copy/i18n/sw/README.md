# Nakili maandishi kwenye ubao wa kunakili

Bofya kitufe ili kunakili maandishi ya kipengele maalum.

- Inaauni kutumia rel kubainisha kiteuzi cha kipengele cha kunakiliwa
- Wakati haijabainishwa, kipengele cha awali cha ndugu kinakiliwa kwa chaguo-msingi.
- Kisanduku cha papo hapo cha Pophover
- Msaada wa kimataifa

## Tumia onyesho

```html
<!-- Nakili vipengele maalum -->
<span id="target">Hapa kuna maandishi ya kunakiliwa</span>
<c-copy rel="#target"></c-copy>

<!-- Vipengele vilivyotangulia vinakiliwa kwa chaguo-msingi -->
<span>maandishi ya kipengele kilichotangulia</span>
<c-copy></c-copy>
```

## kigezo

| mali | onyesha |
| --- | --- |
| rel | Nakili kiteuzi cha kipengele kinacholengwa. Hiari |

## mtindo wa hali

- `c-pophover.Done`: mtindo wa kisanduku cha papo hapo baada ya kunakili kwa ufanisi
