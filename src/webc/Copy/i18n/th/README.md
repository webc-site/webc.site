# คัดลอกข้อความไปยังคลิปบอร์ด

คลิกปุ่มเพื่อคัดลอกข้อความขององค์ประกอบที่ระบุ

- รองรับการใช้ rel เพื่อระบุตัวเลือกองค์ประกอบที่จะคัดลอก
- เมื่อไม่ได้ระบุ องค์ประกอบพี่น้องก่อนหน้าจะถูกคัดลอกตามค่าเริ่มต้น
- กล่องพร้อมท์ Pophover ที่ซ้อนกัน
- การสนับสนุนระหว่างประเทศ

## ใช้การสาธิต

```html
<!-- คัดลอกองค์ประกอบเฉพาะ -->
<span id="target">นี่คือข้อความที่จะคัดลอก</span>
<c-copy rel="#target"></c-copy>

<!-- องค์ประกอบก่อนหน้าจะถูกคัดลอกตามค่าเริ่มต้น -->
<span>ข้อความขององค์ประกอบก่อนหน้า</span>
<c-copy></c-copy>
```

## พารามิเตอร์

| คุณสมบัติ | แสดงให้เห็น |
| --- | --- |
| rel | คัดลอกตัวเลือกขององค์ประกอบเป้าหมาย ไม่จำเป็น |

## สไตล์สถานะ

- `c-pophover.Done`: รูปแบบกล่องข้อความแจ้งหลังจากการคัดลอกสำเร็จ
