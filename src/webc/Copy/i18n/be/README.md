# Скапіраваць тэкст у буфер абмену

Націсніце кнопку, каб скапіяваць тэкст названага элемента.

- Падтрымлівае выкарыстанне rel для ўказання селектара элемента, які трэба скапіяваць
- Калі не пазначана, папярэдні роднасны элемент капіюецца па змаўчанні.
- Укладзенае поле падказкі Pophover
- Міжнародная падтрымка

## Выкарыстоўвайце дэма

```html
<!-- Скапіруйце пэўныя элементы -->
<span id="target">Вось тэкст, які трэба скапіяваць</span>
<c-copy rel="#target"></c-copy>

<!-- Папярэднія элементы капіююцца па змаўчанні -->
<span>тэкст папярэдняга элемента</span>
<c-copy></c-copy>
```

## параметр

| уласнасць | ілюстраваць |
| --- | --- |
| rel | Скапіруйце селектар мэтавага элемента. Дадаткова |

## статусны стыль

- `c-pophover.Done`: стыль поля падказкі пасля паспяховага капіравання
