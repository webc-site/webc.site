# ক্লিপবোর্ডে পাঠ্য অনুলিপি করুন

নির্দিষ্ট উপাদানের পাঠ্য অনুলিপি করতে বোতামে ক্লিক করুন।

- অনুলিপি করা উপাদান নির্বাচক নির্দিষ্ট করতে rel ব্যবহার করে সমর্থন করে
- নির্দিষ্ট করা না থাকলে, পূর্ববর্তী ভাইবোন উপাদানটি ডিফল্টরূপে অনুলিপি করা হয়।
- নেস্টেড পফওভার প্রম্পট বক্স
- আন্তর্জাতিক সমর্থন

## ডেমো ব্যবহার করুন

```html
<!-- নির্দিষ্ট উপাদান অনুলিপি -->
<span id="target">এখানে কপি করা টেক্সট আছে</span>
<c-copy rel="#target"></c-copy>

<!-- পূর্ববর্তী উপাদানগুলি ডিফল্টরূপে অনুলিপি করা হয় -->
<span>পূর্ববর্তী উপাদানের পাঠ্য</span>
<c-copy></c-copy>
```

## প্যারামিটার

| সম্পত্তি | ব্যাখ্যা করা |
| --- | --- |
| rel | টার্গেট এলিমেন্টের সিলেক্টর কপি করুন। ঐচ্ছিক |
