# متن را در کلیپ بورد کپی کنید

روی دکمه کلیک کنید تا متن عنصر مشخص شده را کپی کنید.

- پشتیبانی از استفاده از rel برای تعیین انتخابگر عنصری که باید کپی شود
- زمانی که مشخص نشده باشد، عنصر خواهر و برادر قبلی به طور پیش فرض کپی می شود.
- جعبه اعلان Pophover تو در تو
- حمایت بین المللی

## از دمو استفاده کنید

```html
<!-- عناصر خاص را کپی کنید -->
<span id="target">این متنی است که باید کپی شود</span>
<c-copy rel="#target"></c-copy>

<!-- عناصر قبلی به طور پیش فرض کپی می شوند -->
<span>متن عنصر قبلی</span>
<c-copy></c-copy>
```

## پارامتر

| دارایی | نشان دادن |
| --- | --- |
| rel | انتخابگر عنصر هدف را کپی کنید. اختیاری |

## سبک وضعیت

- `c-pophover.Done`: سبک جعبه اعلان پس از کپی موفقیت آمیز
