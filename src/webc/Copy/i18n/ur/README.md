# متن کو کلپ بورڈ میں کاپی کریں۔

مخصوص عنصر کے متن کو کاپی کرنے کے لیے بٹن پر کلک کریں۔

- کاپی کیے جانے والے عنصر سلیکٹر کی وضاحت کرنے کے لیے rel استعمال کرنے کی حمایت کرتا ہے۔
- جب وضاحت نہ کی گئی ہو، پچھلا بہن بھائی عنصر بطور ڈیفالٹ کاپی کیا جاتا ہے۔
- نیسٹڈ پوف اوور پرامپٹ باکس
- بین الاقوامی حمایت

## ڈیمو استعمال کریں۔

```html
<!-- مخصوص عناصر کو کاپی کریں۔ -->
<span id="target">نقل کرنے کے لیے متن یہ ہے۔</span>
<c-copy rel="#target"></c-copy>

<!-- پچھلے عناصر کو بطور ڈیفالٹ کاپی کیا جاتا ہے۔ -->
<span>پچھلے عنصر کا متن</span>
<c-copy></c-copy>
```

## پیرامیٹر

| جائیداد | مثال دینا |
| --- | --- |
| rel | ہدف عنصر کے سلیکٹر کو کاپی کریں۔ اختیاری |

## حیثیت کا انداز

- `c-pophover.Done`: کامیاب کاپی کرنے کے بعد پرامپٹ باکس اسٹائل
