# ចម្លងអត្ថបទទៅក្ដារតម្បៀតខ្ទាស់

ចុចប៊ូតុងដើម្បីចម្លងអត្ថបទនៃធាតុដែលបានបញ្ជាក់។

- គាំទ្រការប្រើ rel ដើម្បីបញ្ជាក់ឧបករណ៍ជ្រើសរើសធាតុដែលត្រូវចម្លង
- នៅពេលដែលមិនបានបញ្ជាក់ ធាតុបងប្អូនបង្កើតពីមុនត្រូវបានចម្លងតាមលំនាំដើម។
- Nested Popover ប្រអប់បញ្ចូល
- ការគាំទ្រអន្តរជាតិ
