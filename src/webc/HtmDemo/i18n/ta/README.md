# HTML மற்றும் JS குறியீட்டை ஆன்லைனில் பிழைத்திருத்துதல்

- குறியீட்டின் கீழே பிழைத்திருத்த பட்டியை ரெண்டர் செய்து, CodePen மற்றும் JSFiddle பிழைத்திருத்த போர்டல்களை வழங்கவும்
- தேர்ந்தெடுக்கப்பட்ட பிழைத்திருத்த தளத்தை தானாக நினைவில் கொள்க
- HTML, CSS மற்றும் JS குறியீட்டை மாறும் வகையில் உருவாக்க `.gen` பண்புக்கூறு மூலம் கால்பேக்குகளை உட்செலுத்துவதை ஆதரிக்கிறது
- `language-js` வகுப்புப் பெயருடன் குழந்தை உறுப்புகளைத் தானாகக் கண்டறிந்து JS பயன்முறையில் இயக்கவும்

## டெமோவைப் பயன்படுத்தவும்

```html
<c-htm-demo><pre><code class="language-html">
<h1>டெமோ</h1>
</code></pre></c-htm-demo>

<script type="module">
import "webc.site/HtmDemo.js";

const el = document.querySelector("c-htm-demo");
el.gen = (type, text) => {
  // திரும்பவும் [html, css, js]
  return [
    text,
    "h1 { color: red; }",
    "console.log(1)"
  ];
};
</script>
```

## இடைமுக அளவுருக்கள்

### பண்புகள் மற்றும் முறைகள்

- `.gen`: இன்ஜெக்ட் செய்யப்பட்ட கால்பேக் செயல்பாடு. ** பிழைத்திருத்தப் பட்டி காட்டப்படுவதற்கு இந்தக் கால்பேக் செலுத்தப்பட வேண்டும். **
  - அளவுரு:
    - `type`: உட்செலுத்தப்பட்ட உள்ளடக்க வகை (`HTML = 1`, `JS = 2`).
    - `text`: குழந்தை உறுப்பிலிருந்து பிரித்தெடுக்கப்பட்ட குறியீட்டு உரை.
  - திரும்ப மதிப்பு: `[htm, css, js]`
