# Отстраняване на грешки в HTML и JS код онлайн

- Изобразете лента за отстраняване на грешки под кода и осигурете портали за отстраняване на грешки на CodePen и JSFiddle
- Автоматично запомняне на избраната платформа за отстраняване на грешки
- Поддържа инжектиране на обратни извиквания чрез атрибута `.gen` за динамично генериране на HTML, CSS и JS код
- Автоматично идентифицира дъщерни елементи с име на клас `language-js` и стартирайте в JS режим

## Използвайте демонстрацията

```html
<c-htm-demo><pre><code class="language-html">
<h1>Демо</h1>
</code></pre></c-htm-demo>

<script type="module">
import "webc.site/HtmDemo.js";

const el = document.querySelector("c-htm-demo");
el.gen = (type, text) => {
  // Връщане [html, css, js]
  return [
    text,
    "h1 { color: red; }",
    "console.log(1)"
  ];
};
</script>
```

## Параметри на интерфейса

### Свойства и методи

- `.gen`: Инжектирана функция за обратно извикване. **Това обратно извикване трябва да бъде инжектирано, за да се покаже лентата за отстраняване на грешки. **
  - параметър:
    - `type`: Инжектиран тип съдържание (`HTML = 1`, `JS = 2`).
    - `text`: Текстът на кода, извлечен от дъщерния елемент.
  - Върната стойност: `[htm, css, js]`
