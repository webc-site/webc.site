# વર્ચ્યુઅલ સ્ક્રોલ બાર

- **ટુ-વે સ્ક્રોલિંગ**: વર્ટિકલ (`c-vs`) અને હોરીઝોન્ટલ (`c-hs`) સ્ક્રોલિંગ પ્રદાન કરે છે.
- **કદ પ્રતિસાદ**: સામગ્રી અને વિસ્તારના કદના ફેરફારોને મોનિટર કરો, સ્ક્રોલ બારના કદ અને સ્થિતિને આપમેળે અપડેટ કરો અને જ્યારે કોઈ ઓવરફ્લો ન હોય ત્યારે આપમેળે છુપાવો.
- **ઇન્ટરેક્ટિવ સપોર્ટ**: વ્હીલ સ્ક્રોલિંગ, સ્લાઇડર ડ્રેગિંગ અને ટ્રૅક ક્લિક પોઝિશનિંગને સપોર્ટ કરે છે.
- **સ્થિતિ છુપાવો**: હોવર કરતી વખતે અથવા સ્ક્રોલ કરતી વખતે સ્લાઇડરને બતાવો અને પહોળું કરો અને જ્યારે સ્થિર હોય ત્યારે તેને આપમેળે છુપાવો.

## ઊભી સ્ક્રોલ

```html
<div style="height:300px;border:1px solid #000">
  <c-vs>
    <b>1</b>
    <b>2</b>
    <b>3</b>
  </c-vs>
</div>
<style>b { display: flex; border: 2px solid #090; padding: 30px; margin: 30px; }</style>
```

## આડી સ્ક્રોલ

```html
<div style="width:300px;border:1px solid #000">
  <c-hs>
    <b>1</b>
    <b>2</b>
    <b>3</b>
  </c-hs>
</div>
<style>b { display: flex; border: 2px solid #090; margin: 30px; padding: 30px; }</style>
```

## શૈલીના ભાગો

`::part()` દ્વારા શેડો DOM આંતરિક તત્વોની શૈલીને કસ્ટમાઇઝ કરો:

- `::part(scroll)`: સામગ્રી વિસ્તારને સ્ક્રોલ કરો.
- `::part(bar)`: સ્ક્રોલ બાર ટ્રૅક.
- `::part(si)`: સ્ક્રોલ બાર સ્લાઇડર.
- `::part(i)`: આંતરિક રેપિંગ સ્તર.
- `::part(drag)`: ખેંચવાની સ્થિતિમાં સ્ક્રોલ બાર ટ્રૅક.

## શૈલી વર્ગ

- `.drag`: `body` તત્વ પર લાગુ થાય છે, જે દર્શાવે છે કે તે ડ્રેગ-સ્ક્રોલ કરવાની સ્થિતિમાં છે.

## કસ્ટમ ગુણધર્મો

- `--si-bg`: સ્લાઇડર પૃષ્ઠભૂમિ રંગ.
- `--si-anim`: સ્લાઇડર એનિમેશન.
- `--cursorScrollhSvg`: માઉસ કર્સર જ્યારે આડો સ્ક્રોલ બાર હોવર કરે છે.
- `--cursorScrollvSvg`: માઉસ કર્સર જ્યારે વર્ટિકલ સ્ક્રોલ બાર હોવર કરે છે.
