# վիրտուալ ոլորման տող

- **Երկկողմանի ոլորում**. ապահովում է ուղղահայաց (`c-vs`) և հորիզոնական (`c-hs`) ոլորում:
- **Չափի պատասխանը**. Հետևեք բովանդակության և տարածքի չափի փոփոխություններին, ավտոմատ կերպով թարմացրեք ոլորման գծի չափն ու դիրքը և ինքնաբերաբար թաքցրեք, երբ արտահոսք չկա:
- **Ինտերակտիվ աջակցություն **. Աջակցում է անիվի ոլորմանը, սլայդերի քարշմանը և կտտոցների տեղադրմանը հետևելուն:
- **Թաքցնել կարգավիճակը**. ցույց տալ և լայնացնել սահիչը սավառնելիս կամ ոլորելիս, և ինքնաբերաբար թաքցնել այն անշարժ վիճակում:

## ուղղահայաց ոլորել

```html
<div style="height:300px;border:1px solid #000">
  <c-vs>
    <b>1</b>
    <b>2</b>
    <b>3</b>
  </c-vs>
</div>
<style>b { display: flex; border: 2px solid #090; padding: 30px; margin: 30px; }</style>
```

## հորիզոնական ոլորել

```html
<div style="width:300px;border:1px solid #000">
  <c-hs>
    <b>1</b>
    <b>2</b>
    <b>3</b>
  </c-hs>
</div>
<style>b { display: flex; border: 2px solid #090; margin: 30px; padding: 30px; }</style>
```

## ոճի մասեր

Անհատականացրեք Shadow DOM ներքին տարրերի ոճը `::part()`-ի միջոցով:

- `::part(scroll)`. ոլորեք բովանդակության տարածքը:
- `::part(bar)`: Ոլորման գծի հետքերը:
- `::part(si)`: Ոլորման գծի սահիչ:
- `::part(i)`. Ներքին փաթաթման շերտ:
- `::part(drag)`. Ոլորման գծի հետքերը քաշվող վիճակում:

## ոճի դաս

- `.drag`: Կիրառվում է `body` տարրի վրա՝ ցույց տալով, որ այն գտնվում է ոլորման ռեժիմում:

## Պատվերով հատկություններ

- `--si-bg`: Սլայդերի ֆոնի գույնը:
- `--si-anim`. սահող անիմացիա:
- `--cursorScrollhSvg`. մկնիկի կուրսորը, երբ հորիզոնական ոլորման սանդղակը սավառնում է:
