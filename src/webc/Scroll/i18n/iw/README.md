# פס גלילה וירטואלי

- **גלילה דו-כיוונית**: מספקת גלילה אנכית (`c-vs`) ואופקית (`c-hs`).
- **תגובת גודל**: עקוב אחר שינויים בתוכן ובגודל האזור, עדכן אוטומטית את הגודל והמיקום של פס הגלילה והסתרה אוטומטית כאשר אין הצפת יתר.
- **תמיכה אינטראקטיבית**: תומך בגלילת גלגלים, גרירת מחוון ומיקום לחיצה על המסלול.
- **הסתרת סטטוס**: הצג והרחיב את המחוון בעת ​​ריחוף או גלילה, והסתיר אותו באופן אוטומטי כאשר הוא נייח.

## גלילה אנכית

```html
<div style="height:300px;border:1px solid #000">
  <c-vs>
    <b>1</b>
    <b>2</b>
    <b>3</b>
  </c-vs>
</div>
<style>b { display: flex; border: 2px solid #090; padding: 30px; margin: 30px; }</style>
```

## גלילה אופקית

```html
<div style="width:300px;border:1px solid #000">
  <c-hs>
    <b>1</b>
    <b>2</b>
    <b>3</b>
  </c-hs>
</div>
<style>b { display: flex; border: 2px solid #090; margin: 30px; padding: 30px; }</style>
```

## חלקים בסגנון

התאם אישית את הסגנון של אלמנטים פנימיים של Shadow DOM באמצעות `::part()`:

- `::part(scroll)`: גלול באזור התוכן.
- `::part(bar)`: רצועת פס גלילה.
- `::part(si)`: מחוון פס גלילה.
- `::part(i)`: שכבת עטיפה פנימית.
- `::part(drag)`: רצועת פס הגלילה במצב גרירה.

## שיעור סגנון

- `.drag`: הוחל על האלמנט `body`, המציין שהוא נמצא במצב של גרירה-גלילה.

## מאפיינים מותאמים אישית

- `--si-bg`: צבע הרקע של המחוון.
- `--si-anim`: אנימציית סליידר.
- `--cursorScrollhSvg`: סמן העכבר כאשר פס הגלילה האופקי מרחף.
- `--cursorScrollvSvg`: סמן העכבר כאשר פס הגלילה האנכי מרחף.
