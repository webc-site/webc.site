# ვირტუალური გადახვევის ზოლი

- **ორმხრივი გადახვევა**: უზრუნველყოფს ვერტიკალურ (`c-vs`) და ჰორიზონტალურ (`c-hs`) გადახვევას.
- **ზომის პასუხი**: თვალყური ადევნეთ შიგთავსს და არეალის ზომის ცვლილებას, ავტომატურად განაახლეთ გადახვევის ზოლის ზომა და პოზიცია და ავტომატურად დამალვა, როცა არ არის გადაჭარბებული.
- **ინტერაქტიული მხარდაჭერა**: მხარს უჭერს ბორბლების გადახვევას, სლაიდერის გადმოწევას და დაწკაპუნების პოზიციონირებას.
- **სტატუსის დამალვა**: აჩვენეთ და გააფართოვეთ სლაიდერი მაუსის გადაადგილებისას ან გადახვევისას და ავტომატურად დამალეთ, როდესაც სტაციონარულია.

## ვერტიკალური გადახვევა

```html
<div style="height:300px;border:1px solid #000">
  <c-vs>
    <b>1</b>
    <b>2</b>
    <b>3</b>
  </c-vs>
</div>
<style>b { display: flex; border: 2px solid #090; padding: 30px; margin: 30px; }</style>
```

## ჰორიზონტალური გადახვევა

```html
<div style="width:300px;border:1px solid #000">
  <c-hs>
    <b>1</b>
    <b>2</b>
    <b>3</b>
  </c-hs>
</div>
<style>b { display: flex; border: 2px solid #090; margin: 30px; padding: 30px; }</style>
```

## სტილის ნაწილები

Shadow DOM შიდა ელემენტების სტილის მორგება `::part()`-ით:

- `::part(scroll)`: გადაახვიეთ შინაარსის არე.
- `::part(bar)`: გადახვევის ზოლის ჩანაწერი.
- `::part(si)`: გადახვევის ზოლის სლაიდერი.
- `::part(i)`: შიდა შესაფუთი ფენა.
- `::part(drag)`: გადახვევის ზოლის ჩანაწერი გადატანის მდგომარეობაში.

## სტილის კლასი

- `.drag`: გამოიყენება `body` ელემენტზე, რაც მიუთითებს, რომ ის გადაადგილების მდგომარეობაშია.

## მორგებული თვისებები

- `--si-bg`: სლაიდერის ფონის ფერი.
- `--si-anim`: სლაიდერის ანიმაცია.
- `--cursorScrollhSvg`: მაუსის კურსორი, როდესაც ჰორიზონტალური გადახვევის ზოლი არის დაკიდებული.
- `--cursorScrollvSvg`: მაუსის კურსორი, როდესაც ვერტიკალური გადახვევის ზოლი არის დაკიდებული.
