# ورچوئل اسڪرول بار

- **ٻه طرفي اسڪرولنگ**: عمودي (`c-vs`) ۽ افقي (`c-hs`) سکرولنگ مهيا ڪري ٿي.
- **سائز جواب**: مواد ۽ علائقي جي سائيز جي تبديلين کي مانيٽر ڪريو، خودڪار طريقي سان اسڪرول بار جي سائيز ۽ پوزيشن کي اپڊيٽ ڪريو، ۽ خودڪار طريقي سان لڪايو جڏھن ڪو به اوور فلو نه آھي.
- **انٽرايڪٽو سپورٽ**: سپورٽ ڪري ٿو ويل اسڪرولنگ، سلائڊر ڇڪڻ ۽ ٽريڪ ڪلڪ پوزيشن.
- **اسٽيٽس لڪايو**: سلائيڊر کي ڏيکاريو ۽ ويڪرو ڪريو جڏھن ھورنگ يا اسڪرولنگ ڪريو، ۽ پاڻمرادو ان کي لڪايو جڏھن اسٽيشنري.

## عمودي سکرول

```html
<div style="height:300px;border:1px solid #000">
  <c-vs>
    <b>1</b>
    <b>2</b>
    <b>3</b>
  </c-vs>
</div>
<style>b { display: flex; border: 2px solid #090; padding: 30px; margin: 30px; }</style>
```

## افقي اسڪرول

```html
<div style="width:300px;border:1px solid #000">
  <c-hs>
    <b>1</b>
    <b>2</b>
    <b>3</b>
  </c-hs>
</div>
<style>b { display: flex; border: 2px solid #090; margin: 30px; padding: 30px; }</style>
```

## انداز جا حصا

شيڊ ڊوم اندروني عناصر جي انداز کي ترتيب ڏيو `::part()` ذريعي:

- `::part(scroll)`: مواد واري علائقي کي اسڪرول ڪريو.
- `::part(bar)`: اسڪرول بار ٽريڪ.
- `::part(si)`: اسڪرول بار سلائڊر.
- `::part(i)`: اندروني لفافي پرت.
- `::part(drag)`: ڇڪڻ واري حالت ۾ اسڪرول بار ٽريڪ.

## انداز ڪلاس

- `.drag`: `body` عنصر تي لاڳو ڪيو ويو، اهو ظاهر ڪري ٿو ته اهو ڇڪڻ واري حالت ۾ آهي.

## ڪسٽم ملڪيت

- `--si-bg`: سلائڊر پس منظر جو رنگ.
- `--si-anim`: سلائڊر اينيميشن.
- `--cursorScrollhSvg`: ماؤس جو ڪرسر جڏھن ھوريزنٽل اسڪرول بار کي ھور ڪيو ويندو آھي.
- `--cursorScrollvSvg`: مائوس جو ڪرسر جڏهن عمودي اسڪرول بار کي هور ڪيو ويندو آهي.
