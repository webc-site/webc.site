# виртуелна трака за померање

- **Двосмерно померање**: Омогућава вертикално (<б ид="1">) и хоризонтално (<б ид="2">) померање.
- **Odaziv dimenzija**: Prati promene sadržaja i veličine područja, automatski ažurira veličinu i poziciju klizača, i automatski se sakriva kada nema prelivanja.
- **Интерактивна подршка**: Подржава померање точкића, превлачење клизачем и позиционирање праћења кликова.
- **Сакриј статус**: Прикажите и проширите клизач када лебдите или померате, и аутоматски га сакријте када је у стању мировања.

## вертикални скрол

```html
<div style="height:300px;border:1px solid #000">
  <c-vs>
    <b>1</b>
    <b>2</b>
    <b>3</b>
  </c-vs>
</div>
<style>b { display: flex; border: 2px solid #090; padding: 30px; margin: 30px; }</style>
```

## хоризонтални скрол

## стилски делови

Прилагодите стил унутрашњих елемената Схадов ДОМ-а преко <б ид="1">:

- <б ид="1">: Померајте област садржаја.
- <б ид="1">: Трака за померање.
- <б ид="1">: Клизач траке за померање.
- <б ид="1">: Слој унутрашњег омотача.
- <б ид="1">: Трака за померање у стању превлачења.

## стилска класа

- <б ид="1">: Примењено на елемент <б ид="2">, што указује да је у стању померања повлачења.

## Прилагођена својства

- <б ид="1">: Боја позадине клизача.
- <б ид="1">: Анимација клизача.
- <б ид="1">: Курсор миша када се лебди преко хоризонталне траке за померање.
- <б ид="1">: Курсор миша када се лебди вертикална трака за померање.
