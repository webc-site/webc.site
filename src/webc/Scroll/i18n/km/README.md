# របាររមូរនិម្មិត

- **ការរមូរពីរផ្លូវ**៖ ផ្តល់នូវការរមូរបញ្ឈរ (`c-vs`) និងផ្ដេក (`c-hs`)។
- **ការឆ្លើយតបទំហំ**៖ តាមដានការផ្លាស់ប្តូរទំហំផ្ទៃ និងមាតិកា ធ្វើបច្ចុប្បន្នភាពដោយស្វ័យប្រវត្តិនូវទំហំ និងទីតាំងរបាររមូរ ហើយលាក់ដោយស្វ័យប្រវត្តិនៅពេលដែលគ្មានការហៀរចេញ។
- ** ការគាំទ្រអន្តរកម្ម **៖ គាំទ្រការរមូរកង់ ការអូសគ្រាប់រំកិល និងតាមដានទីតាំងចុច។
- **លាក់ស្ថានភាព**៖ បង្ហាញ និងពង្រីកគ្រាប់រំកិលនៅពេលដាក់ ឬរមូរ ហើយលាក់វាដោយស្វ័យប្រវត្តិនៅពេលស្ថានី។

## រមូរបញ្ឈរ

<div style="height:300px;border:1px solid #000">

## ផ្នែករចនាប័ទ្ម

ប្ដូររចនាប័ទ្មនៃធាតុខាងក្នុងរបស់ Shadow DOM តាមបំណងតាមរយៈ `::part()`៖

- `::part(scroll)`៖ រំកិល​ផ្ទៃ​មាតិកា។
- `::part(bar)`៖ របារ​រមូរ​បទ។
- `::part(si)`៖ របាររំកិល គ្រាប់រំកិល។
- `::part(i)`៖ ស្រទាប់រុំខាងក្នុង។
- `::part(drag)`៖ របារ​រមូរ​ក្នុង​ស្ថានភាព​អូស។

## ថ្នាក់រចនាប័ទ្ម

- `.drag`៖ បាន​អនុវត្ត​ទៅ​ធាតុ `body` ដោយ​បង្ហាញ​ថា​វា​ស្ថិត​ក្នុង​ស្ថានភាព​អូស។

## លក្ខណៈសម្បត្តិផ្ទាល់ខ្លួន

- `--si-bg`៖ ពណ៌ផ្ទៃខាងក្រោយគ្រាប់រំកិល។
- `--si-anim`៖ ចលនារំកិល។
- `--cursorScrollhSvg`៖ ទស្សន៍ទ្រនិចកណ្ដុរ នៅពេលរបាររមូរផ្ដេកត្រូវបានដាក់។
- `--cursorScrollvSvg`៖ ទស្សន៍ទ្រនិចកណ្ដុរ នៅពេលរបាររមូរបញ្ឈរត្រូវបានដាក់។
