# виртуалды айналдыру жолағы

- **Екі жақты айналдыру**: тік (`c-vs`) және көлденең (`c-hs`) айналдыруды қамтамасыз етеді.
- **Өлшем жауабы**: Мазмұн мен аумақ өлшемі өзгерістерін бақылаңыз, айналдыру жолағы өлшемі мен орнын автоматты түрде жаңартыңыз және толып кетпегенде автоматты түрде жасырыңыз.
- **Интерактивті қолдау**: доңғалақты айналдыруды, сырғытпаны сүйреуді және басу орнын анықтауды қолдайды.
- **Күйді жасыру**: меңзерді апару немесе айналдыру кезінде сырғытпаны көрсету және кеңейту және қозғалмайтын кезде оны автоматты түрде жасыру.

## тік айналдыру

```html
<div style="height:300px;border:1px solid #000">
  <c-vs>
    <b>1</b>
    <b>2</b>
    <b>3</b>
  </c-vs>
</div>
<style>b { display: flex; border: 2px solid #090; padding: 30px; margin: 30px; }</style>
```

## көлденең айналдыру

```html
<div style="width:300px;border:1px solid #000">
  <c-hs>
    <b>1</b>
    <b>2</b>
    <b>3</b>
  </c-hs>
</div>
<style>b { display: flex; border: 2px solid #090; margin: 30px; padding: 30px; }</style>
```

## стиль бөліктері

Shadow DOM ішкі элементтерінің стилін `::part()` арқылы реттеңіз:

- `::part(scroll)`: мазмұн аймағын айналдырыңыз.
- `::part(bar)`: айналдыру жолағы жолы.
- `::part(si)`: айналдыру жолағы сырғытпасы.
- `::part(i)`: Ішкі орау қабаты.
- `::part(drag)`: сүйреп апару күйіндегі айналдыру жолағы жолы.

## стиль сыныбы

- `.drag`: `body` элементіне қолданылған, оның жылжыту күйінде екенін көрсетеді.

## Пайдаланушы сипаттары

- `--si-bg`: сырғытпа фонының түсі.
- `--si-anim`: жүгірткі анимациясы.
- `--cursorScrollhSvg`: Көлденең айналдыру жолағы меңзерді апарған кездегі тінтуір курсоры.
- `--cursorScrollvSvg`: тік айналдыру жолағы меңзерді апарған кездегі тінтуір курсоры.
