# आभासी स्क्रोल बार

- **टू-वे स्क्रोलिंग**: अनुलंब (`c-vs`) आणि क्षैतिज (`c-hs`) स्क्रोलिंग प्रदान करते.
- **आकार प्रतिसाद**: सामग्री आणि क्षेत्राच्या आकारातील बदलांचे निरीक्षण करा, स्क्रोल बार आकार आणि स्थान स्वयंचलितपणे अद्यतनित करा आणि ओव्हरफ्लो नसताना स्वयंचलितपणे लपवा.
- **परस्परसंवादी सपोर्ट**: व्हील स्क्रोलिंग, स्लाइडर ड्रॅगिंग आणि ट्रॅक क्लिक पोझिशनिंगला सपोर्ट करते.
- **स्थिती लपवा**: फिरवत असताना किंवा स्क्रोल करताना स्लाइडर दाखवा आणि रुंद करा आणि स्थिर असताना ते स्वयंचलितपणे लपवा.

## अनुलंब स्क्रोल

```html
<div style="height:300px;border:1px solid #000">
  <c-vs>
    <b>1</b>
    <b>2</b>
    <b>3</b>
  </c-vs>
</div>
<style>b { display: flex; border: 2px solid #090; padding: 30px; margin: 30px; }</style>
```

## क्षैतिज स्क्रोल

```html
<div style="width:300px;border:1px solid #000">
  <c-hs>
    <b>1</b>
    <b>2</b>
    <b>3</b>
  </c-hs>
</div>
<style>b { display: flex; border: 2px solid #090; margin: 30px; padding: 30px; }</style>
```

## शैली भाग

`::part()` द्वारे शॅडो DOM अंतर्गत घटकांची शैली सानुकूलित करा:

- `::part(scroll)`: सामग्री क्षेत्र स्क्रोल करा.
- `::part(bar)`: स्क्रोल बार ट्रॅक.
- `::part(si)`: स्क्रोल बार स्लाइडर.
- `::part(i)`: अंतर्गत रॅपिंग स्तर.
- `::part(drag)`: ड्रॅगिंग स्थितीत स्क्रोल बार ट्रॅक.

## शैली वर्ग

- `.drag`: `body` घटकावर लागू केले, ते ड्रॅग-स्क्रोलिंग स्थितीत असल्याचे सूचित करते.

## सानुकूल गुणधर्म

- `--si-bg`: स्लाइडर पार्श्वभूमी रंग.
- `--si-anim`: स्लाइडर ॲनिमेशन.
- `--cursorScrollhSvg`: जेव्हा क्षैतिज स्क्रोल बार फिरवला जातो तेव्हा माउस कर्सर.
- `--cursorScrollvSvg`: जेव्हा अनुलंब स्क्रोल बार फिरवला जातो तेव्हा माउस कर्सर.
