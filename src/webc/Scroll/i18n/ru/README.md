# виртуальная полоса прокрутки

- **Двусторонняя прокрутка**: обеспечивает вертикальную (`c-vs`) и горизонтальную (`c-hs`) прокрутку.
- **Ответ на размер**: отслеживайте изменения размера содержимого и области, автоматически обновляйте размер и положение полосы прокрутки и автоматически скрывайте ее при отсутствии переполнения.
- **Интерактивная поддержка**: поддержка прокрутки колесика, перетаскивания ползунка и отслеживания положения щелчка.
- **Скрыть статус**: отображать и расширять ползунок при наведении курсора или прокрутке, а также автоматически скрывать его, когда он неподвижен.

## вертикальная прокрутка

```html
<div style="height:300px;border:1px solid #000">
  <c-vs>
    <b>1</b>
    <b>2</b>
    <b>3</b>
  </c-vs>
</div>
<style>b { display: flex; border: 2px solid #090; padding: 30px; margin: 30px; }</style>
```

## горизонтальная прокрутка

```html
<div style="width:300px;border:1px solid #000">
  <c-hs>
    <b>1</b>
    <b>2</b>
    <b>3</b>
  </c-hs>
</div>
<style>b { display: flex; border: 2px solid #090; margin: 30px; padding: 30px; }</style>
```

## часть стиля

Настройте стиль внутренних элементов Shadow DOM с помощью `::part()`:

- `::part(scroll)`: прокрутка области содержимого.
- `::part(bar)`: дорожка полосы прокрутки.
- `::part(si)`: ползунок полосы прокрутки.
- `::part(i)`: внутренний слой упаковки.
- `::part(drag)`: полоса прокрутки в состоянии перетаскивания.

## класс стиля

- `.drag`: применяется к элементу `body`, указывая, что он находится в состоянии перетаскивания.

## Пользовательские свойства

- `--si-bg`: цвет фона ползунка.
- `--si-anim`: анимация слайдера.
- `--cursorScrollhSvg`: курсор мыши при наведении горизонтальной полосы прокрутки.
- `--cursorScrollvSvg`: курсор мыши при наведении вертикальной полосы прокрутки.
