# віртуальна смуга прокрутки

- **Двостороннє прокручування**: забезпечує вертикальне (`c-vs`) і горизонтальне (`c-hs`) прокручування.
- **Реакція розміру**: відстежуйте зміни розміру вмісту та області, автоматично оновлюйте розмір і положення смуги прокрутки та автоматично приховуйте її, якщо немає переповнення.
- **Інтерактивна підтримка**: підтримує прокручування коліщатком, перетягування повзунком і позиціонування треків.
- **Сховати статус**: відображати та розширювати повзунок під час наведення курсора або прокручування та автоматично приховувати його, коли він нерухомий.

## вертикальна прокрутка

```html
<div style="height:300px;border:1px solid #000">
  <c-vs>
    <b>1</b>
    <b>2</b>
    <b>3</b>
  </c-vs>
</div>
<style>b { display: flex; border: 2px solid #090; padding: 30px; margin: 30px; }</style>
```

## горизонтальна прокрутка

```html
<div style="width:300px;border:1px solid #000">
  <c-hs>
    <b>1</b>
    <b>2</b>
    <b>3</b>
  </c-hs>
</div>
<style>b { display: flex; border: 2px solid #090; margin: 30px; padding: 30px; }</style>
```

## стильові частини

Налаштуйте стиль внутрішніх елементів Shadow DOM за допомогою `::part()`:

- `::part(scroll)`: прокручування області вмісту.
- `::part(bar)`: доріжка смуги прокрутки.
- `::part(si)`: повзунок смуги прокрутки.
- `::part(i)`: внутрішній шар обгортки.
- `::part(drag)`: доріжка смуги прокручування в стані перетягування.

## клас стилю

- `.drag`: Застосовується до елемента `body`, вказуючи, що він перебуває в стані прокручування перетягуванням.

## Спеціальні властивості

- `--si-bg`: фоновий колір повзунка.
- `--si-anim`: анімація повзунка.
- `--cursorScrollhSvg`: курсор миші під час наведення курсора на горизонтальну смугу прокручування.
- `--cursorScrollvSvg`: курсор миші під час наведення курсора на вертикальну смугу прокручування.
