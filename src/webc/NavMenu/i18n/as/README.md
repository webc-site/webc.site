# বিভিন্ন পৰ্দাৰ আকাৰৰ সৈতে খাপ খুৱাই লোৱা নেভিগেচন মেনু

- ডেস্কটপ: সংযোগসমূহ অনুভূমিক টাইলছত প্ৰদৰ্শিত হয়
- মোবাইল সংস্কৰণ: মেনু বুটামত সংকুচিত কৰক, চাইডবাৰ ড্ৰয়াৰৰ পৰা বাহিৰলৈ স্লাইড কৰিবলৈ ক্লিক কৰক
- ৰুট সক্ৰিয়কৰণ: স্বয়ংক্ৰিয়ভাৱে বৰ্তমান পথ আৰু হেচ মান তুলনা কৰক, আৰু সক্ৰিয়কৰণ অৱস্থা শ্ৰেণীৰ নাম মিল থকা সংযোগ `A`লে যোগ কৰক
- স্বয়ংক্ৰিয়ভাৱে বন্ধ কৰক: যেতিয়া মোবাইল টাৰ্মিনেলে কোনো লিংক বা চাইডবাৰৰ বাহ্যিক পটভূমিত ক্লিক কৰে, তেতিয়া চাইডবাৰটো স্বয়ংক্ৰিয়ভাৱে বন্ধ হৈ যায়।

## চিত্ৰিত কৰক

যেতিয়া উপাদানটো আৰম্ভ কৰা হয়, যদি সন্তান উপাদানত `c-side` নাথাকে, মোবাইল মেনু বুটাম আৰু `c-side` কাষবাৰ স্বয়ংক্ৰিয়ভাৱে সৃষ্টি কৰা হ'ব, আৰু সকলো `a` টেগ কাষবাৰলৈ স্থানান্তৰ কৰা হ'ব।
