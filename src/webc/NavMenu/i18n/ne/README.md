# नेभिगेसन मेनु विभिन्न स्क्रिन आकारहरूमा अनुकूलित

- डेस्कटप: लिङ्कहरू तेर्सो टाइलहरूमा प्रदर्शित हुन्छन्
- मोबाइल संस्करण: मेनु बटनमा संक्षिप्त गर्नुहोस्, साइडबार ड्रअर बाहिर स्लाइड गर्न क्लिक गर्नुहोस्
- मार्ग सक्रियता: स्वचालित रूपमा हालको पथ र ह्यास मान तुलना गर्नुहोस्, र मिल्दो लिङ्कमा सक्रियता स्थिति वर्ग नाम थप्नुहोस् `A`
- स्वचालित रूपमा बन्द: जब मोबाइल टर्मिनलले लिङ्क वा साइडबारको बाह्य पृष्ठभूमिमा क्लिक गर्दछ, साइडबार स्वतः बन्द हुन्छ।

## प्रयोग गर्नुहोस्

```html
<c-nav-menu>
  <a href="/">अगाडि पृष्ठ</a>
  <a href="/doc">कागजात</a>
  <a href="/about">बारेमा</a>
</c-nav-menu>
```

## चित्रण गर्नुहोस्

जब कम्पोनेन्ट प्रारम्भ हुन्छ, यदि त्यहाँ चाइल्ड एलिमेन्टमा `c-side` छैन भने, मोबाइल मेनु बटन र `c-side` साइडबार स्वतः सिर्जना हुनेछ, र सबै `a` ट्यागहरू साइडबारमा सारिनेछन्।

## शैली वर्ग

- **A**: सक्रिय लिङ्कले स्वचालित रूपमा कक्षाको नाम `A` थप्नेछ
