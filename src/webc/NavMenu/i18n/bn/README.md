# নেভিগেশন মেনু বিভিন্ন স্ক্রীন আকারে অভিযোজিত

- ডেস্কটপ: লিঙ্কগুলি অনুভূমিক টাইলগুলিতে প্রদর্শিত হয়
- মোবাইল সংস্করণ: মেনু বোতামে সঙ্কুচিত করুন, সাইডবার ড্রয়ারটি স্লাইড করতে ক্লিক করুন
- রুট অ্যাক্টিভেশন: স্বয়ংক্রিয়ভাবে বর্তমান পাথ এবং হ্যাশ মান তুলনা করুন এবং মিলিত লিঙ্কে অ্যাক্টিভেশন স্ট্যাটাস ক্লাস নাম যোগ করুন `A`
- স্বয়ংক্রিয়ভাবে বন্ধ: যখন মোবাইল টার্মিনাল একটি লিঙ্কে বা সাইডবারের বাহ্যিক পটভূমিতে ক্লিক করে, তখন সাইডবার স্বয়ংক্রিয়ভাবে বন্ধ হয়ে যায়।

## ব্যাখ্যা করা

যখন উপাদানটি আরম্ভ করা হয়, যদি চাইল্ড এলিমেন্টে কোন `c-side` না থাকে, মোবাইল মেনু বোতাম এবং `c-side` সাইডবার স্বয়ংক্রিয়ভাবে তৈরি হবে, এবং সমস্ত `a` ট্যাগ সাইডবারে সরানো হবে।
