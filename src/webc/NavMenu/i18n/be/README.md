# Меню навігацыі адаптавана да розных памераў экрана

- Працоўны стол: спасылкі адлюстроўваюцца гарызантальнымі пліткамі
- Мабільная версія: кнопка "Згарнуць у меню", націсніце, каб высунуць скрыню бакавой панэлі
- Маршрут актывацыі: аўтаматычна параўнайце бягучы шлях і хэш-значэнне і дадайце назву класа статусу актывацыі да адпаведнай спасылкі `A`
- Аўтаматычнае закрыццё: калі мабільны тэрмінал націскае на спасылку або знешні фон бакавой панэлі, бакавая панэль аўтаматычна зачыняецца.

## выкарыстоўваць

```html
<c-nav-menu>
  <a href="/">першая старонка</a>
  <a href="/doc">дакумент</a>
  <a href="/about">аб</a>
</c-nav-menu>
```

## ілюстраваць

Пры ініцыялізацыі кампанента, калі ў даччыным элеменце няма `c-side`, кнопка мабільнага меню і бакавая панэль `c-side` будуць створаны аўтаматычна, і ўсе тэгі `a` будуць перамешчаны на бакавую панэль.

## клас стылю

- **A**: актываваная спасылка аўтаматычна дадасць назву класа `A`
