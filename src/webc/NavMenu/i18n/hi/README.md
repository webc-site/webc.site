# नेविगेशन मेनू विभिन्न स्क्रीन आकारों के अनुसार अनुकूलित

- डेस्कटॉप: लिंक क्षैतिज टाइल्स में प्रदर्शित होते हैं
- मोबाइल संस्करण: मेनू बटन को संक्षिप्त करें, साइडबार ड्रॉअर को बाहर स्लाइड करने के लिए क्लिक करें
- रूट सक्रियण: स्वचालित रूप से वर्तमान पथ और हैश मान की तुलना करें, और मिलान लिंक में सक्रियण स्थिति वर्ग का नाम जोड़ें `A`
- स्वचालित रूप से बंद करें: जब मोबाइल टर्मिनल किसी लिंक या साइडबार की बाहरी पृष्ठभूमि पर क्लिक करता है, तो साइडबार स्वचालित रूप से बंद हो जाता है।

## उपयोग

```html
<c-nav-menu>
  <a href="/">मुखपृष्ठ</a>
  <a href="/doc">दस्तावेज़</a>
  <a href="/about">के बारे में</a>
</c-nav-menu>
```

## उदाहरण देकर स्पष्ट करना

जब घटक आरंभ किया जाता है, यदि चाइल्ड तत्व में कोई `c-side` नहीं है, तो मोबाइल मेनू बटन और `c-side` साइडबार स्वचालित रूप से बनाया जाएगा, और सभी `a` टैग साइडबार में ले जाए जाएंगे।

## शैली वर्ग

- **ए**: सक्रिय लिंक स्वचालित रूप से कक्षा का नाम `A` जोड़ देगा
