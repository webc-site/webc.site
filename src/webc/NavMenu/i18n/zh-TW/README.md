# 適合不同螢幕尺寸的導航選單

- 桌面端：連結水平平鋪展示
- 行動端：折疊為選單按鈕，點選滑出側邊欄抽屜
- 路由啟動：自動比對目前路徑與雜湊值，為符合的連結新增啟動狀態類別名稱 `A`
- 自動收起：行動端點擊連結或側邊欄外部背景時，側邊欄自動關閉

## 使用

```html
<c-nav-menu>
  <a href="/">首頁</a>
  <a href="/doc">文件</a>
  <a href="/about">關於</a>
</c-nav-menu>
```

## 說明

元件在初始化時，若子元素中沒有 `c-side`，會自動建立行動端選單按鈕與 `c-side` 側邊欄，並將所有的 `a` 標籤移入側邊欄。

## 樣式類

- **A**：啟動的連結會自動加入類別名稱 `A`
