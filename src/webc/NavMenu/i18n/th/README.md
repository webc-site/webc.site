# เมนูนำทางปรับให้เหมาะกับขนาดหน้าจอที่แตกต่างกัน

- เดสก์ท็อป: ลิงก์จะแสดงเป็นไทล์แนวนอน
- เวอร์ชันมือถือ: ยุบลงในปุ่มเมนู คลิกเพื่อเลื่อนลิ้นชักแถบด้านข้างออก
- การเปิดใช้งานเส้นทาง: เปรียบเทียบเส้นทางปัจจุบันและค่าแฮชโดยอัตโนมัติ และเพิ่มชื่อคลาสสถานะการเปิดใช้งานให้กับลิงก์ที่ตรงกัน `A`
- ปิดโดยอัตโนมัติ: เมื่อเทอร์มินัลมือถือคลิกที่ลิงก์หรือพื้นหลังภายนอกของแถบด้านข้าง แถบด้านข้างจะปิดโดยอัตโนมัติ

## ใช้

```html
<c-nav-menu>
  <a href="/">หน้าแรก</a>
  <a href="/doc">เอกสาร</a>
  <a href="/about">เกี่ยวกับ</a>
</c-nav-menu>
```

## แสดงให้เห็น

เมื่อส่วนประกอบเริ่มต้น หากไม่มี `c-side` ในองค์ประกอบลูก ปุ่มเมนูมือถือและแถบด้านข้าง `c-side` จะถูกสร้างขึ้นโดยอัตโนมัติ และแท็ก `a` ทั้งหมดจะถูกย้ายไปยังแถบด้านข้าง

## คลาสสไตล์

- **A**: ลิงก์ที่เปิดใช้งานจะเพิ่มชื่อคลาส `A` โดยอัตโนมัติ
