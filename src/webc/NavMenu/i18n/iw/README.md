# תפריט ניווט מותאם לגדלי מסך שונים

- שולחן עבודה: קישורים מוצגים באריחים אופקיים
- גרסה ניידת: כווץ לתוך כפתור התפריט, לחץ כדי להחליק החוצה את מגירת סרגל הצד
- הפעלת מסלול: השווה אוטומטית את הנתיב הנוכחי ואת ערך הגיבוב, והוסף את שם מחלקת סטטוס ההפעלה לקישור התואם `A`
- סגירה אוטומטית: כאשר המסוף הנייד לוחץ על קישור או על הרקע החיצוני של סרגל הצד, סרגל הצד נסגר אוטומטית.

## לְהִשְׁתַמֵשׁ

```html
<c-nav-menu>
  <a href="/">עַמוּד רִאשׁוֹן</a>
  <a href="/doc">מִסְמָך</a>
  <a href="/about">אוֹדוֹת</a>
</c-nav-menu>
```

## לְהַדגִים

כאשר הרכיב מאותחל, אם אין `c-side` ברכיב הילד, כפתור התפריט לנייד ו`c-side` סרגל הצד ייווצרו אוטומטית, וכל התגיות `a` יועברו לסרגל הצד.

## שיעור סגנון

- **A**: הקישור המופעל יוסיף אוטומטית את שם הכיתה `A`
