# Әр түрлі экран өлшемдеріне бейімделген шарлау мәзірі

- Жұмыс үстелі: сілтемелер көлденең тақтайшаларда көрсетіледі
- Мобильді нұсқа: «Мәзірге жию» түймесін басыңыз, бүйірлік тақтаның тартпасын сырғытыңыз
- Маршрутты белсендіру: ағымдағы жолды және хэш мәнін автоматты түрде салыстырыңыз және сәйкес сілтемеге белсендіру күйі класының атауын қосыңыз `A`
- Автоматты түрде жабу: мобильді терминал сілтемені немесе бүйірлік тақтаның сыртқы фонын басқанда, бүйірлік тақта автоматты түрде жабылады.

## пайдалану

```html
<c-nav-menu>
  <a href="/">алдыңғы бет</a>
  <a href="/doc">құжат</a>
  <a href="/about">туралы</a>
</c-nav-menu>
```

## суреттеу

Компонент инициализацияланған кезде, еншілес элементте `c-side` болмаса, мобильді мәзір түймесі және `c-side` бүйірлік тақта автоматты түрде жасалады және барлық `a` тегтері бүйірлік тақтаға жылжытылады.

## стиль сыныбы

- **A**: Белсендірілген сілтеме `A` класс атауын автоматты түрде қосады
