# નેવિગેશન મેનૂ વિવિધ સ્ક્રીન માપો માટે અનુકૂળ

- ડેસ્કટોપ: લિંક્સ આડી ટાઇલ્સમાં પ્રદર્શિત થાય છે
- મોબાઇલ સંસ્કરણ: મેનૂ બટનમાં સંકુચિત કરો, સાઇડબાર ડ્રોઅરને સ્લાઇડ કરવા માટે ક્લિક કરો
- રૂટ સક્રિયકરણ: વર્તમાન પાથ અને હેશ મૂલ્યની આપમેળે તુલના કરો અને મેચિંગ લિંક `A`માં સક્રિયકરણ સ્થિતિ વર્ગનું નામ ઉમેરો
- આપમેળે બંધ થાય છે: જ્યારે મોબાઈલ ટર્મિનલ કોઈ લિંક અથવા સાઇડબારની બાહ્ય પૃષ્ઠભૂમિ પર ક્લિક કરે છે, ત્યારે સાઇડબાર આપમેળે બંધ થઈ જાય છે.

## સમજાવવું

જ્યારે ઘટક શરૂ થાય છે, જો ચાઇલ્ડ એલિમેન્ટમાં `c-side` ન હોય, તો મોબાઇલ મેનૂ બટન અને `c-side` સાઇડબાર આપમેળે બનાવવામાં આવશે, અને તમામ `a` ટૅગ્સ સાઇડબારમાં ખસેડવામાં આવશે.
