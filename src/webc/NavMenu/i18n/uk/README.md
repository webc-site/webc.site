# Навігаційне меню, адаптоване до різних розмірів екрана

- Робочий стіл: посилання відображаються в горизонтальних плитках
- Мобільна версія: кнопка «Згорнути в меню», натисніть, щоб висунути ящик бічної панелі
- Активація маршруту: автоматично порівняйте поточний шлях і хеш-значення та додайте назву класу статусу активації до відповідного посилання `A`
- Автоматичне закриття: коли мобільний термінал клацає посилання або зовнішній фон бічної панелі, бічна панель автоматично закривається.

## використовувати

```html
<c-nav-menu>
  <a href="/">титульна сторінка</a>
  <a href="/doc">документ</a>
  <a href="/about">приблизно</a>
</c-nav-menu>
```

## ілюструвати

Під час ініціалізації компонента, якщо в дочірньому елементі немає `c-side`, автоматично буде створено кнопку мобільного меню та бічну панель `c-side`, а всі теги `a` буде переміщено на бічну панель.

## клас стилю

- **A**: активоване посилання автоматично додасть назву класу `A`
