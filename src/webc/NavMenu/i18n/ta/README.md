# வழிசெலுத்தல் மெனு வெவ்வேறு திரை அளவுகளுக்கு ஏற்றது

- டெஸ்க்டாப்: இணைப்புகள் கிடைமட்ட ஓடுகளில் காட்டப்படும்
- மொபைல் பதிப்பு: மெனு பொத்தானில் சுருக்கவும், பக்கப்பட்டி டிராயரை ஸ்லைடு செய்ய கிளிக் செய்யவும்
- பாதை செயல்படுத்தல்: தற்போதைய பாதை மற்றும் ஹாஷ் மதிப்பை தானாக ஒப்பிட்டு, பொருந்தும் இணைப்பில் செயல்படுத்தும் நிலை வகுப்பின் பெயரைச் சேர்க்கவும் `A`
- தானாக மூடு: மொபைல் டெர்மினல் ஒரு இணைப்பை அல்லது பக்கப்பட்டியின் வெளிப்புற பின்னணியைக் கிளிக் செய்யும் போது, ​​பக்கப்பட்டி தானாகவே மூடப்படும்.

## பயன்படுத்த

```html
<c-nav-menu>
  <a href="/">முன் பக்கம்</a>
  <a href="/doc">ஆவணம்</a>
  <a href="/about">பற்றி</a>
</c-nav-menu>
```

## விளக்குகின்றன

கூறு துவக்கப்படும் போது, ​​குழந்தை உறுப்பில் `c-side` இல்லை என்றால், மொபைல் மெனு பொத்தான் மற்றும் `c-side` பக்கப்பட்டி தானாகவே உருவாக்கப்படும், மேலும் அனைத்து `a` குறிச்சொற்களும் பக்கப்பட்டியில் நகர்த்தப்படும்.
