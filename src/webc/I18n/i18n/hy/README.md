# Կոճակներ և բացվող պատուհաններ՝ ինտերֆեյսի լեզուները փոխելու համար

- Սեղմեք կոճակը, որպեսզի բացվի լեզվի ընտրության լողացող շերտը:
- Ավտոմատ ընդգծեք ներկայումս ընտրված լեզուն:

## Օգտագործեք ցուցադրությունը

```html
<c-i18n></c-i18n>

<script type="module">
import "webc.site/I18n.js";
// Լեզվի փոփոխությունները վերահսկելու համար օգտագործեք հետևյալ JS կոդը
import { onLang } from "webc.site/js/i18n.js";

onLang((code, id) => {
  alert(code + ' : ' + id);
});
</script>
```

## Ինտերֆեյսի պարամետրեր

### Հատուկ պիտակ `<c-i18n>`

Հատուկ HTML թեգեր, որոնք ներդրվել են JS-ում և օգտագործվում են HTML-ում:

## Կարգավիճակի կառավարում

Օգտագործեք `src/js/i18n.js`՝ համաշխարհային լեզվի կարգավիճակը կառավարելու համար.

- **`LANG_LI`**. Լեզուների ցանկի զանգված, `[[անուն, ID], ...]` ձևաչափով:
- ***`lang()`**. Ստացեք ընթացիկ լեզվի ID-ն:
- ***`langCode()`**. Ստացեք ընթացիկ լեզվի կոդավորումը:
- ***`langSet(id)`**. Սահմանեք լեզվի ID-ն և տեղեկացրեք բաժանորդներին:
- ***`onLang(func)`**. Բաժանորդագրվեք լեզվական փոփոխություններին: Եթե ​​լեզուն կարգավորված է, հետզանգը անմիջապես կգործարկվի: Վերադարձրեք ապաբաժանորդագրման գործառույթը։
