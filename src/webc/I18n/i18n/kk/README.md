# Интерфейс тілдерін ауыстыруға арналған түймелер мен қалқымалы терезелер

- Тіл таңдау қалқымалы қабатын ашу үшін түймені басыңыз.
- Ағымдағы таңдалған тілді автоматты түрде бөлектеңіз.

## Демонстрацияны пайдаланыңыз

```html
<c-i18n></c-i18n>

<script type="module">
import "webc.site/I18n.js";
// Тіл өзгерістерін бақылау үшін келесі JS кодын пайдаланыңыз
import { onLang } from "webc.site/js/i18n.js";

onLang((code, id) => {
  alert(code + ' : ' + id);
});
</script>
```

## Интерфейс параметрлері

### Арнаулы тег `<c-i18n>`

JS жүйесінде енгізілген және HTML тілінде қолданылатын реттелетін HTML тегтері.

## Күйді басқару

Жаһандық тіл күйін басқару үшін `src/js/i18n.js` пайдаланыңыз:

- **`LANG_LI`**: `[[name, ID], ...]` пішіміндегі тілдер тізімінің массиві.
- **`lang()`**: Ағымдағы тіл идентификаторын алыңыз.
- **`langCode()`**: Ағымдағы тіл кодтауын алыңыз.
- **`langSet(id)`**: тіл идентификаторын орнатыңыз және жазылушыларға хабарлаңыз.
- **`onLang(func)`**: Тіл өзгерістеріне жазылу. Тіл орнатылған болса, кері қоңырау дереу іске қосылады. Жазылымнан бас тарту функциясын қайтарыңыз.
