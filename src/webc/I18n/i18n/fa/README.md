# دکمه ها و پنجره های پاپ آپ برای تغییر زبان های رابط

- روی دکمه کلیک کنید تا لایه شناور انتخاب زبان ظاهر شود.
- زبان انتخاب شده فعلی را به صورت خودکار برجسته کنید.

## از دمو استفاده کنید

```html
<c-i18n></c-i18n>

<script type="module">
import "webc.site/I18n.js";
// از کد JS زیر برای نظارت بر تغییرات زبان استفاده کنید
import { onLang } from "webc.site/js/i18n.js";

onLang((code, id) => {
  alert(code + ' : ' + id);
});
</script>
```

## پارامترهای رابط

### برچسب سفارشی `<c-i18n>`

تگ های HTML سفارشی، در JS معرفی شده و در HTML استفاده می شوند.

## مدیریت وضعیت

از `src/js/i18n.js` برای مدیریت وضعیت زبان جهانی استفاده کنید:

- **`LANG_LI`**: آرایه فهرست زبان، در قالب `[[نام، شناسه]، ...]`.
- ***`lang()`**: شناسه زبان فعلی را دریافت کنید.
- ***`langCode()`**: رمزگذاری زبان فعلی را دریافت کنید.
- ***`langSet(id)`**: شناسه زبان را تنظیم کنید و به مشترکین اطلاع دهید.
- ***`onLang(func)`**: اشتراک در تغییرات زبان. اگر زبان تنظیم شده باشد، پاسخ تماس بلافاصله فعال می شود. تابع لغو اشتراک را برگردانید.
