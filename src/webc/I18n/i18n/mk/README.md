# Копчиња и скокачки прозорци за префрлување јазици на интерфејсот

- Кликнете на копчето за да се појави лебдечки слој за избор на јазик.
- Автоматски означете го тековно избраниот јазик.

## Користете го демо

```html
<c-i18n></c-i18n>

<script type="module">
import "webc.site/I18n.js";
// Користете го следниот JS-код за следење на промените во јазикот
import { onLang } from "webc.site/js/i18n.js";

onLang((code, id) => {
  alert(code + ' : ' + id);
});
</script>
```

## Параметри на интерфејсот

### Приспособена ознака `<c-i18n>`

Прилагодени HTML ознаки, воведени во JS и користени во HTML.

## Управување со статус

Користете `src/js/i18n.js` за управување со статусот на глобалниот јазик:

- **`LANG_LI`**: низа со список на јазици, во формат на `[[име, ID], ...]`.
- ***`lang()`**: Добијте го тековниот ID на јазикот.
- ***`langCode()`**: Добијте го кодирањето на тековниот јазик.
- ***`langSet(id)`**: Поставете го ID на јазикот и известете ги претплатниците.
- ***`onLang(func)`**: Претплатете се за јазични промени. Ако јазикот е поставен, повратниот повик ќе се активира веднаш. Вратете ја функцијата за отпишување.
