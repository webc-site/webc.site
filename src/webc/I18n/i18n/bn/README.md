# ইন্টারফেস ভাষা পরিবর্তনের জন্য বোতাম এবং পপ-আপ উইন্ডো

- ভাষা নির্বাচন ভাসমান স্তর পপ আপ করতে বোতামে ক্লিক করুন।
- বর্তমানে নির্বাচিত ভাষা স্বয়ংক্রিয়ভাবে হাইলাইট করুন।

## ডেমো ব্যবহার করুন

```html
<c-i18n></c-i18n>

<script type="module">
import "webc.site/I18n.js";
// ভাষা পরিবর্তন নিরীক্ষণ করতে নিম্নলিখিত JS কোড ব্যবহার করুন
import { onLang } from "webc.site/js/i18n.js";

onLang((code, id) => {
  alert(code + ' : ' + id);
});
</script>
```

## ইন্টারফেস পরামিতি

### কাস্টম ট্যাগ `<c-i18n>`৷

কাস্টম এইচটিএমএল ট্যাগ, জেএস-এ প্রবর্তিত এবং এইচটিএমএল-এ ব্যবহৃত।

## স্থিতি ব্যবস্থাপনা

বৈশ্বিক ভাষার স্থিতি পরিচালনা করতে `src/js/i18n.js` ব্যবহার করুন:

- **`LANG_LI`**: ভাষা তালিকা বিন্যাস, `[[নাম, আইডি], ...]` এর বিন্যাসে।
- **`lang()`**: বর্তমান ভাষা আইডি পান।
- **`langCode()`**: বর্তমান ভাষা এনকোডিং পান।
- **`langSet(id)`**: ভাষা আইডি সেট করুন এবং গ্রাহকদের অবহিত করুন।
- **`onLang(func)`**: ভাষা পরিবর্তনের জন্য সদস্যতা নিন। ভাষা সেট করা থাকলে, কলব্যাক অবিলম্বে ট্রিগার করা হবে। আনসাবস্ক্রাইব ফাংশন ফেরত দিন।

## শৈলী ক্লাস

### `.BtnC.lang`

আইকন বোতাম।

### `.I18n.Lg`

পপ-আপ উইন্ডোর প্রধান বিন্যাস ভাষা বিকল্পগুলি মোড়ানোর জন্য ব্যবহৃত হয়।
