# Дугмад и искачући прозори за промену језика интерфејса

- Кликните на дугме да бисте отворили плутајући слој за избор језика.
- Аутоматски означи тренутно изабрани језик.

## Користите демо

```html
<c-i18n></c-i18n>

<script type="module">
import "webc.site/I18n.js";
// Користите следећи ЈС код за праћење промена језика
import { onLang } from "webc.site/js/i18n.js";

onLang((code, id) => {
  alert(code + ' : ' + id);
});
</script>
```

## Параметри интерфејса

### Прилагођена ознака <б ид="1">

Прилагођене ХТМЛ ознаке, уведене у ЈС и коришћене у ХТМЛ-у.

## Управљање статусом

Користите <б ид="1"> за управљање глобалним статусом језика:

- **<б ид="1">**: Низ листе језика, у формату <б цласс="ЦОДЕ">[[име, ИД], ...]</б>.
- **<б ид="1">**: Преузмите ИД тренутног језика.
- **<б ид="1">**: Преузмите тренутно кодирање језика.
- **<б ид="1">**: Подесите ИД језика и обавестите претплатнике.
- **<б ид="1">**: Претплатите се на промене језика. Ако је језик подешен, повратни позив ће се одмах покренути. Вратите функцију одјаве.
