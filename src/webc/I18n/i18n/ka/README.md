# ღილაკები და ამომხტარი ფანჯრები ინტერფეისის ენების გადართვისთვის

- დააწკაპუნეთ ღილაკზე ენის შერჩევის მცურავი ფენის გამოსაჩენად.
- ავტომატურად მონიშნეთ ამჟამად არჩეული ენა.

## გამოიყენეთ დემო ვერსია

```html
<c-i18n></c-i18n>

<script type="module">
import "webc.site/I18n.js";
// გამოიყენეთ შემდეგი JS კოდი ენის ცვლილებების მონიტორინგისთვის
import { onLang } from "webc.site/js/i18n.js";

onLang((code, id) => {
  alert(code + ' : ' + id);
});
</script>
```

## ინტერფეისის პარამეტრები

### მორგებული ტეგი `<c-i18n>`

მორგებული HTML ტეგები, დანერგილი JS-ში და გამოიყენება HTML-ში.

## სტატუსის მართვა

გამოიყენეთ `src/js/i18n.js` გლობალური ენის სტატუსის სამართავად:

- **`LANG_LI`**: ენების სიის მასივი, `[[სახელი, ID], ...]` ფორმატში.
- ***`lang()`**: მიიღეთ მიმდინარე ენის ID.
- ***`langCode()`**: მიიღეთ მიმდინარე ენის კოდირება.
- ***`langSet(id)`**: დააყენეთ ენის ID და აცნობეთ აბონენტებს.
- ***`onLang(func)`**: გამოიწერეთ ენის ცვლილებები. თუ ენა დაყენებულია, გამოძახება დაუყოვნებლივ ამოქმედდება. დააბრუნეთ გამოწერის გაუქმების ფუნქცია.

## სტილის კლასი

ხატის ღილაკი.
