# ಇಂಟರ್ಫೇಸ್ ಭಾಷೆಗಳನ್ನು ಬದಲಾಯಿಸಲು ಗುಂಡಿಗಳು ಮತ್ತು ಪಾಪ್-ಅಪ್ ವಿಂಡೋಗಳು

- ಭಾಷಾ ಆಯ್ಕೆ ತೇಲುವ ಪದರವನ್ನು ಪಾಪ್ ಅಪ್ ಮಾಡಲು ಬಟನ್ ಅನ್ನು ಕ್ಲಿಕ್ ಮಾಡಿ.
- ಪ್ರಸ್ತುತ ಆಯ್ಕೆಮಾಡಿದ ಭಾಷೆಯನ್ನು ಸ್ವಯಂಚಾಲಿತವಾಗಿ ಹೈಲೈಟ್ ಮಾಡಿ.

## ಡೆಮೊ ಬಳಸಿ

```html
<c-i18n></c-i18n>

<script type="module">
import "webc.site/I18n.js";
// ಭಾಷೆಯ ಬದಲಾವಣೆಗಳನ್ನು ಮೇಲ್ವಿಚಾರಣೆ ಮಾಡಲು ಕೆಳಗಿನ JS ಕೋಡ್ ಬಳಸಿ
import { onLang } from "webc.site/js/i18n.js";

onLang((code, id) => {
  alert(code + ' : ' + id);
});
</script>
```

## ಇಂಟರ್ಫೇಸ್ ನಿಯತಾಂಕಗಳು

### ಕಸ್ಟಮ್ ಟ್ಯಾಗ್ `<c-i18n>`

ಕಸ್ಟಮ್ HTML ಟ್ಯಾಗ್‌ಗಳು, JS ನಲ್ಲಿ ಪರಿಚಯಿಸಲಾಗಿದೆ ಮತ್ತು HTML ನಲ್ಲಿ ಬಳಸಲಾಗಿದೆ.

## ಸ್ಥಿತಿ ನಿರ್ವಹಣೆ

ಜಾಗತಿಕ ಭಾಷಾ ಸ್ಥಿತಿಯನ್ನು ನಿರ್ವಹಿಸಲು `src/js/i18n.js` ಬಳಸಿ:

- **`LANG_LI`**: ಭಾಷಾ ಪಟ್ಟಿಯ ಅರೇ, `[[ಹೆಸರು, ID], ...]` ಸ್ವರೂಪದಲ್ಲಿ.
- **`lang()`**: ಪ್ರಸ್ತುತ ಭಾಷೆಯ ID ಪಡೆಯಿರಿ.
- **`langCode()`**: ಪ್ರಸ್ತುತ ಭಾಷೆಯ ಎನ್‌ಕೋಡಿಂಗ್ ಪಡೆಯಿರಿ.
- **`langSet(id)`**: ಭಾಷೆಯ ID ಅನ್ನು ಹೊಂದಿಸಿ ಮತ್ತು ಚಂದಾದಾರರಿಗೆ ಸೂಚಿಸಿ.
- **`onLang(func)`**: ಭಾಷೆಯ ಬದಲಾವಣೆಗಳಿಗೆ ಚಂದಾದಾರರಾಗಿ. ಭಾಷೆಯನ್ನು ಹೊಂದಿಸಿದ್ದರೆ, ಕಾಲ್‌ಬ್ಯಾಕ್ ಅನ್ನು ತಕ್ಷಣವೇ ಟ್ರಿಗರ್ ಮಾಡಲಾಗುತ್ತದೆ. ಅನ್‌ಸಬ್‌ಸ್ಕ್ರೈಬ್ ಕಾರ್ಯವನ್ನು ಹಿಂತಿರುಗಿ.

## ಶೈಲಿ ವರ್ಗ

ಐಕಾನ್ ಬಟನ್.
