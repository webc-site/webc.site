# ଶୀର୍ଷ ନାଭିଗେସନ୍ ବାର୍ ଯାହା ପୃଷ୍ଠା ସ୍କ୍ରୋଲ୍ କଲାବେଳେ ସ୍ୱୟଂଚାଳିତ ଭାବରେ ନଷ୍ଟ ହୋଇଯାଏ କିମ୍ବା ବିସ୍ତାର ହୁଏ |

- ** ଅଟୋ ଲୁଚାନ୍ତୁ **: ପୃଷ୍ଠାକୁ ସ୍କ୍ରୋଲ୍ କରିବା ସମୟରେ ସ୍ୱୟଂଚାଳିତ ଭାବରେ ଲୁଚାନ୍ତୁ |
- ** ଅଟୋ-ବିସ୍ତାର **: ପୃଷ୍ଠା ସ୍କ୍ରୋଲ୍ ହୋଇଗଲେ କିମ୍ବା ସ୍କ୍ରୋଲ୍ ବାର୍ ଅଦୃଶ୍ୟ ହେଲେ ସ୍ୱୟଂଚାଳିତ ଭାବରେ ପ୍ରଦର୍ଶିତ ହୁଏ |
- ** କାର୍ଯ୍ୟଦକ୍ଷତା ଅପ୍ଟିମାଇଜେସନ୍ **: ସ୍କ୍ରୋଲିଂକୁ ଥ୍ରଟ କରିବା ପାଇଁ `requestAnimationFrame` ବ୍ୟବହାର କରନ୍ତୁ |
- ** ଉତ୍ସଗୁଡିକ ମୁକ୍ତ କରନ୍ତୁ **: ଉପାଦାନ ନଷ୍ଟ ହେବାପରେ ଇଭେଣ୍ଟ ଶ୍ରୋତା ଏବଂ ପର୍ଯ୍ୟବେକ୍ଷକମାନେ ସ୍ୱୟଂଚାଳିତ ଭାବରେ ଅପସାରିତ ହୁଅନ୍ତି |

## ଡେମୋ ବ୍ୟବହାର କରନ୍ତୁ |

```html
<main>
  <c-vs>
    <c-nav>
      <b class="L">
        <a class="logo" href="#">LOGO</a>
      </b>
      <a class="item" href="#">ଲିଙ୍କ୍ |</a>
    </c-nav>
    <div class="content">ନାଭିଗେସନ୍ ଲୁଚାଇବାକୁ ସ୍ୱାଇପ୍ କରନ୍ତୁ, ଏହାକୁ ଦେଖାଇବାକୁ ସ୍ୱାଇପ୍ କରନ୍ତୁ |</div>
  </c-vs>
</main>
<style>
  main {
    position: relative;
    height: 300px;
    border: 1px solid #000;
  }
  c-nav {
    background: #fffc;
    backdrop-filter: blur(4px);
    border-bottom: 1px solid #ccc;
  }
  .content {

  }

```

## ବ୍ୟବହାର ପାଇଁ ନିର୍ଦ୍ଦେଶାବଳୀ |

`<c-vs>` ର ଶିଶୁ କିମ୍ବା ବଂଶଧର ଉପାଦାନ ଭାବରେ ବ୍ୟବହୃତ ହେବା ଜରୁରୀ |

## ଶ style ଳୀ ସ୍ଥିତି

ଉପାଦାନ ନିମ୍ନ ଶ୍ରେଣୀ ନାମଗୁଡ଼ିକୁ ନିଜ ଭିତରେ ସୁଇଚ୍ କରେ:

- `D`: ସ୍ଥିତି ଦେଖାନ୍ତୁ (ସ୍ଲାଇଡ୍ ଡାଉନ୍ କିମ୍ବା ଉପରେ) |
- `I`: ସ୍ଥାନାନ୍ତରଣ ଅବସ୍ଥା ଲୁଚି ରହିଛି |
- `H`: ଲୁକ୍କାୟିତ ସ୍ଥିତି |

## ଷ୍ଟାଇଲ୍ ଭେରିଏବଲ୍ |

- `--top`: ଉପାଦାନ ଦ୍ୱାରା ସ୍ୱୟଂଚାଳିତ ଭାବରେ ଉଚ୍ଚତାର ଏକ ନକାରାତ୍ମକ ମୂଲ୍ୟରେ ଅପଡେଟ୍, ପୋଜିସନ୍ ଏବଂ ଟ୍ରାନ୍ସଫିସନ୍ ପାଇଁ ବ୍ୟବହୃତ |
