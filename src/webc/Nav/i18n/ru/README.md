# Верхняя панель навигации, которая автоматически сворачивается или расширяется при прокрутке страницы.

- **Автоматическое скрытие**: автоматическое скрытие при прокрутке страницы вниз.
- **Автоматическое расширение**: автоматически отображается при прокрутке страницы вверх или при исчезновении полосы прокрутки.
- **Оптимизация производительности**. Используйте `requestAnimationFrame`, чтобы ограничить прокрутку.
- **Освободить ресурсы**: прослушиватели и наблюдатели событий автоматически удаляются при уничтожении компонента.

## Используйте демо-версию

```html
<main>
  <c-vs>
    <c-nav>
      <b class="L">
        <a class="logo" href="#">LOGO</a>
      </b>
      <a class="item" href="#">Связь</a>
    </c-nav>
    <div class="content">Проведите пальцем вниз, чтобы скрыть навигацию, проведите пальцем вверх, чтобы показать ее.</div>
  </c-vs>
</main>
<style>
  main {
    position: relative;
    height: 300px;
    border: 1px solid #000;
  }
  c-nav {
    background: #fffc;
    backdrop-filter: blur(4px);
    border-bottom: 1px solid #ccc;
  }
  .content {
    height: 200dvh;
    background: linear-gradient(to bottom, #fff, #ccc);
    padding: 80px 20px 20px;
    color: #666;
    text-align: center;
  }
</style>
```

## Инструкция по применению

Должен использоваться как дочерний или дочерний элемент `<c-vs>`.

## статус стиля

Компонент переключает внутри себя следующие имена классов:

- `D`: отображение статуса (скольжение вниз или вверх).
- `I`: переходное состояние скрыто.
- `H`: статус «Скрытый».

## переменные стиля

- `--top`: автоматически обновляется компонентом до отрицательного значения высоты, используемого для позиционирования и перехода.
