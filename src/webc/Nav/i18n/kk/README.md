# Бет жылжытқанда автоматты түрде жиырылатын немесе кеңейетін жоғарғы шарлау жолағы

- **Авто жасыру**: бетті төмен жылжытқанда автоматты түрде жасыру.
- **Автоматты түрде кеңейту**: бет жоғары жылжытылғанда немесе айналдыру жолағы жоғалғанда автоматты түрде көрсетіледі.
- **Өнімділікті оңтайландыру**: айналдыруды азайту үшін `requestAnimationFrame` пайдаланыңыз.
- **Шығару ресурстары**: құрамдас жойылған кезде оқиға тыңдаушылары мен бақылаушылары автоматты түрде жойылады.

## Демонстрацияны пайдаланыңыз

```html
<main>
  <c-vs>
    <c-nav>
      <b class="L">
        <a class="logo" href="#">LOGO</a>
      </b>
      <a class="item" href="#">Сілтеме</a>
    </c-nav>
    <div class="content">Навигацияны жасыру үшін төмен сырғытыңыз, оны көрсету үшін жоғары сырғытыңыз</div>
  </c-vs>
</main>
<style>
  main {
    position: relative;
    height: 300px;
    border: 1px solid #000;
  }
  c-nav {
    background: #fffc;
    backdrop-filter: blur(4px);
    border-bottom: 1px solid #ccc;
  }
  .content {
    height: 200dvh;
    background: linear-gradient(to bottom, #fff, #ccc);
    padding: 80px 20px 20px;
    color: #666;
    text-align: center;
  }
</style>
```

## Қолдану нұсқаулары

`<c-vs>` еншілес немесе ұрпақ элементі ретінде пайдаланылуы керек.

## стиль күйі

Компонент өз ішінде келесі класс атауларын ауыстырады:

- `D`: күйді көрсету (төмен немесе жоғары сырғытыңыз).
- `I`: өтпелі күй жасырылады.
- `H`: жасырын күй.

## стиль айнымалылары

- `--top`: Орналасу және өту үшін пайдаланылатын биіктіктің теріс мәніне құрамдас автоматты түрде жаңартылады.
