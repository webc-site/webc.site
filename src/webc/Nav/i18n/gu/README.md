# ટોચના નેવિગેશન બાર જે જ્યારે પૃષ્ઠ સ્ક્રોલ કરે છે ત્યારે આપમેળે સંકુચિત અથવા વિસ્તૃત થાય છે

- **સ્વતઃ છુપાવો**: પૃષ્ઠ નીચે સ્ક્રોલ કરતી વખતે આપમેળે છુપાવો.
- **સ્વતઃ-વિસ્તૃત**: જ્યારે પૃષ્ઠ ઉપર સ્ક્રોલ કરવામાં આવે અથવા સ્ક્રોલ બાર અદૃશ્ય થઈ જાય ત્યારે આપમેળે પ્રદર્શિત થાય છે.
- **પર્ફોર્મન્સ ઓપ્ટિમાઇઝેશન**: સ્ક્રોલિંગ થ્રોટલ કરવા માટે `requestAnimationFrame` નો ઉપયોગ કરો.
- **પ્રકાશન સંસાધનો**: જ્યારે ઘટક નાશ પામે છે ત્યારે ઇવેન્ટ શ્રોતાઓ અને નિરીક્ષકો આપમેળે દૂર થઈ જાય છે.

## ડેમોનો ઉપયોગ કરો

```html
<main>
  <c-vs>
    <c-nav>
      <b class="L">
        <a class="logo" href="#">LOGO</a>
      </b>
      <a class="item" href="#">લિંક</a>
    </c-nav>
    <div class="content">નેવિગેશન છુપાવવા માટે નીચે સ્વાઇપ કરો, તેને બતાવવા માટે ઉપર સ્વાઇપ કરો</div>
  </c-vs>
</main>
<style>
  main {
    position: relative;
    height: 300px;
    border: 1px solid #000;
  }
  c-nav {
    background: #fffc;
    backdrop-filter: blur(4px);
    border-bottom: 1px solid #ccc;
  }
  .content {
    height: 200dvh;
    background: linear-gradient(to bottom, #fff, #ccc);
    padding: 80px 20px 20px;
    color: #666;
    text-align: center;
  }
</style>
```

## ઉપયોગ માટે સૂચનાઓ

`<c-vs>` ના બાળક અથવા વંશજ તત્વ તરીકે ઉપયોગ કરવો આવશ્યક છે.

## શૈલી સ્થિતિ

ઘટક નીચેના વર્ગના નામોને પોતાની અંદર સ્વિચ કરે છે:

- `D`: સ્થિતિ બતાવો (નીચે અથવા ઉપર સ્લાઇડ કરો).
- `I`: સંક્રમણ સ્થિતિ છુપાવવામાં આવી રહી છે.
- `H`: છુપાયેલ સ્થિતિ.

## શૈલી ચલો

- `--top`: ઘટક દ્વારા ઊંચાઈના નકારાત્મક મૂલ્યમાં આપમેળે અપડેટ થાય છે, જેનો ઉપયોગ સ્થિતિ અને સંક્રમણ માટે થાય છે.
