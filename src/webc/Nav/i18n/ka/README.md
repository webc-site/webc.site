# ზედა ნავიგაციის ზოლი, რომელიც ავტომატურად იშლება ან ფართოვდება გვერდის გადახვევისას

- **ავტომატური დამალვა**: ავტომატურად დამალვა გვერდის ქვემოთ გადახვევისას.
- **ავტომატური გაფართოება**: ავტომატურად გამოჩნდება გვერდის ზემოთ გადახვევისას ან გადახვევის ზოლის გაქრობისას.
- **შესრულების ოპტიმიზაცია**: გამოიყენეთ `requestAnimationFrame` გადახვევისთვის.
- **გამოშვების რესურსები**: ღონისძიების მსმენელები და დამკვირვებლები ავტომატურად იშლება კომპონენტის განადგურებისას.

## გამოიყენეთ დემო ვერსია

```html
<main>
  <c-vs>
    <c-nav>
      <b class="L">
        <a class="logo" href="#">LOGO</a>
      </b>
      <a class="item" href="#">ბმული</a>
    </c-nav>
    <div class="content">გადაფურცლეთ ქვემოთ ნავიგაციის დასამალად, გადაფურცლეთ ზემოთ მის საჩვენებლად</div>
  </c-vs>
</main>
<style>
  main {
    position: relative;
    height: 300px;
    border: 1px solid #000;
  }
  c-nav {
    background: #fffc;
```

## გამოყენების ინსტრუქცია

უნდა იქნას გამოყენებული `<c-vs>`-ის შვილად ან შთამომავალ ელემენტად.

## სტილის სტატუსი

კომპონენტი ცვლის შემდეგ კლასის სახელებს თავის შიგნით:

- `D`: სტატუსის ჩვენება (გასრიალეთ ქვემოთ ან ზემოთ).
- `I`: გარდამავალი მდგომარეობა დამალულია.
- `H`: დამალული სტატუსი.

## სტილის ცვლადები

- `--top`: ავტომატურად განახლდება კომპონენტის მიერ სიმაღლის უარყოფით მნიშვნელობამდე, რომელიც გამოიყენება პოზიციონირებისა და გადასვლისთვის.
