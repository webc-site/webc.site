# ಪುಟ ಸ್ಕ್ರಾಲ್ ಮಾಡಿದಾಗ ಸ್ವಯಂಚಾಲಿತವಾಗಿ ಕುಸಿಯುವ ಅಥವಾ ವಿಸ್ತರಿಸುವ ಉನ್ನತ ನ್ಯಾವಿಗೇಷನ್ ಬಾರ್

- **ಸ್ವಯಂ ಮರೆಮಾಡು**: ಪುಟವನ್ನು ಕೆಳಗೆ ಸ್ಕ್ರೋಲ್ ಮಾಡುವಾಗ ಸ್ವಯಂಚಾಲಿತವಾಗಿ ಮರೆಮಾಡಿ.
- **ಸ್ವಯಂ-ವಿಸ್ತರಿಸು**: ಪುಟವನ್ನು ಮೇಲಕ್ಕೆ ಸ್ಕ್ರಾಲ್ ಮಾಡಿದಾಗ ಅಥವಾ ಸ್ಕ್ರಾಲ್ ಬಾರ್ ಕಣ್ಮರೆಯಾದಾಗ ಸ್ವಯಂಚಾಲಿತವಾಗಿ ಪ್ರದರ್ಶಿಸಲಾಗುತ್ತದೆ.
- **ಪರ್ಫಾರ್ಮೆನ್ಸ್ ಆಪ್ಟಿಮೈಸೇಶನ್**: ಥ್ರೊಟಲ್ ಸ್ಕ್ರೋಲಿಂಗ್ ಮಾಡಲು `requestAnimationFrame` ಬಳಸಿ.
- ** ಬಿಡುಗಡೆ ಸಂಪನ್ಮೂಲಗಳು**: ಘಟಕವು ನಾಶವಾದಾಗ ಈವೆಂಟ್ ಕೇಳುಗರು ಮತ್ತು ವೀಕ್ಷಕರನ್ನು ಸ್ವಯಂಚಾಲಿತವಾಗಿ ತೆಗೆದುಹಾಕಲಾಗುತ್ತದೆ.

## ಬಳಕೆಗೆ ಸೂಚನೆಗಳು

`<c-vs>`ನ ಮಗು ಅಥವಾ ವಂಶಸ್ಥರ ಅಂಶವಾಗಿ ಬಳಸಬೇಕು.

## ಶೈಲಿಯ ಸ್ಥಿತಿ

ಘಟಕವು ಈ ಕೆಳಗಿನ ವರ್ಗದ ಹೆಸರುಗಳನ್ನು ತನ್ನೊಳಗೆ ಬದಲಾಯಿಸುತ್ತದೆ:

- `D`: ಸ್ಥಿತಿಯನ್ನು ತೋರಿಸಿ (ಕೆಳಗೆ ಅಥವಾ ಮೇಲ್ಭಾಗದಲ್ಲಿ ಸ್ಲೈಡ್ ಮಾಡಿ).
- `I`: ಪರಿವರ್ತನೆಯ ಸ್ಥಿತಿಯನ್ನು ಮರೆಮಾಡಲಾಗಿದೆ.
- `H`: ಗುಪ್ತ ಸ್ಥಿತಿ.

## ಶೈಲಿ ಅಸ್ಥಿರ

- `--top`: ಸ್ಥಾನೀಕರಣ ಮತ್ತು ಪರಿವರ್ತನೆಗಾಗಿ ಬಳಸಲಾಗುವ ಎತ್ತರದ ಋಣಾತ್ಮಕ ಮೌಲ್ಯಕ್ಕೆ ಘಟಕದಿಂದ ಸ್ವಯಂಚಾಲಿತವಾಗಿ ನವೀಕರಿಸಲಾಗಿದೆ.
