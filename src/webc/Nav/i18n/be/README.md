# Верхняя панэль навігацыі, якая аўтаматычна згортваецца або пашыраецца пры пракрутцы старонкі

- **Аўтаматычнае хаванне**: аўтаматычна хаваць пры прагортцы старонкі ўніз.
- **Аўтаматычнае разгортванне**: аўтаматычна адлюстроўваецца, калі старонка пракручваецца ўверх або знікае паласа пракруткі.
- **Аптымізацыя прадукцыйнасці**: выкарыстоўвайце `requestAnimationFrame`, каб паменшыць пракрутку.
- **Вызваленне рэсурсаў**: слухачы і назіральнікі падзей аўтаматычна выдаляюцца, калі кампанент знішчаецца.

## Выкарыстоўвайце дэма

```html
<main>
  <c-vs>
    <c-nav>
      <b class="L">
        <a class="logo" href="#">LOGO</a>
      </b>
      <a class="item" href="#">Спасылка</a>
    </c-nav>
    <div class="content">Правядзіце пальцам уніз, каб схаваць навігацыю, правядзіце пальцам уверх, каб паказаць яе</div>
  </c-vs>
</main>
<style>
  main {
    position: relative;
    height: 300px;
    border: 1px solid #000;
  }
  c-nav {
    background: #fffc;
    backdrop-filter: blur(4px);
    border-bottom: 1px solid #ccc;
  }
  .content {
    height: 200dvh;
    background: linear-gradient(to bottom, #fff, #ccc);
    padding: 80px 20px 20px;
    color: #666;
    text-align: center;
  }
</style>
```

## Інструкцыя па ўжыванні

Павінен выкарыстоўвацца ў якасці даччынага элемента або элемента-нашчадка `<c-vs>`.

## статус стылю

Кампанент перамыкае наступныя імёны класаў у сабе:

- `D`: Паказаць статус (ссуньце ўніз або ўверсе).
- `I`: пераходны стан схаваны.
- `H`: Схаваны статус.

## зменныя стылю

- `--top`: аўтаматычна абнаўляецца кампанентам да адмоўнага значэння вышыні, якое выкарыстоўваецца для пазіцыянавання і пераходу.
