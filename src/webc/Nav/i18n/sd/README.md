# مٿيون نيويگيشن بار جيڪو خود بخود ختم ٿئي ٿو يا وڌايو وڃي ٿو جڏهن صفحو اسڪرال ٿئي ٿو

- **خودڪار لڪايو**: پاڻمرادو لڪايو جڏهن صفحو هيٺ ڪيو وڃي.
- **خودڪار وڌاءُ**: خودڪار طور تي ڏيکاريو ويندو آهي جڏهن صفحو اسڪرال ڪيو ويندو آهي يا اسڪرول بار غائب ٿي ويندو آهي.
- **ڪارڪردگي جي اصلاح**: استعمال ڪريو `requestAnimationFrame` طومار ڪرڻ لاءِ.
- **ريليز وسيلا**: واقعا ٻڌندڙن ۽ مبصرن کي خودڪار طريقي سان هٽايو ويندو آهي جڏهن جزو تباهه ٿي ويندو آهي.

## ڊيمو استعمال ڪريو

```html
<main>
  <c-vs>
    <c-nav>
      <b class="L">
        <a class="logo" href="#">LOGO</a>
      </b>
      <a class="item" href="#">ڳنڍ</a>
    </c-nav>
    <div class="content">نيويگيشن کي لڪائڻ لاءِ ھيٺ سوائپ ڪريو، ان کي ڏيکارڻ لاءِ مٿي سوائپ ڪريو</div>
  </c-vs>
</main>
<style>
  main {
    position: relative;
    height: 300px;
    border: 1px solid #000;
  }
  c-nav {
    background: #fffc;
    backdrop-filter: blur(4px);
    border-bottom: 1px solid #ccc;
  }
  .content {
    height: 200dvh;
    background: linear-gradient(to bottom, #fff, #ccc);
    padding: 80px 20px 20px;
    color: #666;
    text-align: center;
  }
</style>
```

## استعمال لاء هدايتون

`<c-vs>` جي ٻار يا نسلي عنصر طور استعمال ڪيو وڃي.

## انداز جي حالت

جزو ھيٺين طبقي جا نالا پاڻ ۾ مٽائي ٿو:

- `D`: ڏيکاريو اسٽيٽس (سائيڊ ھيٺ يا مٿي تي).
- `I`: منتقلي جي حالت لڪايو پيو وڃي.
- `H`: پوشیدہ حالت.

## انداز متغير

- `--top`: خودڪار طور تي جزو طرفان اونچائي جي منفي قدر تائين اپڊيٽ ڪيو ويو، پوزيشن ۽ منتقلي لاء استعمال ڪيو ويو.
