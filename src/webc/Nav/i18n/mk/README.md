# Горна лента за навигација која автоматски се собира или се шири кога страницата се лизга

- **Автоматско сокривање**: Автоматски сокриј при лизгање надолу на страницата.
- **Автоматско проширување**: Автоматски се прикажува кога страницата се лизга нагоре или кога лентата за лизгање исчезнува.
- **Оптимизација на перформанси**: Користете `requestAnimationFrame` за да го пригушите лизгањето.
- **Ослободете ресурси**: Слушачите и набљудувачите на настани автоматски се отстрануваат кога компонентата е уништена.

## Користете го демо

```html
<main>
  <c-vs>
    <c-nav>
      <b class="L">
        <a class="logo" href="#">LOGO</a>
      </b>
      <a class="item" href="#">Врска</a>
    </c-nav>
    <div class="content">Повлечете надолу за да ја скриете навигацијата, повлечете нагоре за да ја прикажете</div>
  </c-vs>
</main>
<style>
  main {
    position: relative;
    height: 300px;
    border: 1px solid #000;
  }
  c-nav {
    background: #fffc;
    backdrop-filter: blur(4px);
    border-bottom: 1px solid #ccc;
  }
  .content {
    height: 200dvh;
    background: linear-gradient(to bottom, #fff, #ccc);
    padding: 80px 20px 20px;
    color: #666;
    text-align: center;
  }
</style>
```

## Упатство за употреба

Мора да се користи како дете или потомок на `<c-vs>`.

## статус на стил

Компонентата ги менува следните имиња на класи во себе:

- `D`: Прикажи статус (лизгајте надолу или на врвот).
- `I`: Состојбата на транзиција е скриена.
- `H`: Скриен статус.

## стилски променливи

- `--top`: Автоматски се ажурира од компонентата до негативна вредност на висината, што се користи за позиционирање и транзиција.
