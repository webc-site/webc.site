# نوار پیمایش بالا که به طور خودکار با پیمایش صفحه جمع می‌شود یا بزرگ می‌شود

- **Auto Hide**: هنگام پایین رفتن صفحه به صورت خودکار مخفی می شود.
- **بسط خودکار**: هنگامی که صفحه به بالا پیمایش می شود یا نوار اسکرول ناپدید می شود، به طور خودکار نمایش داده می شود.
- **بهینه سازی عملکرد**: از `requestAnimationFrame` برای اسکرول کردن دریچه گاز استفاده کنید.
- **منابع انتشار**: شنوندگان و ناظران رویداد به طور خودکار با از بین رفتن مؤلفه حذف می شوند.

## از دمو استفاده کنید

```html
<main>
  <c-vs>
    <c-nav>
      <b class="L">
        <a class="logo" href="#">LOGO</a>
      </b>
      <a class="item" href="#">پیوند</a>
    </c-nav>
    <div class="content">برای پنهان کردن پیمایش، انگشت خود را به پایین بکشید، برای نمایش آن، انگشت خود را به سمت بالا بکشید</div>
  </c-vs>
</main>
<style>
  main {
    position: relative;
    height: 300px;
    border: 1px solid #000;
  }
  c-nav {
    background: #fffc;
    backdrop-filter: blur(4px);
    border-bottom: 1px solid #ccc;
  }
  .content {
    height: 200dvh;
    background: linear-gradient(to bottom, #fff, #ccc);
    padding: 80px 20px 20px;
    color: #666;
    text-align: center;
  }
</style>
```

## دستورالعمل استفاده

باید به عنوان عنصر فرزند یا فرزند `<c-vs>` استفاده شود.

## وضعیت سبک

کامپوننت نام کلاس های زیر را درون خودش تغییر می دهد:

- `D`: نمایش وضعیت (به پایین یا در بالا بکشید).
- `I`: حالت انتقال پنهان است.
- `H`: وضعیت پنهان.

## متغیرهای سبک

- `--top`: به‌طور خودکار توسط مؤلفه به مقدار منفی ارتفاع به‌روزرسانی می‌شود که برای موقعیت‌یابی و انتقال استفاده می‌شود.
