# स्विच करने योग्य टैब

- लेबल की चयनित स्थिति नीचे दिए गए सामग्री क्षेत्र के साथ सहजता से एकीकृत है।
- संरचनात्मक विशिष्टताएँ: चाइल्ड तत्वों को लेबल नेविगेशन के रूप में `nav > a` का उपयोग करने के लिए निर्धारित किया गया है, और प्रत्यक्ष चाइल्ड तत्व `b` को सामग्री क्षेत्र के रूप में उपयोग किया जाता है।
- `a[value]` टैग को `b[slot]` सामग्री नोड के साथ स्वचालित रूप से संबद्ध करें।
- `key` विशेषता के माध्यम से स्वचालित रूप से वर्तमान सक्रिय पृष्ठ को `localStorage` पर बनाए रखने का समर्थन करता है।
- `value` विशेषता के माध्यम से सक्रिय टैब पृष्ठ को निर्दिष्ट करने का समर्थन करता है (यदि कोई कैश नहीं है और `value` है, तो `class="A"` वाला टैग या पहला टैग डिफ़ॉल्ट रूप से चुना जाएगा)।
- लाइटवेट लाइट डोम कार्यान्वयन, कोई छाया डोम बाधा नहीं, शैली अनुकूलन में उच्च स्तर की स्वतंत्रता।

## डेमो का प्रयोग करें

```html
<c-tab key="my-tab-key" value="banana">
  <nav>
    <a value="apple">सेब</a>
    <a value="banana">केला</a>
    <a value="orange">संतरा</a>
  </nav>
  <b slot="apple">सेब एक पौष्टिक फल है</b>
  <b slot="banana">केला एक उष्णकटिबंधीय फल है</b>
  <b slot="orange">संतरे में विटामिन सी प्रचुर मात्रा में होता है</b>
</c-tab>
```

## संपत्ति

- `key`: लोकलस्टोरेज दृढ़ता के लिए उपयोग किया जाने वाला कुंजी नाम
- `value`: वर्तमान में चयनित मान

## कस्टम इवेंट

- `change`: टैब स्विच होने पर ट्रिगर होता है, `e.value` वर्तमान में सक्रिय टैग का `value` मान है।
