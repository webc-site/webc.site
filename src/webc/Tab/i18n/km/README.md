# ផ្ទាំងដែលអាចប្តូរបាន។

- ស្ថានភាពដែលបានជ្រើសរើសនៃស្លាកគឺត្រូវបានរួមបញ្ចូលគ្នាយ៉ាងរលូនជាមួយនឹងផ្ទៃមាតិកាខាងក្រោម។
- ការកំណត់រចនាសម្ព័ន្ធ៖ ធាតុកូនត្រូវបានជួសជុលដើម្បីប្រើ `nav > a`ជាការរុករកស្លាក ហើយធាតុកូនដោយផ្ទាល់ `b`ត្រូវបានប្រើជាផ្ទៃមាតិកា។
- ភ្ជាប់ស្លាក `a[value]`ដោយស្វ័យប្រវត្តិជាមួយថ្នាំងមាតិកា `b[slot]`។
- គាំទ្រការបន្តទំព័រសកម្មបច្ចុប្បន្នដោយស្វ័យប្រវត្តិទៅ `localStorage` តាមរយៈគុណលក្ខណៈ `key`។
- គាំទ្រការបញ្ជាក់ទំព័រផ្ទាំងដែលបានធ្វើឱ្យសកម្មតាមរយៈគុណលក្ខណៈ `value` (ប្រសិនបើមិនមានឃ្លាំងសម្ងាត់ និង `value` ស្លាកដែលមាន `class="A"` ឬស្លាកដំបូងនឹងត្រូវបានជ្រើសរើសតាមលំនាំដើម)។
- ការអនុវត្ត DOM ពន្លឺស្រាល មិនមានការរាំងស្ទះ Shadow DOM កម្រិតសេរីភាពខ្ពស់ក្នុងការកំណត់រចនាប័ទ្មតាមបំណង។

## ប្រើការបង្ហាញ

```html
<c-tab key="my-tab-key" value="banana">
  <nav>
    <a value="apple">ផ្លែប៉ោម</a>
    <a value="banana">ចេក</a>
    <a value="orange">ក្រូចឃ្វិច</a>
  </nav>
  <b slot="apple">ផ្លែប៉ោមគឺជាផ្លែឈើដែលមានជីវជាតិ</b>
  <b slot="banana">ចេកគឺជាផ្លែឈើត្រូពិច</b>
  <b slot="orange">ផ្លែក្រូចសម្បូរទៅដោយវីតាមីន C</b>
</c-tab>
```

## ទ្រព្យសម្បត្តិ

- `key`៖ ឈ្មោះ​គន្លឹះ​ប្រើ​សម្រាប់​ការ​រក្សា​ទុក​ក្នុង​មូលដ្ឋាន
- `value`៖ តម្លៃដែលបានជ្រើសរើសបច្ចុប្បន្ន

## ព្រឹត្តិការណ៍ផ្ទាល់ខ្លួន

- `change`៖ បាន​កេះ​នៅ​ពេល​ផ្ទាំង​ត្រូវ​បាន​ប្ដូរ `e.value` គឺ​ជា `value` តម្លៃ​នៃ​ស្លាក​ដែល​បាន​ធ្វើ​ឱ្យ​សកម្ម​បច្ចុប្បន្ន។
