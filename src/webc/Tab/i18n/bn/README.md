# পরিবর্তনযোগ্য ট্যাব

- লেবেলের নির্বাচিত অবস্থা নিচের বিষয়বস্তুর এলাকার সাথে নির্বিঘ্নে একত্রিত করা হয়েছে।
- স্ট্রাকচারাল স্পেসিফিকেশন: লেবেল নেভিগেশন হিসেবে `nav > a` ব্যবহার করার জন্য চাইল্ড এলিমেন্ট ঠিক করা হয়েছে এবং সরাসরি চাইল্ড এলিমেন্ট `b` কন্টেন্ট এরিয়া হিসেবে ব্যবহার করা হয়েছে।
- স্বয়ংক্রিয়ভাবে `a[value]` ট্যাগটিকে `b[slot]` সামগ্রী নোডের সাথে সংযুক্ত করুন৷
- `key` বৈশিষ্ট্যের মাধ্যমে স্বয়ংক্রিয়ভাবে বর্তমান সক্রিয় পৃষ্ঠাটিকে `localStorage` টিকে সমর্থন করে৷
- `value` অ্যাট্রিবিউটের মাধ্যমে সক্রিয় ট্যাব পৃষ্ঠা নির্দিষ্ট করা সমর্থন করে (যদি কোন ক্যাশে এবং `value` না থাকে, `class="A"` যুক্ত ট্যাগ বা প্রথম ট্যাগটি ডিফল্টরূপে নির্বাচন করা হবে)।
- লাইটওয়েট হালকা DOM বাস্তবায়ন, কোন ছায়া DOM বাধা নেই, শৈলী কাস্টমাইজেশনে উচ্চ মাত্রার স্বাধীনতা।

## ডেমো ব্যবহার করুন

```html
<c-tab key="my-tab-key" value="banana">
  <nav>
    <a value="apple">আপেল</a>
    <a value="banana">কলা</a>
    <a value="orange">ট্যানজারিন</a>
  </nav>
  <b slot="apple">আপেল একটি পুষ্টিকর ফল</b>
  <b slot="banana">কলা একটি ক্রান্তীয় ফল</b>
  <b slot="orange">কমলালেবু ভিটামিন সি সমৃদ্ধ</b>
</c-tab>
```

## সম্পত্তি

- `key`: স্থানীয় স্টোরেজ অধ্যবসায়ের জন্য ব্যবহৃত কী নাম
- `value`: বর্তমানে নির্বাচিত মান

## কাস্টম ঘটনা

- `change`: যখন ট্যাবটি স্যুইচ করা হয় তখন ট্রিগার হয়, `e.value` হল বর্তমানে সক্রিয় ট্যাগের `value` মান।
