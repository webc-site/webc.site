# Εναλλασσόμενες καρτέλες

- Η επιλεγμένη κατάσταση της ετικέτας ενσωματώνεται άψογα με την περιοχή περιεχομένου παρακάτω.
- Δομικές προδιαγραφές: Τα θυγατρικά στοιχεία έχουν καθοριστεί για χρήση του `nav > a` ως πλοήγησης ετικέτας και το άμεσο θυγατρικό στοιχείο `b` χρησιμοποιείται ως περιοχή περιεχομένου.
- Συσχετίστε αυτόματα την ετικέτα `a[value]` με τον κόμβο περιεχομένου `b[slot]`.
- Υποστηρίζει την αυτόματη παραμονή της τρέχουσας ενεργής σελίδας σε `localStorage` μέσω του χαρακτηριστικού `key`.
- Υποστηρίζει τον καθορισμό της ενεργοποιημένης σελίδας καρτέλας μέσω του χαρακτηριστικού `value` (αν δεν υπάρχει προσωρινή μνήμη και `value`, η ετικέτα με `class="A"` ή η πρώτη ετικέτα θα επιλεγεί από προεπιλογή).
- Ελαφριά εφαρμογή Light DOM, χωρίς παρεμπόδιση Shadow DOM, υψηλός βαθμός ελευθερίας στην προσαρμογή στυλ.

## Χρησιμοποιήστε το demo

```html
<c-tab key="my-tab-key" value="banana">
  <nav>
    <a value="apple">μήλο</a>
    <a value="banana">μπανάνα</a>
    <a value="orange">μανταρίνι</a>
  </nav>
  <b slot="apple">Το μήλο είναι ένα θρεπτικό φρούτο</b>
  <b slot="banana">Η μπανάνα είναι ένα τροπικό φρούτο</b>
  <b slot="orange">Τα πορτοκάλια είναι πλούσια σε βιταμίνη C</b>
</c-tab>
```

## ιδιοκτησία

- `key`: όνομα κλειδιού που χρησιμοποιείται για τη διατήρηση της τοπικής αποθήκευσης
- `value`: επιλεγμένη τιμή

## Προσαρμοσμένες εκδηλώσεις

- `change`: Ενεργοποιείται κατά την εναλλαγή της καρτέλας, η `e.value` είναι η τιμή `value` της τρέχουσας ενεργοποιημένης ετικέτας.
