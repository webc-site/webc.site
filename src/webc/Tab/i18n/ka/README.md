# შესაცვლელი ჩანართები

- ეტიკეტის არჩეული მდგომარეობა შეუფერხებლად არის ინტეგრირებული ქვემოთ მოცემულ შიგთავსთან.
- სტრუქტურული სპეციფიკაციები: ბავშვის ელემენტები ფიქსირდება, რათა გამოიყენონ `nav > a` როგორც ლეიბლის ნავიგაცია, ხოლო პირდაპირი შვილობილი ელემენტი `b` გამოიყენება როგორც კონტენტის არე.
- ავტომატურად დააკავშირეთ `a[value]` ტეგი `b[slot]` შინაარსის კვანძთან.
- მხარს უჭერს ამჟამინდელი აქტიური გვერდის ავტომატურად შენარჩუნებას `localStorage` `key` ატრიბუტის მეშვეობით.
- მხარს უჭერს გააქტიურებული ჩანართის გვერდის მითითებას `value` ატრიბუტის მეშვეობით (თუ არ არის ქეში და `value`, ტეგი `class="A"` ან პირველი ტეგი შეირჩევა ნაგულისხმევად).
- მსუბუქი მსუბუქი DOM განხორციელება, Shadow DOM დაბრკოლების გარეშე, სტილის პერსონალიზაციის თავისუფლების მაღალი ხარისხი.

## გამოიყენეთ დემო ვერსია

```html
<c-tab key="my-tab-key" value="banana">
  <nav>
    <a value="apple">ვაშლი</a>
    <a value="banana">ბანანი</a>
    <a value="orange">მანდარინი</a>
  </nav>
  <b slot="apple">ვაშლი მკვებავი ხილია</b>
  <b slot="banana">ბანანი ტროპიკული ხილია</b>
  <b slot="orange">ფორთოხალი მდიდარია C ვიტამინით</b>
</c-tab>
```

## ქონება

- `key`: გასაღების სახელი გამოიყენება localStorage მდგრადობისთვის
- `value`: ამჟამად არჩეული მნიშვნელობა

## მორგებული ღონისძიებები

- `change`: ამოქმედდება ჩანართის გადართვისას, `e.value` არის ამჟამად გააქტიურებული ტეგის `value` მნიშვნელობა.
