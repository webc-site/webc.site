# 切り替え可能なタブ

- ラベルの選択状態は、下のコンテンツ領域とシームレスに統合されます。
- 構造仕様: 子要素はラベル ナビゲーションとして `nav > a` を使用するように固定されており、直接の子要素 ​​`b` はコンテンツ領域として使用されます。
- `a[value]` タグを `b[slot]` コンテンツ ノードに自動的に関連付けます。
- `key` 属性を通じて、現在のアクティブ ページを `localStorage` に自動的に永続化することをサポートします。
- `value` 属性によるアクティブ化されたタブ ページの指定をサポートします (キャッシュがなく `value` の場合、`class="A"` のタグまたは最初のタグがデフォルトで選択されます)。
- 軽量な Light DOM の実装、Shadow DOM の障害がなく、スタイルのカスタマイズの自由度が高い。

## デモを使用する

```html
<c-tab key="my-tab-key" value="banana">
  <nav>
    <a value="apple">りんご</a>
    <a value="banana">バナナ</a>
    <a value="orange">タンジェリン</a>
  </nav>
  <b slot="apple">リンゴは栄養価の高い果物です</b>
  <b slot="banana">バナナはトロピカルフルーツです</b>
  <b slot="orange">オレンジにはビタミンCが豊富に含まれています</b>
</c-tab>
```

## 財産

- `key`: localStorage の永続化に使用されるキー名
- `value`: 現在選択されている値

## カスタムイベント

- `change`: タブが切り替えられるとトリガーされます。`e.value` は、現在アクティブ化されているタグの `value` 値です。
