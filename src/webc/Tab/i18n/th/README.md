# แท็บสลับได้

- สถานะของป้ายกำกับที่เลือกจะผสานรวมเข้ากับพื้นที่เนื้อหาด้านล่างได้อย่างราบรื่น
- ข้อกำหนดด้านโครงสร้าง: องค์ประกอบลูกได้รับการแก้ไขให้ใช้ `nav > a` เป็นการนำทางป้ายกำกับ และองค์ประกอบลูกโดยตรง `b` ถูกใช้เป็นพื้นที่เนื้อหา
- เชื่อมโยงแท็ก `a[value]` กับโหนดเนื้อหา `b[slot]` โดยอัตโนมัติ
- รองรับการคงเพจที่ใช้งานปัจจุบันไว้โดยอัตโนมัติไปยัง `localStorage` ผ่านแอตทริบิวต์ `key`
- รองรับการระบุหน้าแท็บที่เปิดใช้งานผ่านแอตทริบิวต์ `value` (หากไม่มีแคชและ `value` แท็กที่มี `class="A"` หรือแท็กแรกจะถูกเลือกตามค่าเริ่มต้น)
- การใช้ Light DOM น้ำหนักเบา ไม่มีการขัดขวาง Shadow DOM มีอิสระในระดับสูงในการปรับแต่งสไตล์

## ใช้การสาธิต

```html
<c-tab key="my-tab-key" value="banana">
  <nav>
    <a value="apple">แอปเปิล</a>
    <a value="banana">กล้วย</a>
    <a value="orange">ส้มเขียวหวาน</a>
  </nav>
  <b slot="apple">แอปเปิ้ลเป็นผลไม้ที่มีคุณค่าทางโภชนาการ</b>
  <b slot="banana">กล้วยเป็นผลไม้เมืองร้อน</b>
  <b slot="orange">ส้มอุดมไปด้วยวิตามินซี</b>
</c-tab>
```

## คุณสมบัติ

- `key`: ชื่อคีย์ที่ใช้สำหรับการคงอยู่ของ localStorage
- `value`: ค่าที่เลือกในปัจจุบัน

## เหตุการณ์ที่กำหนดเอง

- `change`: ทริกเกอร์เมื่อมีการสลับแท็บ `e.value` คือค่า `value` ของแท็กที่เปิดใช้งานในปัจจุบัน
