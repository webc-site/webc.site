# כרטיסיות שניתן להחליף ביניהן

- המצב הנבחר של התווית משולב בצורה חלקה עם אזור התוכן למטה.
- מפרטים מבניים: רכיבי הצאצא קבועים לשימוש ב`nav > a` בתור ניווט בתווית, ורכיב הצאצא הישיר `b` משמש כאזור התוכן.
- שייך אוטומטית את התג `a[value]` לצומת התוכן `b[slot]`.
- תומך בשמירה אוטומטית של הדף הפעיל הנוכחי ל`localStorage` דרך התכונה `key`.
- תומך בציון דף הכרטיסייה המופעל דרך התכונה `value` (אם אין מטמון ו`value`, התג עם `class="A"` או התג הראשון ייבחר כברירת מחדל).
- יישום DOM קל משקל, ללא חסימת Shadow DOM, רמה גבוהה של חופש בהתאמה אישית של סגנון.

## השתמש בהדגמה

```html
<c-tab key="my-tab-key" value="banana">
  <nav>
    <a value="apple">תַפּוּחַ</a>
    <a value="banana">בָּנָנָה</a>
    <a value="orange">מַנדָרִינָה</a>
  </nav>
  <b slot="apple">תפוח הוא פרי מזין</b>
  <b slot="banana">בננה היא פרי טרופי</b>
  <b slot="orange">תפוזים עשירים בויטמין C</b>
</c-tab>
```

## נֶכֶס

- `key`: שם המפתח המשמש להתמדה של LocalStorage
- `value`: הערך הנבחר כעת

## אירועים מותאמים אישית

- `change`: מופעל כאשר הכרטיסייה משתנה, `e.value` הוא הערך `value` של התג המופעל כעת.
