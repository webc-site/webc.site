# स्विच करण्यायोग्य टॅब

- लेबलची निवडलेली स्थिती खालील सामग्री क्षेत्रासह अखंडपणे एकत्रित केली आहे.
- स्ट्रक्चरल स्पेसिफिकेशन्स: चाइल्ड एलिमेंट्स `nav > a` लेबल नेव्हिगेशन म्हणून वापरण्यासाठी निश्चित केले आहेत आणि डायरेक्ट चाइल्ड एलिमेंट `b` सामग्री क्षेत्र म्हणून वापरले जातात.
- `a[value]` टॅग `b[slot]` सामग्री नोडसह स्वयंचलितपणे संबद्ध करा.
- `key` विशेषताद्वारे वर्तमान सक्रिय पृष्ठास `localStorage` वर आपोआप टिकून राहण्यास समर्थन देते.
- सक्रिय केलेले टॅब पृष्ठ `value` विशेषता (जर कॅशे आणि `value` नसल्यास, `class="A"` सह टॅग किंवा प्रथम टॅग डीफॉल्टनुसार निवडला जाईल) द्वारे निर्दिष्ट करण्यास समर्थन देते.
- लाइटवेट लाइट DOM अंमलबजावणी, छाया DOM अडथळा नाही, शैली सानुकूलनात उच्च पदवी स्वातंत्र्य.

## डेमो वापरा

```html
<c-tab key="my-tab-key" value="banana">
  <nav>
    <a value="apple">सफरचंद</a>
    <a value="banana">केळी</a>
    <a value="orange">टेंजेरिन</a>
  </nav>
  <b slot="apple">सफरचंद हे पौष्टिक फळ आहे</b>
  <b slot="banana">केळी हे उष्णकटिबंधीय फळ आहे</b>
  <b slot="orange">संत्र्यामध्ये व्हिटॅमिन सी भरपूर प्रमाणात असते</b>
</c-tab>
```

## मालमत्ता

- `key`: लोकल स्टोरेज पर्सिस्टन्ससाठी की नाव वापरले
- `value`: सध्या निवडलेले मूल्य

## सानुकूल कार्यक्रम

- `change`: टॅब स्विच केल्यावर ट्रिगर होतो, `e.value` हे सध्या सक्रिय केलेल्या टॅगचे `value` मूल्य आहे.
