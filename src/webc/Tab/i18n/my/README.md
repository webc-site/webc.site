# ပြောင်းနိုင်သော တက်ဘ်များ

- အညွှန်း၏ ရွေးချယ်ထားသော အခြေအနေသည် အောက်ပါ အကြောင်းအရာဧရိယာနှင့် ချောမွေ့စွာ ပေါင်းစပ်ထားသည်။
- ဖွဲ့စည်းပုံဆိုင်ရာ သတ်မှတ်ချက်များ- ကလေးဒြပ်စင်များအား အညွှန်းလမ်းညွှန်မှုအဖြစ် `nav > a`ကို အသုံးပြုရန် ပြင်ဆင်ထားပြီး၊ တိုက်ရိုက်ကလေးဒြပ်စင် `b`ကို အကြောင်းအရာဧရိယာအဖြစ် အသုံးပြုပါသည်။
- `a[value]` tag ကို `b[slot]` အကြောင်းအရာ ဆုံမှတ်နှင့် အလိုအလျောက် ချိတ်ဆက်ပါ။
- `key` ရည်ညွှန်းချက်မှတစ်ဆင့် လက်ရှိအသုံးပြုနေသော စာမျက်နှာကို `localStorage`းသို့ အလိုအလျောက် ဆက်လက်လုပ်ဆောင်ပေးပါသည်။
- `value` ရည်ညွှန်းချက်အားဖြင့် အသက်သွင်းထားသော တက်ဘ်စာမျက်နှာအား သတ်မှတ်ခြင်းကို ပံ့ပိုးပေးသည် (ကက်ရှ်မရှိလျှင် `value`၊ `class="A"` သို့မဟုတ် ပထမတဂ်ကို မူရင်းအတိုင်း ရွေးချယ်မည်)။
- ပေါ့ပါးသော Light DOM ကို အကောင်အထည်ဖော်ခြင်း၊ Shadow DOM အတားအဆီးမရှိ၊ စတိုင်လ်စိတ်ကြိုက်ပြုလုပ်ခြင်းတွင် လွတ်လပ်မှုမြင့်မားခြင်း။

## ဒီမိုကိုသုံးပါ။

```html
<c-tab key="my-tab-key" value="banana">
  <nav>
    <a value="apple">ပန်းသီး</a>
    <a value="banana">ငှက်ပျောသီး</a>
    <a value="orange">အခွံပွလိမ္မော်သီး</a>
  </nav>
  <b slot="apple">ပန်းသီးသည် အာဟာရရှိသော အသီးတစ်မျိုးဖြစ်သည်။</b>
  <b slot="banana">ငှက်ပျောသီးသည် အပူပိုင်းဒေသ အသီးတစ်မျိုးဖြစ်သည်။</b>
  <b slot="orange">လိမ္မော်သီးမှာ ဗီတာမင် C ကြွယ်ဝပါတယ်။</b>
</c-tab>
```

## ပစ္စည်းဥစ္စာ

- `key`: localStorage မြဲမြံမှုအတွက် အသုံးပြုသော သော့အမည်
- `value`- လက်ရှိရွေးချယ်ထားသော တန်ဖိုး

## စိတ်ကြိုက်ပွဲများ

- `change`- တက်ဘ်ကို ပြောင်းလိုက်သောအခါ အစပျိုးလိုက်သည်၊ `e.value`သည် လက်ရှိဖွင့်ထားသည့် တဂ်၏ `value`တန်ဖိုး ဖြစ်သည်။
