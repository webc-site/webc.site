# Markdown စာသားကို လွှင့်ထုတ်ခြင်း

- ဖြည်းဖြည်းချင်း ခွဲခြမ်းစိပ်ဖြာပြီး Markdown ကို တင်ဆက်ပါ။
- အကြောင်းအရာသည် စခရင်ထက်ကျော်လွန်သွားသောအခါ အလိုအလျောက်လှိမ့်ရန် အောက်ခြေအစိတ်အပိုင်းနှင့် ပူးပေါင်းဆောင်ရွက်ပါ။
- နှိပ်သည့်အခါ သို့မဟုတ် အပေါ်သို့ လှိမ့်သွားသည့်အခါ auto scroll ကို ခေတ္တရပ်ပါ၊ အောက်သို့ဆင်းသောအခါတွင် ပြန်လည်စတင်ပါ။
- စာရိုက်ခြင်းအခြေအနေ အညွှန်းများကို အလိုအလျောက်ပြသပါ။

## ဒီမိုကိုသုံးပါ။

```html
<c-vs>
  <c-md-stream id="md-view"></c-md-stream>
</c-vs>

<script type="module">
import "webc.site/MdStream.js";

const el = document.getElementById("md-view"),
  sleep = (ms) => new Promise((resolve) => setTimeout(resolve, ms));

// ပြန်လာသော asynchronous generator လုပ်ဆောင်ချက်မှတစ်ဆင့် ဒေတာကို ထုတ်လွှင့်ခြင်း
el.gen = async function* () {
  အထွက်နှုန်း "# title\n";
  for (let i = 1; i <= 100; ++i) {
    await sleep(400);
    yield "- အပိုင်း " + i + "\n";
  }
};
</script>
```

## ပစ္စည်းဥစ္စာ

| attribute အမည် | အမျိုးအစား | ပုံဥပမာ |
| :--- | :--- | :--- |
| `gen` | `Function` | အပြိုင်အဆိုင်ပြုလုပ်ခြင်း/မီးစက်ကို ပြန်ပေးသည့် လုပ်ဆောင်ချက်၊ အကြောင်းအရာကို ရှင်းလင်းပြီး သတ်မှတ်ပြီးနောက် ပြန်လည်တင်ဆက်ခြင်း |

## စတိုင်

| အတန်း/တက်ဂ် | ပုံဥပမာ |
| :--- | :--- |
| `Md` | တပ်ဆင်သည့်အခါ အတန်းအစားကို အလိုအလျောက်ထည့်သွင်းပြီး Markdown စတိုင်ကို သတ်မှတ်ပါ။ |
| `i.T` | ဒေတာထည့်သွင်းစဉ်အတွင်း စာရိုက်ခြင်းဆိုင်ရာ အခြေအနေညွှန်ကိန်းကို အဆုံးတွင် အလိုအလျောက် ထည့်သွင်းခဲ့သည်။ |
