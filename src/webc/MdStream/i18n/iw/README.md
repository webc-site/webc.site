# עיבוד סטרימינג של טקסט Markdown

- נתח ועיבוד של Markdown בהדרגה
- שיתוף פעולה עם רכיב האב הגלילה כדי לגלול אוטומטית כאשר התוכן חורג מהמסך
- השהה את הגלילה האוטומטית בעת לחיצה או גלילה למעלה, המשך בעת גלילה מטה
- הצג אוטומטית מחווני סטטוס ההקלדה

## השתמש בהדגמה

```html
<c-vs>
  <c-md-stream id="md-view"></c-md-stream>
</c-vs>

<script type="module">
import "webc.site/MdStream.js";

const el = document.getElementById("md-view"),
  sleep = (ms) => new Promise((resolve) => setTimeout(resolve, ms));

// עיבוד נתונים בזרימת דרך פונקציית המחולל האסינכרוני המוחזרת
el.gen = async function* () {
  תשואה "# כותרת\n";
  for (let i = 1; i <= 100; ++i) {
    await sleep(400);
    תשואה "- פריט " + i + "\n";
  }
};
</script>
```

## נֶכֶס

| שם תכונה | סוּג | לְהַדגִים |
| :--- | :--- | :--- |
| `gen` | `Function` | פונקציה שמחזירה איטרטור/מחולל אסינכרוני, מנקה תוכן ומציגה מחדש לאחר הגדרה |

## סִגְנוֹן

| כיתה/תג | לְהַדגִים |
| :--- | :--- |
| `Md` | Class מתווסף אוטומטית לרכיב בעת ההרכבה, הגדר את סגנון Markdown |
| `i.T` | מחוון מצב ההקלדה נוסף אוטומטית בסוף במהלך הזנת הנתונים |
