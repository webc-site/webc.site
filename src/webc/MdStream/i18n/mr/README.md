# मार्कडाउन मजकूराचे स्ट्रीमिंग प्रस्तुतीकरण

- मार्कडाउन वाढत्या प्रमाणात पार्स आणि रेंडर करा
- जेव्हा सामग्री स्क्रीन ओलांडते तेव्हा स्वयंचलितपणे स्क्रोल करण्यासाठी स्क्रोलिंग मूळ घटकास सहकार्य करा
- क्लिक करताना किंवा वर स्क्रोल करताना ऑटोस्क्रोलला विराम द्या, खाली स्क्रोल करताना पुन्हा सुरू करा
- टाइपिंग स्थिती निर्देशक स्वयंचलितपणे प्रदर्शित करा

## डेमो वापरा

```html
<c-vs>
  <c-md-stream id="md-view"></c-md-stream>
</c-vs>

<script type="module">
import "webc.site/MdStream.js";

const el = document.getElementById("md-view"),
  sleep = (ms) => new Promise((resolve) => setTimeout(resolve, ms));

// परत आलेल्या असिंक्रोनस जनरेटर फंक्शनद्वारे प्रवाहित डेटा प्रस्तुतीकरण
el.gen = async function* () {
  उत्पन्न "# शीर्षक\n";
  for (let i = 1; i <= 100; ++i) {
    await sleep(400);
    उत्पन्न "- आयटम " + i + "\n";
  }
};
</script>
```

## मालमत्ता

| विशेषता नाव | प्रकार | स्पष्ट करणे |
| :--- | :--- | :--- |
| `gen` | `Function` | फंक्शन जे एसिंक्रोनस इटरेटर/जनरेटर परत करते, सामग्री साफ करते आणि सेटिंग केल्यानंतर पुन्हा प्रस्तुत करते |

## शैली

| वर्ग/टॅग | स्पष्ट करणे |
| :--- | :--- |
| `Md` | माउंट करताना वर्ग आपोआप घटकामध्ये जोडला जातो, मार्कडाउन शैली सेट करा |
| `i.T` | डेटा एंट्री दरम्यान टायपिंग स्थिती निर्देशक स्वयंचलितपणे शेवटी जोडला जातो |
