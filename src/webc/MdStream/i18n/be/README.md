# Патокавы рэндэрынг тэксту Markdown

- Паступовы аналіз і візуалізацыя Markdown
- Супрацоўнічайце з бацькоўскім элементам пракруткі для аўтаматычнай пракруткі, калі змесціва выходзіць за межы экрана
- Пры пстрычцы або пракрутцы ўверх прыпыняйце аўтапракрутку, аднаўляйце пры пракрутцы ўніз
- Аўтаматычна адлюстроўваць індыкатары стану ўводу

## Выкарыстоўвайце дэма

```html
<c-vs>
  <c-md-stream id="md-view"></c-md-stream>
</c-vs>

<script type="module">
import "webc.site/MdStream.js";

const el = document.getElementById("md-view"),
  sleep = (ms) => new Promise((resolve) => setTimeout(resolve, ms));

// Патокавы рэндэрынг дадзеных праз вернутую функцыю асінхроннага генератара
el.gen = async function* () {
  yield "# назва\n";
  for (let i = 1; i <= 100; ++i) {
    await sleep(400);
    yield "- пункт " + i + "\n";
  }
};
</script>
```

## уласнасць

| імя атрыбута | тыпу | ілюстраваць |
| :--- | :--- | :--- |
| `gen` | `Function` | Функцыя, якая вяртае асінхронны ітэратар/генератар, ачышчае змесціва і паўторна адлюстроўвае пасля ўстаноўкі |

## стыль

| Клас/тэг | ілюстраваць |
| :--- | :--- |
| `Md` | Клас аўтаматычна дадаецца ў кампанент пры мантаванні, задайце стыль Markdown |
| `i.T` | Індыкатар стану ўводу аўтаматычна дадаецца ў канцы падчас уводу даных |
