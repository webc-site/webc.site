# مارڪ ڊائون متن جي اسٽريمنگ رينڊرنگ

- وڌ ۾ وڌ پارس ۽ رينڊر مارڪ ڊائون
- پاڻمرادو اسڪرول ڪرڻ لاءِ اسڪرولنگ پيرن عنصر سان تعاون ڪريو جڏھن مواد اسڪرين کان وڌي وڃي
- بند ڪريو آٽو اسڪرول جڏھن ڪلڪ ڪيو يا مٿي اسڪرول ڪيو، وري شروع ڪريو جڏھن ھيٺ اسڪرول ڪريو
- خودڪار طور تي ٽائپنگ اسٽيٽس اشارو ڏيکاريو

## ڊيمو استعمال ڪريو

```html
<c-vs>
  <c-md-stream id="md-view"></c-md-stream>
</c-vs>

<script type="module">
import "webc.site/MdStream.js";

const el = document.getElementById("md-view"),
  sleep = (ms) => new Promise((resolve) => setTimeout(resolve, ms));

// اسٽريمنگ ڊيٽا رينجرنگ ذريعي واپسي ايسنڪرونس جنريٽر فنڪشن
el.gen = async function* () {
  حاصل "# عنوان\n"؛
  for (let i = 1; i <= 100; ++i) {
    await sleep(400);
    yield "- شيءِ " + i + "\n"؛
  }
};
</script>
```

## ملڪيت

| صفت جو نالو | قسم | بيان ڪرڻ |
| :--- | :--- | :--- |
| `gen` | `Function` | فنڪشن جيڪو واپس ڪري ٿو هڪ هم وقت ساز ائٽيٽر/ جنريٽر، مواد صاف ڪري ٿو ۽ سيٽنگ ڪرڻ کان پوءِ ٻيهر رينڊر ڪري ٿو |

## انداز

| ڪلاس / ٽيگ | بيان ڪرڻ |
| :--- | :--- |
| `Md` | ڪلاس خودڪار طور تي جزو ۾ شامل ڪيو ويو جڏهن چڙهڻ، سيٽ ڪريو مارڪ ڊائون انداز |
| `i.T` | ٽائپنگ اسٽيٽس اشارو خودڪار طريقي سان شامل ڪيو ويو آخر ۾ ڊيٽا انٽري دوران |
