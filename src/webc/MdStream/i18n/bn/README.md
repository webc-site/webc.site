# মার্কডাউন পাঠ্যের স্ট্রিমিং রেন্ডারিং

- ক্রমবর্ধমানভাবে পার্স করুন এবং মার্কডাউন রেন্ডার করুন
- কন্টেন্ট স্ক্রীন ছাড়িয়ে গেলে স্বয়ংক্রিয়ভাবে স্ক্রোল করতে স্ক্রলিং মূল উপাদানের সাথে সহযোগিতা করুন
- ক্লিক করার সময় বা উপরে স্ক্রোল করার সময় অটোস্ক্রোল থামান, নিচে স্ক্রোল করার সময় পুনরায় শুরু করুন
- স্বয়ংক্রিয়ভাবে টাইপিং অবস্থা সূচক প্রদর্শন করুন

## ডেমো ব্যবহার করুন

```html
<c-vs>
  <c-md-stream id="md-view"></c-md-stream>
</c-vs>

<script type="module">
import "webc.site/MdStream.js";

const el = document.getElementById("md-view"),
  sleep = (ms) => new Promise((resolve) => setTimeout(resolve, ms));

// ফিরে আসা অ্যাসিঙ্ক্রোনাস জেনারেটর ফাংশনের মাধ্যমে স্ট্রিমিং ডেটা রেন্ডারিং
el.gen = async function* () {
  ফলন "# শিরোনাম\n";
  for (let i = 1; i <= 100; ++i) {
    await sleep(400);
    ফলন "- আইটেম " + i + "\n";
  }
};
</script>
```

## সম্পত্তি

| বৈশিষ্ট্যের নাম | টাইপ | ব্যাখ্যা করা |
| :--- | :--- | :--- |
| `gen` | `Function` | ফাংশন যা একটি অ্যাসিঙ্ক্রোনাস ইটারেটর/জেনারেটর প্রদান করে, বিষয়বস্তু সাফ করে এবং সেট করার পরে পুনরায় রেন্ডার করে |

## শৈলী

| ক্লাস/ট্যাগ | ব্যাখ্যা করা |
| :--- | :--- |
| `Md` | মাউন্ট করার সময় শ্রেণী স্বয়ংক্রিয়ভাবে উপাদানে যোগ হয়, মার্কডাউন শৈলী সেট করুন |
| `i.T` | ডাটা এন্ট্রির সময় টাইপিং স্ট্যাটাস ইন্ডিকেটর স্বয়ংক্রিয়ভাবে শেষে যোগ করা হয় |
