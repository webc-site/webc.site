# Потокове відтворення тексту Markdown

- Поступовий аналіз і візуалізація Markdown
- Співпрацюйте з батьківським елементом прокручування, щоб автоматично прокручувати, коли вміст виходить за межі екрана
- Призупинити автопрокручування під час натискання або прокручування вгору, відновити під час прокручування вниз
- Автоматично відображати індикатори стану введення

## Використовуйте демо

```html
<c-vs>
  <c-md-stream id="md-view"></c-md-stream>
</c-vs>

<script type="module">
import "webc.site/MdStream.js";

const el = document.getElementById("md-view"),
  sleep = (ms) => new Promise((resolve) => setTimeout(resolve, ms));

// Відтворення потокових даних через повернуту функцію асинхронного генератора
el.gen = async function* () {
  yield "# title\n";
  for (let i = 1; i <= 100; ++i) {
    await sleep(400);
    yield "- елемент " + i + "\n";
  }
};
</script>
```

## власність

| назва атрибута | типу | ілюструвати |
| :--- | :--- | :--- |
| `gen` | `Function` | Функція, яка повертає асинхронний ітератор/генератор, очищає вміст і повторно відображає після налаштування |

## стиль

| Клас/тег | ілюструвати |
| :--- | :--- |
| `Md` | Клас автоматично додається до компонента під час монтування, установіть стиль Markdown |
| `i.T` | Індикатор стану введення автоматично додається в кінці під час введення даних |
