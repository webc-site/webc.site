# ମାର୍କଡାଉନ୍ ପାଠ୍ୟର ଷ୍ଟ୍ରିମିଂ ରେଣ୍ଡରିଂ |

- କ୍ରମାଗତ ଭାବରେ ପାର୍ସଡ୍ ଏବଂ ମାର୍କଡାଉନ୍ ରେଣ୍ଡର୍ କରନ୍ତୁ |
- ବିଷୟବସ୍ତୁ ସ୍କ୍ରିନ ଅତିକ୍ରମ କଲାବେଳେ ସ୍ୱୟଂଚାଳିତ ସ୍କ୍ରୋଲ୍ କରିବାକୁ ସ୍କ୍ରୋଲିଂ ପ୍ୟାରେଣ୍ଟ୍ ଉପାଦାନ ସହିତ ସହଯୋଗ କରନ୍ତୁ |
- କ୍ଲିକ୍ କରିବା କିମ୍ବା ସ୍କ୍ରୋଲ୍ କରିବା ସମୟରେ ଅଟୋସ୍କୋଲ୍କୁ ବିରତି ଦିଅ, ସ୍କ୍ରୋଲ୍ କରିବା ସମୟରେ ପୁନ ume କାର୍ଯ୍ୟ ଆରମ୍ଭ କର |
- ସ୍ୱୟଂଚାଳିତ ଭାବରେ ଟାଇପିଂ ସ୍ଥିତି ସୂଚକ ପ୍ରଦର୍ଶନ କରନ୍ତୁ |

## ଡେମୋ ବ୍ୟବହାର କରନ୍ତୁ |

```html
<c-vs>
  <c-md-stream id="md-view"></c-md-stream>
</c-vs>

<script type="module">
import "webc.site/MdStream.js";

const el = document.getElementById("md-view"),
  sleep = (ms) => new Promise((resolve) => setTimeout(resolve, ms));

// ରିଟର୍ନ ଅସନ୍ତୁଳିତ ଜେନେରେଟର ଫଙ୍କସନ୍ ମାଧ୍ୟମରେ ଷ୍ଟ୍ରିମିଂ ଡାଟା ରେଣ୍ଡରିଂ |
el.gen = async function* () {
  ଅମଳ "# ଆଖ୍ୟା \ n";
  for (let i = 1; i <= 100; ++i) {
    await sleep(400);
    yield "- ତୃତୀୟ " + i + " ଆଇଟମ୍\n";
  }
};
</script>
```

## ସମ୍ପତ୍ତି

| ଗୁଣଧର୍ମ ନାମ | ଟାଇପ୍ କରନ୍ତୁ | ବର୍ଣ୍ଣନା କର |
| :--- | :--- | :--- |
| `gen` | `Function` | କାର୍ଯ୍ୟ ଯାହା ଏକ ଅସନ୍ତୁଳିତ ଇଟରେଟର୍ / ଜେନେରେଟର ରିଟର୍ନ କରେ, ବିଷୟବସ୍ତୁ ସଫା କରେ ଏବଂ ସେଟିଂ ପରେ ପୁନ - ରେଣ୍ଡର୍ କରେ |
