# Markdown පෙළ ප්‍රවාහ විදැහුම්කරණය

- ක්‍රමානුකූලව විග්‍රහ කර Markdown කරන්න
- අන්තර්ගතය තිරය ඉක්මවන විට ස්වයංක්‍රීයව අනුචලනය කිරීමට අනුචලන මාපිය මූලද්‍රව්‍යය සමඟ සහයෝගයෙන් කටයුතු කරන්න
- ක්ලික් කිරීමේදී හෝ ඉහළට අනුචලනය කිරීමේදී ස්වයංක්‍රීය අනුචලනය විරාම කරන්න, පහළට අනුචලනය කිරීමේදී නැවත ආරම්භ කරන්න
- ටයිප් කිරීමේ තත්ව දර්ශක ස්වයංක්‍රීයව සංදර්ශන කරන්න

## Demo භාවිතා කරන්න

```html
<c-vs>
  <c-md-stream id="md-view"></c-md-stream>
</c-vs>

<script type="module">
import "webc.site/MdStream.js";

const el = document.getElementById("md-view"),
  sleep = (ms) => new Promise((resolve) => setTimeout(resolve, ms));

// ආපසු ලබා දුන් අසමමුහුර්ත උත්පාදක ශ්‍රිතය හරහා දත්ත විදැහුම්කරණය ප්‍රවාහ කිරීම
el.gen = async function* () {
  අස්වැන්න "# මාතෘකාව\n";
  for (let i = 1; i <= 100; ++i) {
    await sleep(400);
    අස්වැන්න "- අයිතමය " + i + "\n";
  }
};
</script>
```

## දේපල

| ගුණාංග නාමය | වර්ගය | නිදර්ශනය කරන්න |
| :--- | :--- | :--- |
| `gen` | `Function` | අසමමුහුර්ත පුනරාවර්තකයක්/උත්පාදකයක් ආපසු ලබා දෙන ශ්‍රිතය, අන්තර්ගතය ඉවත් කර සැකසීමෙන් පසු නැවත විදැහුම්කරණය කරයි |
