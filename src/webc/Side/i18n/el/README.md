# Πάνελ συρταριών που γλιστρούν από το πλάι

- Αυτόματη δημιουργία εσωτερικών δομών κύλισης
- Κάντε κλικ στην εξωτερική περιοχή για αυτόματη σύμπτυξη
- Σύρετε με κίνηση προς τα αριστερά και προς τα δεξιά για κλείσιμο
- Πλαϊνή συρόμενη θέση σε αριστερή και δεξιά κατεύθυνση
- Παρέχετε ανοιχτές και κλειστές εκδηλώσεις

## Χρησιμοποιήστε το demo

```html
<button class="open-btn">Ανοιχτό</button>

<c-side class="right">
  <div>Περιεχόμενα συρταριού</div>
</c-side>

<script type="module">
import { open } from "webc.site/Side.js";

const btn = document.querySelector(".open-btn"),
  side = document.querySelector("c-side");

btn.onclick = () => open(side);
</script>
```

## διεπαφή

### μέθοδος

- `open(el)`: Λειτουργία εξαγωγής, ανοιχτό συρτάρι
- `close(el)`: Λειτουργία εξαγωγής, κλείσιμο συρταριού
- `el.open()`: μέθοδος παρουσίας, ανοιχτό συρτάρι
- `el.close()`: μέθοδος εμφάνισης, κλείστε το συρτάρι

### συμβάν

- `open`: Ανάβει όταν είναι ενεργοποιημένο
- `close`: Ανάβει όταν είναι κλειστό

## τάξη στυλ

- `.right`: τοποθέτηση ολίσθησης προς τα δεξιά
- `.top`: Τοποθέτηση ολίσθησης προς τα έξω
- `.bottom`: ολίσθηση εκτός θέσης
- (Εάν δεν έχει οριστεί, η προεπιλογή είναι να σύρετε έξω αριστερά)
- `.open`: ανοιχτή κατάσταση
- `.H`: κρυφή κατάσταση
- `.T`: κατάσταση μετάβασης κινουμένων σχεδίων
