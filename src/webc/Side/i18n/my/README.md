# ဘေးမှလျှောထွက်သောအံဆွဲပြားများ

- အတွင်းပိုင်း လှိမ့်ခြင်း တည်ဆောက်ပုံများကို အလိုအလျောက် ဖန်တီးပါ။
- အလိုအလျောက်ပြိုကျရန် ပြင်ပဧရိယာကို နှိပ်ပါ။
- ပိတ်ရန် ဘယ်ညာ ပွတ်ဆွဲပါ။
- ဘယ်ညာ လမ်းကြောင်းအတိုင်း ဘေးဘက် လျှောချထားခြင်း။
- အဖွင့်အပိတ်ပွဲများကို ပေးဆောင်ပါ။

## ဒီမိုကိုသုံးပါ။

```html
<button class="open-btn">ဖွင့်သည်။</button>

<c-side class="right">
  <div>အံဆွဲ အကြောင်းအရာများ</div>
</c-side>

<script type="module">
import { open } from "webc.site/Side.js";

const btn = document.querySelector(".open-btn"),
  side = document.querySelector("c-side");

btn.onclick = () => open(side);
</script>
```

## ကြားခံ

### နည်းလမ်း

- `open(el)`- လုပ်ဆောင်ချက်ကို ထုတ်ယူပါ၊ အံဆွဲဖွင့်ပါ။
- `close(el)`- လုပ်ဆောင်ချက်ကို ထုတ်ယူပါ၊ အံဆွဲကို ပိတ်ပါ။
- `el.open()`- ဥပမာနည်းလမ်း၊ အံဆွဲဖွင့်ပါ။
- `el.close()`- ဥပမာနည်းလမ်း၊ အံဆွဲကို ပိတ်ပါ။

### အဖြစ်အပျက်

- `open`- ဖွင့်ထားသောအခါတွင် မီးများပွင့်သည်။
- `close`- ပိတ်သောအခါတွင် မီးပျက်သည်။

## စတိုင်အတန်း

- `.right`- ညာဘက်သို့ လျှောထွက်နေရာချထားခြင်း။
- `.top`- နေရာချထားမှုကို ရွှေ့လိုက်ပါ။
- `.bottom`- နေရာချထားခြင်းမှ လျှောကျလိုက်ပါ။
- (သတ်မှတ်မထားပါက၊ ပုံသေသည် ဘယ်ဘက်သို့ လျှောကျရန်ဖြစ်သည်)
- `.open`- အခြေအနေကို ဖွင့်ပါ။
- `.H`- ဝှက်ထားသော အခြေအနေ
- `.T`- ကာတွန်းအကူးအပြောင်း အခြေအနေ
