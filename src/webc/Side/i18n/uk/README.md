# Панелі ящиків, які висуваються збоку

- Автоматично створювати внутрішні структури прокручування
- Натисніть зовнішню область, щоб автоматично згорнути
- Щоб закрити, проведіть ліворуч і праворуч
- Бічне розсувне позиціонування вліво і вправо
- Забезпечити відкриті та закриті заходи

## Використовуйте демо

```html
<button class="open-btn">ВІДЧИНЕНО</button>

<c-side class="right">
  <div>Вміст ящика</div>
</c-side>

<script type="module">
import { open } from "webc.site/Side.js";

const btn = document.querySelector(".open-btn"),
  side = document.querySelector("c-side");

btn.onclick = () => open(side);
</script>
```

## інтерфейс

### метод

- `open(el)`: Функція експорту, відкрити ящик
- `close(el)`: функція експорту, закриття ящика
- `el.open()`: метод екземпляра, відкрити ящик
- `el.close()`: метод екземпляра, закрийте ящик

### подія

- `open`: спрацьовує, коли ввімкнено
- `close`: спрацьовує, коли закрито

## клас стилю

- `.right`: праворуч висунути позиціонування
- `.top`: Розташування ковзання
- `.bottom`: висунути з положення
- (Якщо не встановлено, за замовчуванням ковзає ліворуч)
- `.open`: відкритий статус
- `.H`: прихований стан
- `.T`: стан переходу анімації
