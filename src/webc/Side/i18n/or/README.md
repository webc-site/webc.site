# ଡ୍ରୟର ପ୍ୟାନେଲ ଯାହା ପାର୍ଶ୍ୱରୁ ବାହାରିଥାଏ |

- ସ୍ୱୟଂଚାଳିତ ଭାବରେ ଆଭ୍ୟନ୍ତରୀଣ ସ୍କ୍ରୋଲିଂ ସଂରଚନା ସୃଷ୍ଟି କରନ୍ତୁ |
- ସ୍ୱୟଂଚାଳିତ ଭାବରେ ଭୁଶୁଡ଼ିବା ପାଇଁ ବାହ୍ୟ କ୍ଷେତ୍ର କ୍ଲିକ୍ କରନ୍ତୁ |
- ବନ୍ଦ କରିବାକୁ ବାମ ଏବଂ ଡାହାଣ ଅଙ୍ଗଭଙ୍ଗୀକୁ ସ୍ୱାଇପ୍ କରନ୍ତୁ |
- ବାମ ଏବଂ ଡାହାଣ ଦିଗରେ ସାଇଡ୍ ସ୍ଲାଇଡିଂ ପୋଜିସନ୍ |
- ଖୋଲା ଏବଂ ବନ୍ଦ ଇଭେଣ୍ଟଗୁଡିକ ପ୍ରଦାନ କରନ୍ତୁ |

## ଡେମୋ ବ୍ୟବହାର କରନ୍ତୁ |

```html
<button class="open-btn">ଖୋଲନ୍ତୁ |</button>

<c-side class="right">
  <div>ଡ୍ରୟର ବିଷୟବସ୍ତୁ |</div>
</c-side>

<script type="module">
import { open } from "webc.site/Side.js";

const btn = document.querySelector(".open-btn"),
  side = document.querySelector("c-side");

btn.onclick = () => open(side);
</script>
```

## ଇଣ୍ଟରଫେସ୍

### ପଦ୍ଧତି

- `open(el)`: ରପ୍ତାନି କାର୍ଯ୍ୟ, ଖୋଲା ଡ୍ରୟର |
- `close(el)`: ରପ୍ତାନି କାର୍ଯ୍ୟ, ଡ୍ରୟର ବନ୍ଦ କରନ୍ତୁ |
- `el.open()`: ଉଦାହରଣ ପଦ୍ଧତି, ଖୋଲା ଡ୍ରୟର |
- `el.close()`: ଉଦାହରଣ ପଦ୍ଧତି, ଡ୍ରୟର ବନ୍ଦ କରନ୍ତୁ |

### ଇଭେଣ୍ଟ

- `open`: ଟର୍ନ୍ ଅନ୍ ହୋଇଗଲେ ଅଗ୍ନିକାଣ୍ଡ |
- `close`: ବନ୍ଦ ହେଲେ ଅଗ୍ନିକାଣ୍ଡ |

## ଶ style ଳୀ ଶ୍ରେଣୀ

- `.right`: ଡାହାଣ ସ୍ଲାଇଡ୍ ଆଉଟ୍ ପୋଜିସନ୍ |
- `.top`: ସ୍ଲାଇଡ୍ ଆଉଟ୍ ପୋଜିସନ୍ |
- `.bottom`: ପୋଜିସନ୍ ବାହାରେ ସ୍ଲାଇଡ୍ |
- (ଯଦି ସେଟ୍ ହୋଇନାହିଁ, ଡିଫଲ୍ଟ ହେଉଛି ବାମକୁ ସ୍ଲାଇଡ୍ କରିବା)
- `.open`: ଖୋଲା ସ୍ଥିତି |
- `.H`: ଲୁକ୍କାୟିତ ଅବସ୍ଥା |
- `.T`: ଆନିମେସନ୍ ସ୍ଥାନାନ୍ତରଣ ଅବସ୍ଥା |
