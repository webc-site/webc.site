# ड्रॉवर पॅनेल जे बाजूला सरकतात

- आपोआप अंतर्गत स्क्रोलिंग संरचना तयार करा
- आपोआप संकुचित करण्यासाठी बाह्य क्षेत्रावर क्लिक करा
- बंद करण्यासाठी डावीकडे आणि उजवीकडे जेश्चर स्वाइप करा
- डाव्या आणि उजव्या दिशानिर्देशांमध्ये साइड स्लाइडिंग स्थिती
- खुले आणि बंद कार्यक्रम प्रदान करा

## डेमो वापरा

```html
<button class="open-btn">उघडा</button>

<c-side class="right">
  <div>ड्रॉवर सामग्री</div>
</c-side>

<script type="module">
import { open } from "webc.site/Side.js";

const btn = document.querySelector(".open-btn"),
  side = document.querySelector("c-side");

btn.onclick = () => open(side);
</script>
```

## इंटरफेस

### पद्धत

- `open(el)`: एक्सपोर्ट फंक्शन, ड्रॉवर उघडा
- `close(el)`: एक्सपोर्ट फंक्शन, ड्रॉवर बंद करा
- `el.open()`: उदाहरण पद्धत, ड्रॉवर उघडा
- `el.close()`: उदाहरण पद्धत, ड्रॉवर बंद करा

### कार्यक्रम

- `open`: चालू केल्यावर आग लागते
- `close`: बंद असताना आग लागते

## शैली वर्ग

- `.right`: उजवीकडे स्लाइड आउट पोझिशनिंग
- `.top`: स्लाईड आउट पोझिशनिंग
- `.bottom`: स्थितीनुसार स्लाइड करा
- (सेट न केल्यास, डीफॉल्ट डावीकडे सरकणे आहे)
- `.open`: स्थिती उघडा
- `.H`: लपलेली स्थिती
- `.T`: ॲनिमेशन संक्रमण स्थिती
