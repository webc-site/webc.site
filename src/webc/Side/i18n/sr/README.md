# Панели фиока који се извлаче са стране

- Аутоматски креирајте унутрашње структуре за померање
- Кликните на спољашњу област да бисте се аутоматски скупили
- Превуците налево и надесно да бисте затворили
- Бочно клизно позиционирање у левом и десном смеру
- Обезбедите отворене и затворене догађаје

## Користите демо

```html
<button class="open-btn">Отвори</button>

<c-side class="right">
  <div>Садржај фиоке</div>
</c-side>

<script type="module">
import { open } from "webc.site/Side.js";

const btn = document.querySelector(".open-btn"),
  side = document.querySelector("c-side");

btn.onclick = () => open(side);
</script>
```

## интерфејс

### методом

- <б ид="1">: Функција извоза, отворена фиока
- <б ид="1">: Извоз функција, затвори фиоку
- <б ид="1">: метод инстанце, отвори фиоку
- <б ид="1">: метод инстанце, затворите фиоку

### догађај

- <б ид="1">: Пали се када је укључен
- <б ид="1">: Пали када је затворен

## стилска класа

- <б ид="1">: позиционирање десног извлачења
- <б ид="1">: Извуците позиционирање
- <б ид="1">: извуците из позиционирања
- (Ако није подешено, подразумевано је да се повуче лево)
- <б ид="1">: отворен статус
- <б ид="1">: скривено стање
- <б ид="1">: стање прелаза анимације
