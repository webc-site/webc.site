# دراز پينل جيڪي پاسي کان سلائڊ ڪن ٿا

- پاڻمرادو اندروني اسڪرولنگ ڍانچي ٺاھيو
- ٻاهرئين علائقي کي ڪلڪ ڪريو خودڪار طور تي ختم ڪرڻ لاء
- بند ڪرڻ لاء کاٻي ۽ ساڄي اشارو سوائپ ڪريو
- کاٻي ۽ ساڄي طرفن ۾ سائڊ سلائيڊنگ پوزيشن
- کليل ۽ بند واقعا مهيا ڪريو

## ڊيمو استعمال ڪريو

```html
<button class="open-btn">کليل</button>

<c-side class="right">
  <div>دراز جو مواد</div>
</c-side>

<script type="module">
import { open } from "webc.site/Side.js";

const btn = document.querySelector(".open-btn"),
  side = document.querySelector("c-side");

btn.onclick = () => open(side);
</script>
```

## انٽرفيس

### طريقو

- `open(el)`: ايڪسپورٽ فنڪشن، کليل دراج
- `close(el)`: ايڪسپورٽ فنڪشن، دراز بند ڪريو
- `el.open()`: مثال جو طريقو، کليل دراز
- `el.close()`: مثال جو طريقو، دراز کي بند ڪريو

### واقعو

- `open`: جڏهن چالو ٿئي ته باهه
- `close`: بند ٿيڻ تي باهه لڳندي آهي

## انداز ڪلاس

- `.right`: ساڄي سلائيڊ آئوٽ پوزيشننگ
- `.top`: سلائيڊ آئوٽ پوزيشننگ
- `.bottom`: پوزيشن کان ٻاهر سلائڊ
- (جيڪڏهن مقرر نه ڪيو ويو آهي، ڊفالٽ کاٻي پاسي سلائڊ ڪرڻ آهي)
- `.open`: کليل حيثيت
- `.H`: پوشیدہ حالت
- `.T`: متحرڪ منتقلي واري حالت
