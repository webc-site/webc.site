# Панели за чекмеджета, които се плъзгат отстрани

- Автоматично създаване на вътрешни скролиращи структури
- Щракнете върху външната област, за да се свие автоматично
- Плъзнете наляво и надясно, за да затворите
- Странично плъзгащо се позициониране в лява и дясна посока
- Осигурете отворени и затворени събития

## Използвайте демонстрацията

```html
<button class="open-btn">отворено</button>

<c-side class="right">
  <div>Съдържание на чекмеджето</div>
</c-side>

<script type="module">
import { open } from "webc.site/Side.js";

const btn = document.querySelector(".open-btn"),
  side = document.querySelector("c-side");

btn.onclick = () => open(side);
</script>
```

## интерфейс

### метод

- `open(el)`: Функция за експортиране, отворено чекмедже
- `close(el)`: Функция за експортиране, затваряне на чекмеджето
- `el.open()`: метод на екземпляр, отворено чекмедже
- `el.close()`: метод на екземпляр, затворете чекмеджето

### събитие

- `open`: Задейства се при включване
- `close`: Задейства се при затваряне

## стил клас

- `.right`: дясно плъзгащо се позициониране
- `.top`: Плъзгащо се позициониране
- `.bottom`: плъзнете извън позицията
- (Ако не е зададено, по подразбиране е да се плъзга наляво)
- `.open`: отворен статус
- `.H`: скрито състояние
- `.T`: преходно състояние на анимацията
