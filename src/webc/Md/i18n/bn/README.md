# HTML-এ মার্কডাউন পাঠ্যকে নিরাপদে পার্স করুন এবং রেন্ডার করুন।

- **নিরাপত্তা স্যানিটাইজেশন**: নেটিভ `setHTML` API বা `DOMParser` ক্লিনার ব্যবহার করে XSS আক্রমণ থেকে রক্ষা করুন
- **কন্টেন্ট এক্সট্রাকশন**: ইনিশিয়ালাইজেশনের সময় ডিফল্ট স্লটের টেক্সট কন্টেন্ট স্বয়ংক্রিয়ভাবে পড়ুন এবং রেন্ডার করুন
- **বিল্ট-ইন টাইপসেটিং**: পূর্বনির্ধারিত তালিকা, টেবিল, কোড ব্লক, উদ্ধৃতি এবং সতর্কতা বাক্স শৈলী

## ডেমো ব্যবহার করুন

### স্ট্যাটিক রেন্ডারিং

```html
<c-md>
# শিরোনাম
এটি মার্কডাউন সামগ্রীর একটি অংশ
</c-md>
```

### গতিশীল অ্যাসাইনমেন্ট

```html
<c-md id="md"></c-md>

<script type="module">
import "webc.site/Md.js";
document.querySelector("#md").value = "#নতুন শিরোনাম\nনতুন বিষয়বস্তু";
</script>
```

## ইন্টারফেস পরামিতি

সরাসরি ট্যাগের মধ্যে মার্কডাউন পাঠ্য পাস করুন (ডিফল্ট স্লট হিসাবে), অথবা `value` এর মাধ্যমে গতিশীলভাবে আপডেট করুন৷

## শৈলী বিবরণ

`.Md` স্টাইল ক্লাস স্বয়ংক্রিয়ভাবে যোগ করা হয় যখন উপাদানটি আরম্ভ করা হয়। নিম্নলিখিত মার্কডাউন বর্ধিত সতর্কতা বাক্স লেআউট সমর্থন করে (রেফারেন্স সিনট্যাক্স দ্বারা ট্রিগার করা হয়েছে):

- `[!NOTE]`：প্রম্পট তথ্য
- `[!TIP]`: পরামর্শ/টিপস
- `[!IMPORTANT]`: গুরুত্বপূর্ণ মনোযোগ
- `[!WARNING]`: ঝুঁকির সতর্কতা
- `[!CAUTION]`: অত্যন্ত উচ্চ ঝুঁকির সতর্কতা

## নিরাপত্তা ব্যবস্থা

রেন্ডার করার সময়, নিরাপদে HTML ইনজেক্ট করতে প্রথমে নেটিভ `setHTML` API-কে কল করুন৷ যদি ব্রাউজার এটি সমর্থন না করে, তাহলে এটি স্বয়ংক্রিয়ভাবে `DOMParser` এর উপর ভিত্তি করে একটি পুনরাবৃত্ত ক্লিনারে ডাউনগ্রেড করবে এবং নিম্নলিখিত নিরাপত্তা নীতি বাস্তবায়ন করবে:
- অ-নিরাপদ ট্যাগগুলি সরান (যেমন `script`, `iframe`, ইত্যাদি)
- `on*` ইভেন্টের বৈশিষ্ট্যগুলি সাফ করুন৷
- বিপজ্জনক প্রোটোকল যেমন `javascript:`, `vbscript:` এবং `data:` (`data:image/` ব্যতীত) ফিল্টারিং বৈশিষ্ট্যগুলি ফিল্টার করে XSS আক্রমণের বিরুদ্ধে রক্ষা করুন৷
