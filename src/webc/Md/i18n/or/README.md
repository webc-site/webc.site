# ମାର୍କଡାଉନ୍ ଟେକ୍ସଟ୍ କୁ ସୁରକ୍ଷିତ ଭାବରେ ବିଶ୍ଳେଷଣ କରନ୍ତୁ ଏବଂ HTML ରେ ରେଣ୍ଡର୍ କରନ୍ତୁ |

- ** ସୁରକ୍ଷା ସାନିଟାଇଜେସନ୍ **: ଦେଶୀ `setHTML` API କିମ୍ବା `DOMParser` କ୍ଲିନର୍ ବ୍ୟବହାର କରି XSS ଆକ୍ରମଣରୁ ରକ୍ଷା କରନ୍ତୁ |
- ** ବିଷୟବସ୍ତୁ ନିଷ୍କାସନ **: ପ୍ରାରମ୍ଭିକରଣ ସମୟରେ ଡିଫଲ୍ଟ ସ୍ଲଟ୍ ର ପାଠ୍ୟ ବିଷୟବସ୍ତୁକୁ ସ୍ୱୟଂଚାଳିତ ଭାବରେ ପ read ଏବଂ ଏହାକୁ ଉପସ୍ଥାପନ କର |
- ** ବିଲ୍ଟ-ଇନ୍ ଟାଇପ୍ ସେଟିଂ **: ପ୍ରିସେଟ୍ ତାଲିକା, ଟେବୁଲ୍, କୋଡ୍ ବ୍ଲକ୍, କୋଟ୍ ଏବଂ ଚେତାବନୀ ବାକ୍ସ ଶ yles ଳୀ |

## ଡେମୋ ବ୍ୟବହାର କରନ୍ତୁ |

### ଷ୍ଟାଟିକ୍ ରେଣ୍ଡରିଂ |

```html
<c-md>
# ଆଖ୍ୟା
ଏହା ମାର୍କଡାଉନ୍ ବିଷୟବସ୍ତୁର ଏକ ଖଣ୍ଡ |
</c-md>
```

### ଗତିଶୀଳ କାର୍ଯ୍ୟ

```html
<c-md id="md"></c-md>

<script type="module">
import "webc.site/Md.js";
document.querySelector ("# md")। ମୂଲ୍ୟ = "# ନୂତନ ଆଖ୍ୟା \ n ନୂତନ ବିଷୟବସ୍ତୁ";
</script>
```

## ଇଣ୍ଟରଫେସ୍ ପାରାମିଟରଗୁଡିକ

ମାର୍କଡାଉନ୍ ପାଠ୍ୟରେ ସିଧାସଳଖ ଟ୍ୟାଗ୍ ମଧ୍ୟରେ ପାସ୍ କରନ୍ତୁ (ଡିଫଲ୍ଟ ସ୍ଲଟ୍ ଭାବରେ), କିମ୍ବା `value` ମାଧ୍ୟମରେ ଗତିଶୀଳ ଭାବରେ ଅଦ୍ୟତନ କରନ୍ତୁ |

## ଶ Style ଳୀ ବର୍ଣ୍ଣନା |

ଉପାଦାନ ଆରମ୍ଭ ହେବାପରେ `.Md` ଶ style ଳୀ ଶ୍ରେଣୀ ସ୍ୱୟଂଚାଳିତ ଭାବରେ ଯୋଗ କରାଯାଇଥାଏ | ନିମ୍ନଲିଖିତ ମାର୍କଡାଉନ୍ ବିସ୍ତାରିତ ଚେତାବନୀ ବାକ୍ସ ଲେଆଉଟ୍ (ରେଫରେନ୍ସ ବାକ୍ୟବିନ୍ୟାସ ଦ୍ୱାରା ଟ୍ରିଗର) କୁ ସମର୍ଥନ କରେ:

- `[!NOTE]` ： ତୁରନ୍ତ ସୂଚନା |
- `[!TIP]`: ପରାମର୍ଶ / ଟିପ୍ସ |
- `[!IMPORTANT]`: ଗୁରୁତ୍ୱପୂର୍ଣ୍ଣ ଧ୍ୟାନ |
- `[!WARNING]`: ବିପଦ ଚେତାବନୀ |
- `[!CAUTION]`: ଅତ୍ୟଧିକ ବିପଦ ସଙ୍କେତ |

## ସୁରକ୍ଷା ଯନ୍ତ୍ର

ରେଣ୍ଡରିଂ କରିବାବେଳେ, HTML କୁ ସୁରକ୍ଷିତ ଭାବରେ ଇଞ୍ଜେକ୍ସନ ଦେବା ପାଇଁ ପ୍ରଥମେ ଦେଶୀ `setHTML` API କୁ କଲ୍ କରନ୍ତୁ | ଯଦି ବ୍ରାଉଜର୍ ଏହାକୁ ସମର୍ଥନ କରେ ନାହିଁ, ତେବେ ଏହା ସ୍ୱୟଂଚାଳିତ ଭାବରେ `DOMParser` ଉପରେ ଆଧାର କରି ଏକ ରିସର୍ସିଭ୍ କ୍ଲିନର୍ କୁ ଡାଉନଗ୍ରେଡ୍ ହେବ ଏବଂ ନିମ୍ନଲିଖିତ ସୁରକ୍ଷା ନୀତି କାର୍ଯ୍ୟକାରୀ କରିବ:
- ଅଣ-ସୁରକ୍ଷିତ ଟ୍ୟାଗଗୁଡିକ ଅପସାରଣ କରନ୍ତୁ (ଯଥା `script`, `iframe`, ଇତ୍ୟାଦି)
- `on*` ଇଭେଣ୍ଟ ଗୁଣଗୁଡିକ ସଫା କରନ୍ତୁ |
- ବିପଜ୍ଜନକ ପ୍ରୋଟୋକଲରୁ ଆରମ୍ଭ ହୋଇଥିବା ଗୁଣଗୁଡିକ ଫିଲ୍ଟର କରି XSS ଆକ୍ରମଣରୁ ରକ୍ଷା କରନ୍ତୁ ଯେପରିକି `javascript:`, `vbscript:`, ଏବଂ `data:` (`data:image/` ବ୍ୟତୀତ) |
