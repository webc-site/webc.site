- бһҠбһҖбһ’бһ¶бһҸбһ»бһ…бҹҒбһүбһ–бһё DOM бһҠбҹ„бһҷбһҹбҹ’бһңбҹҗбһҷбһ”бҹ’бһҡбһңбһҸбҹ’бһҸбһ·бһ“бҹ…бһ–бҹҒбһӣбһ”бһ·бһ‘
- бһ‘бһ”бҹӢбһҹбҹ’бһҖбһ¶бһҸбҹӢвҖӢбһҘбһҡбһ·бһҷбһ¶бһ”бһ‘вҖӢбһӣбһ»бһ”бһ…бҹ„бһӣвҖӢбһҹбҹ„ Esc

## бһ”бҹ’бһҡбһҫбһҖбһ¶бһҡбһ”бһ„бҹ’бһ бһ¶бһү

```html
<button>бһ…бһ»бһ…бһҠбһҫбһҳбҹ’бһ”бһёбһӣбҹҒбһ…бһЎбһҫбһ„</button>

<script type="module">
import Box from "webc.site/Box.js";

document.querySelector("button").onclick = () => {
  const dialog = Box(),
    content = document.createElement("div"),
    title = document.createElement("h3"),
    description = document.createElement("p"),
    button = document.createElement("button");

  content.className = "Lg";
  title.textContent = "бһ…бҹҶбһҺбһ„бһҮбһҫбһ„";
  description.textContent = "бһўбһҸбҹ’бһҗбһ”бһ‘";
  button.className = "Btn Main";
  button.textContent = "бһҷбһӣбҹӢбһ–бҹ’бһҡбһҳ";
  button.onclick = () => dialog.close();
  content.append(title, description, button);
  dialog.append(content);
};
</script>
```

### бһҳбһ»бһҒбһ„бһ¶бһҡбһ“бһ¶бҹҶбһ…бҹҒбһүбһӣбҹҶбһ“бһ¶бҹҶбһҠбһҫбһҳ

`Box()`

- **бһҸбһҳбҹ’бһӣбҹғбһҸбҹ’бһҡбһЎбһ”бҹӢ**бҹ– `HTMLDialogElement` бһ’бһ¶бһҸбһ»бһ”бҹ’бһҡбһўбһ”бҹӢбһ“бҹғ `Box`бһҗбҹ’бһ“бһ¶бһҖбҹӢбһҡбһ…бһ“бһ¶бһ”бҹҗбһ‘бҹ’бһҳбһҸбҹ’бһҡбһјбһңбһ”бһ¶бһ“бһ”бһ“бҹ’бһҗбҹӮбһҳбҹ”

## бһҗбҹ’бһ“бһ¶бһҖбҹӢбһҡбһ…бһ“бһ¶бһ”бҹҗбһ‘бҹ’бһҳ

### `.Box`

бһўбһ“бһ»бһңбһҸбҹ’бһҸвҖӢбһ‘бҹ…вҖӢбһ’бһ¶бһҸбһ» `dialog` бһҠбһҫбһҳбҹ’бһ”бһёвҖӢбһҖбҹҶбһҺбһҸбҹӢвҖӢбһ‘бһёбһҸбһ¶бҹҶбһ„ бһҖбһ¶бһҡвҖӢбһҠбһ¶бһҖбҹӢвҖӢбһҖбһҺбҹ’бһҸбһ¶бһӣ бһ“бһ·бһ„вҖӢбһҡбһ”бһ¶бҹҶбһ„вҖӢбһ•бҹ’бһ‘бҹғбһҒбһ¶бһ„бһҖбҹ’бһҡбҹ„бһҷвҖӢбһ–бҹ’бһҡбһ·бһӣбҹ”

### `.Lg`

бһҗбҹ’бһ“бһ¶бһҖбҹӢбһҡбһ…бһ“бһ¶бһ”бҹҗбһ‘бҹ’бһҳбһ”бһ¶бһ“бһўбһ“бһ»бһңбһҸбҹ’бһҸбһ…бҹҶбһ–бҹ„бҹҮбһҒбҹ’бһӣбһ№бһҳбһҹбһ¶бһҡбһҒбһ¶бһ„бһҖбҹ’бһ“бһ»бһ„ бһҖбһ¶бһҡбһҖбҹҶбһҺбһҸбҹӢбһ–бһҺбҹҢбһ•бҹ’бһ‘бҹғбһҒбһ¶бһ„бһҖбҹ’бһҡбҹ„бһҷ бһҮбҹ’бһҡбһ»бһ„бһҳбһјбһӣ бһҹбҹ’бһҡбһҳбҹ„бһӣ бһ“бһ·бһ„бһ”бҹ’бһӣбһ„бҹӢбһ”бһүбҹ’бһҲбһҡ бһ“бһ·бһ„бһҖбһ¶бһҡбһҖбҹҶбһҺбһҸбҹӢбһ”бҹ’бһӣбһ„бҹӢбһҒбһ¶бһ„бһҖбҹ’бһ“бһ»бһ„бһ“бҹғ `h3` бһ“бһ·бһ„ `p`бҹ”
