# ड्रॉप-डाउन चयन, चौड़ाई वर्तमान आइटम के अनुकूल होती है

मूल `select` की चौड़ाई डिफ़ॉल्ट रूप से सबसे लंबे विकल्प टेक्स्ट द्वारा निर्धारित की जाती है। छोटे विकल्पों पर स्विच करते समय, बहुत अधिक खाली स्थान होता है, जो उपस्थिति को प्रभावित करता है।

यह घटक पारदर्शी रूप से `b` टैग के ऊपर मूल `select` को कवर करता है, जिससे चौड़ाई वर्तमान में चयनित पाठ की लंबाई के लिए सटीक रूप से अनुकूलित हो जाती है।

- सभी टर्मिनलों पर मूल `select` ड्रॉपडाउन मेनू शैलियाँ और पहुंच-योग्यता समर्थन बनाए रखें
- संपूर्ण कीबोर्ड टैब नेविगेशन और दिशा कुंजी स्विचिंग जैसे मूल इंटरैक्टिव संचालन का समर्थन करता है।
- अतिरिक्त सफेद स्थान को खत्म करने के लिए चौड़ाई को वर्तमान विकल्प के अनुसार अनुकूलित करें

## डेमो का प्रयोग करें

```html
<c-select>
  <option value="js">JavaScript</option>
  <option value="rs">Rust</option>
  <option value="py">Python</option>
</c-select>
```
