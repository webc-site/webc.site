# ការជ្រើសរើសទម្លាក់ចុះ ទទឹងសម្របទៅនឹងធាតុបច្ចុប្បន្ន

ទទឹងរបស់ដើម `select`ត្រូវបានកំណត់ដោយអត្ថបទជម្រើសវែងបំផុតតាមលំនាំដើម។ នៅពេលប្តូរជម្រើសខ្លីៗ មានចន្លោះពណ៌សច្រើនពេក ដែលប៉ះពាល់ដល់រូបរាង។

សមាសភាគនេះគ្របដណ្តប់ដោយតម្លាភាពនៃប្រភពដើម `select`ខាងលើស្លាក `b` ដែលអនុញ្ញាតឱ្យទទឹងសម្របខ្លួនឱ្យបានត្រឹមត្រូវទៅនឹងប្រវែងនៃអត្ថបទដែលបានជ្រើសរើសបច្ចុប្បន្ន។

- រក្សារចនាប័ទ្មម៉ឺនុយទម្លាក់ចុះដើម `select` និងការគាំទ្រភាពងាយស្រួលនៅលើស្ថានីយទាំងអស់។
- គាំទ្រប្រតិបត្តិការអន្តរកម្មដើម ដូចជាការរុករកផ្ទាំងក្តារចុចពេញលេញ និងការប្តូរគ្រាប់ចុចទិសដៅ។
- សម្រួល​ទទឹង​ទៅ​ជម្រើស​បច្ចុប្បន្ន​ដើម្បី​លុប​បំបាត់​ទំហំ​ស​លើស

## ប្រើការបង្ហាញ

```html
<c-select>
  <option value="js">JavaScript</option>
  <option value="rs">Rust</option>
  <option value="py">Python</option>
</c-select>
```
