# Избор от падащо меню, ширината се адаптира към текущия елемент

Ширината на естествения `select` се определя от най-дългия текст на опцията по подразбиране. При превключване на къси опции има твърде много бяло пространство, което се отразява на външния вид.

Този компонент покрива прозрачно естествения `select` над тага `b`, позволявайки ширината да се адаптира точно към дължината на избрания в момента текст.

- Поддържайте естествени `select` стилове на падащо меню и поддръжка за достъпност на всички терминали
- Поддържа собствени интерактивни операции, като цялостна навигация с раздели на клавиатурата и превключване на клавишите за посока.
- Адаптирайте ширината към текущата опция, за да премахнете излишното бяло пространство

## Използвайте демонстрацията

```html
<c-select>
  <option value="js">JavaScript</option>
  <option value="rs">Rust</option>
  <option value="py">Python</option>
</c-select>
```
