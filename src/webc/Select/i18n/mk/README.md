# Паѓачки избор, ширината се прилагодува на моменталната ставка

Ширината на мајчин `select` стандардно се определува со текстот на најдолгата опција. При префрлување на кратки опции, има премногу бел простор, што влијае на изгледот.

Оваа компонента транспарентно го покрива мајчиниот `select` над ознаката `b`, овозможувајќи ширината прецизно да се прилагоди на должината на моментално избраниот текст.

- Одржувајте стилови на паѓачкото мени `select` и поддршка за пристапност на сите терминали
- Поддржува мајчин интерактивни операции како што се целосна навигација со тастатура со јазичиња и префрлување на копчињата за насока.
- Прилагодете ја ширината на моменталната опција за да го елиминирате вишокот бел простор

## Користете го демо

```html
<c-select>
  <option value="js">JavaScript</option>
  <option value="rs">Rust</option>
  <option value="py">Python</option>
</c-select>
```
