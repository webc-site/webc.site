# Вибір у розкривному списку, ширина адаптується до поточного елемента

Ширина рідного `select` визначається за замовчуванням найдовшим текстом опції. Під час перемикання коротких варіантів залишається забагато білого простору, що позначається на зовнішньому вигляді.

Цей компонент прозоро покриває власний `select` над тегом `b`, дозволяючи ширині точно адаптуватися до довжини вибраного тексту.

- Зберігайте рідні `select` стилі спадного меню та підтримку спеціальних можливостей на всіх терміналах
- Підтримує власні інтерактивні операції, такі як повна навігація вкладками на клавіатурі та перемикання клавіш напрямку.
- Адаптуйте ширину до поточного параметра, щоб усунути зайвий простір

## Використовуйте демо

```html
<c-select>
  <option value="js">JavaScript</option>
  <option value="rs">Rust</option>
  <option value="py">Python</option>
</c-select>
```
