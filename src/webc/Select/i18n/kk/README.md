# Ашылмалы таңдау, ені ағымдағы элементке бейімделеді

Негізгі `select` ені әдепкі бойынша ең ұзын опция мәтінімен анықталады. Қысқа опцияларды ауыстырған кезде тым көп бос орын пайда болады, бұл сыртқы көрініске әсер етеді.

Бұл құрамдас `b` тегінің үстіндегі негізгі `select` ені мөлдір түрде жабады, бұл еннің ағымдағы таңдалған мәтіннің ұзындығына дәл бейімделуіне мүмкіндік береді.

- Түпнұсқа `select` ашылмалы мәзір мәнерлерін және барлық терминалдарда арнайы мүмкіндіктерді қолдау
- Толық пернетақта қойындысын шарлау және бағыт пернелерін ауыстыру сияқты жергілікті интерактивті операцияларды қолдайды.
- Артық бос орынды жою үшін енді ағымдағы опцияға бейімдеңіз

## Демонстрацияны пайдаланыңыз

```html
<c-select>
  <option value="js">JavaScript</option>
  <option value="rs">Rust</option>
  <option value="py">Python</option>
</c-select>
```
