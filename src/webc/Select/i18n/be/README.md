# Выбар у выпадальным меню, шырыня адаптуецца да бягучага элемента

Па змаўчанні шырыня ўласнага `select` вызначаецца самым доўгім тэкстам опцыі. Пры пераключэнні кароткіх варыянтаў занадта шмат прабелаў, што ўплывае на знешні выгляд.

Гэты кампанент празрыста пакрывае родны `select` над тэгам `b`, дазваляючы шырыні дакладна адаптавацца да даўжыні вылучанага тэксту.

- Падтрымлівайце ўласныя стылі выпадальнага меню `select` і падтрымку спецыяльных магчымасцей на ўсіх тэрміналах
- Падтрымлівае ўласныя інтэрактыўныя аперацыі, такія як поўная навігацыя па ўкладках клавіятуры і пераключэнне кнопак кірунку.
- Адаптуйце шырыню да бягучай опцыі, каб пазбавіцца ад лішняга прабелу

## Выкарыстоўвайце дэма

```html
<c-select>
  <option value="js">JavaScript</option>
  <option value="rs">Rust</option>
  <option value="py">Python</option>
</c-select>
```
