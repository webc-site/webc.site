# ડ્રોપ-ડાઉન પસંદગી, પહોળાઈ વર્તમાન આઇટમને અનુકૂળ છે

મૂળ `select`ની પહોળાઈ ડિફૉલ્ટ રૂપે સૌથી લાંબા વિકલ્પ ટેક્સ્ટ દ્વારા નક્કી કરવામાં આવે છે. ટૂંકા વિકલ્પોને સ્વિચ કરતી વખતે, ત્યાં ઘણી બધી સફેદ જગ્યા છે, જે દેખાવને અસર કરે છે.

આ ઘટક `b` ટૅગની ઉપરના મૂળ `select`ને પારદર્શક રીતે આવરી લે છે, જે પહોળાઈને હાલમાં પસંદ કરેલ ટેક્સ્ટની લંબાઈને ચોક્કસ રીતે અનુકૂલિત કરવાની મંજૂરી આપે છે.

- તમામ ટર્મિનલ્સ પર મૂળ `select` ડ્રોપડાઉન મેનૂ શૈલીઓ અને ઍક્સેસિબિલિટી સપોર્ટ જાળવો
- સંપૂર્ણ કીબોર્ડ ટૅબ નેવિગેશન અને દિશા કી સ્વિચિંગ જેવા મૂળ ઇન્ટરેક્ટિવ ઑપરેશનને સપોર્ટ કરે છે.
- વધારાની સફેદ જગ્યાને દૂર કરવા માટે વર્તમાન વિકલ્પ સાથે પહોળાઈને અનુકૂલિત કરો

## ડેમોનો ઉપયોગ કરો

```html
<c-select>
  <option value="js">JavaScript</option>
  <option value="rs">Rust</option>
  <option value="py">Python</option>
</c-select>
```
