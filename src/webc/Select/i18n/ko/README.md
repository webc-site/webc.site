# 드롭다운 선택, 너비가 현재 항목에 맞춰짐

기본 `select`의 너비는 기본적으로 가장 긴 옵션 텍스트에 따라 결정됩니다. 짧은 옵션을 전환할 때 공백이 너무 많아 모양에 영향을 줍니다.

이 구성 요소는 `b` 태그 위의 기본 `select`을 투명하게 덮어 현재 선택한 텍스트의 길이에 맞게 너비를 정확하게 조정할 수 있습니다.

- 모든 터미널에서 기본 `select` 드롭다운 메뉴 스타일 및 접근성 지원을 유지합니다.
- 완전한 키보드 탭 탐색 및 방향 키 전환과 같은 기본 대화형 작업을 지원합니다.
- 과도한 공백을 제거하려면 너비를 현재 옵션에 맞게 조정하세요.

## 데모 사용

```html
<c-select>
  <option value="js">JavaScript</option>
  <option value="rs">Rust</option>
  <option value="py">Python</option>
</c-select>
```
