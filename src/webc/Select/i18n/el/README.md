# Αναπτυσσόμενη επιλογή, το πλάτος προσαρμόζεται στο τρέχον στοιχείο

Το πλάτος του εγγενούς `select` καθορίζεται από το μεγαλύτερο κείμενο επιλογής από προεπιλογή. Κατά την εναλλαγή σύντομων επιλογών, υπάρχει πολύς λευκός χώρος, ο οποίος επηρεάζει την εμφάνιση.

Αυτό το στοιχείο καλύπτει με διαφάνεια την εγγενή `select` πάνω από την ετικέτα `b`, επιτρέποντας στο πλάτος να προσαρμοστεί με ακρίβεια στο μήκος του τρέχοντος επιλεγμένου κειμένου.

- Διατηρήστε τα εγγενή στυλ του αναπτυσσόμενου μενού `select` και την υποστήριξη προσβασιμότητας σε όλα τα τερματικά
- Υποστηρίζει εγγενείς διαδραστικές λειτουργίες, όπως πλήρη πλοήγηση καρτελών πληκτρολογίου και εναλλαγή πλήκτρων κατεύθυνσης.
- Προσαρμόστε το πλάτος στην τρέχουσα επιλογή για να εξαλείψετε τον υπερβολικό χώρο

## Χρησιμοποιήστε το demo

```html
<c-select>
  <option value="js">JavaScript</option>
  <option value="rs">Rust</option>
  <option value="py">Python</option>
</c-select>
```
