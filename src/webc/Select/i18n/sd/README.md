# ڊراپ-ڊائون چونڊ، ويڪر موجوده شيءِ سان ٺهڪي اچي ٿي

اصلي `select` جي ويڪر ڊفالٽ جي لحاظ کان ڊگھي آپشن جي متن سان طئي ڪئي ويندي آهي. جڏهن مختصر اختيارن کي تبديل ڪندي، اتي تمام گهڻو اڇو جاء آهي، جيڪو ظاهري کي متاثر ڪري ٿو.

هي جزو شفاف طور تي مقامي `select` کي ڍڪيندو آهي `b` ٽيگ جي مٿان، ويڪر کي هن وقت چونڊيل متن جي ڊيگهه کي درست ڪرڻ جي اجازت ڏئي ٿو.

- مقامي `select` برقرار رکون مينيو اسٽائلز ۽ سڀني ٽرمينلز تي رسائي جي مدد
- ڏيهي انٽرايڪٽو عملن کي سپورٽ ڪري ٿو جهڙوڪ مڪمل ڪيبورڊ ٽيب نيويگيشن ۽ هدايت ڪيئي سوئچنگ.
- ويڪر کي موجوده اختيار کي اپنائڻ لاء اضافي اڇو جاء کي ختم ڪرڻ لاء

## ڊيمو استعمال ڪريو

```html
<c-select>
  <option value="js">JavaScript</option>
  <option value="rs">Rust</option>
  <option value="py">Python</option>
</c-select>
```
