# கீழ்தோன்றும் தேர்வு, அகலம் தற்போதைய உருப்படிக்கு ஏற்றது

நேட்டிவ் `select`ன் அகலம் இயல்பாகவே நீளமான விருப்ப உரையால் தீர்மானிக்கப்படுகிறது. குறுகிய விருப்பங்களை மாற்றும் போது, ​​அதிகப்படியான வெள்ளை இடம் உள்ளது, இது தோற்றத்தை பாதிக்கிறது.

இந்தக் கூறு `b` குறிச்சொல்லுக்கு மேலே உள்ள சொந்த `select`ஐ வெளிப்படையாக உள்ளடக்கியது, இதன் அகலம் தற்போது தேர்ந்தெடுக்கப்பட்ட உரையின் நீளத்திற்குத் துல்லியமாக மாற்றியமைக்க அனுமதிக்கிறது.

- அனைத்து டெர்மினல்களிலும் சொந்த `select` கீழ்தோன்றும் மெனு பாணிகளையும் அணுகல்தன்மை ஆதரவையும் பராமரிக்கவும்
- முழுமையான விசைப்பலகை தாவல் வழிசெலுத்தல் மற்றும் திசை விசை மாறுதல் போன்ற சொந்த ஊடாடும் செயல்பாடுகளை ஆதரிக்கிறது.
- அதிகப்படியான வெள்ளை இடத்தை அகற்ற தற்போதைய விருப்பத்திற்கு அகலத்தை மாற்றவும்

## டெமோவைப் பயன்படுத்தவும்

```html
<c-select>
  <option value="js">JavaScript</option>
  <option value="rs">Rust</option>
  <option value="py">Python</option>
</c-select>
```
