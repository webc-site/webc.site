# පතන තේරීම, පළල වත්මන් අයිතමයට අනුගත වේ

ස්වදේශීය `select` හි පළල පෙරනිමියෙන් දිගම විකල්ප පෙළ මගින් තීරණය වේ. කෙටි විකල්ප මාරු කරන විට, පෙනුමට බලපාන සුදු පැහැති ඉඩ වැඩියි.

මෙම සංරචකය `b` ටැගයට ඉහලින් ස්වදේශීය `select` විනිවිද පෙනෙන ලෙස ආවරණය කරයි, පළල දැනට තෝරාගෙන ඇති පෙළෙහි දිගට නිවැරදිව අනුවර්තනය වීමට ඉඩ සලසයි.

- සියලුම පර්යන්තවල ස්වදේශීය `select` පතන මෙනු මෝස්තර සහ ප්‍රවේශ්‍යතා සහාය පවත්වා ගන්න
- සම්පූර්ණ යතුරුපුවරු ටැබ් සංචාලනය සහ දිශා යතුරු මාරු කිරීම වැනි ස්වදේශීය අන්තර්ක්‍රියාකාරී මෙහෙයුම් සඳහා සහය දක්වයි.
- අතිරික්ත සුදු අවකාශය ඉවත් කිරීම සඳහා වත්මන් විකල්පයට පළල අනුගත කරන්න

## Demo භාවිතා කරන්න

```html
<c-select>
  <option value="js">JavaScript</option>
  <option value="rs">Rust</option>
  <option value="py">Python</option>
</c-select>
```
