# Падајући избор, ширина се прилагођава тренутној ставци

Ширина изворног <б ид="1"> је подразумевано одређена најдужим текстом опције. Када мењате кратке опције, има превише белог простора, што утиче на изглед.

- Одржавајте изворне <б ид="1"> стилове падајућег менија и подршку за приступачност на свим терминалима
