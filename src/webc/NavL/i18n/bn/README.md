# স্বয়ংক্রিয়ভাবে নেভিগেশন মেনু এবং রিটার্ন বোতাম পরিবর্তন করুন

- নেভিগেশন মেনু এবং রিটার্ন বোতাম গঠন তৈরি করতে স্বয়ংক্রিয়ভাবে শিশু উপাদানগুলিকে মোড়ানো
- স্বয়ংক্রিয়ভাবে রাউটিং ইতিহাস ট্রেস করুন এবং পূর্ববর্তী রিটার্ন পাথ গণনা করুন
- যখন রুটটি মেনু আইটেমের সাথে মেলে তখন নেভিগেশন মেনু প্রদর্শন করুন, অন্যথায় পিছনের বোতামটি প্রদর্শন করুন৷
