# Newid y ddewislen llywio a'r botwm dychwelyd yn awtomatig

- Lapiwch elfennau plentyn yn awtomatig i gynhyrchu dewislen llywio a strwythurau botwm dychwelyd
- Olrhain hanes llwybro yn awtomatig a chyfrifo'r llwybr dychwelyd blaenorol
- Dangoswch y ddewislen llywio pan fydd y llwybr yn cyfateb i'r eitem ddewislen, fel arall dangoswch y botwm yn ôl

## Defnyddiwch y demo

```html
</c-nav-l>
```

## dosbarth arddull

- `B`: yn actifadu cyflwr y botwm dychwelyd. Dangoswch y botwm yn ôl a chuddio'r ddewislen llywio. Wedi'i newid yn awtomatig o fewn y gydran yn seiliedig ar lwybro.
