# નેવિગેશન મેનૂ અને રીટર્ન બટનને આપમેળે સ્વિચ કરો

- નેવિગેશન મેનૂ અને રીટર્ન બટન સ્ટ્રક્ચર્સ જનરેટ કરવા માટે ચાઈલ્ડ એલિમેન્ટ્સને આપમેળે લપેટો
- આપમેળે રૂટીંગ ઇતિહાસને ટ્રેસ કરો અને પાછલા રીટર્ન પાથની ગણતરી કરો
- જ્યારે માર્ગ મેનુ આઇટમ સાથે મેળ ખાતો હોય ત્યારે નેવિગેશન મેનૂ પ્રદર્શિત કરો, અન્યથા પાછળનું બટન પ્રદર્શિત કરો
