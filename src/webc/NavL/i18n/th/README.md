# สลับเมนูนำทางและปุ่มย้อนกลับโดยอัตโนมัติ

- ล้อมองค์ประกอบลูกโดยอัตโนมัติเพื่อสร้างเมนูนำทางและโครงสร้างปุ่มย้อนกลับ
- ติดตามประวัติเส้นทางโดยอัตโนมัติและคำนวณเส้นทางย้อนกลับก่อนหน้า
- แสดงเมนูการนำทางเมื่อเส้นทางตรงกับรายการเมนู หรือแสดงปุ่มย้อนกลับ
