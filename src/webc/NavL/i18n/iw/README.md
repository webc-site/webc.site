# החלף אוטומטית את תפריט הניווט וכפתור החזרה

- עטוף אוטומטית רכיבי צאצא כדי ליצור תפריט ניווט ולהחזיר מבני לחצנים
- עקוב אוטומטית אחר היסטוריית הניתוב וחשב את נתיב ההחזרה הקודם
- הצג את תפריט הניווט כאשר המסלול תואם לפריט התפריט, אחרת הצג את לחצן החזרה

## שיעור סגנון

- `B`: מפעיל את מצב לחצן החזרה. הצג את לחצן 'הקודם' והסתיר את תפריט הניווט. החלפה אוטומטית בתוך הרכיב בהתבסס על ניתוב.
