# Tukar menu navigasi dan butang kembali secara automatik

- Balut elemen kanak-kanak secara automatik untuk menjana menu navigasi dan struktur butang kembali
- Jejak sejarah penghalaan secara automatik dan hitung laluan pulangan sebelumnya
- Paparkan menu navigasi apabila laluan sepadan dengan item menu, jika tidak paparkan butang belakang

## Gunakan demo

```html
<c-nav-l>
  <a href="/">muka depan</a>
  <a href="/doc">dokumen</a>
</c-nav-l>
```

## kelas gaya

- `B`: mengaktifkan keadaan butang kembali. Tunjukkan butang belakang dan sembunyikan menu navigasi. Ditukar secara automatik dalam komponen berdasarkan penghalaan.
