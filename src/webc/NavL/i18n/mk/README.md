# Автоматско менување на менито за навигација и копче за враќање

- Автоматски завиткајте ги детските елементи за да генерирате структури на менито за навигација и копчињата за враќање
- Автоматски следете ја историјата на рутирање и пресметајте ја претходната патека за враќање
- Прикажете го менито за навигација кога маршрутата се совпаѓа со ставката од менито, инаку прикажете го копчето за назад

## Користете го демо

```html
<c-nav-l>
  <a href="/">насловна страница</a>
  <a href="/doc">документ</a>
</c-nav-l>
```

## класа стил

- `B`: ја активира состојбата на копчето за враќање. Покажете го копчето за назад и скријте го менито за навигација. Автоматски се префрлува во рамките на компонентата врз основа на рутирање.
