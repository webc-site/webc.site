# Автоматично превключване на навигационното меню и бутона за връщане

- Автоматично обгръщане на дъщерни елементи, за да генерира навигационно меню и структури на бутони за връщане
- Автоматично проследяване на историята на маршрутите и изчисляване на предишния обратен път
- Показване на навигационното меню, когато маршрутът съответства на елемента от менюто, в противен случай показване на бутона за връщане назад

## Използвайте демонстрацията

```html
<c-nav-l>
  <a href="/">първа страница</a>
  <a href="/doc">документ</a>
</c-nav-l>
```

## стил клас

- `B`: активира състоянието на бутона за връщане. Показване на бутона за връщане назад и скриване на менюто за навигация. Автоматично превключване в рамките на компонента въз основа на маршрутизиране.
