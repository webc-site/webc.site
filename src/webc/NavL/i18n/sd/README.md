# خودڪار طور تي نيويگيشن مينيو ۽ موٽڻ واري بٽڻ کي تبديل ڪريو

- نيويگيشن مينيو ۽ موٽڻ واري بٽڻ جي جوڙجڪ کي پيدا ڪرڻ لاء ٻار جي عناصر کي خودڪار طور تي لپايو
- خودڪار طريقي سان رستي جي تاريخ کي ڇڪيو ۽ پوئين موٽڻ واري رستي جو حساب ڪريو
- نيويگيشن مينيو ڏيکاريو جڏھن رستو مينيو آئٽم سان ملندو آھي، ٻي صورت ۾ پوئتي بٽڻ ڏيکاريو
