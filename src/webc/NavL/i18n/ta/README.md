# வழிசெலுத்தல் மெனு மற்றும் திரும்பும் பொத்தானை தானாக மாற்றவும்

- வழிசெலுத்தல் மெனுவை உருவாக்க மற்றும் பொத்தான் கட்டமைப்புகளைத் தானாக உருவாக்க, குழந்தை உறுப்புகளை தானாக மடிக்கவும்
- ரூட்டிங் வரலாற்றைத் தானாகக் கண்டுபிடித்து முந்தைய திரும்பும் பாதையைக் கணக்கிடுங்கள்
- வழி மெனு உருப்படியுடன் பொருந்தும்போது வழிசெலுத்தல் மெனுவைக் காண்பிக்கவும், இல்லையெனில் பின் பொத்தானைக் காட்டவும்
