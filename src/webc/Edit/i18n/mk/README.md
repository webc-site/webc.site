# копче за уредување

- Содржи копче за икона за уредување со прав агол
- Промов за префрлање на глувчето за уредување, повеќејазична поддршка

## Користете го демо

```html
<c-edit href="https://github.com"></c-edit>
```

## Параметри на интерфејсот

Етикетата на компонентата е `<c-edit>`, а адресата за скокање може да се уредува преку конфигурацијата на атрибутот `href`.
