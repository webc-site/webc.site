# przycisk edycji

- Zawiera przycisk ikony edycji pod kątem prostym
- Po najechaniu myszką monit o edycję, obsługa wielu języków

## Skorzystaj z wersji demonstracyjnej

```html
<c-edit href="https://github.com"></c-edit>
```

## Parametry interfejsu

Etykieta komponentu to `<c-edit>`, a adres skoku można edytować poprzez konfigurację atrybutu `href`.
