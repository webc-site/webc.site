# tlačítko upravit

- Obsahuje tlačítko ikony úpravy pravého úhlu
- Výzva pro úpravy po přejetí myší, podpora více jazyků

## Použijte demo

```html
<c-edit href="https://github.com"></c-edit>
```

## Parametry rozhraní

Označení komponenty je `<c-edit>` a adresu skoku lze upravit pomocí konfigurace atributu `href`.
