# botwm golygu

- Yn cynnwys botwm eicon golygu ongl sgwâr
- Anogwr llygoden ar gyfer golygu, cefnogaeth aml-iaith

## Defnyddiwch y demo

```html
<c-edit href="https://github.com"></c-edit>
```

## Paramedrau rhyngwyneb

Label y gydran yw `<c-edit>`, a gellir golygu'r cyfeiriad naid trwy'r ffurfweddiad priodoledd `href`.
