# pulsante modifica

- Contiene il pulsante dell'icona di modifica dell'angolo retto
- Richiesta di modifica al passaggio del mouse, supporto multilingue

## Utilizza la demo

```html
<c-edit href="https://github.com"></c-edit>
```

## Parametri dell'interfaccia

L'etichetta del componente è `<c-edit>` e l'indirizzo di salto può essere modificato tramite la configurazione dell'attributo `href`.
