# nút chỉnh sửa

- Chứa nút biểu tượng chỉnh sửa góc phải
- Nhắc chuột để chỉnh sửa, hỗ trợ đa ngôn ngữ

## Sử dụng bản trình diễn

```html
<c-edit href="https://github.com"></c-edit>
```

## Thông số giao diện

Nhãn thành phần là `<c-edit>` và địa chỉ bước nhảy có thể được chỉnh sửa thông qua cấu hình thuộc tính `href`.
