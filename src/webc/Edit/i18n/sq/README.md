# butoni i redaktimit

- Përmban butonin e ikonës së modifikimit të këndit të drejtë
- Kërkesa për kthimin e miut për redaktim, mbështetje shumë-gjuhëshe

## Përdorni demonstrimin

```html
<c-edit href="https://github.com"></c-edit>
```

## Parametrat e ndërfaqes

Etiketa e komponentit është `<c-edit>` dhe adresa e kërcimit mund të modifikohet nëpërmjet konfigurimit të atributit `href`.
