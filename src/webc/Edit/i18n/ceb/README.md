# buton sa pag-edit

- Naglangkob sa right angle edit icon button
- Mouse-over prompt para sa pag-edit, suporta sa multi-language

## Gamita ang demo

```html
<c-edit href="https://github.com"></c-edit>
```

## Mga parametro sa interface

Ang component label mao ang `<c-edit>`, ug ang jump address mahimong i-edit pinaagi sa `href` attribute configuration.
