# knop Bewerken

- Bevat een knop voor het bewerken van een rechte hoek
- Mouse-over-prompt voor bewerking, meertalige ondersteuning

## Gebruik de demo

```html
<c-edit href="https://github.com"></c-edit>
```

## Interfaceparameters

Het componentlabel is `<c-edit>` en het sprongadres kan worden bewerkt via de attribuutconfiguratie `href`.
