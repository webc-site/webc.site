# κουμπί επεξεργασίας

- Περιέχει το κουμπί εικονιδίου επεξεργασίας δεξιάς γωνίας
- Προτροπή για επεξεργασία του ποντικιού, υποστήριξη πολλών γλωσσών

## Χρησιμοποιήστε το demo

```html
<c-edit href="https://github.com"></c-edit>
```

## Παράμετροι διεπαφής

Η ετικέτα στοιχείου είναι `<c-edit>` και η διεύθυνση μετάβασης μπορεί να επεξεργαστεί μέσω της διαμόρφωσης χαρακτηριστικών `href`.
