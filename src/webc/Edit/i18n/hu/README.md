# szerkesztés gomb

- Derékszögű szerkesztési ikont tartalmaz
- Az egérmutató szerkesztéséhez, többnyelvű támogatás

## Használd a demót

```html
<c-edit href="https://github.com"></c-edit>
```

## Interfész paraméterei

Az összetevő címke `<c-edit>`, és az ugrás címe az `href` attribútumkonfiguráción keresztül szerkeszthető.
