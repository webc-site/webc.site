# кнопка редактирования

- Содержит кнопку со значком редактирования под прямым углом
- Подсказка при наведении курсора мыши для редактирования, многоязычная поддержка.

## Используйте демо-версию

```html
<c-edit href="https://github.com"></c-edit>
```

## Параметры интерфейса

Метка компонента — `<c-edit>`, а адрес перехода можно редактировать с помощью конфигурации атрибута `href`.
