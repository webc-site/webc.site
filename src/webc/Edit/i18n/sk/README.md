# tlačidlo upraviť

- Obsahuje tlačidlo ikony úpravy pravého uhla
- Výzva po prejdení myšou na úpravu, podpora viacerých jazykov

## Použite demo

```html
<c-edit href="https://github.com"></c-edit>
```

## Parametre rozhrania

Označenie komponentu je `<c-edit>` a adresu skoku je možné upraviť pomocou konfigurácie atribútu `href`.
