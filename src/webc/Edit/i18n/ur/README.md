# ترمیم کے بٹن

- رائٹ اینگل ایڈٹ آئیکن بٹن پر مشتمل ہے۔
- ترمیم کے لیے ماؤس اوور پرامپٹ، کثیر زبان کی مدد

## ڈیمو استعمال کریں۔

```html
<c-edit href="https://github.com"></c-edit>
```

## انٹرفیس کے پیرامیٹرز

اجزاء کا لیبل `<c-edit>` ہے، اور جمپ ایڈریس کو `href` انتساب کنفیگریشن کے ذریعے ایڈٹ کیا جا سکتا ہے۔
