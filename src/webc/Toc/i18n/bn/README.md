# ডিরেক্টরি নেভিগেশন উপাদান

- উল্লম্ব ফ্লেক্স লেআউট
- ডিরেক্টরি ডেটা পাস করার জন্য অভিধান সমর্থন করুন
- মার্কডাউন পাঠ্য স্বয়ংক্রিয়ভাবে লোড করতে ক্লিক করুন
- স্থিতি প্রদর্শন লোড হচ্ছে অ্যানিমেশন

## ডেমো ব্যবহার করুন

```html
<c-toc></c-toc>

<script type="module">
import "webc.site/Toc.js";

const toc = document.querySelector("c-toc");
toc.mdUrl = (key, lang) => "/doc/" + lang + "/" + key + ".md";
toc.li = {
  "README": "WebC.site",
  "doc/use": "উপাদান ব্যবহার করুন"
};
</script>
```

## ইন্টারফেস

### বৈশিষ্ট্য এবং পদ্ধতি

- `toc.li`: একটি ডিরেক্টরি বস্তু বা এন্ট্রিগুলির তালিকা সেট করে, যেমন `{ key: title }`
- `toc.mdUrl`: অনুরোধ URL জেনারেশন ফাংশন `(key, lang) => url` বা প্রিফিক্স স্ট্রিং সেট করুন
- `toc.target`: টার্গেট কন্টেন্ট নোড সেট বা অ্যাসোসিয়েট করুন (যেমন `c-md`), স্বয়ংক্রিয়ভাবে `ing` অপেক্ষা এবং কন্টেন্ট ফিলিং পরিচালনা করুন
- `toc.load(key)`: সংশ্লিষ্ট কীটির লোডিং ম্যানুয়ালি ট্রিগার করুন
