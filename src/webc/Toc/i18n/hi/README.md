# निर्देशिका नेविगेशन घटक

- लंबवत फ्लेक्स लेआउट
- निर्देशिका डेटा को पास करने के लिए शब्दकोश का समर्थन करें
- मार्कडाउन टेक्स्ट को स्वचालित रूप से लोड करने के लिए क्लिक करें
- लोड हो रहा है स्थिति प्रदर्शन आईएनजी एनीमेशन

## डेमो का प्रयोग करें

```html
<c-toc></c-toc>

<script type="module">
import "webc.site/Toc.js";

const toc = document.querySelector("c-toc");
toc.mdUrl = (key, lang) => "/doc/" + lang + "/" + key + ".md";
toc.li = {
  "README": "WebC.site",
  "doc/use": "घटकों का उपयोग करें"
};
</script>
```

## इंटरफ़ेस

### गुण और विधियाँ

- `toc.li`: एक निर्देशिका ऑब्जेक्ट या प्रविष्टियों की सूची सेट करता है, जैसे `{ key: title }`
- `toc.mdUrl`: अनुरोध URL जनरेशन फ़ंक्शन `(key, lang) => url` या उपसर्ग स्ट्रिंग सेट करें
- `toc.target`: लक्ष्य सामग्री नोड को सेट या संबद्ध करें (जैसे `c-md`), स्वचालित रूप से `ing` प्रतीक्षा और सामग्री भरने को संभालें
- `toc.load(key)`: संबंधित कुंजी की लोडिंग को मैन्युअल रूप से ट्रिगर करें
