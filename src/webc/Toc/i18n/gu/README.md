# ડિરેક્ટરી નેવિગેશન ઘટક

- વર્ટિકલ ફ્લેક્સ લેઆઉટ
- ડિરેક્ટરી ડેટા પસાર કરવા માટે સપોર્ટ ડિક્શનરી
- માર્કડાઉન ટેક્સ્ટને આપમેળે લોડ કરવા માટે ક્લિક કરો
- સ્થિતિ પ્રદર્શન એનિમેશન લોડ કરી રહ્યું છે

## ડેમોનો ઉપયોગ કરો

```html
<c-toc></c-toc>

<script type="module">
import "webc.site/Toc.js";

const toc = document.querySelector("c-toc");
toc.mdUrl = (key, lang) => "/doc/" + lang + "/" + key + ".md";
toc.li = {
  "README": "WebC.site",
  "doc/use": "ઘટકોનો ઉપયોગ કરો"
};
</script>
```

## ઈન્ટરફેસ

### ગુણધર્મો અને પદ્ધતિઓ

- `toc.li`: ડિરેક્ટરી ઑબ્જેક્ટ અથવા એન્ટ્રીઓની સૂચિ સેટ કરે છે, જેમ કે `{ key: title }`
- `toc.mdUrl`: વિનંતી URL જનરેશન ફંક્શન `(key, lang) => url` અથવા ઉપસર્ગ સ્ટ્રિંગ સેટ કરો
- `toc.target`: લક્ષ્ય સામગ્રી નોડ (જેમ કે `c-md`) સેટ કરો અથવા સાંકળો, આપોઆપ `ing` પ્રતીક્ષા અને સામગ્રી ભરવાનું સંચાલન કરો
- `toc.load(key)`: સંબંધિત કીના લોડિંગને મેન્યુઅલી ટ્રિગર કરો
