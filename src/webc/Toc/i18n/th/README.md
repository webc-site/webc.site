# ส่วนประกอบการนำทางไดเรกทอรี

- เค้าโครง Flex แนวตั้ง
- รองรับพจนานุกรมเพื่อส่งผ่านข้อมูลไดเร็กทอรี
- คลิกเพื่อโหลดข้อความ Markdown โดยอัตโนมัติ
- กำลังโหลดการแสดงสถานะ Ining แอนิเมชั่น

## ใช้การสาธิต

```html
<c-toc></c-toc>

<script type="module">
import "webc.site/Toc.js";

const toc = document.querySelector("c-toc");
toc.mdUrl = (key, lang) => "/doc/" + lang + "/" + key + ".md";
toc.li = {
  "README": "WebC.site",
  "doc/use": "ใช้ส่วนประกอบ"
};
</script>
```

## อินเตอร์เฟซ

### คุณสมบัติและวิธีการ

- `toc.li`: ตั้งค่าวัตถุไดเรกทอรีหรือรายการรายการ เช่น `{ key: title }`
- `toc.mdUrl`: ตั้งค่าฟังก์ชันการสร้าง URL คำขอ `(key, lang) => url` หรือสตริงคำนำหน้า
- `toc.target`: ตั้งค่าหรือเชื่อมโยงโหนดเนื้อหาเป้าหมาย (เช่น `c-md`) จัดการ `ing` การรอและการเติมเนื้อหาโดยอัตโนมัติ
- `toc.load(key)`: ทริกเกอร์การโหลดคีย์ที่เกี่ยวข้องด้วยตนเอง
