# Գրացուցակի նավիգացիոն բաղադրիչ

- Ուղղահայաց ճկուն դասավորություն
- Աջակցեք բառարանին գրացուցակում տվյալների փոխանցման համար
- Սեղմեք Markdown տեքստն ավտոմատ բեռնելու համար
- Կարգավիճակի ցուցադրման բեռնում Անիմացիա

## Օգտագործեք ցուցադրությունը

```html
<c-toc></c-toc>

<script type="module">
import "webc.site/Toc.js";

const toc = document.querySelector("c-toc");
toc.mdUrl = (key, lang) => "/doc/" + lang + "/" + key + ".md";
toc.li = {
  "README": "WebC.site",
  "doc/use": "Օգտագործել բաղադրիչներ"
};
</script>
```

## ինտերֆեյս

### Հատկություններ և մեթոդներ

- `toc.li`. սահմանում է գրացուցակի օբյեկտ կամ գրառումների ցանկ, օրինակ՝ `{ key: title }`
- `toc.mdUrl`. Սահմանեք հարցումների ստեղծման գործառույթը `(key, lang) => url` կամ նախածանցի տողը
- `toc.target`. սահմանեք կամ կապակցեք թիրախային բովանդակության հանգույցը (օրինակ՝ `c-md`), ավտոմատ կերպով կարգավորեք `ing` սպասումը և բովանդակության լրացումը
- `toc.load(key)`. ձեռքով գործարկել համապատասխան բանալի բեռնումը
