# ডাইৰেকটৰি নেভিগেচন উপাদান

- উলম্ব ফ্লেক্স বিন্যাস
- ডাইৰেকটৰি তথ্যত পাছ কৰিবলে অভিধান সমৰ্থন কৰে
- স্বয়ংক্ৰিয়ভাৱে মাৰ্কডাউন লিখনী লোড কৰিবলৈ ক্লিক কৰক
- অৱস্থা প্ৰদৰ্শন Ing এনিমেচন লোড কৰা হৈছে

## ডেমো ব্যৱহাৰ কৰক

```html
<c-toc></c-toc>

<script type="module">
import "webc.site/Toc.js";

const toc = document.querySelector("c-toc");
toc.mdUrl = (key, lang) => "/doc/" + lang + "/" + key + ".md";
toc.li = {
  "README": "WebC.site",
  "doc/use": "উপাদানসমূহ ব্যৱহাৰ কৰক"
};
</script>
```

## আন্তঃপৃষ্ঠ

### ধৰ্ম আৰু পদ্ধতি

- `toc.li`: এটা পঞ্জিকা বস্তু বা প্ৰৱেশৰ তালিকা সংহতি কৰে, যেনে `{ key: title }`
- `toc.mdUrl`: অনুৰোধ URL সৃষ্টি ফলন `(key, lang) => url` বা উপসৰ্গ ষ্ট্ৰিং নিৰ্ধাৰণ কৰক
- `toc.target`: লক্ষ্য বিষয়বস্তু ন'ড নিৰ্ধাৰণ বা সংযুক্ত কৰক (যেনে `c-md`), স্বয়ংক্ৰিয়ভাৱে `ing` অপেক্ষা আৰু বিষয়বস্তু পূৰণ নিয়ন্ত্ৰণ কৰক
- `toc.load(key)`: সংশ্লিষ্ট কি'ৰ লোডিং হস্তচালিতভাৱে ট্ৰিগাৰ কৰক
