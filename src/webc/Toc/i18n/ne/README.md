# निर्देशिका नेभिगेसन कम्पोनेन्ट

- ठाडो फ्लेक्स लेआउट
- निर्देशिका डाटा पास गर्न शब्दकोश समर्थन गर्नुहोस्
- मार्कडाउन पाठ स्वतः लोड गर्न क्लिक गर्नुहोस्
- स्थिति प्रदर्शन इंग एनिमेसन लोड गर्दै

## डेमो प्रयोग गर्नुहोस्

```html
<c-toc></c-toc>

<script type="module">
import "webc.site/Toc.js";

const toc = document.querySelector("c-toc");
toc.mdUrl = (key, lang) => "/doc/" + lang + "/" + key + ".md";
toc.li = {
  "README": "WebC.site",
  "doc/use": "घटकहरू प्रयोग गर्नुहोस्"
};
</script>
```

## इन्टरफेस

### गुण र विधिहरू

- `toc.li`: डाइरेक्टरी वस्तु वा प्रविष्टिहरूको सूची सेट गर्दछ, जस्तै `{ key: title }`
- `toc.mdUrl`: अनुरोध URL उत्पादन प्रकार्य सेट गर्नुहोस् `(key, lang) => url` वा उपसर्ग स्ट्रिङ
- `toc.target`: लक्षित सामग्री नोड (जस्तै `c-md`) सेट वा सम्बद्ध गर्नुहोस्, स्वचालित रूपमा `ing` प्रतीक्षा र सामग्री भर्ने काम
- `toc.load(key)`: म्यानुअल रूपमा सम्बन्धित कुञ्जीको लोडिङ ट्रिगर गर्नुहोस्
