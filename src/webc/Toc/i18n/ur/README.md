# ڈائرکٹری نیویگیشن جزو

- عمودی فلیکس لے آؤٹ
- ڈائرکٹری ڈیٹا میں پاس کرنے کے لیے لغت کو سپورٹ کریں۔
- مارک ڈاؤن ٹیکسٹ کو خودکار طور پر لوڈ کرنے کے لیے کلک کریں۔
- لوڈ ہو رہا ہے اسٹیٹس ڈسپلے انیمیشن

## ڈیمو استعمال کریں۔

```html
<c-toc></c-toc>

<script type="module">
import "webc.site/Toc.js";

const toc = document.querySelector("c-toc");
toc.mdUrl = (key, lang) => "/doc/" + lang + "/" + key + ".md";
toc.li = {
  "README": "WebC.site",
  "doc/use": "اجزاء استعمال کریں"
};
</script>
```

## انٹرفیس

### خصوصیات اور طریقے

- `toc.li`: ڈائریکٹری آبجیکٹ یا اندراجات کی فہرست سیٹ کرتا ہے، جیسے `{ key: title }`
- `toc.mdUrl`: درخواست URL جنریشن فنکشن `(key, lang) => url` یا سابقہ ​​سٹرنگ سیٹ کریں۔
- `toc.target`: ٹارگٹ مواد نوڈ (جیسے `c-md`) کو سیٹ یا اس سے منسلک کریں، خود بخود `ing` انتظار اور مواد بھرنے کو ہینڈل کریں
- `toc.load(key)`: متعلقہ کلید کی لوڈنگ کو دستی طور پر متحرک کریں۔
