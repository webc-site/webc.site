# សមាសធាតុរុករកថត

- ប្លង់ Flex បញ្ឈរ
- គាំទ្រវចនានុក្រមដើម្បីឆ្លងកាត់ទិន្នន័យថត
- ចុចដើម្បីផ្ទុកអត្ថបទ Markdown ដោយស្វ័យប្រវត្តិ
- កំពុងផ្ទុកការបង្ហាញស្ថានភាព Ing ចលនា

## ប្រើការបង្ហាញ

```html
<c-toc></c-toc>

<script type="module">
import "webc.site/Toc.js";

const toc = document.querySelector("c-toc");
toc.mdUrl = (key, lang) => "/doc/" + lang + "/" + key + ".md";
toc.li = {
  "README": "WebC.site",
  "doc/use": "ប្រើសមាសធាតុ"
};
</script>
```

## ចំណុចប្រទាក់

### លក្ខណៈសម្បត្តិនិងវិធីសាស្រ្ត

- `toc.li`៖ កំណត់​វត្ថុ​ថត ឬ​បញ្ជី​ធាតុ ដូចជា `{ key: title }`
- `toc.mdUrl`៖ កំណត់មុខងារបង្កើត URL សំណើ `(key, lang) => url` ឬខ្សែអក្សរបុព្វបទ
- `toc.target`៖ កំណត់ ឬភ្ជាប់ថ្នាំងមាតិកាគោលដៅ (ដូចជា `c-md`) គ្រប់គ្រងដោយស្វ័យប្រវត្តិ `ing` ការរង់ចាំ និងការបំពេញមាតិកា
- `toc.load(key)`៖ កេះដោយដៃនៃការផ្ទុកសោដែលត្រូវគ្នា។
