# డైరెక్టరీ నావిగేషన్ భాగం

- నిలువు ఫ్లెక్స్ లేఅవుట్
- డైరెక్టరీ డేటాలో పాస్ చేయడానికి డిక్షనరీకి మద్దతు ఇవ్వండి
- మార్క్‌డౌన్ వచనాన్ని స్వయంచాలకంగా లోడ్ చేయడానికి క్లిక్ చేయండి
- స్థితి ప్రదర్శన Ing యానిమేషన్ లోడ్ అవుతోంది

## డెమో ఉపయోగించండి

```html
<c-toc></c-toc>

<script type="module">
import "webc.site/Toc.js";

const toc = document.querySelector("c-toc");
toc.mdUrl = (key, lang) => "/doc/" + lang + "/" + key + ".md";
toc.li = {
  "README": "WebC.site",
  "doc/use": "భాగాలను ఉపయోగించండి"
};
</script>
```

## ఇంటర్ఫేస్

### లక్షణాలు మరియు పద్ధతులు

- `toc.li`: `{ key: title }` వంటి డైరెక్టరీ ఆబ్జెక్ట్ లేదా ఎంట్రీల జాబితాను సెట్ చేస్తుంది
- `toc.mdUrl`: అభ్యర్థన URL ఉత్పత్తి ఫంక్షన్‌ను సెట్ చేయండి `(key, lang) => url` లేదా ప్రిఫిక్స్ స్ట్రింగ్
- `toc.target`: లక్ష్య కంటెంట్ నోడ్‌ను సెట్ చేయండి లేదా అనుబంధించండి (`c-md` వంటివి), స్వయంచాలకంగా `ing` నిరీక్షణ మరియు కంటెంట్ నింపడం
- `toc.load(key)`: సంబంధిత కీ యొక్క లోడ్‌ను మాన్యువల్‌గా ట్రిగ్గర్ చేయండి
