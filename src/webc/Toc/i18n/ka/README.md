# დირექტორია ნავიგაციის კომპონენტი

- ვერტიკალური მოქნილი განლაგება
- ლექსიკონის მხარდაჭერა დირექტორიაში მონაცემების გადასაცემად
- დააწკაპუნეთ Markdown ტექსტის ავტომატურად ჩასატვირთად
- იტვირთება სტატუსის ჩვენება ანიმაციის ჩატვირთვა

## გამოიყენეთ დემო ვერსია

```html
<c-toc></c-toc>

<script type="module">
import "webc.site/Toc.js";

const toc = document.querySelector("c-toc");
toc.mdUrl = (key, lang) => "/doc/" + lang + "/" + key + ".md";
toc.li = {
  "README": "WebC.site",
  "doc/use": "გამოიყენე კომპონენტები"
};
</script>
```

## ინტერფეისი

### თვისებები და მეთოდები

- `toc.li`: აყენებს დირექტორიას ობიექტს ან ჩანაწერების სიას, როგორიცაა `{ key: title }`
- `toc.mdUrl`: დააყენეთ მოთხოვნის URL გენერირების ფუნქცია `(key, lang) => url` ან პრეფიქსის სტრიქონი
- `toc.target`: დააყენეთ ან დააკავშირეთ სამიზნე შინაარსის კვანძი (როგორიცაა `c-md`), ავტომატურად ამუშავებს `ing` ლოდინს და შინაარსის შევსებას
- `toc.load(key)`: ხელით გააქტიურეთ შესაბამისი გასაღების ჩატვირთვა
