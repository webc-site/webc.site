# Компонент навигации по каталогу

- Вертикальный гибкий макет
- Поддержка словаря для передачи данных каталога
- Нажмите, чтобы автоматически загрузить текст Markdown
- Отображение статуса загрузки с анимацией

## Используйте демо-версию

```html
<c-toc></c-toc>

<script type="module">
import "webc.site/Toc.js";

const toc = document.querySelector("c-toc");
toc.mdUrl = (key, lang) => "/doc/" + lang + "/" + key + ".md";
toc.li = {
  "README": "WebC.site",
  "doc/use": "Использовать компоненты"
};
</script>
```

## интерфейс

### Свойства и методы

- `toc.li`: устанавливает объект каталога или список записей, например `{ key: title }`
- `toc.mdUrl`: установите функцию генерации URL-адреса запроса `(key, lang) => url` или строку префикса.
- `toc.target`: установите или свяжите целевой узел контента (например, `c-md`), автоматически обрабатывайте ожидание `ing` и заполнение контента.
- `toc.load(key)`: вручную запустить загрузку соответствующего ключа.
