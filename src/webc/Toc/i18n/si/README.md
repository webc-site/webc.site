# නාමාවලි සංචාලන සංරචකය

- සිරස් Flex පිරිසැලසුම
- ඩිරෙක්ටරි දත්ත හරහා යාමට ශබ්ද කෝෂයට සහය වන්න
- මාර්ක්ඩවුන් පෙළ ස්වයංක්‍රීයව පූරණය කිරීමට ක්ලික් කරන්න
- තත්ව සංදර්ශකය Ing සජීවිකරණය පූරණය කරමින්

## Demo භාවිතා කරන්න

```html
<c-toc></c-toc>

<script type="module">
import "webc.site/Toc.js";

const toc = document.querySelector("c-toc");
toc.mdUrl = (key, lang) => "/doc/" + lang + "/" + key + ".md";
toc.li = {
  "README": "WebC.site",
  "ලේඛනය/භාවිතය": "සංරචක භාවිතා කරන්න"
};
</script>
```

## අතුරු මුහුණත

### ගුණාංග සහ ක්රම

- `toc.li`: `{ key: title }` වැනි ඩිරෙක්ටරි වස්තුවක් හෝ ඇතුළත් කිරීම් ලැයිස්තුවක් සකසයි
- `toc.mdUrl`: ඉල්ලීම් URL උත්පාදන ශ්‍රිතය සකසන්න `(key, lang) => url` හෝ උපසර්ග පෙළ
- `toc.target`: ඉලක්ක අන්තර්ගත නෝඩය (`c-md` වැනි) සකසන්න හෝ සම්බන්ධ කරන්න, `ing` බලා සිටීම සහ අන්තර්ගත පිරවීම ස්වයංක්‍රීයව හසුරුවයි
- `toc.load(key)`: අදාල යතුර පූරණය කිරීම අතින් ක්‍රියාරම්භ කරන්න
