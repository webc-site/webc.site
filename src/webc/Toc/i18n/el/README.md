# Στοιχείο πλοήγησης καταλόγου

- Κάθετη ευέλικτη διάταξη
- Υποστήριξη λεξικού για διαβίβαση δεδομένων καταλόγου
- Κάντε κλικ για αυτόματη φόρτωση του κειμένου Markdown
- Εμφάνιση κατάστασης φόρτωσης Ενεργοποίηση κινούμενων εικόνων

## Χρησιμοποιήστε το demo

```html
<c-toc></c-toc>

<script type="module">
import "webc.site/Toc.js";

const toc = document.querySelector("c-toc");
toc.mdUrl = (key, lang) => "/doc/" + lang + "/" + key + ".md";
toc.li = {
  "README": "WebC.site",
  "doc/use": "Χρήση εξαρτημάτων"
};
</script>
```

## διεπαφή

### Ιδιότητες και μέθοδοι

- `toc.li`: Ορίζει ένα αντικείμενο καταλόγου ή λίστα καταχωρήσεων, όπως `{ key: title }`
- `toc.mdUrl`: Ορισμός συνάρτησης δημιουργίας URL αιτήματος `(key, lang) => url` ή συμβολοσειράς προθέματος
- `toc.target`: Ορίστε ή συσχετίστε τον κόμβο περιεχομένου-στόχου (όπως `c-md`), χειριστείτε αυτόματα `ing` αναμονή και πλήρωση περιεχομένου
- `toc.load(key)`: Μη αυτόματη ενεργοποίηση της φόρτωσης του αντίστοιχου κλειδιού
