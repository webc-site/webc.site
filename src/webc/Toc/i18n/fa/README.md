# مؤلفه ناوبری دایرکتوری

- طرح بندی فلکس عمودی
- پشتیبانی از فرهنگ لغت برای ارسال در داده های دایرکتوری
- برای بارگیری خودکار متن Markdown کلیک کنید
- در حال بارگیری نمایش وضعیت پویانمایی

## از دمو استفاده کنید

```html
<c-toc></c-toc>

<script type="module">
import "webc.site/Toc.js";

const toc = document.querySelector("c-toc");
toc.mdUrl = (key, lang) => "/doc/" + lang + "/" + key + ".md";
toc.li = {
  "README": "WebC.site",
  "doc/use": "استفاده از اجزا"
};
</script>
```

## رابط کاربری

### خواص و روش ها

- `toc.li`: یک شی دایرکتوری یا لیستی از ورودی ها را تنظیم می کند، مانند `{ key: title }`
- `toc.mdUrl`: تابع تولید URL درخواست `(key, lang) => url` یا رشته پیشوند را تنظیم کنید
- `toc.target`: گره محتوای هدف را تنظیم یا مرتبط کنید (مانند `c-md`)، به طور خودکار `ing` انتظار و پر کردن محتوا را مدیریت کند
- `toc.load(key)`: بارگیری کلید مربوطه را به صورت دستی فعال کنید
