# ଡିରେକ୍ଟୋରୀ ନାଭିଗେସନ୍ ଉପାଦାନ |

- ଭର୍ଟିକାଲ୍ ଫ୍ଲେକ୍ସ ଲେଆଉଟ୍ |
- ଡିରେକ୍ଟୋରୀ ତଥ୍ୟରେ ପାସ୍ କରିବାକୁ ଅଭିଧାନକୁ ସମର୍ଥନ କରନ୍ତୁ |
- ମାର୍କଡାଉନ୍ ଟେକ୍ସଟ୍ ସ୍ୱୟଂଚାଳିତ ଭାବରେ ଲୋଡ୍ କରିବାକୁ କ୍ଲିକ୍ କରନ୍ତୁ |
- ଲୋଡ୍ ଷ୍ଟାଟସ୍ ଡିସପ୍ଲେ ଇନ୍ ଆନିମେସନ୍ |

## ଡେମୋ ବ୍ୟବହାର କରନ୍ତୁ |

```html
<c-toc></c-toc>

<script type="module">
import "webc.site/Toc.js";

const toc = document.querySelector("c-toc");
toc.mdUrl = (key, lang) => "/doc/" + lang + "/" + key + ".md";
toc.li = {
  "README": "WebC.site",
  "ଡକ / ବ୍ୟବହାର": "ଉପାଦାନଗୁଡିକ ବ୍ୟବହାର କରନ୍ତୁ"
};
</script>
```

## ଇଣ୍ଟରଫେସ୍

### ଗୁଣ ଏବଂ ପଦ୍ଧତି |

- `toc.li`: ଏକ ଡିରେକ୍ଟୋରୀ ବସ୍ତୁ କିମ୍ବା ଏଣ୍ଟ୍ରିଗୁଡ଼ିକର ତାଲିକା ସେଟ୍ କରେ, ଯେପରିକି `{ key: title }`
- `toc.mdUrl`: ଅନୁରୋଧ URL ଉତ୍ପାଦନ କାର୍ଯ୍ୟ `(key, lang) => url` କିମ୍ବା ଉପସର୍ଗ ଷ୍ଟ୍ରିଙ୍ଗ ସେଟ୍ କରନ୍ତୁ |
- `toc.target`: ଲକ୍ଷ୍ୟ ବିଷୟବସ୍ତୁ ନୋଡ୍ ସେଟ୍ କରନ୍ତୁ କିମ୍ବା ସଂଯୁକ୍ତ କରନ୍ତୁ (ଯେପରିକି `c-md`), ସ୍ୱୟଂଚାଳିତ ଭାବରେ `ing` ଅପେକ୍ଷା ଏବଂ ବିଷୟବସ୍ତୁ ଭରିବା |
- `toc.load(key)`: ସଂପୃକ୍ତ ଚାବିର ଲୋଡିଂକୁ ହସ୍ତକୃତ ଭାବରେ ଟ୍ରିଗର କରନ୍ତୁ |
