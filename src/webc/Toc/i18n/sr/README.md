# Компонента за навигацију директоријума

- Вертикални Флек Лаиоут
- Речник подршке за прослеђивање података директоријума
- Кликните да бисте аутоматски учитали Маркдовн текст
- Приказ статуса учитавања Инг анимација

## Користите демо

```html
<c-toc></c-toc>

<script type="module">
import "webc.site/Toc.js";

const toc = document.querySelector("c-toc");
toc.mdUrl = (key, lang) => "/doc/" + lang + "/" + key + ".md";
toc.li = {
  "README": "WebC.site",
  "доц/усе": "Користите компоненте"
};
</script>
```
