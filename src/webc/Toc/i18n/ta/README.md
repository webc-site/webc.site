# அடைவு வழிசெலுத்தல் கூறு

- செங்குத்து நெகிழ்வு தளவமைப்பு
- கோப்பகத் தரவை அனுப்ப அகராதியை ஆதரிக்கவும்
- மார்க் டவுன் உரையை தானாக ஏற்ற கிளிக் செய்யவும்
- நிலைக் காட்சி இங் அனிமேஷனை ஏற்றுகிறது

## டெமோவைப் பயன்படுத்தவும்

```html
<c-toc></c-toc>

<script type="module">
import "webc.site/Toc.js";

const toc = document.querySelector("c-toc");
toc.mdUrl = (key, lang) => "/doc/" + lang + "/" + key + ".md";
toc.li = {
  "README": "WebC.site",
  "doc/use": "உபயோக கூறுகள்"
};
</script>
```

## இடைமுகம்

### பண்புகள் மற்றும் முறைகள்

- `toc.li`: `{ key: title }` போன்ற கோப்பகப் பொருள் அல்லது உள்ளீடுகளின் பட்டியலை அமைக்கிறது
- `toc.mdUrl`: கோரிக்கை URL உருவாக்க செயல்பாட்டை அமைக்கவும் `(key, lang) => url` அல்லது முன்னொட்டு சரம்
- `toc.target`: இலக்கு உள்ளடக்க முனையை அமைக்கவும் அல்லது இணைக்கவும் (`c-md` போன்றவை), தானாகவே `ing` காத்திருப்பு மற்றும் உள்ளடக்க நிரப்புதலைக் கையாளும்
- `toc.load(key)`: தொடர்புடைய விசையை ஏற்றுவதை கைமுறையாகத் தூண்டவும்
