# Компонент за навигация в директория

- Вертикално гъвкаво оформление
- Поддръжка на речник за предаване на данни от директорията
- Щракнете, за да заредите автоматично Markdown текст
- Показване на състоянието на зареждане Ing анимация

## Използвайте демонстрацията

```html
<c-toc></c-toc>

<script type="module">
import "webc.site/Toc.js";

const toc = document.querySelector("c-toc");
toc.mdUrl = (key, lang) => "/doc/" + lang + "/" + key + ".md";
toc.li = {
  "README": "WebC.site",
  "doc/use": "Използване на компоненти"
};
</script>
```

## интерфейс

### Свойства и методи

- `toc.li`: Задава обект на директория или списък със записи, като например `{ key: title }`
- `toc.mdUrl`: Задайте функция за генериране на URL адрес на заявка `(key, lang) => url` или префиксен низ
- `toc.target`: Задайте или асоциирайте целевия възел на съдържанието (като `c-md`), автоматично обработвайте `ing` изчакване и попълване на съдържание
- `toc.load(key)`: Ръчно задействане на зареждането на съответния ключ
