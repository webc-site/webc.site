# Каталогты шарлау құрамдас бөлігі

- Тік икемді орналасу
- Каталог деректеріне өту үшін сөздікті қолдау
- Markdown мәтінін автоматты түрде жүктеу үшін басыңыз
- Жүктелу күйінің дисплейі анимацияда

## Демонстрацияны пайдаланыңыз

```html
<c-toc></c-toc>

<script type="module">
import "webc.site/Toc.js";

const toc = document.querySelector("c-toc");
toc.mdUrl = (key, lang) => "/doc/" + lang + "/" + key + ".md";
toc.li = {
  "README": "WebC.site",
  "doc/use": "Компоненттерді пайдалану"
};
</script>
```

## интерфейс

### Қасиеттер мен әдістер

- `toc.li`: каталог нысанын немесе `{ key: title }` сияқты жазбалар тізімін орнатады
- `toc.mdUrl`: сұраудың URL мекенжайын құру функциясы `(key, lang) => url` немесе префикс жолын орнату
- `toc.target`: мақсатты мазмұн түйінін орнату немесе байланыстыру (мысалы, `c-md`), `ing` күту мен мазмұнды толтыруды автоматты түрде өңдеу
- `toc.load(key)`: сәйкес кілттің жүктелуін қолмен іске қосу
