# ಡೈರೆಕ್ಟರಿ ನ್ಯಾವಿಗೇಶನ್ ಕಾಂಪೊನೆಂಟ್

- ಲಂಬ ಫ್ಲೆಕ್ಸ್ ಲೇಔಟ್
- ಡೈರೆಕ್ಟರಿ ಡೇಟಾವನ್ನು ರವಾನಿಸಲು ನಿಘಂಟನ್ನು ಬೆಂಬಲಿಸಿ
- ಮಾರ್ಕ್‌ಡೌನ್ ಪಠ್ಯವನ್ನು ಸ್ವಯಂಚಾಲಿತವಾಗಿ ಲೋಡ್ ಮಾಡಲು ಕ್ಲಿಕ್ ಮಾಡಿ
- ಸ್ಥಿತಿ ಪ್ರದರ್ಶನ Ing ಅನಿಮೇಶನ್ ಅನ್ನು ಲೋಡ್ ಮಾಡಲಾಗುತ್ತಿದೆ

## ಡೆಮೊ ಬಳಸಿ

```html
<c-toc></c-toc>

<script type="module">
import "webc.site/Toc.js";

const toc = document.querySelector("c-toc");
toc.mdUrl = (key, lang) => "/doc/" + lang + "/" + key + ".md";
toc.li = {
  "README": "WebC.site",
  "ಡಾಕ್/ಬಳಕೆ": "ಘಟಕಗಳನ್ನು ಬಳಸಿ"
};
</script>
```

## ಇಂಟರ್ಫೇಸ್

### ಗುಣಲಕ್ಷಣಗಳು ಮತ್ತು ವಿಧಾನಗಳು

- `toc.li`: `{ key: title }` ನಂತಹ ಡೈರೆಕ್ಟರಿ ವಸ್ತು ಅಥವಾ ನಮೂದುಗಳ ಪಟ್ಟಿಯನ್ನು ಹೊಂದಿಸುತ್ತದೆ
- `toc.mdUrl`: ವಿನಂತಿ URL ಉತ್ಪಾದನೆಯ ಕಾರ್ಯವನ್ನು ಹೊಂದಿಸಿ `(key, lang) => url` ಅಥವಾ ಪೂರ್ವಪ್ರತ್ಯಯ ಸ್ಟ್ರಿಂಗ್
- `toc.target`: ಟಾರ್ಗೆಟ್ ಕಂಟೆಂಟ್ ನೋಡ್ ಅನ್ನು ಹೊಂದಿಸಿ ಅಥವಾ ಸಂಯೋಜಿಸಿ (ಉದಾಹರಣೆಗೆ `c-md`), ಸ್ವಯಂಚಾಲಿತವಾಗಿ `ing` ಕಾಯುವಿಕೆ ಮತ್ತು ವಿಷಯ ತುಂಬುವಿಕೆಯನ್ನು ನಿರ್ವಹಿಸಿ
- `toc.load(key)`: ಅನುಗುಣವಾದ ಕೀಲಿಯನ್ನು ಲೋಡ್ ಮಾಡುವುದನ್ನು ಹಸ್ತಚಾಲಿತವಾಗಿ ಟ್ರಿಗರ್ ಮಾಡಿ
