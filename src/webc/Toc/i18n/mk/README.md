# Компонента за навигација во директориумот

- Вертикален флекс распоред
- Поддршка за речник за пренесување податоци во директориумот
- Кликнете за автоматско вчитување на текстот на Markdown
- Вчитување на приказот на статусот Вчитување на анимација

## Користете го демо

```html
<c-toc></c-toc>

<script type="module">
import "webc.site/Toc.js";

const toc = document.querySelector("c-toc");
toc.mdUrl = (key, lang) => "/doc/" + lang + "/" + key + ".md";
toc.li = {
  "README": "WebC.site",
  "doc/use": "Користете компоненти"
};
</script>
```

## интерфејс

### Својства и методи

- `toc.li`: поставува објект на директориум или листа на записи, како што е `{ key: title }`
- `toc.mdUrl`: поставете функција за генерирање URL на барање `(key, lang) => url` или низа на префикс
- `toc.target`: поставете или поврзете го јазолот на целната содржина (како што е `c-md`), автоматски се справува со `ing` чекањето и пополнувањето содржина
- `toc.load(key)`: Рачно активирајте го вчитувањето на соодветниот клуч
