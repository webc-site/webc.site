# निर्देशिका नेव्हिगेशन घटक

- अनुलंब फ्लेक्स लेआउट
- डिरेक्टरी डेटा पास करण्यासाठी समर्थन शब्दकोश
- मार्कडाउन मजकूर स्वयंचलितपणे लोड करण्यासाठी क्लिक करा
- ॲनिमेशन स्थिती प्रदर्शन लोड करत आहे

## डेमो वापरा

```html
<c-toc></c-toc>

<script type="module">
import "webc.site/Toc.js";

const toc = document.querySelector("c-toc");
toc.mdUrl = (key, lang) => "/doc/" + lang + "/" + key + ".md";
toc.li = {
  "README": "WebC.site",
  "doc/use": "घटक वापरा"
};
</script>
```

## इंटरफेस

### गुणधर्म आणि पद्धती

- `toc.li`: निर्देशिका ऑब्जेक्ट किंवा नोंदींची सूची सेट करते, जसे की `{ key: title }`
- `toc.mdUrl`: विनंती URL निर्मिती कार्य `(key, lang) => url` किंवा उपसर्ग स्ट्रिंग सेट करा
- `toc.target`: लक्ष्य सामग्री नोड सेट करा किंवा संबद्ध करा (जसे की `c-md`), स्वयंचलितपणे `ing` प्रतीक्षा आणि सामग्री भरणे हाताळा
- `toc.load(key)`: संबंधित की लोड करणे व्यक्तिचलितपणे ट्रिगर करा
