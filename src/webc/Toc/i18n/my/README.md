# လမ်းညွှန်လမ်းညွှန်ချက် အစိတ်အပိုင်း

- ဒေါင်လိုက် Flex Layout
- လမ်းညွှန်ဒေတာတွင် ဖြတ်သန်းရန် အဘိဓာန်ကို ပံ့ပိုးပါ။
- Markdown စာသားကို အလိုအလျောက်တင်ရန် နှိပ်ပါ။
- Ing ကာတွန်းရုပ်ပြပြသမှု အခြေအနေကို ဖွင့်နေသည်။

## ဒီမိုကိုသုံးပါ။

```html
<c-toc></c-toc>

<script type="module">
import "webc.site/Toc.js";

const toc = document.querySelector("c-toc");
toc.mdUrl = (key, lang) => "/doc/" + lang + "/" + key + ".md";
toc.li = {
  "README": "WebC.site",
  "doc/use": "အစိတ်အပိုင်းများကို အသုံးပြုပါ"
};
</script>
```

## ကြားခံ

### ဂုဏ်သတ္တိများနှင့်နည်းလမ်းများ

- `toc.li`- `{ key: title }`ကဲ့သို့သော လမ်းညွှန်အရာဝတ္တု သို့မဟုတ် ထည့်သွင်းမှုစာရင်းကို သတ်မှတ်ပါ
- `toc.mdUrl`- တောင်းဆိုချက် URL ထုတ်လုပ်မှု လုပ်ဆောင်ချက် `(key, lang) => url` သို့မဟုတ် အကြိုစာကြောင်းကို သတ်မှတ်ပါ။
- `toc.target`- ပစ်မှတ်အကြောင်းအရာကို သတ်မှတ်ခြင်း သို့မဟုတ် ဆက်စပ်ခြင်း (`c-md`ကဲ့သို့)၊ အလိုအလျောက် `ing`စောင့်ဆိုင်းခြင်းနှင့် အကြောင်းအရာဖြည့်ခြင်း
- `toc.load(key)`- သက်ဆိုင်ရာကီးကို တင်ခြင်းအား ကိုယ်တိုင်စတင်ပါ။
