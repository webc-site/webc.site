# Кампанент навігацыі па каталогу

- Вертыкальны Flex макет
- Падтрымка слоўніка для перадачы дадзеных каталога
- Націсніце, каб аўтаматычна загрузіць тэкст Markdown
- Адлюстраванне стану загрузкі Ing animation

## Выкарыстоўвайце дэма

```html
<c-toc></c-toc>

<script type="module">
import "webc.site/Toc.js";

const toc = document.querySelector("c-toc");
toc.mdUrl = (key, lang) => "/doc/" + lang + "/" + key + ".md";
toc.li = {
  "README": "WebC.site",
  "doc/use": "Выкарыстоўваць кампаненты"
};
</script>
```

## інтэрфейс

### Уласцівасці і метады

- `toc.li`: Усталёўвае аб'ект каталога або спіс запісаў, напрыклад `{ key: title }`
- `toc.mdUrl`: усталяваць функцыю генерацыі URL запыту `(key, lang) => url` або радок прэфікса
- `toc.target`: усталяваць або звязаць мэтавы вузел кантэнту (напрыклад, `c-md`), аўтаматычна апрацоўваць `ing` чаканне і запаўненне кантэнтам
- `toc.load(key)`: уручную запусціць загрузку адпаведнага ключа
