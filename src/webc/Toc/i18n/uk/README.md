# Компонент навігації каталогом

- Вертикальний гнучкий макет
- Підтримка словника для передачі даних каталогу
- Натисніть, щоб автоматично завантажити текст Markdown
- Відображення статусу завантаження. Анімація

## Використовуйте демо

```html
<c-toc></c-toc>

<script type="module">
import "webc.site/Toc.js";

const toc = document.querySelector("c-toc");
toc.mdUrl = (key, lang) => "/doc/" + lang + "/" + key + ".md";
toc.li = {
  "README": "WebC.site",
  "doc/use": "Використовувати компоненти"
};
</script>
```

## інтерфейс

### Властивості та методи

- `toc.li`: встановлює об’єкт каталогу або список записів, наприклад `{ key: title }`
- `toc.mdUrl`: Установіть функцію створення URL-адреси запиту `(key, lang) => url` або рядок префікса
- `toc.target`: установіть або пов’яжіть цільовий вузол вмісту (наприклад, `c-md`), автоматично обробляйте `ing` очікування та заповнення вмістом
- `toc.load(key)`: вручну запустити завантаження відповідного ключа
