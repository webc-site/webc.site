# घटकों या उपकरण कार्यों को प्रस्तुत करने के लिए दस्तावेज़ीकरण और स्रोत कोड

- दो कॉलम वाला रिस्पॉन्सिव लेआउट, बाईं ओर दस्तावेज़ प्रदर्शित होते हैं और दाईं ओर स्रोत कोड प्रदर्शित होता है
- दस्तावेज़ीकरण और स्रोत कोड स्रोत पते पर जाने के लिए संपादन और कोड शॉर्टकट बटन का समर्थन करते हैं।
- शीर्षक और परिचय शीर्ष पर प्रदर्शित होते हैं, जो i18n स्वचालित अनुवाद का समर्थन करते हैं।

## डेमो का प्रयोग करें

```html
<c-webc-js src="/src/js/dom.js" href="/doc/{code}/js/dom.md"></c-webc-js>
```

## संपत्ति पैरामीटर

- `src`: स्रोत कोड का URL पता।
- `href`: दस्तावेज़ मार्कडाउन का URL पता, वर्तमान भाषा के अनुकूल होने के लिए `{code}` प्लेसहोल्डर का उपयोग करने का समर्थन करता है।
