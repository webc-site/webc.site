# اسناد و کد منبع برای رندر کردن اجزا یا توابع ابزار

- طرح‌بندی پاسخگو دو ستونی، با اسناد نمایش داده شده در سمت چپ و کد منبع نمایش داده شده در سمت راست
- اسناد و کد منبع از دکمه های میانبر ویرایش و کد برای پرش به آدرس منبع پشتیبانی می کنند.
- عنوان و مقدمه در بالا نمایش داده می شود و از ترجمه خودکار i18n پشتیبانی می کند.

## از دمو استفاده کنید

```html
<c-webc-js src="/src/js/dom.js" href="/doc/{code}/js/dom.md"></c-webc-js>
```

## پارامترهای ویژگی

- `src`: آدرس URL کد منبع.
- `href`: آدرس URL علامت‌گذاری سند، از استفاده از نگهدارنده `{code}` برای انطباق با زبان فعلی پشتیبانی می‌کند.
