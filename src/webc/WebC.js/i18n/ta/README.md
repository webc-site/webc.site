# கூறுகள் அல்லது கருவி செயல்பாடுகளை வழங்குவதற்கான ஆவணங்கள் மற்றும் மூல குறியீடு

- இரண்டு நெடுவரிசை பதிலளிக்கக்கூடிய தளவமைப்பு, ஆவணங்கள் இடதுபுறத்தில் காட்டப்படும் மற்றும் மூலக் குறியீடு வலதுபுறத்தில் காட்டப்படும்
- ஆவணப்படுத்தல் மற்றும் மூலக் குறியீடு ஆதரவு மூல முகவரிக்குச் செல்ல, திருத்து மற்றும் குறியீடு குறுக்குவழி பொத்தான்கள்.
- தலைப்பு மற்றும் அறிமுகம் மேலே காட்டப்படும், i18n தானியங்கி மொழிபெயர்ப்பை ஆதரிக்கிறது.

## டெமோவைப் பயன்படுத்தவும்

```html
<c-webc-js src="/src/js/dom.js" href="/doc/{code}/js/dom.md"></c-webc-js>
```

## சொத்து அளவுருக்கள்

- `src`: மூலக் குறியீட்டின் URL முகவரி.
- `href`: ஆவண மார்க் டவுனின் URL முகவரி, தற்போதைய மொழிக்கு ஏற்ப `{code}` ஒதுக்கிடத்தைப் பயன்படுத்துவதை ஆதரிக்கிறது.
