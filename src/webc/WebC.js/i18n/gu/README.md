# રેન્ડરિંગ ઘટકો અથવા સાધન કાર્યો માટે દસ્તાવેજીકરણ અને સ્રોત કોડ

- બે-કૉલમ રિસ્પોન્સિવ લેઆઉટ, ડાબી બાજુએ પ્રદર્શિત દસ્તાવેજો અને જમણી બાજુએ પ્રદર્શિત સ્રોત કોડ સાથે
- દસ્તાવેજીકરણ અને સ્રોત કોડ સ્રોત સરનામાં પર જવા માટે સંપાદિત કરો અને કોડ શોર્ટકટ બટનોને સપોર્ટ કરે છે.
- i18n સ્વચાલિત અનુવાદને સમર્થન આપતા શીર્ષક અને પરિચય ટોચ પર પ્રદર્શિત થાય છે.

## ડેમોનો ઉપયોગ કરો

```html
<c-webc-js src="/src/js/dom.js" href="/doc/{code}/js/dom.md"></c-webc-js>
```

## મિલકત પરિમાણો

- `src`: સ્ત્રોત કોડનું URL સરનામું.
- `href`: દસ્તાવેજ માર્કડાઉનનું URL સરનામું, વર્તમાન ભાષાને અનુકૂલિત કરવા માટે `{code}` પ્લેસહોલ્ડરનો ઉપયોગ કરવાનું સમર્થન કરે છે.
