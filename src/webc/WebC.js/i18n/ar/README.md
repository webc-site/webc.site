# الوثائق والكود المصدري لعرض المكونات أو وظائف الأداة

- تخطيط سريع الاستجابة مكون من عمودين، مع عرض المستندات على اليسار وكود المصدر المعروض على اليمين
- تدعم الوثائق والتعليمات البرمجية المصدرية أزرار الاختصار "تحرير" و"رمز" للانتقال إلى عنوان المصدر.
- يتم عرض العنوان والمقدمة في الأعلى، مع دعم الترجمة الآلية i18n.

## استخدم العرض التوضيحي

```html
<c-webc-js src="/src/js/dom.js" href="/doc/{code}/js/dom.md"></c-webc-js>
```

## معلمات الملكية

- `src`: عنوان URL للكود المصدر.
- `href`: عنوان URL الخاص بتخفيض المستند، ويدعم استخدام العنصر النائب `{code}` للتكيف مع اللغة الحالية.
