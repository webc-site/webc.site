# घटक किंवा टूल फंक्शन्स प्रस्तुत करण्यासाठी दस्तऐवजीकरण आणि स्त्रोत कोड

- दोन-स्तंभ प्रतिसादात्मक लेआउट, डावीकडे प्रदर्शित दस्तऐवज आणि उजवीकडे प्रदर्शित स्त्रोत कोड
- दस्तऐवजीकरण आणि स्त्रोत कोड स्त्रोत पत्त्यावर जाण्यासाठी संपादन आणि कोड शॉर्टकट बटणांना समर्थन देतात.
- i18n स्वयंचलित भाषांतरास समर्थन देणारे शीर्षक आणि परिचय शीर्षस्थानी प्रदर्शित केले जातात.

## डेमो वापरा

```html
<c-webc-js src="/src/js/dom.js" href="/doc/{code}/js/dom.md"></c-webc-js>
```

## मालमत्ता मापदंड

- `src`: स्त्रोत कोडचा URL पत्ता.
- `href`: दस्तऐवज मार्कडाउनचा URL पत्ता, सध्याच्या भाषेशी जुळवून घेण्यासाठी `{code}` प्लेसहोल्डर वापरण्यास समर्थन देतो.
