# დოკუმენტაცია და წყაროს კოდი კომპონენტების ან ხელსაწყოების ფუნქციების გაწევისთვის

- ორსვეტიანი საპასუხო განლაგება, დოკუმენტები ნაჩვენებია მარცხნივ და წყაროს კოდი ნაჩვენებია მარჯვნივ
- დოკუმენტაცია და წყაროს კოდი მხარს უჭერს რედაქტირების და კოდის მალსახმობების ღილაკებს წყაროს მისამართზე გადასასვლელად.
- სათაური და შესავალი ნაჩვენებია ზედა, რომელიც მხარს უჭერს i18n ავტომატურ თარგმნას.

## გამოიყენეთ დემო ვერსია

```html
<c-webc-js src="/src/js/dom.js" href="/doc/{code}/js/dom.md"></c-webc-js>
```

## თვისების პარამეტრები

- `src`: წყარო კოდის URL მისამართი.
- `href`: დოკუმენტის მარკირების URL მისამართი, მხარს უჭერს `{code}` ჩანაცვლების ველერის გამოყენებას მიმდინარე ენასთან ადაპტაციისთვის.
