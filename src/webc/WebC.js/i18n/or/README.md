# ଉପାଦାନ କିମ୍ବା ଉପକରଣ କାର୍ଯ୍ୟଗୁଡ଼ିକ ପାଇଁ ଡକ୍ୟୁମେଣ୍ଟେସନ୍ ଏବଂ ଉତ୍ସ କୋଡ୍ |

- ଦୁଇ ସ୍ତମ୍ଭର ପ୍ରତିକ୍ରିୟାଶୀଳ ଲେଆଉଟ୍, ବାମରେ ପ୍ରଦର୍ଶିତ ଡକ୍ୟୁମେଣ୍ଟ୍ ଏବଂ ଡାହାଣରେ ଉତ୍ସ କୋଡ୍ ପ୍ରଦର୍ଶିତ |
- ଡକ୍ୟୁମେଣ୍ଟେସନ ଏବଂ ଉତ୍ସ କୋଡ୍ ଉତ୍ସ ଠିକଣାକୁ ଡେଇଁବା ପାଇଁ ଏଡିଟ୍ ଏବଂ କୋଡ୍ ସର୍ଟକଟ୍ ବଟନ୍ ସମର୍ଥନ କରେ |
- ଶୀର୍ଷକ ଏବଂ ପରିଚୟ ଶୀର୍ଷରେ ପ୍ରଦର୍ଶିତ ହୋଇଛି, i18n ସ୍ୱୟଂଚାଳିତ ଅନୁବାଦକୁ ସମର୍ଥନ କରେ |

## ଡେମୋ ବ୍ୟବହାର କରନ୍ତୁ |

```html
<c-webc-js src="/src/js/dom.js" href="/doc/{code}/js/dom.md"></c-webc-js>
```

## ସମ୍ପତ୍ତି ପାରାମିଟରଗୁଡିକ |

- `src`: ଉତ୍ସ କୋଡ୍ ର URL ଠିକଣା |
- `href`: ଡକ୍ୟୁମେଣ୍ଟ୍ ମାର୍କଡାଉନ୍ ର URL ଠିକଣା, ସାମ୍ପ୍ରତିକ ଭାଷା ସହିତ ଖାପ ଖାଇବା ପାଇଁ `{code}` ପ୍ଲେସହୋଲ୍ଡର୍ ବ୍ୟବହାର କରିବାକୁ ସମର୍ଥନ କରେ |
