# Τεκμηρίωση και πηγαίος κώδικας για απόδοση στοιχείων ή λειτουργιών εργαλείου

- Διάταξη με απόκριση δύο στηλών, με έγγραφα που εμφανίζονται στα αριστερά και τον πηγαίο κώδικα που εμφανίζεται στα δεξιά
- Η τεκμηρίωση και ο πηγαίος κώδικας υποστηρίζουν τα κουμπιά συντόμευσης Επεξεργασία και Κώδικα για μετάβαση στη διεύθυνση πηγής.
- Ο τίτλος και η εισαγωγή εμφανίζονται στο επάνω μέρος, υποστηρίζοντας την αυτόματη μετάφραση i18n.

## Χρησιμοποιήστε το demo

```html
<c-webc-js src="/src/js/dom.js" href="/doc/{code}/js/dom.md"></c-webc-js>
```

## Παράμετροι ιδιοτήτων

- `src`: διεύθυνση URL του πηγαίου κώδικα.
- `href`: Διεύθυνση URL της σήμανσης εγγράφου, υποστηρίζει τη χρήση σύμβολο κράτησης θέσης `{code}` για προσαρμογή στην τρέχουσα γλώσσα.
