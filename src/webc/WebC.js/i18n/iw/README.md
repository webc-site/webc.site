# תיעוד וקוד מקור לעיבוד רכיבים או פונקציות כלי

- פריסה מגיבה של שתי עמודות, עם מסמכים מוצגים בצד שמאל וקוד המקור מוצג בצד ימין
- תמיכה בתיעוד וקוד מקור כפתורי קיצור עריכה וקוד לקפיצה לכתובת המקור.
- הכותרת והמבוא מוצגים בחלק העליון, ותומכים בתרגום אוטומטי של i18n.

## השתמש בהדגמה

```html
<c-webc-js src="/src/js/dom.js" href="/doc/{code}/js/dom.md"></c-webc-js>
```

## פרמטרים של נכס

- `src`: כתובת URL של קוד המקור.
- `href`: כתובת URL של סימון מסמך, תומך בשימוש ב`{code}` מציין מיקום כדי להתאים לשפה הנוכחית.
