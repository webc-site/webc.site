# Візуалізуйте та відображайте демонстрації компонентів, документацію та інструкції зі встановлення

- Витягніть і проаналізуйте назву, вступ і вміст документа README
- Пісочниця для запуску інтерактивних демонстрацій компонентів і виділення стилів
- Перетворюйте блоки коду в документах на інтерактивні презентації
- Адаптивний макет, широкоекранний дисплей у стовпцях, вузький екран з перемиканням вкладок

## Використовуйте демо

```html
<c-webc-demo></c-webc-demo>
```

```js
import WebCDemo from "./_.js";

const el = document.querySelector("c-webc-demo");

//Ініціалізація та отримання операційного інтерфейсу
const [setMod, setMd, unMount] = WebCDemo(el);
```

## Параметри інтерфейсу

### `WebCDemo(el)`

Ініціалізація компонента.

- **параметр**
  - `el`:`HTMLElement`, контейнер компонентів.
- **Повернене значення**:`[setMod, setMd, unMount]`
  - `setMod(mod)`: Налаштувати демонстраційний модуль.
  - `setMd(name, md, lang)`: установіть назву компонента, вміст документа README та поточну мову.
  - `unMount()`: вивантажте компонент і очистіть обробник подій.

### Атрибут `mod` демонстраційного модуля

- `CSS`:`string`, шлях до таблиці стилів компонента.
- `HTM`:`string`, шаблон HTML компонента.
- `default`:`function(root)`, функція ініціалізації компонента, отримує тіньовий кореневий контейнер.

## клас стилю

### `.M`

область вмісту.

### `.L`

Ліва панель містить інструкції та документацію.

### `.R`

Права панель, що містить демонстрацію.

### `c-nav`

Панель навігації на вузькому екрані.
