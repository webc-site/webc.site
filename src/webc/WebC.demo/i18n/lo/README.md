# ໃຫ້​ແລະ​ສະ​ແດງ​ການ​ສາ​ທິດ​ອົງ​ປະ​ກອບ​, ເອ​ກະ​ສານ​, ແລະ​ຄໍາ​ແນະ​ນໍາ​ການ​ຕິດ​ຕັ້ງ​

- ສະກັດແລະວິເຄາະຫົວຂໍ້, ການແນະນໍາແລະເນື້ອໃນຂອງເອກະສານ README
- Sandbox ເພື່ອເປີດໃຊ້ການສາທິດແບບໂຕ້ຕອບຂອງອົງປະກອບ ແລະຮູບແບບທີ່ໂດດດ່ຽວ
- ແປງບລັອກລະຫັດໃນເອກະສານເຂົ້າໄປໃນການນໍາສະເຫນີແບບໂຕ້ຕອບ
- ຮູບແບບການປັບຕົວ, ຈໍສະແດງຜົນຖັນຈໍກວ້າງ, ຈໍສະແດງຜົນສະຫຼັບແຖບແຄບ

## ໃຊ້ຕົວຢ່າງ

```html
<c-webc-demo></c-webc-demo>
```

```js
import WebCDemo from "./_.js";

const el = document.querySelector("c-webc-demo");

// ເລີ່ມຕົ້ນແລະໄດ້ຮັບການໂຕ້ຕອບການດໍາເນີນງານ
const [setMod, setMd, unMount] = WebCDemo(el);
```

## ຕົວກໍານົດການໃນການໂຕ້ຕອບ

### `WebCDemo(el)`

ເລີ່ມຕົ້ນອົງປະກອບ.

- ** ຕົວ​ກໍາ​ນົດ​ການ **
  - `el`:`HTMLElement`, ບັນຈຸອົງປະກອບ.
- **ຄ່າກັບຄືນ**:`[setMod, setMd, unMount]`
  - `setMod(mod)`: ຕັ້ງຄ່າໂມດູນສາທິດ.
  - `setMd(name, md, lang)`: ຕັ້ງຊື່ອົງປະກອບ, ເນື້ອໃນເອກະສານ README ແລະພາສາປັດຈຸບັນ.
  - `unMount()`: ຖອນອົງປະກອບ ແລະທຳຄວາມສະອາດຕົວຟັງເຫດການ.

### ໂມດູນຕົວຢ່າງ `mod`ຄຸນລັກສະນະ

- `CSS`:`string`, ເສັ້ນທາງສະໄຕລ໌ຊີດຂອງອົງປະກອບ.
- `HTM`:`string`, ແມ່ແບບ HTML ຂອງອົງປະກອບ.
- `default`:`function(root)`, ຟັງຊັນການເລີ່ມຕົ້ນອົງປະກອບ, ໄດ້ຮັບ shadow root container.
