# رينڊر ۽ ڊسپلي جزو ڊيمو، دستاويز، ۽ تنصيب جون هدايتون

- README دستاويز جو عنوان، تعارف ۽ مواد ڪڍيو ۽ پارس ڪريو
- سينڊ باڪس کي هلائڻ لاءِ انٽرايڪٽو ڊيمو اجزاء جا ۽ ڌار ڌار انداز
- دستاويزن ۾ ڪوڊ بلاڪ کي انٽرايڪٽو پريزنٽيشن ۾ تبديل ڪريو
- موافقت واري ترتيب، وسيع اسڪرين ڪالمن ڊسپلي، تنگ اسڪرين ٽيب سوئچنگ ڊسپلي

## ڊيمو استعمال ڪريو

```html
<c-webc-demo></c-webc-demo>
```

```js
import WebCDemo from "./_.js";

const el = document.querySelector("c-webc-demo");

// آپريشن انٽرفيس کي شروع ۽ حاصل ڪريو
const [setMod, setMd, unMount] = WebCDemo(el);
```

## انٽرفيس پيٽرول

### `WebCDemo(el)`

جزو کي شروع ڪريو.

- **پيراميٽر**
  - `el`:`HTMLElement`، جزو ڪنٽينر.
- **واپسي قدر**:`[setMod, setMd, unMount]`
  - `setMod(mod)`: ڊيمو ماڊل سيٽ اپ ڪريو.
  - `setMd(name, md, lang)`: جزو جو نالو مقرر ڪريو، README دستاويزي مواد ۽ موجوده ٻولي.
  - `unMount()`: جزو کي لوڊ ڪريو ۽ واقعي جي ٻڌندڙ کي صاف ڪريو.

### ڊيمو ماڊل `mod` خاصيت

- `CSS`:`string`، جزو جو انداز شيٽ رستو.
- `HTM`:`string`، جزو جو HTML ٽيمپليٽ.
- `default`:`function(root)`، جزو جي شروعاتي فنڪشن، شيڊ روٽ ڪنٽينر وصول ڪري ٿو.
