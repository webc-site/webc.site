# အစိတ်အပိုင်းသရုပ်ပြ၊ စာရွက်စာတမ်းနှင့် တပ်ဆင်မှုလမ်းညွှန်ချက်များကို တင်ဆက်ပြီး ပြသပါ။

- README စာတမ်း၏ ခေါင်းစဉ်၊ နိဒါန်းနှင့် အကြောင်းအရာတို့ကို ထုတ်ယူပြီး ခွဲခြမ်းစိတ်ဖြာပါ။
- အစိတ်အပိုင်းများနှင့် အပြန်အလှန်အကျိုးပြုသော သရုပ်ပြပုံစံများကို လုပ်ဆောင်ရန် Sandbox
- စာရွက်စာတမ်းများရှိ ကုဒ်တုံးများကို အပြန်အလှန်အကျိုးပြုသော တင်ပြချက်များအဖြစ် ပြောင်းပါ။
- လိုက်လျောညီထွေရှိသော အပြင်အဆင်၊ ကျယ်ပြန့်သော စခရင်ကော်လံပြသမှု၊ ကျဉ်းမြောင်းသော မျက်နှာပြင်တက်ဘ်ကို ကူးပြောင်းပြသမှု

## ဒီမိုကိုသုံးပါ။

```html
<c-webc-demo></c-webc-demo>
```

```js
import WebCDemo from "./_.js";

const el = document.querySelector("c-webc-demo");

// လုပ်ဆောင်မှု အင်တာဖေ့စ်ကို စတင်ပြီး ရယူပါ။
const [setMod, setMd, unMount] = WebCDemo(el);
```

## မျက်နှာပြင်ဘောင်များ

### `WebCDemo(el)`

အစိတ်အပိုင်းကို စတင်ပါ။

- ** သတ်မှတ်ချက် **
  - `el`:`HTMLElement`၊ အစိတ်အပိုင်း ကွန်တိန်နာ။
- **ပြန်တန်ဖိုး**:`[setMod, setMd, unMount]`
  - `setMod(mod)`- သရုပ်ပြ မော်ဂျူးကို စနစ်ထည့်သွင်းပါ။
  - `setMd(name, md, lang)`- အစိတ်အပိုင်းအမည်၊ README စာရွက်စာတမ်းအကြောင်းအရာနှင့် လက်ရှိဘာသာစကားကို သတ်မှတ်ပါ။
  - `unMount()`- အစိတ်အပိုင်းကို ဖြုတ်ပြီး အစီအစဉ် နားထောင်သူကို ရှင်းလင်းပါ။

### သရုပ်ပြ module `mod` ရည်ညွှန်းချက်

- `CSS`:`string`၊ အစိတ်အပိုင်း၏ ပုံစံစာရွက် လမ်းကြောင်း။
- `HTM`:`string`၊ အစိတ်အပိုင်း၏ HTML နမူနာ။
- `default`:`function(root)`၊ အစိတ်အပိုင်း အစပြုခြင်း လုပ်ဆောင်ချက်၊ အရိပ်အမြစ် ကွန်တိန်နာကို လက်ခံသည်။

## စတိုင်အတန်း

### `.M`

အကြောင်းအရာဧရိယာ။

### `.L`

ဘယ်ဘက်ဘောင်တွင် ညွှန်ကြားချက်များနှင့် စာရွက်စာတမ်းများ ပါရှိသည်။

### `.R`

ဒီမိုပါဝင်သော ညာဘက်အကန့်။

### `c-nav`

မျက်နှာပြင် ကျဉ်းမြောင်းသော လမ်းကြောင်းပြဘား။
