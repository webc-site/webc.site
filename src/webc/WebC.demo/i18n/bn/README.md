# উপাদান ডেমো, ডকুমেন্টেশন, এবং ইনস্টলেশন নির্দেশাবলী রেন্ডার এবং প্রদর্শন করুন

- README নথির শিরোনাম, ভূমিকা এবং বিষয়বস্তু বের করুন এবং পার্স করুন
- স্যান্ডবক্স উপাদানগুলির ইন্টারেক্টিভ ডেমো এবং বিচ্ছিন্ন শৈলী চালানোর জন্য
- নথিতে থাকা কোড ব্লকগুলিকে ইন্টারেক্টিভ উপস্থাপনায় রূপান্তর করুন
- অভিযোজিত বিন্যাস, প্রশস্ত স্ক্রীন কলাম প্রদর্শন, সংকীর্ণ স্ক্রীন ট্যাব স্যুইচিং প্রদর্শন

## ডেমো ব্যবহার করুন

```html
<c-webc-demo></c-webc-demo>
```

```js
import WebCDemo from "./_.js";

const el = document.querySelector("c-webc-demo");

// অপারেশন ইন্টারফেস শুরু করুন এবং প্রাপ্ত করুন
const [setMod, setMd, unMount] = WebCDemo(el);
```

## ইন্টারফেস পরামিতি

### `WebCDemo(el)`

উপাদানটি শুরু করুন।

- **প্যারামিটার**
  - `el`:`HTMLElement`, উপাদান ধারক।
- **রিটার্ন মান**:`[setMod, setMd, unMount]`
  - `setMod(mod)`: ডেমো মডিউল সেট আপ করুন৷
  - `setMd(name, md, lang)`: উপাদানের নাম, README নথির বিষয়বস্তু এবং বর্তমান ভাষা সেট করুন।
  - `unMount()`: উপাদানটি আনলোড করুন এবং ইভেন্ট শ্রোতাকে পরিষ্কার করুন৷

### ডেমো মডিউল `mod` অ্যাট্রিবিউট

- `CSS`:`string`, উপাদানটির স্টাইল শীট পাথ৷
- `HTM`:`string`, উপাদানটির HTML টেমপ্লেট৷
- `default`:`function(root)`, কম্পোনেন্ট ইনিশিয়ালাইজেশন ফাংশন, শ্যাডো রুট কন্টেইনার গ্রহণ করে।

## শৈলী ক্লাস

### `.M`

বিষয়বস্তু এলাকা।

### `.L`

বাম প্যানেলে নির্দেশাবলী এবং ডকুমেন্টেশন রয়েছে।

### `.R`

ডান প্যানেল, ডেমো ধারণকারী.

### `c-nav`

সংকীর্ণ পর্দা নেভিগেশন বার.
