# Ներկայացրեք և ցուցադրեք բաղադրիչի ցուցադրությունները, փաստաթղթերը և տեղադրման հրահանգները

- Քաղեք և վերլուծեք README փաստաթղթի վերնագիրը, ներածությունը և բովանդակությունը
- Sandbox՝ բաղադրիչների ինտերակտիվ ցուցադրություններ գործարկելու և ոճերը մեկուսացնելու համար
- Փաստաթղթերի կոդերի բլոկները փոխակերպեք ինտերակտիվ ներկայացումների
- Հարմարվողական դասավորություն, լայն էկրանի սյունակի ցուցադրում, նեղ էկրանի ներդիրների անջատման ցուցադրում

## Օգտագործեք ցուցադրությունը

```html
<c-webc-demo></c-webc-demo>
```

```js
import WebCDemo from "./_.js";

const el = document.querySelector("c-webc-demo");

//Նախաձեռնել և ձեռք բերել գործողության միջերեսը
const [setMod, setMd, unMount] = WebCDemo(el);
```

## Ինտերֆեյսի պարամետրեր

### `WebCDemo(el)`

Նախաձեռնեք բաղադրիչը:

- **պարամետր**
  - `el`:`HTMLElement`, բաղադրիչ կոնտեյներ։
- **Վերադարձի արժեքը**:`[setMod, setMd, unMount]`
  - `setMod(mod)`. կարգավորեք ցուցադրական մոդուլը:
  - `setMd(name, md, lang)`: Սահմանեք բաղադրիչի անունը, README փաստաթղթի բովանդակությունը և ընթացիկ լեզուն:
  - `unMount()`. բեռնաթափել բաղադրիչը և մաքրել իրադարձության ունկնդիրը:

### Դեմո մոդուլ `mod` հատկանիշ

- `CSS`:`string`, բաղադրիչի ոճաթերթի ուղին:
- `HTM`:`string`, բաղադրիչի HTML ձևանմուշը:
- `default`:`function(root)`, բաղադրիչի սկզբնավորման ֆունկցիա, ստանում է ստվերային արմատային կոնտեյներ:
