# ଉପାଦାନ ଡେମୋ, ଡକ୍ୟୁମେଣ୍ଟେସନ୍, ଏବଂ ସଂସ୍ଥାପନ ନିର୍ଦ୍ଦେଶାବଳୀ ପ୍ରଦାନ ଏବଂ ପ୍ରଦର୍ଶନ କରନ୍ତୁ |

- README ଡକ୍ୟୁମେଣ୍ଟର ଆଖ୍ୟା, ପରିଚୟ ଏବଂ ବିଷୟବସ୍ତୁକୁ ବାହାର ଏବଂ ବିଶ୍ଳେଷଣ କର |
- ଉପାଦାନଗୁଡ଼ିକର ଇଣ୍ଟରାକ୍ଟିଭ୍ ଡେମୋ ଚଲାଇବା ଏବଂ ଶ yles ଳୀକୁ ପୃଥକ କରିବା ପାଇଁ ସ୍ୟାଣ୍ଡବକ୍ସ |
- ଡକ୍ୟୁମେଣ୍ଟରେ କୋଡ୍ ବ୍ଲକଗୁଡ଼ିକୁ ଇଣ୍ଟରାକ୍ଟିଭ୍ ଉପସ୍ଥାପନାରେ ରୂପାନ୍ତର କରନ୍ତୁ |
- ଆଡାପ୍ଟିଭ୍ ଲେଆଉଟ୍, ୱାଇଡ୍ ସ୍କ୍ରିନ୍ ସ୍ତମ୍ଭ ପ୍ରଦର୍ଶନ, ସଂକୀର୍ଣ୍ଣ ସ୍କ୍ରିନ୍ ଟ୍ୟାବ୍ ସୁଇଚ୍ ଡିସପ୍ଲେ |

## ଡେମୋ ବ୍ୟବହାର କରନ୍ତୁ |

```html
<c-webc-demo></c-webc-demo>
```

```js
import WebCDemo from "./_.js";

const el = document.querySelector("c-webc-demo");

// ଅପରେସନ୍ ଇଣ୍ଟରଫେସ୍ ଆରମ୍ଭ ଏବଂ ପ୍ରାପ୍ତ କରନ୍ତୁ |
const [setMod, setMd, unMount] = WebCDemo(el);
```

## ଇଣ୍ଟରଫେସ୍ ପାରାମିଟରଗୁଡିକ

### `WebCDemo(el)`

ଉପାଦାନକୁ ଆରମ୍ଭ କରନ୍ତୁ |

- ** ପାରାମିଟର **
  - `el`: `HTMLElement`, ଉପାଦାନ ପାତ୍ର |
- ** ଫେରସ୍ତ ମୂଲ୍ୟ **: `[setMod, setMd, unMount]` |
  - `setMod(mod)`: ଡେମୋ ମଡ୍ୟୁଲ୍ ସେଟ୍ ଅପ୍ କରନ୍ତୁ |
  - `setMd(name, md, lang)`: ଉପାଦାନ ନାମ, README ଡକ୍ୟୁମେଣ୍ଟ ବିଷୟବସ୍ତୁ ଏବଂ ସାମ୍ପ୍ରତିକ ଭାଷା ସେଟ୍ କରନ୍ତୁ |
  - `unMount()`: ଉପାଦାନକୁ ଅନଲୋଡ୍ କରନ୍ତୁ ଏବଂ ଇଭେଣ୍ଟ ଶ୍ରୋତାଙ୍କୁ ସଫା କରନ୍ତୁ |

### ଡେମୋ ମଡ୍ୟୁଲ୍ `mod` ଗୁଣ |

- `CSS`: `string`, ଉପାଦାନର ଷ୍ଟାଇଲ୍ ସିଟ୍ ପଥ |
- `HTM`: `string`, ଉପାଦାନର HTML ଟେମ୍ପଲେଟ୍ |
- `default`: `function(root)`, ଉପାଦାନ ପ୍ରାରମ୍ଭିକରଣ କାର୍ଯ୍ୟ, ଛାୟା ମୂଳ ଧାରଣକାରୀ ଗ୍ରହଣ କରେ |

## ଶ style ଳୀ ଶ୍ରେଣୀ

### `.M`

ବିଷୟବସ୍ତୁ କ୍ଷେତ୍ର |

### `.L`

ବାମ ପ୍ୟାନେଲରେ ନିର୍ଦ୍ଦେଶ ଏବଂ ଡକ୍ୟୁମେଣ୍ଟେସନ୍ ଅଛି |

### `.R`

ଡେମୋ ଧାରଣ କରିଥିବା ଡାହାଣ ପ୍ୟାନେଲ୍ |

### `c-nav`

ସଂକୀର୍ଣ୍ଣ ସ୍କ୍ରିନ୍ ନେଭିଗେସନ୍ ବାର୍ |
