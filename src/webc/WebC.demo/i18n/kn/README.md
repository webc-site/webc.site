# ಕಾಂಪೊನೆಂಟ್ ಡೆಮೊಗಳು, ದಸ್ತಾವೇಜನ್ನು ಮತ್ತು ಅನುಸ್ಥಾಪನಾ ಸೂಚನೆಗಳನ್ನು ನಿರೂಪಿಸಿ ಮತ್ತು ಪ್ರದರ್ಶಿಸಿ

- README ಡಾಕ್ಯುಮೆಂಟ್‌ನ ಶೀರ್ಷಿಕೆ, ಪರಿಚಯ ಮತ್ತು ವಿಷಯವನ್ನು ಹೊರತೆಗೆಯಿರಿ ಮತ್ತು ಪಾರ್ಸ್ ಮಾಡಿ
- ಘಟಕಗಳ ಸಂವಾದಾತ್ಮಕ ಡೆಮೊಗಳನ್ನು ಚಲಾಯಿಸಲು ಮತ್ತು ಶೈಲಿಗಳನ್ನು ಪ್ರತ್ಯೇಕಿಸಲು ಸ್ಯಾಂಡ್‌ಬಾಕ್ಸ್
- ಡಾಕ್ಯುಮೆಂಟ್‌ಗಳಲ್ಲಿನ ಕೋಡ್ ಬ್ಲಾಕ್‌ಗಳನ್ನು ಸಂವಾದಾತ್ಮಕ ಪ್ರಸ್ತುತಿಗಳಾಗಿ ಪರಿವರ್ತಿಸಿ
- ಅಡಾಪ್ಟಿವ್ ಲೇಔಟ್, ವೈಡ್ ಸ್ಕ್ರೀನ್ ಕಾಲಮ್ ಡಿಸ್ಪ್ಲೇ, ಕಿರಿದಾದ ಸ್ಕ್ರೀನ್ ಟ್ಯಾಬ್ ಸ್ವಿಚಿಂಗ್ ಡಿಸ್ಪ್ಲೇ

## ಡೆಮೊ ಬಳಸಿ

```html
<c-webc-demo></c-webc-demo>
```

```js
import WebCDemo from "./_.js";

const el = document.querySelector("c-webc-demo");

// ಕಾರ್ಯಾಚರಣೆಯ ಇಂಟರ್ಫೇಸ್ ಅನ್ನು ಪ್ರಾರಂಭಿಸಿ ಮತ್ತು ಪಡೆದುಕೊಳ್ಳಿ
const [setMod, setMd, unMount] = WebCDemo(el);
```

## ಇಂಟರ್ಫೇಸ್ ನಿಯತಾಂಕಗಳು

### `WebCDemo(el)`

ಘಟಕವನ್ನು ಪ್ರಾರಂಭಿಸಿ.

- **ಪ್ಯಾರಾಮೀಟರ್**
  - `el`:`HTMLElement`, ಕಾಂಪೊನೆಂಟ್ ಕಂಟೇನರ್.
- **ಹಿಂತಿರುಗಿ ಮೌಲ್ಯ**:`[setMod, setMd, unMount]`
  - `setMod(mod)`: ಡೆಮೊ ಮಾಡ್ಯೂಲ್ ಅನ್ನು ಹೊಂದಿಸಿ.
  - `setMd(name, md, lang)`: ಘಟಕದ ಹೆಸರು, README ಡಾಕ್ಯುಮೆಂಟ್ ವಿಷಯ ಮತ್ತು ಪ್ರಸ್ತುತ ಭಾಷೆಯನ್ನು ಹೊಂದಿಸಿ.
  - `unMount()`: ಕಾಂಪೊನೆಂಟ್ ಅನ್ನು ಅನ್‌ಲೋಡ್ ಮಾಡಿ ಮತ್ತು ಈವೆಂಟ್ ಆಲಿಸುವವರನ್ನು ಸ್ವಚ್ಛಗೊಳಿಸಿ.

### ಡೆಮೊ ಮಾಡ್ಯೂಲ್ `mod` ಗುಣಲಕ್ಷಣ

- `CSS`:`string`, ಘಟಕದ ಶೈಲಿಯ ಹಾಳೆಯ ಮಾರ್ಗ.
- `HTM`:`string`, ಘಟಕದ HTML ಟೆಂಪ್ಲೇಟ್.
- `default`:`function(root)`, ಕಾಂಪೊನೆಂಟ್ ಇನಿಶಿಯಲೈಸೇಶನ್ ಫಂಕ್ಷನ್, ಶ್ಯಾಡೋ ರೂಟ್ ಕಂಟೇನರ್ ಅನ್ನು ಪಡೆಯುತ್ತದೆ.
