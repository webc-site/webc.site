# కాంపోనెంట్ డెమోలు, డాక్యుమెంటేషన్ మరియు ఇన్‌స్టాలేషన్ సూచనలను రెండర్ చేయండి మరియు ప్రదర్శించండి

- README పత్రం యొక్క శీర్షిక, పరిచయం మరియు కంటెంట్‌ను సంగ్రహించి, అన్వయించండి
- శాండ్‌బాక్స్ కాంపోనెంట్స్ మరియు ఐసోలేట్ స్టైల్స్ యొక్క ఇంటరాక్టివ్ డెమోలను అమలు చేయడానికి
- డాక్యుమెంట్‌లలోని కోడ్ బ్లాక్‌లను ఇంటరాక్టివ్ ప్రెజెంటేషన్‌లుగా మార్చండి
- అడాప్టివ్ లేఅవుట్, వైడ్ స్క్రీన్ కాలమ్ డిస్‌ప్లే, ఇరుకైన స్క్రీన్ ట్యాబ్ స్విచింగ్ డిస్‌ప్లే

## డెమో ఉపయోగించండి

```html
<c-webc-demo></c-webc-demo>
```

```js
import WebCDemo from "./_.js";

const el = document.querySelector("c-webc-demo");

//ఆపరేషన్ ఇంటర్‌ఫేస్‌ను ప్రారంభించండి మరియు పొందండి
const [setMod, setMd, unMount] = WebCDemo(el);
```

## ఇంటర్ఫేస్ పారామితులు

### `WebCDemo(el)`

భాగాన్ని ప్రారంభించండి.

- **పరామితి**
  - `el`:`HTMLElement`, కాంపోనెంట్ కంటైనర్.
- **రిటర్న్ విలువ**:`[setMod, setMd, unMount]`
  - `setMod(mod)`: డెమో మాడ్యూల్‌ని సెటప్ చేయండి.
  - `setMd(name, md, lang)`: భాగం పేరు, README డాక్యుమెంట్ కంటెంట్ మరియు ప్రస్తుత భాషను సెట్ చేయండి.
  - `unMount()`: కాంపోనెంట్‌ను అన్‌లోడ్ చేయండి మరియు ఈవెంట్ లిజనర్‌ను క్లీన్ అప్ చేయండి.

### డెమో మాడ్యూల్ `mod` లక్షణం

- `CSS`:`string`, భాగం యొక్క స్టైల్ షీట్ పాత్.
- `HTM`:`string`, భాగం యొక్క HTML టెంప్లేట్.
- `default`:`function(root)`, కాంపోనెంట్ ఇనిషియలైజేషన్ ఫంక్షన్, షాడో రూట్ కంటైనర్‌ను అందుకుంటుంది.

## శైలి తరగతి

### `.M`

కంటెంట్ ప్రాంతం.

### `.L`

ఎడమ పానెల్‌లో సూచనలు మరియు డాక్యుమెంటేషన్ ఉన్నాయి.

### `.R`

కుడి పానెల్, డెమోను కలిగి ఉంది.

### `c-nav`

ఇరుకైన స్క్రీన్ నావిగేషన్ బార్.
