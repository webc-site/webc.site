# Feilmeldingssiden finnes ikke

- Spørresiden ble ikke funnet, oppgi en knapp for å gå tilbake til startsiden.
- Det lille spøkelsesøyeeplet følger musepekeren.
- Klikk på det lille spøkelset for å gå inn i den sinte tilstanden og avfyre ​​lasere.
- Kontinuerlig klikking vil få det lille spøkelset til å eksplodere og gå i oppløsning, og deretter gjenoppstå etter noen sekunder.
