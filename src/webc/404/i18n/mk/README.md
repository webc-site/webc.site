# Патеката не постои страница со порака за грешка

- Страницата за барање не е пронајдена, обезбедете копче за враќање на почетната страница.
- Очното јаболко од малиот дух го следи покажувачот на глувчето.
- Кликнете на малиот дух за да влезете во лута состојба и да запалите ласери.
- Континуираното кликнување ќе предизвика малиот дух да експлодира и да се распадне, а потоа повторно да се роди по неколку секунди.
