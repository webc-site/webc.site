# Жол қате туралы хабар беті жоқ

- Шақыру беті табылмады, басты бетке оралу үшін түймені беріңіз.
- Кішкентай елес көз алмасы тінтуірдің меңзерін қадағалайды.
- Ашулы күйге кіру және лазерлерді өртеу үшін кішкентай елесті басыңыз.
- Үздіксіз басу кішкентай елестің жарылып, ыдырауына әкеледі, содан кейін бірнеше секундтан кейін қайта пайда болады.
