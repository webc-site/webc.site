# পথ ত্রুটি বার্তা পৃষ্ঠা বিদ্যমান নেই

- প্রম্পট পৃষ্ঠা পাওয়া যায়নি, হোম পেজে ফিরে যেতে একটি বোতাম প্রদান করুন।
- ছোট্ট ভূতের আইবল মাউস পয়েন্টার ট্র্যাক করে।
- রাগান্বিত রাজ্য এবং আগুন লেজারে প্রবেশ করতে ছোট্ট ভূতের উপর ক্লিক করুন।
- ক্রমাগত ক্লিক করার ফলে ছোট্ট ভূতটি বিস্ফোরিত হবে এবং বিচ্ছিন্ন হয়ে যাবে এবং তারপর কয়েক সেকেন্ড পরে পুনরায় জন্ম দেবে।
