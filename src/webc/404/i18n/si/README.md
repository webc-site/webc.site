# මාර්ගය දෝෂ පණිවිඩ පිටුව නොපවතී

- විමසුම් පිටුව හමු නොවීය, මුල් පිටුවට ආපසු යාමට බොත්තමක් සපයන්න.
- කුඩා අවතාර ඇහිබැම මූසික දර්ශකය නිරීක්ෂණය කරයි.
- කෝපාවිෂ්ට තත්වයට ඇතුළු වීමට කුඩා අවතාරය මත ක්ලික් කර ලේසර් වෙඩි තියන්න.
- අඛණ්ඩව ක්ලික් කිරීම කුඩා අවතාරය පිපිරීමට හා විසුරුවා හැරීමට හේතු වනු ඇත, පසුව තත්පර කිහිපයකට පසු නැවත ඉපදේ.
