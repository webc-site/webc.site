# Il percorso non esiste nella pagina dei messaggi di errore

- Pagina di richiesta non trovata, fornire un pulsante per tornare alla home page.
- Il piccolo bulbo oculare fantasma segue il puntatore del mouse.
- Clicca sul piccolo fantasma per entrare nello stato arrabbiato e sparare con i laser.
- Un clic continuo farà esplodere e disintegrare il piccolo fantasma, per poi rinascere dopo pochi secondi.
