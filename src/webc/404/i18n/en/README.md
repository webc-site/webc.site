# Path does not exist error message page

- Prompt page not found, provide a button to return to the home page.
- The little ghost eyeball tracks the mouse pointer.
- Click on the little ghost to enter the angry state and fire lasers.
- Continuous clicking will cause the little ghost to explode and disintegrate, and then respawn after a few seconds.

## Use the demo

```html
<c-404></c-404>
```
