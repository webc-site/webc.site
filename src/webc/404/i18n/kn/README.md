# ಮಾರ್ಗವು ಅಸ್ತಿತ್ವದಲ್ಲಿಲ್ಲ ದೋಷ ಸಂದೇಶ ಪುಟ

- ಪ್ರಾಂಪ್ಟ್ ಪುಟ ಕಂಡುಬಂದಿಲ್ಲ, ಮುಖಪುಟಕ್ಕೆ ಹಿಂತಿರುಗಲು ಬಟನ್ ಅನ್ನು ಒದಗಿಸಿ.
- ಪುಟ್ಟ ಪ್ರೇತ ಕಣ್ಣುಗುಡ್ಡೆಯು ಮೌಸ್ ಪಾಯಿಂಟರ್ ಅನ್ನು ಟ್ರ್ಯಾಕ್ ಮಾಡುತ್ತದೆ.
- ಕೋಪಗೊಂಡ ರಾಜ್ಯ ಮತ್ತು ಬೆಂಕಿ ಲೇಸರ್ಗಳನ್ನು ಪ್ರವೇಶಿಸಲು ಸ್ವಲ್ಪ ಪ್ರೇತದ ಮೇಲೆ ಕ್ಲಿಕ್ ಮಾಡಿ.
- ನಿರಂತರ ಕ್ಲಿಕ್ಕಿಸುವಿಕೆಯು ಚಿಕ್ಕ ಪ್ರೇತವನ್ನು ಸ್ಫೋಟಿಸಲು ಮತ್ತು ವಿಘಟನೆಗೆ ಕಾರಣವಾಗುತ್ತದೆ, ಮತ್ತು ನಂತರ ಕೆಲವು ಸೆಕೆಂಡುಗಳ ನಂತರ ಮತ್ತೆ ಹುಟ್ಟಿಕೊಳ್ಳುತ್ತದೆ.
