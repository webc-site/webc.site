# Путања не постоји страница са поруком о грешци

- Страница са упитом није пронађена, обезбедите дугме за повратак на почетну страницу.
- Мала духовна очна јабучица прати показивач миша.
- Кликните на малог духа да уђете у бесно стање и испалите ласере.
- Континуирано кликање ће узроковати да мали дух експлодира и распадне се, а затим се поново појави након неколико секунди.
