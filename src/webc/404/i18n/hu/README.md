# Az elérési út nem létezik hibaüzenet oldal

- A prompt oldal nem található, adjon meg egy gombot a kezdőlapra való visszatéréshez.
- A kis szellem szemgolyó követi az egérmutatót.
- Kattintson a kis szellemre, hogy belépjen a dühös állapotba, és lőjön lézereket.
- A folyamatos kattintás hatására a kis szellem felrobban és szétesik, majd néhány másodperc múlva újra megjelenik.
