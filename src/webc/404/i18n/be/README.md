# Шлях не існуе старонкі паведамлення пра памылку

- Старонка падказкі не знойдзена, дайце кнопку для вяртання на галоўную старонку.
- Маленькі вочны яблык-прывід адсочвае паказальнік мышы.
- Націсніце на маленькага прывіда, каб увайсці ў стан гневу і страляць лазерамі.
- Пастаяннае націсканне прывядзе да таго, што маленькі прывід выбухне і распадзецца, а потым адрадзіцца праз некалькі секунд.
