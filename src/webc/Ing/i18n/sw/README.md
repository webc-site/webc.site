# Kiashiria cha hali ya upakiaji au usindikaji

- Unganisha kishale katika hali ya kusubiri
- Huonyesha uhuishaji wa ripple ili kuonyesha hali ya uchakataji
- Tumia kinyago cha ukungu kwenye fomu ili kuzuia uwasilishaji unaorudiwa

## Tumia onyesho

```html
<!-- kiashiria cha upakiaji wa kujitegemea -->
<div class="Ing"></div>

<!-- Inapakia fomu na mask -->
<form class="Ing">
  <input type="text" placeholder="Sanduku la kuingiza">
  <button type="submit">wasilisha</button>
</form>
```

## darasa la mtindo

### `.Ing`
- Hali ya kielekezi ni `wait`.
- Kipengele bandia `::before` kinaonyesha uhuishaji wa ripple.
- Inapotumiwa kwenye `form` kipengee, kipengele cha uwongo `::after` hutoa kinyago cha 1px cha ukungu, kuzima mwingiliano na uteuzi wa maandishi.
