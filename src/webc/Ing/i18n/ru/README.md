# Индикатор статуса загрузки или обработки

- Объединить курсор в состояние ожидания
- Показывает пульсирующую анимацию, указывающую состояние обработки.
- Примените маску размытия к форме, чтобы предотвратить дублирование отправок.

## Используйте демо-версию

```html
<!-- независимый индикатор загрузки -->
<div class="Ing"></div>

<!-- Загрузка формы с маской -->
<form class="Ing">
  <input type="text" placeholder="Поле ввода">
  <button type="submit">представлять на рассмотрение</button>
</form>
```

## класс стиля

### `.Ing`
- Статус указателя: `wait`.
- Псевдоэлемент `::before` отображает пульсирующую анимацию.
- При использовании в элементе `form` псевдоэлемент `::after` отображает маску размытия размером 1 пиксель, отключая взаимодействие и выделение текста.
