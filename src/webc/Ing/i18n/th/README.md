# ตัวบ่งชี้สถานะการโหลดหรือการประมวลผล

- รวมเคอร์เซอร์เข้าสู่สถานะรอ
- แสดงภาพเคลื่อนไหวระลอกคลื่นเพื่อระบุสถานะการประมวลผล
- ใช้เบลอมาสก์กับแบบฟอร์มเพื่อป้องกันการส่งซ้ำ

## ใช้การสาธิต

```html
<!-- ตัวบ่งชี้การโหลดอิสระ -->
<div class="Ing"></div>

<!-- กำลังโหลดแบบฟอร์มพร้อมหน้ากาก -->
<form class="Ing">
  <input type="text" placeholder="กล่องขาเข้า">
  <button type="submit">ส่ง</button>
</form>
```

## คลาสสไตล์

### `.Ing`
- สถานะตัวชี้คือ `wait`
- องค์ประกอบหลอก `::before` แสดงผลภาพเคลื่อนไหวระลอกคลื่น
- เมื่อใช้กับองค์ประกอบ `form` องค์ประกอบหลอก `::after` เรนเดอร์มาสก์เบลอ 1px ปิดการใช้งานการโต้ตอบและการเลือกข้อความ
