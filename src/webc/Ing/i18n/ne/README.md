# लोडिङ वा प्रशोधन स्थितिको सूचक

- पर्खाइको अवस्थामा कर्सरलाई एकीकृत गर्नुहोस्
- प्रशोधन स्थिति संकेत गर्न लहर एनिमेसन देखाउँछ
- डुप्लिकेट सबमिशनहरू रोक्नको लागि फारममा ब्लर मास्क लागू गर्नुहोस्

## डेमो प्रयोग गर्नुहोस्

```html
<!-- स्वतन्त्र लोड सूचक -->
<div class="Ing"></div>

<!-- मास्कको साथ फारम लोड गर्दै -->
<form class="Ing">
  <input type="text" placeholder="इनपुट बक्स">
  <button type="submit">पेश गर्नुहोस्</button>
</form>
```

## शैली वर्ग

### `.Ing`
- सूचक स्थिति `wait` हो।
- स्यूडो एलिमेन्ट `::before` ले रिपल एनिमेसन रेन्डर गर्छ।
- जब `form` तत्वमा प्रयोग गरिन्छ, छद्म-तत्व `::after` ले 1px ब्लर मास्क रेन्डर गर्छ, अन्तरक्रिया र पाठ चयन असक्षम पार्छ।
