# લોડિંગ અથવા પ્રોસેસિંગ સ્થિતિનું સૂચક

- કર્સરને પ્રતીક્ષા સ્થિતિમાં એકીકૃત કરો
- પ્રોસેસિંગ સ્ટેટસ દર્શાવવા માટે રિપલ એનિમેશન બતાવે છે
- ડુપ્લિકેટ સબમિશનને રોકવા માટે ફોર્મ પર બ્લર માસ્ક લાગુ કરો

## ડેમોનો ઉપયોગ કરો

```html
<!-- સ્વતંત્ર લોડિંગ સૂચક -->
<div class="Ing"></div>

<!-- માસ્ક સાથે ફોર્મ લોડ કરી રહ્યું છે -->
<form class="Ing">
  <input type="text" placeholder="ઇનપુટ બોક્સ">
  <button type="submit">સબમિટ કરો</button>
</form>
```

## શૈલી વર્ગ

### `.Ing`
- નિર્દેશક સ્થિતિ `wait` છે.
- સ્યુડો એલિમેન્ટ `::before` રિપલ એનિમેશન રેન્ડર કરે છે.
- જ્યારે `form` તત્વ પર ઉપયોગ કરવામાં આવે છે, ત્યારે સ્યુડો-તત્વ `::after` 1px બ્લર માસ્ક રેન્ડર કરે છે, ક્રિયાપ્રતિક્રિયા અને ટેક્સ્ટ પસંદગીને અક્ષમ કરે છે.
