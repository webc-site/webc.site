# লোডিং বা প্ৰচেছিং অৱস্থাৰ সূচক

- কাৰ্চাৰক অপেক্ষাৰ অৱস্থালৈ একত্ৰিত কৰক
- প্ৰচেছিং অৱস্থা সূচাবলৈ ৰিপল এনিমেচন দেখুৱায়
- ডুপ্লিকেট জমা দিয়া ৰোধ কৰিবলৈ এটা প্ৰপত্ৰত এটা ব্লাৰ মাস্ক প্ৰয়োগ কৰক

## ডেমো ব্যৱহাৰ কৰক

```html
<!-- স্বতন্ত্ৰ লোডিং সূচক -->
<div class="Ing"></div>

<!-- মাস্কৰ সৈতে ফৰ্ম লোড কৰা হৈছে -->
<form class="Ing">
  <input type="text" placeholder="ইনপুট বক্স">
  <button type="submit">দাখিল কৰক</button>
</form>
```

## শৈলী শ্ৰেণী

### `.Ing`
- পইণ্টাৰৰ অৱস্থা হৈছে `wait`।
- ছ্যুডো উপাদান `::before` এ এটা ৰিপল এনিমেচন ৰেণ্ডাৰ কৰে।
- `form` উপাদানত ব্যৱহাৰ কৰাৰ সময়ত, ছ্যুডো-উপাদান `::after` এ এটা 1px ব্লাৰ মাস্ক ৰেণ্ডাৰ কৰে, পাৰস্পৰিক ক্ৰিয়া আৰু লিখনী নিৰ্ব্বাচন নিষ্ক্ৰিয় কৰে।
