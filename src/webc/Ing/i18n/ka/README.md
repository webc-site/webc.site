# ჩატვირთვის ან დამუშავების სტატუსის მაჩვენებელი

- გააერთიანეთ კურსორი მოლოდინის მდგომარეობაში
- აჩვენებს ტალღოვან ანიმაციას დამუშავების სტატუსის მითითებისთვის
- წაისვით ბუნდოვანი ნიღაბი ფორმაზე, რათა თავიდან აიცილოთ დუბლიკატები

## გამოიყენეთ დემო ვერსია

```html
<!-- დამოუკიდებელი დატვირთვის მაჩვენებელი -->
<div class="Ing"></div>

<!-- ფორმის ჩატვირთვა ნიღბით -->
<form class="Ing">
  <input type="text" placeholder="შეყვანის ყუთი">
  <button type="submit">წარადგინოს</button>
</form>
```

## სტილის კლასი

### `.Ing`
- მაჩვენებლის სტატუსი არის `wait`.
- ფსევდო ელემენტი `::before` ასახავს ტალღოვან ანიმაციას.
- როდესაც გამოიყენება `form` ელემენტზე, ფსევდო-ელემენტი `::after` წარმოქმნის 1px ბუნდოვან ნიღაბს, რაც თიშავს ინტერაქციას და ტექსტის შერჩევას.
