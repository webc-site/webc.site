# Բեռնման կամ մշակման կարգավիճակի ցուցիչ

- Միավորեք կուրսորը սպասողական վիճակի մեջ
- Ցույց է տալիս ալիքային անիմացիան՝ ցույց տալու մշակման կարգավիճակը
- Կիրառեք բծախնդրության դիմակ ձևի վրա՝ կրկնօրինակ ներկայացումները կանխելու համար

## Օգտագործեք ցուցադրությունը

```html
<!-- անկախ բեռնման ցուցիչ -->
<div class="Ing"></div>

<!-- Բեռնում ձևը դիմակով -->
<form class="Ing">
  <input type="text" placeholder="Ներածման տուփ">
  <button type="submit">ներկայացնել</button>
</form>
```

## ոճի դաս

### `.Ing`
- Ցուցիչի կարգավիճակը `wait` է:
- Կեղծ տարրը `::before` ներկայացնում է ալիքային անիմացիա:
- Երբ օգտագործվում է `form` տարրի վրա, կեղծ տարրը `::after` ցուցադրում է 1px մշուշման դիմակ՝ անջատելով փոխազդեցությունը և տեքստի ընտրությունը:
