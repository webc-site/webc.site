# Автоматическое переключение меню навигации и кнопки возврата

- Автоматически переносить дочерние элементы для создания структур меню навигации и кнопок возврата.
- Автоматически отслеживать историю маршрутизации и рассчитывать предыдущий обратный путь.
- Отобразить меню навигации, если маршрут соответствует пункту меню, в противном случае отобразить кнопку «Назад».

## Используйте демо-версию

```html
<c-nav-l>
  <a href="/">первая страница</a>
  <a href="/doc">документ</a>
</c-nav-l>
```

## класс стиля

- `B`: активирует состояние кнопки возврата. Показать кнопку «Назад» и скрыть меню навигации. Автоматическое переключение внутри компонента на основе маршрутизации.
