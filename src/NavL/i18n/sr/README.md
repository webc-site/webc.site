# Аутоматски пребаците навигациони мени и дугме за повратак

- Аутоматски омотајте подређене елементе да бисте генерисали навигациони мени и структуре дугмади за повратак
- Аутоматски пратите историју рутирања и израчунајте претходну повратну путању
- Прикажите навигациони мени када се рута подудара са ставком менија, у супротном прикажите дугме за назад
