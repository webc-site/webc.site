# Ngalih kanthi otomatis menu navigasi lan tombol bali

- Lebokake unsur anak kanthi otomatis kanggo ngasilake menu navigasi lan struktur tombol bali
- Nglacak riwayat rute kanthi otomatis lan ngitung dalan bali sadurunge
- Nampilake menu navigasi nalika rute cocog karo item menu, yen ora, nampilake tombol bali

## Gunakake demo

```html
<c-nav-l>
  <a href="/">kaca ngarep</a>
  <a href="/doc">dokumen</a>
</c-nav-l>
```

## kelas gaya

- `B`: ngaktifake status tombol bali. Tampilake tombol bali lan ndhelikake menu navigasi. Ngalih kanthi otomatis ing komponen adhedhasar rute.
