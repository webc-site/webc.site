# ナビゲーションメニューと戻るボタンを自動的に切り替えます

- 子要素を自動的にラップしてナビゲーション メニューを生成し、ボタン構造を返す
- ルーティング履歴を自動的にトレースし、以前のリターン パスを計算します。
- ルートがメニュー項目と一致する場合はナビゲーションメニューを表示し、一致しない場合は戻るボタンを表示します

## デモを使用する

```html
<c-nav-l>
  <a href="/">フロントページ</a>
  <a href="/doc">書類</a>
</c-nav-l>
```

## スタイルクラス

- `B`: 戻るボタンの状態をアクティブにします。戻るボタンを表示し、ナビゲーション メニューを非表示にします。ルーティングに基づいてコンポーネント内で自動的に切り替えられます。
