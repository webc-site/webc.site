# Автоматичне перемикання навігаційного меню та кнопки повернення

- Автоматичне обгортання дочірніх елементів для створення структур меню навігації та кнопок повернення
- Автоматично відстежуйте історію маршрутів і обчислюйте попередній шлях повернення
- Відображати навігаційне меню, коли маршрут відповідає пункту меню, інакше відображати кнопку «Назад».

## Використовуйте демо

```html
<c-nav-l>
  <a href="/">титульна сторінка</a>
  <a href="/doc">документ</a>
</c-nav-l>
```

## клас стилю

- `B`: активує стан кнопки повернення. Показати кнопку «Назад» і приховати навігаційне меню. Автоматично перемикається в межах компонента на основі маршрутизації.
