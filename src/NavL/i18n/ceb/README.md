# Awtomatikong ibalhin ang menu sa nabigasyon ug buton sa pagbalik

- Awtomatikong iputos ang mga elemento sa bata aron makamugna ang menu sa nabigasyon ug mga istruktura sa buton sa pagbalik
- Awtomatikong pagsubay sa kasaysayan sa pag-ruta ug pagkalkulo sa miaging agianan sa pagbalik
- Ipakita ang navigation menu kung ang ruta motakdo sa menu item, kung dili ipakita ang back button

## Gamita ang demo

```html
<c-nav-l>
  <a href="/">atubangan nga panid</a>
  <a href="/doc">dokumento</a>
</c-nav-l>
```

## estilo nga klase

- `B`: nagpalihok sa balik nga buton nga kahimtang. Ipakita ang back button ug itago ang navigation menu. Awtomatikong gibalhin sulod sa component base sa routing.
