# Schalt automatesch Navigatiounsmenü a Retour Knäppchen

- Wickelt automatesch Kannerelementer fir Navigatiounsmenü ze generéieren an Knäppercherstrukturen zréckzekommen
- Automatesch Routing Geschicht verfollegen a berechent de fréiere Retour Wee
- Weist de Navigatiounsmenü wann d'Streck dem Menüpunkt entsprécht, soss weist de Back-Knäppchen

## Stil Klass

- `B`: aktivéiert de Retour Knäppchen Staat. Weist de Réck Knäppchen a verstoppt den Navigatiounsmenü. Automatesch gewiesselt bannent der Komponent baséiert op Routing.
