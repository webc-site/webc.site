# Αυτόματη εναλλαγή μενού πλοήγησης και κουμπί επιστροφής

- Τυλίξτε αυτόματα θυγατρικά στοιχεία για να δημιουργήσετε δομές μενού πλοήγησης και κουμπιών επιστροφής
- Αυτόματη ανίχνευση ιστορικού δρομολόγησης και υπολογισμός της προηγούμενης διαδρομής επιστροφής
- Εμφανίστε το μενού πλοήγησης όταν η διαδρομή ταιριάζει με το στοιχείο μενού, διαφορετικά εμφανίστε το κουμπί επιστροφής

## Χρησιμοποιήστε το demo

```html
<c-nav-l>
  <a href="/">πρωτοσέλιδο</a>

```

## τάξη στυλ

- `B`: ενεργοποιεί την κατάσταση του κουμπιού επιστροφής. Εμφάνιση του κουμπιού επιστροφής και απόκρυψη του μενού πλοήγησης. Αυτόματη εναλλαγή εντός του στοιχείου βάσει δρομολόγησης.
