# স্বয়ংক্ৰিয়ভাৱে নেভিগেচন মেনু আৰু উভতি অহা বুটাম সলনি কৰক

- নেভিগেচন মেনু সৃষ্টি কৰিবলে আৰু বুটাম গঠনসমূহ ঘূৰাই দিবলে সন্তান উপাদানসমূহ স্বয়ংক্ৰিয়ভাৱে ৰেপ কৰক
- স্বয়ংক্ৰিয়ভাৱে ৰাউটিং ইতিহাস অনুসৰণ কৰক আৰু পূৰ্বৰ উভতি অহা পথ গণনা কৰক
- নেভিগেচন মেনু প্ৰদৰ্শন কৰক যেতিয়া পথ মেনু বস্তুৰ সৈতে মিলে, অন্যথা পিছৰ বুটাম প্ৰদৰ্শন কৰক
