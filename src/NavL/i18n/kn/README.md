# ನ್ಯಾವಿಗೇಷನ್ ಮೆನು ಮತ್ತು ರಿಟರ್ನ್ ಬಟನ್ ಅನ್ನು ಸ್ವಯಂಚಾಲಿತವಾಗಿ ಬದಲಾಯಿಸಿ

- ನ್ಯಾವಿಗೇಷನ್ ಮೆನು ಮತ್ತು ರಿಟರ್ನ್ ಬಟನ್ ರಚನೆಗಳನ್ನು ರಚಿಸಲು ಮಕ್ಕಳ ಅಂಶಗಳನ್ನು ಸ್ವಯಂಚಾಲಿತವಾಗಿ ಸುತ್ತಿಕೊಳ್ಳಿ
- ರೂಟಿಂಗ್ ಇತಿಹಾಸವನ್ನು ಸ್ವಯಂಚಾಲಿತವಾಗಿ ಪತ್ತೆಹಚ್ಚಿ ಮತ್ತು ಹಿಂದಿನ ಹಿಂತಿರುಗುವ ಮಾರ್ಗವನ್ನು ಲೆಕ್ಕಾಚಾರ ಮಾಡಿ
- ಮಾರ್ಗವು ಮೆನು ಐಟಂಗೆ ಹೊಂದಿಕೆಯಾದಾಗ ನ್ಯಾವಿಗೇಷನ್ ಮೆನುವನ್ನು ಪ್ರದರ್ಶಿಸಿ, ಇಲ್ಲದಿದ್ದರೆ ಹಿಂದಿನ ಬಟನ್ ಅನ್ನು ಪ್ರದರ್ಶಿಸಿ
