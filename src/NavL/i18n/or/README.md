# ସ୍ୱୟଂଚାଳିତ ଭାବରେ ନାଭିଗେସନ୍ ମେନୁ ଏବଂ ରିଟର୍ନ ବଟନ୍ ସୁଇଚ୍ କରନ୍ତୁ |

- ନାଭିଗେସନ୍ ମେନୁ ଏବଂ ରିଟର୍ନ ବଟନ୍ ଗଠନ ପାଇଁ ସ୍ୱୟଂଚାଳିତ ଭାବରେ ଶିଶୁ ଉପାଦାନଗୁଡ଼ିକୁ ଗୁଡ଼ାଇ ଦିଅ |
- ସ୍ୱୟଂଚାଳିତ ଭାବରେ ରାଉଟିଙ୍ଗ୍ ଇତିହାସ ଅନୁସନ୍ଧାନ କରନ୍ତୁ ଏବଂ ପୂର୍ବ ଫେରସ୍ତ ପଥ ଗଣନା କରନ୍ତୁ |
- ମାର୍ଗଟି ମେନୁ ଆଇଟମ୍ ସହିତ ମେଳ ହେଲେ ନାଭିଗେସନ୍ ମେନୁ ପ୍ରଦର୍ଶନ କରନ୍ତୁ, ନଚେତ୍ ପଛ ବଟନ୍ ପ୍ରଦର୍ଶନ କରନ୍ତୁ |
