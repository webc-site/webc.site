# स्वचालित रूपमा नेभिगेसन मेनु र फिर्ता बटन स्विच गर्नुहोस्

- नेभिगेसन मेनु र रिटर्न बटन ढाँचाहरू उत्पन्न गर्न बाल तत्वहरूलाई स्वचालित रूपमा र्‍याप गर्नुहोस्
- स्वचालित रूपमा मार्ग इतिहास ट्रेस गर्नुहोस् र अघिल्लो फिर्ती मार्ग गणना गर्नुहोस्
- नेभिगेसन मेनु प्रदर्शन गर्नुहोस् जब मार्ग मेनु वस्तुसँग मेल खान्छ, अन्यथा पछाडि बटन प्रदर्शन गर्नुहोस्
