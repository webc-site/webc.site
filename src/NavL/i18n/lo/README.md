# ສະຫຼັບເມນູນຳທາງ ແລະປຸ່ມກັບຄືນໂດຍອັດຕະໂນມັດ

- ຫໍ່ອົງປະກອບເດັກນ້ອຍໂດຍອັດຕະໂນມັດເພື່ອສ້າງເມນູນໍາທາງແລະໂຄງສ້າງປຸ່ມກັບຄືນ
- ຕິດຕາມປະຫວັດເສັ້ນທາງອັດຕະໂນມັດແລະຄິດໄລ່ເສັ້ນທາງກັບຄືນທີ່ຜ່ານມາ
- ສະແດງເມນູນໍາທາງເມື່ອເສັ້ນທາງກົງກັບລາຍການເມນູ, ຖ້າບໍ່ດັ່ງນັ້ນຈະສະແດງປຸ່ມກັບຄືນ
