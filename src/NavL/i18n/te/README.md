# నావిగేషన్ మెను మరియు రిటర్న్ బటన్‌ను స్వయంచాలకంగా మార్చండి

- నావిగేషన్ మెను మరియు రిటర్న్ బటన్ నిర్మాణాలను రూపొందించడానికి చైల్డ్ ఎలిమెంట్‌లను ఆటోమేటిక్‌గా చుట్టండి
- స్వయంచాలకంగా రూటింగ్ చరిత్రను కనుగొనండి మరియు మునుపటి రిటర్న్ మార్గాన్ని లెక్కించండి
- మార్గం మెను ఐటెమ్‌తో సరిపోలినప్పుడు నావిగేషన్ మెనుని ప్రదర్శించండి, లేకపోతే బ్యాక్ బటన్‌ను ప్రదర్శించండి

## శైలి తరగతి

- `B`: రిటర్న్ బటన్ స్థితిని సక్రియం చేస్తుంది. వెనుక బటన్‌ను చూపించి, నావిగేషన్ మెనుని దాచండి. రౌటింగ్ ఆధారంగా భాగం లోపల స్వయంచాలకంగా స్విచ్ చేయబడింది.
