# සංචාලන මෙනුව සහ ආපසු බොත්තම ස්වයංක්‍රීයව මාරු කරන්න

- සංචාලන මෙනුව සහ ආපසු බොත්තම් ව්‍යුහයන් ජනනය කිරීමට ළමා මූලද්‍රව්‍ය ස්වයංක්‍රීයව ඔතා
- මාර්ගගත ඉතිහාසය ස්වයංක්‍රීයව සොයා ගෙන පෙර ආපසු එන මාර්ගය ගණනය කරන්න
- මාර්ගය මෙනු අයිතමයට ගැළපෙන විට සංචාලන මෙනුව සංදර්ශන කරන්න, එසේ නොමැතිනම් පසුපස බොත්තම සංදර්ශන කරන්න
