# Badili menyu ya kusogeza na kitufe cha kurudi kiotomatiki

- Funga vipengele vya mtoto kiotomatiki ili kuunda menyu ya kusogeza na miundo ya vitufe vya kurejesha
- Fuatilia kiotomatiki historia ya uelekezaji na ukokote njia ya awali ya kurudi
- Onyesha menyu ya kusogeza wakati njia inalingana na kipengee cha menyu, vinginevyo onyesha kitufe cha nyuma
