# Навигация мәзірін және қайтару түймесін автоматты түрде ауыстырыңыз

- Шарлау мәзірін жасау және түйме құрылымдарын қайтару үшін еншілес элементтерді автоматты түрде орау
- Маршруттау тарихын автоматты түрде қадағалаңыз және алдыңғы қайтару жолын есептеңіз
- Маршрут мәзір элементіне сәйкес келгенде шарлау мәзірін көрсетіңіз, әйтпесе артқа түймешігін көрсетіңіз

## Демонстрацияны пайдаланыңыз

```html
<c-nav-l>
  <a href="/">алдыңғы бет</a>
  <a href="/doc">құжат</a>
</c-nav-l>
```

## стиль сыныбы

- `B`: қайтару түймесі күйін белсендіреді. Артқа түймесін көрсетіңіз және шарлау мәзірін жасырыңыз. Маршрутизация негізінде құрамдас ішінде автоматты түрде ауысады.
