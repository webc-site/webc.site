# Ավտոմատ կերպով փոխեք նավիգացիոն ընտրացանկը և վերադարձի կոճակը

- Ավտոմատ փաթաթեք երեխայի տարրերը՝ նավիգացիոն մենյուի և վերադարձի կոճակների կառուցվածքներ ստեղծելու համար
- Ավտոմատ հետագծել երթուղիների պատմությունը և հաշվարկել նախորդ վերադարձի ուղին
- Ցուցադրել նավիգացիոն ընտրացանկը, երբ երթուղին համընկնում է ընտրացանկի տարրի հետ, հակառակ դեպքում ցուցադրեք «Հետ» կոճակը
