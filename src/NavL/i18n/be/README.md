# Аўтаматычнае пераключэнне меню навігацыі і кнопкі вяртання

- Аўтаматычнае абгортванне даччыных элементаў для стварэння структур меню навігацыі і кнопак вяртання
- Аўтаматычнае адсочванне гісторыі маршрутаў і разлік папярэдняга зваротнага шляху
- Адлюстроўваць меню навігацыі, калі маршрут адпавядае пункту меню, у адваротным выпадку паказваць кнопку "Назад".

## клас стылю

- `B`: актывуе стан кнопкі вяртання. Паказаць кнопку "Назад" і схаваць меню навігацыі. Аўтаматычнае пераключэнне ўнутры кампанента на аснове маршрутызацыі.
