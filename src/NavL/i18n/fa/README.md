# تغییر خودکار منوی ناوبری و دکمه بازگشت

- به طور خودکار عناصر کودک را بپیچید تا ساختارهای منوی ناوبری و دکمه بازگشت ایجاد شود
- ردیابی خودکار تاریخچه مسیریابی و محاسبه مسیر برگشت قبلی
- هنگامی که مسیر با آیتم منو مطابقت دارد، منوی پیمایش را نمایش دهید، در غیر این صورت دکمه برگشت را نمایش دهید
