# Skiptu sjálfkrafa um leiðsöguvalmynd og afturhnapp

- Vefjið sjálfkrafa undireiningum til að búa til leiðsöguvalmynd og skila hnappabyggingum
- Rekja sjálfkrafa leiðarferil og reikna út fyrri afturslóð
- Sýna leiðsöguvalmyndina þegar leiðin passar við valmyndaratriðið, annars birtu til baka hnappinn
