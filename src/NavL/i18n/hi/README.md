# नेविगेशन मेनू और रिटर्न बटन को स्वचालित रूप से स्विच करें

- नेविगेशन मेनू और रिटर्न बटन संरचनाएं उत्पन्न करने के लिए स्वचालित रूप से चाइल्ड तत्वों को लपेटें
- स्वचालित रूप से रूटिंग इतिहास का पता लगाएं और पिछले रिटर्न पथ की गणना करें
- जब मार्ग मेनू आइटम से मेल खाता हो तो नेविगेशन मेनू प्रदर्शित करें, अन्यथा बैक बटन प्रदर्शित करें

## डेमो का प्रयोग करें

```html
<c-nav-l>
  <a href="/">मुखपृष्ठ</a>

```

## शैली वर्ग

- `B`: रिटर्न बटन स्थिति को सक्रिय करता है। बैक बटन दिखाएँ और नेविगेशन मेनू छिपाएँ। रूटिंग के आधार पर घटक के भीतर स्वचालित रूप से स्विच किया जाता है।
