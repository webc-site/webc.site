# نیویگیشن مینو اور ریٹرن بٹن کو خودکار طور پر سوئچ کریں۔

- نیویگیشن مینو اور ریٹرن بٹن کے ڈھانچے بنانے کے لیے بچوں کے عناصر کو خودکار طور پر لپیٹ دیں۔
- خودکار طور پر روٹنگ کی تاریخ کا سراغ لگائیں اور پچھلے واپسی کے راستے کا حساب لگائیں۔
- جب روٹ مینو آئٹم سے میل کھاتا ہے تو نیویگیشن مینو ڈسپلے کریں، بصورت دیگر بیک بٹن ڈسپلے کریں۔
