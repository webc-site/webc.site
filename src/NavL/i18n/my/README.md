# လမ်းကြောင်းပြမီနူးကို အလိုအလျောက်ပြောင်းပြီး ပြန်ပေးသည့် ခလုတ်

- လမ်းကြောင်းပြမီနူးကို ထုတ်လုပ်ရန်နှင့် ခလုတ်ဖွဲ့စည်းပုံများကို ပြန်ပေးရန် ကလေးဒြပ်စင်များကို အလိုအလျောက် ခြုံပါ။
- လမ်းကြောင်းလမ်းကြောင်းမှတ်တမ်းကို အလိုအလျောက်ခြေရာခံပြီး ယခင်ပြန်သွားသည့်လမ်းကြောင်းကို တွက်ချက်ပါ။
- လမ်းကြောင်းသည် မီနူးအကြောင်းအရာနှင့် ကိုက်ညီသည့်အခါ လမ်းကြောင်းပြမီနူးကို ပြသပါ၊ သို့မဟုတ်ပါက နောက်ပြန်ခလုတ်ကို ပြသပါ။
