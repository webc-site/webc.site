- ავტომატურად შეფუთეთ ბავშვის ელემენტები ნავიგაციის მენიუსა და დაბრუნების ღილაკების სტრუქტურების გენერირებისთვის
- მარშრუტიზაციის ისტორიის ავტომატური მიკვლევა და წინა დაბრუნების ბილიკის გამოთვლა
