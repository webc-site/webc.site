# 탐색 메뉴 및 복귀 버튼 자동 전환

- 탐색 메뉴 및 반환 버튼 구조를 생성하기 위해 하위 요소를 자동으로 래핑합니다.
- 라우팅 이력을 자동으로 추적하고 이전 반환 경로를 계산합니다.
- 경로가 메뉴 항목과 일치하면 내비게이션 메뉴를 표시하고, 그렇지 않으면 뒤로 버튼을 표시합니다.

## 데모 사용

```html
<c-nav-l>
  <a href="/">첫 페이지</a>
  <a href="/doc">문서</a>
</c-nav-l>
```

## 스타일 수업

- `B`: 복귀 버튼 상태를 활성화합니다. 뒤로 버튼을 표시하고 탐색 메뉴를 숨깁니다. 라우팅을 기반으로 구성 요소 내에서 자동으로 전환됩니다.
