# नेव्हिगेशन मेनू आणि रिटर्न बटण स्वयंचलितपणे स्विच करा

- नेव्हिगेशन मेनू आणि रिटर्न बटण संरचना व्युत्पन्न करण्यासाठी मूल घटक स्वयंचलितपणे गुंडाळा
- राउटिंग इतिहास स्वयंचलितपणे ट्रेस करा आणि मागील परतीच्या मार्गाची गणना करा
- जेव्हा मार्ग मेनू आयटमशी जुळतो तेव्हा नेव्हिगेशन मेनू प्रदर्शित करा, अन्यथा मागील बटण प्रदर्शित करा
