# Yönlendirme portalı barındırma bileşeni kitaplığı belgeleri ve demo sitesi

- **Tam ekran düzeni**: Mutlak konumlandırma, yerleşik kaydırma ve gezinme ile görünüm alanını doldurur.
- **Dinamik Yönlendirme**: URL yollarını ayrıştırın ve bileşen demolarını veya ana sayfalarını dinamik olarak yükleyin.
- **Çoklu dil**: Gezinme çubuğunu ve sayfa kopyasını gerçek zamanlı olarak yanıtlayın ve güncelleyin.
- **Durum değiştirme**: Yükleme ve ana sayfanın stil sınıflarını otomatik olarak koruyun.

## Demoyu kullanın

```html
<c-webc-index></c-webc-index>
```

## stil sınıfı

Bileşen kendi içinde aşağıdaki sınıf adlarını değiştirecektir:

- `Ing`: Rota yükleme veya dil paketi ayrıştırma.
- `A`: Şu anda ana sayfada.
