# Պորտալի հոստինգի բաղադրիչ գրադարանի փաստաթղթերի և ցուցադրական կայքի երթուղիավորում

- **Ամբողջ էկրանի դասավորություն**. բացարձակ դիրքավորումը լրացնում է տեսադաշտը` ներկառուցված ոլորման և նավիգացիայի միջոցով:
- **Դինամիկ երթուղղում**. վերլուծել URL-ի ուղիները և դինամիկ բեռնել բաղադրիչների ցուցադրությունները կամ գլխավոր էջերը:
- **Բազմալեզու**. Պատասխանեք և թարմացրեք նավիգացիոն տողը և էջի պատճենը իրական ժամանակում:
- **Կարգավիճակի փոխարկում**. ավտոմատ կերպով պահպանում է բեռնման և գլխավոր էջի ոճերի դասերը:
