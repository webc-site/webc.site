# روٹنگ پورٹل ہوسٹنگ جزو لائبریری دستاویزات اور ڈیمو سائٹ

- **مکمل اسکرین لے آؤٹ**: مکمل پوزیشننگ ویوپورٹ کو بھرتی ہے، بلٹ ان اسکرولنگ اور نیویگیشن کے ساتھ۔
- **متحرک روٹنگ**: یو آر ایل کے راستے پارس کریں اور اجزاء کے ڈیمو یا ہوم پیجز کو متحرک طور پر لوڈ کریں۔
- **ملٹی لینگویج**: نیویگیشن بار اور صفحہ کاپی کو حقیقی وقت میں جواب دیں اور اپ ڈیٹ کریں۔
- **اسٹیٹس سوئچنگ**: لوڈنگ اور ہوم پیج کی اسٹائل کلاسز کو خودکار طور پر برقرار رکھیں۔
