# Routing portal hosting komponent bibliotek dokumentation og demo site

- **Fuldskærmslayout**: Absolut positionering fylder visningsporten med indbygget rulning og navigation.
- **Dynamisk routing**: Parse URL-stier, og indlæs dynamisk komponentdemoer eller hjemmesider.
- **Multi-language**: Svar og opdater navigationslinjen og sidekopien i realtid.
- **Statusskift**: Oprethold automatisk stilklasserne for indlæsning og startside.

## Brug demoen

```html
<c-webc-index></c-webc-index>
```

## stil klasse

Komponenten skifter følgende klassenavne i sig selv:

- `Ing`: Ruteindlæsning eller sprogpakke-parsing.
- `A`: Lige nu på hjemmesiden.
