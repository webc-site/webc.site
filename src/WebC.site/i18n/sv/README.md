# Routing portal värd komponent bibliotek dokumentation och demo webbplats

- **Helskärmslayout**: Absolut positionering fyller visningsporten, med inbyggd rullning och navigering.
- **Dynamisk routing**: Analysera webbadressvägar och ladda dynamiskt komponentdemos eller hemsidor.
- **Multi-language**: Svara och uppdatera navigeringsfältet och sidkopian i realtid.
- **Statusväxling**: Upprätthåll automatiskt stilklasserna för laddning och hemsida.

## Använd demon

```html
<c-webc-index></c-webc-index>
```

## stilklass

Komponenten kommer att byta följande klassnamn inom sig själv:

- `Ing`: Ruttladdning eller språkpaketsanalys.
- `A`: För närvarande på hemsidan.
