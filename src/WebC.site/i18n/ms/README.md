# Dokumentasi perpustakaan komponen pengehosan portal penghalaan dan tapak demo

- **Reka letak skrin penuh**: Kedudukan mutlak memenuhi ruang pandang, dengan tatal terbina dalam dan navigasi.
- **Penghalaan Dinamik**: Menghuraikan laluan URL dan memuatkan demo komponen atau halaman utama secara dinamik.
- **Berbilang bahasa**: Balas dan kemas kini bar navigasi dan salinan halaman dalam masa nyata.
- **Penukaran status**: Mengekalkan kelas gaya pemuatan dan halaman utama secara automatik.

## kelas gaya

Komponen akan menukar nama kelas berikut dalam dirinya sendiri:

- `Ing`: Pemuatan laluan atau penghuraian pek bahasa.
- `A`: Pada masa ini di halaman utama.
