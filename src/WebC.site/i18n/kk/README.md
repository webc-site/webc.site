# Маршруттау порталы хостинг құрамдас кітапханасының құжаттамасы және демо-сайт

- **Толық экран орналасуы**: абсолютті орналасу кірістірілген айналдыру және навигация арқылы көру алаңын толтырады.
- **Динамикалық бағыттау**: URL жолдарын талдау және құрамдас көрсетілімдерді немесе басты беттерді динамикалық түрде жүктеңіз.
- **Көп тілді**: нақты уақытта навигациялық жолақ пен бет көшірмесіне жауап беріңіз және жаңартыңыз.
- **Күйді ауыстыру**: Жүктеу мен басты беттің стиль сыныптарын автоматты түрде сақтау.

## Демонстрацияны пайдаланыңыз

```html
<c-webc-index></c-webc-index>
```

## стиль сыныбы

Компонент өз ішінде келесі сынып атауларын ауыстырады:

- `Ing`: маршрутты жүктеу немесе тіл бумасын талдау.
- `A`: Қазіргі уақытта басты бетте.
