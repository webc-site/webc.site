# Sitio de demostración y documentación de la biblioteca de componentes de alojamiento del portal de enrutamiento

- **Diseño de pantalla completa**: el posicionamiento absoluto llena la ventana gráfica, con desplazamiento y navegación integrados.
- **Enrutamiento dinámico**: analiza rutas URL y carga dinámicamente demostraciones de componentes o páginas de inicio.
- **Multilenguaje**: Responde y actualiza la barra de navegación y el texto de la página en tiempo real.
- **Cambio de estado**: mantiene automáticamente las clases de estilo de carga y página de inicio.

## Usa la demostración

```html
<c-webc-index></c-webc-index>
```

## clase de estilo

El componente cambiará los siguientes nombres de clase dentro de sí mismo:

- `Ing`: carga de ruta o análisis de paquetes de idioma.
- `A`: Actualmente en la página de inicio.
