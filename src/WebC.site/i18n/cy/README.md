# Porth llwybro sy'n cynnal dogfennaeth llyfrgell gydran a safle arddangos

- **Cynllun sgrin lawn**: Mae lleoliad absoliwt yn llenwi'r olygfan, gyda sgrolio a llywio integredig.
- ** Llwybro Dynamig **: Dosrannu llwybrau URL a llwytho demos neu hafan cydrannau yn ddeinamig.
- **Aml-iaith**: Ymateb a diweddaru'r bar llywio a chopi tudalen mewn amser real.
- **Newid statws**: Cynnal y dosbarthiadau arddull llwytho a hafan yn awtomatig.

## dosbarth arddull

Bydd y gydran yn newid yr enwau dosbarth canlynol ynddo'i hun:

- `Ing`: Llwytho llwybr neu ddosrannu pecyn iaith.
- `A`: Ar yr hafan ar hyn o bryd.
