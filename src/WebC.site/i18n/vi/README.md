# Cổng thông tin định tuyến lưu trữ tài liệu thư viện thành phần và trang demo

- **Bố cục toàn màn hình**: Định vị tuyệt đối lấp đầy khung nhìn, với tính năng cuộn và điều hướng tích hợp.
- **Định tuyến động**: Phân tích đường dẫn URL và tải động các bản trình diễn hoặc trang chủ thành phần.
- **Đa ngôn ngữ**: Phản hồi và cập nhật thanh điều hướng và bản sao trang theo thời gian thực.
- **Chuyển đổi trạng thái**: Tự động duy trì các lớp kiểu tải và trang chủ.

## Sử dụng bản trình diễn

```html
<c-webc-index></c-webc-index>
```

## lớp phong cách

Thành phần này sẽ chuyển đổi các tên lớp sau trong chính nó:

- `Ing`: Tải tuyến hoặc phân tích gói ngôn ngữ.
- `A`: Hiện có trên trang chủ.
