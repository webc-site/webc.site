# રૂટીંગ પોર્ટલ હોસ્ટિંગ ઘટક પુસ્તકાલય દસ્તાવેજીકરણ અને ડેમો સાઇટ

- **પૂર્ણ સ્ક્રીન લેઆઉટ**: સંપૂર્ણ સ્થિતિ બિલ્ટ-ઇન સ્ક્રોલિંગ અને નેવિગેશન સાથે વ્યૂપોર્ટને ભરે છે.
- **ડાયનેમિક રૂટીંગ**: URL પાથને પાર્સ કરો અને ઘટક ડેમો અથવા હોમપેજને ગતિશીલ રીતે લોડ કરો.
- **મલ્ટી-લેંગ્વેજ**: રીઅલ ટાઇમમાં નેવિગેશન બાર અને પેજ કોપીને પ્રતિસાદ આપો અને અપડેટ કરો.
- **સ્ટેટસ સ્વિચિંગ**: લોડિંગ અને હોમપેજના સ્ટાઇલ ક્લાસને આપમેળે જાળવો.

## શૈલી વર્ગ

ઘટક નીચેના વર્ગના નામોને પોતાની અંદર સ્વિચ કરશે:

- `Ing`: રૂટ લોડિંગ અથવા ભાષા પેક પાર્સિંગ.
- `A`: હાલમાં હોમપેજ પર છે.
