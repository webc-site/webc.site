# Marsruutimisportaali hostimise komponentide teegi dokumentatsioon ja demosait

- **Täisekraani paigutus**: sisseehitatud kerimise ja navigeerimisega täidab vaateava absoluutne positsioneerimine.
- **Dünaamiline marsruutimine**: sõeluge URL-i teid ja laadige dünaamiliselt komponentide demosid või kodulehti.
- **Mitme keel**: vastake ja värskendage navigeerimisriba ja lehe koopiat reaalajas.
- **Oleku vahetamine**: laadimise ja avalehe stiiliklasside automaatne säilitamine.

## Kasutage demo

```html
<c-webc-index></c-webc-index>
```

## stiiliklass

Komponent vahetab enda sees järgmisi klassinimesid:

- `Ing`: marsruudi laadimine või keelepaketi sõelumine.
- `A`: hetkel avalehel.
