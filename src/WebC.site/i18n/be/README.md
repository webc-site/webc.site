# Партал маршрутызацыі, які змяшчае дакументацыю бібліятэкі кампанентаў і дэма-сайт

- **Поўнаэкранны макет**: Абсалютнае пазіцыянаванне запаўняе акно прагляду з убудаванай пракруткай і навігацыяй.
- **Дынамічная маршрутызацыя**: разбірайце URL-шляхі і дынамічна загружайце дэма-версіі кампанентаў або хатнія старонкі.
- **Шматмоўная**: адказвайце і абнаўляйце панэль навігацыі і копію старонкі ў рэжыме рэальнага часу.
- **Пераключэнне стану**: Аўтаматычна падтрымліваць класы стыляў загрузкі і галоўнай старонкі.

## Выкарыстоўвайце дэма

```html
<c-webc-index></c-webc-index>
```

## клас стылю

Кампанент будзе пераключаць наступныя назвы класаў у сабе:

- `Ing`: загрузка маршруту або аналіз моўнага пакета.
- `A`: зараз на галоўнай старонцы.
