# Smerovací portál hosťujúci dokumentáciu knižnice komponentov a demo stránku

- **Rozloženie na celú obrazovku**: Absolútne umiestnenie vyplní zobrazovaciu oblasť so vstavaným posúvaním a navigáciou.
- **Dynamické smerovanie**: Analyzujte cesty URL a dynamicky načítavajte ukážky komponentov alebo domovské stránky.
- **Viacjazyčné**: Odpovedajte a aktualizujte navigačný panel a kópiu stránky v reálnom čase.
- **Prepínanie stavu**: Automaticky udržiava triedy štýlu načítania a domovskej stránky.

## Použite demo

```html
<c-webc-index></c-webc-index>
```

## štýlová trieda

Komponent v sebe prepne nasledujúce názvy tried:

- `Ing`: Načítanie trasy alebo analýza jazykového balíka.
- `A`: Momentálne na domovskej stránke.
