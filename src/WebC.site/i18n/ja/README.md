# コンポーネント ライブラリのドキュメントとデモ サイトをホストするルーティング ポータル

- **全画面レイアウト**: 組み込みのスクロールとナビゲーションを備えた絶対位置がビューポート全体に表示されます。
- **動的ルーティング**: URL パスを解析し、コンポーネントのデモまたはホームページを動的に読み込みます。
- **多言語**: ナビゲーション バーとページ コピーをリアルタイムで応答および更新します。
- **ステータス切り替え**: 読み込みとホームページのスタイル クラスを自動的に維持します。

## デモを使用する

```html
<c-webc-index></c-webc-index>
```

## スタイルクラス

コンポーネントは、それ自体内で次のクラス名を切り替えます。

- `Ing`: ルートの読み込みまたは言語パックの解析。
- `A`: 現在ホームページにあります。
