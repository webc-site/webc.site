# Dokumentasi perpustakaan komponen hosting portal lan situs demo

- ** Tata letak layar wutuh **: Posisi absolut ngisi viewport, kanthi gulung lan navigasi sing dibangun.
- **Dynamic Routing**: Parse URL path lan mbosenke mbukak demo komponen utawa homepages.
- **Multi-basa**: Tanggapi lan nganyari garis navigasi lan salinan kaca kanthi wektu nyata.
- **Ngalih status**: Kanthi otomatis njaga kelas gaya loading lan homepage.

## Gunakake demo

```html
<c-webc-index></c-webc-index>
```

## kelas gaya

Komponen kasebut bakal ngganti jeneng kelas ing ngisor iki:

- `Ing`: Pemuatan rute utawa parsing paket basa.
- `A`: Saiki ing homepage.
