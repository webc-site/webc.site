# Usmerjevalni portal, ki gosti dokumentacijo knjižnice komponent in predstavitveno mesto

- **Celozaslonska postavitev**: Absolutno pozicioniranje zapolni vidno polje z vgrajenim pomikanjem in navigacijo.
- **Dinamično usmerjanje**: razčlenite poti URL-jev in dinamično naložite predstavitve komponent ali domače strani.
- **Večjezičnost**: Odzovite se in posodobite navigacijsko vrstico in kopijo strani v realnem času.
- **Preklapljanje stanja**: Samodejno vzdržuje slogovne razrede nalaganja in domače strani.

## Uporabite demo

```html
<c-webc-index></c-webc-index>
```

## stilski razred

Komponenta bo znotraj sebe zamenjala naslednja imena razredov:

- `Ing`: Nalaganje poti ali razčlenjevanje jezikovnega paketa.
- `A`: Trenutno na domači strani.
