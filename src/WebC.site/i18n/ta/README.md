# ரூட்டிங் போர்டல் ஹோஸ்டிங் கூறு நூலக ஆவணங்கள் மற்றும் டெமோ தளம்

- **முழுத் திரை தளவமைப்பு**: முழுமையான நிலைப்படுத்தல், உள்ளமைக்கப்பட்ட ஸ்க்ரோலிங் மற்றும் வழிசெலுத்தலுடன் காட்சிப் பகுதியை நிரப்புகிறது.
- **டைனமிக் ரூட்டிங்**: URL பாதைகளை அலசவும் மற்றும் கூறு டெமோக்கள் அல்லது முகப்புப் பக்கங்களை மாறும் வகையில் ஏற்றவும்.
- **பல மொழி**: உண்மையான நேரத்தில் வழிசெலுத்தல் பட்டி மற்றும் பக்க நகலுக்கு பதிலளித்து புதுப்பிக்கவும்.
- **நிலை மாறுதல்**: ஏற்றுதல் மற்றும் முகப்புப் பக்கத்தின் நடை வகுப்புகளைத் தானாகப் பராமரிக்கவும்.
