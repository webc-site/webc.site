# Maršrutizavimo portalo prieglobos komponentų bibliotekos dokumentacija ir demonstracinė svetainė

- **Viso ekrano išdėstymas**: absoliuti padėties nustatymas užpildo peržiūros sritį su integruotu slinkimu ir naršymu.
- **Dinaminis maršruto parinkimas**: analizuokite URL kelius ir dinamiškai įkelkite komponentų demonstracines versijas arba pagrindinius puslapius.
- **Kelių kalbų**: atsakykite ir atnaujinkite naršymo juostą bei puslapio kopiją realiuoju laiku.
- **Būsenos perjungimas**: automatiškai palaikykite įkėlimo ir pagrindinio puslapio stiliaus klases.

## Naudokite demonstracinę versiją

```html
<c-webc-index></c-webc-index>
```

## stiliaus klasė

Komponentas pakeis šiuos klasių pavadinimus savyje:

- `Ing`: maršruto įkėlimas arba kalbos paketo analizavimas.
- `A`: šiuo metu yra pagrindiniame puslapyje.
