# রাউটিং পোর্টাল হোস্টিং কম্পোনেন্ট লাইব্রেরি ডকুমেন্টেশন এবং ডেমো সাইট

- **ফুল স্ক্রিন লেআউট**: বিল্ট-ইন স্ক্রোলিং এবং নেভিগেশন সহ সম্পূর্ণ পজিশনিং ভিউপোর্টকে পূর্ণ করে।
- **ডাইনামিক রাউটিং**: ইউআরএল পাথ পার্স করুন এবং ডাইনামিকভাবে কম্পোনেন্ট ডেমো বা হোমপেজ লোড করুন।
- **মাল্টি-ল্যাঙ্গুয়েজ**: রিয়েল টাইমে নেভিগেশন বার এবং পেজ কপি রিপ্লাই এবং আপডেট করুন।
- **স্থিতি স্যুইচিং**: স্বয়ংক্রিয়ভাবে লোডিং এবং হোমপেজের স্টাইল ক্লাস বজায় রাখুন।
