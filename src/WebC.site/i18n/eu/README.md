# Bideratze-atariaren osagai liburutegiaren dokumentazioa eta demo-gunea

- **Pantaila osoko diseinua**: kokapen absolutuak ikuspegia betetzen du, korritze eta nabigazio barneratuta.
- **Bidatze dinamikoa**: analizatu URL bideak eta kargatu dinamikoki osagaien demoak edo hasierako orriak.
- **Hizkuntza anitzeko**: erantzun eta eguneratu nabigazio-barra eta orriaren kopia denbora errealean.
- **Egoera aldatzea**: automatikoki mantendu kargatzeko eta hasierako orriaren estilo-klaseak.

## Erabili demoa

```html
<c-webc-index></c-webc-index>
```

## estilo klasea

Osagaiak klase-izen hauek aldatuko ditu bere baitan:

- `Ing`: ibilbidea kargatzea edo hizkuntza-paketearen analisia.
- `A`: Une honetan hasierako orrian.
