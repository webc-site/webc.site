# Dokumentacja biblioteki komponentów hostingu portalu routingu i witryna demonstracyjna

- **Układ pełnoekranowy**: Pozycjonowanie bezwzględne wypełnia rzutnię, z wbudowanym przewijaniem i nawigacją.
- **Routing dynamiczny**: analizuj ścieżki URL i dynamicznie ładuj prezentacje komponentów lub strony główne.
- **Wielojęzyczny**: Odpowiadaj i aktualizuj pasek nawigacyjny oraz kopię strony w czasie rzeczywistym.
- **Przełączanie statusu**: Automatycznie utrzymuj klasy stylów ładowania i strony głównej.

## Skorzystaj z wersji demonstracyjnej

```html
<c-webc-index></c-webc-index>
```

## klasa stylu

Komponent będzie przełączał w sobie następujące nazwy klas:

- `Ing`: Ładowanie trasy lub analizowanie pakietu językowego.
- `A`: Obecnie na stronie głównej.
