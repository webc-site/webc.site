# Portail de routage hébergeant la documentation de la bibliothèque de composants et le site de démonstration

- **Mise en page plein écran** : le positionnement absolu remplit la fenêtre d'affichage, avec défilement et navigation intégrés.
- **Routage dynamique** : analysez les chemins d'URL et chargez dynamiquement des démos de composants ou des pages d'accueil.
- **Multilingue** : Répondez et mettez à jour la barre de navigation et la copie de page en temps réel.
- **Changement de statut** : conservez automatiquement les classes de style de chargement et de page d'accueil.

## Utilisez la démo

```html
<c-webc-index></c-webc-index>
```

## cours de style

Le composant changera les noms de classe suivants en lui-même :

- `Ing` : chargement de la route ou analyse du module linguistique.
- `A` : actuellement sur la page d'accueil.
