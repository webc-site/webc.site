# Портал за рутирање који хостује документацију библиотеке компоненти и демо сајт

- **Изглед преко целог екрана**: Апсолутно позиционирање испуњава оквир за приказ, са уграђеним померањем и навигацијом.
- **Динамичко рутирање**: Парсирајте УРЛ путање и динамички учитајте демонстрације компоненти или почетне странице.
- **Вишејезично**: Одговорите и ажурирајте траку за навигацију и копију странице у реалном времену.
- **Промена статуса**: Аутоматски одржавајте класе стилова за учитавање и почетну страницу.

## Користите демо

```html
<c-webc-index></c-webc-index>
```

## стилска класа

Компонента ће изменити следећа имена класа унутар себе:

- <б ид="1">: Учитавање руте или рашчлањивање језичког пакета.
- <б ид="1">: Тренутно на почетној страници.
