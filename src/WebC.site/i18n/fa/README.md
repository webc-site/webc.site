# مسیریابی پورتال میزبانی اسناد کتابخانه جزء و سایت آزمایشی

- **طراحی تمام صفحه**: موقعیت یابی مطلق با اسکرول و پیمایش داخلی، درگاه دید را پر می کند.
- **مسیریابی پویا**: مسیرهای URL را تجزیه کنید و به صورت پویا دموی مؤلفه یا صفحات اصلی بارگیری کنید.
- **چند زبانه**: نوار پیمایش و کپی صفحه را به صورت بلادرنگ پاسخ دهید و به روز کنید.
- **تغییر وضعیت**: به طور خودکار کلاس های سبک بارگیری و صفحه اصلی را حفظ می کند.

## از دمو استفاده کنید

```html
<c-webc-index></c-webc-index>
```

## کلاس سبک

کامپوننت نام کلاس های زیر را درون خود تغییر می دهد:

- `Ing`: بارگیری مسیر یا تجزیه بسته زبان.
- `A`: در حال حاضر در صفحه اصلی.
