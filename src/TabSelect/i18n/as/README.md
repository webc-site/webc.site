# ড্ৰপ-ডাউন নিৰ্বাচনৰ সৈতে টেব পৃষ্ঠা

- লেবেলত অভিযোজিত প্ৰস্থ ড্ৰপ-ডাউন নিৰ্ব্বাচন বাকচ (`<c-select>`) এম্বেড কৰাটো সমৰ্থন কৰে, যি কেৱল প্ৰদৰ্শিত হ'ব যেতিয়া বৰ্তমান টেব পৃষ্ঠা সক্ৰিয় কৰা হয়।
- গাঁথনিগত ধাৰ্য্যকৰণ: সন্তান উপাদানে `nav > a` লেবেল নেভিগেচন হিচাপে ব্যৱহাৰ কৰে, আৰু প্ৰত্যক্ষ সন্তান উপাদান `b` এ বিষয়বস্তু অঞ্চল হিচাপে কাম কৰে।
- স্বয়ংক্ৰিয়ভাৱে `a[value]` টেগ, `a`ক `c-select` (গোটৰ মান বৈশিষ্ট্য আৰু নিৰ্বাচিত বিকল্প মানৰ ওপৰত ভিত্তি কৰি) আৰু `b[slot]` বিষয়বস্তু ন'ডৰ সৈতে সংযুক্ত কৰে।
- বৰ্তমানে সক্ৰিয় কৰা বস্তুটোক `key` বৈশিষ্ট্যৰ যোগেদি `localStorage` লৈ স্বয়ংক্ৰিয়ভাৱে ৰখা সমৰ্থন কৰে।
- প্ৰতিটো গোটৰ বাবে ঐতিহাসিক উপ-নিৰ্বাচনসমূহৰ স্বয়ংক্ৰিয় সংৰক্ষণ আৰু পুনৰুদ্ধাৰ সমৰ্থন কৰে `key + '/' + tab` ব্যৱহাৰ কৰি `localStorage` ।
- `value` বৈশিষ্ট্যৰ যোগেদি সক্ৰিয় টেব ধাৰ্য্য কৰা সমৰ্থন কৰে।

## ডেমো ব্যৱহাৰ কৰক

```html
<c-tab-select key="my-tab-select-key" value="language/js">
  <nav>
    <a value="language">
      <span>ভাষা</span>
      <c-select>
        <option value="js">JavaScript</option>
        <option value="rs">Rust</option>
      </c-select>
    </a>
    <a value="os">
      <span>পদ্ধতি</span>
      <c-select>
        <option value="mac">macOS</option>
        <option value="win">Windows</option>
      </c-select>
    </a>
  </nav>
  <b slot="language">
    <b slot="js">জাভাস্ক্রিপ্ট হৈছে এটা লিপি ভাষা</b>
    <b slot="rs">Rust এটা চিস্টেম-স্তৰৰ প্ৰগ্ৰেমিং ভাষা</b>
  </b>
  <b slot="os">
    <b slot="mac">macOS এটা ইউনিক্স-ভিত্তিক গ্ৰাফিকেল অপাৰেটিং চিস্টেম</b>
    <b slot="win">উইণ্ড’জ হৈছে মাইক্ৰ’ছফ্টৰ দ্বাৰা নিৰ্মিত এটা অপাৰেটিং চিষ্টেম</b>
  </b>
</c-tab-select>
```

## সম্পত্তি

- `key`: localStorage স্থায়িত্বৰ বাবে ব্যৱহৃত চাবিৰ নাম
- `value`: বৰ্তমানে নিৰ্বাচিত মান

## কাষ্টম ইভেন্ট

- `change`: টেব সলনি কৰাৰ সময়ত ট্ৰিগাৰ কৰা হয়, `e.value` হৈছে বৰ্তমানে সক্ৰিয় কৰা বিকল্পৰ `value` মান।
