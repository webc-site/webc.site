# ჩანართის გვერდი ჩამოსაშლელი შერჩევით

- მხარს უჭერს ადაპტაციური სიგანის ჩამოსაშლელი შერჩევის ველის (`<c-select>`) ჩაშენებას ლეიბლში, რომელიც მხოლოდ მაშინ გამოჩნდება, როდესაც ჩანართის მიმდინარე გვერდი გააქტიურდება.
- სტრუქტურული სპეციფიკაცია: ბავშვის ელემენტი იყენებს `nav > a` როგორც ლეიბლის ნავიგაციას, ხოლო პირდაპირი შვილობილი ელემენტი `b` ემსახურება კონტენტის ზონას.
- ავტომატურად აკავშირებს `a[value]` ტეგს, `a` `c-select`-თან (ჯგუფის მნიშვნელობის ატრიბუტზე და არჩეული ვარიანტის მნიშვნელობაზე) და `b[slot]` შიგთავსის კვანძთან.
- მხარს უჭერს ამჟამად გააქტიურებული ელემენტის `localStorage` ავტომატურ შენარჩუნებას `key` ატრიბუტის მეშვეობით.
- მხარს უჭერს ისტორიული ქვეარჩევნების ავტომატურ შენახვას და აღდგენას თითოეული ჯგუფისთვის `localStorage` `key + '/' + tab`-ის გამოყენებით.
- მხარს უჭერს აქტიური ჩანართის მითითებას `value` ატრიბუტის მეშვეობით.

## გამოიყენეთ დემო ვერსია

```html
<c-tab-select key="my-tab-select-key" value="language/js">
  <nav>
    <a value="language">
      <span>ენა</span>
      <c-select>
        <option value="js">JavaScript</option>
        <option value="rs">Rust</option>
      </c-select>
    </a>
    <a value="os">
      <span>სისტემა</span>
      <c-select>
        <option value="mac">macOS</option>
        <option value="win">Windows</option>
      </c-select>
    </a>
  </nav>
  <b slot="language">
    <b slot="js">JavaScript არის სკრიპტირების ენა</b>
    <b slot="rs">Rust არის სისტემის დონის პროგრამირების ენა</b>
  </b>
  <b slot="os">
    <b slot="mac">macOS არის Unix-ზე დაფუძნებული გრაფიკული ოპერაციული სისტემა</b>
    <b slot="win">Windows არის ოპერაციული სისტემა, რომელიც შემუშავებულია Microsoft-ის მიერ</b>
  </b>
</c-tab-select>
```

## ქონება

- `key`: გასაღების სახელი გამოიყენება localStorage მდგრადობისთვის
- `value`: ამჟამად არჩეული მნიშვნელობა

## მორგებული ღონისძიებები

- `change`: ამოქმედდება ჩანართების გადართვისას, `e.value` არის ამჟამად გააქტიურებული ვარიანტის `value` მნიშვნელობა.
