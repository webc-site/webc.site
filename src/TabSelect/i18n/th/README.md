# หน้าแท็บพร้อมตัวเลือกแบบเลื่อนลง

- รองรับการฝังช่องการเลือกแบบเลื่อนลงความกว้างที่ปรับเปลี่ยนได้ (`<c-select>`) ในป้ายกำกับ ซึ่งจะแสดงเฉพาะเมื่อเปิดใช้งานหน้าแท็บปัจจุบันเท่านั้น
- ข้อกำหนดโครงสร้าง: องค์ประกอบลูกใช้ `nav > a` เป็นการนำทางป้ายกำกับ และองค์ประกอบลูกโดยตรง `b` ทำหน้าที่เป็นพื้นที่เนื้อหา
- เชื่อมโยงแท็ก `a[value]`, `a` กับ `c-select` โดยอัตโนมัติ (ขึ้นอยู่กับแอตทริบิวต์ค่าของกลุ่มและค่าตัวเลือกที่เลือก) และโหนดเนื้อหา `b[slot]`
- รองรับการคงรายการที่เปิดใช้งานในปัจจุบันไว้โดยอัตโนมัติไปยัง `localStorage` ผ่านแอตทริบิวต์ `key`
- รองรับการบันทึกและกู้คืนการเลือกย่อยในอดีตโดยอัตโนมัติสำหรับแต่ละกลุ่มเพื่อ `localStorage` โดยใช้ `key + '/' + tab`
- รองรับการระบุแท็บที่ใช้งานอยู่ผ่านแอตทริบิวต์ `value`

## ใช้การสาธิต

```html
<c-tab-select key="my-tab-select-key" value="language/js">
  <nav>
    <a value="language">
      <span>ภาษา</span>
      <c-select>
        <option value="js">JavaScript</option>
        <option value="rs">Rust</option>
      </c-select>
    </a>
    <a value="os">
      <span>ระบบ</span>
      <c-select>
        <option value="mac">macOS</option>
        <option value="win">Windows</option>
      </c-select>
    </a>
  </nav>
  <b slot="language">
    <b slot="js">JavaScript เป็นภาษาสคริปต์</b>
    <b slot="rs">Rust เป็นภาษาโปรแกรมระดับระบบ</b>
  </b>
  <b slot="os">
    <b slot="mac">macOS เป็นระบบปฏิบัติการกราฟิกที่ใช้ Unix</b>
    <b slot="win">Windows เป็นระบบปฏิบัติการที่พัฒนาโดย Microsoft</b>
  </b>
</c-tab-select>
```

## คุณสมบัติ

- `key`: ชื่อคีย์ที่ใช้สำหรับการคงอยู่ของ localStorage
- `value`: ค่าที่เลือกในปัจจุบัน

## เหตุการณ์ที่กำหนดเอง

- `change`: ทริกเกอร์เมื่อสลับแท็บ `e.value` คือค่า `value` ของตัวเลือกที่เปิดใช้งานในปัจจุบัน
