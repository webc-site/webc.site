# صفحه برگه با انتخاب کشویی

- از جاسازی کادر انتخاب کشویی عرض تطبیقی ​​(`<c-select>`) در برچسب پشتیبانی می کند، که فقط زمانی نمایش داده می شود که صفحه برگه فعلی فعال شود.
- مشخصات ساختاری: عنصر فرزند از `nav > a` به عنوان پیمایش برچسب استفاده می کند و عنصر فرزند مستقیم `b` به عنوان ناحیه محتوا عمل می کند.
- به طور خودکار برچسب `a[value]`، `a` را با `c-select` (بر اساس ویژگی مقدار گروه و مقدار گزینه انتخاب شده) و گره محتوای `b[slot]` مرتبط می کند.
- از تداوم خودکار مورد فعال فعلی به `localStorage` از طریق ویژگی `key` پشتیبانی می کند.
- از ذخیره و بازیابی خودکار زیرمجموعه های تاریخی برای هر گروه در `localStorage` با استفاده از `key + '/' + tab` پشتیبانی می کند.
- از تعیین برگه فعال از طریق ویژگی `value` پشتیبانی می کند.

## از دمو استفاده کنید

```html
<c-tab-select key="my-tab-select-key" value="language/js">
  <nav>
    <a value="language">
      <span>زبان</span>
      <c-select>
        <option value="js">JavaScript</option>
        <option value="rs">Rust</option>
      </c-select>
    </a>
    <a value="os">
      <span>سیستم</span>
      <c-select>
        <option value="mac">macOS</option>
        <option value="win">Windows</option>
      </c-select>
    </a>
  </nav>
  <b slot="language">
    <b slot="js">جاوا اسکریپت یک زبان برنامه نویسی است</b>
    <b slot="rs">Rust یک زبان برنامه نویسی در سطح سیستم است</b>
  </b>
  <b slot="os">
    <b slot="mac">macOS یک سیستم عامل گرافیکی مبتنی بر یونیکس است</b>
    <b slot="win">ویندوز یک سیستم عامل است که توسط مایکروسافت توسعه یافته است</b>
  </b>
</c-tab-select>
```

## دارایی

- `key`: نام کلید مورد استفاده برای ماندگاری localStorage
- `value`: مقدار در حال حاضر انتخاب شده است

## رویدادهای سفارشی

- `change`: هنگام تعویض برگه ها فعال می شود، `e.value` مقدار `value` گزینه فعال فعلی است.
