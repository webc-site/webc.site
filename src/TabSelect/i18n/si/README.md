# පතන තේරීම සහිත ටැබ් පිටුව

- වත්මන් ටැබ් පිටුව සක්‍රිය වූ විට පමණක් සංදර්ශණය වන ලේබලයේ අනුවර්තී පළල පතන තේරීම් කොටුව (`<c-select>`) කාවැද්දීම සඳහා සහය දක්වයි.
- ව්‍යුහාත්මක පිරිවිතර: ළමා මූලද්‍රව්‍යය ලේබල සංචාලනය ලෙස `nav > a` භාවිතා කරන අතර සෘජු ළමා මූලද්‍රව්‍යය `b` අන්තර්ගත ප්‍රදේශය ලෙස ක්‍රියා කරයි.
- ස්වයංක්‍රීයව `a[value]` ටැගය, `a` `c-select` (සමූහයේ අගය ගුණාංගය සහ තෝරාගත් විකල්ප අගය මත පදනම්ව) සහ `b[slot]` අන්තර්ගත නෝඩය සමඟ සම්බන්ධ කරයි.
- දැනට සක්‍රිය කර ඇති අයිතමය `localStorage` වෙත `key` ගුණාංගය හරහා ස්වයංක්‍රීයව පැවතීමට සහය දක්වයි.
- එක් එක් කණ්ඩායම සඳහා `localStorage` වෙත `key + '/' + tab` භාවිතයෙන් ඉතිහාස උප තේරීම් ස්වයංක්‍රීයව සුරැකීමට සහ ප්‍රතිසාධන කිරීමට සහය දක්වයි.
- `value` ගුණාංගය හරහා සක්‍රිය ටැබය නියම කිරීමට සහය දක්වයි.

## Demo භාවිතා කරන්න

```html
<c-tab-select key="my-tab-select-key" value="language/js">
  <nav>
    <a value="language">
      <span>භාෂාව</span>
      <c-select>
        <option value="js">JavaScript</option>
        <option value="rs">Rust</option>
      </c-select>
    </a>
    <a value="os">
      <span>පද්ධතිය</span>
      <c-select>
        <option value="mac">macOS</option>
        <option value="win">Windows</option>
      </c-select>
    </a>
  </nav>
  <b slot="language">
    <b slot="js">JavaScript යනු scripting language එකකි</b>
    <b slot="rs">රස්ට් යනු පද්ධති මට්ටමේ ක්‍රමලේඛන භාෂාවකි</b>
  </b>
  <b slot="os">
    <b slot="mac">macOS යනු Unix මත පදනම් වූ චිත්‍රක මෙහෙයුම් පද්ධතියකි</b>
    <b slot="win">වින්ඩෝස් යනු මයික්‍රොසොෆ්ට් විසින් නිපදවන ලද මෙහෙයුම් පද්ධතියකි</b>
  </b>
</c-tab-select>
```

## දේපල

- `key`: localStorage කල්පවත්නා සඳහා භාවිතා කරන ප්‍රධාන නම
- `value`: දැනට තෝරාගත් අගය

## අභිරුචි සිදුවීම්

- `change`: ටැබ් මාරු කිරීමේදී ක්‍රියාරම්භක, `e.value` යනු දැනට සක්‍රිය කර ඇති විකල්පයේ `value` අගයයි.
