# ଡ୍ରପ୍ ଡାଉନ୍ ଚୟନ ସହିତ ଟ୍ୟାବ୍ ପୃଷ୍ଠା |

- ଲେବଲ୍ ରେ ଆଡାପ୍ଟିଭ୍ ଓସାର ଡ୍ରପ୍ ଡାଉନ୍ ଚୟନ ବାକ୍ସ (`<c-select>`) ଏମ୍ବେଡିଂକୁ ସମର୍ଥନ କରେ, ଯାହା କେବଳ ସାମ୍ପ୍ରତିକ ଟ୍ୟାବ୍ ପୃଷ୍ଠା ସକ୍ରିୟ ହେବାପରେ ପ୍ରଦର୍ଶିତ ହେବ |
- ଗଠନମୂଳକ ନିର୍ଦ୍ଦିଷ୍ଟକରଣ: ଶିଶୁ ଉପାଦାନ ଲେବଲ୍ ନେଭିଗେସନ୍ ଭାବରେ `nav > a` ବ୍ୟବହାର କରେ, ଏବଂ ପ୍ରତ୍ୟକ୍ଷ ଶିଶୁ ଉପାଦାନ `b` ବିଷୟବସ୍ତୁ କ୍ଷେତ୍ର ଭାବରେ କାର୍ଯ୍ୟ କରେ |
- ସ୍ୱୟଂଚାଳିତ ଭାବରେ `a[value]` ଟ୍ୟାଗ୍, `a` `c-select` (ଗୋଷ୍ଠୀର ମୂଲ୍ୟ ଗୁଣ ଏବଂ ମନୋନୀତ ବିକଳ୍ପ ମୂଲ୍ୟ ଉପରେ ଆଧାର କରି) ଏବଂ `b[slot]` ବିଷୟବସ୍ତୁ ନୋଡ୍ ସହିତ ଜଡିତ |
- ସମ୍ପ୍ରତି ସକ୍ରିୟ ହୋଇଥିବା ଆଇଟମକୁ `localStorage` `key` ଆଟ୍ରିବ୍ୟୁଟ୍ ମାଧ୍ୟମରେ ସ୍ୱୟଂଚାଳିତ ଭାବରେ ସ୍ଥିର କରିବାକୁ ସମର୍ଥନ କରେ |
- ପ୍ରତ୍ୟେକ ଗୋଷ୍ଠୀ ପାଇଁ `localStorage` `key + '/' + tab` ବ୍ୟବହାର କରି historical ତିହାସିକ ଉପ-ଚୟନଗୁଡ଼ିକର ସ୍ୱୟଂଚାଳିତ ସଞ୍ଚୟ ଏବଂ ପୁନରୁଦ୍ଧାରକୁ ସମର୍ଥନ କରେ |
- `value` ଆଟ୍ରିବ୍ୟୁଟ୍ ମାଧ୍ୟମରେ ସକ୍ରିୟ ଟ୍ୟାବ୍ ନିର୍ଦ୍ଦିଷ୍ଟ କରିବାକୁ ସମର୍ଥନ କରେ |

## ଡେମୋ ବ୍ୟବହାର କରନ୍ତୁ |

```html
<c-tab-select key="my-tab-select-key" value="language/js">
  <nav>
    <a value="language">
      <span>ଭାଷା</span>
      <c-select>
        <option value="js">JavaScript</option>
        <option value="rs">Rust</option>
      </c-select>
    </a>
    <a value="os">
      <span>ସିଷ୍ଟମ୍</span>
      <c-select>
        <option value="mac">macOS</option>
        <option value="win">Windows</option>
      </c-select>
    </a>
  </nav>
  <b slot="language">
    <b slot="js">ଜାଭାସ୍କ୍ରିପ୍ଟ ହେଉଛି ଏକ ସ୍କ୍ରିପ୍ଟିଂ ଭାଷା |</b>
    <b slot="rs">ରୁଷ୍ଟ ହେଉଛି ଏକ ସିଷ୍ଟମ୍ ସ୍ତରୀୟ ପ୍ରୋଗ୍ରାମିଂ ଭାଷା |</b>
  </b>
  <b slot="os">
    <b slot="mac">ମାକୋସ୍ ହେଉଛି ଏକ ୟୁନିକ୍ସ-ଆଧାରିତ ଗ୍ରାଫିକାଲ୍ ଅପରେଟିଂ ସିଷ୍ଟମ୍ |</b>
    <b slot="win">ୱିଣ୍ଡୋଜ୍ ହେଉଛି ଏକ ଅପରେଟିଂ ସିଷ୍ଟମ୍ ଯାହା ମାଇକ୍ରୋସଫ୍ଟ ଦ୍ୱାରା ବିକଶିତ |</b>
  </b>
</c-tab-select>
```

## ସମ୍ପତ୍ତି

- `key`: ସ୍ଥାନୀୟ ଷ୍ଟୋରେଜ୍ ସ୍ଥିରତା ପାଇଁ ବ୍ୟବହୃତ ମୁଖ୍ୟ ନାମ |
- `value`: ସମ୍ପ୍ରତି ମନୋନୀତ ମୂଲ୍ୟ |

## କଷ୍ଟମ୍ ଇଭେଣ୍ଟଗୁଡିକ |

- `change`: ଟ୍ୟାବ୍ ସୁଇଚ୍ କରିବା ସମୟରେ ଟ୍ରିଗର୍ ହୋଇଗଲା, `e.value` ହେଉଛି ସମ୍ପ୍ରତି ସକ୍ରିୟ ହୋଇଥିବା ବିକଳ୍ପର `value` ମୂଲ୍ୟ |
