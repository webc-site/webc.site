# Вкладка зі спадним вибором

- Підтримує вбудовування спадного поля вибору адаптивної ширини (`<c-select>`) у мітку, яке відображатиметься, лише якщо активовано поточну сторінку вкладки.
- Структурна специфікація: дочірній елемент використовує `nav > a` як навігацію по мітці, а прямий дочірній елемент `b` служить як область вмісту.
- Автоматично пов’язує тег `a[value]`, `a` з `c-select` (на основі атрибута значення групи та вибраного значення опції) і вузол вмісту `b[slot]`.
- Підтримує автоматичне збереження поточного активованого елемента в `localStorage` через атрибут `key`.
- Підтримує автоматичне збереження та відновлення історичних підвибрань для кожної групи в `localStorage` за допомогою `key + '/' + tab`.
- Підтримує вказівку активної вкладки через атрибут `value`.

## Використовуйте демо

```html
<c-tab-select key="my-tab-select-key" value="language/js">
  <nav>
    <a value="language">
      <span>мова</span>
      <c-select>
        <option value="js">JavaScript</option>
        <option value="rs">Rust</option>
      </c-select>
    </a>
    <a value="os">
      <span>система</span>
      <c-select>
        <option value="mac">macOS</option>
        <option value="win">Windows</option>
      </c-select>
    </a>
  </nav>
  <b slot="language">
    <b slot="js">JavaScript — це мова сценаріїв</b>
    <b slot="rs">Rust — це мова програмування системного рівня</b>
  </b>
  <b slot="os">
    <b slot="mac">macOS — це графічна операційна система на основі Unix</b>
    <b slot="win">Windows — це операційна система, розроблена компанією Microsoft</b>
  </b>
</c-tab-select>
```

## власність

- `key`: назва ключа, що використовується для збереження локального сховища
- `value`: поточне вибране значення

## Спеціальні події

- `change`: активується під час перемикання вкладок, `e.value` це значення `value` поточної активованої опції.
