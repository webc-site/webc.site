# Ներդիրի էջ բացվող ընտրությամբ

- Աջակցում է հարմարվողական լայնության բացվող ընտրության տուփի (`<c-select>`) ներդրմանը պիտակի մեջ, որը կցուցադրվի միայն ընթացիկ ներդիրի էջն ակտիվացնելու դեպքում:
- Կառուցվածքային ճշգրտում. երեխայի տարրը օգտագործում է `nav > a` որպես պիտակի նավարկություն, իսկ ուղղակի երեխա տարրը `b` ծառայում է որպես բովանդակության տարածք:
- Ավտոմատ կերպով կապում է `a[value]` թեգը, `a` `c-select`-ը (հիմնված խմբի արժեքի հատկանիշի և ընտրված տարբերակի արժեքի վրա) և `b[slot]` բովանդակության հանգույցի հետ:
- Աջակցում է այս պահին ակտիվացված տարրի ինքնաբերաբար պահպանումը `localStorage` `key` հատկանիշի միջոցով:
- Աջակցում է յուրաքանչյուր խմբի պատմական ենթընտրությունների ավտոմատ պահպանումն ու վերականգնումը `localStorage`՝ օգտագործելով `key + '/' + tab`:
- Աջակցում է ակտիվ ներդիրի սահմանմանը `value` հատկանիշի միջոցով:

## Օգտագործեք ցուցադրությունը

```html
<c-tab-select key="my-tab-select-key" value="language/js">
  <nav>
    <a value="language">
      <span>լեզուն</span>
      <c-select>
        <option value="js">JavaScript</option>
        <option value="rs">Rust</option>
      </c-select>
    </a>
    <a value="os">
      <span>համակարգ</span>
      <c-select>
        <option value="mac">macOS</option>
        <option value="win">Windows</option>
      </c-select>
    </a>
  </nav>
  <b slot="language">
    <b slot="js">JavaScript-ը սկրիպտավորման լեզու է</b>
    <b slot="rs">Rust-ը համակարգերի մակարդակի ծրագրավորման լեզու է</b>
  </b>
  <b slot="os">
    <b slot="mac">macOS-ը Unix-ի վրա հիմնված գրաֆիկական օպերացիոն համակարգ է</b>
    <b slot="win">Windows-ը Microsoft-ի կողմից մշակված օպերացիոն համակարգ է</b>
  </b>
</c-tab-select>
```

## սեփականություն

- `key`. բանալի անուն, որն օգտագործվում է localStorage-ի կայունության համար
- `value`՝ ներկայումս ընտրված արժեք
