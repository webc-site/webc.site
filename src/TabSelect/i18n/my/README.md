# drop-down ရွေးချယ်မှုနှင့်အတူ တဘ်စာမျက်နှာ

- လက်ရှိတက်ဘ်စာမျက်နှာကို အသက်သွင်းထားသည့်အခါမှသာ ပြသနိုင်မည်ဖြစ်ပြီး အညွှန်းထဲတွင် လိုက်လျောညီထွေရှိသော အကျယ်ပြစက်ရွေးချယ်မှုအကွက် (`<c-select>`) ကို ထည့်သွင်းခြင်းကို ပံ့ပိုးပေးပါသည်။
- ဖွဲ့စည်းတည်ဆောက်ပုံ သတ်မှတ်ချက်- ကလေးဒြပ်စင်သည် `nav > a` အညွှန်းလမ်းညွှန်မှုအဖြစ် အသုံးပြုပြီး တိုက်ရိုက် ကလေးဒြပ်စင် `b` အကြောင်းအရာ ဧရိယာအဖြစ် ဆောင်ရွက်ပါသည်။
- `a[value]`တဂ်၊ `a` (အဖွဲ့၏တန်ဖိုး ရည်ညွှန်းချက်နှင့် ရွေးချယ်ထားသည့် ရွေးချယ်မှုတန်ဖိုးအပေါ် အခြေခံ၍) နှင့် `b[slot]` အကြောင်းအရာ ဆုံမှတ်ကို အလိုအလျောက် ဆက်စပ်ပေးပါသည်။
- `key` ရည်ညွှန်းချက်မှတစ်ဆင့် လက်ရှိအသုံးပြုနေသော အရာအား `localStorage` တွင် အလိုအလျောက် ဆက်လက်တည်မြဲရန် ပံ့ပိုးပေးပါသည်။
- `localStorage` `key + '/' + tab`ကို အသုံးပြု၍ အုပ်စုတစ်ခုစီအတွက် သမိုင်းဝင်ရွေးချယ်မှုခွဲများကို အလိုအလျောက်သိမ်းဆည်းခြင်းနှင့် ပြန်လည်ရယူခြင်းကို ပံ့ပိုးပေးသည်။
- `value` ရည်ညွှန်းချက်အားဖြင့် အသုံးပြုနေသည့် တက်ဘ်အား သတ်မှတ်ခြင်းကို ပံ့ပိုးပေးသည်။

## ဒီမိုကိုသုံးပါ။

```html
<c-tab-select key="my-tab-select-key" value="language/js">
  <nav>
    <a value="language">
      <span>ဘာသာစကား</span>
      <c-select>
        <option value="js">JavaScript</option>
        <option value="rs">Rust</option>
      </c-select>
    </a>
    <a value="os">
      <span>စနစ်</span>
      <c-select>
        <option value="mac">macOS</option>
        <option value="win">Windows</option>
      </c-select>
    </a>
  </nav>
  <b slot="language">
    <b slot="js">JavaScript သည် scripting language တစ်ခုဖြစ်သည်။</b>
    <b slot="rs">Rust သည် စနစ်အဆင့် ပရိုဂရမ်းမင်းဘာသာစကားတစ်ခုဖြစ်သည်။</b>
  </b>
  <b slot="os">
    <b slot="mac">macOS သည် Unix-based graphical operating system တစ်ခုဖြစ်သည်။</b>
    <b slot="win">Windows သည် Microsoft မှထုတ်လုပ်သော operating system တစ်ခုဖြစ်သည်။</b>
  </b>
</c-tab-select>
```

## ပစ္စည်းဥစ္စာ

- `key`: localStorage မြဲမြံမှုအတွက် အသုံးပြုသော သော့အမည်
- `value`- လက်ရှိရွေးချယ်ထားသော တန်ဖိုး

## စိတ်ကြိုက်ပွဲများ

- `change`- တက်ဘ်များကို ပြောင်းသည့်အခါ အစပျိုးခဲ့သည်၊ `e.value`သည် လက်ရှိဖွင့်ထားသည့် ရွေးချယ်မှု၏ `value`တန်ဖိုးဖြစ်သည်။
