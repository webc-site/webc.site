# ڈراپ ڈاؤن انتخاب کے ساتھ ٹیب صفحہ

- لیبل میں انکولی چوڑائی کے ڈراپ ڈاؤن سلیکشن باکس (`<c-select>`) کو سرایت کرنے کی حمایت کرتا ہے، جو صرف موجودہ ٹیب کا صفحہ فعال ہونے پر ظاہر ہوگا۔
- ساختی تفصیلات: چائلڈ عنصر `nav > a` کو لیبل نیویگیشن کے طور پر استعمال کرتا ہے، اور براہ راست چائلڈ عنصر `b` مواد کے علاقے کے طور پر کام کرتا ہے۔
- خودکار طور پر `a[value]` ٹیگ، `a` کو `c-select` کے ساتھ منسلک کرتا ہے (گروپ کی قدر وصف اور منتخب کردہ آپشن ویلیو کی بنیاد پر) اور `b[slot]` مواد نوڈ۔
- `key` انتساب کے ذریعے `localStorage` پر فی الحال فعال کردہ آئٹم کو خود بخود برقرار رکھنے کی حمایت کرتا ہے۔
- `key + '/' + tab` کا استعمال کرتے ہوئے ہر گروپ کے لیے تاریخی ذیلی انتخاب کو `localStorage` میں خودکار طور پر محفوظ کرنے اور بحال کرنے کی حمایت کرتا ہے۔
- `value` انتساب کے ذریعے فعال ٹیب کی وضاحت کرنے کی حمایت کرتا ہے۔

## ڈیمو استعمال کریں۔

```html
<c-tab-select key="my-tab-select-key" value="language/js">
  <nav>
    <a value="language">
      <span>زبان</span>
      <c-select>
        <option value="js">JavaScript</option>
        <option value="rs">Rust</option>
      </c-select>
    </a>
    <a value="os">
      <span>نظام</span>
      <c-select>
        <option value="mac">macOS</option>
        <option value="win">Windows</option>
      </c-select>
    </a>
  </nav>
  <b slot="language">
    <b slot="js">JavaScript ایک اسکرپٹنگ زبان ہے۔</b>
    <b slot="rs">زنگ ایک نظام کی سطح کی پروگرامنگ زبان ہے۔</b>
  </b>
  <b slot="os">
    <b slot="mac">macOS یونکس پر مبنی گرافیکل آپریٹنگ سسٹم ہے۔</b>
    <b slot="win">ونڈوز ایک آپریٹنگ سسٹم ہے جسے مائیکروسافٹ نے تیار کیا ہے۔</b>
  </b>
</c-tab-select>
```

## جائیداد

- `key`: کلیدی نام لوکل سٹوریج استقامت کے لیے استعمال کیا جاتا ہے۔
- `value`: فی الحال منتخب قدر

## حسب ضرورت واقعات

- `change`: ٹیبز کو تبدیل کرتے وقت متحرک ہوتا ہے، `e.value` فی الحال فعال کردہ آپشن کی `value` قدر ہے۔
