# Страница картице са падајућим избором

- Подржава уграђивање падајућег оквира за избор прилагодљиве ширине (<б ид="1">) у ознаку, који ће бити приказан само када је тренутна страница картице активирана.
- Структурна спецификација: подређени елемент користи <б ид="1"> као навигацију ознаком, а директни подређени елемент <б ид="2"> служи као област садржаја.
- Аутоматски повезује ознаку <б ид="1">, <б ид="3"> са <б ид="2"> (на основу атрибута вредности групе и вредности изабране опције) и чвора садржаја <б ид="4">.
- Подржава аутоматско задржавање тренутно активиране ставке на <б ид="2"> преко атрибута <б ид="1">.
- Подржава аутоматско чување и враћање историјских подизбора за сваку групу на <б ид="2"> користећи <б ид="1">.
- Подржава одређивање активне картице преко атрибута <б ид="1">.

## имовине

- <б ид="1">: име кључа који се користи за постојаност лоцалСтораге
- <б ид="1">: тренутно изабрана вредност

## Прилагођени догађаји

- <б ид="1">: Покреће се приликом промене картица, <б ид="2"> је вредност <б ид="3"> тренутно активиране опције.
