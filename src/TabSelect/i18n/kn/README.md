# ಡ್ರಾಪ್-ಡೌನ್ ಆಯ್ಕೆಯೊಂದಿಗೆ ಟ್ಯಾಬ್ ಪುಟ

- ಲೇಬಲ್‌ನಲ್ಲಿ ಹೊಂದಾಣಿಕೆಯ ಅಗಲ ಡ್ರಾಪ್-ಡೌನ್ ಆಯ್ಕೆ ಬಾಕ್ಸ್ (`<c-select>`) ಎಂಬೆಡಿಂಗ್ ಅನ್ನು ಬೆಂಬಲಿಸುತ್ತದೆ, ಪ್ರಸ್ತುತ ಟ್ಯಾಬ್ ಪುಟವನ್ನು ಸಕ್ರಿಯಗೊಳಿಸಿದಾಗ ಮಾತ್ರ ಪ್ರದರ್ಶಿಸಲಾಗುತ್ತದೆ.
- ರಚನಾತ್ಮಕ ವಿವರಣೆ: ಚೈಲ್ಡ್ ಎಲಿಮೆಂಟ್ `nav > a` ಅನ್ನು ಲೇಬಲ್ ನ್ಯಾವಿಗೇಶನ್ ಆಗಿ ಬಳಸುತ್ತದೆ ಮತ್ತು ಡೈರೆಕ್ಟ್ ಚೈಲ್ಡ್ ಎಲಿಮೆಂಟ್ `b` ವಿಷಯ ಪ್ರದೇಶವಾಗಿ ಕಾರ್ಯನಿರ್ವಹಿಸುತ್ತದೆ.
- `a[value]` ಟ್ಯಾಗ್, `a` ಅನ್ನು `c-select` (ಗುಂಪಿನ ಮೌಲ್ಯ ಗುಣಲಕ್ಷಣ ಮತ್ತು ಆಯ್ಕೆಮಾಡಿದ ಆಯ್ಕೆಯ ಮೌಲ್ಯವನ್ನು ಆಧರಿಸಿ) ಮತ್ತು `b[slot]` ವಿಷಯ ನೋಡ್ ಅನ್ನು ಸ್ವಯಂಚಾಲಿತವಾಗಿ ಸಂಯೋಜಿಸುತ್ತದೆ.
- ಪ್ರಸ್ತುತ ಸಕ್ರಿಯವಾಗಿರುವ ಐಟಂ ಅನ್ನು `localStorage` ಗೆ `key` ಗುಣಲಕ್ಷಣದ ಮೂಲಕ ಸ್ವಯಂಚಾಲಿತವಾಗಿ ಬೆಂಬಲಿಸುತ್ತದೆ.
- `key + '/' + tab` ಅನ್ನು ಬಳಸಿಕೊಂಡು ಪ್ರತಿ ಗುಂಪಿಗೆ `localStorage` ಗೆ ಐತಿಹಾಸಿಕ ಉಪ-ಆಯ್ಕೆಗಳ ಸ್ವಯಂಚಾಲಿತ ಉಳಿಸುವಿಕೆ ಮತ್ತು ಮರುಸ್ಥಾಪನೆಯನ್ನು ಬೆಂಬಲಿಸುತ್ತದೆ.
- `value` ಗುಣಲಕ್ಷಣದ ಮೂಲಕ ಸಕ್ರಿಯ ಟ್ಯಾಬ್ ಅನ್ನು ನಿರ್ದಿಷ್ಟಪಡಿಸುವುದನ್ನು ಬೆಂಬಲಿಸುತ್ತದೆ.

## ಡೆಮೊ ಬಳಸಿ

```html
<c-tab-select key="my-tab-select-key" value="language/js">
  <nav>
    <a value="language">
      <span>ಭಾಷೆ</span>
      <c-select>
        <option value="js">JavaScript</option>
        <option value="rs">Rust</option>
      </c-select>
    </a>
    <a value="os">
      <span>ವ್ಯವಸ್ಥೆ</span>
      <c-select>
        <option value="mac">macOS</option>
        <option value="win">Windows</option>
      </c-select>
    </a>
  </nav>
  <b slot="language">
    <b slot="js">ಜಾವಾಸ್ಕ್ರಿಪ್ಟ್ ಒಂದು ಸ್ಕ್ರಿಪ್ಟಿಂಗ್ ಭಾಷೆಯಾಗಿದೆ</b>
    <b slot="rs">ರಸ್ಟ್ ಒಂದು ಸಿಸ್ಟಮ್-ಮಟ್ಟದ ಪ್ರೋಗ್ರಾಮಿಂಗ್ ಭಾಷೆಯಾಗಿದೆ</b>
  </b>
  <b slot="os">
    <b slot="mac">macOS ಯುನಿಕ್ಸ್ ಆಧಾರಿತ ಗ್ರಾಫಿಕಲ್ ಆಪರೇಟಿಂಗ್ ಸಿಸ್ಟಮ್ ಆಗಿದೆ</b>
    <b slot="win">ವಿಂಡೋಸ್ ಮೈಕ್ರೋಸಾಫ್ಟ್ ಅಭಿವೃದ್ಧಿಪಡಿಸಿದ ಆಪರೇಟಿಂಗ್ ಸಿಸ್ಟಮ್ ಆಗಿದೆ</b>
  </b>
</c-tab-select>
```

## ಆಸ್ತಿ

- `key`: ಲೋಕಲ್ ಸ್ಟೋರೇಜ್ ನಿರಂತರತೆಗಾಗಿ ಬಳಸಲಾದ ಪ್ರಮುಖ ಹೆಸರು
- `value`: ಪ್ರಸ್ತುತ ಆಯ್ಕೆಮಾಡಿದ ಮೌಲ್ಯ

## ಕಸ್ಟಮ್ ಘಟನೆಗಳು

- `change`: ಟ್ಯಾಬ್‌ಗಳನ್ನು ಬದಲಾಯಿಸುವಾಗ ಟ್ರಿಗರ್ ಮಾಡಲಾಗಿದೆ, `e.value` ಎಂಬುದು ಪ್ರಸ್ತುತ ಸಕ್ರಿಯವಾಗಿರುವ ಆಯ್ಕೆಯ `value` ಮೌಲ್ಯವಾಗಿದೆ.
