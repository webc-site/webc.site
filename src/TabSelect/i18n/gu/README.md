# ડ્રોપ-ડાઉન પસંદગી સાથે ટૅબ પૃષ્ઠ

- લેબલમાં અનુકૂલનશીલ પહોળાઈ ડ્રોપ-ડાઉન પસંદગી બોક્સ (`<c-select>`) એમ્બેડ કરવાનું સમર્થન કરે છે, જે વર્તમાન ટેબ પૃષ્ઠ સક્રિય થાય ત્યારે જ પ્રદર્શિત થશે.
- માળખાકીય સ્પષ્ટીકરણ: ચાઇલ્ડ એલિમેન્ટ `nav > a`નો લેબલ નેવિગેશન તરીકે ઉપયોગ કરે છે અને ડાયરેક્ટ ચાઇલ્ડ એલિમેન્ટ `b` સામગ્રી વિસ્તાર તરીકે સેવા આપે છે.
- `a[value]` ટેગ, `a` ને `c-select` (જૂથના મૂલ્ય વિશેષતા અને પસંદ કરેલ વિકલ્પ મૂલ્યના આધારે) અને `b[slot]` સામગ્રી નોડ સાથે આપમેળે સાંકળે છે.
- `key` વિશેષતા દ્વારા હાલમાં સક્રિય કરેલ આઇટમને `localStorage` પર આપમેળે ચાલુ રાખવાનું સમર્થન કરે છે.
- `key + '/' + tab` નો ઉપયોગ કરીને દરેક જૂથ માટે `localStorage` માં ઐતિહાસિક પેટા-પસંદગીઓને સ્વચાલિત સાચવવા અને પુનઃસ્થાપિત કરવાનું સમર્થન કરે છે.
- `value` એટ્રિબ્યુટ દ્વારા સક્રિય ટેબનો ઉલ્લેખ કરવાનું સમર્થન કરે છે.

## ડેમોનો ઉપયોગ કરો

```html
<c-tab-select key="my-tab-select-key" value="language/js">
  <nav>
    <a value="language">
      <span>ભાષા</span>
      <c-select>
        <option value="js">JavaScript</option>
        <option value="rs">Rust</option>
      </c-select>
    </a>
    <a value="os">
      <span>સિસ્ટમ</span>
      <c-select>
        <option value="mac">macOS</option>
        <option value="win">Windows</option>
      </c-select>
    </a>
  </nav>
  <b slot="language">
    <b slot="js">JavaScript એ સ્ક્રિપ્ટીંગ ભાષા છે</b>
    <b slot="rs">રસ્ટ એ સિસ્ટમ-સ્તરની પ્રોગ્રામિંગ ભાષા છે</b>
  </b>
  <b slot="os">
    <b slot="mac">macOS એ યુનિક્સ-આધારિત ગ્રાફિકલ ઓપરેટિંગ સિસ્ટમ છે</b>
    <b slot="win">વિન્ડોઝ એ માઇક્રોસોફ્ટ દ્વારા વિકસિત ઓપરેટિંગ સિસ્ટમ છે</b>
  </b>
</c-tab-select>
```

## મિલકત

- `key`: લોકલ સ્ટોરેજ દ્રઢતા માટે વપરાયેલ કી નામ
- `value`: હાલમાં પસંદ કરેલ મૂલ્ય

## કસ્ટમ ઇવેન્ટ્સ

- `change`: ટૅબ્સ સ્વિચ કરતી વખતે ટ્રિગર થાય છે, `e.value` એ હાલમાં સક્રિય કરેલ વિકલ્પનું `value` મૂલ્ય છે.
