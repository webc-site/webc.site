# கீழ்தோன்றும் தேர்வுடன் தாவல் பக்கம்

- லேபிளில் தகவமைப்பு அகல கீழ்தோன்றும் தேர்வுப் பெட்டியை (`<c-select>`) உட்பொதிப்பதை ஆதரிக்கிறது, இது தற்போதைய தாவல் பக்கம் செயல்படுத்தப்படும் போது மட்டுமே காட்டப்படும்.
- கட்டமைப்பு விவரக்குறிப்பு: குழந்தை உறுப்பு `nav > a` ஐ லேபிள் வழிசெலுத்தலாகப் பயன்படுத்துகிறது, மேலும் நேரடி குழந்தை உறுப்பு `b` உள்ளடக்கப் பகுதியாக செயல்படுகிறது.
- `a[value]` டேக், `a` உடன் `c-select` (குழுவின் மதிப்பு பண்புக்கூறு மற்றும் தேர்ந்தெடுக்கப்பட்ட விருப்ப மதிப்பின் அடிப்படையில்) மற்றும் `b[slot]` உள்ளடக்க முனை ஆகியவற்றை தானாகவே இணைக்கிறது.
- `key` பண்புக்கூறு மூலம் தற்போது செயல்படுத்தப்பட்ட உருப்படியை `localStorage` க்கு தானாக தொடர்ந்து ஆதரிக்கிறது.
- `key + '/' + tab`ஐப் பயன்படுத்தி ஒவ்வொரு குழுவிற்கும் `localStorage` க்கு வரலாற்று துணைத் தேர்வுகளைத் தானாகச் சேமித்தல் மற்றும் மீட்டமைத்தல் ஆகியவற்றை ஆதரிக்கிறது.
- `value` பண்புக்கூறு மூலம் செயலில் உள்ள தாவலைக் குறிப்பிடுவதை ஆதரிக்கிறது.

## டெமோவைப் பயன்படுத்தவும்

```html
<c-tab-select key="my-tab-select-key" value="language/js">
  <nav>
    <a value="language">
      <span>மொழி</span>
      <c-select>
        <option value="js">JavaScript</option>
        <option value="rs">Rust</option>
      </c-select>
    </a>
    <a value="os">
      <span>அமைப்பு</span>
      <c-select>
        <option value="mac">macOS</option>
        <option value="win">Windows</option>
      </c-select>
    </a>
  </nav>
  <b slot="language">
    <b slot="js">ஜாவாஸ்கிரிப்ட் ஒரு ஸ்கிரிப்டிங் மொழி</b>
    <b slot="rs">ரஸ்ட் என்பது கணினி நிலை நிரலாக்க மொழியாகும்</b>
  </b>
  <b slot="os">
    <b slot="mac">macOS என்பது Unix அடிப்படையிலான வரைகலை இயக்க முறைமையாகும்</b>
    <b slot="win">விண்டோஸ் என்பது மைக்ரோசாப்ட் உருவாக்கிய இயங்குதளமாகும்</b>
  </b>
</c-tab-select>
```

## சொத்து

- `key`: உள்ளூர் சேமிப்பக நிலைத்தன்மைக்கு பயன்படுத்தப்படும் முக்கிய பெயர்
- `value`: தற்போது தேர்ந்தெடுக்கப்பட்ட மதிப்பு

## விருப்ப நிகழ்வுகள்

- `change`: தாவல்களை மாற்றும்போது தூண்டப்பட்டது, `e.value` என்பது தற்போது செயல்படுத்தப்பட்ட விருப்பத்தின் `value` மதிப்பாகும்.
