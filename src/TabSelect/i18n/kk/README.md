# Ашылмалы таңдауы бар қойынды беті

- Ағымдағы қойынды беті белсендірілгенде ғана көрсетілетін жапсырмаға бейімделгіш ен ашылмалы таңдау жолағын (`<c-select>`) ендіруді қолдайды.
- Құрылымдық сипаттама: еншілес элемент белгіні шарлау ретінде `nav > a` пайдаланады, ал тікелей еншілес элемент `b` мазмұн аймағы ретінде қызмет етеді.
- `a[value]` тегін, `a` `c-select` (топтың мән төлсипаты және таңдалған опция мәні негізінде) және `b[slot]` мазмұн түйінімен автоматты түрде байланыстырады.
- Ағымдағы белсендірілген элементті `key` төлсипаты арқылы `localStorage` күйіне автоматты түрде сақтауды қолдайды.
- Әрбір топ үшін тарихи ішкі таңдауларды `key + '/' + tab` арқылы `localStorage` автоматты түрде сақтауды және қалпына келтіруді қолдайды.
- `value` төлсипаты арқылы белсенді қойындыны көрсетуді қолдайды.

## Демонстрацияны пайдаланыңыз

```html
<c-tab-select key="my-tab-select-key" value="language/js">
  <nav>
    <a value="language">
      <span>тіл</span>
      <c-select>
        <option value="js">JavaScript</option>
        <option value="rs">Rust</option>
      </c-select>
    </a>
    <a value="os">
      <span>жүйесі</span>
      <c-select>
        <option value="mac">macOS</option>
        <option value="win">Windows</option>
      </c-select>
    </a>
  </nav>
  <b slot="language">
    <b slot="js">JavaScript – сценарий жазу тілі</b>
    <b slot="rs">Rust – жүйелік деңгейдегі бағдарламалау тілі</b>
  </b>
  <b slot="os">
    <b slot="mac">macOS — Unix негізіндегі графикалық операциялық жүйе</b>
    <b slot="win">Windows — Microsoft корпорациясы жасаған операциялық жүйе</b>
  </b>
</c-tab-select>
```

## мүлік

- `key`: localStorage тұрақтылығы үшін пайдаланылатын кілт атауы
- `value`: ағымдағы таңдалған мән

## Арнаулы оқиғалар

- `change`: қойындыларды ауыстырған кезде іске қосылады, `e.value` - ағымдағы белсендірілген опцияның `value` мәні.
