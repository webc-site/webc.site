# ड्रॉप-डाउन निवडीसह टॅब पृष्ठ

- लेबलमधील अडॅप्टिव्ह रुंदी ड्रॉप-डाउन निवड बॉक्स (`<c-select>`) एम्बेड करण्यास समर्थन देते, जे वर्तमान टॅब पृष्ठ सक्रिय केल्यावरच प्रदर्शित केले जाईल.
- स्ट्रक्चरल स्पेसिफिकेशन: चाइल्ड एलिमेंट `nav > a` लेबल नेव्हिगेशन म्हणून वापरतो आणि डायरेक्ट चाइल्ड एलिमेंट `b` सामग्री क्षेत्र म्हणून काम करतो.
- `a[value]` टॅग, `a` `c-select` (समूहाच्या मूल्य विशेषता आणि निवडलेल्या पर्याय मूल्यावर आधारित) आणि `b[slot]` सामग्री नोड स्वयंचलितपणे संबद्ध करते.
- `key` विशेषताद्वारे सध्या सक्रिय केलेल्या आयटमला `localStorage` मध्ये आपोआप टिकून राहण्यास समर्थन देते.
- `key + '/' + tab` वापरून `localStorage` मध्ये प्रत्येक गटासाठी ऐतिहासिक उप-निवड स्वयंचलितपणे जतन आणि पुनर्संचयित करण्यास समर्थन देते.
- `value` विशेषता द्वारे सक्रिय टॅब निर्दिष्ट करण्यास समर्थन देते.

## डेमो वापरा

```html
<c-tab-select key="my-tab-select-key" value="language/js">
  <nav>
    <a value="language">
      <span>भाषा</span>
      <c-select>
        <option value="js">JavaScript</option>
        <option value="rs">Rust</option>
      </c-select>
    </a>
    <a value="os">
      <span>प्रणाली</span>
      <c-select>
        <option value="mac">macOS</option>
        <option value="win">Windows</option>
      </c-select>
    </a>
  </nav>
  <b slot="language">
    <b slot="js">JavaScript ही स्क्रिप्टिंग भाषा आहे</b>
    <b slot="rs">रस्ट ही सिस्टम-स्तरीय प्रोग्रामिंग भाषा आहे</b>
  </b>
  <b slot="os">
    <b slot="mac">macOS ही युनिक्स-आधारित ग्राफिकल ऑपरेटिंग सिस्टम आहे</b>
    <b slot="win">विंडोज ही मायक्रोसॉफ्टने विकसित केलेली ऑपरेटिंग सिस्टम आहे</b>
  </b>
</c-tab-select>
```

## मालमत्ता

- `key`: लोकल स्टोरेज पर्सिस्टन्ससाठी की नाव वापरले
- `value`: सध्या निवडलेले मूल्य

## सानुकूल कार्यक्रम

- `change`: टॅब स्विच करताना ट्रिगर केले, `e.value` हे सध्या सक्रिय केलेल्या पर्यायाचे `value` मूल्य आहे.
