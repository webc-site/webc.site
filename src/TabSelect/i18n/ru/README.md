# Вкладка с раскрывающимся выбором

- Поддерживает встраивание раскрывающегося списка адаптивной ширины (`<c-select>`) в метку, которое будет отображаться только при активации текущей вкладки.
- Структурная спецификация: дочерний элемент использует `nav > a` в качестве навигации по меткам, а прямой дочерний элемент `b` служит областью содержимого.
- Автоматически связывает тег `a[value]`, `a` с `c-select` (на основе атрибута value группы и выбранного значения параметра) и узла контента `b[slot]`.
- Поддерживает автоматическое сохранение текущего активированного элемента в `localStorage` с помощью атрибута `key`.
- Поддерживает автоматическое сохранение и восстановление исторических подвыборок для каждой группы в `localStorage` с помощью `key + '/' + tab`.
- Поддерживает указание активной вкладки с помощью атрибута `value`.

## Используйте демо-версию

```html
<c-tab-select key="my-tab-select-key" value="language/js">
  <nav>
    <a value="language">
      <span>язык</span>
      <c-select>
        <option value="js">JavaScript</option>
        <option value="rs">Rust</option>
      </c-select>
    </a>
    <a value="os">
      <span>система</span>
      <c-select>
        <option value="mac">macOS</option>
        <option value="win">Windows</option>
      </c-select>
    </a>
  </nav>
  <b slot="language">
    <b slot="js">JavaScript — это язык сценариев</b>
    <b slot="rs">Rust — это язык программирования системного уровня.</b>
  </b>
  <b slot="os">
    <b slot="mac">macOS — графическая операционная система на базе Unix.</b>
    <b slot="win">Windows — операционная система, разработанная Microsoft</b>
  </b>
</c-tab-select>
```

## свойство

- `key`: имя ключа, используемое для сохранения localStorage.
- `value`: выбранное в данный момент значение.

## Пользовательские события

- `change`: срабатывает при переключении вкладок. `e.value` — это значение `value` текущей активированной опции.
