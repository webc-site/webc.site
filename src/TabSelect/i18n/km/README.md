# ទំព័រផ្ទាំងដែលមានជម្រើសទម្លាក់ចុះ

- គាំទ្រការបង្កប់នូវប្រអប់ជ្រើសរើសទម្លាក់ចុះទទឹងអាដាប់ធ័រ (`<c-select>`) នៅក្នុងស្លាកដែលនឹងត្រូវបានបង្ហាញនៅពេលទំព័រផ្ទាំងបច្ចុប្បន្នត្រូវបានធ្វើឱ្យសកម្មប៉ុណ្ណោះ។
- ការ​បញ្ជាក់​រចនាសម្ព័ន្ធ៖ ធាតុ​កុមារ​ប្រើ `nav > a` ជា​ការ​រុករក​ស្លាក ហើយ​ធាតុ​កូន​ផ្ទាល់ `b` បម្រើ​ជា​ផ្ទៃ​មាតិកា។
- ភ្ជាប់ស្លាក `a[value]`ដោយស្វ័យប្រវត្តិ ជាមួយ `c-select` (ផ្អែកលើគុណលក្ខណៈតម្លៃរបស់ក្រុម និងតម្លៃជម្រើសដែលបានជ្រើសរើស) និងថ្នាំងខ្លឹមសារ `b[slot]`។
- គាំទ្រ​ការ​បន្ត​ដោយ​ស្វ័យ​ប្រវត្តិ​នូវ​ធាតុ​ដែល​បាន​ធ្វើ​ឱ្យ​សកម្ម​បច្ចុប្បន្ន​ទៅ `localStorage` តាមរយៈ​គុណលក្ខណៈ `key`។
- គាំទ្រការរក្សាទុកដោយស្វ័យប្រវត្តិ និងការស្ដារឡើងវិញនូវជម្រើសរងប្រវត្តិសាស្ត្រសម្រាប់ក្រុមនីមួយៗដើម្បី `localStorage`ដោយប្រើ `key + '/' + tab`។
- គាំទ្រការបញ្ជាក់ផ្ទាំងសកម្មតាមរយៈគុណលក្ខណៈ `value`។

## ប្រើការបង្ហាញ

```html
<c-tab-select key="my-tab-select-key" value="language/js">
  <nav>
    <a value="language">
      <span>ភាសា</span>
      <c-select>
        <option value="js">JavaScript</option>
        <option value="rs">Rust</option>
      </c-select>
    </a>
    <a value="os">
      <span>ប្រព័ន្ធ</span>
      <c-select>
        <option value="mac">macOS</option>
        <option value="win">Windows</option>
      </c-select>
    </a>
  </nav>
  <b slot="language">
    <b slot="js">JavaScript គឺជាភាសាស្គ្រីប</b>
    <b slot="rs">Rust គឺជាភាសាសរសេរកម្មវិធីកម្រិតប្រព័ន្ធ</b>
  </b>
  <b slot="os">
    <b slot="mac">macOS គឺជាប្រព័ន្ធប្រតិបត្តិការក្រាហ្វិកដែលមានមូលដ្ឋានលើយូនីក</b>
    <b slot="win">វីនដូគឺជាប្រព័ន្ធប្រតិបត្តិការដែលបង្កើតឡើងដោយក្រុមហ៊ុន Microsoft</b>
  </b>
</c-tab-select>
```

## ទ្រព្យសម្បត្តិ

- `key`៖ ឈ្មោះ​គន្លឹះ​ប្រើ​សម្រាប់​ការ​រក្សា​ទុក​ក្នុង​មូលដ្ឋាន
- `value`៖ តម្លៃដែលបានជ្រើសរើសបច្ចុប្បន្ន

## ព្រឹត្តិការណ៍ផ្ទាល់ខ្លួន

- `change`៖ កេះនៅពេលប្តូរផ្ទាំង `e.value` គឺជាតម្លៃ `value` នៃជម្រើសដែលបានដំណើរការបច្ចុប្បន្ន។
