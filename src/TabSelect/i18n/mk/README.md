# Страница со јазиче со паѓачки избор

- Поддржува вградување на паѓачкото поле за избор на адаптивна ширина (`<c-select>`) во етикетата, што ќе се прикаже само кога ќе се активира тековната страница на картичката.
- Структурна спецификација: Детскиот елемент користи `nav > a` како навигација на етикетата, а директниот детски елемент `b` служи како област за содржина.
- Автоматски ја поврзува ознаката `a[value]`, `a` со `c-select` (врз основа на атрибутот вредност на групата и вредноста на избраната опција) и јазолот за содржина `b[slot]`.
- Поддржува автоматско задржување на моментално активираната ставка на `localStorage` преку атрибутот `key`.
- Поддржува автоматско зачувување и враќање на историските подизбори за секоја група во `localStorage` користејќи `key + '/' + tab`.
- Поддржува одредување на активното јазиче преку атрибутот `value`.

## Користете го демо

```html
<c-tab-select key="my-tab-select-key" value="language/js">
  <nav>
    <a value="language">
      <span>јазикот</span>
      <c-select>
        <option value="js">JavaScript</option>
        <option value="rs">Rust</option>
      </c-select>
    </a>
    <a value="os">
      <span>систем</span>
      <c-select>
        <option value="mac">macOS</option>
        <option value="win">Windows</option>
      </c-select>
    </a>
  </nav>
  <b slot="language">
    <b slot="js">JavaScript е јазик за скриптирање</b>
    <b slot="rs">Rust е програмски јазик на системско ниво</b>
  </b>
  <b slot="os">
    <b slot="mac">macOS е графички оперативен систем базиран на Unix</b>
    <b slot="win">Windows е оперативен систем развиен од Microsoft</b>
  </b>
</c-tab-select>
```

## имот

- `key`: име на клуч што се користи за упорност на локалната меморија
- `value`: моментално избрана вредност

## Прилагодени настани

- `change`: Се активира при префрлување јазичиња, `e.value` е вредноста `value` на моментално активираната опција.
