# ໜ້າແຖບທີ່ມີການເລືອກແບບເລື່ອນລົງ

- ສະ​ຫນັບ​ສະ​ຫນູນ​ການ​ຝັງ​ກ່ອງ​ການ​ຄັດ​ເລືອກ​ເລື່ອນ​ລົງ​ຄວາມ​ກວ້າງ​ຂອງ​ການ​ປັບ (`<c-select>`​) ໃນ​ປ້າຍ​ຊື່​, ຊຶ່ງ​ຈະ​ສະ​ແດງ​ພຽງ​ແຕ່​ໃນ​ເວ​ລາ​ທີ່​ຫນ້າ​ແຖບ​ໃນ​ປັດ​ຈຸ​ບັນ​ໄດ້​ຮັບ​ການ​ກະ​ຕຸ້ນ​.
- ການລະບຸໂຄງສ້າງ: ອົງປະກອບລູກໃຊ້ `nav > a`ເປັນຕົວນໍາທາງຂອງປ້າຍກຳກັບ, ແລະອົງປະກອບລູກໂດຍກົງ `b`ເຮັດໜ້າທີ່ເປັນພື້ນທີ່ເນື້ອຫາ.
- ເຊື່ອມໂຍງແທັກ `a[value]`ອັດຕະໂນມັດ, `a`ກັບ `c-select` (ໂດຍອ້າງອີງຈາກຄຸນສົມບັດຂອງກຸ່ມ ແລະຄ່າທາງເລືອກທີ່ເລືອກ) ແລະ`b[slot]`ເນື້ອຫາ.
- ສະ​ຫນັບ​ສະ​ຫນູນ​ອັດ​ຕະ​ໂນ​ມັດ​ຕໍ່​ກັບ​ລາຍ​ການ​ທີ່​ເປີດ​ໃຊ້​ໃນ​ປັດ​ຈຸ​ບັນ​ເປັນ `localStorage` ໂດຍ​ຜ່ານ​ຄຸນ​ລັກ​ສະ​ນະ `key`.
- ຮອງຮັບການບັນທຶກອັດຕະໂນມັດ ແລະການຟື້ນຟູການເລືອກຍ່ອຍປະຫວັດສາດສໍາລັບແຕ່ລະກຸ່ມເພື່ອ `localStorage`ໂດຍໃຊ້ `key + '/' + tab`.
- ຮອງຮັບການລະບຸແຖບທີ່ໃຊ້ໄດ້ຜ່ານຄຸນສົມບັດ `value`.

## ໃຊ້ຕົວຢ່າງ

```html
<c-tab-select key="my-tab-select-key" value="language/js">
  <nav>
    <a value="language">
      <span>ພາສາ</span>
      <c-select>
        <option value="js">JavaScript</option>
        <option value="rs">Rust</option>
      </c-select>
    </a>
    <a value="os">
      <span>ລະບົບ</span>
      <c-select>
        <option value="mac">macOS</option>
        <option value="win">Windows</option>
      </c-select>
    </a>
  </nav>
  <b slot="language">
    <b slot="js">JavaScript ເປັນພາສາຂຽນ</b>
    <b slot="rs">Rust ແມ່ນພາສາການຂຽນໂປຼແກຼມລະດັບລະບົບ</b>
  </b>
  <b slot="os">
    <b slot="mac">macOS ແມ່ນລະບົບປະຕິບັດການກາຟິກທີ່ອີງໃສ່ Unix</b>
    <b slot="win">Windows ເປັນລະບົບປະຕິບັດການທີ່ພັດທະນາໂດຍ Microsoft</b>
  </b>
</c-tab-select>
```

## ຊັບສິນ

- `key`: ຊື່ກະແຈທີ່ໃຊ້ສຳລັບ localStorage persistence
- `value`: ຄ່າທີ່ເລືອກໃນປັດຈຸບັນ

## ເຫດການແບບກຳນົດເອງ

- `change`: ຖືກກະຕຸ້ນເມື່ອປ່ຽນແຖບ, `e.value`ແມ່ນຄ່າ `value`ຂອງຕົວເລືອກທີ່ເປີດໃຊ້ໃນປັດຈຸບັນ.
