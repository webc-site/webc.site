# ड्रप-डाउन चयनको साथ ट्याब पृष्ठ

- लेबलमा इम्बेडिङ अनुकूली चौडाइ ड्रप-डाउन चयन बाकस (`<c-select>`) लाई समर्थन गर्दछ, जुन हालको ट्याब पृष्ठ सक्रिय हुँदा मात्र प्रदर्शित हुनेछ।
- संरचनात्मक विशिष्टता: बाल तत्वले लेबल नेभिगेसनको रूपमा `nav > a` प्रयोग गर्दछ, र प्रत्यक्ष बाल तत्व `b` सामग्री क्षेत्रको रूपमा कार्य गर्दछ।
- स्वचालित रूपमा `a[value]` ट्याग, `a` लाई `c-select` (समूहको मान विशेषता र चयन गरिएको विकल्प मानमा आधारित) र `b[slot]` सामग्री नोडसँग सम्बद्ध गर्दछ।
- स्वचालित रूपमा हाल सक्रिय वस्तुलाई `localStorage` मा `key` विशेषता मार्फत जारी राख्न समर्थन गर्दछ।
- `localStorage` `key + '/' + tab` प्रयोग गरेर प्रत्येक समूहको लागि ऐतिहासिक उप-चयनहरूको स्वचालित बचत र पुनर्स्थापनालाई समर्थन गर्दछ।
- `value` विशेषता मार्फत सक्रिय ट्याब निर्दिष्ट गर्न समर्थन गर्दछ।

## डेमो प्रयोग गर्नुहोस्

```html
<c-tab-select key="my-tab-select-key" value="language/js">
  <nav>
    <a value="language">
      <span>भाषा</span>
      <c-select>
        <option value="js">JavaScript</option>
        <option value="rs">Rust</option>
      </c-select>
    </a>
    <a value="os">
      <span>प्रणाली</span>
      <c-select>
        <option value="mac">macOS</option>
        <option value="win">Windows</option>
      </c-select>
    </a>
  </nav>
  <b slot="language">
    <b slot="js">JavaScript एक स्क्रिप्टिङ भाषा हो</b>
    <b slot="rs">Rust एक प्रणाली-स्तर प्रोग्रामिङ भाषा हो</b>
  </b>
  <b slot="os">
    <b slot="mac">macOS युनिक्समा आधारित ग्राफिकल अपरेटिङ सिस्टम हो</b>
    <b slot="win">Windows Microsoft द्वारा विकसित एक अपरेटिङ सिस्टम हो</b>
  </b>
</c-tab-select>
```

## सम्पत्ति

- `key`: स्थानीय भण्डारण स्थिरताको लागि प्रयोग गरिएको कुञ्जी नाम
- `value`: हाल चयन गरिएको मान

## अनुकूलन घटनाहरू

- `change`: ट्याबहरू स्विच गर्दा ट्रिगर गरिएको, `e.value` हाल सक्रिय विकल्पको `value` मान हो।
