# शीर्ष नेव्हिगेशन बार जो पृष्ठ स्क्रोल केल्यावर आपोआप कोसळतो किंवा विस्तृत होतो

- **स्वयं लपवा**: पृष्ठ खाली स्क्रोल करताना स्वयंचलितपणे लपवा.
- **स्वयं-विस्तार**: जेव्हा पृष्ठ वर स्क्रोल केले जाते किंवा स्क्रोल बार अदृश्य होते तेव्हा स्वयंचलितपणे प्रदर्शित होते.
- **कार्यप्रदर्शन ऑप्टिमायझेशन**: स्क्रोलिंग थ्रॉटल करण्यासाठी `requestAnimationFrame` वापरा.
- **रिलीझ संसाधने**: घटक नष्ट झाल्यावर इव्हेंट श्रोते आणि निरीक्षक आपोआप काढून टाकले जातात.

## डेमो वापरा

```html
<main>
  <c-vs>
    <c-nav>
      <b class="L">
        <a class="logo" href="#">LOGO</a>
      </b>
      <a class="item" href="#">दुवा</a>
    </c-nav>
    <div class="content">नेव्हिगेशन लपवण्यासाठी खाली स्वाइप करा, ते दाखवण्यासाठी वर स्वाइप करा</div>
  </c-vs>
</main>
<style>
  main {
    position: relative;
    height: 300px;
    border: 1px solid #000;
  }
  c-nav {
    background: #fffc;
    backdrop-filter: blur(4px);
    border-bottom: 1px solid #ccc;
  }
  .content {
    height: 200dvh;
    background: linear-gradient(to bottom, #fff, #ccc);
    padding: 80px 20px 20px;
    color: #666;
    text-align: center;
  }
</style>
```

## वापरासाठी सूचना

`<c-vs>` चा मूल किंवा वंशज घटक म्हणून वापरला जाणे आवश्यक आहे.

## शैली स्थिती

घटक खालील वर्गांची नावे स्वतःमध्ये बदलतो:

- `D`: स्थिती दर्शवा (खाली किंवा वर स्लाइड करा).
- `I`: संक्रमण स्थिती लपवली जात आहे.
- `H`: लपलेली स्थिती.

## शैली व्हेरिएबल्स

- `--top`: घटकाद्वारे स्वयंचलितपणे उंचीच्या नकारात्मक मूल्यावर अद्यतनित केले जाते, स्थान आणि संक्रमणासाठी वापरले जाते.
