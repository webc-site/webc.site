# စာမျက်နှာ လှိမ့်သည့်အခါ အလိုအလျောက် ပြိုကျ သို့မဟုတ် ချဲ့ထွင်သည့် ထိပ်တန်းလမ်းညွှန်ဘား

- **Auto Hide**- စာမျက်နှာကို အောက်သို့ရွှေ့သည့်အခါ အလိုအလျောက်ဖျောက်ပါ။
- **အလိုအလျောက် ချဲ့ထွင်ခြင်း**- စာမျက်နှာကို အပေါ်သို့ လှိမ့်လိုက်သည့်အခါ သို့မဟုတ် scroll bar ပျောက်သွားသည့်အခါ အလိုအလျောက် ပြသသည်။
- **စွမ်းဆောင်ရည် ပိုမိုကောင်းမွန်အောင် ပြုလုပ်ခြင်း**- အရှိန်မြှင့်ရန် `requestAnimationFrame`ကို အသုံးပြုပါ။
- **ဖြန့်ချိသည့်ရင်းမြစ်များ**- အစိတ်အပိုင်းကို ဖျက်ဆီးလိုက်သောအခါတွင် ပွဲနားထောင်သူများနှင့် စောင့်ကြည့်သူများ အလိုအလျောက် ဖယ်ရှားခံရသည်။

## ဒီမိုကိုသုံးပါ။

```html
<main>
  <c-vs>
    <c-nav>
      <b class="L">
        <a class="logo" href="#">LOGO</a>
      </b>
      <a class="item" href="#">လင့်</a>
    </c-nav>
    <div class="content">လမ်းညွှန်ချက်ကို ဖျောက်ရန် အောက်သို့ ပွတ်ဆွဲပါ၊ ၎င်းကို ပြသရန် အပေါ်သို့ ပွတ်ဆွဲပါ။</div>
  </c-vs>
</main>
<style>
  main {
    position: relative;
    height: 300px;
    border: 1px solid #000;
  }
  c-nav {
    background: #fffc;
    backdrop-filter: blur(4px);
    border-bottom: 1px solid #ccc;
  }
  .content {
    height: 200dvh;
    background: linear-gradient(to bottom, #fff, #ccc);

    color: #666;
    text-align: center;
  }
</style>
```

## အသုံးပြုရန်လမ်းညွှန်ချက်များ

`<c-vs>` ၏ ကလေး သို့မဟုတ် ဆင်းသက်လာသော အစိတ်အပိုင်းအဖြစ် အသုံးပြုရပါမည်။

## ပုံစံအနေအထား

အစိတ်အပိုင်းသည် အောက်ပါ အတန်းအမည်များကို သူ့အလိုလိုပြောင်းသည်-

- `D`- အခြေအနေကို ပြပါ (အပေါ်ဘက်သို့ ပွတ်ဆွဲပါ)။
- `I`- အသွင်ကူးပြောင်းမှုအခြေအနေကို ဝှက်ထားသည်။
- `H`- ဝှက်ထားသော အခြေအနေ။

## စတိုင်ပြောင်းလဲမှုများ

- `--top`- နေရာချထားခြင်းနှင့် ကူးပြောင်းခြင်းအတွက် အသုံးပြုသည့် အမြင့်၏ အနုတ်တန်ဖိုးတစ်ခုသို့ အစိတ်အပိုင်းမှ အလိုအလျောက် အပ်ဒိတ်လုပ်သည်။
