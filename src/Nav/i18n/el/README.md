# Κορυφαία γραμμή πλοήγησης που συμπτύσσεται ή επεκτείνεται αυτόματα κατά την κύλιση της σελίδας

- **Αυτόματη απόκρυψη**: Αυτόματη απόκρυψη κατά την κύλιση προς τα κάτω στη σελίδα.
- **Αυτόματη επέκταση**: Εμφανίζεται αυτόματα όταν η σελίδα πραγματοποιεί κύλιση προς τα επάνω ή εξαφανίζεται η γραμμή κύλισης.
- **Βελτιστοποίηση απόδοσης**: Χρησιμοποιήστε το `requestAnimationFrame` για να μειώσετε την κύλιση.
- **Πόροι απελευθέρωσης**: Οι ακροατές και οι παρατηρητές συμβάντων αφαιρούνται αυτόματα όταν καταστραφεί το στοιχείο.

## Χρησιμοποιήστε το demo

```html
<main>
  <c-vs>
    <c-nav>
      <b class="L">
        <a class="logo" href="#">LOGO</a>
      </b>
      <a class="item" href="#">Σύνδεσμος</a>
    </c-nav>
    <div class="content">Σύρετε προς τα κάτω για να αποκρύψετε την πλοήγηση, σύρετε προς τα επάνω για να την εμφανίσετε</div>
  </c-vs>
</main>
<style>
  main {
    position: relative;
    height: 300px;
    border: 1px solid #000;
  }
  c-nav {
    background: #fffc;
    backdrop-filter: blur(4px);
    border-bottom: 1px solid #ccc;
  }
  .content {
    height: 200dvh;
    background: linear-gradient(to bottom, #fff, #ccc);
    padding: 80px 20px 20px;
    color: #666;
    text-align: center;
  }
</style>
```

## Οδηγίες χρήσης

Πρέπει να χρησιμοποιείται ως θυγατρικό ή απόγονο στοιχείο του `<c-vs>`.

## κατάσταση στυλ

Το στοιχείο αλλάζει τα ακόλουθα ονόματα κλάσεων μέσα του:

- `D`: Εμφάνιση κατάστασης (σύρετε προς τα κάτω ή στο επάνω μέρος).
- `I`: Η κατάσταση μετάβασης είναι απόκρυψη.
- `H`: Κρυφή κατάσταση.

## μεταβλητές στυλ

- `--top`: Ενημερώνεται αυτόματα από το στοιχείο σε μια αρνητική τιμή ύψους, που χρησιμοποιείται για τοποθέτηση και μετάβαση.
