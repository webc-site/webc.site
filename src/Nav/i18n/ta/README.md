# பக்கம் ஸ்க்ரோல் செய்யும் போது தானாகவே சரிந்து அல்லது விரிவடையும் மேல் வழிசெலுத்தல் பட்டி

- **தானாக மறை**: பக்கத்தை கீழே உருட்டும் போது தானாகவே மறை.
- **தானியங்கு-விரிவாக்கு**: பக்கத்தை மேலே ஸ்க்ரோல் செய்யும் போது அல்லது ஸ்க்ரோல் பார் மறைந்தால் தானாகவே காட்டப்படும்.
- **செயல்திறன் மேம்படுத்தல்**: ஸ்க்ரோலிங் செய்ய `requestAnimationFrame`ஐப் பயன்படுத்தவும்.
- **வெளியீட்டு ஆதாரங்கள்**: கூறு அழிக்கப்படும் போது நிகழ்வு கேட்பவர்களும் பார்வையாளர்களும் தானாகவே அகற்றப்படுவார்கள்.

## டெமோவைப் பயன்படுத்தவும்

```html
<main>
  <c-vs>
    <c-nav>
      <b class="L">
        <a class="logo" href="#">LOGO</a>
      </b>
      <a class="item" href="#">இணைப்பு</a>
    </c-nav>
    <div class="content">வழிசெலுத்தலை மறைக்க கீழே ஸ்வைப் செய்யவும், அதைக் காட்ட மேலே ஸ்வைப் செய்யவும்</div>
  </c-vs>
</main>
<style>
  main {
    position: relative;
    height: 300px;
    border: 1px solid #000;
  }
  c-nav {
    background: #fffc;
    backdrop-filter: blur(4px);
    border-bottom: 1px solid #ccc;
  }
  .content {
    height: 200dvh;
    background: linear-gradient(to bottom, #fff, #ccc);
    padding: 80px 20px 20px;
    color: #666;
    text-align: center;
  }
</style>
```

## பயன்பாட்டிற்கான வழிமுறைகள்

குழந்தையாகவோ அல்லது `<c-vs>`ன் வழித்தோன்றலாகவோ பயன்படுத்தப்பட வேண்டும்.

## பாணி நிலை

கூறு பின்வரும் வகுப்புப் பெயர்களை தனக்குள் மாற்றிக் கொள்கிறது:

- `D`: நிலையைக் காட்டு (கீழே அல்லது மேலே ஸ்லைடு செய்யவும்).
- `I`: மாறுதல் நிலை மறைக்கப்பட்டுள்ளது.
- `H`: மறைக்கப்பட்ட நிலை.

## பாணி மாறிகள்

- `--top`: நிலைப்படுத்தல் மற்றும் மாற்றத்திற்குப் பயன்படுத்தப்படும் உயரத்தின் எதிர்மறை மதிப்புக்கு கூறுகளால் தானாகவே புதுப்பிக்கப்படும்.
