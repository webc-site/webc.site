# Վերևի նավիգացիոն գոտին, որն ինքնաբերաբար փլվում կամ ընդլայնվում է, երբ էջը ոլորվում է

- **Ավտոմատ թաքցնել**. ավտոմատ կերպով թաքցնել էջը ներքև ոլորելիս:
- **Ավտոմատ ընդլայնում**. ավտոմատ կերպով ցուցադրվում է, երբ էջը ոլորվում է վերև կամ ոլորման տողը անհետանում է:
- **Կատարման օպտիմիզացում**. Օգտագործեք `requestAnimationFrame`` ոլորումը արագացնելու համար:
- **Թողարկման ռեսուրսներ**. Իրադարձությունների ունկնդիրներն ու դիտորդները ավտոմատ կերպով հեռացվում են, երբ բաղադրիչը ոչնչացվում է:

## Օգտագործեք ցուցադրությունը

```html
<main>
  <c-vs>
    <c-nav>
      <b class="L">
        <a class="logo" href="#">LOGO</a>
      </b>
      <a class="item" href="#">Հղում</a>
    </c-nav>
    <div class="content">Նավիգացիան թաքցնելու համար սահեցրեք ներքև, այն ցուցադրելու համար սահեցրեք վերև</div>
  </c-vs>
</main>
<style>
  main {
    position: relative;
    height: 300px;
    border: 1px solid #000;
  }
  c-nav {
    background: #fffc;
    backdrop-filter: blur(4px);
    border-bottom: 1px solid #ccc;
  }
  .content {
    height: 200dvh;
    background: linear-gradient(to bottom, #fff, #ccc);
    padding: 80px 20px 20px;
    color: #666;
    text-align: center;
  }
</style>
```
