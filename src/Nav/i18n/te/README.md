# పేజీ స్క్రోల్ చేసినప్పుడు స్వయంచాలకంగా కుప్పకూలుతుంది లేదా విస్తరిస్తుంది టాప్ నావిగేషన్ బార్

- **స్వీయ దాచు**: పేజీని క్రిందికి స్క్రోల్ చేస్తున్నప్పుడు స్వయంచాలకంగా దాచు.
- **ఆటో-విస్తరించండి**: పేజీ పైకి స్క్రోల్ చేయబడినప్పుడు లేదా స్క్రోల్ బార్ అదృశ్యమైనప్పుడు స్వయంచాలకంగా ప్రదర్శించబడుతుంది.
- **పనితీరు ఆప్టిమైజేషన్**: స్క్రోలింగ్‌ను థ్రోటిల్ చేయడానికి `requestAnimationFrame`ని ఉపయోగించండి.
- **విడుదల వనరులు**: భాగం నాశనం అయినప్పుడు ఈవెంట్ శ్రోతలు మరియు పరిశీలకులు స్వయంచాలకంగా తీసివేయబడతారు.

## డెమో ఉపయోగించండి

```html
<main>
  <c-vs>
    <c-nav>
      <b class="L">
        <a class="logo" href="#">LOGO</a>
      </b>
      <a class="item" href="#">లింక్</a>
    </c-nav>
    <div class="content">నావిగేషన్‌ను దాచడానికి క్రిందికి స్వైప్ చేయండి, దానిని చూపడానికి పైకి స్వైప్ చేయండి</div>
  </c-vs>
</main>
<style>
  main {
    position: relative;
    height: 300px;
    border: 1px solid #000;
  }
  c-nav {
    background: #fffc;
    backdrop-filter: blur(4px);
    border-bottom: 1px solid #ccc;
  }
  .content {
    height: 200dvh;
    background: linear-gradient(to bottom, #fff, #ccc);
    padding: 80px 20px 20px;
    color: #666;
    text-align: center;
  }
</style>
```

## ఉపయోగం కోసం సూచనలు

తప్పనిసరిగా `<c-vs>` యొక్క చైల్డ్ లేదా డిసెండెంట్ ఎలిమెంట్‌గా ఉపయోగించబడాలి.

## శైలి స్థితి

కాంపోనెంట్ క్రింది తరగతి పేర్లను దానిలోనే మారుస్తుంది:

- `D`: స్థితిని చూపు (క్రిందికి లేదా ఎగువన స్లయిడ్ చేయండి).
- `I`: పరివర్తన స్థితి దాచబడింది.
- `H`: దాచబడిన స్థితి.

## శైలి వేరియబుల్స్

- `--top`: పొజిషనింగ్ మరియు ట్రాన్సిషన్ కోసం ఉపయోగించబడుతుంది, ఎత్తు యొక్క ప్రతికూల విలువకు కాంపోనెంట్ ద్వారా ఆటోమేటిక్‌గా అప్‌డేట్ చేయబడింది.
