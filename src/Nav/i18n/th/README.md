# แถบนำทางด้านบนที่จะยุบหรือขยายโดยอัตโนมัติเมื่อเลื่อนหน้า

- **ซ่อนอัตโนมัติ**: ซ่อนอัตโนมัติเมื่อเลื่อนหน้าลง
- **ขยายอัตโนมัติ**: แสดงโดยอัตโนมัติเมื่อเลื่อนหน้าขึ้นหรือแถบเลื่อนหายไป
- **การเพิ่มประสิทธิภาพประสิทธิภาพ**: ใช้ `requestAnimationFrame` เพื่อควบคุมการเลื่อน
- **เผยแพร่ทรัพยากร**: ผู้ฟังเหตุการณ์และผู้สังเกตการณ์จะถูกลบออกโดยอัตโนมัติเมื่อส่วนประกอบถูกทำลาย

## ใช้การสาธิต

```html
<main>
  <c-vs>
    <c-nav>
      <b class="L">
        <a class="logo" href="#">LOGO</a>
      </b>
      <a class="item" href="#">ลิงค์</a>
    </c-nav>
    <div class="content">ปัดลงเพื่อซ่อนการนำทาง ปัดขึ้นเพื่อแสดง</div>
  </c-vs>
</main>
<style>
  main {
    position: relative;
    height: 300px;
    border: 1px solid #000;
  }
  c-nav {
    background: #fffc;
    backdrop-filter: blur(4px);
    border-bottom: 1px solid #ccc;
  }
  .content {
    height: 200dvh;
    background: linear-gradient(to bottom, #fff, #ccc);
    padding: 80px 20px 20px;
    color: #666;
    text-align: center;
  }
</style>
```

## คำแนะนำสำหรับการใช้งาน

จะต้องใช้เป็นองค์ประกอบย่อยหรือลูกหลานของ `<c-vs>`

## สถานะสไตล์

ส่วนประกอบจะเปลี่ยนชื่อคลาสต่อไปนี้ภายในตัวเอง:

- `D`: แสดงสถานะ (เลื่อนลงหรือด้านบน)
- `I`: สถานะการเปลี่ยนภาพถูกซ่อนอยู่
- `H`: สถานะที่ซ่อนอยู่

## ตัวแปรสไตล์

- `--top`: อัปเดตอัตโนมัติโดยส่วนประกอบให้เป็นค่าความสูงติดลบ ใช้สำหรับการวางตำแหน่งและการเปลี่ยน
