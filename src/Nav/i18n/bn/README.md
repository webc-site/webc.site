# শীর্ষ নেভিগেশন বার যা পৃষ্ঠাটি স্ক্রোল করার সময় স্বয়ংক্রিয়ভাবে ভেঙে যায় বা প্রসারিত হয়

- **স্বয়ংক্রিয়ভাবে লুকান**: পৃষ্ঠাটি স্ক্রোল করার সময় স্বয়ংক্রিয়ভাবে লুকান।
- **স্বয়ংক্রিয়ভাবে প্রসারিত**: পৃষ্ঠাটি স্ক্রোল করা হলে বা স্ক্রোল বারটি অদৃশ্য হয়ে গেলে স্বয়ংক্রিয়ভাবে প্রদর্শিত হয়।
- **পারফরম্যান্স অপ্টিমাইজেশান**: থ্রোটল স্ক্রলিং করতে `requestAnimationFrame` ব্যবহার করুন।
- **রিলিজ রিসোর্স**: ইভেন্ট শ্রোতা এবং পর্যবেক্ষক স্বয়ংক্রিয়ভাবে সরানো হয় যখন উপাদানটি ধ্বংস হয়ে যায়।

## ডেমো ব্যবহার করুন

```html
<main>
  <c-vs>
    <c-nav>
```

## ব্যবহারের জন্য নির্দেশাবলী

`<c-vs>` এর একটি শিশু বা বংশধর উপাদান হিসাবে ব্যবহার করা আবশ্যক৷

## শৈলী অবস্থা

উপাদানটি নিজের মধ্যে নিম্নলিখিত শ্রেণীর নামগুলি পরিবর্তন করে:

- `D`: স্থিতি দেখান (নিচে বা উপরে স্লাইড করুন)।
- `I`: রূপান্তর অবস্থা লুকানো হচ্ছে৷
- `H`: লুকানো অবস্থা।

## শৈলী ভেরিয়েবল

- `--top`: স্বয়ংক্রিয়ভাবে উপাদান দ্বারা উচ্চতার একটি ঋণাত্মক মানের আপডেট করা হয়, যা অবস্থান এবং স্থানান্তরের জন্য ব্যবহৃত হয়।
