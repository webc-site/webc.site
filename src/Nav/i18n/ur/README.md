# ٹاپ نیویگیشن بار جو صفحہ سکرول ہونے پر خود بخود منہدم یا پھیل جاتا ہے۔

- **خودکار چھپائیں**: صفحہ کو نیچے سکرول کرتے وقت خودکار طور پر چھپ جائیں۔
- **خودکار توسیع**: صفحہ کو اسکرول کرنے یا اسکرول بار غائب ہونے پر خودکار طور پر ظاہر ہوتا ہے۔
- **پرفارمنس آپٹیمائزیشن**: تھروٹل سکرولنگ کے لیے `requestAnimationFrame` استعمال کریں۔
- **ریلیز کے وسائل**: جزو کے تباہ ہونے پر ایونٹ کے سننے والے اور مبصرین خود بخود ہٹا دیئے جاتے ہیں۔

## ڈیمو استعمال کریں۔

```html
<main>
  <c-vs>
    <c-nav>
      <b class="L">
        <a class="logo" href="#">LOGO</a>
      </b>
      <a class="item" href="#">لنک</a>
    </c-nav>
    <div class="content">نیویگیشن کو چھپانے کے لیے نیچے سوائپ کریں، اسے دکھانے کے لیے اوپر سوائپ کریں۔</div>
  </c-vs>
</main>
<style>
  main {
    position: relative;
    height: 300px;
    border: 1px solid #000;
  }
  c-nav {
    background: #fffc;
    backdrop-filter: blur(4px);
    border-bottom: 1px solid #ccc;
  }
  .content {
    height: 200dvh;
    background: linear-gradient(to bottom, #fff, #ccc);
    padding: 80px 20px 20px;
    color: #666;
    text-align: center;
  }
</style>
```

## استعمال کے لیے ہدایات

`<c-vs>` کے بچے یا نسلی عنصر کے طور پر استعمال کیا جانا چاہیے۔

## طرز کی حیثیت

جزو اپنے اندر درج ذیل کلاس کے ناموں کو تبدیل کرتا ہے:

- `D`: حالت دکھائیں (نیچے یا اوپر سلائیڈ کریں)۔
- `I`: منتقلی کی حالت چھپی ہوئی ہے۔
- `H`: پوشیدہ حیثیت۔

## طرز متغیرات

- `--top`: جزو کے ذریعہ اونچائی کی منفی قدر میں خودکار طور پر اپ ڈیٹ کیا جاتا ہے، جو پوزیشننگ اور منتقلی کے لیے استعمال ہوتا ہے۔
