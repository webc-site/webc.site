# শীৰ্ষ নেভিগেচন বাৰ যি পৃষ্ঠা স্ক্ৰল কৰাৰ সময়ত স্বয়ংক্ৰিয়ভাৱে সংকুচিত বা প্ৰসাৰিত হয়

- **স্বয়ংক্ৰিয়ভাৱে লুকুৱাওক**: পৃষ্ঠাটো তললৈ স্ক্ৰল কৰাৰ সময়ত স্বয়ংক্ৰিয়ভাৱে লুকুৱাওক।
- **স্বয়ংক্ৰিয়ভাৱে প্ৰসাৰিত**: স্বয়ংক্ৰিয়ভাৱে প্ৰদৰ্শিত হয় যেতিয়া পৃষ্ঠাটো ওপৰলৈ স্ক্ৰল কৰা হয় বা স্ক্ৰল বাৰ অদৃশ্য হয়।
- **কাৰ্য্যক্ষমতা অনুকূলন**: স্ক্ৰলিং থ্ৰ'টল কৰিবলৈ `requestAnimationFrame` ব্যৱহাৰ কৰক।
- **উন্মোচন সম্পদ**: ইভেন্ট শ্ৰোতা আৰু পৰ্যবেক্ষক স্বয়ংক্ৰিয়ভাৱে আঁতৰোৱা হয় যেতিয়া উপাদান ধ্বংস কৰা হয়।

## ডেমো ব্যৱহাৰ কৰক

```html
<main>
  <c-vs>
    <c-nav>
      <b class="L">
        <a class="logo" href="#">LOGO</a>
      </b>
      <a class="item" href="#">লিংক</a>
    </c-nav>
    <div class="content">নেভিগেচন লুকুৱাবলৈ তললৈ চুইপ কৰক, ইয়াক দেখুৱাবলৈ ওপৰলৈ চুইপ কৰক</div>
  </c-vs>
</main>
<style>

  }
```

## ব্যৱহাৰৰ বাবে নিৰ্দেশনা

`<c-vs>` ৰ সন্তান বা বংশধৰ উপাদান হিচাপে ব্যৱহাৰ কৰিব লাগিব।

## শৈলীৰ অৱস্থা

উপাদানটোৱে নিজৰ ভিতৰত নিম্নলিখিত শ্ৰেণী নামসমূহ সলনি কৰে:

- `D`: অৱস্থা দেখুৱাওক (তল বা ওপৰত স্লাইড কৰক)।
- `I`: পৰিৱৰ্তন অৱস্থা লুকুৱাই ৰখা হৈছে।
- `H`: লুকাই থকা অৱস্থা।

## শৈলী চলকসমূহ

- `--top`: উপাদানৰ দ্বাৰা স্বয়ংক্ৰিয়ভাৱে উচ্চতাৰ এটা ঋণাত্মক মানলৈ আপডেইট কৰা হয়, অৱস্থান আৰু পৰিৱৰ্তনৰ বাবে ব্যৱহাৰ কৰা হয়।
