# Горна навигационна лента, която автоматично се свива или разширява, когато страницата се превърта

- **Автоматично скриване**: Автоматично скриване при превъртане надолу по страницата.
- **Автоматично разширяване**: Показва се автоматично, когато страницата се превърти нагоре или лентата за превъртане изчезне.
- **Оптимизиране на производителността**: Използвайте `requestAnimationFrame`, за да ограничите превъртането.
- **Ресурси за освобождаване**: Слушателите и наблюдателите на събития се премахват автоматично, когато компонентът бъде унищожен.

## Използвайте демонстрацията

```html
<main>
  <c-vs>
    <c-nav>
      <b class="L">
        <a class="logo" href="#">LOGO</a>
      </b>
      <a class="item" href="#">Връзка</a>
    </c-nav>
    <div class="content">Плъзнете надолу, за да скриете навигацията, плъзнете нагоре, за да я покажете</div>
  </c-vs>
</main>
<style>
  main {
    position: relative;
    height: 300px;
    border: 1px solid #000;
  }
  c-nav {
    background: #fffc;
    backdrop-filter: blur(4px);
    border-bottom: 1px solid #ccc;
  }
  .content {
    height: 200dvh;
    background: linear-gradient(to bottom, #fff, #ccc);
    padding: 80px 20px 20px;
    color: #666;
    text-align: center;
  }
</style>
```

## Инструкции за употреба

Трябва да се използва като дъщерен или наследствен елемент на `<c-vs>`.

## статус на стил

Компонентът превключва следните имена на класове в себе си:

- `D`: Показване на състоянието (плъзнете надолу или отгоре).
- `I`: Преходното състояние е скрито.
- `H`: Скрит статус.

## стилови променливи

- `--top`: Автоматично актуализирано от компонента до отрицателна стойност на височината, използвана за позициониране и преход.
