# පිටුව අනුචලනය වන විට ස්වයංක්‍රීයව කඩා වැටෙන හෝ පුළුල් වන ඉහළ සංචාලන තීරුව

- **ස්වයං සඟවන්න**: පිටුව පහළට අනුචලනය කිරීමේදී ස්වයංක්‍රීයව සඟවන්න.
- **ස්වයං-ප්‍රසාරණය**: පිටුව ඉහළට අනුචලනය වන විට හෝ අනුචලන තීරුව අතුරුදහන් වූ විට ස්වයංක්‍රීයව දර්ශනය වේ.
- **කාර්ය සාධන ප්‍රශස්තකරණය**: අනුචලනය තල්ලු කිරීමට `requestAnimationFrame` භාවිතා කරන්න.
- **සම්පත් නිකුත් කරන්න**: සංරචකය විනාශ වූ විට සිදුවීම් අසන්නන් සහ නිරීක්ෂකයින් ස්වයංක්‍රීයව ඉවත් කරනු ලැබේ.

## භාවිතය සඳහා උපදෙස්

`<c-vs>` හි දරුවෙකු හෝ පැවත එන අංගයක් ලෙස භාවිතා කළ යුතුය.

## ශෛලිය තත්ත්වය

සංරචකය පහත සඳහන් පන්ති නාම තමා තුළම මාරු කරයි:

- `D`: තත්ත්වය පෙන්වන්න (පහළට හෝ ඉහළට ලිස්සා යන්න).
- `I`: සංක්‍රාන්ති තත්ත්වය සඟවා ඇත.
- `H`: සැඟවුණු තත්ත්වය.

## ශෛලිය විචල්යයන්

- `--top`: ස්ථානගත කිරීම සහ සංක්‍රමණය සඳහා භාවිතා කරන උසෙහි සෘණ අගයකට සංරචකය විසින් ස්වයංක්‍රීයව යාවත්කාලීන කර ඇත.
