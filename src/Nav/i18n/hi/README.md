# शीर्ष नेविगेशन बार जो पृष्ठ स्क्रॉल करने पर स्वचालित रूप से ढह जाता है या विस्तारित हो जाता है

- **स्वतः छिपाएँ**: पृष्ठ को नीचे स्क्रॉल करने पर स्वचालित रूप से छिपाएँ।
- **स्वत: विस्तार**: पृष्ठ को ऊपर स्क्रॉल करने या स्क्रॉल बार गायब होने पर स्वचालित रूप से प्रदर्शित होता है।
- **प्रदर्शन अनुकूलन**: स्क्रॉलिंग को कम करने के लिए `requestAnimationFrame` का उपयोग करें।
- **रिलीज़ संसाधन**: घटक नष्ट होने पर इवेंट श्रोता और पर्यवेक्षक स्वचालित रूप से हटा दिए जाते हैं।

## डेमो का प्रयोग करें

```html
<main>
  <c-vs>
    <c-nav>
      <b class="L">
        <a class="logo" href="#">LOGO</a>
      </b>
      <a class="item" href="#">जोड़ना</a>
    </c-nav>
    <div class="content">नेविगेशन को छिपाने के लिए नीचे की ओर स्वाइप करें, इसे दिखाने के लिए ऊपर की ओर स्वाइप करें</div>
  </c-vs>
</main>
<style>
  main {
    position: relative;
    height: 300px;
    border: 1px solid #000;
  }
  c-nav {
    background: #fffc;
    backdrop-filter: blur(4px);
    border-bottom: 1px solid #ccc;
  }
  .content {
    height: 200dvh;
    background: linear-gradient(to bottom, #fff, #ccc);
    padding: 80px 20px 20px;
    color: #666;
    text-align: center;
  }
</style>
```

## उपयोग के लिए निर्देश

`<c-vs>` के बच्चे या वंशज तत्व के रूप में उपयोग किया जाना चाहिए।

## शैली की स्थिति

घटक निम्नलिखित वर्ग नामों को अपने भीतर स्विच करता है:

- `D`: स्थिति दिखाएं (नीचे या शीर्ष पर स्लाइड करें)।
- `I`: संक्रमण स्थिति छिपाई जा रही है।
- `H`: छिपी हुई स्थिति।

## शैली चर

- `--top`: घटक द्वारा ऊंचाई के नकारात्मक मान पर स्वचालित रूप से अपडेट किया जाता है, जिसका उपयोग स्थिति और संक्रमण के लिए किया जाता है।
