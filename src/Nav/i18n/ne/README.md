# शीर्ष नेभिगेसन पट्टी जुन पृष्ठ स्क्रोल गर्दा स्वचालित रूपमा पतन वा विस्तार हुन्छ

- **स्वतः लुकाउनुहोस्**: पृष्ठ तल स्क्रोल गर्दा स्वचालित रूपमा लुकाउनुहोस्।
- **स्वत: विस्तार**: पृष्ठ स्क्रोल गर्दा वा स्क्रोल बार गायब हुँदा स्वचालित रूपमा प्रदर्शित हुन्छ।
- **कार्यसम्पादन अप्टिमाइजेसन**: थ्रॉटल स्क्रोल गर्न `requestAnimationFrame` प्रयोग गर्नुहोस्।
- **रिलिज स्रोतहरू**: कम्पोनेन्ट नष्ट हुँदा घटना श्रोता र पर्यवेक्षकहरू स्वचालित रूपमा हटाइन्छ।

## डेमो प्रयोग गर्नुहोस्

```html
<main>
  <c-vs>
    <c-nav>
      <b class="L">
        <a class="logo" href="#">LOGO</a>
      </b>
      <a class="item" href="#">लिङ्क</a>
    </c-nav>
    <div class="content">नेभिगेसन लुकाउन तल स्वाइप गर्नुहोस्, यसलाई देखाउन माथि स्वाइप गर्नुहोस्</div>
  </c-vs>
</main>
<style>
  main {
    position: relative;
    height: 300px;
    border: 1px solid #000;
  }
  c-nav {
    background: #fffc;
    backdrop-filter: blur(4px);
    border-bottom: 1px solid #ccc;
  }
  .content {
    height: 200dvh;
    background: linear-gradient(to bottom, #fff, #ccc);
    padding: 80px 20px 20px;
    color: #666;
    text-align: center;
  }
</style>
```

## प्रयोगको लागि निर्देशनहरू

बच्चा वा `<c-vs>` को वंशज तत्वको रूपमा प्रयोग गरिनु पर्छ।

## शैली स्थिति

कम्पोनेन्टले निम्न वर्ग नामहरू आफै भित्र स्विच गर्दछ:

- `D`: स्थिति देखाउनुहोस् (तल वा शीर्षमा स्लाइड गर्नुहोस्)।
- `I`: संक्रमण अवस्था लुकाइएको छ।
- `H`: लुकेको स्थिति।

## शैली चर

- `--top`: कम्पोनेन्टद्वारा स्वचालित रूपमा उचाइको ऋणात्मक मानमा अद्यावधिक हुन्छ, स्थिति र संक्रमणको लागि प्रयोग गरिन्छ।
