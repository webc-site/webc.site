# ແຖບນໍາທາງເທິງສຸດທີ່ຫຍໍ້ລົງ ຫຼືຂະຫຍາຍໂດຍອັດຕະໂນມັດເມື່ອໜ້າເລື່ອນ

- ** ເຊື່ອງ​ອັດ​ຕະ​ໂນ​ມັດ **​: ເຊື່ອງ​ອັດ​ຕະ​ໂນ​ມັດ​ໃນ​ເວ​ລາ​ທີ່​ເລື່ອນ​ລົງ​ຫນ້າ​.
- **ຂະຫຍາຍອັດຕະໂນມັດ**: ສະແດງອັດຕະໂນມັດເມື່ອໜ້າຖືກເລື່ອນຂຶ້ນ ຫຼືແຖບເລື່ອນຫາຍໄປ.
- **ການເພີ່ມປະສິດທິພາບການເພີ່ມປະສິດທິພາບ**: ໃຊ້ `requestAnimationFrame` ເພື່ອກະຕຸ້ນການເລື່ອນ.
- **ຊັບພະຍາກອນການປ່ອຍຕົວ**: ຜູ້ຟັງເຫດການ ແລະຜູ້ສັງເກດການຈະຖືກລຶບອອກໂດຍອັດຕະໂນມັດເມື່ອອົງປະກອບຖືກທໍາລາຍ.

## ໃຊ້ຕົວຢ່າງ

```html
<main>
  <c-vs>
    <c-nav>
      <b class="L">
        <a class="logo" href="#">LOGO</a>
      </b>
      <a class="item" href="#">ເຊື່ອມຕໍ່</a>
    </c-nav>
    <div class="content">ປັດລົງເພື່ອເຊື່ອງການນຳທາງ, ປັດຂຶ້ນເພື່ອສະແດງມັນ</div>
  </c-vs>
</main>
<style>
  main {
    position: relative;
    height: 300px;
    border: 1px solid #000;
  }
  c-nav {
    background: #fffc;
    backdrop-filter: blur(4px);
    border-bottom: 1px solid #ccc;
  }
  .content {
    height: 200dvh;
    background: linear-gradient(to bottom, #fff, #ccc);
    padding: 80px 20px 20px;
    color: #666;
    text-align: center;
  }
</style>
```

## ຄໍາແນະນໍາສໍາລັບການນໍາໃຊ້

ຈະຕ້ອງໃຊ້ເປັນອົງປະກອບຂອງລູກ ຫຼືລູກຫຼານຂອງ `<c-vs>`.

## ສະຖານະຮູບແບບ

ອົງປະກອບປ່ຽນຊື່ຊັ້ນຮຽນຕໍ່ໄປນີ້ພາຍໃນຕົວຂອງມັນເອງ:

- `D`: ສະແດງສະຖານະ (ເລື່ອນລົງ ຫຼືຢູ່ເທິງສຸດ).
- `I`: ສະຖານະການປ່ຽນແປງກຳລັງຖືກເຊື່ອງໄວ້.
- `H`: ສະຖານະທີ່ເຊື່ອງໄວ້.

## ຕົວແປຮູບແບບ

- `--top`: ປັບປຸງອັດຕະໂນມັດໂດຍອົງປະກອບເປັນຄ່າລົບຂອງຄວາມສູງ, ໃຊ້ສໍາລັບການຈັດຕໍາແໜ່ງ ແລະການປ່ຽນແປງ.
