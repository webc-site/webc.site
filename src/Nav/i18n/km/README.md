# របាររុករកកំពូលដែលបង្រួម ឬពង្រីកដោយស្វ័យប្រវត្តិ នៅពេលទំព័ររមូរ

- **លាក់ដោយស្វ័យប្រវត្តិ**៖ លាក់ដោយស្វ័យប្រវត្តិនៅពេលរំកិលចុះក្រោមទំព័រ។
- **ពង្រីកដោយស្វ័យប្រវត្តិ**៖ បង្ហាញដោយស្វ័យប្រវត្តិនៅពេលដែលទំព័រត្រូវបានរមូរឡើងលើ ឬរបាររមូរបាត់។
- **ការបង្កើនប្រសិទ្ធភាពដំណើរការ**៖ ប្រើ `requestAnimationFrame` ដើម្បីបិទបើកការរមូរ។
- **ធនធានចេញផ្សាយ**៖ អ្នកស្តាប់ព្រឹត្តិការណ៍ និងអ្នកសង្កេតការណ៍ត្រូវបានដកចេញដោយស្វ័យប្រវត្តិ នៅពេលដែលសមាសធាតុត្រូវបានបំផ្លាញ។

## ការណែនាំសម្រាប់ការប្រើប្រាស់

ត្រូវតែប្រើជាធាតុកូន ឬកូនចៅនៃ `<c-vs>`។

## ស្ថានភាពរចនាប័ទ្ម

សមាសភាគប្តូរឈ្មោះថ្នាក់ខាងក្រោមនៅក្នុងខ្លួនវា៖

- `D`៖ បង្ហាញស្ថានភាព (រំកិលចុះក្រោម ឬពីលើ)។
- `I`៖ ស្ថានភាពផ្លាស់ប្តូរកំពុងត្រូវបានលាក់។
- `H`៖ ស្ថានភាពលាក់។

## អថេររចនាប័ទ្ម
