# Верхня панель навігації, яка автоматично згортається або розгортається під час прокручування сторінки

- **Автоматичне приховування**: автоматично приховувати під час прокручування сторінки вниз.
- **Автоматичне розгортання**: автоматично відображається, коли сторінка прокручується вгору або смуга прокручування зникає.
- **Оптимізація продуктивності**: використовуйте `requestAnimationFrame`, щоб зменшити прокручування.
- **Випуск ресурсів**: слухачі подій і спостерігачі автоматично видаляються, коли компонент знищується.

## Використовуйте демо

```html
<main>
  <c-vs>
    <c-nav>
      <b class="L">
        <a class="logo" href="#">LOGO</a>
      </b>
      <a class="item" href="#">Посилання</a>
    </c-nav>
    <div class="content">Проведіть пальцем вниз, щоб приховати навігацію, проведіть пальцем угору, щоб показати її</div>
  </c-vs>
</main>
<style>
  main {
    position: relative;
    height: 300px;
    border: 1px solid #000;
  }
  c-nav {
    background: #fffc;
    backdrop-filter: blur(4px);
    border-bottom: 1px solid #ccc;
  }
  .content {
    height: 200dvh;
    background: linear-gradient(to bottom, #fff, #ccc);
    padding: 80px 20px 20px;
    color: #666;
    text-align: center;
  }
</style>
```

## Інструкція із застосування

Має використовуватися як дочірній або нащадковий елемент `<c-vs>`.

## статус стилю

Компонент перемикає в собі такі імена класів:

- `D`: Показати статус (повзунок вниз або вгорі).
- `I`: перехідний стан приховано.
- `H`: Прихований статус.

## змінні стилю

- `--top`: автоматично оновлюється компонентом до від’ємного значення висоти, що використовується для позиціонування та переходу.
