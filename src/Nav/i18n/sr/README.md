# Горња трака за навигацију која се аутоматски скупља или шири када се страница помера

- **Аутоматско сакривање**: Аутоматски се сакријте када се померате надоле по страници.
- **Аутоматско проширење**: Аутоматски се приказује када се страница помера нагоре или трака за померање нестане.
- **Оптимизација перформанси**: Користите <б ид="1"> да бисте смањили померање.
- **Ослободите ресурсе**: Слушаоци и посматрачи догађаја се аутоматски уклањају када се компонента уништи.

## Користите демо

```html
<main>
  <c-vs>
    <c-nav>
      <b class="L">
        <a class="logo" href="#">LOGO</a>
      </b>
      <a class="item" href="#">Линк</a>
    </c-nav>
    <div class="content">Превуците надоле да бисте сакрили навигацију, превуците нагоре да бисте је приказали</div>
  </c-vs>
</main>
<style>
  main {
    position: relative;
    height: 300px;
    border: 1px solid #000;
  }
  c-nav {
    background: #fffc;
    backdrop-filter: blur(4px);
    border-bottom: 1px solid #ccc;
  }
  .content {
    height: 200dvh;
    background: linear-gradient(to bottom, #fff, #ccc);
    padding: 80px 20px 20px;
    color: #666;
    text-align: center;
  }
</style>
```

## Упутство за употребу

Мора се користити као подређени или потомак елемента <б ид="1">.

## статус стила

Компонента мења следећа имена класа унутар себе:

- <б ид="1">: Прикажи статус (клизање надоле или на врх).
- <б ид="1">: Прелазно стање је скривено.
- <б ид="1">: Скривен статус.

## стилске променљиве

- <б ид="1">: Компонента аутоматски ажурира на негативну вредност висине, која се користи за позиционирање и прелаз.
