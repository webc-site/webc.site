# ឯកសារ និងកូដប្រភពសម្រាប់បង្ហាញសមាសធាតុ ឬមុខងារឧបករណ៍

- ប្លង់ឆ្លើយតបពីរជួរ ដោយឯកសារបង្ហាញនៅខាងឆ្វេង និងកូដប្រភពបង្ហាញនៅខាងស្តាំ
- ឯកសារ និងកូដប្រភពគាំទ្រប៊ូតុងផ្លូវកាត់ កែសម្រួល និងកូដ ដើម្បីលោតទៅអាសយដ្ឋានប្រភព។
- ចំណងជើង និងការណែនាំត្រូវបានបង្ហាញនៅផ្នែកខាងលើ ដែលគាំទ្រការបកប្រែដោយស្វ័យប្រវត្តិ i18n ។

## ប្រើការបង្ហាញ

```html
<c-webc-js src="/src/js/dom.js" href="/doc/{code}/js/dom.md"></c-webc-js>
```

## ប៉ារ៉ាម៉ែត្រអចលនទ្រព្យ

- `src`៖ អាសយដ្ឋាន URL នៃកូដប្រភព។
- `href`៖ អាសយដ្ឋាន URL នៃឯកសារ markdown គាំទ្រដោយប្រើ `{code}`កន្លែងដាក់ ដើម្បីសម្របទៅនឹងភាសាបច្ចុប្បន្ន។
