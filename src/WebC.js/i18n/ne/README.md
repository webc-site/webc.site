# कम्पोनेन्ट वा उपकरण कार्यहरू प्रतिपादन गर्नका लागि कागजात र स्रोत कोड

- दुई-स्तम्भ उत्तरदायी लेआउट, बायाँमा प्रदर्शित कागजातहरू र दायाँमा स्रोत कोड प्रदर्शित
- कागजात र स्रोत कोड स्रोत ठेगानामा हाम फाल्न सम्पादन र कोड सर्टकट बटन समर्थन गर्दछ।
- शीर्षक र परिचय शीर्षमा प्रदर्शित हुन्छ, i18n स्वचालित अनुवादलाई समर्थन गर्दै।

## डेमो प्रयोग गर्नुहोस्

```html
<c-webc-js src="/src/js/dom.js" href="/doc/{code}/js/dom.md"></c-webc-js>
```

## सम्पत्ति मापदण्डहरू

- `src`: स्रोत कोडको URL ठेगाना।
- `href`: कागजात मार्कडाउनको URL ठेगाना, हालको भाषामा अनुकूलन गर्न `{code}` प्लेसहोल्डर प्रयोग गर्न समर्थन गर्दछ।
