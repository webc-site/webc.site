# ເອກະສານແລະລະຫັດແຫຼ່ງສໍາລັບການສະແດງອົງປະກອບຫຼືຫນ້າທີ່ເຄື່ອງມື

- ຮູບ​ແບບ​ການ​ຕອບ​ສະ​ຫນອງ​ສອງ​ຖັນ​, ມີ​ເອ​ກະ​ສານ​ສະ​ແດງ​ຢູ່​ເບື້ອງ​ຊ້າຍ​ແລະ​ລະ​ຫັດ​ແຫຼ່ງ​ສະ​ແດງ​ຢູ່​ທາງ​ຂວາ
- ເອກະສານແລະລະຫັດແຫຼ່ງສະຫນັບສະຫນູນການແກ້ໄຂແລະລະຫັດປຸ່ມລັດເພື່ອເຕັ້ນໄປຫາທີ່ຢູ່ແຫຼ່ງ.
- ຫົວຂໍ້ແລະການແນະນໍາແມ່ນສະແດງຢູ່ເທິງສຸດ, ສະຫນັບສະຫນູນການແປພາສາອັດຕະໂນມັດ i18n.

## ໃຊ້ຕົວຢ່າງ

```html
<c-webc-js src="/src/js/dom.js" href="/doc/{code}/js/dom.md"></c-webc-js>
```

## ຕົວກໍານົດການຊັບສິນ

- `src`: ທີ່ຢູ່ URL ຂອງລະຫັດແຫຼ່ງ.
- `href`: ທີ່ຢູ່ URL ຂອງເອກະສານ markdown, ຮອງຮັບການໃຊ້ `{code}` placeholder ເພື່ອປັບຕົວເຂົ້າກັບພາສາປັດຈຸບັນ.
