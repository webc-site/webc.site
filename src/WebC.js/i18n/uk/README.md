# Документація та вихідний код для відтворення компонентів або функцій інструменту

- Адаптивний макет у дві колонки з документами, що відображаються ліворуч, і вихідним кодом, що відображається праворуч
- Документація та вихідний код підтримують кнопки швидкого доступу «Редагувати» та «Код», щоб перейти до адреси джерела.
- Заголовок і вступ відображаються вгорі, підтримуючи автоматичний переклад i18n.

## Використовуйте демо

```html
<c-webc-js src="/src/js/dom.js" href="/doc/{code}/js/dom.md"></c-webc-js>
```

## Параметри властивості

- `src`: URL-адреса вихідного коду.
- `href`: URL-адреса розмітки документа, підтримує використання заповнювача `{code}` для адаптації до поточної мови.
