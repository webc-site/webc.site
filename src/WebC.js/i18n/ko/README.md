# 구성 요소 또는 도구 기능을 렌더링하기 위한 문서 및 소스 코드

- 왼쪽에 문서가 표시되고 오른쪽에 소스 코드가 표시되는 2열 반응형 레이아웃
- 문서 및 소스 코드는 소스 주소로 이동하는 편집 및 코드 바로 가기 버튼을 지원합니다.
- 제목과 소개가 상단에 표시되며 i18n 자동 번역을 지원합니다.

## 데모 사용

```html
<c-webc-js src="/src/js/dom.js" href="/doc/{code}/js/dom.md"></c-webc-js>
```

## 속성 매개변수

- `src`: 소스 코드의 URL 주소입니다.
- `href`: 문서 마크다운의 URL 주소, 현재 언어에 적응하기 위해 `{code}` 자리 표시자 사용을 지원합니다.
