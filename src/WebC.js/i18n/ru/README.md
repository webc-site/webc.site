# Документация и исходный код для рендеринга компонентов или функций инструмента.

- Адаптивный макет с двумя столбцами: документы отображаются слева, а исходный код — справа.
- Поддержка документации и исходного кода. Кнопки быстрого доступа «Редактировать» и «Код» для перехода к исходному адресу.
- Название и введение отображаются вверху, что поддерживает автоматический перевод i18n.

## Используйте демо-версию

```html
<c-webc-js src="/src/js/dom.js" href="/doc/{code}/js/dom.md"></c-webc-js>
```

## Параметры недвижимости

- `src`: URL-адрес исходного кода.
- `href`: URL-адрес уценки документа, поддерживает использование заполнителя `{code}` для адаптации к текущему языку.
