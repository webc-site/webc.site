# Документација и изворни код за приказивање компоненти или функција алата

- Изглед са две колоне који одговара, са документима приказаним на левој страни и изворним кодом приказаним на десној страни
- Документација и изворни код подржавају дугмад за пречице за уређивање и код за прелазак на изворну адресу.
- Наслов и увод су приказани на врху, подржавајући и18н аутоматски превод.

## Користите демо

```html
<c-webc-js src="/src/js/dom.js" href="/doc/{code}/js/dom.md"></c-webc-js>
```

## Параметри својства

- <б ид="1">: УРЛ адреса изворног кода.
- <б ид="1">: УРЛ адреса умањивања документа, подржава коришћење чувара места <б ид="2"> за прилагођавање тренутном језику.
