# Dokumentatioun a Quellcode fir Komponente oder Toolfunktiounen ze maachen

- Zwee-Kolonn reaktiounsfäeger Layout, mat Dokumenter lénks ugewisen an Quellcode op der rietser ugewisen
- Dokumentatioun a Quellcode ënnerstëtzen Änneren a Code Ofkierzungsknäpper fir op d'Quelladress ze sprangen.
- Den Titel an d'Aféierung ginn uewen ugewisen, ënnerstëtzt i18n automatesch Iwwersetzung.

## Benotzt d'Demo

```html
<c-webc-js src="/src/js/dom.js" href="/doc/{code}/js/dom.md"></c-webc-js>
```

## Property Parameteren

- `src`: URL Adress vum Quellcode.
- `href`: URL Adress vum Dokumentmarkdown, ënnerstëtzt d'Benotzung vun `{code}` Plazhalter fir un déi aktuell Sprooch unzepassen.
