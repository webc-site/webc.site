# রেন্ডারিং উপাদান বা টুল ফাংশন জন্য ডকুমেন্টেশন এবং উত্স কোড

- দুই-কলামের প্রতিক্রিয়াশীল লেআউট, বাম দিকে প্রদর্শিত নথি এবং ডানদিকে সোর্স কোড প্রদর্শিত হয়
- ডকুমেন্টেশন এবং সোর্স কোড সোর্স অ্যাড্রেসে যেতে সম্পাদনা এবং কোড শর্টকাট বোতাম সমর্থন করে।
- শিরোনাম এবং ভূমিকা শীর্ষে প্রদর্শিত হয়, i18n স্বয়ংক্রিয় অনুবাদ সমর্থন করে।

## ডেমো ব্যবহার করুন

```html
<c-webc-js src="/src/js/dom.js" href="/doc/{code}/js/dom.md"></c-webc-js>
```

## সম্পত্তি পরামিতি

- `src`: উৎস কোডের URL ঠিকানা।
- `href`: ডকুমেন্ট মার্কডাউনের URL ঠিকানা, বর্তমান ভাষার সাথে খাপ খাইয়ে নিতে `{code}` স্থানধারক ব্যবহার করে সমর্থন করে।
