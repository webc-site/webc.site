# Komponentlarni yoki asbob funktsiyalarini ko'rsatish uchun hujjatlar va manba kodi

- Ikki ustunli javob beruvchi tartib, chap tomonda hujjatlar va o'ng tomonda manba kodi ko'rsatilgan
- Hujjatlar va manba kodi manba manziliga o'tish uchun Tahrirlash va Kod yorliqlarini qo'llab-quvvatlaydi.
- Sarlavha va kirish i18n avtomatik tarjimasini qo'llab-quvvatlovchi yuqori qismida ko'rsatiladi.

## Ko‘rsatma ishlatish

```html
<c-webc-js src="/src/js/dom.js" href="/doc/{code}/js/dom.md"></c-webc-js>
```

## Atribut parametrlari

- `src`: manba kodining URL manzili.
- `href`: hujjatni belgilashning URL manzili, joriy tilga moslashish uchun `{code}` to'ldiruvchidan foydalanishni qo'llab-quvvatlaydi.
