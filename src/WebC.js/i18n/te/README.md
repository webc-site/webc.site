# రెండరింగ్ భాగాలు లేదా టూల్ ఫంక్షన్‌ల కోసం డాక్యుమెంటేషన్ మరియు సోర్స్ కోడ్

- రెండు-నిలువు వరుస ప్రతిస్పందించే లేఅవుట్, పత్రాలు ఎడమవైపు ప్రదర్శించబడతాయి మరియు సోర్స్ కోడ్ కుడి వైపున ప్రదర్శించబడతాయి
- డాక్యుమెంటేషన్ మరియు సోర్స్ కోడ్ మద్దతు మూలం చిరునామాకు వెళ్లడానికి సవరణ మరియు కోడ్ సత్వరమార్గ బటన్‌లు.
- i18n స్వయంచాలక అనువాదానికి మద్దతునిస్తూ శీర్షిక మరియు పరిచయం ఎగువన ప్రదర్శించబడతాయి.

## డెమో ఉపయోగించండి

```html
<c-webc-js src="/src/js/dom.js" href="/doc/{code}/js/dom.md"></c-webc-js>
```

## ఆస్తి పారామితులు

- `src`: సోర్స్ కోడ్ యొక్క URL చిరునామా.
- `href`: డాక్యుమెంట్ మార్క్‌డౌన్ యొక్క URL చిరునామా, ప్రస్తుత భాషకి అనుగుణంగా `{code}` ప్లేస్‌హోల్డర్‌ని ఉపయోగించడం సపోర్ట్ చేస్తుంది.
