# Дакументацыя і зыходны код для рэндэрынгу кампанентаў або функцый інструмента

- Адаптыўны макет у два слупкі, дакументы адлюстроўваюцца злева, а зыходны код - справа
- Дакументацыя і зыходны код падтрымліваюць кнопкі хуткага доступу "Праўка" і "Код", каб перайсці да зыходнага адрасу.
- Загаловак і ўвядзенне адлюстроўваюцца ўверсе, падтрымліваючы аўтаматычны пераклад i18n.

## Выкарыстоўвайце дэма

```html
<c-webc-js src="/src/js/dom.js" href="/doc/{code}/js/dom.md"></c-webc-js>
```

## Параметры ўласцівасці

- `src`: URL-адрас зыходнага кода.
- `href`: URL-адрас разметкі дакумента, падтрымлівае выкарыстанне запаўняльніка `{code}` для адаптацыі да бягучай мовы.
