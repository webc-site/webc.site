# উপাদানসমূহ বা সঁজুলি ফলনসমূহ ৰেণ্ডাৰ কৰাৰ বাবে দস্তাবেজ আৰু উৎস ক'ড

- দুটা-স্তম্ভৰ প্ৰতিক্ৰিয়াশীল পৰিকল্পনা, বাওঁফালে প্ৰদৰ্শিত দস্তাবেজ আৰু সোঁফালে উৎস ক'ড প্ৰদৰ্শিত
- দস্তাবেজ আৰু উৎস ক'ডে উৎস ঠিকনালৈ জপিয়াই যাবলৈ সম্পাদনা আৰু ক'ড চৰ্টকাট বুটামসমূহ সমৰ্থন কৰে।
- শিৰোনাম আৰু পৰিচয় ওপৰত প্ৰদৰ্শিত হয়, i18n স্বয়ংক্ৰিয় অনুবাদ সমৰ্থন কৰে।

## ডেমো ব্যৱহাৰ কৰক

```html
<c-webc-js src="/src/js/dom.js" href="/doc/{code}/js/dom.md"></c-webc-js>
```

## বৈশিষ্ট্য প্ৰাচলসমূহ

- `src`: উৎস ক'ডৰ URL ঠিকনা।
- `href`: দস্তাবেজ মাৰ্কডাউনৰ URL ঠিকনা, বৰ্তমান ভাষাৰ সৈতে খাপ খুৱাবলৈ `{code}` প্লেচহোল্ডাৰ ব্যৱহাৰ কৰা সমৰ্থন কৰে।
