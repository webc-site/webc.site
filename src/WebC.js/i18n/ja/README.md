# コンポーネントまたはツール機能をレンダリングするためのドキュメントとソース コード

- 2 列のレスポンシブ レイアウト。左側にドキュメントが表示され、右側にソース コードが表示されます。
- ドキュメントとソース コードは、ソース アドレスにジャンプするための [編集] および [コード] ショートカット ボタンをサポートしています。
- タイトルと紹介文が上部に表示され、i18n自動翻訳に対応しています。

## デモを使用する

```html
<c-webc-js src="/src/js/dom.js" href="/doc/{code}/js/dom.md"></c-webc-js>
```

## プロパティパラメータ

- `src`: ソース コードの URL アドレス。
- `href`: ドキュメント マークダウンの URL アドレス。現在の言語に適応するための `{code}` プレースホルダーの使用をサポートします。
