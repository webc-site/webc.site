# اجزاء یا ٹول کے افعال کو پیش کرنے کے لیے دستاویزات اور سورس کوڈ

- دو کالم جوابی ترتیب، بائیں طرف دکھائے جانے والے دستاویزات کے ساتھ اور دائیں طرف ظاہر کردہ سورس کوڈ
- دستاویزی اور سورس کوڈ ایڈٹ اور کوڈ کے شارٹ کٹ بٹنوں کو سورس ایڈریس پر جانے کے لیے سپورٹ کرتے ہیں۔
- i18n خودکار ترجمہ کی حمایت کرتے ہوئے عنوان اور تعارف سب سے اوپر دکھائے گئے ہیں۔

## ڈیمو استعمال کریں۔

```html
<c-webc-js src="/src/js/dom.js" href="/doc/{code}/js/dom.md"></c-webc-js>
```

## پراپرٹی کے پیرامیٹرز

- `src`: سورس کوڈ کا URL پتہ۔
- `href`: دستاویز کے مارک ڈاؤن کا URL ایڈریس، موجودہ زبان کے مطابق ڈھالنے کے لیے `{code}` پلیس ہولڈر استعمال کرنے کی حمایت کرتا ہے۔
