# Документација и изворен код за рендерирање на компоненти или функции на алатката

- Распоред кој одговара на две колони, со документи прикажани лево и изворен код прикажан надесно
- Документацијата и изворниот код ги поддржуваат копчињата за кратенка Уреди и Код за да скокаат до изворната адреса.
- Насловот и воведот се прикажани на врвот, поддржувајќи автоматски превод на i18n.

## Користете го демо

```html
<c-webc-js src="/src/js/dom.js" href="/doc/{code}/js/dom.md"></c-webc-js>
```

## Параметри на својствата

- `src`: URL адреса на изворниот код.
- `href`: URL адреса на означување на документот, поддржува користење `{code}` место за адаптација на тековниот јазик.
