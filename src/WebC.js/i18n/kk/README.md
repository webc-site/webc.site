# Құрамдас бөліктерді немесе құрал функцияларын көрсетуге арналған құжаттама және бастапқы код

- Екі бағанды ​​жауап беретін орналасу, құжаттар сол жақта және бастапқы код оң жақта көрсетіледі
- Құжаттама және бастапқы код бастапқы мекенжайға өту үшін Өңдеу және Код таңбаша түймелерін қолдайды.
- Тақырып пен кіріспе i18n автоматты аудармасын қолдайтын жоғарғы жағында көрсетіледі.

## Демонстрацияны пайдаланыңыз

```html
<c-webc-js src="/src/js/dom.js" href="/doc/{code}/js/dom.md"></c-webc-js>
```

## Меншік параметрлері

- `src`: бастапқы кодтың URL мекенжайы.
- `href`: құжатты белгілеудің URL мекенжайы, ағымдағы тілге бейімделу үшін `{code}` толтырғышты пайдалануды қолдайды.
