# Dogfennaeth a chod ffynhonnell ar gyfer rendro cydrannau neu swyddogaethau offer

- Cynllun ymatebol dwy golofn, gyda dogfennau wedi'u harddangos ar y chwith a chod ffynhonnell i'w gweld ar y dde
- Mae dogfennaeth a chod ffynhonnell yn cefnogi botymau llwybr byr Golygu a Chodio i neidio i'r cyfeiriad ffynhonnell.
- Mae'r teitl a'r cyflwyniad yn cael eu harddangos ar y brig, gan gefnogi i18n cyfieithu awtomatig.

## Defnyddiwch y demo

```html
<c-webc-js src="/src/js/dom.js" href="/doc/{code}/js/dom.md"></c-webc-js>
```

## Paramedrau eiddo

- `src`: cyfeiriad URL y cod ffynhonnell.
- `href`: cyfeiriad URL marcio'r ddogfen, yn cefnogi defnyddio `{code}` dalfan i addasu i'r iaith gyfredol.
