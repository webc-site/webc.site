# เอกสารประกอบและซอร์สโค้ดสำหรับการเรนเดอร์ส่วนประกอบหรือฟังก์ชันของเครื่องมือ

- รูปแบบตอบสนองแบบสองคอลัมน์ โดยมีเอกสารแสดงทางด้านซ้ายและซอร์สโค้ดแสดงทางด้านขวา
- รองรับเอกสารและซอร์สโค้ดแก้ไขและปุ่มลัดรหัสเพื่อข้ามไปยังที่อยู่ต้นทาง
- ชื่อเรื่องและคำนำจะแสดงที่ด้านบน รองรับการแปลอัตโนมัติของ i18n

## ใช้การสาธิต

```html
<c-webc-js src="/src/js/dom.js" href="/doc/{code}/js/dom.md"></c-webc-js>
```

## พารามิเตอร์คุณสมบัติ

- `src`: ที่อยู่ URL ของซอร์สโค้ด
- `href`: ที่อยู่ URL ของมาร์กดาวน์เอกสาร รองรับการใช้ตัวยึดตำแหน่ง `{code}` เพื่อปรับให้เข้ากับภาษาปัจจุบัน
