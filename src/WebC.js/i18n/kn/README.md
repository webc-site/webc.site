# ರೆಂಡರಿಂಗ್ ಘಟಕಗಳು ಅಥವಾ ಟೂಲ್ ಕಾರ್ಯಗಳಿಗಾಗಿ ದಾಖಲೆ ಮತ್ತು ಮೂಲ ಕೋಡ್

- ಎರಡು-ಕಾಲಮ್ ರೆಸ್ಪಾನ್ಸಿವ್ ಲೇಔಟ್, ಡಾಕ್ಯುಮೆಂಟ್‌ಗಳನ್ನು ಎಡಭಾಗದಲ್ಲಿ ಪ್ರದರ್ಶಿಸಲಾಗುತ್ತದೆ ಮತ್ತು ಮೂಲ ಕೋಡ್ ಅನ್ನು ಬಲಭಾಗದಲ್ಲಿ ಪ್ರದರ್ಶಿಸಲಾಗುತ್ತದೆ
- ಡಾಕ್ಯುಮೆಂಟೇಶನ್ ಮತ್ತು ಮೂಲ ಕೋಡ್ ಬೆಂಬಲ ಮೂಲ ವಿಳಾಸಕ್ಕೆ ನೆಗೆಯಲು ಸಂಪಾದಿಸಿ ಮತ್ತು ಕೋಡ್ ಶಾರ್ಟ್‌ಕಟ್ ಬಟನ್‌ಗಳು.
- i18n ಸ್ವಯಂಚಾಲಿತ ಅನುವಾದವನ್ನು ಬೆಂಬಲಿಸುವ ಶೀರ್ಷಿಕೆ ಮತ್ತು ಪರಿಚಯವನ್ನು ಮೇಲ್ಭಾಗದಲ್ಲಿ ಪ್ರದರ್ಶಿಸಲಾಗುತ್ತದೆ.

## ಡೆಮೊ ಬಳಸಿ

```html
<c-webc-js src="/src/js/dom.js" href="/doc/{code}/js/dom.md"></c-webc-js>
```

## ಆಸ್ತಿ ನಿಯತಾಂಕಗಳು

- `src`: ಮೂಲ ಕೋಡ್‌ನ URL ವಿಳಾಸ.
- `href`: ಡಾಕ್ಯುಮೆಂಟ್ ಮಾರ್ಕ್‌ಡೌನ್‌ನ URL ವಿಳಾಸ, ಪ್ರಸ್ತುತ ಭಾಷೆಗೆ ಹೊಂದಿಕೊಳ್ಳಲು `{code}` ಪ್ಲೇಸ್‌ಹೋಲ್ಡರ್ ಅನ್ನು ಬೆಂಬಲಿಸುತ್ತದೆ.
