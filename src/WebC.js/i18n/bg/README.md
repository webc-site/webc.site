# Документация и изходен код за изобразяване на компоненти или функции на инструменти

- Адаптивно оформление с две колони, с документи, показани отляво, и изходен код, показан отдясно
- Документацията и изходният код поддържат бутони за бърз достъп Редактиране и Код за прескачане до адреса на източника.
- Заглавието и въведението се показват в горната част, поддържайки i18n автоматичен превод.

## Използвайте демонстрацията

```html
<c-webc-js src="/src/js/dom.js" href="/doc/{code}/js/dom.md"></c-webc-js>
```

## Параметри на имота

- `src`: URL адрес на изходния код.
- `href`: URL адрес на маркирането на документа, поддържа използване на `{code}` контейнер за адаптиране към текущия език.
