# အစိတ်အပိုင်းများ သို့မဟုတ် ကိရိယာလုပ်ဆောင်ချက်များကို တင်ဆက်ရန်အတွက် စာရွက်စာတမ်းနှင့် အရင်းအမြစ်ကုဒ်

- ဘယ်ဘက်တွင် ပြထားသည့် စာရွက်စာတမ်းများနှင့် အရင်းအမြစ်ကုဒ်ကို ညာဘက်တွင် ပြသထားသည့် ကော်လံနှစ်ခု တုံ့ပြန်မှုပုံစံ
- စာရွက်စာတမ်းနှင့် အရင်းအမြစ်ကုဒ်သည် အရင်းအမြစ်လိပ်စာသို့သွားရန် တည်းဖြတ်ရန်နှင့် ကုဒ်ဖြတ်လမ်းခလုတ်များကို ပံ့ပိုးပေးသည်။
- i18n အလိုအလျောက်ဘာသာပြန်ခြင်းကို ပံ့ပိုးပေးသည့် ခေါင်းစဉ်နှင့် နိဒါန်းကို ထိပ်တွင် ပြသထားသည်။

## ဒီမိုကိုသုံးပါ။

```html
<c-webc-js src="/src/js/dom.js" href="/doc/{code}/js/dom.md"></c-webc-js>
```

## ပိုင်ဆိုင်မှုဘောင်များ

- `src`- အရင်းအမြစ်ကုဒ်၏ URL လိပ်စာ။
- `href`- စာရွက်စာတမ်းအမှတ်အသားပြုခြင်း၏ URL လိပ်စာ၊ လက်ရှိဘာသာစကားနှင့်လိုက်လျောညီထွေဖြစ်အောင်ပြုလုပ်ရန် `{code}`နေရာကိုင်ဆောင်သူကို အသုံးပြု၍ ပံ့ပိုးပေးသည်။
