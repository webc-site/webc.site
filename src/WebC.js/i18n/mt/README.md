# Dokumentazzjoni u kodiċi tas-sors għall-għoti ta' komponenti jew funzjonijiet ta' għodda

- Tqassim b'żewġ kolonni li jirrispondu, bid-dokumenti murija fuq ix-xellug u l-kodiċi tas-sors murija fuq il-lemin
- Dokumentazzjoni u appoġġ tal-kodiċi tas-sors Editja u buttuni ta' shortcut tal-Kodiċi biex taqbeż għall-indirizz tas-sors.
- It-titlu u l-introduzzjoni jintwerew fil-parti ta 'fuq, u jappoġġjaw it-traduzzjoni awtomatika i18n.

## Uża d-demo

```html
<c-webc-js src="/src/js/dom.js" href="/doc/{code}/js/dom.md"></c-webc-js>
```

## Parametri tal-proprjetà

- `src`: Indirizz URL tal-kodiċi tas-sors.
- `href`: L-indirizz URL tal-markdown tad-dokument, jappoġġja l-użu ta' `{code}` placeholder biex tadatta għal-lingwa attwali.
