# دستاويز ۽ ماخذ ڪوڊ رينڊرنگ اجزاء يا اوزار جي ڪم لاء

- ٻه ڪالمن جوابي ترتيب، کاٻي پاسي ڏيکاريل دستاويزن سان ۽ ساڄي پاسي ڏيکاريل ماخذ ڪوڊ
- دستاويزن ۽ سورس ڪوڊ سپورٽ ايڊٽ ۽ ڪوڊ شارٽ ڪٽ بٽڻ کي ماخذ ايڊريس تي وڃڻ لاءِ.
- عنوان ۽ تعارف مٿي تي ڏيکاريل آهن، i18n خودڪار ترجمي جي حمايت ڪندي.

## ڊيمو استعمال ڪريو

```html
<c-webc-js src="/src/js/dom.js" href="/doc/{code}/js/dom.md"></c-webc-js>
```

## ملڪيت جا پيراگراف

- `src`: سورس ڪوڊ جو URL پتو.
- `href`: URL ائڊريس جو ڊاڪيومينٽ مارڪ ڊائون، استعمال ڪرڻ جي سپورٽ ڪري ٿو `{code}` پليس هولڊر کي موجوده ٻولي سان موافقت ڪرڻ لاءِ.
