## Demo භාවිතා කරන්න

```html
<c-webc-js src="/src/js/dom.js" href="/doc/{code}/js/dom.md"></c-webc-js>
```

## දේපල පරාමිතීන්

- `src`: මූලාශ්‍ර කේතයේ URL ලිපිනය.
- `href`: ලේඛනයේ URL ලිපිනය සලකුණු කිරීම, වත්මන් භාෂාවට අනුවර්තනය වීමට `{code}` ස්ථාන දරන්නා භාවිතා කිරීමට සහය දක්වයි.
