# Komponentləri və ya alət funksiyalarını göstərmək üçün sənədlər və mənbə kodu

- Solda göstərilən sənədlər və sağda göstərilən mənbə kodu ilə iki sütunlu cavab verən tərtibat
- Sənədləşdirmə və mənbə kodu mənbə ünvanına keçmək üçün Redaktə və Kod qısayol düymələrini dəstəkləyir.
- Başlıq və giriş yuxarıda göstərilir, i18n avtomatik tərcüməsini dəstəkləyir.

## Demodan istifadə edin

```html
<c-webc-js src="/src/js/dom.js" href="/doc/{code}/js/dom.md"></c-webc-js>
```

## Mülkiyyət parametrləri

- `src`: mənbə kodunun URL ünvanı.
- `href`: sənəd qeydinin URL ünvanı, cari dilə uyğunlaşmaq üçün `{code}` yertutandan istifadəni dəstəkləyir.
