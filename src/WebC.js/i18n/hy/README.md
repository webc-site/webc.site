# Փաստաթղթեր և սկզբնական կոդը բաղադրիչների կամ գործիքի գործառույթների մատուցման համար

- Երկու սյունակ արձագանքող դասավորություն՝ ձախ կողմում ցուցադրված փաստաթղթերով, իսկ աջ կողմում՝ սկզբնաղբյուրային կոդով
- Փաստաթղթերը և սկզբնական կոդը աջակցում են «Խմբագրել» և «Կոդ» դյուրանցման կոճակները՝ սկզբնաղբյուր հասցեին անցնելու համար:
- Վերնագիրը և ներածությունը ցուցադրվում են վերևում՝ աջակցելով i18n ավտոմատ թարգմանությանը:

## Օգտագործեք ցուցադրությունը

```html
<c-webc-js src="/src/js/dom.js" href="/doc/{code}/js/dom.md"></c-webc-js>
```

## Գույքի պարամետրեր

- `src`՝ սկզբնաղբյուրի URL հասցեն:
- `href`. փաստաթղթի նշման URL հասցեն, աջակցում է `{code}` տեղապահի օգտագործումը ընթացիկ լեզվին հարմարվելու համար:
