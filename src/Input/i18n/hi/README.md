# इनपुट बॉक्स जो प्लेसहोल्डर स्थिति को स्वचालित रूप से स्विच करता है

- दर्ज न होने पर प्लेसहोल्डर दिखाएं
- सामग्री टाइप करते समय प्लेसहोल्डर आसानी से स्केल करते हैं और ऊपर तैरते हैं
- शुद्ध सीएसएस कार्यान्वयन, कोई जावास्क्रिप्ट निर्भरता नहीं
- फ्रॉस्टेड ग्लास बैकग्राउंड और होवर ट्रांज़िशन के साथ डिफ़ॉल्ट

## डेमो का प्रयोग करें

```html
<b class="Input Lg">
  <input id="email" type="text" placeholder=" ">
  <label for="email">ईमेल</label>
</b>
```

## संरचनात्मक आवश्यकताएँ

- कंटेनर: इनपुट बॉक्स और लेबल होस्ट करने के लिए `.Input` वर्ग के तत्वों का उपयोग करें।
- इनपुट बॉक्स: `input` तत्व, `placeholder=" "` (एक प्लेसहोल्डर जिसमें एक स्थान होता है) को राज्य स्विच को ट्रिगर करने के लिए सेट किया जाना चाहिए।
- टिप टैग: `label` तत्व, `input` के तुरंत बाद।

## शैली वर्ग

- `.Input`: मूल शैली, इनपुट बॉक्स संरचना और संक्रमण एनीमेशन प्रदान करती है।
- `.Lg`: हाइलाइट/फ्रॉस्टेड ग्लास संशोधन शैली।
