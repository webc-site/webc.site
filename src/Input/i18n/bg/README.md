# Полето за въвеждане, което автоматично превключва състоянието на контейнера

- Показване на контейнер, когато не е въведен
- Заместителите плавно се мащабират и изплуват отгоре, когато въвеждате съдържание
- Чисто внедряване на CSS, без зависимости на JavaScript

## Използвайте демонстрацията

```html
<b class="Input Lg">
  <input id="email" type="text" placeholder=" ">
  <label for="email">Имейл</label>
</b>
```

## структурни изисквания

- Контейнер: Използвайте елементи от класа `.Input` за хостване на полета за въвеждане и етикети.
- Поле за въвеждане: `input` елемент, `placeholder=" "` (заместител, съдържащ интервал) трябва да бъде настроен, за да задейства превключването на състоянието.
- Таг за съвет: `label` елемент, непосредствено след `input`.

## стил клас

- `.Input`: Основен стил, осигуряващ структура на полето за въвеждане и анимация на прехода.
- `.Lg`: Стил на модификация на акценти/матово стъкло.
