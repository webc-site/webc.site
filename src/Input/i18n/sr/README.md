# Оквир за унос који аутоматски мења стање чувара места

- Прикажи чувар места када није унет
- Чувари места се глатко скалирају и лебде изнад када куцате садржај
- Чиста имплементација ЦСС-а, без ЈаваСцрипт зависности

## Користите демо

```html
<b class="Input Lg">
  <input id="email" type="text" placeholder=" ">
  <label for="email">Емаил</label>
</b>
```

## структурни захтеви

- Контејнер: Користите елементе класе <б ид="1"> за хостовање поља за унос и ознака.
- Поље за унос: <б ид="1"> елемент, <б ид="2"> (чувар места који садржи размак) мора бити подешен да покрене прекидач стања.
- Ознака савета: <б ид="1"> елемент, одмах после <б ид="2">.

## стилска класа

- <б ид="1">: Основни стил, пружа структуру поља за унос и анимацију прелаза.
- <б ид="1">: Стил модификације истицања/матирано стакло.
