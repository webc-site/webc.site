# Толтырғыш күйін автоматты түрде ауыстыратын енгізу терезесі

- Енгізілмеген кезде толтырғышты көрсету
- Мазмұнды теру кезінде толтырғыштар біркелкі масштабталады және жоғарыда өзгереді
- Таза CSS енгізу, JavaScript тәуелділігі жоқ

## Демонстрацияны пайдаланыңыз

```html
<b class="Input Lg">
  <input id="email" type="text" placeholder=" ">
  <label for="email">Электрондық пошта</label>
</b>
```

## құрылымдық талаптар

- Контейнер: енгізу өрістері мен белгілерді орналастыру үшін `.Input` класының элементтерін пайдаланыңыз.
- Енгізу жолағы: `input` элементі, `placeholder=" "` (бос орыны бар толтырғыш) күй ауыстырғышын іске қосу үшін орнатылуы керек.
- Кеңес тегі: `label` элементі, бірден `input` кейін.

## стиль сыныбы

- `.Input`: енгізу жолағы құрылымын және өту анимациясын қамтамасыз ететін негізгі стиль.
- `.Lg`: бөлектеу/мұздатылған әйнекті өзгерту стилі.
