# תיבת קלט שמחליפה אוטומטית מצב מציין מיקום

- הצג מציין מיקום כאשר לא הוזן
- מצייני מיקום משתנים בצורה חלקה ומרחפים מעל בעת הקלדת תוכן
- הטמעת CSS טהורה, ללא תלות ב-JavaScript

## השתמש בהדגמה

```html
<b class="Input Lg">
  <input id="email" type="text" placeholder=" ">
  <label for="email">אֶלֶקטרוֹנִי</label>
</b>
```

## דרישות מבניות

- מיכל: השתמש באלמנטים של המחלקה `.Input` כדי לארח תיבות קלט ותוויות.
- תיבת קלט: `input` אלמנט, `placeholder=" "` (מציין מיקום המכיל רווח) חייב להיות מוגדר כדי להפעיל את מתג המצב.
- תג טיפ: `label` אלמנט, מיד אחרי `input`.

## שיעור סגנון

- `.Input`: סגנון בסיסי, מספק מבנה תיבת קלט ואנימציית מעבר.
- `.Lg`: סגנון שינוי הדגשה/זכוכית חלבית.
