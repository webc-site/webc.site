# ইনপুট বাক্স যা স্বয়ংক্রিয়ভাবে স্থানধারক অবস্থা পরিবর্তন করে

- প্রবেশ না করলে স্থানধারক দেখান
- কন্টেন্ট টাইপ করার সময় প্লেসহোল্ডার মসৃণভাবে স্কেল করে এবং উপরে ভাসতে থাকে
- বিশুদ্ধ CSS বাস্তবায়ন, কোন জাভাস্ক্রিপ্ট নির্ভরতা নেই
- ফ্রস্টেড গ্লাস ব্যাকগ্রাউন্ড এবং হোভার ট্রানজিশন সহ ডিফল্ট

## ডেমো ব্যবহার করুন

```html
<b class="Input Lg">
  <input id="email" type="text" placeholder=" ">
  <label for="email">ইমেইল</label>
</b>
```

## কাঠামোগত প্রয়োজনীয়তা

- ধারক: ইনপুট বাক্স এবং লেবেল হোস্ট করতে `.Input` ক্লাসের উপাদানগুলি ব্যবহার করুন৷
- ইনপুট বাক্স: `input` উপাদান, `placeholder=" "` (একটি স্থান ধারণকারী স্থানধারক) অবশ্যই স্টেট সুইচ ট্রিগার করতে সেট করতে হবে।
- টিপ ট্যাগ: `label` উপাদান, `input` এর পরপরই।

## শৈলী ক্লাস

- `.Input`: মৌলিক শৈলী, ইনপুট বক্স গঠন এবং ট্রানজিশন অ্যানিমেশন প্রদান করে।
- `.Lg`: হাইলাইট/ফ্রস্টেড গ্লাস পরিবর্তন শৈলী।
