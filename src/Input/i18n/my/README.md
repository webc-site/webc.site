# နေရာယူမှုအခြေအနေကို အလိုအလျောက်ပြောင်းပေးသော အကွက်

- မဝင်သည့်အခါ နေရာထိန်းသူကို ပြပါ။
- အကြောင်းအရာကို စာရိုက်သောအခါတွင် နေရာကိုင်ဆောင်သူများသည် ချောမွေ့စွာ စကေးနှင့် မျှောနေပါသည်။
- သန့်ရှင်းသော CSS အကောင်အထည်ဖော်မှု၊ JavaScript မှီခိုမှုမရှိပါ။

## ဒီမိုကိုသုံးပါ။

```html
<b class="Input Lg">
  <input id="email" type="text" placeholder=" ">
  <label for="email">အီးမေးလ်</label>
</b>
```

## ဖွဲ့စည်းပုံလိုအပ်ချက်များ

- ကွန်တိန်နာ- ထည့်သွင်းမှုသေတ္တာများနှင့် အညွှန်းများကို လက်ခံကျင်းပရန် `.Input`အတန်း၏ အစိတ်အပိုင်းများကို အသုံးပြုပါ။
- ထည့်သွင်းမှုအကွက်- `input`ဒြပ်စင်၊ `placeholder=" "` (နေရာလွတ်တစ်ခုပါရှိသော နေရာကိုင်ဆောင်သူ) ကို ပြည်နယ်ခလုတ်ကို စတင်ရန် သတ်မှတ်ရပါမည်။
- အကြံပြုချက်- `label`ဒြပ်စင် `input` ပြီးနောက် ချက်ချင်း။

## စတိုင်အတန်း

- `.Input`- အခြေခံပုံစံ၊ ထည့်သွင်းဘောက်စ်ဖွဲ့စည်းပုံနှင့် အသွင်ကူးပြောင်းမှု ကာတွန်းကို ပံ့ပိုးပေးသည်။
- `.Lg`- မီးမောင်းထိုးပြခြင်း/နှင်းခဲမှန် ပြုပြင်မွမ်းမံမှုပုံစံ။
