# ස්ථාන දරණ තත්ත්වය ස්වයංක්‍රීයව මාරු කරන ආදාන පෙට්ටිය

- ඇතුළත් නොකළ විට ස්ථාන දරන්නා පෙන්වන්න
- අන්තර්ගතය ටයිප් කරන විට ස්ථාන දරන්නන් සුමටව පරිමාණය කර ඉහළින් පාවෙයි
- පිරිසිදු CSS ක්‍රියාත්මක කිරීම, JavaScript පරායත්තතා නොමැත

## Demo භාවිතා කරන්න

```html
<b class="Input Lg">
  <input id="email" type="text" placeholder=" ">
  <label for="email">ඊමේල් කරන්න</label>
</b>
```

## ව්යුහාත්මක අවශ්යතා

- බහාලුම්: ආදාන කොටු සහ ලේබල සත්කාරක කිරීමට `.Input` පන්තියේ මූලද්‍රව්‍ය භාවිතා කරන්න.
- ආදාන පෙට්ටිය: `input` මූලද්‍රව්‍යය, `placeholder=" "` (හිස්තැනක් අඩංගු තැන් දරන්නා) රාජ්‍ය ස්විචය ක්‍රියාරම්භ කිරීමට සැකසිය යුතුය.
- ඉඟි ටැගය: `label` මූලද්‍රව්‍යය, `input`ට පසු වහාම.

## ශෛලිය පන්තිය

- `.Input`: මූලික විලාසය, ආදාන කොටු ව්‍යුහය සහ සංක්‍රාන්ති සජීවිකරණය සපයයි.
- `.Lg`: Highlight/frosted glass modification style.
