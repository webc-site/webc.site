# შეყვანის ველი, რომელიც ავტომატურად ცვლის ადგილსამყოფელის მდგომარეობას

- ჩანაცვლების ჩანაცვლების ჩვენება, როდესაც არ არის შეყვანილი
- პლაცჰოლდერები შეუფერხებლად მასშტაბირებენ და ცურავს ზემოთ შინაარსის აკრეფისას
- სუფთა CSS იმპლემენტაცია, JavaScript-ის დამოკიდებულების გარეშე

## გამოიყენეთ დემო ვერსია

```html
<b class="Input Lg">
  <input id="email" type="text" placeholder=" ">
  <label for="email">ელფოსტა</label>
</b>
```

## სტრუქტურული მოთხოვნები

- კონტეინერი: გამოიყენეთ `.Input` კლასის ელემენტები შეყვანის ველებისა და ლეიბლების განსათავსებლად.
- შეყვანის ველი: `input` ელემენტი, `placeholder=" "` (ჩანაცვლების ველი, რომელიც შეიცავს სივრცეს) უნდა იყოს დაყენებული მდგომარეობის გადამრთველის გასააქტიურებლად.
- რჩევის ტეგი: `label` ელემენტი, `input`-ის შემდეგ.

## სტილის კლასი

- `.Input`: ძირითადი სტილი, რომელიც უზრუნველყოფს შეყვანის ყუთის სტრუქტურას და გარდამავალ ანიმაციას.
- `.Lg`: მონიშვნა/მოყინული მინის მოდიფიკაციის სტილი.
