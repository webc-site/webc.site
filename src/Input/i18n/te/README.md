# ప్లేస్‌హోల్డర్ స్థితిని స్వయంచాలకంగా మార్చే ఇన్‌పుట్ బాక్స్

- నమోదు చేయనప్పుడు ప్లేస్‌హోల్డర్‌ని చూపండి
- కంటెంట్‌ను టైప్ చేస్తున్నప్పుడు ప్లేస్‌హోల్డర్‌లు సజావుగా స్కేల్ చేయబడతాయి మరియు పైకి తేలుతాయి
- స్వచ్ఛమైన CSS అమలు, జావాస్క్రిప్ట్ డిపెండెన్సీలు లేవు

## డెమో ఉపయోగించండి

```html
<b class="Input Lg">
  <input id="email" type="text" placeholder=" ">
  <label for="email">ఇమెయిల్</label>
</b>
```

## నిర్మాణ అవసరాలు

- కంటైనర్: ఇన్‌పుట్ బాక్స్‌లు మరియు లేబుల్‌లను హోస్ట్ చేయడానికి `.Input` తరగతి మూలకాలను ఉపయోగించండి.
- ఇన్‌పుట్ బాక్స్: `input` మూలకం, `placeholder=" "` (స్పేస్‌ని కలిగి ఉన్న ప్లేస్‌హోల్డర్) స్టేట్ స్విచ్‌ను ట్రిగ్గర్ చేయడానికి తప్పనిసరిగా సెట్ చేయాలి.
- చిట్కా ట్యాగ్: `label` మూలకం, `input` తర్వాత వెంటనే.

## శైలి తరగతి

- `.Input`: ప్రాథమిక శైలి, ఇన్‌పుట్ బాక్స్ నిర్మాణం మరియు పరివర్తన యానిమేషన్‌ను అందిస్తుంది.
- `.Lg`: హైలైట్/ఫ్రాస్టెడ్ గ్లాస్ సవరణ శైలి.
