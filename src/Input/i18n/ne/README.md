# स्वचालित रूपमा प्लेसहोल्डर स्थिति स्विच गर्ने इनपुट बाकस

- प्रविष्ट नगरिएको बेला प्लेसहोल्डर देखाउनुहोस्
- सामग्री टाइप गर्दा प्लेसहोल्डरहरू सजिलै मापन र माथि फ्लोट हुन्छन्
- शुद्ध CSS कार्यान्वयन, कुनै जाभास्क्रिप्ट निर्भरता छैन

## डेमो प्रयोग गर्नुहोस्

```html
<b class="Input Lg">
  <input id="email" type="text" placeholder=" ">
  <label for="email">इमेल</label>
</b>
```

## संरचनात्मक आवश्यकताहरू

- कन्टेनर: इनपुट बाकस र लेबलहरू होस्ट गर्न `.Input` वर्गका तत्वहरू प्रयोग गर्नुहोस्।
- इनपुट बक्स: `input` तत्व, `placeholder=" "` (स्पेस भएको प्लेसहोल्डर) राज्य स्विच ट्रिगर गर्न सेट हुनुपर्छ।
- सुझाव ट्याग: `label` तत्व, तुरुन्तै `input` पछि।

## शैली वर्ग

- `.Input`: आधारभूत शैली, इनपुट बक्स संरचना र संक्रमण एनिमेसन प्रदान गर्दै।
- `.Lg`: हाइलाइट/फ्रोस्टेड ग्लास परिमार्जन शैली।
