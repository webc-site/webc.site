# مربع الإدخال الذي يقوم بتبديل حالة العنصر النائب تلقائيًا

- إظهار العنصر النائب عند عدم إدخاله
- يتم تغيير حجم العناصر النائبة بسلاسة وتطفو في الأعلى عند كتابة المحتوى
- تنفيذ CSS خالص، لا توجد تبعيات لجافا سكريبت
- الافتراضي مع خلفية زجاجية بلوري وانتقال التحويم

## استخدم العرض التوضيحي

```html
<b class="Input Lg">
  <input id="email" type="text" placeholder=" ">
  <label for="email">بريد إلكتروني</label>
</b>
```

## المتطلبات الهيكلية

- الحاوية: استخدم عناصر فئة `.Input` لاستضافة صناديق الإدخال والتسميات.
- مربع الإدخال: يجب تعيين عنصر `input`، `placeholder=" "` (عنصر نائب يحتوي على مسافة) لتشغيل مفتاح الحالة.
- علامة نصيحة: عنصر `label`، مباشرة بعد `input`.

## فئة النمط

- `.Input`: النمط الأساسي، الذي يوفر بنية مربع الإدخال والرسوم المتحركة الانتقالية.
- `.Lg`: نمط تعديل الزجاج المظلل/المتجمد.
