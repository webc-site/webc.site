# Πλαίσιο εισαγωγής που αλλάζει αυτόματα την κατάσταση κράτησης θέσης

- Εμφάνιση κράτησης θέσης όταν δεν έχει εισαχθεί
- Τα σύμβολα θέσης κλιμακώνονται ομαλά και αιωρούνται επάνω κατά την πληκτρολόγηση περιεχομένου
- Καθαρή υλοποίηση CSS, χωρίς εξαρτήσεις JavaScript

## Χρησιμοποιήστε το demo

```html
<b class="Input Lg">
  <input id="email" type="text" placeholder=" ">
  <label for="email">E-mail</label>
</b>
```

## δομικές απαιτήσεις

- Container: Χρησιμοποιήστε στοιχεία της κλάσης `.Input` για να φιλοξενήσετε πλαίσια εισαγωγής και ετικέτες.
- Πλαίσιο εισαγωγής: Το στοιχείο `input`, το `placeholder=" "` (ένα σύμβολο κράτησης θέσης που περιέχει ένα διάστημα) πρέπει να ρυθμιστεί για να ενεργοποιήσει τον διακόπτη κατάστασης.
- Ετικέτα συμβουλής: στοιχείο `label`, αμέσως μετά το `input`.

## τάξη στυλ

- `.Input`: Βασικό στυλ, που παρέχει δομή πλαισίου εισόδου και κινούμενα σχέδια μετάβασης.
- `.Lg`: Στυλ τροποποίησης επισήμανσης/γυαλιού.
