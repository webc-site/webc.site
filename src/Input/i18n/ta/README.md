# உள்ளீட்டு பெட்டி தானாகவே ஒதுக்கிட நிலையை மாற்றும்

- உள்ளிடாத போது ஒதுக்கிடத்தைக் காட்டு
- உள்ளடக்கத்தை தட்டச்சு செய்யும் போது பெட்டிகள் சீராக அளவிடப்பட்டு மேலே மிதக்கும்
- தூய CSS செயல்படுத்தல், JavaScript சார்புகள் இல்லை
- உறைந்த கண்ணாடி பின்னணி மற்றும் மிதவை மாற்றத்துடன் இயல்புநிலை

## டெமோவைப் பயன்படுத்தவும்

```html
<b class="Input Lg">
  <input id="email" type="text" placeholder=" ">
  <label for="email">மின்னஞ்சல்</label>
</b>
```

## கட்டமைப்பு தேவைகள்

- கொள்கலன்: உள்ளீட்டு பெட்டிகள் மற்றும் லேபிள்களை ஹோஸ்ட் செய்ய `.Input` வகுப்பின் கூறுகளைப் பயன்படுத்தவும்.
- உள்ளீட்டுப் பெட்டி: `input` உறுப்பு, `placeholder=" "` (இடைவெளியைக் கொண்ட ஒதுக்கிடமானது) நிலை மாறுதலைத் தூண்டுவதற்கு அமைக்கப்பட வேண்டும்.
- குறிப்பு குறிச்சொல்: `label` உறுப்பு, உடனடியாக `input`க்குப் பிறகு.

## பாணி வகுப்பு

- `.Input`: அடிப்படை நடை, உள்ளீட்டு பெட்டி அமைப்பு மற்றும் மாற்றம் அனிமேஷனை வழங்குகிறது.
- `.Lg`: ஹைலைட்/உறைந்த கண்ணாடி மாற்றியமைக்கும் பாணி.
