# ಪ್ಲೇಸ್‌ಹೋಲ್ಡರ್ ಸ್ಥಿತಿಯನ್ನು ಸ್ವಯಂಚಾಲಿತವಾಗಿ ಬದಲಾಯಿಸುವ ಇನ್‌ಪುಟ್ ಬಾಕ್ಸ್

- ನಮೂದಿಸದಿದ್ದಾಗ ಪ್ಲೇಸ್‌ಹೋಲ್ಡರ್ ತೋರಿಸಿ
- ವಿಷಯವನ್ನು ಟೈಪ್ ಮಾಡುವಾಗ ಪ್ಲೇಸ್‌ಹೋಲ್ಡರ್‌ಗಳು ಸರಾಗವಾಗಿ ಅಳೆಯುತ್ತವೆ ಮತ್ತು ಮೇಲೆ ತೇಲುತ್ತವೆ
- ಶುದ್ಧ CSS ಅನುಷ್ಠಾನ, ಯಾವುದೇ JavaScript ಅವಲಂಬನೆಗಳಿಲ್ಲ
- ಫ್ರಾಸ್ಟೆಡ್ ಗ್ಲಾಸ್ ಹಿನ್ನೆಲೆ ಮತ್ತು ಹೋವರ್ ಪರಿವರ್ತನೆಯೊಂದಿಗೆ ಡೀಫಾಲ್ಟ್

## ಡೆಮೊ ಬಳಸಿ

```html
<b class="Input Lg">
  <input id="email" type="text" placeholder=" ">
  <label for="email">ಇಮೇಲ್</label>
</b>
```

## ರಚನಾತ್ಮಕ ಅವಶ್ಯಕತೆಗಳು

- ಕಂಟೈನರ್: ಇನ್‌ಪುಟ್ ಬಾಕ್ಸ್‌ಗಳು ಮತ್ತು ಲೇಬಲ್‌ಗಳನ್ನು ಹೋಸ್ಟ್ ಮಾಡಲು `.Input` ವರ್ಗದ ಅಂಶಗಳನ್ನು ಬಳಸಿ.
- ಇನ್‌ಪುಟ್ ಬಾಕ್ಸ್: `input` ಅಂಶ, `placeholder=" "` (ಸ್ಪೇಸ್ ಹೊಂದಿರುವ ಪ್ಲೇಸ್‌ಹೋಲ್ಡರ್) ಸ್ಟೇಟ್ ಸ್ವಿಚ್ ಅನ್ನು ಟ್ರಿಗರ್ ಮಾಡಲು ಹೊಂದಿಸಬೇಕು.
- ಸಲಹೆ ಟ್ಯಾಗ್: `label` ಅಂಶ, `input` ನಂತರ ತಕ್ಷಣವೇ.

## ಶೈಲಿ ವರ್ಗ

- `.Input`: ಮೂಲ ಶೈಲಿ, ಇನ್‌ಪುಟ್ ಬಾಕ್ಸ್ ರಚನೆ ಮತ್ತು ಪರಿವರ್ತನೆಯ ಅನಿಮೇಷನ್ ಅನ್ನು ಒದಗಿಸುತ್ತದೆ.
- `.Lg`: ಹೈಲೈಟ್/ಫ್ರಾಸ್ಟೆಡ್ ಗ್ಲಾಸ್ ಮಾರ್ಪಾಡು ಶೈಲಿ.
