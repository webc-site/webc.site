# इनपुट बॉक्स जो आपोआप प्लेसहोल्डर स्थिती बदलतो

- प्रविष्ट केलेले नसताना प्लेसहोल्डर दर्शवा
- सामग्री टाइप करताना प्लेसहोल्डर सहजतेने स्केल करतात आणि वर फ्लोट करतात
- शुद्ध CSS अंमलबजावणी, JavaScript अवलंबित्व नाही
- फ्रॉस्टेड ग्लास पार्श्वभूमी आणि होव्हर संक्रमणासह डीफॉल्ट

## डेमो वापरा

```html
<b class="Input Lg">
  <input id="email" type="text" placeholder=" ">
  <label for="email">ईमेल</label>
</b>
```

## संरचनात्मक आवश्यकता

- कंटेनर: इनपुट बॉक्स आणि लेबल होस्ट करण्यासाठी `.Input` वर्गाचे घटक वापरा.
- इनपुट बॉक्स: `input` घटक, `placeholder=" "` (स्पेस असलेले प्लेसहोल्डर) स्टेट स्विच ट्रिगर करण्यासाठी सेट करणे आवश्यक आहे.
- टीप टॅग: `label` घटक, `input` नंतर लगेच.

## शैली वर्ग

- `.Input`: मूलभूत शैली, इनपुट बॉक्स संरचना आणि संक्रमण ॲनिमेशन प्रदान करते.
- `.Lg`: हायलाइट/फ्रॉस्टेड ग्लास सुधारणा शैली.
