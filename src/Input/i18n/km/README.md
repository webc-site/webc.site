# бһ”бҹ’бһҡбһўбһ”бҹӢбһ”бһүбҹ’бһ…бһјбһӣбһҠбҹӮбһӣбһ”бҹ’бһҸбһјбһҡбһҹбҹ’бһҗбһ¶бһ“бһ—бһ¶бһ–бһҖбһ“бҹ’бһӣбҹӮбһ„бһҠбһ¶бһҖбҹӢбһҠбҹ„бһҷбһҹбҹ’бһңбҹҗбһҷбһ”бҹ’бһҡбһңбһҸбҹ’бһҸбһ·

- бһ”бһ„бҹ’бһ бһ¶бһүбһҖбһ“бҹ’бһӣбҹӮбһ„бһҠбһ¶бһҖбҹӢбһ“бҹ…бһ–бҹҒбһӣбһҳбһ·бһ“бһ”бһ¶бһ“бһ”бһүбҹ’бһ…бһјбһӣ
- бһҖбһ“бҹ’бһӣбҹӮбһ„бһҠбһ¶бһҖбҹӢбһҖбһҳбҹ’бһҡбһ·бһҸбһҷбҹүбһ¶бһ„бһҡбһӣбһјбһ“ бһ“бһ·бһ„бһўбһҺбҹ’бһҸбҹӮбһҸбһ–бһёбһӣбһҫбһ“бҹ…бһ–бҹҒбһӣбһңбһ¶бһҷбһҳбһ¶бһҸбһ·бһҖбһ¶
- бһҖбһ¶бһҡбһўбһ“бһ»бһңбһҸбҹ’бһҸ CSS бһҹбһ»бһ‘бҹ’бһ’ бһӮбҹ’бһҳбһ¶бһ“ JavaScript бһўбһ¶бһҹбҹ’бһҡбҹҗбһҷ

## бһ”бҹ’бһҡбһҫбһҖбһ¶бһҡбһ”бһ„бҹ’бһ бһ¶бһү

```html
<b class="Input Lg">
  <input id="email" type="text" placeholder=" ">
  <label for="email">бһўбҹҠбһёбһҳбҹӮбһӣ</label>
</b>
```

## бһҸбһҳбҹ’бһҡбһјбһңбһҖбһ¶бһҡбһҡбһ…бһ“бһ¶бһҹбһҳбҹ’бһ–бҹҗбһ“бҹ’бһ’

- бһҖбһ»бһ„бһҸбһәбһ“бҹҗбһҡбҹ– бһ”бҹ’бһҡбһҫбһ’бһ¶бһҸбһ»бһ“бҹғбһҗбҹ’бһ“бһ¶бһҖбҹӢ `.Input` бһҠбһҫбһҳбҹ’бһ”бһёбһҡбҹҖбһ”бһ…бҹҶбһ”бҹ’бһҡбһўбһ”бҹӢбһ”бһүбҹ’бһ…бһјбһӣ бһ“бһ·бһ„бһҹбҹ’бһӣбһ¶бһҖбҹ”
- бһ”бҹ’бһҡбһўбһ”бҹӢбһ”бһүбҹ’бһ…бһјбһӣбҹ– `input`бһ’бһ¶бһҸбһ» `placeholder=" "` (бһҖбһ“бҹ’бһӣбҹӮбһ„бһҠбһ¶бһҖбҹӢбһҠбҹӮбһӣбһҳбһ¶бһ“бһҠбһҖбһғбҹ’бһӣбһ¶) бһҸбҹ’бһҡбһјбһңбһҸбҹӮбһҖбҹҶбһҺбһҸбҹӢбһҠбһҫбһҳбҹ’бһ”бһёбһҖбҹҒбҹҮбһҖбһ»бһ„бһҸбһ¶бһҖбҹӢбһҡбһҠбҹ’бһӢбҹ”
- бһҹбҹ’бһӣбһ¶бһҖбһӮбһ“бҹ’бһӣбһ№бҹҮбҹ– `label` бһ’бһ¶бһҸбһ» бһ—бҹ’бһӣбһ¶бһҳбҹ—бһ”бһ“бҹ’бһ‘бһ¶бһ”бҹӢбһ–бһё `input`бҹ”

## бһҗбҹ’бһ“бһ¶бһҖбҹӢбһҡбһ…бһ“бһ¶бһ”бҹҗбһ‘бҹ’бһҳ

- `.Input`бҹ– бһҡбһ…бһ“бһ¶бһ”бҹҗбһ‘бҹ’бһҳбһҳбһјбһӣбһҠбҹ’бһӢбһ¶бһ“ бһ•бҹ’бһҸбһӣбҹӢбһ“бһјбһңбһҡбһ…бһ“бһ¶бһҹбһҳбҹ’бһ–бҹҗбһ“бҹ’бһ’бһ”бҹ’бһҡбһўбһ”бҹӢбһ”бһүбҹ’бһ…бһјбһӣ бһ“бһ·бһ„бһ…бһӣбһ“бһ¶бһ•бҹ’бһӣбһ¶бһҹбҹӢбһ”бҹ’бһҸбһјбһҡбҹ”
- `.Lg`бҹ– бһ”бһ“бҹ’бһӣбһ·бһ…/бһҡбһ…бһ“бһ¶бһ”бҹҗбһ‘бҹ’бһҳвҖӢбһҖбһ¶бһҡвҖӢбһҖбҹӮбһ”бҹ’бһҡбҹӮвҖӢбһҖбһүбҹ’бһ…бһҖбҹӢвҖӢбһҹбһ¶бһҷбҹ”
