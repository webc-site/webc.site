# Ներածման տուփ, որն ավտոմատ կերպով փոխում է տեղապահի վիճակը

- Ցուցադրել տեղապահը, երբ մուտքագրված չէ
- Տեղապահները սահուն չափվում և լողում են վերևում՝ բովանդակություն մուտքագրելիս
- Մաքուր CSS իրականացում, JavaScript-ի կախվածություն չկա

## Օգտագործեք ցուցադրությունը

```html
<b class="Input Lg">
  <input id="email" type="text" placeholder=" ">
  <label for="email">Էլ</label>
</b>
```

- Container. Օգտագործեք `.Input` դասի տարրերը մուտքագրման տուփերը և պիտակները տեղադրելու համար:
- Մուտքագրման տուփ. `input` տարրը, `placeholder=" "` (բացատ պարունակող տեղապահ) պետք է սահմանվի վիճակի անջատիչը գործարկելու համար:
