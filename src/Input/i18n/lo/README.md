# ກ່ອງປ້ອນຂໍ້ມູນທີ່ປ່ຽນສະຖານະຕົວຍຶດອັດຕະໂນມັດ

- ສະແດງຕົວຍຶດເມື່ອບໍ່ໄດ້ເຂົ້າ
- ການຈັດວາງຈະປັບຂະ ໜາດ ແລະລອຍຢູ່ເທິງສຸດເມື່ອພິມເນື້ອຫາ
- ການປະຕິບັດ CSS ບໍລິສຸດ, ບໍ່ມີການຂຶ້ນກັບ JavaScript

## ໃຊ້ຕົວຢ່າງ

```html
<b class="Input Lg">
  <input id="email" type="text" placeholder=" ">
  <label for="email">ອີເມວ</label>
</b>
```

## ຄວາມຕ້ອງການໂຄງສ້າງ

- Container: ໃຊ້ອົງປະກອບຂອງຫ້ອງຮຽນ `.Input`ເພື່ອໂຮດກ່ອງປ້ອນຂໍ້ມູນ ແລະປ້າຍກຳກັບ.
- ກ່ອງປ້ອນຂໍ້ມູນ: `input`ອົງປະກອບ, `placeholder=" "` (ຕົວຍຶດບ່ອນທີ່ມີຊ່ອງຫວ່າງ) ຕ້ອງຖືກຕັ້ງເພື່ອກະຕຸ້ນສະວິດລັດ.
- ແທັກເຄັດລັບ: `label`ອົງປະກອບ, ທັນທີຫຼັງຈາກ `input`.

## ຫ້ອງຮຽນແບບ

- `.Input`: ຮູບແບບພື້ນຖານ, ສະໜອງໂຄງສ້າງກ່ອງປ້ອນຂໍ້ມູນ ແລະພາບເຄື່ອນໄຫວໃນການປ່ຽນແປງ.
- `.Lg`: ຮູບແບບການດັດແປງແກ້ວທີ່ມີຈຸດເດັ່ນ/ໜາວເຢັນ.
