# 자리 표시자 상태를 자동으로 전환하는 입력 상자

- 입력되지 않은 경우 자리 표시자 표시
- 콘텐츠를 입력할 때 자리 표시자가 부드럽게 크기를 조정하고 위에 떠 있습니다.
- 순수한 CSS 구현, JavaScript 종속성 없음
- 반투명 유리 배경 및 호버 전환이 포함된 기본값

## 데모 사용

```html
<b class="Input Lg">
  <input id="email" type="text" placeholder=" ">
  <label for="email">이메일</label>
</b>
```

## 구조적 요구 사항

- 컨테이너: `.Input` 클래스의 요소를 사용하여 입력 상자와 레이블을 호스팅합니다.
- 입력 상자: `input` 요소, `placeholder=" "`(공백이 포함된 자리 표시자)는 상태 전환을 트리거하도록 설정되어야 합니다.
- 팁 태그: `label` 요소, `input` 바로 뒤.

## 스타일 수업

- `.Input`: 기본 스타일, 입력 상자 구조 및 전환 애니메이션을 제공합니다.
- `.Lg`: 하이라이트/반투명 유리 수정 스타일.
