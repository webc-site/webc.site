# ان پٽ باڪس جيڪو خودڪار طريقي سان جڳه دار رياست کي تبديل ڪري ٿو

- جڏهن داخل نه ٿيو ته جڳهه هولڊر ڏيکاريو
- جڳھ ھولڊر آسانيءَ سان ماپ ڪن ٿا ۽ مٿي فلوٽ ڪن ٿا جڏھن مواد ٽائپ ڪن ٿا
- خالص CSS عمل درآمد، جاوا اسڪرپٽ انحصار نه

## ڊيمو استعمال ڪريو

```html
<b class="Input Lg">
  <input id="email" type="text" placeholder=" ">
  <label for="email">اي ميل</label>
</b>
```

## ساخت جي گهرج

- ڪنٽينر: ان پٽ باڪس ۽ ليبلز کي ميزباني ڪرڻ لاءِ `.Input` ڪلاس جا عنصر استعمال ڪريو.
- ان پٽ باڪس: `input` عنصر، `placeholder=" "` (هڪ جاءِ هولڊر جنهن ۾ اسپيس شامل هجي) لازمي طور تي اسٽيٽ سوئچ کي ٽرگر ڪرڻ لاءِ مقرر ڪيو وڃي.
- ٽپڻي ٽيگ: `label` عنصر، فوري بعد `input`.

## انداز ڪلاس

- `.Input`: بنيادي انداز، ان پٽ باڪس جي جوڙجڪ ۽ منتقلي اينيميشن مهيا ڪرڻ.
- `.Lg`: نمايان/فروسٽ ٿيل شيشي جي ترميمي انداز.
