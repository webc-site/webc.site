# ઇનપુટ બોક્સ જે આપમેળે પ્લેસહોલ્ડર સ્ટેટને સ્વિચ કરે છે

- જ્યારે દાખલ ન હોય ત્યારે પ્લેસહોલ્ડર બતાવો
- સામગ્રી ટાઇપ કરતી વખતે પ્લેસહોલ્ડર્સ સરળતાથી સ્કેલ કરે છે અને ઉપર ફ્લોટ કરે છે
- શુદ્ધ CSS અમલીકરણ, કોઈ JavaScript નિર્ભરતા નથી

## ડેમોનો ઉપયોગ કરો

```html
<b class="Input Lg">
  <input id="email" type="text" placeholder=" ">
  <label for="email">ઈમેલ</label>
</b>
```

## માળખાકીય જરૂરિયાતો

- કન્ટેનર: ઇનપુટ બોક્સ અને લેબલ્સને હોસ્ટ કરવા માટે `.Input` વર્ગના ઘટકોનો ઉપયોગ કરો.
- ઇનપુટ બોક્સ: `input` એલિમેન્ટ, `placeholder=" "` (એક જગ્યા ધરાવતું પ્લેસહોલ્ડર) સ્ટેટ સ્વિચને ટ્રિગર કરવા માટે સેટ કરવું આવશ્યક છે.
- ટીપ ટેગ: `label` તત્વ, `input` પછી તરત જ.

## શૈલી વર્ગ

- `.Input`: મૂળભૂત શૈલી, ઇનપુટ બોક્સ માળખું અને સંક્રમણ એનિમેશન પ્રદાન કરે છે.
- `.Lg`: હાઇલાઇટ/ફ્રોસ્ટેડ ગ્લાસ ફેરફાર શૈલી.
