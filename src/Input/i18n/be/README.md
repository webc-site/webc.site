# Поле ўводу, якое аўтаматычна пераключае стан запаўняльніка

- Паказваць запаўняльнік, калі ён не ўведзены
- Запаўняльнікі плаўна маштабуюцца і ўсплываюць уверсе пры ўводзе змесціва
- Чыстая рэалізацыя CSS, ніякіх залежнасцей ад JavaScript

## Выкарыстоўвайце дэма

```html
<b class="Input Lg">
  <input id="email" type="text" placeholder=" ">
  <label for="email">Электронная пошта</label>
</b>
```

## структурныя патрабаванні

- Кантэйнер: выкарыстоўвайце элементы класа `.Input` для размяшчэння палёў уводу і этыкетак.
- Поле ўводу: элемент `input`, `placeholder=" "` (запаўняльнік, які змяшчае прабел) павінен быць усталяваны для запуску пераключэння стану.
- Тэг падказкі: элемент `label`, адразу пасля `input`.

## клас стылю

- `.Input`: Базавы стыль, забяспечваючы структуру поля ўводу і анімацыю пераходу.
- `.Lg`: Стыль мадыфікацыі падсветкі/матавага шкла.
