# Поле ввода, которое автоматически переключает состояние заполнителя

- Показывать заполнитель, если он не введен
- Заполнители плавно масштабируются и плавают над поверхностью при вводе контента.
- Чистая реализация CSS, без зависимостей JavaScript.
- По умолчанию с фоном из матового стекла и переходом при наведении

## Используйте демо-версию

```html
<b class="Input Lg">
  <input id="email" type="text" placeholder=" ">
  <label for="email">Электронная почта</label>
</b>
```

## структурные требования

- Контейнер: используйте элементы класса `.Input` для размещения полей ввода и меток.
- Поле ввода: элемент `input`, `placeholder=" "` (заполнитель, содержащий пробел) должен быть установлен для запуска переключения состояния.
- Тег подсказки: элемент `label`, сразу после `input`.

## класс стиля

- `.Input`: базовый стиль, обеспечивающий структуру поля ввода и анимацию перехода.
- `.Lg`: стиль модификации «Высветление/матовое стекло».
