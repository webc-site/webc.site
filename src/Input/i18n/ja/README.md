# プレースホルダーの状態を自動的に切り替える入力ボックス

- 入力されていない場合はプレースホルダーを表示
- コンテンツを入力すると、プレースホルダーが滑らかに拡大縮小して上に浮かび上がります
- 純粋な CSS 実装、JavaScript 依存関係なし
- すりガラスの背景とホバートランジションを備えたデフォルト

## デモを使用する

```html
<b class="Input Lg">
  <input id="email" type="text" placeholder=" ">
  <label for="email">電子メール</label>
</b>
```

## 構造要件

- コンテナ: `.Input` クラスの要素を使用して、入力ボックスとラベルをホストします。
- 入力ボックス: 状態の切り替えをトリガーするには、`input` 要素、`placeholder=" "` (スペースを含むプレースホルダー) を設定する必要があります。
- ヒントタグ: `input` の直後の `label` 要素。

## スタイルクラス

- `.Input`: 基本スタイル。入力ボックス構造と遷移アニメーションを提供します。
- `.Lg`: ハイライト/曇りガラスの変更スタイル。
