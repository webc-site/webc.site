# کادر ورودی که به طور خودکار وضعیت نگهدارنده مکان را تغییر می دهد

- هنگامی که وارد نشده است، مکان نگهدار را نشان دهید
- هنگام تایپ محتوا، متغیرهای مکان به آرامی مقیاس می شوند و در بالا شناور می شوند
- پیاده سازی CSS خالص، بدون وابستگی به جاوا اسکریپت
- پیش‌فرض با پس‌زمینه شیشه‌ای مات و انتقال شناور

## از دمو استفاده کنید

```html
<b class="Input Lg">
  <input id="email" type="text" placeholder=" ">
  <label for="email">ایمیل</label>
</b>
```

## الزامات ساختاری

- Container: از عناصر کلاس `.Input` برای میزبانی جعبه های ورودی و برچسب ها استفاده کنید.
- کادر ورودی: عنصر `input`، `placeholder=" "` (یک مکان نگهدار حاوی یک فاصله) باید برای فعال کردن سوئیچ حالت تنظیم شود.
- برچسب نکته: عنصر `label`، بلافاصله بعد از `input`.

## کلاس سبک

- `.Input`: سبک پایه، ارائه ساختار جعبه ورودی و انیمیشن انتقال.
- `.Lg`: سبک اصلاح برجسته/تصاویر شیشه مات.
