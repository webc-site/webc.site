# Поле за внесување што автоматски ја менува состојбата на место за место

- Прикажи место кога не е внесено
- Местата непречено се зголемуваат и лебдат над кога пишувате содржина
- Чиста имплементација на CSS, без зависност од JavaScript

## Користете го демо

```html
<b class="Input Lg">
  <input id="email" type="text" placeholder=" ">
  <label for="email">Е-пошта</label>
</b>
```

## структурни барања

- Контејнер: користете елементи од класата `.Input` за да ги хостирате влезните полиња и етикети.
- Поле за внесување: `input` елемент, `placeholder=" "` (местодржач што содржи празно место) мора да се постави за да го активира прекинувачот за состојбата.
- Ознака за совет: `label` елемент, веднаш по `input`.

## класа стил

- `.Input`: Основен стил, обезбедува структура на влезното поле и анимација на транзиција.
- `.Lg`: Стил на модификација на нагласување/матено стакло.
