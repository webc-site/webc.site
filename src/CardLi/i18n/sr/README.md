# Учитајте и прикажите листу картица асинхроно

- **Асинхроно учитавање**: Добијте податке преко асинхроне функције
- **Приказ статуса**: Подржава три стања: учитавање, празни подаци и картица за приказивање.
- **Флексибилни распоред**: Картице су поређане у редове

## Користите демо

```html
<script type="module">
  import CardLi from "webc.site/CardLi.js";

  const sleep = (ms) => new Promise((resolve) => setTimeout(resolve, ms));

  const card_li = CardLi(async () => {
    await sleep(2000);
    const data = [];
    for (let i = 1; i <= 13; ++i) {
      дата.пусх(["Наслов " + и, "Опис " + и, "#" + и]);
    }
    return data;
  });

  document.body.append(card_li);
</script>
```

## Параметри интерфејса

### CardLi(loader)

- **учитавач**: <б ид="1">, функција асинхроног учитавања података. Формат сваког поткључа је <б цласс="ЦОДЕ">[име, увод, веза]</б>.

## стилска класа

- <б ид="1">: спољни елемент
- <б ид="1">: елемент картице
- <б ид="1">: Учитавање анимираног елемента
