# Вчитајте и прикажувајте список со картички асинхроно

- **Асинхроно вчитување**: Добијте податоци преку асинхрона функција
- **Приказ на статус**: Поддржува три состојби: вчитување, празни податоци и картичка за прикажување.
- **Флексибилен распоред**: картичките се распоредени во редови

## Користете го демо

```html
<script type="module">
  import CardLi from "webc.site/CardLi.js";

  const sleep = (ms) => new Promise((resolve) => setTimeout(resolve, ms));

  const card_li = CardLi(async () => {
    await sleep(2000);
    const data = [];
    for (let i = 1; i <= 13; ++i) {
      data.push(["Наслов " + i, "Опис" + i, "#" + i]);
    }
    return data;
  });

  document.body.append(card_li);
</script>
```

## Параметри на интерфејсот

### CardLi(loader)

- **натоварувач**: `() => Promise<Array<[string, string, string]>>`, функција за асинхроно вчитување податоци. Форматот на секој потклуч е `[име, вовед, врска]`.

## класа стил

- `.CardLi`: надворешен елемент
- `.Card`: елемент на картичката
- `.Ing`: се вчитува анимиран елемент
