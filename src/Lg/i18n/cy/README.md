# Cefndir gwydr barugog ac effaith amlygu ymyl

Mae ffin lled-dryloyw a chysgod mewnol yn creu uchafbwyntiau ymyl micro-rhyddhad.
Fel dibyniaeth arddull sylfaenol cydrannau botwm a blwch mewnbwn.

## Defnyddiwch y demo

```html
<main class="Lg">
  <article>
    <h3>teitl</h3>
    <pre>Testun cynnwys</pre>
  </article>
</main>
```

## dosbarth arddull

### `.Lg`
Arddull sylfaenol y cerdyn. Defnyddiwch `backdrop-filter` niwl, `border` ffin boglynnog micro, a `box-shadow` cysgod mewnol. Gosod `svg/glass.svg#h` hidlydd ar hofran.

### `main.Lg`
Prif gynhwysydd bloc y dudalen. Corneli crwn `24px` gyda phadin ymatebol (`16px` os yw'r lled yn llai na `800px`, `24px` fel arall).

### `.Btn`, `.BtnC`, `.Input`
Enw dosbarth cydran sylfaenol. Ar ôl cyflwyno'r arddull hon, byddwch yn awtomatig yn cael cynllun hyblyg, perthnasoedd hierarchaidd a phriodweddau sylfaenol gwydr barugog.
