# ყინვაგამძლე მინის ფონი და კიდეების ხაზგასმის ეფექტი

ნახევრად გამჭვირვალე საზღვარი და შიდა ჩრდილი ქმნის მიკრო რელიეფურ ხაზებს.
როგორც ღილაკისა და შეყვანის ყუთის კომპონენტების ძირითადი სტილის დამოკიდებულება.

## გამოიყენეთ დემო ვერსია

```html
<main class="Lg">
  <article>
    <h3>სათაური</h3>
    <pre>შინაარსის ტექსტი</pre>
  </article>
</main>
```

## სტილის კლასი

### `.Lg`
ბარათის ძირითადი სტილი. გამოიყენეთ `backdrop-filter` ბუნდოვანი, `border` მიკრო რელიეფური საზღვრები და `box-shadow` შიდა ჩრდილი. გამოიყენეთ `svg/glass.svg#h` ფილტრი ჰოვერზე.

### `main.Lg`
გვერდის მთავარი ბლოკის კონტეინერი. მომრგვალებული კუთხეები `24px` საპასუხო შიგთავსით (`16px` თუ სიგანე `800px`-ზე ნაკლებია, `24px` წინააღმდეგ შემთხვევაში).

### `.Btn`, `.BtnC`, `.Input`
ძირითადი კომპონენტის კლასის სახელი. ამ სტილის დანერგვის შემდეგ თქვენ ავტომატურად მიიღებთ მოქნილ განლაგებას, იერარქიულ კავშირებს და ყინვაგამძლე მინის ძირითად თვისებებს.
