# Latar belakang kaca beku dan kesan sorotan tepi

Jidar separa lutsinar dan bayang dalam mencipta sorotan tepi pelega mikro.
Sebagai pergantungan gaya asas butang dan komponen kotak input.

## Gunakan demo

```html
<main class="Lg">
  <article>
    <h3>tajuk</h3>
    <pre>Teks kandungan</pre>
  </article>
</main>
```

## kelas gaya

### `.Lg`
Gaya asas kad. Gunakan `backdrop-filter` kabur, `border` jidar timbul mikro dan `box-shadow` bayang dalam. Gunakan penapis `svg/glass.svg#h` pada tuding.

### `main.Lg`
Bekas blok utama halaman. Sudut bulat `24px` dengan pelapik responsif (`16px` jika lebar kurang daripada `800px`, `24px` sebaliknya).

### `.Btn`, `.BtnC`, `.Input`
Nama kelas komponen asas. Selepas memperkenalkan gaya ini, anda secara automatik akan memperoleh susun atur fleksibel, hubungan hierarki dan sifat asas kaca beku.
