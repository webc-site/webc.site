# Φόντο από παγωμένο γυαλί και εφέ τονισμού των άκρων

Το ημιδιαφανές περίγραμμα και η εσωτερική σκιά δημιουργούν μικροανάγλυφες ανταύγειες στις άκρες.
Ως βασική εξάρτηση στυλ των στοιχείων του κουμπιού και του κουτιού εισόδου.

## Χρησιμοποιήστε το demo

```html
<main class="Lg">
  <article>
    <h3>τίτλος</h3>
    <pre>Κείμενο περιεχομένου</pre>
  </article>
</main>
```

## τάξη στυλ

### `.Lg`
Κάρτα βασικό στυλ. Εφαρμόστε `backdrop-filter` θόλωση, `border` μικροανάγλυφο περίγραμμα και `box-shadow` εσωτερική σκιά. Εφαρμόστε το φίλτρο `svg/glass.svg#h` στο δείκτη του ποντικιού.

### `main.Lg`
Το κύριο μπλοκ κοντέινερ της σελίδας. Στρογγυλεμένες γωνίες `24px` με αποκριτική επένδυση (`16px` εάν το πλάτος είναι μικρότερο από `800px`, `24px` διαφορετικά).

### `.Btn`, `.BtnC`, `.Input`
Όνομα κλάσης βασικού στοιχείου. Μετά την εισαγωγή αυτού του στυλ, θα αποκτήσετε αυτόματα ευέλικτη διάταξη, ιεραρχικές σχέσεις και βασικές ιδιότητες του παγωμένου γυαλιού.
