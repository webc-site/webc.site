# Ефект матового скла та підсвічування країв

Напівпрозора рамка і внутрішня тінь створюють мікрорельєфні краї.
Як базова залежність стилю компонентів кнопки та поля введення.

## Використовуйте демо

```html
<main class="Lg">
  <article>
    <h3>назва</h3>
    <pre>Текст змісту</pre>
  </article>
</main>
```

## клас стилю

### `.Lg`
Базовий стиль картки. Застосуйте `backdrop-filter` розмиття, `border` межу з мікротисненням і `box-shadow` внутрішню тінь. Застосувати фільтр `svg/glass.svg#h` при наведенні.

### `main.Lg`
Контейнер основного блоку сторінки. Заокруглені кути `24px` із адаптивним відступом (`16px`, якщо ширина менше `800px`, `24px` інакше).

### `.Btn`, `.BtnC`, `.Input`
Назва класу базового компонента. Після введення цього стилю ви автоматично отримаєте гнучке розташування, ієрархічні зв'язки та основні властивості матового скла.
