# พื้นหลังกระจกฝ้าและเอฟเฟกต์ไฮไลต์ขอบ

ขอบกึ่งโปร่งใสและเงาด้านในสร้างไฮไลท์ขอบนูนขนาดเล็ก
เป็นการพึ่งพารูปแบบพื้นฐานของปุ่มและส่วนประกอบกล่องอินพุต

## ใช้การสาธิต

```html
<main class="Lg">
  <article>
    <h3>ชื่อ</h3>
    <pre>ข้อความเนื้อหา</pre>
  </article>
</main>
```

## คลาสสไตล์

### `.Lg`
การ์ดสไตล์พื้นฐาน ใช้ `backdrop-filter` เบลอ `border` ขอบนูนขนาดเล็ก และ `box-shadow` เงาด้านใน ใช้ตัวกรอง `svg/glass.svg#h` เมื่อโฮเวอร์

### `main.Lg`
คอนเทนเนอร์บล็อกหลักของเพจ มุมโค้งมน `24px` พร้อมช่องว่างภายในแบบตอบสนอง (`16px` หากความกว้างน้อยกว่า `800px`, `24px` มิฉะนั้น)

### `.Btn`, `.BtnC`, `.Input`
ชื่อคลาสส่วนประกอบพื้นฐาน หลังจากแนะนำสไตล์นี้ คุณจะได้รับโครงร่างที่ยืดหยุ่น ความสัมพันธ์แบบลำดับชั้น และคุณสมบัติพื้นฐานของกระจกฝ้าโดยอัตโนมัติ
