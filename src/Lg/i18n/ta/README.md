# உறைந்த கண்ணாடி பின்னணி மற்றும் விளிம்பு ஹைலைட் விளைவு

அரை-வெளிப்படையான பார்டர் மற்றும் உள் நிழல் நுண்ணிய நிவாரண விளிம்பு சிறப்பம்சங்களை உருவாக்குகிறது.
பொத்தான் மற்றும் உள்ளீட்டு பெட்டி கூறுகளின் அடிப்படை பாணி சார்பு.

## டெமோவைப் பயன்படுத்தவும்

```html
<main class="Lg">
  <article>
    <h3>தலைப்பு</h3>
    <pre>உள்ளடக்க உரை</pre>
  </article>
</main>
```

## பாணி வகுப்பு

### `.Lg`
அட்டை அடிப்படை பாணி. `backdrop-filter` தெளிவின்மை, `border` மைக்ரோ-எம்போஸ்டு பார்டர் மற்றும் `box-shadow` உள் நிழலைப் பயன்படுத்தவும். மிதவையில் `svg/glass.svg#h` வடிப்பானைப் பயன்படுத்தவும்.

### `main.Lg`
பக்கத்தின் முக்கிய தொகுதி கொள்கலன். வட்டமான மூலைகள் `24px` பதிலளிக்கக்கூடிய திணிப்புடன் (`16px` அகலம் `800px`ஐ விட குறைவாக இருந்தால், `24px` இல்லையெனில்).

### `.Btn`, `.BtnC`, `.Input`
அடிப்படை கூறு வகுப்பு பெயர். இந்த பாணியை அறிமுகப்படுத்திய பிறகு, நீங்கள் தானாகவே நெகிழ்வான தளவமைப்பு, படிநிலை உறவுகள் மற்றும் உறைந்த கண்ணாடியின் அடிப்படை பண்புகள் ஆகியவற்றைப் பெறுவீர்கள்.
