# Frosted glass နောက်ခံနှင့် အစွန်းများကို မီးမောင်းထိုးပြသည့်အကျိုးသက်ရောက်မှု

တစ်ပိုင်းမြင်သာသောဘောင်နှင့် အတွင်းပိုင်းအရိပ်သည် သေးငယ်သောရုပ်ကြွအစွန်းများကို မီးမောင်းထိုးပြသည်။
ခလုတ်နှင့် input box အစိတ်အပိုင်းများ၏ အခြေခံစတိုင်ကို မှီခိုမှုအဖြစ်။

## ဒီမိုကိုသုံးပါ။

```html
<main class="Lg">
  <article>
    <h3>ခေါင်းစဉ်</h3>
    <pre>အကြောင်းအရာ စာသား</pre>
  </article>
</main>
```

## စတိုင်အတန်း

### `.Lg`
ကတ်အခြေခံစတိုင်။ `backdrop-filter`မှုန်ဝါးခြင်း၊ `border`မိုက်ခရိုဖောင်းကြွဘောင်နှင့် `box-shadow`အတွင်းပိုင်းအရိပ်ကို အသုံးချပါ။ `svg/glass.svg#h` စစ်ထုတ်မှုကို နှိပ်၍ နှိပ်ပါ။

### `main.Lg`
စာမျက်နှာ၏ ပင်မပိတ်ဆို့ ကွန်တိန်နာ။ အဝိုင်းထောင့် `24px`နှင့် တုံ့ပြန်မှုအကွက်များ (`16px`အကျယ်သည် `800px`၊ `24px`မဟုတ်ပါက)။

### `.Btn`, `.BtnC`, `.Input`
အခြေခံ အစိတ်အပိုင်း အတန်းအမည်။ ဤစတိုင်ကို မိတ်ဆက်ပြီးနောက်၊ ဖန်သားပြင်၏ ကွေးညွှတ်နိုင်သော အပြင်အဆင်၊ အထက်အောက် ဆက်ဆံရေးနှင့် ဖန်သားပြင်၏ အခြေခံ ဂုဏ်သတ္တိများကို အလိုအလျောက် ရရှိမည်ဖြစ်သည်။
