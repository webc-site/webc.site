# ශීත කළ වීදුරු පසුබිම සහ දාර උද්දීපන බලපෑම

අර්ධ විනිවිද පෙනෙන මායිම සහ අභ්‍යන්තර සෙවනැල්ල ක්ෂුද්‍ර සහන දාර උද්දීපනය කරයි.
බොත්තම් සහ ආදාන කොටු සංරචකවල මූලික ශෛල පරායත්තතාව ලෙස.

## Demo භාවිතා කරන්න

```html
<main class="Lg">
  <article>
    <h3>මාතෘකාව</h3>
    <pre>අන්තර්ගත පෙළ</pre>
  </article>
</main>
```

## ශෛලිය පන්තිය

### `.Lg`
කාඩ්පත් මූලික ශෛලිය. `backdrop-filter` නොපැහැදිලි, `border` ක්ෂුද්‍ර-එම්බෝස්ඩ් බෝඩරය, සහ `box-shadow` අභ්‍යන්තර සෙවන යොදන්න. හොවර් මත `svg/glass.svg#h` පෙරහන යොදන්න.

### `main.Lg`
පිටුවේ ප්‍රධාන වාරණ බහාලුම. ප්‍රතිචාරාත්මක පිරවුමක් සහිත වටකුරු කොන් `24px` (`16px` පළල `800px`ට වඩා අඩු නම්, `24px` වෙනත් ආකාරයකින්).

### `.Btn`, `.BtnC`, `.Input`
මූලික සංරචක පන්තියේ නම. මෙම විලාසය හඳුන්වා දීමෙන් පසු, ඔබට ස්වයංක්‍රීයව නම්‍යශීලී පිරිසැලසුම, ධූරාවලි සම්බන්ධතා සහ ශීත කළ වීදුරු වල මූලික ගුණාංග ලැබෙනු ඇත.
