# Мұздатылған шыны фон және жиекті бөлектеу әсері

Жартылай мөлдір жиек және ішкі көлеңке микрорельефті жиектерді бөлектеуді жасайды.
Түйме және кіріс жәшігінің компоненттерінің негізгі стиль тәуелділігі ретінде.

## Демонстрацияны пайдаланыңыз

```html
<main class="Lg">
  <article>
    <h3>тақырып</h3>
    <pre>Мазмұн мәтіні</pre>
  </article>
</main>
```

## стиль сыныбы

### `.Lg`
Карточканың негізгі стилі. `backdrop-filter` бұлыңғырлау, `border` микро-бедер жиегі және `box-shadow` ішкі көлеңке қолданыңыз. Меңзерді апарғанда `svg/glass.svg#h` сүзгісін қолданыңыз.

### `main.Lg`
Беттің негізгі блок контейнері. Жауапты толтырулары бар дөңгелектелген бұрыштар `24px` (`16px` ені `800px`-ден аз болса, `24px` әйтпесе).

### `.Btn`, `.BtnC`, `.Input`
Негізгі компонент класының аты. Бұл стильді енгізгеннен кейін сіз автоматты түрде икемді орналасуды, иерархиялық қатынастарды және мұздатылған әйнектің негізгі қасиеттерін аласыз.
