# ফ্ৰষ্টেড গ্লাছ বেকগ্ৰাউণ্ড আৰু এজ হাইলাইট ইফেক্ট

অৰ্ধ-স্বচ্ছ সীমা আৰু ভিতৰৰ ছাঁই মাইক্ৰ'-ৰিলিফ এজ হাইলাইট সৃষ্টি কৰে।
বুটাম আৰু ইনপুট বাকচ উপাদানসমূহৰ মূল শৈলী নিৰ্ভৰশীলতা হিচাপে।

## ডেমো ব্যৱহাৰ কৰক

```html
<main class="Lg">
  <article>
    <h3>অলপ</h3>
    <pre>বিষয়বস্তুৰ লিখনী</pre>
  </article>
</main>
```

## শৈলী শ্ৰেণী

### `.Lg`
কাৰ্ড বেচিক ষ্টাইল। `backdrop-filter` ব্লাৰ, `border` মাইক্ৰ'-এম্ব'ছড বৰ্ডাৰ, আৰু `box-shadow` ভিতৰৰ ছাঁ প্ৰয়োগ কৰক। হোভাৰত `svg/glass.svg#h` ফিল্টাৰ প্ৰয়োগ কৰক।

### `main.Lg`
পৃষ্ঠাৰ মূল ব্লক পাত্ৰ। প্ৰতিক্ৰিয়াশীল পেডিঙৰ সৈতে ঘূৰণীয়া চুক `24px` (`16px` যদি প্ৰস্থ `800px`তকৈ কম হয়, অন্যথা `24px`)।

### `.Btn`, `.BtnC`, `.Input`
মূল উপাদান শ্ৰেণীৰ নাম। এই শৈলী প্ৰৱৰ্তন কৰাৰ পিছত, আপুনি স্বয়ংক্ৰিয়ভাৱে নমনীয় বিন্যাস, স্তৰভিত্তিক সম্পৰ্ক আৰু ফ্ৰষ্টেড কাঁচৰ মৌলিক বৈশিষ্ট্যসমূহ লাভ কৰিব।
