# فراسٹڈ شیشے کا پس منظر اور کنارے نمایاں اثر

نیم شفاف بارڈر اور اندرونی سائے مائیکرو ریلیف ایج ہائی لائٹس بناتے ہیں۔
بٹن اور ان پٹ باکس کے اجزاء کی بنیادی طرز انحصار کے طور پر۔

## ڈیمو استعمال کریں۔

```html
<main class="Lg">
  <article>
    <h3>عنوان</h3>
    <pre>مواد کا متن</pre>
  </article>
</main>
```

## سٹائل کلاس

### `.Lg`
کارڈ بنیادی سٹائل. `backdrop-filter` دھندلاپن، `border` مائیکرو ایمبسڈ بارڈر، اور `box-shadow` اندرونی سائے کا اطلاق کریں۔ ہوور پر `svg/glass.svg#h` فلٹر لگائیں۔

### `main.Lg`
صفحہ کا مرکزی بلاک کنٹینر۔ جوابی پیڈنگ کے ساتھ گول کونے `24px` (`16px` اگر چوڑائی `800px` سے کم ہے، `24px` دوسری صورت میں)۔

### `.Btn`, `.BtnC`, `.Input`
بنیادی اجزاء کی کلاس کا نام۔ اس انداز کو متعارف کروانے کے بعد، آپ خود بخود لچکدار ترتیب، درجہ بندی کے تعلقات اور فراسٹڈ شیشے کی بنیادی خصوصیات حاصل کر لیں گے۔
