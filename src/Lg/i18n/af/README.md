# Matglas agtergrond en rand hoogtepunt effek

Semi-deursigtige rand en binneskadu skep mikro-reliëf randhoogtepunte.
As die basiese stylafhanklikheid van knoppie- en invoerbokskomponente.

## Gebruik die demo

```html
<main class="Lg">
  <article>
    <h3>titel</h3>
    <pre>Inhoud teks</pre>
  </article>
</main>
```

## styl klas

### `.Lg`
Kaart basiese styl. Pas `backdrop-filter` vervaging, `border` mikro-reliëfrand en `box-shadow` binneskadu toe. Pas `svg/glass.svg#h` filter toe wanneer jy beweeg.

### `main.Lg`
Die bladsy se hoofblokhouer. Afgeronde hoeke `24px` met responsiewe vulling (`16px` as die breedte minder as `800px` is, `24px` andersins).

### `.Btn`, `.BtnC`, `.Input`
Basiese komponent klas naam. Nadat u hierdie styl bekendgestel het, sal u outomaties buigsame uitleg, hiërargiese verhoudings en basiese eienskappe van matglas verkry.
