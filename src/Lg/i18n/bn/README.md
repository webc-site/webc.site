# ফ্রস্টেড গ্লাস পটভূমি এবং প্রান্ত হাইলাইট প্রভাব

আধা-স্বচ্ছ সীমানা এবং ভিতরের ছায়া মাইক্রো-রিলিফ প্রান্ত হাইলাইট তৈরি করে।
বোতাম এবং ইনপুট বক্স উপাদান মৌলিক শৈলী নির্ভরতা হিসাবে.

## ডেমো ব্যবহার করুন

```html
<main class="Lg">
  <article>
    <h3>শিরোনাম</h3>
    <pre>বিষয়বস্তু পাঠ্য</pre>
  </article>
</main>
```

## শৈলী ক্লাস

### `.Lg`
কার্ড মৌলিক শৈলী. `backdrop-filter` ব্লার, `border` মাইক্রো-এমবসড বর্ডার এবং `box-shadow` ভিতরের ছায়া প্রয়োগ করুন। হোভারে `svg/glass.svg#h` ফিল্টার প্রয়োগ করুন।

### `main.Lg`
পৃষ্ঠার প্রধান ব্লক কন্টেইনার। বৃত্তাকার কোণগুলি `24px` প্রতিক্রিয়াশীল প্যাডিং সহ (`16px` যদি প্রস্থ `800px` এর কম হয়, `24px` অন্যথায়)।

### `.Btn`, `.BtnC`, `.Input`
মৌলিক উপাদান শ্রেণীর নাম। এই শৈলীটি প্রবর্তন করার পরে, আপনি স্বয়ংক্রিয়ভাবে নমনীয় বিন্যাস, শ্রেণিবদ্ধ সম্পর্ক এবং ফ্রস্টেড গ্লাসের মৌলিক বৈশিষ্ট্যগুলি পাবেন।
