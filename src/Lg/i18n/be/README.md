# Фон з матавага шкла і эфект падсвечвання краёў

Напаўпразрыстая рамка і ўнутраная цень ствараюць мікрарэльефныя блікі па краях.
У якасці асноўнай залежнасці ад стылю кампанентаў кнопкі і поля ўводу.

## Выкарыстоўвайце дэма

```html
<main class="Lg">
  <article>
    <h3>назва</h3>
    <pre>Змест тэксту</pre>
  </article>
</main>
```

## клас стылю

### `.Lg`
Асноўны стыль карты. Прымяніце `backdrop-filter` размыццё, `border` мікрарэльефную мяжу і `box-shadow` унутраную цень. Прымяніць фільтр `svg/glass.svg#h` пры навядзенні.

### `main.Lg`
Кантэйнер галоўнага блока старонкі. Закругленыя вуглы `24px` з адаптыўным запаўненнем (`16px`, калі шырыня менш за `800px`, `24px` інакш).

### `.Btn`, `.BtnC`, `.Input`
Назва класа базавага кампанента. Пасля ўвядзення гэтага стылю вы аўтаматычна атрымаеце гнуткую планіроўку, іерархічныя адносіны і асноўныя ўласцівасці матавага шкла.
