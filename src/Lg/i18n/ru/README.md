# Фон из матового стекла и эффект выделения краев

Полупрозрачная кайма и внутренняя тень создают микрорельефные блики по краям.
В качестве базовой зависимости стиля компонентов кнопки и поля ввода.

## Используйте демо-версию

```html
<main class="Lg">
  <article>
    <h3>заголовок</h3>
    <pre>Текст контента</pre>
  </article>
</main>
```

## класс стиля

### `.Lg`
Основной стиль карты. Примените `backdrop-filter` размытие, `border` микротисненую рамку и `box-shadow` внутреннюю тень. Примените фильтр `svg/glass.svg#h` при наведении курсора.

### `main.Lg`
Основной блок-контейнер страницы. Закругленные углы `24px` с адаптивным заполнением (`16px`, если ширина меньше `800px`, `24px` в противном случае).

### `.Btn`, `.BtnC`, `.Input`
Имя класса базового компонента. Введя этот стиль, вы автоматически получите гибкую компоновку, иерархические связи и основные свойства матового стекла.
