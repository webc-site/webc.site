# רקע זכוכית חלבית ואפקט הדגשת קצה

גבול שקוף למחצה וצללית פנימית יוצרים הדגשות קצה של מיקרו-תבליט.
כתלות בסגנון הבסיסי של רכיבי כפתור ותיבת קלט.

## השתמש בהדגמה

```html
<main class="Lg">
  <article>
    <h3>כּוֹתֶרֶת</h3>
    <pre>טקסט תוכן</pre>
  </article>
</main>
```

## שיעור סגנון

### `.Lg`
סגנון בסיסי של כרטיס. החל `backdrop-filter` טשטוש, `border` גבול מוטבע במיקרו ו`box-shadow` צל פנימי. החל מסנן `svg/glass.svg#h` על ריחוף.

### `main.Lg`
מיכל הבלוק הראשי של הדף. פינות מעוגלות `24px` עם ריפוד מגיב (`16px` אם הרוחב קטן מ`800px`, `24px` אחרת).

### `.Btn`, `.BtnC`, `.Input`
שם מחלקה של רכיב בסיסי. לאחר הצגת סגנון זה, תקבל באופן אוטומטי פריסה גמישה, יחסים היררכיים ומאפיינים בסיסיים של זכוכית חלבית.
