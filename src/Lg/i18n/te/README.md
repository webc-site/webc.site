# తుషార గాజు నేపథ్యం మరియు అంచు హైలైట్ ప్రభావం

సెమీ పారదర్శక అంచు మరియు లోపలి నీడ మైక్రో-రిలీఫ్ ఎడ్జ్ హైలైట్‌లను సృష్టిస్తాయి.
బటన్ మరియు ఇన్‌పుట్ బాక్స్ భాగాల ప్రాథమిక శైలి డిపెండెన్సీగా.

## డెమో ఉపయోగించండి

```html
<main class="Lg">
  <article>
    <h3>శీర్షిక</h3>
    <pre>కంటెంట్ టెక్స్ట్</pre>
  </article>
</main>
```

## శైలి తరగతి

### `.Lg`
కార్డ్ ప్రాథమిక శైలి. `backdrop-filter` బ్లర్, `border` మైక్రో-ఎంబాస్డ్ అంచు మరియు `box-shadow` లోపలి నీడను వర్తింపజేయండి. హోవర్‌లో `svg/glass.svg#h` ఫిల్టర్‌ని వర్తింపజేయండి.

### `main.Lg`
పేజీ యొక్క ప్రధాన బ్లాక్ కంటైనర్. ప్రతిస్పందించే పాడింగ్‌తో గుండ్రంగా ఉన్న మూలలు `24px` (`16px` వెడల్పు `800px` కంటే తక్కువగా ఉంటే, `24px` లేకపోతే).

### `.Btn`, `.BtnC`, `.Input`
ప్రాథమిక భాగం తరగతి పేరు. ఈ శైలిని పరిచయం చేసిన తర్వాత, మీరు స్వయంచాలకంగా సౌకర్యవంతమైన లేఅవుట్, క్రమానుగత సంబంధాలు మరియు తుషార గాజు యొక్క ప్రాథమిక లక్షణాలను పొందుతారు.
