# ಫ್ರಾಸ್ಟೆಡ್ ಗ್ಲಾಸ್ ಹಿನ್ನೆಲೆ ಮತ್ತು ಅಂಚಿನ ಹೈಲೈಟ್ ಪರಿಣಾಮ

ಅರೆ-ಪಾರದರ್ಶಕ ಗಡಿ ಮತ್ತು ಒಳ ನೆರಳು ಮೈಕ್ರೋ-ರಿಲೀಫ್ ಅಂಚಿನ ಮುಖ್ಯಾಂಶಗಳನ್ನು ರಚಿಸುತ್ತದೆ.
ಬಟನ್ ಮತ್ತು ಇನ್‌ಪುಟ್ ಬಾಕ್ಸ್ ಘಟಕಗಳ ಮೂಲ ಶೈಲಿಯ ಅವಲಂಬನೆಯಂತೆ.

## ಡೆಮೊ ಬಳಸಿ

```html
<main class="Lg">
  <article>
    <h3>ಶೀರ್ಷಿಕೆ</h3>
    <pre>ವಿಷಯ ಪಠ್ಯ</pre>
  </article>
</main>
```

## ಶೈಲಿ ವರ್ಗ

### `.Lg`
ಕಾರ್ಡ್ ಮೂಲ ಶೈಲಿ. `backdrop-filter` ಬ್ಲರ್, `border` ಮೈಕ್ರೋ-ಎಂಬಾಸ್ಡ್ ಬಾರ್ಡರ್ ಮತ್ತು `box-shadow` ಒಳ ನೆರಳು ಅನ್ವಯಿಸಿ. ಹೋವರ್‌ನಲ್ಲಿ `svg/glass.svg#h` ಫಿಲ್ಟರ್ ಅನ್ನು ಅನ್ವಯಿಸಿ.

### `main.Lg`
ಪುಟದ ಮುಖ್ಯ ಬ್ಲಾಕ್ ಕಂಟೇನರ್. ರೆಸ್ಪಾನ್ಸಿವ್ ಪ್ಯಾಡಿಂಗ್‌ನೊಂದಿಗೆ ದುಂಡಾದ ಮೂಲೆಗಳು `24px` (`16px` ಅಗಲವು `800px` ಗಿಂತ ಕಡಿಮೆಯಿದ್ದರೆ, `24px` ಇಲ್ಲದಿದ್ದರೆ).

### `.Btn`, `.BtnC`, `.Input`
ಮೂಲ ಘಟಕ ವರ್ಗದ ಹೆಸರು. ಈ ಶೈಲಿಯನ್ನು ಪರಿಚಯಿಸಿದ ನಂತರ, ನೀವು ಸ್ವಯಂಚಾಲಿತವಾಗಿ ಹೊಂದಿಕೊಳ್ಳುವ ಲೇಔಟ್, ಕ್ರಮಾನುಗತ ಸಂಬಂಧಗಳು ಮತ್ತು ಫ್ರಾಸ್ಟೆಡ್ ಗ್ಲಾಸ್ನ ಮೂಲ ಗುಣಲಕ್ಷಣಗಳನ್ನು ಪಡೆಯುತ್ತೀರಿ.
