# پس زمینه شیشه مات و جلوه برجسته لبه

حاشیه نیمه شفاف و سایه داخلی، برجسته‌سازی لبه‌های برجسته‌ای را ایجاد می‌کند.
به عنوان وابستگی سبک اساسی اجزای دکمه و جعبه ورودی.

## از دمو استفاده کنید

```html
<main class="Lg">
  <article>
    <h3>عنوان</h3>
    <pre>متن محتوا</pre>
  </article>
</main>
```

## کلاس سبک

### `.Lg`
سبک پایه کارت `backdrop-filter` تاری، `border` حاشیه ریز برجسته، و `box-shadow` سایه داخلی را اعمال کنید. فیلتر `svg/glass.svg#h` را روی شناور اعمال کنید.

### `main.Lg`
محفظه بلوک اصلی صفحه. گوشه‌های گرد `24px` با لایه‌بندی پاسخگو (`16px` اگر عرض کمتر از `800px` باشد، در غیر این صورت `24px`).

### `.Btn`, `.BtnC`, `.Input`
نام کلاس جزء اصلی پس از معرفی این سبک، به طور خودکار چیدمان انعطاف پذیر، روابط سلسله مراتبی و ویژگی های اساسی شیشه مات را به دست خواهید آورد.
