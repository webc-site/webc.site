# 젖빛 유리 배경 및 가장자리 강조 효과

반투명 테두리와 내부 그림자는 미세한 양각 가장자리 하이라이트를 만듭니다.
버튼 및 입력 상자 구성 요소의 기본 스타일 종속성입니다.

## 데모 사용

```html
<main class="Lg">
  <article>
    <h3>제목</h3>
    <pre>콘텐츠 텍스트</pre>
  </article>
</main>
```

## 스타일 수업

### `.Lg`
카드 기본 스타일. `backdrop-filter` 흐림, `border` 미세 양각 테두리 및 `box-shadow` 내부 그림자를 적용합니다. 마우스를 올리면 `svg/glass.svg#h` 필터를 적용합니다.

### `main.Lg`
페이지의 기본 블록 컨테이너입니다. 반응형 패딩이 있는 둥근 모서리 `24px`(너비가 `800px`보다 작은 경우 `16px`, 그렇지 않은 경우 `24px`).

### `.Btn`, `.BtnC`, `.Input`
기본 구성 요소 클래스 이름입니다. 이 스타일을 도입한 후에는 자동으로 젖빛 유리의 유연한 레이아웃, 계층적 관계 및 기본 속성을 얻게 됩니다.
