# خلفية زجاجية بلوري وتأثير تسليط الضوء على الحافة

تعمل الحدود شبه الشفافة والظل الداخلي على إنشاء إبرازات حافة بارزة.
باعتبارها تبعية النمط الأساسي لمكونات الزر ومربع الإدخال.

## استخدم العرض التوضيحي

```html
<main class="Lg">
  <article>
    <h3>عنوان</h3>
    <pre>نص المحتوى</pre>
  </article>
</main>
```

## فئة النمط

### `.Lg`
النمط الأساسي للبطاقة. طبّق `backdrop-filter` تمويهًا و`border` حدودًا مزخرفة بشكل دقيق و`box-shadow` ظلًا داخليًا. قم بتطبيق مرشح `svg/glass.svg#h` عند التمرير.

### `main.Lg`
حاوية الكتلة الرئيسية للصفحة. زوايا مستديرة `24px` مع حشوة سريعة الاستجابة (`16px` إذا كان العرض أقل من `800px`، `24px` بخلاف ذلك).

### `.Btn`, `.BtnC`, `.Input`
اسم فئة المكون الأساسي. بعد تقديم هذا النمط، سوف تحصل تلقائيًا على تخطيط مرن وعلاقات هرمية وخصائص أساسية للزجاج المصنفر.
