# ផ្ទៃខាងក្រោយកញ្ចក់សាយ និងឥទ្ធិពលបន្លិចគែម

ស៊ុម​ពាក់កណ្តាល​ថ្លា និង​ស្រមោល​ខាងក្នុង​បង្កើត​ការរំលេច​គែម​សង្គ្រោះ​ខ្នាតតូច។
ក្នុងនាមជារចនាប័ទ្មមូលដ្ឋានអាស្រ័យនៃប៊ូតុង និងសមាសភាគប្រអប់បញ្ចូល។

## ប្រើការបង្ហាញ

```html
<main class="Lg">
  <article>
    <h3>ចំណងជើង</h3>
    <pre>អត្ថបទមាតិកា</pre>
  </article>
</main>
```

## ថ្នាក់រចនាប័ទ្ម

### `.Lg`
រចនាប័ទ្មមូលដ្ឋាននៃកាត។ អនុវត្ត `backdrop-filter`ព្រិល, `border`ស៊ុមក្រឡោតខ្នាតតូច និង`box-shadow`ស្រមោលខាងក្នុង។ អនុវត្ត `svg/glass.svg#h`តម្រង​លើ​ការ​ដាក់។

### `main.Lg`
ធុងប្លុកសំខាន់របស់ទំព័រ។ ជ្រុងរាងមូល `24px`ជាមួយបន្ទះឆ្លើយតប (`16px`ប្រសិនបើទទឹងតិចជាង `800px` `24px`បើមិនដូច្នេះទេ)។

### `.Btn`, `.BtnC`, `.Input`
ឈ្មោះថ្នាក់សមាសភាគមូលដ្ឋាន។ បន្ទាប់ពីការណែនាំរចនាប័ទ្មនេះ អ្នកនឹងទទួលបានប្លង់ដែលអាចបត់បែនបាន ទំនាក់ទំនងតាមឋានានុក្រម និងលក្ខណៈសម្បត្តិជាមូលដ្ឋាននៃកញ្ចក់កកដោយស្វ័យប្រវត្តិ។
