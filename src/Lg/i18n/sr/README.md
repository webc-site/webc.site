# Ефекат истицања позадине и ивица од мат стакла

Полупровидна ивица и унутрашња сенка стварају микрорељефне ивице.
Као основна стилска зависност компоненти дугмета и поља за унос.

## Користите демо

```html
<main class="Lg">
  <article>
    <h3>наслов</h3>
    <pre>Текст садржаја</pre>
  </article>
</main>
```

## стилска класа

### `.Lg`
Основни стил картице. Примените <б ид="1"> замућење, <б ид="2"> микро-рељефну ивицу и <б ид="3"> унутрашњу сенку. Примените филтер <б ид="4"> на курсор.

### `main.Lg`
Контејнер главног блока странице. Заобљени углови <б ид="1"> са респонзивним допунама (<б ид="3"> ако је ширина мања од <б ид="2">, <б ид="4"> у супротном).

### `.Btn`, `.BtnC`, `.Input`
Име класе основне компоненте. Након увођења овог стила, аутоматски ћете добити флексибилан распоред, хијерархијске односе и основна својства мат стакла.
