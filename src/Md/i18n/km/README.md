# ញែកដោយសុវត្ថិភាព និងបង្ហាញអត្ថបទ Markdown ទៅជា HTML ។

- **Security Sanitization**៖ ការពារប្រឆាំងនឹងការវាយប្រហារ XSS ដោយប្រើ `setHTML` API ឬ `DOMParser`អ្នកសម្អាត
- **ការស្រង់ចេញមាតិកា**៖ អានមាតិកាអត្ថបទដោយស្វ័យប្រវត្តិនៃរន្ធដោតលំនាំដើម អំឡុងពេលចាប់ផ្តើម ហើយបង្ហាញវា
- **ការកំណត់ប្រភេទដែលភ្ជាប់មកជាមួយ**៖ បញ្ជីកំណត់ជាមុន តារាង ប្លុកកូដ សម្រង់ និងរចនាប័ទ្មប្រអប់ព្រមាន

## ប្រើការបង្ហាញ

### ការបង្ហាញឋិតិវន្ត

```html
<c-md>
# ចំណងជើង
នេះគឺជាខ្លឹមសារនៃ Markdown
</c-md>
```

### កិច្ចការថាមវន្ត

```html
<c-md id="md-el"></c-md>

<script type="module">
  const el = document.querySelector("#md-el");
  el.textContent = "#ចំណងជើងថ្មី\nមាតិកាថ្មី";
</script>
```

## ប៉ារ៉ាម៉ែត្រចំណុចប្រទាក់

បញ្ចូលក្នុងអត្ថបទ Markdown ដោយផ្ទាល់នៅក្នុងស្លាក (ជារន្ធដោតលំនាំដើម) ឬធ្វើបច្ចុប្បន្នភាពថាមវន្តតាមរយៈ `textContent`។

## ការពិពណ៌នារចនាប័ទ្ម

ថ្នាក់រចនាប័ទ្ម `.Md`ត្រូវបានបន្ថែមដោយស្វ័យប្រវត្តិ នៅពេលដែលសមាសធាតុត្រូវបានចាប់ផ្តើម។ គាំទ្រប្លង់ប្រអប់ព្រមានបន្ថែម Markdown ខាងក្រោម (បង្កឡើងដោយវាក្យសម្ព័ន្ធយោង)៖

- `[!NOTE]`៖ ព័ត៌មាន​ភ្លាមៗ
- `[!TIP]`៖ ការណែនាំ/គន្លឹះ
- `[!IMPORTANT]`៖ ការយកចិត្តទុកដាក់សំខាន់
- `[!WARNING]`៖ ការព្រមានអំពីហានិភ័យ
- `[!CAUTION]`៖ ការព្រមានអំពីហានិភ័យខ្ពស់ខ្លាំង

## យន្តការសន្តិសុខ

នៅពេលបង្ហាញ សូមហៅទៅកាន់ API ដើម `setHTML`ដំបូង ដើម្បីបញ្ចូល HTML ដោយសុវត្ថិភាព។ ប្រសិនបើកម្មវិធីរុករកតាមអ៊ីនធឺណិតមិនគាំទ្រវាទេ វានឹងទម្លាក់ចំណាត់ថ្នាក់ដោយស្វ័យប្រវត្តិទៅកម្មវិធីសម្អាតឡើងវិញដោយផ្អែកលើ `DOMParser` ហើយអនុវត្តគោលការណ៍សុវត្ថិភាពខាងក្រោម៖
- លុបស្លាកដែលមិនមានសុវត្ថិភាព (ឧ. `script`, `iframe` ។ល។)
- សម្អាត `on*`លក្ខណៈព្រឹត្តិការណ៍
- ការពារប្រឆាំងនឹងការវាយប្រហារ XSS ដោយត្រងគុណលក្ខណៈដែលចាប់ផ្តើមដោយពិធីការគ្រោះថ្នាក់ដូចជា `javascript:`, `vbscript:` និង `data:` (លើកលែងតែ `data:image/`)។
