# مارڪ ڊائون ٽيڪسٽ کي HTML ۾ محفوظ طور تي پارس ۽ رينڈر ڪريو.

- ** حفاظتي صفائي **: XSS حملن جي خلاف دفاع ڪريو مقامي `setHTML` API يا `DOMParser` صاف ڪندڙ
- ** مواد ڪڍڻ **: شروعاتي ڪرڻ دوران ڊفالٽ سلاٽ جي متن جي مواد کي خودڪار طور تي پڙهو ۽ ان کي پيش ڪريو
- **بدليل ٽائپ سيٽنگ**: اڳواٽ لسٽ، ٽيبل، ڪوڊ بلاڪ، اقتباس ۽ ڊيڄاريندڙ باڪس انداز

## ڊيمو استعمال ڪريو

### جامد رينجرنگ

```html
<c-md>
# عنوان
هي مارڪ ڊائون مواد جو هڪ ٽڪرو آهي
</c-md>
```

### متحرڪ تفويض

```html
<c-md id="md-el"></c-md>

<script type="module">
  const el = document.querySelector("#md-el");
  el.textContent = "#نئون عنوان\nنئون مواد"؛
</script>
```

## انٽرفيس پيٽرول

مارڪ ڊائون ٽيڪسٽ ۾ داخل ڪريو سڌو ٽيگ جي اندر (جيئن ڊفالٽ سلاٽ)، يا متحرڪ طور تي `textContent` ذريعي تازه ڪاري ڪريو.

## انداز بيان

`.Md` اسلوب ڪلاس خودڪار طريقي سان شامل ڪيو ويندو آهي جڏهن جزو شروع ڪيو ويندو آهي. ھيٺ ڏنل مارڪ ڊائون کي وڌايو وارننگ باڪس لي آئوٽ کي سپورٽ ڪري ٿو (حوالو نحو ذريعي شروع ٿيل):

- `[!NOTE]`: فوري معلومات
- `[!TIP]`: صلاحون/تجويز
- `[!IMPORTANT]`: اهم ڌيان
- `[!WARNING]`: خطري جي ڊيڄاريندڙ
- `[!CAUTION]`: انتهائي خطرناڪ ڊيڄاريندڙ

## سيڪيورٽي ميڪانيزم

رينڊرنگ ڪرڻ وقت، HTML کي محفوظ طور تي انجڻ ڪرڻ لاءِ پھرين مقامي `setHTML` API کي ڪال ڪريو. جيڪڏهن برائوزر ان کي سپورٽ نٿو ڪري، اهو خودڪار طور تي `DOMParser` جي بنياد تي ٻيهر ريسرسيو ڪلينر ڏانهن گهٽجي ويندو ۽ هيٺ ڏنل سيڪيورٽي پاليسي لاڳو ڪندو:
- غير محفوظ ٽيگ هٽايو (مثال طور `script`، `iframe`، وغيره.)
- صاف ڪريو `on*` واقعي جون خاصيتون
- خطرناڪ پروٽوڪولن سان شروع ٿيندڙ خاصيتن کي فلٽر ڪندي XSS حملن جي خلاف دفاع ڪريو جيئن ته `javascript:`، `vbscript:`، ۽ `data:` (سواءِ `data:image/`).
