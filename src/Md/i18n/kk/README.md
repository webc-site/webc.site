# Markdown мәтінін HTML тіліне қауіпсіз түрде талдаңыз және көрсетіңіз.

- **Қауіпсіздікті тазарту**: жергілікті `setHTML` API немесе `DOMParser` тазалағыштарды пайдаланып XSS шабуылдарынан қорғаңыз
- **Мазмұнды шығару**: инициализация кезінде әдепкі слоттың мәтіндік мазмұнын автоматты түрде оқып, оны көрсету
- **Кірістірілген теру**: алдын ала орнатылған тізім, кесте, код блогы, дәйексөз және ескерту өрісі мәнерлері

## Демонстрацияны пайдаланыңыз

### статикалық көрсету

```html
<c-md>
# тақырып
Бұл Markdown мазмұнының бір бөлігі
</c-md>
```

### динамикалық тапсырма

```html
<c-md id="md-el"></c-md>

<script type="module">
  const el = document.querySelector("#md-el");
  el.textContent = "#Жаңа тақырып\nЖаңа мазмұн";
</script>
```

## Интерфейс параметрлері

Markdown мәтінін тікелей тег ішінде (әдепкі ұяшық ретінде) жіберіңіз немесе `textContent` арқылы динамикалық түрде жаңартыңыз.

## Стиль сипаттамасы

Компонент инициализацияланған кезде `.Md` стиль класы автоматты түрде қосылады. Келесі Markdown кеңейтілген ескерту өрісінің орналасуын қолдайды (анықтамалық синтаксис арқылы іске қосылған):

- `[!NOTE]`:Шұғыл ақпарат
- `[!TIP]`: ұсыныстар/кеңестер
- `[!IMPORTANT]`: Маңызды назар
- `[!WARNING]`: қауіп туралы ескерту
- `[!CAUTION]`: өте жоғары қауіп туралы ескерту

## қауіпсіздік механизмі

Көрсету кезінде HTML қауіпсіз енгізу үшін алдымен жергілікті `setHTML` API-ге қоңырау шалыңыз. Браузер оған қолдау көрсетпесе, ол `DOMParser` негізіндегі рекурсивті тазартқышқа автоматты түрде төмендейді және келесі қауіпсіздік саясатын жүзеге асырады:
- Қауіпсіз тегтерді жою (мысалы: `script`, `iframe`, т.б.)
- `on*` оқиға атрибуттарын өшіріңіз
- `javascript:`, `vbscript:` және `data:` (`data:image/` қоспағанда) сияқты қауіпті протоколдардан басталатын атрибуттарды сүзу арқылы XSS шабуылдарынан қорғаңыз.
