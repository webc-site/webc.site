# Безбедно анализирајте и прикажите Маркдовн текст у ХТМЛ.

- **Сигурносна санација**: Одбраните се од КССС напада користећи изворне <б ид="1"> АПИ или <б ид="2"> чистаче
- **Издвајање садржаја**: Аутоматски прочитајте текстуални садржај подразумеваног слота током иницијализације и рендерујте га
- **Уграђено постављање текста**: унапред подешена листа, табела, блок кодова, стилови оквира за цитате и упозорења

## Користите демо

### статичко приказивање

```html
<c-md>
# наслов
Ово је део Маркдовн садржаја
</c-md>
```

### динамички задатак

```html
<c-md id="md-el"></c-md>

<script type="module">
  const el = document.querySelector("#md-el");
  ел.валуе = "#Нови наслов\нНови садржај";
</script>
```

## Параметри интерфејса

Проследите Маркдовн текст директно унутар ознаке (као подразумевани слот) или ажурирајте динамички преко <б ид="1">.

Класа стила <б ид="1"> се аутоматски додаје када се компонента иницијализује. Подржава следећи изглед Маркдовн проширеног оквира упозорења (покреће референтна синтакса):

- <б ид="1">: Брза информација
- <б ид="1">: Предлози/Савети
- <б ид="1">: Важна пажња
- <б ид="1">: Упозорење о ризику
- <б ид="1">: Упозорење о изузетно високом ризику

## сигурносни механизам

Приликом приказивања, прво позовите изворни <б ид="1"> АПИ да бисте безбедно убацили ХТМЛ. Ако га прегледач не подржава, аутоматски ће се вратити на рекурзивни чистач заснован на <б ид="2"> и примениће следећу безбедносну политику:
- Уклоните ознаке које нису безбедне (нпр. <б ид="1">, <б ид="2">, итд.)
- Обришите атрибуте догађаја <б ид="1">
- Одбраните се од КССС напада филтрирањем атрибута почевши од опасних протокола као што су <б ид="1">, <б ид="2"> и <б ид="3"> (осим <б ид="4">).
