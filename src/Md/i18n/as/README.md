# মাৰ্কডাউন লিখনীক নিৰাপদে বিশ্লেষণ আৰু HTML লৈ ৰেণ্ডাৰ কৰক।

- **সুৰক্ষা অনাময়**: স্থানীয় `setHTML` API বা `DOMParser` ক্লিনাৰ ব্যৱহাৰ কৰি XSS আক্ৰমণৰ বিৰুদ্ধে প্ৰতিৰক্ষা কৰক
- **সামগ্ৰী নিষ্কাশন**: আৰম্ভণিৰ সময়ত অবিকল্পিত স্লটৰ লিখনী বিষয়বস্তু স্বয়ংক্ৰিয়ভাৱে পঢ়ক আৰু ইয়াক ৰেণ্ডাৰ কৰক
- **বিল্ট-ইন টাইপছেটিং**: প্ৰিছেট তালিকা, টেবুল, ক'ড ব্লক, উদ্ধৃতি আৰু সতৰ্কবাণী বাকচ শৈলী

## ডেমো ব্যৱহাৰ কৰক

### ষ্টেটিক ৰেণ্ডাৰ

```html
<c-md>
# অলপ
এইটো এটা মাৰ্কডাউন কন্টেন্ট
</c-md>
```

### গতিশীল নিযুক্তি

```html
<c-md id="md-el"></c-md>

<script type="module">
  const el = document.querySelector("#md-el");
  el.value = "#নতুন শিৰোনাম\nনতুন বিষয়বস্তু";
</script>
```

## আন্তঃপৃষ্ঠ প্ৰাচলসমূহ

টেগৰ ভিতৰত পোনপটীয়াকৈ মাৰ্কডাউন লিখনী পাছ কৰক (অবিকল্পিত স্লট হিচাপে), বা `value` ৰ যোগেদি পৰিবৰ্তনশীলভাৱে আপডেইট কৰক ।

## শৈলীৰ বিৱৰণ

`.Md` শৈলী শ্ৰেণী স্বয়ংক্ৰিয়ভাৱে যোগ কৰা হয় যেতিয়া উপাদান আৰম্ভ কৰা হয় । নিম্নলিখিত মাৰ্কডাউন সম্প্ৰসাৰিত সতৰ্কবাণী বাকচ পৰিকল্পনা সমৰ্থন কৰে (উল্লেখ বাক্যবিন্যাসৰ দ্বাৰা ট্ৰিগাৰ কৰা):

- `[!NOTE]`:উল্লেখযোগ্য তথ্য
- `[!TIP]`: পৰামৰ্শ/টিপছ
- `[!IMPORTANT]`: গুৰুত্বপূৰ্ণ মনোযোগ
- `[!WARNING]`: বিপদৰ সতৰ্কবাণী
- `[!CAUTION]`: অতি উচ্চ বিপদৰ সতৰ্কবাণী

## নিৰাপত্তা ব্যৱস্থা

ৰেণ্ডাৰ কৰাৰ সময়ত, নিৰাপদে HTML ইনজেক্ট কৰিবলে প্ৰথমে স্থানীয় `setHTML` API কল কৰক। যদি ব্ৰাউজাৰে ইয়াক সমৰ্থন নকৰে, ই স্বয়ংক্ৰিয়ভাৱে `DOMParser` ৰ ওপৰত ভিত্তি কৰি এটা পুনৰাবৃত্তিমূলক ক্লিনাৰলৈ ডাউনগ্ৰেড কৰিব আৰু নিম্নলিখিত সুৰক্ষা নীতি প্ৰণয়ন কৰিব:
- অসুৰক্ষিত টেগসমূহ আঁতৰাওক (যেনে `script`, `iframe`, ইত্যাদি)
- `on*` ইভেন্ট বৈশিষ্ট্যসমূহ পৰিষ্কাৰ কৰক
- বিপজ্জনক প্ৰটোকল যেনে `javascript:`, `vbscript:`, আৰু `data:` (`data:image/` ৰ বাহিৰে) ৰ পৰা আৰম্ভ কৰি বৈশিষ্ট্যসমূহ ফিল্টাৰ কৰি XSS আক্ৰমণৰ বিৰুদ্ধে প্ৰতিৰক্ষা কৰক।
