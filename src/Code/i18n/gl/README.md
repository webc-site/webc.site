# botón de código

- Botón de icona Conter código
- Código de aviso para pasar o rato do rato, admite varios idiomas

## Use a demostración

```html
<c-code href="https://github.com"></c-code>

<script type="module">
import "webc.site/Code/_.js";
</script>
```

## Parámetros da interface

A etiqueta do compoñente é `<c-code>` e o enderezo de salto de código pódese configurar mediante o atributo `href`.
