# ປຸ່ມລະຫັດ

- ມີປຸ່ມໄອຄອນລະຫັດ
- ເມົ້າເລື່ອນໃສ່ລະຫັດເຕືອນ, ຮອງຮັບຫຼາຍພາສາ

## ໃຊ້ຕົວຢ່າງ

```html
<c-code href="https://github.com"></c-code>

<script type="module">
import "webc.site/Code/_.js";
</script>
```

## ຕົວກໍານົດການໃນການໂຕ້ຕອບ

ປ້າຍກຳກັບອົງປະກອບແມ່ນ `<c-code>`, ແລະທີ່ຢູ່ໂດດລະຫັດສາມາດຖືກຕັ້ງຄ່າໄດ້ຜ່ານຄຸນສົມບັດ `href`.
