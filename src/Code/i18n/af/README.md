# kode knoppie

- Bevat kode-ikoonknoppie
- Muis sweef prompt kode, ondersteun verskeie tale

## Gebruik die demo

```html
<c-code href="https://github.com"></c-code>

<script type="module">
import "webc.site/Code/_.js";
</script>
```

## Interface parameters

Die komponentetiket is `<c-code>`, en die kodesprongadres kan deur die `href`-kenmerk gekonfigureer word.
