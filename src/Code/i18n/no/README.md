# kodeknapp

- Inneholder kodeikonknapp
- Kode for musepeker, støtter flere språk

## Bruk demoen

```html
<c-code href="https://github.com"></c-code>

<script type="module">
import "webc.site/Code/_.js";
</script>
```

## Grensesnittparametere

Komponentetiketten er `<c-code>`, og kodehoppadressen kan konfigureres gjennom attributtet `href`.
