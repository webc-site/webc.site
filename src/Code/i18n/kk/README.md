# код түймесі

- Код белгішесін қамтитын түйме
- Тінтуірді апару шақыру коды, бірнеше тілді қолдайды

## Демонстрацияны пайдаланыңыз

```html
<c-code href="https://github.com"></c-code>
```

## Интерфейс параметрлері

Құрамдас белгі `<c-code>` және кодты көшіру мекенжайын `href` төлсипаты арқылы конфигурациялауға болады.
