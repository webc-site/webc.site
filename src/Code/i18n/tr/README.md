# kod düğmesi

- Kod simgesi düğmesini içerir
- Fareyle üzerine gelme komut istemi kodu, birden çok dili destekler

## Demoyu kullanın

```html
<c-code href="https://github.com"></c-code>
```

## Arayüz parametreleri

Bileşen etiketi `<c-code>`'dir ve kod atlama adresi `href` özelliği aracılığıyla yapılandırılabilir.
