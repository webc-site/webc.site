# кнопка кода

- Кнопка са значком "Утрымліваць код".
- Код падказкі пры навядзенні мышы, падтрымлівае некалькі моў

## Выкарыстоўвайце дэма

```html
<c-code href="https://github.com"></c-code>

<script type="module">
import "webc.site/Code/_.js";
</script>
```

## Параметры інтэрфейсу

Пазнака кампанента - `<c-code>`, а адрас пераходу кода можна наладзіць праз атрыбут `href`.
