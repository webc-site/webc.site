# buttuna tal-kodiċi

- Fih il-buttuna tal-ikona tal-kodiċi
- Kodiċi fil-pront tal-maws li jtajjar, jappoġġja lingwi multipli

## Uża d-demo

```html

```

## Parametri tal-interface

It-tikketta tal-komponent hija `<c-code>`, u l-indirizz tal-qabża tal-kodiċi jista 'jiġi kkonfigurat permezz tal-attribut `href`.
