# kód gomb

- Kódot tartalmazó ikon gomb
- Az egérmutató prompt kódja több nyelvet is támogat

## Használd a demót

```html
<c-code href="https://github.com"></c-code>

<script type="module">
import "webc.site/Code/_.js";
</script>
```

## Interfész paraméterei

Az összetevő címke `<c-code>`, a kódugrás címe pedig az `href` attribútummal konfigurálható.
