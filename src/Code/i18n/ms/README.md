# butang kod

- Mengandungi butang ikon kod
- Kod gesaan tuding tetikus, menyokong berbilang bahasa

## Gunakan demo

```html
<c-code href="https://github.com"></c-code>
```

## Parameter antara muka

Label komponen ialah `<c-code>` dan alamat lompat kod boleh dikonfigurasikan melalui atribut `href`.
