# kodni gumb

- Gumb ikone Vsebuj kodo
- Koda poziva za lebdenje miške, podpira več jezikov

## Uporabite demo

```html
<c-code href="https://github.com"></c-code>

<script type="module">
import "webc.site/Code/_.js";
</script>
```

## Parametri vmesnika

Oznaka komponente je `<c-code>`, naslov skoka kode pa je mogoče konfigurirati prek atributa `href`.
