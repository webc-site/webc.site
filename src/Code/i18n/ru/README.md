# кнопка кода

- Кнопка со значком кода
- Код приглашения при наведении мыши, поддерживает несколько языков

## Используйте демо-версию

```html
<c-code href="https://github.com"></c-code>
```

## Параметры интерфейса

Метка компонента — `<c-code>`, а адрес перехода кода можно настроить с помощью атрибута `href`.
