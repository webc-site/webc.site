# کوڈ بٹن

- کوڈ آئیکن بٹن پر مشتمل ہے۔
- ماؤس ہوور پرامپٹ کوڈ، متعدد زبانوں کو سپورٹ کرتا ہے۔

## ڈیمو استعمال کریں۔

```html
<c-code href="https://github.com"></c-code>

<script type="module">
import "webc.site/Code/_.js";
</script>
```

## انٹرفیس کے پیرامیٹرز

اجزاء کا لیبل `<c-code>` ہے، اور کوڈ جمپ ایڈریس کو `href` انتساب کے ذریعے ترتیب دیا جا سکتا ہے۔
