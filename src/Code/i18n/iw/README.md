# כפתור קוד

- מכיל לחצן סמל קוד
- קוד הנחיה לרחף עכבר, תומך במספר שפות

## השתמש בהדגמה

```html
<c-code href="https://github.com"></c-code>

<script type="module">
import "webc.site/Code/_.js";
</script>
```

## פרמטרים של ממשק

תווית הרכיב היא `<c-code>`, וניתן להגדיר את כתובת קפיצת הקוד באמצעות התכונה `href`.
