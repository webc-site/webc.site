# копче за код

- Копче за икона содржи код
- Промптен код за лебдење на глувчето, поддржува повеќе јазици

## Користете го демо

```html
<c-code href="https://github.com"></c-code>

<script type="module">
import "webc.site/Code/_.js";
</script>
```

## Параметри на интерфејсот

Етикетата на компонентата е `<c-code>`, а адресата за скокање код може да се конфигурира преку атрибутот `href`.
