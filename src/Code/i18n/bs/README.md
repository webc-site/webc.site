# dugme koda

- Sadrži dugme ikone koda
- Kod promptnog lebdenja mišem, podržava više jezika

## Koristite demo

```html
<c-code href="https://github.com"></c-code>

<script type="module">
import "webc.site/Code/_.js";
</script>
```

## Parametri interfejsa

Oznaka komponente je `<c-code>`, a adresa za skok koda može se konfigurirati preko `href` atributa.
