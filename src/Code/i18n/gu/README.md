# કોડ બટન

- કોડ આયકન બટન ધરાવે છે
- માઉસ હોવર પ્રોમ્પ્ટ કોડ, બહુવિધ ભાષાઓને સપોર્ટ કરે છે

## ડેમોનો ઉપયોગ કરો

```html
<c-code href="https://github.com"></c-code>

<script type="module">
import "webc.site/Code/_.js";
</script>
```

## ઇન્ટરફેસ પરિમાણો

ઘટક લેબલ `<c-code>` છે, અને કોડ જમ્પ સરનામું `href` વિશેષતા દ્વારા ગોઠવી શકાય છે.
