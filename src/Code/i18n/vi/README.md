# nút mã

- Chứa nút biểu tượng mã
- Mã nhắc chuột di chuột, hỗ trợ nhiều ngôn ngữ

## Sử dụng bản trình diễn

```html
<c-code href="https://github.com"></c-code>

<script type="module">
import "webc.site/Code/_.js";
</script>
```

## Thông số giao diện

Nhãn thành phần là `<c-code>` và địa chỉ nhảy mã có thể được định cấu hình thông qua thuộc tính `href`.
