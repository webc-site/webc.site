# Code-Taste

- Symbolschaltfläche „Code enthalten“.
- Eingabeaufforderungscode für den Mauszeiger, unterstützt mehrere Sprachen

## Nutzen Sie die Demo

```html
<c-code href="https://github.com"></c-code>

<script type="module">
import "webc.site/Code/_.js";
</script>
```

## Schnittstellenparameter

Die Komponentenbezeichnung lautet `<c-code>` und die Codesprungadresse kann über das Attribut `href` konfiguriert werden.
