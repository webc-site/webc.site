# gumb koda

- Gumb ikone Sadrži kod
- Kôd upita za lebdenje mišem, podržava više jezika

## Koristite demo

```html
<c-code href="https://github.com"></c-code>

<script type="module">
import "webc.site/Code/_.js";
</script>
```

## Parametri sučelja

Oznaka komponente je `<c-code>`, a adresa skoka koda može se konfigurirati putem atributa `href`.
