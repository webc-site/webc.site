# кнопка коду

- Кнопка значка коду
- Код підказки при наведенні миші, підтримує кілька мов

## Використовуйте демо

```html
<c-code href="https://github.com"></c-code>
```

## Параметри інтерфейсу

Мітка компонента — `<c-code>`, а адресу переходу коду можна налаштувати за допомогою атрибута `href`.
