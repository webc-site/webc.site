# codeknop

- Bevat codepictogramknop
- Muis hover prompt code, ondersteunt meerdere talen

## Gebruik de demo

```html
<c-code href="https://github.com"></c-code>

<script type="module">
import "webc.site/Code/_.js";
</script>
```

## Interfaceparameters

Het componentlabel is `<c-code>` en het codesprongadres kan worden geconfigureerd via het `href` attribuut.
