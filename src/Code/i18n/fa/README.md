# دکمه کد

- شامل دکمه نماد کد
- کد اعلان شناور ماوس، از چندین زبان پشتیبانی می کند

## از دمو استفاده کنید

```html
<c-code href="https://github.com"></c-code>

<script type="module">
import "webc.site/Code/_.js";
</script>
```

## پارامترهای رابط

برچسب مؤلفه `<c-code>` است و آدرس پرش کد را می توان از طریق ویژگی `href` پیکربندی کرد.
