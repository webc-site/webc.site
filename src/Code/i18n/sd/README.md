# ڪوڊ بٽڻ

- ڪوڊ آئڪن بٽڻ تي مشتمل
- ماؤس هور پرامپٽ ڪوڊ، ڪيترن ئي ٻولين کي سپورٽ ڪري ٿو

## ڊيمو استعمال ڪريو

```html
<c-code href="https://github.com"></c-code>

<script type="module">
import "webc.site/Code/_.js";
</script>
```

## انٽرفيس پيٽرول

جزو جو ليبل `<c-code>` آهي، ۽ ڪوڊ جمپ ايڊريس کي ترتيب ڏئي سگهجي ٿو `href` وصف ذريعي.
