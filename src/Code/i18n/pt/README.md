# botão de código

- Contém botão de ícone de código
- Código de prompt de passar o mouse, suporta vários idiomas

## Use a demonstração

```html
<c-code href="https://github.com"></c-code>

<script type="module">
import "webc.site/Code/_.js";
</script>
```

## Parâmetros de interface

O rótulo do componente é `<c-code>` e o endereço de salto do código pode ser configurado através do atributo `href`.
