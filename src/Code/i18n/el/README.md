# κουμπί κωδικού

- Κουμπί εικονιδίου με κωδικό
- Κώδικας προτροπής τοποθέτησης ποντικιού, υποστηρίζει πολλές γλώσσες

## Χρησιμοποιήστε το demo

```html
<c-code href="https://github.com"></c-code>

<script type="module">
import "webc.site/Code/_.js";
</script>
```

## Παράμετροι διεπαφής

Η ετικέτα στοιχείου είναι `<c-code>` και η διεύθυνση μετάβασης κώδικα μπορεί να διαμορφωθεί μέσω του χαρακτηριστικού `href`.
