# pindutan ng code

- Button ng icon na naglalaman ng code
- Mouse hover prompt code, sumusuporta sa maraming wika

## Gamitin ang demo

```html
<c-code href="https://github.com"></c-code>
```

## Mga parameter ng interface

Ang component label ay `<c-code>`, at ang code jump address ay maaaring i-configure sa pamamagitan ng `href` attribute.
