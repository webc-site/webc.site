# przycisk kodu

- Zawiera przycisk z ikoną kodu
- Kod zachęty po najechaniu myszką, obsługuje wiele języków

## Skorzystaj z wersji demonstracyjnej

```html
<c-code href="https://github.com"></c-code>
```

## Parametry interfejsu

Etykieta komponentu to `<c-code>`, a adres skoku kodu można skonfigurować za pomocą atrybutu `href`.
