# botón de código

- Contiene el botón del icono de código
- Código de aviso al pasar el mouse, admite múltiples idiomas

## Usa la demostración

```html
<c-code href="https://github.com"></c-code>

<script type="module">
import "webc.site/Code/_.js";
</script>
```

## Parámetros de interfaz

La etiqueta del componente es `<c-code>` y la dirección de salto del código se puede configurar a través del atributo `href`.
