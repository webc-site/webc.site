# botó de codi

- Botó d'icona Conté codi
- Codi d'indicació del ratolí, admet diversos idiomes

## Utilitzeu la demostració

```html
<c-code href="https://github.com"></c-code>

<script type="module">
import "webc.site/Code/_.js";
</script>
```

## Paràmetres de la interfície

L'etiqueta del component és `<c-code>` i l'adreça de salt del codi es pot configurar mitjançant l'atribut `href`.
