# kódové tlačidlo

- Tlačidlo ikony obsahujúca kód
- Kód výzvy po umiestnení myši, podporuje viacero jazykov

## Použite demo

```html
<c-code href="https://github.com"></c-code>
```

## Parametre rozhrania

Označenie komponentu je `<c-code>` a adresu skoku kódu je možné nakonfigurovať pomocou atribútu `href`.
