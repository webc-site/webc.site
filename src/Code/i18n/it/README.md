# pulsante del codice

- Contiene il pulsante dell'icona del codice
- Codice prompt al passaggio del mouse, supporta più lingue

## Utilizza la demo

```html
<c-code href="https://github.com"></c-code>
```

## Parametri dell'interfaccia

L'etichetta del componente è `<c-code>` e l'indirizzo del salto del codice può essere configurato tramite l'attributo `href`.
