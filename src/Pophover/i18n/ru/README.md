# Подсказка о плавающем пузыре

Отображать информацию всплывающей подсказки при наведении указателя мыши или активации элемента.

- Основан на собственном веб-компоненте, легкий и высокопроизводительный.
- Поддерживает адаптивное пространственное позиционирование в четырех направлениях (class="top/bottom/left/right")
- Он использует градиентный фон с текстурой черного матового стекла и белые символы с изысканными физическими контурами контуров.
- Встроенный адаптивный расчет положения автоматически регулирует направление всплывающего окна в соответствии с оставшимся пространством в окне просмотра.
- Используйте `slot="pop"` для отображения содержимого плавающей подсказки.

## Используйте демо-версию

```html
<!-- Автоматически рассчитывать направление (рекомендуется, не нужно объявлять класс) -->
<c-pophover>
  <button>автоматический</button>
  <div slot="pop">Это кончик пузыря</div>
</c-pophover>

<!-- фиксированное направление -->
<c-pophover class="left">
  <button>Левый</button>
  <div slot="pop">Это кончик пузыря</div>
</c-pophover>

<c-pophover class="right">
  <button>верно</button>
  <div slot="pop">Это кончик пузыря</div>
</c-pophover>

<c-pophover class="bottom">
  <button>Вниз</button>
  <div slot="pop">Это кончик пузыря</div>
</c-pophover>

<c-pophover class="top">
  <button>начальство</button>
  <div slot="pop">Это кончик пузыря</div>
</c-pophover>
```

## CSS-переменные

Пользовательские стили поддерживаются с помощью следующих переменных CSS:

| имя переменной | значение по умолчанию | иллюстрировать |
| :--- | :--- | :--- |
| `--pophover-gap` | `8px` | Пространство между всплывающей подсказкой и триггерным элементом |
| `--pophover-margin` | `24px` | Запас безопасности края видового экрана во время расчета адаптивного позиционирования |
| `--pophover-bg-top` | `#18181ce0` | Пузырьковый градиент фона верхнего цвета |
| `--pophover-bg-mid` | `#121215e5` | Цвет фона стрелок слева и справа от пузырька. |
| `--pophover-bg-bottom` | `#0c0c0feb` | Цвет нижней, верхней и нижней боковых стрелок градиентного фона пузырька. |

## Стиль подсказки значка (`.Ico`)

Для кнопок плавающих подсказок чистого типа значков (таких как «Изменить», «Удалить» и т. д.) вы можете применить единый значок и стиль наведения, добавив имя класса `.Ico` к внешнему контейнеру или самому компоненту.

### Пример структуры
```html
<b class="Ico">
  <c-pophover>
    <a></a>
    <div slot="pop">редактировать</div>
  </c-pophover>
</b>
```

### расширение стиля
Базовый макет, размер (18x18 пикселей) и изменения цвета при наведении были унифицированы в `.Ico > c-pophover > a`. Разработчикам нужно лишь указать соответствующий файл значка отдельно:
```stylus
c-edit > c-pophover > a
  mask-image url("./svg/edit.svg")
```
