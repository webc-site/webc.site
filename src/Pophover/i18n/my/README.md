# floating bubble ကိုနှိပ်ပါ။

မောက်စ်ကို ရွှေ့ထားသည့်အခါ သို့မဟုတ် ဒြပ်စင်ကို အသက်သွင်းထားသည့်အခါတွင် ရွှေ့ရန် ကိရိယာအကြံပြုချက် အချက်အလက်ကို ပြသပါ။

- မူရင်း Web Component ကို အခြေခံ၍ ပေါ့ပါးပြီး စွမ်းဆောင်ရည်မြင့်သည်။
- လမ်းညွှန်လေးချက်ဖြင့် လိုက်လျောညီထွေရှိသော spatial positioning ကို ပံ့ပိုးပေးသည် (class="top/bottom/left/right")
- ၎င်းသည် အနက်ရောင် နှင်းခဲထားသော ဖန်သားပြင်ကို အရောင်ခြယ်သည့် နောက်ခံနှင့် အဖြူရောင် စာလုံးများကို အသုံးပြုကာ လက်ရာမြောက်သော ရုပ်ပိုင်းဆိုင်ရာ အစွန်းအထင်းများကို ပေါ်လွင်စေသည်။
- Built-in အနေအထား အလိုက်သင့် တွက်ချက်မှု၊ မြင်ကွင်းပို့တ်ရှိ ကျန်နေရာများအလိုက် ပေါ့ပ်အပ်လမ်းကြောင်းကို အလိုအလျောက်ချိန်ညှိသည်
- မျောနေသော အချက်ပြအကြောင်းအရာကို ပြသရန် `slot="pop"`ကို အသုံးပြုပါ။

## ဒီမိုကိုသုံးပါ။

```html
<!-- ဦးတည်ချက်ကို အလိုအလျောက် တွက်ချက်ခြင်း (အကြံပြုထားသည်၊ အတန်းကို ကြေညာရန် မလိုအပ်ပါ) -->
<c-pophover>
  <button>အလိုအလျောက်</button>
  <div slot="pop">ဒါက ပူဖောင်းထိပ်ဖျားပါ။</div>
</c-pophover>

<!-- ပုံသေ ဦးတည်ချက် -->
<c-pophover class="left">
  <button>ဝဲ</button>
  <div slot="pop">ဒါက ပူဖောင်းထိပ်ဖျားပါ။</div>
</c-pophover>

<c-pophover class="right">
  <button>မှန်တယ်</button>
  <div slot="pop">ဒါက ပူဖောင်းထိပ်ဖျားပါ။</div>
</c-pophover>

<c-pophover class="bottom">
  <button>ဆင်းတယ်။</button>
  <div slot="pop">ဒါက ပူဖောင်းထိပ်ဖျားပါ။</div>
</c-pophover>

<c-pophover class="top">
  <button>သာလွန်သည်။</button>
  <div slot="pop">ဒါက ပူဖောင်းထိပ်ဖျားပါ။</div>
</c-pophover>
```

## CSS ကိန်းရှင်များ

အောက်ပါ CSS variables များမှ စိတ်ကြိုက်စတိုင်များကို ပံ့ပိုးပေးသည်-

| ပြောင်းလဲနိုင်သောအမည် | မူရင်းတန်ဖိုး | ပုံဥပမာ |
| :--- | :--- | :--- |
| `--pophover-gap` | `8px` | bubble tooltip နှင့် triggering element အကြား နေရာလွတ် |
| `--pophover-margin` | `24px` | လိုက်လျောညီထွေရှိသော နေရာချထားမှုကို တွက်ချက်နေစဉ် Viewport edge ဘေးကင်းရေးအနားသတ် |
| `--pophover-bg-top` | `#18181ce0` | Bubble gradient နောက်ခံအပေါ်ဆုံးအရောင် |
| `--pophover-bg-mid` | `#121215e5` | ပူဖောင်း၏ ဘယ်ဘက်နှင့် ညာဘက်ရှိ မြှားများ၏ နောက်ခံအရောင် |
| `--pophover-bg-bottom` | `#0c0c0feb` | bubble gradient နောက်ခံ၏ အောက်ခြေနှင့် အထက်နှင့် အောက်ဘက်မြှားများ၏ အရောင် |

## အိုင်ကွန် ကိရိယာအကြံပြုချက်စတိုင် (`.Ico`)

စင်ကြယ်သောအိုင်ကွန်အမျိုးအစားအတွက် ရေပေါ်အမှာစာခလုတ်များ (ဥပမာ "တည်းဖြတ်ရန်" "ဖျက်ရန်" စသည်ဖြင့်)၊ သင်သည် `.Ico`အတန်းအမည်ကို ပြင်ပကွန်တိန်နာ သို့မဟုတ် အစိတ်အပိုင်းကိုယ်တိုင်ထည့်သွင်းခြင်းဖြင့် ပေါင်းစည်းထားသော အိုင်ကွန်နှင့် မောက်စ်ပုံစံကို အသုံးပြုနိုင်သည်။

### ဖွဲ့စည်းပုံဥပမာ
```html
<b class="Ico">
  <c-pophover>
    <a></a>
    <div slot="pop">တည်းဖြတ်ပါ။</div>
  </c-pophover>
</b>
```

### စတိုင်တိုးချဲ့မှု
အခြေခံ အပြင်အဆင်၊ အရွယ်အစား (18x18px) နှင့် ရွှေ့ထားသော အရောင်ပြောင်းလဲမှုများကို `.Ico > c-pophover > a` တွင် ပေါင်းစည်းထားသည်။ ဆော့ဖ်ဝဲရေးသားသူများသည် သက်ဆိုင်ရာ အိုင်ကွန်ဖိုင်ကို သီးခြားသတ်မှတ်ရန်သာ လိုအပ်သည်-
```stylus
c-edit > c-pophover > a
  mask-image url("./svg/edit.svg")
```
