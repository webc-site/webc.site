# फ्लोटिंग बबल प्रम्प्ट

माउस होभर गरिएको वा तत्व सक्रिय हुँदा होभर टूलटिप जानकारी प्रदर्शन गर्नुहोस्।

- नेटिभ वेब कम्पोनेन्टमा आधारित, हल्का र उच्च प्रदर्शन
- चार दिशामा अनुकूली स्थानिक स्थिति समर्थन गर्दछ (वर्ग = "माथि/तल/बायाँ/दायाँ")
- यसले कालो फ्रस्टेड गिलास बनावट ढाँचा पृष्ठभूमि र सेतो वर्णहरू प्रयोग गर्दछ, उत्कृष्ट भौतिक किनारा समोच्च हाइलाइटहरू सहित।
- बिल्ट-इन स्थिति अनुकूली गणना, स्वचालित रूपमा भ्यूपोर्टमा बाँकी ठाउँ अनुसार पप-अप दिशा समायोजन गर्दछ
- फ्लोटिंग प्रम्प्ट सामग्री प्रदर्शन गर्न `slot="pop"` प्रयोग गर्नुहोस्

## डेमो प्रयोग गर्नुहोस्

```html
<!-- स्वचालित रूपमा दिशा गणना गर्नुहोस् (सिफारिस गरिएको, कक्षा घोषणा गर्न आवश्यक छैन) -->
<c-pophover>
  <button>स्वचालित</button>
  <div slot="pop">यो बबल टिप हो</div>
</c-pophover>

<!-- निश्चित दिशा -->
<c-pophover class="left">
  <button>बायाँ</button>
  <div slot="pop">यो बबल टिप हो</div>
</c-pophover>

<c-pophover class="right">
  <button>सही</button>
  <div slot="pop">यो बबल टिप हो</div>
</c-pophover>

<c-pophover class="bottom">
  <button>तल</button>
  <div slot="pop">यो बबल टिप हो</div>
</c-pophover>

<c-pophover class="top">
  <button>उच्च</button>
  <div slot="pop">यो बबल टिप हो</div>
</c-pophover>
```

## CSS चरहरू

अनुकूलन शैलीहरू निम्न CSS चरहरू मार्फत समर्थित छन्:

| चर नाम | पूर्वनिर्धारित मान | चित्रण गर्नुहोस् |
| :--- | :--- | :--- |
| `--pophover-gap` | `8px` | बबल टुलटिप र ट्रिगर गर्ने तत्व बीचको खाली ठाउँ |
| `--pophover-margin` | `24px` | अनुकूली स्थिति गणना गर्दा भ्युपोर्ट किनारा सुरक्षा मार्जिन |
| `--pophover-bg-top` | `#18181ce0` | बबल ग्रेडियन्ट पृष्ठभूमि शीर्ष रङ |
| `--pophover-bg-mid` | `#121215e5` | बबलको बायाँ र दायाँ छेउमा तीरहरूको पृष्ठभूमि रङ |
| `--pophover-bg-bottom` | `#0c0c0feb` | बबल ग्रेडियन्ट पृष्ठभूमिको तल र माथिल्लो र तल्लो छेउको तीरहरूको रङ |

## आइकन टुलटिप शैली (`.Ico`)

शुद्ध आइकन प्रकारको फ्लोटिंग प्रम्प्ट बटनहरू (जस्तै "सम्पादन गर्नुहोस्", "मेट्नुहोस्", आदि) को लागि, तपाईंले बाहिरी कन्टेनर वा कम्पोनेन्टमा `.Ico` वर्ग नाम थपेर एकीकृत प्रतिमा र होभर शैली लागू गर्न सक्नुहुन्छ।

### संरचना उदाहरण
```html
<b class="Ico">
  <c-pophover>
    <a></a>
    <div slot="pop">सम्पादन गर्नुहोस्</div>
  </c-pophover>
</b>
```

### शैली विस्तार
आधारभूत लेआउट, आकार (18x18px), र होभर रंग परिवर्तनहरू `.Ico > c-pophover > a` मा एकीकृत गरिएको छ। विकासकर्ताहरूले मात्र सम्बन्धित प्रतिमा फाइल छुट्टै निर्दिष्ट गर्न आवश्यक छ:
```stylus
c-edit > c-pophover > a
  mask-image url("./svg/edit.svg")
```
