# подкана за плаващ балон

Показване на информация за подсказка при задържане на мишката, когато мишката е задържана или елементът е активиран.

- Базиран на собствен уеб компонент, лек и високопроизводителен
- Поддържа адаптивно пространствено позициониране в четири посоки (class="top/bottom/left/right")
- Той използва градиентен фон с черна матирана стъклена текстура и бели знаци, с изящни отблясъци на физически контури.
- Вградено адаптивно изчисление на позицията, автоматично настройва посоката на изскачащи прозорци според оставащото пространство в прозореца за изглед
- Използвайте `slot="pop"` за показване на плаващо подканващо съдържание

## Използвайте демонстрацията

```html
<!-- Автоматично изчисляване на посоката (препоръчително, няма нужда да декларирате клас) -->
<c-pophover>
  <button>автоматичен</button>
  <div slot="pop">Това е балонен връх</div>
</c-pophover>

<!-- фиксирана посока -->
<c-pophover class="left">
  <button>наляво</button>
  <div slot="pop">Това е балонен връх</div>
</c-pophover>

<c-pophover class="right">
  <button>точно</button>
  <div slot="pop">Това е балонен връх</div>
</c-pophover>

<c-pophover class="bottom">
  <button>Надолу</button>
  <div slot="pop">Това е балонен връх</div>
</c-pophover>

<c-pophover class="top">
  <button>превъзхождащ</button>
  <div slot="pop">Това е балонен връх</div>
</c-pophover>
```

## CSS променливи

Персонализираните стилове се поддържат чрез следните CSS променливи:

| име на променлива | стойност по подразбиране | илюстрирам |
| :--- | :--- | :--- |
| `--pophover-gap` | `8px` | Пространството между подсказката на балончето и задействащия елемент |
| `--pophover-margin` | `24px` | Марж за безопасност на ръба на прозореца за изглед по време на изчисляване на адаптивното позициониране |
| `--pophover-bg-top` | `#18181ce0` | Балон градиент фон цвят на горната част |
| `--pophover-bg-mid` | `#121215e5` | Цветът на фона на стрелките от лявата и дясната страна на балона |
| `--pophover-bg-bottom` | `#0c0c0feb` | Цветът на долната и горната и долната странични стрелки на балонния градиентен фон |

## Стил на подсказка за икона (`.Ico`)

За плаващи подканващи бутони от чист тип икона (като „Редактиране“, „Изтриване“ и т.н.) можете да приложите унифицирана икона и стил на задържане, като добавите името на класа `.Ico` към самия външен контейнер или компонент.

### Пример за структура
```html
<b class="Ico">
  <c-pophover>
    <a></a>
    <div slot="pop">редактиране</div>
  </c-pophover>
</b>
```

### разширение на стила
Основното оформление, размерът (18x18px) и промените в цвета при задържане на мишката са унифицирани в `.Ico > c-pophover > a`. Разработчиците трябва само да посочат отделно съответния файл с икона:
```stylus
c-edit > c-pophover > a
  mask-image url("./svg/edit.svg")
```
