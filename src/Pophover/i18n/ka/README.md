# მცურავი ბუშტის მოთხოვნა

მაუსის დაჭერისას ან ელემენტის გააქტიურებისას აჩვენეთ მაუსის მინიშნების ინფორმაცია.

- ვებ კომპონენტზე დაფუძნებული, მსუბუქი და მაღალი ხარისხის
- მხარს უჭერს ადაპტირებულ სივრცულ პოზიციონირებას ოთხი მიმართულებით (class="top/bottom/left/right")
- იგი იყენებს შავი ყინვაგამძლე მინის ტექსტურის გრადიენტურ ფონს და თეთრ სიმბოლოებს, დახვეწილი ფიზიკური კიდეების კონტურის ხაზგასმა.
- ჩამონტაჟებული პოზიციის ადაპტაციური გაანგარიშება, ავტომატურად არეგულირებს ამომხტარის მიმართულებას ხედის პორტში დარჩენილი სივრცის მიხედვით
- გამოიყენეთ `slot="pop"` მცურავი მოთხოვნის შინაარსის საჩვენებლად

## გამოიყენეთ დემო ვერსია

```html
<!-- მიმართულების ავტომატური გამოთვლა (რეკომენდირებულია, არ არის საჭირო კლასის გამოცხადება) -->
<c-pophover>
  <button>ავტომატური</button>
  <div slot="pop">ეს არის ბუშტის წვერი</div>
</c-pophover>

<!-- ფიქსირებული მიმართულება -->
<c-pophover class="left">
  <button>მარცხენა</button>
  <div slot="pop">ეს არის ბუშტის წვერი</div>
</c-pophover>

<c-pophover class="right">
  <button>უფლება</button>
  <div slot="pop">ეს არის ბუშტის წვერი</div>
</c-pophover>

<c-pophover class="bottom">
  <button>ქვემოთ</button>
  <div slot="pop">ეს არის ბუშტის წვერი</div>
</c-pophover>

<c-pophover class="top">
  <button>უმაღლესი</button>
  <div slot="pop">ეს არის ბუშტის წვერი</div>
</c-pophover>
```

## CSS ცვლადები

მორგებული სტილები მხარდაჭერილია შემდეგი CSS ცვლადების მეშვეობით:

| ცვლადის სახელი | ნაგულისხმევი მნიშვნელობა | ილუსტრირება |
| :--- | :--- | :--- |
| `--pophover-gap` | `8px` | სივრცე ბუშტის ხელსაწყოს მინიშნებასა და გამომწვევ ელემენტს შორის |
| `--pophover-margin` | `24px` | ხედვის პორტის კიდეების უსაფრთხოების ზღვარი ადაპტური პოზიციონირების გაანგარიშებისას |
| `--pophover-bg-top` | `#18181ce0` | ბუშტის გრადიენტის ფონის ზედა ფერი |
| `--pophover-bg-mid` | `#121215e5` | ისრების ფონის ფერი ბუშტის მარცხენა და მარჯვენა მხარეს |
| `--pophover-bg-bottom` | `#0c0c0feb` | ბუშტის გრადიენტის ფონის ქვედა და ზედა და ქვედა გვერდითი ისრების ფერი |

## ხატულა ხელსაწყოს მინიშნების სტილი (`.Ico`)

წმინდა ხატის ტიპის მცურავი მოთხოვნის ღილაკებისთვის (როგორიცაა "რედაქტირება", "წაშლა" და ა.შ.), შეგიძლიათ გამოიყენოთ ერთიანი ხატულა და გადაადგილების სტილი `.Ico` კლასის სახელის დამატებით გარე კონტეინერში ან კომპონენტში.

### სტრუქტურის მაგალითი
```html
<b class="Ico">
  <c-pophover>
    <a></a>
    <div slot="pop">რედაქტირება</div>
  </c-pophover>
</b>
```

### სტილის გაფართოება
ძირითადი განლაგება, ზომა (18x18 პიქსელი) და ჰოვერის ფერის ცვლილებები გაერთიანებულია `.Ico > c-pophover > a`-ში. დეველოპერებს მხოლოდ შესაბამისი ხატის ფაილი ცალკე უნდა მიუთითონ:
```stylus
c-edit > c-pophover > a
  mask-image url("./svg/edit.svg")
```
