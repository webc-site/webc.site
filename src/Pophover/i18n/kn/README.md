# ತೇಲುವ ಬಬಲ್ ಪ್ರಾಂಪ್ಟ್

ಮೌಸ್ ಅನ್ನು ಸುಳಿದಾಡಿದಾಗ ಅಥವಾ ಅಂಶವನ್ನು ಸಕ್ರಿಯಗೊಳಿಸಿದಾಗ ಹೋವರ್ ಟೂಲ್ಟಿಪ್ ಮಾಹಿತಿಯನ್ನು ಪ್ರದರ್ಶಿಸಿ.

- ಸ್ಥಳೀಯ ವೆಬ್ ಕಾಂಪೊನೆಂಟ್, ಹಗುರವಾದ ಮತ್ತು ಹೆಚ್ಚಿನ ಕಾರ್ಯಕ್ಷಮತೆಯನ್ನು ಆಧರಿಸಿದೆ
- ನಾಲ್ಕು ದಿಕ್ಕುಗಳಲ್ಲಿ ಹೊಂದಾಣಿಕೆಯ ಪ್ರಾದೇಶಿಕ ಸ್ಥಾನವನ್ನು ಬೆಂಬಲಿಸುತ್ತದೆ (ವರ್ಗ = "ಮೇಲ್ಭಾಗ/ಕೆಳಗೆ/ಎಡ/ಬಲ")
- ಇದು ಕಪ್ಪು ಫ್ರಾಸ್ಟೆಡ್ ಗ್ಲಾಸ್ ಟೆಕ್ಸ್ಚರ್ ಗ್ರೇಡಿಯಂಟ್ ಹಿನ್ನೆಲೆ ಮತ್ತು ಬಿಳಿ ಅಕ್ಷರಗಳನ್ನು ಬಳಸುತ್ತದೆ, ಅಂದವಾದ ಭೌತಿಕ ಅಂಚಿನ ಬಾಹ್ಯರೇಖೆಯ ಮುಖ್ಯಾಂಶಗಳೊಂದಿಗೆ.
- ಅಂತರ್ನಿರ್ಮಿತ ಸ್ಥಾನ ಹೊಂದಾಣಿಕೆಯ ಲೆಕ್ಕಾಚಾರ, ವೀಕ್ಷಣೆ ಪೋರ್ಟ್‌ನಲ್ಲಿ ಉಳಿದಿರುವ ಜಾಗಕ್ಕೆ ಅನುಗುಣವಾಗಿ ಪಾಪ್-ಅಪ್ ದಿಕ್ಕನ್ನು ಸ್ವಯಂಚಾಲಿತವಾಗಿ ಸರಿಹೊಂದಿಸುತ್ತದೆ
- ತೇಲುವ ಪ್ರಾಂಪ್ಟ್ ವಿಷಯವನ್ನು ಪ್ರದರ್ಶಿಸಲು `slot="pop"` ಬಳಸಿ

## ಡೆಮೊ ಬಳಸಿ

```html
<!-- ದಿಕ್ಕನ್ನು ಸ್ವಯಂಚಾಲಿತವಾಗಿ ಲೆಕ್ಕಾಚಾರ ಮಾಡಿ (ಶಿಫಾರಸು ಮಾಡಲಾಗಿದೆ, ವರ್ಗವನ್ನು ಘೋಷಿಸುವ ಅಗತ್ಯವಿಲ್ಲ) -->
<c-pophover>
  <button>ಸ್ವಯಂಚಾಲಿತ</button>
  <div slot="pop">ಇದು ಬಬಲ್ ಟಿಪ್ ಆಗಿದೆ</div>
</c-pophover>

<!-- ಸ್ಥಿರ ದಿಕ್ಕು -->
<c-pophover class="left">
  <button>ಎಡಕ್ಕೆ</button>
  <div slot="pop">ಇದು ಬಬಲ್ ಟಿಪ್ ಆಗಿದೆ</div>
</c-pophover>

<c-pophover class="right">
  <button>ಬಲ</button>
  <div slot="pop">ಇದು ಬಬಲ್ ಟಿಪ್ ಆಗಿದೆ</div>
</c-pophover>

<c-pophover class="bottom">
  <button>ಕೆಳಗೆ</button>
  <div slot="pop">ಇದು ಬಬಲ್ ಟಿಪ್ ಆಗಿದೆ</div>
</c-pophover>

<c-pophover class="top">
  <button>ಉನ್ನತವಾದ</button>
  <div slot="pop">ಇದು ಬಬಲ್ ಟಿಪ್ ಆಗಿದೆ</div>
</c-pophover>
```

## CSS ಅಸ್ಥಿರ

ಕೆಳಗಿನ CSS ವೇರಿಯೇಬಲ್‌ಗಳ ಮೂಲಕ ಕಸ್ಟಮ್ ಶೈಲಿಗಳನ್ನು ಬೆಂಬಲಿಸಲಾಗುತ್ತದೆ:

| ವೇರಿಯಬಲ್ ಹೆಸರು | ಡೀಫಾಲ್ಟ್ ಮೌಲ್ಯ | ವಿವರಿಸಲು |
| :--- | :--- | :--- |
| `--pophover-gap` | `8px` | ಬಬಲ್ ಟೂಲ್ಟಿಪ್ ಮತ್ತು ಟ್ರಿಗ್ಗರಿಂಗ್ ಎಲಿಮೆಂಟ್ ನಡುವಿನ ಅಂತರ |
| `--pophover-margin` | `24px` | ಅಡಾಪ್ಟಿವ್ ಪೊಸಿಷನಿಂಗ್ ಲೆಕ್ಕಾಚಾರದ ಸಮಯದಲ್ಲಿ ವೀಕ್ಷಿಸಿ ಪೋರ್ಟ್ ಅಂಚಿನ ಸುರಕ್ಷತೆ ಅಂಚು |
| `--pophover-bg-top` | `#18181ce0` | ಬಬಲ್ ಗ್ರೇಡಿಯಂಟ್ ಹಿನ್ನೆಲೆ ಮೇಲಿನ ಬಣ್ಣ |
| `--pophover-bg-mid` | `#121215e5` | ಬಬಲ್‌ನ ಎಡ ಮತ್ತು ಬಲ ಬದಿಯಲ್ಲಿರುವ ಬಾಣಗಳ ಹಿನ್ನೆಲೆ ಬಣ್ಣ |
| `--pophover-bg-bottom` | `#0c0c0feb` | ಬಬಲ್ ಗ್ರೇಡಿಯಂಟ್ ಹಿನ್ನೆಲೆಯ ಕೆಳಗಿನ ಮತ್ತು ಮೇಲಿನ ಮತ್ತು ಕೆಳಗಿನ ಬದಿಯ ಬಾಣಗಳ ಬಣ್ಣ |

## ಐಕಾನ್ ಟೂಲ್‌ಟಿಪ್ ಶೈಲಿ (`.Ico`)

ಶುದ್ಧ ಐಕಾನ್ ಪ್ರಕಾರದ ಫ್ಲೋಟಿಂಗ್ ಪ್ರಾಂಪ್ಟ್ ಬಟನ್‌ಗಳಿಗಾಗಿ ("ಸಂಪಾದಿಸು", "ಅಳಿಸು", ಇತ್ಯಾದಿ), ನೀವು `.Ico` ವರ್ಗದ ಹೆಸರನ್ನು ಹೊರಗಿನ ಕಂಟೇನರ್ ಅಥವಾ ಘಟಕಕ್ಕೆ ಸೇರಿಸುವ ಮೂಲಕ ಏಕೀಕೃತ ಐಕಾನ್ ಮತ್ತು ಹೋವರ್ ಶೈಲಿಯನ್ನು ಅನ್ವಯಿಸಬಹುದು.

### ರಚನೆಯ ಉದಾಹರಣೆ
```html
<b class="Ico">
  <c-pophover>
    <a></a>
    <div slot="pop">ಸಂಪಾದಿಸು</div>
  </c-pophover>
</b>
```

### ಶೈಲಿ ವಿಸ್ತರಣೆ
ಮೂಲ ವಿನ್ಯಾಸ, ಗಾತ್ರ (18x18px), ಮತ್ತು ಹೂವರ್ ಬಣ್ಣ ಬದಲಾವಣೆಗಳನ್ನು `.Ico > c-pophover > a` ನಲ್ಲಿ ಏಕೀಕರಿಸಲಾಗಿದೆ. ಡೆವಲಪರ್‌ಗಳು ಅನುಗುಣವಾದ ಐಕಾನ್ ಫೈಲ್ ಅನ್ನು ಪ್ರತ್ಯೇಕವಾಗಿ ನಿರ್ದಿಷ್ಟಪಡಿಸುವ ಅಗತ್ಯವಿದೆ:
```stylus
c-edit > c-pophover > a
  mask-image url("./svg/edit.svg")
```
