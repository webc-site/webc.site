# ફ્લોટિંગ બબલ પ્રોમ્પ્ટ

જ્યારે માઉસ હોવર કરવામાં આવે અથવા તત્વ સક્રિય થાય ત્યારે હોવર ટૂલટિપ માહિતી દર્શાવો.

- મૂળ વેબ ઘટક, હલકો અને ઉચ્ચ-પ્રદર્શન પર આધારિત
- ચાર દિશાઓમાં અનુકૂલનશીલ અવકાશી સ્થિતિને સમર્થન આપે છે (વર્ગ = "ટોપ/નીચે/ડાબે/જમણે")
- તે કાળા ફ્રોસ્ટેડ ગ્લાસ ટેક્સચર ગ્રેડિયન્ટ બેકગ્રાઉન્ડ અને સફેદ અક્ષરોનો ઉપયોગ કરે છે, જેમાં ઉત્કૃષ્ટ ભૌતિક ધાર કોન્ટૂર હાઇલાઇટ્સ છે.
- બિલ્ટ-ઇન પોઝિશન અનુકૂલનશીલ ગણતરી, વ્યુપોર્ટમાં બાકીની જગ્યા અનુસાર પોપ-અપ દિશાને આપમેળે ગોઠવે છે
- ફ્લોટિંગ પ્રોમ્પ્ટ સામગ્રી પ્રદર્શિત કરવા માટે `slot="pop"` નો ઉપયોગ કરો

## ડેમોનો ઉપયોગ કરો

```html
<!-- આપમેળે દિશાની ગણતરી કરો (ભલામણ કરેલ, વર્ગ જાહેર કરવાની જરૂર નથી) -->
<c-pophover>
  <button>આપોઆપ</button>
  <div slot="pop">આ એક બબલ ટીપ છે</div>
</c-pophover>

<!-- નિશ્ચિત દિશા -->
<c-pophover class="left">
  <button>ડાબી</button>
  <div slot="pop">આ એક બબલ ટીપ છે</div>
</c-pophover>

<c-pophover class="right">
  <button>અધિકાર</button>
  <div slot="pop">આ એક બબલ ટીપ છે</div>
</c-pophover>

<c-pophover class="bottom">
  <button>નીચે</button>
  <div slot="pop">આ એક બબલ ટીપ છે</div>
</c-pophover>

<c-pophover class="top">
  <button>શ્રેષ્ઠ</button>
  <div slot="pop">આ એક બબલ ટીપ છે</div>
</c-pophover>
```

## CSS ચલો

કસ્ટમ શૈલીઓ નીચેના CSS ચલો દ્વારા સમર્થિત છે:

| ચલ નામ | ડિફૉલ્ટ મૂલ્ય | સમજાવવું |
| :--- | :--- | :--- |
| `--pophover-gap` | `8px` | બબલ ટૂલટિપ અને ટ્રિગરિંગ ઘટક વચ્ચેની જગ્યા |
| `--pophover-margin` | `24px` | અનુકૂલનશીલ સ્થિતિની ગણતરી દરમિયાન વ્યુપોર્ટ એજ સલામતી માર્જિન |
| `--pophover-bg-top` | `#18181ce0` | બબલ ગ્રેડિયન્ટ પૃષ્ઠભૂમિ ટોચનો રંગ |
| `--pophover-bg-mid` | `#121215e5` | બબલની ડાબી અને જમણી બાજુએ તીરોનો પૃષ્ઠભૂમિ રંગ |
| `--pophover-bg-bottom` | `#0c0c0feb` | બબલ ગ્રેડિયન્ટ બેકગ્રાઉન્ડના તળિયે અને ઉપલા અને નીચલા બાજુના તીરોનો રંગ |

## આયકન ટૂલટીપ શૈલી (`.Ico`)

શુદ્ધ આઇકન પ્રકારના ફ્લોટિંગ પ્રોમ્પ્ટ બટનો માટે (જેમ કે "સંપાદિત કરો", "કાઢી નાખો", વગેરે), તમે બાહ્ય કન્ટેનર અથવા ઘટકમાં જ `.Ico` વર્ગ નામ ઉમેરીને એકીકૃત આઇકન અને હોવર શૈલી લાગુ કરી શકો છો.

### માળખું ઉદાહરણ
```html
<b class="Ico">
  <c-pophover>
    <a></a>
    <div slot="pop">ફેરફાર કરો</div>
  </c-pophover>
</b>
```

### શૈલી વિસ્તરણ
મૂળભૂત લેઆઉટ, કદ (18x18px), અને હોવર રંગ ફેરફારો `.Ico > c-pophover > a` માં એકીકૃત કરવામાં આવ્યા છે. વિકાસકર્તાઓએ માત્ર અનુરૂપ આઇકોન ફાઇલને અલગથી સ્પષ્ટ કરવાની જરૂર છે:
```stylus
c-edit > c-pophover > a
  mask-image url("./svg/edit.svg")
```
