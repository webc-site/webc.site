# تیرتا ہوا بلبلا پرامپٹ

ہوور ٹول ٹِپ کی معلومات ڈسپلے کریں جب ماؤس ہور ہو یا عنصر فعال ہو۔

- مقامی ویب اجزاء پر مبنی، ہلکا پھلکا اور اعلی کارکردگی
- چار سمتوں میں انکولی مقامی پوزیشننگ کی حمایت کرتا ہے (کلاس = اوپر/نیچے/بائیں/دائیں")
- یہ سیاہ فروسٹڈ شیشے کی ساخت کے گراڈینٹ پس منظر اور سفید حروف کا استعمال کرتا ہے، جس میں شاندار فزیکل ایج کونٹور ہائی لائٹس ہیں۔
- بلٹ ان پوزیشن ایڈاپٹیو کیلکولیشن، ویو پورٹ میں باقی جگہ کے مطابق خود بخود پاپ اپ سمت کو ایڈجسٹ کرتا ہے
- فلوٹنگ پرامپٹ مواد ڈسپلے کرنے کے لیے `slot="pop"` استعمال کریں۔

## ڈیمو استعمال کریں۔

```html
<!-- خود بخود سمت کا حساب لگائیں (تجویز کردہ، کلاس کا اعلان کرنے کی ضرورت نہیں) -->
<c-pophover>
  <button>خودکار</button>
  <div slot="pop">یہ ایک بلبلا ٹپ ہے۔</div>
</c-pophover>

<!-- مقررہ سمت -->
<c-pophover class="left">
  <button>بائیں</button>
  <div slot="pop">یہ ایک بلبلا ٹپ ہے۔</div>
</c-pophover>

<c-pophover class="right">
  <button>صحیح</button>
  <div slot="pop">یہ ایک بلبلا ٹپ ہے۔</div>
</c-pophover>

<c-pophover class="bottom">
  <button>نیچے</button>
  <div slot="pop">یہ ایک بلبلا ٹپ ہے۔</div>
</c-pophover>

<c-pophover class="top">
  <button>اعلی</button>
  <div slot="pop">یہ ایک بلبلا ٹپ ہے۔</div>
</c-pophover>
```

## سی ایس ایس متغیرات

حسب ضرورت طرزیں درج ذیل CSS متغیرات کے ذریعے سپورٹ کی جاتی ہیں:

| متغیر نام | پہلے سے طے شدہ قدر | مثال دینا |
| :--- | :--- | :--- |
| `--pophover-gap` | `8px` | بلبلا ٹول ٹپ اور محرک عنصر کے درمیان کی جگہ |
| `--pophover-margin` | `24px` | انکولی پوزیشننگ کیلکولیشن کے دوران ویو پورٹ ایج سیفٹی مارجن |
| `--pophover-bg-top` | `#18181ce0` | بلبلا گریڈینٹ پس منظر کا سب سے اوپر کا رنگ |
| `--pophover-bg-mid` | `#121215e5` | بلبلے کے بائیں اور دائیں جانب تیروں کے پس منظر کا رنگ |
| `--pophover-bg-bottom` | `#0c0c0feb` | ببل گریڈینٹ پس منظر کے نیچے اور اوپری اور نچلے طرف والے تیروں کا رنگ |

## آئیکن ٹول ٹِپ اسٹائل (`.Ico`)

خالص آئیکن ٹائپ فلوٹنگ پرامپٹ بٹن (جیسے "ترمیم"، "حذف کریں" وغیرہ) کے لیے، آپ بیرونی کنٹینر یا جزو میں ہی `.Ico` کلاس کا نام شامل کرکے ایک متحد آئیکن اور ہوور اسٹائل کا اطلاق کرسکتے ہیں۔

### ساخت کی مثال
```html
<b class="Ico">
  <c-pophover>
    <a></a>
    <div slot="pop">ترمیم کریں</div>
  </c-pophover>
</b>
```

### سٹائل کی توسیع
بنیادی ترتیب، سائز (18x18px) اور ہوور رنگ کی تبدیلیوں کو `.Ico > c-pophover > a` میں یکجا کر دیا گیا ہے۔ ڈویلپرز کو صرف متعلقہ آئیکن فائل کو الگ سے بتانے کی ضرورت ہے:
```stylus
c-edit > c-pophover > a
  mask-image url("./svg/edit.svg")
```
