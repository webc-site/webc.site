# พรอมต์ฟองลอย

แสดงข้อมูลคำแนะนำเครื่องมือโฮเวอร์เมื่อวางเมาส์หรือองค์ประกอบถูกเปิดใช้งาน

- อิงตาม Web Component ดั้งเดิม น้ำหนักเบาและประสิทธิภาพสูง
- รองรับการวางตำแหน่งเชิงพื้นที่แบบปรับได้ในสี่ทิศทาง (class="top/bottom/left/right")
- ใช้พื้นหลังไล่ระดับพื้นผิวกระจกฝ้าสีดำและตัวอักษรสีขาว พร้อมไฮไลท์เส้นขอบทางกายภาพอันงดงาม
- การคำนวณแบบปรับได้ในตัวจะปรับทิศทางป๊อปอัปโดยอัตโนมัติตามพื้นที่ที่เหลืออยู่ในวิวพอร์ต
- ใช้ `slot="pop"` เพื่อแสดงเนื้อหาพร้อมท์ลอย

## ใช้การสาธิต

```html
<!-- คำนวณทิศทางอัตโนมัติ (แนะนำ ไม่ต้องประกาศคลาส) -->
<c-pophover>
  <button>อัตโนมัติ</button>
  <div slot="pop">นี่คือเคล็ดลับฟอง</div>
</c-pophover>

<!-- ทิศทางคงที่ -->
<c-pophover class="left">
  <button>ซ้าย</button>
  <div slot="pop">นี่คือเคล็ดลับฟอง</div>
</c-pophover>

<c-pophover class="right">
  <button>ขวา</button>
  <div slot="pop">นี่คือเคล็ดลับฟอง</div>
</c-pophover>

<c-pophover class="bottom">
  <button>ลง</button>
  <div slot="pop">นี่คือเคล็ดลับฟอง</div>
</c-pophover>

<c-pophover class="top">
  <button>เหนือกว่า</button>
  <div slot="pop">นี่คือเคล็ดลับฟอง</div>
</c-pophover>
```

## ตัวแปรซีเอสเอส

สไตล์ที่กำหนดเองได้รับการสนับสนุนผ่านตัวแปร CSS ต่อไปนี้:

| ชื่อตัวแปร | ค่าเริ่มต้น | แสดงให้เห็น |
| :--- | :--- | :--- |
| `--pophover-gap` | `8px` | ช่องว่างระหว่างเคล็ดลับเครื่องมือบับเบิลและองค์ประกอบที่ทริกเกอร์ |
| `--pophover-margin` | `24px` | อัตราความปลอดภัยของขอบวิวพอร์ตระหว่างการคำนวณตำแหน่งที่ปรับเปลี่ยนได้ |
| `--pophover-bg-top` | `#18181ce0` | สีด้านบนของพื้นหลังไล่ระดับสีฟอง |
| `--pophover-bg-mid` | `#121215e5` | สีพื้นหลังของลูกศรด้านซ้ายและด้านขวาของฟอง |
| `--pophover-bg-bottom` | `#0c0c0feb` | สีของลูกศรด้านล่างและด้านบนและด้านล่างของพื้นหลังไล่ระดับสีแบบฟอง |

## รูปแบบคำแนะนำเครื่องมือไอคอน (`.Ico`)

สำหรับปุ่มพรอมต์แบบลอยประเภทไอคอนล้วน (เช่น "แก้ไข", "ลบ" ฯลฯ ) คุณสามารถใช้ไอคอนแบบรวมและสไตล์โฮเวอร์โดยการเพิ่มชื่อคลาส `.Ico` ให้กับคอนเทนเนอร์ภายนอกหรือส่วนประกอบเอง

### ตัวอย่างโครงสร้าง
```html
<b class="Ico">
  <c-pophover>
    <a></a>
    <div slot="pop">แก้ไข</div>
  </c-pophover>
</b>
```

### ส่วนขยายสไตล์
การเปลี่ยนแปลงเค้าโครงพื้นฐาน ขนาด (18x18px) และสีโฮเวอร์ได้รับการรวมเป็นหนึ่งเดียวใน `.Ico > c-pophover > a` นักพัฒนาจำเป็นต้องระบุไฟล์ไอคอนที่เกี่ยวข้องแยกกัน:
```stylus
c-edit > c-pophover > a
  mask-image url("./svg/edit.svg")
```
