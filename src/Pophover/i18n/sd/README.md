# سچل بلبل جو اشارو

هوور ٽول ٽائپ جي معلومات ڏيکاريو جڏهن ماؤس هور هجي يا عنصر چالو ٿئي.

- مقامي ويب اجزاء جي بنياد تي، هلڪو وزن ۽ اعلي ڪارڪردگي
- چار طرفن ۾ موافقت واري فضائي پوزيشن کي سپورٽ ڪري ٿو (ڪلاس = "مٿي / هيٺان / کاٻي / ساڄي")
- اهو هڪ ڪارو ڀريل شيشي جي بناوت گريجوئيٽ پس منظر ۽ اڇا ڪردارن کي استعمال ڪري ٿو، شاندار فزيڪل ايج ڪنٽور نمايانن سان.
- بلٽ ان پوزيشن موافقت وارو حساب، خودڪار طريقي سان پاپ اپ جي هدايت کي ترتيب ڏئي ٿو وييو پورٽ ۾ باقي خلا جي مطابق
- استعمال ڪريو `slot="pop"` سچل تڪڙي مواد ڏيکارڻ لاءِ

## ڊيمو استعمال ڪريو

```html
<!-- خودڪار طريقي سان حساب ڪريو (سفارش ٿيل، ڪلاس جو اعلان ڪرڻ جي ضرورت ناهي) -->
<c-pophover>
  <button>خودڪار</button>
  <div slot="pop">هي هڪ بلبل ٽپ آهي</div>
</c-pophover>

<!-- مقرر سمت -->
<c-pophover class="left">
  <button>کاٻو</button>
  <div slot="pop">هي هڪ بلبل ٽپ آهي</div>
</c-pophover>

<c-pophover class="right">
  <button>ساڄو</button>
  <div slot="pop">هي هڪ بلبل ٽپ آهي</div>
</c-pophover>

<c-pophover class="bottom">
  <button>هيٺ</button>
  <div slot="pop">هي هڪ بلبل ٽپ آهي</div>
</c-pophover>

<c-pophover class="top">
  <button>اعليٰ</button>
  <div slot="pop">هي هڪ بلبل ٽپ آهي</div>
</c-pophover>
```

## CSS متغير

ڪسٽم اسلوب کي ھيٺ ڏنل CSS متغيرن ذريعي سپورٽ ڪيو ويو آھي:

| متغير جو نالو | ڊفالٽ قدر | بيان ڪرڻ |
| :--- | :--- | :--- |
| `--pophover-gap` | `8px` | بلبل ٽول ٽائپ ۽ محرڪ عنصر جي وچ ۾ خلا |
| `--pophover-margin` | `24px` | موافقت واري پوزيشن جي حساب ڪتاب جي دوران Viewport کنڊ حفاظت مارجن |
| `--pophover-bg-top` | `#18181ce0` | بلبل گريجوئيٽ پس منظر مٿي جو رنگ |
| `--pophover-bg-mid` | `#121215e5` | بلبل جي کاٻي ۽ ساڄي پاسن تي تيرن جو پس منظر رنگ |
| `--pophover-bg-bottom` | `#0c0c0feb` | بلبل گريڊيئينٽ پس منظر جي ھيٺان ۽ مٿئين ۽ ھيٺئين پاسي واري تيرن جو رنگ |

## آئڪن ٽول ٽائپ جو انداز (`.Ico`)

خالص آئڪن ٽائپ فلوٽنگ پرامپٽ بٽڻن لاءِ (جهڙوڪ "ايڊٽ ڪريو"، "ڊيليٽ" وغيره)، توهان هڪ گڏيل آئڪن ۽ هوور انداز لاڳو ڪري سگهو ٿا `.Ico` ڪلاس جو نالو شامل ڪرڻ سان.

### ساخت جو مثال
```html
<b class="Ico">
  <c-pophover>
    <a></a>
    <div slot="pop">ترميم ڪريو</div>
  </c-pophover>
</b>
```

### انداز جي واڌ
بنيادي ترتيب، سائيز (18x18px)، ۽ هور رنگ تبديلين کي متحد ڪيو ويو آهي `.Ico > c-pophover > a` ۾. ڊولپرز کي صرف الڳ الڳ آئڪن فائل کي بيان ڪرڻ جي ضرورت آهي:
```stylus
c-edit > c-pophover > a
  mask-image url("./svg/edit.svg")
```
