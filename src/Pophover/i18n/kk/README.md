# қалқымалы көпіршікті шақыру

Тінтуір меңзерін апарғанда немесе элемент белсендірілгенде, меңзерді апару кеңесі ақпаратын көрсету.

- Жеңіл және өнімділігі жоғары жергілікті веб-компонентке негізделген
- Төрт бағытта адаптивті кеңістікті орналастыруды қолдайды (класс = "жоғары/төменгі/сол/оң")
- Ол қара аязды әйнек текстурасының градиенті фон мен ақ таңбаларды пайдаланады, жиектер контурының керемет физикалық ерекшеліктері бар.
- Кірістірілген позицияға бейімделген есептеу, қарау терезесіндегі қалған кеңістікке сәйкес қалқымалы бағытты автоматты түрде реттейді
- Қалқымалы шақыру мазмұнын көрсету үшін `slot="pop"` пайдаланыңыз

## Демонстрацияны пайдаланыңыз

```html
<!-- Бағытты автоматты түрде есептеу (ұсынылады, сыныпты жариялаудың қажеті жоқ) -->
<c-pophover>
  <button>автоматты</button>
  <div slot="pop">Бұл көпіршікті кеңес</div>
</c-pophover>

<!-- бекітілген бағыт -->
<c-pophover class="left">
  <button>Солға</button>
  <div slot="pop">Бұл көпіршікті кеңес</div>
</c-pophover>

<c-pophover class="right">
  <button>дұрыс</button>
  <div slot="pop">Бұл көпіршікті кеңес</div>
</c-pophover>

<c-pophover class="bottom">
  <button>Төмен</button>
  <div slot="pop">Бұл көпіршікті кеңес</div>
</c-pophover>

<c-pophover class="top">
  <button>жоғары</button>
  <div slot="pop">Бұл көпіршікті кеңес</div>
</c-pophover>
```

## CSS айнымалылары

Теңшелетін стильдерге келесі CSS айнымалылары арқылы қолдау көрсетіледі:

| айнымалы атауы | әдепкі мән | суреттеу |
| :--- | :--- | :--- |
| `--pophover-gap` | `8px` | Көпіршікті құрал кеңесі мен іске қосу элементі арасындағы бос орын |
| `--pophover-margin` | `24px` | Бейімделетін позициялауды есептеу кезінде көрініс портының жиегі қауіпсіздік маржасы |
| `--pophover-bg-top` | `#18181ce0` | Көпіршікті градиент фонының үстіңгі түсі |
| `--pophover-bg-mid` | `#121215e5` | Көпіршіктің сол және оң жағындағы көрсеткілердің фондық түсі |
| `--pophover-bg-bottom` | `#0c0c0feb` | Көпіршікті градиент фонының төменгі және жоғарғы және төменгі бүйірлік көрсеткілерінің түсі |

## Белгіше кеңесінің стилі (`.Ico`)

Таза белгіше түріндегі қалқымалы шақыру түймелері үшін («Өңдеу», «Жою» және т.б.) сыртқы контейнерге немесе құрамдастың өзіне `.Ico` сынып атауын қосу арқылы біртұтас белгішені және меңзерді апару мәнерін қолдануға болады.

### Құрылым үлгісі
```html
<b class="Ico">
  <c-pophover>
    <a></a>
    <div slot="pop">өңдеу</div>
  </c-pophover>
</b>
```

### стиль кеңейтімі
Негізгі орналасу, өлшем (18x18px) және меңзердегі түс өзгерістері `.Ico > c-pophover > a` ішінде біріктірілді. Әзірлеушілер тек тиісті белгіше файлын бөлек көрсетуі керек:
```stylus
c-edit > c-pophover > a
  mask-image url("./svg/edit.svg")
```
