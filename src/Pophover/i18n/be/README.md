# падказка з плаваючай бурбалкай

Адлюстраванне ўсплывальнай падказкі пры навядзенні мышы або пры актывацыі элемента.

- Заснаваны на ўласным вэб-кампаненце, лёгкі і высокапрадукцыйны
- Падтрымлівае адаптыўнае прасторавае пазіцыянаванне ў чатырох кірунках (class="top/bottom/left/right")
- Ён выкарыстоўвае градыентны фон тэкстуры чорнага матавага шкла і белыя сімвалы з вытанчанымі блікамі фізічных контураў краёў.
- Убудаваны адаптыўны разлік пазіцыі, аўтаматычна рэгулюе кірунак усплываючых вокнаў у залежнасці ад пакінутай прасторы ў акне прагляду
- Выкарыстоўвайце `slot="pop"` для адлюстравання плаваючага змесціва падказкі

## Выкарыстоўвайце дэма

```html
<!-- Аўтаматычны разлік напрамку (рэкамендуецца, не трэба аб'яўляць клас) -->
<c-pophover>
  <button>аўтаматычны</button>
  <div slot="pop">Гэта наканечнік бурбалкі</div>
</c-pophover>

<!-- фіксаваны кірунак -->
<c-pophover class="left">
  <button>налева</button>
  <div slot="pop">Гэта наканечнік бурбалкі</div>
</c-pophover>

<c-pophover class="right">
  <button>справа</button>
  <div slot="pop">Гэта наканечнік бурбалкі</div>
</c-pophover>

<c-pophover class="bottom">
  <button>Уніз</button>
  <div slot="pop">Гэта наканечнік бурбалкі</div>
</c-pophover>

<c-pophover class="top">
  <button>вышэйшы</button>
  <div slot="pop">Гэта наканечнік бурбалкі</div>
</c-pophover>
```

## Зменныя CSS

Карыстальніцкія стылі падтрымліваюцца з дапамогай наступных зменных CSS:

| імя зменнай | значэнне па змаўчанні | ілюстраваць |
| :--- | :--- | :--- |
| `--pophover-gap` | `8px` | Прастора паміж падказкай бурбалкі і запускаючым элементам |
| `--pophover-margin` | `24px` | Надзейны запас краю акна прагляду падчас разліку адаптыўнага пазіцыянавання |
| `--pophover-bg-top` | `#18181ce0` | Верхні колер градыентнага фону бурбалкі |
| `--pophover-bg-mid` | `#121215e5` | Колер фону стрэлак злева і справа ад бурбалкі |
| `--pophover-bg-bottom` | `#0c0c0feb` | Колер ніжняй і верхняй і ніжняй бакавых стрэлак градыентнага фону бурбалкі |

## Стыль падказкі значка (`.Ico`)

Для плаваючых кнопак падказкі чыстага тыпу значкоў (напрыклад, "Рэдагаваць", "Выдаліць" і г.д.) вы можаце ўжыць адзіны значок і стыль навядзення, дадаўшы імя класа `.Ico` да знешняга кантэйнера або самога кампанента.

### Прыклад структуры
```html
<b class="Ico">
  <c-pophover>
    <a></a>
    <div slot="pop">рэдагаваць</div>
  </c-pophover>
</b>
```

### пашырэнне стылю
Базавы макет, памер (18x18 пікселяў) і змены колеру пры навядзенні курсора былі ўніфікаваны ў `.Ico > c-pophover > a`. Распрацоўшчыкам трэба толькі асобна ўказаць адпаведны файл значка:
```stylus
c-edit > c-pophover > a
  mask-image url("./svg/edit.svg")
```
