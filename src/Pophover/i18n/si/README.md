# පාවෙන බුබුලු විමසුම

මූසිකය සැරිසරන විට හෝ මූලද්‍රව්‍යය සක්‍රිය කර ඇති විට හෝවර් මෙවලම් ඉඟි තොරතුරු පෙන්වන්න.

- ස්වදේශීය වෙබ් සංරචක මත පදනම්ව, සැහැල්ලු සහ ඉහළ කාර්ය සාධනය
- දිශාවන් හතරකට අනුවර්තී අවකාශීය ස්ථානගත කිරීම සඳහා සහය දක්වයි (පන්තිය = "ඉහළ/පහළ/වම/දකුණ")
- එය අතිවිශිෂ්ට භෞතික දාර සමෝච්ඡ උද්දීපනයන් සහිත කළු තුහීන වීදුරු වයනය අනුක්‍රමණ පසුබිමක් සහ සුදු අක්ෂර භාවිතා කරයි.
- තනා ඇති ස්ථාන අනුවර්තන ගණනය, දර්ශන තොටේ ඉතිරි ඉඩ අනුව උත්පතන දිශාව ස්වයංක්‍රීයව සකසයි
- පාවෙන ක්ෂණික අන්තර්ගතය සංදර්ශන කිරීමට `slot="pop"` භාවිතා කරන්න

## CSS විචල්යයන්

අභිරුචි විලාසයන් පහත CSS විචල්‍යයන් හරහා සහය දක්වයි:

| විචල්ය නාමය | පෙරනිමි අගය | නිදර්ශනය කරන්න |
| :--- | :--- | :--- |
| `--pophover-gap` | `8px` | බුබුලු මෙවලම් ඉඟිය සහ ප්‍රේරක මූලද්‍රව්‍යය අතර අවකාශය |
| `--pophover-margin` | `24px` | අනුවර්තන ස්ථානගත කිරීම ගණනය කිරීමේදී පෝට් දාර ආරක්ෂණ ආන්තිකය |
| `--pophover-bg-top` | `#18181ce0` | බුබුලු ශ්‍රේණියේ පසුබිමේ ඉහළ වර්ණය |
| `--pophover-bg-mid` | `#121215e5` | බුබුලේ වම් සහ දකුණු පස ඇති ඊතලවල පසුබිම් වර්ණය |
| `--pophover-bg-bottom` | `#0c0c0feb` | බුබුලු ශ්‍රේණියේ පසුබිමේ පහළ සහ ඉහළ සහ පහළ පැති ඊතලවල වර්ණය |

## අයිකන මෙවලම් ඉඟි විලාසය (`.Ico`)

පිරිසිදු අයිකන ආකාරයේ පාවෙන විමසුම් බොත්තම් සඳහා ("සංස්කරණය", "මකන්න", ආදිය), ඔබට `.Ico` පන්තියේ නම බාහිර බහාලුමට හෝ සංරචකයට එක් කිරීමෙන් ඒකාබද්ධ අයිකනයක් සහ හොවර් විලාසයක් යෙදිය හැක.

### ව්යුහය උදාහරණය
```html
<b class="Ico">
  <c-pophover>
    <a></a>
    <div slot="pop">සංස්කරණය කරන්න</div>
  </c-pophover>
</b>
```

### ශෛලිය දිගුව
මූලික පිරිසැලසුම, ප්‍රමාණය (18x18px), සහ hover වර්ණ වෙනස්කම් `.Ico > c-pophover > a` තුළ ඒකාබද්ධ කර ඇත. සංවර්ධකයින්ට අවශ්‍ය වන්නේ අනුරූප අයිකන ගොනුව වෙන වෙනම සඳහන් කිරීමට පමණි:
```stylus
c-edit > c-pophover > a
  mask-image url("./svg/edit.svg")
```
