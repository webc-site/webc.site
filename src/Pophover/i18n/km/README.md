# ប្រអប់បញ្ចូលពពុះអណ្តែត

បង្ហាញព័ត៌មានជំនួយឧបករណ៍ដាក់កណ្ដុរ នៅពេលដែលកណ្តុរត្រូវបានដាក់ ឬធាតុត្រូវបានធ្វើឱ្យសកម្ម។

- ផ្អែកលើសមាសធាតុគេហទំព័រដើម ទម្ងន់ស្រាល និងដំណើរការខ្ពស់។
- គាំទ្រ​ការ​កំណត់​ទីតាំង​លំហ​ដែល​ប្រែប្រួល​ក្នុង​ទិស​ទាំង​បួន (ថ្នាក់ = "កំពូល/បាត/ឆ្វេង/ស្ដាំ")
- វាប្រើផ្ទៃខាងក្រោយពណ៌ជម្រាលវាយនភាពកញ្ចក់ពណ៌ខ្មៅ និងតួអក្សរពណ៌ស ជាមួយនឹងការបន្លិចវណ្ឌវង្កតាមគែមដ៏ស្រស់ស្អាត។
- ការ​គណនា​ការ​សម្រប​ខ្លួន​ក្នុង​ទីតាំង​ដែល​ភ្ជាប់​មក​ជាមួយ លៃតម្រូវ​ទិស​លេច​ឡើង​ដោយ​ស្វ័យ​ប្រវត្តិ​តាម​ទំហំ​ដែល​នៅ​សល់​ក្នុង​ច្រក​មើល
- ប្រើ `slot="pop"` ដើម្បីបង្ហាញមាតិកាប្រអប់បញ្ចូលអណ្តែត

## ប្រើការបង្ហាញ

```html
<!-- គណនាទិសដៅដោយស្វ័យប្រវត្តិ (បានណែនាំ មិនចាំបាច់ប្រកាសថ្នាក់) -->
<c-pophover>
  <button>ស្វ័យប្រវត្តិ</button>
  <div slot="pop">នេះគឺជាព័ត៌មានជំនួយពពុះ</div>
</c-pophover>

<!-- ទិសដៅថេរ -->
<c-pophover class="left">
  <button>ឆ្វេង</button>
  <div slot="pop">នេះគឺជាព័ត៌មានជំនួយពពុះ</div>
</c-pophover>

<c-pophover class="right">
  <button>ត្រឹមត្រូវ។</button>
  <div slot="pop">នេះគឺជាព័ត៌មានជំនួយពពុះ</div>
</c-pophover>

<c-pophover class="bottom">
  <button>ចុះ</button>
  <div slot="pop">នេះគឺជាព័ត៌មានជំនួយពពុះ</div>
</c-pophover>

<c-pophover class="top">
  <button>ឧត្តម</button>
  <div slot="pop">នេះគឺជាព័ត៌មានជំនួយពពុះ</div>
</c-pophover>
```

## អថេរ CSS

រចនាប័ទ្មផ្ទាល់ខ្លួនត្រូវបានគាំទ្រតាមរយៈអថេរ CSS ខាងក្រោម៖

| ឈ្មោះអថេរ | តម្លៃលំនាំដើម | បង្ហាញ |
| :--- | :--- | :--- |
| `--pophover-gap` | `8px` | ចន្លោះរវាងព័ត៌មានជំនួយពពុះ និងធាតុកេះ |
| `--pophover-margin` | `24px` | Viewport edge រឹមសុវត្ថិភាព កំឡុងពេលគណនាទីតាំងប្រែប្រួល |
| `--pophover-bg-top` | `#18181ce0` | ពណ៌​ផ្ទៃ​ខាង​ក្រោយ​ជម្រាល​ពពុះ |
| `--pophover-bg-mid` | `#121215e5` | ពណ៌ផ្ទៃខាងក្រោយនៃព្រួញនៅខាងឆ្វេង និងខាងស្តាំនៃពពុះ |
| `--pophover-bg-bottom` | `#0c0c0feb` | ពណ៌នៃព្រួញផ្នែកខាងក្រោម និងខាងលើ និងខាងក្រោមនៃផ្ទៃខាងក្រោយពណ៌ជម្រាលពពុះ |

## រចនាប័ទ្មព័ត៌មានជំនួយរូបតំណាង (`.Ico`)

សម្រាប់​ប្រភេទ​រូបតំណាង​សុទ្ធ ប៊ូតុង​ប្រអប់បញ្ចូល​អណ្តែត​ទឹក (ដូចជា "កែសម្រួល", "លុប" ជាដើម) អ្នកអាច​អនុវត្ត​រូបតំណាង​បង្រួបបង្រួម និង​រចនាប័ទ្ម​ដាក់​ដោយ​បន្ថែម​ឈ្មោះ​ថ្នាក់ `.Ico`ទៅ​កុងតឺន័រ​ខាងក្រៅ ឬ​សមាសធាតុ​ខ្លួនវា។

### ឧទាហរណ៍រចនាសម្ព័ន្ធ
```html
<b class="Ico">
  <c-pophover>
    <a></a>
    <div slot="pop">កែសម្រួល</div>
  </c-pophover>
</b>
```

### ផ្នែកបន្ថែមរចនាប័ទ្ម
ប្លង់​មូលដ្ឋាន ទំហំ (18x18px) និង​ការ​ផ្លាស់​ប្តូរ​ពណ៌​ដាក់​ត្រូវ​បាន​បង្រួប​បង្រួម​ក្នុង `.Ico > c-pophover > a`។ អ្នកអភិវឌ្ឍន៍គ្រាន់តែត្រូវបញ្ជាក់ឯកសាររូបតំណាងដែលត្រូវគ្នាដាច់ដោយឡែកពីគ្នា៖
```stylus
c-edit > c-pophover > a
  mask-image url("./svg/edit.svg")
```
