# ভাসমান বুদ্বুদ প্রম্পট

মাউস ঘোরানো বা উপাদান সক্রিয় করা হলে হোভার টুলটিপ তথ্য প্রদর্শন করুন।

- নেটিভ ওয়েব কম্পোনেন্ট, লাইটওয়েট এবং হাই-পারফরম্যান্সের উপর ভিত্তি করে
- চার দিকে অভিযোজিত স্থানিক অবস্থান সমর্থন করে (class="top/bottom/left/right")
- এটি একটি কালো ফ্রস্টেড গ্লাস টেক্সচার গ্রেডিয়েন্ট ব্যাকগ্রাউন্ড এবং সাদা অক্ষর ব্যবহার করে, চমৎকার ফিজিক্যাল এজ কনট্যুর হাইলাইট সহ।
- অন্তর্নির্মিত অবস্থান অভিযোজিত গণনা, ভিউপোর্টের অবশিষ্ট স্থান অনুযায়ী স্বয়ংক্রিয়ভাবে পপ-আপ দিক সামঞ্জস্য করে
- ভাসমান প্রম্পট সামগ্রী প্রদর্শন করতে `slot="pop"` ব্যবহার করুন৷

## ডেমো ব্যবহার করুন

```html
<!-- স্বয়ংক্রিয়ভাবে দিকনির্দেশ গণনা করুন (প্রস্তাবিত, ক্লাস ঘোষণা করার দরকার নেই) -->
<c-pophover>
  <button>স্বয়ংক্রিয়</button>
  <div slot="pop">এটি একটি বুদবুদ টিপ</div>
</c-pophover>

<!-- নির্দিষ্ট দিক -->
<c-pophover class="left">
  <button>বাম</button>
  <div slot="pop">এটি একটি বুদবুদ টিপ</div>
</c-pophover>

<c-pophover class="right">
  <button>অধিকার</button>
  <div slot="pop">এটি একটি বুদবুদ টিপ</div>
</c-pophover>

<c-pophover class="bottom">
  <button>নিচে</button>
  <div slot="pop">এটি একটি বুদবুদ টিপ</div>
</c-pophover>

<c-pophover class="top">
  <button>উচ্চতর</button>
  <div slot="pop">এটি একটি বুদবুদ টিপ</div>
</c-pophover>
```

## CSS ভেরিয়েবল

কাস্টম শৈলী নিম্নলিখিত CSS ভেরিয়েবলের মাধ্যমে সমর্থিত:

| পরিবর্তনশীল নাম | ডিফল্ট মান | ব্যাখ্যা করা |
| :--- | :--- | :--- |
| `--pophover-gap` | `8px` | বুদবুদ টুলটিপ এবং ট্রিগারিং উপাদানের মধ্যে স্থান |
| `--pophover-margin` | `24px` | অভিযোজিত পজিশনিং গণনার সময় ভিউপোর্ট প্রান্ত নিরাপত্তা মার্জিন |
| `--pophover-bg-top` | `#18181ce0` | বুদ্বুদ গ্রেডিয়েন্ট ব্যাকগ্রাউন্ড টপ কালার |
| `--pophover-bg-mid` | `#121215e5` | বুদবুদের বাম এবং ডান দিকে তীরের পটভূমির রঙ |
| `--pophover-bg-bottom` | `#0c0c0feb` | বুদবুদ গ্রেডিয়েন্ট ব্যাকগ্রাউন্ডের নীচের এবং উপরের এবং নীচের দিকের তীরগুলির রঙ৷ |

## আইকন টুলটিপ শৈলী (`.Ico`)

বিশুদ্ধ আইকন টাইপ ফ্লোটিং প্রম্পট বোতামগুলির জন্য (যেমন "সম্পাদনা", "মুছুন" ইত্যাদি), আপনি বাইরের ধারক বা উপাদানে `.Ico` ক্লাস নাম যোগ করে একটি ইউনিফাইড আইকন এবং হোভার শৈলী প্রয়োগ করতে পারেন৷

### গঠন উদাহরণ
```html
<b class="Ico">
  <c-pophover>
    <a></a>
    <div slot="pop">সম্পাদনা</div>
  </c-pophover>
</b>
```

### শৈলী এক্সটেনশন
মৌলিক লেআউট, আকার (18x18px), এবং হোভার রঙ পরিবর্তনগুলি `.Ico > c-pophover > a` এ একীভূত করা হয়েছে৷ ডেভেলপারদের শুধুমাত্র সংশ্লিষ্ট আইকন ফাইলটি আলাদাভাবে উল্লেখ করতে হবে:
```stylus
c-edit > c-pophover > a
  mask-image url("./svg/edit.svg")
```
