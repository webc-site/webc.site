# רשום והצג את כל הרכיבים בספריית הרכיבים

- מוצג על ידי סגנון טהור, רכיבים בסיסיים ורכיבים מרוכבים
- קיבוץ תמיכה לפי כללים קבועים
- טען אוטומטית את רשימת הרכיבים והתיאור של השפה המתאימה בעת החלפת שפות
- החלף אוטומטית מחלקות מצב במהלך טעינה אסינכרונית

## השתמש בהדגמה

```html
<c-webc-ls></c-webc-ls>
```

## שיעור סגנון

### `.Ing`

נוסף אוטומטית בעת טעינת נתונים, משמש להסתרת תוכן רשימה ולהחלת סגנון טעינה.
