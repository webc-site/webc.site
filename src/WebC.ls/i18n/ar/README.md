# سرد وعرض كافة المكونات في مكتبة المكونات

- يتم عرضه من خلال النمط النقي والمكونات الأساسية والمكونات المركبة
- دعم التجميع وفقا للقواعد العادية
- قم بتحميل قائمة المكونات ووصف اللغة المقابلة تلقائيًا عند تبديل اللغات
- تبديل فئات الحالة تلقائيًا أثناء التحميل غير المتزامن

## استخدم العرض التوضيحي

```html
<c-webc-ls></c-webc-ls>
```

## فئة النمط

### `.Ing`

تتم إضافتها تلقائيًا عند تحميل البيانات، وتستخدم لإخفاء محتوى القائمة وتطبيق نمط التحميل.
