# فهرست و نمایش تمام اجزای موجود در کتابخانه کامپوننت

- با سبک خالص، اجزای اصلی و اجزای ترکیبی نمایش داده می شود
- از گروه بندی طبق قوانین منظم پشتیبانی کنید
- هنگام تعویض زبان، فهرست مؤلفه ها و توضیحات زبان مربوطه به طور خودکار بارگیری می شود
- تغییر خودکار کلاس های حالت در حین بارگذاری ناهمزمان

## از دمو استفاده کنید


## کلاس سبک

### `.Ing`

به طور خودکار هنگام بارگیری داده ها اضافه می شود، برای پنهان کردن محتوای لیست و اعمال سبک بارگیری استفاده می شود.
