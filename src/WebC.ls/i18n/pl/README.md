# Wyświetl listę i wyświetl wszystkie komponenty w bibliotece komponentów

- Prezentowany przez czysty styl, podstawowe komponenty i komponenty kompozytowe
- Wspieraj grupowanie według stałych zasad
- Automatycznie ładuj listę komponentów i opis odpowiedniego języka podczas zmiany języka
- Automatycznie przełączaj klasy stanu podczas ładowania asynchronicznego

## Skorzystaj z wersji demonstracyjnej

```html
<c-webc-ls></c-webc-ls>
```

## klasa stylu

### `.Ing`

Dodawane automatycznie podczas ładowania danych, używane do ukrywania zawartości listy i stosowania stylu ładowania.
