# Répertorier et afficher tous les composants de la bibliothèque de composants

- Affiché par style pur, composants de base et composants composites
- Regroupement de support selon les règles habituelles
- Charge automatiquement la liste des composants et la description de la langue correspondante lors du changement de langue
- Changer automatiquement de classe d'état pendant le chargement asynchrone

## Utilisez la démo

```html
<c-webc-ls></c-webc-ls>
```

## cours de style

### `.Ing`

Ajouté automatiquement lors du chargement des données, utilisé pour masquer le contenu de la liste et appliquer le style de chargement.
