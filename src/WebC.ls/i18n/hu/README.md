# Listázza és jelenítse meg az összetevőkönyvtár összes összetevőjét

- Tiszta stílus, alapkomponensek és kompozit alkatrészek jelennek meg
- Támogassa a csoportosítást a szokásos szabályok szerint
- Nyelvváltáskor automatikusan betölti a megfelelő nyelv összetevőlistáját és leírását
- Állapotosztályok automatikus váltása aszinkron betöltés során
