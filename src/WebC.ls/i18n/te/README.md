# కాంపోనెంట్ లైబ్రరీలోని అన్ని భాగాలను జాబితా చేయండి మరియు ప్రదర్శించండి

- స్వచ్ఛమైన శైలి, ప్రాథమిక భాగాలు మరియు మిశ్రమ భాగాల ద్వారా ప్రదర్శించబడుతుంది
- సాధారణ నియమాల ప్రకారం సమూహానికి మద్దతు ఇవ్వండి
- భాషలను మార్చేటప్పుడు సంబంధిత భాష యొక్క భాగాల జాబితా మరియు వివరణను స్వయంచాలకంగా లోడ్ చేయండి
- అసమకాలిక లోడింగ్ సమయంలో రాష్ట్ర తరగతులను స్వయంచాలకంగా మార్చండి
