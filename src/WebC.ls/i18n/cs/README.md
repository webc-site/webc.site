# Seznam a zobrazení všech komponent v knihovně komponent

- Zobrazuje se čistým stylem, základními komponenty a kompozitními komponenty
- Podporujte seskupování podle běžných pravidel
- Automaticky načíst seznam komponent a popis odpovídajícího jazyka při přepínání jazyků
- Automaticky přepínat stavové třídy během asynchronního načítání

## Použijte demo

```html
<c-webc-ls></c-webc-ls>
```

## stylová třída

### `.Ing`

Automaticky přidán při načítání dat, používá se ke skrytí obsahu seznamu a použití stylu načítání.
