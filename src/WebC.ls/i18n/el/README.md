# Λίστα και εμφάνιση όλων των στοιχείων στη βιβλιοθήκη στοιχείων

- Εμφανίζεται με καθαρό στυλ, βασικά εξαρτήματα και σύνθετα στοιχεία
- Υποστήριξη ομαδοποίησης σύμφωνα με τους κανονικούς κανόνες
- Αυτόματη φόρτωση της λίστας στοιχείων και της περιγραφής της αντίστοιχης γλώσσας κατά την εναλλαγή γλωσσών
- Αυτόματη εναλλαγή τάξεων κατάστασης κατά την ασύγχρονη φόρτωση

## Χρησιμοποιήστε το demo

```html
<c-webc-ls></c-webc-ls>
```

## τάξη στυλ

### `.Ing`

Προστίθεται αυτόματα κατά τη φόρτωση δεδομένων, χρησιμοποιείται για την απόκρυψη περιεχομένου λίστας και την εφαρμογή στυλ φόρτωσης.
