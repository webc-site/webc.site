# အစိတ်အပိုင်းဒစ်ဂျစ်တိုက်ရှိ အစိတ်အပိုင်းအားလုံးကို စာရင်းပြုစုပြီး ပြသပါ။

- သန့်ရှင်းသောစတိုင်၊ အခြေခံအစိတ်အပိုင်းများနှင့် ပေါင်းစပ်အစိတ်အပိုင်းများဖြင့် ပြသထားသည်။
- ပုံမှန်စည်းမျဉ်းများအတိုင်း အုပ်စုဖွဲ့ခြင်းကို ပံ့ပိုးကူညီပါ။
- ဘာသာစကားများကို ပြောင်းသည့်အခါ သက်ဆိုင်ရာဘာသာစကား၏ အစိတ်အပိုင်းစာရင်းနှင့် ဖော်ပြချက်ကို အလိုအလျောက်တင်ပါ။
- အပြိုင်အဆိုင်ဖွင့်နေစဉ်အတွင်း ပြည်နယ်အတန်းများကို အလိုအလျောက်ပြောင်းပါ။
