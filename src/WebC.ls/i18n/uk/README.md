# Список і відображення всіх компонентів у бібліотеці компонентів

- Відображається чистим стилем, основними компонентами та композитними компонентами
- Підтримуйте групування за звичайними правилами
- Автоматично завантажувати список компонентів і опис відповідної мови під час перемикання мов
- Автоматично перемикати класи стану під час асинхронного завантаження

## Використовуйте демо

```html
<c-webc-ls></c-webc-ls>
```

## клас стилю

### `.Ing`

Автоматично додається під час завантаження даних, використовується для приховання вмісту списку та застосування стилю завантаження.
