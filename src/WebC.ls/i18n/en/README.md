# List and display all components in the component library

- Displayed by pure style, basic components and composite components
- Support grouping according to regular rules
- Automatically load the component list and description of the corresponding language when switching languages
- Automatically switch state classes during asynchronous loading

## Use the demo

```html
<c-webc-ls></c-webc-ls>
```

## Style Class

### `.Ing`

Automatically added when loading data, used to hide list content and apply loading style.
