# கூறு நூலகத்தில் உள்ள அனைத்து கூறுகளையும் பட்டியலிட்டு காண்பிக்கவும்

- தூய நடை, அடிப்படை கூறுகள் மற்றும் கலப்பு கூறுகள் மூலம் காட்டப்படும்
- வழக்கமான விதிகளின்படி குழுவை ஆதரிக்கவும்
- மொழிகளை மாற்றும்போது கூறுகளின் பட்டியல் மற்றும் தொடர்புடைய மொழியின் விளக்கத்தை தானாக ஏற்றவும்
- ஒத்திசைவற்ற ஏற்றுதலின் போது மாநில வகுப்புகளை தானாக மாற்றவும்
