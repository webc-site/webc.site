# ઘટક લાઇબ્રેરીમાં તમામ ઘટકોની સૂચિ બનાવો અને પ્રદર્શિત કરો

- શુદ્ધ શૈલી, મૂળભૂત ઘટકો અને સંયુક્ત ઘટકો દ્વારા પ્રદર્શિત
- નિયમિત નિયમો અનુસાર જૂથને સમર્થન આપો
- ભાષાઓ સ્વિચ કરતી વખતે ઘટક સૂચિ અને અનુરૂપ ભાષાનું વર્ણન આપમેળે લોડ કરો
- અસુમેળ લોડિંગ દરમિયાન રાજ્ય વર્ગોને આપમેળે સ્વિચ કરો

## શૈલી વર્ગ

### `.Ing`

ડેટા લોડ કરતી વખતે આપમેળે ઉમેરવામાં આવે છે, જેનો ઉપયોગ સૂચિ સામગ્રીને છુપાવવા અને લોડિંગ શૈલી લાગુ કરવા માટે થાય છે.
