# コンポーネントライブラリ内のすべてのコンポーネントをリストして表示します。

- 純粋なスタイル、基本コンポーネント、および複合コンポーネントによって表示されます。
- 定期的なルールに従ってグループ化をサポートする
- 言語を切り替えるときに、対応する言語のコンポーネントリストと説明を自動的にロードします
- 非同期読み込み中に状態クラスを自動的に切り替える

## デモを使用する

```html
<c-webc-ls></c-webc-ls>
```

## スタイルクラス

### `.Ing`

データのロード時に自動的に追加され、リストのコンテンツを非表示にし、ロード スタイルを適用するために使用されます。
