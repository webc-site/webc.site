# Elenka u uri l-komponenti kollha fil-librerija tal-komponenti

- Muri minn stil pur, komponenti bażiċi u komponenti komposti
- Appoġġ grupp skond regoli regolari
- Awtomatikament tagħbija l-lista tal-komponenti u d-deskrizzjoni tal-lingwa korrispondenti meta taqleb il-lingwi
- Awtomatikament taqleb il-klassijiet tal-istat waqt it-tagħbija asinkronika

## Uża d-demo

```html
<c-webc-ls></c-webc-ls>
```

## klassi tal-istil

### `.Ing`

Miżjud awtomatikament waqt it-tagħbija tad-dejta, użat biex jaħbi l-kontenut tal-lista u japplika l-istil tat-tagħbija.
