# ಘಟಕ ಲೈಬ್ರರಿಯಲ್ಲಿ ಎಲ್ಲಾ ಘಟಕಗಳನ್ನು ಪಟ್ಟಿ ಮಾಡಿ ಮತ್ತು ಪ್ರದರ್ಶಿಸಿ

- ಶುದ್ಧ ಶೈಲಿ, ಮೂಲ ಘಟಕಗಳು ಮತ್ತು ಸಂಯೋಜಿತ ಘಟಕಗಳಿಂದ ಪ್ರದರ್ಶಿಸಲಾಗುತ್ತದೆ
- ನಿಯಮಿತ ನಿಯಮಗಳ ಪ್ರಕಾರ ಬೆಂಬಲ ಗುಂಪು
- ಭಾಷೆಗಳನ್ನು ಬದಲಾಯಿಸುವಾಗ ಅನುಗುಣವಾದ ಭಾಷೆಯ ಘಟಕ ಪಟ್ಟಿ ಮತ್ತು ವಿವರಣೆಯನ್ನು ಸ್ವಯಂಚಾಲಿತವಾಗಿ ಲೋಡ್ ಮಾಡಿ
- ಅಸಮಕಾಲಿಕ ಲೋಡ್ ಸಮಯದಲ್ಲಿ ಸ್ವಯಂಚಾಲಿತವಾಗಿ ರಾಜ್ಯದ ತರಗತಿಗಳನ್ನು ಬದಲಿಸಿ
