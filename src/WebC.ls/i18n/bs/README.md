# Navedite i prikažite sve komponente u biblioteci komponenti

- Prikazano čistim stilom, osnovnim komponentama i kompozitnim komponentama
- Podržite grupisanje prema redovnim pravilima
- Automatski učitajte listu komponenti i opis odgovarajućeg jezika prilikom promjene jezika
- Automatski prebacite klase stanja tokom asinhronog učitavanja

## Koristite demo

```html
<c-webc-ls></c-webc-ls>
```

## stilska klasa

### `.Ing`

Automatski se dodaje prilikom učitavanja podataka, koristi se za skrivanje sadržaja liste i primjenu stila učitavanja.
