# Ilista at ipakita ang lahat ng bahagi sa component library

- Ipinapakita ng purong istilo, mga pangunahing bahagi at pinagsama-samang bahagi
- Suportahan ang pagpapangkat ayon sa mga regular na tuntunin
- Awtomatikong i-load ang listahan ng bahagi at paglalarawan ng kaukulang wika kapag nagpapalit ng mga wika
- Awtomatikong lumipat ng mga klase ng estado sa panahon ng asynchronous na paglo-load

## Gamitin ang demo

```html
<c-webc-ls></c-webc-ls>
```

## klase ng istilo

### `.Ing`

Awtomatikong idinagdag kapag naglo-load ng data, ginagamit upang itago ang nilalaman ng listahan at ilapat ang istilo ng paglo-load.
