# Ցուցադրել և ցուցադրել բաղադրիչների գրադարանի բոլոր բաղադրիչները

- Ցուցադրվում է մաքուր ոճով, հիմնական բաղադրիչներով և կոմպոզիտային բաղադրիչներով
- Աջակցեք խմբավորմանը ըստ կանոնավոր կանոնների
- Լեզուն փոխարկելիս ավտոմատ կերպով բեռնել բաղադրիչների ցանկը և համապատասխան լեզվի նկարագրությունը
- Ավտոմատ կերպով փոխեք վիճակի դասերը ասինխրոն բեռնման ժամանակ
