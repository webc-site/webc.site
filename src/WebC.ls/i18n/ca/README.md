# Llista i mostra tots els components de la biblioteca de components

- Es mostra per estil pur, components bàsics i components compostos
- Donar suport a l'agrupació segons normes habituals
- Carregueu automàticament la llista de components i la descripció de l'idioma corresponent quan canvieu d'idioma
- Canvia automàticament les classes d'estat durant la càrrega asíncrona

## Utilitzeu la demostració

```html
<c-webc-ls></c-webc-ls>
```

## classe d'estil

### `.Ing`

S'afegeix automàticament en carregar dades, s'utilitza per ocultar el contingut de la llista i aplicar l'estil de càrrega.
