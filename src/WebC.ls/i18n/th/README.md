# แสดงรายการและแสดงส่วนประกอบทั้งหมดในไลบรารีส่วนประกอบ

- จัดแสดงด้วยสไตล์ล้วนๆ ส่วนประกอบพื้นฐาน และส่วนประกอบประกอบ
- รองรับการจัดกลุ่มตามกฎปกติ
- โหลดรายการส่วนประกอบและคำอธิบายของภาษาที่เกี่ยวข้องโดยอัตโนมัติเมื่อเปลี่ยนภาษา
- สลับคลาสสถานะโดยอัตโนมัติระหว่างการโหลดแบบอะซิงโครนัส

## ใช้การสาธิต

```html
<c-webc-ls></c-webc-ls>
```

## คลาสสไตล์

### `.Ing`

เพิ่มโดยอัตโนมัติเมื่อโหลดข้อมูล ใช้เพื่อซ่อนเนื้อหารายการและใช้รูปแบบการโหลด
