# Компоненттер кітапханасындағы барлық құрамдастардың тізімін көрсетіңіз және көрсетіңіз

- Таза стиль, негізгі құрамдас бөліктер және құрама компоненттер арқылы көрсетіледі
- Тұрақты ережелер бойынша топтастыруды қолдау
- Тілдерді ауыстырған кезде тиісті тілдің құрамдас тізімі мен сипаттамасын автоматты түрде жүктеңіз
- Асинхронды жүктеу кезінде күй кластарын автоматты түрде ауыстырыңыз
