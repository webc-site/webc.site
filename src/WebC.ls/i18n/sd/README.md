# اجزاء لائبريري ۾ سڀني اجزاء جي فهرست ۽ ڊسپلي ڪريو

- ظاهر ڪيل خالص انداز، بنيادي اجزاء ۽ جامع اجزاء
- باقاعده ضابطن جي مطابق گروپ جي حمايت ڪريو
- پاڻمرادو لوڊ ڪريو جزو جي فهرست ۽ لاڳاپيل ٻولي جي وضاحت جڏهن ٻولين کي تبديل ڪندي
- خودڪار طور تي تبديل ڪريو رياستي طبقن کي غير مطابقت واري لوڊ ڪرڻ دوران
