# Seznam in prikaz vseh komponent v knjižnici komponent

- Prikazan s čistim slogom, osnovnimi komponentami in sestavljenimi komponentami
- Podprite združevanje v skupine po rednih pravilih
- Samodejno naloži seznam komponent in opis ustreznega jezika pri preklopu jezikov
- Samodejno preklapljajte med razredi stanja med asinhronim nalaganjem

## Uporabite demo

```html
<c-webc-ls></c-webc-ls>
```

### `.Ing`
