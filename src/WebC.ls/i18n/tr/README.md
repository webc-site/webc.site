# Bileşen kitaplığındaki tüm bileşenleri listeleyin ve görüntüleyin

- Saf stil, temel bileşenler ve kompozit bileşenlerle görüntülenir
- Düzenli kurallara göre gruplamayı destekleyin
- Dilleri değiştirirken ilgili dilin bileşen listesini ve açıklamasını otomatik olarak yükle
- Eşzamansız yükleme sırasında durum sınıflarını otomatik olarak değiştir

## Demoyu kullanın

```html
<c-webc-ls></c-webc-ls>
```

## stil sınıfı

### `.Ing`

Veriler yüklenirken otomatik olarak eklenir, liste içeriğini gizlemek ve yükleme stilini uygulamak için kullanılır.
