# Рендеринг и отображение демонстраций компонентов, документации и инструкций по установке.

- Извлеките и проанализируйте заголовок, введение и содержимое документа README.
- Песочница для запуска интерактивных демонстраций компонентов и изоляции стилей.
- Преобразуйте блоки кода в документах в интерактивные презентации.
- Адаптивный макет, широкоэкранное отображение столбцов, узкоэкранное переключение вкладок

## Используйте демо-версию

```html
<c-webc-demo></c-webc-demo>
```

```js
import WebCDemo from "./_.js";

const el = document.querySelector("c-webc-demo");

//Инициализируем и получаем интерфейс операции
const [setMod, setMd, unMount] = WebCDemo(el);
```

## Параметры интерфейса

### `WebCDemo(el)`

Инициализируйте компонент.

- **параметр**
  - `el`:`HTMLElement`, контейнер компонента.
- **Возвращаемое значение**:`[setMod, setMd, unMount]`
  - `setMod(mod)`: Настройте демонстрационный модуль.
  - `setMd(name, md, lang)`: укажите имя компонента, содержимое документа README и текущий язык.
  - `unMount()`: выгрузите компонент и очистите прослушиватель событий.

### Атрибут `mod` демонстрационного модуля

- `CSS`:`string`, путь к таблице стилей компонента.
- `HTM`:`string`, HTML-шаблон компонента.
- `default`:`function(root)`, функция инициализации компонента, получает теневой корневой контейнер.

## класс стиля

### `.M`

область контента.

### `.L`

Левая панель содержит инструкции и документацию.

### `.R`

Правая панель, содержащая демо.

### `c-nav`

Узкая экранная панель навигации.
