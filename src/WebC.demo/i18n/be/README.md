# Рэндэрынг і адлюстраванне дэманстрацый кампанентаў, дакументацыі і інструкцый па ўсталёўцы

- Выняць і разабраць загаловак, увядзенне і змест дакумента README
- Пясочніца для запуску інтэрактыўных дэманстрацый кампанентаў і ізаляцыі стыляў
- Пераўтварыце блокі кода ў дакументах у інтэрактыўныя прэзентацыі
- Адаптыўны макет, шырокаэкранны дысплей у калонках, вузкі экран з пераключэннем укладак

## Выкарыстоўвайце дэма

```html
<c-webc-demo></c-webc-demo>
```

```js
import WebCDemo from "./_.js";

const el = document.querySelector("c-webc-demo");

//Ініцыялізаваць і атрымаць аперацыйны інтэрфейс
const [setMod, setMd, unMount] = WebCDemo(el);
```

## Параметры інтэрфейсу

### `WebCDemo(el)`

Ініцыялізаваць кампанент.

- **параметр**
  - `el`:`HTMLElement`, кантэйнер кампанентаў.
- **Вяртанае значэнне**:`[setMod, setMd, unMount]`
  - `setMod(mod)`: Наладзьце дэма-модуль.
  - `setMd(name, md, lang)`: Усталюйце назву кампанента, змест дакумента README і бягучую мову.
  - `unMount()`: Выгрузіце кампанент і ачысціце слухач падзей.

### Атрыбут `mod` дэманстрацыйнага модуля

- `CSS`:`string`, шлях да табліцы стыляў кампанента.
- `HTM`:`string`, шаблон HTML кампанента.
- `default`:`function(root)`, функцыя ініцыялізацыі кампанента, атрымлівае ценявы каранёвы кантэйнер.

## клас стылю

### `.M`

вобласць кантэнту.

### `.L`

Левая панэль змяшчае інструкцыі і дакументацыю.

### `.R`

Правая панэль, якая змяшчае дэманстрацыю.

### `c-nav`

Вузкая панэль навігацыі экрана.
