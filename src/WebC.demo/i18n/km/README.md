# បង្ហាញ និងបង្ហាញការបង្ហាញសមាសធាតុ ឯកសារ និងការណែនាំអំពីការដំឡើង

- ស្រង់ចេញ និងញែកចំណងជើង សេចក្តីផ្តើម និងខ្លឹមសារនៃឯកសារ README
- Sandbox ដើម្បីដំណើរការការបង្ហាញអន្តរកម្មនៃសមាសភាគ និងរចនាប័ទ្មដាច់ដោយឡែក
- បំលែងប្លុកកូដក្នុងឯកសារទៅជាបទបង្ហាញអន្តរកម្ម
- ប្លង់អាដាប់ធ័រ ការបង្ហាញជួរឈរអេក្រង់ធំទូលាយ ការបង្ហាញការប្តូរផ្ទាំងអេក្រង់តូចចង្អៀត

## ប្រើការបង្ហាញ

```html
<c-webc-demo></c-webc-demo>
```

```js
import WebCDemo from "./_.js";

const el = document.querySelector("c-webc-demo");

// ចាប់ផ្តើម និងទទួលបានចំណុចប្រទាក់ប្រតិបត្តិការ
const [setMod, setMd, unMount] = WebCDemo(el);
```

## ប៉ារ៉ាម៉ែត្រចំណុចប្រទាក់

### `WebCDemo(el)`

ចាប់ផ្តើមធាតុផ្សំ។

- ** ប៉ារ៉ាម៉ែត្រ **
  - `el`:`HTMLElement` ធុងផ្ទុកសមាសធាតុ។
- **តម្លៃត្រឡប់**:`[setMod, setMd, unMount]`
  - `setMod(mod)`៖ ដំឡើងម៉ូឌុលសាកល្បង។
  - `setMd(name, md, lang)`៖ កំណត់ឈ្មោះសមាសភាគ មាតិកាឯកសារ README និងភាសាបច្ចុប្បន្ន។
  - `unMount()`៖ ដោះចេញសមាសភាគ ហើយសម្អាតកម្មវិធីស្តាប់ព្រឹត្តិការណ៍។

### ម៉ូឌុលសាកល្បង `mod`គុណលក្ខណៈ

- `CSS`:`string` ផ្លូវសន្លឹករចនាប័ទ្មនៃសមាសភាគ។
- `HTM`:`string` គំរូ HTML របស់សមាសភាគ។
- `default`:`function(root)`មុខងារ​ចាប់ផ្ដើម​សមាសភាគ ទទួល​ធុង​ឫស​ស្រមោល។

## ថ្នាក់រចនាប័ទ្ម

### `.M`

តំបន់មាតិកា។

### `.L`

បន្ទះខាងឆ្វេងមានការណែនាំ និងឯកសារ។

### `.R`

បន្ទះខាងស្តាំ មានការបង្ហាញ។

### `c-nav`

របាររុករកអេក្រង់តូចចង្អៀត។
