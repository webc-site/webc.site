# Αποδώστε και εμφανίστε επιδείξεις στοιχείων, τεκμηρίωση και οδηγίες εγκατάστασης

- Εξαγωγή και ανάλυση του τίτλου, της εισαγωγής και του περιεχομένου του εγγράφου README
- Sandbox για εκτέλεση διαδραστικών επιδείξεων στοιχείων και απομόνωση στυλ
- Μετατροπή μπλοκ κώδικα σε έγγραφα σε διαδραστικές παρουσιάσεις
- Προσαρμοστική διάταξη, προβολή στήλης ευρείας οθόνης, εμφάνιση εναλλαγής καρτελών στενής οθόνης

## Χρησιμοποιήστε το demo

```html
<c-webc-demo></c-webc-demo>
```

```js
import WebCDemo from "./_.js";

const el = document.querySelector("c-webc-demo");

//Εκκίνηση και λήψη της διεπαφής λειτουργίας
const [setMod, setMd, unMount] = WebCDemo(el);
```

## Παράμετροι διεπαφής

### `WebCDemo(el)`

Αρχικοποιήστε το στοιχείο.

- **παράμετρος**
  - `el`:`HTMLElement`, κοντέινερ συστατικού.
- **Επιστρεφόμενη τιμή**:`[setMod, setMd, unMount]`
  - `setMod(mod)`: Ρυθμίστε τη μονάδα επίδειξης.
  - `setMd(name, md, lang)`: Ορίστε το όνομα του στοιχείου, το περιεχόμενο του εγγράφου README και την τρέχουσα γλώσσα.
  - `unMount()`: Ξεφορτώστε το στοιχείο και καθαρίστε το πρόγραμμα ακρόασης συμβάντος.

### Μονάδα επίδειξης χαρακτηριστικό `mod`

- `CSS`:`string`, η διαδρομή του φύλλου στυλ του στοιχείου.
- `HTM`:`string`, το πρότυπο HTML του στοιχείου.
- `default`:`function(root)`, συνάρτηση αρχικοποίησης στοιχείου, λαμβάνει σκιώδη ριζικό κοντέινερ.

## τάξη στυλ

### `.M`

περιοχή περιεχομένου.

### `.L`

Το αριστερό πλαίσιο περιέχει οδηγίες και τεκμηρίωση.

### `.R`

Δεξί πίνακα, που περιέχει το demo.

### `c-nav`

Στενή γραμμή πλοήγησης οθόνης.
