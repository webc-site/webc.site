# উপাদান ডেমো, আলেখ্যন, আৰু সংস্থাপন নিৰ্দেশসমূহ ৰেণ্ডাৰ আৰু প্ৰদৰ্শন কৰক

- README দস্তাবেজৰ শিৰোনাম, পৰিচয় আৰু বিষয়বস্তু এক্সট্ৰেক্ট আৰু বিশ্লেষণ কৰক
- উপাদানসমূহৰ পাৰস্পৰিক ডেমো চলাবলৈ আৰু শৈলীসমূহ পৃথক কৰিবলৈ ছেণ্ডবক্স
- দস্তাবেজসমূহত ক'ড ব্লকসমূহক পাৰস্পৰিক উপস্থাপনলৈ ৰূপান্তৰ কৰক
- অভিযোজিত পৰিকল্পনা, বহল পৰ্দা স্তম্ভ প্ৰদৰ্শন, সংকীৰ্ণ পৰ্দা টেব চুইচিং প্ৰদৰ্শন

## ডেমো ব্যৱহাৰ কৰক

```html
<c-webc-demo></c-webc-demo>
```

```js
import WebCDemo from "./_.js";

const el = document.querySelector("c-webc-demo");

//অপাৰেচন আন্তঃপৃষ্ঠ আৰম্ভ কৰক আৰু লাভ কৰক
const [setMod, setMd, unMount] = WebCDemo(el);
```

## আন্তঃপৃষ্ঠ প্ৰাচলসমূহ

### `WebCDemo(el)`

উপাদানটো আৰম্ভ কৰক।

- **প্যাৰামিটাৰ**
  - `el`:`HTMLElement`, উপাদান ধাৰক।
- **মান ঘূৰাই দিয়ক**:`[setMod, setMd, unMount]`
  - `setMod(mod)`: ডেমো মডিউল সংস্থাপন কৰক।
  - `setMd(name, md, lang)`: উপাদানৰ নাম, README দস্তাবেজৰ বিষয়বস্তু আৰু বৰ্তমান ভাষা নিৰ্ধাৰণ কৰক।
  - `unMount()`: উপাদান আনলোড কৰক আৰু ইভেন্ট লিষ্টাৰ পৰিষ্কাৰ কৰক।

### ডেমো মডিউল `mod` বৈশিষ্ট্য

- `CSS`:`string`, উপাদানৰ শৈলী পত্ৰিকা পথ।
- `HTM`:`string`, উপাদানৰ HTML সাঁচ।
- `default`:`function(root)`, উপাদান আৰম্ভণি ফলন, ছাঁ ৰূট ধাৰক গ্ৰহণ কৰে।

## শৈলী শ্ৰেণী

### `.M`

বিষয়বস্তুৰ এলেকা।

### `.L`

বাওঁফালৰ পেনেলত নিৰ্দেশনা আৰু নথিপত্ৰ থাকে।

### `.R`

সোঁফালৰ পেনেল, ডেমো থকা।

### `c-nav`

সংকীৰ্ণ পৰ্দা নেভিগেচন বাৰ।
