# Рендерујте и прикажите демонстрације компоненти, документацију и упутства за инсталацију

- Издвојите и рашчланите наслов, увод и садржај РЕАДМЕ документа
- Сандбок за покретање интерактивних демонстрација компоненти и изоловање стилова
- Претворите блокове кода у документима у интерактивне презентације
- Прилагодљиви распоред, приказ колона на широком екрану, екран за пребацивање картица у уском екрану

## Користите демо

```html
<c-webc-demo></c-webc-demo>
```

```js
import WebCDemo from "./_.js";

const el = document.querySelector("c-webc-demo");

//Иницијализација и добијање оперативног интерфејса
const [setMod, setMd, unMount] = WebCDemo(el);
```

Иницијализујте компоненту.

- **параметар**
  - <б ид="1">:<б ид="2">, контејнер компоненте.
- **Повратна вредност**:<б ид="1">
  - <б ид="1">: Подесите демо модул.

## стилска класа

### `.M`

област садржаја.

### `.L`

Лева табла садржи упутства и документацију.

### `.R`

Десни панел, који садржи демо.

### `c-nav`

Уска трака за навигацију на екрану.
