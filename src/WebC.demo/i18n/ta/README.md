# கூறு டெமோக்கள், ஆவணங்கள் மற்றும் நிறுவல் வழிமுறைகளை வழங்குதல் மற்றும் காட்சிப்படுத்துதல்

- README ஆவணத்தின் தலைப்பு, அறிமுகம் மற்றும் உள்ளடக்கத்தை பிரித்தெடுத்து அலசவும்
- சாண்ட்பாக்ஸ் கூறுகளின் ஊடாடும் டெமோக்கள் மற்றும் தனிமைப்படுத்தப்பட்ட பாணிகளை இயக்கும்
- ஆவணங்களில் உள்ள குறியீடு தொகுதிகளை ஊடாடும் விளக்கக்காட்சிகளாக மாற்றவும்
- தகவமைப்பு தளவமைப்பு, பரந்த திரை நெடுவரிசை காட்சி, குறுகிய திரை தாவல் மாறுதல் காட்சி

## டெமோவைப் பயன்படுத்தவும்

```html
<c-webc-demo></c-webc-demo>
```

```js
import WebCDemo from "./_.js";

const el = document.querySelector("c-webc-demo");

//ஆபரேஷன் இடைமுகத்தை துவக்கி பெறவும்
const [setMod, setMd, unMount] = WebCDemo(el);
```

## இடைமுக அளவுருக்கள்

### `WebCDemo(el)`

கூறுகளைத் தொடங்கவும்.

- **அளவுரு**
  - `el`:`HTMLElement`, கூறு கொள்கலன்.
- **மதிப்பு திரும்ப**:`[setMod, setMd, unMount]`
  - `setMod(mod)`: டெமோ தொகுதியை அமைக்கவும்.
  - `setMd(name, md, lang)`: கூறுகளின் பெயர், README ஆவண உள்ளடக்கம் மற்றும் தற்போதைய மொழியை அமைக்கவும்.
  - `unMount()`: கூறுகளை இறக்கி, நிகழ்வு கேட்பவரை சுத்தம் செய்யவும்.

### டெமோ தொகுதி `mod` பண்புக்கூறு

- `CSS`:`string`, கூறுகளின் நடை தாள் பாதை.
- `HTM`:`string`, கூறுகளின் HTML டெம்ப்ளேட்.
- `default`:`function(root)`, கூறு துவக்க செயல்பாடு, நிழல் ரூட் கொள்கலனைப் பெறுகிறது.

## பாணி வகுப்பு

### `.M`

உள்ளடக்க பகுதி.

### `.L`

இடது பேனலில் வழிமுறைகள் மற்றும் ஆவணங்கள் உள்ளன.

### `.R`

வலது பேனல், டெமோவைக் கொண்டுள்ளது.

### `c-nav`

குறுகிய திரை வழிசெலுத்தல் பட்டி.
