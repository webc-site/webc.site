# Рендерирајте и прикажете демо, документација и инструкции за инсталација на компонентите

- Извадете го и анализирајте го насловот, воведот и содржината на документот README
- Sandbox за извршување на интерактивни демо на компоненти и изолирање на стилови
- Претворете ги кодните блокови во документите во интерактивни презентации
- Приспособлив распоред, приказ на колона со широк екран, приказ на префрлување на јазичињата со тесен екран

## Користете го демо

```html
<c-webc-demo></c-webc-demo>
```

```js
import WebCDemo from "./_.js";

const el = document.querySelector("c-webc-demo");

//Иницијализирајте и добијте го интерфејсот за работа
const [setMod, setMd, unMount] = WebCDemo(el);
```

## Параметри на интерфејсот

### `WebCDemo(el)`

Иницијализирајте ја компонентата.

- **параметар**
  - `el`:`HTMLElement`, контејнер за компоненти.
- **Повратна вредност**:`[setMod, setMd, unMount]`
  - `setMod(mod)`: поставете го демо-модулот.
  - `setMd(name, md, lang)`: Поставете го името на компонентата, содржината на документот README и тековниот јазик.
  - `unMount()`: исклучете ја компонентата и исчистете го слушателот на настани.

### Демо-модул `mod` атрибут

- `CSS`:`string`, патеката на листот со стилови на компонентата.
- `HTM`:`string`, HTML-шаблон на компонентата.
- `default`:`function(root)`, функција за иницијализација на компонентата, прима контејнер со корен во сенка.

## класа стил

### `.M`

област на содржина.

### `.L`

Левиот панел содржи инструкции и документација.

### `.R`

Десен панел, кој ја содржи демонстрацијата.

### `c-nav`

Лента за навигација на тесен екран.
