# عرض وعرض العروض التوضيحية للمكونات والوثائق وتعليمات التثبيت

- قم باستخراج وتحليل عنوان ومقدمة ومحتوى مستند README
- Sandbox لتشغيل العروض التوضيحية التفاعلية للمكونات وعزل الأنماط
- تحويل كتل التعليمات البرمجية في المستندات إلى عروض تقديمية تفاعلية
- تخطيط متكيف، عرض عمودي على شاشة عريضة، عرض تبديل علامة تبويب الشاشة الضيقة

## استخدم العرض التوضيحي

```html
<c-webc-demo></c-webc-demo>
```

```js
import WebCDemo from "./_.js";

const el = document.querySelector("c-webc-demo");

// تهيئة واجهة التشغيل والحصول عليها
const [setMod, setMd, unMount] = WebCDemo(el);
```

## معلمات الواجهة

### `WebCDemo(el)`

تهيئة المكون.

- **المعلمة**
  - `el`:`HTMLElement`، حاوية المكون.
- **قيمة الإرجاع**:`[setMod, setMd, unMount]`
  - `setMod(mod)`: قم بإعداد الوحدة التجريبية.
  - `setMd(name, md, lang)`: قم بتعيين اسم المكون ومحتوى مستند README واللغة الحالية.
  - `unMount()`: قم بإلغاء تحميل المكون وتنظيف مستمع الحدث.

### سمة الوحدة التجريبية `mod`

- `CSS`:`string`، مسار ورقة الأنماط للمكون.
- `HTM`:`string`، قالب HTML الخاص بالمكون.
- `default`:`function(root)`، وظيفة تهيئة المكون، تتلقى حاوية جذر الظل.

## فئة النمط

### `.M`

منطقة المحتوى.

### `.L`

تحتوي اللوحة اليسرى على تعليمات ووثائق.

### `.R`

اللوحة اليمنى، التي تحتوي على العرض التوضيحي.

### `c-nav`

شريط التنقل الضيق على الشاشة.
