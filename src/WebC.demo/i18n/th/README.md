# เรนเดอร์และแสดงการสาธิตส่วนประกอบ เอกสารประกอบ และคำแนะนำในการติดตั้ง

- แยกและแยกวิเคราะห์ชื่อเรื่อง บทนำ และเนื้อหาของเอกสาร README
- แซนด์บ็อกซ์เพื่อเรียกใช้การสาธิตส่วนประกอบแบบโต้ตอบและแยกสไตล์
- แปลงบล็อคโค้ดในเอกสารให้เป็นการนำเสนอเชิงโต้ตอบ
- รูปแบบที่ปรับเปลี่ยนได้ การแสดงคอลัมน์หน้าจอกว้าง การสลับแท็บหน้าจอแคบ

## ใช้การสาธิต

```html
<c-webc-demo></c-webc-demo>
```

```js
import WebCDemo from "./_.js";

const el = document.querySelector("c-webc-demo");

// เริ่มต้นและรับอินเทอร์เฟซการดำเนินการ
const [setMod, setMd, unMount] = WebCDemo(el);
```

## พารามิเตอร์อินเทอร์เฟซ

### `WebCDemo(el)`

เริ่มต้นส่วนประกอบ

- **พารามิเตอร์**
  - `el`:`HTMLElement` คอนเทนเนอร์ส่วนประกอบ
- **ค่าส่งคืน**:`[setMod, setMd, unMount]`
  - `setMod(mod)`: ตั้งค่าโมดูลสาธิต
  - `setMd(name, md, lang)`: ตั้งชื่อส่วนประกอบ เนื้อหาเอกสาร README และภาษาปัจจุบัน
  - `unMount()`: ยกเลิกการโหลดส่วนประกอบและล้างข้อมูลตัวฟังเหตุการณ์

### โมดูลสาธิต `mod` แอตทริบิวต์

- `CSS`:`string` เส้นทางสไตล์ชีตของส่วนประกอบ
- `HTM`:`string` เทมเพลต HTML ของส่วนประกอบ
- `default`:`function(root)` ฟังก์ชันการเริ่มต้นส่วนประกอบ รับคอนเทนเนอร์รูทเงา

## คลาสสไตล์

### `.M`

พื้นที่เนื้อหา

### `.L`

แผงด้านซ้ายประกอบด้วยคำแนะนำและเอกสารประกอบ

### `.R`

แผงด้านขวาที่มีการสาธิต

### `c-nav`

แถบนำทางหน้าจอแคบ
