# घटक डेमो, दस्तावेज़ीकरण और स्थापना निर्देश प्रस्तुत और प्रदर्शित करें

- README दस्तावेज़ का शीर्षक, परिचय और सामग्री निकालें और पार्स करें
- घटकों के इंटरैक्टिव डेमो चलाने और शैलियों को अलग करने के लिए सैंडबॉक्स
- दस्तावेज़ों में कोड ब्लॉक को इंटरैक्टिव प्रस्तुतियों में बदलें
- अनुकूली लेआउट, चौड़ी स्क्रीन कॉलम डिस्प्ले, संकीर्ण स्क्रीन टैब स्विचिंग डिस्प्ले

## डेमो का प्रयोग करें

```html
<c-webc-demo></c-webc-demo>
```

```js
import WebCDemo from "./_.js";

const el = document.querySelector("c-webc-demo");

// ऑपरेशन इंटरफ़ेस प्रारंभ करें और प्राप्त करें
const [setMod, setMd, unMount] = WebCDemo(el);
```

## इंटरफ़ेस पैरामीटर

### `WebCDemo(el)`

घटक को प्रारंभ करें.

- **पैरामीटर**
  - `el`:`HTMLElement`, घटक कंटेनर।
- **वापसी मूल्य**:`[setMod, setMd, unMount]`
  - `setMod(mod)`: डेमो मॉड्यूल सेट करें।
  - `setMd(name, md, lang)`: घटक का नाम, README दस्तावेज़ सामग्री और वर्तमान भाषा सेट करें।
  - `unMount()`: घटक को अनलोड करें और ईवेंट श्रोता को साफ़ करें।

### डेमो मॉड्यूल `mod` विशेषता

- `CSS`:`string`, घटक का स्टाइल शीट पथ।
- `HTM`:`string`, घटक का HTML टेम्पलेट।
- `default`:`function(root)`, घटक आरंभीकरण फ़ंक्शन, छाया रूट कंटेनर प्राप्त करता है।

## शैली वर्ग

### `.M`

इच्छुक क्षेत्र।

### `.L`

बाएँ पैनल में निर्देश और दस्तावेज़ शामिल हैं।

### `.R`

दायां पैनल, जिसमें डेमो है।

### `c-nav`

संकीर्ण स्क्रीन नेविगेशन बार.
