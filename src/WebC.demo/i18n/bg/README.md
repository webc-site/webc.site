# Изобразете и покажете демонстрации на компоненти, документация и инструкции за инсталиране

- Извлечете и анализирайте заглавието, въведението и съдържанието на документа README
- Sandbox за стартиране на интерактивни демонстрации на компоненти и изолиране на стилове
- Преобразувайте кодови блокове в документи в интерактивни презентации
- Адаптивно оформление, широкоекранен дисплей с колони, тесен екран с превключване на раздели

## Използвайте демонстрацията

```html
<c-webc-demo></c-webc-demo>
```

```js
import WebCDemo from "./_.js";

const el = document.querySelector("c-webc-demo");

//Инициализиране и получаване на операционния интерфейс
const [setMod, setMd, unMount] = WebCDemo(el);
```

## Параметри на интерфейса

### `WebCDemo(el)`

Инициализирайте компонента.

- **параметър**
  - `el`:`HTMLElement`, контейнер за компоненти.
- **Върната стойност**:`[setMod, setMd, unMount]`
  - `setMod(mod)`: Настройте демонстрационния модул.
  - `setMd(name, md, lang)`: Задайте името на компонента, съдържанието на документа README и текущия език.
  - `unMount()`: Разтоварете компонента и почистете слушателя на събития.

### Атрибут `mod` на демо модул

- `CSS`:`string`, пътят на листа със стилове на компонента.
- `HTM`:`string`, HTML шаблонът на компонента.
- `default`:`function(root)`, функция за инициализация на компонент, получава скрит коренов контейнер.

## стил клас
