# සංරචක ආදර්ශන, ලේඛන සහ ස්ථාපන උපදෙස් ලබා දීම සහ ප්‍රදර්ශනය කිරීම

- README ලේඛනයේ මාතෘකාව, හැඳින්වීම සහ අන්තර්ගතය උපුටා ගෙන විග්‍රහ කරන්න
- සංරචක සහ හුදකලා විලාසවල අන්තර්ක්‍රියාකාරී ආදර්ශන ධාවනය කිරීමට සෑන්ඩ්බොක්ස්
- ලේඛනවල ඇති කේත කුට්ටි අන්තර්ක්‍රියාකාරී ඉදිරිපත් කිරීම් බවට පරිවර්තනය කරන්න
- අනුවර්තන පිරිසැලසුම, පුළුල් තිර තීරු සංදර්ශකය, පටු තිර ටැබ් මාරු කිරීමේ සංදර්ශකය

## Demo භාවිතා කරන්න

```html
<c-webc-demo></c-webc-demo>
```

```js
import WebCDemo from "./_.js";

const el = document.querySelector("c-webc-demo");

//මෙහෙයුම් අතුරුමුහුණත ආරම්භ කර ලබා ගන්න
const [setMod, setMd, unMount] = WebCDemo(el);
```

## අතුරුමුහුණත් පරාමිතීන්

### `WebCDemo(el)`

සංරචකය ආරම්භ කරන්න.

- **පරාමිතිය**
  - `el`:`HTMLElement`, සංරචක බහාලුම.
- **ආපසු අගය**:`[setMod, setMd, unMount]`
  - `setMod(mod)`: ආදර්ශන මොඩියුලය සකසන්න.
  - `setMd(name, md, lang)`: සංරචක නාමය, README ලේඛන අන්තර්ගතය සහ වත්මන් භාෂාව සකසන්න.
  - `unMount()`: සංරචකය බාගෙන සිදුවීම් සවන්දෙන්නා පිරිසිදු කරන්න.

### Demo module `mod` attribute

- `CSS`:`string`, සංරචකයේ විලාස පත්‍ර මාර්ගය.
- `HTM`:`string`, සංරචකයේ HTML අච්චුව.
- `default`:`function(root)`, සංරචක ආරම්භක ශ්‍රිතය, සෙවනැලි මූල කන්ටේනරය ලබා ගනී.

## ශෛලිය පන්තිය

### `.M`

අන්තර්ගත ප්රදේශය.

### `.L`

වම් පුවරුවේ උපදෙස් සහ ලියකියවිලි අඩංගු වේ.

### `.R`

ආදර්ශනය අඩංගු දකුණු පුවරුව.

### `c-nav`

පටු තිර සංචාලන තීරුව.
