# رینڈر اور ڈسپلے جزو ڈیمو، دستاویزات، اور تنصیب کی ہدایات

- README دستاویز کے عنوان، تعارف اور مواد کو نکالیں اور پارس کریں۔
- اجزاء کے انٹرایکٹو ڈیمو اور الگ تھلگ اسٹائل چلانے کے لیے سینڈ باکس
- دستاویزات میں کوڈ بلاکس کو انٹرایکٹو پریزنٹیشنز میں تبدیل کریں۔
- انکولی لے آؤٹ، وسیع اسکرین کالم ڈسپلے، تنگ اسکرین ٹیب سوئچنگ ڈسپلے

## ڈیمو استعمال کریں۔

```html
<c-webc-demo></c-webc-demo>
```

```js
import WebCDemo from "./_.js";

const el = document.querySelector("c-webc-demo");

// آپریشن انٹرفیس کو شروع کریں اور حاصل کریں۔
const [setMod, setMd, unMount] = WebCDemo(el);
```

## انٹرفیس کے پیرامیٹرز

### `WebCDemo(el)`

جزو کو شروع کریں۔

- **پیرامیٹر**
  - `el`:`HTMLElement`، جزو کنٹینر۔
- **واپسی قدر**:`[setMod, setMd, unMount]`
  - `setMod(mod)`: ڈیمو ماڈیول سیٹ اپ کریں۔
  - `setMd(name, md, lang)`: اجزاء کا نام، README دستاویز کا مواد اور موجودہ زبان سیٹ کریں۔
  - `unMount()`: جزو کو اتاریں اور ایونٹ سننے والے کو صاف کریں۔

### ڈیمو ماڈیول `mod` وصف

- `CSS`:`string`، جزو کا اسٹائل شیٹ پاتھ۔
- `HTM`:`string`، جزو کا HTML ٹیمپلیٹ۔
- `default`:`function(root)`، جزو شروع کرنے کا فنکشن، شیڈو روٹ کنٹینر وصول کرتا ہے۔
