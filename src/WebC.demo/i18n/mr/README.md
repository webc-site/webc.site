# घटक डेमो, दस्तऐवजीकरण आणि स्थापना सूचना प्रस्तुत करा आणि प्रदर्शित करा

- README दस्तऐवजाचे शीर्षक, परिचय आणि सामग्री काढा आणि पार्स करा
- घटकांचे परस्पर डेमो आणि पृथक् शैली चालविण्यासाठी सँडबॉक्स
- दस्तऐवजांमधील कोड ब्लॉक्सना परस्पर सादरीकरणांमध्ये रूपांतरित करा
- ॲडॉप्टिव्ह लेआउट, रुंद स्क्रीन कॉलम डिस्प्ले, अरुंद स्क्रीन टॅब स्विचिंग डिस्प्ले

## डेमो वापरा

```html
<c-webc-demo></c-webc-demo>
```

```js
import WebCDemo from "./_.js";

const el = document.querySelector("c-webc-demo");

// ऑपरेशन इंटरफेस सुरू करा आणि प्राप्त करा
const [setMod, setMd, unMount] = WebCDemo(el);
```

## इंटरफेस पॅरामीटर्स

### `WebCDemo(el)`

घटक सुरू करा.

- **मापदंड**
  - `el`:`HTMLElement`, घटक कंटेनर.
- **परताव्याचे मूल्य**:`[setMod, setMd, unMount]`
  - `setMod(mod)`: डेमो मॉड्यूल सेट करा.
  - `setMd(name, md, lang)`: घटकाचे नाव, README दस्तऐवज सामग्री आणि वर्तमान भाषा सेट करा.
  - `unMount()`: घटक अनलोड करा आणि इव्हेंट श्रोता साफ करा.

### डेमो मॉड्यूल `mod` विशेषता

- `CSS`:`string`, घटकाचा स्टाईल शीट पथ.
- `HTM`:`string`, घटकाचे HTML टेम्पलेट.
- `default`:`function(root)`, घटक आरंभिकरण कार्य, शॅडो रूट कंटेनर प्राप्त करते.

## शैली वर्ग

### `.M`

सामग्री क्षेत्र.

### `.L`

डाव्या पॅनेलमध्ये सूचना आणि कागदपत्रे आहेत.

### `.R`

उजवे पॅनेल, ज्यामध्ये डेमो आहे.

### `c-nav`

अरुंद स्क्रीन नेव्हिगेशन बार.
