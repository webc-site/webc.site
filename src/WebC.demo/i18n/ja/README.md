# コンポーネントのデモ、ドキュメント、インストール手順のレンダリングと表示

- README ドキュメントのタイトル、紹介、内容を抽出して解析します。
- コンポーネントのインタラクティブなデモを実行し、スタイルを分離するためのサンドボックス
- ドキュメント内のコード ブロックをインタラクティブなプレゼンテーションに変換する
- アダプティブレイアウト、ワイド画面列表示、狭画面タブ切り替え表示

## デモを使用する

```html
<c-webc-demo></c-webc-demo>
```

```js
import WebCDemo from "./_.js";

const el = document.querySelector("c-webc-demo");

//操作インターフェースの初期化と取得
const [setMod, setMd, unMount] = WebCDemo(el);
```

## インターフェースパラメータ

### `WebCDemo(el)`

コンポーネントを初期化します。

- **パラメータ**
  - `el`:`HTMLElement`、コンポーネントコンテナ。
- **戻り値**:`[setMod, setMd, unMount]`
  - `setMod(mod)`: デモ モジュールをセットアップします。
  - `setMd(name, md, lang)`: コンポーネント名、README ドキュメントの内容、および現在の言語を設定します。
  - `unMount()`: コンポーネントをアンロードし、イベント リスナーをクリーンアップします。

### デモモジュールの `mod` 属性

- `CSS`:`string`、コンポーネントのスタイルシートのパス。
- `HTM`:`string`、コンポーネントの HTML テンプレート。
- `default`:`function(root)`、コンポーネント初期化関数は、シャドウ ルート コンテナを受け取ります。

## スタイルクラス

### `.M`

コンテンツエリア。

### `.L`

左側のパネルには手順とドキュメントが含まれています。

### `.R`

右側のパネルにはデモが含まれています。

### `c-nav`

狭い画面のナビゲーション バー。
