# कम्पोनेन्ट डेमो, कागजात, र स्थापना निर्देशनहरू रेन्डर र प्रदर्शन गर्नुहोस्

- README कागजातको शीर्षक, परिचय र सामग्री निकाल्नुहोस् र पार्स गर्नुहोस्
- कम्पोनेन्ट र आइसोलेट शैलीहरूको अन्तरक्रियात्मक डेमोहरू चलाउन स्यान्डबक्स
- कागजातहरूमा कोड ब्लकहरूलाई अन्तरक्रियात्मक प्रस्तुतीकरणहरूमा रूपान्तरण गर्नुहोस्
- अनुकूलन लेआउट, चौडा स्क्रिन स्तम्भ प्रदर्शन, साँघुरो स्क्रिन ट्याब स्विच प्रदर्शन

## डेमो प्रयोग गर्नुहोस्

```html
<c-webc-demo></c-webc-demo>
```

```js
import WebCDemo from "./_.js";

const el = document.querySelector("c-webc-demo");

// अपरेशन इन्टरफेस प्रारम्भ र प्राप्त गर्नुहोस्
const [setMod, setMd, unMount] = WebCDemo(el);
```

## इन्टरफेस प्यारामिटरहरू

### `WebCDemo(el)`

कम्पोनेन्ट प्रारम्भ गर्नुहोस्।

- ** प्यारामिटर**
  - `el`:`HTMLElement`, कम्पोनेन्ट कन्टेनर।
- **फिर्ता मान**:`[setMod, setMd, unMount]`
  - `setMod(mod)`: डेमो मोड्युल सेटअप गर्नुहोस्।
  - `setMd(name, md, lang)`: कम्पोनेन्ट नाम, README कागजात सामग्री र हालको भाषा सेट गर्नुहोस्।
  - `unMount()`: कम्पोनेन्ट अनलोड गर्नुहोस् र घटना श्रोतालाई सफा गर्नुहोस्।

### डेमो मोड्युल `mod` विशेषता

- `CSS`:`string`, कम्पोनेन्टको शैली पाना मार्ग।
- `HTM`:`string`, कम्पोनेन्टको HTML टेम्प्लेट।
- `default`:`function(root)`, कम्पोनेन्ट प्रारम्भिक कार्य, छाया रूट कन्टेनर प्राप्त गर्दछ।

## शैली वर्ग

### `.M`

सामग्री क्षेत्र।

### `.L`

बायाँ प्यानलले निर्देशन र कागजात समावेश गर्दछ।

### `.R`

दायाँ प्यानल, डेमो समावेश।

### `c-nav`

साँघुरो स्क्रिन नेभिगेसन पट्टी।
