# نمایش دموی مؤلفه، مستندات، و دستورالعمل های نصب را ارائه و نمایش دهید

- عنوان، مقدمه و محتوای سند README را استخراج و تجزیه کنید
- Sandbox برای اجرای دموهای تعاملی اجزا و جداسازی سبک ها
- بلوک های کد موجود در اسناد را به ارائه های تعاملی تبدیل کنید
- چیدمان تطبیقی، نمایش ستون عریض، صفحه نمایش تغییر زبانه باریک

## از دمو استفاده کنید

```html
<c-webc-demo></c-webc-demo>
```

```js
import WebCDemo from "./_.js";

const el = document.querySelector("c-webc-demo");

//Initialize و رابط عملیات را بدست آورید
const [setMod, setMd, unMount] = WebCDemo(el);
```

## پارامترهای رابط

### `WebCDemo(el)`

کامپوننت را مقدار دهی اولیه کنید.

- **پارامتر**
  - `el`:`HTMLElement`، محفظه مؤلفه.
- **مقدار برگشتی**:`[setMod, setMd, unMount]`
  - `setMod(mod)`: ماژول نمایشی را تنظیم کنید.
  - `setMd(name, md, lang)`: نام مؤلفه، محتوای سند README و زبان فعلی را تنظیم کنید.
  - `unMount()`: مؤلفه را بارگیری کنید و شنونده رویداد را تمیز کنید.

### ویژگی ماژول نمایشی `mod`

- `CSS`:`string`، مسیر برگه سبک کامپوننت.
- `HTM`:`string`، الگوی HTML مؤلفه.
- `default`:`function(root)`، تابع اولیه سازی مؤلفه، محفظه ریشه سایه را دریافت می کند.

## کلاس سبک

### `.M`

حوزه محتوا

### `.L`

پانل سمت چپ حاوی دستورالعمل ها و مستندات است.

### `.R`

پانل سمت راست، حاوی نسخه نمایشی.

### `c-nav`

نوار ناوبری صفحه نمایش باریک.
