# ઘટક ડેમો, દસ્તાવેજીકરણ અને ઇન્સ્ટોલેશન સૂચનાઓ રેન્ડર અને પ્રદર્શિત કરો

- README દસ્તાવેજના શીર્ષક, પરિચય અને સામગ્રીને બહાર કાઢો અને તેનું વિશ્લેષણ કરો
- ઘટકોના ઇન્ટરેક્ટિવ ડેમો ચલાવવા અને શૈલીઓને અલગ કરવા માટે સેન્ડબોક્સ
- દસ્તાવેજોમાંના કોડ બ્લોક્સને ઇન્ટરેક્ટિવ પ્રસ્તુતિઓમાં કન્વર્ટ કરો
- અનુકૂલનશીલ લેઆઉટ, વિશાળ સ્ક્રીન કૉલમ ડિસ્પ્લે, સાંકડી સ્ક્રીન ટેબ સ્વિચિંગ ડિસ્પ્લે

## ડેમોનો ઉપયોગ કરો

```html
<c-webc-demo></c-webc-demo>
```

```js
import WebCDemo from "./_.js";

const el = document.querySelector("c-webc-demo");

// ઓપરેશન ઇન્ટરફેસ શરૂ કરો અને મેળવો
const [setMod, setMd, unMount] = WebCDemo(el);
```

## ઇન્ટરફેસ પરિમાણો

### `WebCDemo(el)`

ઘટક શરૂ કરો.

- **પેરામીટર**
  - `el`:`HTMLElement`, ઘટક કન્ટેનર.
- **વળતર મૂલ્ય**:`[setMod, setMd, unMount]`
  - `setMod(mod)`: ડેમો મોડ્યુલ સેટ કરો.
  - `setMd(name, md, lang)`: ઘટકનું નામ, README દસ્તાવેજ સામગ્રી અને વર્તમાન ભાષા સેટ કરો.
  - `unMount()`: ઘટકને અનલોડ કરો અને ઇવેન્ટ સાંભળનારને સાફ કરો.

### ડેમો મોડ્યુલ `mod` વિશેષતા

- `CSS`:`string`, ઘટકનો સ્ટાઈલ શીટ પાથ.
- `HTM`:`string`, ઘટકનો HTML ટેમ્પલેટ.
- `default`:`function(root)`, ઘટક પ્રારંભ કાર્ય, શેડો રૂટ કન્ટેનર મેળવે છે.

## શૈલી વર્ગ

### `.M`

સામગ્રી વિસ્તાર.

### `.L`

ડાબી પેનલમાં સૂચનાઓ અને દસ્તાવેજો છે.

### `.R`

જમણી પેનલ, જેમાં ડેમો છે.

### `c-nav`

સાંકડી સ્ક્રીન નેવિગેશન બાર.
