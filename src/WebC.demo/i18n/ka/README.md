- ამოიღეთ და გააანალიზეთ README დოკუმენტის სათაური, შესავალი და შინაარსი
- Sandbox კომპონენტების ინტერაქტიული დემოს გასაშვებად და სტილების იზოლირებისთვის
- გადააკეთეთ დოკუმენტების კოდის ბლოკები ინტერაქტიულ პრეზენტაციებად
- ადაპტირებული განლაგება, ფართო ეკრანის სვეტის ჩვენება, ვიწრო ეკრანის ჩანართის გადართვის ჩვენება

## გამოიყენეთ დემო ვერსია

```html
<c-webc-demo></c-webc-demo>
```

```js
import WebCDemo from "./_.js";

const el = document.querySelector("c-webc-demo");

//ინიციალიზება და ოპერაციის ინტერფეისის მიღება
const [setMod, setMd, unMount] = WebCDemo(el);
```

## ინტერფეისის პარამეტრები

### `WebCDemo(el)`

კომპონენტის ინიციალიზაცია.

- **პარამეტრი**
  - `el`:`HTMLElement`, კომპონენტის კონტეინერი.
- **დაბრუნების მნიშვნელობა**:`[setMod, setMd, unMount]`
  - `setMod(mod)`: დააყენეთ დემო მოდული.
  - `setMd(name, md, lang)`: დააყენეთ კომპონენტის სახელი, README დოკუმენტის შინაარსი და მიმდინარე ენა.
  - `unMount()`: ჩამოტვირთეთ კომპონენტი და გაასუფთავეთ ღონისძიების მსმენელი.

### დემო მოდული `mod` ატრიბუტი

- `CSS`:`string`, კომპონენტის სტილის ფურცლის გზა.
- `HTM`:`string`, კომპონენტის HTML შაბლონი.
- `default`:`function(root)`, კომპონენტის ინიციალიზაციის ფუნქცია, იღებს ჩრდილოვანი ფესვის კონტეინერს.

## სტილის კლასი

### `.M`

შინაარსის არეალი.

### `.L`

### `.R`

მარჯვენა პანელი, შეიცავს დემო.

### `c-nav`

ვიწრო ეკრანის ნავიგაციის ზოლი.
