# Компонент демонстрацияларын, құжаттаманы және орнату нұсқауларын көрсету және көрсету

- README құжатының тақырыбын, кіріспесін және мазмұнын шығарып алыңыз және талдаңыз
- Құрамдастардың интерактивті демонстрацияларын іске қосуға және стильдерді оқшаулауға арналған құмсалғыш
- Құжаттардағы код блоктарын интерактивті презентацияларға түрлендіру
- Бейімделетін орналасу, кең экранды баған дисплейі, тар экран қойындысын ауыстыру дисплейі

## Демонстрацияны пайдаланыңыз

```html
<c-webc-demo></c-webc-demo>
```

```js
import WebCDemo from "./_.js";

const el = document.querySelector("c-webc-demo");

//Операциялық интерфейсті инициализациялаңыз және алыңыз
const [setMod, setMd, unMount] = WebCDemo(el);
```

## Интерфейс параметрлері

### `WebCDemo(el)`

Компонентті инициализациялаңыз.

- **параметр**
  - `el`:`HTMLElement`, құрамдас контейнер.
- **Қайтарылатын мән**:`[setMod, setMd, unMount]`
  - `setMod(mod)`: демонстрациялық модульді орнату.
  - `setMd(name, md, lang)`: Құрамдас атын, README құжатының мазмұнын және ағымдағы тілді орнатыңыз.
  - `unMount()`: құрамдас бөлікті босатып, оқиға тыңдаушысын тазалаңыз.

### Демо-модуль `mod` төлсипаты

- `CSS`:`string`, компоненттің стильдер кестесінің жолы.
- `HTM`:`string`, компоненттің HTML үлгісі.
- `default`:`function(root)`, құрамдас баптандыру функциясы, көлеңкелі түбірлік контейнерді қабылдайды.
