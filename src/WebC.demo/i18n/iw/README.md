# עבד והצג הדגמות של רכיבים, תיעוד והוראות התקנה

- חלץ ונתח את הכותרת, המבוא והתוכן של מסמך README
- ארגז חול להפעלת הדגמות אינטראקטיביות של רכיבים ובידוד סגנונות
- המרת בלוקי קוד במסמכים למצגות אינטראקטיביות
- פריסה מותאמת, תצוגת עמודות מסך רחב, תצוגת החלפת כרטיסיות מסך צר

## השתמש בהדגמה

```html
<c-webc-demo></c-webc-demo>
```

```js
import WebCDemo from "./_.js";

const el = document.querySelector("c-webc-demo");

//אתחול והשג את ממשק הפעולה
const [setMod, setMd, unMount] = WebCDemo(el);
```

## פרמטרים של ממשק

### `WebCDemo(el)`

אתחל את הרכיב.

- **פָּרָמֶטֶר**
  - `el`:`HTMLElement`, מיכל רכיב.
- **ערך החזרה**:`[setMod, setMd, unMount]`
  - `setMod(mod)`: הגדר את מודול ההדגמה.
  - `setMd(name, md, lang)`: הגדר את שם הרכיב, תוכן מסמך README והשפה הנוכחית.
  - `unMount()`: פרק את הרכיב ונקה את מאזין האירועים.

### תכונת `mod` מודול הדגמה

- `CSS`:`string`, נתיב גיליון הסגנונות של הרכיב.
- `HTM`:`string`, תבנית ה-HTML של הרכיב.
- `default`:`function(root)`, פונקציית אתחול רכיב, מקבל מיכל שורש צל.

## שיעור סגנון

### `.M`

אזור תוכן.

### `.L`

הלוח השמאלי מכיל הוראות ותיעוד.

### `.R`

לוח ימין, המכיל את ההדגמה.

### `c-nav`

סרגל ניווט מסך צר.
