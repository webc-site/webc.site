# Стил на листа на врски со лентата за навигација

- Хоризонтален распоред врска деца
- Содржи стандардни, подвижни и активни состојби
- Вградени анимации за транзиција и клик-за-зумирање

## Користете го демо

```html
<div class="NavALi">
  <a class="Card" href="#home">Врска</a>
  <a class="Card A" href="#active">линк за активирање</a>
</div>
```

## статус на стил

### `.NavALi`

Листата со надворешни врски применува хоризонтален распоред и растојание.

### `.Card`

Поврзете ги децата.

### `.A`

Кога е активирано, се применува ефектот на нагласување.
