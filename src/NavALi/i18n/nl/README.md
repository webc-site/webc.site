# Stijl van lijst met koppelingen in de navigatiebalk

- Horizontale layout link kinderen
- Bevat standaard-, hover- en actieve statussen
- Ingebouwde overgangs- en klik-om-zoom-animaties

## Gebruik de demo

```html
<div class="NavALi">
  <a class="Card" href="#home">Link</a>
  <a class="Card A" href="#active">activatielink</a>
</div>
```

## stijl status

### `.NavALi`

De buitenste linklijst past een horizontale lay-out en spatiëring toe.

### `.Card`

Kinderen koppelen.

### `.A`

Indien geactiveerd, wordt het highlight-effect toegepast.
