# Stili i listës së lidhjeve të shiritit të navigimit

- Lidhja e shtrirjes horizontale për fëmijët
- Përmban gjendjet e parazgjedhura, hover dhe aktive
- Animacionet e integruara të tranzicionit dhe kliko-për-zoom

## Përdorni demonstrimin

```html
<div class="NavALi">
  <a class="Card" href="#home">Lidhje</a>
  <a class="Card A" href="#active">lidhjen e aktivizimit</a>
</div>
```

## statusi i stilit

### `.NavALi`

Lista e lidhjeve të jashtme zbaton paraqitjen dhe ndarjen horizontale.

### `.Card`

Lidhni fëmijët.

### `.A`

Kur aktivizohet, zbatohet efekti i theksimit.
