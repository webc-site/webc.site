# Стил на списък с връзки на лентата за навигация

- Хоризонтално оформление връзка деца
- Съдържа състояния по подразбиране, при задържане и активни състояния
- Вградени анимации за преход и мащабиране с кликване

## Използвайте демонстрацията

```html
<div class="NavALi">
  <a class="Card" href="#home">Връзка</a>
  <a class="Card A" href="#active">връзка за активиране</a>
</div>
```

## статус на стил

### `.NavALi`

Списъкът с външни връзки прилага хоризонтално оформление и разстояние.

### `.Card`

Линк деца.

### `.A`

Когато се активира, се прилага ефектът на подчертаване.
