# Navigationslinje-linklistestil

- Vandret layout forbinder børn
- Indeholder standard, hover og aktive tilstande
- Indbyggede overgange og klik-for-zoom-animationer

## Brug demoen

```html
<div class="NavALi">
  <a class="Card" href="#home">Forbindelse</a>
  <a class="Card A" href="#active">aktiveringslink</a>
</div>
```

## stilstatus

### `.NavALi`

Den ydre linkliste anvender vandret layout og mellemrum.

### `.Card`

Forbind børn.

### `.A`

Når den er aktiveret, anvendes highlight-effekten.
