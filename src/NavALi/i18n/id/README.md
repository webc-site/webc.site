# Gaya daftar tautan bilah navigasi

- Tautan tata letak horizontal anak-anak
- Berisi status default, hover, dan aktif
- Transisi bawaan dan animasi klik untuk memperbesar

## Gunakan demonya

```html
<div class="NavALi">
  <a class="Card" href="#home">Link</a>
  <a class="Card A" href="#active">tautan aktivasi</a>
</div>
```

## status gaya

### `.NavALi`

Daftar tautan luar menerapkan tata letak dan spasi horizontal.

### `.Card`

Tautkan anak-anak.

### `.A`

Saat diaktifkan, efek sorotan diterapkan.
