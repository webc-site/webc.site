# Estilo de lista de enlaces de la barra de navegación

- Hijos de enlace de diseño horizontal
- Contiene estados predeterminado, flotante y activo.
- Transiciones integradas y animaciones de hacer clic para ampliar

## Usa la demostración

```html
<div class="NavALi">
  <a class="Card" href="#home">Enlace</a>
  <a class="Card A" href="#active">enlace de activación</a>
</div>
```

## estado de estilo

### `.NavALi`

La lista de enlaces externos aplica diseño y espaciado horizontal.

### `.Card`

Vincular a los niños.

### `.A`

Cuando se activa, se aplica el efecto de resaltado.
