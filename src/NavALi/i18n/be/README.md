# Стыль спісу спасылак панэлі навігацыі

- Спасылка гарызантальнай разметкі дзяцей
- Змяшчае стан па змаўчанні, навядзенне і актыўны стан
- Убудаваныя анімацыі пераходу і маштабавання пстрычкай

## Выкарыстоўвайце дэма

```html
<div class="NavALi">
  <a class="Card" href="#home">Спасылка</a>
  <a class="Card A" href="#active">спасылка для актывацыі</a>
</div>
```

## статус стылю

### `.NavALi`

Спіс знешніх спасылак прымяняе гарызантальную кампаноўку і інтэрвал.

### `.Card`

Спасылка дзяцей.

### `.A`

Пры актывацыі прымяняецца эфект падсветкі.
