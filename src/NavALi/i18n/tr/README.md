# Gezinme çubuğu bağlantı listesi stili

- Yatay düzen bağlantısı çocukları
- Varsayılan, vurgulu ve etkin durumları içerir
- Yerleşik geçiş ve tıklatarak yakınlaştırma animasyonları

## Demoyu kullanın

```html
<div class="NavALi">
  <a class="Card" href="#home">Bağlantı</a>
  <a class="Card A" href="#active">etkinleştirme bağlantısı</a>
</div>
```

## stil durumu

### `.NavALi`

Dış bağlantı listesinde yatay düzen ve aralık uygulanır.

### `.Card`

Çocukları bağlayın.

### `.A`

Etkinleştirildiğinde vurgu efekti uygulanır.
