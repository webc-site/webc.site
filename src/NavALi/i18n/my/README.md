# လမ်းကြောင်းပြဘားလင့်ခ်စာရင်းပုံစံ

- အလျားလိုက် လက်ကွက် လင့်လေးတွေပါ။
- ပုံသေ၊ ရွှေ့ခြင်းနှင့် အသက်ဝင်သော အခြေအနေများ ပါရှိသည်။
- Built-in အကူးအပြောင်းနှင့် ကလစ်နှိပ်ရန် ဇူးမ်ကာတွန်းများ

## ဒီမိုကိုသုံးပါ။

```html
<div class="NavALi">
  <a class="Card" href="#home">လင့်</a>
  <a class="Card A" href="#active">activation လင့်ခ်</a>
</div>
```

## ပုံစံအနေအထား

### `.NavALi`

ပြင်ပလင့်ခ်စာရင်းသည် အလျားလိုက် အပြင်အဆင်နှင့် အကွာအဝေးကို သက်ရောက်မှုရှိသည်။

### `.Card`

လင့်လေးတွေပါ။

### `.A`

စဖွင့်သောအခါ၊ အထူးအသားပေးအကျိုးသက်ရောက်မှုကို အသုံးပြုသည်။
