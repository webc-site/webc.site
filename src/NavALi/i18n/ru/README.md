# Стиль списка ссылок на панели навигации

- Горизонтальная компоновка ссылок дочерних элементов
- Содержит состояния по умолчанию, при наведении и активные состояния.
- Встроенная анимация перехода и масштабирования по щелчку.

## Используйте демо-версию

```html
<div class="NavALi">
  <a class="Card" href="#home">Связь</a>
  <a class="Card A" href="#active">ссылка активации</a>
</div>
```

## статус стиля

### `.NavALi`

В списке внешних ссылок применяется горизонтальное расположение и интервалы.

### `.Card`

Связать детей.

### `.A`

При активации применяется эффект выделения.
