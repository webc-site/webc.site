# Slog seznama povezav navigacijske vrstice

- Vodoravna postavitev povezava otrok
- Vsebuje privzeto stanje, stanje lebdenja in aktivno stanje
- Vgrajen prehod in animacije s klikom za povečavo

## Uporabite demo

```html
<div class="NavALi">
  <a class="Card" href="#home">Povezava</a>
  <a class="Card A" href="#active">aktivacijska povezava</a>
</div>
```

## stilski status

### `.NavALi`

Seznam zunanjih povezav uporablja vodoravno postavitev in razmik.

### `.Card`

Povežite otroke.

### `.A`

Ko je aktiviran, se uporabi učinek osvetlitve.
