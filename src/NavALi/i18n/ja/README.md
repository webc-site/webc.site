# ナビゲーション バーのリンク リストのスタイル

- 水平レイアウトのリンクの子
- デフォルト、ホバー、アクティブ状態が含まれます
- 組み込みのトランジションおよびクリックしてズームするアニメーション

## デモを使用する

```html
<div class="NavALi">
  <a class="Card" href="#home">リンク</a>
  <a class="Card A" href="#active">アクティベーションリンク</a>
</div>
```

## スタイルステータス

### `.NavALi`

外側のリンク リストには、水平方向のレイアウトと間隔が適用されます。

### `.Card`

子どもたちをリンクします。

### `.A`

有効にすると、ハイライト効果が適用されます。
