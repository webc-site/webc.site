# नेव्हिगेशन बार लिंक सूची शैली

- क्षैतिज लेआउट लिंक मुले
- डीफॉल्ट, फिरवा आणि सक्रिय स्थिती समाविष्टीत आहे
- अंगभूत संक्रमण आणि क्लिक-टू-झूम ॲनिमेशन

## डेमो वापरा

```html
<div class="NavALi">
  <a class="Card" href="#home">दुवा</a>
  <a class="Card A" href="#active">सक्रियकरण दुवा</a>
</div>
```

## शैली स्थिती

### `.NavALi`

बाह्य लिंक सूची क्षैतिज मांडणी आणि अंतर लागू करते.

### `.Card`

मुलांना लिंक करा.

### `.A`

सक्रिय केल्यावर, हायलाइट प्रभाव लागू केला जातो.
