# Stil liste linkova trake za navigaciju

- Horizontalni raspored povezuje djecu
- Sadrži zadano, lebdeće i aktivno stanje
- Ugrađeni prijelaz i animacije klikni za zumiranje

## Koristite demo

```html
<div class="NavALi">
  <a class="Card" href="#home">Link</a>
  <a class="Card A" href="#active">link za aktivaciju</a>
</div>
```

## status stila

### `.NavALi`

Lista vanjskih veza primjenjuje horizontalni raspored i razmake.

### `.Card`

Povežite djecu.

### `.A`

Kada se aktivira, primjenjuje se efekat isticanja.
