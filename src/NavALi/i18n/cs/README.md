# Styl seznamu odkazů navigační lišty

- Vodorovné rozvržení odkaz děti
- Obsahuje výchozí stavy, stavy přechodu a aktivní
- Vestavěné animace přechodů a přiblížení kliknutím

## Použijte demo

```html
<div class="NavALi">
  <a class="Card" href="#home">Odkaz</a>
  <a class="Card A" href="#active">aktivační odkaz</a>
</div>
```

## stav stylu

### `.NavALi`

Seznam vnějších odkazů používá vodorovné rozvržení a mezery.

### `.Card`

Propojit děti.

### `.A`

Při aktivaci se použije efekt zvýraznění.
