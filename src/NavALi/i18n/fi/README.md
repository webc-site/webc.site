# Navigointipalkin linkkiluettelon tyyli

- Vaakasuora asettelu linkki lapset
- Sisältää oletus-, hiiri- ja aktiiviset tilat
- Sisäänrakennettu siirtymä- ja zoomausanimaatiot

## Käytä demoa

```html
<div class="NavALi">
  <a class="Card" href="#home">Linkki</a>
  <a class="Card A" href="#active">aktivointilinkki</a>
</div>
```

## tyylin tila

### `.NavALi`

Ulompi linkkiluettelo koskee vaakasuuntaista asettelua ja välilyöntejä.

### `.Card`

Linkitä lapset.

### `.A`

Kun tämä on aktivoitu, korostustehoste otetaan käyttöön.
