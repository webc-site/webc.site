# Gaya senarai pautan bar navigasi

- Pautan susun atur mendatar kanak-kanak
- Mengandungi keadaan lalai, tuding dan aktif
- Animasi peralihan terbina dalam dan klik untuk zum

## Gunakan demo

```html
<div class="NavALi">
  <a class="Card" href="#home">Pautan</a>
  <a class="Card A" href="#active">pautan pengaktifan</a>
</div>
```

## status gaya

### `.NavALi`

Senarai pautan luar menggunakan reka letak mendatar dan jarak.

### `.Card`

Pautkan kanak-kanak.

### `.A`

Apabila diaktifkan, kesan serlahan digunakan.
