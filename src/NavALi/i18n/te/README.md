# నావిగేషన్ బార్ లింక్ జాబితా శైలి

- క్షితిజసమాంతర లేఅవుట్ లింక్ పిల్లలు
- డిఫాల్ట్, హోవర్ మరియు యాక్టివ్ స్టేట్‌లను కలిగి ఉంటుంది
- అంతర్నిర్మిత పరివర్తన మరియు క్లిక్-టు-జూమ్ యానిమేషన్లు

## డెమో ఉపయోగించండి

```html
<div class="NavALi">
  <a class="Card" href="#home">లింక్</a>
  <a class="Card A" href="#active">యాక్టివేషన్ లింక్</a>
</div>
```

## శైలి స్థితి

### `.NavALi`

బయటి లింక్ జాబితా క్షితిజ సమాంతర లేఅవుట్ మరియు అంతరాన్ని వర్తింపజేస్తుంది.

### `.Card`

పిల్లలను లింక్ చేయండి.

### `.A`

సక్రియం చేసినప్పుడు, హైలైట్ ప్రభావం వర్తించబడుతుంది.
