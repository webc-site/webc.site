# រចនាប័ទ្មបញ្ជីតំណរបាររុករក

- ប្លង់ផ្ដេកភ្ជាប់កុមារ
- មានលំនាំដើម សង្កត់លើ និងស្ថានភាពសកម្ម
- ការផ្លាស់ប្តូរដែលភ្ជាប់មកជាមួយ និងចលនាចុចដើម្បីពង្រីក

## ប្រើការបង្ហាញ

```html
<div class="NavALi">
  <a class="Card" href="#home">តំណភ្ជាប់</a>
  <a class="Card A" href="#active">តំណភ្ជាប់ធ្វើឱ្យសកម្ម</a>
</div>
```

## ស្ថានភាពរចនាប័ទ្ម

### `.NavALi`

បញ្ជីតំណខាងក្រៅអនុវត្តប្លង់ផ្ដេក និងគម្លាត។

### `.Card`

ភ្ជាប់កុមារ។

### `.A`

នៅពេលដំណើរការ ឥទ្ធិពលបន្លិចត្រូវបានអនុវត្ត។
