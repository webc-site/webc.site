# Стил листе линкова траке за навигацију

- Хоризонтални изглед везе деца
- Садржи подразумевана, лебдећа и активна стања
- Уграђени прелаз и анимације са кликом за зумирање

## Користите демо

```html
<div class="NavALi">
  <a class="Card" href="#home">Линк</a>
  <a class="Card A" href="#active">линк за активацију</a>
</div>
```

## статус стила

### `.NavALi`

Листа спољних веза примењује хоризонтални распоред и размаке.

### `.Card`

Повежите децу.

### `.A`

Када се активира, примењује се ефекат истицања.
