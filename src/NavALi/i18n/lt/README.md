# Naršymo juostos nuorodų sąrašo stilius

- Horizontalus išdėstymas nuoroda vaikams
- Yra numatytosios, užvedimo ir aktyvios būsenos
- Integruotas perėjimas ir spustelėjus, kad padidintumėte mastelį, animacija

## Naudokite demonstracinę versiją

```html
<div class="NavALi">
  <a class="Card" href="#home">Nuoroda</a>
  <a class="Card A" href="#active">aktyvinimo nuoroda</a>
</div>
```

## stiliaus statusas

### `.NavALi`

Išoriniame nuorodų sąraše taikomas horizontalus išdėstymas ir tarpai.

### `.Card`

Susieti vaikus.

### `.A`

Kai suaktyvinta, taikomas paryškinimo efektas.
