# Stil der Linkliste der Navigationsleiste

- Untergeordnete Elemente im horizontalen Layout verknüpfen
- Enthält Standard-, Hover- und Aktivstatus
- Integrierte Übergangs- und Click-to-Zoom-Animationen

## Nutzen Sie die Demo

```html
<div class="NavALi">
  <a class="Card" href="#home">Link</a>
  <a class="Card A" href="#active">Aktivierungslink</a>
</div>
```

## Stilstatus

### `.NavALi`

Die äußere Linkliste wendet horizontales Layout und Abstand an.

### `.Card`

Kinder verknüpfen.

### `.A`

Bei Aktivierung wird der Hervorhebungseffekt angewendet.
