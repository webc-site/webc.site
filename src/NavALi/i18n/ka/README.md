# ნავიგაციის ზოლის ბმულების სიის სტილი

- ჰორიზონტალური განლაგების ბმული ბავშვები
- შეიცავს ნაგულისხმევ, მავალ და აქტიურ მდგომარეობებს
- ჩაშენებული გადასვლისა და დაწკაპუნებით მასშტაბირების ანიმაციები

## გამოიყენეთ დემო ვერსია

```html
<div class="NavALi">
  <a class="Card" href="#home">ბმული</a>
  <a class="Card A" href="#active">აქტივაციის ლინკი</a>
</div>
```

## სტილის სტატუსი

### `.NavALi`

გარე ბმულების სია იყენებს ჰორიზონტალურ განლაგებას და ინტერვალს.

### `.Card`

დააკავშირეთ ბავშვები.

### `.A`

როდესაც გააქტიურებულია, ხაზგასმული ეფექტი გამოიყენება.
