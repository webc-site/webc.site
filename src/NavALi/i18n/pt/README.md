# Estilo de lista de links da barra de navegação

- Layout horizontal link filhos
- Contém estados padrão, flutuante e ativo
- Transição integrada e animações clique para ampliar

## Use a demonstração

```html
<div class="NavALi">
  <a class="Card" href="#home">Link</a>
  <a class="Card A" href="#active">link de ativação</a>
</div>
```

## status de estilo

### `.NavALi`

A lista de links externos aplica layout e espaçamento horizontais.

### `.Card`

Vincule crianças.

### `.A`

Quando ativado, o efeito de destaque é aplicado.
