# Štýl zoznamu odkazov navigačného panela

- Horizontálne rozloženie odkaz deti
- Obsahuje predvolené stavy, stavy kurzora a aktívne stavy
- Vstavané animácie prechodu a priblíženia kliknutím

## Použite demo

```html
<div class="NavALi">
  <a class="Card" href="#home">Odkaz</a>
  <a class="Card A" href="#active">aktivačný odkaz</a>
</div>
```

## stav štýlu

### `.NavALi`

Vonkajší zoznam odkazov používa horizontálne rozloženie a medzery.

### `.Card`

Prepojiť deti.

### `.A`

Po aktivácii sa použije efekt zvýraznenia.
