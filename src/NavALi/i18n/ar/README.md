# نمط قائمة ارتباط شريط التنقل

- ربط الأطفال بالتخطيط الأفقي
- يحتوي على الحالات الافتراضية والتمرير والحالات النشطة
- انتقال مدمج ورسوم متحركة للنقر للتكبير

## استخدم العرض التوضيحي

```html
<div class="NavALi">
  <a class="Card" href="#home">وصلة</a>
  <a class="Card A" href="#active">رابط التنشيط</a>
</div>
```

## حالة النمط

### `.NavALi`

تطبق قائمة الارتباطات الخارجية التخطيط الأفقي والتباعد.

### `.Card`

ربط الأطفال.

### `.A`

عند التنشيط، يتم تطبيق تأثير التمييز.
