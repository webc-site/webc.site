# नेविगेशन बार लिंक सूची शैली

- क्षैतिज लेआउट लिंक बच्चे
- इसमें डिफ़ॉल्ट, होवर और सक्रिय स्थितियाँ शामिल हैं
- अंतर्निहित संक्रमण और क्लिक-टू-ज़ूम एनिमेशन

## डेमो का प्रयोग करें

```html
<div class="NavALi">
  <a class="Card" href="#home">जोड़ना</a>
  <a class="Card A" href="#active">सक्रियण लिंक</a>
</div>
```

## शैली की स्थिति

### `.NavALi`

बाहरी लिंक सूची क्षैतिज लेआउट और रिक्ति लागू करती है।

### `.Card`

बच्चों को जोड़ें.

### `.A`

सक्रिय होने पर, हाइलाइट प्रभाव लागू होता है।
