# নেভিগেচন বাৰ লিংক তালিকা শৈলী

- অনুভূমিক বিন্যাস লিংক শিশু
- অবিকল্পিত, হ'ভাৰ আৰু সক্ৰিয় অৱস্থাসমূহ অন্তৰ্ভুক্ত কৰে
- বিল্ট-ইন পৰিৱৰ্তন আৰু ক্লিক-টু-জুম এনিমেচন

## ডেমো ব্যৱহাৰ কৰক

```html
<div class="NavALi">
  <a class="Card" href="#home">লিংক</a>
  <a class="Card A" href="#active">সক্ৰিয়কৰণ লিংক</a>
</div>
```

## শৈলীৰ অৱস্থা

### `.NavALi`

বাহিৰৰ সংযোগ তালিকাই অনুভূমিক বিন্যাস আৰু ব্যৱধান প্ৰয়োগ কৰে।

### `.Card`

ল'ৰা-ছোৱালীক লিংক কৰক।

### `.A`

সক্ৰিয় হ'লে হাইলাইট ইফেক্ট প্ৰয়োগ কৰা হয়।
