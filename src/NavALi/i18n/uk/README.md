# Стиль списку посилань панелі навігації

- Дочірні посилання горизонтального макета
- Містить стан за замовчуванням, наведення та активний стан
- Вбудовані анімації переходів і збільшення при натисканні

## Використовуйте демо

```html
<div class="NavALi">
  <a class="Card" href="#home">Посилання</a>
  <a class="Card A" href="#active">посилання для активації</a>
</div>
```

## статус стилю

### `.NavALi`

У списку зовнішніх посилань застосовуються горизонтальне розташування та інтервали.

### `.Card`

Посилання дітей.

### `.A`

При активації застосовується ефект підсвічування.
