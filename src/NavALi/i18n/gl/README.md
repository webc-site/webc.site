# Estilo de lista de ligazóns da barra de navegación

- Ligazón de deseño horizontal fillos
- Contén estados predeterminados, de paso e activo
- Animacións de transición e zoom incorporadas

## Use a demostración

```html
<div class="NavALi">
  <a class="Card" href="#home">Ligazón</a>
  <a class="Card A" href="#active">enlace de activación</a>
</div>
```

## estado de estilo

### `.NavALi`

A lista de ligazóns externas aplica un deseño horizontal e un espazo.

### `.Card`

Vincular os nenos.

### `.A`

Cando está activado, aplícase o efecto de resaltado.
