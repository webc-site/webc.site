# Estilo sa listahan sa link sa navigation bar

- Horizontal nga layout link sa mga bata
- Naglangkob sa default, hover ug aktibo nga estado
- Gitukod-sa transisyon ug click-to-zoom animation

## Gamita ang demo

```html
<div class="NavALi">
  <a class="Card" href="#home">Link</a>
  <a class="Card A" href="#active">link sa pagpaaktibo</a>
</div>
```

## kahimtang sa estilo

### `.NavALi`

Ang lista sa gawas nga link magamit ang pinahigda nga layout ug gilay-on.

### `.Card`

I-link ang mga bata.

### `.A`

Kung gi-aktibo, ang epekto sa highlight magamit.
