# سبک لیست پیوند نوار پیمایش

- پیوند طرح افقی کودکان
- شامل حالت های پیش فرض، شناور و فعال است
- انیمیشن های انتقال داخلی و کلیک به زوم

## از دمو استفاده کنید

```html
<div class="NavALi">
  <a class="Card" href="#home">پیوند</a>
  <a class="Card A" href="#active">لینک فعال سازی</a>
</div>
```

## وضعیت سبک

### `.NavALi`

فهرست پیوندهای بیرونی، طرح بندی افقی و فاصله را اعمال می کند.

### `.Card`

کودکان را پیوند دهید

### `.A`

هنگامی که فعال می شود، جلوه برجسته اعمال می شود.
