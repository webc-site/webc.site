# Länkliststil i navigeringsfältet

- Horisontell layout länk barn
- Innehåller standardtillstånd, hovring och aktiva tillstånd
- Inbyggda övergångs- och klicka-för-zoomningsanimationer

## Använd demon

```html
<div class="NavALi">
  <a class="Card" href="#home">Länk</a>
  <a class="Card A" href="#active">aktiveringslänk</a>
</div>
```

## stilstatus

### `.NavALi`

Den yttre länklistan tillämpar horisontell layout och avstånd.

### `.Card`

Länka barn.

### `.A`

När den är aktiverad tillämpas markeringseffekten.
