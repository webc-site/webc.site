# Naviqasiya paneli keçid siyahısı stili

- Uşaqların üfüqi düzülüşü
- Defolt, hover və aktiv vəziyyətləri ehtiva edir
- Daxili keçid və böyütmək üçün kliklə animasiyalar

## Demodan istifadə edin

```html
<div class="NavALi">
  <a class="Card" href="#home">Link</a>
  <a class="Card A" href="#active">aktivləşdirmə linki</a>
</div>
```

## stil statusu

### `.NavALi`

Xarici keçid siyahısı üfüqi düzülmə və boşluq tətbiq edir.

### `.Card`

Uşaqları bağlayın.

### `.A`

Aktivləşdirildikdə, vurğulama effekti tətbiq olunur.
