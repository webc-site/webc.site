# Στυλ λίστας συνδέσμων γραμμής πλοήγησης

- Παιδιά σύνδεσμος οριζόντιας διάταξης
- Περιέχει προεπιλεγμένες καταστάσεις, αιώρηση και ενεργές καταστάσεις
- Ενσωματωμένα κινούμενα σχέδια μετάβασης και κλικ για ζουμ

## Χρησιμοποιήστε το demo

```html
<div class="NavALi">
  <a class="Card" href="#home">Σύνδεσμος</a>
  <a class="Card A" href="#active">σύνδεσμος ενεργοποίησης</a>
</div>
```

## κατάσταση στυλ

### `.NavALi`

Η λίστα εξωτερικών συνδέσμων εφαρμόζει οριζόντια διάταξη και απόσταση.

### `.Card`

Συνδέστε τα παιδιά.

### `.A`

Όταν ενεργοποιηθεί, εφαρμόζεται το εφέ επισήμανσης.
