# Шарлау жолағы сілтеме тізімінің стилі

- Балалардың көлденең орналасуы
- Әдепкі, меңзерді және белсенді күйлерді қамтиды
- Кірістірілген өту және масштабтау үшін басу анимациялары

## Демонстрацияны пайдаланыңыз

```html
<div class="NavALi">
  <a class="Card" href="#home">Сілтеме</a>
  <a class="Card A" href="#active">белсендіру сілтемесі</a>
</div>
```

## стиль күйі

### `.NavALi`

Сыртқы сілтемелер тізімі көлденең орналасуды және аралықты қолданады.

### `.Card`

Балаларды байланыстыру.

### `.A`

Іске қосылған кезде бөлектеу әсері қолданылады.
