# Styl listy łączy paska nawigacji

- Elementy podrzędne łącza układu poziomego
- Zawiera stany domyślne, najechanie i aktywne
- Wbudowane animacje przejść i kliknięcia, aby powiększyć

## Skorzystaj z wersji demonstracyjnej

```html
<div class="NavALi">
  <a class="Card" href="#home">Połączyć</a>
  <a class="Card A" href="#active">link aktywacyjny</a>
</div>
```

## stan stylu

### `.NavALi`

Zewnętrzna lista linków stosuje układ poziomy i odstępy.

### `.Card`

Połącz dzieci.

### `.A`

Po włączeniu zastosowany zostanie efekt podświetlenia.
