# Linklistestil for navigasjonslinjen

- Horisontal layout link barn
- Inneholder standard, hover og aktive tilstander
- Innebygd overgang og klikk-for-zoom-animasjoner

## Bruk demoen

```html
<div class="NavALi">
  <a class="Card" href="#home">Link</a>
  <a class="Card A" href="#active">aktiveringslenke</a>
</div>
```

## stilstatus

### `.NavALi`

Den ytre lenkelisten bruker horisontal layout og avstand.

### `.Card`

Koble barn.

### `.A`

Når den er aktivert, brukes fremhevingseffekten.
