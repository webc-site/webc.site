# Navigatiounsbar Link Lëscht Stil

- Horizontal Layout Link Kanner
- Enthält Standard, Hover an aktiv Staaten
- Built-in Transitioun an Klick-zu-Zoom Animatiounen

## Benotzt d'Demo

```html
<div class="NavALi">
  <a class="Card" href="#home">Link</a>
  <a class="Card A" href="#active">Aktivéierungslink</a>
</div>
```

## Stil Status

### `.NavALi`

Déi baussenzeg Link Lëscht gëlt horizontal Layout an Abstand.

### `.Card`

Link Kanner.

### `.A`

Wann aktivéiert gëtt den Highlight Effekt applizéiert.
