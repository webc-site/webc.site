# Gaya dhaptar link bar navigasi

- Layout horisontal link anak
- Ngandhut status standar, hover lan aktif
- Animasi transisi lan klik-kanggo-zoom sing dibangun

## Gunakake demo

```html
<div class="NavALi">
  <a class="Card" href="#home">Link</a>
  <a class="Card A" href="#active">link aktivasi</a>
</div>
```

## status gaya

### `.NavALi`

Dhaptar pranala njaba ditrapake tata letak lan spasi horisontal.

### `.Card`

Link anak.

### `.A`

Nalika diaktifake, efek sorotan ditrapake.
