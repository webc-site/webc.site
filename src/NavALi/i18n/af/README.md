# Navigasiebalkskakellysstyl

- Horisontale uitleg skakel kinders
- Bevat verstek-, beweeg- en aktiewe toestande
- Ingeboude oorgang en klik-om-te-zoem-animasies

## Gebruik die demo

```html
<div class="NavALi">
  <a class="Card" href="#home">Skakel</a>
  <a class="Card A" href="#active">aktivering skakel</a>
</div>
```

## styl status

### `.NavALi`

Die buitenste skakellys pas horisontale uitleg en spasiëring toe.

### `.Card`

Koppel kinders.

### `.A`

Wanneer dit geaktiveer word, word die hoogtepunt-effek toegepas.
