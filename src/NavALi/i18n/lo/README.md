# ຮູບແບບລາຍຊື່ລິ້ງແຖບນໍາທາງ

- ຮູບແບບແນວນອນເຊື່ອມຕໍ່ເດັກນ້ອຍ
- ປະກອບດ້ວຍຄ່າເລີ່ມຕົ້ນ, ເລື່ອນ ແລະສະຖານະທີ່ເຄື່ອນໄຫວ
- ການປ່ຽນແປງໃນຕົວ ແລະອະນິເມຊັນຄລິກເພື່ອຊູມ

## ສະຖານະຮູບແບບ

### `.NavALi`

ລາຍຊື່ການເຊື່ອມຕໍ່ພາຍນອກໃຊ້ການຈັດວາງແນວນອນ ແລະໄລຍະຫ່າງ.

### `.Card`

ເດັກນ້ອຍເຊື່ອມຕໍ່.

### `.A`

ເມື່ອເປີດໃຊ້ງານແລ້ວ, ເອັບເຟັກໄຮໄລ້ຈະຖືກນຳໃຊ້.
