# ନାଭିଗେସନ୍ ବାର୍ ଲିଙ୍କ୍ ତାଲିକା ଶ style ଳୀ |

- ଭୂସମାନ୍ତର ଲେଆଉଟ୍ ପିଲାମାନଙ୍କୁ ଲିଙ୍କ୍ କରେ |
- ଡିଫଲ୍ଟ, ହୋଭର ଏବଂ ସକ୍ରିୟ ଅବସ୍ଥା ଧାରଣ କରେ |
- ବିଲ୍ଟ-ଇନ୍ ଟ୍ରାନ୍ସମିସନ୍ ଏବଂ କ୍ଲିକ୍-ଟୁ-ଜୁମ୍ ଆନିମେସନ୍ |

## ଶ style ଳୀ ସ୍ଥିତି

### `.NavALi`

ବାହ୍ୟ ଲିଙ୍କ୍ ତାଲିକା ଭୂସମାନ୍ତର ଲେଆଉଟ୍ ଏବଂ ବ୍ୟବଧାନ ପ୍ରୟୋଗ କରେ |

### `.Card`

ପିଲାମାନଙ୍କୁ ଲିଙ୍କ୍ କରନ୍ତୁ |

### `.A`

ସକ୍ରିୟ ହେଲେ, ହାଇଲାଇଟ୍ ଇଫେକ୍ଟ ପ୍ରୟୋଗ କରାଯାଏ |
