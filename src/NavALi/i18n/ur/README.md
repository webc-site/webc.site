# نیویگیشن بار لنک لسٹ اسٹائل

- افقی ترتیب لنک بچوں
- پہلے سے طے شدہ، ہوور اور فعال حالتوں پر مشتمل ہے۔
- بلٹ ان ٹرانزیشن اور کلک ٹو زوم اینیمیشن

## ڈیمو استعمال کریں۔

```html
<div class="NavALi">
  <a class="Card" href="#home">لنک</a>
  <a class="Card A" href="#active">ایکٹیویشن لنک</a>
</div>
```

## طرز کی حیثیت

### `.NavALi`

بیرونی لنک کی فہرست افقی ترتیب اور وقفہ کاری کا اطلاق کرتی ہے۔

### `.Card`

بچوں کو لنک کریں۔

### `.A`

چالو ہونے پر، نمایاں اثر لاگو ہوتا ہے۔
