# סגנון רשימת קישורים בסרגל הניווט

- פריסה אופקית קישור ילדים
- מכיל ברירת מחדל, מצבי ריחוף ופעילים
- אנימציות מעבר ולחיצה לזום מובנות

## השתמש בהדגמה

```html
<div class="NavALi">
  <a class="Card" href="#home">לְקַשֵׁר</a>
  <a class="Card A" href="#active">קישור הפעלה</a>
</div>
```

## מצב סגנון

### `.NavALi`

רשימת הקישורים החיצונית חלה פריסה אופקית ומרווחים.

### `.Card`

קישור ילדים.

### `.A`

כאשר מופעל, אפקט ההדגשה מוחל.
