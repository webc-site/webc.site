# Style de liste de liens de la barre de navigation

- Enfants de lien de disposition horizontale
- Contient les états par défaut, en survol et actif
- Transitions intégrées et animations par clic pour zoomer

## Utilisez la démo

```html
<div class="NavALi">
  <a class="Card" href="#home">Lien</a>
  <a class="Card A" href="#active">lien d'activation</a>
</div>
```

## statut de style

### `.NavALi`

La liste de liens externes applique une disposition et un espacement horizontaux.

### `.Card`

Reliez les enfants.

### `.A`

Lorsqu'il est activé, l'effet de surbrillance est appliqué.
