# پانل های کشو که از کناره بیرون می آیند

- به طور خودکار ساختارهای پیمایش داخلی ایجاد کنید
- روی ناحیه بیرونی کلیک کنید تا به طور خودکار جمع شود
- برای بستن، ژست چپ و راست را بکشید
- موقعیت کشویی جانبی در جهت چپ و راست
- رویدادهای باز و بسته را ارائه دهید

## از دمو استفاده کنید

```html
<button class="open-btn">باز کنید</button>

<c-side class="right">
  <div>محتویات کشو</div>
</c-side>

<script type="module">
import { open } from "webc.site/Side/_.js";

const btn = document.querySelector(".open-btn"),
  side = document.querySelector("c-side");

btn.onclick = () => open(side);
</script>
```

## رابط کاربری

### روش

- `open(el)`: تابع صادرات، کشو باز
- `close(el)`: تابع صادرات، کشو را ببندید
- `el.open()`: روش نمونه، کشو باز
- `el.close()`: روش نمونه، کشو را ببندید

### رویداد

- `open`: وقتی روشن می شود آتش می گیرد
- `close`: هنگام بسته شدن آتش می گیرد

## کلاس سبک

- `.right`: موقعیت یابی به سمت راست
- `.top`: موقعیت یابی را به بیرون بکشید
- `.bottom`: از موقعیت‌یابی خارج شوید
- (اگر تنظیم نشده باشد، پیش‌فرض این است که به سمت چپ حرکت کنید)
- `.open`: وضعیت باز
- `.H`: حالت پنهان
- `.T`: وضعیت انتقال انیمیشن
