# Панели ящиков, которые выдвигаются сбоку

- Автоматически создавать внутренние структуры прокрутки
- Нажмите внешнюю область, чтобы автоматически свернуть
- Проведите пальцем влево и вправо, чтобы закрыть
- Боковое скользящее позиционирование в левом и правом направлениях
- Проведение открытых и закрытых мероприятий

## Используйте демо-версию

```html
<button class="open-btn">Открыть</button>

<c-side class="right">
  <div>Содержимое ящика</div>
</c-side>

<script type="module">
import { open } from "webc.site/Side/_.js";

const btn = document.querySelector(".open-btn"),
  side = document.querySelector("c-side");

btn.onclick = () => open(side);
</script>
```

## интерфейс

### метод

- `open(el)`: функция экспорта, открытый ящик.
- `close(el)`: функция экспорта, закрытие ящика.
- `el.open()`: метод экземпляра, открытый ящик
- `el.close()`: метод экземпляра, закройте ящик

### событие

- `open`: срабатывает при включении.
- `close`: срабатывает при закрытии.

## класс стиля

- `.right`: положение выдвижения вправо
- `.top`: расположение выдвижения
- `.bottom`: выйти из позиционирования
- (Если не установлено, по умолчанию выдвигается влево)
- `.open`: статус открыт
- `.H`: скрытое состояние
- `.T`: состояние перехода анимации.
