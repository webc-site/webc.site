# వైపు నుండి జారిపోయే డ్రాయర్ ప్యానెల్లు

- అంతర్గత స్క్రోలింగ్ నిర్మాణాలను స్వయంచాలకంగా సృష్టించండి
- స్వయంచాలకంగా కుప్పకూలడానికి బయటి ప్రాంతంపై క్లిక్ చేయండి
- మూసివేయడానికి ఎడమ మరియు కుడి సంజ్ఞలను స్వైప్ చేయండి
- ఎడమ మరియు కుడి దిశలలో సైడ్ స్లైడింగ్ పొజిషనింగ్
- ఓపెన్ మరియు క్లోజ్ ఈవెంట్‌లను అందించండి

## డెమో ఉపయోగించండి

```html
<button class="open-btn">తెరవండి</button>

<c-side class="right">
  <div>డ్రాయర్ విషయాలు</div>
</c-side>

<script type="module">
import { open } from "webc.site/Side.js";

const btn = document.querySelector(".open-btn"),
  side = document.querySelector("c-side");

btn.onclick = () => open(side);
</script>
```

## ఇంటర్ఫేస్

### పద్ధతి

- `open(el)`: ఎగుమతి ఫంక్షన్, ఓపెన్ డ్రాయర్
- `close(el)`: ఎగుమతి ఫంక్షన్, డ్రాయర్‌ను మూసివేయండి
- `el.open()`: ఉదాహరణ పద్ధతి, ఓపెన్ డ్రాయర్
- `el.close()`: ఉదాహరణ పద్ధతి, డ్రాయర్‌ను మూసివేయండి

### సంఘటన

- `open`: ఆన్ చేసినప్పుడు మంటలు
- `close`: మూసివేయబడినప్పుడు మంటలు

## శైలి తరగతి

- `.right`: కుడివైపు స్లయిడ్ అవుట్ పొజిషనింగ్
- `.top`: స్లైడ్ అవుట్ పొజిషనింగ్
- `.bottom`: పొజిషనింగ్ నుండి స్లయిడ్
- (సెట్ చేయకపోతే, డిఫాల్ట్ ఎడమవైపుకు జారడం)
- `.open`: ఓపెన్ స్టేటస్
- `.H`: దాచిన స్థితి
- `.T`: యానిమేషన్ పరివర్తన స్థితి
