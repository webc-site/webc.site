# दराज के पैनल जो किनारे से बाहर की ओर खिसकते हैं

- स्वचालित रूप से आंतरिक स्क्रॉलिंग संरचनाएँ बनाएँ
- स्वचालित रूप से संक्षिप्त करने के लिए बाहरी क्षेत्र पर क्लिक करें
- बंद करने के लिए बाएँ और दाएँ इशारा स्वाइप करें
- बाएँ और दाएँ दिशाओं में साइड स्लाइडिंग स्थिति
- खुले और बंद ईवेंट प्रदान करें

## डेमो का प्रयोग करें

```html
<button class="open-btn">खुला</button>

<c-side class="right">
  <div>दराज की सामग्री</div>
</c-side>

<script type="module">
import { open } from "webc.site/Side.js";

const btn = document.querySelector(".open-btn"),
  side = document.querySelector("c-side");

btn.onclick = () => open(side);
</script>
```

## इंटरफ़ेस

### तरीका

- `open(el)`: निर्यात फ़ंक्शन, खुला दराज
- `close(el)`: निर्यात फ़ंक्शन, दराज बंद करें
- `el.open()`: उदाहरण विधि, खुली दराज
- `el.close()`: उदाहरण विधि, दराज बंद करें

### आयोजन

- `open`: चालू होने पर आग लग जाती है
- `close`: बंद होने पर आग लग जाती है

## शैली वर्ग

- `.right`: दाईं ओर स्लाइड आउट स्थिति
- `.top`: स्लाइड आउट पोजीशनिंग
- `.bottom`: स्थिति से बाहर स्लाइड करें
- (यदि सेट नहीं है, तो डिफ़ॉल्ट बाईं ओर स्लाइड करना है)
- `.open`: खुली स्थिति
- `.H`: छिपी हुई स्थिति
- `.T`: एनीमेशन संक्रमण स्थिति
