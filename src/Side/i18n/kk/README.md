# Бүйірден сырғып шығатын тартпа панельдері

- Ішкі айналдыру құрылымдарын автоматты түрде жасаңыз
- Автоматты түрде жию үшін сыртқы аймақты басыңыз
- Жабу үшін солға және оңға сырғытыңыз
- Бүйірлік жылжымалы орналасу сол және оң бағытта
- Ашық және жабық оқиғаларды қамтамасыз етіңіз

## Демонстрацияны пайдаланыңыз

```html
<button class="open-btn">Ашық</button>

<c-side class="right">
  <div>Тартпаның мазмұны</div>
</c-side>

<script type="module">
import { open } from "webc.site/Side/_.js";

const btn = document.querySelector(".open-btn"),
  side = document.querySelector("c-side");

btn.onclick = () => open(side);
</script>
```

## интерфейс

### әдіс

- `open(el)`: Экспорттау функциясы, тартпаны ашу
- `close(el)`: Экспорттау функциясы, тартпаны жабу
- `el.open()`: дана әдісі, тартпаны ашу
- `el.close()`: дана әдісі, тартпаны жабыңыз

### оқиға

- `open`: қосулы кезде жанады
- `close`: жабылған кезде жанады

## стиль сыныбы

- `.right`: оң жаққа сырғыту орнын анықтау
- `.top`: орналасуды сырғыту
- `.bottom`: орналастырудан сырғыту
- (Орнатылмаған болса, әдепкі солға сырғыту болып табылады)
- `.open`: ашық күй
- `.H`: жасырын күй
- `.T`: анимацияның өту күйі
