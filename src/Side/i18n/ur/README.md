# دراز کے پینل جو سائیڈ سے باہر نکلتے ہیں۔

- خودکار طور پر اندرونی سکرولنگ ڈھانچے بنائیں
- خودکار طور پر گرنے کے لیے بیرونی حصے پر کلک کریں۔
- بند کرنے کے لیے بائیں اور دائیں اشارے کو سوائپ کریں۔
- بائیں اور دائیں سمتوں میں سائیڈ سلائیڈنگ پوزیشننگ
- کھلے اور قریبی واقعات فراہم کریں۔

## ڈیمو استعمال کریں۔

```html
<button class="open-btn">کھولیں۔</button>

<c-side class="right">
  <div>دراز کا مواد</div>
</c-side>

<script type="module">
import { open } from "webc.site/Side/_.js";

const btn = document.querySelector(".open-btn"),
  side = document.querySelector("c-side");

btn.onclick = () => open(side);
</script>
```

## انٹرفیس

### طریقہ

- `open(el)`: ایکسپورٹ فنکشن، دراز کھولیں۔
- `close(el)`: ایکسپورٹ فنکشن، بند دراز
- `el.open()`: مثال کا طریقہ، دراز کھولیں۔
- `el.close()`: مثال کا طریقہ، دراز کو بند کریں۔

### واقعہ

- `open`: آن ہونے پر آگ لگ جاتی ہے۔
- `close`: بند ہونے پر آگ لگ جاتی ہے۔

## سٹائل کلاس

- `.right`: دائیں سلائیڈ آؤٹ پوزیشننگ
- `.top`: سلائیڈ آؤٹ پوزیشننگ
- `.bottom`: پوزیشننگ سے باہر سلائیڈ
- (اگر سیٹ نہیں ہے تو، ڈیفالٹ بائیں طرف سلائیڈ کرنا ہے)
- `.open`: کھلی حیثیت
- `.H`: پوشیدہ حالت
- `.T`: حرکت پذیری کی منتقلی کی حالت
