# ألواح الأدراج التي تنزلق للخارج من الجانب

- إنشاء هياكل التمرير الداخلية تلقائيًا
- انقر فوق المنطقة الخارجية لطيها تلقائيًا
- اسحب لفتة اليسار واليمين للإغلاق
- وضع الانزلاق الجانبي في الاتجاهين الأيسر والأيمن
- توفير الأحداث المفتوحة والمغلقة

## استخدم العرض التوضيحي

```html
<button class="open-btn">يفتح</button>

<c-side class="right">
  <div>محتويات الدرج</div>
</c-side>

<script type="module">
import { open } from "webc.site/Side/_.js";

const btn = document.querySelector(".open-btn"),
  side = document.querySelector("c-side");

btn.onclick = () => open(side);
</script>
```

## واجهة

### طريقة

- `open(el)`: وظيفة التصدير، الدرج المفتوح
- `close(el)`: وظيفة التصدير، إغلاق الدرج
- `el.open()`: طريقة المثيل، الدرج المفتوح
- `el.close()`: طريقة المثيل، أغلق الدرج

### حدث

- `open`: يتم تشغيله عند تشغيله
- `close`: يتم تشغيله عند إغلاقه

## فئة النمط

- `.right`: وضع الشريحة للخارج لليمين
- `.top`: وضع الشريحة للخارج
- `.bottom`: انزلق خارج الموضع
- (إذا لم يتم تعيينه، فإن الإعداد الافتراضي هو التمرير للخارج لليسار)
- `.open`: الحالة مفتوحة
- `.H`: الحالة المخفية
- `.T`: حالة انتقال الرسوم المتحركة
