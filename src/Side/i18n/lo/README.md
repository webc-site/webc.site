# ແຖບລິ້ນຊັກທີ່ເລື່ອນອອກມາຈາກດ້ານຂ້າງ

- ສ້າງໂຄງສ້າງເລື່ອນພາຍໃນໂດຍອັດຕະໂນມັດ
- ຄລິກພື້ນທີ່ນອກເພື່ອຍຸບອັດຕະໂນມັດ
- ປັດອອກທ່າທາງຊ້າຍ ແລະຂວາເພື່ອປິດ
- ການຈັດຕໍາແຫນ່ງເລື່ອນດ້ານຂ້າງໃນທິດທາງຊ້າຍແລະຂວາ
- ໃຫ້ເຫດການເປີດແລະປິດ

## ໃຊ້ຕົວຢ່າງ

```html
<button class="open-btn">ເປີດ</button>

<c-side class="right">
  <div>ເນື້ອໃນຂອງລິ້ນຊັກ</div>
</c-side>

<script type="module">
import { open } from "webc.site/Side/_.js";

const btn = document.querySelector(".open-btn"),
  side = document.querySelector("c-side");

btn.onclick = () => open(side);
</script>
```

## ການໂຕ້ຕອບ

### ວິທີການ

- `open(el)`: ຟັງຊັນສົ່ງອອກ, ເປີດລິ້ນຊັກ
- `close(el)`: ຟັງຊັນສົ່ງອອກ, ປິດລິ້ນຊັກ
- `el.open()`: ວິທີການຕົວຢ່າງ, ເປີດລິ້ນຊັກ
- `el.close()`: ວິທີການຕົວຢ່າງ, ປິດລິ້ນຊັກ

### ເຫດການ

- `open`: ໄຟໄໝ້ເມື່ອເປີດ
- `close`: ໄຟໄໝ້ເມື່ອປິດ

## ຫ້ອງຮຽນແບບ

- `.right`: ເລື່ອນອອກຂວາຈັດຕຳແໜ່ງ
- `.top`: ການວາງຕຳແໜ່ງເລື່ອນອອກ
- `.bottom`: ເລື່ອນອອກຈາກຕຳແໜ່ງ
- (ຖ້າ​ບໍ່​ໄດ້​ຕັ້ງ​ຄ່າ, ຄ່າ​ເລີ່ມ​ຕົ້ນ​ແມ່ນ​ໃຫ້​ເລື່ອນ​ອອກ​ໄປ​ຊ້າຍ​)
- `.open`: ສະຖານະເປີດ
- `.H`: ສະຖານະທີ່ເຊື່ອງໄວ້
- `.T`: ສະຖານະການປ່ຽນແປງອະນິເມຊັນ
