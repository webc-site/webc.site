# แผงลิ้นชักที่เลื่อนออกจากด้านข้าง

- สร้างโครงสร้างการเลื่อนภายในโดยอัตโนมัติ
- คลิกที่พื้นที่ด้านนอกเพื่อยุบโดยอัตโนมัติ
- ปัดไปทางซ้ายและขวาเพื่อปิด
- ตำแหน่งเลื่อนด้านข้างในทิศทางซ้ายและขวา
- จัดให้มีกิจกรรมเปิดและปิด

## ใช้การสาธิต

```html
<button class="open-btn">เปิด</button>

<c-side class="right">
  <div>เนื้อหาในลิ้นชัก</div>
</c-side>

<script type="module">
import { open } from "webc.site/Side.js";

const btn = document.querySelector(".open-btn"),
  side = document.querySelector("c-side");

btn.onclick = () => open(side);
</script>
```

## อินเตอร์เฟซ

### วิธี

- `open(el)`: ฟังก์ชั่นส่งออก เปิดลิ้นชัก
- `close(el)`: ฟังก์ชั่นส่งออก ปิดลิ้นชัก
- `el.open()`: วิธีการอินสแตนซ์ เปิดลิ้นชัก
- `el.close()`: วิธีการอินสแตนซ์ ปิดลิ้นชัก

### เหตุการณ์

- `open`: เริ่มทำงานเมื่อเปิดเครื่อง
- `close`: ไฟไหม้เมื่อปิด

## คลาสสไตล์

- `.right`: เลื่อนตำแหน่งออกทางขวา
- `.top`: เลื่อนตำแหน่งออก
- `.bottom`: เลื่อนออกจากตำแหน่ง
- (หากไม่ได้ตั้งค่าไว้ ค่าเริ่มต้นคือ เลื่อนออกทางซ้าย)
- `.open`: สถานะเปิด
- `.H`: สถานะที่ซ่อนอยู่
- `.T`: สถานะการเปลี่ยนภาพเคลื่อนไหว
