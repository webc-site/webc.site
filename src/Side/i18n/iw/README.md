# לוחות מגירות המחליקים החוצה מהצד

- צור באופן אוטומטי מבני גלילה פנימיים
- לחץ על האזור החיצוני כדי להתכווץ אוטומטית
- החלק תנועות ימינה ושמאלה כדי לסגור
- מיקום הזזה בצד לכיוונים שמאלה וימינה
- לספק אירועים פתוחים וסגורים

## השתמש בהדגמה

```html
<button class="open-btn">לִפְתוֹחַ</button>

<c-side class="right">
  <div>תכולת מגירה</div>
</c-side>

<script type="module">
import { open } from "webc.site/Side.js";

const btn = document.querySelector(".open-btn"),
  side = document.querySelector("c-side");

btn.onclick = () => open(side);
</script>
```

## מִמְשָׁק

### שִׁיטָה

- `open(el)`: פונקציית ייצוא, פתח את המגירה
- `close(el)`: פונקציית ייצוא, סגור את המגירה
- `el.open()`: שיטת מופע, מגירה פתוחה
- `el.close()`: שיטת מופע, סגור את המגירה

### מִקרֶה

- `open`: מופעל כאשר הוא מופעל
- `close`: יורה כשהוא סגור

## שיעור סגנון

- `.right`: מיקום החלקה ימינה החוצה
- `.top`: מיקום החלק החוצה
- `.bottom`: החלק אל מחוץ למיקום
- (אם לא מוגדר, ברירת המחדל היא להחליק החוצה שמאלה)
- `.open`: מצב פתוח
- `.H`: מצב מוסתר
- `.T`: מצב מעבר הנפשה
