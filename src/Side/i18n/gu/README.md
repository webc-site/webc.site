# ડ્રોઅર પેનલ્સ કે જે બાજુમાંથી બહાર નીકળે છે

- આપમેળે આંતરિક સ્ક્રોલિંગ સ્ટ્રક્ચર્સ બનાવો
- આપમેળે સંકુચિત થવા માટે બાહ્ય વિસ્તાર પર ક્લિક કરો
- બંધ કરવા માટે ડાબે અને જમણા હાવભાવને સ્વાઇપ કરો
- ડાબી અને જમણી દિશામાં સાઇડ સ્લાઇડિંગ સ્થિતિ
- ખુલ્લી અને બંધ ઘટનાઓ પ્રદાન કરો

## ડેમોનો ઉપયોગ કરો

```html
<button class="open-btn">ખોલો</button>

<c-side class="right">
  <div>ડ્રોઅર સમાવિષ્ટો</div>
</c-side>

<script type="module">
import { open } from "webc.site/Side/_.js";

const btn = document.querySelector(".open-btn"),
  side = document.querySelector("c-side");

btn.onclick = () => open(side);
</script>
```

## ઈન્ટરફેસ

### પદ્ધતિ

- `open(el)`: નિકાસ કાર્ય, ડ્રોઅર ખોલો
- `close(el)`: નિકાસ કાર્ય, ડ્રોઅર બંધ કરો
- `el.open()`: ઉદાહરણ પદ્ધતિ, ડ્રોઅર ખોલો
- `el.close()`: ઉદાહરણ પદ્ધતિ, ડ્રોઅર બંધ કરો

### ઘટના

- `open`: જ્યારે ચાલુ હોય ત્યારે આગ લાગે છે
- `close`: બંધ હોય ત્યારે આગ લાગે છે

## શૈલી વર્ગ

- `.right`: જમણી સ્લાઇડ આઉટ પોઝિશનિંગ
- `.top`: સ્લાઇડ આઉટ પોઝિશનિંગ
- `.bottom`: સ્થિતિની બહાર સ્લાઇડ
- (જો સેટ કરેલ નથી, તો ડિફોલ્ટ ડાબી બાજુ સ્લાઇડ કરવાનું છે)
- `.open`: ઓપન સ્ટેટસ
- `.H`: છુપાયેલ સ્થિતિ
- `.T`: એનિમેશન સંક્રમણ સ્થિતિ
