# छेउबाट बाहिर सर्ने दराज प्यानलहरू

- स्वचालित रूपमा आन्तरिक स्क्रोलिङ संरचनाहरू सिर्जना गर्नुहोस्
- स्वचालित रूपमा पतन गर्न बाहिरी क्षेत्रमा क्लिक गर्नुहोस्
- बन्द गर्न बायाँ र दायाँ इशारा स्वाइप गर्नुहोस्
- बायाँ र दायाँ दिशाहरूमा साइड स्लाइडिङ स्थिति
- खुला र बन्द घटनाहरू प्रदान गर्नुहोस्

## डेमो प्रयोग गर्नुहोस्

```html
<button class="open-btn">खोल्नुहोस्</button>

<c-side class="right">
  <div>दराज सामग्री</div>
</c-side>

<script type="module">
import { open } from "webc.site/Side/_.js";

const btn = document.querySelector(".open-btn"),
  side = document.querySelector("c-side");

btn.onclick = () => open(side);
</script>
```

## इन्टरफेस

### विधि

- `open(el)`: निर्यात प्रकार्य, दराज खोल्नुहोस्
- `close(el)`: निर्यात प्रकार्य, दराज बन्द गर्नुहोस्
- `el.open()`: उदाहरण विधि, दराज खोल्नुहोस्
- `el.close()`: उदाहरण विधि, दराज बन्द गर्नुहोस्

### घटना

- `open`: सक्रिय हुँदा आगो
- `close`: बन्द हुँदा आगो

## शैली वर्ग

- `.right`: दायाँ स्लाइड आउट स्थिति
- `.top`: स्लाइड आउट स्थिति
- `.bottom`: स्थितिको बाहिर स्लाइड
- (यदि सेट छैन भने, पूर्वनिर्धारित बायाँ बाहिर स्लाइड गर्न हो)
- `.open`: खुला स्थिति
- `.H`: लुकेको अवस्था
- `.T`: एनिमेसन संक्रमण अवस्था
