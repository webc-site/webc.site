# පැත්තෙන් ලිස්සා යන ලාච්චු පැනල්

- අභ්‍යන්තර අනුචලන ව්‍යුහයන් ස්වයංක්‍රීයව සාදන්න
- ස්වයංක්‍රීයව කඩා වැටීමට පිටත ප්‍රදේශය ක්ලික් කරන්න
- වැසීමට වම් සහ දකුණු අභිනය ස්වයිප් කරන්න
- වම් සහ දකුණු දිශාවන්හි පැති ස්ලයිඩින් ස්ථානගත කිරීම
- විවෘත හා සමීප සිදුවීම් ලබා දෙන්න

## Demo භාවිතා කරන්න

```html
<button class="open-btn">විවෘත කරන්න</button>

<c-side class="right">
  <div>ලාච්චු අන්තර්ගතය</div>
</c-side>

<script type="module">
import { open } from "webc.site/Side/_.js";

const btn = document.querySelector(".open-btn"),
  side = document.querySelector("c-side");

btn.onclick = () => open(side);
</script>
```

## අතුරු මුහුණත

### ක්රමය

- `open(el)`: අපනයන කාර්යය, විවෘත ලාච්චුව
- `close(el)`: අපනයන කාර්යය, ලාච්චුව වසන්න
- `el.open()`: උදාහරණ ක්‍රමය, විවෘත ලාච්චුව
- `el.close()`: උදාහරණ ක්‍රමය, ලාච්චුව වසන්න

### සිදුවීම

- `open`: ක්‍රියාත්මක කළ විට ගිනි ගනී
- `close`: වසා ඇති විට ගිනි ගනී

## ශෛලිය පන්තිය

- `.right`: ස්ථානගත කිරීම දකුණට ලිස්සා යන්න
- `.top`: ස්ථානගත කිරීම පිටතට ලිස්සා යාම
- `.bottom`: ස්ථානගත කිරීමෙන් පිටතට ලිස්සා යන්න
- (සකසා නොමැති නම්, පෙරනිමිය වමට ලිස්සා යාමයි)
- `.open`: විවෘත තත්ත්වය
- `.H`: සැඟවුණු තත්වය
- `.T`: සජීවිකරණ සංක්‍රාන්ති තත්ත්වය
