# பக்கவாட்டில் இருந்து வெளியேறும் டிராயர் பேனல்கள்

- உள் ஸ்க்ரோலிங் கட்டமைப்புகளை தானாக உருவாக்கவும்
- தானாக சரிவதற்கு வெளிப்புறப் பகுதியைக் கிளிக் செய்யவும்
- மூடுவதற்கு இடது மற்றும் வலது சைகையை ஸ்வைப் செய்யவும்
- இடது மற்றும் வலது திசைகளில் பக்க நெகிழ் நிலைப்பாடு
- திறந்த மற்றும் நெருக்கமான நிகழ்வுகளை வழங்கவும்

## டெமோவைப் பயன்படுத்தவும்

```html
<button class="open-btn">திற</button>

<c-side class="right">
  <div>அலமாரியின் உள்ளடக்கங்கள்</div>
</c-side>

<script type="module">
import { open } from "webc.site/Side.js";

const btn = document.querySelector(".open-btn"),
  side = document.querySelector("c-side");

btn.onclick = () => open(side);
</script>
```

## இடைமுகம்

### முறை

- `open(el)`: ஏற்றுமதி செயல்பாடு, திறந்த அலமாரி
- `close(el)`: ஏற்றுமதி செயல்பாடு, அலமாரியை மூடவும்
- `el.open()`: நிகழ்வு முறை, திறந்த அலமாரி
- `el.close()`: உதாரண முறை, டிராயரை மூடவும்

### நிகழ்வு

- `open`: ஆன் செய்யும்போது தீப்பிடிக்கும்
- `close`: மூடப்படும் போது தீப்பிடிக்கும்

## பாணி வகுப்பு

- `.right`: வலது ஸ்லைடு அவுட் பொசிஷனிங்
- `.top`: ஸ்லைடு அவுட் பொசிஷனிங்
- `.bottom`: நிலைநிறுத்தலுக்கு வெளியே ஸ்லைடு
- (அமைக்கப்படவில்லை என்றால், இடதுபுறமாக ஸ்லைடு செய்வதே இயல்பு)
- `.open`: திறந்த நிலை
- `.H`: மறைக்கப்பட்ட நிலை
- `.T`: அனிமேஷன் மாற்றம் நிலை
