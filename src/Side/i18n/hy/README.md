# Գզրոցների վահանակներ, որոնք դուրս են սահում կողքից

- Ավտոմատ կերպով ստեղծել ներքին ոլորման կառուցվածքներ
- Սեղմեք արտաքին տարածքը՝ ինքնաբերաբար փլուզվելու համար
- Փակելու համար սահեցրեք ձախ և աջ ժեստը
- Կողային սահող դիրքավորում ձախ և աջ ուղղություններով
- Տրամադրել բաց և փակ միջոցառումներ

## Օգտագործեք ցուցադրությունը

```html
<button class="open-btn">Բաց</button>

<c-side class="right">
  <div>Գզրոցի պարունակությունը</div>
</c-side>

<script type="module">
import { open } from "webc.site/Side.js";

const btn = document.querySelector(".open-btn"),
  side = document.querySelector("c-side");

btn.onclick = () => open(side);
</script>
```

## ինտերֆեյս

### մեթոդ

- `open(el)`. Արտահանման գործառույթ, բաց դարակը
- `close(el)`. Արտահանման գործառույթ, փակել դարակը
- `el.open()`՝ օրինակի մեթոդ, բաց դարակ
- `el.close()`: օրինակի մեթոդ, փակիր դարակը

### իրադարձություն

- `open`. վառվում է, երբ միացված է
- `close`. բռնկվում է փակ վիճակում

## ոճի դաս

- `.right`: աջ սահեցրեք դիրքավորումը
- `.top`: Դիրքորոշումը դուրս սահեցրեք
- `.bottom`. սահեցրեք դիրքավորումից
- (Եթե կարգավորված չէ, կանխադրված է ձախ սահելը)
- `.open`՝ բաց կարգավիճակ
- `.H`՝ թաքնված վիճակ
- `.T`՝ անիմացիայի անցումային վիճակ
