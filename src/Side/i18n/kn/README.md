# ಬದಿಯಿಂದ ಸ್ಲೈಡ್ ಮಾಡುವ ಡ್ರಾಯರ್ ಫಲಕಗಳು

- ಆಂತರಿಕ ಸ್ಕ್ರೋಲಿಂಗ್ ರಚನೆಗಳನ್ನು ಸ್ವಯಂಚಾಲಿತವಾಗಿ ರಚಿಸಿ
- ಸ್ವಯಂಚಾಲಿತವಾಗಿ ಕುಸಿಯಲು ಹೊರಗಿನ ಪ್ರದೇಶವನ್ನು ಕ್ಲಿಕ್ ಮಾಡಿ
- ಮುಚ್ಚಲು ಎಡ ಮತ್ತು ಬಲ ಗೆಸ್ಚರ್ ಅನ್ನು ಸ್ವೈಪ್ ಮಾಡಿ
- ಎಡ ಮತ್ತು ಬಲ ದಿಕ್ಕುಗಳಲ್ಲಿ ಸೈಡ್ ಸ್ಲೈಡಿಂಗ್ ಸ್ಥಾನೀಕರಣ
- ಮುಕ್ತ ಮತ್ತು ನಿಕಟ ಘಟನೆಗಳನ್ನು ಒದಗಿಸಿ

## ಡೆಮೊ ಬಳಸಿ

```html
<button class="open-btn">ತೆರೆಯಿರಿ</button>

<c-side class="right">
  <div>ಡ್ರಾಯರ್ ವಿಷಯಗಳು</div>
</c-side>

<script type="module">
import { open } from "webc.site/Side/_.js";

const btn = document.querySelector(".open-btn"),
  side = document.querySelector("c-side");

btn.onclick = () => open(side);
</script>
```

## ಇಂಟರ್ಫೇಸ್

### ವಿಧಾನ

- `open(el)`: ರಫ್ತು ಕಾರ್ಯ, ತೆರೆದ ಡ್ರಾಯರ್
- `close(el)`: ರಫ್ತು ಕಾರ್ಯ, ಡ್ರಾಯರ್ ಅನ್ನು ಮುಚ್ಚಿ
- `el.open()`: ನಿದರ್ಶನ ವಿಧಾನ, ತೆರೆದ ಡ್ರಾಯರ್
- `el.close()`: ನಿದರ್ಶನ ವಿಧಾನ, ಡ್ರಾಯರ್ ಅನ್ನು ಮುಚ್ಚಿ

### ಘಟನೆ

- `open`: ಆನ್ ಮಾಡಿದಾಗ ಉರಿಯುತ್ತದೆ
- `close`: ಮುಚ್ಚಿದಾಗ ಉರಿಯುತ್ತದೆ

## ಶೈಲಿ ವರ್ಗ

- `.right`: ಬಲಕ್ಕೆ ಸ್ಲೈಡ್ ಔಟ್ ಸ್ಥಾನೀಕರಣ
- `.top`: ಸ್ಥಾನೀಕರಣವನ್ನು ಸ್ಲೈಡ್ ಔಟ್ ಮಾಡಿ
- `.bottom`: ಸ್ಥಾನದಿಂದ ಹೊರಗೆ ಸ್ಲೈಡ್ ಮಾಡಿ
- (ಹೊಂದಿಸದಿದ್ದರೆ, ಎಡಕ್ಕೆ ಸ್ಲೈಡ್ ಮಾಡುವುದು ಡೀಫಾಲ್ಟ್ ಆಗಿದೆ)
- `.open`: ತೆರೆದ ಸ್ಥಿತಿ
- `.H`: ಗುಪ್ತ ಸ್ಥಿತಿ
- `.T`: ಅನಿಮೇಷನ್ ಪರಿವರ್ತನೆಯ ಸ್ಥಿತಿ
