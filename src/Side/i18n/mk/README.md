# Панели за фиоки што се лизгаат од страна

- Автоматски креирајте внатрешни структури за лизгање
- Кликнете на надворешната област за автоматски да се склопи
- Повлечете лево и десно гест за затворање
- Странично лизгачко позиционирање во лева и десна насока
- Обезбедете отворени и затворени настани

## Користете го демо

```html
<button class="open-btn">Отвори</button>

<c-side class="right">
  <div>Содржина на фиоката</div>
</c-side>

<script type="module">
import { open } from "webc.site/Side/_.js";

const btn = document.querySelector(".open-btn"),
  side = document.querySelector("c-side");

btn.onclick = () => open(side);
</script>
```

## интерфејс

### метод

- `open(el)`: Функција за извоз, отворена фиока
- `close(el)`: Функција за извоз, затворете ја фиоката
- `el.open()`: метод на пример, отворена фиока
- `el.close()`: метод на пример, затворете ја фиоката

### настан

- `open`: пали кога е вклучено
- `close`: пожари кога се затворени

## класа стил

- `.right`: позиционирање со лизгање надесно
- `.top`: позиционирање со лизгање
- `.bottom`: излизгајте од позиционирањето
- (Ако не е поставено, стандардно е да се лизне лево)
- `.open`: отворен статус
- `.H`: скриена состојба
- `.T`: состојба на транзиција на анимација
