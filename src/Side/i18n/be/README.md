# Панэлі скрынь, якія рассоўваюцца збоку

- Аўтаматычна ствараць унутраныя структуры пракруткі
- Націсніце вонкавую вобласць, каб аўтаматычна згарнуць
- Каб закрыць, правядзіце пальцам улева і ўправа
- Бакавое слізгальнае пазіцыянаванне ў левым і правым напрамках
- Забяспечце адкрытыя і закрытыя мерапрыемствы

## Выкарыстоўвайце дэма

```html
<button class="open-btn">Адкрыты</button>

<c-side class="right">
  <div>Змесціва скрыні</div>
</c-side>

<script type="module">
import { open } from "webc.site/Side.js";

const btn = document.querySelector(".open-btn"),
  side = document.querySelector("c-side");

btn.onclick = () => open(side);
</script>
```

## інтэрфейс

### метад

- `open(el)`: функцыя экспарту, адкрытая скрыня
- `close(el)`: функцыя экспарту, закрыццё скрыні
- `el.open()`: метад асобніка, адкрытая скрыня
- `el.close()`: метад асобніка, зачыніць скрыню

### мерапрыемства

- `open`: Спрацоўвае пры ўключэнні
- `close`: Загараецца, калі зачынены

## клас стылю

- `.right`: правае слізгаценне
- `.top`: размяшчэнне слізгацення
- `.bottom`: высунуць з пазіцыянавання
- (Калі не ўстаноўлена, па змаўчанні - ссоўванне налева)
- `.open`: адкрыты статус
- `.H`: схаваны стан
- `.T`: стан пераходу анімацыі
