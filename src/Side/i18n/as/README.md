# কাষৰ পৰা ওলাই যোৱা ড্ৰয়াৰ পেনেল

- স্বয়ংক্ৰিয়ভাৱে আভ্যন্তৰীণ স্ক্ৰলিং গঠন সৃষ্টি কৰক
- স্বয়ংক্ৰিয়ভাৱে ভাঙি যাবলৈ বাহিৰৰ অঞ্চলটো ক্লিক কৰক
- বন্ধ কৰিবলৈ বাওঁ আৰু সোঁ ইংগিত চুইপ কৰক
- বাওঁ আৰু সোঁ দিশত চাইড স্লাইডিং পজিচনিং
- মুকলি আৰু বন্ধ অনুষ্ঠানৰ ব্যৱস্থা কৰা

## ডেমো ব্যৱহাৰ কৰক

```html
<button class="open-btn">খোলা</button>

<c-side class="right">
  <div>ড্ৰয়াৰৰ বিষয়বস্তু</div>
</c-side>

<script type="module">
import { open } from "webc.site/Side/_.js";

const btn = document.querySelector(".open-btn"),
  side = document.querySelector("c-side");

btn.onclick = () => open(side);
</script>
```

## আন্তঃপৃষ্ঠ

### পদ্ধতি

- `open(el)`: ৰপ্তানি কাৰ্য্য, খোলা ড্ৰয়াৰ
- `close(el)`: ৰপ্তানি কাৰ্য্য, ড্ৰয়াৰ বন্ধ কৰক
- `el.open()`: উদাহৰণ পদ্ধতি, খোলা ড্ৰয়াৰ
- `el.close()`: উদাহৰণ পদ্ধতি, ড্ৰয়াৰ বন্ধ কৰক

### কাৰ্যক্ৰম

- `open`: অন কৰিলে জুই লাগে
- `close`: বন্ধ কৰিলে জুই লাগে

## শৈলী শ্ৰেণী

- `.right`: ৰাইট স্লাইড আউট পজিচনিং
- `.top`: পজিচনিং স্লাইড আউট
- `.bottom`: অৱস্থানৰ পৰা বাহিৰলৈ স্লাইড কৰক
- (যদি সংহতি কৰা হোৱা নাই, অবিকল্পিতভাৱে বাওঁফালে স্লাইড কৰা)
- `.open`: খোলা অৱস্থা
- `.H`: লুকাই থকা অৱস্থা
- `.T`: এনিমেচন পৰিৱৰ্তন অৱস্থা
