# უჯრის პანელები, რომლებიც სრიალებს გვერდიდან

- ავტომატურად შექმენით შიდა გადახვევის სტრუქტურები
- დააწკაპუნეთ გარე ზონაზე ავტომატურად დასაკეცად
- დახურვისთვის გადაფურცლეთ მარცხნივ და მარჯვნივ
- გვერდითი მოცურების განლაგება მარცხენა და მარჯვენა მიმართულებით
- უზრუნველყოს ღია და დახურული ღონისძიებები

## გამოიყენეთ დემო ვერსია

```html
<button class="open-btn">გახსენით</button>

<c-side class="right">
  <div>უჯრის შიგთავსი</div>
</c-side>

<script type="module">
import { open } from "webc.site/Side/_.js";

const btn = document.querySelector(".open-btn"),
  side = document.querySelector("c-side");

btn.onclick = () => open(side);
</script>
```

## ინტერფეისი

### მეთოდი

- `open(el)`: ექსპორტის ფუნქცია, გახსენით უჯრა
- `close(el)`: ექსპორტის ფუნქცია, უჯრის დახურვა
- `el.open()`: ინსტანციის მეთოდი, ღია უჯრა
- `el.close()`: მაგალითის მეთოდი, დახურეთ უჯრა

### ღონისძიება

- `open`: ჩართულია
- `close`: იწვის დახურვისას

## სტილის კლასი

- `.right`: პოზიციონირება მარჯვნივ
- `.top`: გამორთეთ პოზიციონირება
- `.bottom`: გასრიალეთ პოზიციიდან
- (თუ არ არის დაყენებული, ნაგულისხმევია მარცხნივ გაცურვა)
- `.open`: ღია სტატუსი
- `.H`: ფარული მდგომარეობა
- `.T`: ანიმაციის გარდამავალი მდგომარეობა
