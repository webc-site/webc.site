# ড্রয়ার প্যানেল যে পাশ থেকে স্লাইড আউট

- স্বয়ংক্রিয়ভাবে অভ্যন্তরীণ স্ক্রোলিং কাঠামো তৈরি করুন
- স্বয়ংক্রিয়ভাবে ভেঙে পড়তে বাইরের এলাকায় ক্লিক করুন
- বন্ধ করতে বাম এবং ডান অঙ্গভঙ্গি সোয়াইপ করুন
- বাম এবং ডান দিকে সাইড স্লাইডিং পজিশনিং
- খোলা এবং বন্ধ ইভেন্ট প্রদান

## ডেমো ব্যবহার করুন

```html
<button class="open-btn">খোলা</button>

<c-side class="right">
  <div>ড্রয়ার বিষয়বস্তু</div>
</c-side>

<script type="module">
import { open } from "webc.site/Side.js";

const btn = document.querySelector(".open-btn"),
  side = document.querySelector("c-side");

btn.onclick = () => open(side);
</script>
```

## ইন্টারফেস

### পদ্ধতি

- `open(el)`: এক্সপোর্ট ফাংশন, খোলা ড্রয়ার
- `close(el)`: এক্সপোর্ট ফাংশন, ড্রয়ার বন্ধ করুন
- `el.open()`: উদাহরণ পদ্ধতি, খোলা ড্রয়ার
- `el.close()`: উদাহরণ পদ্ধতি, ড্রয়ার বন্ধ করুন

### ঘটনা

- `open`: চালু হলে আগুন জ্বলে
- `close`: বন্ধ হলে আগুন লাগে

## শৈলী ক্লাস

- `.right`: ডান স্লাইড আউট অবস্থান
- `.top`: স্লাইড আউট পজিশনিং
- `.bottom`: অবস্থানের বাইরে স্লাইড
- (যদি সেট করা না থাকে, ডিফল্ট হল বাম দিকে স্লাইড করা)
- `.open`: খোলা অবস্থা
- `.H`: লুকানো অবস্থা
- `.T`: অ্যানিমেশন ট্রানজিশন অবস্থা
