# бһҖбһ¶бһҸ Capsule бһҠбҹӮбһӣбһҳбһ¶бһ“бһ–бҹ’бһҡбһҪбһүбһҒбһ¶бһ„бһҹбҹ’бһҸбһ¶бҹҶбһ”бһ„бҹ’бһ бһ¶бһүбһ“бҹ…бһҖбҹ’бһ“бһ»бһ„бһҖбһ¶бһҡбһ–бҹ’бһҷбһҪбһҡ

- бһҠбһ¶бһҖбҹӢбһ–бҹ’бһҡбһҪбһүбһҒбһ¶бһ„бһҹбҹ’бһҸбһ¶бҹҶ
- бһҖбһ¶бһҡбһ”бһ“бҹ’бһӣбһ·бһ…бһ–бһҺбҹҢбһҒбҹҖбһңбһҹбһҖбһҳбҹ’бһҳ
- бһ…бһ»бһ…бһҠбһҫбһҳбҹ’бһ”бһёбһ–бһ„бҹ’бһҡбһёбһҖбһҳбһҸбһ·бһҖбҹӮбһӣбһҳбҹ’бһў

## бһ”бҹ’бһҡбһҫбһҖбһ¶бһҡбһ”бһ„бҹ’бһ бһ¶бһү

```html
<main style="display:flex;gap:16px">
  <a class="Card A" href="#">
    <h2>бһ…бҹҶбһҺбһ„бһҮбһҫбһ„ бҹЎ</h2>
    <i>бһҖбһ¶бһҡбһ–бһ·бһ–бһҺбҹҢбһ“бһ¶ бҹЎ</i>
  </a>
  <a class="Card" href="#">
    <h2>бһ…бҹҶбһҺбһ„бһҮбһҫбһ„ бҹў</h2>
    <i>бһҖбһ¶бһҡбһ–бһ·бһ–бһҺбҹҢбһ“бһ¶ бҹў</i>
  </a>
</main>

<script type="module">
  import { On } from "webc.site/js/On.js";

  const el = document.querySelector("main"),
    CARD = ".Card",
    A = "A",
    CARD_A = CARD + "." + A;

  On(el, {
    click: (e) => {
      const card = e.target.closest(CARD);
      if (card) {
        const active = el.querySelector(CARD_A);
        if (active && active !== card) {
          active.classList.remove(A);
        }
        card.classList.add(A);
      }
    },
  });
</script>
```

## бһҗбҹ’бһ“бһ¶бһҖбҹӢбһҡбһ…бһ“бһ¶бһ”бҹҗбһ‘бҹ’бһҳ

- `.Card`бҹ– бһҖбһ¶бһҸбһҳбһјбһӣбһҠбҹ’бһӢбһ¶бһ“
- `.Card.A`бҹ– бһ’бҹ’бһңбһҫбһұбҹ’бһҷбһҖбһ¶бһҡбһ”бһ“бҹ’бһӣбһ·бһ…бһҹбһҖбһҳбҹ’бһҳ
- `> h2`бҹ– бһ…бҹҶбһҺбһ„бһҮбһҫбһ„
- `> i`бҹ– бһҖбһ¶бһҡбһ–бһ·бһ–бһҺбҹҢбһ“бһ¶
