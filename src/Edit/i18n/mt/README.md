# buttuna editja

- Fih buttuna tal-ikona tal-editjar tal-angolu rett
- Mouse-over fil-pront għall-editjar, appoġġ multi-lingwi

## Uża d-demo

```html
<c-edit href="https://github.com"></c-edit>
```

## Parametri tal-interface

It-tikketta tal-komponent hija `<c-edit>`, u l-indirizz tal-qabża jista' jiġi editjat permezz tal-konfigurazzjoni tal-attribut `href`.
