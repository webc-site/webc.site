# Lehekülje või dokumendi redigeerimise nupp

- Sisaldab täisnurga muutmise ikooni nuppu
- Sisseehitatud Pophoveri hõljutusviibad
- Toetab hüppe URL-i edastamist ja uue akna avamist

## Kasutage demo

```html
<c-edit href="https://github.com"></c-edit>

<script type="module">
import "webc.site/Edit/_.js";
</script>
```

## Liidese parameetrid

Komponendi silt on `<c-edit>` ja hüppeaadressi saab redigeerida `href` atribuudi konfiguratsiooni kaudu.
