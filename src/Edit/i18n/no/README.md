# rediger-knappen

- Inneholder rettvinklet redigeringsikonknapp
- Beskjed med musepekeren for redigering, flerspråklig støtte

## Bruk demoen

```html
<c-edit href="https://github.com"></c-edit>
```

## Grensesnittparametere

Komponentetiketten er `<c-edit>`, og hoppadressen kan redigeres gjennom `href` attributtkonfigurasjonen.
