# සංස්කරණ බොත්තම

- සෘජු කෝණ සංස්කරණ නිරූපක බොත්තම අඩංගු වේ
- සංස්කරණය සඳහා Mouse-over prompt, බහු භාෂා සහාය

## Demo භාවිතා කරන්න

```html
<c-edit href="https://github.com"></c-edit>

<script type="module">
import "webc.site/Edit/_.js";
</script>
```

## අතුරුමුහුණත් පරාමිතීන්

සංරචක ලේබලය `<c-edit>` වන අතර, `href` ගුණාංග වින්‍යාසය හරහා පැනීමේ ලිපිනය සංස්කරණය කළ හැක.
