# кнопка редагування

- Містить кнопку для редагування під прямим кутом
- Підказка для редагування при наведенні миші, підтримка кількох мов

## Використовуйте демо

```html
<c-edit href="https://github.com"></c-edit>

<script type="module">
import "webc.site/Edit/_.js";
</script>
```

## Параметри інтерфейсу

Мітка компонента — `<c-edit>`, а адресу переходу можна редагувати за допомогою конфігурації атрибута `href`.
