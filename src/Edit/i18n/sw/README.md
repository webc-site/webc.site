# kitufe cha kuhariri

- Ina kitufe cha aikoni ya pembe ya kulia
- Kidokezo cha panya cha kuhariri, usaidizi wa lugha nyingi

## Tumia onyesho

```html
<c-edit href="https://github.com"></c-edit>
```

## Vigezo vya interface

Lebo ya kipengele ni `<c-edit>`, na anwani ya kuruka inaweza kuhaririwa kupitia `href` usanidi wa sifa.
