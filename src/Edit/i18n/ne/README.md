# पृष्ठ वा कागजात सम्पादन बटन

- दायाँ कोण सम्पादन आइकन बटन समावेश गर्दछ
- बिल्ट-इन पोफोभर होभर प्रम्प्ट गर्दछ
- जम्प URL मा पास गर्न र नयाँ विन्डो खोल्न समर्थन गर्दछ

## डेमो प्रयोग गर्नुहोस्

```html
<c-edit href="https://github.com"></c-edit>

<script type="module">
import "webc.site/Edit/_.js";
</script>
```

## इन्टरफेस प्यारामिटरहरू

कम्पोनेन्ट लेबल `<c-edit>` हो, र जम्प ठेगाना `href` विशेषता कन्फिगरेसन मार्फत सम्पादन गर्न सकिन्छ।
