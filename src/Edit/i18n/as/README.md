# পৃষ্ঠা বা দস্তাবেজ সম্পাদনা বুটাম

- সোঁ কোণ সম্পাদনা আইকন বুটাম অন্তৰ্ভুক্ত
- বিল্ট-ইন Pophover হোভাৰ প্ৰমপ্ট
- জাম্প URL পাছ কৰা আৰু নতুন উইণ্ড' খোলা সমৰ্থন কৰে

## ডেমো ব্যৱহাৰ কৰক

```html
<c-edit href="https://github.com"></c-edit>

<script type="module">
import "webc.site/Edit/_.js";
</script>
```

## আন্তঃপৃষ্ঠ প্ৰাচলসমূহ

উপাদান লেবেল `<c-edit>`, আৰু জাম্প ঠিকনাক `href` বৈশিষ্ট্য বিন্যাসৰ যোগেদি সম্পাদনা কৰিব পাৰি।
