# தொகு பொத்தான்

- வலது கோணத்தில் திருத்து ஐகான் பட்டன் உள்ளது
- எடிட்டிங் செய்ய மவுஸ்-ஓவர் ப்ராம்ட், பல மொழி ஆதரவு

## டெமோவைப் பயன்படுத்தவும்

```html
<c-edit href="https://github.com"></c-edit>
```

## இடைமுக அளவுருக்கள்

கூறு லேபிள் `<c-edit>` ஆகும், மேலும் ஜம்ப் முகவரியை `href` பண்புக்கூறு உள்ளமைவு மூலம் திருத்தலாம்.
