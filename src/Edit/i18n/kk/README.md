# өңдеу түймесі

- Тік бұрышты өңдеу белгішесі түймешігін қамтиды
- Өңдеу үшін тінтуірдің үстінен жылжыту, көп тілді қолдау

## Демонстрацияны пайдаланыңыз

```html
<c-edit href="https://github.com"></c-edit>
```

## Интерфейс параметрлері

Құрамдас белгі `<c-edit>` және өту мекенжайын `href` атрибут конфигурациясы арқылы өңдеуге болады.
