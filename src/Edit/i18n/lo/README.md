- ມີປຸ່ມໄອຄອນແກ້ໄຂມຸມຂວາ
- ການເຕືອນເມົ້າຜ່ານການແກ້ໄຂ, ຮອງຮັບຫຼາຍພາສາ

## ໃຊ້ຕົວຢ່າງ

```html
<c-edit href="https://github.com"></c-edit>

<script type="module">

</script>
```

## ຕົວກໍານົດການໃນການໂຕ້ຕອບ

ປ້າຍກຳກັບອົງປະກອບແມ່ນ `<c-edit>`, ແລະທີ່ຢູ່ໂດດສາມາດແກ້ໄຂໄດ້ຜ່ານການຕັ້ງຄ່າຄຸນສົມບັດ `href`.
