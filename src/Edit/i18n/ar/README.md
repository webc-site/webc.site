# زر التحرير

- يحتوي على زر أيقونة تحرير الزاوية اليمنى
- موجه الماوس فوق للتحرير، ودعم متعدد اللغات

## استخدم العرض التوضيحي

```html
<c-edit href="https://github.com"></c-edit>

<script type="module">
import "webc.site/Edit/_.js";
</script>
```

## معلمات الواجهة

تسمية المكون هي `<c-edit>`، ويمكن تحرير عنوان الانتقال من خلال تكوين السمة `href`.
