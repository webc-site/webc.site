# redigera knappen

- Innehåller knapp för redigeringsikon i höger vinkel
- Mouse-over prompt för redigering, stöd för flera språk

## Använd demon

```html
<c-edit href="https://github.com"></c-edit>
```

## Gränssnittsparametrar

Komponentetiketten är `<c-edit>` och hoppadressen kan redigeras genom attributkonfigurationen `href`.
