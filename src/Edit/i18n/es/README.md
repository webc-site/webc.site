# botón editar

- Contiene el botón de icono de edición en ángulo recto
- Mensaje al pasar el mouse para editar, soporte en varios idiomas

## Usa la demostración

```html
<c-edit href="https://github.com"></c-edit>

<script type="module">
import "webc.site/Edit/_.js";
</script>
```

## Parámetros de interfaz

La etiqueta del componente es `<c-edit>` y la dirección de salto se puede editar mediante la configuración del atributo `href`.
