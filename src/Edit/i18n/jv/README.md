# tombol sunting

- Ngandhut tombol lambang suntingan sudut tengen
- Pandhuan mouse-over kanggo nyunting, dhukungan multi-basa

## Gunakake demo

```html
<c-edit href="https://github.com"></c-edit>

<script type="module">
import "webc.site/Edit/_.js";
</script>
```

## Paramèter antarmuka

Label komponen yaiku `<c-edit>`, lan alamat lompat bisa diowahi liwat konfigurasi atribut `href`.
