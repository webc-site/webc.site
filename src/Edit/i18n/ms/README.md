# butang edit

- Mengandungi butang ikon edit sudut kanan
- Gesaan alih tetikus untuk pengeditan, sokongan berbilang bahasa

## Gunakan demo

```html
<c-edit href="https://github.com"></c-edit>
```

## Parameter antara muka

Label komponen ialah `<c-edit>` dan alamat lompat boleh diedit melalui konfigurasi atribut `href`.
