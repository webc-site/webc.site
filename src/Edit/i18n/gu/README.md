# સંપાદિત કરો બટન

- જમણો કોણ સંપાદિત કરો આયકન બટન ધરાવે છે
- સંપાદન, બહુ-ભાષા સપોર્ટ માટે માઉસ-ઓવર પ્રોમ્પ્ટ

## ડેમોનો ઉપયોગ કરો

```html
<c-edit href="https://github.com"></c-edit>

<script type="module">
import "webc.site/Edit/_.js";
</script>
```

## ઇન્ટરફેસ પરિમાણો

ઘટક લેબલ `<c-edit>` છે, અને જમ્પ સરનામું `href` વિશેષતા રૂપરેખાંકન દ્વારા સંપાદિત કરી શકાય છે.
