# gumb za uređivanje

- Sadrži gumb ikone za uređivanje pod pravim kutom
- Upit za uređivanje mišem, podrška za više jezika

## Koristite demo

```html
<c-edit href="https://github.com"></c-edit>

<script type="module">
import "webc.site/Edit/_.js";
</script>
```

## Parametri sučelja

Oznaka komponente je `<c-edit>`, a adresa skoka može se uređivati ​​kroz konfiguraciju atributa `href`.
