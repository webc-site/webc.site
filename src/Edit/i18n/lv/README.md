# rediģēšanas poga

- Satur taisnā leņķa rediģēšanas ikonas pogu
- Novietojiet peles kursoru virs uzvednes rediģēšanai, vairāku valodu atbalsts

## Izmantojiet demonstrāciju

```html
<c-edit href="https://github.com"></c-edit>
```

## Interfeisa parametri

Komponenta etiķete ir `<c-edit>`, un pārejas adresi var rediģēt, izmantojot `href` atribūta konfigurāciju.
