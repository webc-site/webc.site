# botão editar

- Contém botão de ícone de edição em ângulo reto
- Solicitação do mouse para edição, suporte multilíngue

## Use a demonstração

```html
<c-edit href="https://github.com"></c-edit>

<script type="module">
import "webc.site/Edit/_.js";
</script>
```

## Parâmetros de interface

O rótulo do componente é `<c-edit>` e o endereço de salto pode ser editado por meio da configuração do atributo `href`.
