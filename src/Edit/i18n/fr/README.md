# bouton modifier

- Contient un bouton d'icône d'édition à angle droit
- Invite de passage de la souris pour l'édition, prise en charge multilingue

## Utilisez la démo

```html
<c-edit href="https://github.com"></c-edit>
```

## Paramètres d'interface

L'étiquette du composant est `<c-edit>` et l'adresse de saut peut être modifiée via la configuration de l'attribut `href`.
