# စာမျက်နှာ သို့မဟုတ် စာရွက်စာတမ်း တည်းဖြတ်ခြင်းခလုတ်

- ထောင့်မှန် တည်းဖြတ်ရန် အိုင်ကွန်ခလုတ် ပါရှိသည်။
- Built-in Pophover ခလုတ်များကို ရွှေ့ပါ။
- Jump URL တွင်ဖြတ်သန်းခြင်းနှင့် ဝင်းဒိုးအသစ်ဖွင့်ခြင်းကို ပံ့ပိုးပေးသည်။

## ဒီမိုကိုသုံးပါ။

```html
<c-edit href="https://github.com"></c-edit>

<script type="module">
import "webc.site/Edit/_.js";
</script>
```

## မျက်နှာပြင်ဘောင်များ

အစိတ်အပိုင်းအညွှန်းသည် `<c-edit>`ဖြစ်ပြီး၊ ခုန်လိပ်စာကို `href` ရည်ညွှန်းချက်ဖွဲ့စည်းမှုပုံစံဖြင့် တည်းဖြတ်နိုင်သည်။
