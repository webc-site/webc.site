# бутон за редактиране

- Съдържа бутон за редактиране на прав ъгъл
- Подкана за редактиране с мишката върху нея, многоезична поддръжка

## Използвайте демонстрацията

```html
<c-edit href="https://github.com"></c-edit>
```

## Параметри на интерфейса

Етикетът на компонента е `<c-edit>`, а адресът за преход може да се редактира чрез конфигурацията на атрибута `href`.
