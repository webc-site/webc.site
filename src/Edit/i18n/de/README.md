# Bearbeiten-Schaltfläche

- Enthält eine Schaltfläche zum Bearbeiten des rechten Winkels
- Eingabeaufforderung per Mouseover zur Bearbeitung, Unterstützung mehrerer Sprachen

## Nutzen Sie die Demo

```html
<c-edit href="https://github.com"></c-edit>

<script type="module">
import "webc.site/Edit/_.js";
</script>
```

## Schnittstellenparameter

Die Komponentenbezeichnung lautet `<c-edit>` und die Sprungadresse kann über die Attributkonfiguration `href` bearbeitet werden.
