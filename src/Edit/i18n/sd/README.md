# صفحو يا دستاويز جي تبديلي بٽڻ

- ساڄي زاويه ايڊٽ آئڪن بٽڻ تي مشتمل آهي
- تعمير ٿيل Pophover hover prompts
- جمپ URL ۾ پاس ڪرڻ ۽ نئين ونڊو کولڻ جي حمايت ڪري ٿو

## ڊيمو استعمال ڪريو

```html
<c-edit href="https://github.com"></c-edit>

<script type="module">
import "webc.site/Edit/_.js";
</script>
```

## انٽرفيس پيٽرول

جزو جو ليبل `<c-edit>` آهي، ۽ جمپ ايڊريس کي `href` خصوصيت جي ترتيب ذريعي تبديل ڪري سگهجي ٿو.
