# pindutang i-edit

- Naglalaman ng right angle edit icon button
- Mouse-over prompt para sa pag-edit, suporta sa maraming wika

## Gamitin ang demo

```html
<c-edit href="https://github.com"></c-edit>
```

## Mga parameter ng interface

Ang sangkap na label ay `<c-edit>`, at ang jump address ay maaaring i-edit sa pamamagitan ng `href` configuration ng attribute.
