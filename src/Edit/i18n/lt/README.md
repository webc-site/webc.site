# redagavimo mygtuką

- Yra dešiniojo kampo redagavimo piktogramos mygtukas
- Užveskite pelės žymeklį virš redagavimo, kelių kalbų palaikymas

## Naudokite demonstracinę versiją

```html
<c-edit href="https://github.com"></c-edit>
```

## Sąsajos parametrai

Komponento etiketė yra `<c-edit>`, o šuolio adresą galima redaguoti naudojant `href` atributo konfigūraciją.
