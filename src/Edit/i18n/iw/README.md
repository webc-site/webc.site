# כפתור עריכה

- מכיל כפתור סמל עריכה בזווית ישרה
- הנחיה של מעבר עכבר לעריכה, תמיכה בריבוי שפות

## השתמש בהדגמה

```html
<c-edit href="https://github.com"></c-edit>

<script type="module">
import "webc.site/Edit/_.js";
</script>
```

## פרמטרים של ממשק

תווית הרכיב היא `<c-edit>`, וניתן לערוך את כתובת הקפיצה באמצעות תצורת התכונה `href`.
