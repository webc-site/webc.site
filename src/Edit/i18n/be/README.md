# кнопка рэдагавання

- Змяшчае кнопку рэдагавання пад прамым вуглом
- Падказка для рэдагавання пры навядзенні мышы, падтрымка некалькіх моў

## Выкарыстоўвайце дэма

```html
<c-edit href="https://github.com"></c-edit>
```

## Параметры інтэрфейсу

Метка кампанента - `<c-edit>`, а адрас пераходу можна рэдагаваць праз канфігурацыю атрыбута `href`.
