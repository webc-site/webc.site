# botón editar

- Contén o botón de icona de edición en ángulo recto
- Solicitude de paso do rato para editar, compatibilidade con varios idiomas

## Use a demostración

```html
<c-edit href="https://github.com"></c-edit>

<script type="module">
import "webc.site/Edit/_.js";
</script>
```

## Parámetros da interface

A etiqueta do compoñente é `<c-edit>` e o enderezo de salto pódese editar mediante a configuración do atributo `href`.
