# düzenle düğmesi

- Dik açılı düzenleme simgesi düğmesini içerir
- Düzenleme için fareyle üzerine gelme istemi, çoklu dil desteği

## Demoyu kullanın

```html
<c-edit href="https://github.com"></c-edit>

<script type="module">
import "webc.site/Edit/_.js";
</script>
```

## Arayüz parametreleri

Bileşen etiketi `<c-edit>` şeklindedir ve atlama adresi `href` öznitelik yapılandırması aracılığıyla düzenlenebilir.
