# ปุ่มแก้ไข

- มีปุ่มแก้ไขไอคอนมุมขวา
- พรอมต์เมาส์เหนือสำหรับการแก้ไข รองรับหลายภาษา

## ใช้การสาธิต

```html
<c-edit href="https://github.com"></c-edit>

<script type="module">
import "webc.site/Edit/_.js";
</script>
```

## พารามิเตอร์อินเทอร์เฟซ

ป้ายกำกับส่วนประกอบคือ `<c-edit>` และที่อยู่การกระโดดสามารถแก้ไขได้ผ่านการกำหนดค่าแอตทริบิวต์ `href`
