# butonul de editare

- Conține butonul pictogramă de editare în unghi drept
- Solicitare trecerea mouse-ului pentru editare, suport în mai multe limbi

## Utilizați demonstrația

```html
<c-edit href="https://github.com"></c-edit>
```

## Parametrii interfeței

Eticheta componentei este `<c-edit>`, iar adresa de salt poate fi editată prin configurația atributului `href`.
