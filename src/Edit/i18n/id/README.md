# tombol edit

- Berisi tombol ikon edit sudut kanan
- Prompt mouse-over untuk mengedit, dukungan multi-bahasa

## Gunakan demonya

```html
<c-edit href="https://github.com"></c-edit>

<script type="module">
import "webc.site/Edit/_.js";
</script>
```

## Parameter antarmuka

Label komponennya adalah `<c-edit>`, dan alamat lompat dapat diedit melalui konfigurasi atribut `href`.
