# botó d'edició

- Conté un botó d'icona d'edició en angle recte
- Petició del ratolí per a l'edició, suport en diversos idiomes

## Utilitzeu la demostració

```html
<c-edit href="https://github.com"></c-edit>
```

## Paràmetres de la interfície

L'etiqueta del component és `<c-edit>` i l'adreça de salt es pot editar mitjançant la configuració de l'atribut `href`.
