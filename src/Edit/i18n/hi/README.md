# संपादित करें बटन

- इसमें समकोण संपादन आइकन बटन शामिल है
- संपादन के लिए माउस-ओवर प्रॉम्प्ट, बहुभाषी समर्थन

## डेमो का प्रयोग करें

```html
<c-edit href="https://github.com"></c-edit>
```

## इंटरफ़ेस पैरामीटर

घटक लेबल `<c-edit>` है, और जंप एड्रेस को `href` विशेषता कॉन्फ़िगरेशन के माध्यम से संपादित किया जा सकता है।
