# redaktə düyməsi

- Düzbucaqlı redaktə nişanı düyməsini ehtiva edir
- Redaktə üçün siçan üzərinə sürüşmə, çoxdilli dəstək

## Demodan istifadə edin

```html
<c-edit href="https://github.com"></c-edit>
```

## İnterfeys parametrləri

Komponent etiketi `<c-edit>`-dir və keçid ünvanı `href` atribut konfiqurasiyası vasitəsilə redaktə edilə bilər.
