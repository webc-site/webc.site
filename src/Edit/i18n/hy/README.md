# Էջի կամ փաստաթղթի խմբագրման կոճակ

- Պարունակում է ուղիղ անկյան խմբագրման պատկերակի կոճակ
- Ներկառուցված «Pophover»-ի սավառնման հուշումները
- Աջակցում է անցնել ցատկ URL-ը և բացել նոր պատուհան

## Օգտագործեք ցուցադրությունը

```html
<c-edit href="https://github.com"></c-edit>

<script type="module">
import "webc.site/Edit/_.js";
</script>
```
