# խմբագրել կոճակը

- Պարունակում է ուղիղ անկյան խմբագրման պատկերակի կոճակ
- Խմբագրման հուշում մկնիկի վրա, բազմալեզու աջակցություն

## Օգտագործեք ցուցադրությունը

```html
<c-edit href="https://github.com"></c-edit>
```

## Ինտերֆեյսի պարամետրեր
