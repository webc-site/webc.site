# editatzeko botoia

- Angelu zuzeneko editatzeko ikonoaren botoia dauka
- Sagua pasatzeko gonbita editatzeko, hizkuntza anitzeko laguntza

## Erabili demoa

```html
<c-edit href="https://github.com"></c-edit>
```

## Interfazearen parametroak

Osagaien etiketa `<c-edit>` da, eta salto-helbidea `href` atributuaren konfigurazioaren bidez edita daiteke.
