# breyta hnappi

- Inniheldur hægri horn breytingatáknhnapp
- Mús yfir hvetja til að breyta, stuðningur á mörgum tungumálum

## Notaðu kynninguna

```html
<c-edit href="https://github.com"></c-edit>

<script type="module">
import "webc.site/Edit/_.js";
</script>
```

## Viðmótsbreytur

Hlutamerkið er `<c-edit>` og hægt er að breyta stökkvistfanginu í gegnum `href` eigindauppsetninguna.
