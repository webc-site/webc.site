# מחוון של מצב טעינה או עיבוד

- אחד את הסמן למצב המתנה
- מציג אנימציית אדוות כדי לציין מצב עיבוד
- החל מסכת טשטוש על טופס כדי למנוע הגשות כפולות

## השתמש בהדגמה

```html
<!-- מחוון טעינה עצמאי -->
<div class="Ing"></div>

<!-- טופס טעינה עם מסכה -->
<form class="Ing">
  <input type="text" placeholder="תיבת קלט">
  <button type="submit">לְהַגִישׁ</button>
</form>
```

## שיעור סגנון

### `.Ing`
- מצב המצביע הוא `wait`.
- אלמנט פסאודו `::before` יוצר אנימציית גלים.
- בשימוש באלמנט `form`, האלמנט הפסאודו `::after` יוצר מסכת טשטוש של 1 פיקסלים, משבית אינטראקציה ובחירת טקסט.
