# लोडिंग या प्रोसेसिंग स्थिति का संकेतक

- कर्सर को प्रतीक्षा स्थिति में एकीकृत करें
- प्रसंस्करण स्थिति को इंगित करने के लिए तरंग एनीमेशन दिखाता है
- डुप्लिकेट सबमिशन को रोकने के लिए फॉर्म पर ब्लर मास्क लगाएं

## डेमो का प्रयोग करें

```html
<!-- स्वतंत्र लोडिंग सूचक -->
<div class="Ing"></div>

<!-- मास्क के साथ फॉर्म लोड हो रहा है -->
<form class="Ing">
  <बी आईडी='1'>
  <button type="submit">जमा करना</button>
</form>
```

## शैली वर्ग

### `.Ing`
- सूचक स्थिति `wait` है।
- छद्म तत्व `::before` एक तरंग एनीमेशन प्रस्तुत करता है।
- जब `form` तत्व पर उपयोग किया जाता है, तो छद्म-तत्व `::after` 1px ब्लर मास्क प्रस्तुत करता है, जिससे इंटरैक्शन और टेक्स्ट चयन अक्षम हो जाता है।
