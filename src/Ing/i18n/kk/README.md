# Жүктеу немесе өңдеу күйінің көрсеткіші

- Курсорды күту күйіне біріктіріңіз
- Өңдеу күйін көрсету үшін толқынды анимацияны көрсетеді
- Қайталанатын жіберулерді болдырмау үшін пішінге бұлыңғырлық маскасын қолданыңыз

## Демонстрацияны пайдаланыңыз

```html
<!-- тәуелсіз жүктеу көрсеткіші -->
<div class="Ing"></div>

<!-- Бетперде бар пішін жүктелуде -->
<form class="Ing">
  <input type="text" placeholder="Енгізу терезесі">
  <button type="submit">жіберу</button>
</form>
```

## стиль сыныбы

### `.Ing`
- Көрсеткіш күйі `wait`.
- Pseudo элементі `::before` толқындық анимацияны көрсетеді.
- `form` элементінде пайдаланылған кезде, псевдоэлемент `::after` өзара әрекеттесу мен мәтінді таңдауды өшіріп, 1px бұлыңғырлық маскасын көрсетеді.
