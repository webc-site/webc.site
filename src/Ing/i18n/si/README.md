# පැටවීමේ හෝ සැකසීමේ තත්ත්වය පිළිබඳ දර්ශකය

- කර්සරය පොරොත්තු තත්ත්වයට ඒකාබද්ධ කරන්න
- සැකසුම් තත්ත්වය දැක්වීමට රැළි සජීවිකරණය පෙන්වයි
- අනුපිටපත් ඉදිරිපත් කිරීම් වැළැක්වීමට පෝරමයකට බොඳ වෙස් මුහුණක් යොදන්න

## Demo භාවිතා කරන්න

```html
<!-- ස්වාධීන පැටවීමේ දර්ශකය -->
<div class="Ing"></div>

<!-- ආවරණ සහිත පෝරමය පූරණය කිරීම -->
<form class="Ing">
  <input type="text" placeholder="ආදාන පෙට්ටිය">
  <button type="submit">ඉදිරිපත් කරන්න</button>
</form>
```

## ශෛලිය පන්තිය

### `.Ing`
- පොයින්ටර් තත්ත්වය `wait` වේ.
- ව්‍යාජ මූලද්‍රව්‍ය `::before` තරංග සජීවිකරණයක් ලබා දෙයි.
- `form` මූලද්‍රව්‍ය මත භාවිතා කරන විට, ව්‍යාජ මූලද්‍රව්‍යය `::after` 1px බොඳ වෙස් මුහුණක් ලබා දෙයි, අන්තර්ක්‍රියා සහ පෙළ තේරීම අබල කරයි.
