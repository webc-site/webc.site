# Індыкатар стану загрузкі або апрацоўкі

- Перавесці курсор у стан чакання
- Паказвае анімацыю рабізны, каб паказаць стан апрацоўкі
- Прымяніце маску размыцця да формы, каб прадухіліць дублікат адпраўкі

## Выкарыстоўвайце дэма

```html
<!-- незалежны індыкатар загрузкі -->
<div class="Ing"></div>

<!-- Форма загрузкі з маскай -->
<form class="Ing">
  <input type="text" placeholder="Поле ўводу">
  <button type="submit">прадставіць</button>
</form>
```

## клас стылю

### `.Ing`
- Статус паказальніка - `wait`.
- Псеўдаэлемент `::before` адлюстроўвае анімацыю рабізны.
- Пры выкарыстанні ў элеменце `form` псеўдаэлемент `::after` адлюстроўвае маску размыцця памерам 1 пікс, адключаючы ўзаемадзеянне і выбар тэксту.
