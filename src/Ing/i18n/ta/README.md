# ஏற்றுதல் அல்லது செயலாக்க நிலையின் காட்டி

- கர்சரை காத்திருக்கும் நிலைக்கு ஒருங்கிணைக்கவும்
- செயலாக்க நிலையைக் குறிக்க சிற்றலை அனிமேஷனைக் காட்டுகிறது
- நகல் சமர்ப்பிப்புகளைத் தடுக்க, படிவத்தில் மங்கலான முகமூடியைப் பயன்படுத்தவும்

## டெமோவைப் பயன்படுத்தவும்

```html
<!-- சுயாதீன ஏற்றுதல் காட்டி -->
<div class="Ing"></div>

<!-- முகமூடியுடன் படிவத்தை ஏற்றுகிறது -->
<form class="Ing">
  <input type="text" placeholder="உள்ளீட்டு பெட்டி">
  <button type="submit">சமர்ப்பிக்க</button>
</form>
```

## பாணி வகுப்பு

### `.Ing`
- சுட்டி நிலை `wait`.
- போலி உறுப்பு `::before` ஒரு சிற்றலை அனிமேஷனை வழங்குகிறது.
- `form` உறுப்பில் பயன்படுத்தப்படும் போது, ​​போலி உறுப்பு `::after` ஒரு 1px மங்கலான முகமூடியை வழங்குகிறது, தொடர்பு மற்றும் உரைத் தேர்வை முடக்குகிறது.
