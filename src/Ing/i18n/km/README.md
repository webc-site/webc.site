# សូចនាករនៃស្ថានភាពផ្ទុកឬដំណើរការ

- បង្រួបបង្រួមទស្សន៍ទ្រនិចទៅក្នុងស្ថានភាពរង់ចាំ
- បង្ហាញចលនា ripple ដើម្បីបង្ហាញពីស្ថានភាពដំណើរការ
- អនុវត្តរបាំងព្រិលទៅទម្រង់មួយដើម្បីការពារការបញ្ជូនស្ទួន

## ប្រើការបង្ហាញ

```html
<!-- សូចនាករផ្ទុកឯករាជ្យ -->
<div class="Ing"></div>

<!-- កំពុងផ្ទុកទម្រង់ជាមួយរបាំង -->
<form class="Ing">
  <input type="text" placeholder="ប្រអប់បញ្ចូល">
  <button type="submit">ដាក់ស្នើ</button>
</form>
```

## ថ្នាក់រចនាប័ទ្ម

### `.Ing`
- ស្ថានភាពទ្រនិចគឺ `wait`។
- ធាតុ Pseudo `::before` បង្ហាញចលនារំញ័រ។
- នៅពេលប្រើនៅលើធាតុ `form` ធាតុ pseudo-element `::after` បង្ហាញរបាំងព្រិល 1px ដោយបិទដំណើរការអន្តរកម្ម និងការជ្រើសរើសអត្ថបទ។
