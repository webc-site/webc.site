# لوڈنگ یا پروسیسنگ کی حیثیت کا اشارہ

- کرسر کو انتظار کی حالت میں متحد کریں۔
- پروسیسنگ کی حیثیت کی نشاندہی کرنے کے لیے ریپل اینیمیشن دکھاتا ہے۔
- ڈپلیکیٹ گذارشات کو روکنے کے لیے فارم پر بلر ماسک لگائیں۔

## ڈیمو استعمال کریں۔

```html
<!-- آزاد لوڈنگ اشارے -->
<div class="Ing"></div>

<!-- ماسک کے ساتھ فارم لوڈ ہو رہا ہے۔ -->
<form class="Ing">
  <input type="text" placeholder="ان پٹ باکس">
  <button type="submit">جمع کروائیں</button>
</form>
```

## سٹائل کلاس

### `.Ing`
- پوائنٹر کی حیثیت `wait` ہے۔
- چھدم عنصر `::before` ریپل اینیمیشن پیش کرتا ہے۔
- جب `form` عنصر پر استعمال کیا جاتا ہے تو، pseudo-element `::after` ایک 1px بلر ماسک پیش کرتا ہے، تعامل اور متن کے انتخاب کو غیر فعال کرتا ہے۔
