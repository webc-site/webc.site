# लोडिंग किंवा प्रोसेसिंग स्थितीचे सूचक

- कर्सरला प्रतीक्षा स्थितीत एकत्र करा
- प्रक्रिया स्थिती दर्शवण्यासाठी रिपल ॲनिमेशन दाखवते
- डुप्लिकेट सबमिशन टाळण्यासाठी फॉर्मवर ब्लर मास्क लावा

## डेमो वापरा

```html
<!-- स्वतंत्र लोडिंग सूचक -->
<div class="Ing"></div>

<!-- मास्कसह फॉर्म लोड करत आहे -->
<form class="Ing">
  <input type="text" placeholder="इनपुट बॉक्स">
  <button type="submit">सबमिट करा</button>
</form>
```

## शैली वर्ग

### `.Ing`
- पॉइंटर स्थिती `wait` आहे.
- स्यूडो एलिमेंट `::before` रिपल ॲनिमेशन रेंडर करते.
- `form` घटकावर वापरल्यावर, स्यूडो-एलिमेंट `::after` 1px ब्लर मास्क रेंडर करतो, परस्परसंवाद आणि मजकूर निवड अक्षम करतो.
