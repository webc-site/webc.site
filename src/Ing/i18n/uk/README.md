# Індикатор стану завантаження або обробки

- Переведіть курсор у стан очікування
- Показує брижу анімації, яка вказує на стан обробки
- Застосуйте маску розмиття до форми, щоб запобігти подвійному надсиланню

## Використовуйте демо

```html
<!-- незалежний індикатор завантаження -->
<div class="Ing"></div>

<!-- Завантажувальна форма з маскою -->
<form class="Ing">
  <input type="text" placeholder="Поле введення">
  <button type="submit">подати</button>
</form>
```

## клас стилю

### `.Ing`
- Статус покажчика: `wait`.
- Псевдоелемент `::before` рендерить анімацію хвилі.
- При використанні в елементі `form` псевдоелемент `::after` відтворює маску розмиття розміром 1 пікс, вимикаючи взаємодію та виділення тексту.
