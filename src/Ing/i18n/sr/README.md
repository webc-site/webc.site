# Индикатор статуса утовара или обраде

- Обједините курсор у стање чекања
- Приказује анимацију таласа која указује на статус обраде
- Примените маску замућења на образац да бисте спречили дуплирање слања

## Користите демо

```html
<!-- независни индикатор оптерећења -->
<div class="Ing"></div>

<!-- Учитавање формулара са маском -->
<form class="Ing">
  <б ид="1">
  <button type="submit">поднети</button>
</form>
```

## стилска класа

### `.Ing`
- Статус показивача је <б ид="1">.
- Псеудо елемент <б ид="1"> приказује таласасту анимацију.
- Када се користи на елементу <б ид="1">, псеудоелемент <б ид="2"> приказује маску замућења од 1пк, онемогућујући интеракцију и избор текста.
