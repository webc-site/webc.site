# 로드 또는 처리 상태 표시기

- 커서를 대기 상태로 통합
- 처리 상태를 나타내는 잔물결 애니메이션 표시
- 중복 제출을 방지하기 위해 양식에 흐림 마스크 적용

## 데모 사용

```html
<!-- 독립 로딩 표시기 -->
<div class="Ing"></div>

<!-- 마스크가 있는 로딩 양식 -->
<form class="Ing">
  <input type="text" placeholder="입력 상자">
  <button type="submit">제출하다</button>
</form>
```

## 스타일 수업

### `.Ing`
- 포인터 상태는 `wait`입니다.
- 의사 요소 `::before`는 잔물결 애니메이션을 렌더링합니다.
- `form` 요소에 사용될 때 의사 요소 `::after`는 1px 흐림 마스크를 렌더링하여 상호 작용 및 텍스트 선택을 비활성화합니다.
