# لوڊشيڊنگ يا پروسيسنگ جي حالت جو اشارو

- انتظار جي حالت ۾ ڪرسر کي متحد ڪريو
- ريپل اينيميشن ڏيکاري ٿو پروسيسنگ اسٽيٽس کي ظاهر ڪرڻ لاءِ
- هڪ بلر ماسڪ لاڳو ڪريو هڪ فارم تي نقل ڪرڻ کان بچڻ لاءِ

## ڊيمو استعمال ڪريو

```html
<!-- آزاد لوڊشيڊنگ اشارو -->
<div class="Ing"></div>

<!-- ماسڪ سان فارم لوڊ ڪندي -->
<form class="Ing">
  <input type="text" placeholder="ان پٽ باڪس">
  <button type="submit">جمع ڪرائڻ</button>
</form>
```

## انداز ڪلاس

### `.Ing`
- پوائنٽر جي حيثيت `wait` آهي.
- Pseudo عنصر `::before` ريپل اينيميشن پيش ڪري ٿو.
- جڏهن `form` عنصر تي استعمال ڪيو وڃي ٿو، pseudo-element `::after` هڪ 1px بلر ماسڪ پيش ڪري ٿو، رابطي ۽ متن جي چونڊ کي غير فعال ڪرڻ.
