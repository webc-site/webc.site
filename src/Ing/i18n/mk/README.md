# Индикатор за статусот на вчитување или обработка

- Унифицирајте го курсорот во состојба на чекање
- Прикажува брановидна анимација за да го покаже статусот на обработка
- Нанесете маска за заматување на формуларот за да спречите дупликат поднесоци

## Користете го демо

```html
<!-- независен индикатор за вчитување -->
<div class="Ing"></div>

<!-- Вчитување на формуларот со маска -->
<form class="Ing">
  <input type="text" placeholder="Влезна кутија">
  <button type="submit">поднесете</button>
</form>
```

## класа стил

### `.Ing`
- Статусот на покажувачот е `wait`.
- Псевдо елементот `::before` прикажува брановидна анимација.
- Кога се користи на елементот `form`, псевдоелементот `::after` прикажува маска за замаглување од 1 px, оневозможувајќи ја интеракцијата и изборот на текст.
