# ଲୋଡିଂ କିମ୍ବା ପ୍ରକ୍ରିୟାକରଣ ସ୍ଥିତିର ସୂଚକ |

- କର୍ସର୍ କୁ ଅପେକ୍ଷା ଅବସ୍ଥାରେ ଏକତ୍ର କର |
- ପ୍ରକ୍ରିୟାକରଣ ସ୍ଥିତି ସୂଚାଇବା ପାଇଁ ରିପଲ୍ ଆନିମେସନ୍ ଦେଖାଏ |
- ନକଲ ଉପସ୍ଥାପନାକୁ ରୋକିବା ପାଇଁ ଏକ ଫର୍ମରେ ଏକ ଅସ୍ପଷ୍ଟ ମାସ୍କ ଲଗାନ୍ତୁ |

## ଡେମୋ ବ୍ୟବହାର କରନ୍ତୁ |

```html
<!-- ସ୍ independent ାଧୀନ ଲୋଡିଂ ସୂଚକ | -->
<div class="Ing"></div>

<!-- ମାସ୍କ ସହିତ ଫର୍ମ ଲୋଡିଂ | -->
<form class="Ing">
  <input type="text" placeholder="ଇନପୁଟ୍ ବାକ୍ସ"> |
  <button type="submit">ଦାଖଲ କର |</button>
</form>
```

## ଶ style ଳୀ ଶ୍ରେଣୀ

### `.Ing`
- ସୂଚକ ସ୍ଥିତି ହେଉଛି `wait` |
- ସିଉଡୋ ଉପାଦାନ `::before` ଏକ ରିପଲ୍ ଆନିମେସନ୍ ପ୍ରଦାନ କରେ |
- ଯେତେବେଳେ `form` ଉପାଦାନରେ ବ୍ୟବହୃତ ହୁଏ, ଛଉ-ଉପାଦାନ `::after` ପାରସ୍ପରିକ କ୍ରିୟା ଏବଂ ପାଠ୍ୟ ଚୟନକୁ ଅକ୍ଷମ କରି 1px ବ୍ଲୁର୍ ମାସ୍କ ପ୍ରଦାନ କରିଥାଏ |
