# 読み込みまたは処理ステータスのインジケーター

- カーソルを待機状態に統合する
- 処理状況を示す波紋アニメーションを表示します
- 重複送信を防ぐためにフォームにぼかしマスクを適用する

## デモを使用する

```html
<!-- 独立した荷重インジケーター -->
<div class="Ing"></div>

<!-- マスク付きフォームの読み込み -->
<form class="Ing">
  <input type="text" placeholder="入力ボックス">
  <button type="submit">提出する</button>
</form>
```

## スタイルクラス

### `.Ing`
- ポインタのステータスは`wait`です。
- 疑似要素`::before`は波紋アニメーションをレンダリングします。
- `form` 要素で使用すると、擬似要素 `::after` は 1 ピクセルのぼかしマスクをレンダリングし、インタラクションとテキストの選択を無効にします。
