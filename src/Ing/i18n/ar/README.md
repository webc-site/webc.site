# مؤشر لحالة التحميل أو المعالجة

- توحيد المؤشر في حالة الانتظار
- يُظهر الرسوم المتحركة المموجة للإشارة إلى حالة المعالجة
- قم بتطبيق قناع تمويه على نموذج لمنع عمليات الإرسال المكررة

## استخدم العرض التوضيحي

```html
<!-- مؤشر تحميل مستقل -->
<div class="Ing"></div>

<!-- تحميل النموذج بالقناع -->
<form class="Ing">
  <ب معرف = "1">
  <button type="submit">يُقدِّم</button>
</form>
```

## فئة النمط

### `.Ing`
- حالة المؤشر هي `wait`.
- يعرض العنصر الزائف `::before` رسمًا متحركًا مموجًا.
- عند استخدامه مع العنصر `form`، يعرض العنصر الزائف `::after` قناعًا ضبابيًا بحجم 1 بكسل، مما يؤدي إلى تعطيل التفاعل واختيار النص.
