# లోడింగ్ లేదా ప్రాసెసింగ్ స్థితి యొక్క సూచిక

- కర్సర్‌ని వెయిటింగ్ స్టేట్‌గా ఏకీకృతం చేయండి
- ప్రాసెసింగ్ స్థితిని సూచించడానికి అలల యానిమేషన్‌ను చూపుతుంది
- నకిలీ సమర్పణలను నిరోధించడానికి ఫారమ్‌కు బ్లర్ మాస్క్‌ని వర్తింపజేయండి

## డెమో ఉపయోగించండి

```html
<!-- స్వతంత్ర లోడింగ్ సూచిక -->
<div class="Ing"></div>

<!-- ముసుగుతో ఫారమ్ లోడ్ అవుతోంది -->
<form class="Ing">
  <input type="text" placeholder="ఇన్‌పుట్ బాక్స్">
  <button type="submit">సమర్పించండి</button>
</form>
```

## శైలి తరగతి

### `.Ing`
- పాయింటర్ స్థితి `wait`.
- సూడో మూలకం `::before` ఒక అలల యానిమేషన్‌ను అందిస్తుంది.
- `form` మూలకంపై ఉపయోగించినప్పుడు, నకిలీ మూలకం `::after` పరస్పర చర్య మరియు వచన ఎంపికను నిలిపివేసి, 1px బ్లర్ మాస్క్‌ని అందిస్తుంది.
