# ຕົວຊີ້ວັດສະຖານະການໂຫຼດ ຫຼືການປະມວນຜົນ

- ຮວມຕົວກະພິບເຂົ້າໄປໃນສະຖານະລໍຖ້າ
- ສະແດງພາບເຄື່ອນໄຫວເພື່ອຊີ້ບອກສະຖານະການປະມວນຜົນ
- ນຳໃຊ້ໜ້າກາກມົວໃສ່ແບບຟອມເພື່ອປ້ອງກັນການສົ່ງຂໍ້ມູນຊໍ້າກັນ

## ໃຊ້ຕົວຢ່າງ

```html
<!-- ຕົວຊີ້ວັດການໂຫຼດເອກະລາດ -->
<div class="Ing"></div>

<!-- ກຳລັງໂຫຼດແບບຟອມດ້ວຍໜ້າກາກ -->
<form class="Ing">
  <input type="text" placeholder="ກ່ອງປ້ອນຂໍ້ມູນ">
  <button type="submit">ສົ່ງ</button>
</form>
```

## ຫ້ອງຮຽນແບບ

### `.Ing`
- ສະຖານະຕົວຊີ້ແມ່ນ `wait`.
- ອົງປະກອບ Pseudo `::before` renders a ripple animation.
- ເມື່ອໃຊ້ໃນອົງປະກອບ `form`, ອົງປະກອບ pseudo-`::after`ຈະສະແດງໜ້າກາກມົວ 1px, ປິດການໂຕ້ຕອບ ແລະການເລືອກຂໍ້ຄວາມ.
