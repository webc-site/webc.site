# نشانگر وضعیت بارگیری یا پردازش

- مکان نما را در حالت انتظار یکسان کنید
- انیمیشن ریپل را برای نشان دادن وضعیت پردازش نشان می دهد
- برای جلوگیری از ارسال‌های تکراری، ماسک تاری را روی فرم اعمال کنید

## از دمو استفاده کنید

```html
<!-- نشانگر بارگذاری مستقل -->
<div class="Ing"></div>

<!-- بارگیری فرم با ماسک -->
<form class="Ing">
  <input type="text" placeholder="جعبه ورودی">
  <button type="submit">ارسال کنید</button>
</form>
```

## کلاس سبک

### `.Ing`
- وضعیت نشانگر `wait` است.
- عنصر شبه `::before` یک انیمیشن موج دار را ارائه می دهد.
- هنگامی که در عنصر `form` استفاده می شود، شبه عنصر `::after` یک ماسک تاری 1 پیکسلی را ارائه می دهد و تعامل و انتخاب متن را غیرفعال می کند.
