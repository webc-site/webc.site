# တင်ခြင်း သို့မဟုတ် လုပ်ဆောင်ခြင်းအခြေအနေ အညွှန်း

- ကာဆာကို စောင့်ဆိုင်းနေသည့်အခြေအနေသို့ ပေါင်းစည်းပါ။
- လုပ်ဆောင်နေသည့် အခြေအနေကို ညွှန်ပြရန် တုန်လှုပ်နေသော ကာတွန်းကို ပြသည်။
- ထပ်နေသောတင်ပြမှုများကို တားဆီးရန် မှုန်ဝါးသောမျက်နှာဖုံးကို ဖောင်တစ်ခုတွင် ထည့်ပါ။

## ဒီမိုကိုသုံးပါ။

```html
<!-- လွတ်လပ်သော loading ညွှန်ပြချက် -->
<div class="Ing"></div>

<!-- မျက်နှာဖုံးဖြင့် ဖောင်တင်နေသည်။ -->
<form class="Ing">
  <input type="text" placeholder="ထည့်သွင်းပုံး">
  <button type="submit">တင်ပြပါ။</button>
</form>
```

## စတိုင်အတန်း

### `.Ing`
- ညွှန်ပြမှုအခြေအနေသည် `wait`။
- Pseudo ဒြပ်စင် `::before` လှုပ်လှုပ်ရှားရှား လှုပ်ရှားသက်ဝင်မှုကို တင်ဆက်သည်။
- `form`ဒြပ်စင်ပေါ်တွင် အသုံးပြုသောအခါ၊ pseudo-ဒြပ်စင် `::after`သည် 1px မှုန်ဝါးသောမျက်နှာဖုံးကို လုပ်ဆောင်ပေးကာ အပြန်အလှန်ဆက်သွယ်မှုနှင့် စာသားရွေးချယ်မှုကို ပိတ်ထားသည်။
