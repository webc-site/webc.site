# Индикатор за състояние на зареждане или обработка

- Обединете курсора в състояние на изчакване
- Показва пулсираща анимация, за да покаже статус на обработка
- Приложете маска за размазване към формуляр, за да предотвратите дублиране на подавания

## Използвайте демонстрацията

```html
<!-- независим индикатор за натоварване -->
<div class="Ing"></div>

<!-- Форма за зареждане с маска -->
<form class="Ing">
  <input type="text" placeholder="Кутия за въвеждане">
  <button type="submit">подавам</button>
</form>
```

## стил клас

### `.Ing`
- Състоянието на показалеца е `wait`.
- Псевдоелементът `::before` изобразява пулсираща анимация.
- Когато се използва върху елемента `form`, псевдоелементът `::after` изобразява маска за размазване от 1px, деактивирайки взаимодействието и избора на текст.
