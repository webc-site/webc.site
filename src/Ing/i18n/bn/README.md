# লোডিং বা প্রসেসিং স্ট্যাটাসের সূচক

- কার্সারকে অপেক্ষমাণ অবস্থায় একীভূত করুন
- প্রসেসিং স্ট্যাটাস নির্দেশ করতে রিপল অ্যানিমেশন দেখায়
- ডুপ্লিকেট জমা প্রতিরোধ করতে একটি ফর্মে একটি অস্পষ্ট মাস্ক প্রয়োগ করুন

## ডেমো ব্যবহার করুন

```html
<!-- স্বাধীন লোডিং সূচক -->
<div class="Ing"></div>

<!-- মাস্ক সহ ফর্ম লোড হচ্ছে -->
<form class="Ing">
  <input type="text" placeholder="ইনপুট বক্স">
  <button type="submit">জমা</button>
</form>
```

## শৈলী ক্লাস

### `.Ing`
- পয়েন্টার স্ট্যাটাস হল `wait`৷
- সিউডো এলিমেন্ট `::before` একটি রিপল অ্যানিমেশন রেন্ডার করে।
- `form` উপাদানে ব্যবহার করা হলে, ছদ্ম-উপাদান `::after` একটি 1px ব্লার মাস্ক রেন্ডার করে, মিথস্ক্রিয়া এবং পাঠ্য নির্বাচন অক্ষম করে।
