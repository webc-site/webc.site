# Ένδειξη κατάστασης φόρτωσης ή επεξεργασίας

- Ενοποιήστε τον κέρσορα σε κατάσταση αναμονής
- Εμφανίζει κινούμενα σχέδια κυματισμού για να υποδείξει την κατάσταση επεξεργασίας
- Εφαρμόστε μια μάσκα θαμπώματος σε μια φόρμα για να αποτρέψετε τις διπλές υποβολές

## Χρησιμοποιήστε το demo

```html
<!-- ανεξάρτητη ένδειξη φόρτωσης -->
<div class="Ing"></div>

<!-- Φόρτωση φόρμας με μάσκα -->
<form class="Ing">
  <input type="text" placeholder="Κουτί εισόδου">
  <button type="submit">υποτάσσομαι</button>
</form>
```

## τάξη στυλ

### `.Ing`
- Η κατάσταση του δείκτη είναι `wait`.
- Το ψευδοστοιχείο `::before` αποδίδει μια κινούμενη εικόνα κυματισμού.
- Όταν χρησιμοποιείται στο στοιχείο `form`, το ψευδοστοιχείο `::after` αποδίδει μια μάσκα θολώματος 1 px, απενεργοποιώντας την αλληλεπίδραση και την επιλογή κειμένου.
