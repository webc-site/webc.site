# ಲೋಡ್ ಅಥವಾ ಪ್ರಕ್ರಿಯೆ ಸ್ಥಿತಿಯ ಸೂಚಕ

- ಕರ್ಸರ್ ಅನ್ನು ಕಾಯುವ ಸ್ಥಿತಿಗೆ ಏಕೀಕರಿಸಿ
- ಸಂಸ್ಕರಣೆಯ ಸ್ಥಿತಿಯನ್ನು ಸೂಚಿಸಲು ಏರಿಳಿತದ ಅನಿಮೇಶನ್ ಅನ್ನು ತೋರಿಸುತ್ತದೆ
- ನಕಲಿ ಸಲ್ಲಿಕೆಗಳನ್ನು ತಡೆಯಲು ಫಾರ್ಮ್‌ಗೆ ಬ್ಲರ್ ಮಾಸ್ಕ್ ಅನ್ನು ಅನ್ವಯಿಸಿ

## ಡೆಮೊ ಬಳಸಿ

```html
<!-- ಸ್ವತಂತ್ರ ಲೋಡಿಂಗ್ ಸೂಚಕ -->
<div class="Ing"></div>

<!-- ಮುಖವಾಡದೊಂದಿಗೆ ಫಾರ್ಮ್ ಅನ್ನು ಲೋಡ್ ಮಾಡಲಾಗುತ್ತಿದೆ -->
<form class="Ing">
  <input type="text" placeholder="ಇನ್ಪುಟ್ ಬಾಕ್ಸ್">
  <button type="submit">ಸಲ್ಲಿಸಿ</button>
</form>
```

## ಶೈಲಿ ವರ್ಗ

### `.Ing`
- ಪಾಯಿಂಟರ್ ಸ್ಥಿತಿಯು `wait` ಆಗಿದೆ.
- ಹುಸಿ ಅಂಶ `::before` ಒಂದು ತರಂಗ ಅನಿಮೇಶನ್ ಅನ್ನು ನಿರೂಪಿಸುತ್ತದೆ.
- `form` ಅಂಶದಲ್ಲಿ ಬಳಸಿದಾಗ, ಹುಸಿ-ಎಲಿಮೆಂಟ್ `::after` 1px ಮಸುಕು ಮುಖವಾಡವನ್ನು ನೀಡುತ್ತದೆ, ಸಂವಹನ ಮತ್ತು ಪಠ್ಯ ಆಯ್ಕೆಯನ್ನು ನಿಷ್ಕ್ರಿಯಗೊಳಿಸುತ್ತದೆ.
