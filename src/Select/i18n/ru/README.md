# Выбор раскрывающегося списка, ширина адаптируется к текущему элементу

Ширина собственного `select` по умолчанию определяется самым длинным текстом параметра. При переключении коротких опций остается слишком много пустого пространства, что ухудшает внешний вид.

Этот компонент прозрачно закрывает собственный тег `select` над тегом `b`, позволяя точно адаптировать ширину к длине выделенного в данный момент текста.

- Поддерживать встроенные стили раскрывающегося меню `select` и поддержку специальных возможностей на всех терминалах.
- Поддерживает встроенные интерактивные операции, такие как полная навигация по вкладкам клавиатуры и переключение клавиш направления.
- Адаптируйте ширину к текущему параметру, чтобы устранить лишнее пустое пространство.

## Используйте демо-версию

```html
<c-select>
  <option value="js">JavaScript</option>
  <option value="rs">Rust</option>
  <option value="py">Python</option>
</c-select>
```
