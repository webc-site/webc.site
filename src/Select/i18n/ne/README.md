# ड्रप-डाउन चयन, चौडाइ हालको वस्तुमा अनुकूल हुन्छ

नेटिभ `select` को चौडाइ पूर्वनिर्धारित रूपमा सबैभन्दा लामो विकल्प पाठद्वारा निर्धारण गरिन्छ। छोटो विकल्पहरू स्विच गर्दा, त्यहाँ धेरै सेतो ठाउँ हुन्छ, जसले उपस्थितिलाई असर गर्छ।

यस कम्पोनेन्टले पारदर्शी रूपमा नेटिभ `select` लाई `b` ट्याग माथि कभर गर्दछ, चौडाइलाई हाल चयन गरिएको पाठको लम्बाइमा सही रूपमा अनुकूलन गर्न अनुमति दिँदै।

- नेटिभ `select` ड्रपडाउन मेनु शैलीहरू र सबै टर्मिनलहरूमा पहुँच समर्थन कायम राख्नुहोस्
- पूर्ण किबोर्ड ट्याब नेभिगेसन र दिशा कुञ्जी स्विचिङ जस्ता नेटिभ अन्तरक्रियात्मक कार्यहरूलाई समर्थन गर्दछ।
- अतिरिक्त सेतो ठाउँ हटाउनको लागि हालको विकल्पमा चौडाइ अनुकूलन गर्नुहोस्

## डेमो प्रयोग गर्नुहोस्

```html
<c-select>
  <option value="js">JavaScript</option>
  <option value="rs">Rust</option>
  <option value="py">Python</option>
</c-select>
```
