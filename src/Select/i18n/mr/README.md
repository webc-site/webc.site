# ड्रॉप-डाउन निवड, रुंदी वर्तमान आयटमशी जुळवून घेते

मूळ `select` ची रुंदी डीफॉल्टनुसार सर्वात लांब पर्याय मजकुराद्वारे निर्धारित केली जाते. लहान पर्याय स्विच करताना, खूप पांढरी जागा आहे, जे देखावा प्रभावित करते.

हा घटक पारदर्शकपणे मूळ `select` कव्हर करतो `b` टॅगच्या वर, रुंदीला सध्या निवडलेल्या मजकुराच्या लांबीशी अचूकपणे जुळवून घेण्यास अनुमती देतो.

- सर्व टर्मिनल्सवर मूळ `select` ड्रॉपडाउन मेनू शैली आणि प्रवेशयोग्यता समर्थन ठेवा
- संपूर्ण कीबोर्ड टॅब नेव्हिगेशन आणि दिशा की स्विचिंग सारख्या मूळ परस्पर क्रियांना समर्थन देते.
- जादा पांढरी जागा काढून टाकण्यासाठी सध्याच्या पर्यायाशी रुंदी जुळवून घ्या

## डेमो वापरा

```html
<c-select>
  <option value="js">JavaScript</option>
  <option value="rs">Rust</option>
  <option value="py">Python</option>
</c-select>
```
