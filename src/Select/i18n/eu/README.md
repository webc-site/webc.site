# Goitibeherako hautapena, zabalera uneko elementura egokitzen da

`select` jatorrizkoaren zabalera aukera-testu luzeenaren arabera zehazten da lehenespenez. Aukera laburrak aldatzean, espazio zuri gehiegi dago eta horrek itxurari eragiten dio.

Osagai honek gardenki estaltzen du `select` etiketaren gaineko jatorrizko `select`, zabalera unean hautatutako testuaren luzerara zehaztasunez egokitzeko aukera emanez.

- Mantendu jatorrizko `select` goitibeherako menu-estiloak eta irisgarritasun-laguntza terminal guztietan
- Jatorrizko eragiketa interaktiboak onartzen ditu, hala nola teklatuaren fitxa osoa nabigazioa eta norabide-tekla aldatzea.
- Egokitu zabalera uneko aukerara gehiegizko zuriuneak ezabatzeko

## Erabili demoa

```html
<c-select>
  <option value="js">JavaScript</option>
  <option value="rs">Rust</option>
  <option value="py">Python</option>
</c-select>
```
