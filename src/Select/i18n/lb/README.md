# Drop-Down Auswiel, Breet passt un aktuellen Artikel

D'Breet vum gebiertege `select` gëtt par défaut vum längsten Optiounstext bestëmmt. Wann Dir kuerz Optiounen wiesselt, gëtt et ze vill wäiss Plaz, wat d'Erscheinung beaflosst.

Dëse Bestanddeel deckt transparent de gebiertege `select` iwwer dem `b` Tag, sou datt d'Breet präzis un d'Längt vum aktuell gewielten Text adaptéiert.

- Erhalen gebierteg `select` Dropdown-Menü Stiler an Accessibilitéit Ënnerstëtzung op all Terminals
- Ënnerstëtzt gebierteg interaktiv Operatiounen wéi komplett Tastatur Tab Navigatioun a Richtungsschalter.
- Passt d'Breet un déi aktuell Optioun un fir iwwerschësseg wäiss Plaz ze eliminéieren

## Benotzt d'Demo

```html
<c-select>
  <option value="js">JavaScript</option>
  <option value="rs">Rust</option>
  <option value="py">Python</option>
</c-select>
```
