# డ్రాప్-డౌన్ ఎంపిక, వెడల్పు ప్రస్తుత అంశానికి అనుగుణంగా ఉంటుంది

స్థానిక `select` వెడల్పు డిఫాల్ట్‌గా పొడవైన ఎంపిక వచనం ద్వారా నిర్ణయించబడుతుంది. చిన్న ఎంపికలను మార్చినప్పుడు, చాలా ఎక్కువ ఖాళీ స్థలం ఉంది, ఇది ప్రదర్శనను ప్రభావితం చేస్తుంది.

ఈ భాగం `b` ట్యాగ్‌కి ఎగువన ఉన్న స్థానిక `select`ని పారదర్శకంగా కవర్ చేస్తుంది, దీని వెడల్పు ప్రస్తుతం ఎంచుకున్న టెక్స్ట్ యొక్క పొడవుకు ఖచ్చితంగా అనుగుణంగా ఉండేలా చేస్తుంది.

- అన్ని టెర్మినల్స్‌లో స్థానిక `select` డ్రాప్‌డౌన్ మెను స్టైల్స్ మరియు యాక్సెసిబిలిటీ సపోర్ట్‌ను నిర్వహించండి
- పూర్తి కీబోర్డ్ ట్యాబ్ నావిగేషన్ మరియు డైరెక్షన్ కీ మార్పిడి వంటి స్థానిక ఇంటరాక్టివ్ ఆపరేషన్‌లకు మద్దతు ఇస్తుంది.
- అదనపు ఖాళీ స్థలాన్ని తొలగించడానికి వెడల్పును ప్రస్తుత ఎంపికకు అనుగుణంగా మార్చండి

## డెమో ఉపయోగించండి

```html
<c-select>
  <option value="js">JavaScript</option>
  <option value="rs">Rust</option>
  <option value="py">Python</option>
</c-select>
```
