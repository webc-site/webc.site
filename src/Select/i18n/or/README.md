# ଡ୍ରପ୍-ଡାଉନ୍ ଚୟନ, ମୋଟେଇ ସାମ୍ପ୍ରତିକ ଆଇଟମ୍ ସହିତ ଖାପ ଖାଏ |

ଦେଶୀ `select` ର ମୋଟେଇ ଡିଫଲ୍ଟ ଭାବରେ ଲମ୍ବା ବିକଳ୍ପ ପାଠ୍ୟ ଦ୍ୱାରା ନିର୍ଣ୍ଣୟ କରାଯାଏ | ସର୍ଟ ଅପ୍ସନ୍ ସୁଇଚ୍ କରିବାବେଳେ ସେଠାରେ ବହୁତ ଧଳା ସ୍ଥାନ ଅଛି, ଯାହା ରୂପକୁ ପ୍ରଭାବିତ କରିଥାଏ |

ଏହି ଉପାଦାନଟି ସ୍ ently ଚ୍ଛ ଭାବରେ ଦେଶୀ `select` `b` ଟ୍ୟାଗ୍ ଉପରେ ଆବୃତ କରେ, ଯାହା ସମ୍ପ୍ରତି ମନୋନୀତ ପାଠ୍ୟର ଦ length ର୍ଘ୍ୟ ସହିତ ସଠିକ୍ ଭାବରେ ଖାପ ଖୁଆଇବାକୁ ଅନୁମତି ଦିଏ |

- ସମସ୍ତ ଟର୍ମିନାଲରେ ଦେଶୀ `select` ଡ୍ରପଡାଉନ୍ ମେନୁ ଶ yles ଳୀ ଏବଂ ଆକ୍ସେସିବିଲିଟି ସମର୍ଥନ ବଜାୟ ରଖନ୍ତୁ |
- ଦେଶୀ ଇଣ୍ଟରାକ୍ଟିଭ୍ ଅପରେସନ୍ସ ଯେପରିକି ସଂପୂର୍ଣ୍ଣ କୀବୋର୍ଡ୍ ଟ୍ୟାବ୍ ନେଭିଗେସନ୍ ଏବଂ ଡାଇରେକ୍ଟ୍ କି ସୁଇଚ୍ କୁ ସମର୍ଥନ କରେ |
- ଅତ୍ୟଧିକ ଧଳା ସ୍ଥାନକୁ ଦୂର କରିବା ପାଇଁ ମୋଟେଇକୁ ସାମ୍ପ୍ରତିକ ବିକଳ୍ପ ସହିତ ଆଡାପ୍ଟ୍ କରନ୍ତୁ |

## ଡେମୋ ବ୍ୟବହାର କରନ୍ତୁ |

```html
<c-select>
  <option value="js">JavaScript</option>
  <option value="rs">Rust</option>
  <option value="py">Python</option>
</c-select>
```
