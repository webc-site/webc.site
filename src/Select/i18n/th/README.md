# การเลือกแบบเลื่อนลง ความกว้างจะปรับตามรายการปัจจุบัน

ความกว้างของเนทิฟ `select` ถูกกำหนดโดยข้อความตัวเลือกที่ยาวที่สุดตามค่าเริ่มต้น เมื่อเปลี่ยนตัวเลือกแบบสั้น มีพื้นที่สีขาวมากเกินไป ซึ่งส่งผลต่อลักษณะที่ปรากฏ

ส่วนประกอบนี้ครอบคลุมข้อความดั้งเดิม `select` เหนือแท็ก `b` อย่างโปร่งใส ช่วยให้ความกว้างสามารถปรับให้เข้ากับความยาวของข้อความที่เลือกในปัจจุบันได้อย่างแม่นยำ

- รักษารูปแบบเมนูแบบเลื่อนลงดั้งเดิมของ `select` และการสนับสนุนการเข้าถึงบนเทอร์มินัลทั้งหมด
- รองรับการทำงานแบบโต้ตอบดั้งเดิม เช่น การนำทางแท็บด้วยแป้นพิมพ์ที่สมบูรณ์และการสลับปุ่มทิศทาง
- ปรับความกว้างให้เข้ากับตัวเลือกปัจจุบันเพื่อกำจัดพื้นที่สีขาวส่วนเกิน

## ใช้การสาธิต

```html
<c-select>
  <option value="js">JavaScript</option>
  <option value="rs">Rust</option>
  <option value="py">Python</option>
</c-select>
```
