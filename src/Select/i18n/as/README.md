# ড্ৰপ-ডাউন নিৰ্বাচন, প্ৰস্থ বৰ্তমান বস্তুৰ সৈতে খাপ খায়

স্থানীয় `select` ৰ প্ৰস্থ অবিকল্পিতভাৱে আটাইতকৈ দীঘল বিকল্প লিখনীৰ দ্বাৰা নিৰ্ধাৰণ কৰা হয়। চুটি বিকল্প সলনি কৰাৰ সময়ত বগা ঠাই বেছি থাকে, যাৰ ফলত ৰূপত প্ৰভাৱ পৰে।

এই উপাদানে স্বচ্ছভাৱে `select` টেগৰ ওপৰৰ স্থানীয় `select` আবৰি ৰাখে, প্ৰস্থক বৰ্তমানে নিৰ্বাচিত লিখনীৰ দৈৰ্ঘ্যৰ সৈতে সঠিকভাৱে খাপ খুৱাবলৈ অনুমতি দিয়ে।

- স্থানীয় `select` ড্ৰপডাউন তালিকা শৈলীসমূহ আৰু অভিগম্যতা সমৰ্থন সকলো টাৰ্মিনেলত ৰক্ষণাবেক্ষণ কৰক
- স্থানীয় পাৰস্পৰিক কাৰ্য্যসমূহ যেনে সম্পূৰ্ণ কিবৰ্ড টেব নেভিগেচন আৰু দিশ কি' চুইচিং সমৰ্থন কৰে।
- অতিৰিক্ত বগা স্থান আঁতৰাবলৈ বৰ্তমান বিকল্পৰ সৈতে প্ৰস্থ খাপ খুৱাই লওক

## ডেমো ব্যৱহাৰ কৰক

```html
<c-select>
  <option value="js">JavaScript</option>
  <option value="rs">Rust</option>
  <option value="py">Python</option>
</c-select>
```
