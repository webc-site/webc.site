# ჩამოსაშლელი არჩევანი, სიგანე ადაპტირდება მიმდინარე ელემენტთან

საწყისი `select` სიგანე განისაზღვრება ნაგულისხმევად ყველაზე გრძელი ვარიანტის ტექსტით. მოკლე ვარიანტების გადართვისას ძალიან ბევრი თეთრი სივრცეა, რაც გავლენას ახდენს გარეგნობაზე.

ეს კომპონენტი გამჭვირვალედ ფარავს `select` ტეგს ზემოთ `b`, რაც საშუალებას აძლევს სიგანეს ზუსტად მოერგოს ამჟამად არჩეული ტექსტის სიგრძეს.

- შეინარჩუნეთ ჩამოსაშლელი მენიუს მშობლიური `select` სტილები და ხელმისაწვდომობის მხარდაჭერა ყველა ტერმინალზე
- მხარს უჭერს მშობლიურ ინტერაქტიულ ოპერაციებს, როგორიცაა კლავიატურის სრული ნავიგაცია და მიმართულების ღილაკის გადართვა.
- დაარეგულირეთ სიგანე მიმდინარე ვარიანტზე ზედმეტი თეთრი სივრცის აღმოსაფხვრელად

<c-select>
