# اختيار القائمة المنسدلة، يتكيف العرض مع العنصر الحالي

يتم تحديد عرض `select` الأصلي بواسطة نص الخيار الأطول افتراضيًا. عند تبديل الخيارات القصيرة، توجد مساحة بيضاء كبيرة جدًا، مما يؤثر على المظهر.

يغطي هذا المكون بشفافية العلامة `select` الأصلية أعلى العلامة `b`، مما يسمح للعرض بالتكيف بدقة مع طول النص المحدد حاليًا.

- حافظ على أنماط القائمة المنسدلة `select` الأصلية ودعم إمكانية الوصول على جميع الأجهزة الطرفية
- يدعم العمليات التفاعلية الأصلية مثل التنقل الكامل بعلامة تبويب لوحة المفاتيح وتبديل مفتاح الاتجاه.
- قم بتكييف العرض مع الخيار الحالي للتخلص من المساحة البيضاء الزائدة

## استخدم العرض التوضيحي

```html
<c-select>
  <option value="js">JavaScript</option>
  <option value="rs">Rust</option>
  <option value="py">Python</option>
</c-select>
```
