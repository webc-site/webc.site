# Dewisiad cwymplen, lled yn addasu i eitem gyfredol

Mae lled y brodorol `select` yn cael ei bennu gan y testun opsiwn hiraf yn ddiofyn. Wrth newid opsiynau byr, mae gormod o le gwyn, sy'n effeithio ar yr edrychiad.

Mae'r gydran hon yn gorchuddio'r brodorol `select` uwchben y tag `b` yn dryloyw, gan ganiatáu i'r lled addasu'n gywir i hyd y testun a ddewiswyd ar hyn o bryd.

- Cynnal arddulliau dewislen brodorol `select` a chymorth hygyrchedd ar bob terfynell
- Yn cefnogi gweithrediadau rhyngweithiol brodorol fel llywio Tab bysellfwrdd cyflawn a newid bysell cyfeiriad.
- Addaswch y lled i'r opsiwn presennol i ddileu gofod gwyn gormodol

## Defnyddiwch y demo

```html
<c-select>
  <option value="js">JavaScript</option>
  <option value="rs">Rust</option>
  <option value="py">Python</option>
</c-select>
```
