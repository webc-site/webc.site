# ড্রপ-ডাউন নির্বাচন, প্রস্থ বর্তমান আইটেমের সাথে খাপ খায়

নেটিভ `select` এর প্রস্থ ডিফল্টরূপে দীর্ঘতম বিকল্প পাঠ্য দ্বারা নির্ধারিত হয়৷ সংক্ষিপ্ত বিকল্পগুলি স্যুইচ করার সময়, খুব বেশি সাদা স্থান থাকে, যা চেহারাকে প্রভাবিত করে।

এই উপাদানটি স্বচ্ছভাবে `b` ট্যাগের উপরে নেটিভ `select` কভার করে, প্রস্থকে বর্তমানে নির্বাচিত পাঠ্যের দৈর্ঘ্যের সাথে সঠিকভাবে মানিয়ে নিতে দেয়।

- সমস্ত টার্মিনালে নেটিভ `select` ড্রপডাউন মেনু শৈলী এবং অ্যাক্সেসিবিলিটি সমর্থন বজায় রাখুন
- সম্পূর্ণ কীবোর্ড ট্যাব নেভিগেশন এবং দিক কী স্যুইচিংয়ের মতো নেটিভ ইন্টারেক্টিভ ক্রিয়াকলাপগুলিকে সমর্থন করে।
- অতিরিক্ত সাদা স্থান দূর করতে বর্তমান বিকল্পের সাথে প্রস্থকে মানিয়ে নিন

## ডেমো ব্যবহার করুন

```html
<c-select>
  <option value="js">JavaScript</option>
  <option value="rs">Rust</option>
  <option value="py">Python</option>
</c-select>
```
