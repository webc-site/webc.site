# انتخاب کشویی، عرض با مورد فعلی سازگار می شود

عرض `select` بومی با طولانی ترین متن گزینه به طور پیش فرض تعیین می شود. هنگام تعویض گزینه های کوتاه، فضای سفید زیادی وجود دارد که بر ظاهر تأثیر می گذارد.

این مؤلفه به طور شفاف `select` بومی بالای تگ `b` را پوشش می‌دهد و به عرض آن اجازه می‌دهد تا به طور دقیق با طول متن انتخاب‌شده فعلی سازگار شود.

- سبک‌های منوی کشویی بومی `select` و پشتیبانی دسترسی را در همه پایانه‌ها حفظ کنید
- از عملیات تعاملی بومی مانند ناوبری کامل صفحه کلید صفحه کلید و تغییر کلید جهت پشتیبانی می کند.
- عرض را با گزینه فعلی تطبیق دهید تا فضای سفید اضافی حذف شود

## از دمو استفاده کنید

```html
<c-select>
  <option value="js">JavaScript</option>
  <option value="rs">Rust</option>
  <option value="py">Python</option>
</c-select>
```
