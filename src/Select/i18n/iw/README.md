# בחירה נפתחת, רוחב מתאים לפריט הנוכחי

הרוחב של `select` מקורי נקבע על ידי טקסט האפשרות הארוך ביותר כברירת מחדל. כאשר מחליפים אפשרויות קצרות, יש יותר מדי שטח לבן, מה שמשפיע על המראה.

רכיב זה מכסה בשקיפות את ה`select` המקורי מעל התג `b`, ומאפשר לרוחב להתאים במדויק לאורך הטקסט הנבחר הנוכחי.

- שמור על סגנונות תפריט נפתחים `select` מקוריים ותמיכה בנגישות בכל המסופים
- תומך בפעולות אינטראקטיביות מקוריות כגון ניווט מלא במקלדת בכרטיסיות והחלפת מקשי כיוון.
- התאם את הרוחב לאפשרות הנוכחית כדי למנוע עודף שטח לבן

## השתמש בהדגמה

```html
<c-select>
  <option value="js">JavaScript</option>
  <option value="rs">Rust</option>
  <option value="py">Python</option>
</c-select>
```
