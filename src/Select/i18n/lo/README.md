# ການເລືອກແບບເລື່ອນລົງ, ຄວາມກວ້າງປັບຕົວເຂົ້າກັບລາຍການປະຈຸບັນ

ຄວາມກວ້າງຂອງ native `select`ຖືກກຳນົດໂດຍຂໍ້ຄວາມທາງເລືອກທີ່ຍາວທີ່ສຸດຕາມຄ່າເລີ່ມຕົ້ນ. ເມື່ອປ່ຽນຕົວເລືອກສັ້ນ, ມີພື້ນທີ່ສີຂາວຫຼາຍເກີນໄປ, ເຊິ່ງມີຜົນກະທົບຕໍ່ຮູບລັກສະນະ.

ອົງ​ປະ​ກອບ​ນີ້​ກວມ​ເອົາ​ຢ່າງ​ໂປ່ງ​ໃສ​ປົກ​ຫຸ້ມ​ຂອງ​ຕົ້ນ​ກໍາ​ເນີດ `select` ຂ້າງ​ເທິງ​ໂຄດ​ຄໍາ​ສັ່ງ `b`​, ປ່ອຍ​ໃຫ້​ຄວາມ​ກວ້າງ​ຂອງ​ການ​ປັບ​ໄດ້​ຢ່າງ​ຖືກ​ຕ້ອງ​ກັບ​ຄວາມ​ຍາວ​ຂອງ​ຂໍ້​ຄວາມ​ທີ່​ເລືອກ​ໃນ​ປັດ​ຈຸ​ບັນ​.

- ຮັກສາຮູບແບບເມນູແບບເລື່ອນລົງແບບເດີມ `select` ແລະຮອງຮັບການຊ່ວຍເຂົ້າເຖິງຢູ່ໃນທຸກເຄື່ອງໝາຍປາຍທາງ
- ຮອງຮັບການດຳເນີນການໂຕ້ຕອບແບບເດີມ ເຊັ່ນ: ການນຳທາງແຖບແປ້ນພິມທີ່ສົມບູນ ແລະການປ່ຽນປຸ່ມທິດທາງ.
- ປັບຄວາມກວ້າງໃຫ້ກັບຕົວເລືອກປະຈຸບັນເພື່ອກໍາຈັດພື້ນທີ່ສີຂາວເກີນ

## ໃຊ້ຕົວຢ່າງ

```html
<c-select>
  <option value="js">JavaScript</option>
  <option value="rs">Rust</option>
  <option value="py">Python</option>
</c-select>
```
