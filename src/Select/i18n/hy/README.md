# Բացվող ընտրություն, լայնությունը հարմարվում է ընթացիկ տարրին

Մայրենի `select`-ի լայնությունը որոշվում է լռելյայն տարբերակի ամենաերկար տեքստով: Կարճ տարբերակները փոխելու ժամանակ չափազանց շատ սպիտակ տարածություն կա, ինչը ազդում է արտաքին տեսքի վրա:

Այս բաղադրիչը թափանցիկ կերպով ծածկում է `b` պիտակի վերևում գտնվող բնիկ `select`-ը, ինչը թույլ է տալիս լայնությունը ճշգրտորեն հարմարվել ներկայումս ընտրված տեքստի երկարությանը:

- Պահպանեք բնիկ `select` բացվող ընտրացանկի ոճերը և հասանելիության աջակցությունը բոլոր տերմինալներում
- Աջակցում է բնիկ ինտերակտիվ գործողություններին, ինչպիսիք են ստեղնաշարի ներդիրների ամբողջական նավարկությունը և ուղղության ստեղնաշարի փոխարկումը:
- Լայնությունը հարմարեցրեք ընթացիկ տարբերակին՝ ավելորդ սպիտակ տարածությունը վերացնելու համար

## Օգտագործեք ցուցադրությունը

```html
<c-select>
  <option value="js">JavaScript</option>
  <option value="rs">Rust</option>
  <option value="py">Python</option>
</c-select>
```
