# အကွက်ချရွေးချယ်မှု၊ အကျယ်သည် လက်ရှိအရာနှင့် လိုက်လျောညီထွေဖြစ်စေသည်။

မူရင်း `select`၏ အကျယ်ကို ပုံသေအားဖြင့် အရှည်ဆုံးရွေးချယ်စရာစာသားဖြင့် ဆုံးဖြတ်သည်။ တိုတောင်းသော ရွေးချယ်မှုများကို ပြောင်းသည့်အခါ အသွင်အပြင်ကို ထိခိုက်စေသည့် အဖြူကွက်များ အလွန်များသည်။

ဤအစိတ်အပိုင်းသည် `b`တဂ်အထက်ရှိ မူရင်း `select`ကို ပွင့်လင်းမြင်သာစွာ ဖုံးအုပ်ထားပြီး အကျယ်အား လက်ရှိရွေးချယ်ထားသော စာသား၏အရှည်နှင့် တိကျစွာလိုက်လျောညီထွေဖြစ်စေရန် ခွင့်ပြုသည်။

- မူရင်း `select` dropdown menu စတိုင်များနှင့် terminals အားလုံးတွင် ဝင်ရောက်နိုင်မှု ပံ့ပိုးမှုကို ထိန်းသိမ်းပါ။
- ပြီးပြည့်စုံသော ကီးဘုတ် Tab လမ်းညွှန်ခြင်းနှင့် ဦးတည်ချက်ခလုတ်ပြောင်းခြင်းကဲ့သို့သော မူရင်းအပြန်အလှန်လုပ်ဆောင်မှုများကို ပံ့ပိုးပေးပါသည်။
- ပိုလျှံနေသော အဖြူရောင်နေရာကို ဖယ်ရှားရန် အကျယ်ကို လက်ရှိရွေးချယ်မှုသို့ ချိန်ညှိပါ။

## ဒီမိုကိုသုံးပါ။

```html
<c-select>
  <option value="js">JavaScript</option>
  <option value="rs">Rust</option>
  <option value="py">Python</option>
</c-select>
```
