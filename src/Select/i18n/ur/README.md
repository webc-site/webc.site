# ڈراپ ڈاؤن انتخاب، چوڑائی موجودہ آئٹم کے مطابق ہوتی ہے۔

مقامی `select` کی چوڑائی کا تعین بطور ڈیفالٹ طویل ترین آپشن ٹیکسٹ سے ہوتا ہے۔ مختصر اختیارات کو تبدیل کرتے وقت، بہت زیادہ سفید جگہ ہوتی ہے، جو ظاہری شکل کو متاثر کرتی ہے۔

یہ جزو شفاف طور پر مقامی `select` کا احاطہ کرتا ہے `b` ٹیگ کے اوپر، چوڑائی کو موجودہ منتخب متن کی لمبائی کے مطابق درست طریقے سے ڈھالنے کی اجازت دیتا ہے۔

- تمام ٹرمینلز پر مقامی `select` ڈراپ ڈاؤن مینو اسٹائل اور ایکسیسبیلٹی سپورٹ کو برقرار رکھیں
- مقامی انٹرایکٹو آپریشنز کو سپورٹ کرتا ہے جیسے مکمل کی بورڈ ٹیب نیویگیشن اور ڈائریکشن کلید سوئچنگ۔
- اضافی سفید جگہ کو ختم کرنے کے لیے چوڑائی کو موجودہ آپشن کے مطابق بنائیں

## ڈیمو استعمال کریں۔

```html
<c-select>
  <option value="js">JavaScript</option>
  <option value="rs">Rust</option>
  <option value="py">Python</option>
</c-select>
```
