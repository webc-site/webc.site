# ドロップダウン選択、幅は現在の項目に適応します

ネイティブ `select` の幅は、デフォルトでは最長のオプション テキストによって決まります。短いオプションを切り替えると、空白が多すぎるため、外観に影響します。

このコンポーネントは、`b` タグの上のネイティブ `select` を透過的にカバーし、幅を現在選択されているテキストの長さに正確に適応させることができます。

- すべての端末でネイティブの`select`ドロップダウン メニュー スタイルとアクセシビリティ サポートを維持します。
- 完全なキーボードのタブ ナビゲーションや方向キーの切り替えなど、ネイティブの対話型操作をサポートします。
- 幅を現在のオプションに調整して、余分な空白を削除します。

## デモを使用する

```html
<c-select>
  <option value="js">JavaScript</option>
  <option value="rs">Rust</option>
  <option value="py">Python</option>
</c-select>
```
