# ସୁଇଚ୍ ଟ୍ୟାବ୍ |

- ଲେବଲ୍ ର ମନୋନୀତ ଅବସ୍ଥା ନିମ୍ନରେ ଥିବା ବିଷୟବସ୍ତୁ କ୍ଷେତ୍ର ସହିତ ନିରବଚ୍ଛିନ୍ନ ଭାବରେ ସଂଯୁକ୍ତ |
- ଗଠନମୂଳକ ନିର୍ଦ୍ଦିଷ୍ଟକରଣ: ଶିଶୁ ଉପାଦାନଗୁଡିକ `nav > a` ଲେବଲ୍ ନେଭିଗେସନ୍ ଭାବରେ ବ୍ୟବହାର କରିବାକୁ ସ୍ଥିର ହୋଇଛି, ଏବଂ ସିଧାସଳଖ ଶିଶୁ ଉପାଦାନ `b` ବିଷୟବସ୍ତୁ କ୍ଷେତ୍ର ଭାବରେ ବ୍ୟବହୃତ ହୁଏ |
- `a[value]` ଟ୍ୟାଗକୁ ସ୍ୱୟଂଚାଳିତ ଭାବରେ `b[slot]` ବିଷୟବସ୍ତୁ ନୋଡ ସହିତ ଯୋଡନ୍ତୁ |
- ସାମ୍ପ୍ରତିକ ସକ୍ରିୟ ପୃଷ୍ଠାକୁ `localStorage` `key` ଆଟ୍ରିବ୍ୟୁଟ୍ ମାଧ୍ୟମରେ ସ୍ୱୟଂଚାଳିତ ଭାବରେ ସ୍ଥିର କରିବାକୁ ସମର୍ଥନ କରେ |
- `value` ଆଟ୍ରିବ୍ୟୁଟ୍ ମାଧ୍ୟମରେ ସକ୍ରିୟ ଟ୍ୟାବ୍ ପୃଷ୍ଠା ନିର୍ଦ୍ଦିଷ୍ଟ କରିବାକୁ ସମର୍ଥନ କରେ (ଯଦି କ c ଣସି କ୍ୟାଚ୍ ନାହିଁ ଏବଂ `value`, `class="A"` କିମ୍ବା ପ୍ରଥମ ଟ୍ୟାଗ୍ ଡିଫଲ୍ଟ ଭାବରେ ଚୟନ ହେବ) |
- ହାଲୁକା ହାଲୁକା DOM କାର୍ଯ୍ୟକାରିତା, କ Sha ଣସି ଛାୟା DOM ପ୍ରତିବନ୍ଧକ, ଶ style ଳୀ କଷ୍ଟମାଇଜେସନ୍ରେ ଉଚ୍ଚତର ସ୍ୱାଧୀନତା |

## ଡେମୋ ବ୍ୟବହାର କରନ୍ତୁ |

```html
<c-tab key="my-tab-key" value="banana">
  <nav>
    <a value="apple">ଆପଲ୍</a>
    <a value="banana">କଦଳୀ</a>
    <a value="orange">ଟାଙ୍ଗରାଇନ୍ |</a>
  </nav>
  <b slot="apple">ଆପଲ୍ ଏକ ପୁଷ୍ଟିକର ଫଳ |</b>
  <b slot="banana">କଦଳୀ ହେଉଛି ଏକ ଟ୍ରପିକାଲ୍ ଫଳ |</b>
  <b slot="orange">କମଳା ଭିଟାମିନ୍ ସି ରେ ଭରପୂର ଅଟେ |</b>
</c-tab>
```

## ସମ୍ପତ୍ତି

- `key`: ସ୍ଥାନୀୟ ଷ୍ଟୋରେଜ୍ ସ୍ଥିରତା ପାଇଁ ବ୍ୟବହୃତ ମୁଖ୍ୟ ନାମ |
- `value`: ସମ୍ପ୍ରତି ମନୋନୀତ ମୂଲ୍ୟ |

## କଷ୍ଟମ୍ ଇଭେଣ୍ଟଗୁଡିକ |

- `change`: ଟ୍ୟାବ୍ ସୁଇଚ୍ ହେବାବେଳେ ଟ୍ରିଗର୍ ହୋଇଯାଏ, `e.value` ହେଉଛି ବର୍ତ୍ତମାନ ସକ୍ରିୟ ହୋଇଥିବା ଟ୍ୟାଗ୍ ର `value` ମୂଲ୍ୟ |
