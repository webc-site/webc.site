# ಬದಲಾಯಿಸಬಹುದಾದ ಟ್ಯಾಬ್‌ಗಳು

- ಲೇಬಲ್‌ನ ಆಯ್ಕೆಮಾಡಿದ ಸ್ಥಿತಿಯನ್ನು ಕೆಳಗಿನ ವಿಷಯ ಪ್ರದೇಶದೊಂದಿಗೆ ಮನಬಂದಂತೆ ಸಂಯೋಜಿಸಲಾಗಿದೆ.
- ರಚನಾತ್ಮಕ ವಿಶೇಷಣಗಳು: ಲೇಬಲ್ ನ್ಯಾವಿಗೇಷನ್ ಆಗಿ `nav > a` ಅನ್ನು ಬಳಸಲು ಚೈಲ್ಡ್ ಎಲಿಮೆಂಟ್‌ಗಳನ್ನು ನಿಗದಿಪಡಿಸಲಾಗಿದೆ ಮತ್ತು ನೇರ ಚೈಲ್ಡ್ ಎಲಿಮೆಂಟ್ `b` ಅನ್ನು ವಿಷಯ ಪ್ರದೇಶವಾಗಿ ಬಳಸಲಾಗುತ್ತದೆ.
- `a[value]` ಟ್ಯಾಗ್ ಅನ್ನು `b[slot]` ವಿಷಯ ನೋಡ್‌ನೊಂದಿಗೆ ಸ್ವಯಂಚಾಲಿತವಾಗಿ ಸಂಯೋಜಿಸಿ.
- ಪ್ರಸ್ತುತ ಸಕ್ರಿಯ ಪುಟವನ್ನು `localStorage` ಗೆ `key` ಗುಣಲಕ್ಷಣದ ಮೂಲಕ ಸ್ವಯಂಚಾಲಿತವಾಗಿ ಬೆಂಬಲಿಸುತ್ತದೆ.
- `value` ಗುಣಲಕ್ಷಣದ ಮೂಲಕ ಸಕ್ರಿಯಗೊಳಿಸಿದ ಟ್ಯಾಬ್ ಪುಟವನ್ನು ನಿರ್ದಿಷ್ಟಪಡಿಸುವುದನ್ನು ಬೆಂಬಲಿಸುತ್ತದೆ (ಯಾವುದೇ ಕ್ಯಾಶ್ ಮತ್ತು `value` ಇಲ್ಲದಿದ್ದರೆ, `class="A"` ಟ್ಯಾಗ್ ಅಥವಾ ಮೊದಲ ಟ್ಯಾಗ್ ಅನ್ನು ಡಿಫಾಲ್ಟ್ ಆಗಿ ಆಯ್ಕೆ ಮಾಡಲಾಗುತ್ತದೆ).
- ಹಗುರವಾದ ಲೈಟ್ DOM ಅನುಷ್ಠಾನ, ನೆರಳು DOM ಅಡಚಣೆಯಿಲ್ಲ, ಶೈಲಿಯ ಗ್ರಾಹಕೀಕರಣದಲ್ಲಿ ಹೆಚ್ಚಿನ ಮಟ್ಟದ ಸ್ವಾತಂತ್ರ್ಯ.

## ಡೆಮೊ ಬಳಸಿ

```html
<c-tab key="my-tab-key" value="banana">
  <nav>
    <a value="apple">ಸೇಬು</a>
    <a value="banana">ಬಾಳೆಹಣ್ಣು</a>
    <a value="orange">ಟ್ಯಾಂಗರಿನ್</a>
  </nav>
  <b slot="apple">ಸೇಬು ಒಂದು ಪೌಷ್ಟಿಕ ಹಣ್ಣು</b>
  <b slot="banana">ಬಾಳೆಹಣ್ಣು ಉಷ್ಣವಲಯದ ಹಣ್ಣು</b>
  <b slot="orange">ಕಿತ್ತಳೆಯಲ್ಲಿ ವಿಟಮಿನ್ ಸಿ ಸಮೃದ್ಧವಾಗಿದೆ</b>
</c-tab>
```

## ಆಸ್ತಿ

- `key`: ಲೋಕಲ್ ಸ್ಟೋರೇಜ್ ನಿರಂತರತೆಗಾಗಿ ಬಳಸಲಾದ ಪ್ರಮುಖ ಹೆಸರು
- `value`: ಪ್ರಸ್ತುತ ಆಯ್ಕೆಮಾಡಿದ ಮೌಲ್ಯ

## ಕಸ್ಟಮ್ ಘಟನೆಗಳು

- `change`: ಟ್ಯಾಬ್ ಸ್ವಿಚ್ ಮಾಡಿದಾಗ ಟ್ರಿಗರ್ ಆಗುತ್ತದೆ, `e.value` ಎಂಬುದು ಪ್ರಸ್ತುತ ಸಕ್ರಿಯವಾಗಿರುವ ಟ್ಯಾಗ್‌ನ `value` ಮೌಲ್ಯವಾಗಿದೆ.
