# మారగల ట్యాబ్‌లు

- లేబుల్ యొక్క ఎంచుకున్న స్థితి దిగువ కంటెంట్ ప్రాంతంతో సజావుగా ఏకీకృతం చేయబడింది.
- నిర్మాణ నిర్దేశాలు: లేబుల్ నావిగేషన్‌గా `nav > a`ని ఉపయోగించడానికి చైల్డ్ ఎలిమెంట్‌లు స్థిరపరచబడ్డాయి మరియు ప్రత్యక్ష చైల్డ్ ఎలిమెంట్ `b` కంటెంట్ ప్రాంతంగా ఉపయోగించబడుతుంది.
- `a[value]` ట్యాగ్‌ని `b[slot]` కంటెంట్ నోడ్‌తో ఆటోమేటిక్‌గా అనుబంధించండి.
- `key` లక్షణం ద్వారా ప్రస్తుత క్రియాశీల పేజీని `localStorage`కి స్వయంచాలకంగా కొనసాగించడానికి మద్దతు ఇస్తుంది.
- యాక్టివేట్ చేయబడిన ట్యాబ్ పేజీని `value` లక్షణం ద్వారా పేర్కొనడానికి మద్దతు ఇస్తుంది (కాష్ మరియు `value` లేకుంటే, `class="A"`తో ట్యాగ్ లేదా మొదటి ట్యాగ్ డిఫాల్ట్‌గా ఎంపిక చేయబడుతుంది).
- తేలికపాటి లైట్ DOM అమలు, షాడో DOM అడ్డంకి లేదు, శైలి అనుకూలీకరణలో అధిక స్థాయి స్వేచ్ఛ.

## డెమో ఉపయోగించండి

```html
<c-tab key="my-tab-key" value="banana">
  <nav>
    <a value="apple">ఆపిల్</a>
    <a value="banana">అరటిపండు</a>
    <a value="orange">టాన్జేరిన్</a>
  </nav>
  <b slot="apple">యాపిల్ ఒక పోషకమైన పండు</b>
  <b slot="banana">అరటి ఒక ఉష్ణమండల పండు</b>
  <b slot="orange">నారింజలో విటమిన్ సి పుష్కలంగా ఉంటుంది</b>
</c-tab>
```

## ఆస్తి

- `key`: స్థానిక నిల్వ నిలకడ కోసం కీ పేరు ఉపయోగించబడుతుంది
- `value`: ప్రస్తుతం ఎంచుకున్న విలువ

## కస్టమ్ ఈవెంట్స్

- `change`: ట్యాబ్ మారినప్పుడు ట్రిగ్గర్ చేయబడింది, `e.value` అనేది ప్రస్తుతం యాక్టివేట్ చేయబడిన ట్యాగ్ యొక్క `value` విలువ.
