# سوئچ ایبل ٹیبز

- لیبل کی منتخب حالت بغیر کسی رکاوٹ کے نیچے مواد کے علاقے کے ساتھ مربوط ہے۔
- ساختی وضاحتیں: چائلڈ عناصر کو `nav > a` کو لیبل نیویگیشن کے طور پر استعمال کرنے کے لیے مقرر کیا گیا ہے، اور براہ راست چائلڈ عنصر `b` کو مواد کے علاقے کے طور پر استعمال کیا جاتا ہے۔
- خودکار طور پر `a[value]` ٹیگ کو `b[slot]` مواد نوڈ کے ساتھ منسلک کریں۔
- `key` انتساب کے ذریعے خود بخود موجودہ فعال صفحہ کو `localStorage` پر برقرار رکھنے کی حمایت کرتا ہے۔
- `value` انتساب (اگر کوئی کیش اور `value` نہیں ہے تو، `class="A"` والا ٹیگ یا پہلے ٹیگ کو بطور ڈیفالٹ منتخب کیا جائے گا) کے ذریعے فعال ٹیب صفحہ کی وضاحت کرنے کی حمایت کرتا ہے۔
- ہلکا پھلکا DOM کا نفاذ، کوئی شیڈو DOM رکاوٹ نہیں، طرز کی تخصیص میں اعلیٰ ڈگری۔

## ڈیمو استعمال کریں۔

```html
<c-tab key="my-tab-key" value="banana">
  <nav>
    <a value="apple">سیب</a>
    <a value="banana">کیلا</a>
    <a value="orange">ٹینجیرین</a>
  </nav>
  <b slot="apple">سیب ایک غذائیت سے بھرپور پھل ہے۔</b>
  <b slot="banana">کیلا ایک اشنکٹبندیی پھل ہے۔</b>
  <b slot="orange">سنترے وٹامن سی سے بھرپور ہوتے ہیں۔</b>
</c-tab>
```

## جائیداد

- `key`: کلیدی نام لوکل سٹوریج استقامت کے لیے استعمال کیا جاتا ہے۔
- `value`: فی الحال منتخب قدر

## حسب ضرورت واقعات

- `change`: ٹیب کے سوئچ ہونے پر متحرک ہوتا ہے، `e.value` فی الحال چالو ٹیگ کی `value` قدر ہے۔
