# Փոխարկվող ներդիրներ

- Պիտակի ընտրված վիճակը անխափան կերպով ինտեգրված է ներքևի բովանդակության տարածքին:
- Կառուցվածքային առանձնահատկություններ. երեխայի տարրերը ամրագրված են `nav > a` որպես պիտակի նավիգացիա օգտագործելու համար, իսկ ուղղակի երեխայի տարրը `b` օգտագործվում է որպես բովանդակության տարածք:
- Ավտոմատ կերպով կապեք `a[value]` թեգը `b[slot]` բովանդակության հանգույցի հետ:
- Աջակցում է ընթացիկ ակտիվ էջի ավտոմատ պահպանումը `localStorage` `key` հատկանիշի միջոցով:
- Աջակցում է ակտիվացված ներդիրի էջը նշելը `value` հատկանիշի միջոցով (եթե չկա քեշ և `value`, ապա `class="A"` պիտակը կամ առաջին պիտակը կընտրվի լռելյայնորեն):
- Թեթև թեթև DOM-ի իրականացում, առանց ստվերային DOM-ի խոչընդոտների, ոճի հարմարեցման ազատության բարձր աստիճան:

## Օգտագործեք ցուցադրությունը

```html
<c-tab key="my-tab-key" value="banana">
  <nav>
    <a value="apple">խնձոր</a>
    <a value="banana">բանան</a>
    <a value="orange">մանդարին</a>
  </nav>
  <b slot="apple">Խնձորը սննդարար միրգ է</b>
  <b slot="banana">Բանանը արևադարձային միրգ է</b>
  <b slot="orange">Նարինջը հարուստ է վիտամին C-ով</b>
</c-tab>
```

## սեփականություն

- `key`. բանալի անուն, որն օգտագործվում է localStorage-ի կայունության համար
- `value`՝ ներկայումս ընտրված արժեք
