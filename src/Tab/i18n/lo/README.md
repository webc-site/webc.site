# ແຖບປ່ຽນໄດ້

- ສະຖານະທີ່ເລືອກຂອງປ້າຍກຳກັບນັ້ນຖືກລວມເຂົ້າກັບພື້ນທີ່ເນື້ອໃນຂ້າງລຸ່ມນີ້.
- ຂໍ້ມູນສະເພາະຂອງໂຄງສ້າງ: ອົງປະກອບຂອງເດັກນ້ອຍຖືກແກ້ໄຂເພື່ອໃຊ້ `nav > a`ເປັນການນໍາທາງຂອງປ້າຍກຳກັບ, ແລະອົງປະກອບລູກໂດຍກົງ `b`ຖືກໃຊ້ເປັນພື້ນທີ່ເນື້ອຫາ.
- ເຊື່ອມໂຍງແທັກ `a[value]` ໂດຍອັດຕະໂນມັດກັບໂນດເນື້ອຫາ `b[slot]`.
- ຮອງຮັບການສືບຕໍ່ໜ້າທີ່ເຄື່ອນໄຫວປັດຈຸບັນໂດຍອັດຕະໂນມັດໄປຫາ `localStorage`ຜ່ານຄຸນລັກສະນະ `key`.
- ຮອງຮັບການລະບຸໜ້າແຖບທີ່ເປີດໃຊ້ງານຜ່ານຄຸນສົມບັດ `value` (ຖ້າບໍ່ມີ cache ແລະ `value`, ແທັກທີ່ມີ `class="A"` ຫຼືແທັກທຳອິດຈະຖືກເລືອກຕາມຄ່າເລີ່ມຕົ້ນ).
- ການປະຕິບັດ DOM ທີ່ມີນ້ໍາຫນັກເບົາ, ບໍ່ມີການຂັດຂວາງ Shadow DOM, ລະດັບສູງຂອງອິດສະລະໃນການປັບແຕ່ງແບບ.

## ໃຊ້ຕົວຢ່າງ

```html
<c-tab key="my-tab-key" value="banana">
  <nav>
    <a value="apple">ໝາກໂປມ</a>
    <a value="banana">ກ້ວຍ</a>
    <a value="orange">ໝາກກ້ຽງ</a>
  </nav>
  <b slot="apple">ໝາກ ແອບເປີ້ນເປັນ ໝາກ ໄມ້ທີ່ມີທາດອາຫານ</b>
  <b slot="banana">ໝາກກ້ວຍເປັນໝາກໄມ້ເຂດຮ້ອນ</b>
  <b slot="orange">ໝາກກ້ຽງອຸດົມໄປດ້ວຍວິຕາມິນຊີ</b>
</c-tab>
```

## ຊັບສິນ

- `key`: ຊື່ກະແຈທີ່ໃຊ້ສຳລັບ localStorage persistence
- `value`: ຄ່າທີ່ເລືອກໃນປັດຈຸບັນ

## ເຫດການແບບກຳນົດເອງ

- `change`: ຖືກກະຕຸ້ນເມື່ອແຖບຖືກປ່ຽນ, `e.value`ແມ່ນ `value`ຄ່າຂອງແທັກທີ່ເປີດໃຊ້ໃນປັດຈຸບັນ.
