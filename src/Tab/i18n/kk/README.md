# Ауыстырылатын қойындылар

- Белгінің таңдалған күйі төмендегі мазмұн аймағымен үздіксіз біріктірілген.
- Құрылымдық сипаттамалар: еншілес элементтер белгіні шарлау ретінде `nav > a` пайдалану үшін бекітілген және тікелей еншілес элемент `b` мазмұн аймағы ретінде пайдаланылады.
- `a[value]` тегін `b[slot]` мазмұн түйінімен автоматты түрде байланыстырыңыз.
- Ағымдағы белсенді бетті `key` төлсипаты арқылы `localStorage` күйіне автоматты түрде сақтауды қолдайды.
- `value` төлсипаты арқылы белсендірілген қойынды бетін көрсетуді қолдайды (егер кэш және `value` болмаса, `class="A"` тег немесе бірінші тег әдепкі бойынша таңдалады).
- Жеңіл жеңіл DOM енгізу, көлеңкелі DOM кедергісі жоқ, стильді теңшеудегі еркіндіктің жоғары дәрежесі.

## Демонстрацияны пайдаланыңыз

```html
<c-tab key="my-tab-key" value="banana">
  <nav>
    <a value="apple">алма</a>
    <a value="banana">банан</a>
    <a value="orange">мандарин</a>
  </nav>
  <b slot="apple">Алма - қоректік жеміс</b>
  <b slot="banana">Банан - тропикалық жеміс</b>
  <b slot="orange">Апельсиндер С дәруменіне бай</b>
</c-tab>
```

## мүлік

- `key`: localStorage тұрақтылығы үшін пайдаланылатын кілт атауы
- `value`: ағымдағы таңдалған мән

## Арнаулы оқиғалар

- `change`: қойынды ауыстырылған кезде іске қосылады, `e.value` - ағымдағы белсендірілген тегтің `value` мәні.
