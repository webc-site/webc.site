# تبديل ٿيندڙ ٽيب

- ليبل جي چونڊيل حالت هيٺ ڏنل مواد واري علائقي سان بيحد ضم ٿي وئي آهي.
- ساخت جون وضاحتون: ٻارن جي عنصرن کي استعمال ڪرڻ لاءِ مقرر ڪيو ويو آهي `nav > a` ليبل نيويگيشن طور، ۽ سڌو ٻار عنصر `b` استعمال ڪيو ويندو مواد جي علائقي طور.
- پاڻمرادو ڳنڍيو `a[value]` ٽيگ کي `b[slot]` مواد جي نوڊ سان.
- خود بخود موجوده فعال صفحي کي `localStorage` جي ذريعي `key` خاصيت برقرار رکڻ جي حمايت ڪري ٿو.
- فعال ٿيل ٽيب صفحي کي `value` وصف ذريعي بيان ڪرڻ جي حمايت ڪري ٿو (جيڪڏهن ڪيش نه هجي ۽ `value`، ٽيگ سان `class="A"` يا پھريون ٽيگ ڊفالٽ طور چونڊيو ويندو).
- ھلڪو ھلڪو DOM عمل درآمد، ڪو به شيڊو DOM رڪاوٽ نه، انداز جي ڪسٽمائيزيشن ۾ اعليٰ درجي جي آزادي.

## ڊيمو استعمال ڪريو

```html
<c-tab key="my-tab-key" value="banana">
  <nav>
    <a value="apple">انب</a>
    <a value="banana">ڪيلا</a>
    <a value="orange">tangerine</a>
  </nav>
  <b slot="apple">ايپل هڪ غذائي ميوو آهي</b>
  <b slot="banana">ڪيلا هڪ اڀرندڙ ميوو آهي</b>
  <b slot="orange">نارنگي وٽامن سي سان مالا مال آهي</b>
</c-tab>
```

## ملڪيت

- `key`: مکيه نالو استعمال ڪيو ويو مقامي اسٽوريج جي استحڪام لاءِ
- `value`: في الحال چونڊيل قدر

## حسب ضرورت واقعا

- `change`: ٽئب کي مٽائڻ تي شروع ڪيو ويو، `e.value` آهي `value` في الحال چالو ٿيل ٽيگ جو قدر.
