# Переключаемые вкладки

- Выбранное состояние метки легко интегрируется с областью содержимого ниже.
- Структурные характеристики: дочерние элементы фиксированно используют `nav > a` в качестве навигации по меткам, а прямой дочерний элемент `b` используется в качестве области содержимого.
- Автоматически связывайте тег `a[value]` с узлом контента `b[slot]`.
- Поддерживает автоматическое сохранение текущей активной страницы в `localStorage` с помощью атрибута `key`.
- Поддерживает указание активированной вкладки через атрибут `value` (при отсутствии кеша и `value` по умолчанию будет выбран тег с `class="A"` или первый тег).
- Легкая реализация Light DOM, отсутствие препятствий для Shadow DOM, высокая степень свободы в настройке стиля.

## Используйте демо-версию

```html
<c-tab key="my-tab-key" value="banana">
  <nav>
    <a value="apple">яблоко</a>
    <a value="banana">банан</a>
    <a value="orange">мандарин</a>
  </nav>
  <b slot="apple">Яблоко – питательный фрукт</b>
  <b slot="banana">Банан – тропический фрукт</b>
  <b slot="orange">Апельсины богаты витамином С</b>
</c-tab>
```

## свойство

- `key`: имя ключа, используемое для сохранения localStorage.
- `value`: выбранное в данный момент значение.

## Пользовательские события

- `change`: активируется при переключении вкладки. `e.value` — это значение `value` текущего активированного тега.
