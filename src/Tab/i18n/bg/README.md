# Превключваеми раздели

- Избраното състояние на етикета е безпроблемно интегрирано със съдържанието по-долу.
- Структурни спецификации: дъщерните елементи са фиксирани да използват `nav > a` като навигация на етикета, а директният дъщерен елемент `b` се използва като област на съдържанието.
- Автоматично свързване на етикета `a[value]` с възела на съдържанието `b[slot]`.
- Поддържа автоматично запазване на текущата активна страница до `localStorage` чрез атрибута `key`.
- Поддържа указване на активираната разделна страница чрез атрибута `value` (ако няма кеш и `value`, етикетът с `class="A"` или първият етикет ще бъдат избрани по подразбиране).
- Лека реализация на Light DOM, без препятствия на Shadow DOM, висока степен на свобода при персонализиране на стила.

## Използвайте демонстрацията

```html
<c-tab key="my-tab-key" value="banana">
  <nav>
    <a value="apple">ябълка</a>
    <a value="banana">банан</a>
    <a value="orange">мандарина</a>
  </nav>
  <b slot="apple">Ябълката е питателен плод</b>
  <b slot="banana">Бананът е тропически плод</b>
  <b slot="orange">Портокалите са богати на витамин С</b>
</c-tab>
```

## собственост

- `key`: име на ключ, използвано за постоянство на localStorage
- `value`: текущо избрана стойност

## Персонализирани събития

- `change`: Задейства се при превключване на раздела, `e.value` е стойността `value` на текущо активирания маркер.
