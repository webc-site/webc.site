# Перемикання вкладок

- Вибраний стан мітки бездоганно інтегрується з областю вмісту нижче.
- Специфікації структури: дочірні елементи фіксовано для використання `nav > a` як навігації по мітках, а прямий дочірній елемент `b` використовується як область вмісту.
- Автоматично пов’язуйте тег `a[value]` із вузлом вмісту `b[slot]`.
- Підтримує автоматичне збереження поточної активної сторінки `localStorage` через атрибут `key`.
- Підтримується вказівка ​​активованої сторінки вкладки за допомогою атрибута `value` (якщо кеш відсутній і `value`, за замовчуванням буде вибрано тег з `class="A"` або перший тег).
- Легка реалізація Light DOM, відсутність перешкод Shadow DOM, високий ступінь свободи в налаштуванні стилю.

## Використовуйте демо

```html
<c-tab key="my-tab-key" value="banana">
  <nav>
    <a value="apple">яблуко</a>
    <a value="banana">банан</a>
    <a value="orange">мандарин</a>
  </nav>
  <b slot="apple">Яблуко - поживний фрукт</b>
  <b slot="banana">Банан - тропічний фрукт</b>
  <b slot="orange">Апельсини багаті на вітамін С</b>
</c-tab>
```

## власність

- `key`: назва ключа, що використовується для збереження локального сховища
- `value`: поточне вибране значення

## Спеціальні події

- `change`: запускається під час перемикання вкладки, `e.value` є значенням `value` поточного активованого тегу.
