# Променљиве картице

- Изабрано стање ознаке је неприметно интегрисано у област садржаја испод.
- Структурне спецификације: подређени елементи су фиксни да користе <б ид="1"> као навигацију ознака, а директни подређени елемент <б ид="2"> се користи као област садржаја.
- Аутоматски повежите ознаку <б ид="1"> са садржајним чвором <б ид="2">.
- Подржава аутоматско задржавање тренутне активне странице до <б ид="2"> преко атрибута <б ид="1">.
- Подржава навођење активиране странице картице преко атрибута <б ид="1"> (ако нема кеш меморије и <б ид="2">, подразумевано ће бити изабрана ознака са <б ид="3"> или првом ознаком).
- Лагана Лигхт ДОМ имплементација, нема опструкције Схадов ДОМ-а, висок степен слободе у прилагођавању стила.

## Користите демо

```html
<c-tab key="my-tab-key" value="banana">
  <nav>
    <a value="apple">јабука</a>
    <a value="banana">банана</a>
    <a value="orange">мандарина</a>
  </nav>
  <b slot="apple">Јабука је хранљиво воће</b>
  <b slot="banana">Банана је тропско воће</b>
  <b slot="orange">Наранџе су богате витамином Ц</b>
</c-tab>
```

## имовине

- <б ид="1">: име кључа који се користи за постојаност лоцалСтораге
- <б ид="1">: тренутно изабрана вредност

## Прилагођени догађаји

- <б ид="1">: Покреће се када се картица промени, <б ид="2"> је вредност <б ид="3"> тренутно активиране ознаке.
