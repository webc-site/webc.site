# زبانه های قابل تعویض

- حالت انتخاب شده برچسب به طور یکپارچه با قسمت محتوای زیر یکپارچه شده است.
- مشخصات ساختاری: عناصر فرزند برای استفاده از `nav > a` به عنوان پیمایش برچسب، و عنصر فرزند مستقیم `b` به عنوان ناحیه محتوا استفاده می‌شود.
- به طور خودکار برچسب `a[value]` را با گره محتوای `b[slot]` مرتبط کنید.
- از تداوم خودکار صفحه فعال فعلی تا `localStorage` از طریق ویژگی `key` پشتیبانی می کند.
- از تعیین صفحه برگه فعال شده از طریق ویژگی `value` پشتیبانی می کند (اگر حافظه نهان و `value` وجود نداشته باشد، برچسب با `class="A"` یا اولین تگ به طور پیش فرض انتخاب می شود).
- اجرای سبک Light DOM، بدون انسداد Shadow DOM، درجه آزادی بالا در سفارشی سازی سبک.

## از دمو استفاده کنید

```html
<c-tab key="my-tab-key" value="banana">
  <nav>
    <a value="apple">سیب</a>
    <a value="banana">موز</a>
    <a value="orange">نارنگی</a>
  </nav>
  <b slot="apple">سیب یک میوه مغذی است</b>
  <b slot="banana">موز یک میوه گرمسیری است</b>
  <b slot="orange">پرتقال سرشار از ویتامین C است</b>
</c-tab>
```

## دارایی

- `key`: نام کلید مورد استفاده برای ماندگاری localStorage
- `value`: مقدار در حال حاضر انتخاب شده است

## رویدادهای سفارشی

- `change`: هنگامی که برگه تغییر می کند فعال می شود، `e.value` مقدار `value` تگ فعال فعلی است.
