# চুইচযোগ্য টেবসমূহ

- লেবেলৰ নিৰ্বাচিত অৱস্থা তলৰ বিষয়বস্তু অঞ্চলৰ সৈতে নিৰৱচ্ছিন্নভাৱে সংহত কৰা হৈছে।
- গাঁথনিগত ধাৰ্য্যকৰণসমূহ: সন্তান উপাদানসমূহক `nav > a` লেবেল নেভিগেচন হিচাপে ব্যৱহাৰ কৰিবলে নিৰ্ধাৰণ কৰা হৈছে, আৰু প্ৰত্যক্ষ সন্তান উপাদান `b` ক বিষয়বস্তু অঞ্চল হিচাপে ব্যৱহাৰ কৰা হয়।
- `a[value]` টেগক `b[slot]` বিষয়বস্তু ন'ডৰ সৈতে স্বয়ংক্ৰিয়ভাৱে সংযুক্ত কৰক।
- বৰ্তমান সক্ৰিয় পৃষ্ঠাক `key` বৈশিষ্ট্যৰ যোগেদি স্বয়ংক্ৰিয়ভাৱে `localStorage` লৈ ৰখা সমৰ্থন কৰে।
- সক্ৰিয় টেব পৃষ্ঠাক `value` বৈশিষ্ট্যৰ যোগেদি ধাৰ্য্য কৰাটো সমৰ্থন কৰে (যদি কোনো কেশ্ব নাই আৰু `value`, `class="A"` বা প্ৰথম টেগ থকা টেগ অবিকল্পিতভাৱে নিৰ্ব্বাচিত হ'ব) ।
- লঘু পোহৰ DOM প্ৰণয়ন, কোনো ছাঁ DOM বাধা নাই, শৈলী স্বনিৰ্বাচনত উচ্চ মাত্ৰাৰ স্বাধীনতা।

## ডেমো ব্যৱহাৰ কৰক

```html
<c-tab key="my-tab-key" value="banana">
  <nav>
    <a value="apple">আপেল</a>
    <a value="banana">কল</a>
    <a value="orange">টেঞ্জাৰিন</a>
  </nav>
  <b slot="apple">আপেল এবিধ পুষ্টিকৰ ফল</b>
  <b slot="banana">কল গ্রীষ্মমণ্ডলীয় এবিধ ফল৷</b>
  <b slot="orange">কমলাত প্ৰচুৰ পৰিমাণে ভিটামিন চি থাকে</b>
</c-tab>
```

## সম্পত্তি

- `key`: localStorage স্থায়িত্বৰ বাবে ব্যৱহৃত চাবিৰ নাম
- `value`: বৰ্তমানে নিৰ্বাচিত মান

## কাষ্টম ইভেন্ট

- `change`: টেব সলনি কৰাৰ সময়ত ট্ৰিগাৰ কৰা হয়, `e.value` হৈছে বৰ্তমানে সক্ৰিয় কৰা টেগৰ `value` মান।
