# स्विच गर्न मिल्ने ट्याबहरू

- लेबलको चयन गरिएको अवस्थालाई तलको सामग्री क्षेत्रसँग सहज रूपमा एकीकृत गरिएको छ।
- संरचनात्मक विनिर्देशहरू: बाल तत्वहरूलाई लेबल नेभिगेसनको रूपमा `nav > a` प्रयोग गर्न निश्चित गरिएको छ, र प्रत्यक्ष बाल तत्व `b` सामग्री क्षेत्रको रूपमा प्रयोग गरिन्छ।
- स्वचालित रूपमा `a[value]` ट्यागलाई `b[slot]` सामग्री नोडसँग सम्बद्ध गर्नुहोस्।
- स्वचालित रूपमा हालको सक्रिय पृष्ठलाई `localStorage` मा `key` विशेषता मार्फत समर्थन गर्दछ।
- `value` विशेषता (यदि त्यहाँ कुनै क्यास र `value` छैन भने, `class="A"` वा पहिलो ट्याग पूर्वनिर्धारित रूपमा चयन गरिनेछ) मार्फत सक्रिय ट्याब पृष्ठ निर्दिष्ट गर्न समर्थन गर्दछ।
- लाइटवेट लाइट DOM कार्यान्वयन, छाया DOM अवरोध छैन, शैली अनुकूलनमा उच्च डिग्री स्वतन्त्रता।

## डेमो प्रयोग गर्नुहोस्

```html
<c-tab key="my-tab-key" value="banana">
  <nav>
    <a value="apple">स्याउ</a>
    <a value="banana">केरा</a>
    <a value="orange">ट्याङ्गेरिन</a>
  </nav>
  <b slot="apple">स्याउ एक पौष्टिक फल हो</b>
  <b slot="banana">केरा एक उष्णकटिबंधीय फल हो</b>
  <b slot="orange">सुन्तलामा भिटामिन सी प्रचुर मात्रामा पाइन्छ</b>
</c-tab>
```

## सम्पत्ति

- `key`: स्थानीय भण्डारण स्थिरताको लागि प्रयोग गरिएको कुञ्जी नाम
- `value`: हाल चयन गरिएको मान

## अनुकूलन घटनाहरू

- `change`: ट्याब स्विच गर्दा ट्रिगर हुन्छ, `e.value` हाल सक्रिय ट्यागको `value` मान हो।
