# மாறக்கூடிய தாவல்கள்

- லேபிளின் தேர்ந்தெடுக்கப்பட்ட நிலை கீழே உள்ள உள்ளடக்கப் பகுதியுடன் தடையின்றி ஒருங்கிணைக்கப்பட்டுள்ளது.
- கட்டமைப்பு விவரக்குறிப்புகள்: லேபிள் வழிசெலுத்தலாக `nav > a` ஐப் பயன்படுத்த குழந்தை உறுப்புகள் சரி செய்யப்படுகின்றன, மேலும் நேரடி குழந்தை உறுப்பு `b` உள்ளடக்கப் பகுதியாகப் பயன்படுத்தப்படுகிறது.
- `a[value]` குறிச்சொல்லை `b[slot]` உள்ளடக்க முனையுடன் தானாக இணைக்கவும்.
- `key` பண்புக்கூறு மூலம் தற்போதைய செயலில் உள்ள பக்கத்தை `localStorage`க்கு தானாக தொடர்வதை ஆதரிக்கிறது.
- `value` பண்புக்கூறு மூலம் செயல்படுத்தப்பட்ட தாவல் பக்கத்தைக் குறிப்பிடுவதை ஆதரிக்கிறது (கேச் மற்றும் `value` இல்லை என்றால், `class="A"` அல்லது முதல் குறிச்சொல் இயல்புநிலையாக தேர்ந்தெடுக்கப்படும்).
- லைட்வெயிட் லைட் DOM செயல்படுத்தல், நிழல் DOM தடை இல்லை, பாணி தனிப்பயனாக்கத்தில் அதிக சுதந்திரம்.

## டெமோவைப் பயன்படுத்தவும்

```html
<c-tab key="my-tab-key" value="banana">
  <nav>
    <a value="apple">ஆப்பிள்</a>
    <a value="banana">வாழைப்பழம்</a>
    <a value="orange">டேன்ஜரின்</a>
  </nav>
  <b slot="apple">ஆப்பிள் ஒரு சத்தான பழம்</b>
  <b slot="banana">வாழைப்பழம் ஒரு வெப்பமண்டலப் பழம்</b>
  <b slot="orange">ஆரஞ்சு பழத்தில் வைட்டமின் சி நிறைந்துள்ளது</b>
</c-tab>
```

## சொத்து

- `key`: உள்ளூர் சேமிப்பக நிலைத்தன்மைக்கு பயன்படுத்தப்படும் முக்கிய பெயர்
- `value`: தற்போது தேர்ந்தெடுக்கப்பட்ட மதிப்பு

## விருப்ப நிகழ்வுகள்

- `change`: தாவல் மாறும்போது தூண்டப்பட்டது, `e.value` என்பது தற்போது செயல்படுத்தப்பட்ட குறிச்சொல்லின் `value` மதிப்பு.
