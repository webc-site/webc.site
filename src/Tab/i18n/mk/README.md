# Заменливи јазичиња

- Избраната состојба на етикетата е беспрекорно интегрирана со областа за содржина подолу.
- Структурни спецификации: Детските елементи се фиксирани да користат `nav > a` како навигација на етикетата, а директниот детски елемент `b` се користи како област за содржина.
- Автоматски поврзете ја ознаката `a[value]` со јазолот за содржина `b[slot]`.
- Поддржува автоматско опстојување на моменталната активна страница до `localStorage` преку атрибутот `key`.
- Поддржува одредување на страницата со активирана картичка преку атрибутот `value` (ако нема кеш и `value`, ознаката со `class="A"` или првата ознака ќе биде стандардно избрана).
- Лесна имплементација на Light DOM, без опструкција на Shadow DOM, висок степен на слобода во прилагодувањето на стилот.

## Користете го демо

```html
<c-tab key="my-tab-key" value="banana">
  <nav>
    <a value="apple">јаболко</a>
    <a value="banana">банана</a>
    <a value="orange">мандарина</a>
  </nav>
  <b slot="apple">Јаболкото е хранливо овошје</b>
  <b slot="banana">Бананата е тропско овошје</b>
  <b slot="orange">Портокалите се богати со витамин Ц</b>
</c-tab>
```

## имот

- `key`: име на клуч што се користи за упорност на локалната меморија
- `value`: моментално избрана вредност

## Прилагодени настани

- `change`: се активира кога ќе се префрли картичката, `e.value` е вредноста `value` на моментално активираната ознака.
