# સ્વિચ કરવા યોગ્ય ટૅબ્સ

- લેબલની પસંદ કરેલી સ્થિતિ નીચેની સામગ્રી વિસ્તાર સાથે એકીકૃત રીતે સંકલિત છે.
- માળખાકીય વિશિષ્ટતાઓ: ચાઇલ્ડ એલિમેન્ટ્સ `nav > a` ને લેબલ નેવિગેશન તરીકે વાપરવા માટે નિશ્ચિત છે અને ડાયરેક્ટ ચાઇલ્ડ એલિમેન્ટ `b` નો ઉપયોગ સામગ્રી વિસ્તાર તરીકે થાય છે.
- `a[value]` ટેગને `b[slot]` સામગ્રી નોડ સાથે આપમેળે સાંકળો.
- `key` વિશેષતા દ્વારા `localStorage` પર વર્તમાન સક્રિય પૃષ્ઠને આપમેળે ચાલુ રાખવાનું સમર્થન કરે છે.
- સક્રિય કરેલ ટેબ પૃષ્ઠને `value` વિશેષતા દ્વારા નિર્દિષ્ટ કરવાનું સમર્થન કરે છે (જો ત્યાં કોઈ કેશ અને `value` ન હોય તો, `class="A"` સાથેનો ટેગ અથવા પ્રથમ ટેગ ડિફોલ્ટ રૂપે પસંદ કરવામાં આવશે).
- લાઇટવેઇટ લાઇટ DOM અમલીકરણ, શેડો DOM અવરોધ નહીં, શૈલી કસ્ટમાઇઝેશનમાં ઉચ્ચ ડિગ્રી સ્વતંત્રતા.

## ડેમોનો ઉપયોગ કરો

```html
<c-tab key="my-tab-key" value="banana">
  <nav>
    <a value="apple">સફરજન</a>
    <a value="banana">કેળા</a>
    <a value="orange">ટેન્જેરીન</a>
  </nav>
  <b slot="apple">સફરજન એક પૌષ્ટિક ફળ છે</b>
  <b slot="banana">કેળા એ ઉષ્ણકટિબંધીય ફળ છે</b>
  <b slot="orange">નારંગીમાં વિટામિન સી ભરપૂર માત્રામાં હોય છે</b>
</c-tab>
```

## મિલકત

- `key`: લોકલ સ્ટોરેજ દ્રઢતા માટે વપરાયેલ કી નામ
- `value`: હાલમાં પસંદ કરેલ મૂલ્ય

## કસ્ટમ ઇવેન્ટ્સ

- `change`: જ્યારે ટેબ સ્વિચ કરવામાં આવે ત્યારે ટ્રિગર થાય છે, `e.value` એ હાલમાં સક્રિય કરેલ ટેગનું `value` મૂલ્ય છે.
