# මාරු කළ හැකි ටැබ්

- ලේබලයේ තෝරාගත් තත්ත්වය පහත අන්තර්ගත ප්‍රදේශය සමඟ බාධාවකින් තොරව ඒකාබද්ධ වේ.
- ව්‍යුහාත්මක පිරිවිතර: ලේබල සංචාලනය ලෙස `nav > a` භාවිත කිරීමට ළමා මූලද්‍රව්‍ය සවි කර ඇති අතර අන්තර්ගත ප්‍රදේශය ලෙස සෘජු ළමා මූලද්‍රව්‍යය `b` භාවිත කෙරේ.
- `a[value]` ටැගය `b[slot]` අන්තර්ගත නෝඩය සමඟ ස්වයංක්‍රීයව සම්බන්ධ කරන්න.
- වත්මන් සක්‍රිය පිටුව `localStorage` වෙත `key` ගුණාංගය හරහා ස්වයංක්‍රීයව පැවතීමට සහය දක්වයි.
- `value` ගුණාංගය හරහා සක්‍රිය ටැබ් පිටුව සඳහන් කිරීමට සහය දක්වයි (හැඹිලිය සහ `value` නොමැති නම්, `class="A"` සහිත ටැගය හෝ පළමු ටැගය පෙරනිමියෙන් තෝරා ගනු ලැබේ).
- සැහැල්ලු සැහැල්ලු DOM ක්‍රියාත්මක කිරීම, සෙවනැලි DOM බාධාවක් නැත, විලාසිතා අභිරුචිකරණයේ ඉහළ නිදහස.

## Demo භාවිතා කරන්න

```html
<c-tab key="my-tab-key" value="banana">
  <nav>
    <a value="apple">ඇපල්</a>
    <a value="banana">කෙසෙල්</a>
    <a value="orange">ජවනාරන්</a>
  </nav>
  <b slot="apple">ඇපල් යනු පෝෂ්‍යදායී පලතුරකි</b>
  <b slot="banana">කෙසෙල් යනු නිවර්තන පලතුරකි</b>
  <b slot="orange">දොඩම් විටමින් C වලින් පොහොසත්</b>
</c-tab>
```

## දේපල

- `key`: localStorage කල්පවත්නා සඳහා භාවිතා කරන ප්‍රධාන නම
- `value`: දැනට තෝරාගත් අගය

## අභිරුචි සිදුවීම්

- `change`: ටැබය මාරු වූ විට ක්‍රියාරම්භක වේ, `e.value` යනු දැනට සක්‍රිය කර ඇති ටැගයේ `value` අගයයි.
