# Стилі типографіки, оптимізовані для читання

- **Базовий набір**: адаптація до заголовків, абзаців, напівжирного шрифту, курсиву, закреслення, вбудованого коду, малюнків і розділових ліній.
- **Анімація посилання**: відображати анімацію підкреслення під час наведення курсора.
- **Список проектів**: невпорядковані списки використовують маркери, упорядковані списки використовують округлені фонові серійні номери та підтримують багаторівневе вкладення.
- **Список завдань**: надає стиль прапорця, підтримує статус справ і завершено.
- **Поради щодо попередження**: підтримує п’ять типів вікон із попередженнями: Примітка, Порада, Важливо, Попередження та Увага.
- **Таблиці та код**: забезпечує виділення рядка таблиці при наведенні курсора та блоки коду з тінями та градієнтним фоном.

## Використовуйте демо

```html
<div class="Md">
  <h1>назва</h1>
  <p>Текст абзацу та <a href="#">посилання</a></p>
  <hr>
  <h2>Назва другого рівня</h2>
  <ul><li><input type="checkbox" checked="" disabled=""> Елемент завдання</li></ul>
  <blockquote class="q note">
    <b class="qt"><i class="qi"></i>ПРИМІТКА`<p>Вміст примітки</p>
  </blockquote>
</div>
```

## клас стилю

Стилі застосовуються до елементів HTML у контейнері `.Md`.

### Стиль вікна сповіщення

Додайте таке ім’я класу до елемента `blockquote` з іменем класу `.q`, щоб перемкнути вікно сповіщення:

- `note`: Примітки
- `tip`: Порада
- `important`: важливо
- `warning`: попередження
- `caution`: Примітка
