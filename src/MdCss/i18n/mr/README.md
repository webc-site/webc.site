# वाचन अनुभवासाठी टायपोग्राफी शैली अनुकूलित

- **मूलभूत टाइपसेटिंग**: शीर्षके, परिच्छेद, ठळक, तिर्यक, स्ट्राइकथ्रू, इनलाइन कोड, चित्रे आणि विभाजन रेषा यांच्याशी जुळवून घ्या.
- **लिंक ॲनिमेशन**: फिरत असताना अधोरेखित ॲनिमेशन प्रदर्शित करा.
- **प्रोजेक्ट सूची**: क्रम नसलेल्या याद्या बुलेट वापरतात, ऑर्डर केलेल्या याद्या गोलाकार पार्श्वभूमी अनुक्रमांक वापरतात आणि बहु-स्तरीय नेस्टिंगला समर्थन देतात.
- **कार्य सूची**: चेक बॉक्स शैली प्रदान करते, टू-डू आणि पूर्ण स्थितीचे समर्थन करते.
- **चेतावणी टिपा**: पाच प्रकारच्या चेतावणी बॉक्सचे समर्थन करते: टीप, टीप, महत्वाचे, चेतावणी आणि लक्ष.
- **टेबल्स आणि कोड**: टेबल पंक्ती हॉवर हायलाइटिंग आणि छाया आणि ग्रेडियंट बॅकग्राउंडसह कोड ब्लॉक प्रदान करते.

## डेमो वापरा

```html
<div class="Md">
  <h1>शीर्षक</h1>
  <p>परिच्छेद मजकूर आणि <a href="#">लिंक</a></p>
  <hr>
  <h2>द्वितीय स्तरावरील शीर्षक</h2>
  <ul><li><input type="checkbox" checked="" disabled=""> कार्य आयटम</li></ul>
  <blockquote class="q note">
    <b class="qt"><i class="qi"></i>सूचना`<p>सामग्री टिपा</p>
  </blockquote>
</div>
```

## शैली वर्ग

शैली `.Md` कंटेनर अंतर्गत HTML घटकांवर लागू केल्या जातात.

### अलर्ट बॉक्स शैली

ॲलर्ट बॉक्स टॉगल करण्यासाठी `blockquote` घटकावर `.q` वर्ग नावासह खालील वर्गाचे नाव जोडा:

- `note`: टिपा
- `tip`: टीप
- `important`: महत्त्वाचे
- `warning`: चेतावणी
- `caution`: टीप
