# 독서 경험에 최적화된 타이포그래피 스타일

- **기본 조판**: 제목, 단락, 굵게, 기울임꼴, 취소선, 인라인 코드, 그림 및 구분선에 맞게 조정됩니다.
- **링크 애니메이션**: 마우스를 올리면 밑줄 애니메이션이 표시됩니다.
- **프로젝트 목록**: 순서가 지정되지 않은 목록은 글머리 기호를 사용하고, 순서가 지정된 목록은 둥근 배경 일련 번호를 사용하며 다중 레벨 중첩을 지원합니다.
- **작업 목록**: 체크박스 스타일을 제공하고 할 일 및 완료 상태를 지원합니다.
- **경고 팁**: 참고, 팁, 중요, 경고, 주의 등 5가지 유형의 경고 상자를 지원합니다.
- **테이블 및 코드**: 테이블 행 호버 강조 표시와 그림자 및 그라데이션 배경이 있는 코드 블록을 제공합니다.

## 데모 사용

```html
<div class="Md">
  <h1>제목</h1>
  <p>단락 텍스트 및 <a href="#">링크</a></p>
  <hr>
  <h2>두 번째 수준 제목</h2>
  <ul><li><input type="checkbox" checked="" disabled=""> 작업 항목</li></ul>
  <blockquote class="q note">
    <b class="qt"><i class="qi"></i>참고`<p>참고 내용</p>
  </blockquote>
</div>
```

## 스타일 수업

스타일은 `.Md` 컨테이너 아래의 HTML 요소에 적용됩니다.

### 경고 상자 스타일

경고 상자를 전환하려면 `.q` 클래스 이름을 사용하여 `blockquote` 요소에 다음 클래스 이름을 추가하세요.

- `note`: 참고사항
- `tip`: 팁
- `important`: 중요
- `warning`: 경고
- `caution`: 참고
