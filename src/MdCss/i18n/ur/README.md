# نوع ٹائپ کی طرزیں پڑھنے کے تجربے کے لیے موزوں ہیں۔

- **بنیادی ٹائپ سیٹنگ**: ٹائٹلز، پیراگراف، بولڈ، اٹالکس، اسٹرائیک تھرو، ان لائن کوڈ، تصاویر اور تقسیم لائنوں کے مطابق ڈھالیں۔
- **لنک اینیمیشن**: منڈلاتے وقت انڈر لائن اینیمیشن ڈسپلے کریں۔
- **پروجیکٹ لسٹ**: غیر ترتیب شدہ فہرستیں گولیوں کا استعمال کرتی ہیں، ترتیب شدہ فہرستیں گول پس منظر کے سیریل نمبرز کا استعمال کرتی ہیں، اور ملٹی لیول نیسٹنگ کو سپورٹ کرتی ہیں۔
- **ٹاسک لسٹ**: چیک باکس اسٹائل فراہم کرتا ہے، کام کرنے اور مکمل ہونے کی حیثیت کو سپورٹ کرتا ہے۔
- **انتباہی تجاویز**: پانچ قسم کے انتباہی خانوں کو سپورٹ کرتا ہے: نوٹ، ٹپ، اہم، انتباہ، اور توجہ۔
- **ٹیبلز اور کوڈ**: ٹیبل قطار ہوور کو ہائی لائٹنگ فراہم کرتا ہے، اور سائے اور تدریجی پس منظر کے ساتھ کوڈ بلاکس فراہم کرتا ہے۔

## ڈیمو استعمال کریں۔

```html
<div class="Md">
  <h1>عنوان</h1>
  <p>پیراگراف کا متن اور <a href="#">لنک</a></p>
  <hr>
  <h2>دوسرے درجے کا عنوان</h2>
  <ul><li><input type="checkbox" checked="" disabled=""> ٹاسک آئٹم</li></ul>
  <blockquote class="q note">
    <b class="qt"><i class="qi"></i>نوٹ`<p>نوٹ مواد</p>
  </blockquote>
</div>
```

## سٹائل کلاس

طرزیں `.Md` کنٹینر کے تحت HTML عناصر پر لاگو ہوتی ہیں۔

### الرٹ باکس اسٹائل

الرٹ باکس کو ٹوگل کرنے کے لیے `blockquote` عنصر پر `.q` کلاس کے نام کے ساتھ درج ذیل کلاس کا نام شامل کریں:

- `note`: نوٹس
- `tip`: ٹپ
- `important`: اہم
- `warning`: وارننگ
- `caution`: نوٹ
