# વાંચન અનુભવ માટે ઑપ્ટિમાઇઝ ટાઇપોગ્રાફી શૈલીઓ

- **મૂળભૂત ટાઇપસેટિંગ**: શીર્ષકો, ફકરા, બોલ્ડ, ત્રાંસા, સ્ટ્રાઇકથ્રુ, ઇનલાઇન કોડ, ચિત્રો અને વિભાજન રેખાઓ સાથે અનુકૂલન કરો.
- **લિંક એનિમેશન**: હોવર કરતી વખતે અન્ડરલાઇન એનિમેશન પ્રદર્શિત કરો.
- **પ્રોજેક્ટ સૂચિ**: બિનક્રમાંકિત સૂચિઓ બુલેટનો ઉપયોગ કરે છે, ક્રમાંકિત સૂચિ ગોળાકાર પૃષ્ઠભૂમિ સીરીયલ નંબરનો ઉપયોગ કરે છે અને બહુ-સ્તરીય માળખાને સમર્થન આપે છે.
- **કાર્ય સૂચિ**: ચેક બૉક્સ શૈલી પ્રદાન કરે છે, કરવા-કરવા અને પૂર્ણ સ્થિતિને સમર્થન આપે છે.
- **ચેતવણી ટીપ્સ**: પાંચ પ્રકારના ચેતવણી બોક્સને સપોર્ટ કરે છે: નોંધ, ટીપ, મહત્વપૂર્ણ, ચેતવણી અને ધ્યાન.
- **કોષ્ટકો અને કોડ**: ટેબલ પંક્તિ હોવર હાઇલાઇટિંગ અને પડછાયાઓ અને ગ્રેડિયન્ટ બેકગ્રાઉન્ડ સાથે કોડ બ્લોક્સ પ્રદાન કરે છે.

## ડેમોનો ઉપયોગ કરો

```html
<div class="Md">
  <h1>શીર્ષક</h1>
  <p>ફકરા ટેક્સ્ટ અને <a href="#">લિંક</a></p>
  <hr>
  <h2>બીજા સ્તરનું શીર્ષક</h2>
  <ul><li><input type="checkbox" checked="" disabled=""> કાર્ય આઇટમ</li></ul>
  <blockquote class="q note">
    <b class="qt"><i class="qi"></i>નોંધ`<p>સામગ્રીની નોંધ કરો</p>
  </blockquote>
</div>
```

## શૈલી વર્ગ

શૈલીઓ `.Md` કન્ટેનર હેઠળ HTML ઘટકો પર લાગુ થાય છે.

### ચેતવણી બોક્સ શૈલી

ચેતવણી બૉક્સને ટૉગલ કરવા માટે `.q` વર્ગના નામ સાથે `blockquote` ઘટક પર નીચેના વર્ગનું નામ ઉમેરો:

- `note`: નોંધો
- `tip`: ટીપ
- `important`: મહત્વપૂર્ણ
- `warning`: ચેતવણી
- `caution`: નોંધ
