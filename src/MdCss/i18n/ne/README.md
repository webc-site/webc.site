# टाइपोग्राफी शैलीहरू पढ्ने अनुभवको लागि अनुकूलित

- **आधारभूत टाइपसेटिङ**: शीर्षक, अनुच्छेद, बोल्ड, इटालिक, स्ट्राइकथ्रु, इनलाइन कोड, चित्र र विभाजन रेखाहरूमा अनुकूलन गर्नुहोस्।
- **लिंक एनिमेसन**: होभर गर्दा अंडरलाइन एनिमेसन प्रदर्शन गर्नुहोस्।
- **परियोजना सूची**: क्रमरहित सूचीहरूले बुलेटहरू प्रयोग गर्दछ, क्रमबद्ध सूचीहरूले गोलाकार पृष्ठभूमि क्रम संख्याहरू प्रयोग गर्दछ, र बहु-स्तर नेस्टिङलाई समर्थन गर्दछ।
- **कार्य सूची**: चेक बाकस शैली प्रदान गर्दछ, गर्न र पूरा स्थिति समर्थन गर्दछ।
- **चेतावनी सुझावहरू**: पाँच प्रकारका चेतावनी बाकसहरूलाई समर्थन गर्दछ: नोट, टिप, महत्त्वपूर्ण, चेतावनी, र ध्यान।
- **तालिका र कोड**: तालिका पङ्क्ति होभर हाइलाइटिङ, र छाया र ग्रेडियन्ट पृष्ठभूमि संग कोड ब्लक प्रदान गर्दछ।

## डेमो प्रयोग गर्नुहोस्

```html
<div class="Md">
  <h1>शीर्षक</h1>
  <p>अनुच्छेद पाठ र <a href="#">लिंक</a></p>
  <hr>
  <h2>दोस्रो स्तरको शीर्षक</h2>
  <ul><li><input type="checkbox" checked="" disabled=""> कार्य वस्तु</li></ul>
  <blockquote class="q note">
    <b class="qt"><i class="qi"></i>नोट`<p>सामग्री नोट गर्नुहोस्</p>
  </blockquote>
</div>
```

## शैली वर्ग

शैलीहरू `.Md` कन्टेनर अन्तर्गत HTML तत्वहरूमा लागू हुन्छन्।

### अलर्ट बक्स शैली

अलर्ट बाकस टगल गर्न `blockquote` `.q` वर्ग नामको साथमा निम्न वर्गको नाम थप्नुहोस्:

- `note`: नोटहरू
- `tip`: टिप
- `important`: महत्त्वपूर्ण
- `warning`: चेतावनी
- `caution`: नोट
