# รูปแบบตัวอักษรที่ปรับให้เหมาะกับประสบการณ์การอ่าน

- **การเรียงพิมพ์ขั้นพื้นฐาน**: ปรับให้เข้ากับชื่อเรื่อง ย่อหน้า ตัวหนา ตัวเอียง ขีดทับ โค้ดในบรรทัด รูปภาพ และเส้นแบ่ง
- **ภาพเคลื่อนไหวของลิงก์**: แสดงภาพเคลื่อนไหวที่ขีดเส้นใต้เมื่อวางเมาส์เหนือ
- **รายการโครงการ**: รายการที่ไม่เรียงลำดับใช้สัญลักษณ์แสดงหัวข้อย่อย รายการที่เรียงลำดับใช้หมายเลขซีเรียลพื้นหลังแบบโค้งมน และรองรับการซ้อนหลายระดับ
- **รายการงาน**: จัดเตรียมรูปแบบช่องทำเครื่องหมาย รองรับสิ่งที่ต้องทำและสถานะที่เสร็จสมบูรณ์
- **เคล็ดลับคำเตือน**: รองรับกล่องคำเตือนห้าประเภท: หมายเหตุ เคล็ดลับ สำคัญ คำเตือน และความสนใจ
- **ตารางและโค้ด**: ให้การไฮไลต์แบบโฮเวอร์แถวตาราง และบล็อกโค้ดที่มีเงาและพื้นหลังแบบไล่ระดับสี

## ใช้การสาธิต

```html
<div class="Md">
  <h1>ชื่อ</h1>
  <p>ข้อความย่อหน้าและ <a href="#">ลิงก์</a></p>
  <hr>
  <h2>ชื่อระดับที่สอง</h2>
  <ul><li><input type="checkbox" checked="" disabled=""> รายการงาน</li></ul>
  <blockquote class="q note">
    <b class="qt"><i class="qi"></i>หมายเหตุ`<p>เนื้อหาหมายเหตุ</p>
  </blockquote>
</div>
```

## คลาสสไตล์

สไตล์จะถูกนำไปใช้กับองค์ประกอบ HTML ภายใต้คอนเทนเนอร์ `.Md`

### รูปแบบกล่องแจ้งเตือน

เพิ่มชื่อคลาสต่อไปนี้ในองค์ประกอบ `blockquote` ด้วยชื่อคลาส `.q` เพื่อสลับกล่องการแจ้งเตือน:

- `note`: หมายเหตุ
- `tip`: เคล็ดลับ
- `important`: สำคัญ
- `warning`: คำเตือน
- `caution`: หมายเหตุ
