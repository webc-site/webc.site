# టైపోగ్రఫీ శైలులు పఠన అనుభవం కోసం ఆప్టిమైజ్ చేయబడ్డాయి

- **ప్రాథమిక టైప్‌సెట్టింగ్**: శీర్షికలు, పేరాగ్రాఫ్‌లు, బోల్డ్, ఇటాలిక్‌లు, స్ట్రైక్‌త్రూ, ఇన్‌లైన్ కోడ్, చిత్రాలు మరియు విభజన పంక్తులకు అనుకూలం.
- **లింక్ యానిమేషన్**: హోవర్ చేస్తున్నప్పుడు అండర్‌లైన్ యానిమేషన్‌ను ప్రదర్శించండి.
- **ప్రాజెక్ట్ జాబితా**: క్రమం చేయని జాబితాలు బుల్లెట్‌లను ఉపయోగిస్తాయి, ఆర్డర్ చేసిన జాబితాలు గుండ్రని నేపథ్య క్రమ సంఖ్యలను ఉపయోగిస్తాయి మరియు బహుళ-స్థాయి గూడుకు మద్దతు ఇస్తాయి.
- **టాస్క్ లిస్ట్**: చెక్ బాక్స్ స్టైల్‌ను అందిస్తుంది, చేయవలసిన మరియు పూర్తి చేసిన స్థితికి మద్దతు ఇస్తుంది.
- **హెచ్చరిక చిట్కాలు**: ఐదు రకాల హెచ్చరిక పెట్టెలకు మద్దతు ఇస్తుంది: గమనిక, చిట్కా, ముఖ్యమైనది, హెచ్చరిక మరియు శ్రద్ధ.
- **టేబుల్స్ మరియు కోడ్**: టేబుల్ రో హోవర్ హైలైటింగ్ మరియు కోడ్ బ్లాక్‌లను షాడోస్ మరియు గ్రేడియంట్ బ్యాక్‌గ్రౌండ్‌లను అందిస్తుంది.

## డెమో ఉపయోగించండి

```html
<div class="Md">
  <h1>శీర్షిక</h1>
  <p>పేరా టెక్స్ట్ మరియు <a href="#">లింక్</a></p>
  <hr>
  <h2>రెండవ స్థాయి టైటిల్</h2>
  <ul><li><input type="checkbox" checked="" disabled=""> టాస్క్ అంశం</li></ul>
  <blockquote class="q note">
    <b class="qt"><i class="qi"></i>గమనిక`<p>గమనిక కంటెంట్</p>
  </blockquote>
</div>
```

## శైలి తరగతి

`.Md` కంటైనర్‌లో ఉన్న HTML మూలకాలకు శైలులు వర్తింపజేయబడతాయి.

### హెచ్చరిక పెట్టె శైలి

హెచ్చరిక పెట్టెను టోగుల్ చేయడానికి `.q` తరగతి పేరుతో `blockquote` మూలకంపై క్రింది తరగతి పేరును జోడించండి:

- `note`: గమనికలు
- `tip`: చిట్కా
- `important`: ముఖ్యమైనది
- `warning`: హెచ్చరిక
- `caution`: గమనిక
