# টাইপোগ্রাফি শৈলী পড়ার অভিজ্ঞতার জন্য অপ্টিমাইজ করা হয়েছে

- **বেসিক টাইপসেটিং**: শিরোনাম, অনুচ্ছেদ, বোল্ড, তির্যক, স্ট্রাইকথ্রু, ইনলাইন কোড, ছবি এবং বিভাজন লাইনের সাথে মানিয়ে নিন।
- **লিংক অ্যানিমেশন**: হোভার করার সময় আন্ডারলাইন অ্যানিমেশন প্রদর্শন করুন।
- **প্রকল্প তালিকা**: ক্রমবিহীন তালিকাগুলি বুলেট ব্যবহার করে, অর্ডারকৃত তালিকাগুলি বৃত্তাকার ব্যাকগ্রাউন্ড সিরিয়াল নম্বর ব্যবহার করে এবং বহু-স্তরের নেস্টিং সমর্থন করে।
- **টাস্ক লিস্ট**: চেক বক্স শৈলী প্রদান করে, করণীয় এবং সম্পূর্ণ অবস্থা সমর্থন করে।
- **সতর্কতা টিপস**: পাঁচ ধরনের সতর্কতা বাক্স সমর্থন করে: নোট, টিপ, গুরুত্বপূর্ণ, সতর্কতা এবং মনোযোগ।
- **টেবিল এবং কোড**: টেবিল সারি হোভার হাইলাইটিং এবং ছায়া ও গ্রেডিয়েন্ট ব্যাকগ্রাউন্ড সহ কোড ব্লক প্রদান করে।

## ডেমো ব্যবহার করুন

```html
<div class="Md">
  <h1>শিরোনাম</h1>
  <p>অনুচ্ছেদ পাঠ্য এবং <a href="#">লিঙ্ক</a>৷</p>
  <hr>
  <h2>দ্বিতীয় স্তরের শিরোনাম</h2>
  <ul><li><input type="checkbox" checked="" disabled=""> টাস্ক আইটেম</li></ul>
  <blockquote class="q note">
    <b class="qt"><i class="qi"></i>দ্রষ্টব্য`<p>সামগ্রী নোট করুন</p>
  </blockquote>
</div>
```

## শৈলী ক্লাস

`.Md` কন্টেইনারের অধীনে HTML উপাদানগুলিতে শৈলী প্রয়োগ করা হয়৷

### সতর্কতা বক্স শৈলী

সতর্কতা বাক্সে টগল করতে `blockquote` এলিমেন্টে `.q` ক্লাসের নামের সাথে নিম্নলিখিত ক্লাসের নাম যোগ করুন:

- `note`: নোট
- `tip`: টিপ
- `important`: গুরুত্বপূর্ণ
- `warning`: সতর্কতা
- `caution`: দ্রষ্টব্য
