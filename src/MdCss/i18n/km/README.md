# រចនាប័ទ្មវាយអក្សរត្រូវបានធ្វើឱ្យប្រសើរសម្រាប់បទពិសោធន៍អាន

- **ការ​កំណត់​ប្រភេទ​មូលដ្ឋាន**៖ សម្រប​តាម​ចំណង​ជើង កថាខណ្ឌ ដិត ទ្រេត កូដកម្ម កូដ​ក្នុង​ជួរ រូបភាព និង​បន្ទាត់​បែងចែក។
- **Link animation**៖ បង្ហាញ​ចលនា​គូស​បន្ទាត់​ក្រោម ពេល​ដាក់។
- **បញ្ជីគម្រោង**៖ បញ្ជីគ្មានតម្រៀបប្រើចំណុច បញ្ជីលំដាប់ប្រើលេខសៀរៀលផ្ទៃខាងក្រោយមូល និងគាំទ្រការដាក់សំបុកពហុកម្រិត។
- **បញ្ជីកិច្ចការ**៖ ផ្តល់នូវរចនាប័ទ្មប្រអប់ធីក គាំទ្រការងារត្រូវធ្វើ និងស្ថានភាពដែលបានបញ្ចប់។
- ** គន្លឹះព្រមាន **៖ គាំទ្រប្រអប់ព្រមានប្រាំប្រភេទ៖ ចំណាំ គន្លឹះ សំខាន់ ព្រមាន និងយកចិត្តទុកដាក់។
- **តារាង និងលេខកូដ**៖ ផ្តល់នូវការរំលេចលើជួរដេកតារាង និងប្លុកកូដជាមួយនឹងស្រមោល និងផ្ទៃខាងក្រោយពណ៌ជម្រាល។

## ប្រើការបង្ហាញ

```html
<div class="Md">
  <h1>ចំណងជើង</h1>
  <p>អត្ថបទកថាខណ្ឌ និង <a href="#">តំណ</a></p>
  <hr>
  <h2>ចំណងជើងកម្រិតទីពីរ</h2>
  <ul><li><input type="checkbox" checked="" disabled=""> ធាតុភារកិច្ច</li></ul>
  <blockquote class="q note">
    <b class="qt"><i class="qi"></i>ចំណាំ`<p>មាតិកាចំណាំ</p>
  </blockquote>
</div>
```

## ថ្នាក់រចនាប័ទ្ម

រចនាប័ទ្ម​ត្រូវ​បាន​អនុវត្ត​ចំពោះ​ធាតុ HTML ក្រោម​កុងតឺន័រ `.Md`។

### រចនាប័ទ្មប្រអប់ជូនដំណឹង

បន្ថែមឈ្មោះថ្នាក់ខាងក្រោមនៅលើធាតុ `blockquote` ជាមួយឈ្មោះថ្នាក់ `.q` ដើម្បីបិទបើកប្រអប់ជូនដំណឹង៖

- `note`៖ កំណត់ចំណាំ
- `tip`៖ គន្លឹះ
- `important`៖ សំខាន់
- `warning`៖ ការព្រមាន
- `caution`៖ ចំណាំ
