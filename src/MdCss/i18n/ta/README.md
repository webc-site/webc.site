# வாசிப்பு அனுபவத்திற்கு உகந்த அச்சுக்கலை பாணிகள்

- **அடிப்படை தட்டச்சு அமைப்பு**: தலைப்புகள், பத்திகள், தடிமனான, சாய்வு, ஸ்ட்ரைக்த்ரூ, இன்லைன் குறியீடு, படங்கள் மற்றும் பிரிக்கும் கோடுகளுக்கு ஏற்ப.
- **இணைப்பு அனிமேஷன்**: வட்டமிடும்போது அண்டர்லைன் அனிமேஷனைக் காண்பி.
- **திட்டப் பட்டியல்**: வரிசைப்படுத்தப்படாத பட்டியல்கள் தோட்டாக்களைப் பயன்படுத்துகின்றன, ஆர்டர் செய்யப்பட்ட பட்டியல்கள் வட்டமான பின்னணி வரிசை எண்களைப் பயன்படுத்துகின்றன, மேலும் பல-நிலைக் கூட்டை ஆதரிக்கின்றன.
- **பணி பட்டியல்**: செக் பாக்ஸ் ஸ்டைலை வழங்குகிறது, செய்ய வேண்டியவை மற்றும் முடிக்கப்பட்ட நிலையை ஆதரிக்கிறது.
- **எச்சரிக்கை குறிப்புகள்**: ஐந்து வகையான எச்சரிக்கை பெட்டிகளை ஆதரிக்கிறது: குறிப்பு, உதவிக்குறிப்பு, முக்கியமானது, எச்சரிக்கை மற்றும் கவனம்.
- **அட்டவணைகள் மற்றும் குறியீடு**: டேபிள் வரிசை ஹோவர் ஹைலைட்டிங் மற்றும் குறியீடு தொகுதிகளை நிழல்கள் மற்றும் சாய்வு பின்னணியுடன் வழங்குகிறது.

## டெமோவைப் பயன்படுத்தவும்

```html
<div class="Md">
  <h1>தலைப்பு</h1>
  <p>பத்தி உரை மற்றும் <a href="#">இணைப்பு</a></p>
  <hr>
  <h2>இரண்டாம் நிலை தலைப்பு</h2>
  <ul><li><input type="checkbox" checked="" disabled=""> பணி உருப்படி</li></ul>
  <blockquote class="q note">
    <b class="qt"><i class="qi"></i>குறிப்பு`<p>குறிப்பு உள்ளடக்கம்</p>
  </blockquote>
</div>
```

## பாணி வகுப்பு

`.Md` கண்டெய்னரின் கீழ் HTML உறுப்புகளுக்கு ஸ்டைல்கள் பயன்படுத்தப்படுகின்றன.

### எச்சரிக்கை பெட்டி பாணி

விழிப்பூட்டல் பெட்டியை நிலைமாற்ற, `blockquote` உறுப்பில் `.q` வகுப்புப் பெயருடன் பின்வரும் வகுப்புப் பெயரைச் சேர்க்கவும்:

- `note`: குறிப்புகள்
- `tip`: உதவிக்குறிப்பு
- `important`: முக்கியமானது
- `warning`: எச்சரிக்கை
- `caution`: குறிப்பு
