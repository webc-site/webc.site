# 为阅读体验优化的排版样式

- **基础排版**：适配标题、段落、粗体、斜体、删除线、行内代码、图片与分割线。
- **链接动效**：悬停时显示下划线动画。
- **项目列表**：无序列表使用项目符号，有序列表使用圆角背景序号，支持多级嵌套。
- **任务列表**：提供复选框样式，支持待办和已办状态。
- **警告提示**：支持便签、提示、重要、警告、注意五种样式的警告框。
- **表格与代码**：提供表格行悬停高亮，以及带阴影与渐变背景的代码块。

## 使用演示

```html
<div class="Md">
  <h1>标题</h1>
  <p>段落文本与 <a href="#">链接</a></p>
  <hr>
  <h2>二级标题</h2>
  <ul><li><input type="checkbox" checked disabled> 任务项目</li></ul>
  <blockquote class="q note">
    <b class="qt"><i class="qi"></i>NOTE</b><p>便签提示内容</p>
  </blockquote>
</div>
```

## 样式类

样式应用于 `.Md` 容器下的 HTML 元素。

### 警告框样式

在带有 `.q` 类名的 `blockquote` 元素上添加以下类名以切换警告框：

- `note`：便签
- `tip`：提示
- `important`：重要
- `warning`：警告
- `caution`：注意
