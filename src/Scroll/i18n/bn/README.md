# ভার্চুয়াল স্ক্রল বার

- **টু-ওয়ে স্ক্রোলিং**: উল্লম্ব (`c-vs`) এবং অনুভূমিক (`c-hs`) স্ক্রোলিং প্রদান করে।
- **আকারের প্রতিক্রিয়া**: বিষয়বস্তু এবং এলাকার আকার পরিবর্তনগুলি মনিটর করুন, স্বয়ংক্রিয়ভাবে স্ক্রোল বারের আকার এবং অবস্থান আপডেট করুন এবং কোনো ওভারফ্লো না থাকলে স্বয়ংক্রিয়ভাবে লুকান।
- **ইন্টারেক্টিভ সাপোর্ট**: হুইল স্ক্রলিং, স্লাইডার ড্র্যাগিং এবং ট্র্যাক ক্লিক পজিশনিং সমর্থন করে।
- **স্থিতি লুকান**: ঘোরানো বা স্ক্রোলিং করার সময় স্লাইডারটি দেখান এবং প্রশস্ত করুন এবং স্থির থাকলে স্বয়ংক্রিয়ভাবে লুকান৷

## উল্লম্ব স্ক্রোল

```html
<div style="height:300px;border:1px solid #000">
  <c-vs>
    <b>1</b>
    <b>2</b>
    <b>3</b>
  </c-vs>
</div>
<style>b { display: flex; border: 2px solid #090; padding: 30px; margin: 30px; }</style>
```

## অনুভূমিক স্ক্রোল

```html
<div style="width:300px;border:1px solid #000">
  <c-hs>
    <b>1</b>
    <b>2</b>
    <b>3</b>
  </c-hs>
</div>
<style>b { display: flex; border: 2px solid #090; margin: 30px; padding: 30px; }</style>
```

## শৈলী অংশ

`::part()` এর মাধ্যমে ছায়া DOM অভ্যন্তরীণ উপাদানগুলির শৈলী কাস্টমাইজ করুন:

- `::part(scroll)`: বিষয়বস্তু এলাকা স্ক্রোল করুন।
- `::part(bar)`: স্ক্রোল বার ট্র্যাক।
- `::part(si)`: স্ক্রোল বার স্লাইডার।
- `::part(i)`: অভ্যন্তরীণ মোড়ানো স্তর।
- `::part(drag)`: ড্র্যাগিং অবস্থায় স্ক্রোল বার ট্র্যাক৷

## শৈলী ক্লাস

- `.drag`: `body` উপাদানে প্রয়োগ করা হয়েছে, এটি নির্দেশ করে যে এটি একটি ড্র্যাগ-স্ক্রলিং অবস্থায় রয়েছে৷

## কাস্টম বৈশিষ্ট্য

- `--si-bg`: স্লাইডারের পটভূমির রঙ।
- `--si-anim`: স্লাইডার অ্যানিমেশন।
- `--cursorScrollhSvg`: অনুভূমিক স্ক্রল বারটি ঘোরালে মাউস কার্সার।
- `--cursorScrollvSvg`: উল্লম্ব স্ক্রল বারটি ঘোরালে মাউস কার্সার।
