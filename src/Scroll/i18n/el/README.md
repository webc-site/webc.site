# εικονική γραμμή κύλισης

- **Αμφίδρομη κύλιση**: Παρέχει κάθετη (`c-vs`) και οριζόντια (`c-hs`) κύλιση.
- **Απόκριση μεγέθους**: Παρακολουθήστε τις αλλαγές του περιεχομένου και του μεγέθους της περιοχής, ενημερώνετε αυτόματα το μέγεθος και τη θέση της γραμμής κύλισης και αποκρύπτεται αυτόματα όταν δεν υπάρχει υπερχείλιση.
- **Διαδραστική υποστήριξη**: Υποστηρίζει κύλιση τροχού, σύρσιμο ρυθμιστικού και εντοπισμό θέσης κλικ παρακολούθησης.
- **Κατάσταση Απόκρυψη**: Εμφάνιση και διεύρυνση του ρυθμιστικού όταν τοποθετείτε το δείκτη του ποντικιού ή κάνετε κύλιση και αποκρύπτεται αυτόματα όταν είναι ακίνητο.

## κάθετη κύλιση

```html
<div style="height:300px;border:1px solid #000">
  <c-vs>
    <b>1</b>
    <b>2</b>
    <b>3</b>
  </c-vs>
</div>
<style>b { display: flex; border: 2px solid #090; padding: 30px; margin: 30px; }</style>
```

## οριζόντια κύλιση

```html
<div style="width:300px;border:1px solid #000">
  <c-hs>
    <b>1</b>
    <b>2</b>
    <b>3</b>
  </c-hs>
</div>
<style>b { display: flex; border: 2px solid #090; margin: 30px; padding: 30px; }</style>
```

## μέρη στυλ

Προσαρμόστε το στυλ των εσωτερικών στοιχείων του Shadow DOM μέσω `::part()`:

- `::part(scroll)`: Κάντε κύλιση στην περιοχή περιεχομένου.
- `::part(bar)`: Κομμάτι γραμμής κύλισης.
- `::part(si)`: Ρυθμιστικό γραμμής κύλισης.
- `::part(i)`: Εσωτερικό επίπεδο αναδίπλωσης.
- `::part(drag)`: Το κομμάτι της γραμμής κύλισης σε κατάσταση μεταφοράς.

## τάξη στυλ

- `.drag`: Εφαρμόζεται στο στοιχείο `body`, υποδεικνύοντας ότι βρίσκεται σε κατάσταση μεταφοράς κύλισης.

## Προσαρμοσμένες ιδιότητες

- `--si-bg`: Χρώμα φόντου ρυθμιστικού.
- `--si-anim`: Κινούμενα σχέδια ρυθμιστικού.
- `--cursorScrollhSvg`: Ο κέρσορας του ποντικιού όταν τοποθετείται η οριζόντια γραμμή κύλισης.
- `--cursorScrollvSvg`: Ο κέρσορας του ποντικιού όταν τοποθετείται η κάθετη γραμμή κύλισης.
