# ورچوئل اسکرول بار

- **دو طرفہ سکرولنگ**: عمودی (`c-vs`) اور افقی (`c-hs`) سکرولنگ فراہم کرتا ہے۔
- **سائز جواب**: مواد اور علاقے کے سائز کی تبدیلیوں کی نگرانی کریں، اسکرول بار کے سائز اور پوزیشن کو خود بخود اپ ڈیٹ کریں، اور جب کوئی اوور فلو نہ ہو تو خود بخود چھپ جائیں۔
- **انٹرایکٹو سپورٹ**: وہیل اسکرولنگ، سلائیڈر ڈریگنگ اور ٹریک کلک پوزیشننگ کو سپورٹ کرتا ہے۔
- **اسٹیٹس ہائیڈ**: منڈلاتے یا اسکرول کرتے وقت سلائیڈر کو دکھائیں اور چوڑا کریں، اور جب اسٹیشنری ہو تو اسے خود بخود چھپائیں۔

## عمودی طومار

```html
<div style="height:300px;border:1px solid #000">
  <c-vs>
    <b>1</b>
    <b>2</b>
    <b>3</b>
  </c-vs>
</div>
<style>b { display: flex; border: 2px solid #090; padding: 30px; margin: 30px; }</style>
```

## افقی طومار

```html
<div style="width:300px;border:1px solid #000">
  <c-hs>
    <b>1</b>
    <b>2</b>
    <b>3</b>
  </c-hs>
</div>
<style>b { display: flex; border: 2px solid #090; margin: 30px; padding: 30px; }</style>
```

## سٹائل حصوں

شیڈو DOM اندرونی عناصر کے انداز کو `::part()` کے ذریعے اپنی مرضی کے مطابق بنائیں:

- `::part(scroll)`: مواد کے علاقے کو اسکرول کریں۔
- `::part(bar)`: اسکرول بار ٹریک۔
- `::part(si)`: اسکرول بار سلائیڈر۔
- `::part(i)`: اندرونی لپیٹنے والی پرت۔
- `::part(drag)`: ڈریگنگ حالت میں اسکرول بار ٹریک۔

## سٹائل کلاس

- `.drag`: `body` عنصر پر لاگو ہوتا ہے، یہ بتاتا ہے کہ یہ ڈریگ اسکرولنگ حالت میں ہے۔

## حسب ضرورت خصوصیات

- `--si-bg`: سلائیڈر پس منظر کا رنگ۔
- `--si-anim`: سلائیڈر اینیمیشن۔
- `--cursorScrollhSvg`: ماؤس کرسر جب افقی اسکرول بار منڈلاتا ہے۔
- `--cursorScrollvSvg`: جب عمودی اسکرول بار منڈلاتا ہے تو ماؤس کا کرسر۔
