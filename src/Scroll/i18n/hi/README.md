# वर्चुअल स्क्रॉल बार

- **दोतरफा स्क्रॉलिंग**: लंबवत (`c-vs`) और क्षैतिज (`c-hs`) स्क्रॉलिंग प्रदान करता है।
- **आकार प्रतिक्रिया**: सामग्री और क्षेत्र के आकार में परिवर्तन की निगरानी करें, स्क्रॉल बार के आकार और स्थिति को स्वचालित रूप से अपडेट करें, और कोई अतिप्रवाह न होने पर स्वचालित रूप से छिपाएं।
- **इंटरैक्टिव समर्थन**: व्हील स्क्रॉलिंग, स्लाइडर ड्रैगिंग और ट्रैक क्लिक पोजिशनिंग का समर्थन करता है।
- **स्थिति छुपाएं**: होवर या स्क्रॉल करते समय स्लाइडर को दिखाएं और चौड़ा करें, और स्थिर होने पर इसे स्वचालित रूप से छिपाएं।

## लंबवत स्क्रॉल

```html
<div style="height:300px;border:1px solid #000">
  <c-vs>
    <b>1</b>
    <b>2</b>
    <b>3</b>
  </c-vs>
</div>
<style>b { display: flex; border: 2px solid #090; padding: 30px; margin: 30px; }</style>
```


## शैली भाग

`::part()` के माध्यम से शैडो DOM आंतरिक तत्वों की शैली को अनुकूलित करें:

- `::part(scroll)`: सामग्री क्षेत्र को स्क्रॉल करें।
- `::part(bar)`: स्क्रॉल बार ट्रैक।
- `::part(si)`: स्क्रॉल बार स्लाइडर।
- `::part(i)`: आंतरिक रैपिंग परत।
- `::part(drag)`: स्क्रॉल बार ट्रैक खींचने की स्थिति में है।

## शैली वर्ग

- `.drag`: `body` तत्व पर लागू, यह दर्शाता है कि यह ड्रैग-स्क्रॉलिंग स्थिति में है।

## कस्टम गुण

- `--si-bg`: स्लाइडर पृष्ठभूमि रंग.
- `--si-anim`: स्लाइडर एनीमेशन.
- `--cursorScrollhSvg`: जब क्षैतिज स्क्रॉल पट्टी मँडराती है तो माउस कर्सर।
- `--cursorScrollvSvg`: जब ऊर्ध्वाधर स्क्रॉल बार को घुमाया जाता है तो माउस कर्सर।
