# virtual scroll bar

- **နှစ်လမ်းသွားလှိမ့်ခြင်း**- ဒေါင်လိုက် (`c-vs`) နှင့် အလျားလိုက် (`c-hs`) လှိမ့်ခြင်းကို ပံ့ပိုးပေးသည်။
- **အရွယ်အစားတုံ့ပြန်မှု**- အကြောင်းအရာနှင့် ဧရိယာအရွယ်အစားပြောင်းလဲမှုများကို စောင့်ကြည့်စစ်ဆေးပါ၊ လှိမ့်ဘားအရွယ်အစားနှင့် အနေအထားကို အလိုအလျောက် အပ်ဒိတ်လုပ်ကာ လျှံခြင်းမရှိသည့်အခါ အလိုအလျောက် ဖျောက်ပါ။
- **အပြန်အလှန် ပံ့ပိုးမှု**- ဘီးလှိမ့်ခြင်း၊ လျှောဆွဲခြင်းနှင့် ခြေရာခံ ကလစ်တည်နေရာကို ပံ့ပိုးပေးသည်။
- **အခြေအနေ ဖျောက်ထားရန်**- ပျံဝဲနေချိန် သို့မဟုတ် လှိမ့်နေချိန်တွင် ဆလိုက်ဒါကို ပြပြီး ချဲ့ကာ လျှို့ဝှက်ထားသည့်အခါ ၎င်းကို အလိုအလျောက် ဝှက်ထားသည်။

## ဒေါင်လိုက်

```html
<div style="height:300px;border:1px solid #000">
  <c-vs>
    <b>1</b>
    <b>2</b>
    <b>3</b>
  </c-vs>
</div>
<style>b { display: flex; border: 2px solid #090; padding: 30px; margin: 30px; }</style>
```

```html
    <b>3</b>
  </c-hs>

```

## စတိုင်အစိတ်အပိုင်းများ

`::part()`မှတဆင့် Shadow DOM အတွင်းပိုင်းဒြပ်စင်များ၏စတိုင်ကို စိတ်ကြိုက်ပြင်ဆင်ပါ-

- `::part(scroll)`- အကြောင်းအရာဧရိယာကို ရွှေ့ပါ။
- `::part(bar)`- Scroll bar လမ်းကြောင်း။
- `::part(si)`- Scroll bar slider
- `::part(i)`- အတွင်းပိုင်းထုပ်ပိုးမှုအလွှာ။
- `::part(drag)`- ဆွဲယူသည့်အခြေအနေရှိ scroll bar လမ်းကြောင်း။

## စတိုင်အတန်း

- `.drag`: ဆွဲယူခြင်း အခြေအနေတွင် ရှိနေကြောင်း ညွှန်ပြသော `body` ဒြပ်စင်သို့ အသုံးချသည်။

## စိတ်ကြိုက်လုပ်မယ်။

- `--si-bg`- ဆလိုက်ဒါ နောက်ခံအရောင်။
- `--si-anim`- ဆလိုက်ဒါကာတွန်း။
- `--cursorScrollhSvg`- အလျားလိုက် scroll bar ကို ရွှေ့သည့်အခါ မောက်စ် ကာဆာ။
- `--cursorScrollvSvg`- ဒေါင်လိုက် scroll bar ကို ရွှေ့ထားသည့်အခါ မောက်စ် ကာဆာ။
