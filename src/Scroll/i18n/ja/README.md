# 仮想スクロールバー

- **双方向スクロール**: 垂直 (`c-vs`) スクロールと水平 (`c-hs`) スクロールを提供します。
- **サイズ応答**: コンテンツと領域のサイズの変化を監視し、スクロール バーのサイズと位置を自動的に更新し、オーバーフローがない場合は自動的に非表示にします。
- **インタラクティブ サポート**: ホイール スクロール、スライダーのドラッグ、トラック クリックの位置決めをサポートします。
- **ステータス非表示**: ホバーまたはスクロールするとスライダーが表示および拡大され、静止すると自動的に非表示になります。

## 縦スクロール

```html
<div style="height:300px;border:1px solid #000">
  <c-vs>
    <b>1</b>
    <b>2</b>
    <b>3</b>
  </c-vs>
</div>
<style>b { display: flex; border: 2px solid #090; padding: 30px; margin: 30px; }</style>
```

## 横スクロール

```html
<div style="width:300px;border:1px solid #000">
  <c-hs>
    <b>1</b>
    <b>2</b>
    <b>3</b>
  </c-hs>
</div>
<style>b { display: flex; border: 2px solid #090; margin: 30px; padding: 30px; }</style>
```

## スタイル部分

`::part()` を使用して Shadow DOM 内部要素のスタイルをカスタマイズします。

- `::part(scroll)`: コンテンツ領域をスクロールします。
- `::part(bar)`: スクロール バー トラック。
- `::part(si)`: スクロールバーのスライダー。
- `::part(i)`: 内部ラッピング層。
- `::part(drag)`: ドラッグ状態のスクロール バー トラック。

## スタイルクラス

- `.drag`: `body` 要素に適用され、ドラッグ スクロール状態にあることを示します。

## カスタムプロパティ

- `--si-bg`: スライダーの背景色。
- `--si-anim`: スライダー アニメーション。
- `--cursorScrollhSvg`: 水平スクロール バーをホバーしたときのマウス カーソル。
- `--cursorScrollvSvg`: 垂直スクロール バーをホバーしたときのマウス カーソル。
