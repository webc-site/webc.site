# மெய்நிகர் உருள் பட்டை

- **இருவழி ஸ்க்ரோலிங்**: செங்குத்து (`c-vs`) மற்றும் கிடைமட்ட (`c-hs`) ஸ்க்ரோலிங் வழங்குகிறது.
- **அளவு பதில்**: உள்ளடக்கம் மற்றும் பகுதி அளவு மாற்றங்களைக் கண்காணித்து, ஸ்க்ரோல் பார் அளவு மற்றும் நிலையை தானாகப் புதுப்பித்து, நிரம்பி வழியும் போது தானாக மறையும்.
- ** ஊடாடும் ஆதரவு**: வீல் ஸ்க்ரோலிங், ஸ்லைடர் இழுத்தல் மற்றும் ட்ராக் கிளிக் பொசிஷனிங் ஆகியவற்றை ஆதரிக்கிறது.
- **நிலை மறை**: வட்டமிடும்போது அல்லது ஸ்க்ரோலிங் செய்யும் போது ஸ்லைடரைக் காண்பிக்கவும் விரிவுபடுத்தவும், மேலும் நிலையானதாக இருக்கும்போது தானாகவே மறைக்கவும்.

## செங்குத்து சுருள்

```html
<div style="height:300px;border:1px solid #000">
  <c-vs>
    <b>1</b>
    <b>2</b>
    <b>3</b>
  </c-vs>
</div>
<style>b { display: flex; border: 2px solid #090; padding: 30px; margin: 30px; }</style>
```

## பாணி பாகங்கள்

`::part()` வழியாக நிழல் DOM உள் உறுப்புகளின் பாணியைத் தனிப்பயனாக்கவும்:

- `::part(scroll)`: உள்ளடக்க பகுதியை உருட்டவும்.
- `::part(bar)`: ஸ்க்ரோல் பார் டிராக்.
- `::part(si)`: ஸ்க்ரோல் பார் ஸ்லைடர்.
- `::part(i)`: உள் மடக்கு அடுக்கு.
- `::part(drag)`: இழுக்கும் நிலையில் உள்ள ஸ்க்ரோல் பார் டிராக்.

## பாணி வகுப்பு

- `.drag`: `body` உறுப்புக்குப் பயன்படுத்தப்பட்டது, இது இழுக்கும்-ஸ்க்ரோலிங் நிலையில் இருப்பதைக் குறிக்கிறது.

## தனிப்பயன் பண்புகள்

- `--si-bg`: ஸ்லைடர் பின்னணி நிறம்.
- `--si-anim`: ஸ்லைடர் அனிமேஷன்.
- `--cursorScrollhSvg`: கிடைமட்ட ஸ்க்ரோல் பட்டியை நகர்த்தும்போது மவுஸ் கர்சர்.
- `--cursorScrollvSvg`: செங்குத்து உருள் பட்டை நகர்த்தும்போது மவுஸ் கர்சர்.
