# ແຖບເລື່ອນ virtual

- **ການເລື່ອນສອງທາງ**: ໃຫ້ການເລື່ອນແນວຕັ້ງ (`c-vs`) ແລະແນວນອນ (`c-hs`) ເລື່ອນ.
- ** ການ​ຕອບ​ສະ​ຫນອງ​ຂະ​ຫນາດ **​: ຕິດ​ຕາມ​ກວດ​ກາ​ເນື້ອ​ໃນ​ແລະ​ການ​ປ່ຽນ​ແປງ​ຂະ​ຫນາດ​ຂອງ​ພື້ນ​ທີ່​, ການ​ປັບ​ປຸງ​ອັດ​ຕະ​ໂນ​ມັດ​ການ​ປັບ​ຂະ​ຫນາດ​ແຖບ​ເລື່ອນ​ແລະ​ຕໍາ​ແຫນ່ງ​, ແລະ​ອັດ​ຕະ​ໂນ​ມັດ​ຊ່ອນ​ໃນ​ເວ​ລາ​ທີ່​ບໍ່​ມີ overflow​.
- **ສະ​ຫນັບ​ສະ​ຫນູນ​ການ​ໂຕ້​ຕອບ **​: ສະ​ຫນັບ​ສະ​ຫນູນ​ການ​ເລື່ອນ​ລໍ້​, slider drag ແລະ​ຕິດ​ຕາມ​ຕໍາ​ແຫນ່ງ​ຄລິກ​.
- **ເຊື່ອງສະຖານະ**: ສະແດງ ແລະຂະຫຍາຍຕົວເລື່ອນເມື່ອເລື່ອນ ຫຼືເລື່ອນ, ແລະເຊື່ອງມັນໂດຍອັດຕະໂນມັດເມື່ອຢູ່ສະຖາທີ່.

## ເລື່ອນແນວຕັ້ງ

```html
<div style="height:300px;border:1px solid #000">
  <c-vs>
    <b>1</b>
    <b>2</b>
    <b>3</b>
  </c-vs>
</div>
<style>b { display: flex; border: 2px solid #090; padding: 30px; margin: 30px; }</style>
```

## ພາກສ່ວນສະໄຕ

ປັບແຕ່ງຮູບແບບຂອງອົງປະກອບພາຍໃນຂອງ Shadow DOM ຜ່ານ `::part()`:

- `::part(scroll)`: ເລື່ອນພື້ນທີ່ເນື້ອຫາ.
- `::part(bar)`: ແຖບແຖບເລື່ອນ.
- `::part(si)`: ແຖບເລື່ອນແຖບເລື່ອນ.
- `::part(i)`: ຊັ້ນຫໍ່ພາຍໃນ.
- `::part(drag)`: ແຖບເລື່ອນຢູ່ໃນສະຖານະລາກ.

## ຫ້ອງຮຽນແບບ

- `.drag`: ນຳໃຊ້ກັບອົງປະກອບ `body`, ຊີ້ບອກວ່າມັນຢູ່ໃນສະຖານະແບບລາກເລື່ອນ.

## ຄຸນສົມບັດທີ່ກໍາຫນົດເອງ

- `--si-bg`: ສີພື້ນຫຼັງຕົວເລື່ອນ.
- `--si-anim`: ອະນິເມຊັນຕົວເລື່ອນ.
- `--cursorScrollhSvg`: ຕົວຊີ້ເມົ້າ ເມື່ອແຖບເລື່ອນແນວນອນຖືກເລື່ອນ.
- `--cursorScrollvSvg`: ຕົວຊີ້ເມົ້າ ເມື່ອແຖບເລື່ອນແນວຕັ້ງຖືກເລື່ອນ.
