# نوار پیمایش مجازی

- **پیمایش دو طرفه**: اسکرول عمودی (`c-vs`) و افقی (`c-hs`) را فراهم می کند.
- **پاسخ اندازه**: تغییرات محتوا و اندازه منطقه را بررسی کنید، اندازه و موقعیت نوار پیمایش را به طور خودکار به روز کنید و در صورت عدم سرریز به طور خودکار پنهان شوید.
- **پشتیبانی تعاملی**: پشتیبانی از اسکرول چرخ، کشیدن نوار لغزنده و تعیین موقعیت کلیک ردیابی.
- **Status Hide**: نمایش و باز کردن نوار لغزنده هنگام شناور کردن یا پیمایش، و به طور خودکار پنهان کردن آن در صورت ثابت بودن.

## اسکرول عمودی

```html
<div style="height:300px;border:1px solid #000">
  <c-vs>
    <b>1</b>
    <b>2</b>
    <b>3</b>
  </c-vs>
</div>
<style>b { display: flex; border: 2px solid #090; padding: 30px; margin: 30px; }</style>
```

## اسکرول افقی

```html
<div style="width:300px;border:1px solid #000">
  <c-hs>
    <b>1</b>
    <b>2</b>
    <b>3</b>
  </c-hs>
</div>
<style>b { display: flex; border: 2px solid #090; margin: 30px; padding: 30px; }</style>
```

## قطعات سبک

سفارشی کردن سبک عناصر داخلی Shadow DOM از طریق `::part()`:

- `::part(scroll)`: ناحیه محتوا را پیمایش کنید.
- `::part(bar)`: مسیر نوار پیمایش.
- `::part(si)`: نوار لغزنده پیمایش.
- `::part(i)`: لایه بسته بندی داخلی.
- `::part(drag)`: مسیر نوار پیمایش در حالت کشیدن.

## کلاس سبک

- `.drag`: روی عنصر `body` اعمال می‌شود که نشان می‌دهد در حالت کشیدن اسکرول است.

## خواص سفارشی

- `--si-bg`: رنگ پس‌زمینه لغزنده.
- `--si-anim`: انیمیشن لغزنده.
- `--cursorScrollhSvg`: مکان‌نمای ماوس هنگامی که نوار اسکرول افقی شناور است.
- `--cursorScrollvSvg`: مکان‌نمای ماوس هنگامی که نوار پیمایش عمودی شناور است.
