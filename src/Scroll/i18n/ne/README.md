# भर्चुअल स्क्रोल पट्टी

- **दुई-तर्फी स्क्रोलिङ**: ठाडो (`c-vs`) र तेर्सो (`c-hs`) स्क्रोलिङ प्रदान गर्दछ।
- **आकार प्रतिक्रिया**: सामग्री र क्षेत्र आकार परिवर्तनहरू निगरानी गर्नुहोस्, स्वचालित रूपमा स्क्रोल बार आकार र स्थिति अद्यावधिक गर्नुहोस्, र कुनै ओभरफ्लो नभएको बेला स्वचालित रूपमा लुकाउनुहोस्।
- **अन्तर्क्रियात्मक समर्थन**: व्हील स्क्रोलिङ, स्लाइडर ड्र्यागिङ र ट्र्याक क्लिक स्थिति समर्थन गर्दछ।
- **स्थिति लुकाउनुहोस्**: होभर गर्दा वा स्क्रोल गर्दा स्लाइडर देखाउनुहोस् र फराकिलो गर्नुहोस्, र स्थिर हुँदा स्वचालित रूपमा लुकाउनुहोस्।

## ठाडो स्क्रोल

```html
<div style="height:300px;border:1px solid #000">
  <c-vs>
    <b>1</b>
    <b>2</b>
    <b>3</b>
  </c-vs>
</div>
<style>b { display: flex; border: 2px solid #090; padding: 30px; margin: 30px; }</style>
```

## तेर्सो स्क्रोल

```html
<div style="width:300px;border:1px solid #000">
  <c-hs>
    <b>1</b>
    <b>2</b>
    <b>3</b>
  </c-hs>
</div>
<style>b { display: flex; border: 2px solid #090; margin: 30px; padding: 30px; }</style>
```

## शैली भागहरू

`::part()` मार्फत छाया DOM आन्तरिक तत्वहरूको शैली अनुकूलित गर्नुहोस्:

- `::part(scroll)`: सामग्री क्षेत्र स्क्रोल गर्नुहोस्।
- `::part(bar)`: स्क्रोल बार ट्र्याक।
- `::part(si)`: स्क्रोल बार स्लाइडर।
- `::part(i)`: आन्तरिक र्‍यापिङ तह।
- `::part(drag)`: ड्र्यागिङ अवस्थामा स्क्रोल बार ट्र्याक।

## शैली वर्ग

- `.drag`: `body` तत्वमा लागू गरियो, यो ड्र्याग-स्क्रोलिङ अवस्थामा रहेको जनाउँछ।

## अनुकूलन गुणहरू

- `--si-bg`: स्लाइडर पृष्ठभूमि रङ।
- `--si-anim`: स्लाइडर एनिमेसन।
- `--cursorScrollhSvg`: तेर्सो स्क्रोल पट्टी होभर हुँदा माउस कर्सर।
- `--cursorScrollvSvg`: ठाडो स्क्रोल बार होभर हुँदा माउस कर्सर।
