# వర్చువల్ స్క్రోల్ బార్

- **రెండు-మార్గం స్క్రోలింగ్**: నిలువు (`c-vs`) మరియు క్షితిజ సమాంతర (`c-hs`) స్క్రోలింగ్‌ను అందిస్తుంది.
- **పరిమాణ ప్రతిస్పందన**: కంటెంట్ మరియు ప్రాంతం పరిమాణం మార్పులను పర్యవేక్షించండి, స్క్రోల్ బార్ పరిమాణం మరియు స్థానాన్ని స్వయంచాలకంగా నవీకరించండి మరియు ఓవర్‌ఫ్లో లేనప్పుడు స్వయంచాలకంగా దాచండి.
- **ఇంటరాక్టివ్ సపోర్ట్**: వీల్ స్క్రోలింగ్, స్లయిడర్ లాగడం మరియు ట్రాక్ క్లిక్ పొజిషనింగ్‌కు మద్దతు ఇస్తుంది.
- ** స్థితి దాచు**: హోవర్ చేస్తున్నప్పుడు లేదా స్క్రోలింగ్ చేస్తున్నప్పుడు స్లయిడర్‌ను చూపండి మరియు విస్తరించండి మరియు నిశ్చలంగా ఉన్నప్పుడు స్వయంచాలకంగా దాచండి.

## నిలువు స్క్రోల్

```html
<div style="height:300px;border:1px solid #000">
  <c-vs>
    <b>1</b>
    <b>2</b>
    <b>3</b>
  </c-vs>
</div>
<style>b { display: flex; border: 2px solid #090; padding: 30px; margin: 30px; }</style>
```

## శైలి భాగాలు

షాడో DOM అంతర్గత మూలకాల శైలిని `::part()` ద్వారా అనుకూలీకరించండి:

- `::part(scroll)`: కంటెంట్ ప్రాంతాన్ని స్క్రోల్ చేయండి.
- `::part(bar)`: స్క్రోల్ బార్ ట్రాక్.
- `::part(si)`: స్క్రోల్ బార్ స్లయిడర్.
- `::part(i)`: అంతర్గత చుట్టే పొర.
- `::part(drag)`: డ్రాగ్ చేసే స్థితిలో ఉన్న స్క్రోల్ బార్ ట్రాక్.

## శైలి తరగతి

- `.drag`: `body` మూలకానికి వర్తింపజేయబడింది, ఇది డ్రాగ్-స్క్రోలింగ్ స్థితిలో ఉందని సూచిస్తుంది.

## అనుకూల లక్షణాలు

- `--si-bg`: స్లైడర్ నేపథ్య రంగు.
- `--si-anim`: స్లైడర్ యానిమేషన్.
- `--cursorScrollhSvg`: క్షితిజ సమాంతర స్క్రోల్ బార్ హోవర్ చేసినప్పుడు మౌస్ కర్సర్.
- `--cursorScrollvSvg`: నిలువు స్క్రోల్ బార్ హోవర్ చేసినప్పుడు మౌస్ కర్సర్.
