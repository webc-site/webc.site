# ভাৰ্চুৱেল স্ক্ৰল বাৰ

- **দুমুখীয়া স্ক্ৰলিং**: উলম্ব (`c-vs`) আৰু অনুভূমিক (`c-hs`) স্ক্ৰলিং প্ৰদান কৰে।
- **আকাৰৰ সঁহাৰি**: বিষয়বস্তু আৰু এলেকাৰ আকাৰ পৰিবৰ্তনসমূহ নিৰীক্ষণ কৰক, স্ক্ৰল বাৰৰ আকাৰ আৰু অৱস্থান স্বয়ংক্ৰিয়ভাৱে আপডেইট কৰক, আৰু কোনো অভাৰফ্ল' নহ'লে স্বয়ংক্ৰিয়ভাৱে লুকুৱাওক।
- **পাৰস্পৰিক সমৰ্থন**: চকা স্ক্ৰলিং, স্লাইডাৰ ড্ৰেগিং আৰু ট্ৰেক ক্লিক অৱস্থান সমৰ্থন কৰে।
- **অৱস্থা লুকুৱাওক**: হ'ভাৰ বা স্ক্ৰল কৰাৰ সময়ত স্লাইডাৰ দেখুৱাওক আৰু বহল কৰক, আৰু স্থবিৰ হ'লে স্বয়ংক্ৰিয়ভাৱে লুকুৱাওক।

## উলম্ব স্ক্ৰল

```html
<div style="height:300px;border:1px solid #000">
  <c-vs>
    <b>1</b>
    <b>2</b>
    <b>3</b>
  </c-vs>
</div>
<style>b { display: flex; border: 2px solid #090; padding: 30px; margin: 30px; }</style>
```

## অনুভূমিক স্ক্ৰল


## শৈলীৰ অংশ

`::part()` ৰ যোগেদি ছাঁয়া DOM অভ্যন্তৰীণ উপাদানসমূহৰ শৈলী স্বনিৰ্বাচিত কৰক:

- `::part(scroll)`: বিষয়বস্তু অঞ্চল স্ক্ৰল কৰক।
- `::part(bar)`: স্ক্ৰল বাৰ ট্ৰেক।
- `::part(si)`: স্ক্ৰল বাৰ স্লাইডাৰ।
- `::part(i)`: আভ্যন্তৰীণ ৰেপিং স্তৰ।
- `::part(drag)`: ড্ৰেগিং অৱস্থাত স্ক্ৰল বাৰ ট্ৰেক।

## শৈলী শ্ৰেণী

- `.drag`: `body` উপাদানত প্ৰয়োগ কৰা হৈছে, ই এটা ড্ৰেগ-স্ক্ৰলিং অৱস্থাত আছে বুলি সূচায়।

## স্বনিৰ্বাচিত বৈশিষ্ট্যসমূহ

- `--si-bg`: স্লাইডাৰৰ পটভূমিৰ ৰং।
- `--si-anim`: স্লাইডাৰ এনিমেচন।
- `--cursorScrollhSvg`: মাউছ কাৰ্চাৰ যেতিয়া অনুভূমিক স্ক্ৰল বাৰ ৰখা হয়।
- `--cursorScrollvSvg`: মাউছ কাৰ্চাৰ যেতিয়া উলম্ব স্ক্ৰল বাৰ ৰখা হয়।
