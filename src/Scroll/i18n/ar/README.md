# شريط التمرير الظاهري

- **التمرير في اتجاهين**: يوفر التمرير الرأسي (`c-vs`) والأفقي (`c-hs`).
- **استجابة الحجم**: مراقبة تغيرات حجم المحتوى والمنطقة، وتحديث حجم شريط التمرير وموضعه تلقائيًا، والإخفاء تلقائيًا عند عدم وجود تجاوز.
- **الدعم التفاعلي**: يدعم التمرير على العجلة وسحب شريط التمرير وتحديد موضع النقر.
- **إخفاء الحالة**: إظهار شريط التمرير وتوسيعه عند التمرير أو التمرير، وإخفائه تلقائيًا عند الثبات.

## التمرير العمودي

```html
<div style="height:300px;border:1px solid #000">
  <c-vs>
    <b>1</b>
    <b>2</b>
    <b>3</b>
  </c-vs>
</div>
<style>b { display: flex; border: 2px solid #090; padding: 30px; margin: 30px; }</style>
```

## التمرير الأفقي

```html
<div style="width:300px;border:1px solid #000">
  <c-hs>
    <b>1</b>
    <b>2</b>
    <b>3</b>
  </c-hs>
</div>
<style>b { display: flex; border: 2px solid #090; margin: 30px; padding: 30px; }</style>
```

## أجزاء النمط

قم بتخصيص نمط عناصر Shadow DOM الداخلية عبر `::part()`:

- `::part(scroll)`: قم بالتمرير في منطقة المحتوى.
- `::part(bar)`: مسار شريط التمرير.
- `::part(si)`: منزلق شريط التمرير.
- `::part(i)`: طبقة التغليف الداخلية.
- `::part(drag)`: مسار شريط التمرير في حالة السحب.

## فئة النمط

- `.drag`: يتم تطبيقه على العنصر `body`، مما يشير إلى أنه في حالة سحب التمرير.

## خصائص مخصصة

- `--si-bg`: لون خلفية شريط التمرير.
- `--si-anim`: الرسوم المتحركة المنزلقة.
- `--cursorScrollhSvg`: مؤشر الماوس عند تحريك شريط التمرير الأفقي.
- `--cursorScrollvSvg`: مؤشر الماوس عند تحريك شريط التمرير العمودي.
