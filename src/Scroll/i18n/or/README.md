# ଭର୍ଚୁଆଲ୍ ସ୍କ୍ରୋଲ୍ ବାର୍ |

- ** ଦୁଇ-ମାର୍ଗ ସ୍କ୍ରୋଲିଂ **: ଭୂଲମ୍ବ (`c-vs`) ଏବଂ ଭୂସମାନ୍ତର (`c-hs`) ସ୍କ୍ରୋଲିଂ ପ୍ରଦାନ କରେ |
- ** ଆକାର ପ୍ରତିକ୍ରିୟା **: ବିଷୟବସ୍ତୁ ଏବଂ କ୍ଷେତ୍ର ଆକାର ପରିବର୍ତ୍ତନ ଉପରେ ନଜର ରଖନ୍ତୁ, ସ୍କ୍ରୋଲ୍ ବାର୍ ଆକାର ଏବଂ ସ୍ଥିତିକୁ ସ୍ୱୟଂଚାଳିତ ଭାବରେ ଅଦ୍ୟତନ କରନ୍ତୁ, ଏବଂ ଯେତେବେଳେ କ over ଣସି ଓଭରଫ୍ଲୋ ନଥାଏ ସ୍ୱୟଂଚାଳିତ ଭାବରେ ଲୁଚାନ୍ତୁ |
- ** ଇଣ୍ଟରାକ୍ଟିଭ୍ ସପୋର୍ଟ **: ଚକ ସ୍କ୍ରୋଲିଂ, ସ୍ଲାଇଡର୍ ଡ୍ରାଗ୍ ଏବଂ କ୍ଲିକ୍ ପୋଜିସନ୍ ଟ୍ରାକ୍ କୁ ସମର୍ଥନ କରେ |
- ** ସ୍ଥିତି ଲୁଚାନ୍ତୁ **: ଘୁଞ୍ଚିବା କିମ୍ବା ସ୍କ୍ରୋଲ୍ କରିବା ସମୟରେ ସ୍ଲାଇଡର୍ ଦେଖାନ୍ତୁ ଏବଂ ପ୍ରଶସ୍ତ କରନ୍ତୁ, ଏବଂ ସ୍ଥିର ହେଲେ ସ୍ୱୟଂଚାଳିତ ଭାବରେ ଏହାକୁ ଲୁଚାନ୍ତୁ |

## ଭୂଲମ୍ବ ସ୍କ୍ରୋଲ୍ |

```html
<div style="height:300px;border:1px solid #000">
  <c-vs>
    <b>1</b>
    <b>2</b>
    <b>3</b>
  </c-vs>
</div>
<style>b { display: flex; border: 2px solid #090; padding: 30px; margin: 30px; }</style>
```

## ଭୂସମାନ୍ତର ସ୍କ୍ରୋଲ୍ |

```html
<div style="width:300px;border:1px solid #000">
  <c-hs>
    <b>1</b>
    <b>2</b>
    <b>3</b>
  </c-hs>
</div>
<style>b { display: flex; border: 2px solid #090; margin: 30px; padding: 30px; }</style>
```

## ଶ style ଳୀ ଅଂଶଗୁଡିକ |

ଛାୟା DOM ଆଭ୍ୟନ୍ତରୀଣ ଉପାଦାନଗୁଡିକର ଶ style ଳୀକୁ `::part()` ମାଧ୍ୟମରେ କଷ୍ଟମାଇଜ୍ କରନ୍ତୁ:

- `::part(scroll)`: ବିଷୟବସ୍ତୁ କ୍ଷେତ୍ରକୁ ସ୍କ୍ରୋଲ୍ କରନ୍ତୁ |
- `::part(bar)`: ବାର୍ ଟ୍ରାକ୍ ସ୍କ୍ରୋଲ୍ କରନ୍ତୁ |
- `::part(si)`: ସ୍ଲାଇଡ୍ ବାର୍ ସ୍ଲାଇଡର୍ |
- `::part(i)`: ଆଭ୍ୟନ୍ତରୀଣ ରାପିଙ୍ଗ୍ ସ୍ତର |
- `::part(drag)`: ଡ୍ରାଗ୍ ଅବସ୍ଥାରେ ସ୍କ୍ରୋଲ୍ ବାର୍ ଟ୍ରାକ୍ |

## ଶ style ଳୀ ଶ୍ରେଣୀ

- `.drag`: `body` ଉପାଦାନରେ ପ୍ରଯୁଜ୍ୟ, ଏହା ଦର୍ଶାଏ ଯେ ଏହା ଏକ ଡ୍ରାଗ୍ ସ୍କ୍ରୋଲିଂ ଅବସ୍ଥାରେ ଅଛି |

## କଷ୍ଟମ୍ ଗୁଣ

- `--si-bg`: ସ୍ଲାଇଡର୍ ପୃଷ୍ଠଭୂମି ରଙ୍ଗ |
- `--si-anim`: ସ୍ଲାଇଡର୍ ଆନିମେସନ୍ |
- `--cursorScrollhSvg`: ଯେତେବେଳେ ଭୂସମାନ୍ତର ସ୍କ୍ରୋଲ୍ ବାର୍ ହୋଭର୍ ହୁଏ ମାଉସ୍ କର୍ସର୍ |
- `--cursorScrollvSvg`: ଯେତେବେଳେ ଭୂଲମ୍ବ ସ୍କ୍ରୋଲ୍ ବାର୍ ହୋଭର୍ ହୁଏ ମାଉସ୍ କର୍ସର୍ |
