# ವರ್ಚುವಲ್ ಸ್ಕ್ರಾಲ್ ಬಾರ್

- **ದ್ವಿಮುಖ ಸ್ಕ್ರೋಲಿಂಗ್**: ಲಂಬ (`c-vs`) ಮತ್ತು ಅಡ್ಡ (`c-hs`) ಸ್ಕ್ರೋಲಿಂಗ್ ಅನ್ನು ಒದಗಿಸುತ್ತದೆ.
- **ಗಾತ್ರದ ಪ್ರತಿಕ್ರಿಯೆ**: ವಿಷಯ ಮತ್ತು ಪ್ರದೇಶದ ಗಾತ್ರ ಬದಲಾವಣೆಗಳನ್ನು ಮೇಲ್ವಿಚಾರಣೆ ಮಾಡಿ, ಸ್ಕ್ರಾಲ್ ಬಾರ್ ಗಾತ್ರ ಮತ್ತು ಸ್ಥಾನವನ್ನು ಸ್ವಯಂಚಾಲಿತವಾಗಿ ನವೀಕರಿಸಿ ಮತ್ತು ಯಾವುದೇ ಓವರ್‌ಫ್ಲೋ ಇಲ್ಲದಿದ್ದಾಗ ಸ್ವಯಂಚಾಲಿತವಾಗಿ ಮರೆಮಾಡಿ.
- ** ಸಂವಾದಾತ್ಮಕ ಬೆಂಬಲ **: ಚಕ್ರ ಸ್ಕ್ರೋಲಿಂಗ್, ಸ್ಲೈಡರ್ ಡ್ರ್ಯಾಗ್ ಮತ್ತು ಟ್ರ್ಯಾಕ್ ಕ್ಲಿಕ್ ಸ್ಥಾನೀಕರಣವನ್ನು ಬೆಂಬಲಿಸುತ್ತದೆ.
- ** ಸ್ಥಿತಿ ಮರೆಮಾಡಿ**: ತೂಗಾಡುತ್ತಿರುವಾಗ ಅಥವಾ ಸ್ಕ್ರೋಲಿಂಗ್ ಮಾಡುವಾಗ ಸ್ಲೈಡರ್ ಅನ್ನು ತೋರಿಸಿ ಮತ್ತು ವಿಸ್ತರಿಸಿ ಮತ್ತು ಸ್ಥಾಯಿಯಾಗಿರುವಾಗ ಅದನ್ನು ಸ್ವಯಂಚಾಲಿತವಾಗಿ ಮರೆಮಾಡಿ.

## ಲಂಬ ಸ್ಕ್ರಾಲ್

```html
<div style="height:300px;border:1px solid #000">
  <c-vs>
    <b>1</b>
    <b>2</b>
    <b>3</b>
  </c-vs>
</div>
<style>b { display: flex; border: 2px solid #090; padding: 30px; margin: 30px; }</style>
```


## ಶೈಲಿಯ ಭಾಗಗಳು

`::part()` ಮೂಲಕ Shadow DOM ಆಂತರಿಕ ಅಂಶಗಳ ಶೈಲಿಯನ್ನು ಕಸ್ಟಮೈಸ್ ಮಾಡಿ:

- `::part(scroll)`: ವಿಷಯ ಪ್ರದೇಶವನ್ನು ಸ್ಕ್ರಾಲ್ ಮಾಡಿ.
- `::part(bar)`: ಸ್ಕ್ರಾಲ್ ಬಾರ್ ಟ್ರ್ಯಾಕ್.
- `::part(si)`: ಸ್ಕ್ರಾಲ್ ಬಾರ್ ಸ್ಲೈಡರ್.
- `::part(i)`: ಆಂತರಿಕ ಸುತ್ತುವ ಪದರ.
- `::part(drag)`: ಡ್ರ್ಯಾಗ್ ಮಾಡುವ ಸ್ಥಿತಿಯಲ್ಲಿ ಸ್ಕ್ರಾಲ್ ಬಾರ್ ಟ್ರ್ಯಾಕ್.

## ಶೈಲಿ ವರ್ಗ

- `.drag`: `body` ಅಂಶಕ್ಕೆ ಅನ್ವಯಿಸಲಾಗಿದೆ, ಇದು ಡ್ರ್ಯಾಗ್-ಸ್ಕ್ರೋಲಿಂಗ್ ಸ್ಥಿತಿಯಲ್ಲಿದೆ ಎಂದು ಸೂಚಿಸುತ್ತದೆ.

## ಕಸ್ಟಮ್ ಗುಣಲಕ್ಷಣಗಳು

- `--si-bg`: ಸ್ಲೈಡರ್ ಹಿನ್ನೆಲೆ ಬಣ್ಣ.
- `--si-anim`: ಸ್ಲೈಡರ್ ಅನಿಮೇಷನ್.
- `--cursorScrollhSvg`: ಸಮತಲವಾದ ಸ್ಕ್ರಾಲ್ ಬಾರ್ ಅನ್ನು ಹೋವರ್ ಮಾಡಿದಾಗ ಮೌಸ್ ಕರ್ಸರ್.
- `--cursorScrollvSvg`: ಲಂಬ ಸ್ಕ್ರಾಲ್ ಬಾರ್ ಅನ್ನು ಹೋವರ್ ಮಾಡಿದಾಗ ಮೌಸ್ ಕರ್ಸರ್.
