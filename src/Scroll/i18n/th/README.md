# แถบเลื่อนเสมือน

- **การเลื่อนแบบสองทาง**: ให้การเลื่อนในแนวตั้ง (`c-vs`) และแนวนอน (`c-hs`)
- **การตอบสนองขนาด**: ตรวจสอบเนื้อหาและการเปลี่ยนแปลงขนาดพื้นที่ อัปเดตขนาดและตำแหน่งของแถบเลื่อนโดยอัตโนมัติ และซ่อนโดยอัตโนมัติเมื่อไม่มีการล้น
- **การสนับสนุนแบบโต้ตอบ**: รองรับการเลื่อนล้อ การลากตัวเลื่อน และการติดตามตำแหน่งการคลิก
- **ซ่อนสถานะ**: แสดงและขยายแถบเลื่อนเมื่อเลื่อนเมาส์หรือเลื่อน และซ่อนโดยอัตโนมัติเมื่ออยู่กับที่

## เลื่อนแนวตั้ง

```html
<div style="height:300px;border:1px solid #000">
  <c-vs>
    <b>1</b>
    <b>2</b>
    <b>3</b>
  </c-vs>
</div>
<style>b { display: flex; border: 2px solid #090; padding: 30px; margin: 30px; }</style>
```

## เลื่อนแนวนอน

```html
<div style="width:300px;border:1px solid #000">
  <c-hs>
    <b>1</b>
    <b>2</b>
    <b>3</b>
  </c-hs>
</div>
<style>b { display: flex; border: 2px solid #090; margin: 30px; padding: 30px; }</style>
```

## ส่วนสไตล์

ปรับแต่งสไตล์ขององค์ประกอบภายใน Shadow DOM ผ่าน `::part()`:

- `::part(scroll)`: เลื่อนพื้นที่เนื้อหา
- `::part(bar)`: แทร็กแถบเลื่อน
- `::part(si)`: ตัวเลื่อนแถบเลื่อน
- `::part(i)`: เลเยอร์การห่อภายใน
- `::part(drag)`: แทร็กแถบเลื่อนในสถานะการลาก

## คลาสสไตล์

- `.drag`: ใช้กับองค์ประกอบ `body` ซึ่งบ่งชี้ว่าอยู่ในสถานะลากเลื่อน

## คุณสมบัติที่กำหนดเอง

- `--si-bg`: สีพื้นหลังของแถบเลื่อน
- `--si-anim`: ภาพเคลื่อนไหวของแถบเลื่อน
- `--cursorScrollhSvg`: เคอร์เซอร์ของเมาส์เมื่อเลื่อนแถบเลื่อนแนวนอน
- `--cursorScrollvSvg`: เคอร์เซอร์ของเมาส์เมื่อเลื่อนแถบเลื่อนแนวตั้ง
