# අතථ්‍ය අනුචලන තීරුව

- **ද්වි-මාර්ග අනුචලනය**: සිරස් (`c-vs`) සහ තිරස් (`c-hs`) අනුචලනය සපයයි.
- **ප්‍රමාණයේ ප්‍රතිචාරය**: අන්තර්ගතය සහ ප්‍රදේශයේ ප්‍රමාණයේ වෙනස්කම් නිරීක්ෂණය කරන්න, අනුචලන තීරුවේ ප්‍රමාණය සහ ස්ථානය ස්වයංක්‍රීයව යාවත්කාලීන කරන්න, සහ පිටාර ගැලීමක් නොමැති විට ස්වයංක්‍රීයව සඟවන්න.
- ** අන්තර්ක්‍රියාකාරී සහාය**: රෝද අනුචලනය, ස්ලයිඩරය ඇදගෙන යාම සහ ට්‍රැක් ක්ලික් ස්ථානගත කිරීම සඳහා සහය දක්වයි.
- **තත්ත්වය සඟවන්න**: සැරිසරන විට හෝ අනුචලනය කරන විට ස්ලයිඩරය පෙන්වන්න සහ පුළුල් කරන්න, සහ ස්ථාවර විට එය ස්වයංක්‍රීයව සඟවන්න.

## සිරස් අනුචලනය

```html
<div style="height:300px;border:1px solid #000">
  <c-vs>
    <b>1</b>
    <b>2</b>
    <b>3</b>
  </c-vs>
</div>
<style>b { display: flex; border: 2px solid #090; padding: 30px; margin: 30px; }</style>
```

## ශෛලිය කොටස්

`::part()` හරහා Shadow DOM අභ්‍යන්තර මූලද්‍රව්‍යවල විලාසය අභිරුචිකරණය කරන්න:

- `::part(scroll)`: අන්තර්ගත ප්‍රදේශය අනුචලනය කරන්න.
- `::part(bar)`: අනුචලන තීරු ධාවන පථය.
- `::part(si)`: අනුචලන තීරු ස්ලයිඩරය.
- `::part(i)`: අභ්‍යන්තර එතීමේ ස්තරය.
- `::part(drag)`: ඇදගෙන යන තත්වයේ ඇති අනුචලන තීරු ඛණ්ඩය.

## ශෛලිය පන්තිය

- `.drag`: `body` මූලද්‍රව්‍යයට යොදන ලද, එය ඇදගෙන යාමේ-අනුචලන තත්වයක පවතින බව දක්වයි.

## අභිරුචි ගුණාංග

- `--si-bg`: ස්ලයිඩර් පසුබිම් වර්ණය.
- `--si-anim`: ස්ලයිඩර් සජීවිකරණය.
- `--cursorScrollhSvg`: තිරස් අනුචලන තීරුව සැරිසරන විට මූසික කර්සරය.
- `--cursorScrollvSvg`: සිරස් අනුචලන තීරුව සැරිසරන විට මූසික කර්සරය.
