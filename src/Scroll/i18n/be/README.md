# віртуальная паласа пракруткі

- **Двухбаковая пракрутка**: Забяспечвае вертыкальную (`c-vs`) і гарызантальную (`c-hs`) пракрутку.
- **Адказ па памеры**: Кантралюйце змены памеру змесціва і вобласці, аўтаматычна абнаўляйце памер і становішча паласы пракруткі і аўтаматычна хавайце, калі няма перапаўнення.
- **Інтэрактыўная падтрымка**: падтрымлівае прагортку колы, перацягванне паўзунка і пазіцыянаванне трэка.
- **Схаваць статус**: паказваць і пашыраць паўзунок пры навядзенні або прагортцы і аўтаматычна хаваць яго, калі ён нерухомы.

## вертыкальная пракрутка

```html
<div style="height:300px;border:1px solid #000">
  <c-vs>
    <b>1</b>
    <b>2</b>
    <b>3</b>
  </c-vs>
</div>
<style>b { display: flex; border: 2px solid #090; padding: 30px; margin: 30px; }</style>
```

## гарызантальная пракрутка

```html
<div style="width:300px;border:1px solid #000">
  <c-hs>
    <b>1</b>
    <b>2</b>
    <b>3</b>
  </c-hs>
</div>
<style>b { display: flex; border: 2px solid #090; margin: 30px; padding: 30px; }</style>
```

## часткі стылю

Наладзьце стыль унутраных элементаў Shadow DOM праз `::part()`:

- `::part(scroll)`: пракруціць вобласць змесціва.
- `::part(bar)`: дарожка паласы пракруткі.
- `::part(si)`: паўзунок паласы пракруткі.
- `::part(i)`: унутраны слой абгорткі.
- `::part(drag)`: дарожка паласы пракруткі ў стане перацягвання.

## клас стылю

- `.drag`: Ужываецца да элемента `body`, паказваючы, што ён знаходзіцца ў стане пракруткі перацягвання.

## Карыстальніцкія ўласцівасці

- `--si-bg`: фонавы колер паўзунка.
- `--si-anim`: Анімацыя паўзунка.
- `--cursorScrollhSvg`: Курсор мышы пры навядзенні на гарызантальную паласу пракруткі.
- `--cursorScrollvSvg`: Курсор мышы пры навядзенні на вертыкальную паласу пракруткі.
