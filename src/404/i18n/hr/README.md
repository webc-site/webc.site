# Put ne postoji stranica s porukom o pogrešci

- Stranica upita nije pronađena, navedite gumb za povratak na početnu stranicu.
- Mala očna jabučica duha prati pokazivač miša.
- Kliknite na malog duha da uđete u stanje ljutnje i ispalite lasere.
- Kontinuirano klikanje uzrokovat će eksploziju i raspadanje malog duha, a zatim se ponovno pojaviti nakon nekoliko sekundi.
