# رستو موجود ناهي غلطي پيغام وارو صفحو

- پرامٽ صفحو نه مليو، گھر جي صفحي ڏانھن موٽڻ لاء ھڪڙو بٽڻ ڏيو.
- ننڍڙو ڀوت اکين جو گول مائوس جي پوائنٽر کي ٽريڪ ڪري ٿو.
- ناراض رياست ۽ فائر ليزرز ۾ داخل ٿيڻ لاءِ ننڍڙو گھوسٽ تي ڪلڪ ڪريو.
- مسلسل ڪلڪ ڪرڻ سان ننڍڙو ڀوت ڦاٽڻ ۽ ٽٽڻ جو سبب بڻجندو، ۽ پوءِ ڪجهه سيڪنڊن کان پوءِ ٻيهر پيدا ٿيندو.
