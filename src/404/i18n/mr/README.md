# पथ त्रुटी संदेश पृष्ठ अस्तित्वात नाही

- प्रॉम्प्ट पृष्ठ आढळले नाही, मुख्यपृष्ठावर परत जाण्यासाठी एक बटण प्रदान करा.
- लहान भूत नेत्रगोलक माउस पॉइंटरचा मागोवा घेतो.
- क्रोधित अवस्थेत आणि फायर लेसरमध्ये प्रवेश करण्यासाठी छोट्या भूतावर क्लिक करा.
- सतत क्लिक केल्याने लहान भुताचा स्फोट होईल आणि त्याचे विघटन होईल आणि काही सेकंदांनंतर पुन्हा तयार होईल.
