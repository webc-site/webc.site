# మార్గం లోపం సందేశ పేజీ ఉనికిలో లేదు

- ప్రాంప్ట్ పేజీ కనుగొనబడలేదు, హోమ్ పేజీకి తిరిగి రావడానికి బటన్‌ను అందించండి.
- చిన్న దెయ్యం ఐబాల్ మౌస్ పాయింటర్‌ను ట్రాక్ చేస్తుంది.
- కోపంతో ఉన్న స్థితిలోకి ప్రవేశించడానికి మరియు లేజర్‌లను కాల్చడానికి చిన్న దెయ్యంపై క్లిక్ చేయండి.
- నిరంతర క్లిక్ చేయడం వలన చిన్న దెయ్యం పేలిపోయి, విచ్చిన్నమై, కొన్ని సెకన్ల తర్వాత మళ్లీ పుంజుకుంటుంది.
