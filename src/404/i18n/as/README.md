# পথৰ অস্তিত্ব নাই ভুল বাৰ্তা পৃষ্ঠা

- প্ৰমপ্ট পৃষ্ঠা পোৱা নগ'ল, হোম পৃষ্ঠালৈ উভতি যাবলৈ এটা বুটাম প্ৰদান কৰক।
- সৰু ভূতৰ চকুৰ মণিটোৱে মাউছৰ পইণ্টাৰটোক অনুসৰণ কৰে।
- ক্ৰোধিত অৱস্থাত প্ৰৱেশ কৰিবলৈ সৰু ভূতটোত ক্লিক কৰক আৰু লেজাৰ জুই জ্বলায়।
- অবিৰত ক্লিক কৰিলে সৰু ভূতটো বিস্ফোৰণ আৰু বিভাজিত হ'ব, আৰু তাৰ পিছত কেইছেকেণ্ডমানৰ পিছত পুনৰ জন্ম হ'ব।
