# O camiño non existe páxina de mensaxes de erro

- Non se atopou a páxina de solicitude, proporciona un botón para volver á páxina de inicio.
- O pequeno globo ocular pantasma segue o punteiro do rato.
- Fai clic na pequena pantasma para entrar no estado de rabia e disparar láseres.
- Facer clic continuamente fará que a pequena pantasma explote e se desintegre, e despois reapareza despois duns segundos.
