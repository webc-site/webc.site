# Sökvägen finns inte felmeddelandesida

- Uppmaningssidan hittades inte, ange en knapp för att återgå till startsidan.
- Den lilla spökögongloben spårar muspekaren.
- Klicka på det lilla spöket för att komma in i det arga tillståndet och avfyra lasrar.
- Kontinuerlig klickning kommer att få det lilla spöket att explodera och sönderfalla och sedan återuppstå efter några sekunder.
