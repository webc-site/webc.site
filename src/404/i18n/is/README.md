# Slóð er ekki til villuboðasíða

- Hvetjandi síða fannst ekki, gefðu upp hnapp til að fara aftur á heimasíðuna.
- Litli draugaaugnappinn fylgist með músarbendlinum.
- Smelltu á litla drauginn til að komast inn í reiðiríkið og skjóta leysigeislum.
- Stöðugur smellur mun valda því að litli draugurinn springur og sundrast og sprettur svo aftur eftir nokkrar sekúndur.
