# Calea nu există pagina cu mesaj de eroare

- Pagina promptă nu a fost găsită, furnizați un buton pentru a reveni la pagina de pornire.
- Micul glob ocular fantomă urmărește indicatorul mouse-ului.
- Faceți clic pe micuța fantomă pentru a intra în starea de furie și trage cu lasere.
- Un clic continuu va face ca mica fantomă să explodeze și să se dezintegreze, iar apoi să reapare după câteva secunde.
