# Страница сообщения об ошибке «Путь не существует»

- Страница подсказки не найдена. Предоставьте кнопку для возврата на домашнюю страницу.
- Маленький глаз-призрак отслеживает указатель мыши.
- Нажмите на маленького призрака, чтобы войти в состояние гнева и запустить лазеры.
- Непрерывное нажатие приведет к тому, что маленькое привидение взорвется и распадется, а затем возродится через несколько секунд.
