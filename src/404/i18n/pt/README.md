# Caminho não existe página de mensagem de erro

- Página de prompt não encontrada, forneça um botão para retornar à página inicial.
- O pequeno globo ocular fantasma rastreia o ponteiro do mouse.
- Clique no pequeno fantasma para entrar no estado de raiva e disparar lasers.
- Clicar continuamente fará com que o pequeno fantasma exploda e se desintegre, e reapareça após alguns segundos.
