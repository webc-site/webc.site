# Njia haipo ukurasa wa ujumbe wa hitilafu

- Ukurasa wa haraka haujapatikana, toa kitufe ili kurudi kwenye ukurasa wa nyumbani.
- Jicho dogo la mzimu hufuata kielekezi cha kipanya.
- Bofya kwenye roho kidogo ili kuingia katika hali ya hasira na lasers za moto.
- Kubofya mara kwa mara kutasababisha mzimu mdogo kulipuka na kugawanyika, na kisha kuzaa tena baada ya sekunde chache.
