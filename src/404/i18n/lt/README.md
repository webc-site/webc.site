# Kelio nėra klaidos pranešimo puslapyje

- Raginimo puslapis nerastas, pateikite mygtuką, kad grįžtumėte į pagrindinį puslapį.
- Mažas vaiduoklio akies obuolys seka pelės žymeklį.
- Spustelėkite mažą vaiduoklį, kad patektumėte į piktą būseną ir paleistumėte lazerius.
- Nuolat spustelėjus mažasis vaiduoklis sprogs ir suirs, o po kelių sekundžių vėl atgims.
