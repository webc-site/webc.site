# Yo‘l xato xabari sahifasi mavjud emas

- So'rov sahifasi topilmadi, bosh sahifaga qaytish uchun tugmani taqdim eting.
- Kichkina sharpa ko'z olmasi sichqoncha ko'rsatkichini kuzatib boradi.
- G'azablangan holatga kirish va lazerlarni yoqish uchun kichkina sharpani bosing.
- Doimiy bosish kichik sharpaning portlashi va parchalanishiga olib keladi va keyin bir necha soniyadan keyin qayta tug'iladi.
