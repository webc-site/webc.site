# Laluan tidak wujud halaman mesej ralat

- Halaman gesaan tidak ditemui, sediakan butang untuk kembali ke halaman utama.
- Bola mata hantu kecil menjejaki penunjuk tetikus.
- Klik pada hantu kecil untuk memasuki keadaan marah dan laser api.
- Klik berterusan akan menyebabkan hantu kecil itu meletup dan hancur, dan kemudian muncul semula selepas beberapa saat.
