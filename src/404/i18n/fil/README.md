# Ang landas ay walang pahina ng mensahe ng error

- Hindi nahanap ang prompt page, magbigay ng button para bumalik sa home page.
- Sinusubaybayan ng maliit na ghost eyeball ang pointer ng mouse.
- Mag-click sa maliit na multo upang makapasok sa galit na estado at mga laser ng apoy.
- Ang tuluy-tuloy na pag-click ay magiging sanhi ng pagsabog at pagkawatak-watak ng munting multo, at pagkatapos ay muling lalabas pagkatapos ng ilang segundo.
