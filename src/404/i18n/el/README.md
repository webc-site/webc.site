# Η διαδρομή δεν υπάρχει σελίδα μηνύματος σφάλματος

- Η σελίδα προτροπής δεν βρέθηκε, δώστε ένα κουμπί για να επιστρέψετε στην αρχική σελίδα.
- Το μικρό βολβό του ματιού φάντασμα παρακολουθεί τον δείκτη του ποντικιού.
- Κάντε κλικ στο μικρό φάντασμα για να μπείτε στην κατάσταση θυμού και να πυροβολήσετε λέιζερ.
- Το συνεχές κλικ θα προκαλέσει το μικρό φάντασμα να εκραγεί και να αποσυντεθεί και στη συνέχεια να αναπαραχθεί μετά από λίγα δευτερόλεπτα.
