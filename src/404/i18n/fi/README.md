# Polkua ei ole olemassa virheilmoitussivua

- Kehotussivua ei löydy, anna painike palataksesi etusivulle.
- Pieni kummitussilmämuna seuraa hiiren osoitinta.
- Napsauta pientä haamua päästäksesi vihaiseen tilaan ja ampuaksesi lasereita.
- Jatkuva napsautus saa pienen aaveen räjähtämään ja hajoamaan ja ilmaantumaan uudelleen muutaman sekunnin kuluttua.
