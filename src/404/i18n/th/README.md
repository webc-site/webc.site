# เส้นทางไม่มีหน้าข้อความแสดงข้อผิดพลาด

- ไม่พบหน้าแจ้ง ให้ปุ่มเพื่อกลับไปยังหน้าแรก
- ลูกตาผีตัวน้อยติดตามตัวชี้เมาส์
- คลิกที่ผีตัวน้อยเพื่อเข้าสู่สถานะโกรธและยิงเลเซอร์
- การคลิกอย่างต่อเนื่องจะทำให้ผีตัวน้อยระเบิดและสลายตัว จากนั้นจะเกิดใหม่อีกครั้งในไม่กี่วินาที
