# La ruta no existe página de mensaje de error

- Página de aviso no encontrada, proporcione un botón para regresar a la página de inicio.
- El pequeño globo ocular fantasma sigue el puntero del ratón.
- Haz clic en el pequeño fantasma para entrar en el estado de ira y disparar láseres.
- Al hacer clic continuo, el pequeño fantasma explotará y se desintegrará, y luego reaparecerá después de unos segundos.
