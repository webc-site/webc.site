# It-triq ma teżistix paġna tal-messaġġ ta' żball

- Paġna fil-pront ma nstabx, ipprovdi buttuna biex terġa' lura għall-paġna ewlenija.
- Il-boċċa tal-għajnejn tal-fatat żgħir isegwi l-pointer tal-maws.
- Ikklikkja fuq il-fatat żgħir biex tidħol fl-istat rrabjat u l-lasers tan-nar.
- L-ikklikkjar kontinwu se jikkawża li l-fatat żgħir jisplodi u jiddiżintegra, u mbagħad jerġa 'jibda wara ftit sekondi.
