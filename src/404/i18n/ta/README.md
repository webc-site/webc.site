# பாதை பிழை செய்தி பக்கம் இல்லை

- உடனடிப் பக்கம் காணப்படவில்லை, முகப்புப் பக்கத்திற்குத் திரும்புவதற்கு ஒரு பொத்தானை வழங்கவும்.
- சிறிய பேய் ஐபால் மவுஸ் பாயிண்டரைக் கண்காணிக்கிறது.
- கோபமான நிலைக்குச் சென்று லேசர்களைச் சுட சிறிய பேயின் மீது கிளிக் செய்யவும்.
- தொடர்ந்து கிளிக் செய்வதால், சிறிய பேய் வெடித்துச் சிதறி, சில நொடிகளுக்குப் பிறகு மீண்டும் தோன்றும்.
