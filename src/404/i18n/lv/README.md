# Ceļš nepastāv kļūdas ziņojuma lapa

- Uzvednes lapa nav atrasta. Nodrošiniet pogu, lai atgrieztos sākumlapā.
- Mazais spoka acs ābols izseko peles rādītāju.
- Noklikšķiniet uz mazā spoka, lai iekļūtu dusmīgajā stāvoklī un izšautu lāzerus.
- Nepārtraukta klikšķināšana liks mazajam spokam eksplodēt un sadalīties, un pēc dažām sekundēm tas atkal parādīsies.
