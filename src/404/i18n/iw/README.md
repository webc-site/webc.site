- דף הנחיה לא נמצא, ספק כפתור כדי לחזור לדף הבית.
- גלגל העין הקטן של הרפאים עוקב אחר מצביע העכבר.
- לחץ על רוח הרפאים הקטנה כדי להיכנס למצב הכעס ולירות לייזרים.
