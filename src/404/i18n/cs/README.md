# Stránka s chybovou zprávou cesta neexistuje

- Stránka s výzvou nenalezena, poskytněte tlačítko pro návrat na domovskou stránku.
- Oční koule malého ducha sleduje ukazatel myši.
- Klikněte na malého ducha, abyste vstoupili do rozzlobeného stavu a vypálili lasery.
- Nepřetržité klikání způsobí, že malý duch exploduje a rozpadne se a po několika sekundách se znovu objeví.
