- کره چشم شبح کوچک نشانگر ماوس را ردیابی می کند.
- برای وارد شدن به حالت عصبانی و آتش زدن لیزر روی روح کوچک کلیک کنید.
- کلیک کردن مداوم باعث می شود روح کوچک منفجر شود و از هم بپاشد و بعد از چند ثانیه دوباره به وجود بیاید.
