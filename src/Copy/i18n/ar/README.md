# نسخ النص إلى الحافظة

انقر فوق الزر لنسخ نص العنصر المحدد.

- يدعم استخدام rel لتحديد محدد العنصر المراد نسخه
- عندما لا يتم تحديده، يتم نسخ العنصر الشقيق السابق افتراضيًا.
- مربع المطالبة Pophover المتداخل
- الدعم الدولي

## استخدم العرض التوضيحي

```html
<!-- نسخ عناصر محددة -->
<span id="target">هنا هو النص المراد نسخه</span>
<c-copy rel="#target"></c-copy>

<!-- يتم نسخ العناصر السابقة بشكل افتراضي -->
<span>نص العنصر السابق</span>
<c-copy></c-copy>
```

## المعلمة

| ملكية | يوضح |
| --- | --- |
| rel | انسخ محدد العنصر الهدف. خياري |

## نمط الحالة

- `c-pophover.Done`: نمط مربع المطالبة بعد النسخ الناجح
