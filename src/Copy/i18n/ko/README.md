# 텍스트를 클립보드에 복사

버튼을 클릭하면 지정된 요소의 텍스트가 복사됩니다.

- 복사할 요소 선택기를 지정하기 위해 rel 사용 지원
- 지정하지 않으면 기본적으로 이전 형제 요소가 복사됩니다.
- 중첩된 Popover 프롬프트 상자
- 국제적인 지원

## 데모 사용

```html
<!-- 특정 요소 복사 -->
<span id="target">복사할 텍스트는 다음과 같습니다.</span>
<c-copy rel="#target"></c-copy>

<!-- 기본적으로 이전 요소가 복사됩니다. -->
<span>이전 요소의 텍스트</span>
<c-copy></c-copy>
```

## 매개변수

| 재산 | 설명하다 |
| --- | --- |
| rel | 대상 요소의 선택기를 복사합니다. 선택 과목 |

## 상태 스타일

- `c-pophover.Done`: 복사 성공 후 프롬프트 상자 스타일
