# వచనాన్ని క్లిప్‌బోర్డ్‌కు కాపీ చేయండి

పేర్కొన్న మూలకం యొక్క వచనాన్ని కాపీ చేయడానికి బటన్‌ను క్లిక్ చేయండి.

- కాపీ చేయవలసిన ఎలిమెంట్ సెలెక్టర్‌ను పేర్కొనడానికి relని ఉపయోగించడాన్ని సపోర్ట్ చేస్తుంది
- పేర్కొనబడనప్పుడు, మునుపటి తోబుట్టువుల మూలకం డిఫాల్ట్‌గా కాపీ చేయబడుతుంది.
- నెస్టెడ్ పోఫోవర్ ప్రాంప్ట్ బాక్స్
- అంతర్జాతీయ మద్దతు

## డెమో ఉపయోగించండి

```html
<!-- నిర్దిష్ట అంశాలను కాపీ చేయండి -->
<span id="target">కాపీ చేయవలసిన వచనం ఇక్కడ ఉంది</span>
<c-copy rel="#target"></c-copy>

<!-- మునుపటి అంశాలు డిఫాల్ట్‌గా కాపీ చేయబడతాయి -->
<span>మునుపటి మూలకం యొక్క వచనం</span>
<c-copy></c-copy>
```

## పరామితి

| ఆస్తి | వర్ణించండి |
| --- | --- |
| rel | లక్ష్య మూలకం యొక్క ఎంపిక సాధనాన్ని కాపీ చేయండి. ఐచ్ఛికం |

## స్థితి శైలి
