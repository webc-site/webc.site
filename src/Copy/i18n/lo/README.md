# ສຳເນົາຂໍ້ຄວາມໃສ່ຄລິບບອດ

ຄລິກປຸ່ມເພື່ອສຳເນົາຂໍ້ຄວາມຂອງອົງປະກອບທີ່ລະບຸໄວ້.

- ຮອງຮັບການໃຊ້ rel ເພື່ອລະບຸຕົວເລືອກອົງປະກອບທີ່ຈະຄັດລອກ
- ເມື່ອບໍ່ໄດ້ລະບຸ, ອົງປະກອບຂອງອ້າຍເອື້ອຍນ້ອງທີ່ຜ່ານມາຈະຖືກຄັດລອກໂດຍຄ່າເລີ່ມຕົ້ນ.
- Nested Pophover ກ່ອງເຕືອນ
- ສະຫນັບສະຫນູນສາກົນ
