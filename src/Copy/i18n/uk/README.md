# Копіювати текст у буфер обміну

Натисніть кнопку, щоб скопіювати текст зазначеного елемента.

- Підтримує використання rel для визначення селектора елемента, який потрібно скопіювати
- Якщо не вказано, попередній однорідний елемент копіюється за замовчуванням.
- Вкладене вікно підказки Pophover
- Міжнародна підтримка

## Використовуйте демо

```html
<!-- Скопіюйте окремі елементи -->
<span id="target">Ось текст для копіювання</span>
<c-copy rel="#target"></c-copy>

<!-- Попередні елементи копіюються за замовчуванням -->
<span>текст попереднього елемента</span>
<c-copy></c-copy>
```

## параметр

| власність | ілюструвати |
| --- | --- |
| rel | Скопіюйте селектор цільового елемента. Додатково |

## статусний стиль

- `c-pophover.Done`: стиль вікна підказки після успішного копіювання
