# Копирај текст во таблата со исечоци

Кликнете на копчето за да го копирате текстот на наведениот елемент.

- Поддржува користење rel за одредување на избирачот на елементи што треба да се копира
- Кога не е одредено, претходниот братски елемент стандардно се копира.
- Вгнездено промптно поле Pophover
- Меѓународна поддршка

## Користете го демо

```html
<!-- Копирајте специфични елементи -->
<span id="target">Еве го текстот што треба да се копира</span>
<c-copy rel="#target"></c-copy>

<!-- Претходните елементи се стандардно копирани -->
<span>текстот на претходниот елемент</span>
<c-copy></c-copy>
```

## параметар

| имот | илустрираат |
| --- | --- |
| rel | Копирајте го избирачот на целниот елемент. Факултативно |

## статусен стил

- `c-pophover.Done`: стил на прашалник по успешното копирање
