# 复制文本到剪贴板

点击按钮复制指定元素的文本。

- 支持用 rel 指定要复制的元素选择器
- 未指定时，默认复制前一个兄弟元素
- 嵌套 Pophover 提示框
- 国际化支持

## 使用演示

```html
<!-- 复制特定元素 -->
<span id="target">这里是待复制的文本</span>
<c-copy rel="#target"></c-copy>

<!-- 默认复制前置元素 -->
<span>前一个元素的文本</span>
<c-copy></c-copy>
```

## 参数

| 属性 | 说明 |
| --- | --- |
| rel | 复制目标元素的选择器。可选 |

## 状态样式

- `c-pophover.Done`: 复制成功后的提示框样式
