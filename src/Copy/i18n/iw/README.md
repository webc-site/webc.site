# העתק טקסט ללוח

לחץ על הלחצן כדי להעתיק את הטקסט של האלמנט שצוין.

- תומך בשימוש ב-rel כדי לציין את בורר האלמנטים שיש להעתיק
- כאשר לא מצוין, הרכיב האח הקודם מועתק כברירת מחדל.
- תיבת הנחיות Pophover מקוננת
- תמיכה בינלאומית

## השתמש בהדגמה

```html
<!-- העתק אלמנטים ספציפיים -->
<span id="target">הנה הטקסט שיש להעתיק</span>
<c-copy rel="#target"></c-copy>

<!-- הרכיבים הקודמים מועתקים כברירת מחדל -->
<span>הטקסט של האלמנט הקודם</span>
<c-copy></c-copy>
```

## פָּרָמֶטֶר

| נֶכֶס | לְהַדגִים |
| --- | --- |
| rel | העתק את הבורר של רכיב היעד. אופציונלי |

## סגנון סטטוס

- `c-pophover.Done`: סגנון תיבת הנחיה לאחר העתקה מוצלחת
