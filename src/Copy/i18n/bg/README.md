# Копиране на текст в клипборда

Щракнете върху бутона, за да копирате текста на посочения елемент.

- Поддържа използването на rel за указване на селектора на елемента, който да бъде копиран
- Когато не е указано, предишният братски елемент се копира по подразбиране.
- Вложено поле за подкани Pophover
- Международна подкрепа

## Използвайте демонстрацията

```html
<!-- Копирайте конкретни елементи -->
<span id="target">Ето текста за копиране</span>
<c-copy rel="#target"></c-copy>

<!-- Предходните елементи се копират по подразбиране -->
<span>текста на предишния елемент</span>
<c-copy></c-copy>
```

## параметър

| собственост | илюстрирам |
| --- | --- |
| rel | Копирайте селектора на целевия елемент. Не е задължително |

## статус стил

- `c-pophover.Done`: стил на полето за подкана след успешно копиране
