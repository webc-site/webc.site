# ಕ್ಲಿಪ್‌ಬೋರ್ಡ್‌ಗೆ ಪಠ್ಯವನ್ನು ನಕಲಿಸಿ

ನಿರ್ದಿಷ್ಟಪಡಿಸಿದ ಅಂಶದ ಪಠ್ಯವನ್ನು ನಕಲಿಸಲು ಬಟನ್ ಕ್ಲಿಕ್ ಮಾಡಿ.

- ನಕಲು ಮಾಡಬೇಕಾದ ಅಂಶ ಸೆಲೆಕ್ಟರ್ ಅನ್ನು ನಿರ್ದಿಷ್ಟಪಡಿಸಲು rel ಅನ್ನು ಬೆಂಬಲಿಸುತ್ತದೆ
- ನಿರ್ದಿಷ್ಟಪಡಿಸದಿದ್ದಾಗ, ಹಿಂದಿನ ಒಡಹುಟ್ಟಿದ ಅಂಶವನ್ನು ಪೂರ್ವನಿಯೋಜಿತವಾಗಿ ನಕಲಿಸಲಾಗುತ್ತದೆ.
- ನೆಸ್ಟೆಡ್ ಪೋಪೋವರ್ ಪ್ರಾಂಪ್ಟ್ ಬಾಕ್ಸ್
- ಅಂತರರಾಷ್ಟ್ರೀಯ ಬೆಂಬಲ
