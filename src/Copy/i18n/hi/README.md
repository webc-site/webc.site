# टेक्स्ट को क्लिपबोर्ड पर कॉपी करें

निर्दिष्ट तत्व के पाठ की प्रतिलिपि बनाने के लिए बटन पर क्लिक करें।

- कॉपी किए जाने वाले तत्व चयनकर्ता को निर्दिष्ट करने के लिए rel का उपयोग करने का समर्थन करता है
- जब निर्दिष्ट नहीं किया जाता है, तो पिछला सहोदर तत्व डिफ़ॉल्ट रूप से कॉपी किया जाता है।
- नेस्टेड पॉपओवर प्रॉम्प्ट बॉक्स
- अंतर्राष्ट्रीय समर्थन

## डेमो का प्रयोग करें

```html
<!-- विशिष्ट तत्वों की प्रतिलिपि बनाएँ -->
<span id="target">यहां कॉपी किया जाने वाला टेक्स्ट है</span>
<c-copy rel="#target"></c-copy>

<!-- पूर्ववर्ती तत्व डिफ़ॉल्ट रूप से कॉपी किए जाते हैं -->
<span>पिछले तत्व का पाठ</span>
<c-copy></c-copy>
```

## पैरामीटर

| संपत्ति | उदाहरण देकर स्पष्ट करना |
| --- | --- |
| rel | लक्ष्य तत्व के चयनकर्ता की प्रतिलिपि बनाएँ। वैकल्पिक |
