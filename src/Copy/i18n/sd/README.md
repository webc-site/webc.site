# ڪلپ بورڊ تي متن نقل ڪريو

مخصوص عنصر جي متن کي نقل ڪرڻ لاء بٽڻ تي ڪلڪ ڪريو.

- نقل ڪرڻ لاءِ عنصر چونڊيندڙ کي بيان ڪرڻ لاءِ rel استعمال ڪندي سپورٽ ڪري ٿو
- جڏهن بيان نه ڪيو ويو آهي، پوئين ڀائر عنصر ڊفالٽ طور نقل ڪيو ويندو آهي.
- Nested Pophover Prompt باڪس
- بين الاقوامي حمايت

## ڊيمو استعمال ڪريو

```html
<!-- مخصوص عناصر کي نقل ڪريو -->
<span id="target">هتي نقل ڪرڻ جو متن آهي</span>
<c-copy rel="#target"></c-copy>

<!-- اڳوڻي عناصر ڊفالٽ طور نقل ڪيا ويا آهن -->
<span>پوئين عنصر جو متن</span>
<c-copy></c-copy>
```

## پيرا ميٽر

| ملڪيت | بيان ڪرڻ |
| --- | --- |
| rel | ھدف جي عنصر جي چونڊيندڙ کي نقل ڪريو. اختياري |

## اسٽيٽس جو انداز

- `c-pophover.Done`: ڪامياب ڪاپي ڪرڻ کان پوءِ تڪڙي باڪس انداز
