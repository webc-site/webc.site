# Мәтінді алмасу буферіне көшіру

Көрсетілген элементтің мәтінін көшіру үшін түймені басыңыз.

- Көшірілетін элемент селекторын көрсету үшін rel пайдалануды қолдайды
- Көрсетілмеген кезде, алдыңғы қатардағы элемент әдепкі бойынша көшіріледі.
- Кірістірілген Pophover шақыру терезесі
- Халықаралық қолдау

## Демонстрацияны пайдаланыңыз

```html
<!-- Арнайы элементтерді көшіру -->
<span id="target">Міне көшірілетін мәтін</span>
<c-copy rel="#target"></c-copy>

<!-- Алдыңғы элементтер әдепкі бойынша көшіріледі -->
<span>алдыңғы элементтің мәтіні</span>
<c-copy></c-copy>
```

## параметр

| мүлік | суреттеу |
| --- | --- |
| rel | Мақсатты элементтің селекторын көшіріңіз. Қосымша |

## күй стилі

- `c-pophover.Done`: сәтті көшіруден кейін шақыру өрісі стилі
