# Պատճենել տեքստը clipboard-ում

Սեղմեք կոճակը՝ նշված տարրի տեքստը պատճենելու համար:

- Աջակցում է rel-ի՝ պատճենվող տարրի ընտրիչը նշելու համար
- Երբ նշված չէ, նախորդ եղբոր տարրը պատճենվում է լռելյայն:
- Nested Pophover հուշման տուփ
- Միջազգային աջակցություն

## Օգտագործեք ցուցադրությունը

```html
<!-- Պատճենել կոնկրետ տարրեր -->
<span id="target">Ահա պատճենվող տեքստը</span>
<c-copy rel="#target"></c-copy>

<!-- Նախորդ տարրերը պատճենվում են լռելյայն -->
<span>նախորդ տարրի տեքստը</span>
<c-copy></c-copy>
```

## պարամետր

| սեփականություն | նկարազարդել |
| --- | --- |
| rel | Պատճենեք թիրախային տարրի ընտրիչը: Ընտրովի |

## կարգավիճակի ոճը

- `c-pophover.Done`. հուշագրի ոճը հաջող պատճենումից հետո
