# စာသားကို ကလစ်ဘုတ်သို့ ကူးယူပါ။

သတ်မှတ်ထားသောဒြပ်စင်၏စာသားကိုကူးယူရန်ခလုတ်ကိုနှိပ်ပါ။

- ကူးယူမည့် ဒြပ်စင်ရွေးချယ်မှုအား သတ်မှတ်ရန် rel ကို အသုံးပြု၍ ပံ့ပိုးပေးသည်။
- သတ်မှတ်ထားခြင်းမရှိသောအခါ၊ ယခင်ပေါက်ဖော်ဒြပ်စင်ကို မူရင်းအတိုင်း ကူးယူပါသည်။
- Nested Popover prompt box
- နိုင်ငံတကာက ထောက်ခံတယ်။

## ဒီမိုကိုသုံးပါ။

```html
<!-- သီးခြားအစိတ်အပိုင်းများကို ကူးယူပါ။ -->
<span id="target">ဤသည်မှာ ကူးယူရမည့် စာသားဖြစ်ပါသည်။</span>
<c-copy rel="#target"></c-copy>

<!-- ရှေ့ဒြပ်စင်များကို မူရင်းအတိုင်း ကူးယူပါသည်။ -->
<span>ယခင်ဒြပ်စင်၏စာသား</span>
<c-copy></c-copy>
```

## ကန့်သတ်ချက်

| ပစ္စည်းဥစ္စာ | ပုံဥပမာ |
| --- | --- |
| rel | ပစ်မှတ်ဒြပ်စင်၏ရွေးချယ်မှုအား ကူးယူပါ။ ရွေးချယ်ခွင့် |

## အခြေအနေစတိုင်

- `c-pophover.Done`- အောင်မြင်စွာ ကူးယူပြီးနောက် ပေါ်လာသော အကွက်ပုံစံ
