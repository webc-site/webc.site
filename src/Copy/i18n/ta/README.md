# கிளிப்போர்டுக்கு உரையை நகலெடுக்கவும்

குறிப்பிட்ட உறுப்பின் உரையை நகலெடுக்க பொத்தானைக் கிளிக் செய்யவும்.

- நகலெடுக்கப்பட வேண்டிய உறுப்பு தேர்வியைக் குறிப்பிட rel ஐப் பயன்படுத்துவதை ஆதரிக்கிறது
- குறிப்பிடப்படாதபோது, ​​முந்தைய உடன்பிறப்பு உறுப்பு இயல்புநிலையாக நகலெடுக்கப்படும்.
- உள்ளமைக்கப்பட்ட போபோவர் ப்ராம்ட் பாக்ஸ்
- சர்வதேச ஆதரவு

## டெமோவைப் பயன்படுத்தவும்

```html
<!-- குறிப்பிட்ட கூறுகளை நகலெடுக்கவும் -->
<span id="target">நகலெடுக்க வேண்டிய உரை இதோ</span>
<c-copy rel="#target"></c-copy>

<!-- முந்தைய கூறுகள் இயல்பாக நகலெடுக்கப்படும் -->
<span>முந்தைய உறுப்பு உரை</span>
<c-copy></c-copy>
```
