# ટેક્સ્ટને ક્લિપબોર્ડ પર કૉપિ કરો

ઉલ્લેખિત તત્વના ટેક્સ્ટની નકલ કરવા માટે બટન પર ક્લિક કરો.

- કોપી કરવા માટે તત્વ પસંદગીકારનો ઉલ્લેખ કરવા માટે rel નો ઉપયોગ કરવાનું સમર્થન કરે છે
- જ્યારે ઉલ્લેખિત ન હોય, ત્યારે પાછલા ભાઈ તત્વની ડિફૉલ્ટ રૂપે કૉપિ કરવામાં આવે છે.
- નેસ્ટેડ પોફોવર પ્રોમ્પ્ટ બોક્સ
- આંતરરાષ્ટ્રીય આધાર

## ડેમોનો ઉપયોગ કરો

```html
<!-- ચોક્કસ ઘટકોની નકલ કરો -->
<span id="target">અહીં કોપી કરવાનું લખાણ છે</span>
<c-copy rel="#target"></c-copy>

<!-- પૂર્વવર્તી તત્વો મૂળભૂત રીતે નકલ કરવામાં આવે છે -->
<span>પાછલા તત્વનું લખાણ</span>
<c-copy></c-copy>
```

## પરિમાણ

| મિલકત | સમજાવવું |
| --- | --- |
| rel | લક્ષ્ય તત્વના પસંદગીકારની નકલ કરો. વૈકલ્પિક |

## સ્થિતિ શૈલી

- `c-pophover.Done`: સફળ નકલ કર્યા પછી પ્રોમ્પ્ટ બોક્સ શૈલી
