# テキストをクリップボードにコピーする

ボタンをクリックして、指定した要素のテキストをコピーします。

- コピーする要素セレクターを指定するための rel の使用をサポートします
- 指定しない場合、デフォルトで前の兄弟要素がコピーされます。
- ネストされた Pophover プロンプト ボックス
- 国際的なサポート

## デモを使用する

```html
<!-- 特定の要素をコピーする -->
<span id="target">コピーするテキストは次のとおりです</span>
<c-copy rel="#target"></c-copy>

<!-- デフォルトでは前の要素がコピーされます -->
<span>前の要素のテキスト</span>
<c-copy></c-copy>
```

## パラメータ

| 財産 | 説明する |
| --- | --- |
| rel | 対象要素のセレクターをコピーします。オプション |

## ステータススタイル

- `c-pophover.Done`: コピーが成功した後のプロンプト ボックスのスタイル
