# Копирајте текст у међуспремник

Кликните на дугме да бисте копирали текст наведеног елемента.

- Подржава коришћење рел за одређивање селектора елемената који ће се копирати
- Када није наведен, претходни сродни елемент се подразумевано копира.
- Угнежђени оквир са промптовима Попховер
- Међународна подршка

## Користите демо

```html
<!-- Копирајте одређене елементе -->
<span id="target">Ево текста за копирање</span>
<c-copy rel="#target"></c-copy>

<!-- Претходни елементи се подразумевано копирају -->
<span>текст претходног елемента</span>
<c-copy></c-copy>
```

## параметар

| имовине | илуструју |
| --- | --- |
| rel | Копирајте селектор циљног елемента. Опционо |

## статусни стил

- <б ид="1">: стил оквира упита након успешног копирања
