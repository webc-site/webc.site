# ტექსტის კოპირება ბუფერში

დააწკაპუნეთ ღილაკზე მითითებული ელემენტის ტექსტის დასაკოპირებლად.

- მხარს უჭერს rel-ის გამოყენებას, რათა დააკოპიროთ ელემენტის ამომრჩეველი
- როდესაც არ არის მითითებული, წინა ძმის ელემენტი კოპირდება ნაგულისხმევად.
- ჩადგმული Popover-ის მოთხოვნის ყუთი
- საერთაშორისო მხარდაჭერა

## გამოიყენეთ დემო ვერსია

```html
<!-- დააკოპირეთ კონკრეტული ელემენტები -->
<span id="target">აქ არის ტექსტი, რომელიც უნდა დააკოპიროთ</span>
<c-copy rel="#target"></c-copy>

<!-- წინა ელემენტები კოპირებულია ნაგულისხმევად -->
<span>წინა ელემენტის ტექსტი</span>
<c-copy></c-copy>
```

## პარამეტრი

| ქონება | ილუსტრირება |
| --- | --- |
| rel | დააკოპირეთ სამიზნე ელემენტის ამომრჩეველი. სურვილისამებრ |

## სტატუსის სტილი

- `c-pophover.Done`: მოთხოვნის ყუთის სტილი წარმატებული კოპირების შემდეგ
