# 複製文字到剪貼簿

點選按鈕複製指定元素的文字。

- 支援用 rel 指定要複製的元素選擇器
- 未指定時，預設複製前一個兄弟元素
- 嵌套 Pophover 提示框
- 國際化支持

## 使用演示

```html
<!-- 複製特定元素 -->
<span id="target">這裡是待複製的文本</span>
<c-copy rel="#target"></c-copy>

<!-- 預設複製前置元素 -->
<span>前一個元素的文本</span>
<c-copy></c-copy>
```

## 參數

| 屬性 | 說明 |
| --- | --- |
| rel | 複製目標元素的選擇器。可選 |

## 狀態樣式

- `c-pophover.Done`: 複製成功後的提示框樣式
