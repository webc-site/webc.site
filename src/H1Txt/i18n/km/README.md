# бһ…бҹҶбһҺбһ„бһҮбһҫбһ„бһңбһ¶бһҷбһ“бһ—бһ¶бһ– бһ“бһ·бһ„бһҖбһ¶бһҡбһ–бһ·бһ–бһҺбҹҢбһ“бһ¶

бһ…бҹҶбһҺбһ„бһҮбһҫбһ„вҖӢбһҮбһҳбҹ’бһҡбһ¶бһӣвҖӢбһӣбһёбһ“бҹҒбһўбҹҠбҹӮбһҡвҖӢбһҮбһ¶бһҳбһҪбһҷвҖӢбһҸбһҳбҹ’бһҡбһ„вҖӢбһҠбһ¶бһ…бҹӢвҖӢбһҹбһҡбһҹбҹғбһҲбһ¶бһҳвҖӢбһҒбһҪбһҡбһҖбҹ’бһ”бһ¶бһӣ бһ“бһ·бһ„вҖӢбһҹбҹ’бһҡбһҳбҹ„бһӣ

## бһ”бҹ’бһҡбһҫбһҖбһ¶бһҡбһ”бһ„бҹ’бһ бһ¶бһү

бһ”бһ“бҹ’бһҗбҹӮбһҳбһҗбҹ’бһ“бһ¶бһҖбҹӢ `H1Txt`бһ‘бҹ…бһ’бһ¶бһҸбһ» `header`бҹ”

```html
<header class="H1Txt">
  <h1>бһ…бҹҶбһҺбһ„бһҮбһҫбһ„</h1>
  <b>бһ–бһ·бһ–бһҺбҹҢбһ“бһ¶</b>
</header>
```
