# Кнопкі і ўсплывальныя вокны для пераключэння моў інтэрфейсу

- Націсніце кнопку, каб адкрыць плаваючы пласт выбару мовы.
- Аўтаматычна вылучаць выбраную мову.

## Выкарыстоўвайце дэма

```html
<c-i18n></c-i18n>

<script type="module">
import "webc.site/I18n/_.js";
// Выкарыстоўвайце наступны код JS, каб сачыць за зменамі мовы
import { onLang } from "webc.site/js/i18n.js";

onLang((code, id) => {
  alert(code + ' : ' + id);
});
</script>
```

## Параметры інтэрфейсу

### Карыстальніцкі тэг `<c-i18n>`

Карыстальніцкія тэгі HTML, уведзеныя ў JS і выкарыстоўваюцца ў HTML.

## Кіраванне статусам

Выкарыстоўвайце `src/js/i18n.js` для кіравання глабальным статусам мовы:

- **`LANG_LI`**: масіў спісу моў у фармаце `[[імя, ID], ...]`.
- **`lang()`**: атрымаць ідэнтыфікатар бягучай мовы.
- **`langCode()`**: атрымаць бягучую кадыроўку мовы.
- **`langSet(id)`**: усталяваць ідэнтыфікатар мовы і паведаміць абанентам.
- **`onLang(func)`**: падпісацца на моўныя змены. Калі мова была ўстаноўлена, зваротны выклік будзе запушчаны неадкладна. Вярнуць функцыю адпіскі.
