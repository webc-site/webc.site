# انٹرفیس کی زبانوں کو تبدیل کرنے کے لیے بٹن اور پاپ اپ ونڈوز

- زبان کے انتخاب کی تیرتی پرت کو پاپ اپ کرنے کے لیے بٹن پر کلک کریں۔
- موجودہ منتخب زبان کو خودکار طور پر نمایاں کریں۔

## ڈیمو استعمال کریں۔

```html
<c-i18n></c-i18n>

<script type="module">
import "webc.site/I18n/_.js";
// زبان کی تبدیلیوں کی نگرانی کے لیے درج ذیل جے ایس کوڈ کا استعمال کریں۔
import { onLang } from "webc.site/js/i18n.js";

onLang((code, id) => {
  alert(code + ' : ' + id);
});
</script>
```

## انٹرفیس کے پیرامیٹرز

### حسب ضرورت ٹیگ `<c-i18n>`

حسب ضرورت HTML ٹیگز، JS میں متعارف کرائے گئے اور HTML میں استعمال کیے گئے۔

## اسٹیٹس مینجمنٹ

عالمی زبان کی حیثیت کو منظم کرنے کے لیے `src/js/i18n.js` استعمال کریں:

- **`LANG_LI`**: زبان کی فہرست کی صف، `[[نام، ID]، ...]` کی شکل میں۔
- **`lang()`**: موجودہ زبان کی شناخت حاصل کریں۔
- **`langCode()`**: موجودہ زبان کی انکوڈنگ حاصل کریں۔
- **`langSet(id)`**: زبان کی شناخت سیٹ کریں اور سبسکرائبرز کو مطلع کریں۔
- **`onLang(func)`**: زبان کی تبدیلیوں کے لیے سبسکرائب کریں۔ اگر زبان سیٹ کر دی گئی ہے تو کال بیک فوری طور پر متحرک ہو جائے گا۔ ان سبسکرائب فنکشن کو واپس کریں۔

## سٹائل کلاس

### `.BtnC.lang`

آئیکن بٹن.

### `.I18n.Lg`

پاپ اپ ونڈو کا مرکزی لے آؤٹ زبان کے اختیارات کو سمیٹنے کے لیے استعمال کیا جاتا ہے۔
