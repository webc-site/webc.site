# ઈન્ટરફેસ ભાષાઓ બદલવા માટે બટનો અને પોપ-અપ વિન્ડો

- ભાષા પસંદગી ફ્લોટિંગ લેયરને પોપ અપ કરવા માટે બટન પર ક્લિક કરો.
- હાલમાં પસંદ કરેલી ભાષાને આપમેળે હાઇલાઇટ કરો.

## ડેમોનો ઉપયોગ કરો

```html
<c-i18n></c-i18n>

<script type="module">
import "webc.site/I18n.js";
// ભાષાના ફેરફારોને મોનિટર કરવા માટે નીચેના JS કોડનો ઉપયોગ કરો
import { onLang } from "webc.site/js/i18n.js";

onLang((code, id) => {
  alert(code + ' : ' + id);
});
</script>
```

## ઇન્ટરફેસ પરિમાણો

### કસ્ટમ ટૅગ `<c-i18n>`

કસ્ટમ HTML ટૅગ્સ, જેએસમાં રજૂ કરવામાં આવ્યા છે અને HTMLમાં ઉપયોગમાં લેવાય છે.

## સ્થિતિ વ્યવસ્થાપન

વૈશ્વિક ભાષા સ્થિતિનું સંચાલન કરવા માટે `src/js/i18n.js` નો ઉપયોગ કરો:

- **`LANG_LI`**: ભાષાની સૂચિ એરે, `[[નામ, ID], ...]`ના ફોર્મેટમાં.
- **`lang()`**: વર્તમાન ભાષા ID મેળવો.
- **`langCode()`**: વર્તમાન ભાષાનું એન્કોડિંગ મેળવો.
- **`langSet(id)`**: ભાષા ID સેટ કરો અને સબ્સ્ક્રાઇબર્સને સૂચિત કરો.
- **`onLang(func)`**: ભાષા ફેરફારો માટે સબ્સ્ક્રાઇબ કરો. જો ભાષા સેટ કરવામાં આવી હોય, તો કૉલબેક તરત જ ટ્રિગર થશે. અનસબ્સ્ક્રાઇબ ફંક્શન પરત કરો.

## શૈલી વર્ગ

### `.BtnC.lang`

આઇકોન બટન.

### `.I18n.Lg`

પોપ-અપ વિન્ડોના મુખ્ય લેઆઉટનો ઉપયોગ ભાષા વિકલ્પોને લપેટવા માટે થાય છે.
