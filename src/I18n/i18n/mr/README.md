# इंटरफेस भाषा बदलण्यासाठी बटणे आणि पॉप-अप विंडो

- भाषा निवड फ्लोटिंग लेयर पॉप अप करण्यासाठी बटणावर क्लिक करा.
- सध्या निवडलेली भाषा स्वयंचलितपणे हायलाइट करा.

## डेमो वापरा

```html
<c-i18n></c-i18n>

<script type="module">
import "webc.site/I18n/_.js";
// भाषेतील बदलांचे निरीक्षण करण्यासाठी खालील JS कोड वापरा
import { onLang } from "webc.site/js/i18n.js";

onLang((code, id) => {
  alert(code + ' : ' + id);
});
</script>
```

## इंटरफेस पॅरामीटर्स

### सानुकूल टॅग `<c-i18n>`

सानुकूल एचटीएमएल टॅग, जेएसमध्ये सादर केले गेले आणि एचटीएमएलमध्ये वापरले गेले.

## स्थिती व्यवस्थापन

जागतिक भाषा स्थिती व्यवस्थापित करण्यासाठी `src/js/i18n.js` वापरा:

- **`LANG_LI`**: भाषा सूची ॲरे, `[[नाव, आयडी], ...]` च्या स्वरूपात.
- **`lang()`**: वर्तमान भाषा आयडी मिळवा.
- **`langCode()`**: वर्तमान भाषा एन्कोडिंग मिळवा.
- **`langSet(id)`**: भाषा आयडी सेट करा आणि सदस्यांना सूचित करा.
- **`onLang(func)`**: भाषा बदलांसाठी सदस्यता घ्या. भाषा सेट केली असल्यास, कॉलबॅक त्वरित ट्रिगर केला जाईल. सदस्यता रद्द फंक्शन परत करा.

## शैली वर्ग

### `.BtnC.lang`

चिन्ह बटण.

### `.I18n.Lg`

पॉप-अप विंडोचा मुख्य लेआउट भाषा पर्याय गुंडाळण्यासाठी वापरला जातो.
