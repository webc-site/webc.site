# इंटरफ़ेस भाषाएँ बदलने के लिए बटन और पॉप-अप विंडो

- भाषा चयन फ़्लोटिंग परत को पॉप अप करने के लिए बटन पर क्लिक करें।
- वर्तमान में चयनित भाषा को स्वचालित रूप से हाइलाइट करें।

## डेमो का प्रयोग करें

```html
<c-i18n></c-i18n>

<script type="module">
import "webc.site/I18n/_.js";
// भाषा परिवर्तनों की निगरानी के लिए निम्नलिखित JS कोड का उपयोग करें
import { onLang } from "webc.site/js/i18n.js";

onLang((code, id) => {
  alert(code + ' : ' + id);
});
</script>
```

## इंटरफ़ेस पैरामीटर

### कस्टम टैग `<c-i18n>`

कस्टम HTML टैग, JS में पेश किए गए और HTML में उपयोग किए गए।

## स्थिति प्रबंधन

वैश्विक भाषा स्थिति प्रबंधित करने के लिए `src/js/i18n.js` का उपयोग करें:

- **`LANG_LI`**: भाषा सूची सरणी, <b class='CODE'>[[नाम, आईडी], ...]` के प्रारूप में।
- **`lang()`**: वर्तमान भाषा आईडी प्राप्त करें।
- **`langCode()`**: वर्तमान भाषा एन्कोडिंग प्राप्त करें।
- **`langSet(id)`**: भाषा आईडी सेट करें और ग्राहकों को सूचित करें।
- **`onLang(func)`**: भाषा परिवर्तन की सदस्यता लें। यदि भाषा सेट की गई है, तो कॉलबैक तुरंत चालू हो जाएगा। सदस्यता समाप्त फ़ंक्शन वापस करें.
