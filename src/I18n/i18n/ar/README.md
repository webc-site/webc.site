# الأزرار والنوافذ المنبثقة لتبديل لغات الواجهة

- انقر فوق الزر لتظهر الطبقة العائمة لاختيار اللغة.
- قم بتمييز اللغة المحددة حاليًا تلقائيًا.

## استخدم العرض التوضيحي

```html
<c-i18n></c-i18n>

<script type="module">
import "webc.site/I18n/_.js";
// استخدم كود JS التالي لمراقبة تغييرات اللغة
import { onLang } from "webc.site/js/i18n.js";

onLang((code, id) => {
  alert(code + ' : ' + id);
});
</script>
```

## معلمات الواجهة

### علامة مخصصة `<c-i18n>`

علامات HTML المخصصة، المقدمة في JS والمستخدمة في HTML.

## إدارة الحالة

استخدم `src/js/i18n.js` لإدارة حالة اللغة العالمية:

- **`LANG_LI`**: مصفوفة قائمة اللغات، بتنسيق `[[name, ID], ...]`.
- **`lang()`**: احصل على معرف اللغة الحالي.
- **`langCode()`**: احصل على ترميز اللغة الحالي.
- **`langSet(id)`**: قم بتعيين معرف اللغة وإخطار المشتركين.
- **`onLang(func)`**: اشترك في تغييرات اللغة. إذا تم ضبط اللغة، فسيتم تشغيل رد الاتصال على الفور. إعادة وظيفة إلغاء الاشتراك.
