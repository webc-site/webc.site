# ປຸ່ມ ແລະໜ້າຕ່າງປັອບອັບສຳລັບການປ່ຽນພາສາໃນການໂຕ້ຕອບ

- ຄລິກທີ່ປຸ່ມເພື່ອເປີດຊັ້ນເລືອກພາສາທີ່ເລື່ອນໄດ້.
- ເນັ້ນອັດຕະໂນມັດພາສາທີ່ເລືອກໃນປັດຈຸບັນ.

## ໃຊ້ຕົວຢ່າງ

```html
<c-i18n></c-i18n>

<script type="module">
import "webc.site/I18n/_.js";
// ໃຊ້ລະຫັດ JS ຕໍ່ໄປນີ້ເພື່ອຕິດຕາມການປ່ຽນແປງພາສາ
import { onLang } from "webc.site/js/i18n.js";

onLang((code, id) => {
  alert(code + ' : ' + id);
});
</script>
```

## ຕົວກໍານົດການໃນການໂຕ້ຕອບ

### ແທັກແບບກຳນົດເອງ `<c-i18n>`

ແທັກ HTML ແບບກຳນົດເອງ, ແນະນຳໃນ JS ແລະໃຊ້ໃນ HTML.

## ການຄຸ້ມຄອງສະຖານະພາບ

ໃຊ້ `src/js/i18n.js` ເພື່ອຈັດການສະຖານະພາສາທົ່ວໂລກ:

- **`LANG_LI`**: array ລາຍການພາສາ, ໃນຮູບແບບຂອງ `[[name, ID], ...]`.
- **`lang()`**: ເອົາ ID ພາສາປັດຈຸບັນ.
- **`langCode()`**: ເອົາການເຂົ້າລະຫັດພາສາປັດຈຸບັນ.
- **`langSet(id)`**: ຕັ້ງ ID ພາສາ ແລະແຈ້ງເຕືອນຜູ້ສະໝັກໃຊ້.
- **`onLang(func)`**: ຕິດຕາມການປ່ຽນແປງພາສາ. ຖ້າ​ຫາກ​ວ່າ​ພາ​ສາ​ໄດ້​ຖືກ​ຕັ້ງ​ໄວ້​, callback ຈະ​ໄດ້​ຮັບ​ການ​ກະ​ຕຸ້ນ​ທັນ​ທີ​. ກັບຄືນຟັງຊັນຍົກເລີກການຈອງ.
