# 인터페이스 언어 전환을 위한 버튼 및 팝업 창

- 버튼을 클릭하면 언어 선택 플로팅 레이어가 팝업됩니다.
- 현재 선택한 언어를 자동으로 강조 표시합니다.

## 데모 사용

```html
<c-i18n></c-i18n>

<script type="module">
import "webc.site/I18n.js";
// 언어 변경을 모니터링하려면 다음 JS 코드를 사용하세요.
import { onLang } from "webc.site/js/i18n.js";

onLang((code, id) => {
  alert(code + ' : ' + id);
});
</script>
```

## 인터페이스 매개변수

### 맞춤 태그 `<c-i18n>`

JS에 도입되어 HTML에 사용되는 사용자 정의 HTML 태그입니다.

## 현황관리

글로벌 언어 상태를 관리하려면 `src/js/i18n.js`을 사용하세요.

- **`LANG_LI`**: `[[이름, ID], ...]` 형식의 언어 목록 배열입니다.
- **`lang()`**: 현재 언어 ID를 가져옵니다.
- **`langCode()`**: 현재 언어 인코딩을 가져옵니다.
- **`langSet(id)`**: 언어 ID를 설정하고 구독자에게 알립니다.
- **`onLang(func)`**: 언어 변경 사항을 구독하세요. 언어가 설정된 경우 콜백이 즉시 트리거됩니다. 구독 취소 기능을 반환합니다.

## 스타일 수업

### `.BtnC.lang`

아이콘 버튼입니다.

### `.I18n.Lg`

팝업 창의 기본 레이아웃은 언어 옵션을 래핑하는 데 사용됩니다.
