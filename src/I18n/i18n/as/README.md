# আন্তঃপৃষ্ঠ ভাষা সলনি কৰাৰ বাবে বুটাম আৰু পপ-আপ উইণ্ড'সমূহ

- ভাষা নিৰ্বাচন ভাসমান স্তৰ পপ আপ কৰিবলে বুটাম ক্লিক কৰক।
- বৰ্তমানে নিৰ্বাচিত ভাষাটো স্বয়ংক্ৰিয়ভাৱে হাইলাইট কৰক।

## ডেমো ব্যৱহাৰ কৰক

```html
<c-i18n></c-i18n>

<script type="module">
import "webc.site/I18n/_.js";
// ভাষা পৰিবৰ্তন নিৰীক্ষণ কৰিবলে নিম্নলিখিত JS ক'ড ব্যৱহাৰ কৰক
import { onLang } from "webc.site/js/i18n.js";

onLang((code, id) => {
  alert(code + ' : ' + id);
});
</script>
```

## আন্তঃপৃষ্ঠ প্ৰাচলসমূহ

### স্বনিৰ্বাচিত টেগ `<c-i18n>`

কাষ্টম HTML টেগ, JS ত প্ৰৱৰ্তিত আৰু HTML ত ব্যৱহাৰ কৰা।

## অৱস্থা ব্যৱস্থাপনা

গোলকীয় ভাষাৰ অৱস্থা পৰিচালনা কৰিবলে `src/js/i18n.js` ব্যৱহাৰ কৰক:

- **`LANG_LI`**: ভাষা তালিকা এৰে, `[[name, ID], ...]` ৰ বিন্যাসত।
- **`lang()`**: বৰ্তমান ভাষা আইডি লাভ কৰক।
- **`langCode()`**: বৰ্তমান ভাষা এনকোডিং লাভ কৰক।
- **`langSet(id)`**: ভাষা আইডি নিৰ্ধাৰণ কৰক আৰু গ্ৰাহকসকলক অৱগত কৰক।
- **`onLang(func)`**: ভাষা পৰিৱৰ্তনৰ বাবে চাবস্ক্ৰাইব কৰক। যদি ভাষাটো ছেট কৰা হৈছে, কলবেক তৎক্ষণাত ট্ৰিগাৰ কৰা হ'ব। আনচাবস্ক্ৰাইব ফাংচন ঘূৰাই দিয়ক।

## শৈলী শ্ৰেণী

### `.BtnC.lang`

আইকন বুটাম।

### `.I18n.Lg`

পপ-আপ উইণ্ড'ৰ মূল বিন্যাস ভাষা বিকল্পসমূহ ৰেপ কৰিবলৈ ব্যৱহাৰ কৰা হয়।
