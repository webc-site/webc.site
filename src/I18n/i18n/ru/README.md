# Кнопки и всплывающие окна для переключения языков интерфейса

- Нажмите кнопку, чтобы открыть плавающий слой выбора языка.
- Автоматически выделять выбранный в данный момент язык.

## Используйте демо-версию

```html
<c-i18n></c-i18n>

<script type="module">
import "webc.site/I18n.js";
// Используйте следующий код JS для отслеживания изменений языка
import { onLang } from "webc.site/js/i18n.js";

onLang((code, id) => {
  alert(code + ' : ' + id);
});
</script>
```

## Параметры интерфейса

### Пользовательский тег `<c-i18n>`

Пользовательские теги HTML, представленные в JS и используемые в HTML.

## Управление статусами

Используйте `src/js/i18n.js` для управления глобальным статусом языка:

- **`LANG_LI`**: массив списка языков в формате `[[name, ID], ...]`.
- **`lang()`**: получение текущего идентификатора языка.
- **`langCode()`**: получение текущей языковой кодировки.
- **`langSet(id)`**: установите идентификатор языка и уведомите подписчиков.
- **`onLang(func)`**: подписаться на изменения языка. Если язык был установлен, обратный вызов будет запущен немедленно. Верните функцию отписки.
