# இடைமுக மொழிகளை மாற்றுவதற்கான பொத்தான்கள் மற்றும் பாப்-அப் சாளரங்கள்

- மொழி தேர்வு மிதக்கும் லேயரை பாப் அப் செய்ய பட்டனை கிளிக் செய்யவும்.
- தற்போது தேர்ந்தெடுக்கப்பட்ட மொழியைத் தானாக முன்னிலைப்படுத்தவும்.

## டெமோவைப் பயன்படுத்தவும்

```html
<c-i18n></c-i18n>

<script type="module">
import "webc.site/I18n/_.js";
// மொழி மாற்றங்களைக் கண்காணிக்க பின்வரும் JS குறியீட்டைப் பயன்படுத்தவும்
import { onLang } from "webc.site/js/i18n.js";

onLang((code, id) => {
  alert(code + ' : ' + id);
});
</script>
```

## இடைமுக அளவுருக்கள்

### தனிப்பயன் குறிச்சொல் `<c-i18n>`

தனிப்பயன் HTML குறிச்சொற்கள், JS இல் அறிமுகப்படுத்தப்பட்டு HTML இல் பயன்படுத்தப்படுகின்றன.

## நிலை மேலாண்மை

உலகளாவிய மொழி நிலையை நிர்வகிக்க `src/js/i18n.js` ஐப் பயன்படுத்தவும்:

- **`LANG_LI`**: மொழி பட்டியல் வரிசை, `[[பெயர், ஐடி], ...]` வடிவத்தில்.
- **`lang()`**: தற்போதைய மொழி ஐடியைப் பெறவும்.
- **`langCode()`**: தற்போதைய மொழி குறியாக்கத்தைப் பெறவும்.
- **`langSet(id)`**: மொழி ஐடியை அமைத்து சந்தாதாரர்களுக்கு தெரிவிக்கவும்.
- **`onLang(func)`**: மொழி மாற்றங்களுக்கு குழுசேரவும். மொழி அமைக்கப்பட்டிருந்தால், திரும்ப அழைப்பது உடனடியாகத் தூண்டப்படும். குழுவிலகுதல் செயல்பாட்டைத் திரும்பு.

## பாணி வகுப்பு

### `.BtnC.lang`

ஐகான் பொத்தான்.

### `.I18n.Lg`

பாப்-அப் சாளரத்தின் முக்கிய தளவமைப்பு மொழி விருப்பங்களை மடிக்க பயன்படுத்தப்படுகிறது.
