# Κουμπιά και αναδυόμενα παράθυρα για εναλλαγή γλωσσών διεπαφής

- Κάντε κλικ στο κουμπί για να εμφανιστεί το αιωρούμενο επίπεδο επιλογής γλώσσας.
- Αυτόματη επισήμανση της τρέχουσας επιλεγμένης γλώσσας.

## Χρησιμοποιήστε το demo

```html
<c-i18n></c-i18n>

<script type="module">
import "webc.site/I18n.js";
// Χρησιμοποιήστε τον ακόλουθο κώδικα JS για να παρακολουθείτε τις αλλαγές γλώσσας
import { onLang } from "webc.site/js/i18n.js";

onLang((code, id) => {
  alert(code + ' : ' + id);
});
</script>
```

## Παράμετροι διεπαφής

### Προσαρμοσμένη ετικέτα `<c-i18n>`

Προσαρμοσμένες ετικέτες HTML, που εισήχθησαν στο JS και χρησιμοποιούνται σε HTML.

## Διαχείριση κατάστασης

Χρησιμοποιήστε το `src/js/i18n.js` για να διαχειριστείτε την καθολική κατάσταση της γλώσσας:

- **`LANG_LI`**: Συστοιχία λίστας γλωσσών, σε μορφή `[[όνομα, αναγνωριστικό], ...]`.
- ***`lang()`**: Λάβετε το αναγνωριστικό της τρέχουσας γλώσσας.
- **`langCode()`**: Λάβετε την τρέχουσα κωδικοποίηση γλώσσας.
- **`langSet(id)`**: Ορίστε το αναγνωριστικό γλώσσας και ειδοποιήστε τους συνδρομητές.
- ***`onLang(func)`**: Εγγραφείτε στις αλλαγές γλώσσας. Εάν η γλώσσα έχει οριστεί, η επανάκληση θα ενεργοποιηθεί αμέσως. Επιστρέψτε τη λειτουργία κατάργησης εγγραφής.

## τάξη στυλ

### `.BtnC.lang`

εικονίδιο κουμπί.

### `.I18n.Lg`

Η κύρια διάταξη του αναδυόμενου παραθύρου χρησιμοποιείται για την αναδίπλωση των επιλογών γλώσσας.
