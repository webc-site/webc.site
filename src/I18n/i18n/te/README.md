# ఇంటర్‌ఫేస్ భాషలను మార్చడానికి బటన్‌లు మరియు పాప్-అప్ విండోలు

- భాష ఎంపిక ఫ్లోటింగ్ లేయర్‌ను పాప్ అప్ చేయడానికి బటన్‌ను క్లిక్ చేయండి.
- ప్రస్తుతం ఎంచుకున్న భాషను స్వయంచాలకంగా హైలైట్ చేయండి.

## డెమో ఉపయోగించండి

```html
<c-i18n></c-i18n>

<script type="module">
import "webc.site/I18n.js";
// భాష మార్పులను పర్యవేక్షించడానికి క్రింది JS కోడ్‌ని ఉపయోగించండి
import { onLang } from "webc.site/js/i18n.js";

onLang((code, id) => {
  alert(code + ' : ' + id);
});
</script>
```

## ఇంటర్ఫేస్ పారామితులు

### అనుకూల ట్యాగ్ `<c-i18n>`

అనుకూల HTML ట్యాగ్‌లు, JSలో పరిచయం చేయబడ్డాయి మరియు HTMLలో ఉపయోగించబడ్డాయి.

## స్థితి నిర్వహణ

ప్రపంచ భాషా స్థితిని నిర్వహించడానికి `src/js/i18n.js`ని ఉపయోగించండి:

- **`LANG_LI`**: భాష జాబితా శ్రేణి, `[[పేరు, ID], ...]` ఆకృతిలో.
- **`lang()`**: ప్రస్తుత భాషా IDని పొందండి.
- **`langCode()`**: ప్రస్తుత భాషా ఎన్‌కోడింగ్‌ను పొందండి.
- **`langSet(id)`**: భాష IDని సెట్ చేయండి మరియు చందాదారులకు తెలియజేయండి.
- **`onLang(func)`**: భాష మార్పులకు సభ్యత్వం పొందండి. భాష సెట్ చేయబడితే, కాల్‌బ్యాక్ వెంటనే ట్రిగ్గర్ చేయబడుతుంది. అన్‌సబ్‌స్క్రైబ్ ఫంక్షన్‌ని తిరిగి ఇవ్వండి.

## శైలి తరగతి

### `.BtnC.lang`

చిహ్నం బటన్.

### `.I18n.Lg`

భాషా ఎంపికలను చుట్టడానికి పాప్-అప్ విండో యొక్క ప్రధాన లేఅవుట్ ఉపయోగించబడుతుంది.
