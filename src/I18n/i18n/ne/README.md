# इन्टरफेस भाषाहरू स्विच गर्नका लागि बटनहरू र पप-अप विन्डोहरू

- भाषा चयन फ्लोटिंग तह पप अप गर्न बटन क्लिक गर्नुहोस्।
- हाल चयन गरिएको भाषा स्वचालित रूपमा हाइलाइट गर्नुहोस्।

## डेमो प्रयोग गर्नुहोस्

```html
<c-i18n></c-i18n>

<script type="module">
import "webc.site/I18n/_.js";
// भाषा परिवर्तनहरू निगरानी गर्न निम्न JS कोड प्रयोग गर्नुहोस्
import { onLang } from "webc.site/js/i18n.js";

onLang((code, id) => {
  alert(code + ' : ' + id);
});
</script>
```

## इन्टरफेस प्यारामिटरहरू

### आफू अनुकूल ट्याग `<c-i18n>`

अनुकूलन HTML ट्यागहरू, JS मा पेश गरियो र HTML मा प्रयोग गरियो।

## स्थिति व्यवस्थापन

विश्वव्यापी भाषा स्थिति व्यवस्थापन गर्न `src/js/i18n.js` प्रयोग गर्नुहोस्:

- **`LANG_LI`**: भाषा सूची एरे, `[[नाम, ID], ...]` को ढाँचामा।
- **`lang()`**: हालको भाषा ID प्राप्त गर्नुहोस्।
- **`langCode()`**: हालको भाषा इन्कोडिङ प्राप्त गर्नुहोस्।
- **`langSet(id)`**: भाषा ID सेट गर्नुहोस् र सदस्यहरूलाई सूचित गर्नुहोस्।
- **`onLang(func)`**: भाषा परिवर्तनहरूको सदस्यता लिनुहोस्। यदि भाषा सेट गरिएको छ भने, कलब्याक तुरुन्तै ट्रिगर हुनेछ। सदस्यता रद्द गर्ने प्रकार्य फर्काउनुहोस्।
