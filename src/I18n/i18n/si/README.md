# අතුරු මුහුණත් භාෂා මාරු කිරීම සඳහා බොත්තම් සහ උත්පතන කවුළු

- භාෂා තේරීමේ පාවෙන ස්තරය උත්පතන කිරීමට බොත්තම ක්ලික් කරන්න.
- දැනට තෝරාගත් භාෂාව ස්වයංක්‍රීයව උද්දීපනය කරන්න.

## Demo භාවිතා කරන්න

```html
<c-i18n></c-i18n>

<script type="module">
import "webc.site/I18n/_.js";
// භාෂා වෙනස්කම් නිරීක්ෂණය කිරීමට පහත JS කේතය භාවිතා කරන්න
import { onLang } from "webc.site/js/i18n.js";

onLang((code, id) => {
  alert(code + ' : ' + id);
});
</script>
```

## අතුරුමුහුණත් පරාමිතීන්

### අභිරුචි ටැගය `<c-i18n>`

අභිරුචි HTML ටැග්, JS හි හඳුන්වා දී HTML හි භාවිතා වේ.

## තත්ව කළමනාකරණය

ගෝලීය භාෂා තත්ත්වය කළමනාකරණය කිරීමට `src/js/i18n.js` භාවිතා කරන්න:

- **`LANG_LI`**: භාෂා ලැයිස්තු අරාව, `[[නම, ID], ...]` ආකෘතියෙන්.
- **`lang()`**: වත්මන් භාෂා හැඳුනුම්පත ලබා ගන්න.
- **`langCode()`**: වත්මන් භාෂා කේතනය ලබා ගන්න.
- **`langSet(id)`**: භාෂා හැඳුනුම්පත සකසා ග්‍රාහකයින්ට දැනුම් දෙන්න.
- **`onLang(func)`**: භාෂා වෙනස්කම් සඳහා දායක වන්න. භාෂාව සකසා ඇත්නම්, ආපසු ඇමතීම වහාම ක්‍රියාත්මක වේ. දායකත්වයෙන් ඉවත් වීමේ කාර්යය ආපසු දෙන්න.

## ශෛලිය පන්තිය

### `.BtnC.lang`

අයිකන බොත්තම.

### `.I18n.Lg`

උත්පතන කවුළුවේ ප්‍රධාන පිරිසැලසුම භාෂා විකල්ප එතීමට භාවිතා කරයි.
