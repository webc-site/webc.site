# бһ”бҹҠбһјбһҸбһ»бһ„ бһ“бһ·бһ„бһ”бһ„бҹ’бһўбһҪбһ…бһӣбҹҒбһ…бһЎбһҫбһ„бһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһ”бҹ’бһҸбһјбһҡбһ—бһ¶бһҹбһ¶бһ…бҹҶбһҺбһ»бһ…бһ”бҹ’бһҡбһ‘бһ¶бһҖбҹӢ

- бһ…бһ»бһ…вҖӢбһ”бҹҠбһјбһҸбһ»бһ„вҖӢбһҠбһҫбһҳбҹ’бһ”бһёвҖӢбһ”бһ„бҹ’бһ бһ¶бһүвҖӢбһҹбҹ’бһҡбһ‘бһ¶бһ”бҹӢвҖӢбһ”бһҺбҹ’бһҠбҹӮбһҸвҖӢбһҮбһҳбҹ’бһҡбһҫбһҹвҖӢбһ—бһ¶бһҹбһ¶бҹ”
- бһҡбҹҶбһӣбҹҒбһ…бһ—бһ¶бһҹбһ¶бһҠбҹӮбһӣбһ”бһ¶бһ“бһҮбҹ’бһҡбһҫбһҹбһҡбһҫбһҹбһ”бһ…бҹ’бһ…бһ»бһ”бҹ’бһ”бһ“бҹ’бһ“бһҠбҹ„бһҷбһҹбҹ’бһңбҹҗбһҷбһ”бҹ’бһҡбһңбһҸбҹ’бһҸбһ·бҹ”

## бһ”бҹ’бһҡбһҫбһҖбһ¶бһҡбһ”бһ„бҹ’бһ бһ¶бһү

```html
<c-i18n></c-i18n>

<script type="module">
import "webc.site/I18n/_.js";
// бһ”бҹ’бһҡбһҫбһҖбһјбһҠ JS бһҒбһ¶бһ„бһҖбҹ’бһҡбҹ„бһҳбһҠбһҫбһҳбҹ’бһ”бһёбһҸбһ¶бһҳбһҠбһ¶бһ“бһҖбһ¶бһҡбһ•бҹ’бһӣбһ¶бһҹбҹӢбһ”бҹ’бһҸбһјбһҡбһ—бһ¶бһҹбһ¶
import { onLang } from "webc.site/js/i18n.js";

onLang((code, id) => {
  alert(code + ' : ' + id);
});
</script>
```

## бһ”бҹүбһ¶бһҡбҹүбһ¶бһҳбҹүбҹӮбһҸбҹ’бһҡбһ…бҹҶбһҺбһ»бһ…бһ”бҹ’бһҡбһ‘бһ¶бһҖбҹӢ

### бһҹбҹ’бһӣбһ¶бһҖбһ•бҹ’бһ‘бһ¶бһӣбҹӢбһҒбҹ’бһӣбһҪбһ“ `<c-i18n>`

бһҹбҹ’бһӣбһ¶бһҖ HTML бһ•бҹ’бһ‘бһ¶бһӣбҹӢбһҒбҹ’бһӣбһҪбһ“ бһҺбҹӮбһ“бһ¶бҹҶбһҖбҹ’бһ“бһ»бһ„ JS бһ“бһ·бһ„бһ”бҹ’бһҡбһҫбһҖбҹ’бһ“бһ»бһ„ HTML бҹ”

## бһҖбһ¶бһҡбһӮбҹ’бһҡбһ”бҹӢбһӮбҹ’бһҡбһ„бһҹбҹ’бһҗбһ¶бһ“бһ—бһ¶бһ–

бһ”бҹ’бһҡбһҫ `src/js/i18n.js` бһҠбһҫбһҳбҹ’бһ”бһёбһӮбҹ’бһҡбһ”бҹӢбһӮбҹ’бһҡбһ„бһҹбҹ’бһҗбһ¶бһ“бһ—бһ¶бһ–бһ—бһ¶бһҹбһ¶бһҹбһҖбһӣбҹ–

- **`LANG_LI`**бҹ– бһўбһ¶бһҡбҹҒбһ”бһүбҹ’бһҮбһёбһ—бһ¶бһҹбһ¶ бһҖбҹ’бһ“бһ»бһ„бһ‘бһҳбҹ’бһҡбһ„бҹӢ `[[name, ID], ...]`бҹ”
- **`lang()`**бҹ– бһ‘бһ‘бһҪбһӣбһ”бһ¶бһ“бһӣбҹҒбһҒбһҹбһҳбҹ’бһӮбһ¶бһӣбҹӢбһ—бһ¶бһҹбһ¶бһ”бһ…бҹ’бһ…бһ»бһ”бҹ’бһ”бһ“бҹ’бһ“бҹ”
- **`langCode()`**бҹ– бһ‘бһ‘бһҪбһӣбһҖбһ¶бһҡбһўбҹҠбһ·бһ“бһҖбһјбһҠбһ—бһ¶бһҹбһ¶бһ”бһ…бҹ’бһ…бһ»бһ”бҹ’бһ”бһ“бҹ’бһ“бҹ”
- **`langSet(id)`**бҹ– бһҖбҹҶбһҺбһҸбҹӢбһӣбҹҒбһҒбһҹбһҳбҹ’бһӮбһ¶бһӣбҹӢбһ—бһ¶бһҹбһ¶ бһ“бһ·бһ„бһҮбһјбһ“бһҠбҹҶбһҺбһ№бһ„бһҠбһӣбҹӢбһўбҹ’бһ“бһҖбһҮбһ¶бһңбҹ”
- **`onLang(func)`**бҹ– бһҮбһ¶бһңбһҖбһ¶бһҡбһ•бҹ’бһӣбһ¶бһҹбҹӢбһ”бҹ’бһҸбһјбһҡбһ—бһ¶бһҹбһ¶бҹ” бһ”бҹ’бһҡбһҹбһ·бһ“бһ”бһҫбһ—бһ¶бһҹбһ¶бһҸбҹ’бһҡбһјбһңбһ”бһ¶бһ“бһҖбҹҶбһҺбһҸбҹӢ бһҖбһ¶бһҡбһ бҹ…бһҸбҹ’бһҡбһЎбһ”бҹӢбһҳбһҖбһңбһ·бһүбһ“бһ№бһ„бһҸбҹ’бһҡбһјбһңбһ”бһ¶бһ“бһҖбҹҒбҹҮбһ—бҹ’бһӣбһ¶бһҳбҹ—бҹ” бһҸбҹ’бһҡбһЎбһ”бҹӢбһҳбһ»бһҒбһ„бһ¶бһҡбһҲбһ”бҹӢбһҮбһ¶бһңбҹ”

## бһҗбҹ’бһ“бһ¶бһҖбҹӢбһҡбһ…бһ“бһ¶бһ”бҹҗбһ‘бҹ’бһҳ

### `.BtnC.lang`

бһ”бҹҠбһјбһҸбһ»бһ„бһҡбһјбһ”бһҸбҹҶбһҺбһ¶бһ„бҹ”

### `.I18n.Lg`

бһ”бҹ’бһӣбһ„бҹӢбһ…бһҳбҹ’бһ”бһ„бһ“бҹғбһ”бһ„бҹ’бһўбһҪбһ…бһӣбҹҒбһ…бһЎбһҫбһ„бһҸбҹ’бһҡбһјбһңбһ”бһ¶бһ“бһ”бҹ’бһҡбһҫбһҠбһҫбһҳбҹ’бһ”бһёбһҡбһ»бҹҶбһҮбһҳбҹ’бһҡбһҫбһҹбһ—бһ¶бһҹбһ¶бҹ”
