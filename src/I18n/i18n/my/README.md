# အင်တာဖေ့စ်ဘာသာစကားများကို ပြောင်းရန်အတွက် ခလုတ်များနှင့် ပေါ့ပ်အပ်ဝင်းဒိုးများ

- ဘာသာစကားရွေးချယ်မှု ရေပေါ်အလွှာကို ပေါ်လာရန် ခလုတ်ကို နှိပ်ပါ။
- လက်ရှိရွေးချယ်ထားသော ဘာသာစကားကို အလိုအလျောက် မီးမောင်းထိုးပြပါ။

## ဒီမိုကိုသုံးပါ။

```html
<c-i18n></c-i18n>

<script type="module">
import "webc.site/I18n/_.js";
// ဘာသာစကားပြောင်းလဲမှုများကို စောင့်ကြည့်ရန် အောက်ပါ JS ကုဒ်ကို အသုံးပြုပါ။
import { onLang } from "webc.site/js/i18n.js";

onLang((code, id) => {
  alert(code + ' : ' + id);
});
</script>
```

## မျက်နှာပြင်ဘောင်များ

### စိတ်ကြိုက်တဂ် `<c-i18n>`

JS တွင်မိတ်ဆက်ပြီး HTML တွင်အသုံးပြုသော စိတ်ကြိုက် HTML တဂ်များ။

## အဆင့်အတန်းစီမံခန့်ခွဲမှု

ကမ္ဘာလုံးဆိုင်ရာဘာသာစကားအဆင့်အတန်းကိုစီမံခန့်ခွဲရန် `src/js/i18n.js`ကိုသုံးပါ-

- **`LANG_LI`**- `[[အမည်၊ ID]၊ ...]` ဖော်မတ်ဖြင့် ဘာသာစကားစာရင်း ခင်းကျင်းမှု။
- **`lang()`**- လက်ရှိ ဘာသာစကား ID ကို ရယူပါ။
- **`langCode()`**- လက်ရှိ ဘာသာစကား ကုဒ်ကို ရယူပါ။
- **`langSet(id)`**- ဘာသာစကား ID ကို သတ်မှတ်ပြီး စာရင်းသွင်းသူများကို အကြောင်းကြားပါ။
- **`onLang(func)`**- ဘာသာစကား ပြောင်းလဲမှုများကို စာရင်းသွင်းပါ။ ဘာသာစကားကို သတ်မှတ်ပြီးပါက၊ ဖုန်းပြန်ခေါ်ခြင်းကို ချက်ချင်းစတင်ပါမည်။ စာရင်းသွင်းခြင်းမှ ရပ်ဆိုင်းခြင်း လုပ်ဆောင်ချက်ကို ပြန်ပေးပါ။
