# Кнопки та спливаючі вікна для перемикання мов інтерфейсу

- Натисніть кнопку, щоб відкрити плаваючий шар вибору мови.
- Автоматично виділяти поточну вибрану мову.

## Використовуйте демо

```html
<c-i18n></c-i18n>

<script type="module">
import "webc.site/I18n/_.js";
// Використовуйте наступний код JS для моніторингу змін мови
import { onLang } from "webc.site/js/i18n.js";

onLang((code, id) => {
  alert(code + ' : ' + id);
});
</script>
```

## Параметри інтерфейсу

### Спеціальний тег `<c-i18n>`

Спеціальні теги HTML, представлені в JS і використані в HTML.

## Управління статусом

Використовуйте `src/js/i18n.js` для керування глобальним статусом мови:

- **`LANG_LI`**: масив списку мов у форматі `[[ім’я, ідентифікатор], ...]`.
- **`lang()`**: отримати ідентифікатор поточної мови.
- **`langCode()`**: отримати кодування поточної мови.
- **`langSet(id)`**: установіть ідентифікатор мови та повідомте підписників.
- **`onLang(func)`**: підписка на мовні зміни. Якщо мову встановлено, зворотний дзвінок буде запущено негайно. Повернути функцію скасування підписки.
