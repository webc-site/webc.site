# ଇଣ୍ଟରଫେସ୍ ଭାଷା ବଦଳାଇବା ପାଇଁ ବଟନ୍ ଏବଂ ପପ୍-ଅପ୍ ୱିଣ୍ଡୋ |

- ଭାଷା ଚୟନ ଭାସମାନ ସ୍ତର ପପ୍ ଅପ୍ କରିବାକୁ ବଟନ୍ କ୍ଲିକ୍ କରନ୍ତୁ |
- ସମ୍ପ୍ରତି ମନୋନୀତ ଭାଷାକୁ ସ୍ୱୟଂଚାଳିତ ଭାବରେ ହାଇଲାଇଟ୍ କରନ୍ତୁ |

## ଡେମୋ ବ୍ୟବହାର କରନ୍ତୁ |

```html
<c-i18n></c-i18n>

<script type="module">
import "webc.site/I18n/_.js";
// ଭାଷା ପରିବର୍ତ୍ତନ ଉପରେ ନଜର ରଖିବା ପାଇଁ ନିମ୍ନଲିଖିତ JS କୋଡ୍ ବ୍ୟବହାର କରନ୍ତୁ |
import { onLang } from "webc.site/js/i18n.js";

onLang((code, id) => {
  alert(code + ' : ' + id);
});
</script>
```

## ଇଣ୍ଟରଫେସ୍ ପାରାମିଟରଗୁଡିକ

### କଷ୍ଟମ୍ ଟ୍ୟାଗ୍ `<c-i18n>` |

କଷ୍ଟମ୍ HTML ଟ୍ୟାଗ୍, JS ରେ ପରିଚିତ ଏବଂ HTML ରେ ବ୍ୟବହୃତ |

## ସ୍ଥିତି ପରିଚାଳନା

ବିଶ୍ୱ ଭାଷା ସ୍ଥିତିକୁ ପରିଚାଳନା କରିବା ପାଇଁ `src/js/i18n.js` ବ୍ୟବହାର କରନ୍ତୁ:

- ** `LANG_LI` **: ଭାଷା ତାଲିକା ଆରେ, <b ଶ୍ରେଣୀ = "କୋଡ୍"> [[ନାମ, ID], ...] ` ଫର୍ମାଟରେ |
- ** `lang()` **: ସାମ୍ପ୍ରତିକ ଭାଷା ID ପ୍ରାପ୍ତ କରନ୍ତୁ |
- ** `langCode()` **: ସାମ୍ପ୍ରତିକ ଭାଷା ଏନକୋଡିଂ ପାଆନ୍ତୁ |
- ** `langSet(id)` **: ଭାଷା ID ସେଟ୍ କରନ୍ତୁ ଏବଂ ଗ୍ରାହକମାନଙ୍କୁ ଜଣାନ୍ତୁ |
- ** `onLang(func)` **: ଭାଷା ପରିବର୍ତ୍ତନକୁ ସବସ୍କ୍ରାଇବ କରନ୍ତୁ | ଯଦି ଭାଷା ସେଟ୍ ହୋଇଛି, ତୁରନ୍ତ କଲବ୍ୟାକ୍ ଟ୍ରିଗର ହେବ | ସଦସ୍ୟତା ଫଙ୍କସନ୍ ଫେରସ୍ତ କରନ୍ତୁ |

## ଶ style ଳୀ ଶ୍ରେଣୀ

### `.BtnC.lang`

ଆଇକନ୍ ବଟନ୍ |
