# Бутони и изскачащи прозорци за превключване на езиците на интерфейса

- Щракнете върху бутона, за да изскочите плаващия слой за избор на език.
- Автоматично маркиране на текущо избрания език.

## Използвайте демонстрацията

```html
<c-i18n></c-i18n>

<script type="module">
import "webc.site/I18n.js";
// Използвайте следния JS код, за да наблюдавате езиковите промени
import { onLang } from "webc.site/js/i18n.js";

onLang((code, id) => {
  alert(code + ' : ' + id);
});
</script>
```

## Параметри на интерфейса

### Персонализиран етикет `<c-i18n>`

Персонализирани HTML тагове, въведени в JS и използвани в HTML.

## Управление на състоянието

Използвайте `src/js/i18n.js`, за да управлявате състоянието на глобалния език:

- **`LANG_LI`**: масив от езиков списък във формат `[[име, ID], ...]`.
- **`lang()`**: Вземете текущия идентификатор на езика.
- **`langCode()`**: Вземете текущото езиково кодиране.
- **`langSet(id)`**: Задайте ID на езика и уведомете абонатите.
- **`onLang(func)`**: Абонирайте се за езикови промени. Ако езикът е зададен, обратното повикване ще се задейства незабавно. Върнете функцията за отписване.
