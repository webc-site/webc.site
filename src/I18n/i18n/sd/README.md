# انٽرفيس ٻولين کي مٽائڻ لاءِ بٽڻ ۽ پاپ اپ ونڊوز

- ٻولي جي چونڊ سچل پرت کي پاپ اپ ڪرڻ لاءِ بٽڻ تي ڪلڪ ڪريو.
- خودڪار طور تي ھاڻي منتخب ٿيل ٻولي کي نمايان ڪريو.

## ڊيمو استعمال ڪريو

```html
<c-i18n></c-i18n>

<script type="module">
import "webc.site/I18n/_.js";
// ٻولي جي تبديلين جي نگراني ڪرڻ لاء ھيٺ ڏنل JS ڪوڊ استعمال ڪريو
import { onLang } from "webc.site/js/i18n.js";

onLang((code, id) => {
  alert(code + ' : ' + id);
});
</script>
```

## انٽرفيس پيٽرول

### ڪسٽم ٽيگ `<c-i18n>`

ڪسٽم HTML ٽيگ، JS ۾ متعارف ڪرايو ۽ HTML ۾ استعمال ڪيو ويو.

## اسٽيٽس مينيجمينٽ

استعمال ڪريو `src/js/i18n.js` عالمي ٻولي جي صورتحال کي منظم ڪرڻ لاء:

- **`LANG_LI`**: ٻوليءَ جي فهرست جي صف، فارميٽ ۾ `[[نالو، ID]، ...]`.
- **`lang()`**: موجوده ٻولي جي سڃاڻپ حاصل ڪريو.
- **`langCode()`**: موجوده ٻولي انڪوڊنگ حاصل ڪريو.
- **`langSet(id)`**: ٻولي جي سڃاڻپ مقرر ڪريو ۽ رڪنن کي اطلاع ڏيو.
- **`onLang(func)`**: ٻولي تبديلين لاءِ رڪنيت حاصل ڪريو. جيڪڏهن ٻولي مقرر ڪئي وئي آهي، ڪال بيڪ فوري طور تي شروع ڪيو ويندو. رڪنيت ختم ڪرڻ واري فنڪشن کي واپس ڏيو.
