# לחצנים וחלונות קופצים להחלפת שפות ממשק

- לחץ על הלחצן כדי להקפיץ את השכבה הצפה של בחירת השפה.
- סמן אוטומטית את השפה הנבחרת כעת.

## השתמש בהדגמה

```html
<c-i18n></c-i18n>

<script type="module">
import "webc.site/I18n.js";
// השתמש בקוד JS הבא כדי לפקח על שינויים בשפה
import { onLang } from "webc.site/js/i18n.js";

onLang((code, id) => {
  alert(code + ' : ' + id);
});
</script>
```

## פרמטרים של ממשק

### תג מותאם אישית `<c-i18n>`

תגי HTML מותאמים אישית, הוצגו ב-JS ונעשה בהם שימוש ב-HTML.

## ניהול סטטוס

השתמש ב-`src/js/i18n.js` לניהול סטטוס שפה גלובלי:

- **`LANG_LI`**: מערך רשימת שפות, בפורמט של `[[שם, מזהה], ...]`.
- ***`lang()`**: קבל את מזהה השפה הנוכחי.
- **`langCode()`**: קבל את קידוד השפה הנוכחי.
- **`langSet(id)`**: הגדר את מזהה השפה והודע למנויים.
- ***`onLang(func)`**: הירשם לשינויי שפה. אם השפה הוגדרה, ההתקשרות חזרה תופעל מיד. החזר את פונקציית ביטול הרישום.

## שיעור סגנון

### `.BtnC.lang`

לחצן סמל.

### `.I18n.Lg`

הפריסה הראשית של החלון הקופץ משמשת לעטוף את אפשרויות השפה.
