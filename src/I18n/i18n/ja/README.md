# インターフェイス言語を切り替えるためのボタンとポップアップ ウィンドウ

- ボタンをクリックすると、言語選択のフローティング レイヤーがポップアップ表示されます。
- 現在選択されている言語を自動的に強調表示します。

## デモを使用する

```html
<c-i18n></c-i18n>

<script type="module">
import "webc.site/I18n.js";
// 次の JS コードを使用して言語の変更を監視します
import { onLang } from "webc.site/js/i18n.js";

onLang((code, id) => {
  alert(code + ' : ' + id);
});
</script>
```

## インターフェースパラメータ

### カスタムタグ `<c-i18n>`

カスタム HTML タグ。JS で導入され、HTML で使用されます。

## ステータス管理

`src/js/i18n.js` を使用してグローバル言語ステータスを管理します。

- **`LANG_LI`**: `[[name, ID], ...]` の形式の言語リスト配列。
- **`lang()`**: 現在の言語 ID を取得します。
- **`langCode()`**: 現在の言語エンコーディングを取得します。
- **`langSet(id)`**: 言語 ID を設定し、購読者に通知します。
- **`onLang(func)`**: 言語の変更を購読します。言語が設定されている場合、コールバックはすぐにトリガーされます。購読解除関数を返します。

## スタイルクラス

### `.BtnC.lang`

アイコンボタン。

### `.I18n.Lg`

ポップアップ ウィンドウのメイン レイアウトは、言語オプションをラップするために使用されます。
