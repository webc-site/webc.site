# ปุ่มและหน้าต่างป๊อปอัปสำหรับเปลี่ยนภาษาอินเทอร์เฟซ

- คลิกปุ่มเพื่อเปิดเลเยอร์การเลือกภาษาแบบลอยขึ้นมา
- ไฮไลท์ภาษาที่เลือกในปัจจุบันโดยอัตโนมัติ

## ใช้การสาธิต

```html
<c-i18n></c-i18n>

<script type="module">
import "webc.site/I18n/_.js";
// ใช้รหัส JS ต่อไปนี้เพื่อตรวจสอบการเปลี่ยนแปลงภาษา
import { onLang } from "webc.site/js/i18n.js";

onLang((code, id) => {
  alert(code + ' : ' + id);
});
</script>
```

## พารามิเตอร์อินเทอร์เฟซ

### แท็กที่กำหนดเอง `<c-i18n>`

แท็ก HTML แบบกำหนดเอง เปิดตัวใน JS และใช้ใน HTML

## การจัดการสถานะ

ใช้ `src/js/i18n.js` เพื่อจัดการสถานะภาษาสากล:

- **`LANG_LI`**: อาร์เรย์รายการภาษา ในรูปแบบของ `[[ชื่อ, ID], ...]`
- ***`lang()`**: รับรหัสภาษาปัจจุบัน
- ***`langCode()`**: รับการเข้ารหัสภาษาปัจจุบัน
- **`langSet(id)`**: ตั้งค่ารหัสภาษาและแจ้งเตือนสมาชิก
- ***`onLang(func)`**: สมัครรับข้อมูลการเปลี่ยนแปลงภาษา หากตั้งค่าภาษาไว้ ระบบจะโทรกลับทันที ส่งคืนฟังก์ชันยกเลิกการสมัคร
