# par ključev vrednosti

Slog značke kapsule, razporejen v dvojnih besedilnih blokih, ki se pogosto uporablja za prikaz statusa, različice ali informacij o ključni vrednosti.
- Prikaz v dveh stolpcih: podpira dva besedilna bloka, razporejena levo in desno ter lahko razlikuje imena atributov in vsebino poziva
- Tekoče steklo: sledi matiranemu prosojnemu dizajnu s subtilnimi notranjimi sencami in povratnimi informacijami mikro zooma pri lebdenju
- Podpora za vsebnik: podpira en zaslon in prilagodljivo razporeditev razmikov v vsebnikih s seznami

## Uporabite demo
```html
<main class="KvLi">
  <a href="#">
    <b>Version</b>
    <b>1.0.0</b>
  </a>
  <a href="#">
    <b>License</b>
    <b>MIT</b>
  </a>
</main>

<a class="Kv" href="#">
  <b>Status</b>
  <b>Active</b>
</a>
```
