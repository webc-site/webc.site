# пара ключ-значение

Стиль значка капсулы, организованный в виде двойных текстовых блоков, часто используется для отображения информации о статусе, версии или ключевом значении.
- Отображение двух столбцов: поддерживает два текстовых блока, расположенных слева и справа, и позволяет различать имена атрибутов и содержимое подсказок.
- Жидкое стекло: повторяет матовый полупрозрачный дизайн с тонкими внутренними тенями и микрозумом при наведении курсора.
- Поддержка контейнеров: поддерживает одиночный дисплей и адаптивное расположение интервалов в контейнерах списков.

## Используйте демо-версию
```html
<main class="KvLi">
  <a href="#">
    <b>Version</b>
    <b>1.0.0</b>
  </a>
  <a href="#">
    <b>License</b>
    <b>MIT</b>
  </a>
</main>

<a class="Kv" href="#">
  <b>Status</b>
  <b>Active</b>
</a>
```
