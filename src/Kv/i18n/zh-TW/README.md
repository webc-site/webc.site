# 鍵值對

雙文本區塊排列的膠囊徽章樣式，常用於展示狀態、版本或鍵值資訊。
- 雙欄展示：支援兩個文字區塊左右排列，可區分屬性名稱與提示內容
- 液態玻璃：遵循磨砂半透明設計，具有細膩的內陰影與懸停微縮放回饋
- 容器支援：支援單體展示，也支援在列表容器中自適應間距排列

## 使用演示
```html
<main class="KvLi">
  <a href="#">
    <b>Version</b>
    <b>1.0.0</b>
  </a>
  <a href="#">
    <b>License</b>
    <b>MIT</b>
  </a>
</main>

<a class="Kv" href="#">
  <b>Status</b>
  <b>Active</b>
</a>
```
