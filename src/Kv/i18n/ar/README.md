# زوج القيمة الرئيسية

نمط شارة كبسولة مرتب في كتل نصية مزدوجة، يُستخدم غالبًا لعرض الحالة أو الإصدار أو معلومات القيمة الأساسية.
- عرض عمود مزدوج: يدعم كتلتين نصيتين تم ترتيبهما على اليسار واليمين، ويمكنهما التمييز بين أسماء السمات ومحتوى المطالبة
- الزجاج السائل: يتبع تصميمًا شفافًا بلوريًا مع ظلال داخلية دقيقة وردود فعل تكبير/تصغير دقيقة عند التمرير
- دعم الحاوية: يدعم العرض الفردي وترتيب التباعد التكيفي في حاويات القائمة
