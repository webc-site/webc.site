# gako-balio bikotea

Testu-bloke bikoitzetan antolatutako kapsula-txapa estiloa, sarritan egoera, bertsioa edo gako-balioaren informazioa bistaratzeko erabiltzen da.
- Zutabe bikoitzeko pantaila: ezkerreko eta eskuineko bi testu-bloke onartzen ditu eta atributuen izenak eta gonbidapen-edukia bereiz ditzake.
- Beira likidoa: diseinu zeharrargi izoztu bati jarraitzen dio, barruko itzal sotilekin eta mikrozoom-adierazpena igarotzean.
- Edukiontzien euskarria: pantaila bakarra eta tarte egokitzeko antolaketa onartzen ditu zerrenda-edukiontzietan
