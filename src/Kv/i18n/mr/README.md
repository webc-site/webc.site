# की मूल्य जोडी

दुहेरी मजकूर ब्लॉकमध्ये व्यवस्था केलेली कॅप्सूल बॅज शैली, बहुतेकदा स्थिती, आवृत्ती किंवा मुख्य मूल्य माहिती प्रदर्शित करण्यासाठी वापरली जाते.
- डबल-कॉलम डिस्प्ले: डावीकडे आणि उजवीकडे व्यवस्था केलेल्या दोन मजकूर ब्लॉकला समर्थन देते आणि विशेषता नावे आणि प्रॉम्प्ट सामग्री वेगळे करू शकतात
- लिक्विड ग्लास: होवरवर सूक्ष्म आतील सावल्या आणि मायक्रो-झूम फीडबॅकसह फ्रॉस्टेड अर्धपारदर्शक डिझाइनचे अनुसरण करते
- कंटेनर सपोर्ट: सूचीच्या कंटेनरमध्ये सिंगल डिस्प्ले आणि अडॅप्टिव्ह स्पेसिंग व्यवस्था समर्थित करते
