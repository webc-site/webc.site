# nøkkelverdipar

En kapselmerkestil arrangert i doble tekstblokker, ofte brukt til å vise status, versjon eller nøkkelverdiinformasjon.
- Visning med dobbel kolonne: støtter to tekstblokker arrangert til venstre og høyre, og kan skille attributtnavn og ledetekstinnhold
- Flytende glass: Følger en frostet gjennomskinnelig design med subtile indre skygger og tilbakemelding med mikrozoom ved sveving
- Beholderstøtte: støtter enkeltdisplay og adaptivt avstandsarrangement i listebeholdere

## Bruk demoen
```html
<main class="KvLi">
  <a href="#">
    <b>Version</b>
    <b>1.0.0</b>
  </a>
  <a href="#">
    <b>License</b>
    <b>MIT</b>
  </a>
</main>

<a class="Kv" href="#">
  <b>Status</b>
  <b>Active</b>
</a>
```
