# သော့တန်ဖိုးအတွဲ

အခြေအနေ၊ ဗားရှင်း သို့မဟုတ် သော့တန်ဖိုးအချက်အလက်ကိုပြသရန် မကြာခဏအသုံးပြုသည့် စာသားဘလောက်နှစ်ထပ်တွင် ကာဆူးလ်တံဆိပ်ပုံစံ။
- နှစ်ထပ်ကော်လံပြသခြင်း- ဘယ်နှင့်ညာ စီထားသော စာသားဘလောက်နှစ်ခုကို ပံ့ပိုးပေးကာ ရည်ညွှန်းချက်အမည်များနှင့် အချက်ပြအကြောင်းအရာများကို ခွဲခြားနိုင်သည်
- Liquid Glass- နှင်းခဲထားသော အလင်းဝင်သည့် ဒီဇိုင်းကို အောက်ခြေတွင် အတွင်းမှ အရိပ်များနှင့် မိုက်ခရိုချုံ့ချဲ့ တုံ့ပြန်ချက်တို့နှင့်အတူ လိုက်ပါသည်
- ကွန်တိန်နာပံ့ပိုးမှု- စာရင်းပုံးများတွင် တစ်ခုတည်းသောပြသမှုနှင့် လိုက်လျောညီထွေရှိသော အကွာအဝေးအစီအစဉ်ကို ပံ့ပိုးပေးသည်။
