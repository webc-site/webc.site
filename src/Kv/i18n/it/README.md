# coppia chiave-valore

Uno stile badge capsula disposto in blocchi di testo doppi, spesso utilizzato per visualizzare informazioni sullo stato, sulla versione o sul valore chiave.
- Visualizzazione a doppia colonna: supporta due blocchi di testo disposti a sinistra e a destra e può distinguere i nomi degli attributi e il contenuto dei prompt
- Vetro liquido: segue un design traslucido satinato con sottili ombre interne e feedback micro-zoom al passaggio del mouse
- Supporto contenitori: supporta la visualizzazione singola e la disposizione della spaziatura adattiva nei contenitori di elenchi

## Utilizza la demo
```html
<main class="KvLi">
  <a href="#">
    <b>Version</b>
    <b>1.0.0</b>
  </a>
  <a href="#">
    <b>License</b>
    <b>MIT</b>
  </a>
</main>

<a class="Kv" href="#">
  <b>Status</b>
  <b>Active</b>
</a>
```
