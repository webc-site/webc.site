# sleutelwaardepaar

'n Kapsulekentekenstyl wat in dubbele teksblokke gerangskik is, wat dikwels gebruik word om status, weergawe of sleutelwaarde-inligting te vertoon.
- Dubbelkolomvertoning: ondersteun twee teksblokke wat links en regs gerangskik is, en kan kenmerkname en vinnige inhoud onderskei
- Vloeibare glas: Volg 'n ryp deurskynende ontwerp met subtiele binneskaduwees en mikrozoom-terugvoer wanneer jy beweeg
- Houerondersteuning: ondersteun enkelvertoning en aanpasbare spasiëringsrangskikking in lyshouers

## Gebruik die demo
```html
<main class="KvLi">
  <a href="#">
    <b>Version</b>
    <b>1.0.0</b>
  </a>
  <a href="#">
    <b>License</b>
    <b>MIT</b>
  </a>
</main>

<a class="Kv" href="#">
  <b>Status</b>
  <b>Active</b>
</a>
```
