# par ključeva vrijednosti

Stil značke kapsule raspoređen u dvostruke tekstualne blokove, koji se često koriste za prikaz informacija o statusu, verziji ili ključnoj vrijednosti.
- Prikaz u dva stupca: podržava dva tekstualna bloka raspoređena lijevo i desno i može razlikovati nazive atributa i sadržaj upita
- Tekuće staklo: prati matirani proziran dizajn sa suptilnim unutarnjim sjenama i povratnom informacijom mikro zumiranja pri lebdenju
- Podrška za spremnike: podržava jedan prikaz i prilagodljiv raspored razmaka u spremnicima popisa

## Koristite demo
```html
<main class="KvLi">
  <a href="#">
    <b>Version</b>
    <b>1.0.0</b>
  </a>
  <a href="#">
    <b>License</b>
    <b>MIT</b>
  </a>
</main>

<a class="Kv" href="#">
  <b>Status</b>
  <b>Active</b>
</a>
```
