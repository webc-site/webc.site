# пар кључна вредност

Стил капсуле распоређен у двоструке текстуалне блокове, који се често користи за приказ информација о статусу, верзији или вредности кључа.
- Приказ у две колоне: подржава два текстуална блока распоређена лево и десно, и може да разликује имена атрибута и садржај упита
- Течно стакло: Прати мат прозирни дизајн са суптилним унутрашњим сенкама и повратним информацијама о микро зумирању при лебдењу
- Подршка за контејнере: подржава један екран и прилагодљиви распоред размака у контејнерима листе

## Користите демо
```html
<main class="KvLi">
  <a href="#">
    <b>Version</b>
    <b>1.0.0</b>
  </a>
  <a href="#">
    <b>License</b>
    <b>MIT</b>
  </a>
</main>

<a class="Kv" href="#">
  <b>Status</b>
  <b>Active</b>
</a>
```
