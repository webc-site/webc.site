# pereche cheie valoare

Un stil de insignă capsulă aranjat în blocuri de text duble, adesea folosit pentru a afișa informații despre stare, versiune sau valoare cheie.
- Afișaj cu două coloane: acceptă două blocuri de text aranjate la stânga și la dreapta și poate distinge numele atributelor și conținutul prompt
- Sticla lichidă: Urmează un design translucid înghețat cu umbre interioare subtile și feedback cu microzoom la hover
- Suport container: acceptă afișaj unic și aranjament adaptiv de spațiere în containerele de listă

## Utilizați demonstrația
```html
<main class="KvLi">
  <a href="#">
    <b>Version</b>
    <b>1.0.0</b>
  </a>
  <a href="#">
    <b>License</b>
    <b>MIT</b>
  </a>
</main>

<a class="Kv" href="#">
  <b>Status</b>
  <b>Active</b>
</a>
```
