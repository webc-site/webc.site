# কী মান যোৰ

এটা কেপচুল বেজ শৈলী দুটা লিখনী ব্লকত সজোৱা, প্ৰায়ে অৱস্থা, সংস্কৰণ বা চাবি মূল্য তথ্য প্ৰদৰ্শন কৰিবলে ব্যৱহাৰ কৰা হয়।
- ডাবল-স্তম্ভ প্ৰদৰ্শন: বাওঁ আৰু সোঁফালে সজোৱা দুটা লিখনী ব্লক সমৰ্থন কৰে, আৰু বৈশিষ্ট্যৰ নাম আৰু প্ৰমপ্ট বিষয়বস্তু পৃথক কৰিব পাৰে
- লিকুইড গ্লাছ: হ'ভাৰত সূক্ষ্ম ভিতৰৰ ছাঁ আৰু মাইক্ৰ'-জুম প্ৰতিক্ৰিয়াৰ সৈতে এটা ফ্ৰষ্টেড অৰ্ধস্বচ্ছ ডিজাইন অনুসৰণ কৰে
- ধাৰক সমৰ্থন: তালিকা ধাৰকসমূহত একক প্ৰদৰ্শন আৰু অভিযোজিত ব্যৱধান ব্যৱস্থা সমৰ্থন কৰে
