# par clave-valor

Un estilo de insignia de cápsula organizado en bloques de texto dobles, que a menudo se usa para mostrar información de estado, versión o valor clave.
- Visualización de dos columnas: admite dos bloques de texto dispuestos a izquierda y derecha y puede distinguir nombres de atributos y contenido de mensajes
- Liquid Glass: sigue un diseño translúcido esmerilado con sutiles sombras interiores y respuesta de microzoom al pasar el mouse
- Compatibilidad con contenedores: admite visualización única y disposición de espaciado adaptable en contenedores de listas

## Usa la demostración
```html
<main class="KvLi">
  <a href="#">
    <b>Version</b>
    <b>1.0.0</b>
  </a>
  <a href="#">
    <b>License</b>
    <b>MIT</b>
  </a>
</main>

<a class="Kv" href="#">
  <b>Status</b>
  <b>Active</b>
</a>
```
