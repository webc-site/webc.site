# 键值对

双文本块排列的胶囊徽章样式，常用于展示状态、版本或键值信息。
- 双栏展示：支持两个文本块左右排列，可区分属性名与提示内容
- 液态玻璃：遵循磨砂半透明设计，具有细腻的内阴影与悬停微缩放反馈
- 容器支持：支持单体展示，也支持在列表容器中自适应间距排列

## 使用演示
```html
<main class="KvLi">
  <a href="#">
    <b>Version</b>
    <b>1.0.0</b>
  </a>
  <a href="#">
    <b>License</b>
    <b>MIT</b>
  </a>
</main>

<a class="Kv" href="#">
  <b>Status</b>
  <b>Active</b>
</a>
```
