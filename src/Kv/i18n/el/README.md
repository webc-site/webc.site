# ζεύγος βασικών τιμών

Ένα στυλ σήματος κάψουλας διατεταγμένο σε διπλά μπλοκ κειμένου, που χρησιμοποιείται συχνά για την εμφάνιση πληροφοριών κατάστασης, έκδοσης ή βασικής τιμής.
- Εμφάνιση διπλής στήλης: υποστηρίζει δύο μπλοκ κειμένου τακτοποιημένα αριστερά και δεξιά και μπορεί να διακρίνει ονόματα χαρακτηριστικών και περιεχόμενο προτροπής
- Υγρό γυαλί: Ακολουθεί ένα ημιδιαφανές σχέδιο με λεπτές εσωτερικές σκιές και ανάδραση μικροζουμ κατά την αιώρηση
- Υποστήριξη κοντέινερ: υποστηρίζει ενιαία οθόνη και προσαρμοστική διάταξη διαστήματος σε δοχεία λίστας
