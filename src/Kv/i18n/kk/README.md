# негізгі мән жұбы

Күй, нұсқа немесе негізгі мән туралы ақпаратты көрсету үшін жиі пайдаланылатын қос мәтіндік блоктарда реттелген капсула белгішесі стилі.
- Қос бағанды ​​дисплей: сол және оң жақта реттелген екі мәтіндік блокты қолдайды және атрибут атаулары мен шақыру мазмұнын ажырата алады
- Сұйық әйнек: меңзерді жылжытқанда нәзік ішкі көлеңкелері және микро масштабты кері байланысы бар аязды мөлдір дизайнды ұстанады.
- Контейнерді қолдау: тізім контейнерлерінде бір дисплейді және адаптивті аралық орналасуын қолдайды
