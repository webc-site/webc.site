# முக்கிய மதிப்பு ஜோடி

ஒரு கேப்சூல் பேட்ஜ் பாணி இரட்டை உரைத் தொகுதிகளில் அமைக்கப்பட்டுள்ளது, இது பெரும்பாலும் நிலை, பதிப்பு அல்லது முக்கிய மதிப்புத் தகவலைக் காட்டப் பயன்படுகிறது.
- இரட்டை நெடுவரிசை காட்சி: இடது மற்றும் வலது வரிசைப்படுத்தப்பட்ட இரண்டு உரைத் தொகுதிகளை ஆதரிக்கிறது, மேலும் பண்புக்கூறு பெயர்கள் மற்றும் உடனடி உள்ளடக்கத்தை வேறுபடுத்தி அறியலாம்
- திரவ கண்ணாடி: மெல்லிய உள் நிழல்கள் மற்றும் மிதவையில் மைக்ரோ-ஜூம் பின்னூட்டத்துடன் உறைந்த ஒளிஊடுருவக்கூடிய வடிவமைப்பைப் பின்பற்றுகிறது
- கொள்கலன் ஆதரவு: பட்டியல் கண்டெய்னர்களில் ஒற்றை காட்சி மற்றும் அடாப்டிவ் ஸ்பேசிங் ஏற்பாட்டை ஆதரிக்கிறது
