# Schlüssel-Wert-Paar

Ein Kapsel-Badge-Stil, der in doppelten Textblöcken angeordnet ist und häufig zur Anzeige von Status-, Versions- oder Schlüsselwertinformationen verwendet wird.
- Doppelspaltige Anzeige: Unterstützt zwei links und rechts angeordnete Textblöcke und kann Attributnamen und Eingabeaufforderungsinhalte unterscheiden
- Liquid Glass: Folgt einem mattierten, durchscheinenden Design mit subtilen Innenschatten und Mikrozoom-Feedback beim Schweben
- Containerunterstützung: Unterstützt Einzelanzeige und adaptive Abstandsanordnung in Listencontainern

## Nutzen Sie die Demo
```html
<main class="KvLi">
  <a href="#">
    <b>Version</b>
    <b>1.0.0</b>
  </a>
  <a href="#">
    <b>License</b>
    <b>MIT</b>
  </a>
</main>

<a class="Kv" href="#">
  <b>Status</b>
  <b>Active</b>
</a>
```
