# par valur ewlieni

Stil tal-badge tal-kapsula rranġat fi blokki ta 'test doppju, spiss użat biex juri informazzjoni dwar l-istatus, il-verżjoni jew il-valur ewlieni.
- Display b'kolonna doppja: jappoġġja żewġ blokki ta 'test irranġati fuq ix-xellug u fuq il-lemin, u jista' jiddistingwi l-ismijiet tal-attributi u l-kontenut fil-pront
- Ħġieġ Likwidu: Isegwi disinn trasluċidu bil-ġlata b'dellijiet ta 'ġewwa sottili u feedback ta' mikro-zoom fuq id-depożitu
- Appoġġ tal-kontenitur: jappoġġja wiri wieħed u arranġament ta 'spazjar adattiv f'kontenituri tal-lista
