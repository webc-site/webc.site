# pasangan nilai kunci

Gaya lencana kapsul sing disusun ing blok teks dobel, asring digunakake kanggo nampilake informasi status, versi utawa nilai kunci.
- Tampilan kolom kaping pindho: ndhukung rong blok teks sing disusun kiwa lan tengen, lan bisa mbedakake jeneng atribut lan konten cepet.
- Kaca Cairan: Nderek desain tembus es kanthi bayangan batin sing halus lan umpan balik zoom mikro ing hover
- Dhukungan kontainer: ndhukung tampilan tunggal lan pengaturan jarak adaptif ing wadhah dhaptar

## Gunakake demo
```html
<main class="KvLi">
  <a href="#">
    <b>Version</b>
    <b>1.0.0</b>
  </a>
  <a href="#">
    <b>License</b>
    <b>MIT</b>
  </a>
</main>

<a class="Kv" href="#">
  <b>Status</b>
  <b>Active</b>
</a>
```
