# пара ключ-значення

Стиль значка капсули, упорядкованого подвійними текстовими блоками, які часто використовуються для відображення інформації про статус, версію чи значення ключа.
- Дисплей у дві колонки: підтримує два текстові блоки, розташовані ліворуч і праворуч, і може розрізняти назви атрибутів і вміст підказок
- Рідке скло: повторює матовий напівпрозорий дизайн із тонкими внутрішніми тінями та мікрозумом під час наведення
- Підтримка контейнерів: підтримує один дисплей і адаптивне розміщення інтервалів у контейнерах списку

## Використовуйте демо
```html
<main class="KvLi">
  <a href="#">
    <b>Version</b>
    <b>1.0.0</b>
  </a>
  <a href="#">
    <b>License</b>
    <b>MIT</b>
  </a>
</main>

<a class="Kv" href="#">
  <b>Status</b>
  <b>Active</b>
</a>
```
