# pasangan nilai kunci

Gaya lencana kapsul yang disusun dalam blok teks berganda, selalunya digunakan untuk memaparkan status, versi atau maklumat nilai utama.
- Paparan lajur dua: menyokong dua blok teks yang disusun kiri dan kanan, dan boleh membezakan nama atribut dan kandungan segera
- Kaca Cecair: Mengikut reka bentuk lut sinar beku dengan bayang-bayang dalaman yang halus dan maklum balas zum mikro pada tuding
- Sokongan bekas: menyokong paparan tunggal dan susunan jarak suai dalam bekas senarai

## Gunakan demo
```html
<main class="KvLi">
  <a href="#">
    <b>Version</b>
    <b>1.0.0</b>
  </a>
  <a href="#">
    <b>License</b>
    <b>MIT</b>
  </a>
</main>

<a class="Kv" href="#">
  <b>Status</b>
  <b>Active</b>
</a>
```
