# par ključnih vrijednosti

Stil kapsule raspoređen u dvostruke tekstualne blokove, koji se često koristi za prikaz informacija o statusu, verziji ili vrijednosti ključa.
- Prikaz u dva stupca: podržava dva tekstualna bloka raspoređena lijevo i desno i može razlikovati imena atributa i sadržaj upita
- Tečno staklo: Prati mat prozirni dizajn sa suptilnim unutrašnjim senkama i povratnim informacijama o mikro zumiranju pri lebdenju
- Podrška za kontejnere: podržava jedan prikaz i prilagodljivi raspored razmaka u kontejnerima liste

## Koristite demo
```html
<main class="KvLi">
  <a href="#">
    <b>Version</b>
    <b>1.0.0</b>
  </a>
  <a href="#">
    <b>License</b>
    <b>MIT</b>
  </a>
</main>

<a class="Kv" href="#">
  <b>Status</b>
  <b>Active</b>
</a>
```
