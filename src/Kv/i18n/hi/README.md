# कुंजी मान युग्म

दोहरे टेक्स्ट ब्लॉकों में व्यवस्थित एक कैप्सूल बैज शैली, जिसका उपयोग अक्सर स्थिति, संस्करण या कुंजी मूल्य जानकारी प्रदर्शित करने के लिए किया जाता है।
- डबल-कॉलम डिस्प्ले: बाएं और दाएं व्यवस्थित दो टेक्स्ट ब्लॉक का समर्थन करता है, और विशेषता नाम और शीघ्र सामग्री को अलग कर सकता है
- लिक्विड ग्लास: सूक्ष्म आंतरिक छाया और होवर पर माइक्रो-ज़ूम फीडबैक के साथ एक फ्रॉस्टेड पारभासी डिज़ाइन का अनुसरण करता है
- कंटेनर समर्थन: सूची कंटेनरों में एकल डिस्प्ले और अनुकूली रिक्ति व्यवस्था का समर्थन करता है
