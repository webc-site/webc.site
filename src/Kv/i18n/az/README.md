# əsas dəyər cütü

Çox vaxt status, versiya və ya əsas dəyər məlumatlarını göstərmək üçün istifadə olunan ikiqat mətn bloklarında təşkil edilmiş kapsul nişanı üslubu.
- İki sütunlu ekran: sola və sağa düzülmüş iki mətn blokunu dəstəkləyir və atribut adlarını və məlumat məzmununu ayırd edə bilir
- Maye Şüşə: İncə daxili kölgələr və hover zamanı mikro-zoom rəyi ilə şaxtalı şəffaf dizaynı izləyir.
- Konteyner dəstəyi: siyahı konteynerlərində tək displey və adaptiv aralıq təşkilini dəstəkləyir

## Demodan istifadə edin
```html
<main class="KvLi">
  <a href="#">
    <b>Version</b>
    <b>1.0.0</b>
  </a>
  <a href="#">
    <b>License</b>
    <b>MIT</b>
  </a>
</main>

<a class="Kv" href="#">
  <b>Status</b>
  <b>Active</b>
</a>
```
