# avainarvopari

Kaksoistekstilohkoihin järjestetty kapselimerkkityyli, jota käytetään usein näyttämään tila-, versio- tai avainarvotietoja.
- Kaksisarakkeinen näyttö: tukee kahta tekstilohkoa, jotka on järjestetty vasemmalle ja oikealle, ja pystyy erottamaan määritteiden nimet ja kehotteen sisällön
- Liquid Glass: Seuraa himmeästi läpikuultavaa muotoilua, jossa on hienovaraisia ​​sisävarjoja ja mikrozoom-palaute leijuttaessa
- Säiliötuki: tukee yhtä näyttöä ja mukautuvaa välitysjärjestelyä luettelosäiliöissä

## Käytä demoa
```html
<main class="KvLi">
  <a href="#">
    <b>Version</b>
    <b>1.0.0</b>
  </a>
  <a href="#">
    <b>License</b>
    <b>MIT</b>
  </a>
</main>

<a class="Kv" href="#">
  <b>Status</b>
  <b>Active</b>
</a>
```
