# کلیدی قدر جوڑی

ایک کیپسول بیج اسٹائل جو ڈبل ٹیکسٹ بلاکس میں ترتیب دیا گیا ہے، جو اکثر اسٹیٹس، ورژن یا کلیدی قدر کی معلومات کو ظاہر کرنے کے لیے استعمال ہوتا ہے۔
- ڈبل کالم ڈسپلے: بائیں اور دائیں ترتیب والے دو ٹیکسٹ بلاکس کو سپورٹ کرتا ہے، اور انتساب کے ناموں اور فوری مواد میں فرق کر سکتا ہے۔
- مائع شیشہ: ہوور پر ٹھیک ٹھیک اندرونی سائے اور مائیکرو زوم فیڈ بیک کے ساتھ فراسٹڈ پارباسی ڈیزائن کی پیروی کرتا ہے۔
- کنٹینر سپورٹ: فہرست کنٹینرز میں سنگل ڈسپلے اور انکولی وقفہ کاری کے انتظام کی حمایت کرتا ہے۔
