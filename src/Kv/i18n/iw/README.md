# צמד ערכי מפתח

סגנון תג קפסולה מסודר בקוביות טקסט כפולות, המשמש לעתים קרובות להצגת מידע על מצב, גרסה או ערך מפתח.
- תצוגת עמודה כפולה: תומכת בשני בלוקי טקסט המסודרים משמאל וימין, ויכולה להבחין בין שמות מאפיינים ותוכן בקשות
- זכוכית נוזלית: עוקב אחר עיצוב שקוף חלבי עם צללים פנימיים עדינים ומשוב מיקרו זום בעת ריחוף
- תמיכת מיכל: תומך בתצוגה יחידה ובסידור מרווחים אדפטיבי במכולות רשימה
