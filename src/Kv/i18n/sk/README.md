# pár kľúč-hodnota

Štýl odznaku kapsuly usporiadaný do dvojitých textových blokov, ktorý sa často používa na zobrazenie informácií o stave, verzii alebo kľúčovej hodnote.
- Dvojstĺpcové zobrazenie: podporuje dva textové bloky usporiadané vľavo a vpravo a dokáže rozlíšiť názvy atribútov a obsah výzvy
- Tekuté sklo: Vyznačuje sa matným priesvitným dizajnom s jemnými vnútornými tieňmi a spätnou väzbou mikrozoomu pri umiestnení kurzora myši.
- Podpora kontajnerov: podporuje jedno zobrazenie a adaptívne usporiadanie medzier v zoznamových kontajneroch

## Použite demo
```html
<main class="KvLi">
  <a href="#">
    <b>Version</b>
    <b>1.0.0</b>
  </a>
  <a href="#">
    <b>License</b>
    <b>MIT</b>
  </a>
</main>

<a class="Kv" href="#">
  <b>Status</b>
  <b>Active</b>
</a>
```
