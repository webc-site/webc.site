# sleutelwaardepaar

Een capsule-badgestijl gerangschikt in dubbele tekstblokken, vaak gebruikt om status-, versie- of sleutelwaarde-informatie weer te geven.
- Weergave met dubbele kolommen: ondersteunt twee tekstblokken die links en rechts zijn gerangschikt, en kan attribuutnamen en promptinhoud onderscheiden
- Liquid Glass: volgt een mat, doorschijnend ontwerp met subtiele innerlijke schaduwen en microzoomfeedback bij het zweven
- Containerondersteuning: ondersteunt enkele weergave en adaptieve spatiëring in lijstcontainers

## Gebruik de demo
```html
<main class="KvLi">
  <a href="#">
    <b>Version</b>
    <b>1.0.0</b>
  </a>
  <a href="#">
    <b>License</b>
    <b>MIT</b>
  </a>
</main>

<a class="Kv" href="#">
  <b>Status</b>
  <b>Active</b>
</a>
```
