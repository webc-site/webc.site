# કી મૂલ્યની જોડી

ડબલ ટેક્સ્ટ બ્લોક્સમાં ગોઠવાયેલ કેપ્સ્યુલ બેજ શૈલી, જે ઘણીવાર સ્થિતિ, સંસ્કરણ અથવા મુખ્ય મૂલ્યની માહિતી પ્રદર્શિત કરવા માટે વપરાય છે.
- ડબલ-કૉલમ ડિસ્પ્લે: ડાબે અને જમણે ગોઠવાયેલા બે ટેક્સ્ટ બ્લોક્સને સપોર્ટ કરે છે, અને એટ્રિબ્યુટ નામો અને પ્રોમ્પ્ટ સામગ્રીને અલગ કરી શકે છે
- લિક્વિડ ગ્લાસ: હોવર પર સૂક્ષ્મ આંતરિક પડછાયાઓ અને માઇક્રો-ઝૂમ પ્રતિસાદ સાથે હિમાચ્છાદિત અર્ધપારદર્શક ડિઝાઇનને અનુસરે છે
- કન્ટેનર સપોર્ટ: સૂચિ કન્ટેનરમાં સિંગલ ડિસ્પ્લે અને અનુકૂલનશીલ અંતર ગોઠવણને સપોર્ટ કરે છે
