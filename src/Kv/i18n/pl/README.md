# para klucz-wartość

Styl plakietki kapsuły ułożonej w podwójne bloki tekstu, często używany do wyświetlania informacji o statusie, wersji lub wartości klucza.
- Wyświetlanie dwukolumnowe: obsługuje dwa bloki tekstu ułożone po lewej i prawej stronie oraz umożliwia rozróżnienie nazw atrybutów i treści podpowiedzi
- Płynne szkło: ma matową, półprzezroczystą konstrukcję z subtelnymi cieniami wewnętrznymi i informacją o mikrozoomie po najechaniu myszką
- Obsługa kontenerów: obsługuje pojedynczy wyświetlacz i adaptacyjne rozmieszczenie odstępów w kontenerach list

## Skorzystaj z wersji demonstracyjnej
```html
<main class="KvLi">
  <a href="#">
    <b>Version</b>
    <b>1.0.0</b>
  </a>
  <a href="#">
    <b>License</b>
    <b>MIT</b>
  </a>
</main>

<a class="Kv" href="#">
  <b>Status</b>
  <b>Active</b>
</a>
```
