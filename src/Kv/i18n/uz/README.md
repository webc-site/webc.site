# kalit qiymat juftligi

Ikki matnli bloklarda joylashgan kapsula nishoni uslubi, odatda holat, versiya yoki asosiy qiymat ma'lumotlarini ko'rsatish uchun ishlatiladi.
- Ikki ustunli displey: chap va o'ngga joylashtirilgan ikkita matn blokini qo'llab-quvvatlaydi va atribut nomlari va so'rov mazmunini ajrata oladi
- Suyuq shisha: sichqonchani ko'targanda nozik ichki soyalar va mikro masshtabli fikr-mulohazalarga ega muzli shaffof dizaynga amal qiladi.
- Konteynerni qo'llab-quvvatlash: ro'yxat konteynerlarida bitta displey va moslashtirilgan intervallarni tartibga solishni qo'llab-quvvatlaydi
