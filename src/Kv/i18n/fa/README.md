# جفت ارزش کلیدی

سبک نشان کپسولی که در بلوک‌های متنی دوتایی مرتب شده است، که اغلب برای نمایش وضعیت، نسخه یا اطلاعات ارزش کلیدی استفاده می‌شود.
- نمایش دو ستونی: از دو بلوک متنی که به چپ و راست چیده شده اند پشتیبانی می کند و می تواند نام ویژگی ها و محتوای درخواستی را تشخیص دهد.
- شیشه مایع: از طراحی شفاف مات با سایه‌های داخلی ظریف و بازخورد میکرو زوم هنگام شناور پیروی می‌کند.
- پشتیبانی از کانتینر: از صفحه نمایش منفرد و ترتیب فاصله تطبیقی ​​در ظروف لیست پشتیبانی می کند
