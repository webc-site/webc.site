# parell de valors clau

Un estil d'insígnia de càpsula disposat en blocs de text dobles, que sovint s'utilitza per mostrar informació sobre l'estat, la versió o el valor clau.
- Pantalla de doble columna: admet dos blocs de text disposats a l'esquerra i a la dreta, i pot distingir els noms dels atributs i el contingut del missatge
- Vidre líquid: segueix un disseny translúcid esmerilat amb ombres interiors subtils i retroalimentació de micro-zoom en passar el ratolí.
- Suport de contenidors: admet una pantalla única i una disposició adaptativa d'espaiat en contenidors de llista

## Utilitzeu la demostració
```html
<main class="KvLi">
  <a href="#">
    <b>Version</b>
    <b>1.0.0</b>
  </a>
  <a href="#">
    <b>License</b>
    <b>MIT</b>
  </a>
</main>

<a class="Kv" href="#">
  <b>Status</b>
  <b>Active</b>
</a>
```
