# par de valores-chave

Um estilo de emblema de cápsula organizado em blocos de texto duplos, geralmente usado para exibir informações de status, versão ou valor-chave.
- Exibição de coluna dupla: suporta dois blocos de texto organizados à esquerda e à direita e pode distinguir nomes de atributos e conteúdo de prompt
- Vidro líquido: segue um design translúcido fosco com sombras internas sutis e feedback de micro zoom ao passar o mouse
- Suporte a contêineres: suporta exibição única e arranjo de espaçamento adaptável em contêineres de lista

## Use a demonstração
```html
<main class="KvLi">
  <a href="#">
    <b>Version</b>
    <b>1.0.0</b>
  </a>
  <a href="#">
    <b>License</b>
    <b>MIT</b>
  </a>
</main>

<a class="Kv" href="#">
  <b>Status</b>
  <b>Active</b>
</a>
```
