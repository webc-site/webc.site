# atslēgu vērtību pāris

Kapsulas emblēmas stils, kas sakārtots divos teksta blokos, ko bieži izmanto statusa, versijas vai atslēgas vērtības informācijas attēlošanai.
- Divu kolonnu displejs: atbalsta divus teksta blokus, kas sakārtoti pa kreisi un pa labi, un var atšķirt atribūtu nosaukumus un uzvednes saturu
- Šķidrais stikls: atbilst matētam caurspīdīgam dizainam ar smalkām iekšējām ēnām un mikro tālummaiņas atgriezenisko saiti, virzot kursoru.
- Konteineru atbalsts: atbalsta vienu displeju un adaptīvu atstarpju izvietojumu sarakstu konteineros

## Izmantojiet demonstrāciju
```html
<main class="KvLi">
  <a href="#">
    <b>Version</b>
    <b>1.0.0</b>
  </a>
  <a href="#">
    <b>License</b>
    <b>MIT</b>
  </a>
</main>

<a class="Kv" href="#">
  <b>Status</b>
  <b>Active</b>
</a>
```
