# គូតម្លៃសំខាន់

រចនាប័ទ្មផ្លាកសញ្ញាកន្សោមត្រូវបានរៀបចំជាប្លុកអត្ថបទពីរដង ដែលជារឿយៗប្រើដើម្បីបង្ហាញស្ថានភាព កំណែ ឬព័ត៌មានតម្លៃគន្លឹះ។
- ការ​បង្ហាញ​ជួរ​ឈរ​ពីរ៖ គាំទ្រ​ប្លុក​អត្ថបទ​ពីរ​ដែល​បាន​រៀប​ចំ​ឆ្វេង និង​ស្ដាំ ហើយ​អាច​បែងចែក​ឈ្មោះ​គុណលក្ខណៈ និង​មាតិកា​ប្រអប់​បញ្ចូល
- កញ្ចក់រាវ៖ ធ្វើតាមការរចនាបែបរលោងដែលមានភាពត្រជាក់ជាមួយនឹងស្រមោលខាងក្នុងដ៏ស្រទន់ និងមតិកែលម្អមីក្រូពង្រីកនៅពេលដាក់
- ការគាំទ្រកុងតឺន័រ៖ គាំទ្រការបង្ហាញតែមួយ និងការរៀបចំគម្លាតដែលអាចសម្របបាននៅក្នុងធុងបញ្ជី
