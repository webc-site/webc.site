# คู่ค่าคีย์

รูปแบบป้ายแคปซูลที่จัดเรียงเป็นบล็อกข้อความคู่ มักใช้เพื่อแสดงข้อมูลสถานะ เวอร์ชัน หรือค่าคีย์
- การแสดงคอลัมน์คู่: รองรับบล็อกข้อความสองบล็อกที่จัดเรียงทางซ้ายและขวา และสามารถแยกแยะชื่อแอตทริบิวต์และเนื้อหาพร้อมท์ได้
- กระจกเหลว: เป็นไปตามการออกแบบโปร่งแสงแบบฝ้าพร้อมเงาภายในอันละเอียดอ่อนและการตอบรับการซูมแบบไมโครเมื่อวางเมาส์เหนือ
- การสนับสนุนคอนเทนเนอร์: รองรับจอแสดงผลเดี่ยวและการจัดเรียงระยะห่างแบบปรับได้ในคอนเทนเนอร์รายการ
