# କି ମୂଲ୍ୟ ଯୁଗଳ |

ଡବଲ୍ ଟେକ୍ସଟ୍ ବ୍ଲକ୍ରେ ଏକ କ୍ୟାପସୁଲ୍ ବ୍ୟାଜ୍ ଶ style ଳୀ, ସ୍ଥିତି, ସଂସ୍କରଣ କିମ୍ବା କି ମୂଲ୍ୟ ସୂଚନା ପ୍ରଦର୍ଶନ ପାଇଁ ବ୍ୟବହୃତ ହୁଏ |
- ଡବଲ୍-ସ୍ତମ୍ଭ ପ୍ରଦର୍ଶନ: ବାମ ଏବଂ ଡାହାଣରେ ବ୍ୟବହୃତ ଦୁଇଟି ପାଠ୍ୟ ବ୍ଲକକୁ ସମର୍ଥନ କରେ, ଏବଂ ଗୁଣଧର୍ମ ନାମ ଏବଂ ପ୍ରମ୍ପ୍ଟ ବିଷୟବସ୍ତୁକୁ ପୃଥକ କରିପାରିବ |
- ତରଳ ଗ୍ଲାସ୍: ସୂକ୍ଷ୍ମ ଆଭ୍ୟନ୍ତରୀଣ ଛାୟା ଏବଂ ହୋଭର ଉପରେ ମାଇକ୍ରୋ-ଜୁମ୍ ଫିଡବ୍ୟାକ୍ ସହିତ ଏକ ତରକାରୀ ସ୍ୱଚ୍ଛ ଡିଜାଇନ୍ ଅନୁସରଣ କରେ |
- ଧାରଣକାରୀ ସମର୍ଥନ: ତାଲିକା ପାତ୍ରରେ ଏକକ ପ୍ରଦର୍ଶନ ଏବଂ ଆଡାପ୍ଟିଭ୍ ବ୍ୟବଧାନ ବ୍ୟବସ୍ଥାକୁ ସମର୍ଥନ କରେ |
