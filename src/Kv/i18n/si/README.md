# ප්රධාන අගය යුගලය

කැප්සියුල ලාංඡන විලාසය ද්විත්ව පෙළ බ්ලොක් වලින් සකස් කර ඇත, බොහෝ විට තත්වය, අනුවාදය හෝ ප්‍රධාන අගය තොරතුරු ප්‍රදර්ශනය කිරීමට භාවිතා කරයි.
- ද්විත්ව තීරු සංදර්ශකය: වම සහ දකුණට සකසන ලද පෙළ කොටස් දෙකකට සහය දක්වයි, සහ ගුණාංග නම් සහ ක්ෂණික අන්තර්ගතයන් වෙන්කර හඳුනාගත හැකිය
- ද්‍රව වීදුරු: සියුම් අභ්‍යන්තර සෙවනැලි සහ හොවර් මත ක්ෂුද්‍ර විශාලන ප්‍රතිපෝෂණ සහිත තුහීන පාරභාසක මෝස්තරයක් අනුගමනය කරයි
- බහාලුම් සහාය: ලැයිස්තු බහාලුම්වල තනි සංදර්ශකය සහ අනුවර්තන පරතරය සැකසීම සඳහා සහය දක්වයි
