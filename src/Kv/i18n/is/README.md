# lykilgildi par

Hylkismerkisstíll raðað í tvöföldum textablokkum, oft notaður til að sýna upplýsingar um stöðu, útgáfu eða lykilgildi.
- Tvöfaldur dálkaskjár: styður tvo textabubba raðað til vinstri og hægri, og getur greint eigindaheiti og hvetjandi efni
- Fljótandi gler: Fylgir mattri hálfgagnsærri hönnun með fíngerðum innri skuggum og endurgjöf með öraðdrætti við sveima
- Gámastuðningur: styður stakan skjá og aðlögunarbil í listagámum

## Notaðu kynninguna
```html
<main class="KvLi">
  <a href="#">
    <b>Version</b>
    <b>1.0.0</b>
  </a>
  <a href="#">
    <b>License</b>
    <b>MIT</b>
  </a>
</main>

<a class="Kv" href="#">
  <b>Status</b>
  <b>Active</b>
</a>
```
