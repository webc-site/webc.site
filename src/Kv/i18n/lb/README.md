# Schlëssel Wäert Pair

E Kapsel Badge Stil arrangéiert an duebel Textblocken, dacks benotzt fir Status, Versioun oder Schlësselwäertinformatioun ze weisen.
- Duebelkolonne Display: ënnerstëtzt zwee Textblocken, déi lénks a riets arrangéiert sinn, a kënnen Attributnumm a prompt Inhalt z'ënnerscheeden
- Liquid Glass: Follegt e frostéiert transluzenten Design mat subtile bannenzege Schatten a Mikro-Zoom Feedback beim Hover
- Container Ënnerstëtzung: Ënnerstëtzt eenzeg Display an adaptiven Abstandsarrangement a Lëschtbehälter
