# двойка ключ стойност

Капсулен стил на значка, подреден в двойни текстови блокове, често използвани за показване на информация за състояние, версия или ключова стойност.
- Дисплей с две колони: поддържа два текстови блока, подредени отляво и отдясно, и може да разграничава имената на атрибутите и съдържанието на подканите
- Течно стъкло: Следва матиран полупрозрачен дизайн с фини вътрешни сенки и обратна връзка с микроувеличение при задържане
- Поддръжка на контейнери: поддържа единичен дисплей и адаптивно подреждане на разстоянието в контейнерите със списъци

## Използвайте демонстрацията
```html
<main class="KvLi">
  <a href="#">
    <b>Version</b>
    <b>1.0.0</b>
  </a>
  <a href="#">
    <b>License</b>
    <b>MIT</b>
  </a>
</main>

<a class="Kv" href="#">
  <b>Status</b>
  <b>Active</b>
</a>
```
