# пар клучна вредност

Стил на значка на капсула распореден во двојни текстуални блокови, често се користи за прикажување на информации за статус, верзија или клучна вредност.
- Приказ со двојна колона: поддржува два текстуални блока наредени лево и десно, и може да разликува имиња на атрибути и содржина на барање
- Течно стакло: Следи замрзнат проѕирен дизајн со суптилни внатрешни сенки и повратни информации за микро-зум при лебдење
- Поддршка за контејнер: поддржува единечен приказ и распоред на адаптивни растојание во контејнерите со список

## Користете го демо
```html
<main class="KvLi">
  <a href="#">
    <b>Version</b>
    <b>1.0.0</b>
  </a>
  <a href="#">
    <b>License</b>
    <b>MIT</b>
  </a>
</main>

<a class="Kv" href="#">
  <b>Status</b>
  <b>Active</b>
</a>
```
