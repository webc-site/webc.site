# par valor clave

Un estilo de distintivo de cápsula disposto en bloques de texto dobres, que adoita usar para mostrar información de estado, versión ou valor clave.
- Visualización de dobre columna: admite dous bloques de texto dispostos á esquerda e á dereita, e pode distinguir os nomes de atributos e o contido do aviso
- Vidro líquido: segue un deseño translúcido xeado con sombras interiores sutís e retroalimentación de micro-zoom ao pasar o cursor.
- Soporte de contedores: admite unha visualización única e unha disposición de espazo adaptable nos contedores de lista

## Use a demostración
```html
<main class="KvLi">
  <a href="#">
    <b>Version</b>
    <b>1.0.0</b>
  </a>
  <a href="#">
    <b>License</b>
    <b>MIT</b>
  </a>
</main>

<a class="Kv" href="#">
  <b>Status</b>
  <b>Active</b>
</a>
```
