# සංචාලන මෙනුව විවිධ තිර ප්‍රමාණයන්ට අනුවර්තනය කර ඇත

- ඩෙස්ක්ටොප්: ලින්ක් තිරස් ටයිල් වලින් පෙන්වනු ලැබේ
- ජංගම අනුවාදය: මෙනු බොත්තමට හකුළන්න, පැති තීරු ලාච්චුවෙන් ලිස්සා යාමට ක්ලික් කරන්න
- මාර්ග සක්‍රිය කිරීම: වත්මන් මාර්ගය සහ හැෂ් අගය ස්වයංක්‍රීයව සංසන්දනය කරන්න, සහ ගැළපෙන සබැඳියට සක්‍රිය තත්ව පන්තියේ නම එක් කරන්න `A`
- ස්වයංක්‍රීයව වසන්න: ජංගම පර්යන්තය සබැඳියක් හෝ පැති තීරුවේ බාහිර පසුබිම මත ක්ලික් කළ විට, පැති තීරුව ස්වයංක්‍රීයව වැසෙයි.

## නිදර්ශනය කරන්න

සංරචකය ආරම්භ කළ විට, ළමා මූලද්‍රව්‍යයේ `c-side` නොමැති නම්, ජංගම මෙනු බොත්තම සහ `c-side` පැති තීරුව ස්වයංක්‍රීයව සාදනු ලබන අතර, සියලු `a` ටැග් පැති තීරුව වෙත ගෙන යනු ඇත.
