# さまざまな画面サイズに適応したナビゲーション メニュー

- デスクトップ: リンクは水平タイルで表示されます
- モバイル版: メニュー ボタンに折りたたんで、クリックしてサイドバーの引き出しをスライドさせます。
- ルートのアクティベーション: 現在のパスとハッシュ値を自動的に比較し、アクティベーション ステータス クラス名を一致するリンクに追加します`A`
- 自動的に閉じる: モバイル端末がリンクまたはサイドバーの外部背景をクリックすると、サイドバーは自動的に閉じます。

## 使用

```html
<c-nav-menu>
  <a href="/">フロントページ</a>
  <a href="/doc">書類</a>
  <a href="/about">について</a>
</c-nav-menu>
```

## 説明する

コンポーネントの初期化時に、子要素に `c-side` がない場合、モバイル メニュー ボタンと `c-side` サイドバーが自動的に作成され、すべての `a` タグがサイドバーに移動されます。

## スタイルクラス

- **A**: アクティブ化されたリンクにより、クラス名 `A` が自動的に追加されます。
