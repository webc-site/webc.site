# Меню навигации адаптировано под разные размеры экрана

- Рабочий стол: ссылки отображаются в виде горизонтальных плиток.
- Мобильная версия: кнопка «Свернуть в меню», нажмите, чтобы выдвинуть ящик боковой панели.
- Активация маршрута: автоматическое сравнение текущего пути и значения хеш-функции, а также добавление имени класса статуса активации в соответствующую ссылку `A`
- Автоматически закрывать: когда мобильный терминал нажимает на ссылку или внешний фон боковой панели, боковая панель автоматически закрывается.

## использовать

```html
<c-nav-menu>
  <a href="/">первая страница</a>
  <a href="/doc">документ</a>
  <a href="/about">о</a>
</c-nav-menu>
```

## иллюстрировать

При инициализации компонента, если в дочернем элементе нет `c-side`, кнопка мобильного меню и боковая панель `c-side` будут автоматически созданы, а все теги `a` будут перемещены на боковую панель.

## класс стиля

- **A**: активированная ссылка автоматически добавит имя класса `A`
