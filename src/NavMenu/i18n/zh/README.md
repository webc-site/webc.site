# 适配不同屏幕尺寸的导航菜单

- 桌面端：链接水平平铺展示
- 移动端：折叠为菜单按钮，点击滑出侧边栏抽屉
- 路由激活：自动比对当前路径与哈希值，为匹配的链接添加激活状态类名 `A`
- 自动收起：移动端点击链接或侧边栏外部背景时，侧边栏自动关闭

## 使用

```html
<c-nav-menu>
  <a href="/">首页</a>
  <a href="/doc">文档</a>
  <a href="/about">关于</a>
</c-nav-menu>
```

## 说明

组件在初始化时，若子元素中没有 `c-side`，会自动创建移动端菜单按钮与 `c-side` 侧边栏，并将所有的 `a` 标签移入侧边栏中。

## 样式类

- **A**：激活的链接会自动添加类名 `A`
