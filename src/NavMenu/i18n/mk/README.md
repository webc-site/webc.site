# Мени за навигација прилагодено на различни големини на екранот

- Десктоп: Врските се прикажани во хоризонтални плочки
- Мобилна верзија: Соберете во копче од менито, кликнете за да ја извлечете фиоката на страничната лента
- Активирање маршрута: автоматски споредете ја тековната патека и вредноста на хашот и додајте го името на класата за статус на активирање на врската што одговара `A`
- Автоматско затворање: кога мобилниот терминал ќе кликне на врска или на надворешната позадина на страничната лента, страничната лента автоматски се затвора.

## употреба

```html
<c-nav-menu>
  <a href="/">насловна страница</a>
  <a href="/doc">документ</a>
  <a href="/about">за</a>
</c-nav-menu>
```

## илустрираат

Кога компонентата е иницијализирана, ако нема `c-side` во елементот дете, копчето за мобилни мени и `c-side` страничната лента автоматски ќе се креираат, а сите ознаки `a` ќе се преместат во страничната лента.

## класа стил

- **А**: Активираната врска автоматски ќе го додаде името на класата `A`
