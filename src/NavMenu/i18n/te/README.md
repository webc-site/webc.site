# నావిగేషన్ మెను వివిధ స్క్రీన్ పరిమాణాలకు అనుగుణంగా మార్చబడింది

- డెస్క్‌టాప్: లింక్‌లు క్షితిజ సమాంతర టైల్స్‌లో ప్రదర్శించబడతాయి
- మొబైల్ వెర్షన్: మెను బటన్‌లోకి కుదించండి, సైడ్‌బార్ డ్రాయర్ నుండి స్లయిడ్ చేయడానికి క్లిక్ చేయండి
- రూట్ యాక్టివేషన్: ప్రస్తుత మార్గం మరియు హాష్ విలువను స్వయంచాలకంగా సరిపోల్చండి మరియు సరిపోలే లింక్ `A`కి యాక్టివేషన్ స్థితి తరగతి పేరును జోడించండి
- స్వయంచాలకంగా మూసివేయండి: మొబైల్ టెర్మినల్ లింక్ లేదా సైడ్‌బార్ బాహ్య నేపథ్యంపై క్లిక్ చేసినప్పుడు, సైడ్‌బార్ స్వయంచాలకంగా మూసివేయబడుతుంది.

## ఉపయోగించండి

```html
<c-nav-menu>
  <a href="/">మొదటి పేజీ</a>
  <a href="/doc">పత్రం</a>
  <a href="/about">గురించి</a>
</c-nav-menu>
```

## వర్ణించండి

భాగం ప్రారంభించబడినప్పుడు, చైల్డ్ ఎలిమెంట్‌లో `c-side` లేకపోతే, మొబైల్ మెను బటన్ మరియు `c-side` సైడ్‌బార్ స్వయంచాలకంగా సృష్టించబడతాయి మరియు అన్ని `a` ట్యాగ్‌లు సైడ్‌బార్‌లోకి తరలించబడతాయి.
