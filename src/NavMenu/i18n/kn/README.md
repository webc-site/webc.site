# ನ್ಯಾವಿಗೇಶನ್ ಮೆನು ವಿವಿಧ ಪರದೆಯ ಗಾತ್ರಗಳಿಗೆ ಅಳವಡಿಸಲಾಗಿದೆ

- ಡೆಸ್ಕ್‌ಟಾಪ್: ಲಿಂಕ್‌ಗಳನ್ನು ಸಮತಲ ಟೈಲ್‌ಗಳಲ್ಲಿ ಪ್ರದರ್ಶಿಸಲಾಗುತ್ತದೆ
- ಮೊಬೈಲ್ ಆವೃತ್ತಿ: ಮೆನು ಬಟನ್‌ಗೆ ಸಂಕುಚಿಸಿ, ಸೈಡ್‌ಬಾರ್ ಡ್ರಾಯರ್ ಅನ್ನು ಸ್ಲೈಡ್ ಮಾಡಲು ಕ್ಲಿಕ್ ಮಾಡಿ
- ಮಾರ್ಗ ಸಕ್ರಿಯಗೊಳಿಸುವಿಕೆ: ಪ್ರಸ್ತುತ ಮಾರ್ಗ ಮತ್ತು ಹ್ಯಾಶ್ ಮೌಲ್ಯವನ್ನು ಸ್ವಯಂಚಾಲಿತವಾಗಿ ಹೋಲಿಸಿ, ಮತ್ತು ಹೊಂದಾಣಿಕೆಯ ಲಿಂಕ್‌ಗೆ ಸಕ್ರಿಯಗೊಳಿಸುವ ಸ್ಥಿತಿ ವರ್ಗದ ಹೆಸರನ್ನು ಸೇರಿಸಿ `A`
- ಸ್ವಯಂಚಾಲಿತವಾಗಿ ಮುಚ್ಚಿ: ಮೊಬೈಲ್ ಟರ್ಮಿನಲ್ ಲಿಂಕ್ ಅಥವಾ ಸೈಡ್‌ಬಾರ್‌ನ ಬಾಹ್ಯ ಹಿನ್ನೆಲೆಯಲ್ಲಿ ಕ್ಲಿಕ್ ಮಾಡಿದಾಗ, ಸೈಡ್‌ಬಾರ್ ಸ್ವಯಂಚಾಲಿತವಾಗಿ ಮುಚ್ಚುತ್ತದೆ.

## ವಿವರಿಸಲು

ಘಟಕವನ್ನು ಪ್ರಾರಂಭಿಸಿದಾಗ, ಚೈಲ್ಡ್ ಎಲಿಮೆಂಟ್‌ನಲ್ಲಿ ಯಾವುದೇ `c-side` ಇಲ್ಲದಿದ್ದರೆ, ಮೊಬೈಲ್ ಮೆನು ಬಟನ್ ಮತ್ತು `c-side` ಸೈಡ್‌ಬಾರ್ ಅನ್ನು ಸ್ವಯಂಚಾಲಿತವಾಗಿ ರಚಿಸಲಾಗುತ್ತದೆ ಮತ್ತು ಎಲ್ಲಾ `a` ಟ್ಯಾಗ್‌ಗಳನ್ನು ಸೈಡ್‌ಬಾರ್‌ಗೆ ಸರಿಸಲಾಗುತ್ತದೆ.
