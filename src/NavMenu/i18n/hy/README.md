# Նավիգացիոն ընտրացանկ՝ հարմարեցված էկրանի տարբեր չափերին

- Աշխատասեղան. Հղումները ցուցադրվում են հորիզոնական սալիկներով
- Բջջային տարբերակ. Ծալեք ընտրացանկի կոճակը, սեղմեք՝ կողագոտու դարակից դուրս սահելու համար
- Երթուղու ակտիվացում. ավտոմատ կերպով համեմատեք ընթացիկ ուղին և հեշ արժեքը և ավելացրեք ակտիվացման կարգավիճակի դասի անունը համապատասխան հղմանը `A`
- Ավտոմատ փակում. Երբ շարժական տերմինալը սեղմում է հղման կամ կողագոտու արտաքին ֆոնի վրա, կողագոտին ավտոմատ կերպով փակվում է:

## օգտագործել

```html
<c-nav-menu>
  <a href="/">առաջին էջ</a>
  <a href="/doc">փաստաթուղթ</a>
  <a href="/about">մասին</a>
</c-nav-menu>
```

## նկարազարդել

Երբ բաղադրիչը սկզբնավորվում է, եթե երեխայի տարրում չկա `c-side`, շարժական մենյուի կոճակը և `c-side` կողագոտին ավտոմատ կստեղծվեն, և բոլոր `a` պիտակները կտեղափոխվեն կողագոտ:

## ոճի դաս

- **A**. ակտիվացված հղումը ավտոմատ կերպով կավելացնի դասի անունը `A`
