# ନାଭିଗେସନ୍ ମେନୁ ବିଭିନ୍ନ ସ୍କ୍ରିନ୍ ଆକାର ସହିତ ଅନୁକୂଳ ହୋଇଛି |

- ଡେସ୍କଟପ୍: ଭୂସମାନ୍ତର ଟାଇଲରେ ଲିଙ୍କ୍ ପ୍ରଦର୍ଶିତ ହୁଏ |
- ମୋବାଇଲ୍ ସଂସ୍କରଣ: ମେନୁ ବଟନ୍ ରେ ପତନ, ପାର୍ଶ୍ୱ ପଟି ଡ୍ରୟରକୁ ସ୍ଲାଇଡ୍ କରିବାକୁ କ୍ଲିକ୍ କରନ୍ତୁ |
- ମାର୍ଗ ସକ୍ରିୟତା: ସ୍ୱୟଂଚାଳିତ ଭାବରେ ସାମ୍ପ୍ରତିକ ପଥ ଏବଂ ହ୍ୟାସ୍ ମୂଲ୍ୟ ତୁଳନା କରନ୍ତୁ, ଏବଂ ମେଳଣ ଲିଙ୍କରେ ସକ୍ରିୟତା ସ୍ଥିତି ଶ୍ରେଣୀ ନାମ ଯୋଡନ୍ତୁ `A`
- ସ୍ୱୟଂଚାଳିତ ଭାବରେ ବନ୍ଦ: ଯେତେବେଳେ ମୋବାଇଲ୍ ଟର୍ମିନାଲ୍ ଏକ ଲିଙ୍କ୍ କିମ୍ବା ପାର୍ଶ୍ୱ ପଟିର ବାହ୍ୟ ପୃଷ୍ଠଭୂମି ଉପରେ କ୍ଲିକ୍ କରେ, ପାର୍ଶ୍ୱ ପଟି ସ୍ୱୟଂଚାଳିତ ଭାବରେ ବନ୍ଦ ହୋଇଯାଏ |

## ବର୍ଣ୍ଣନା କର

ଯେତେବେଳେ ଉପାଦାନଟି ପ୍ରାରମ୍ଭିକୃତ ହୁଏ, ଯଦି ଶିଶୁ ଉପାଦାନରେ `c-side` ନଥାଏ, ମୋବାଇଲ୍ ମେନୁ ବଟନ୍ ଏବଂ `c-side` ପାର୍ଶ୍ୱ ପଟି ସ୍ୱୟଂଚାଳିତ ଭାବରେ ସୃଷ୍ଟି ହେବ ଏବଂ ସମସ୍ତ `a` ଟ୍ୟାଗ୍ ପାର୍ଶ୍ୱ ପଟିକୁ ସ୍ଥାନାନ୍ତରିତ ହେବ |
