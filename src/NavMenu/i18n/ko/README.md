# 다양한 화면 크기에 맞게 조정된 탐색 메뉴

- 데스크톱: 링크가 가로 타일에 표시됩니다.
- 모바일 버전: 메뉴 버튼으로 접고 사이드바 서랍을 밀어내려면 클릭하세요.
- 경로 활성화: 현재 경로와 해시 값을 자동으로 비교하고 일치하는 링크에 활성화 상태 클래스 이름을 추가합니다. `A`
- 자동 닫기: 모바일 단말기에서 사이드바의 링크나 외부 배경을 클릭하면 사이드바가 자동으로 닫힙니다.

## 사용

```html
<c-nav-menu>
  <a href="/">첫 페이지</a>
  <a href="/doc">문서</a>
  <a href="/about">~에 대한</a>
</c-nav-menu>
```

## 설명하다

컴포넌트 초기화 시 하위 요소에 `c-side`이 없으면 모바일 메뉴 버튼과 `c-side` 사이드바가 자동으로 생성되고 모든 `a` 태그가 사이드바로 이동됩니다.

## 스타일 수업

- **답**: 활성화된 링크는 자동으로 클래스 이름 `A`을 추가합니다.
