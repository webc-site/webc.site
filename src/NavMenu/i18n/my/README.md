# မတူညီသော မျက်နှာပြင်အရွယ်အစားများအတွက် လမ်းညွှန်ချက်မီနူး

- ဒက်စ်တော့- လင့်ခ်များကို အလျားလိုက် အကွက်များဖြင့် ပြသထားသည်။
- မိုဘိုင်းဗားရှင်း- မီနူးခလုတ်သို့ ခေါက်သိမ်းပါ၊ ဘေးဘားအံဆွဲကို ဆွဲထုတ်ရန် နှိပ်ပါ။
- လမ်းကြောင်းစဖွင့်ခြင်း- လက်ရှိလမ်းကြောင်းနှင့် hash တန်ဖိုးကို အလိုအလျောက် နှိုင်းယှဉ်ပြီး ကိုက်ညီသည့်လင့်ခ်သို့ အသက်သွင်းမှု အခြေအနေ အတန်းအမည်ကို ပေါင်းထည့်ပါ။
- အလိုအလျောက်ပိတ်ခြင်း- မိုဘိုင်းဂိတ်သည် လင့်ခ်တစ်ခု သို့မဟုတ် ဘေးဘား၏ ပြင်ပနောက်ခံကို နှိပ်သောအခါ ဘေးဘားသည် အလိုအလျောက် ပိတ်သွားမည်ဖြစ်သည်။

## အသုံးပြု

```html
<c-nav-menu>
  <a href="/">ရှေ့စာမျက်နှာ</a>
  <a href="/doc">စာရွက်စာတမ်း</a>
  <a href="/about">အကြောင်း</a>
</c-nav-menu>
```

## ပုံဥပမာ

အစိတ်အပိုင်းကို စတင်လုပ်ဆောင်သောအခါ၊ ကလေးဒြပ်စင်တွင် `c-side`မရှိပါက၊ မိုဘိုင်းမီနူးခလုတ်နှင့် `c-side` ဘေးဘားကို အလိုအလျောက် ဖန်တီးမည်ဖြစ်ပြီး၊ တဂ်အားလုံးကို ဘေးဘားသို့ ရွှေ့မည်ဖြစ်သည်။

## စတိုင်အတန်း

- **A**- အသက်သွင်းထားသော လင့်ခ်သည် အတန်းအမည် `A`ကို အလိုအလျောက် ထည့်သွင်းပါမည်။
