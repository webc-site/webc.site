# ເມນູນໍາທາງທີ່ປັບໃຫ້ເຫມາະສົມກັບຂະຫນາດຫນ້າຈໍທີ່ແຕກຕ່າງກັນ

- Desktop: ລິ້ງຖືກສະແດງຢູ່ໃນກະເບື້ອງແນວນອນ
- ລຸ້ນມືຖື: ຫຍໍ້ລົງໃສ່ປຸ່ມເມນູ, ຄລິກເພື່ອເລື່ອນອອກແຖບດ້ານຂ້າງ
- ການເປີດໃຊ້ເສັ້ນທາງ: ປຽບທຽບເສັ້ນທາງປະຈຸບັນ ແລະຄ່າ hash ໂດຍອັດຕະໂນມັດ, ແລະເພີ່ມຊື່ຊັ້ນສະຖານະການເປີດໃຊ້ງານໃສ່ລິ້ງທີ່ກົງກັນ `A`
- ປິດອັດຕະໂນມັດ: ເມື່ອເຄື່ອງໃຊ້ມືຖືຄລິກໃສ່ລິ້ງ ຫຼືພື້ນຫຼັງພາຍນອກຂອງແຖບດ້ານຂ້າງ, ແຖບດ້ານຂ້າງຈະປິດໂດຍອັດຕະໂນມັດ.

## ສະແດງໃຫ້ເຫັນ

ເມື່ອອົງປະກອບເລີ່ມຕົ້ນ, ຖ້າບໍ່ມີ `c-side`ໃນອົງປະກອບລູກ, ປຸ່ມເມນູມືຖື ແລະ `c-side`ແຖບດ້ານຂ້າງຈະຖືກສ້າງໂດຍອັດຕະໂນມັດ, ແລະທຸກ `a`ແທັກຈະຖືກຍ້າຍໄປຢູ່ໃນແຖບດ້ານຂ້າງ.
