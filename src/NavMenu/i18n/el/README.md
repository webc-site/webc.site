# Μενού πλοήγησης προσαρμοσμένο σε διαφορετικά μεγέθη οθόνης

- Επιφάνεια εργασίας: Οι σύνδεσμοι εμφανίζονται σε οριζόντια πλακίδια
- Έκδοση για κινητά: Σύμπτυξη στο κουμπί μενού, κάντε κλικ για να σύρετε έξω το συρτάρι της πλαϊνής γραμμής
- Ενεργοποίηση διαδρομής: συγκρίνετε αυτόματα την τρέχουσα διαδρομή και την τιμή κατακερματισμού και προσθέστε το όνομα κλάσης κατάστασης ενεργοποίησης στον αντίστοιχο σύνδεσμο `A`
- Αυτόματο κλείσιμο: Όταν το τερματικό κινητής τηλεφωνίας κάνει κλικ σε έναν σύνδεσμο ή στο εξωτερικό φόντο της πλαϊνής γραμμής, η πλαϊνή γραμμή κλείνει αυτόματα.

## χρήση

```html
<c-nav-menu>
  <a href="/">πρωτοσέλιδο</a>
  <a href="/doc">έγγραφο</a>
  <a href="/about">για</a>
</c-nav-menu>
```

## εικονογραφώ

Όταν αρχικοποιηθεί το στοιχείο, εάν δεν υπάρχει `c-side` στο θυγατρικό στοιχείο, το κουμπί μενού για κινητά και η πλευρική γραμμή `c-side` θα δημιουργηθούν αυτόματα και όλες οι ετικέτες `a` θα μετακινηθούν στην πλαϊνή γραμμή.

## τάξη στυλ

- **Α**: Ο ενεργοποιημένος σύνδεσμος θα προσθέσει αυτόματα το όνομα της τάξης `A`
