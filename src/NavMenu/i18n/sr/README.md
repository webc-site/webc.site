# Навигациони мени прилагођен различитим величинама екрана

- Радна површина: Везе се приказују у хоризонталним плочицама
- Мобилна верзија: Дугме скупи у мени, кликните да бисте извукли фиоку бочне траке
- Активација руте: аутоматски упореди тренутну путању и хеш вредност и додајте назив класе статуса активације на одговарајућу везу <б ид="1">
- Аутоматско затварање: Када мобилни терминал кликне на везу или спољну позадину бочне траке, бочна трака се аутоматски затвара.

## користити

```html
<c-nav-menu>
  <a href="/">насловна страна</a>
  <a href="/doc">документ</a>
  <a href="/about">о</a>
</c-nav-menu>
```

## илуструју

Када се компонента иницијализује, ако у подређеном елементу нема <б ид="1">, дугме мобилног менија и бочна трака <б ид="2"> ће се аутоматски креирати, а све ознаке <б ид="3"> ће бити премештене у бочну траку.

## стилска класа

- **А**: Активирана веза ће аутоматски додати име класе <б ид="1">
