# नेव्हिगेशन मेनू वेगवेगळ्या स्क्रीन आकारांमध्ये रुपांतरित केला

- डेस्कटॉप: लिंक्स क्षैतिज टाइलमध्ये प्रदर्शित केले जातात
- मोबाइल आवृत्ती: मेनू बटणावर संक्षिप्त करा, साइडबार ड्रॉवर स्लाइड करण्यासाठी क्लिक करा
- मार्ग सक्रियकरण: वर्तमान मार्ग आणि हॅश मूल्याची आपोआप तुलना करा आणि जुळणाऱ्या लिंकवर सक्रियकरण स्थिती वर्ग नाव जोडा `A`
- स्वयंचलितपणे बंद करा: जेव्हा मोबाइल टर्मिनल लिंकवर किंवा साइडबारच्या बाह्य पार्श्वभूमीवर क्लिक करते तेव्हा साइडबार आपोआप बंद होतो.

## वापर

```html
<c-nav-menu>
  <a href="/">मुखपृष्ठ</a>
  <a href="/doc">दस्तऐवज</a>
  <a href="/about">बद्दल</a>
</c-nav-menu>
```

## स्पष्ट करणे

घटक सुरू झाल्यावर, मूल घटकामध्ये `c-side` नसल्यास, मोबाइल मेनू बटण आणि `c-side` साइडबार आपोआप तयार होतील आणि सर्व `a` टॅग साइडबारमध्ये हलवले जातील.

## शैली वर्ग

- **A**: सक्रिय केलेली लिंक आपोआप वर्गाचे नाव `A` जोडेल
