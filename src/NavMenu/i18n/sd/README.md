# نيويگيشن مينيو مختلف اسڪرين جي سائزن سان ترتيب ڏنل

- ڊيسڪ ٽاپ: ڪڙيون افقي ٽائلس ۾ ڏيکاريل آھن
- موبائل ورزن: مينيو بٽڻ ۾ کوليو، سائڊبار دراز کي سلائيڊ ڪرڻ لاءِ ڪلڪ ڪريو
- روٽ چالو ڪرڻ: خودڪار طريقي سان موجوده رستي ۽ هيش ويل جو مقابلو ڪريو، ۽ ايڪٽيويشن اسٽيٽس ڪلاس جو نالو ملندڙ لنڪ ۾ شامل ڪريو `A`
- خودڪار طور تي بند ڪريو: جڏهن موبائل ٽرمينل لنڪ تي ڪلڪ ڪري ٿو يا سائڊبار جي ٻاهرين پس منظر تي، سائڊبار خودڪار طور تي بند ٿي ويندو آهي.

## بيان ڪرڻ

جڏهن جزو شروع ڪيو ويندو، جيڪڏهن ٻار جي عنصر ۾ ڪو به `c-side` نه هوندو، موبائل مينيو بٽڻ ۽ `c-side` سائڊبار پاڻمرادو ٺاهيا ويندا، ۽ سڀئي `a` ٽيگ سائڊبار ۾ منتقل ڪيا ويندا.
