# نیویگیشن مینو اسکرین کے مختلف سائز کے مطابق ڈھال لیا گیا۔

- ڈیسک ٹاپ: لنکس افقی ٹائلوں میں دکھائے جاتے ہیں۔
- موبائل ورژن: مینو بٹن میں سمٹیں، سائڈبار دراز کو سلائیڈ کرنے کے لیے کلک کریں۔
- روٹ ایکٹیویشن: خود بخود موجودہ پاتھ اور ہیش ویلیو کا موازنہ کریں، اور ایکٹیویشن اسٹیٹس کلاس کا نام مماثل لنک میں شامل کریں `A`
- خودکار طور پر بند: جب موبائل ٹرمینل کسی لنک یا سائڈبار کے بیرونی پس منظر پر کلک کرتا ہے، تو سائڈبار خود بخود بند ہو جاتا ہے۔

## استعمال کریں

```html
<c-nav-menu>
  <a href="/">صفحہ اول</a>
  <a href="/doc">دستاویز</a>
  <a href="/about">کے بارے میں</a>
</c-nav-menu>
```

## مثال دینا

جب جزو کو شروع کیا جاتا ہے، اگر چائلڈ عنصر میں کوئی `c-side` نہیں ہے تو، موبائل مینو بٹن اور `c-side` سائڈبار خود بخود بن جائیں گے، اور تمام `a` ٹیگز سائڈبار میں منتقل ہو جائیں گے۔

## سٹائل کلاس

- **A**: فعال لنک خود بخود کلاس کا نام `A` شامل کر دے گا۔
