# قائمة التنقل تتكيف مع أحجام الشاشات المختلفة

- سطح المكتب: يتم عرض الروابط في مربعات أفقية
- إصدار الهاتف المحمول: قم بالطي في زر القائمة، وانقر لتحريك درج الشريط الجانبي للخارج
- تنشيط المسار: قم تلقائيًا بمقارنة المسار الحالي وقيمة التجزئة، وأضف اسم فئة حالة التنشيط إلى الرابط المطابق `A`
- الإغلاق تلقائيًا: عندما تنقر محطة الهاتف المحمول على رابط أو على الخلفية الخارجية للشريط الجانبي، يتم إغلاق الشريط الجانبي تلقائيًا.

## يستخدم

```html
<c-nav-menu>
  <a href="/">الصفحة الأولى</a>
  <a href="/doc">وثيقة</a>
  <a href="/about">عن</a>
</c-nav-menu>
```

## يوضح

عند تهيئة المكون، إذا لم يكن هناك `c-side` في العنصر الفرعي، فسيتم إنشاء زر قائمة الهاتف المحمول والشريط الجانبي `c-side` تلقائيًا، وسيتم نقل جميع علامات `a` إلى الشريط الجانبي.

## فئة النمط

- **A**: سيضيف الرابط المنشط اسم الفصل تلقائيًا `A`
