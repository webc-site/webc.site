# منوی ناوبری متناسب با اندازه های مختلف صفحه نمایش

- دسکتاپ: پیوندها در کاشی های افقی نمایش داده می شوند
- نسخه موبایل: جمع کردن در دکمه منو، کلیک کنید تا کشوی نوار کناری به بیرون بکشید
- فعال‌سازی مسیر: به‌طور خودکار مسیر فعلی و مقدار هش را مقایسه کنید و نام کلاس وضعیت فعال‌سازی را به پیوند منطبق اضافه کنید `A`
- بسته شدن خودکار: وقتی ترمینال تلفن همراه روی پیوند یا پس‌زمینه خارجی نوار کناری کلیک می‌کند، نوار کناری به‌طور خودکار بسته می‌شود.

## استفاده کنید

```html
<c-nav-menu>
  <a href="/">صفحه اول</a>
  <a href="/doc">سند</a>
  <a href="/about">در مورد</a>
</c-nav-menu>
```

## نشان دادن

هنگامی که مؤلفه مقدار دهی اولیه می شود، اگر در عنصر فرزند `c-side` وجود نداشته باشد، دکمه منوی تلفن همراه و نوار کناری `c-side` به طور خودکار ایجاد می شود و همه تگ های `a` به نوار کناری منتقل می شوند.

## کلاس سبک

- **A**: پیوند فعال شده به طور خودکار نام کلاس `A` را اضافه می کند
