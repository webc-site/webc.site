# மூடு பட்டனுடன் பாப்அப் லேயரை உருவாக்கி காண்பிக்கவும்

- பாப்அப் லேயரை உருவாக்கி காட்டுவதற்கான செயல்பாட்டு அழைப்பு
- உள்ளமைந்த மூட பட்டன், DOM இலிருந்து உறுப்பை மூட மற்றும் அகற்ற கிளிக் செய்யவும்
- உள்ளீட்டு பெட்டியை டிஃபோகஸ் செய்ய Esc ஐ அழுத்தவும் அல்லது பாப்அப் லேயரை மூடவும்

## டெமோவைப் பயன்படுத்தவும்

```html
<button>பாப் அப் செய்ய கிளிக் செய்யவும்</button>

<script type="module">
import BoxX from "webc.site/BoxX.js";

document.querySelector("button").onclick = () => {
  const dialog = BoxX(),
    main = document.createElement("main"),
    h3 = document.createElement("h3");
  main.classList.add("Lg");
  h3.textContent = "உரை";
  main.appendChild(h3);
  dialog.appendChild(main);
};
</script>
```


## இடைமுக அளவுருக்கள்

இயல்புநிலை ஏற்றுமதி செயல்பாடு `HTMLDialogElement`ஐ வழங்குகிறது, மேலும் `Box` மற்றும் `X` நடை வகுப்புகள் உரையாடல் உறுப்புடன் சேர்க்கப்பட்டுள்ளன.

## பாணி வகுப்பு

### `.Box.X`

`dialog` உறுப்புக்கு பயன்படுத்தப்பட்டது, உரையாடல் பாணியைக் குறிப்பிடுகிறது.

### `a.X`

`a` உறுப்புக்கு பயன்படுத்தப்பட்டது, மூடும் பொத்தான் பாணியைக் குறிப்பிடுகிறது.
