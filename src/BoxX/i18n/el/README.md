# Δημιουργήστε και εμφανίστε ένα αναδυόμενο επίπεδο με ένα κουμπί κλεισίματος

- Λειτουργική κλήση για δημιουργία και εμφάνιση αναδυόμενου στρώματος
- Ενσωματωμένο κουμπί κλεισίματος, κάντε κλικ για να κλείσετε και να αφαιρέσετε το στοιχείο από το DOM
- Πατήστε Esc για να αποεστιάσετε το πλαίσιο εισόδου ή να κλείσετε το αναδυόμενο επίπεδο

## Χρησιμοποιήστε το demo

```html
<button>Κάντε κλικ για να εμφανιστεί</button>

<script type="module">
import BoxX from "webc.site/BoxX.js";

document.querySelector("button").onclick = () => {
  const dialog = BoxX(),
    main = document.createElement("main"),
    h3 = document.createElement("h3");
  main.classList.add("Lg");
  h3.textContent = "κείμενο";
  main.appendChild(h3);
  dialog.appendChild(main);
};
</script>
```


## Παράμετροι διεπαφής

Η προεπιλεγμένη συνάρτηση εξαγωγής επιστρέφει `HTMLDialogElement` και οι κλάσεις στυλ `Box` και `X` έχουν προστεθεί στο στοιχείο διαλόγου.

## τάξη στυλ

### `.Box.X`

Εφαρμόζεται στο στοιχείο `dialog`, καθορίζει το στυλ διαλόγου.

### `a.X`

Εφαρμόζεται στο στοιχείο `a`, καθορίζει το στυλ του κουμπιού κλεισίματος.
