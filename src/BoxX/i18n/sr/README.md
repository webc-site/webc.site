# Направите и прикажите искачући слој помоћу дугмета за затварање

- Функционални позив за креирање и приказ попуп слоја
- Уграђено дугме за затварање, кликните да бисте затворили и уклонили елемент из ДОМ-а
- Притисните Есц да дефокусирате поље за унос или затворите искачући слој

## Користите демо

```html
<button>Кликните да бисте искочили</button>

<script type="module">
import BoxX from "webc.site/BoxX/_.js";

document.querySelector("button").onclick = () => {
  const dialog = BoxX(),
    main = document.createElement("main"),
    h3 = document.createElement("h3");
  main.classList.add("Lg");
  х3.тектЦонтент = "текст";
  main.appendChild(h3);
  dialog.appendChild(main);
};
</script>
```


## Параметри интерфејса

Подразумевана функција извоза враћа <б ид="1">, а стилске класе <б ид="2"> и <б ид="3"> су додате елементу дијалога.

## стилска класа

### `.Box.X`

Примењено на елемент <б ид="1">, одређује стил дијалога.

### `a.X`

Примењено на елемент <б ид="1">, одређује стил дугмета за затварање.
