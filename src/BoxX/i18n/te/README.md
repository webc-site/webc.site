# క్లోజ్ బటన్‌తో పాప్‌అప్ లేయర్‌ని సృష్టించండి మరియు ప్రదర్శించండి

- పాప్అప్ లేయర్‌ని సృష్టించడానికి మరియు ప్రదర్శించడానికి ఫంక్షనల్ కాల్
- అంతర్నిర్మిత క్లోజ్ బటన్, DOM నుండి మూలకాన్ని మూసివేయడానికి మరియు తీసివేయడానికి క్లిక్ చేయండి
- ఇన్‌పుట్ బాక్స్‌ను డిఫోకస్ చేయడానికి Esc నొక్కండి లేదా పాపప్ లేయర్‌ను మూసివేయండి

## డెమో ఉపయోగించండి

```html
<button>పాప్ అప్ చేయడానికి క్లిక్ చేయండి</button>

<script type="module">
import BoxX from "webc.site/BoxX/_.js";

document.querySelector("button").onclick = () => {
  const dialog = BoxX(),
    main = document.createElement("main"),
    h3 = document.createElement("h3");
  main.classList.add("Lg");
  h3.textContent = "టెక్స్ట్";
  main.appendChild(h3);
  dialog.appendChild(main);
};
</script>
```


## ఇంటర్ఫేస్ పారామితులు

డిఫాల్ట్ ఎగుమతి ఫంక్షన్ `HTMLDialogElement`ని అందిస్తుంది మరియు `Box` మరియు `X` స్టైల్ క్లాసులు డైలాగ్ ఎలిమెంట్‌కు జోడించబడ్డాయి.

## శైలి తరగతి

### `.Box.X`

`dialog` మూలకానికి వర్తింపజేయబడింది, డైలాగ్ శైలిని పేర్కొంటుంది.

### `a.X`

`a` మూలకానికి వర్తింపజేయబడింది, క్లోజ్ బటన్ శైలిని పేర్కొంటుంది.
