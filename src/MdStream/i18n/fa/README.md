# رندر جریانی متن Markdown

- Markdown را به صورت تدریجی تجزیه و رندر کنید
- با عنصر والد پیمایشی همکاری کنید تا زمانی که محتوا از صفحه نمایش بیشتر شود، به طور خودکار پیمایش کنید
- هنگام کلیک کردن یا پیمایش به بالا، اسکرول خودکار را مکث کنید، با اسکرول کردن به پایین ادامه دهید
- نمایش خودکار نشانگرهای وضعیت تایپ

## از دمو استفاده کنید

```html
<c-vs>
  <c-md-stream id="md-view"></c-md-stream>
</c-vs>

<script type="module">
  import "webc.site/MdStream/_.js";

  const el = document.getElementById("md-view"),
    sleep = (ms) => new Promise((resolve) => setTimeout(resolve, ms));

  // پخش جریانی داده ها از طریق تابع مولد ناهمزمان برگشتی
  el.gen = async function* () {
    بازده "# عنوان\n"؛
    for (let i = 1; i <= 100; ++i) {
      await sleep(400);
      بازده "- آیتم " + i + "\n";
    }
  };
</script>
```

## دارایی

| نام ویژگی | نوع | نشان دادن |
| :--- | :--- | :--- |
| `gen` | `Function` | عملکردی که یک تکرار/مولد ناهمزمان را برمی‌گرداند، محتوا را پاک می‌کند و پس از تنظیم دوباره ارائه می‌شود. |

## سبک

| کلاس/برچسب | نشان دادن |
| :--- | :--- |
| `Md` | کلاس هنگام نصب به طور خودکار به مؤلفه اضافه می شود، سبک Markdown را تنظیم کنید |
| `i.T` | نشانگر وضعیت تایپ به طور خودکار در هنگام ورود داده ها در پایان اضافه می شود |
