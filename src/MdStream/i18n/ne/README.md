# मार्कडाउन पाठको स्ट्रिमिङ रेन्डरिङ

- बढ्दो रूपमा पार्स र मार्कडाउन रेन्डर गर्नुहोस्
- सामग्री स्क्रिन नाघेको बेला स्वचालित रूपमा स्क्रोल गर्न स्क्रोलिङ अभिभावक तत्वसँग सहकार्य गर्नुहोस्
- क्लिक गर्दा वा माथि स्क्रोल गर्दा अटोस्क्रोल रोक्नुहोस्, तल स्क्रोल गर्दा पुन: सुरु गर्नुहोस्
- स्वचालित रूपमा टाइपिङ स्थिति संकेतकहरू प्रदर्शन गर्नुहोस्

## डेमो प्रयोग गर्नुहोस्

```html
<c-vs>
  <c-md-stream id="md-view"></c-md-stream>
</c-vs>

<script type="module">
  import "webc.site/MdStream/_.js";

  const el = document.getElementById("md-view"),
    sleep = (ms) => new Promise((resolve) => setTimeout(resolve, ms));

  // फर्काइएको एसिन्क्रोनस जेनेरेटर प्रकार्य मार्फत स्ट्रिमिङ डेटा रेन्डरिङ
  el.gen = async function* () {
    उपज "# शीर्षक\n";
    for (let i = 1; i <= 100; ++i) {
      await sleep(400);
      yield "- दफा " + i + " वस्तु\n";
    }
  };
</script>
```

## सम्पत्ति

| विशेषता नाम | प्रकार | चित्रण गर्नुहोस् |
| :--- | :--- | :--- |
| `gen` | `Function` | कार्य जसले एसिन्क्रोनस इटरेटर/जेनेरेटर फर्काउँछ, सामग्री खाली गर्दछ र सेटिङ पछि पुन: रेन्डर गर्दछ |

## शैली

| वर्ग/ट्याग | चित्रण गर्नुहोस् |
| :--- | :--- |
| `Md` | माउन्ट गर्दा कक्षा स्वतः कम्पोनेन्टमा थपियो, मार्कडाउन शैली सेट गर्नुहोस् |
| `i.T` | टाइपिङ स्थिति सूचक स्वचालित रूपमा डेटा प्रविष्टिको समयमा अन्तमा थपियो |
