# మార్క్‌డౌన్ టెక్స్ట్ యొక్క స్ట్రీమింగ్ రెండరింగ్

- క్రమానుగతంగా అన్వయించండి మరియు మార్క్‌డౌన్‌ను రెండర్ చేయండి
- కంటెంట్ స్క్రీన్‌ను మించినప్పుడు స్వయంచాలకంగా స్క్రోల్ చేయడానికి స్క్రోలింగ్ పేరెంట్ ఎలిమెంట్‌తో సహకరించండి
- క్లిక్ చేసినప్పుడు లేదా పైకి స్క్రోల్ చేస్తున్నప్పుడు ఆటోస్క్రోల్‌ను పాజ్ చేయండి, క్రిందికి స్క్రోల్ చేస్తున్నప్పుడు పునఃప్రారంభించండి
- టైపింగ్ స్థితి సూచికలను స్వయంచాలకంగా ప్రదర్శిస్తుంది

## డెమో ఉపయోగించండి

```html
<c-vs>
  <c-md-stream id="md-view"></c-md-stream>
</c-vs>

<script type="module">
  import "webc.site/MdStream/_.js";

  const el = document.getElementById("md-view"),
    sleep = (ms) => new Promise((resolve) => setTimeout(resolve, ms));

  // తిరిగి వచ్చిన అసమకాలిక జనరేటర్ ఫంక్షన్ ద్వారా స్ట్రీమింగ్ డేటా రెండరింగ్
  el.gen = async function* () {
    దిగుబడి "# టైటిల్\n";
    for (let i = 1; i <= 100; ++i) {
      await sleep(400);
      దిగుబడి "- అంశం " + i + "\n";
    }
  };
</script>
```

## ఆస్తి

| లక్షణం పేరు | రకం | వర్ణించండి |
| :--- | :--- | :--- |
| `gen` | `Function` | అసమకాలిక ఇటరేటర్/జనరేటర్‌ను తిరిగి ఇచ్చే ఫంక్షన్, కంటెంట్‌ను క్లియర్ చేస్తుంది మరియు సెట్టింగ్ తర్వాత మళ్లీ రెండర్ చేస్తుంది |

## శైలి

| `i.T` | డేటా నమోదు సమయంలో ముగింపులో టైపింగ్ స్థితి సూచిక స్వయంచాలకంగా జోడించబడింది |
