# Потоковое рендеринг текста Markdown

- Поэтапный анализ и рендеринг Markdown
- Взаимодействуйте с родительским элементом прокрутки для автоматической прокрутки, когда содержимое выходит за пределы экрана.
- Приостанавливать автопрокрутку при нажатии или прокрутке вверх, возобновлять при прокрутке вниз
- Автоматически отображать индикаторы статуса набора текста

## Используйте демо-версию

```html
<c-vs>
  <c-md-stream id="md-view"></c-md-stream>
</c-vs>

<script type="module">
  import "webc.site/MdStream.js";

  const el = document.getElementById("md-view"),
    sleep = (ms) => new Promise((resolve) => setTimeout(resolve, ms));

  // Потоковая отрисовка данных через возвращаемую функцию асинхронного генератора
  el.gen = async function* () {
    доходность "# заголовок\n";
    for (let i = 1; i <= 100; ++i) {
      await sleep(400);
      выход "- элемент " + i + "\n";
    }
  };
</script>
```

## свойство

| имя атрибута | тип | иллюстрировать |
| :--- | :--- | :--- |
| `gen` | `Function` | Функция, которая возвращает асинхронный итератор/генератор, очищает содержимое и повторно отображает после установки |

## стиль

| Класс/Тег | иллюстрировать |
| :--- | :--- |
| `Md` | Класс автоматически добавляется к компоненту при монтировании, устанавливает стиль Markdown |
| `i.T` | Индикатор статуса набора автоматически добавляется в конце во время ввода данных |
