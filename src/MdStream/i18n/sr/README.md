# Стримовање приказивања Маркдовн текста

- Постепено анализирајте и рендерујте Маркдовн
- Сарађујте са родитељским елементом који се помера да бисте се аутоматски померали када садржај премаши екран
- Паузирајте аутоматско померање када кликнете или померате нагоре, наставите када се померате надоле
- Аутоматски приказ индикатора статуса куцања

## Користите демо

```html
<c-vs>
  <c-md-stream id="md-view"></c-md-stream>
</c-vs>

<script type="module">
  import "webc.site/MdStream.js";

  const el = document.getElementById("md-view"),
    sleep = (ms) => new Promise((resolve) => setTimeout(resolve, ms));

  // Стримовање приказивања података кроз враћену функцију асинхроног генератора
  el.gen = async function* () {
    ииелд "# наслов\н";
    for (let i = 1; i <= 100; ++i) {
      await sleep(400);
      принос "- ставка " + и + "\н";
    }
  };
</script>
```

## имовине

| назив атрибута | тип | илуструју |
| :--- | :--- | :--- |
| `gen` | `Function` | Функција која враћа асинхрони итератор/генератор, брише садржај и поново рендерује након подешавања |

## стил

| Цласс/Таг | илуструју |
| :--- | :--- |
| `Md` | Класа се аутоматски додаје компоненти приликом монтирања, подесите Маркдовн стил |
| `i.T` | Индикатор статуса куцања се аутоматски додаје на крају током уноса података |
