# การสตรีมการเรนเดอร์ข้อความ Markdown

- แยกวิเคราะห์และแสดงผล Markdown แบบค่อยเป็นค่อยไป
- ร่วมมือกับองค์ประกอบหลักในการเลื่อนเพื่อเลื่อนโดยอัตโนมัติเมื่อเนื้อหาเกินหน้าจอ
- หยุดการเลื่อนอัตโนมัติชั่วคราวเมื่อคลิกหรือเลื่อนขึ้น และดำเนินการต่อเมื่อเลื่อนลง
- แสดงตัวบ่งชี้สถานะการพิมพ์โดยอัตโนมัติ

## ใช้การสาธิต

```html
<c-vs>
  <c-md-stream id="md-view"></c-md-stream>
</c-vs>

<script type="module">
  import "webc.site/MdStream.js";

  const el = document.getElementById("md-view"),
    sleep = (ms) => new Promise((resolve) => setTimeout(resolve, ms));

  // การสตรีมข้อมูลที่เรนเดอร์ผ่านฟังก์ชันตัวสร้างอะซิงโครนัสที่ส่งคืน
  el.gen = async function* () {
    ให้ผลผลิต # ชื่อ \ n ";
    for (let i = 1; i <= 100; ++i) {
      await sleep(400);
      ผลผลิต "- รายการ" + i + "\n";
    }
  };
</script>
```

## คุณสมบัติ

| ชื่อแอตทริบิวต์ | พิมพ์ | แสดงให้เห็น |
| :--- | :--- | :--- |
| `gen` | `Function` | ฟังก์ชันที่ส่งคืนตัววนซ้ำ/ตัวสร้างแบบอะซิงโครนัส ล้างเนื้อหา และเรนเดอร์ใหม่หลังจากการตั้งค่า |

## สไตล์

| คลาส/แท็ก | แสดงให้เห็น |
| :--- | :--- |
| `Md` | เพิ่มคลาสให้กับส่วนประกอบโดยอัตโนมัติเมื่อทำการติดตั้ง ตั้งค่าสไตล์ Markdown |
| `i.T` | ตัวบ่งชี้สถานะการพิมพ์จะถูกเพิ่มโดยอัตโนมัติเมื่อสิ้นสุดระหว่างการป้อนข้อมูล |
