# ການຖ່າຍທອດການສະແດງຂໍ້ຄວາມ Markdown

- ແຍກວິເຄາະ ແລະສະແດງ Markdown ເພີ່ມຂຶ້ນ
- ຮ່ວມມືກັບອົງປະກອບຫຼັກທີ່ເລື່ອນໄດ້ເພື່ອເລື່ອນອັດຕະໂນມັດເມື່ອເນື້ອຫາເກີນໜ້າຈໍ
- ຢຸດເລື່ອນອັດຕະໂນມັດເມື່ອຄລິກ ຫຼືເລື່ອນຂຶ້ນ, ສືບຕໍ່ເມື່ອເລື່ອນລົງ
- ສະແດງຕົວຊີ້ບອກສະຖານະການພິມໂດຍອັດຕະໂນມັດ

## ໃຊ້ຕົວຢ່າງ

```html
<c-vs>
  <c-md-stream id="md-view"></c-md-stream>
</c-vs>

<script type="module">
  import "webc.site/MdStream.js";

  const el = document.getElementById("md-view"),
    sleep = (ms) => new Promise((resolve) => setTimeout(resolve, ms));

  // ການຖ່າຍທອດຂໍ້ມູນການສະແດງຜົນຜ່ານຟັງຊັນເຄື່ອງກໍາເນີດແບບ asynchronous ທີ່ສົ່ງຄືນ
  el.gen = async function* () {
    ໃຫ້ "# ຫົວຂໍ້\n";
    for (let i = 1; i <= 100; ++i) {
      await sleep(400);
      ຜົນຜະລິດ "- ລາຍການ " + i + "\n";
    }
  };
</script>
```

## ຊັບສິນ

| ຊື່ຄຸນສົມບັດ | ປະເພດ | ສະແດງໃຫ້ເຫັນ |
| :--- | :--- | :--- |
| `gen` | `Function` | ຟັງຊັນທີ່ສົ່ງຄືນເຄື່ອງເຮັດຊ້ຳ/ເຄື່ອງປັ່ນໄຟແບບບໍ່ຊິ້ງໂຄນ, ລຶບເນື້ອຫາ ແລະສະແດງຜົນຄືນໃໝ່ຫຼັງຈາກການຕັ້ງຄ່າ |

## ແບບ

| ໝວດໝູ່/ແທັກ | ສະແດງໃຫ້ເຫັນ |
| :--- | :--- |
| `Md` | ຫ້ອງຮຽນຖືກເພີ່ມໃສ່ອົງປະກອບໂດຍອັດຕະໂນມັດເມື່ອຕິດຕັ້ງ, ຕັ້ງຮູບແບບ Markdown |
| `i.T` | ຕົວຊີ້ບອກສະຖານະການພິມຖືກເພີ່ມໂດຍອັດຕະໂນມັດໃນຕອນທ້າຍໃນລະຫວ່າງການປ້ອນຂໍ້ມູນ |
