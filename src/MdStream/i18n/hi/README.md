# मार्कडाउन टेक्स्ट का स्ट्रीमिंग रेंडरिंग

- मार्कडाउन को क्रमिक रूप से पार्स और रेंडर करें
- जब सामग्री स्क्रीन से अधिक हो जाए तो स्वचालित रूप से स्क्रॉल करने के लिए स्क्रॉलिंग पैरेंट तत्व के साथ सहयोग करें
- क्लिक करते या ऊपर स्क्रॉल करते समय ऑटोस्क्रॉल रोकें, नीचे स्क्रॉल करते समय फिर से शुरू करें
- स्वचालित रूप से टाइपिंग स्थिति संकेतक प्रदर्शित करें

## डेमो का प्रयोग करें

```html
<c-vs>
  <c-md-stream id="md-view"></c-md-stream>
</c-vs>

<script type="module">
  import "webc.site/MdStream.js";

  const el = document.getElementById("md-view"),
    sleep = (ms) => new Promise((resolve) => setTimeout(resolve, ms));

  // लौटाए गए एसिंक्रोनस जनरेटर फ़ंक्शन के माध्यम से डेटा रेंडरिंग को स्ट्रीम करना
  el.gen = async function* () {
    उपज "# शीर्षक\n";
    for (let i = 1; i <= 100; ++i) {
      await sleep(400);
      उपज "-आइटम" + i + "\n";
    }
  };
</script>
```

## संपत्ति

| विशेषता नाम | प्रकार | उदाहरण देकर स्पष्ट करना |
| :--- | :--- | :--- |
| `gen` | `Function` | फ़ंक्शन जो एक एसिंक्रोनस इटरेटर/जनरेटर लौटाता है, सामग्री को साफ़ करता है और सेटिंग के बाद पुन: प्रस्तुत करता है |

## शैली

| कक्षा/टैग | उदाहरण देकर स्पष्ट करना |
| :--- | :--- |
| `Md` | माउंट करते समय क्लास स्वचालित रूप से घटक में जुड़ जाती है, मार्कडाउन शैली सेट करें |
| `i.T` | डेटा प्रविष्टि के दौरान टाइपिंग स्थिति संकेतक स्वचालित रूप से अंत में जुड़ जाता है |
