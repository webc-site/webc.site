# Απόδοση σε ροή του κειμένου Markdown

- Σταδιακή ανάλυση και απόδοση Markdown
- Συνεργαστείτε με το κυλιόμενο γονικό στοιχείο για αυτόματη κύλιση όταν το περιεχόμενο υπερβαίνει την οθόνη
- Παύση της αυτόματης κύλισης όταν κάνετε κλικ ή κάνετε κύλιση προς τα πάνω, συνεχίστε όταν κάνετε κύλιση προς τα κάτω
- Αυτόματη εμφάνιση ενδείξεων κατάστασης πληκτρολόγησης

## Χρησιμοποιήστε το demo

```html
<c-vs>
  <c-md-stream id="md-view"></c-md-stream>
</c-vs>

<script type="module">
  import "webc.site/MdStream.js";

  const el = document.getElementById("md-view"),
    sleep = (ms) => new Promise((resolve) => setTimeout(resolve, ms));

  // Απόδοση δεδομένων ροής μέσω της επιστρεφόμενης συνάρτησης ασύγχρονης γεννήτριας
  el.gen = async function* () {
    απόδοση "# τίτλος\n";
    for (let i = 1; i <= 100; ++i) {
      await sleep(400);
      απόδοση "- στοιχείο " + i + "\n";
    }
  };
</script>
```

## ιδιοκτησία

| όνομα χαρακτηριστικού | τύπος | εικονογραφώ |
| :--- | :--- | :--- |
| `gen` | `Function` | Λειτουργία που επιστρέφει έναν ασύγχρονο επαναλήπτη/γεννήτρια, διαγράφει το περιεχόμενο και αποδίδει εκ νέου μετά τη ρύθμιση |

## στυλ

| Τάξη/Ετικέτα | εικονογραφώ |
| :--- | :--- |
| `Md` | Η κλάση προστίθεται αυτόματα στο στοιχείο κατά την τοποθέτηση, ορίστε το στυλ Markdown |
| `i.T` | Η ένδειξη κατάστασης πληκτρολόγησης προστίθεται αυτόματα στο τέλος κατά την εισαγωγή δεδομένων |
