# მარკდაუნის ტექსტის სტრიმინგი

- თანდათანობით გააანალიზეთ და გამოიტანეთ Markdown
- ითანამშრომლეთ გადახვევის მშობელ ელემენტთან, რათა ავტომატურად გადახვიდეთ, როდესაც კონტენტი აჭარბებს ეკრანს
- დააპაუზეთ ავტომატური გადახვევა დაწკაპუნებისას ან ზემოთ გადახვევისას, განაახლეთ ქვევით გადახვევისას
- აკრეფის სტატუსის ინდიკატორების ავტომატურად ჩვენება

## გამოიყენეთ დემო ვერსია

```html
<c-vs>
  <c-md-stream id="md-view"></c-md-stream>
</c-vs>

<script type="module">
  import "webc.site/MdStream.js";

  const el = document.getElementById("md-view"),
    sleep = (ms) => new Promise((resolve) => setTimeout(resolve, ms));

  // მონაცემთა სტრიმინგი დაბრუნებული ასინქრონული გენერატორის ფუნქციის მეშვეობით
  el.gen = async function* () {
    yield "# title\n";
    for (let i = 1; i <= 100; ++i) {
      await sleep(400);
      yield "- item " + i + "\n";
    }
  };
</script>
```

## ქონება

| ატრიბუტის სახელი | ტიპი | ილუსტრირება |
| :--- | :--- | :--- |
| `gen` | `Function` | ფუნქცია, რომელიც აბრუნებს ასინქრონულ იტერატორს/გენერატორს, ასუფთავებს კონტენტს და ხელახლა ასახავს დაყენების შემდეგ |

## სტილი

| კლასი/ტეგი | ილუსტრირება |
| :--- | :--- |
| `Md` | კლასი ავტომატურად ემატება კომპონენტს დამონტაჟებისას, დააყენეთ Markdown სტილი |
| `i.T` | აკრეფის სტატუსის ინდიკატორი ავტომატურად დაემატება ბოლოს მონაცემების შეყვანისას |
