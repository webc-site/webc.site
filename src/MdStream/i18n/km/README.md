# бһҹбҹ’бһ‘бҹ’бһҡбһёбһҳбһ”бһ„бҹ’бһ бһ¶бһүбһўбһҸбҹ’бһҗбһ”бһ‘ Markdown

- бһ”бһ„бҹ’бһҖбһҫбһ“бһҖбһ¶бһҡбһүбҹӮбһҖ бһ“бһ·бһ„бһ”бһ„бҹ’бһ бһ¶бһү Markdown
- бһҹбһ бһҖбһ¶бһҡвҖӢбһҮбһ¶бһҳбһҪбһҷвҖӢбһ’бһ¶бһҸбһ»вҖӢбһҳбҹҒвҖӢбһҡбһҳбһјбһҡвҖӢбһҠбһҫбһҳбҹ’бһ”бһёвҖӢбһҡбһҳбһјбһҡвҖӢбһҠбҹ„бһҷвҖӢбһҹбҹ’бһңбҹҗбһҷвҖӢбһ”бҹ’бһҡбһңбһҸбҹ’бһҸбһ·вҖӢбһ“бҹ…вҖӢбһ–бҹҒбһӣвҖӢбһҳбһ¶бһҸбһ·бһҖбһ¶вҖӢбһӣбһҫбһҹвҖӢбһўбҹҒбһҖбҹ’бһҡбһ„бҹӢ
- бһ•бҹ’бһўбһ¶бһҖбһҖбһ¶бһҡбһҡбһҳбһјбһҡбһҠбҹ„бһҷбһҹбҹ’бһңбҹҗбһҷбһ”бҹ’бһҡбһңбһҸбҹ’бһҸбһ·бһ“бҹ…бһ–бҹҒбһӣбһ…бһ»бһ… бһ¬бһҡбһҳбһјбһҡбһЎбһҫбһ„бһӣбһҫ бһ”бһ“бҹ’бһҸбһ“бҹ…бһ–бҹҒбһӣбһҡбҹҶбһҖбһ·бһӣбһ…бһ»бҹҮбһҖбҹ’бһҡбҹ„бһҳ
- бһ”бһ„бҹ’бһ бһ¶бһүбһҹбһјбһ…бһ“бһ¶бһҖбһҡбһҹбҹ’бһҗбһ¶бһ“бһ—бһ¶бһ–бһңбһ¶бһҷбһ”бһүбҹ’бһ…бһјбһӣбһҠбҹ„бһҷбһҹбҹ’бһңбҹҗбһҷбһ”бҹ’бһҡбһңбһҸбҹ’бһҸбһ·

## бһ”бҹ’бһҡбһҫбһҖбһ¶бһҡбһ”бһ„бҹ’бһ бһ¶бһү

```html
<c-vs>
  <c-md-stream id="md-view"></c-md-stream>
</c-vs>

<script type="module">
  import "webc.site/MdStream.js";

  const el = document.getElementById("md-view"),
    sleep = (ms) => new Promise((resolve) => setTimeout(resolve, ms));

  // бһҹбҹ’бһ‘бҹ’бһҡбһёбһҳбһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷбһҠбҹӮбһӣбһ”бһ„бҹ’бһ бһ¶бһүбһҸбһ¶бһҳбһҡбһҷбҹҲбһҳбһ»бһҒбһ„бһ¶бһҡбһҳбҹүбһ¶бһҹбҹҠбһёбһ“бһ—бҹ’бһӣбһҫбһ„бһўбһҹбһҳбһҖбһ¶бһӣбһҠбҹӮбһӣбһ”бһ¶бһ“бһҸбҹ’бһҡбһЎбһ”бҹӢбһҳбһҖбһңбһ·бһү
  el.gen = async function* () {
    бһ‘бһ·бһ“бҹ’бһ“бһ•бһӣ "# бһ…бҹҶбһҺбһ„бһҮбһҫбһ„\n";
    for (let i = 1; i <= 100; ++i) {
      await sleep(400);
      yield "- бһҸбҹҶбһҺбһҫбһҡ " + i + " бһ‘бҹҶбһ“бһ·бһү\n";
    }
  };
</script>
```

## бһ‘бҹ’бһҡбһ–бҹ’бһҷбһҹбһҳбҹ’бһ”бһҸбҹ’бһҸбһ·

| бһҲбҹ’бһҳбҹ„бҹҮбһӮбһ»бһҺбһӣбһҖбҹ’бһҒбһҺбҹҲ | бһ”бҹ’бһҡбһ—бҹҒбһ‘ | бһ”бһ„бҹ’бһ бһ¶бһү |
| :--- | :--- | :--- |
| `gen` | `Function` | бһўбһ“бһ»бһӮбһҳбһ“бҹҚвҖӢбһҠбҹӮбһӣвҖӢбһҸбҹ’бһҡбһЎбһ”бҹӢвҖӢбһ§бһ”бһҖбһҡбһҺбҹҚвҖӢбһ”бҹҶбһ”бҹ’бһӣбҹӮбһ„вҖӢбһўбһҹбһҳбһҖбһ¶бһӣ/вҖӢбһҳбҹүбһ¶бһҹбҹҠбһёбһ“вҖӢбһ—бҹ’бһӣбһҫбһ„ бһҹбһҳбҹ’бһўбһ¶бһҸвҖӢбһҒбҹ’бһӣбһ№бһҳбһҹбһ¶бһҡ бһ“бһ·бһ„вҖӢбһ”бһ„бҹ’бһ бһ¶бһүвҖӢбһЎбһҫбһ„бһңбһ·бһүвҖӢбһ”бһ“бҹ’бһ‘бһ¶бһ”бҹӢвҖӢбһ–бһёвҖӢбһҖбһ¶бһҡвҖӢбһҖбҹҶбһҺбһҸбҹӢ |

## бһҡбһ…бһ“бһ¶бһ”бҹҗбһ‘бҹ’бһҳ
