# માર્કડાઉન ટેક્સ્ટનું સ્ટ્રીમિંગ રેન્ડરિંગ

- માર્કડાઉનને ક્રમિક રીતે પાર્સ અને રેન્ડર કરો
- જ્યારે સામગ્રી સ્ક્રીનને ઓળંગી જાય ત્યારે આપમેળે સ્ક્રોલ કરવા માટે સ્ક્રોલિંગ પેરેન્ટ ઘટક સાથે સહકાર આપો
- ક્લિક કરતી વખતે અથવા ઉપર સ્ક્રોલ કરતી વખતે ઑટોસ્ક્રોલને થોભાવો, જ્યારે નીચે સ્ક્રોલ કરો ત્યારે ફરી શરૂ કરો
- ટાઇપિંગ સ્થિતિ સૂચકાંકો આપમેળે પ્રદર્શિત કરો

## ડેમોનો ઉપયોગ કરો

```html
<c-vs>
  <c-md-stream id="md-view"></c-md-stream>
</c-vs>

<script type="module">
  import "webc.site/MdStream.js";

  const el = document.getElementById("md-view"),
    sleep = (ms) => new Promise((resolve) => setTimeout(resolve, ms));

  // પરત કરેલ અસુમેળ જનરેટર કાર્ય દ્વારા સ્ટ્રીમિંગ ડેટા રેન્ડરીંગ
  el.gen = async function* () {
    ઉપજ "# શીર્ષક\n";
    for (let i = 1; i <= 100; ++i) {
      await sleep(400);
      ઉપજ "- આઇટમ " + i + "\n";
    }
  };
</script>
```

## મિલકત

| લક્ષણ નામ | પ્રકાર | સમજાવવું |
| :--- | :--- | :--- |
| `gen` | `Function` | કાર્ય જે અસુમેળ ઇટરરેટર/જનરેટર પરત કરે છે, સામગ્રીને સાફ કરે છે અને સેટ કર્યા પછી ફરીથી રેન્ડર કરે છે |

## શૈલી

| વર્ગ/ટેગ | સમજાવવું |
| :--- | :--- |
| `Md` | માઉન્ટ કરતી વખતે વર્ગ આપમેળે ઘટકમાં ઉમેરાય છે, માર્કડાઉન શૈલી સેટ કરો |
| `i.T` | ડેટા એન્ટ્રી દરમિયાન ટાઈપિંગ સ્ટેટસ ઈન્ડિકેટર આપમેળે અંતમાં ઉમેરાઈ જાય છે |
