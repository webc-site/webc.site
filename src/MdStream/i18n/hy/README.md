# Markdown տեքստի հոսքային մատուցում

- Աստիճանաբար վերլուծել և արտապատկերել Markdown-ը
- Համագործակցեք պտտվող մայր տարրի հետ՝ ավտոմատ ոլորելու համար, երբ բովանդակությունը գերազանցում է էկրանը
- Դադարեցնել ավտոմատ ոլորումը սեղմելիս կամ վերև ոլորելիս, վերսկսել ներքև ոլորելիս
- Ավտոմատ ցուցադրել մուտքագրման կարգավիճակի ցուցիչները

## Օգտագործեք ցուցադրությունը

```html
<c-vs>
  <c-md-stream id="md-view"></c-md-stream>
</c-vs>

<script type="module">
  import "webc.site/MdStream.js";

  const el = document.getElementById("md-view"),
    sleep = (ms) => new Promise((resolve) => setTimeout(resolve, ms));

  // Տվյալների հոսքային մատուցում վերադարձված ասինխրոն գեներատորի ֆունկցիայի միջոցով
  el.gen = async function* () {
    ելքը «# վերնագիր\n»;
    for (let i = 1; i <= 100; ++i) {
      await sleep(400);
      yield "- item " + i + "\n";
    }
  };
</script>
```

## սեփականություն

| հատկանիշի անունը | տեսակը | նկարազարդել |
| :--- | :--- | :--- |
| `gen` | `Function` | Գործառույթ, որը վերադարձնում է ասինխրոն կրկնող/գեներատոր, մաքրում է բովանդակությունը և կարգավորումից հետո վերարտադրվում |

## ոճը

| Դաս/Թագ | նկարազարդել |
| :--- | :--- |
| `Md` | Դասը ավտոմատ կերպով ավելացվում է բաղադրիչին, երբ տեղադրվում է, սահմանեք Markdown ոճը |
| `i.T` | Տվյալների մուտքագրման ժամանակ վերջում ավտոմատ կերպով ավելացվել է մուտքագրման կարգավիճակի ցուցիչը |
