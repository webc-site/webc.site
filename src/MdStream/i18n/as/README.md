# মাৰ্কডাউন টেক্সটৰ ষ্ট্ৰীমিং ৰেণ্ডাৰ

- ক্ৰমান্বয়ে মাৰ্কডাউন বিশ্লেষণ আৰু ৰেণ্ডাৰ কৰক
- স্ক্ৰলিং পিতৃ উপাদানৰ সৈতে সহযোগ কৰক যেতিয়া বিষয়বস্তুৱে পৰ্দা অতিক্ৰম কৰে স্বয়ংক্ৰিয়ভাৱে স্ক্ৰল কৰিবলে
- ক্লিক বা ওপৰলৈ স্ক্ৰল কৰাৰ সময়ত অটোস্ক্ৰল স্থগিত ৰাখক, তললৈ স্ক্ৰল কৰাৰ সময়ত পুনৰাৰম্ভ কৰক
- টাইপিং অৱস্থা সূচকসমূহ স্বয়ংক্ৰিয়ভাৱে প্ৰদৰ্শন কৰক

## ডেমো ব্যৱহাৰ কৰক

```html
<c-vs>
  <c-md-stream id="md-view"></c-md-stream>
</c-vs>

<script type="module">
  import "webc.site/MdStream.js";

  const el = document.getElementById("md-view"),
    sleep = (ms) => new Promise((resolve) => setTimeout(resolve, ms));

  // ঘূৰাই দিয়া এচিনক্ৰ'নাছ জেনেৰেটৰ ফাংচনৰ জৰিয়তে ডাটা ৰেণ্ডাৰ ষ্ট্ৰীমিং কৰা
  el.gen = async function* () {
    উৎপাদন "# শিৰোনাম\n";
    for (let i = 1; i <= 100; ++i) {
      await sleep(400);
      উৎপাদন "- বস্তু " + i + "\n";
    }
  };
</script>
```

## সম্পত্তি

| বৈশিষ্ট্যৰ নাম | প্ৰকাৰ | চিত্ৰিত কৰক |
| :--- | :--- | :--- |
| `gen` | `Function` | ফাংচন যি এটা এচিনক্ৰ'নাছ ইটাৰেটৰ/জেনেৰেটৰ ঘূৰাই দিয়ে, বিষয়বস্তু পৰিষ্কাৰ কৰে আৰু সংহতিৰ পিছত পুনৰায় ৰেণ্ডাৰ কৰে |

## শৈলী

| ক্লাছ/টেগ | চিত্ৰিত কৰক |
