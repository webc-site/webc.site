# Поточно изобразяване на Markdown текст

- Постепенно анализирайте и изобразете Markdown
- Сътрудничете с превъртащия родителски елемент за автоматично превъртане, когато съдържанието надхвърли екрана
- Пауза на автоматичното превъртане при щракване или превъртане нагоре, възобновяване при превъртане надолу
- Автоматично показване на индикатори за състояние на писане

## Използвайте демонстрацията

```html
<c-vs>
  <c-md-stream id="md-view"></c-md-stream>
</c-vs>

<script type="module">
  import "webc.site/MdStream/_.js";

  const el = document.getElementById("md-view"),
    sleep = (ms) => new Promise((resolve) => setTimeout(resolve, ms));

  // Поточно изобразяване на данни чрез върнатата функция за асинхронен генератор
  el.gen = async function* () {
    добив "# заглавие\n";
    for (let i = 1; i <= 100; ++i) {
      await sleep(400);
      yield "- елемент " + i + "\n";
    }
  };
</script>
```

## собственост

| име на атрибут | тип | илюстрирам |
| :--- | :--- | :--- |
| `gen` | `Function` | Функция, която връща асинхронен итератор/генератор, изчиства съдържанието и рендерира отново след настройка |

## стил

| Клас/Етикет | илюстрирам |
| :--- | :--- |
| `Md` | Класът се добавя автоматично към компонента при монтиране, задайте стил Markdown |
| `i.T` | Индикаторът за състояние на въвеждане се добавя автоматично в края по време на въвеждане на данни |
