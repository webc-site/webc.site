# Markdown мәтінінің ағынды көрсетілімі

- Markdown-ды біртіндеп талдау және көрсету
- Мазмұн экраннан асып кеткенде автоматты түрде айналдыру үшін айналдыратын негізгі элементпен бірлесіп жұмыс жасаңыз
- Басқан кезде немесе жоғары жылжытқанда автоайналдыруды кідірту, төмен айналдыру кезінде жалғастыру
- Теру күйінің көрсеткіштерін автоматты түрде көрсету

## Демонстрацияны пайдаланыңыз

```html
<c-vs>
  <c-md-stream id="md-view"></c-md-stream>
</c-vs>

<script type="module">
  import "webc.site/MdStream/_.js";

  const el = document.getElementById("md-view"),
    sleep = (ms) => new Promise((resolve) => setTimeout(resolve, ms));

  // Қайтарылған асинхронды генератор функциясы арқылы ағындық деректерді көрсету
  el.gen = async function* () {
    кірістілік "# тақырып\n";
    for (let i = 1; i <= 100; ++i) {
      await sleep(400);
      кірістілік "- тармақ " + i + "\n";
    }
  };
</script>
```

## мүлік

| атрибут атауы | түрі | суреттеу |
| :--- | :--- | :--- |
| `gen` | `Function` | Асинхронды итераторды/генераторды қайтаратын, мазмұнды тазартатын және орнатқаннан кейін қайта көрсететін функция |

## стиль

| Класс/тег | суреттеу |
| :--- | :--- |
| `Md` | Орнату кезінде компонентке класс автоматты түрде қосылады, Markdown мәнерін орнатыңыз |
| `i.T` | Теру күйінің индикаторы деректерді енгізу кезінде соңында автоматты түрде қосылады |
