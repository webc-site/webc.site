# Стриминг рендерирање на Markdown текст

- Инкрементално анализирајте и рендерирајте Markdown
- Соработувајте со лизгачкиот родителски елемент за автоматско лизгање кога содржината го надминува екранот
- Паузирајте го автоматското лизгање при кликнување или лизгање нагоре, продолжете кога лизгате надолу
- Автоматски прикажува индикатори за статусот на пишување

## Користете го демо

```html
<c-vs>
  <c-md-stream id="md-view"></c-md-stream>
</c-vs>

<script type="module">
  import "webc.site/MdStream/_.js";

  const el = document.getElementById("md-view"),
    sleep = (ms) => new Promise((resolve) => setTimeout(resolve, ms));

  // Стриминг рендерирање на податоци преку вратената функција на асинхрони генератор
  el.gen = async function* () {
    дава „# наслов\n“;
    for (let i = 1; i <= 100; ++i) {
      await sleep(400);
      принос "- ставка " + i + "\n";
    }
  };
</script>
```

## имот

| име на атрибут | тип | илустрираат |
| :--- | :--- | :--- |
| `gen` | `Function` | Функција која враќа асинхрони итератор/генератор, ја брише содржината и повторно ја прикажува по поставувањето |

## стил

| Класа/ознака | илустрираат |
| :--- | :--- |
| `Md` | Класата автоматски се додава на компонентата при монтажа, поставете го стилот Markdown |
| `i.T` | Индикаторот за статус на пишување автоматски се додава на крајот при внесување податоци |
