# مارک ڈاؤن ٹیکسٹ کی سٹریمنگ رینڈرنگ

- بتدریج تجزیہ کریں اور مارک ڈاؤن کو رینڈر کریں۔
- جب مواد اسکرین سے بڑھ جائے تو خود بخود اسکرول کرنے کے لیے اسکرولنگ پیرنٹ عنصر کے ساتھ تعاون کریں۔
- کلک کرنے یا اوپر سکرول کرتے وقت آٹو سکرول کو روکیں، نیچے سکرول کرتے وقت دوبارہ شروع کریں۔
- ٹائپنگ اسٹیٹس کے اشارے خود بخود ڈسپلے کریں۔

## ڈیمو استعمال کریں۔

```html
<c-vs>
  <c-md-stream id="md-view"></c-md-stream>
</c-vs>

<script type="module">
  import "webc.site/MdStream.js";

  const el = document.getElementById("md-view"),
    sleep = (ms) => new Promise((resolve) => setTimeout(resolve, ms));

  // واپس کردہ غیر مطابقت پذیر جنریٹر فنکشن کے ذریعے سٹریمنگ ڈیٹا رینڈرنگ
  el.gen = async function* () {
    حاصل کریں "# عنوان\n"؛
    for (let i = 1; i <= 100; ++i) {
      await sleep(400);
      پیداوار "- آئٹم " + i + "\n"؛
    }
  };
</script>
```

## جائیداد

| وصف کا نام | قسم | مثال دینا |
| :--- | :--- | :--- |
| `gen` | `Function` | فنکشن جو ایک غیر مطابقت پذیر تکرار کرنے والا/جنریٹر واپس کرتا ہے، مواد کو صاف کرتا ہے اور ترتیب کے بعد دوبارہ پیش کرتا ہے |

## انداز

| کلاس/ٹیگ | مثال دینا |
| :--- | :--- |
| `Md` | کلاس بڑھتے وقت جزو میں خود بخود شامل ہوجاتی ہے، مارک ڈاؤن اسٹائل سیٹ کریں۔ |
| `i.T` | ڈیٹا انٹری کے دوران ٹائپنگ اسٹیٹس انڈیکیٹر خود بخود آخر میں شامل ہو جاتا ہے۔ |
