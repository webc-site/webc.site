# عرض متدفق للنص Markdown

- تحليل تدريجي وتقديم Markdown
- تعاون مع عنصر التمرير الأصلي للتمرير تلقائيًا عندما يتجاوز المحتوى الشاشة
- قم بإيقاف التمرير التلقائي مؤقتًا عند النقر أو التمرير لأعلى، واستأنفه عند التمرير لأسفل
- عرض مؤشرات حالة الكتابة تلقائيًا

## استخدم العرض التوضيحي

```html
<c-vs>
  <c-md-stream id="md-view"></c-md-stream>
</c-vs>

<script type="module">
  import "webc.site/MdStream.js";

  const el = document.getElementById("md-view"),
    sleep = (ms) => new Promise((resolve) => setTimeout(resolve, ms));

  // عرض البيانات المتدفقة من خلال وظيفة المولد غير المتزامن التي تم إرجاعها
  el.gen = async function* () {
    العائد "# عنوان \ n"؛
    for (let i = 1; i <= 100; ++i) {
      await sleep(400);
      العائد "- العنصر" + i + "\n"؛
    }
  };
</script>
```

## ملكية

| اسم السمة | يكتب | يوضح |
| :--- | :--- | :--- |
| `gen` | `Function` | دالة تقوم بإرجاع مكرر/مولد غير متزامن، ومسح المحتوى وإعادة العرض بعد الإعداد |

## أسلوب

| فئة/علامة | يوضح |
| :--- | :--- |
| `Md` | تتم إضافة الفئة تلقائيًا إلى المكون عند التركيب، وضبط نمط Markdown |
| `i.T` | تمت إضافة مؤشر حالة الكتابة تلقائيًا في النهاية أثناء إدخال البيانات |
