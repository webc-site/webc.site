# மார்க் டவுன் உரையின் ஸ்ட்ரீமிங் ரெண்டரிங்

- பெருகிய முறையில் பாகுபடுத்தி மார்க் டவுனை ரெண்டர் செய்யவும்
- உள்ளடக்கம் திரையை மீறும் போது தானாக உருட்ட ஸ்க்ரோலிங் பெற்றோர் உறுப்புடன் ஒத்துழைக்கவும்
- கிளிக் செய்யும் போது அல்லது மேலே ஸ்க்ரோல் செய்யும் போது ஆட்டோஸ்க்ரோலை இடைநிறுத்தவும், கீழே ஸ்க்ரோல் செய்யும் போது மீண்டும் தொடரவும்
- தட்டச்சு நிலை குறிகாட்டிகளைத் தானாகக் காண்பிக்கும்

## டெமோவைப் பயன்படுத்தவும்

```html
<c-vs>
  <c-md-stream id="md-view"></c-md-stream>
</c-vs>

<script type="module">
  import "webc.site/MdStream/_.js";

  const el = document.getElementById("md-view"),
    sleep = (ms) => new Promise((resolve) => setTimeout(resolve, ms));

  // திரும்பிய ஒத்திசைவற்ற ஜெனரேட்டர் செயல்பாட்டின் மூலம் ஸ்ட்ரீமிங் டேட்டா ரெண்டரிங்
  el.gen = async function* () {
    "# தலைப்பு\n";
    for (let i = 1; i <= 100; ++i) {
      await sleep(400);
      விளைச்சல் "- உருப்படி " + i + "\n";
    }
  };
</script>
```

## சொத்து

| பண்பு பெயர் | வகை | விளக்குகின்றன |
| :--- | :--- | :--- |
| `gen` | `Function` | ஒத்திசைவற்ற இட்டேட்டர்/ஜெனரேட்டரை வழங்கும் செயல்பாடு, உள்ளடக்கத்தை அழிக்கிறது மற்றும் அமைத்த பிறகு மீண்டும் ரெண்டர் செய்கிறது |

## பாணி

| வகுப்பு/குறிச்சொல் | விளக்குகின்றன |
| :--- | :--- |
| `Md` | மவுண்ட் செய்யும் போது கிளாஸ் தானாக பாகத்தில் சேர்க்கப்படும், மார்க் டவுன் ஸ்டைலை அமைக்கவும் |
| `i.T` | தரவு உள்ளீட்டின் போது தட்டச்சு நிலை காட்டி தானாகவே இறுதியில் சேர்க்கப்படும் |
