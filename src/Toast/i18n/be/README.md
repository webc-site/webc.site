# Усплывальнае акно падказкі на краі старонкі

- **Выклік функцыі**: Забяспечвае звычайны інтэрфейс і інтэрфейс паведамлення аб памылках
- **Аўтаматычны стос**: Некалькі падказак аўтаматычна разлічваюць інтэрвал і складаюць вертыкальна
- **Запланаванае і ручное выключэнне**: падтрымлівае запланаванае і ручное выключэнне

## Выкарыстоўвайце дэма

```js
import { toast, toastErr } from "webc.site/js/Toast.js";

toast((el) => {
  el.textContent = "Аператыўны змест";
});

toastErr((el) => {
  el.textContent = "Змест памылкі";
});

toast(
  (el) => {
    el.textContent = "Аўтаматычна закрываць праз 99 секунд";
  },
  99
);
```

## Параметры інтэрфейсу

### toast(render, timeout)

Стварэнне і адлюстраванне падказак.

- `render`: `(el) => void`, візуалізацыя зваротнага выкліку, `el` - элемент поля падказкі.
- `timeout`: `Number`, тайм-аўт у секундах. Па змаўчанні 9. Усталюйце 0, каб не выключацца аўтаматычна.
- Вяртанае значэнне: элемент поля падказкі. Падтрымлівае `el.close()` выкл.

### toastErr(render, timeout)

Стварэнне і адлюстраванне паведамленняў пра памылкі. Параметры такія ж, як `toast`, з класам стылю `.ERR`.

## клас стылю

- `.Toast`: базавы клас поля падказкі.
- `.ERR`: Статус памылкі.
- `.animated` / `.fadeInR` / `.fadeOutR`: аніміраваны пераход.
- `.x`: Кнопка закрыцця.
