# صفحي جي ڪنڊ تي پاپ اپ ونڊو کي فوري طور تي

- ** فنڪشن ڪال **: مهيا ڪري ٿو عام ۽ غلطي فوري انٽرفيس
- **آٽو اسٽيڪ**: گھڻا ٽوٽڪا پاڻمرادو فاصلي کي ڳڻيو ۽ عمودي طور تي اسٽيڪ ڪريو
- ** شيڊول ٿيل ۽ دستي بند ڪرڻ **: شيڊول ۽ دستي بندش کي سپورٽ ڪري ٿو

## ڊيمو استعمال ڪريو

```js
import { toast, toastErr } from "webc.site/js/Toast.js";

toast((el) => {
  el.textContent = "فوري مواد"؛
});

toastErr((el) => {
  el.textContent = "غلطي وارو مواد"؛
});

toast(
  (el) => {
    el.textContent = "99 سيڪنڊن کان پوءِ پاڻمرادو بند"؛
  },
  99
);
```

## انٽرفيس پيٽرول

### toast(render, timeout)

ٺاھيو ۽ ڏيکاريو اشارو.

- `render`: `(el) => void`، رينڊرنگ ڪال بڪ، `el` پرامٽ باڪس عنصر آهي.
- `timeout`: `Number`، ٽائيم ختم سيڪنڊ. ڊفالٽ 9. خودڪار طور تي بند نه ڪرڻ لاء 0 تي سيٽ ڪريو.
- واپسي جي قيمت: پرامپٹ باڪس عنصر. سپورٽ `el.close()` بند.

### toastErr(render, timeout)

غلطي پيغام ٺاھيو ۽ ڏيکاريو. پيرا ميٽرس ساڳيا آهن `toast`، `.ERR` طرز طبقي سان.

## انداز ڪلاس

- `.Toast`: پرامپٹ باڪس بيس ڪلاس.
- `.ERR`: نقص جي حالت.
- `.animated` / `.fadeInR` / `.fadeOutR`: متحرڪ منتقلي.
- `.x`: بند ڪريو بٽڻ.
