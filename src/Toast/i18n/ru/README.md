# Всплывающее окно подсказки на краю страницы

- **Вызов функции**: обеспечивает нормальный интерфейс и интерфейс подсказки об ошибке.
- **Автоматическое сложение**: несколько насадок автоматически рассчитывают расстояние и складываются вертикально.
- **Запланированное и ручное выключение**: поддерживается запланированное и ручное выключение.

## Используйте демо-версию

```js
import { toast, toastErr } from "webc.site/js/Toast.js";

toast((el) => {
  el.textContent = "Содержимое запроса";
});

toastErr((el) => {
  el.textContent = "Содержимое ошибки";
});

toast(
  (el) => {
    el.textContent = "Автоматически закрывается через 99 секунд";
  },
  99
);
```

## Параметры интерфейса

### toast(render, timeout)

Создание и отображение подсказок.

- `render`: `(el) => void`, обратный вызов рендеринга, `el` — элемент поля подсказки.
- `timeout`: `Number`, тайм-аут в секундах. По умолчанию 9. Установите значение 0, чтобы не отключать автоматическое выключение.
- Возвращаемое значение: элемент поля подсказки. Поддерживает `el.close()` выключение.

### toastErr(render, timeout)

Создание и отображение сообщений об ошибках. Параметры такие же, как у `toast`, с классом стиля `.ERR`.

## класс стиля

- `.Toast`: базовый класс окна подсказки.
- `.ERR`: статус ошибки.
- `.animated` / `.fadeInR` / `.fadeOutR`: анимированный переход.
- `.x`: кнопка закрытия.
