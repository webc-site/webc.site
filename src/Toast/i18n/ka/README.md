# მოთხოვნის ამომხტარი ფანჯარა გვერდის კიდეზე

- **ფუნქციის გამოძახება**: უზრუნველყოფს ნორმალურ და შეცდომის მოთხოვნის ინტერფეისებს
- **ავტომატური დასტა**: მრავალი რჩევა ავტომატურად ითვლის მანძილს და დაწყობს ვერტიკალურად
- ** დაგეგმილი და ხელით გამორთვა **: მხარს უჭერს დაგეგმილ და ხელით გამორთვას

## გამოიყენეთ დემო ვერსია

```js
import { toast, toastErr } from "webc.site/js/Toast.js";

toast((el) => {
  el.textContent = "მოთხოვნის შინაარსი";
});

toastErr((el) => {
  el.textContent = "შეცდომის შინაარსი";
});

toast(
  (el) => {
    el.textContent = "ავტომატური დახურვა 99 წამის შემდეგ";
  },
  99
);
```

## ინტერფეისის პარამეტრები

### toast(render, timeout)

მოთხოვნის შექმნა და ჩვენება.

- `render`: `(el) => void`, გამოძახების რენდერი, `el` არის მოთხოვნის ველი ელემენტი.
- `timeout`: `Number`, ვადის ამოწურვა წამი. ნაგულისხმევი 9. დააყენეთ 0-ზე, რათა ავტომატურად არ დაიხუროს.
- დაბრუნების მნიშვნელობა: მოთხოვნის ყუთის ელემენტი. მხარს უჭერს `el.close()` გამორთვას.

### toastErr(render, timeout)

შეცდომის შეტყობინებების შექმნა და ჩვენება. პარამეტრები იგივეა, რაც `toast`, `.ERR` სტილის კლასით.

## სტილის კლასი

- `.Toast`: მოთხოვნის ყუთის საბაზისო კლასი.
- `.ERR`: შეცდომის სტატუსი.
- `.animated` / `.fadeInR` / `.fadeOutR`: ანიმაციური გადასვლა.
- `.x`: დახურვის ღილაკი.
