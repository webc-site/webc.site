# Изскачащ прозорец с подкана в края на страницата

- **Извикване на функция**: Осигурява нормален интерфейс и интерфейс за подкана за грешка
- **Автоматично подреждане**: Няколко съвета автоматично изчисляват разстоянието и подреждат вертикално
- **Планирано и ръчно изключване**: Поддържа планирано и ръчно изключване

## Използвайте демонстрацията

```js
import { toast, toastErr } from "webc.site/js/Toast.js";

toast((el) => {
  el.textContent = "Подсказващо съдържание";
});

toastErr((el) => {
  el.textContent = "Съдържание на грешка";
});

toast(
  (el) => {
    el.textContent = "Автоматично затваряне след 99 секунди";
  },
  99
);
```

## Параметри на интерфейса

### toast(render, timeout)

Създавайте и показвайте подкани.

- `render`: `(el) => void`, изобразяване на обратно извикване, `el` е елементът на полето за подкана.
- `timeout`: `Number`, изчакване секунди. По подразбиране 9. Задайте 0, за да не се изключва автоматично.
- Върната стойност: елемент на полето за подкана. Поддържа `el.close()` изключено.

### toastErr(render, timeout)

Създаване и показване на съобщения за грешки. Параметрите са същите като `toast`, с `.ERR` стил клас.

## стил клас

- `.Toast`: Базов клас на кутия за подкани.
- `.ERR`: Състояние на грешка.
- `.animated` / `.fadeInR` / `.fadeOutR`: анимиран преход.
- `.x`: Бутон за затваряне.
