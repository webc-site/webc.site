# نافذة منبثقة موجهة على حافة الصفحة

- **استدعاء الوظيفة**: يوفر واجهات عادية وواجهات موجهة للأخطاء
- **المكدس التلقائي**: تقوم النصائح المتعددة تلقائيًا بحساب التباعد والتكديس عموديًا
- **إيقاف التشغيل المجدول واليدوي**: يدعم إيقاف التشغيل المجدول واليدوي

## استخدم العرض التوضيحي

```js
import { toast, toastErr } from "webc.site/js/Toast.js";

toast((el) => {
  el.textContent = "محتوى موجه";
});

toastErr((el) => {
  el.textContent = "محتوى الخطأ"؛
});

toast(
  (el) => {
    el.textContent = "الإغلاق تلقائيًا بعد 99 ثانية";
  },
  99
);
```

## معلمات الواجهة

### toast(render, timeout)

إنشاء وعرض المطالبات.

- `render`: `(el) => void`، عرض رد الاتصال، `el` هو عنصر مربع المطالبة.
- `timeout`: `Number`، المهلة بالثواني. الافتراضي 9. اضبط على 0 لعدم إيقاف التشغيل تلقائيًا.
- قيمة الإرجاع: عنصر مربع المطالبة. يدعم إيقاف `el.close()`.

### toastErr(render, timeout)

إنشاء وعرض رسائل الخطأ. المعلمات هي نفسها `toast`، مع فئة النمط `.ERR`.

## فئة النمط

- `.Toast`: الفئة الأساسية لمربع المطالبة.
- `.ERR`: حالة الخطأ.
- `.animated` / `.fadeInR` / `.fadeOutR`: انتقال متحرك.
- `.x`: زر الإغلاق.
