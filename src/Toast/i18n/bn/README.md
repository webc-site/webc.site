# পৃষ্ঠার প্রান্তে প্রম্পট পপ-আপ উইন্ডো

- **ফাংশন কল**: স্বাভাবিক এবং ত্রুটি প্রম্পট ইন্টারফেস প্রদান করে
- **অটো স্ট্যাক**: একাধিক টিপ স্বয়ংক্রিয়ভাবে ব্যবধান গণনা করে এবং উল্লম্বভাবে স্ট্যাক করে
- **নির্ধারিত এবং ম্যানুয়াল শাটডাউন**: নির্ধারিত এবং ম্যানুয়াল শাটডাউন সমর্থন করে

## ডেমো ব্যবহার করুন

```js
import { toast, toastErr } from "webc.site/js/Toast.js";

toast((el) => {
  el.textContent = "প্রম্পট কন্টেন্ট";
});

toastErr((el) => {
  el.textContent = "ত্রুটি বিষয়বস্তু";
});

toast(
  (el) => {
    el.textContent = "99 সেকেন্ড পরে স্বয়ংক্রিয়ভাবে বন্ধ";
  },
  99
);
```

## ইন্টারফেস পরামিতি

### toast(render, timeout)

প্রম্পট তৈরি করুন এবং প্রদর্শন করুন।

- `render`: `(el) => void`, রেন্ডারিং কলব্যাক, `el` হল প্রম্পট বক্স উপাদান৷
- `timeout`: `Number`, সময় শেষ সেকেন্ড। ডিফল্ট 9. স্বয়ংক্রিয়ভাবে বন্ধ না হওয়ার জন্য 0 এ সেট করুন।
- রিটার্ন মান: প্রম্পট বক্স উপাদান। `el.close()` বন্ধ সমর্থন করে।

### toastErr(render, timeout)

ত্রুটি বার্তা তৈরি এবং প্রদর্শন. প্যারামিটারগুলি `toast` এর মতই, `.ERR` শৈলী ক্লাস সহ।

## শৈলী ক্লাস

- `.Toast`: প্রম্পট বক্স বেস ক্লাস।
- `.ERR`: ত্রুটি স্থিতি।
- `.animated` / `.fadeInR` / `.fadeOutR`: অ্যানিমেটেড ট্রানজিশন৷
- `.x`: বন্ধ করুন বোতাম।
