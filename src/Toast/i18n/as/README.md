# পৃষ্ঠাৰ প্ৰান্তত পপ-আপ উইণ্ড' প্ৰমপ্ট কৰক

- **ফাংচন কল**: সাধাৰণ আৰু ভুল প্ৰমপ্ট আন্তঃপৃষ্ঠসমূহ প্ৰদান কৰে
- **স্বয়ংক্ৰিয় ষ্টেক**: একাধিক টিপে স্বয়ংক্ৰিয়ভাৱে ব্যৱধান গণনা কৰে আৰু উলম্বভাৱে ষ্টেক কৰে
- **নিৰ্ধাৰিত আৰু হস্তচালিত বন্ধ**: নিৰ্ধাৰিত আৰু হস্তচালিত বন্ধ সমৰ্থন কৰে

## ডেমো ব্যৱহাৰ কৰক

```js
import { toast, toastErr } from "webc.site/js/Toast.js";

toast((el) => {
  el.textContent = "প্ৰমপ্ট বিষয়বস্তু";
});

toastErr((el) => {
  el.textContent = "ত্ৰুটিৰ বিষয়বস্তু";
});

toast(
  (el) => {
    el.textContent = "৯৯ ছেকেণ্ডৰ পিছত স্বয়ংক্ৰিয়ভাৱে বন্ধ";
  },
  99
);
```

## আন্তঃপৃষ্ঠ প্ৰাচলসমূহ

### toast(render, timeout)

প্ৰমপ্ট সৃষ্টি আৰু প্ৰদৰ্শন কৰক।

- `render`: `(el) => void`, কলবেক ৰেণ্ডাৰ কৰা, `el` হৈছে প্ৰমপ্ট বাকচ উপাদান।
- `timeout`: `Number`, সময়সীমা চেকেণ্ড। অবিকল্পিত 9. স্বয়ংক্ৰিয়ভাৱে বন্ধ নহ'বলৈ 0 লে সংহতি কৰক।
- ৰিটাৰ্ণ মান: প্ৰমপ্ট বাকচ উপাদান। `el.close()` বন্ধ সমৰ্থন কৰে।

### toastErr(render, timeout)

ভুল বাৰ্তা সৃষ্টি আৰু প্ৰদৰ্শন কৰক। প্ৰাচলসমূহ `toast`ৰ সৈতে একে, `.ERR` শৈলী শ্ৰেণীৰ সৈতে।

## শৈলী শ্ৰেণী

- `.Toast`: প্ৰমপ্ট বাকচ ভিত্তি শ্ৰেণী।
- `.ERR`: ত্ৰুটিৰ অৱস্থা।
- `.animated` / `.fadeInR` / `.fadeOutR`: এনিমেটেড পৰিৱৰ্তন।
- `.x`: বন্ধ বুটাম।
