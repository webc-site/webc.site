# पृष्ठाच्या काठावर प्रॉम्प्ट पॉप-अप विंडो

- **फंक्शन कॉल**: सामान्य आणि त्रुटी प्रॉम्प्ट इंटरफेस प्रदान करते
- **ऑटो स्टॅक**: एकाधिक टिपा स्वयंचलितपणे अंतराची गणना करतात आणि अनुलंब स्टॅक करतात
- **अनुसूचित आणि मॅन्युअल शटडाउन**: अनुसूचित आणि मॅन्युअल शटडाउनला समर्थन देते

## डेमो वापरा

```js
import { toast, toastErr } from "webc.site/js/Toast.js";

toast((el) => {
  el.textContent = "प्रॉम्प्ट सामग्री";
});

toastErr((el) => {
  el.textContent = "त्रुटी सामग्री";
});

toast(
  (el) => {
    el.textContent = "99 सेकंदांनंतर स्वयंचलितपणे बंद करा";
  },
  99
);
```

## इंटरफेस पॅरामीटर्स

### toast(render, timeout)

प्रॉम्प्ट तयार करा आणि प्रदर्शित करा.

- `render`: `(el) => void`, रेंडरिंग कॉलबॅक, `el` हा प्रॉम्प्ट बॉक्स घटक आहे.
- `timeout`: `Number`, कालबाह्य सेकंद. डीफॉल्ट 9. स्वयंचलितपणे बंद न होण्यासाठी 0 वर सेट करा.
- रिटर्न व्हॅल्यू: प्रॉम्प्ट बॉक्स घटक. सपोर्ट `el.close()` बंद.

### toastErr(render, timeout)

त्रुटी संदेश तयार करा आणि प्रदर्शित करा. पॅरामीटर्स `toast` सारखेच आहेत, `.ERR` शैली वर्गासह.

## शैली वर्ग

- `.Toast`: प्रॉम्प्ट बॉक्स बेस क्लास.
- `.ERR`: त्रुटी स्थिती.
- `.animated` / `.fadeInR` / `.fadeOutR`: ॲनिमेटेड संक्रमण.
- `.x`: बंद करा बटण.
