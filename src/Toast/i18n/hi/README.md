# पृष्ठ के किनारे पर शीघ्र पॉप-अप विंडो

- **फ़ंक्शन कॉल**: सामान्य और त्रुटि शीघ्र इंटरफ़ेस प्रदान करता है
- **ऑटो स्टैक**: एकाधिक युक्तियाँ स्वचालित रूप से रिक्ति की गणना करती हैं और लंबवत रूप से स्टैक करती हैं
- **अनुसूचित और मैन्युअल शटडाउन**: अनुसूचित और मैन्युअल शटडाउन का समर्थन करता है

## डेमो का प्रयोग करें

```js
import { toast, toastErr } from "webc.site/js/Toast.js";

toast((el) => {
  el.textContent = "प्रॉम्प्ट सामग्री";
});

toastErr((el) => {
  el.textContent = "त्रुटि सामग्री";
});

toast(
  (el) => {
    el.textContent = "99 सेकंड के बाद स्वचालित रूप से बंद करें";
  },
  99
);
```

## इंटरफ़ेस पैरामीटर

### toast(render, timeout)

संकेत बनाएं और प्रदर्शित करें.

- `render`: `(el) => void`, कॉलबैक प्रस्तुत करते हुए, `el` प्रॉम्प्ट बॉक्स तत्व है।
- `timeout`: `Number`, टाइमआउट सेकंड। डिफ़ॉल्ट 9. स्वचालित रूप से बंद न होने के लिए 0 पर सेट करें।
- वापसी मान: प्रॉम्प्ट बॉक्स तत्व। `el.close()` को बंद करने का समर्थन करता है।

### toastErr(render, timeout)

त्रुटि संदेश बनाएं और प्रदर्शित करें. पैरामीटर `toast` के समान हैं, `.ERR` शैली वर्ग के साथ।

## शैली वर्ग

- `.Toast`: प्रॉम्प्ट बॉक्स बेस क्लास।
- `.ERR`: त्रुटि स्थिति.
- `.animated` / `.fadeInR` / `.fadeOutR`: एनिमेटेड संक्रमण।
- `.x`: बंद करें बटन।
