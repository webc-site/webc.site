# Հուշել թռուցիկ պատուհանը էջի եզրին

- **Ֆունկցիայի կանչ**. Ապահովում է նորմալ և սխալի արագ միջերեսներ
- **Auto stack**. Բազմաթիվ խորհուրդներ ավտոմատ կերպով հաշվարկում են տարածությունը և շարվում ուղղահայաց
- **Պլանավորված և ձեռքով անջատում**. աջակցում է պլանավորված և ձեռքով անջատումը

## Օգտագործեք ցուցադրությունը

```js
import { toast, toastErr } from "webc.site/js/Toast.js";

toast((el) => {
  el.textContent = «Հրաժեշտ բովանդակություն»;
});

toastErr((el) => {
  el.textContent = «Սխալի բովանդակություն»;
});

toast(
  (el) => {
    el.textContent = «Ավտոմատ փակել 99 վայրկյանից հետո»;
  },
  99
);
```

## Ինտերֆեյսի պարամետրեր

### toast(render, timeout)

Ստեղծեք և ցուցադրեք հուշումներ:

- `render`. `(el) => void`, ետ կանչելու համար, `el` հուշման տուփի տարրն է:
- `timeout`՝ `Number`, ժամանակի ավարտ վայրկյաններ: Կանխադրված 9. Սահմանեք 0, որպեսզի ինքնաբերաբար չփակվի:
- Վերադարձի արժեքը՝ հուշման տուփի տարր: Աջակցում է `el.close()` անջատված:

### toastErr(render, timeout)
