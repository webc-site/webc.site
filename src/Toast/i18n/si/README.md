# පිටුවේ කෙළවරේ ඇති ක්ෂණික උත්පතන කවුළුව

- **Function call**: සාමාන්‍ය සහ දෝෂ ඉක්මන් අතුරුමුහුණත් සපයයි
- **ස්වයං තොගය**: බහුවිධ ඉඟි ස්වයංක්‍රීයව පරතරය ගණනය කර සිරස් අතට අඳින්න
- ** කාලසටහන්ගත සහ අතින් වසා දැමීම**: කාලසටහන්ගත සහ අතින් වසා දැමීම සඳහා සහය දක්වයි

## Demo භාවිතා කරන්න

```js
import { toast, toastErr } from "webc.site/js/Toast.js";

toast((el) => {
  el.textContent = "ඉක්මන් අන්තර්ගතය";
});

toastErr((el) => {
  el.textContent = "දෝෂ අන්තර්ගතය";
});

toast(
  (el) => {
    el.textContent = "තත්පර 99කට පසු ස්වයංක්‍රීයව වසන්න";
  },
  99
);
```

## අතුරුමුහුණත් පරාමිතීන්

### toast(render, timeout)

විමසුම් සාදන්න සහ සංදර්ශන කරන්න.

- `render`: `(el) => void`, නැවත ඇමතුම ලබා දීම, `el` යනු ප්‍රේරක කොටු මූලද්‍රව්‍යය වේ.
- `timeout`: `Number`, කල් ඉකුත් වූ තත්පර. පෙරනිමිය 9. ස්වයංක්‍රීයව වසා නොදැමීමට 0 ලෙස සකසන්න.
- ප්‍රතිලාභ අගය: ප්‍රේරක කොටු මූලද්‍රව්‍යය. `el.close()` ක්‍රියා විරහිත කිරීමට සහය දක්වයි.

### toastErr(render, timeout)

දෝෂ පණිවිඩ සාදන්න සහ ප්රදර්ශනය කරන්න. පරාමිති `toast`, `.ERR` style class සමග සමාන වේ.

## ශෛලිය පන්තිය

- `.Toast`: Prompt box base class.
- `.ERR`: දෝෂ තත්ත්වය.
- `.animated` / `.fadeInR` / `.fadeOutR`: සජීවිකරණ සංක්‍රාන්තිය.
- `.x`: වසන්න බොත්තම.
