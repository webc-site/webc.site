# ପୃଷ୍ଠାର ଧାରରେ ତୁରନ୍ତ ପପ୍-ଅପ୍ ୱିଣ୍ଡୋ |

- ** ଫଙ୍କସନ୍ କଲ୍ **: ସାଧାରଣ ଏବଂ ତ୍ରୁଟି ପ୍ରମ୍ପ୍ଟ ଇଣ୍ଟରଫେସ୍ ପ୍ରଦାନ କରେ |
- ** ଅଟୋ ଷ୍ଟାକ **: ଏକାଧିକ ଟିପ୍ସ ସ୍ୱୟଂଚାଳିତ ଭାବରେ ବ୍ୟବଧାନ ଏବଂ ଷ୍ଟାକକୁ ଭୂଲମ୍ବ ଭାବରେ ଗଣନା କରେ |
- ** ଅନୁସୂଚିତ ଏବଂ ମାନୁଆଲ ବନ୍ଦ କରିବା **: ଅନୁସୂଚିତ ଏବଂ ମାନୁଆଲ ବନ୍ଦକୁ ସମର୍ଥନ କରେ |

## ଡେମୋ ବ୍ୟବହାର କରନ୍ତୁ |

```js
import { toast, toastErr } from "webc.site/js/Toast.js";

toast((el) => {
  el.textContent = "ତୁରନ୍ତ ବିଷୟବସ୍ତୁ";
});

toastErr((el) => {
  el.textContent = "ତ୍ରୁଟି ବିଷୟବସ୍ତୁ";
});

toast(
  (el) => {
    el.textContent = "99 ସେକେଣ୍ଡ ପରେ ସ୍ୱୟଂଚାଳିତ ଭାବରେ ବନ୍ଦ";
  },
  99
);
```

## ଇଣ୍ଟରଫେସ୍ ପାରାମିଟରଗୁଡିକ

### toast(render, timeout)

ପ୍ରମ୍ପ୍ଟ ସୃଷ୍ଟି ଏବଂ ପ୍ରଦର୍ଶନ କରନ୍ତୁ |

- `render`: `(el) => void`, କଲବ୍ୟାକ୍ ରେଣ୍ଡରିଂ, `el` ହେଉଛି ପ୍ରମ୍ପ୍ଟ ବକ୍ସ ଉପାଦାନ |
- `timeout`: `Number`, ସମୟ ସମାପ୍ତ ସେକେଣ୍ଡ | ଡିଫଲ୍ଟ 9। ସ୍ୱୟଂଚାଳିତ ଭାବରେ ବନ୍ଦ ନହେବା ପାଇଁ 0 କୁ ସେଟ୍ କରନ୍ତୁ |
- ଫେରସ୍ତ ମୂଲ୍ୟ: ପ୍ରମ୍ପ୍ଟ ବକ୍ସ ଉପାଦାନ | `el.close()` ବନ୍ଦକୁ ସମର୍ଥନ କରେ |

### toastErr(render, timeout)

ତ୍ରୁଟି ସନ୍ଦେଶ ସୃଷ୍ଟି ଏବଂ ପ୍ରଦର୍ଶନ କରନ୍ତୁ | ପାରାମିଟରଗୁଡିକ `toast` ସହିତ ସମାନ, `.ERR` ଶ style ଳୀ ଶ୍ରେଣୀ ସହିତ |

## ଶ style ଳୀ ଶ୍ରେଣୀ

- `.Toast`: ପ୍ରମ୍ପ୍ଟ ବକ୍ସ ବେସ୍ କ୍ଲାସ୍ |
- `.ERR`: ତ୍ରୁଟି ସ୍ଥିତି |
- `.animated` / `.fadeInR` / `.fadeOutR`: ଆନିମେଟେଡ୍ ସ୍ଥାନାନ୍ତରଣ |
- `.x`: ବଟନ୍ ବନ୍ଦ କରନ୍ତୁ |
