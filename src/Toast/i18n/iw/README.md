# חלון מוקפץ בקשה בקצה העמוד

- **שיחת ​​פונקציה**: מספקת ממשקים רגילים וממשקי שגיאה
- **ערימה אוטומטית**: טיפים מרובים מחשבים אוטומטית את המרווח והערימה אנכית
- **כיבוי מתוזמן וידני**: תומך בכיבוי מתוזמן וידני

## השתמש בהדגמה

```js
import { toast, toastErr } from "webc.site/js/Toast.js";

toast((el) => {
  el.textContent = "תוכן בקשה";
});

toastErr((el) => {
  el.textContent = "תוכן שגיאה";
});

toast(
  (el) => {
    el.textContent = "נסגר אוטומטית לאחר 99 שניות";
  },
  99
);
```

## פרמטרים של ממשק

### toast(render, timeout)

צור והצג הנחיות.

- `render`: `(el) => void`, עיבוד התקשרות חוזרת, `el` הוא רכיב ה-prompt box.
- `timeout`: `Number`, שניות זמן קצוב. ברירת מחדל 9. הגדר ל-0 כדי לא לכבות אוטומטית.
- ערך החזרה: רכיב תיבת הנחיה. תומך ב`el.close()` כבוי.

### toastErr(render, timeout)

צור והצג הודעות שגיאה. הפרמטרים זהים ל-`toast`, עם `.ERR` מחלקה בסגנון.

## שיעור סגנון

- `.Toast`: מחלקת הבסיס של תיבת הנחיה.
- `.ERR`: מצב שגיאה.
- `.animated` / `.fadeInR` / `.fadeOutR`: מעבר מונפש.
- `.x`: לחצן סגירה.
