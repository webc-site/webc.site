# पृष्ठको छेउमा पप-अप विन्डो प्रम्प्ट गर्नुहोस्

- ** प्रकार्य कल **: सामान्य र त्रुटि प्रम्प्ट इन्टरफेस प्रदान गर्दछ
- **अटो स्ट्याक**: धेरै सुझावहरूले स्वचालित रूपमा स्पेसिङ गणना गर्दछ र ठाडो रूपमा स्ट्याक गर्दछ
- **अनुसूचित र म्यानुअल बन्द **: अनुसूचित र म्यानुअल बन्द समर्थन गर्दछ

## डेमो प्रयोग गर्नुहोस्

```js
import { toast, toastErr } from "webc.site/js/Toast.js";

toast((el) => {
  el.textContent = "प्रम्प्ट सामग्री";
});

toastErr((el) => {
  el.textContent = "त्रुटि सामग्री";
});

toast(
  (el) => {
    el.textContent = "99 सेकेन्ड पछि स्वचालित रूपमा बन्द";
  },
  99
);
```

## इन्टरफेस प्यारामिटरहरू

### toast(render, timeout)

प्रम्प्टहरू सिर्जना र प्रदर्शन गर्नुहोस्।

- `render`: `(el) => void`, रेन्डरिङ कलब्याक, `el` प्रोम्प्ट बक्स तत्व हो।
- `timeout`: `Number`, टाइमआउट सेकेन्ड। पूर्वनिर्धारित ९. स्वचालित रूपमा बन्द नहुनको लागि ० मा सेट गर्नुहोस्।
- फिर्ता मान: प्रम्प्ट बक्स तत्व। `el.close()` बन्द समर्थन गर्दछ।

### toastErr(render, timeout)

त्रुटि सन्देशहरू सिर्जना र प्रदर्शन गर्नुहोस्। प्यारामिटरहरू `toast` जस्तै हुन्, `.ERR` शैली वर्गसँग।

## शैली वर्ग

- `.Toast`: प्रम्प्ट बक्स आधार वर्ग।
- `.ERR`: त्रुटि स्थिति।
- `.animated` / `.fadeInR` / `.fadeOutR`: एनिमेटेड संक्रमण।
- `.x`: बन्द गर्नुहोस् बटन।
