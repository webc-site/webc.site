# ページの端にプロンプ​​ト ポップアップ ウィンドウを表示する

- **関数呼び出し**: 通常およびエラー プロンプト インターフェイスを提供します
- **自動スタック**: 複数のチップが自動的に間隔を計算し、垂直方向にスタックします。
- **スケジュールされたシャットダウンと手動シャットダウン**: スケジュールされたシャットダウンと手動シャットダウンをサポートします

## デモを使用する

```js
import { toast, toastErr } from "webc.site/js/Toast.js";

toast((el) => {
  el.textContent = "プロンプトコンテンツ";
});

toastErr((el) => {
  el.textContent = "エラー内容";
});

toast(
  (el) => {
    el.textContent = "99 秒後に自動的に終了";
  },
  99
);
```

## インターフェースパラメータ

### toast(render, timeout)

プロンプトを作成して表示します。

- `render`: `(el) => void`、レンダリング コールバック、`el` はプロンプト ボックス要素です。
- `timeout`: `Number`、タイムアウト秒。デフォルトは 9。自動的にシャットダウンしない場合は 0 に設定します。
- 戻り値: プロンプト ボックス要素。オフ`el.close()`をサポートします。

### toastErr(render, timeout)

エラーメッセージを作成して表示します。パラメータは `toast` と同じで、 `.ERR` スタ​​イル クラスが使用されます。

## スタイルクラス

- `.Toast`: プロンプト ボックスの基本クラス。
- `.ERR`: エラーステータス。
- `.animated` / `.fadeInR` / `.fadeOutR`: アニメーションによるトランジション。
- `.x`: 閉じるボタン。
