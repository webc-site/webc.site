# Беттің шетіндегі қалқымалы терезе

- **Функция шақыруы**: Қалыпты және қате туралы хабарлау интерфейстерін қамтамасыз етеді
- **Авто стек**: Бірнеше кеңестер аралық пен стекті тігінен автоматты түрде есептейді
- **Жоспарлы және қолмен өшіру**: Жоспарланған және қолмен өшіруді қолдайды

## Демонстрацияны пайдаланыңыз

```js
import { toast, toastErr } from "webc.site/js/Toast.js";

toast((el) => {
  el.textContent = "Шұғыл мазмұн";
});

toastErr((el) => {
  el.textContent = "Қате мазмұны";
});

toast(
  (el) => {
    el.textContent = "99 секундтан кейін автоматты түрде жабу";
  },
  99
);
```

## Интерфейс параметрлері

### toast(render, timeout)

Сұрауларды жасау және көрсету.

- `render`: `(el) => void`, кері шақыруды көрсету, `el` - шақыру терезесінің элементі.
- `timeout`: `Number`, күту уақыты секундтары. Әдепкі 9. Автоматты түрде өшпеу үшін 0 мәніне орнатыңыз.
- Қайтарылатын мән: шақыру жолағы элементі. `el.close()` өшіруді қолдайды.

### toastErr(render, timeout)

Қате туралы хабарларды жасаңыз және көрсетіңіз. Параметрлер `toast`, `.ERR` стиль сыныбымен бірдей.

## стиль сыныбы

- `.Toast`: Шақыру терезесінің негізгі класы.
- `.ERR`: қате күйі.
- `.animated` / `.fadeInR` / `.fadeOutR`: анимациялық көшу.
- `.x`: жабу түймесі.
