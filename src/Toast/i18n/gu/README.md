# પૃષ્ઠની ધાર પર પોપ-અપ વિન્ડોને પ્રોમ્પ્ટ કરો

- **ફંક્શન કૉલ**: સામાન્ય અને ભૂલ પ્રોમ્પ્ટ ઇન્ટરફેસ પ્રદાન કરે છે
- **ઓટો સ્ટેક**: બહુવિધ ટીપ્સ આપમેળે અંતરની ગણતરી કરે છે અને ઊભી રીતે સ્ટેક કરે છે
- **સુનિશ્ચિત અને મેન્યુઅલ શટડાઉન**: સુનિશ્ચિત અને મેન્યુઅલ શટડાઉનને સપોર્ટ કરે છે

## ડેમોનો ઉપયોગ કરો

```js
import { toast, toastErr } from "webc.site/js/Toast.js";

toast((el) => {
  el.textContent = "પ્રોમ્પ્ટ સામગ્રી";
});

toastErr((el) => {
  el.textContent = "ભૂલ સામગ્રી";
});

toast(
  (el) => {
    el.textContent = "99 સેકન્ડ પછી આપોઆપ બંધ";
  },
  99
);
```

## ઇન્ટરફેસ પરિમાણો

### toast(render, timeout)

પ્રોમ્પ્ટ્સ બનાવો અને પ્રદર્શિત કરો.

- `render`: `(el) => void`, રેન્ડરિંગ કૉલબેક, `el` એ પ્રોમ્પ્ટ બોક્સ ઘટક છે.
- `timeout`: `Number`, સમય સમાપ્તિ સેકંડ. ડિફૉલ્ટ 9. આપમેળે બંધ ન થવા માટે 0 પર સેટ કરો.
- વળતર મૂલ્ય: પ્રોમ્પ્ટ બોક્સ ઘટક. `el.close()` બંધને સપોર્ટ કરે છે.

### toastErr(render, timeout)

ભૂલ સંદેશાઓ બનાવો અને પ્રદર્શિત કરો. પેરામીટર્સ `toast` જેવા જ છે, `.ERR` શૈલી વર્ગ સાથે.

## શૈલી વર્ગ

- `.Toast`: પ્રોમ્પ્ટ બોક્સ બેઝ ક્લાસ.
- `.ERR`: ભૂલ સ્થિતિ.
- `.animated` / `.fadeInR` / `.fadeOutR`: એનિમેટેડ સંક્રમણ.
- `.x`: બંધ કરો બટન.
