# စာမျက်နှာ၏အစွန်းရှိ ပေါ်လာသော ဝင်းဒိုးကို အချက်ပြပါ။

- **Function call**- ပုံမှန်နှင့် error prompt interface များကို ပေးသည်။
- **Auto stack**- အကြံပြုချက်များစွာသည် အကွာအဝေးကို အလိုအလျောက် တွက်ချက်ပြီး ဒေါင်လိုက် အစုအဝေးကို တွက်ချက်သည်။
- **အချိန်ဇယားဆွဲထားပြီး လူကိုယ်တိုင် ပိတ်ခြင်း**- စီစဉ်ထားပြီး လူကိုယ်တိုင် ပိတ်ခြင်းကို ပံ့ပိုးပေးသည်။

## ဒီမိုကိုသုံးပါ။

```js
import { toast, toastErr } from "webc.site/js/Toast.js";

toast((el) => {
  el.textContent = "အမှာစာ အကြောင်းအရာ";
});

toastErr((el) => {
  el.textContent = "အမှား အကြောင်းအရာ";
});

toast(
  (el) => {
    el.textContent = "99 စက္ကန့်အကြာတွင် အလိုအလျောက် ပိတ်သည်";
  },
  99
);
```

## မျက်နှာပြင်ဘောင်များ

### toast(render, timeout)

အကြံပြုချက်များကိုဖန်တီးပြီးပြသပါ။

- `render`: `(el) => void`၊ ပြန်ခေါ်ခြင်းကို တင်ဆက်နေသည်၊ `el` သည် အချက်ပြဘောက်စ် အစိတ်အပိုင်း ဖြစ်သည်။
- `timeout`: `Number`၊ စက္ကန့်ကုန်သွားပါပြီ။ ပုံသေ 9. အလိုအလျောက်ပိတ်ခြင်းမပြုရန် 0 ကို သတ်မှတ်ပါ။
- ပြန်ပို့သည့်တန်ဖိုး- အချက်ပြဘောက်စ် အစိတ်အပိုင်း။ `el.close()` ပိတ်ခြင်းကို ပံ့ပိုးသည်။

### toastErr(render, timeout)

အမှားအယွင်း မက်ဆေ့ချ်များကို ဖန်တီးပြီး ပြသပါ။ ကန့်သတ်ချက်များသည် `toast`စတိုင်အတန်းနှင့် တူညီပါသည်။

## စတိုင်အတန်း

- `.Toast`- အကွက်အကွက် အခြေခံ အတန်းအစား။
- `.ERR`- အမှားအယွင်းအခြေအနေ။
- `.animated` / `.fadeInR` / `.fadeOutR`- ကာတွန်းအကူးအပြောင်း။
- `.x`- ခလုတ်ကို ပိတ်ပါ။
