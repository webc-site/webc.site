# Спливаюче вікно підказки на краю сторінки

- **Виклик функції**: забезпечує звичайний інтерфейс і інтерфейс повідомлення про помилку
- **Автоматичне складання**: кілька підказок автоматично обчислюють відстань і складають вертикально
- **Вимкнення за розкладом і вручну**: підтримує вимкнення за розкладом і вручну

## Використовуйте демо

```js
import { toast, toastErr } from "webc.site/js/Toast.js";

toast((el) => {
  el.textContent = "Підказувати вміст";
});

toastErr((el) => {
  el.textContent = "Вміст помилки";
});

toast(
  (el) => {
    el.textContent = "Автоматично закривати через 99 секунд";
  },
  99
);
```

## Параметри інтерфейсу

### toast(render, timeout)

Створення та відображення підказок.

- `render`: `(el) => void`, рендеринг зворотного виклику, `el` є елементом вікна запиту.
- `timeout`: `Number`, час очікування секунд. За замовчуванням 9. Встановіть 0, щоб не вимикати автоматично.
- Повернене значення: елемент вікна запиту. Підтримує `el.close()` off.

### toastErr(render, timeout)

Створення та відображення повідомлень про помилки. Параметри такі самі, як `toast`, з класом стилю `.ERR`.

## клас стилю

- `.Toast`: базовий клас вікна запиту.
- `.ERR`: Статус помилки.
- `.animated` / `.fadeInR` / `.fadeOutR`: анімований перехід.
- `.x`: кнопка закриття.
