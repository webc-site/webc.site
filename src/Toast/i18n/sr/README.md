# Промпт искачући прозор на ивици странице

- **Позив функције**: Пружа нормалне интерфејсе и интерфејсе за брзе грешке
- **Аутоматско слагање**: Више савета аутоматски израчунава размак и наслага вертикално
- **Планирано и ручно искључивање**: Подржава заказано и ручно искључивање

## Користите демо

```js
import { toast, toastErr } from "webc.site/js/Toast.js";

toast((el) => {
  ел.тектЦонтент = "Садржај упита";
});

toastErr((el) => {
  ел.тектЦонтент = "Садржај грешке";
});

toast(
  (el) => {
    ел.тектЦонтент = "Аутоматски затвори након 99 секунди";
  },
  99
);
```

## Параметри интерфејса

### toast(render, timeout)

Креирајте и прикажите упите.

- <б ид="1">: <б ид="2">, рендеровање повратног позива, <б ид="3"> је елемент оквира за промпт.
- <б ид="1">: <б ид="2">, временско ограничење у секундама. Подразумевано 9. Поставите на 0 да се не искључи аутоматски.
- Повратна вредност: елемент оквира за промпт. Подржава <б ид="1"> искључено.

### toastErr(render, timeout)

Креирајте и прикажите поруке о грешкама. Параметри су исти као <б ид="1">, са <б ид="2"> класом стила.

## стилска класа

- <б ид="1">: Основна класа оквира упита.
- <б ид="1">: Статус грешке.
- <б ид="1"> / <б ид="2"> / <б ид="3">: анимирани прелаз.
- <б ид="1">: Дугме за затварање.
