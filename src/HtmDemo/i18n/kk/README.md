# HTML және JS кодын желіде жөндеу

- Кодтың астында жөндеу жолағын көрсетіңіз және CodePen және JSFiddle жөндеу порталдарын қамтамасыз етіңіз
- Таңдалған жөндеу платформасын автоматты түрде есте сақтаңыз
- HTML, CSS және JS кодын динамикалық түрде жасау үшін `.gen` атрибуты арқылы кері қоңырауларды енгізуді қолдайды.
- `language-js` сынып аты бар еншілес элементтерді автоматты түрде анықтаңыз және JS режимінде іске қосыңыз

## Демонстрацияны пайдаланыңыз

```html
<c-htm-demo><pre><code class="language-html">
<h1>Демо</h1>
</code></pre></c-htm-demo>

<script type="module">
import "webc.site/HtmDemo/_.js";

const el = document.querySelector("c-htm-demo");
el.gen = (type, text) => {
  // Қайтару [html, css, js]
  return [
    text,
    "h1 { color: red; }",
    "console.log(1)"
  ];
};
</script>
```

## Интерфейс параметрлері

### Қасиеттер мен әдістер

- `.gen`: енгізілген кері шақыру функциясы. **Бұл кері шақыру отладтау жолағын көрсету үшін енгізілуі керек. **
  - параметр:
    - `type`: енгізілген мазмұн түрі (`HTML = 1`, `JS = 2`).
    - `text`: еншілес элементтен алынған код мәтіні.
  - Қайтарылатын мән: `[htm, css, js]`
