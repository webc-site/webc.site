# ఆన్‌లైన్‌లో HTML మరియు JS కోడ్‌లను డీబగ్గింగ్ చేస్తోంది

- కోడ్ క్రింద డీబగ్గింగ్ బార్‌ను రెండర్ చేయండి మరియు కోడ్‌పెన్ మరియు JSFiddle డీబగ్గింగ్ పోర్టల్‌లను అందించండి
- ఎంచుకున్న డీబగ్గింగ్ ప్లాట్‌ఫారమ్‌ను స్వయంచాలకంగా గుర్తుంచుకోండి
- HTML, CSS మరియు JS కోడ్‌ను డైనమిక్‌గా రూపొందించడానికి `.gen` లక్షణం ద్వారా కాల్‌బ్యాక్‌లను ఇంజెక్ట్ చేయడానికి మద్దతు ఇస్తుంది
- `language-js` తరగతి పేరుతో చైల్డ్ ఎలిమెంట్‌లను స్వయంచాలకంగా గుర్తించి, JS మోడ్‌లో అమలు చేయండి

## డెమో ఉపయోగించండి

```html
<c-htm-demo><pre><code class="language-html">
<h1>డెమో</h1>
</code></pre></c-htm-demo>

<script type="module">
import "webc.site/HtmDemo.js";

const el = document.querySelector("c-htm-demo");
el.gen = (type, text) => {
  // రిటర్న్ [html, css, js]
  return [
    text,
    "h1 { color: red; }",
    "console.log(1)"
  ];
};
</script>
```

## ఇంటర్ఫేస్ పారామితులు

### లక్షణాలు మరియు పద్ధతులు

- `.gen`: ఇంజెక్ట్ చేయబడిన కాల్‌బ్యాక్ ఫంక్షన్. **డీబగ్ బార్ ప్రదర్శించబడాలంటే ఈ కాల్‌బ్యాక్ తప్పనిసరిగా ఇంజెక్ట్ చేయబడాలి. **
  - పరామితి:
    - `type`: ఇంజెక్ట్ చేయబడిన కంటెంట్ రకం (`HTML = 1`, `JS = 2`).
    - `text`: చైల్డ్ ఎలిమెంట్ నుండి సంగ్రహించబడిన కోడ్ టెక్స్ట్.
  - రిటర్న్ విలువ: `[htm, css, js]`
