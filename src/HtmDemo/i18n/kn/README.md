# HTML ಮತ್ತು JS ಕೋಡ್ ಅನ್ನು ಆನ್‌ಲೈನ್‌ನಲ್ಲಿ ಡೀಬಗ್ ಮಾಡಲಾಗುತ್ತಿದೆ

- ಕೋಡ್‌ನ ಕೆಳಗೆ ಡೀಬಗ್ ಬಾರ್ ಅನ್ನು ರೆಂಡರ್ ಮಾಡಿ ಮತ್ತು ಕೋಡ್‌ಪೆನ್ ಮತ್ತು ಜೆಎಸ್‌ಫಿಡ್ಲ್ ಡೀಬಗ್ ಮಾಡುವ ಪೋರ್ಟಲ್‌ಗಳನ್ನು ಒದಗಿಸಿ
- ಆಯ್ಕೆಮಾಡಿದ ಡೀಬಗ್ ಮಾಡುವ ವೇದಿಕೆಯನ್ನು ಸ್ವಯಂಚಾಲಿತವಾಗಿ ನೆನಪಿಸಿಕೊಳ್ಳಿ
- HTML, CSS ಮತ್ತು JS ಕೋಡ್ ಅನ್ನು ಕ್ರಿಯಾತ್ಮಕವಾಗಿ ಉತ್ಪಾದಿಸಲು `.gen` ಗುಣಲಕ್ಷಣದ ಮೂಲಕ ಕಾಲ್‌ಬ್ಯಾಕ್‌ಗಳನ್ನು ಇಂಜೆಕ್ಟ್ ಮಾಡುವುದನ್ನು ಬೆಂಬಲಿಸುತ್ತದೆ
- `language-js` ವರ್ಗದ ಹೆಸರಿನೊಂದಿಗೆ ಮಕ್ಕಳ ಅಂಶಗಳನ್ನು ಸ್ವಯಂಚಾಲಿತವಾಗಿ ಗುರುತಿಸಿ ಮತ್ತು JS ಮೋಡ್‌ನಲ್ಲಿ ರನ್ ಮಾಡಿ

## ಡೆಮೊ ಬಳಸಿ

```html
<c-htm-demo><pre><code class="language-html">
<h1>ಡೆಮೊ</h1>
</code></pre></c-htm-demo>

<script type="module">
import "webc.site/HtmDemo/_.js";

const el = document.querySelector("c-htm-demo");
el.gen = (type, text) => {
  // ಹಿಂತಿರುಗಿ [html, css, js]
  return [
    text,
    "h1 { color: red; }",
    "console.log(1)"
  ];
};
</script>
```

## ಇಂಟರ್ಫೇಸ್ ನಿಯತಾಂಕಗಳು

### ಗುಣಲಕ್ಷಣಗಳು ಮತ್ತು ವಿಧಾನಗಳು

- `.gen`: ಇಂಜೆಕ್ಟ್ ಮಾಡಲಾದ ಕಾಲ್‌ಬ್ಯಾಕ್ ಕಾರ್ಯ. ** ಡೀಬಗ್ ಬಾರ್ ಅನ್ನು ಪ್ರದರ್ಶಿಸಲು ಈ ಕಾಲ್‌ಬ್ಯಾಕ್ ಅನ್ನು ಇಂಜೆಕ್ಟ್ ಮಾಡಬೇಕು. **
  - ನಿಯತಾಂಕ:
    - `type`: ಇಂಜೆಕ್ಟ್ ಮಾಡಲಾದ ವಿಷಯ ಪ್ರಕಾರ (`HTML = 1`, `JS = 2`).
    - `text`: ಚೈಲ್ಡ್ ಎಲಿಮೆಂಟ್‌ನಿಂದ ಹೊರತೆಗೆಯಲಾದ ಕೋಡ್ ಪಠ್ಯ.
  - ಮೌಲ್ಯವನ್ನು ಹಿಂತಿರುಗಿಸಿ: `[htm, css, js]`
