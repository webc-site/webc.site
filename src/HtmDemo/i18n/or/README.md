# ଅନଲାଇନରେ HTML ଏବଂ JS କୋଡ୍ ଡିବଗ୍ କରିବା |

- କୋଡ୍ ତଳେ ଏକ ତ୍ରୁଟି ନିବାରଣ ଦଣ୍ଡ ଦିଅନ୍ତୁ ଏବଂ CodePen ଏବଂ JSFiddle ତ୍ରୁଟି ନିବାରଣ ପୋର୍ଟାଲଗୁଡିକ ପ୍ରଦାନ କରନ୍ତୁ |
- ମନୋନୀତ ତ୍ରୁଟି ନିବାରଣ ପ୍ଲାଟଫର୍ମକୁ ସ୍ୱୟଂଚାଳିତ ଭାବରେ ମନେରଖ |
- HTML, CSS ଏବଂ JS କୋଡ୍ ଗତିଶୀଳ ଭାବରେ ସୃଷ୍ଟି କରିବାକୁ `.gen` ଆଟ୍ରିବ୍ୟୁଟ୍ ମାଧ୍ୟମରେ କଲବ୍ୟାକ୍ ଇଞ୍ଜେକ୍ସନ୍ କୁ ସମର୍ଥନ କରେ |
- `language-js` ଶ୍ରେଣୀ ନାମ ସହିତ ଶିଶୁ ଉପାଦାନଗୁଡ଼ିକୁ ସ୍ୱୟଂଚାଳିତ ଭାବରେ ଚିହ୍ନଟ କରନ୍ତୁ ଏବଂ JS ମୋଡରେ ଚଲାନ୍ତୁ |

## ଡେମୋ ବ୍ୟବହାର କରନ୍ତୁ |

```html
<c-htm-demo><pre><code class="language-html">
<h1>ଡେମୋ |</h1>
</code></pre></c-htm-demo>

<script type="module">
import "webc.site/HtmDemo/_.js";

const el = document.querySelector("c-htm-demo");
el.gen = (type, text) => {
  // ଫେରସ୍ତ [html, css, js]
  return [
    text,
    "h1 { color: red; }",
    "console.log(1)"
  ];
};
</script>
```

## ଇଣ୍ଟରଫେସ୍ ପାରାମିଟରଗୁଡିକ

### ଗୁଣ ଏବଂ ପଦ୍ଧତି |

- `.gen`: ଇଞ୍ଜେକ୍ସନ୍ କଲବ୍ୟାକ୍ ଫଙ୍କସନ୍ | ** ଡିବଗ୍ ବାର୍ ପ୍ରଦର୍ଶିତ ହେବା ପାଇଁ ଏହି କଲବ୍ୟାକ୍ ଇଞ୍ଜେକ୍ସନ ଦେବା ଜରୁରୀ | **
  - ପାରାମିଟର:
    - `type`: ଇଞ୍ଜେକ୍ସନ ଦିଆଯାଇଥିବା ବିଷୟବସ୍ତୁ ପ୍ରକାର (`HTML = 1`, `JS = 2`) |
    - `text`: ଶିଶୁ ଉପାଦାନରୁ ବାହାର କରାଯାଇଥିବା କୋଡ୍ ପାଠ୍ୟ |
  - ଫେରସ୍ତ ମୂଲ୍ୟ: `[htm, css, js]` |
