# Отклањање грешака у ХТМЛ и ЈС коду на мрежи

- Рендерирајте траку за отклањање грешака испод кода и обезбедите портале за отклањање грешака ЦодеПен и ЈСФиддле
- Аутоматски запамтите изабрану платформу за отклањање грешака
- Подржава убацивање повратних позива преко атрибута <б ид="1"> ради динамичког генерисања ХТМЛ, ЦСС и ЈС кода
- Аутоматски идентификујте подређене елементе са <б ид="1"> именом класе и покрените у ЈС режиму

## Користите демо

```html
<c-htm-demo><pre><code class="language-html">
<h1>Демо</h1>
</code></pre></c-htm-demo>

<script type="module">
import "webc.site/HtmDemo/_.js";

const el = document.querySelector("c-htm-demo");
el.gen = (type, text) => {
  // Врати [хтмл, цсс, јс]
  return [
    text,
    "h1 { color: red; }",
    "console.log(1)"
  ];
};
</script>
```

## Параметри интерфејса

### Особине и методе

- <б ид="1">: Ињектирана функција повратног позива. **Овај повратни позив мора бити уведен да би се приказала трака за отклањање грешака. **
  - параметар:
    - <б ид="1">: Тип убаченог садржаја (<б ид="2">, <б ид="3">).
    - <б ид="1">: Текст кода екстрахован из подређеног елемента.
  - Повратна вредност: <б ид="1">
