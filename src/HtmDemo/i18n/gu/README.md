# HTML અને JS કોડને ઑનલાઇન ડિબગ કરવું

- કોડની નીચે ડીબગીંગ બાર રેન્ડર કરો અને CodePen અને JSFiddle ડીબગીંગ પોર્ટલ પ્રદાન કરો
- પસંદ કરેલ ડીબગીંગ પ્લેટફોર્મ આપમેળે યાદ રાખો
- HTML, CSS અને JS કોડને ગતિશીલ રીતે જનરેટ કરવા માટે `.gen` એટ્રિબ્યુટ દ્વારા ઇન્જેક્શન કૉલબેક્સને સપોર્ટ કરે છે
- `language-js` વર્ગના નામ સાથે ચાઇલ્ડ એલિમેન્ટ્સને આપમેળે ઓળખો અને JS મોડમાં ચલાવો

## ડેમોનો ઉપયોગ કરો

```html
<c-htm-demo><pre><code class="language-html">
<h1>ડેમો</h1>
</code></pre></c-htm-demo>

<script type="module">
import "webc.site/HtmDemo/_.js";

const el = document.querySelector("c-htm-demo");
el.gen = (type, text) => {
  // વળતર [html, css, js]
  return [
    text,
    "h1 { color: red; }",
    "console.log(1)"
  ];
};
</script>
```

## ઇન્ટરફેસ પરિમાણો

### ગુણધર્મો અને પદ્ધતિઓ

- `.gen`: ઇન્જેક્ટેડ કૉલબેક કાર્ય. **ડિબગ બારને પ્રદર્શિત કરવા માટે આ કૉલબેક ઇન્જેક્ટેડ હોવું આવશ્યક છે. **
  - પરિમાણ:
    - `type`: ઇન્જેક્ટ કરેલ સામગ્રી પ્રકાર (`HTML = 1`, `JS = 2`).
    - `text`: ચાઇલ્ડ એલિમેન્ટમાંથી કાઢવામાં આવેલ કોડ ટેક્સ્ટ.
  - વળતર મૂલ્ય: `[htm, css, js]`
