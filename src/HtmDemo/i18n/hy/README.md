# HTML և JS կոդի վրիպազերծում առցանց

- Ստեղծեք վրիպազերծման տող կոդի տակ և տրամադրեք CodePen և JSFiddle վրիպազերծման պորտալներ
- Ավտոմատ կերպով հիշեք ընտրված վրիպազերծման հարթակը
- Աջակցում է հետադարձ զանգերի ներարկումը `.gen` հատկանիշի միջոցով՝ դինամիկ HTML, CSS և JS կոդ ստեղծելու համար:
- Ավտոմատ կերպով նույնականացրեք մանկական տարրերը `language-js` դասի անունով և գործարկեք JS ռեժիմում

## Օգտագործեք ցուցադրությունը

```html
<c-htm-demo><pre><code class="language-html">
<h1>Դեմո</h1>
</code></pre></c-htm-demo>

<script type="module">
import "webc.site/HtmDemo.js";

const el = document.querySelector("c-htm-demo");
el.gen = (type, text) => {
  // Վերադարձեք [html, css, js]
  return [
    text,
    "h1 { color: red; }",
    "console.log(1)"
  ];
};
</script>
```

## Ինտերֆեյսի պարամետրեր

### Հատկություններ և մեթոդներ

- `.gen`. ներարկված հետ կանչի գործառույթ: **Այս հետ կանչը պետք է ներարկվի, որպեսզի վրիպազերծման տողը ցուցադրվի: **
  - պարամետր:
    - `type`. Ներարկված բովանդակության տեսակ (`HTML = 1`, `JS = 2`):
    - `text`. կոդի տեքստը հանված է երեխայի տարրից:
  - Վերադարձի արժեքը՝ `[htm, css, js]`
