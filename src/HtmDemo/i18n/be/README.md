# Адладка кода HTML і JS онлайн

- Рэндэрынг панэлі адладкі пад кодам і прадастаўленне парталаў адладкі CodePen і JSFiddle
- Аўтаматычна запамінаць выбраную платформу адладкі
- Падтрымлівае ўвядзенне зваротных выклікаў праз атрыбут `.gen` для дынамічнай генерацыі кода HTML, CSS і JS
- Аўтаматычна вызначаць даччыныя элементы з назвай класа `language-js` і запускаць у рэжыме JS

## Выкарыстоўвайце дэма

```html
<c-htm-demo><pre><code class="language-html">
<h1>Дэманстрацыя</h1>
</code></pre></c-htm-demo>

<script type="module">
import "webc.site/HtmDemo.js";

const el = document.querySelector("c-htm-demo");
el.gen = (type, text) => {
  // Вярнуць [html, css, js]
  return [
    text,
    "h1 { color: red; }",
    "console.log(1)"
  ];
};
</script>
```

## Параметры інтэрфейсу

### Уласцівасці і метады

- `.gen`: уведзеная функцыя зваротнага выкліку. **Гэты зваротны выклік павінен быць уведзены для адлюстравання панэлі адладкі. **
  - параметр:
    - `type`: Тып уведзенага кантэнту (`HTML = 1`, `JS = 2`).
    - `text`: тэкст кода, выняты з даччынага элемента.
  - Вяртанае значэнне: `[htm, css, js]`
