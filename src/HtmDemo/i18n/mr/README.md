# HTML आणि JS कोड ऑनलाइन डीबग करणे

- कोडच्या खाली डीबगिंग बार रेंडर करा आणि CodePen आणि JSFiddle डीबगिंग पोर्टल प्रदान करा
- निवडलेले डीबगिंग प्लॅटफॉर्म स्वयंचलितपणे लक्षात ठेवा
- HTML, CSS आणि JS कोड डायनॅमिकपणे व्युत्पन्न करण्यासाठी `.gen` विशेषताद्वारे कॉलबॅक इंजेक्ट करण्यास समर्थन देते
- `language-js` वर्ग नावाने मूल घटक आपोआप ओळखा आणि JS मोडमध्ये चालवा

## डेमो वापरा

```html
<c-htm-demo><pre><code class="language-html">
<h1>डेमो</h1>
</code></pre></c-htm-demo>

<script type="module">
import "webc.site/HtmDemo.js";

const el = document.querySelector("c-htm-demo");
el.gen = (type, text) => {
  // रिटर्न [html, css, js]
  return [
    text,
    "h1 { color: red; }",
    "console.log(1)"
  ];
};
</script>
```

## इंटरफेस पॅरामीटर्स

### गुणधर्म आणि पद्धती

- `.gen`: इंजेक्ट केलेले कॉलबॅक फंक्शन. **डीबग बार प्रदर्शित होण्यासाठी हा कॉलबॅक इंजेक्ट करणे आवश्यक आहे. **
  - पॅरामीटर:
    - `type`: इंजेक्ट केलेला सामग्री प्रकार (`HTML = 1`, `JS = 2`).
    - `text`: मूल घटकातून काढलेला कोड मजकूर.
  - परतावा मूल्य: `[htm, css, js]`
