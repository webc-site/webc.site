# Εντοπισμός σφαλμάτων HTML και κώδικα JS στο διαδίκτυο

- Αποδώστε μια γραμμή εντοπισμού σφαλμάτων κάτω από τον κώδικα και παρέχετε πύλες εντοπισμού σφαλμάτων CodePen και JSFiddle
- Αυτόματη απομνημόνευση της επιλεγμένης πλατφόρμας εντοπισμού σφαλμάτων
- Υποστηρίζει την εισαγωγή επανάκλησης μέσω του χαρακτηριστικού `.gen` για τη δυναμική δημιουργία κώδικα HTML, CSS και JS
- Αυτόματη αναγνώριση θυγατρικών στοιχείων με `language-js` όνομα κλάσης και εκτέλεση σε λειτουργία JS

## Χρησιμοποιήστε το demo

```html
<c-htm-demo><pre><code class="language-html">
<h1>Διαδήλωση</h1>
</code></pre></c-htm-demo>

<script type="module">
import "webc.site/HtmDemo/_.js";

const el = document.querySelector("c-htm-demo");
el.gen = (type, text) => {
  // Επιστροφή [html, css, js]
  return [
    text,
    "h1 { color: red; }",
    "console.log(1)"
  ];
};
</script>
```

## Παράμετροι διεπαφής

### Ιδιότητες και μέθοδοι

- `.gen`: Λειτουργία επανάκλησης με έγχυση. **Αυτή η επανάκληση πρέπει να εγχυθεί για να εμφανιστεί η γραμμή εντοπισμού σφαλμάτων. **
  - παράμετρος:
    - `type`: Τύπος περιεχομένου με ένεση (`HTML = 1`, `JS = 2`).
    - `text`: Το κείμενο κώδικα που εξήχθη από το θυγατρικό στοιχείο.
  - Επιστρεφόμενη τιμή: `[htm, css, js]`
