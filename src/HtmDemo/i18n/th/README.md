# การดีบักโค้ด HTML และ JS ออนไลน์

- เรนเดอร์แถบการดีบักด้านล่างโค้ด และจัดเตรียมพอร์ทัลการดีบัก CodePen และ JSFiddle
- จดจำแพลตฟอร์มการดีบักที่เลือกโดยอัตโนมัติ
- รองรับการแทรกการโทรกลับผ่านแอตทริบิวต์ `.gen` เพื่อสร้างโค้ด HTML, CSS และ JS แบบไดนามิก
- ระบุองค์ประกอบลูกโดยอัตโนมัติด้วยชื่อคลาส `language-js` และทำงานในโหมด JS

## ใช้การสาธิต

```html
<c-htm-demo><pre><code class="language-html">
<h1>สาธิต</h1>
</code></pre></c-htm-demo>

<script type="module">
import "webc.site/HtmDemo/_.js";

const el = document.querySelector("c-htm-demo");
el.gen = (type, text) => {
  // กลับ [html, css, js]
  return [
    text,
    "h1 { color: red; }",
    "console.log(1)"
  ];
};
</script>
```

## พารามิเตอร์อินเทอร์เฟซ

### คุณสมบัติและวิธีการ

- `.gen`: ฟังก์ชันการโทรกลับที่ถูกแทรก ** จะต้องฉีดการเรียกกลับนี้เพื่อให้แถบดีบั๊กปรากฏขึ้น **
  - พารามิเตอร์:
    - `type`: ประเภทเนื้อหาที่แทรก (`HTML = 1`, `JS = 2`)
    - `text`: ข้อความโค้ดที่ดึงมาจากองค์ประกอบลูก
  - ส่งคืนค่า: `[htm, css, js]`
