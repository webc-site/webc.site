# Дебагирање на HTML и JS код онлајн

- Направете лента за дебагирање под кодот и обезбедете портали за дебагирање CodePen и JSFiddle
- Автоматски запомнете ја избраната платформа за дебагирање
- Поддржува инјектирање повратни повици преку атрибутот `.gen` за динамичко генерирање HTML, CSS и JS код
- Автоматски идентификувајте детски елементи со `language-js` име на класа и стартувајте во режим JS

## Користете го демо

```html
<c-htm-demo><pre><code class="language-html">
<h1>Демо</h1>
</code></pre></c-htm-demo>

<script type="module">
import "webc.site/HtmDemo/_.js";

const el = document.querySelector("c-htm-demo");
el.gen = (type, text) => {
  // Врати [html, css, js]
  return [
    text,
    "h1 { color: red; }",
    "console.log(1)"
  ];
};
</script>
```

## Параметри на интерфејсот

### Својства и методи

- `.gen`: Инјектирана функција за повратен повик. **Овој повратен повик мора да се инјектира за да се прикаже лентата за отстранување грешки. **
  - параметар:
    - `type`: тип на инјектирана содржина (`HTML = 1`, `JS = 2`).
    - `text`: Текстот на кодот извлечен од детскиот елемент.
  - Повратна вредност: `[htm, css, js]`
