# HTML და JS კოდის გამართვა ონლაინ

- გამართეთ გამართვის ზოლი კოდის ქვემოთ და მიაწოდეთ CodePen და JSFiddle გამართვის პორტალები
- ავტომატურად დამახსოვრება შერჩეული გამართვის პლატფორმა
- მხარს უჭერს გამოძახების ინექციას `.gen` ატრიბუტის მეშვეობით HTML, CSS და JS კოდების დინამიურად გენერირებისთვის
- ავტომატურად იდენტიფიცირება ბავშვის ელემენტების `language-js` კლასის სახელით და გაშვება JS რეჟიმში

## გამოიყენეთ დემო ვერსია

```html
<c-htm-demo><pre><code class="language-html">
<h1>დემო</h1>
</code></pre></c-htm-demo>

<script type="module">
import "webc.site/HtmDemo/_.js";

const el = document.querySelector("c-htm-demo");
el.gen = (type, text) => {
  // დაბრუნება [html, css, js]
  return [
    text,
    "h1 { color: red; }",
    "console.log(1)"
  ];
};
</script>
```

## ინტერფეისის პარამეტრები

### თვისებები და მეთოდები

- `.gen`: ინექციური გამოძახების ფუნქცია. **ეს გამოძახება უნდა იყოს შეყვანილი, რათა გამოჩნდეს გამართვის ზოლი. **
  - პარამეტრი:
    - `type`: ინექციური კონტენტის ტიპი (`HTML = 1`, `JS = 2`).
    - `text`: კოდის ტექსტი ამოღებულია ბავშვის ელემენტიდან.
  - დაბრუნების მნიშვნელობა: `[htm, css, js]`
