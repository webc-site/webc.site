# HTML और JS कोड को ऑनलाइन डिबग करना

- कोड के नीचे एक डिबगिंग बार रेंडर करें और CodePen और JSFiddle डिबगिंग पोर्टल प्रदान करें
- चयनित डिबगिंग प्लेटफ़ॉर्म को स्वचालित रूप से याद रखें
- HTML, CSS और JS कोड को गतिशील रूप से उत्पन्न करने के लिए `.gen` विशेषता के माध्यम से कॉलबैक इंजेक्ट करने का समर्थन करता है।
- `language-js` वर्ग नाम के साथ बाल तत्वों को स्वचालित रूप से पहचानें और जेएस मोड में चलाएं

## डेमो का प्रयोग करें

```html
<c-htm-demo><pre><code class="language-html">
<h1>डेमो</h1>
</code></pre></c-htm-demo>

<script type="module">
import "webc.site/HtmDemo/_.js";

const el = document.querySelector("c-htm-demo");
el.gen = (type, text) => {
  // वापसी [एचटीएमएल, सीएसएस, जेएस]
  return [
    text,
    "h1 { color: red; }",
    "console.log(1)"
  ];
};
</script>
```

## इंटरफ़ेस पैरामीटर

### गुण और विधियाँ

- `.gen`: इंजेक्टेड कॉलबैक फ़ंक्शन। **डिबग बार प्रदर्शित करने के लिए इस कॉलबैक को इंजेक्ट किया जाना चाहिए। **
  - पैरामीटर:
    - `type`: इंजेक्ट की गई सामग्री प्रकार (`HTML = 1`, `JS = 2`).
    - `text`: चाइल्ड एलिमेंट से निकाला गया कोड टेक्स्ट।
  - वापसी मान: `[htm, css, js]`
