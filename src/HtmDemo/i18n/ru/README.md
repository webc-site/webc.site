# Отладка HTML и JS-кода онлайн

- Отобразите панель отладки под кодом и предоставьте порталы отладки CodePen и JSFiddle.
- Автоматически запоминать выбранную платформу отладки
- Поддерживает внедрение обратных вызовов через атрибут `.gen` для динамического создания кода HTML, CSS и JS.
- Автоматически определять дочерние элементы по имени класса `language-js` и запускать их в режиме JS.

## Используйте демо-версию

```html
<c-htm-demo><pre><code class="language-html">
<h1>Демо</h1>
</code></pre></c-htm-demo>

<script type="module">
import "webc.site/HtmDemo/_.js";

const el = document.querySelector("c-htm-demo");
el.gen = (type, text) => {
  // Возвращаем [html, css, js]
  return [
    text,
    "h1 { color: red; }",
    "console.log(1)"
  ];
};
</script>
```

## Параметры интерфейса

### Свойства и методы

- `.gen`: внедренная функция обратного вызова. **Этот обратный вызов необходимо внедрить для отображения панели отладки. **
  - параметр:
    - `type`: тип внедренного контента (`HTML = 1`, `JS = 2`).
    - `text`: текст кода, извлеченный из дочернего элемента.
  - Возвращаемое значение: `[htm, css, js]`
