# Налагодження коду HTML і JS онлайн

- Відобразіть панель налагодження під кодом і надайте портали налагодження CodePen і JSFiddle
- Автоматично запам'ятовувати вибрану платформу налагодження
- Підтримує введення зворотних викликів через атрибут `.gen` для динамічного створення коду HTML, CSS і JS
- Автоматично ідентифікувати дочірні елементи з назвою класу `language-js` і запускати в режимі JS

## Використовуйте демо

```html
<c-htm-demo><pre><code class="language-html">
<h1>Демо</h1>
</code></pre></c-htm-demo>

<script type="module">
import "webc.site/HtmDemo/_.js";

const el = document.querySelector("c-htm-demo");
el.gen = (type, text) => {
  // Повернення [html, css, js]
  return [
    text,
    "h1 { color: red; }",
    "console.log(1)"
  ];
};
</script>
```

## Параметри інтерфейсу

### Властивості та методи

- `.gen`: вбудована функція зворотного виклику. **Цей зворотний виклик має бути впроваджено, щоб відобразилася панель налагодження. **
  - параметр:
    - `type`: тип введеного вмісту (`HTML = 1`, `JS = 2`).
    - `text`: текст коду, отриманий із дочірнього елемента.
  - Повернене значення: `[htm, css, js]`
